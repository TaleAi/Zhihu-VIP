## 简介
本专栏由孙歌教授主讲，重新讲解明治维新的历史，解除对明治维新的误解，从另一个角度解读日本的发展。


  



**明治维新**，对于中国与日本来说，是截然不同的。


在我们中国人看来，日本开始超越中国，是从明治维新开始的。


在此之前，中国是文化悠久、经济繁荣的宗主国，


在此之后，日本摇身一变成了发达国家，将中国远远甩在后面。


然而在**明治维新 150 周年的今年**，


日本却举办了万人集会，公开批判明治维新。


  



明治维新真的仅仅是开启日本现代化的钥匙吗？


**着名日本思想史学者孙歌的答案是：并不。**


 


150 年前，这场标志着日本现代化的变革发生了，


但是结果是使日本失去了主权独立，


也使亚洲邻国以及日本国人，


陷入了水深火热的境地。


那么，明治维新究竟有什么意义？


**明治维新，就是读懂现代日本的起点。**


**打破历史窠臼，就是照见我们的未来。**


  



孙歌教授的观点深刻而犀利


既从宏观视角俯视历史


又在细节中照见真实


**她将在课程中告诉我们**：


 


**·我们对明治维新有什么误解？**


明治维新并不是值得羡慕的现代化方式


**·关于明治维新的想象，忽略了什么？**


维新的成功给了日本盲目自信，开始侵略战争


**·今天为什么要重新提起明治维新？**


扩张和侵略并不可取，和平发展才是王道


  



你将获得：
-----


**刷新**你看待历史的眼光：如果明治维新失败了，日本会更好？


**颠覆**你理解世界的角度：日本竟然不是一个主权独立的国家？


**突破**你观察问题的深度：为什么冲绳万人集会批判明治维新？


**启发**你思考自身的契机：怎么创造属于中国自己的现代化国家？
