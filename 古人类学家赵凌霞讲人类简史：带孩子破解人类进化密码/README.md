## 简介
人类是女娲造出来的吗？最早的人类长什么样？为什么说直立人的出现是人类进化的新起点？……本专栏特邀中国科学院古脊椎动物与古人类研究所研究员、古人类学家赵凌霞老师，为孩子们讲述人类演化的历史，带领孩子们了解人类演化的科学奥秘，引导孩子们对于生命历史的兴趣。


人类是女娲造出来的吗？最早的人类长什么样？


早期智人的生活是什么样子的？为什么说直立人的出现是人类进化的新起点？


……


我们每个人都拥有着强烈的好奇心和求知欲。回想一下，其实在很小的时候，每个人都会像科学家一样在追问各种问题。


恐怕你也一定问过你的父母：「我是从哪里来的？」


这是一个特别有意思的问题，无论小孩子还是科学家，都想知道它的答案。而古人类学研究，就是在试着回答这个问题：「人类，究竟是如何而来的？」


孩童时期的好奇心是很宝贵的，希望我们都能保护好它。不管孩子们提了什么问题，希望我们作为家长，都能好好聆听，引导孩子思考，和孩子一起探索问题，培养孩子对事物的兴趣和观察力。


本专栏特邀中国科学院古脊椎动物与古人类研究所研究员、古人类学家赵凌霞老师，为孩子们讲述人类演化的历史，带领孩子们了解人类演化的科学奥秘，引导孩子们对于生命历史的兴趣。
