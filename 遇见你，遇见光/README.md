## 简介
「太好哭」的治愈救赎甜文合集。本专栏包含 6 个男女主相互治愈救赎的言情故事。何其有幸，遇见了你，也遇见了光。


**《席心为木》**


一句话简介：他可以为她做任何事，唯独不敢爱上她。


席业接我回家那年，我只有十四岁，我妈妈刚死于一场车祸。他举着一把黑伞站在派出所门口，问我：「你愿不愿意跟我回家。」而他是那个传闻中我妈妈养在外面的男艺人。只是妈妈喜欢的男人，没想到我也会喜欢。


后来我进了娱乐圈，和当红偶像传绯闻，跟流量明星拍吻戏，都是希望他会吃醋，但他表现如常，默默推拒。直到那次我的吻戏，导演说不能借位，席业当着全剧组的面带走我。他将我逼在墙角，声音隐忍而低沉：「木木，我会疯的。」


原来这段感情里，不只我一人沉沦。


**《吱吱》**


一句话简介：他活在黑暗里，她却带着光闯进来。


被邪教洗脑的柳吱吱，一心想为民除害杀掉暴政狗皇帝。进宫前听说皇帝喜欢太监，师兄就塞她进宫假扮太监。可是狗皇帝简直是个绯闻制造机！她还什么都没做，宫里就传遍了狗皇帝宠爱她的各种谣言。救命，他们真的什么都没有发生啊！


就在柳吱吱想要动手时，却发现狗皇帝其实不像教主说的那样暴政，甚至让人有些心疼，她突然就不想杀他了。后来，成为皇后的柳吱吱逛御花园，听见假山后有两人唠嗑，「听说如今的皇后娘娘，就是因为长得像陛下当初宠幸的那个小公公，才被陛下瞧上的……」


好的，又一轮谣言开始了。
