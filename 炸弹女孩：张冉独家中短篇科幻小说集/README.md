## 简介
中国当代知名科幻作家，科学与幻想成长基金发起人——张冉首个独家在线专栏上线知乎！本专栏用 11 篇故事涵盖了非常丰富的科幻命题，风格跃动多变，包括中美合拍电影《赵师傅》故事原文，以及首次发表的中篇故事《最后的喜马拉雅》等。探讨了科幻背景下，平凡普通人的情感，张冉独特的人文、青春的浪漫情怀贯穿始终，令人动容！


**《赵师傅》**


每天准点给我送外卖的赵师傅，似乎有未卜先知的本领，


后来他告诉我，他的真实身份是一位时间旅行者。


在妻子被绝症折磨的那一年，他出了车祸，


醒来后却拥有了这种在自己命运的无数条支线中穿梭的能力……


使得他原本的生命被无限地拉长，


使得他无时无刻不在面对着，漫长岁月模糊掉真实和虚幻的恐惧。


他的内心被迫长满年轮，


他是一位活在自己世界里的长者。


  



**《回家》**


世界毁灭三年，老罗父子已习以为常。


老罗是个在维加斯工作的中餐馆厨子，末日来临的第二年，他就再没见过其他活人。


他到处找吃的，收集物资，劈柴烧水，煮饭熬汤，养活儿子小罗。


这天老罗撬开中国超市的门，找着本几年前的日历，上面印着中国字。


后来，老罗对小罗说：「我们回家嘛，回家过年。」


农历八月十六，他们开车踏上生死不明的归乡之路……


  



**《最后的喜马拉雅》**


2050 年全球地震频发，全球性季风成为灾难的源头。


联合国决定用核弹炸掉几座关键山脉，以平缓季风对各个板块的冲击，


喜马拉雅山脉正是其中之一。


前提是，对藏传佛教有着重要意义的 34 座寺庙、8 座神山，以及 3 个圣湖必须整体迁移。


像古老的神话故事里那样，以人力搬山移海？


没人有把握自己能做到，但没人选择退缩。


岩浆在地底沸腾，雪山向下沉降，八千万年前的地壳运动正在逆向重演。


印度洋季风掠过灼热的山脉，化为瓢泼大雨，蒸汽柱升上七千米高空。


这是最后的喜马拉雅，这是崭新的未来。
