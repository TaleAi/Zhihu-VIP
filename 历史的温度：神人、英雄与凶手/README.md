## 简介
每个细节都会引发历史格局的动荡


「二战」中与希特勒和墨索里尼并肩的「法西斯三巨头」东条英机，一路叫嚣，把整个日本捆绑上了疯狂自杀的战车，而在日本战败后，作为甲级战犯的东条英机，却忽然失去了当初所谓的「勇气」……


提到聂耳，可能只有一句话：「音乐家，国歌的作者」。但他究竟是怎样一个人？他为什么在 23 岁的年纪就去世了？他的死亡背后有没有什么阴谋？


尼古拉·特斯拉是现代交流电系统的奠基人，但很多人说他的成就远不止这些，比如：说他智商超过 700，乃至 1000；说他其实已经实现了无线输电技术；说他其实早就发现了 X 射线；说他曾获得过 11 次诺贝尔奖，但他都拒绝了；说他以一己之力，创造了人类的 20 世纪，乃至 21 世纪……真相是不是这样呢？


我们常常会把医生称为「白衣天使」，因为他们悬壶济世，因为他们救死扶伤。但如果有一些人，顶着一个「医生」的头衔，做的却是连地狱恶魔都可能做不出来的事呢？


历史风云故事，有很多我们不了解的细节，看馒头大师张玮细细道来。
