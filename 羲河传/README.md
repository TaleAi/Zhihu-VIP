## 简介
国破家亡之际，身为君主理应殉国，身为公主又当如何？


羲河公主少时跋扈任性，


十六岁时国破家亡，只身带着六岁的储君在乱世中求生。


敌国少年将军宸冬，早已看穿她不是一个普通的少女，却还是爱上了她。


然而，国仇家恨岂能忘却？


她终究走上了一条满是荆棘的复仇之路。


伤痕累累之际，尚有一个纯净的少年，冲破了所有阻碍走到她身边，


拥抱了她。
