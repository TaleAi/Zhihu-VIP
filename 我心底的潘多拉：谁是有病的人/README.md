## 简介
心理病因素悬疑反转故事合集。


迷丛都市里，几乎每个人都或多或少地带着一些心理病。


有人被这些心理病自我毁灭，有人则选择去毁灭别人。


8 个心理病故事，有救赎，有复仇，也有纯粹的罪孽……在这些难以预料到结局的故事里，人性深不可测的一角也被悄然揭开。


「去另一个世界遇见你」


患有抑郁症的陈念跳楼自杀，死前听到一个陌生女孩的呼唤，那是他早在 12 年前就自杀的同桌顾盼。自杀后的陈念意识穿越回 13 年前，遇到了即将在 1 年后自杀的女孩。两人相识相恋，陈念得知了顾盼一年后自杀的真相，也知道了顾盼藏了 18 年的秘密。


「叫我钮祜禄氏」


我室友何小蕊疯了。她半夜爬上我的床，说自己是甄嬛，要对她们复仇。因为一次考试作弊，何小蕊被寝室其他四个女生一起霸凌了小半年。终于在她生日的那个周末，小蕊被那些人带走后，回来疯了。她说自己是甄嬛，甚至开始对其他几个人一一复仇。先是借着富家女温亚楠的朋友圈素材伪造了一个微博账号，将温亚楠塑造成拜金女，引起全网对她的网暴。借着利用一张 PS 的艾滋病检测报告成功吓唬住了郭莹莹和刘玲玲姐妹，让她俩反目成仇。何小蕊复仇成功，顺利专升本脱离了这个环境。


而多年后在一次同学聚会上，我才忽然意识到当年的一切，其实掩藏在另一个更大的秘密之下。
