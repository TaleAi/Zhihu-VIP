## 简介
如果只剩三个小时就要出现丧尸，你会干什么?


平凡的超市兼职大学生小庄收到了一条奇怪的短信：


「三小时后，将出现丧尸，倒计时两个小时五十九分。」


收到警报的小庄将信将疑地开始筹集物资，没想到的是，本来以为是恶作剧的警报，竟然成真了：


街道上的行人开始无规则地相互撕咬、攻击，惨叫声此起彼伏。整个城市一朝沦陷。


世界末日，真的来临了！


机缘巧合下，物资充足的小庄，和跆拳道少女、断臂退伍军人、天才宅男走到了一起。


在兵荒马乱的末世，他们如何报团取暖，找到活下去的方法？


那条警示的短信又是谁发来的？TA 又是如何得知末日就要到来的？


超市失踪的老板娘为何给小庄留下了诸多资源，她去了哪里？


……


一切，在这个专栏见分晓。
