## 简介
越是足不出户，人类内心越是渴望外面的世界。美苏两个超级大国当年投入了海量的资源，冒着巨大的风险，展开了持续几十年的太空竞赛。如今中美两国新一代载人飞船成功上天，人类似乎再次站到了探索太空的新起点上。施洋老师将从美苏冷战的角度，解读人类航天发展史。随着当时机密文件和资料的解禁，更好地复盘美苏两国当时的竞赛。


从地球走向太空，是人类文明的一大进步。飞出地球，奔向月球，漫游太空，这些梦想吸引着无数英雄竞折腰。而探索太空的源动力，来自美国、苏联两大阵营之间的意识形态之争。


与其他领域的竞争不同，太空竞赛中美苏的起点水平相当，基本都是「从零开始」；围绕着占领太空技术的制高点，双方展开了激烈的、极具冒险性的竞争，甚至会为此付出生命的代价。


但他们仍然选择前赴后继，勇往直前。最终拍板决定实施「阿波罗登月计划」的美国总统肯尼迪，在那场经典演讲中说道：「我们决定登月。我们决定在这十年间登上月球并实现更多梦想，并非它们容易，而正是因为它们困难......」正是这种面对未知充满自信的态度，才让这段历史在后世看来如此波澜壮阔。


美苏太空竞赛只是冷战中的一条线索，但顺着这条线，可以分析航天战略的成败得失，一览美苏科技竞赛的全貌。
