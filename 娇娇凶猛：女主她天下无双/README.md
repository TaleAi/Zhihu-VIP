《我在锦衣卫混日子》


相传锦衣卫的指挥使是个吃人不吐骨头的主儿。


我扮男装去当差的时候，指挥使左一句骂我娘不唧唧，右一句骂我白脸草包。


没日没夜地压榨我干活，还指鼻子瞪眼地骂我没有猪崽子勤快。


我恢复女装再去之时——


杜重云奴颜媚骨，「沈姑娘，这石榴剥皮累吧，快要不我喂你？」


「累？再累也没有您让我批十二小时文书累啊！」


「瞧你说的，我哪舍得呀！」


「呵呵。」
