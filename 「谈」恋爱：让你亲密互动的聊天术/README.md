## 简介
亲密关系，从好好聊天开始。


一边是忙碌的工作和生活，一边是无比快捷的即时通讯，让现代人与人之间的交往，更多建立在「聊天」之上。


但随着聊天重要性愈发凸显，「不会聊天」这个问题，产生的影响也越来越大，特别是在「恋爱」上。


毕竟现在的年轻人，从相识到在一起，大多数时候的情感推动，都是通过聊天来完成的——谈论自己、了解对方、沟通三观……


所以不会聊天的人，难免出现这样的困惑——


· 我只是想找喜欢的人聊天，为什么对方总觉得我很烦？


· 为什么我一张嘴，就把天给聊死了？


· 我只是想了解她，为什么总说我在「查户口」？


本专栏针对恋爱中的亲密沟通，从底层逻辑到实用技巧，帮助读者学会聊天，能够好好「谈」恋爱。
