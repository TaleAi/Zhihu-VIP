## 简介
花样年轻人的情感故事集


***网恋奔现的小甜饼：《保护我方蔡鸡》***


我这个人没什么兴趣，就喜欢玩王者荣耀。


现实中我是个刚刚上班还未转正的职场小菜鸡，但在峡谷里，我却备受尊重。


***冷面男神 X 沙雕社畜的奇妙相遇：《男友常来救火》***


前男友发朋友圈找到了新女友。


我不甘心，我喝多了。


坐最后一班地铁时，我疲惫的躺在空空荡荡的座椅上。


然后，地铁的晃动，把我的头直接送进了座椅扶手里。


卡住了……


***自作自受的渣恋故事：《一份渣女自述》***


你的大学都在干什么？是不是卧槽这个好好吃卧槽那个也好好吃。


太巧了，我也是。


可我的室友不是。


在我还在思考是去吃一食堂的麻辣拌，还是南门的烤冷面时，我的室友领证结婚了，然后三个月又离婚了，现在，她又准备二婚了。
