## 简介
离奇、诡异的都市怪谈，可能就发生在你我身边。在本专栏的 6 个故事里，你将窥见人性最深处的邪恶与善良，找到自己内心的答案。


你有没有这种经历，朝夕相处的恋人忽然变得面目可憎，爱你敬你的下属忽然捅你一刀，两肋插刀的朋友将你卖给敌人……


他们没变，只是你一直没发现，怪物潜藏在你身边。


本专栏包含了几个普通人的故事。


他们和你我一样，在不经意间卷入了匪夷所思的事件。


他们恐惧、他们懊悔。


当恶意化成实体，敲响心门，人们才能想起——过去的自己也曾有秘密。


  



《电影角色替换事件》


我有一种超能力，可以让电影中的角色和现实中的人替换，坏人都被我换掉了。可我的母亲却坚持要和我看电影？


  



《聂小倩是我的相亲对象》


我以为遇到了最喜欢的人，对方却是「亡人」。当我下定决心和这个不同寻常的女人结婚，我才发现她的真实意图。


……


  



我们都以为自己是个还不错的人，至少是个罪不至死的人。但当你想起深埋心底的那些恶意，不要忘了，自己究竟是谁。
