## 简介
互联网时代的高薪人士都有哪些特点？——能写。互联网改变了人与人之间的沟通方式，也改变了职场的需求形态。文字功底直接反映了一个人的「职业质感」，关乎职场人的「前途」与「钱途」。本门课程将从多维度场景入手，一站式解决你关于职场写作的各种困惑。


无论你是否靠写字谋生，写作都是绕不开的一项职场生存技能。面对五花八门的文字沟通/表达场景，我们难免陷入「辞穷」的境地：


社交网络：如何让自己微信群和朋友圈的文字，给人留下专业又不失风趣的印象？


正式邮件：每次重要邮件总是要修改半天，发出去后，还是觉得少了一点专业度？


汇报文书：从会议发言到会议纪要，从周报到季度总结，如何优雅地展现个人实力？


自媒体：全民自媒体的时代，突然被老板要求写公众号/发微博，什么样的文字自带传播属性？


优秀的文字表达可以帮助我们促成合作、说服他人、获得理解、树立边界。在互联网时代，优秀的文字打破了传播的边界，从而更容易被转化为赚钱的能力。


本课程已被应用于光大银行、平安人寿、恒大地产、联合利华等国内外知名企业的培训中，切实有效地帮助广大职场人士提升了自己的写作水平。
