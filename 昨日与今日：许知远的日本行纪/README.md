## 简介
用许知远的眼睛，看日本。一个冒险者、思想者、游荡者，不畏惧「冲突」，不回避「误解」，一层层拨开现代日本的迷雾，探寻真实的日本细节。无数个活色生香的时刻，转化成一个个日本的侧写。这里有无数个充满争议的问题；这里，没有所谓的「正确答案」。
