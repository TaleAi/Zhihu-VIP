## 简介

**临渊和煦**


长公主沈嘉懿是西陵大魔头，嫖男妓，捧戏子，蓄面首。她的情人们都与首辅大人季临渊生得几分相像。季临渊只把长公主当作地下情人，他要娶的是曹将军的掌上明珠。喜宴上，长公主不邀自来，艳压群芳，季临渊以为她是为了自己盛装出席，没想到她三言两语就勾搭上了新科状元安和煦。季临渊在自己的婚礼上，荒诞地控制不住对她的占有欲。沈嘉懿，不能成为别人的女人。只能是他的。


**女山鬼**


西陵幽冥山，有一个女山鬼阿芜。她凭着变化多端的模样，蛊惑仙君、魔君、人君，与她交合，借着他们至阳至纯的精气，她避过了三次天劫。她的天劫，每三万年一次，到了这第十二万年，该死的天劫又要来了。她必须出山，再去找个至阳至纯的男人，吸取精气。这次她决定对轮回后的人君陆行止下手。在她把他又一次睡了之后，正要溜之大吉的时候。陆行止反手就给她上铐了，含情脉脉说：「爱妃，乖，这回，不要跑了。」


**玄樱**


西陵朝，江氏一族权势显赫。江家嫡女江云樱十五岁入宫，与皇帝沈玄澈青梅竹马，日久生情，顺理成章成了沈玄澈的皇后。直到有一天，她的父亲卷入一宗贪污案，沈玄澈欲借机发作，对江氏一族痛下杀手，江云樱才醒悟，沈玄澈娶她，是为了夺权。她求沈玄澈放过江家。沈玄澈答应了她，却只是权宜之计。直到诞下皇子，江云樱才知道，江家人，处死的处死，流放的流放。而她，成了沈玄澈彻头彻尾的玩偶。


……


病娇来势汹汹，偏执、腹黑、占有欲强……只能成为他们的掌中之物吗？
