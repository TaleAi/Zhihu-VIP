## 简介
本课程由陶瓷文化研究者涂睿明主讲，带你走近陶瓷，解读陶器背后的历史和变迁。本课程将用一个个的小故事，将历史、文化和工艺融为一体，让听众形象感受到瓷器的诞生、瓷器的成长，瓷器的故事。通过它，去了解中华文明的发展历程，中国传统美学的真正所在。


 越地长歌不散，我翘首以盼重逢！ 


没有多少东西，可以穿越时光，她可以。 


远古沉船中的故事，新发掘墓葬中的遗物，纸张、绢帛，都已随岁月灰飞烟灭，唯有她依然留存。 


她是穿越时光，带着先辈记忆，活在现代的传统文化！ 


她是中国最有代表性的器物，唐诗宋词、明月汉关、江山万里如画，都在她的身体上栩栩如生。 


乾隆的「花式」审美，雍正的简约「性冷淡」，所有先辈的故事，我们都可以通过她去找到痕迹。


她就是 CHINA，中国瓷！   


瓷器，为何千年来经久不衰？ 


青瓷白瓷珐琅彩，制瓷工艺的变迁又记录了怎样的历史？ 


为何我们说「纵有家财万贯，不如钧瓷一片」？ 


景德镇的瓷器为什么会被称为「假玉器」？ 


…… 


瓷器的身上，我们能读出太多。 


一块瓷片，便能引出一段故事，一些知识，虽然只是一片片瓷器史的局部，但把它们拼起来，便是一部中国的文明发展史。 


本课程将用一个个的小故事，将历史、文化和工艺融为一体，让听众形象感受到瓷器的诞生、瓷器的成长，瓷器的故事。通过它，去了解中华文明的发展历程，中国传统美学的真正所在。
