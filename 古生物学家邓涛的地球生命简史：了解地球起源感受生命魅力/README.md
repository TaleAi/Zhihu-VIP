## 简介
小朋友们对我们生活的地球有强烈的好奇心，经常会听到这样的问题：地球是从哪儿来的？它一开始就是现在这个模样吗？地球上的生命从细胞开始，经历了漫长的进化历程。本专栏通过有趣的讲解，带你了解地球生物的那些事儿。


地球上的生命，从最原始的无细胞结构状态进化为有细胞结构的原核生物，从原核生物进化为真核单细胞生物，然后按照不同方向发展，出现了真菌界、植物界和动物界。


生命来自深海还是外太空？ 最早的生物什么样？


世界上第一条鱼是什么时候出现的？


你爱吃的海苔，可是植物祖先的亲戚？


海底的「大虫子」，比你见过的所有虫子都要大？


......


对于地球而言，我们人类也只是生物当中的一个平凡的生命。然而，即便是这样，地球上一个平凡的生命，也是经历了千百万年不断演化而成。


本专栏古生物学家讲解有趣的自然历史，将新近破译的化石奥秘传授给小朋友们，培养孩子对大自然的认知和好奇心，在知识的海洋中遨游，了解关于地球的生命简史。
