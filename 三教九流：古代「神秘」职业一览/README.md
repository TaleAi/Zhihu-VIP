## 简介
特定时期诞生的职业，是时代的重要剪影。


看待历史的角度，可以是宏大的，大到朝代更替、家国情怀；也可以是微观的，小到一个职业、一个从事这个职业的人。


有些职业是随着科技的进步被淘汰；有些职业是因为法制的健全而逐渐被取缔……


虽然这些职业都泯灭于时间的长河，但不能忽视的是，在职业存续期内，有那么一些人，从事于它、谋生于它，因为它而获得「生存」的口粮，也因为它遭受「命运的折磨」。


本专栏从历史中的小人物切入，用他的特殊职业经历，以及这份「职业」给他带来的命运变化，带领读者了解这些「神秘」职业背后，折射出的社会现状与历史制度——


* 为什么刽子手一生只能砍 99 个头？面对死刑犯时，他们真的能内心毫无波动吗？
* 为什么一个巫师可以操纵一个国家？「神权」和「王权」经历了哪些争斗？
* ……
