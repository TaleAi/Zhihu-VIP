# 哲学家们都干了些什么？

### 作者：林欣浩

### 分类：社科

### 状态：已完结，共 45 节

### 存档日期：2022-05-24

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1231640840251805696


## 简介
用穷凶极恶的吐槽和喜闻乐见的八卦，彻底瓦解你对哲学史的成见，
翻开任何一页，都是欲罢不能的哲学故事。


宇宙的外面是什么？


世界会不会是假的？


我是不是活在虚拟世界/别人梦中？


上帝到底存在还是不存在？


这个世界有没有最终真理？


……


你知道吗，你想过或者没想过的这些奇形怪状的问题，早在几千年前就被哲学家们一本正经地提出，挖空心思地找证据，面红耳赤地争论过了。


在整个世界忙着探索、开垦、打仗、上天、发展的漫长历史中，这帮哲学家们却神经质地死磕这个世界的真相和人生的意义，就好像一个网络游戏里的角色试图琢磨自己手里武器的代码是什么……


史上最严谨又最不严肃的哲学史，遍布穷凶极恶的吐槽和喜闻乐见的八卦。


这些人类历史上最具智慧的头脑，用他们令人不可思议、啼笑皆非的方式，


为我们提交了一张张无法判分的奇妙答卷。
