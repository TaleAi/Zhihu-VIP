## 简介
你会用什么词来形容现在的年轻人？


在外界的眼里，这届年轻人似乎是不能吃苦的一代。


然而，现实真的如此吗？


这个专栏，我们聚焦疲惫的年轻人，从三个方面入手，探讨激活新青年的实操手册。


第一部分，我们从经济学角度分析种种现状产生的原因：


* 为什么现在的年轻人不再以劳动为荣了？
* 有什么影响到了年轻人的消费习惯？

第二部分，我们根据青年一代的切身需求，从生涯规划、思维转变、理财建议等方面提供相应的建议：


* 收入一般的人怎么理财？
* 对未来一片茫然，怎么规划自己的人生？
* 还有必要买房吗？
* 如何从「穷人思维」过渡到「富人思维」？

第三部分，我们邀请心理学专业人士，真诚地告诉你：


面对如此庞大的社会压力，你真的不必把自己逼得太狠，无需太过焦虑。


愿你从中找到面对现实困境的方法和勇气。
