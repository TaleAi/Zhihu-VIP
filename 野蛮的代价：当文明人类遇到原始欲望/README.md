## 简介
在人类漫长的历史中，总有一些埋藏于历史档案尘埃之下的往事。我们除了惊叹和感慨，更可以领略揭秘事实真相的乐趣，以及思考那个人类学的终极问题：人类为何会成为今日的我们？


人类有哪些细思恐极的事？下面这些故事中，科普作家眠眠，带你重返历史现场，还原那些惊悚吊诡的瞬间。


更多惊悚瞬间，请戳[《神秘事件簿：奥兹冰人、天启大爆炸和迪亚特洛夫事件》](https://www.zhihu.com/xen/market/remix/paid_column/1215386413346488320)


飞机坠毁还有生还的可能吗？1972 年，一架民航客机跌落在安第斯山脉间，飞机上的二十多名乘客，奇迹般地存活了下来。不过他们所面临的，是比坠机更恐怖的事情。弹尽粮绝的情形下，他们最终选择割下已经死去的同伴的肉，然后吃掉……


古老的阿兹特克文明有一种残酷的人牲祭祀活动——将活人开膛，取出仍在跳动的心脏献给神明，这种臭名昭著的祭祀行为，最终令阿兹特克文明，走向了消亡。


1966 年，美国西弗吉尼亚州的五名掘墓人，发现了一个拥有人形却长着巨大翅膀的怪物从天上飞过，犹如史前巨兽。三天后，同样的地点，又有巨兽从天略过。更为诡异的是，在小镇发现巨兽整一年后，一座石桥突然断裂，数十辆车坠入水中……


了不起的居里夫人在发现镭元素的时候，一定没有料到，多年后它会被爱美的女孩涂在指甲上作荧光色甲油。紧接着，女孩骇人地惨死，牙齿迅速脱落，下颌骨像朽木一样千疮百孔，充满了大型的坏疽……


人类究竟能否死而复生？1940 年，苏联科学家为了验证这一问题，将一只实验狗的所有血液抽出，使其进入临床死亡状态，然后将其放置 10 分钟并连接到心肺机，心肺机逐渐将血液回流到动物的血液循环中。几分钟后，狗的心脏重新颤动，科学家认为自己掌握了死而复生奥秘，他声称将在人体上进行试验……
