## 简介
婚姻是条双行道，有人偏离轨道，就有人受伤害，是时候来一场酣畅淋漓的婚内反杀了。婚姻案板上，他为鱼肉，我为刀俎。


婚礼那天我也相信过爱情。


直到岁月磨穿了真情，现实暴露了人性……


婚姻战场，胜者为王。


  



你见过抠到给老婆送假护肤品的男人吗？


我老公就是，他把给其他女人用完了的 La mer 罐子装了大宝送给我。


陪他省吃俭用十年，我却看着别的女人搂着他告诉我：「La mer 掺了大宝，配你刚刚好。」


  



——《铁公鸡活不到天明》


  



你问我绝不能接受舍友做什么？


我绝不能接受那个虚荣且惹人厌的舍友，竟然想做我后妈！


  



——《同岁后妈》


  



我陪着英俊而优秀的丈夫远嫁，直到我生下了一个大脑发育不全的孩子，我的丈夫突然像是变了一个人，他对我控制欲极强，限制我的人身自由，把我囚禁在家。


上天无处，下地无门，我要逃离这个恐怖的男人，为了我，也为了我的孩子。


  



——《囚牢》


  



……


婚姻是条双行道，有人偏离轨道，就有人受伤害；


你变道给我戴帽，我就让你社死现场，气急败坏；


你害我净身出户，我就让你倾家荡产，永失所爱；


你设计阴谋诡计，我就让你渣都不剩，砍瓜切菜；


是时候来一场酣畅淋漓的婚内反杀了。


婚姻案板上，他为鱼肉，我为刀俎。
