## 29.人际沟通的理论
    一、相互作用分析理论    

    1．相互作用分析理论的内容    

    加拿大精神科医生、心理学家艾瑞克·柏恩（       Eric Berne      ）1964 年提出了人际沟通的相互作用分析理论（又称 PAC 理论）。他认为，人们在交往过程会出现三种不同的心理状态，或称自我状态。这三种自我状态分别为父母（parent，简称 P）、成年人（adult，简称 A）和儿童（child，简称 C）状态。其中，P 状态是一种权威、强制、教训的类型，讲起话来总是「你应该……」「你不能……」「你必须……」；A 状态是一种理智和客观的类型，语言上的反应是「为什么……」「何时……」「何处……」「……如何」「我个人的想法是……」「我觉得……」「能否……」等；C 状态是一种冲动任性的类型，通常的语言是「我不知道……」「我要……」「我不管……」。他认为，每个人的人格结构中都包含有这三种成分，只是各自的成分构成比重不同，并随着交往对象和环境的变化不断进行调整变化。三种类型反映了人们不同的交往态度，人际交往实际上就是人与人之间 PAC 的相互作用。当与具有不同自我状态的人进行沟通时，我们的沟通行为和社会交互方式会具有不同的表现形式。    

    （1）父母自我状态（P）。持父母自我状态的人往往具有较强的控制欲，比较保守，往往以长者自居，会照顾人，对人比较严厉。持这种自我状态人以权威和优越感为心理标志，其行为主要表现为权威式、命令式、家长式，待人处事比较主观、独断专行。    

    （2）成人自我状态（A）。持成人自我状态的人一般比较理智，心理状态成熟，实事求是，冷静客观，处理事情比较理性、慎重。待人处事公正、民主，尊重对方的意见。持这种状态的人能够客观、冷静地根据各种事实和研究数据来预测要实施事情的可行性。    

    （3）儿童自我状态（C）。儿童自我状态是指在处理问题时比较容易产生类似于儿童成长过程中出现的一些不成熟的情绪。比如对某项指示的逆反心理。持儿童自我状态的人往往比较冲动、幼稚、无主见，比较习惯于感情用事。像儿童一样希望得到他人的肯定并得到赞赏。    

    以上三种自我状态一起构成个体的性格，在每个个体的心理和行为中均有不同的表现。每个个体三种状态的比重也是不相同的，成人比重一般比较稳定，但会随着环境和交往对象的变化而有部分变化。根据三种自我状态的不同比重，人的特征可以分为以下几种。    

    （1）专制幼稚型：P 高 A 低 C 高，其行为特征是具有较强的控制欲和权利欲，有一定的决断能力，情绪波动较大，且喜欢被人吹捧。难以相处和共事。    

    （2）专制型：P 高 A 低 C 低，其行为特征是比较刻板，墨守成规，对下属十分严格。由于办事缺乏灵活性，容易犯教条主义和经验主义的错误。    

    （3）幼稚型：P 低 A 低 C 高，其行为特征是办事幼稚，在工作中总是有不切实际的幻想，不实事求是，做出的决策是理想的但是可行性低。与员工关系不错，但是工作效果较差。    

    （4）正统成人型：P 低 A 高 C 低，其行为特征是工作冷静，客观而且重事实。只关心任务的完成，较少考虑到员工的感受和需求。难以和他人共事。    

    （5）父母成人型：P 高 A 高 C 低，其行为特征是同时具备父母行为特征和成人行为特征，具备优秀的经理人的潜质。如果加以培养和指导，克服权力欲和多关心员工，能够成为优秀的经理。    

    （6）成人儿童型：P 低 A 高 C 高，其行为特征是将儿童和成人的行为特征结合在一起，儿童的行为特征是比较注重员工关系，成人的行为特征是比较关心工作。能同时处理好员工与任务的关系，是优秀的职业经理。

    2．人际沟通的行为分析    

    在人际沟通过程中，根据沟通双方所持有的自我状态，可以将沟通分为两类：平行互应性沟通（       complementary transactions      ）和交叉交错性沟通（       crossed transactions      ），具体如下。    

    （1）平行互应性沟通。如果交往的每一方均按照对方所期望的自我状态去反应，这样的沟通就是平行或互补沟通（如 AA-AA、PC-CP、AC-CA、AP-PA 等）。此类沟通是互应性的，双方关系融洽。例如上级对下级进行沟通时，上级习惯将自己定位在父母自我状态，将自己的下属定位在儿童自我状态，当进行沟通时，下级如果也将自己的自我状态定位为儿童状态，将上级定位为父母状态，那么这样的沟通就是平行的，这种关系就可以继续维持下去。最稳定持久的沟通模式是「成人-成人」（AA-AA）类型，双方都比较理性，能够获得彼此的真实想法。    

    （2）交叉交错性沟通。如果交往的另一方并不以对方期望的状态做出反应，这样的沟通就是交叉或非互补性的（如 PC-PC、CP-CP、AA-PC 等），沟通就会受阻，导致矛盾、误会与争吵。例如上级对下级进行沟通时以父母-儿童的状态进行，但是下级却以成人-成人的状态给予反应，这样的沟通就是交叉型沟通。这种沟通由于人格状态的差异，可能会影响双方的人际关系，给工作造成一定的障碍。

     二、自我呈现理论    

    1．自我呈现理论的内容    

    自我呈现理论是由美国社会学家尔文·戈夫曼（       Erving Goffman      ）于 1959 年提出的。他将社会结构比作一个舞台，人们的社会行为就是舞台表演，观众就是社会中真实存在的其他人或者是人们自己想象中的人。在表演中，人们的内心是希望没有任何约束地去做自己想做的事，但是这些事可能不是观众所期望的，甚至是恰好相反的。作为一名优秀的表演者，我们必须按照观众所期待的方式行动，所以就隐藏了真实的自我。戈夫曼用舞台前和舞台后比作理想化的自我和真实的自我，为了在舞台上完美的表演，真实的自我必须被隐藏起来，表现出来的是乔装、美化后的自我，这也叫做印象管理。印象管理主要有三种类型，具体如下。    

    （1）误导性表演。为了给观众呈现理想化的表演，表演者在进行表演时会采取部分误导性的表演。首先，表演者所要表演的内容是要积极向上的，符合社会公认的道德标准，为大家所认同；其次，表演者可以通过自己的表演掩盖自己的错误；再次，掩盖产生结果的过程；最后，试图让观众认为他们之间的关系比较亲密，以便达到自己的目的。    

    （2）神秘化表演。所谓神秘化表演就是将一些本不是秘密的东西故意隐藏起来，以引起观众的好奇心。在表演过程前后，就要防止观众与表演者交往过于亲密，以便维持这种神秘感。人们都有一种好奇心理，越是藏着掖着不让了解，就越是感兴趣想要接触了解。    

    （3）补救表演。在表演过程中可能会出现一些意外情况，导致表演失败。这时表演者在观众心目中的理想化形象就会被破坏，因此需要表演的参与者进行补救。剧班成员全身心的投入、表演者的表演涵养、小心谨慎、事先做好各种突发情况的准备以及观众和局外人提供的保护措施等都能够帮助表演顺利进行。    

    2．自我呈现理论对人际沟通分析    

    戈夫曼的自我呈现理论重视人的主观能动性，认为人际沟通是一种社会互动活动。人们在遵守社会规范的前提下，充分发挥自己的主观能动性去行动。在人际交往过程中，为了给他人留下较好的印象，人们需要考虑到他人的反应，然后来调整自己的行为，使这种沟通能够顺利地进行下去。同时，人们会借助于印象管理向他人展示有利于自己的方面，而故意隐藏一些不利于自己的一面，以达到塑造自己良好形象的目的。通过呈现自我的良好形象来引导与他人之间的交往，以控制他人对待自己的方式和态度。
