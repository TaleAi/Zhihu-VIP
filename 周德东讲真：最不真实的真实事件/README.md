## 简介
你是否经历过某些离奇的事情？之后，你跟谁说谁都不信……本次，中国悬疑教父周德东记录的这些人，他们用自己的亲身经历告诉你：你，不是一个人。


**现实中常见的事，却藏着的你永远想不通的谜题。**


**这些人，他们跟你一样……**


张安，女，26 岁，陕西汉中佛坪县人。  

「我的闺蜜被猫抓伤后，出现了奇怪的反应，之后，她好像能听懂动物说话了。」


皮曦久，男，29 岁，安徽人。  

「我在上海 CBD 从事金融业，公司所在的大厦是一栋超高层建筑，总共 87 层，高 443 米，某次电梯失控了，显示楼层 88……」


赵赣乡，男，33 岁，山东临沂郯城县人  

「前年，我坐火车出差去了一趟西安，从卫生间出来发现，车厢内竟然空无一人，外面一片漆黑，我下了车，顺着铁道往前走，最后被一堵墙挡住了去路……」
