## 简介
中国恶魔第二季，还原特大罪案令人震惊的真相。


2011 年 6 月，南半球即将入冬。


深夜的海洋下起瓢泼大雨，一艘名为鲁荣渔 2682 号的中国渔船，正随着海浪颠簸起伏。船上人们身披黑色雨衣，站在高瓦灯光下，不断将海中鱿鱼提起。


太平洋公海上这艘渔船，没人敢想会发生这样的惨案。船上 33 人，最后只剩下 11 个。


——「鲁荣渔 2682 号惨案」


1992 年，中国有个平远镇，里面的罪恶让人瞠目结舌。


在平远街只存在一种罪，那就是穷。人人高喊着「坐牢一阵子，幸福一辈子」的口号，只要能搞来钱，杀人掠货，卖枪贩毒，无恶不作！ 


——「平远街事件：犯罪天堂的覆灭」
