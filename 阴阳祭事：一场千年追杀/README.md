## 简介
这是一篇长篇灵异恐怖小说。我十八岁生日那天，小芸嫂难产，她诞下一子，却命丧黄泉。我为她守夜时，变故横生，大火烧了小芸嫂的灵堂，我身上发生了灵异事件……后来我才知道，自己是一场持续了千年的追杀的目标。


出生之夜，林生的母亲阿凉看到了父亲的尸体，悲恸难产，随后，生下了林生撒手人寰。


姑姑林三月将林生抚养长大，直到十八岁生日这天，同村的小芸嫂难产，完美复刻了十八年前林生母亲难产的那一夜。


而阴阳双子一词的出现，也彻底打破了林生平静的生活，直到姑姑死亡，林生身为阴阳双子之阳生子的身份也浮出了水面，随之而来的，就是十八年前不为人知的秘辛。


整个村子的人为了守住这个秘密，自杀的自杀，去死的去死，甚至，当着林生的面，将自己的舌头割断也不敢吐露半个字。


随后出现的神秘人更加验证了他对自己身份的猜想。


直到叶秋白的及时出现，才从伥鬼的手下将林生救出了照镜村。


只是，他从叶秋白的口中得知，姑姑还没有死。


于是，回到照镜村救出姑姑，成了林生这一路走下去的唯一目的。


而照镜山上，迷月浓雾，你永远都不知道迎面走来的是活人还是死人……
