## 简介
本专栏集合了知乎上最热门的「灵魂拷问」，我们邀请了知乎心理学领域最顶尖的心理咨询师，为我们回答生活中那些无法回避的问题。如果你是一位勇于向内探索、思考人性，渴望改变自我且富于行动力的人，那么恭喜你，你会在这些回答中找到那些难题的解决方案。


【恋情篇】


***老师你好，最近我有了一个喜欢的人，可我好像很害怕接近 ta 呀，我为什么会有这样的心态呢？***


***我有个相处了很久的男朋友，可我最近好像遇到了一个更好看、跟我兴趣更相投的男孩，请问我应该离开我的现任吗？可是他并没有做错什么……***


***...***


别着急，你的心理学导师安慰记店长和暗涌，正在前往帮你解决问题的路上！他们会从心理咨询师的角度，帮助你找到这些恋爱烦恼的根源。


【家庭篇】


***我是个 90 后，我的父母是 60 后，我觉得根本没法跟他们沟通。我听说「家不是一个讲道理的地方」，可我的问题如果不通过跟他们讲道理解决，就永远是个悬而未决的麻烦啊……***


***中国普遍存在「代际创伤」吗？产生的原因是什么？***


【绝妙的生活篇】


***最近接到公司的大任务，deadline 十万火急，请问我该如何缓解巨大的压力和焦虑感？有时真的觉得自己还不如死掉啊，就不用面对这一切了……***


***老是听说身边有人表现出「反差萌」，为什么觉得能做到「反差们」的人，都很受欢迎？***


***老师你好，请问我可以做到像电影上的 FBI 探员一样，判断一个人是否在撒谎吗！？***


*...*


别慌张，这些问题都能在暗涌和曾旻老师这里找到靠谱的解决方案。一个不行，老师给你三个！


这里有最贴心的恋爱心理学小能手，最有同理心的原生家庭问题解答，最脑洞大开的反常识心理学问题。希望能给你的人生，增添一些聪明和快乐。
