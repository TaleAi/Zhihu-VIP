## 简介
甜虐得当的古现言，脑筋不大对劲儿的男主大合集。一会是病娇，一会是霸总，一会说我可爱，一会又要「鲨」了我？水逆水逆快走开，我只想好好谈恋爱。


**《荣宠》**


李怀莳宠爱我，我入宫数夕就得封贵妃。但举宫里最鄙夷我的人，似乎也是李怀莳。事出有因，我美艳有余，脑子却不太灵。


是宫里的人常在暗地里笑我，说我无谋无智，光会耍赖，没有半分将门千金的风范。


我要的是恩宠，至于那风范，要来干什么?


**《冬姝》**


「我要你的心头血做药引，」夏渊连索求时都怀着远山寒月般的清冷，「事后我会娶你的牌位。」


闻言后我依旧保持着跪地的姿态，只是情不自禁地挪动双膝，一点点地近了夏渊的身，「二爷当真？」


**《傀儡》**


他忽地折身过来，英挺的眉眼垂下，眸光陷在我的锁骨上，「朕突然觉得就不该让苏胤舟那么轻易地就走了，留他下来，好好看我们怎么欢愉的，不更好吗？」


「皇弟，」我重重地唤他，「别让楚妃候久了。」


「都来皇姐这了，还找别人做什么？朕说了，今晚谁那都不去。」


**《区区白月光》**  

宁厌声不知是何时来的，他坐在树干上，宝光熠熠的黑衫被风撩起下摆，露出精致的靴子。


我头脑不发热的时候，心里门儿清，再修炼个五十年我都打不过他，于是——


「你杀了我一次，没杀死，我打了你几下，没打死，我们和解吧 。」我诚恳道。


「报两次仇，没有这样的规矩。」宁厌声阴恻恻地说。


两次？他记错了吧。


**《当反派来临时》**


「听说我是你的小说男主？」面容冷艳的男人支着头问，「那你为什么不敢看我？」


「误会了。你只是个反派。」


「林子澜，抬头。」男人的声线里隐隐透着威慑感。


我把头昂起来，首次认真地打量起这个在半小时前闯入我家里的人。


男人的内搭是深灰色衬衫，外套是黑色长西装，连领带都系得极有美感。


我写小说才敢这么堆砌辞藻的美貌此时真切地映在我眼帘里。


他很有霸总那味。
