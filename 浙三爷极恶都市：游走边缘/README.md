## 简介
感受钢铁丛林的生存法则，体验你不曾知晓的城市黑暗。


这一次，我会用一连串的悬疑故事，带大家走进城市黑暗面，里面既有小人物的攀爬挣扎，也有人性的温暖光芒，亦或是你从未想到的离奇故事。


人嘛，一定要感受极致的抑郁，却又死死地抱着希望。黎明是最黑暗的，可总会天亮，总会迎来曙光。
