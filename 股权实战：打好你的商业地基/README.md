## 简介
股权是创业和企业发展的利器，用得好威力无边，用错了则自伤筋脉。雷军、徐小平的私人法律顾问、中国资深创投律师张明若将通过 3 大模块 10 节课带你掌握股权的核心逻辑，针对创业和企业发展过程中的相关问题提供解决思路，带你真正从企业发展和战略的高度理解股权。


追求财富是人类永恒的主题，而其中最重要的竞争，是制度的竞争。**股权制度是公司在诞生之初就自带的力量基因**，它本身所蕴含的力量之巨大，使荷兰东印度公司横空出世，并击败了当时所有的竞争对手。


**股权制度设计，是现代组织的基础制度。它的设计是否科学合理将会在根本上影响相关各方的利益安排，影响组织的成败。**


科学合理的股权设计和分配，会成为公司发展的源动力，有效地解决关于人的问题、钱的问题、资源的问题、融资的问题、和公司控制权的问题。可以说，一个企业如果不能发挥股权的力量，就等于放弃了成功路上的最大武器。


然而，真实的商业世界中，也不乏各种股权设计失败的案例，它会导致本来蒸蒸日上的事业，轰然倒塌；它会让原本非常好的朋友，变成了不共戴天的仇人。


创业者如何设计股权结构更易吸引投资机构？企业家如何设计股权布局来确保企业的长治久安？创始人如何保护自己对公司的控制权？新的创业项目，合伙人该如何分股权？创业失败，创业者要不要去偿还投资人的投资款？回购权制度合理吗？传统企业、成熟企业要不要做股权激励?
…...


在这里，你都能找到答案！本课程针对创业者、企业家在创业、企业经营管理过程中的常见股权问题提供了针对性的解决方案，得到了雷军、徐小平、王强、黎万强、薛蛮子、牛文文、熊明华、佟大为等业界大咖的认可和支持。

人生最悲哀的事情是年轻时不懂爱情，创业时不懂股权。
