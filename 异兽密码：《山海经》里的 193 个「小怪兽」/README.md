## 简介
古人的「脑洞」能有多大？快来《山海经》里看看。在这门课中，一颉老师将带你解密《山海经》，看遍 193 种珍奇异兽，唤醒异界密码。


有一本书，它纵观山海，提笔化神。


是上古人文、自然宝典，充满了光怪陆离的想象力。


它，就是《山海经》。


在《山海经》这本书里，古老的文字记录了中国的创世史诗，其中记载了许多极具神秘感的奇异怪兽。比如：


峳峳，它长得像马，却长着四只角、羊一样的眼睛、牛一样的尾巴，声音如同狗叫。是上古凶兽，预示着天灾。


䱻鱼，它的形状与鱼相似，但长着鸟一样的翅膀，在水中出入时身体会闪闪发光。只要它一出现，天下就会有大旱灾发生。


乘黄，形体像一般的狐狸，脊背上有角，人要是骑上它就能活两千岁的寿命。


……


《山海经》里的异兽还有很多，了解它们不仅可以满足我们的好奇心，还可以知道许多关于古时候生态环境的知识以及古人对一些未知事物的理解和想象。


在这门课程里，一颉老师将用生动形象的语言讲述 193 种令人眼界大开的异兽故事，带你解密上古奇观，唤醒异界密码。
