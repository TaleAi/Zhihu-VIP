## 简介
本课程由资深律师李魁军主讲，为你讲解高难度的再审案件办理技巧。课程中，李魁军律师总结了一套独特的再审案件胜诉三大法宝，教你将事实真相直接、有效的呈现给法庭并说服法官，最终赢得案件的胜诉，大大的提高了此类案件的办理效率。


 


再审案件因为「立案难、改判难、案件复杂」的特点，一直以来都是律师执业的「痛点」，再审案件「很难搞」，也几乎成了所有律师不谋而合的共识。


 


通常案件到了再审阶段，一般都经历了一审、二审阶段，再审案件一般在立案审查后被裁定驳回，要想获得成功，无疑是绝境中寻找希望。


 


因此复杂疑难的再审案件，想要胜诉不仅需要律师投入大量的精力，更需要律师充分的准备和全面的考虑，去不留任何死角的掌握和利用好一、二审判决、裁定、双方证据、庭审笔录、谈话笔录、情况说明等与案卷中的所有材料，再有技巧地确定再审策略、法律关系定位、再审请求的基础。


 


律师代理再审案件，应该要怎样才能将事实真相直接、有效的呈现给法庭并说服法官，最终赢得案件的胜诉？


 


再审案件作为申请一方的胜诉率极低，为了解决这一问题，我们特邀请到李魁军律师，开设再审胜诉技巧课程。课程中，李魁军律师总结了一套独特的再审案件胜诉三大法宝，大大的提高了此类案件的办理效率，可复制，很实用。
