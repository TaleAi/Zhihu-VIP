# 与君绝：维以不永伤

### 作者：吾玉

### 分类：古代言情

### 状态：已完结，共 64 节

### 存档日期：2022-05-25

## 简介
这是一个情节百转千回、看完难以释怀的古风故事专栏。


她这辈子，做过奶娘，嫁过太监，还养大了一个孩子，几十年人世浮沉中，哭过笑过爱过恨过，如果说还有什么遗憾的，大概是直至最后离别时，都没能亲口告诉那个孩子，她有多么想守在他身边，哼着歌谣，唱到他白发苍苍。


她的父亲被判斩立决，未婚夫娶了那个陷害她全家的女人，更绝望的是，她被当众羞辱。只要能换她父亲一命，她愿意。谁知羞辱过后，那个女人携她的未婚夫扬长而去。刑场上，她已来迟，父亲早已人头落地。她失去的，要让仇人十倍奉还。


她在深宫，他在江湖。失宠的皇后不如一条狗。只有这个会飞檐走壁的人夜夜陪她喝酒。她心底还舍不得皇上，还在等皇上无法兑现的承诺。风云突变，物是人非，没有人能再回到从前。


读完故事，独留怅然：参不透爱情，做不到绝情，只能与君绝，维以不永伤。
