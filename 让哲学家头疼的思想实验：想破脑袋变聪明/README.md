## 简介
研究哲学上的思想实验和悖论，让独到而又有趣的问题在头脑中肆意盘旋，借着这些问题来检验那些人生中最基本的、最习以为常的想法，破除思维枷锁，开辟出一条见所未见的新奇道路，让我们看到另一边的风景。


在很多人的印象里哲学是一门艰深晦涩、让人皓首穷经的玄学，因此望而却步。


但思想实验能够让你 get 哲学思考的魅力，依靠丰富的想象力和逻辑方法构建故事场景，在场景中阐明抽象的哲学概念、检验哲学理论。


知识的革命和世界的改变，往往诞生于大胆的狂想与高妙的假设中，每一个伟大的思想实验，都会引发深刻的思考乃至洞察。


在大多数时间里，我们对于这些最基本的观念和想法不假思索，按照约定俗成、显而易见的方式去理解和实践，而哲学恰恰是要挑战这种态度。


我们甄选了最具生命力的十个思想实验，去反思时间、空间、自我、理性、正义等哲学中最重要的概念。


借由这些思想实验我们将一起检验那些人生中最基本的、最习以为常的想法，用一次思想的探险，抽丝剥茧、深度解读与我们每一个人息息相关的重大问题。
