## 简介
虐妻一时爽，追妻火葬场的小故事。


傲娇狗男人不好追，拿到了修罗剧本怎么办？


看我使出浑身解数，施展撩汉宝典，把他们一个个都收入囊中。


呵呵，就喜欢你们一副看不上我，却还会对我动心吃醋的样子。


《城春草木深》


我承认，我是个贪财、虚荣、矫情、风骚、善妒的坏女人。旁人皆说许津恪杀伐太重，才会遭报应，养了我这么个胸大无脑的败家娘儿们。


但听他们这么说~


我也不是一无是处嘛，至少胸大就赢在起跑线上了啊！


许大督军见惯了端庄的，突然碰见我这么一个骚的，那还能挪得开眼？


只是他不知道，那些风月缱绻啊，其实都是我装的。


《红荔不相思》


我是第二位寿王妃，上一位寿王妃是宫里的杨贵妃，我因容貌与她五分相像而备受恩宠。


偶然间，我发现了寿王李瑁藏在美人画卷里的秘密，原来这一切的骗局，源自两代帝王家的爱恨情仇……


《东宫美人》


我身负家族使命嫁太子萧礼为正妃，后宫险恶，生存不易，我不得不想法设法取悦太子、稳固地位。刚坐稳位子，我又招来亲妹妹的嫉妒，她自诩美貌才华在我之上，趁我难产血崩时入宫引诱萧礼，意图将我取而代之。


既然她不顾念手足情谊，要和我窝里斗，我不妨正面刚上去应敌。算计、陷害，一关又一关考验，接连横亘在我面前，我能否顺利过关，走上权利的巅峰？


《红袖招》


那年惊鸿一瞥，我对萧繁动了心，我不顾他已有妻室，执意入宫做了他的妃。


亲眼目睹了萧繁对结发之妻颜芷的宠爱，我嫉妒入骨。我虽为皇贵妃，可百年之后帝后同陵，我不过是个陪衬。我不甘心，我要争，争他的人，更争他的心。


我由着性子盲目争宠，终于争来了颜芷身死，争来了我的皇后之位，成了萧繁生而同衾死同穴的那个人。可没想到啊，颜芷早给我设下了无双之局，这个局不仅葬送了我的爱情，更葬送了整个霍氏家族。
