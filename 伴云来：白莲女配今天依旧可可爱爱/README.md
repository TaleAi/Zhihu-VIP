## 简介
穿成恶毒女配后，我决定先抢走男主的金手指，再找个靓仔谈一段甜甜的恋爱。有一天，轮回镜告诉我，这已经不是我第一次穿越了！？


我穿成了一个仙女，肤白貌美大长腿，和广寒宫里的嫦娥上仙平分秋色，觊觎我美色的小仙男都快踏破了门槛。


但我的未婚夫——凌风上仙却一点都不满意这门婚事，他已有心上人。


原作中，我从此开启了以拆散男女主为己任的作死之路，最后我抑制不住嫉妒之心，把女主推下了诛仙台，让女主差点魂飞魄散。真相大白后我被仙界驱逐，又被男主亲手杀死，魂魄都不剩下。


可是，从 21 世纪穿过来的我，只想继续做一个无忧无虑的仙女，而不是惨死男主剑下。


为了自保，我决定抢先一步前往秘境，拿走男主后期的金手指——龙纹玉佩。


在秘境里，我撞上了一个小妖怪，他眉如青山，目如朗月，挺鼻如峰，朱唇皓齿，干净又耀眼。这世间，怎么会有长得这么好看的人？


但他很奇怪，他知道我喜欢吃什么，他偷看我的一举一动，每次跟我说话他的声音都温柔得能掐出水来……


直到我被吸入轮回镜中，才知晓他的真实身份，也才发现我自己真实的身份和宿命！
