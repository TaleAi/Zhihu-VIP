假如军阀混战的时候有新闻联播，那么画风可能会是这样的：


亲爱的观众朋友们，今天是中华民国 X 年 X 月 X 日，今天的主要新闻有：


震惊！孙中山竟然亲自炮击观音山！国民党内部频传不和究竟为哪般？


可怕！国会议员集体化身猪崽，有议员声称自己千里投票只为钱！


无语！日本对华借款再攀新高峰，寺内内阁与段祺瑞政府又双叒叕了一项 2000 万日元借款协议！


无奈！四国银行集团在伦敦举行会议，抗议中国滥发债券重复抵押各项税款！发言人表示今后要对借款用途进行监控！
