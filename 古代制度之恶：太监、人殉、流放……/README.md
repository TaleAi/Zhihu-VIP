## 简介
太监、人殉、流放、和亲、屠城……这些残忍、变态的古代制度，是如何制定的？执行流程是什么？又是何时退出历史舞台的？


中国历史上，封建皇权下的威压，衍生了很多如今看来毫无人性、违背人权的制度，比如太监、人殉、流放、和亲、屠城……


这些制度，我们看似熟悉，但真正细思起来，脑袋里都是大大的问号。


比如：


* 太监是切阴茎，还是切睾丸？
* 殉葬的妃子们不愿死怎么办？
* 被流放的人不可以偷溜回来吗？
* 古代公主和亲为什么不偷偷换人？
* 古代屠城是每个人都要找出来处死吗？

……


该专栏从这些制度的某个有趣细节切入，结合当时有关的历史事件、历史人物，尽可能还原这些制度的执行流程，分析这些制度为什么根深蒂固地存在了数千年，以及这些制度是如何造就万千臣民的悲惨人生的。
