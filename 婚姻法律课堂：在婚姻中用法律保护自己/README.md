## 简介
婚姻中除了爱情的牵绊，也有很多生活的摩擦：结婚不像谈恋爱，它涉及很多财产、房产、抚养权等方面的问题。在很多婚姻案件中，「房子」、「银子」、「孩子」是三大永恒的主题。本专栏通过为你讲解经典婚姻案件，教你如何应对婚姻纠纷中的难题，了解关于婚姻法的那些事儿。


我们经常会在影视剧中看到：


马上要结婚的小夫妻，为房产证上写谁的名字吵得不可开交；怀疑对方有外遇，为搜寻对方的出轨证据费劲心思；做出离婚的决定，为争夺孩子的抚养权大打出手......


在现实生活中：


婚前财产公证得到了越来越多年轻人的认同，「离婚冷静期」的生效使大家对婚姻关系有了更多的思考。


还没有进入婚姻的你，是否了解婚前和婚后买房，写谁的名字有什么区别？婚前婚后的财产如何区分，什么属于夫妻共同财产？


在婚姻中遇到难题的你，是否了解关于婚姻的法律知识，能够运用法律的武器保护自己？


无论是婚前有备无患，还是婚后未雨绸缪，了解婚姻法才更有利于维持和谐的婚姻关系，在婚姻中更好地保护自己。本专栏主讲人于琦律师代理离婚案件 400 多件，从法律人的角度为你科普婚姻案件中的法律法规。
