## 简介
给你教育大师的传家智慧，让孩子健康成长、快乐成才。


作为宝娃的爸妈，你是否时常感觉自己孩子坏习惯很多，或者比较叛逆？


而在教育孩子这方面，你是否发现周围的大家都在进行教育输出，「不让孩子输在起跑线上」，因此感觉十分焦虑？


有关孩子的教育，尤其是我们的家庭教育，在当下时代确实是一个重大的问题。


其实，中国自晚清近代以来，涌现出许多教育大师，有的培养出优秀的子女，有的为我们留下宝贵的教育思想，都是非常适合中国家庭的传家智慧。


这些教育智慧，不仅是一种精神财富、家风传承，也有助于破除当下的教育焦虑，不「鸡娃」、不攀比，告别教育陷阱，回归教育本质。


本专栏精选曾国藩、梁启超、蔡元培、鲁迅、陶行知 5 位教育巨匠，为你提供历经风雨锤炼、宜学易用的家教心法，让孩子都可以健康成长、快乐成才。
