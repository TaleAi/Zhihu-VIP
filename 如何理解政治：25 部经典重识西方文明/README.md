## 简介
3个人及以上就会产生政治，它和我们息息相关。国际局势风云变幻，看清事件表象下的真实动因。本期专栏通过解读25部经典,100+个专业概念，帮助你掌握结构化认知，以政治背景做依托，建立政治学的思维模式，做出大环境下的正确选择。


在我们熟悉的文学作品和影视剧中，政治总是一场和阴谋、野心、权力操纵在一起「过关斩将」的竞争游戏。


「权力如浮影游墙。即便是矮小之人，也能投射出巨大的影子。」


「用右手握手,但左手要握着石头。」


如何训练出「政治头脑」从而看清事件表象下的潜在利益链?


如何看清历史长河中的暗潮汹涌，准确把握时代发展趋势?


这门课将颠覆你对政治的认知，解读 25 部经典,100+个专业概念，摒弃枯燥术语，从经典文本出发，引出公民、城邦、民主、共和、君主、政党等基本概念，助你习得一手知识，深度剖析热点话题。
