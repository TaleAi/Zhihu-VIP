## 简介
当穿越不局限于千篇一律，不必拘泥于俗套剧情，而是让你穿越到另一种人生，会发生什么？现代人类穿越后，到底会发生哪些离谱行为？


我是策划部被人遗忘在角落的 PPT 女工，他是策略部指点江山光芒万丈的大帅比。


如此悬殊的差距，让我只敢暗中仰望……


没想到，我竟然获得了一个和他滚到一起的机会。


可是怎么滚的……毛茸茸的？？？


——《汪汪？这就是爱情！》


  



我出车祸了，是被人蓄意谋杀的。


我知道，有人盯上了我的公司，而且凶手就是我身边的人。


这一切证据，竟然都是在我成为植物人后找到的？


因为我的身体，好像发生了奇怪的变化……


——《与猫共舞的日子》


  



我是一名女警察，一次值班时不小心睡着了，醒来竟然变成了被霸凌的女高中生？！


这我能忍？？？


妹妹别怕，姐姐帮你教训他们！


——《热血女高我来了》
