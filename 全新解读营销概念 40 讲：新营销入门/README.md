## 简介
这是一本活的「营销辞典」，40 堂课根据「基本概念、传播手段、行业术语、近年热词」四大版块划分。每个版块精选的概念，都直指你工作中遇到的实际问题，没有抽象语言，而是通过大量的方法和案例帮你打造属于自己的专业知识体系。


为什么杜蕾斯每出一条文案都能红遍全网？


为什么西少爷肉夹馍能在短短几年内，从街边小店，发展成拥有四十多家连锁的网红餐饮店？


为什么看着别人家的营销方案，一夜爆红，望洋兴叹。自己学来学去却总是照猫画虎，四不像？


本课程请到了资深营销人黎媛老师。黎媛老师曾在沃尔玛担任营销工作，并且策划了沃尔玛在中国历史上的第一个「社会化营销战役」，使沃尔玛的品牌形象在中国深入人心。


黎媛老师将从自己多年的工作经历出发，为你深入浅出、通俗易懂的讲解 40 个营销人员必懂的基础概念，为你想出新奇动人的营销点子，打下坚实基础。


本次课程共分四大板块：


第一大块，黎媛老师将讲述品牌、营销、目标用户、细分市场、社会化营销等，这些营销人必备的营销概念。


第二大块将讲述营销人所需的各种传播手段，例如，内容营销、社会化营销、公关……


第三大块将讲述各类营销术语的含义。


第四大块将分析近年来各种营销热词是如何一夜爆红的，又有那些值得学习和借鉴的地方。


同时，每堂课引用大量的实际案例与解决方法，直指营销工作中的实际问题，由浅入深，帮你打造属于自己的专业营销知识体系。
