## 简介
表达能力是一个人工作生活中的关键能力，它能帮助你在众多场景中脱颖而出，让他人更好的理解自己。《超级演说家》总冠军刘媛媛将用自己的成功实践帮你学会表达，成就自己。


生活中不被重视的人，


往往都是不会说话的人。


  



你或许曾遇到这样的场景：


班级社团竞选——一上台就紧张，词不达意；


工作汇报方案——总是词不达意，逻辑不清；


想做说话有魅力的领导——开会台下昏昏欲睡，效率极低；


聚会饭局上想成为焦点——不知道聊什么，处境尴尬。


再也不想把努力变成徒劳了，


这需要提高当众说话的能力，


因为当众说话的能力=影响力。


学会说话，


才能展现真实的自己，


才能在关键时刻脱颖而出。


  



就像她，


从毫无演讲经验的学生，


一路逆袭成为《超级演说家》总冠军。


刘媛媛,北大法学硕士、《超级演说家》总冠军


代表演讲作品：《寒门贵子》


她的单个演讲作品点击量过亿，是很多演讲机构的教学素材。


至今演讲场次已超过 200 场，开发线上演讲课程，已有数万学员受益，线下分享会更是座无虚席。


从默默无闻到总冠军的经历，


让她知道说话和表达的重要性。


她总结了一套最实用的方法论，


帮你解决说话问题。


五大板块，层层深入，


升级你的表达力。


**基础表达：让别人听的进去；**


**技巧表达：告别紧张，逻辑清晰；**


**高级表达：让说话成为你的武器；**


**魅力表达：打造属于你的影响力；**


**场景表达：即学即用。**


任何场合都能展现最光彩的自己。


  



**课程特色：**


5 大板块：从基础到高阶，学习内容层层递进；


30 节课程：从底层原理到实用技巧再到实践场景，多维学习；


演讲视频：每节课配相关主题演讲，结合实践，身临其境。


  



**你将获得：**


一套全面系统的演讲表达术；


提高情商，说话更具说服力；


职场上汇报沟通清晰有逻辑，快速脱颖而出；


饭局上开口就是焦点，收获人脉；


告别紧张，竞选开会，塑造完美气场。
