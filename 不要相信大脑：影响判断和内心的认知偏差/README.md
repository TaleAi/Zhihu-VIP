## 简介
你的大脑，远比你想象中更会偷懒！


* 我们看到的世界是外界事物的真实反馈吗？
* 为什么有些很明显的谣言还有很多人信？
* 为什么我们劝不动听信谣言的长辈们？
* 为什么出租车司机大多喜欢聊政治？
* 灾难来临前，为什么总有「钉子户」不听劝说，执意留在家中？

这一切，都是「认知偏差」在作祟！


各式各样的「认知偏差」就像是我们复杂的人脑系统里的 BUG 一样。


它们深深植根于我们日常的思维方式中，并且其中许多是无意识的。


它们使我们难以准确地交换信息或得出客观的事实真相，扰乱甚至摧毁了我们的批判性思维，从而导致可能长久存在的错误想法或行为，对自己以及他人造成伤害。


**在本专栏中，拥有脑科学和心理学双重专业背景的叶赌徒，将为你解读影响判断和内心的「认知偏差」。**


**学会识别我们在日常交往中所经历的「认知偏差」，了解我们的心理和思维过程如何运作，从而做出更好的、更明智的决策吧！**
