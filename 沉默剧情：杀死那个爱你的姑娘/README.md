## 简介
年少时候时候总以为自己的深爱重值万金，但漫长的岁月总会教你什么叫做自作多情。诗梦易做，爱意难消，当你从感情沼泽里狼狈脱身，才能发现远处的巍峨——你当像鸟飞向你的山。



《上岸》


他和我说婚讯的时候，我手里正燃着的烟头不小心掉在了脚面上。


然而就在片刻之前，我们还有过一夜欢愉。


他知道我爱他，但他拒绝了我的求婚。


还说，他永远不会爱我。


《奶狗有诈》


我看上的小奶狗是个乖巧且聪明的替身，目标明确，兢兢业业勤勤恳恳。


但我不是个聪明的金主，把人赶走了自己却栽了。


套娃懂吗？它有多好看，我就有多笨。


《情投难意合》


我至今没谈恋爱的原因很简单。


我喜欢的那个人，看似不给我希望，却又总是在他最难过的时候想起我。


让我有一种错觉，他好像随时会爱上我。


为了这份游走不定的爱，我时刻准备着……
