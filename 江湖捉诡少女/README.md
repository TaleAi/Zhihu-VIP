## 简介
少女可不会按照江湖套路行事！


这是个脑洞大开的言情故事合集。


**· 惊！穿到修仙小说后，我靠 C++ 走上人生巅峰。**


倒霉的我跌了一跤，醒来就成了修仙小说里的恶毒女配。没活够的现代奇女子我本人，决心修正不得好死的结局。


修仙要天赋？看我靠 C++ 逆天改命！


男女主闹矛盾？我绝不允许美好的爱情就这样破裂！


只是，谁能告诉我为什么仙侠小说里还会闹鬼？


这个，臣妾真不会啊！


**· 急！色诱魔尊被抓包，沙雕女孩怎么当好双面间谍？**


咱也不懂，上一秒我还在梦里调戏大帅哥呢，下一秒怎么就被这个反派大魔王给抓住了？


被抓也就算了，他为啥还要抽走咱的一魂呢？有咱的身子还不够吗？


他抽走我的一魂也就算了，还让我这么个正直善良的小女孩去当间谍！这是人干的事吗？！


什么？他以我的性命逼迫我？


那我，我就同意吧。


反正，他不知道，离开这个世界的机会就在眼前。


我是不会错过的。
