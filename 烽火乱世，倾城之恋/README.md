## 简介
乱世佳人对抗命运的爱情绝唱


专栏收录了各种有勇有谋的乱世佳人的故事，她们中有被褐怀珠、对抗命运的青楼女子，也有堕入泥淖、不屈不挠的名门贵女……


深陷漩涡，她们不卑不亢；身处乱世，她们坚如磐石，用智慧与坚韧，演绎可歌可泣的倾城之恋。


  



**暮禽**


我是出身富贵，喜好男色的江家二小姐。路边捡了个美惨强，顺手巧取豪夺，要将他吃干抹净。


直到那一日，我全家遭他血洗，被屠个精光。而我也被他放走，远渡重洋。


多年后，我留洋归来，再次宴会偶遇，他已是一方军阀，目光如炬。


这回，轮到他吃了我。


 


**欲穷年**


我是乱世之中，双手沾血的卖笑女。


承蒙一张像极了督军三公子白月光的脸，和好巧不巧同音的名字，被送进他的房，却近不了他的身。


直到有一日，他的白月光回来了，以他后妈的身份，臣服在他爹的脚边……
