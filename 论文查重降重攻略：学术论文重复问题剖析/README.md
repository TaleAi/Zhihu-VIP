## 简介
本专栏将为你剖析中英文论文的查重原理，提出针对性的降重方法，结合案例实操，帮大家掌握降重的方法和技巧，有效降低重复率。


在科研和求学之路上，毕业、晋升、择校、就业，都伴随着发表一定数量的中英文论文的需要。近年来，随着中英文论文投稿量增大，论文重复等学术道德问题屡屡出现，不仅使论文作者声誉受损，作者单位也会因此蒙羞。


与此同时，国内外期刊对论文重复率的要求也在逐步提高，如果重复率过高，大部分期刊编辑会直接拒稿甚至不给再次投稿的机会；即使侥幸发表，如果之后被举报和证实有重复问题，后果会更加严重。


然而，很多论文作者并不清楚中英文论文查重的原理，总以为连续多个单词、汉字相同就算重复了，不经意间造成文字重复过高，实在可惜。


为了让广大科研人员和学生避开论文重复过高的大坑，本专栏将为你剖析中英文论文的查重原理，提出针对性的降重方法，结合案例实操，帮大家掌握降重的方法和技巧，有效降低重复率。
