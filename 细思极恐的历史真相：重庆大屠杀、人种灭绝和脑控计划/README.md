## 简介
本专栏通过大量考证，带你走进一部「人类残害同胞史」，让你看清那些由人类亲手制造的惨剧，如何支配了我们今天的生活。


二战时期，中情局坚信，


敌人在审讯时用了「某种未知力量」操控战俘心理。


于是「心灵控制计划」启动了：


洗脑、催眠、lsd 致幻剂、间谍审讯……由此诞生。


他们确信，人脑还有更多可以改造的地方。


——《为了对付中国人，CIA 琢磨了 20 多年脑控计划》


  



纳粹当年是怎么精准找到犹太人的？


IBM 曾作为「知识分子突击队」，


让数据统计成了大屠杀的有力工具；


技术狂人们通过改进荷兰人口统计局的分类方法、和霍尔瑞斯穿孔流程，


精确辨识，驱逐了荷兰 10.7 万犹太人，10.2 万人惨死。 


这样的技术成就，无愧于统计局的口号：记录即服务。 


——《这家公司曾参与一场 600 万人的大屠杀，你可能正在用它的产品》


  



1940 年的重庆人都经历过什么？


从第一次遭遇大轰炸时，无知围观造成的惨烈现场，


到熟练「跑警报」、在地毯式轰炸间隙顽强生存，


甚至雾季来临时，还能穿上耀眼衣服赛龙舟……


生生不息，「嘞是雾都」。


——《大轰炸来临时，重庆上万人死在防空洞》


  



专门研究中国传统鞋饰的藏家有一双三寸金莲鞋，


蒂凡尼蓝，滚精致的棕边，内里衬着时髦的蓝白方格布。


鞋面上绣着巨大的英文：THE BABY。


这是一双民国时代裹脚妇人的潮鞋，


比一万双潮鞋还要有力量。


——《民国女孩穿衣解放历程》
