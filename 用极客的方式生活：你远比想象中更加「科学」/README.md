## 简介
科技改变世界的同时，文化会被重新塑造。这不是一个最好读的专栏，但其中充满了新鲜的知识和能量，相信它会带给你全新的「浸入式」阅读体验。


一、旧术犹新：过去未来的惊奇科技
----------------


几乎任何技术都不会消亡，至少在全球层面上不会消亡。常常会有某个地点的某个人，仍然在使用一种最古老的技术。——凯文·凯利，《一种消亡的媒介》


对科技发展的一个脉络式梳理。科技在任何时候都是先锋的、持续进化的，但同时也都存在着关于科技的反面。科技正反两面，在长久的历史发展和现状中的对比呈现，人同时也在平衡与科技的关系。


每一个阶段的科技都有令人惊奇之处，也有需要人们反思的关于科技的「无处不在」。在我们善用科技的今天，留给生活一些「反科技」的空间。


通过对技术与人文的交叉讨论，还原科技的两面性。


二、隐蔽而伟大的设计之力
------------


包括大人的玩具、无用的艺术、自动贩卖机、可供性四个话题。


探寻看似简单的产品背后隐藏着的设计力和创造力，对习以为常的「简单」产品背后所暗藏的设计和创新进行剖析。


三、当代中青年困顿生活
-----------


包括熬夜、网络上瘾、内向者、亲密关系、虚拟偶像等五个话题。可以理解为「当代人生活烦恼的答疑解惑」，通过这些常被讨论和调侃，也是最有代表性的焦虑——熬夜、网瘾、追星、社恐，来展现年轻人生活不为人知的困境。


用科学的精神和科普的方式，来解释出现这些焦虑的根源，并给出建议的解决方案。


四、厨房里的技术宅：写给美味的硬核情书
-------------------


厨房中不止有美食家，还有极客。以「科学」的极客精神，来重新演绎厨房中的科学和文化，展示当代年轻人生活中的独特趣味。生活方式不只是文化的，也是科学的。


包括咖啡、日本料理、意大利面、炸鸡四个典型话题。从科普和日常的角度全新角度切入年轻人的生活场景，定义年轻一代对食物的全新理解。可作为大众读者日常轻阅读，也可给低龄读者做科普读物。
