## 简介
创业者不仅需要了解枯燥的公司法条，还必须掌握具体的实务案例操作。如何利用股权控制公司？如何在公司治理中实现股东权利？本专栏用丰富翔实的案例，解读公司法及企业经营中的股权博弈，轻松掌握与公司法有关的那些事儿。


从初创企业到行业巨头、从情感维系到资本入局，股东们从同心协力、同甘共苦，到同床异梦甚至同室操戈。


公司与股东们一路「过关斩将」时，会面临哪些法律风险和纠纷？


中国裁判文书网发布的案例统计，中国内地法院自 2011 至 2020 年 10 月间，与公司有关的民事纠纷案件共计约 35 万宗，其中近半数是股权转让纠纷，约 17 万宗。其他与公司有关的纠纷主要是股东资格、股东出资、股东知情权、损害公司利益、公司解散纠纷。


本专栏将以《中华人民共和国公司法》为基础，结合公司频发的各类纠纷，辅以丰富案例，明晰企业成立到解散全流程的法律要点及注意事项，解答与公司有关的重要问题。
