## 简介
末日来了，与其正面硬刚，不如选择躺平，安稳求生。


未来世界，来自外宇宙的未知 X 病毒席卷全球，爆发了恐怖的丧尸危机。


一对中国兄妹在欧洲、五十星帝国开始了各自求生、进化的经历。


**鹿零**蛰居在欧洲海边小国的一座灯塔上，观察丧尸演变、应对时不时找上门的种种危机，等待远在五十星帝国的哥哥脱离困境，来带自己回家。


在此过程中，鹿零和一个神秘男子——老鬼逐渐产生交集，也得罪了对方身边的少年——洛西法，最终导致 X 病毒终极体——大阿尔法的到来，零被其掳走，并产生被动进化。


一番历险后，鹿零兄妹、老鬼回归东方，人类应对外宇宙威胁的进化元年，就此正式开启。
