## 简介
从哥伦布到加勒比海盗，从伊萨贝拉女王的资助，到英法西在南美的争霸，和你一起重温加勒比往事。


哥伦布，一个历史书中如雷贯耳的名字，一个凭借「冒险」和「勇气」，扩宽人类对「世界尽头」认知的英雄。 


但同时，他也是一个对权力和黄金充满渴望的野心家；一个做了糟糕的航海决策，以至于一直以为古巴是个半岛的航海家；也是一个到死都没搞明白自己到的到底是日本还是中国的「新大陆」发现者…… 


经过了 4 次航行，找到了从西班牙通往加勒比只需 21 天的航线，在大海的狂暴中九死一生，而对哥伦布来说，更为艰难的是，如何面对海员的恐慌和叛乱、如何满足西班牙皇室对黄金的极度渴求，如何在日后贩卖对待他极为友好的原住民…… 


「加勒比往事」的开头中，我们把时间线拉回 1492 年，加入哥伦布的船队，和他一起面对风暴，穿越大西洋抵达尽头，发现和面对未知……


而对于生活在加勒比区域的人来说，这是一段被人发现的历史。而这次的「被发现」，意味着「一切都被改变」。在「加勒比往事」系列中，你将获得双重视角，重新来看待哥伦布、奴隶贸易、加勒比海盗，以及英法西在南美的争霸这一段历史……


作者司洋，知乎 ID「十一点半」，投资行业人士，在知乎分享的优质内容已获得 26 万赞同。作者的《以色列：一个国家的诞生》系列电子书广受欢迎。有读者评论道：「终于弄清楚新闻联播里约旦河西岸，加沙地带的意思。这是一本读起来会自带画面感的书。」 


这一次，以作者一贯的写故事手法，我们来为你讲述自带画面感的，加勒比的故事。
