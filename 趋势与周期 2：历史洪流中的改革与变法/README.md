## 简介
了解历史深处的改革与个人创造价值的途径。


在这个世界上，唯一不变的是变化本身。


历史上有名的改革者，如商鞅、管仲、张居正等，他们的事迹我们虽耳熟能详，但其中包含的思想和社会背景，尤其是经济、市场和土地方面的涵义，我们了解得并不到位：


* 商鞅变法：改革是战国时代最时髦的工作？
* 张居正新政：活着的时候风光无限，死后儿子却被饿死，为什么？
* 雍正新政：历史上的第一肝帝，怎么就被全网黑？

历史是螺旋上升的，即便已经过去了 2000 多年，封建统治下的那些改革与变法依然有很多值得借鉴，值得学习的地方。


作者叶泊枫将用诙谐简练的语言带领大家从经济、文化等多个角度去理解历史洪流中的改革和变法。
