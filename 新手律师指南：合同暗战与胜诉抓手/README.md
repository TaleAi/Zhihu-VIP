什么是合同暗战？


合同暗战就是指通过多个层次、多个阶段的结构性的安排，在对方不知不觉中打造能够让己方胜诉的抓手。


所谓多层次，举个例子来讲，就是在合同目的、条款词句、合同履行的行为等等各个层面作出结构性的安排。这几个层面、因素凑到一起，可以形成一个胜诉的抓手。


所谓多阶段就是指在合同的发生、发展、消灭的整个过程中，具体来讲包括协商、签订合同、具体订立合同的条款、履行合同解除合同、还有就是发生纠纷之后的协商解决五个阶段。


纠纷可以发生在合同的前述的各个阶段，那么胜诉抓手也可以产生在前述的各个阶段，纠纷和胜诉抓手，既可以发生在同一个阶段，也可以在不同阶段的发生。当然通常来讲，胜诉抓手要早于纠纷的发生而产生。


 


什么是胜诉抓手？


胜诉抓手简单的讲就是，你在诉讼仲裁案件中要想胜诉，你必须解决的问题，必须找到的基点。


这分两个层面，第一个层面是在裁判价值观上，在裁判价值观上就是要让裁判者愿意判你赢，如果裁判者不愿意判你赢的话，你自己觉得你再有理也很难胜诉。


第二个层面就是在裁判方法论上，要让裁判者可以判你赢，也就是说如果裁判者愿意判你赢，但是他没有办法判你赢，也很难做出对你有利的裁判结果。
