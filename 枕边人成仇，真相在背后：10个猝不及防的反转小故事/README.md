## 简介
你所经历的都不是真相，你所看见的都不是终章，怎么办？


她出差回家，撞见了非常俗套的一幕：丈夫和闺蜜滚在了一起，在她的床上，在她怀孕 3 个月的时候。丈夫好整以暇，闺蜜落落大方，只有她必须是打碎牙齿和血吞的那个，她只配黯然离场？


——《全员皆输》


老公不但出轨，还觉得自己仁至义尽，黄脸婆净身出户，还要感恩戴德。这到底是什么世道？洛云安静地接受，有条不紊地按心中计划一步步反击，她给到自己的期限是 3 年，没想到半年，对方就已经招架不住……


——《最终的情人》


枕边人陌路，陌生人入侵，生活不再一潭死水，宁静被彻底打破，想触碰真相又收回手，但到底要迎着暴风雨，不能退缩。


生活的转角，有惊疑，也有转机。
