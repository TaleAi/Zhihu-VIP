## 简介
这是一篇长篇权谋言情小说。讲述了相府长女一朝沦落神秘的「易园」成为不由自主的侍女，周旋在各方势力之间步步惊心的故事，堪称古代版的「密狱逃生」。


十一岁之前，我是相府千金。


十一岁之后，我毁容，被送入易园，变成了其中最微不足道的侍女。相貌平凡，没了姓名。


还得呕心沥血辛苦伺候各位主子。


  



天下易园，可不只是一座普普通通的销金享乐窟而已。


它的背后，是一股神秘的皇家势力。


来这里的贵客公子犹如过江之鲫，我每日如履薄冰，不得不打起十二分精神和他们周旋在其中。


但某一天，一名神秘来客打破了我费尽心机的遮掩与平衡……


我还能拿回我失去的一切吗？
