## 3.全面认识元宇宙，还需了解这些概念

大家好，欢迎大家收看「辨伪存真·揭秘元宇宙十三讲」之第三讲：全面认识元宇宙概念


上一讲我们重点讲解了看起来像游戏一样的纯虚拟元宇宙具备哪些普通游戏不具备的特征，使得其有潜质发展成为新的虚拟经济体。目前元宇宙的发展中，纯虚拟元宇宙是占比重比较大的。不过除此之外，你也许还听到过以下和元宇宙相关的概念：沉浸式互联网、Web3.0、工业元宇宙。它们也是目前元宇宙概念中的重要拼图，影响着元宇宙的未来走向。要想更加全面地认识元宇宙，也必须对这些概念有所了解。


首先来看**沉浸式互联网**。


简单来讲，我们现在的互联网是 2D 的，而沉浸式互联网是 3D 的，「空间互联网」、「Web XR」说的也都是沉浸式互联网。沉浸式互联网通过 3D 建模、人工智能、虚拟现实/增强现实/混合现实等技术让互联网的内容层变得更加具有真实感和交互性。但是沉浸式互联网是否就是元宇宙呢？目前大部分互联网巨头所谓对于元宇宙的尝试，其实更像是打造沉浸式互联网。比如微软的虚拟办公，Facebook 的虚拟社交，亚马逊和阿里的虚拟购物等等，都是在原有的 2D 互联网的基础上，增加了沉浸式体验的维度。当然，当原有事物附着了新的维度后，可能会引发很多变化，互联网在加入了沉浸式体验后，不仅仅是从 2D 到 3D 这么简单，有可能伴随的是新的交互方式、新的交易行为、新的业态的出现。但也有可能，沉浸式互联网只是传统互联网红利退去后，强行拉来的一根救命稻草，如果不加入其他驱动因素，比如像元宇宙游戏中的去中心化经济激励机制等，沉浸式互联网只能停留在内容升级层面，而无法发展成为基于现实的半虚拟数字经济元宇宙。


再来看 **Web3.**0。


简单来说 Web3.0 就是由区块链支撑的价值互联网。Web1.0 时代是首页+超链，可以被称之为互联网只读时代；Web2.0 可以允许用户进行内容生产，进入读/写/交互时代；而 Web3.0 在区块链的支撑下，可以使互联网上的所有元素在理论上都可以资产化，形成所有权，进入了读/写/拥有的时代，即价值互联网。那么 Web3.0 是否是元宇宙的另一个代名词呢？不得不重申，元宇宙的定义到现在还没有一个达成共识的标准答案，我们在本课程中根据其背后的供需关系、未来走向，以及综合各方的观点，将其定义为一个以明确身份可交互、可交易的新虚拟数字经济体。那么 Web3.0 看起来更像是这个虚拟经济体的价值层，如果上层没有更丰富的内容和可以资产化的要素，恐怕 Web3.0 也无法发展起来。 


说到这有没有觉得，元宇宙也许就是 Web3.0 +沉浸式互联网+去中心化游戏？其实早在 1995 年乔布斯就说过，伟大的技术是看不见的。现在关于元宇宙还有这么多的名字、概念，也许是因为这些「伟大」的技术还没有真正走入我们的生活，我们还停留在探讨而不是使用它们的阶段。也许再过若干年，沉浸式互联网、Web3.0、元宇宙等等概念都会消失，因为它们已经成为我们生活的一部分。


最后我们来说说**工业元宇宙**。


想要了解工业元宇宙，首先要知道什么是数字孪生。数字孪生的英文是「Digital Twin」, 数字双胞胎；百度百科里对数字孪生的定义是通过物理模型、传感器更新、运行历史等数据，集成多学科、多物理量、多尺度、多概率的仿真过程，在虚拟空间中完成映射，从而反映相对应的实体装备的全生命周期过程。简单来讲，就是通过一系列操作，依据现实的工业生产建立了一个虚拟的、一模一样的生产模型；说得更大一点，更宏观一点，就是建立一个平行于现实工业生产外的另一个虚拟工业生产世界，自从元宇宙这个概念兴起后，这个世界开始被有些人称之为工业元宇宙。 


目前很多从事工业互联网、数字孪生等事业的业内人士非常看好工业元宇宙的前景，相比于对于人的现实价值还没有完全搞清楚的以人为主的元宇宙，由机器和物构成的工业元宇宙对于实际的工业生产已经显现出明确的价值。比如波音、BMW、现代等企业都已经开始利用工业元宇宙，通过数字孪生的方式将部分制造或生产流程首先在线上模拟，然后再在现实生产中采取行动，以此获得更好的决策来指导现实的制造或生产，降低因决策带来的成本风险。


好了，这就是我们第一节厘清概念中的三讲内容了，这一讲中我们探讨了元宇宙的本质、重点研究了元宇宙和游戏的关系，又进一步了解了沉浸式互联网、Web3.0、工业元宇宙等和元宇宙相关的热点概念。希望现在的你已经对元宇宙有一个较为全面和清晰的认知。
