## 简介
丁香医生医学总监、孕产科专家田吉顺，精心打磨 13 堂孕产通识课，从备孕、怀孕、分娩到产后恢复，为所有计划备孕、正在怀孕以及产后恢复的家庭，提供一套完整的孕产期身心指导。


**怀孕是一场心理和生理上的硬仗，是整个家庭的大事。**


对于没有经验的家庭来说，孕产面临着各式各样的困惑：


→备孕期检查到底该做哪些？


→早孕期该如何保胎才能避免流产？


→腿抽筋、便秘、失眠、反酸、腰疼怎么办？


→想顺产但又怕疼怎么办？


→生完到底怎么坐月子，怎么恢复身材？


而随着二胎政策的放开，很多家庭面临高龄生产的问题，那么年龄太大生孩子风险到底有多大？


针对准爸爸准妈妈们关于怀孕面临的各种问题，妇产科专家田吉顺精心打造了这套孕产通识课。 **从备孕、怀孕、分娩、坐月子到产后恢复整个孕期过程， 田吉顺为大家提供了一套非常完整的孕产期身心指导方案。**


破除朋友圈谣言和陈旧孕产观念， 针对情绪疏导、饮食运动、孕期检查、孕期常见注意事项和医学知识几大维度， 为大家提供先进的医学指导，避免因受到误导陷入不必要的麻烦。


**学习本次课程你将获得**


**1. 除了医学知识和生活建议之外，还会提供心理疏导的方法，让大家不止是身体上轻松，也要心理上愉快**


**2. 比医院咨询更加系统，比网络海量信息更加精准的全方位孕产观念解读**


**3. 除了准妈妈之外，准爸爸也可以全程参与的指导建议**


愿你和家人能享受人生中这非常重要的 40 周，轻松愉快地怀孕生孩子，健康地度过整个孕产期。
