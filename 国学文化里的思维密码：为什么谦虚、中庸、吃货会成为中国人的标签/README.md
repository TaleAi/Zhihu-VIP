## 简介
本课程将从汉字、《易经》、礼仪、禅宗、春秋，这五个版块出发，揭示中国人的思维源头与特点。


中国文化在今天是否还适用？在将来是否会发展？会朝哪个方向发展？


这些问题都不是短短几万字可以讨论清楚的？我们内容的重点也不在于此！


无论你对中国传统文化是什么态度？


反对 or 支持 or 无感，我的内容都适合你。


我并不想吹捧中国文化，也不想贬低它！


我的目的是从这些文化典籍中揭示一个又一个暗藏在我们心底的思维方式。


为什么中国人都是吃货？为什么中国人特别会写诗，能在唐代就到达艺术的高峰？


为什么中国是礼仪之邦？为什么我们那么注重历史？


不管你喜欢还是不喜欢，这些问题都是我们中国人底层的思维方式。


因此，我们会从具体的历史环境出发，选取了五个最典型的版块，每个版块都会从五个维度去揭示其背后的思维世界。


去看一看这些文化现象和思维究竟是怎么产生的。


以及，这些思维如何影响后来的中国和现在的我们。
