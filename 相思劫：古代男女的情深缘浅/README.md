## 简介
揭露古代后宅的阴谋算计，讲述皇家世族的爱恨纠葛！被富人逼嫁给傻子的女子、被皇帝逼下城楼的皇后、被庶母陷害嫁给乞儿的千金……古代生活不易，女子生存更不易。在封建制度的荼毒中，她们能否涅槃重生，最终追求到属于自己的幸福？


七娘洗个澡就变成了董家大少奶奶，这速度委实有点快……面对傻兮兮的丈夫，七娘却在这场虚假的爱情中越陷越深。直到她发现，傻子不傻，傻的是她。


——《七娘》


在宫人看来，帝后相携赴宴，真真是恩爱羡煞旁人。可只有春生知道，在笑得那般百意温柔的舒越心中，只有疏离。皇后本不应有爱，她却渐渐动心；皇帝也不应有爱，他却发现，当她对着他笑得不再虚情假意的时候，他已无路可逃……


——《画堂春》


孟大小姐被姨娘庶兄陷害，为了名节只能嫁给一个乞儿。可乞儿身份存疑，她竟然从低嫁变成了高攀……哪怕相爱，他们也终究沦为了争权夺利的牺牲品。


——《续春》
