## 简介
如何理清战国的历史？生在战国时期是一种怎样的体验？怎么才能活出战国诸侯的感觉？


战国时代，被大争精神贯穿，催生了层叠林立的原典，造就了接踵比肩的巨人，生发了深刻壮阔的变法浪潮……


所有这一切，都汇成了浩浩荡荡的历史大潮，直接推动秦帝国实现了统一中国文明的历史跨越。


专栏分为上下两篇，上篇解读了韩、赵、燕、魏、楚、齐六国灭亡的原因，以史为鉴可知兴替。


下篇聚焦战国时代的传奇人物，秦孝公、吴起、墨子、吕不韦、荀子、韩非、赵武灵王、赵佗，一睹两千年前先贤的风采。


大争之世，多事之时；凡有血气，必有争心。**以古论今，我们仍未走出战国时代！**
