## 简介
本课程是以心理学为依托的「情感经营方法论」，针对 14 个亲密关系中的行为误区，给出 14 个使用的解决方案。更有 3 种提升亲密关系的实战经验，助你建立更稳定的亲密关系。


「从此，公主和王子幸福地生活在一起。」「爱一个人就是要毫无保留地付出！」


关于爱情，你过去知道的这些都是错的。


「只有绿茶才撒娇！」「同居是结婚的美好前奏和序曲。」


关于情感经营，你以前经常做的也是错的。


恋爱其实是两种能力：爱一个人的能力，和让对方爱你的能力。情感经营其实是两个功课：认识爱情和学会经营。


本课程将打破从小「被迫害」建立的四个错误认知，树立正确的爱情观。用案例式教学法支出情感生活中 14 个最常见的错误行为，矫正经营动作。一起来学习让爱情升温的技巧，爱到老，学到老。
