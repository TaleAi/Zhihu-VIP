## 简介
世界上最稀有的植物是什么？植物的生存能力可以多强大？


在我们这颗蓝色星球上，植物的出现无疑是一个最大的奇迹：它们不像动物一样可以逃避不利的环境，也不像微生物一样通过快速变异适应环境。


植物选择了一条最艰难的道路：去改变环境。


在我们熟悉的植物的背后，有着许多不为人知的故事。


植物的演化之路，就是一部惊心动魄的生命创业史，它们创造市场、改造环境、修造壁垒、打造品牌、塑造用户。


看完这个专栏，相信你会对身边的植物有着不一样的理解和好奇。


**你将获得：**


1. 关于植物的有趣知识


2. 这些有趣知识背后的深层道理


3. 了解植物是如何适应和改变地球环境的


4. 了解其他生物是如何因植物而改变的


5. 了解到植物未来的演化道路将会通往何方


6. 学到一些关于生存、竞争以及抉择的智慧


**适合谁听：**


1. 适合对植物学感兴趣的人群，你可以在这里学到植物学的知识


2. 适合科学爱好者群体，你可以在这里学到科学思维和科学精神


**3.** 适合创业者或者尝试创业的人群，你会在这里学到生存的智慧
