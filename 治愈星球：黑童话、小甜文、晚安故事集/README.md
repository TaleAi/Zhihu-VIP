## 简介
女王、人鱼、霸道总裁、无心人，形形色色的故事轮番上演，带你开启一场治愈人心的奇幻之旅。


从前有一颗治愈星球，可以治疗人们心中的忧伤和烦恼。


公主、恶魔、女巫、人鱼，现实于童话中的人物都住在这颗星球上，他们相安无事的生活着。


公主吻醒了恶魔，不断请求恶魔完成自己愿望的同时，却不断失去自己最重要的东西；


女孩阿宁不冲马桶，这不是她的坏习惯，而是生理自然反应，这给她带来了一系列的烦恼；


人鱼与人类男友恋爱，办了入籍移居陆地，当她和男友分手时，在回大海办手续时又遇到了种种困难。


这颗星球上的故事，每天都在上演。


有些很温暖，有些很美好，有些很奇幻，有些触动人心。


现在，欢迎你来到治愈星球，在现实与童话的世界之间穿梭。
