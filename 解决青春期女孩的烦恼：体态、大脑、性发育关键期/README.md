## 简介
青春期，是女性走向成熟的一个重要时期，这个时期的女孩子，面临着形态、功能、素质和内分泌等一系列的剧烈变化，在生理上成熟的同时，也面临着心理上的新特点和新体验。本专栏将视角集中在青春期的女孩子，通过「我在长大」、「我是女生」、「我会更棒」三个板块，10 个知识点的讲授，围绕青春期的女孩子关于体态发育、性发育和大脑发育过程中的营养促进、运动改善健康的特殊进行解答。


要长高，就要多喝骨头汤或者吃激素？错！长高，吃「它」比什么都强。


预防近视就要少看电子产品？错！足够的户外运动才是关键。


大姨妈痛就多喝红糖水？错！想要不痛，一日三餐吃什么很关键。


青春期，是女性走向成熟的一个重要时期。在这个特殊的时期，女孩子们有其不同的生理生化特点和营养需求，她们的某些营养和健康问题又区别于男性。面对这一未知的时期，很多女孩子表现出了恐慌和不知所措，而她们的家长，可能也从未跟自己的孩子谈论过相关的问题。


北京体育大学的郭娴讲师将介绍青春期女性的营养促进的基本知识，带来一系列在家就可以学习的简单、有效的运动锻炼手段。本专栏希望让青春期的女孩子们得到更多的人文关怀和爱护，让家长们教会女孩子们懂得如何保护自己、关爱自己、宣传自己，绽放出更加耀眼的光彩。
