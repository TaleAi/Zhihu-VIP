## 简介
本课程基于失眠认知行为疗法，它是美国睡眠医学会（ AASM ）等多个权威机构推荐的失眠治疗首选方案。中国睡眠研究会理事及副秘书长、南方医院精神心理科主任张斌老师，总结 20 余年对睡眠医学研究和临床经验，针对职场人睡眠问题的特点，打造了这门系统课程。


**你可能被下面的问题困扰？**


**明明很累，**可就是**睡不着**；


躺在床上，脑子却**不受控制**地**想事**；


睡眠很浅，一有动静就特别**容易醒**；


不管睡多长时间，**醒来**还是感觉**特别累**；


睡觉**认床**，住酒店睡不好……


睡不好，工作和学习状态不好，长此以往，还可能会**记忆力衰退、情绪暴躁、抑郁、抵抗力下降、皮肤衰老、脱发……**


**也尝试过各种各样的方法：**


担心失眠**提早上床**；


睡不着也**坚持躺着**；


睡前**喝杯酒**助眠；


服用各种**助眠药品**；


……


然而，这些做法都是**错误的**。


错误的认识和做法，不仅无法改善睡眠质量，反而会让你的睡眠问题**越来越严重**，如何才能快速入睡？


**这门课将教给你：一套科学、有效、无副作用的方案——失眠认知行为疗法**


**让你晚上快速入睡，白天精神百倍。**


  




**从这门课中，你将获得：**


**1 个失眠原因分析模型：**系统分析失眠的原因，找到你失眠的根源


**7 大睡眠调节核心技能：**解决失眠问题的系统技能。


**6 类典型失眠案例及其应对方案：**对有代表性的失眠案例进行分析，并提供针对性解决方案。
