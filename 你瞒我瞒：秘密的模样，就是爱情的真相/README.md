## 简介
有人说，真正的爱情，背后没有秘密。说这话的人，既不明白爱情，也不明白秘密。


秘密是底牌，你不先把牌翻开，往往难见对方的真相。


秘密是软肋，是负担，然而两人共同拥抱秘密，也可以是一种温暖。


本专栏聚焦情感中的秘密，既包括双向暗恋暧昧的甜文，也包括爱而不得的虐文。


  



《赝品》


今年是我跟沈肆之的第四年。


虽然被他叫了四年的徐渺渺，可我其实是颜秋意。


这四年，他让我穿和徐渺渺一样的衣服，让我模仿她的言行举止，甚至他对着我喊她的名字。


我曾经有过疑惑，明明他那样地爱徐渺渺，为何偏偏要隐藏自己的心意？


后来跟他的时间久了，我终于弄清楚了原因，原来徐渺渺是沈肆之的堂弟沈西檀的女友，这三人是青梅竹马，而徐渺渺却只钟情于沈西檀。于是碍于这层关系，她成了他爱而不得的人。


在我之前，沈肆之其实有过几个替身，她们或多或少和徐渺渺相似，她们跟过沈肆之的时间也有长有短，而我，是跟沈肆之最长的一位。


这是我的秘密——徐渺渺不爱他，可是我这个赝品爱他。


  



《青梅呀竹马》


当我得知暗恋的男神和别人在一起了，我灌了两瓶二锅头。


然后晕晕乎乎爬到竹马付思源的房间，捧起他的脸就亲，晕晕乎乎地和他…… 


第二天我落荒而逃，朋友圈却变了天：大家都知道我把付思源给睡了，而且不止一次。


其实……我知道把我压在身下的男人是谁。


  



《心知肚明》


我现在在公司里最怕看见的人是许惟乐。


因为……我做了好几次有关他的带颜色的梦，所以只要见到真人，我就会无耻地开始想象他的身体，让工作对话变得难以进行。


这事说来也怪。


从我对男性身体感到好奇开始，只要喜欢谁就会开始做有关那个人的春梦。


那天许惟乐在公司内部的沟通软件上私信我，让我到他工位上去，说有工作需要沟通。


「你直接打字跟我讲不可以吗？」我有点怕看见他。


「如果打字跟你讲，还要不断给你截图我调整的版式，来来回回的，会打扰到你的工作，所以还是麻烦你直接过来一趟吧。」


来来回回的……天，我好想跟你来来回回的啊……
