## 简介
本专栏的 6 个故事，有关人性真善美与假恶丑，让我们在这些跌宕起伏故事中，窥见人心中单纯动物性与复杂的人性。


人类学家说，我们每个人身上都具有动物性和社会性的二重性，动物性，贪婪、自私、残暴、掠夺等等，是动物发展进化而带来的本性，遵循丛林法则，饿了就要吃，掠夺其他物种甚至同类。


而人类社会是靠人类共有的道德规范来维系，充满约束和条条框框……但更多的，却是像这六个故事里难以下定义的复杂人性，和在人类社会中游荡的不安动物灵魂。


《牛之血》


江城夜市摊老板老王，手艺精湛，为人忠厚，但谁也想不到，他背地里竟然是连犯下十几条人命的冷血杀手，这一切只为了赚钱给年轻漂亮的老婆挥霍。而某夜，任劳任怨如黄牛，甚至不惜犯案杀人的老王，却意外发现自己对老婆所知甚少，背后的隐情也正将他一步步引向深渊……


《马之途》


90 年代，华北平原小城，一起运钞车抢劫案，两个家庭支离破碎。犯罪嫌疑人马识途逃亡二十三年，抓捕未果，年轻警察胡仁义，看着师傅在谢师宴上痛哭流涕，悬案也成了他的心结。更让他难以启齿的是，高中好友的邵坤，正是凶手马识途的儿子。同样怀着警察梦的一对好友，在毕业后渐行渐远。直到某天，邵坤接到电话，逃亡二十三年的父亲马识途，要偷偷逃回小城，他们的命运又一次开始转折……


《犬之梦》


十六岁离家，工伤断手指，出了医院睡大街，差点被人撞死。我的人生总不那么顺遂。差点撞死我的人，是个催债的私家侦探，我跟着他入行八九年，我熬出来，他死了，生活在玩弄我，能握在手里的不剩下什么。一个富二代请我找欠债的老赖，为了钱，我装作不知道，他要找的人，其实是个杀人犯。赚钱路上，总以为未来光明灿烂，谁也猜不到，人生背面，永远更灰暗。


 《雀之山》


1875 年的美国，华人翟阿细年幼时随大哥远赴美国淘金，梦想中的金山却不复存在，大哥在雪崩中意外失踪，阿细在美国大陆游荡十年，俨然已是枪法极准、狠辣毒绝的地道牛仔。大雪封山的小镇上，阿细出手击退印第安人，但命运无常，失踪的哥哥、友情、爱情、黄金以及死亡和厄运接踵而至，狂风暴雪中，快马、猎枪、雪白、血红，阿细只想如雀儿般，飞回故乡的高山。


……


这些故事，有今天，有昨日，有善恶，有是非。唯独不变的是，人性复杂，万物可歌。
