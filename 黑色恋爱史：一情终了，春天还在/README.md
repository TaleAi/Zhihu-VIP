## 简介
听说过很多糟糕的感情故事，源头多是金钱、家庭、私欲，在这个故事集中，作者用日常且细腻的笔触，将种种不堪重新组合，它们可能是黑色的，可能是灰色的，但经历后，谁又能说不会再见春天。


男友什么事都要和我 AA，我该分手么？


男友不求上进，看我努力还骂我，我到底做错了什么？


嫁人嫁的是他的原生家庭，如何和他那得理不饶人的妈妈、姐姐相处？


有一个做摄影师的男友，却把女模特拍得异常性感，他俩有什么暧昧？


......


谈一段情，伤一次心。


陷入感情漩涡里的人，总会碰到各种各样的难题。


其实，跟随心性，又何尝不是一种生活方式呢。


当你想分手的时候，就分手。


当你想恋爱的时候，就去吧。
