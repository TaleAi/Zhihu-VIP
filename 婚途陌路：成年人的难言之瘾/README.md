## 简介
成年人的生活中充斥着各种谎言、迂回、借口和难言之隐……


不必凡事问个究竟，我的绝口不提，是我心里最隐秘的伤痛。


《宠物的觉醒》


备受老公宠溺的我，在生日那天被他当众暴打，从此，我的婚姻被撕开了一道血口子，充斥着家暴、侮辱与贬低。我决定反击，带着女儿们逃离这段地狱般的婚姻。


《我的完美妻子》


娶了一个美女老婆，她却榨干我全部身家去帮扶弟弟，我忍气吞声，只想踏实过日子。可某天我意外得知妻子怀的竟然是她「弟弟」的孩子！一个隐忍多年，被骗财骗色的男人幡然醒悟，开启复仇之路……


《变成女人去爱你》


我有一个秘密，除了父母外只有一个挚友知道：我是个双性人，一个以男孩身份长大的双性人。直到我碰到他，我才第一次开始利用我女性的一面。
