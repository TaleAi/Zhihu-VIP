## 简介
与你擦肩而过的那个都市丽人，妖艳背后却藏着细思极恐的怪谈。午夜寂静无声，唯有心跳加快。


我一直被脱发困扰，在一家装潢诡异的假发店内买了一款逼真假发，整个人都精神焕发了起来。但这假发背后的真相，等我得知时，为时已晚……


——《美人骨》


女孩在背上纹了一只漂亮的蝴蝶，可晚上睡觉的时候总能感觉背后像是有什么东西在拱自己，神情也憔悴了起来。后来在与男朋友亲密的时候，男朋友问她为什么纹了一个狗头在身后，女主发现自己身上的纹身变了样子。


——《变异纹身》


妖艳靓丽背后，深渊欢迎你。
