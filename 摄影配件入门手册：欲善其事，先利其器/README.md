你是否经常为自己买了专业的单反/微单，但拍出来感觉还不如手机而郁闷？


初入摄影大门，你是否面对玲琅满目的品牌和林林总总的摄影配件挑花了眼？


入手相机之后，为一套完整的装备花费不菲，结果却被收了「智商税」，如何避开摄影路上的大坑？


「工欲善其事，必先利其器」，了解并选购合适的配件，不仅能在不花冤枉钱的前提下保护好自己的相机和镜头，也能妥善地应对五花八门的拍摄场景，让拍摄进行的更加顺利，在现场不会「掉链子」。


本专栏由职业摄影师平老虎结合自己丰富的摄影实践与培训经验，从实战出发，由浅及深为大家讲解不同摄影题材中摄影配件的选购与使用要点，帮你用尽可能少的投入获得尽可能好的拍摄效果，让你的照片与众不同。
