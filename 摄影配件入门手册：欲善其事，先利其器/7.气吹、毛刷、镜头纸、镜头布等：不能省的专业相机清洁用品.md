## 7.气吹、毛刷、镜头纸、镜头布等：不能省的专业相机清洁用品
有些朋友，买相机之前各种看专业的评测，下载大图仔细查看某台机身配合某支镜头的画质；入门摄影之后，买了各种贴纸甚至硅胶套把相机保护起来，却在相机、镜头的清理和清洁用品上不舍得花钱。


镜头上有灰尘了，就用嘴吹一下，最多哈口气，或者用饭店里顺来的免费纸巾或者眼镜布，甚至衣襟擦一下就完事了。


其实，如果要最大限度改善相机拍照的画质，在重大拍摄，尤其是逆光、小光圈拍摄之前，利用专业工具，采用专业手法清理相机镜头，以及对相机镜头进行清理清洁，都是不可或缺的。


下图就是相机镜头在拍摄夜景之前没有盖镜头盖，也没有清理，结果正好在这朵白云位置最好的时候拍摄了一张慢门，等拍完之后，才发现镜头脏了，照片废了，也没有机会再拍摄第二张。


![img](https://pic1.zhimg.com/v2-d6c76f47e11542b9842e49eff95f95f6.webp)

用嘴吹是不科学的清理相机的方法
---------------


这点来具体分析一下：嘴里有各种食物残渣和唾液，会造成器材受潮和霉菌繁殖。在没有 UV 镜或者保护镜的镜头上，食物残渣或者唾液都会诱发霉菌生长。在湿度超过 80% 的环境里，镜头长了霉菌，会在逆光拍摄时候大大影响画质，并在镜头出二手的时候损失不少钱（一般是多打二折）。


因此，使用专用气吹、毛刷吹去沙尘，使用专用毛刷刷去顽固灰尘，才是正确清理相机和镜头的第一步。


此外，使用湿性镜头纸，是初学者比较适用的清洁相机镜头的方式。


![img](https://pic1.zhimg.com/v2-c431d222b79d526293dfcf3036a8bd10.webp)

我的相机曾经在洒红节里变成上面样子，回到宾馆，小凡用牙刷加相机专用毛刷、气吹和湿性镜头纸，清理干净。正常使用。


说到气吹，有些朋友门上举手要求抢答了：气吹我有！上次买相机，商家送给我一支！


多次参加摄影活动，总会被送很多气吹，这是我的一些好气吹和送的气吹在一起：


![img](https://pic4.zhimg.com/v2-b7fce72001c80450414e1b19c068d68e.webp)

一般商家送的气吹，售价才几块钱，成本不明，采用抗老化性非常差的橡胶，有些甚至采用回收的劣质橡胶，首先闻起来气味就是刺鼻的气味，而且在吹气、捏放过程中，内部的劣质材料很容易掉渣，反而污染了相机内部。


![img](https://pic2.zhimg.com/v2-6524ac4c2fc5ec19cbe44109fb1fd255.webp)

最可怕的是，这种做工简单、用料劣质的气吹，头部的喷嘴非常不牢靠，最危险的情况是在清理相机内部时候，气门嘴（有些还是金属的）吹出去，会击打在相机的感光元件上，造成污点或者损坏，而感光元件损坏一般是几千元的成本。


还有五年后老化就堪比握力器的廉价不好用的气吹，请注意它的头部是金属的，虽然套了塑料软管，还是很危险，容易碰伤相机反光板或者感光元件：


![img](https://pic1.zhimg.com/v2-691f1ef828d560323b25c7fa7b48a999.webp)

当年在青岛信息城地下，还有某品牌气吹（现在已经不做了），外观色调明快，结果买了几年后，橡胶老化，成了我健身时候的握力器。


**正确的相机、镜头清洁步骤：**


1. 清理相机的环境要在无灰尘、油烟的室内
2. 使用气吹吹掉镜头上的灰尘和杂物，可以同时使用毛刷清理固态沙粒或者毛屑
3. 对顽固的附着物使用炭笔等清理
4. 使用湿性镜头纸或者镜头水滴在一次性擦镜布上，单向擦拭镜头
5. 如果有残留毛屑，使用气吹或者毛刷清理

**不推荐的相机、镜头清洁方式**
=================


前面讲过，有些朋友凑合式的嘴吹、餐巾纸、卫生纸、甚至衣襟等等清理镜头相机的方式，是非常不规范的。


**临时的方法**
---------


如果手头就是没有专业的相机清理用品，如何清理呢？


1. 选择空气洁净度高、照明充足的地方进行；
2. 用一包新的、比较高档的无屑面巾纸纸巾，取出后，把纸巾卷成圆筒状，用端面，代替毛刷，清理掉镜头上的沙粒和油脂颗粒等；
3. 对于 UV 镜或者保护镜上的油脂，可以通过加有几滴洗洁精的干净水清洗，然后用干净水冲洗，甩掉水珠；
4. 重复第二的步骤，制作一个毛刷来擦干净 UV 镜或者保护镜。

**清洁要点和禁忌**
-----------


**要点：**


选择空气洁净度高、照明充足的地方进行；


重大拍摄之前，尤其是逆光、小光圈拍摄、星空拍摄等之前，要提前清理相机镜头；

清洁用品使用完毕，要马上装回到密封的整理收纳包里，避免灰尘、沙粒的污染；


在环境恶劣地方使用相机，尤其是除了佳能 EOS R、EOS R6、EOS R5 之外的，没有机身帘幕的微单拍摄时候，要特别注意有无杂物落在相机的感光元件上；

检测相机感光元件脏污的方法：


使用光圈优先的 f/22 光圈，手动对焦，对着一面白墙或者白纸拍摄，到电脑上，使用 Lightroom 快捷键 Q 启动「污点去除」功能，照片栏勾选「显现污点」，拖动滑块到右边 8 成左右，来判断有无影响拍摄的污点。


![img](https://pic3.zhimg.com/v2-beba422a56f4f959996ec41999c44e01.webp)

相机的感光元件清理，最好请专业人士清洁；


如果从单反取景器能看到灰尘，但是用前面提及的感光元件脏点检测方法，没有发现明显脏污的时候，不用清理相机的感光元件。实在难受的，可以去找心理医生作咨询（开玩笑），因为单反相机取景器取景和拍摄，不是一个光学通道，你看到取景器或者反光板上有灰尘难受，这是洁癖。


**禁忌：**


️千万不要在船只、拖拉机、越野车等的柴油机排烟口、空调机外机前面等环境更换镜头；


️不要在灰尘大的场景下更换镜头；


️使用眼镜布、麂皮擦拭镜头，不是一个好习惯，因为镜头布、麂皮处在被污染的环境中，镜头布会落上沙粒、油脂颗粒、打喷嚏的残渣等，反而污染镜头；


️不要使用高溶解能力的化学试剂比如丙酮、甚至汽油等清理镜头或者有镀膜的 UV 镜或者保护镜表面，容易造成镀膜损伤；


️苹果笔记本屏幕清洁时候要求极高，不要使用本文中的任何清洁剂、湿巾等清洁以免造成不可逆的伤害；


️不要使用小饭店里劣质的掉毛很严重的面巾纸等清洁镜头、UV 镜或者保护镜；


️购买相机、镜头时候赠送的无品牌的气吹、清洁液等物品，建议不要用于昂贵器材的清理，可以用在贴了膜的手机、电脑显示器、键盘、多肉等的清洁上——
