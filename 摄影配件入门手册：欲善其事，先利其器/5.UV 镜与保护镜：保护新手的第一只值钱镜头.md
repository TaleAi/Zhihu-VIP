## 5.UV 镜与保护镜：保护新手的第一只值钱镜头
为什么比较贵的镜头要选配 UV 镜或者保护镜？


我对 UV 镜或者保护镜的定义是：大多数场景下不耽误拍摄的，保护镜头的透明镜头盖。


![img](https://pic4.zhimg.com/v2-7ff1bbac6ee50020221ea2faa4ede28f.webp)

有的时候，各种原因镜头会失手摔在地上，如果没有遮光罩和保护镜，镜头的前组镜片会直接摔碎，导致不能使用，发生在旅途中，接下来的旅行中抱着破碎的镜头和相机没法拍照片，实在是非常悲惨的事。


但是如果镜头前面有 UV 镜或者保护镜，镜头最前端的 UV 镜或者保护镜会被摔碎，牺牲了自己，而镜头往往是被保住了。


![img](https://pic3.zhimg.com/v2-ec66d33126865163bfbb04a3b6b885fe.webp)

有的时候，在浪花飞溅甚至是泥浆甩出的场景下拍摄，为了保证拍摄时机和拍摄效果，镜头就不免被迸上液体甚至泥沙，比如下图拍摄完毕，相机镜片里就都是海水了：


![img](https://pic4.zhimg.com/v2-3dee349f4d52e3f767ba6fac9eda0b4d.webp)

还有的时候，一时找不到镜头盖，挂着相机逛街、吃饭，也不免弄脏镜头。


如果在家，我们清理弄脏的镜头时候，可以使用棉签等粘去泥沙，然后用湿性镜头纸擦拭镜头，甚至使用超声波清洗机来清洗滤镜；但在旅行中，可能就没有条件细致清理，贸然清理可能导致镜头前组镜片划伤等，导致逆光拍摄效果极差，而更换一片前组滤镜，要花费几百元。


这个时候，如果镜头前面装配了价格比较便宜的保护镜，就可以大胆地清理了。反正到了逆光拍摄日出日落等场景时候，保护镜都是要拆掉的。


当然了，比如 EF 50mm f/1.8 STM 这样售价 700 多元的镜头，就没必要再配置保护镜了，如果前组镜片划伤了，再重新买一个，或者维修，也花不了多少钱。


UV 镜和保护镜有什么区别？
--------------


UV 镜的 UV 是就是「UltraViolet」紫外线的意思，UV 镜可以过滤掉特定波长的紫外线，使用胶片、CCD 相机在高原、海边、雪地等环境下来说能减弱因为肉眼不可见的紫外线引起的紫色调。


很多新手朋友问，UV 镜有必要买么？


正确的回答应该是：作为保护作用的 UV 镜应该买。


其实直接选购保护镜，效果也是一样的，起到物理上的防护，替代镜头承受海浪、灰尘等等，但是作为紫外线过滤来考虑，意义不算大。


![img](https://pic2.zhimg.com/v2-9f03160e1349af1742fd23751fb39a51.webp)

**选择要点：**


和所有滤镜一样，UV 镜或者保护镜的选择和相机、镜头的品牌无关。作为镜头上的的滤镜，它的选购只和镜头口径（直径），符号为「Φ」有关。

不推荐的选购 UV 镜或者保护镜的方式
-------------------


不推荐购买价格过高的 UV 镜或者保护镜，有些厂家的 UV 镜或者保护镜，会宣传有多少镀膜，如何使用之后使得拍摄锐度增加、去除雾霾等等。


如果效果那么好，为啥镜头厂家不直接在前组镜片上直接应用呢？


但是在数码摄影的现在，我们在 Lightroom 或者 Adobe CameraRaw 中，略调整「清晰度」和「去朦胧」就可以轻松改善。所以不建议在这方面花太多钱。


还是开始的那句话，UV 镜或者保护镜就是：透明的镜头盖。


所以，遇到逆光、高分辨率拍摄的场合，拆掉 UV 镜或者保护镜，并小心防护镜头就可以了。


UV 镜或者保护镜的使用要点和禁忌
-----------------


滤镜装配到镜头时候，最好是把机身或者镜头放平整，滤镜平放在镜头前，手指稍向下施加压力，平行顺时针轻柔旋转，直到两者螺纹完好契合旋入，顺滑旋转几圈，到头，略用力旋紧。


如果开始时候旋入式后发现滤镜和镜头不平行，旋转受阻，那就是螺纹相错了，要松开之后重新旋入。尤其是超薄、82mm 大口径镜片更容易出现这样问题，千万不可以蛮力拧入，否则可能损坏镜头。


![img](https://pic2.zhimg.com/v2-2a9be4f0aac7a974d79713aecdf4f0f1.webp)

拍摄日出、日落等大逆光场景、日落后夜景的大光比场景时候，最好拆掉 UV 镜或者保护镜，并用气吹清理镜头，小心保护镜头即可。否则保护镜的折射会造成奇怪的眩光影子


![img](https://pic2.zhimg.com/v2-3b4f0dca2c11836b9dd257c89f90336d.webp)

正确的清理方式应该是：在没有灰尘的室内，先用气吹吹掉镜片上的砂尘、然后使用湿性镜头纸单方向擦拭、晾干，清理时候不要对着镜片咳嗽或者打喷嚏。


如果比较脏，可以使用清洗眼镜、首饰的超声波清洗设备清洗。


每次重大拍摄或者重要旅行之前，最好都清洗一下 UV 镜或者保护镜。


沙尘天、土比较大的场合，要注意经常清理 UV 镜或者保护镜。


外出旅行时候，丢失镜头盖是常见的事儿，建议购买副厂镜头盖，日常使用，并放在行李箱中备用。待镜头卖二手时候，再换上原装镜头盖。


**使用禁忌：**


️不要用嘴吹镜片前面灰尘，要使用气吹。嘴吹容易导致镜片长霉斑。


️摄影包中，不要把金属物品和镜头放在一起，避免镜头盖丢失情况下，镜片受到撞击破损。


️使用偏振镜等其他滤镜时候，尤其是超广角镜头、逆光拍摄场景下，最好把 UV 镜或者保护镜拆掉，再装配其他滤镜。


![img](https://pic1.zhimg.com/v2-7a18ebdc9db7aa63fdb54b9d88ffe357.webp)

️UV 镜可能过滤掉流星，在流星雨的拍摄中，最好不要使用 UV 镜。
