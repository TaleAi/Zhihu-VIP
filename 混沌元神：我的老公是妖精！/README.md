二人再见时女乞儿已是后宫贵妃，小和尚成了护国圣僧。女主因命格受尽折磨，却为了小和尚性命不得不留在后宫，就在女主姓名垂危之际小和尚杀入后宫，经历过种种磨难，二人终于跳脱命运修成正果。
