## 简介
在人类眼中，一头猪的宿命就是被人吃掉。但在猪的眼中，向人类献出自己的肉体，不管是用作战场上的炮筒刷，潜在地影响着国际局势，还是作为人类的器官移植供体，帮人类实现永生的梦想，都更像是一种聪明的生存策略。


猪表面上平平无奇，背地里却做了很多惊人的事。比如：


* 猪悄悄影响了二战战场上的胜负。
* 猪使女明星摆脱了素面朝天的困扰。
* 猪可以提高儿童的平均身高。
* 猪让冰淇淋吃起来更加清新爽口。
* ……

可以说，我们每天的生活，几乎每一分每一秒，都在和猪进行着全方位的零距离接触。


正是我们对猪的这种依赖，让猪得以在众多家畜和家禽中脱颖而出，成为世界各地餐桌上的常客，也暂时逃过了物种灭绝的命运。


在这个专栏中，生物学博士、《疯狂人类进化史》的作者史钧再次用推理小说般的行文，告诉你猪的进化故事：


* 行动如风的野猪为什么进化成了好吃懒做的家猪？
* 猪肉为什么独得中国人的恩宠？
* 器官移植、长生不老……猪如何被人类寄予厚望？
* 如果猪消失了，世界格局会发生怎样的变化、我们胃里的空缺由谁来填满？

希望读完这个专栏，你能对我们身边的平凡动物抱有不一样的观察视角，并纠正我们人类的自负和偏见。
