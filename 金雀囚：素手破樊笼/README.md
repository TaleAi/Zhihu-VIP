## 简介
妾有鸿鹄志，不意金雀囚。
红颜岂甘困，素手破樊笼。


潇洒女主到底有多香。


追妻追不回的火葬场，身处后宫却不宫斗的妃子，


贱奴步步为营登上女帝之位，虐文女主清醒以后夺回人生……


  



**「凌波曲」**  

我当过大盛的皇后，太后，皇后。  

这个顺序很明显，我嫁过老的，也睡过小的。  

美名，骂名，我都受过，但那又如何？  

只要我没有道德，就没人可以绑架我。


  



  

**「引银瓶」**  

那日我的夫君带回一位姑娘。  

她精通诗词歌赋，吟得一两句便惊材绝艳，才女之名名满京城。  

她是美明珠，我是鱼眼睛。  

冷漠是他，要休妻的是他。  

可四年前那个跪在冬雨里，任母亲打断了三根藤条，依旧要娶我为妻的少年，也是他。
