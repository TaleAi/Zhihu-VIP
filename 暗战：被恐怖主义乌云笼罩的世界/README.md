## 简介
他们无差别攻击世界。


世界发展史上，恐怖组织与恐怖行动总是层出不穷的，他们或通过绑架要员、民众来胁迫国家，或通过爆炸、撞击来报复社会，给所有手无寸铁的民众，带来暗藏的危机。


我们将聚焦于危害世界的恐怖行动，以及情报机关、特种部队与之的交锋——


* 俄式营救是怎样炼成的：劫持大亨调教「阿尔法」
* 没有尽头的复仇：摩萨德猎杀「黑九月」
* 俄罗斯硬核人质：骗飞机逃出生天，还能反杀塔利班？

救援还是以杀止杀？


看得见的和平下，常常藏着看不见的阴谋与短兵相接。


希望本专栏，能让你对这场持续已久的「暗战」，有更多的了解。
