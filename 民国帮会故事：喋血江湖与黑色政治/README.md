## 简介
了解时代背景，开启一场与民国江湖的深度对话；走进民国黑帮人物的双面人生，从民国江湖了解整个民国的制度变化和观念发展。


众帮之首的洪门，一手遮天的青帮，历来都吸引着人们的关注，却又让人摸不到了解的门路。


一个手不能提、肩不能扛的秀才，是怎么一步步成为威震江湖的「远东第一杀手」？


在那个对女性包容度较低的年代，「鉴湖女侠」为何能够在社会上混的风生水起？


......


本专栏将从清末民初帮会急剧膨胀的源头开始，描绘民国时期的喋血江湖和黑色政治。


另外，在此宏观的历史构架下，采取以小见大的叙述方式，以极具代表性的人物故事为基础，来揭秘民国帮会的内幕，讲述民国帮会的「厚黑术」。
