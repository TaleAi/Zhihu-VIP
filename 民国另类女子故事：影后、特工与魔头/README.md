## 简介
本专栏为民国传奇故事第一系列，主要讲的是民国时期那些震动时代的奇女子，这里有在刀尖上跳舞的隐秘女特工，有柳絮才高的文艺才女，还有风华绝代、八面玲珑的交际花。让我们一起品味每个传奇女子的身世与故事，勾勒出民国风雨飘摇的时代画卷。


说起民国，你想起的是什么？ 


是「桃花灼灼，宜室宜家」这样的婚姻誓言和爱情，还是「我行过许多地方的桥，看过许多次数的云，喝过许多种类的酒，却只爱过一个正当最好年龄的人」，这样的美好诗篇和多情； 


是身着青衣布衫、手持长笔铁杆的文人志士，还是戎马半生、身陷利欲泥潭的江湖军阀； 


是半旧社会的封建挣扎，还是半新社会的歌舞升平？ 


民国是兵荒马乱的，也是人才辈出的，谈起民国总觉得有一种说不清道不明的气质。这个时代的人与事曲折深邃，情与仇纷繁难解，造就出一批能人志士，怀揣着或磅礴或离奇的传奇故事。 


民国故事系列课程精选 50 个动人心弦的故事，帮你还原一个有血有肉的民国时代，走近那些被岁月尘封和遗忘，又无法抹去痕迹的历史人物的内心世界。让我们一起体味在那个动乱又美好的民国时代，那一个个撩动人心的灵魂。
