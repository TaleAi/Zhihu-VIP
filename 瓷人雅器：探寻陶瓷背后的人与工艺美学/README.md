## 简介
本课程由丁焱老师主讲，带你走近陶瓷，倾听它们背后的故事。每一件瓷器，从设计到彩绘、从成型到烧成，这背后都蕴含着民间手工制瓷艺人坚守的决心。本课程将用一个个手工艺人的小故事，伴随着优美的旋律，在娓娓道来的叙述之中让你品味瓷器之美，体会大国工匠精神的传承和创新。


「天青色等烟雨，而我在等你。」


「素胚勾勒」、「釉色渲染」、「色白花青」……


一首《青花瓷》将我们带回那烟雨朦胧的江南，眼前仿佛出现了那些古朴典雅，幽倩美观的瓷器。


说到瓷器，你一定知道景德镇。


在这个制瓷重地，有着随处可见的瓷器商店，精美的瓷器琳琅满目，让人觉得目不暇接。


而这背后，是数以万计的制瓷人，在世世代代传承和发扬着传统制瓷工艺。他们自由地将创作灵感，倾注于瓷土之中，成就点土成瓷的艺术之梦。


他们的故事你想倾听吗？


在这门课中，我们将伴随丁焱老师去走访景德镇手艺人，带你了解你不懂的瓶瓶罐罐，发掘手工艺术背后的故事。


相信即使穿越时光，繁华沧桑，瓷器背后的故事依然令人着迷和神往。
