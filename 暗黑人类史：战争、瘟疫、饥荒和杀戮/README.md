## 简介
顺着人类历史的脉络，探索战争、瘟疫、饥荒和杀戮的另外一面。



2000 年前写就的《启示录》中描述到，天启四骑士来到人间，分别带来了战争、瘟疫、饥荒和杀戮。


直到今天，我们还会偶尔瞥见如幽灵一般笼罩在人间的，四骑士的身影。


记录历史，是为了警醒世人，不要重蹈历史的覆辙。


本专栏将从多个角度出发，讲述生活在和平年代的我们所不知道的人间真相：


* 战争：想要坐稳联合国五常之位，一定要注意什么？
* 杀戮：人类历史上最悲壮的时刻是什么？
* 瘟疫：西方的神权，为什么会坍塌于一场黑死病大流行？
* 饥荒：一粒粮食也能难倒一个国家？

作者远方青木视角独特，知识渊博，擅长旁征博引，将一些看似毫无关联的事件以一种精巧的方式组合到一起。除了流畅的阅读体验，本专栏也会带给你更多的历史新知。
