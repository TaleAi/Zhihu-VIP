## 简介
苏家有女名润，一朝登顶临凤阙。


看不聪慧、不美貌、不受宠的贤德皇后，如何将人间烟火洒满寂寂深宫？


我是皇后。


我不聪明，不美貌。既没有好看的皮囊，也没有有趣的灵魂。没有光荣显赫的家世，没有举世无双的才华。太后不喜，皇上不爱，妃嫔不敬，朝臣不尊。甚至没有自己的儿子。


可我知道，一个无才无貌的女人想要在这世上立足只有一条出路，便是以贤德良善为武器， 唯有将这贤德二字做到极处才会有人真心敬你，即便无才无貌，也可以在后宫容身立足了。
