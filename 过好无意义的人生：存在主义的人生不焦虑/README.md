## 简介
焦虑是生存的一种状态，也是人生必须面对的一种情境。往往是生活中的琐碎小事累积起来，把人慢慢击溃。本门课将与你畅谈西方哲学里一个特别的存在「存在主义思潮」，使你从十大问题中感悟存在主义，探析存在主义先驱的思想历程，打开一扇思辨的大门，用这些智慧反思自我、反思当下生活。


焦虑的现代人如何在自处？  

如何找到存在的价值、并肯定自我的存在？  

面对瞬息万变的社会现实，人究竟该如何过好短暂的一生？  

又该如何在永恒的人生难题中找到自己的答案？     


存在主义哲学关注的恰恰就是生命中最重要、最个人化的问题并提供思路：  

我们是谁？为什么我们是现在这个样子？  

如果我们不信仰上帝，我们能否找到生命的意义？  

我们需要承担什么样的责任？我们要做什么？  


被生活裹挟的现代人，面对世事无常、悲喜剧交替上演、生命意义难寻的生存境遇，像浮萍一般漂浮在各种各样的可能性之间，充满了焦虑和寂寞。


而存在主义将引导我们，在真实的人生处境中，用行动不断寻找自我存在的价值，同时让我们学会为自己的人生负起责任。


我们比想象中更自由，更需要意义，也更需要存在主义。本门课将由人大哲学院讲师雷思温为你讲解存在主义。
