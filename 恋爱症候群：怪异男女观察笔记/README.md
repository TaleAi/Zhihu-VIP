## 简介
失败的艺术家、生活艰难的打工者、冷漠疏离的感情、劳燕分飞的婚姻，还有那些转瞬即逝的火花、不可靠的梦想……少见的当代男女，在都市生活的裹挟下，变得畸形、古怪，但即使有着疾病症候，他们还是坚持去爱。


**另一种泡妞身份**


作为农村来的打工人，开着老板的车，被拜金女误认为是老板的儿子。  

而打工人顺势，用这层身份与拜金女谈恋爱、怀孕，身份暴露后，遭到拜金女的疯狂报复。


**婚恋市场**


我，早上起床，被母亲逼迫必须在下午两点半之前结婚。经人介绍，我去了一块专门的区域，方便被逼迫结婚的人自由结合，这块领域叫做「婚恋市场」。


**83**


陈小姐从小就患了一种病，在她眼中，每个人都变成了一个数值。  

条件好的人，数值大一些，反之，则少一些。原来，大多数人连及格分都拿不到。  

陈小姐认识了 75，立刻跟他建立了婚约，可是结婚前又认识了 83，这成为了陈小姐的心病，与 75 的婚约，也显得不香了。在陈小姐百般纠结之时，更高的数字主动出现了……
