## 简介
揭秘西方媒体口中的苏联奇闻异事。


冷战时期，苏联造出了许多奇葩武器，也做了很多疯狂实验。


西方媒体捕风捉影，制造了苏联正在进行「洗脑术」、「食人恶魔」等非人性实验的舆论。


为舆论战提供弹药的同时，更是渲染了战斗民族的神秘和荒诞。


例如：


**西方传说中的「洗脑术」——巴甫洛夫的「吐真剂」与莫斯科公审**

**睡眠剥夺实验——把常人变成「食人恶魔」的秘密实验**


**现实中的「沉睡山谷」——辐射下的卡拉奇「睡美人村」**


**………**


本专栏将从史实本身出发，还原「怪谈」背后的真实情境。


也将盘点苏俄社会文化的独特性，分析苏联历史环境下产生这种怪谈的政治和社会原因。


一起来看看冷战时期的战斗民族吧。
