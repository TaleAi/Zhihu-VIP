# 艾滋病的传染与预防：正确认识 HIV，告别「恐艾」

## 简介
本专栏由皮肤科医生徐宏俊带来 HIV 与艾滋病相关科普，帮助大家认识、正视与警惕艾滋病，做到洁身自好，正确防护。


如今，艾滋病的发展趋势，已经成为全社会都关注，乃至世界性的问题。处于性活跃时期，且自控能力不够的年轻人成为艾滋病感染的主要群体，特别在某些城市情势尤为严峻。


医生徐宏俊在公立三甲医院皮肤科工作已经 13 年，已诊断 HIV 感染的患者超过百人，他们要么是主动前来医院筛查艾滋病，要么是在进行其他性病的诊疗过程中抽血查出。


得知自己感染 HIV 的患者，痛哭流涕、强制镇定者皆有之。在他的印象中，他们问的最多的一句话是：


**大夫，我还能活多长时间呢？**


很多人不清楚 HIV 感染和艾滋病的区别，在高危性生活后出现任何一点不舒服，都以为自己感染了艾滋病，甚至每个月都去医院检查 HIV，整整一年都在「恐艾」中；有的人则相反，根本不敢去医院，悄悄从网上购买艾滋病的试纸检测；有的人担心隐私被暴露，或是确诊后不知道该如何向自己的家人和爱人坦白，甚至隐瞒自己的病情。


一旦被确诊感染了 HIV，你知道如何进行 HIV 的科学规范化治疗吗？或者当你一旦出现了高危的行为以后，该如何弥补呢？为此，皮肤科医生徐宏俊推出关于 HIV 和艾滋病的科普专栏，希望能让广大读者了解、正视与警惕。


遏制艾滋，人人有责。
