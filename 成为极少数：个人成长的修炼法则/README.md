文章和读者深度共鸣


你以为你整天抱怨只是因为工作不顺，你以为情绪失控只是因为那天心情不好，你以为爱情离你而去只是因为没钱，你以为买不起房只是因为房价太高，所有的自以为是里，都包含着人生失败的种子。
