## 简介
来自暗网的神秘少年、操纵比特币的幕后之手、代表着未来的区块链，逼迫我与时间赛跑。我是莫飞，一个置之死地而后生的创业者，在在举国加快推动区块链技术的时代背景下，我和其他创业者，面临互联网战场的生死厮杀和巅峰对决，我们可以对抗神秘力量，打赢这场商战吗？


莫飞是当代创业者的典型代表，他曾无限接近成功，却遭人陷害，被逐出了自己的公司。几年后，他再次创业，成立新脑公司，并拿到了美国投行的融资邀约。


可融资的前提，要求他去四川的深山密林里，拿下一笔巨额的商业订单。莫飞沿着 GPS，寻找到一座废弃水坝，正当他要解开这里的秘密时，北京警方却在他家楼下，发现了一具尸体。而这具尸体，竟跟水坝有着千丝万缕的联系。


同一时间，新脑公司遭受了大量的网络攻击，而攻击的源头，居然来自于一个由点对点组成的、去中心化的区块链网络。祸不单行，一家名为 Swiftly 的创业公司横空出世，可它的系统、算法、甚至源代码都和新脑如出一辙……


多重打击之下，莫飞面前只有一条路：拿下订单，完成融资，为公司续命。


就在他突破重重阻碍，即将交付订单时，忽然，他主导的开发基地里，发生了一场惊天的爆炸……


来自暗网的神秘少年、操纵比特币的幕后之手、代表着未来的区块链，正义与邪恶、死亡与复活，环环相扣，重重计谋，跨国互联网之战，一触即发，他如何才能赢得胜利？


**我要声明：我不鼓励大家在区块链上花一分钱。在这场击鼓传花的游戏中，希望你不要成为食物链底层，珍爱生命，远离数字货币。**
