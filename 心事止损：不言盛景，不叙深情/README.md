## 简介
狗男人回头了？对不起，晚了。年轻俊美又能干的小狼狗不香吗？
世界那么大，有人明着行恶，有人暗里做渣，而聪明的女孩永远人间清醒。时间会吹走蒙在真心上的尘埃，因为爱不是自我怀疑和自我失去，而是自我成长和自我救赎。



《暗度陈仓》


我对陈维清一见钟情，但他不爱我。我没尊严地爱了他多年，用一场无爱婚姻将他紧紧捆绑在身边。我以为，在一起的时间久了，哪怕是冰块也能被捂热。


我捂了很久都没能捂热陈维清的心，直到他的女神级前女友回来了，我决定放手另寻新欢。可出乎意料，陈维清居然不愿意离婚……


《故作情深》


人人都奇怪，为什么帅气又优秀的邱近云会和平凡普通的我在一起。我以前也不懂，直到我看到了他前女友的照片。


我累了，不想当替身了。可邱近云又回头来找我了。他以为我还爱着他。真的很对不起，我已经翻篇了，我的新男友爱我如生命。


《朋友之上》


我暗恋了八年的男人终于和他女朋友分手了。我以为，我的机会来了，多年的暗恋终于可以暴露在阳光之下了。可我没想到，薛一凡一边跟我暧昧，一边跟女友复合了……


在最失落，以为自己被全世界抛弃的时候，有一束光照亮了我，沈之澜的出现，让我尝到了幸福的终极滋味。
