## 简介
你相信「人定胜天」吗？


中国历史上，有很多传奇故事，让人高呼不可能，写小说都不敢这么写。


但它们实实在在发生了。


**· 一人灭国王玄策**


大唐的使团碰见异国政变，反唐派成功上位，杀光了使团的人，王玄策仅以身免。


于是，那个国家的噩梦开始了。


王玄策没有回大唐，而是辗转借兵，联军，取得指挥权，亲自杀了回来。


然后覆灭了一国政权。


**· 封狼居胥霍去病**


23 岁的霍去病究竟是如何完成了封狼居胥的壮举？


他绕行沙漠究竟是怎么穿过去的？


他偷袭匈奴王庭为何能悄无声息？


这件几乎不可能完成的功业里，其实除了运气成分，还有满满的细节。


**· 力挽狂澜诸葛亮**


白帝城托孤后，诸葛亮是怎么撑起蜀汉的？


在内忧外困的情况下，诸葛亮又怎么打得司马懿畏蜀如虎？


诸葛亮的诸多言语中，「一夫之死，皆亮之罪」比「鞠躬尽瘁，死而后已」更让人动容。

**他们铸造着这片土地的脊梁与灵魂，他们是中国历史中的奇迹。**
