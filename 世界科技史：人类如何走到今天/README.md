## 简介
学习历史是为了解决当下的问题。作者提供了 5 大科技变革的脉络，从文明源头出发，延展出将近 100 个具体发明、理论，来俯瞰整部人类文明史，帮你弄清科技在人类历史中的地位和作用。你可以从科技发展的侧面，找到科学成功的「必然性」，掌握科学思维方法，摆脱错误归因，提高解决问题的能力，做好当下的事，更好地把握科技发展规律和趋势，从而抓住科技进步带来的巨大机遇和红利。


我们身处一个科技爆炸的时代，国家间科技军备竞赛，商业巨头们的专利斗争，都让我们对眼前技术的发展感到眼花缭乱。虚拟现实、人工智能、无人驾驶技术、基因编辑、大数据医疗、区块链和虚拟货币……我们每天在社交媒体上看到的这些概念意味着什么？对我们将产生什么影响？


**解决这些问题，学习科技史是绕不开的方式。**


科技史，能帮你深刻理解科技在人类历史中的地位和作用——它在人类的经济和社会生活中究竟曾经扮演过什么样的角色，有着怎样的发展规律和趋势，如何影响和推动人类文明进程。


回望科技史，把握科技进化的来龙去脉，清楚我们今天所处的位置，掌握关键要素之间的联系，看清世界变化的规律。这个专栏，作者以新颖有趣的角度，重新看待我们生活中常见的技术和发明，挖掘表面看起来毫无关联的领域之间所存在的意想不到的微妙联系，理清科技发展的脉络，让你重新审视人类的宗教、社会和技术之间的关系，思考科技是如何演变的，最终将引导我们走向何方。


【内容亮点】


* **俯瞰概括整部人类文明史**

掌握科技变革的规律，判断清楚我们今天所处的位置，预测未来科技发展的趋势。


作者从人类文明源头出发，以一种技术和发明的全新视角探索人类历史，延展出将近 100  

个具体发明、理论，用一种新颖有趣的科技历史观来阐明科技发明是如何引领人类走到今天，并走向富足、繁荣的未来。


* **专注于探索里程碑式创新的发明轨迹**

「轨道电车为什么让供电网络和家用电器开始普及？」、「为什么现在主要的飞行工具是飞机，而非热气球或飞艇？」、「鼠疫如何催生中世纪欧洲的文艺复兴和资本主义萌芽？」、「为什么我们今天广泛使用的是交流电，而不是直流电？」、「计算机最早的雏形竟然是一台织布机？」、「世界上第一位程序员居然是 19 世纪英国著名诗人拜伦的贵族女儿？」……


以「科技与发明」推动人类发展的历史观，将历史串联起来。将人类发展不同时期的重大发明作为线索、标尺，展开该时期人类经济、文化、政治生活的方方面面。


* **以史为鉴掌握科学思维提高解决问题的能力**

了解历史是为了把握今天、预测明天，你可以从对科技历史发展规律的陈述中，找到科学家成功的「必然性」，通过掌握科学和技术发明之间的逻辑关系，提高自己发现问题、分析问题、提出假设、实验验证和归纳总结的科学思维能力，创造性地解决当下的问题。
