## 简介
和耶鲁大学哲学博士傅佩荣一起学哲学，用哲学回归理性，架构思考问题的底层逻辑，提升个人思辨和批判能力，成就更好的自己。


**哲学有什么用？**


康德将人的认识能力分为的三个层次，就是感性、知性和理性。


我们需要不断地积累知识、保持理性的思维方式，才能化解生活中遇到的种种问题。


而哲学可以重新梳理我们的底层思维架构，拥有思辨和批判的能力，看清事情的本质，找到正确的前进方向，而不是盲从、追随，忘记自己是谁。


**谁讲的哲学，值得一听？**


**主讲人：傅佩荣**


▸ **专业背景，资质过硬**


台湾大学哲学研究所硕士，美国耶鲁大学哲学博士，师从余英时。曾在央视《百家讲坛》主讲《孟子的智慧》。


▸ **知识有深度，讲法很轻松**


如邻家长者一样，把哲学知识溶解到人生中，全面地给你必备的哲学知识，从西方哲学到东方哲学，从苏格拉底到孔子，从神话到教育话题，从艺术审美到生命探索，无所不包，带你走进丰富的哲学世界。


**16 章 259 节海量内容**，无论你在上班路上、早餐桌上、还是晚上睡觉前，轻轻一点，便可以聆听傅老师的智慧。


————————————————


**你将得到：**


1. 绕过复杂的西方哲学思想体系，以人物和名言为线索一步步了解西方哲学史和东方哲学经典。


2. 看透制度、法律、风俗背后的哲学逻辑，更懂人生，才会过好这一生。


3. 让正确的哲学思维方式成为你的本能，对复杂的现代社会做出恰如其分的反应。


4. 提高个人修养和品位，增长见识，提高对抽象理念的鉴赏水平。


————————————————


**适合人群：**


1. 喜欢哲学，但不知从哪里开始学起的你。


2. 想要透彻了解人生，探索命运的你。


3. 想要用科学的思维方式解决生活中棘手问题的你。
