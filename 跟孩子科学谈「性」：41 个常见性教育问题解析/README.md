## 简介
性教育不可缺席，那么，家长们如何从小跟孩子科学谈性？


性是与生俱来的，从出生到死亡，伴随人们一生。同样，性教育在孩子成长的过程中不可或缺。 


曾经，我们的父母或许对性教育选择回避、敷衍；现在，我们希望通过科学的方法，给孩子真正的性教育。 


性教育不古板、不尴尬，家长们也不必害羞。本专栏里，作者就像家长们的一个亲切朋友，结合自己近年来接访的近千件相关案例，提炼家长和孩子常见的性教育问题，就各种家庭性教育中的常见问题，提供了详细而全面的解答，带给家长们全新的性教育观念。 


希望通过这些专业知识和沟通技巧，能帮助父母理解孩子们行为背后的诉求，在跟孩子沟通性的话题时，学会科学、妥当地回答孩子的各种提问。


愿每个孩子都拥有接受性教育的机会，也愿父母在跟孩子谈性时，不再因种种原因而难以启齿。


祝你阅读愉快～
