## 简介
甜、虐、权谋……古言控不能错过。


这是一个女主视角的古代言情故事合集。


在她们的故事里，会遇到心心相印的恋人，相交惺惺相惜的朋友；也会对上狠毒的对手，受制于捉弄人心的命运。


但最终，她们总会成为自己最喜欢的自己，过上想要的人生，活出独一无二的精彩！


《男主养成攻略》


穿越后，我成了原书里养育男主的世外高人，在他遇到女主后就该功成身退。


诡异的是，助攻男主复仇的女主迟迟没上线，本该她干的活全压在了我身上。


直到那天，在被男主拉着去京城复仇的路上，我偶遇了一个跟我长得一模一样的男人。


而他，是原书里女主的双胞胎哥哥……


《愿者上钩》


我是最受宠的公主，本来只想混到出宫立府，过上逍遥自在的大好生活。


但没想到因为太受宠，在皇子们斗得死去活来之后，我一跃成了摄政的长公主。


为了帮助年幼的小皇帝坐稳江山，我和太傅开始合作，合力教养小皇帝。


但在这日复一日的「工作」中，我和太傅曾经被皇帝剪断的情缘似乎又要连起来了……
