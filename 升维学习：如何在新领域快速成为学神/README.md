## 简介
进入新领域，一定要从 0 学起吗？



升维学习，指的是将学习的要素，用升维的方法去解构和分析，使学习者跳出眼前问题的限制和常规解法，快速达到学习目的，并能灵活运用的学习方法。


古希腊哲学家亚里士多德有一句名言：「每个系统中存在一个最基本的命题，他不能被违背或者删除。」这个基本命题，就是所有复杂理论根植的土壤。这种观点被埃隆·马斯克疯狂推崇，他是一个善于将事情升华到最根本的真理，然后从最核心处开始推理的升维学习者，这为他解决了无数的商业难题。


在新概念层出不穷的「元宇宙」时代，我们更需要掌握升维学习法去面对一个个崭新未知的领域。


那么，作为一个普通学习者，我们应该如何找出领域所需的基本命题，又该如何用升维的方法，成为领域内的学神？


本专栏从方法论、实操和精进三个角度，共 8 篇文章，向读者介绍先降维分解，再全面升维的学习方法。
