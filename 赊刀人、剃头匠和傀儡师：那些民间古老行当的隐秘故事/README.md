## 简介
三百六十行，行行出状元。赊刀人、赶尸人、剃头匠，这些曾经十分古老的行当，各自有着悠久的传奇，不为人知的隐秘，却逐渐消失在现代人视野中。本专栏揭秘民间十大神秘行当，用十个奇诡异事，带领你了解它们的神秘真相。



赊刀人从不收钱，只赠送一则「预言」，等到预言成真的那天，再上门收刀钱。我二叔没有收不回的刀钱，直到那天，预言成真，他却再也当不了赊刀人……


 你听说过湘西赶尸人吗？ 我的一个朋友，恰恰就是一位赶尸人。他们选择夜半出行，白天休息，行走之地，大多是荒郊野岭……


他家世代都是傀儡师，操控木偶演戏的师傅，一场农村「天价彩礼」骗婚事件，让他把祖传傀儡师的秘方用错了地方……


关于这些传统匠人的民间传说，从未消停过，或神秘离奇，或荒诞不经，以下，就是这些神秘的民间行当背后，不为人知的故事。
