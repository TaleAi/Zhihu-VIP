## 52.永远要有 B 计划
一个领导者必备的素质，就是必须永远要有 B 计划，不论是在用人上要有备用人选，还是在任何一个你自己执行的计划方案的时候要有 B 计划，甚至你下属执行的方案，你也要有 B 计划，B 计划的目的就是要保证在出现任何情况的变化的时候，变故的时候，意外的时候，你都有一个后手准备来保证你主体目标的达成。


B 计划是什么？首先为什么要有 B 计划？因为我们做企业、做事业都有一定的冒险性，凡事，百分之百的有把握的事这是上帝的事，百分之八九十有把握的事是行业领军人物他能够把握的机会，而对于创业者，对于早期的公司来讲，百分之五六十把握的事就应该去努力，去争，去抢。失败了怎么办？我做决策的习惯是，首先设想最坏的情况是什么，就如果我失败了会出现什么样的情况。这个最坏的情况如果出现了，我会怎么样？我还能不能够有东山再起的本钱，还是说我败了，我肉体上就被消灭了，而我是比较激进型的这种决策风格，我认为我首先要保证的是，如果失败了，我还留有一点点东山再起的本钱，这是我能承受的底线，在这个底线之上我就会激进地去选择和决策。当然有的人会相对保守一点，他会选择说，如果我败了，最坏情况下会损失到什么程度，我就同意去做这样一个选择，如果最坏情况可能会超过这个损失程度，就放弃这个机会，不管你是激进型的选择还是保守型的选择，我们做任何一个决策都有可能失败，怎么样能够让我们最终达成成功呢？B 计划非常重要。


首先 B 计划可以保证我们的成功把握，如果我们只有 A 计划，一旦遇到变故和意外，我们就失败了。如果我们 A 计划之外还有 B 计划和 C 计划，首先它就可以加大我们最终达成目标的这种可能性。另外一个，当我们有 B 计划的时候，我们在执行 A 计划也会相对的顺畅，所以作为领导人，一定要保留一个习惯，就是凡事都有 B 计划。首先是你自己亲自执行的方案一定要有 B 计划，哪怕 A 计划把握性再大，再完美，你手里边必须我有 B 计划，这个 B 计划在战争中就叫做战略预备队，一个优秀的指挥官永远会留有预备队，甚至是仗都打完了，打胜利了，这个预备队还没有动用，它永远不会把预备队派到战场上去，因为一旦他把预备队派到战场上去，他就没有任何的后手机会了，所以往往是到最关键的时刻，才把预备队派上去，没有到最关键的时刻，他预备队就不派，也许战斗没有到最关键的时刻就取得了胜利，预备队就一直以逸待劳。


拿破仑著名的失败案例滑铁卢，其实打的就是预备队，当时在滑铁卢这个小村庄，拿破仑跟英国的惠灵顿将军两军激烈的交锋，激烈的争夺，谁也战胜不了谁，这时候双方都在等待自己的援军，而拿破仑的援军是内伊元帅率领的法国的骑兵，但是他没有赶到，而是普鲁士的将军率领的普鲁士军队先赶到了战场，所以滑铁卢，拿破仑就败了。这就是援军的重要性。


在战争中，援军就是每个指挥官留下的预备队。在我们企业经营过程中，当然它不是你死我活的战争，我们的预备队是什么？我们的预备队就是任何一个计划都要有 B 计划，B 计划就是我们的预备队。B 计划的目的是在 A 计划执行不下去的时候马上启用 B 计划去达成同样的目标，所以凡事要有 B 计划，首先是自己去负责的任何一个项目都必须要有 B 计划。


**第二个要有 B 计划的方面就是你的下属的任何计划你也要有 B 计划**，不管下属有没有 B 计划，你作为上级，你必须要有 B 计划，来防备一旦下属的 A 计划、B 计划甚至 C 计划都失败了之后，你的 B 计划是怎么样的。你的 B 计划有可能是去救他，让他的计划翻盘、成功，也可能是放弃他的计划，在他的计划失败情况下你用一个什么样的方案来达成你的计划。如果没有 B 计划，就等于把你的成败，把你的未来完全寄托在下属计划的成败的基础之上，这对一个指挥官来讲是不负责任的。一个伟大的指挥官应该是永远把命运牢牢的握在自己的手里，不应因任何人的计划失败和变化来影响自己目标的达成，无论是自己的下属，自己的友军，甚至是自己的合作者，任何一个方面，他们的计划变了或者他们变卦了，都不会影响你的目标的达成，这才是一个伟大的指挥官。


**凡事要有 B 计划的第三个方面就是人员上要有 B 计划**，你再信任的一个人，再看好的一个人，你也要有他的备用人选，你要防备一旦他不干了甚至是他生病了怎么办，尤其是越关键的岗位，你越要有手头的 B 计划人选，如果没有 B 计划人选，你的风险就非常大，所以我曾经提炼总结过，作为优秀的领导者有三个小工具来提升自己的领导能力，因为领导能力很大程度上是天生的，后天的训练各有各的流派，我认为有三个是非常有助于提升你领导能力的小工具，一个是向前看，就是你要有预见性。另外一个是向后看，就是你要学会复盘，不断地总结经验教训，在自己的经历中去提升自己的功力。还有一个就是 B 计划。凡事你只要有 B 计划，你心里就不慌，所以 B 计划不论是对于战争来讲，还是对于企业经营来讲，都非常之重要。我喜欢永远把命运把握在自己手里，把命运把握在自己手里的核心就是你凡事都要有 B 计划，不论是你自己做的事还是你下属做的事或者是你的安排的一些下属的人选，方方面面都要有 B 计划，B 计划其实也是预见性中的一种，因为你预见到了任何一个事情都可能出现让你意外的情况，所以你永远多准备一些冗余和备份，来以便应对这样的一些变化，来确保自己的最终成功。


可见作为创始人来讲，你是一个需要思考非常多的人，也是需要一个做事非常多的人。我们作为创始人，作为领导者，要做的事不是亲自撸胳膊、挽袖子自己去干，也不是替代下属去干，最重要的是要去干下属干不了的事情，需要看到下属看不到的地方，做下属做不了的事情，并且时刻做好准备，当下属出现马失前蹄的时候，自己如何用其他的方案把它补上。


所以凡事要有 B 计划，这是对于一个领导者非常重要和关键的事情，如果我们没有 B 计划，我们永远是心里不踏实的，任何出现的意外的情况都有可能导致我们达不成目标，而如果当我们有了 B 计划，我们不但达成目标的把握性增强了，我们在执行 A 计划的时候，我们的心态也会更加平和，我们的动作也会更加的成熟和流畅，反而会加大 A 计划成功的可能性，就像一个关键的岗位，如果你有 B 计划的人选，这个岗位它本身的压力也不会大，你也不会担心这个岗位的人的心态、情绪各个方面的变化对你造成的影响，尤其是现在企业竞争中彼此之中挖人的情况非常的激烈，如果你任何一个岗位都有 B 角色，你就不会太担心说这个岗位一旦被人挖走，你的事情会不会垮下来，而这个岗位他知道，他永远有一个 B 角色在儿等着、候着他，他可能也不会轻易地被人挖走，而且他的工作往往也会更加努力，因为人都是这样，当他发现他自己不是可替代的，他就会骄傲自满，他就会松懈，甚至是说在别人来挖他的时候，他就会跟公司谈一谈条件，而当他发现他这个位置有很多人盯着，有很多人可以替代他的时候，他就不会有这种居功自傲的想法，而且他自己会加倍的努力，所以 B 计划其实是非常好的一个工作习惯，有了这个工作习惯，不但可以让你在出现各种意外的时候也保证目标的达成，而且会降低意外的发生。
