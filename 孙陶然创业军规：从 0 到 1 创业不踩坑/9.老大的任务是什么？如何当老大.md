## 9.老大的任务是什么？如何当老大
一个队伍的老大是这个队伍的领路人，也是主心骨，所以显而易见，老大的第一个任务就是做所有事情的最后的后援，所有的人都可以说我已经尽力了，我没有办法、解决不了；我要不就放弃了，要么呢，就请求支援，只有老大不可以说这句话。因为天塌下来要有高个儿顶着，老大就是队伍中个子最高的那个人，所有人都可以放弃，老大不能放弃，老大一旦放弃，就意味着整件事情放弃了，所有人都可以去请求支援，老大无人可请求支援，因为他就是最后的防线。


所以老大的任务是什么？老大的任务就是去解决所有别人解决不了的问题，扛住所有别人扛不住的压力，这就是老大的任务。当然了，这是根本上的任务，那具体而言，一个组织的老大，他该干什么呢？**核心是两件事儿，找方向和找人。**


找方向，首先老大必须知道我们要去哪个方向，这是一切事业的核心根本，如果我们不知道我们应该去哪个方向，东一榔头西一棒子，或者是东边撞撞西边撞撞，那我们永远不可能到达目的地。当然，如果想到达目的地，仅仅知道方向还不够，我们还必须知道方向和方位有很大的差别，方向是一个面，方位是一个点，真正能够引导我们到达目的地的是方位，我们仅仅知道大方向，那么我们可以向这个方向去走，但最后能否到达目的地，取决于我们能不能够找到这个目的地的方位，实际上，绝大多数的失败都不是不知道方向，而是不知道方位，绝大多数到不了目的地都是在目的地周围来转悠。所以知道方向不难，不是本事，考校我们本事的是具体的方位，我们能不能在一个大方向之中找到精准目的地的方位是我们成功的关键。


有很多老大很粗心大意，认为我已经知道了方向在东北偏北，那我们往那边去走就好了，但是往往徒劳无功，就是因为不准确，而只有花心思找到精准的方位，我们才能够成功，这是老大该干的第一件事儿，就是要为这个队伍指明方向，找到方位。就像一个船的船长是老大，他最重要的是知道我们如果要从伦敦到纽约正确的航线是什么，在这个航线中要怎么走，才能够避开航线上的暗礁冰山，并且走一条比较短的、比较节省燃料的一个航线到达目的地，这是一个船的老大最重要的任务。至于说锅炉烧的水平怎么样，或者是餐厅的餐饮水平怎么样，这些都有那些水手和厨师去管。


第二个老大应该干的事儿就是他要知道，什么人可以辅助他来把这艘船带到目的地，因为我们要做事儿都不是一个单打独斗的事，我们需要一组人去干成一件事。既然需要一组人，那么你就需要几个直接向你汇报的助手，可能是班子成员，也可能是团队成员，但不管怎么说，你需要有人来帮你，因为现在任何事情都不是一个人可以单挑搞下来的。作为老大最重要的一件事情就是你要知道你需要什么人，例如说你要驾驶一艘船横渡大西洋，那你应该知道需要二副，你需要水手，你需要轮机长，甚至你需要通讯员，你要知道为了到达目的地完成使命，你需要什么人？


第三个，就是你要能够找到这些人，你知道你需要这些人，你找不到这些人也没有作用，我们历史上大家都知道，刘邦、刘备这些人的故事，其实刘邦和刘备都是一个典型的老大，就是论单兵技能、论个人专长，他们可能不如很多人，例如刘邦说运筹帷幄，决胜千里之外他不如张良，统筹粮草保证后勤他不如萧何，领兵打仗他不如韩信，但是最重要的他有一个能力，他知道他需要韩信萧何张良，并且他有本事能够说服他们出山，纳入到自己的麾下去辅助他，所以刘邦是老大，他最后能夺得天下。


当老大，就是要知道你需要什么样的人，并且能找到这些人。往往是这样你越需要的人都是越有能力的人，而越有能力的人就会越有个性。所以我们古代有很多这样的故事，三顾茅庐、程门立雪，几次去请姜子牙出山，凡是这种要当老大的人都应该有一个宽广的胸怀，应该能够礼贤下士，百折不挠去做什么呢？最核心的不是自己去干某一件事情，而是去请到能够帮助他干这件事情的人。


在找人这个方面其实刘邦的例子是非常著名的，他通过各种各样的努力把需要的人请到了，也是这样的一个人三顾茅庐，按照很多人的想法，我已经亲自去请你了，你还不来，那算了就放弃了，但是要知道你如果放弃了这个人，那么很可能你也就放弃了你要做的事业。


再换个角度来看当老大，如果连请一个人都请不到，那你怎么样能够去率领一群人成就一番事业呢？当老大如果连一点点的委屈都受不了，那你怎么能够组织到一大群的精兵良将、能人志士来服务你、帮助你一起去达成目标呢？所以当老大能够找到你需要的人，这是你必须要干的一件事儿。


第四件，当老大要干的事儿就是要能够让你手下的这些能人愉快的协同工作，还是那句话，越是有能力的人个性越强，脾气越大，往往也倾向于单打独斗。但是一支部队要想达成他的目标就必须能够协同配合、各展所长，所以老大一个非常重要的任务，就是能够让自己手下的这些非常有能力的人各自发挥自己的所长，并且能够愉快地配合和协同。你要去调动他们的积极性，要去协调他们的矛盾，解决他们可能出现的问题。怎么样能够做到这一点，这是老大非常需要花心思的一件事情。


第五个，老大要做的事情就是永远要有 B 计划 C 计划，这个也是我自己的一个管理心得。作为一个领军的老大，你做任何事情必须要有 B 计划，你使用任何一个人，必须要有 B 计划，总而言之做任何事情你都必须有备选方案，这样才能够让你永远不翻船。


我们做任何一件事情，计划得再好，一旦开始实施都有可能不行，因为情况是瞬息万变的，在军事上有一个著名的话叫枪声一响，预案作废。军事上你要准备一场战役，可能提前要几个月做准备，反复的在沙盘上去制定进攻的计划，但是只要战斗打响，敌人并不听你的指挥，原来设想你从左路进攻，敌人会做出什么样的反应，但是往往敌人做出的反应跟预想不一样，这个时候你的原计划就必须去变，你如果没有一个备选的方案，不能够随机应变，那你的进攻就会失败了。


我们经营企业也一样，我们规划的再好的一个产品，再好的一个营销计划，再好的一个市场规划，一旦开始实施，很有可能遇到这样那样的变数让我们原计划无法实行，这个时候，你有没有准备好 B 计划，马上调整实施 B 计划，如果 B 计划不行，马上调整实施 C 计划？这是作为老大非常重要的要干的事儿。也许具体的执行人，他会把全部的注意力和精力放在他想去做的这件事上，但是你作为他的上级，你必须有一个备选方案。


在用人上也一样，我们每个人都管理着很多人，这里边有主力部队也有辅助部队，任何一个公司都有可能某一条产品线占了公司整个收入的百分之六十百分之七十，那么这样的产品现在领军人物就非常关键了。很多公司碰到这种情况，一个重要分支机构的人跳槽了，导致公司整个阵脚大乱，业绩下滑，作为老大，你永远在使用任何人的时候都必须有他的备用人选，就是这个人一旦不行了，有没有人马上去顶上，而且顶上的时候能不能够延续原来的工作，甚至能够去做得更好，这个我们就叫做备用的人选。


在拉卡拉前年开始推行一个计划叫二把手计划，就要求任何一个部门都必须有二把手，而且要让上级的上级看到二把手是谁，我们也会针对二把手做一些培训，并且有意识的安排二把手去接触全面的工作，目的有两个：第一个就是一旦一把手出现了变化，不论是组织上要把他调走，还是说他自己有了变动，二把手要随时能够顶上来，保证我们任何一个部门工作的延续性。第二个目的就是因为二把手的存在，也会激励刺激一把手，不断地进取，做得更好。如果你觉得小富即安，既可以了，如果你觉得做得差不多就行了，那你身边有一个虎视眈眈的二把手，他随时准备上位，那这个时候有就逼迫你不断的往前去走。当然我们还有另外一个想法，就是希望一把手能和二把手配合工作。我们讲要用一个班子来工作，为什么呢？因为只有用一个班子来工作，才能够发挥大家的力量，才能够抑制一把手的冲动和独断专行。如果部门和机构比较小，你刻意的去搞一个 3 个人 5 个人的班子，那就有点就有点装了，但是最小的班子是两个人，就是有一把手二把手两个人来搭班子来，形成一种群策群力，形成一种对冲动和独断的制约。


所以二把手其实就是我们在人上的 B 计划。我们不但做事要有 B 计划 C 计划，用人上也要有 B 方案 C 方案，这样的话才能保证我们的终极目标可以实现。


所以总结一下老大应该干什么，重要的不是钻到某一个具体的事情中去。我历史上曾经这个遇到过有的老大因为是技术专业出身，他对技术非常的热爱，所以他经常把时间大量的花在跟某个程序员讨论矛盾程序应该如何开发，或者是说某个页面应该如何设计上这些。我认为老大应该关注而且在不同的阶段可能也应该花主要的精力在这个上面，但是前提是你把老大该干的事干好，如果一个组织的老大没有把他该干的事干好，这个组织如同盲人骑瞎马是非常非常危险的，你想一个组织没有一个胜任的老大在老大的位置上，那么这个组织不就像一辆无人驾驶的汽车在一个高速公路上行驶，那是非常非常危险的。


老大到底应该干什么呢？就是刚才我们谈到的这一点，第一个指方向和方位，你必须为你的这支部队找到方位，并且指明了带领大家往这个方向去走。第二个你必须知道要成功要达到目的地，需要一些什么样的关键人。第三个你必须有能力把这些人找来加入到队伍之中来工作。第四个你必须让他们能够愉快的协同工作。第五个任何事儿任何岗位你都必须有 B 计划 C 计划，这 5 件事情就是一个老大应该干的事儿，这些事你干好了，这支队伍就会不断地从一个胜利走向另外一个胜利，这些事你没有干，这个队伍是不可能胜利的，不论是对于初创的公司还是成熟公司来讲都是如此。


那简单的来讲，我曾经提到一个观点，初创的公司 3 件大事，找方向找人找资源，这就是老大核心要做的事儿，那刚才我们谈到了方向，谈到了人力和资源，这方面其实人是最重要的一个资源，另外一个资源就是融资，初创公司的融资也非常重要的，这也是老大应该亲自去干的事儿，但是核心是如果你有了方向，你就很可能有人，如果有了方向有了人，钱应该上门来找到你，或者你出去找钱的时候，别人也会非常容易的认同和投进来。


老大应该做好自己该做的事儿，我们喜欢说一句话叫不要用战术上的勤奋掩盖战略上的懒惰，这句话是什么意思呢？就是老大的核心是制定战略，你的精力重点应该放在制定战略上，而不是放在每天忙那些不重要的事情上。有很多老大每天第一个上班最后一个下班，每天坐在办公室里面任何一个字都他要亲自去签，他认为这样是负责任，我认为这是最大的不负责任。我还有一个观点，我认为一个老大一个领军人物，他对组织最大的投入应该是他脑力上的投入，他心理上的投入，而不是他身体上的投入。虽然很多人看到我每年有很多的时间在全世界各地去旅行，并且我也热心地去参与一些社会活动，他们认为你是不是没有把全部的心思放在整个拉卡拉的工作上？我说不是因为我 7×24 小时都在思考拉卡拉的事情，只不过我思考的是老大应该思考的，例如方向的问题，人的问题，B 计划的问题，而我没有 7×24 小时的坐在我办公室里边去处理那些不应该是老大做的事情。所以作为老大来讲，要想成为一个合格的老大，首先要搞清楚我们应该干什么，不应该干什么，我们的核心任务是什么。
