创业是充满问题的荆棘之路，从零到一、从无到有的把一个公司建立起来，并让公司生存下去，赚取利润很难，让公司脱离温饱线，健康可持续地发展则更是一场严酷的长征。


你需要一款接地气的国民创业实战教程。


为什么要听孙陶然讲？
----------


* 他是一名连续跨界创业者，覆盖了金融服务、基金、电子产品、媒体、广告、公关等众多领域；
* 他创办的蓝色光标成为全球排名前十的中国本土公关公司、拉卡拉成为综合性金融服务上市公司；
* 他创作的《创业 36 条军规》，销量超过百万册，被誉为中国的创业圣经。

你将获得什么？
-------


* 从创业实际需求出发，提供清晰可行的对策；
* 抛开浮躁，句句干货，带你借鉴前人成功、吸取失败经验；
* 轻理论，重实战案例，给创业者更实际的解决方案。
