## 简介
我们的 960 万平方公里从何而来？


先来看两幅地图：


![img](https://pic3.zhimg.com/v2-3f10e335590d809998a011630d15bbe5.jpg?source=6a64a727)

* 明万历十年（1582 年）疆域示意图（土黄色区域为明领土）。
* 图源：《中国历史地图集》（谭其骧）

![img](https://pica.zhimg.com/v2-56aa21e1bf6e76ea360698f5675ffdfd.jpeg?source=6a64a727)

* 清嘉庆二十五年（1820 年）疆域示意图，其中各省区划分与如今相似。
* 图源：《中国历史地图集》（谭其骧）

从明到清，中国正统王朝，从偏居南部，到拥有约 1300 万平方公里宏伟版图，版图扩张约 3 倍。


我国现在有 960 万平方公里的辽阔领土，位居世界第三，清朝功不可没。


**当时的领土扩张是如何实现的？**


**康熙、乾隆、雍正各自为扩张做出了哪些贡献？**


**本专栏带你走进清朝的大扩张时代。**
