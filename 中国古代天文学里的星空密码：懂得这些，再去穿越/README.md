## 简介
本课程由北京天文学会副秘书长张杰老师主讲。课程中张杰老师会从古代生活中的天文学、古人心中的星空、占星与天象、文化基因底层的古代天文学 4 大模块出发，带领你一起探寻天文学的奥秘，共同了解那个陌生的领域，你会发现我们每个人的身上都有天文学的痕迹。


 本课程从中国古代文学出发，带你领略天文学的秘密，探究占星与天象，发现每个人身上的天文学痕迹。 


本课程由北京天文学会副秘书长张杰主讲。 


张杰老师将本课程分为古代生活中的天文学、古人心中的星空、占星与天象以及文化基因底层的古代天文学 4 大模块，带领你一起了解中国古人是如何利用天文学确定方向，测量时节，占卜吉凶和治理国家的。 


并在课堂上从实用层面来深入分析天文学的存在价值和在生活中发挥的作用。 


为你介绍中国古代星官和三垣二十八宿系统，让你清晰地掌握中国古代的天文学基础知识。 


全方位剖析中国古代的占星和术数理论，说不定你也可以成为一名「占星师」。 


为你讲解古代天文学对中国文化和中国人的深层影响，一起发现我们日常生活中暗藏的天文学知识。 


让我们感慨人类的渺小，感叹宇宙的浩瀚之余，像哲学家那样思考世界观。
