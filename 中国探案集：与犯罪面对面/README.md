## 简介
我曾目睹过 10 个罪犯的人生，贩毒、杀人、强奸、窃听、盗窃……兽性和人性在他们体内博弈，为人的底线不断在刷新。10 个与恶相关的故事，读懂阴影，才能看见光明。


我少年时代的兄弟是个强奸杀人犯，而我已经成为他口中的「叛徒、警狗」，他把仅有的四发子弹瞄准了我；


阿龙是我从警后送走的第一个死刑犯，死刑前一晚，他和我讲述了他先做神枪手又成为富二代，最终成为杀人犯的一生；


为了保外就医，他「得了」十几年偏瘫，脚踩图钉面不改色，我们敬他是监狱中的奥斯卡影帝……


一个警察的工作，要经历多少艰难凶险；一个少年的成长，又要面临多少悬崖、岔路的考验；一个犯人的罪恶行凶过程，又有多少丧心病狂，多少身不由己？


这些罪恶值得被铭记，防范未知，警惕人性。在不为人知的角落，那些无法克制的疯狂心跳和无法抑制的绝望泪水，都值得被记录下来。


读懂阴影，才能看见光明。
