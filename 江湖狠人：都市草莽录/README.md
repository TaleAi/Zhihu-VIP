## 简介
讲述那些身怀绝技与梦想的小人物。


这些人，生活在现代城市的「灰色地带」，他们有的是隐姓埋名的习武之人，有的混迹夜场却身怀绝技，有的坑蒙拐骗、无恶不作。


那个世界，有热血，有遗憾，不变的是，他们个个都是「狠人」，个个都是传奇。


在繁忙的都市生活中停下来，看欧阳乾为你带来的热血江湖，仿佛那快意恩仇的世界，从未远去。


  



自从棍子来到这个城市，就一直过着苦行僧式的生活，因为他的身份是一个「青年武术家」，背负的责任说起来很好笑：光大自己的门派。他出身于北方一个小拳种，门下没几个人了，他就是整个门派的希望。我们都对他的理想嗤之以鼻，但风情万种的小腰却被棍子给深深吸引了，无奈棍子并不领情……


——《一个人的门派》


我刚毕业的时候在酒吧干人体沙袋，供顾客击打发泄。一个老板看我格斗水平不错，便邀请我去打黑拳。在一次比赛中，我遇到了以前的队友，眼睁睁地看着他被人打死。至此，我恍然大悟，便写了一封检举信，却没想到老板手眼通天，他没事，我却被抓了起来……


——《人体沙袋》
