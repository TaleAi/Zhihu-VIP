## 简介
不通过这场考试，我们都会死。


某天，你忽然失去意识，再醒来时，发现置身于一个全封闭的考场当中。


而考试的内容，是杀死考场里的其他人……


近年来，各地多次出现名为「死亡考试」的异象，背后的主办方并非人类，而是人类无法理解的未知力量。


被迫参与考试的人当中，有一批不守规矩的社会边缘人：


我是作弊肄业的女大学生，遇见了贪污停职的警察、想要通过死亡考试暴富的疯狂赌徒、为了自保可以随时出卖队友的胆小厂妹……


为了活下去，为了抓捕考试主办方归案，为了各自的目的，


我们踏上了参加每一场死亡考试的旅程。
