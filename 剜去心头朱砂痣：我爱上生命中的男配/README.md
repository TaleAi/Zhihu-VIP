## 简介
如果爱情是道选择题，你会选白月光，还是朱砂痣？


**《摘星》**


我心里有位温润儒雅的郎君，我觉得世上再没有比他更好看的人，但我却被父亲执意嫁给了我不喜欢的公子。公子待我极好，甚至会对我发些小脾气，兜兜转转，最后我才发现，心中的朱砂痣不过是空长了副好皮囊。


  



**《又岂在朝朝暮暮》**


我青梅竹马的公子心上有别人。为了躲避皇帝采选，我和他成为合约夫妻，约定成婚一年后和离。大婚前夕，公子翻墙进来陪我喝酒，告诉我他心上有人，让我猜这个人究竟是哪家姑娘，谁家小姐。半醒半醉间，他留下一句，「是你」。


  



**《婉归迟》**


我被迫嫁给了父亲的仇敌摄政王，朝夕相处生出了感情。没想到，王爷突然从外地带回一个女子，藏在府里。原来，那个女子是王爷青梅竹马的白月光，而我不过是替代品。
