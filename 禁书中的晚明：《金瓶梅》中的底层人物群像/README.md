## 简介
《金瓶梅》才是真正的明史。


**（作者说：因近期俗事缠身，上段时间专栏更新有点慢，感谢大家的谅解和耐心等待，之后我会尽量保证按一周一节的频次更新，非常感谢大家对专栏的喜爱和支持，我会加油更新的～）**


在风月禁书的外衣之下，《金瓶梅》其实描绘了一幅晚明时期的底层人物群像图。


贪婪的官员，无耻的嫖客，淫邪的妻妾，堕落的文人，龌蹉的僧人**……**


这些市井人物，把晚明时期上上下下、里里外外的丑恶，都翻了出来。


该专栏用一种轻松诙谐的语言，讲述这些底层人物在情欲和物欲中挣扎的故事，刻画了这个时代的世风和世相。同时，着重分析这些人物的形象与动机，并对其生命价值和生存形态进行思索。


**比如西门庆是纵欲无度的花花公子，但也是个敏感的孤独者。**


对于女人，他来者不拒。大龄剩女吴月娘，官妓李娇，私娼卓二姐，寡妇孟玉楼，富婆李瓶儿……都被他收入「后宫」。他是有精神创伤的，父母双双离世时，他才 17 岁，被迫匆匆结束东游西逛的富二代生活，走上当家人的位置。这让他成为一个敏感的孤独者，终其一生，他想要的只是一份热闹。


**比如潘金莲是毒杀亲夫的恶女，但也曾在心底真正爱过一个人。**


15 岁她被亲妈第二次卖给别人，18-24 岁她正在床上伺候 60 岁的张大爷，24 岁被迫与短小不精悍的武大郎做夫妻。那本该是女人一生最美好的光景。25 岁，她爱上了生命中的第一个男人——自己的小叔子武松。但善良的武松坚守着人世间的道义，一直都把她当娘。从此之后，她的行事风格多有「性决定大脑」的倾向，最终以悲惨结局收场。


他们所经历的痛楚，都是当时这个即将崩塌的末世带来的惨痛。


如果你没看过《金瓶梅》，这个专栏可以带你一起轻松读完整个故事；如果你看过《金瓶梅》，这个专栏可以帮你站在末世的余光中重新理解潘金莲、西门庆……
