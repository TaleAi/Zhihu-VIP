## 简介
关于深爱的古代言情故事，任生活雨打风吹，与爱人同行方能无坚不摧。


**《折霜花》**


我叫霜花，在坤宁宫做宫女五年，最大的秘密，是我一直都心仪坤宁宫的暗卫，影。


而我也知道影最大的秘密——他的心上人，是当今圣上最宠爱的皇后娘娘。


他守护着皇后娘娘，我守护着他。如果不是陈王殿下突然出现，我以为我可以一直像这样在这深宫中待到死去。


陈王来到我面前，仔仔细细端详过我的脸之后，便执意要求带我离开坤宁宫，离开皇后。他告诉我，害我变成宫女，失去过去所有记忆的女人，就是皇后。


**《九皇子》**


我万没想到，眼前这个落魄的男人，是皇帝的九皇子。


这是我在冷宫的第三年，伺候的都是失了宠、犯了错的主子们，不曾想，也不敢想九皇子这样的人，倒乐意跟我这样一个小丫鬟做朋友。


他还说要带我离开这个冰冷的地方。


我却不知前朝早已风云大变，为了讨得给我治病的药，他与其他皇子做了交易，答应辅佐新王，并且每日服毒，以表忠诚。


我只知自己将重获自由，却不知九皇子用命，换来了我余生的幸福。
