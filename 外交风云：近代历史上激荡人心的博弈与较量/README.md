## 简介
历史上有哪些大快人心的外交事件？




世界历史舞台风云际会，国与国之间有合作，也有争端。大国之间的外交，什么时候该强硬，什么时候该妥协，都大有学问。


本专栏里，专门研究近现代史的皮耶霍，以充满镜头感的语言，聚焦外交这一主题，带我们重回近代历史上，那些惊心动魄的博弈与较量。


1919 年，巴黎和会上，面对日本政府的阴谋和各大国的步步紧逼，年轻的外交家顾维钧贡献了一场精彩绝伦的辩论。内忧外患的艰难环境下，顾维钧是如何据理力争，捍卫了祖国的尊严和利益？


1971 年，联合国大会上，外交家乔冠华用一番不卑不亢的精彩发言，展现了中国的大国风貌。这也是中国恢复联合国合法席位后的第一次发言。二十年来，在重返五大常任理事国的艰难道路上，我们究竟做了哪些努力？ 


1993 年，北京申奥失败的消息震惊了全国。在票数一路领先的情况下，期盼已久的奥运会却在最后时刻与我们失之交臂。直到 2000 年，我们才再次申奥成功。从失败到成功，背后发生了哪些鲜为人知的故事？


······ 


较量与博弈之间，是一代代外交家运用自己的智慧和才华，不屈不让，据理力争，捍卫祖国尊严的努力和抗争。
