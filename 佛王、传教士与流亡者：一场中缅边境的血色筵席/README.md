## 简介
一个征税的传教士、一个久远的弥赛亚传说、一个中缅边界的宗教政权，以及一个延续近百年、富庶强悍的土司家族，最终却被一个十余年隐姓埋名的语言学天才兼流亡者逐一击破。


20 世纪初，美国传教士永伟里来到中国云南的澜沧县糯福乡，他假借一百多年前的云南佛教传说（救世主会降临），自诩为当地拉祜人的救世主，迅速在云南拉祜族地区建立了一个宗教国家。


永家在与云南政府和土司家族的斡旋中，步步获利，逐渐成为当地权势最大的一股力量。


而他万没有想到的是，正当自己的势力达到巅峰时，一个被国民党驱逐到云南糯福的共产党员，凭借高超的语言天赋，在逃亡之路上，习得傣语、拉祜语、佤语、英语，最终在永家的政权边，悄悄地栖息下来。


他叫李晓村，他的信仰很简单：推翻旧社会的势力，为父亲和生活在底层的所有穷苦农民复仇。


仅凭一己之力，二十年间，他在边境隐秘地组织起了武装力量。1948 年，在与党组织重新搭线成功后，他制作了详细周密的计划，最终一举端掉了永家。


与永家一同被李晓村驱逐出边境的，还有世袭大土司家族。


这是共产党员李晓村的故事，也是一个语言学天才兼流亡者的复仇之歌。
