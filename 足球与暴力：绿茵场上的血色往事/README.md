## 简介
激情与暴力、胜利与失意、呐喊与唏嘘……足球被誉为世界上最性感、最浪漫的运动，本专栏带你揭开足球运动的背面，那些血腥的、阴暗的、不公平的细节，也终究是它绕不开的命运。


本专栏是一部现代足球小史，那些足球史上令人唏嘘的比赛、天才球星的陨落、无处不在的暴力争端，都是我们热爱这项运动的理由。


你不看足球，你不会知道 1989 年的「希尔斯堡惨案」，至今都是利物浦人心中挥洒不去的阴影。如今的利物浦为荣誉而战，更为死去的 96 名球迷而战；


你不看足球，你不会知道在中国足球的巅峰时代，远在英国效力水晶宫队的范志毅曾拒绝过利物浦的邀请——「**而立之年，还是想回国，还是想再冲击一次世界杯。**」


你不看足球，你没有见过 2002 年日韩世界杯那些令人瞠目结舌的动作和判决。也许，作为东道主的他们，的确侮辱了足球。


你看了足球，也许就真的等来了中国足球再次冲进世界杯的那一天。


请翻开这部足球小史，与它再次亲密接触。
