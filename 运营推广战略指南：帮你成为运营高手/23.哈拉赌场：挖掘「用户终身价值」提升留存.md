## 23.哈拉赌场：挖掘「用户终身价值」提升留存
终身价值是一个庞大的富矿，企业可以通过自己的模式来慢慢的挖掘，细水长流的获得收益。在这个模式里面，最核心的不是推广营销以换取更多用户，而是在已有用户的群体里面深度开发，加大每一个用户的黏度和价值。在这个基础上，确保每一位新进来的用户也到达了黏度和价值的顶峰。


案例一：地产公司 Build


Build 最早是卖精装修的房子，卖一栋房子赚一笔，然后发现这个生意不太好做。后来就转变了思路，开始关注房子里的各种配件，比如门把手、沐浴喷头、马桶、灯罩、墙壁上的空调遥控器等，他们发现用户买房子平均会住上个十几年，在这漫长的居住时光中，会需要更换上万件的小东西。等这十几年过去他们换了一栋房子继续住，还是会继续更换房子里的各种零件，这是一个很大而且很长期的痛点。


于是 Build 就转型了，开始做房子里的各种配件，发现这真的是一门好生意。为什么呢？因为首先房子里的配件如果损坏了，要找一模一样或者至少尺寸风格相符很麻烦，但是 Build 作为房子的供应商，肯定有大把的符合要求的配件可供选择；其次，这也是最核心的，当用户习惯在 Build 这里找配件更换配件的时候，他们就懒得再去花时间花精力去其它地方淘了，以后所有的配件单子都变成 Build 的了。


后来 Build 也不好好卖房子，直接转型做配件零件的平台，然后签了很多优质的供应商，把以前竞争对手的用户也全吸了过来，大家在谁那买的房子不重要，重要的是 Build 有所有的配件，欢迎选购。


通过这种模式，Build 挖掘到了用户的终身价值，只要人在房子里住着，就会时不时发生需要维护或者换配件的情况，而这种情况一旦发生，就是 Build 的一个订单。而且因为 Build 做配件做的很溜，其它房地产商也乐意牵线搭桥，让自己的用户接触 Build，这种靠第三方服务就能提升用户体验的事情，不做白不做。


Build 已经经营很多年了，回过头来看，他最早是直接卖房子的，卖掉一栋这一单也就到此为止了，属于短期生意，但是后来转型做配件，就无限延长了自己的用户生意关系，只要用户还住在房子里，就会源源不断的产生新的需求，由 Build 来不断满足，最大化一个用户的价值，也就是这节课的主题，终身价值。


案例二：哈拉


哈拉是拉斯维加斯的一家赌场，一般来说，赌场做大生意的方式都比较简单粗暴，首先是计算清楚赔率，确保用户们无论怎么赌最后赌场都是赚钱的；然后再通过周边的服务吸引用户前来。


比如价格非常便宜实惠的五星级酒店客房，赌场就开在酒店一楼，让住店用户赌起来特别方便。然后赌场还会安排最有特色的厨师和餐厅，提供优质但是低价的美食和美酒等等。但是哈拉赌场有些不同，它看重的是每一位现有赌客身上的潜力。


举个例子，对于其它赌场来说，每增加一位赌客，可能会给赌场带来 100 美元的利润，于是赌场就会花 50 美元在广告和餐饮住宿上去吸引他们过来。但是对于哈拉赌场来说，如果把经费投资在客户管理上面，可以让每个赌客带来 300 美元的利润，那么这个时候广告投放和其它推广方面的费用肯定就要缩减了，就会把核心资源放在客户管理上面。


对此，哈拉赌场甚至开发出了一个赌客数据库，叫做赢家信息网。这个数据库跟建户口一样的把每一个在哈拉赌场娱乐的用户都建立了档案，档案里面细致到连赌客爱喝什么口味的饮料都有。这个数据库最重要的功能，是直接估算出一个用户的终身价值，通过这个人的赌场过去的表现、自身的信息以及算法的结果，哈拉可以判断出每一位用户对于赌场的价值到底是多少，然后根据这个数值的结果来给他们设计不一样的奖励机制。


对于终身价值有限的用户，哈拉赌场就做一个普通的奖励机制，甚至可以把他让给其它竞争的赌场。但是对于终身价值极高的用户，哈拉赌场会豁出去的为其提供各种福利，唯一的目的就是把对方长期留住，因为只要留住了，就绝对稳赚不赔。


案例三：英国企业 Quintessentially Group


Quintessentially Group，他们做的是无微不至的 VIP 服务，也就是说，只要用户把年费付了，他就可以提出任何要求，然后这家企业就会不惜一切代价为之完成，当然，都得是合理合法的要求。


比如说，会员临时需要在米其林三星的餐厅定一个位置，或者需要为自己的孩子找一个优质的小学，或者需要巴黎时装周的 VIP 票，都可以向这家企业提出，他们都会帮忙做到。


这种服务模式会把用户的习惯培养的越来越刁，他们将不会再花时间为不同的需求找不一样的供应商，而是会全部打包丢给这家服务企业，而相对应的，这家企业也就长期留住了自己的用户，以及他们每年支付的那一笔不菲的费用。这也是一个终身价值的锁定。


企业是如何挖掘用户的终身价值的？首先是锁定他们长期的需求，然后通过非常细致的关注来确保他们会长期留住，源源不断地贡献着价值。
