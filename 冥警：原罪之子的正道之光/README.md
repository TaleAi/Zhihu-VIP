## 简介
孤儿黄石石一直耻于提起父母的罪恶身份，却在工作中找到了久违的关爱。搭档、伙伴、同事……身边的人，因为自己接连被牵扯到多摩会的事件中，生死离别再一次降临到黄石石的身上。


黄石石是西川市冥警局，一名正式的人界冥警，与冥界冥警陆渔结为搭档，处理各种家长里短的报警与不明所以的乌龙案件。


一日，黄石石听闻女高中生小晴的撞鬼事件，到小晴家中拜访，发现小晴已经神智不清，但是其父亲魏国发却声称女儿只是因为学习压力大。


拜访当中，黄石石无意间发现客厅神柜上有用黑布遮住的异物，看起来当事人有所隐瞒。


黄石石和陆渔并不相信魏国发所言，去学校查探小晴精神失常的原因，却接到小晴失踪的消息……


再一次见到魏国发，他以肉眼可见的速度瞬间白头，才说出事因：他为了给小晴报仇，已将灵魂出卖给多摩会，多磨邪神却夺走小晴的身体，意图颠覆世间秩序……


但众人不知道，多摩邪神行动的第一步是要找到黄石石，理清多摩会与冥警局的羁绊。


……


多摩会的目的、正义的真相、黄石石的真实身份……各种问题在几天内冲击着黄石石的脑袋，该何去何从？


当黄石石决定为正义走一遭时，却浮现出又一个的谜团，他该怎么办？


一切，都要从一纸契约说起。
