## 简介
本课程由北京大学中文系博士林品主讲。本课程主要介绍了《魔戒》《纳尼亚传奇》《冰与火之歌》和《哈利波特》这 4 部最为著名且经典的奇幻文学著作。同时，还讲解了托尔金、C.S.刘易斯、乔治•马丁和 J.K.罗琳这 4 位作家的创作理念与写作思路，为我们打开了一扇通往魔法世界的大门。


你是否也曾幻想过收到霍格沃茨的来信？


你是否也对魔法世界充满了好奇？


哪一部科幻文学作品离你心目中的魔法世界更接近一些？


又是否有哪一位魔法世界中的人物曾给你留下深刻印象？


本课程将为你打开魔法世界的大门，带你一窥魔法世界的神秘面纱，让你真正认识科幻文学作者们心目中的魔法世界。


本课程主要介绍了《魔戒》《纳尼亚传奇》《冰与火之歌》和《哈利波特》这 4 部奇幻文学著作，包括不同作品的世界观设定，叙事手法和现实参照等诸多内容。同时，课程中还介绍了托尔金、C.S.刘易斯、乔治·马丁和 J.K.罗琳这 4 位作家在撰写这些奇幻文学著作时的理念与思路。


走进本课程，让我们走进神秘的魔法世界。
