日本的动漫、游戏早已风靡全世界，而动漫、游戏也不再是单单的娱乐品，它们包含了无数的社会、历史和文化元素。



这一档专栏我们将从日本最具人气的那些二次元作品为切入点，聊一下**二次元中所包含的真实**，尽可能地**将纸片中的那个现实世界，还原呈现出来**：


* 日本的高中生是不是真的轻轻松松地读书，高高兴兴地恋爱？
* 现实世界，娶一位樱花妹到底有多「幸福」？
* 日本警察是什么样的？会不会邀请柯南破案？

* 美少女战士为何频频穿越《灌篮高手》？
* 任天堂最开始居然是卖扑克牌的？
* 日本漫画家到底挣不挣钱？
