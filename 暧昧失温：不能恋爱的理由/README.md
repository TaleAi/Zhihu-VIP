## 简介
我们之间，超过了友情，还不到爱情。



甜蜜又苦涩的暧昧关系里，我们小心翼翼，患得患失。


所以，他到底爱不爱我？


这段不确定的感情带来的忐忑和揪心，已经要消磨掉当初的心动和热爱。


不能恋爱的理由，也让漫长的等待变得没有意义。


是时候放弃了。


**暧昧失温**


最近，我跟喜欢的男孩关系突飞猛进，他甚至把家里的备用钥匙交给了我。直到他的白月光前任突然回来了。他的冷漠疏离让我怀疑，这几个月来的温柔以待，只是我一个人的错觉。后来我终于放弃，他却突然跑来质问我，为什么离开他？


······


感情里，如果迟迟得不到肯定回应，或许，是他单纯不够喜欢你而已。


不如试试，勇敢挥别一段不确定的错位关系，迎接一场盛大而甜蜜的双向奔赴。
