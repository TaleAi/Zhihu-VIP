## 简介
世间多奇诡，人性傍深渊，窥恐怖秘辛，看人生百态。


这世界上真的有鬼吗？


中邪、撞鬼、附身……


玄之又玄的故事总是在民间传诵，经久不衰。


所有灵异事件都能被心理学理论及猜想完美解释，


心理学博士孙愚一直这样认为。


直到他开始走访各个恐怖命案的案发地，更多惊悚诡异的事件涌向他：


东北少妇为了出马自毁容貌；


富二代少女剖腹后上吊自杀；


金融中心顶层连续九起纹身者跳楼；


女大学生豢养小鬼杀人；


寿衣店老板被强行剥皮……


事态开始失控，一次比一次凶恶，无从解释，无从抵抗。


孙愚决定收手远离，但他不知道的是，更大的危机在背后等着他……
