## 简介
用生活的故事向大脑发问


都说大脑是人体的指挥官，正影响着我们所思、所见、所说、所做、所经历的一切......


但它似乎总有那么一点不靠谱，常常让我们感受到情绪的困扰，引导我们做出一些费解的决策，比如：


*为什么总是陷入容貌焦虑的怪圈？*


*为什么明知道 DDL 快到了却还是继续拖延？*


*同样都是九年义务教育的优秀毕业者，凭啥你的「情商」比我高？*


*多巴胺带来的真的是快乐吗？*


*... ...* 


当我们感受到这些困惑时，我们的大脑到底在干嘛？


本专栏将基于身边的日常琐碎和困惑，向大脑提出十个问题；从神经科学和心理学层面，尽量轻松地去解读相关情境下的大脑工作机制，探讨大脑与个人认知、情绪、心智、行为及决策的关系。 


希望读者们能够更多地看到大脑可爱却也纠结的一面，也能够从神经科学的角度解释身边常见的困扰情绪和行为；也希望本专栏的内容能帮助你解决一些困惑，带来一点快乐，对他人和自己多一些包容和理解。


毕竟神秘的大脑常常不着调啊！
