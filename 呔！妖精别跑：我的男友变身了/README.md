《王子变牛娃》：


炒了老板以后，我和同事打算一起去吃牛蛙庆祝。


万万没想到，牛蛙居然开口说话了。


他说他其实是个被女巫诅咒的王子，只要我亲他一口，他就能变回去……


《欠我的狗粮用什么来还》：


早上我开门的时候，一条素不相识的狗突然闯进了家里。


它吃光了我家狗盆里的所有狗粮然后扬长而去。


我跟着狗，发现它是隔壁帅哥家的。


帅哥很负责任，他决定子债父偿，用他自己偿还我的狗粮。


《前男友天天说我坏话》：


我捡到一只八哥。


这个八哥嘴很溜，天天大声说主人前女友的坏话。


八哥才来了一天，我就知道了它主人的前女友脾气暴躁、小心眼、冷暴力还无情无义。


我很好奇这个怨种前女友到底是谁。


后来我才知道，怨种前女友竟是我自己。
