## 简介
本专栏详细讲述了有关「军情六处」的传奇故事，还穿插军情六处与军情五处的斗争，引人入胜。 


英国秘密情报局（SIS），通常称为「军情六处」（MI6），创建于 1909 年。在其创建后长达八十多年的时间里，「军情六处」的运作一直是绝密信息，官方也不承认它的存在，直到 1994 年英国《情报局法例》颁布后，它才逐渐被外界知晓。


本专栏详细讲述了那些「军情六处」的传奇故事，创始人卡明、继任者辛克莱及孟席斯、「白衣夫人团体」、威廉•斯蒂芬森、作家毛姆及麦肯齐、「间谍之王」赖利、解密专家阿兰•图灵、「月亮女神」艾米•索普、「孤胆英雄」奥拉尔、「鼹鼠」金•菲尔比与乔治•布莱克等人在此悉数登场，还穿插着军情六处与军情五处的斗争，引人入胜。
