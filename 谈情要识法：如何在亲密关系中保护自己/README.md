## 简介
懂法是为了让爱情更安全。



> 杭州三套房离婚案中，婚前丈夫全款购买的房子，离婚时被判分给妻子 25%；



> 浙江台州全职妈妈为家庭付出所有，离婚时只收到 1.5 万补偿；



> 湖南省新化县订婚男女分手，除了领证外，办酒、同居、生子一样不落的情况下，女方被判返还所有彩礼……


一桩桩、一件件的判决，给了处于婚恋年龄的男女——「每天一个恐婚小技巧」。


**相爱简单，但保护自己太难。**


提到恋爱和结婚，我们脑中满满都是问题——


* 如何保护婚前财产？
* 彩礼问题到底关乎哪些风险？
* 是不是只要不离婚，共同财产就是安全的？
* 亲密关系中遭遇暴力我该怎么做？
* 单方面想离婚，我能离掉吗？

更别提很多骚操作，比如债务转移、骗婚等等。


为了帮助单身读者放心追求爱情；帮助已经处在亲密关系中的读者，能够识别「不安全因素」；帮助已经面临问题的读者，找到能够有效保护自己的合法手段，我们特此制作了这个专栏。


专栏作者易轶律师，是一个从业 13 年的资深婚姻家事律师，曾带领团队处理过 3000+婚姻、情感、家庭财产纠纷，她同时也是中央电视台《法律讲堂》主讲律师、北京电视台《第三调解室》常驻调解专家，擅长用普通人能听得懂、学得会的方法，科普情感、婚姻中的法律问题。


在专栏中，她将从真实的情感纠纷案例入手，科普当下男女在恋爱、婚姻等亲密关系中需要注意的法律问题，以及面对这些问题时保护自己的实操方法，让大家能在保护好自己的前提下，放心相爱。
