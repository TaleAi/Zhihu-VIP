## 简介
只要认真观察生活，就会发现生活和热物理学息息相关。科学家们将科学理论知识运用到生活中，才使我们的生活有了巨大的改变。本专栏带你了解生活中的热物理学知识，在日常生活中培养科学思维。


在日常生活中，你是否留意到以下这些现象：


炎热的夏天，我们会用空调和电扇降温；


高压锅做出来的米饭更好吃；


寒冷的冬天，钻进有有暖气的屋子可以抵御严寒；


听到烟花爆竹的声响，才能更好地感受过年的味道；


......


这些都是我们生活中最常见的事情，它们背后蕴含着很多热物理学的知识。学习物理小知识，对于小朋友们来说，并不是需要了解所有科学原理，而是要培养一种科学的思维方式。


本专栏热物理学家从生活现象入手，带小朋友们了解热物理学的基本常识，从小探索科学世界，呵护孩子的好奇心。
