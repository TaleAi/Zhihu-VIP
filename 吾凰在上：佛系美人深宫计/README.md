## 简介
佛系美人在宫墙内的生存指南。


「在后宫，不争宠如何苟到大结局？」


这里的人间清醒大女主，表面上不争不抢不声不响，不斗绿茶不搞雌竞，却深藏功与名，笑到了最后。


**《白头不守约》**


我是京城第一美人，宁繁锦。


爹爹说，希望我这一生都可以繁花似锦，但是，打脸来得太快。


主要还是爹爹不给力，官途不顺，让本来就不富裕的家庭又雪上加霜。妹妹年幼，娘亲病弱，加上自从贬官之后就天天叹气的爹爹，这日子没法过了。


作为长女，我自觉挑起了生活的重担。


这个时候，脸的作用就凸显出来了，虽然我爹和我娘只是一般般的英俊、一般般的美貌，但我不一样啊，我天生丽质。


正好赶上选秀，我决定去参加。但爹爹不同意，娘亲也不同意。


入宫，这条路，在爹娘眼里就是绝路。但他们也不想想，家里都这样了，还不能奋力一搏吗？


毕竟，我这张脸生得这么美，我怎么舍得辜负了它。


**《后宫升职记》**


父亲让我代替姐姐入宫，我欣然应允，毕竟，我只是失去了爱情，却得到了荣华富贵的后半辈子。


老皇帝一家很好，把我这个冒牌货当作吉祥物，让我从皇后一步步晋升到太皇太后，为了回报他们，我特意求了皇帝，把我那喜欢沽名钓誉却又有点能力的父亲下放到穷苦之地，造福百姓。


我在宫里衣食无忧，听曲儿、养猫，还遇到了和我志同道合的小皇孙，简直就是神仙日子。


而我那恋爱脑的姐姐却觉得我过着暗无天日的生活，但实际上她才是拿着虐文剧本。
