## 简介
来，推理谜题开始了，不服来战！我，凌雅瞳，H 市公安局刑侦大队总队长，人称霹雳警花。在我惊心动魄的职业生涯中，侦办的全都是H市最棘手、最烧脑、最具有挑战性的刑事案件：密室杀人、当众杀人、完美的不在场证明……所有你们在推理小说中看到的不可能犯罪统统穿越了时空，在H市隆重登场。你们即将领略的，是闻所未闻的犯罪诡计，想都想不到的恐怖阴谋；当然，还有我独当一面、精妙绝伦的推理。怎么样，敢不敢来挑战我，霹雳警花凌雅瞳？我在这里等着你们。


**第一案：蹦极惊魂！山谷内惊现尸体，报警后尸体又莫名消失**


第一次蹦极，山谷中惊现受害人尸体，下山去看时尸体却奇怪地消失不见；而第二次蹦极时却又诡异地再次出现在山谷当中。


整个案发期间，所有人都有完美的不在场证明。


尸体究竟为什么会失而复现？凶手到底使用了怎样离奇的诡计？


  



**第二案：疗养院院长深夜偷情，与情妇双双被杀，医生病人都有不在场证明**


疗养院院长和情妇偷情时双双毙命办公室。


案发期间无外人闯入，凶手就在当晚为数不多的医生、护士和病人当中。


然而调查下来，竟发现所有人都有完美的不在场证明。


究竟凶手采用了怎样奇妙的诡计？


  



**第三案：被害人被困住手脚丢入池塘活活溺死，所有人都有完美不在场证明**


受害人被捆住手脚，丢入池塘活活溺死，整个小镇却无一人听见呼喊救命；


案发期间，所有嫌疑人都有完美不在场证明！


凶手究竟采用了怎样精妙的诡计？


决定性的证据又藏在何处？


  



**第四案：艺术学院研究生惨死宿舍，三名嫌疑人振振有词，凶手究竟是谁**


艺术学院研究生惨死宿舍，凶器神秘消失；


作案现场出现怪异景象：艺术品石膏像碎了一地；


三名嫌疑人振振有词，看似无懈可击，无从下手。


凶手到底是谁？


揭开真相的关键证据究竟隐藏在什么地方？


  



**第五案：高颜值校花坠楼砸毁保时捷，她真的是跳楼自杀吗**


高颜值校花跳楼，砸毁保时捷豪车。


是自杀？还是凶手设下的诡计？


唯一有嫌疑的人，却有着无懈可击的不在场证明！


决定性的证据究竟藏在何处？


  



**第六案：公园双重密室杀人！凶手有完美不在场证明，又是如何穿越双重密室**


明珠公园内的儿童乐园里发生离奇凶杀案！


案发期间，明珠公园和儿童乐园都是完全封闭的空间。


大密室中套着小密室，凶手究竟如何进出双重密室？


案发期间，所有人都在公园门外吃烧烤，所有人都有完美的不在场证明。


凶手到底是谁？究竟使用了怎样的诡计制造了这起匪夷所思的不可能犯罪？


  



**第七案：广义雪地密室杀人！雪地上没有凶手的脚印，所有人都有完美不在场证明**


受害人死在雪地中央，现场只有受害人进来的脚印，却没有凶手进出的脚印；


广义雪地密室！凶手究竟如何实施的这场不可能犯罪？


案发期间，所有嫌疑人全都有完美的不在场证明！


白茫茫一片的雪地背后，究竟隐藏着怎样宏大的诡计？


  



**第八案：网红直播带货却被杀死在密室当中，凶手如何进入密室**


万圣夜，网红直播带货，却当着万千粉丝的面被神秘凶手杀害。


凶案现场竟然是一间密室，凶手究竟如何进入的房间？


所有的嫌疑人都有完美的不在场证明。


网上猜测众说纷纭，究竟哪一种才是正解？


  



**第九案：新娘血溅婚博会现场，凶手如何进出密室**


完全封闭的婚博会场馆内发现女尸，凶手究竟是如何进出密室的呢？


受害人死亡多时之后竟然还会发生诡异的短距离位移，究竟是什么外力导致的？


唯一有嫌疑的两个人却有着完美的不在场证明，他们究竟使用了怎样的诡计制造了「密室杀人」和「完美不在场证明」的呢？


罕见的双重不可能犯罪等你来破解！
