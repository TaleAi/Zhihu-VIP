## 简介
面对丈夫出轨，有的女人选择隐忍，有的女人选择离开，有的女人则选择了复仇……



老公带着一家人不仅吃她的，喝她的，还背着她养小三，更过分的是这个小三还是昔日仇人。拿我的爱情当狗屁？老娘不玩了！玩手段，耍心机，我怕过谁？


——《妻子的复仇》


那一夜我站在冰冷的地板上，听我最爱的老公和婆婆在商量怎么打掉我肚子里的孩子，怎么神不知鬼不觉地占有我的房子……


——《破婚：滚蛋吧，妈宝！》


老公游手好闲，还以夫妻名义网贷巨款；婆婆跋扈刁钻，对我欺凌压榨；绿茶闺蜜更是要败坏我的名声。既然都来逼我，那就请君入瓮，别怪我以恶治恶了。


——《请君入瓮》


领养两年的女儿竟然是老公跟小三生的？就在拿到女儿 DNA 检测报告的当天，自己的亲生女儿坠楼身亡，这是意外还是其他？


——《秘密》


丈夫不仅花光了我的钱，还将我打进医院，他却一走了之。我借钱交住院费，却意外发现了丈夫不止借了高利贷，还在外面有其他女人。面对这一桩桩的打击，我决定要他血债血偿。


——《金蝉脱壳》


遭遇骗婚，孩子惨死，丈夫还伙同外人要把我关进大牢当替罪羊！我以为一切的不幸都是从结婚那刻才开始的，却不想我早就陷入了一场精心策划了十年的阴谋中……


——《反杀》


如果不是我复明那天亲眼看到袁杰让我填写一份巨额保险，或许不会一直假装还在失明中。


——《盲妻》


我用一根口红试出丈夫出轨，对方竟是我的学生。


我装作不知道，冷眼看他替小三偷我学术成果，还伙同婆婆设局让我净身出乎。


我是一个催眠师……


——《催眠师》


我从一张照片发现老公不对劲，我略施小计，便亲眼看见他和他的白月光的偷吃现场。


这两个人，花着我的钱，嘲笑我的脸，还想霸占我的房！


我佯装不知，继续养着他们的欲望……


猪，要养肥了，再杀！


——《白月光与朱砂痣》
