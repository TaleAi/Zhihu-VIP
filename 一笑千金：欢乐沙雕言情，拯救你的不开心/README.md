## 简介
人，就是需要点画风清奇的言情故事来调剂琐碎生活吧。


穿成修仙虐文里的恶毒女配，怎么办？


当霸道总裁遇上沙雕女主，会是怎样的情景？


成为史上最年轻太后，却被狗皇帝搂在怀里，什么情况？


.......


本专栏涵盖修仙、古言、现言几大类型，文风轻松，沙雕不无脑，帮你忘记生活的苦，体味爱情的酸甜。在谈笑间，明了情为何物。


  



《梦事三年》


沙雕少女偶然间穿进一本没看完的修仙小说中，地狱开局，刚穿进来就已经对男主用了思情咒，把男女主和门派长老得罪了个遍，人品值 down 到极点。认怂超快加上一张气死人不偿命的嘴，且看炮灰女配如何顺利苟到大结局，拥抱幸福生活。


 


《寒江雪》


二十二岁就当了太后，还是垂帘听政独揽大权的太后。可那个原本因为体弱多病才被选为新帝的便宜儿子，却摇身一变成了能呼风唤雨的帝王。傀儡就该有个傀儡的样子啊，两人对抗到结局。到底是用情最深的人，受伤最深。


 


《克星》


男女主泡温泉时一同落水，穿越到昭国，一个成了昏庸女帝，一个成了权势滔天的摄政王。试过种种方式都不能穿回现代后，两人决定，来都来了，玩把大的：开设通商口岸，大修土木，设立科学研究部……原本怀揣着理想与希望的两人最后才发现，历史的洪流，不可逆转，有些改变，是没有意义的。还好最后，他们身边还有彼此。
