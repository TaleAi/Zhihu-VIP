## 简介
史书上的大佬们，真人会很严肃吗？


当我们拜读史书时，常常艳羡于古代仁人志士挥斥方裘的样子。


殊不知，他们也有在微末中艰难求生，在生活中家长里短的时候。


* 五十人就敢劫营的勇将辛弃疾，其实私下里是个写词哄妻子的温柔丈夫？
* 政坛文坛「两开花」的王安石，其实是个不爱洗澡的「臭棋篓子」？
* 山水田园派诗人韦应物，其实在二十岁前是个横行霸道的典型二世祖？

他们或者热血满腔，或者柔肠百转，或者有这样那样的怪癖，但都分外的鲜活可爱。


在本专栏中，房昊以轻松愉悦的笔触，讲述历史上知名人物或跌宕起伏、或趣味横生的非常规故事，为你揭开古代名人的「庐山真面目」。
