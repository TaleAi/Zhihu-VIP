## 25.你不说，他怎么知道自己很小气？
也许是在外面放花时着了凉，我从大年初二就开始发烧咳嗽，吃了三天药还不见好转。贝勒送我到医院检查，竟然是急性肺炎。


我万般不情愿地住进了医院，贝勒用我的手机给所有女会员发了短信，没说我生病的事，只是让她们这段时间有什么事先找雪碧处理一下。


但八喜和夏一还是知道了实情，两个人都来医院看了我，不过是分别来的。夏一来的时候正好看到贝勒守在床边喂我吃饭，心里明白是怎么回事，脸上便有些讪讪的，礼节性地说了几句好好养病之类的话，就匆匆离开了。


八喜则是满面春风跑来的，刚刚和夏一搬到一起的她还处在兴奋状态中，说从来没想到跟一个男人共同生活是这么有意思的事情。呵呵，想当年我刚跟夏一同住的时候也曾经像她一样，觉得一切都那么新鲜美好，对未来充满了信心和期待，但愿她不要像我一样，没过多久就开始失望，到后来简直就是幻灭了。


雪碧也来病房坐了一会儿，没说太多的话，只是忍不住一会儿看看贝勒、一会儿看看我。看来我们俩的关系是彻底瞒不过去了，算了，随它去吧，也没人规定女老板和男下属就不能搞到一起啊！


在贝勒的精心照料下，我在三天后出了院，一星期后完全恢复了健康。上班的第一天，贝勒还是不放心我独自赶路，坚持陪我一起去了公司。在办公室见到雪碧，我很真诚地感谢她在我生病期间一直独自支撑公司的事务，雪碧一副宠辱不惊的样子，只是淡淡地说了句「别客气」。


但是还没过一个小时，我对雪碧的感激之情就灰飞烟灭，因为小 Q 打来电话，说她准备跟小戴离婚了。


起因还是小 Q 和小戴春节期间的这趟老家之旅，但问题并不是出在小戴的家人身上，而就是出在小戴本身。按照小 Q 的说法，小戴一直对钱看得比较重，总是能省就省，小 Q 平时手里不缺钱，对此倒没有太强烈的感觉。但是这次跟小戴回老家之前，小 Q 大手笔地给小戴家人买了很多礼物，把手头的现金全都花光了，想着反正跟小戴在一起，又不怕没钱用，就很放心地跟着小戴启程上路了。没想到，就因为得从小戴手里拿钱，小 Q 在这七天的行程里受尽了委屈——坐火车的时候想去餐车吃饭，小戴说餐车的饭又贵又难吃，吃他带的泡面就行了；在小戴的老家逛街，小 Q 看见路边的小吃想尝尝，小戴说回家让我妈给你做，比这个做得好；小 Q 在街上逛累了想要打车回家，小戴说他老家这边的出租司机可宰人了，硬是拉着小 Q 在刮着四五级大风的街上等了 20 分钟的公共汽车；最过分的是，小 Q 突然来月经，想去商店里买包卫生巾，小戴却从家里翻出一包不知多少年前他妈用剩下的卫生巾拿给小 Q，说先用这个，要是不够再去买。


小 Q 欲哭无泪，在公婆家里又不好发作，忍到回了北京，又已经事过境迁，再翻后账也觉得没什么意思了。可终究是憋了一肚子的火儿，本想找我诉苦，偏偏我又病倒撂挑子了，于是只好跟雪碧说了事情的经过，没想到雪碧比小 Q 本人还要义愤填膺，说小戴这种男人根本就不配称作是男人，跟他过一辈子怎么能有幸福可言？现在你身体健康、自己挣钱、还贴钱给他家人买东西，他都能这么对你，将来万一你有个病有个灾的，他肯出钱给你治吗？还不得立刻卷了财产跑路啊！


小 Q 听了雪碧的话，顿觉心灰意冷，反反复复想了几天，觉得这个婚姻实在没什么维持下去的必要了，于是就在我恢复上班的这一天，决定跟小戴离婚，只是跟小戴摊牌之前，她决定还是先打电话知会我们一声。


万幸小 Q 打了这个电话，更万幸我正好在今天回来上班了，我让小 Q 先别冲动，我这就过去找她聊这件事。放下电话，我直接冲到了雪碧的办公桌前，拼命压抑着想把桌上的一摞资料摔到她脸上的冲动，低声咆哮着质问道：「你还知不知道自己是干什么的？你怎么能随便撺掇客户离婚呢？」


「你说谁啊？我撺掇谁离婚了？」雪碧一脸无辜地看着我。


我更火大了：「小 Q 啊！你自己说过的话自己不记得吗？」


「我可没说让她离婚啊！」雪碧仍然一副理直气壮的样子，「我只是客观地发表了一下自己的看法，让她好好想想再做决定。」


「这有区别吗？」我的声音又高了两个八度，「跟你说了多少次了，不要火上浇油、不要火上浇油，你怎么就是不听呢？咱们是婚介公司，不是离婚服务公司！」


「那也得分情况吧，像小 Q 老公那号吝啬鬼，跟他在一起生活能幸福吗？我也是本着对客户负责任的想法……」


「你负个鬼的责任！哪对夫妻在一起过日子没点矛盾？遇上点儿事就离婚，你真以为婚姻是小孩子过家家啊？」


「过家家倒不至于，但我觉得结婚至少应该让自己开心，如果根本就不开心的话，为什么不能推倒重来？」


我气得完全丧失了理智，在桌子上重重地拍了一下：「你一定要跟我强词夺理是不是？我说一句你有十句等着我，我是不是该管你叫老板？」


雪碧不说话了，目光却并没有退缩，仍然不服气地和我对视着。


贝勒被我们这边的吵闹声引了过来，见此情景，忙把我拉到了外面。


「消消气，消消气！雪碧什么性格你又不是不知道，何必跟她较真儿。」贝勒在楼道里劝我。


「别的事儿我都可以不较真儿，客户的事怎么能随便拿来开玩笑？要是都为这点事就让人家去离婚，咱这公司还开不开了？」我气咻咻地说道。


贝勒笑着点点头：「这孩子是过了点儿，回头我说说她。」


「说不说随你，反正这种事情只要再有下次，我立马让她走人，绝对没有二话！本来看她这几天挺辛苦，还打算给她发点儿奖金呢，这下玩儿去吧，我不倒扣她工资算是厚道的了！」我理了理衣服，平息了一下情绪，「行了，我还得赶去收拾烂摊子呢，中午你们俩自己吃饭不用管我了。」


我心急火燎地赶到小 Q 的服装店时，一对情侣正在店里挑衣服。女孩试了件大衣，一副很喜欢的样子，男孩二话不说地付了钱，女孩欢天喜地地抱着衣服离开了。


「看见没有？自打过完春节，我每次看见这样的客人都想哭。」小 Q 红着眼睛对我说，「人家男的给女的花钱都大方着呢，可我家那位，从认识到现在就没给我买过一件像样的礼物，倒是我一有钱就想着给他买这买那的。都说舍得给你花钱的男人不一定爱你，但不舍得给你花钱的男人一定不爱你，我现在觉得这句话真是绝对的真理！以前也没想那么多，糊里糊涂就过来了，可这次是真寒了心了，怎么说也是两口子，最不济也得有点儿亲情吧？我大包小包地给你家人买礼物，到了你家那边，人生地不熟的，手里又没钱，你反过来就这么对我？就是普通的同事朋友也不能这么干啊！」


「是是，这事儿换了谁都得生气。」我拉着小 Q 在店里的长凳上坐了下来，「不过，听你这么说，小戴这抠门儿的毛病也不是一天两天了，怎么以前也没听你抱怨过啊？」


「嗨，咱们北京女孩的脾气你还不知道吗？总觉着老提钱的事儿太俗、伤感情。刚结婚的时候小戴提出过让我把自己挣的钱给他管，我没答应，他为这事儿闹腾了好一阵，看我实在不愿意也就算了，但是他挣的钱当然也不会给我。我呢，反正自己又不缺钱，想要什么自己买就是了，所以也懒得跟他较劲。可是雪碧说的话我觉得有道理，现在我是自己能挣，大家可以相安无事，以后万一哪天我病了、挣不动了，就冲他现在这态度，我能指望他养着我？连这么点儿起码的安全感都没有，这日子还过个什么劲啊！」


我连连点头：「你说的没错，这种情况一定不能姑息纵容，咱嫁人又不是为了当包子的！但是……也不至于立刻就上升到离婚的高度吧？咱们是不是可以先试着沟通沟通、调教调教，看能不能让他有所改变？要是实在改不了，再离婚也不迟啊。」


「嘁，江山易改，本性难移，我对他根本就不抱期望。」


虽然小 Q 说的是流传了几百年的老话，但老话有时候也并不一定绝对正确。我给小 Q 讲了一个发生在我身边的故事：以前我有一个跟我关系不错的师弟，各方面看着都挺好的一个男孩子，可不知怎么的，就是搞不上对象，谈了好多女孩，都莫名其妙地就吹了。有一次，他又跟一个女孩开始交往，起初一切都挺顺利的，后来有天晚上他们俩通了一个电话，那之后女孩突然就对他冷淡了。他百思不得其解，跑来找我发牢骚，我问他是不是在电话里说什么不该说的话了，他说没有啊，就是当时他在外地漫游，女孩打给他的话会比他打过去便宜一些，所以他当时给女孩拨通了电话就挂断了，等着女孩再给他打回来的。


我听到这儿已经大概猜到是怎么回事了，就问他平时跟女孩在一起是不是经常会算计这些小钱。他说是啊，不是只跟女孩算计，他从小到大都这样，觉得钱能省就一定要省着花。这下事实就很清楚了——从小到大都算计钱，所以从小到大都找不着女朋友。


我把症结所在告诉这个师弟之后，他才恍然大悟，还埋怨我怎么不早点儿跟他说。后来，等他碰到下一个女孩的时候，就有意识地克服了这个毛病，于是恋爱谈得很顺利，现在孩子都满地跑了。


我想通过这个故事告诉小 Q 的是：有些男人，如果他们长期以来养成了省钱的习惯，当他们和女孩恋爱以后，也不会那么自觉地醒悟过来，因为他们的消费观念和你不一样，他们根本就不知道怎样的尺度才是恰当的、是能被女孩所接受的，如果女方没有对他明确表示过不满，他就更不明白自己有什么不对了。可是很多女孩的想法都跟小 Q 一样，觉得说钱的事太俗、显得自己太计较，但心里又确实不舒服，所以干脆分手了事，而被甩掉的小气男还根本就没搞清楚状况呢。


我跟小 Q 说：「如果你和小戴还没结婚，你也可以这么做。但是你们已经结婚了啊，既然还从来没有就这个问题沟通过，至少应该先试试看吧？」


小 Q 依然一脸的愤懑：「可是我就不明白了，有些事情难道真的非得明确告诉他，他才能懂吗？就比如说我在街上提出想吃什么东西，你给我买了我就高兴、你不给我买我就不高兴，这么简单的道理难道还需要别人说？要真的买不起也就算了，又不是这么回事，傻子也能知道别人心里是什么感受吧？有时候别人不说，也是想给他留点儿面子，不想伤他自尊而已。」


我点点头：「嗯，确实是，有的事他也未必就真的不知道自己做得有问题，但人都是有惰性的，有时候明知道自己做得不对，但如果没人点出来，那就得过且过了；可一旦有人指出来他做得不对呢？这时候人的自尊心就通常会战胜惰性心理，促使自己做出改变。具体到小戴身上，你一天不把事情说破，他就一天觉得还是能糊弄得过去；一旦你说破了，他就不得不去面对这样一个选择——到底是要做一个被女人看不起的小气猥琐男呢，还是要做一个被女人喜欢的豪爽男子汉呢？相信我，当题目很明确的时候，大多数男人都还是会选后者的，只要选了，他们就得朝着那个方向努力，不信你就试试看。」


小 Q 犹犹豫豫地点了点头：「那……好吧，晚上回去我试着跟他谈谈，可他要是食古不化，那就别怪我绝情了！」


「行，要真是那样，你尽管找我，我负责给你退货！」


第二天一大早，我就接到了小 Q 打来的电话：「苏浣，还真的让你说中了！我昨天晚上回到家，当着小戴的面把所有的不满都说了出来，小戴竟然被我说得痛哭流涕，做了深刻的自我检讨。他说他从小家庭条件就不好，所以在钱的方面特别缺乏安全感，总觉得每一分钱都得尽量省着花才行。可他省着花钱，就是因为怕将来有急需，所以如果真是我有病了什么的，他是不可能还捂着钱不拿出来的。不过现在他知道自己错了，光是想着以后，却没想到在平时就已经委屈了我，作为男人他觉得自己很失败，以后再也不会这样了。后来他还非拉着我去了商场，给我买了好多东西，说是要把以前亏待我的都给补回来。唉，其实我后来想想，觉得也能理解他了，我是从小没尝过缺钱花的滋味，要是生活环境像他一样，肯定也不敢大手大脚的。再说除了花钱这件事，他在其他方面对我还是挺不错的，我之前离婚的想法确实有点儿偏激。」


「呵呵，祝贺祝贺！」我感到无限的欣慰，「不过我还得提醒你，一个人是不可能在一天之内就彻底脱胎换骨的，肯定还会有一个反复的过程，毕竟几十年养成的习惯，不是说改就能完全改掉的。以后他要是又犯老毛病，你千万记得今天说的话，对他多点儿耐心。对了，你把这件事跟雪碧也说一下吧，毕竟你是先找的她，也得跟她交代一下结果。」


「没问题，我一会儿就给她打电话。」


我一整天都没正眼看雪碧，下班的时候，雪碧自己讪讪地蹭了过来。


「老板，我昨天态度不好，你别往心里去。」雪碧低头玩着自己的手指，口气听上去也并不是多么地诚恳。


「没事儿，过去就过去了。」我故作大度地说道，「以后记得别这么冲动，咱们的原则永远是尽一切可能维护客户的婚姻稳定，而不是挑拨他们的关系，明白吗？」


「嗯。那没什么事我先走了。」


雪碧转身向门外走去，我的手犹豫着伸进抽屉，摸到了之前准备好的红包，抬头看了看雪碧冷冰冰的背影，终于还是没有开口叫住她。
