## 简介
一个从入门到今天已有27年的Marketer公开工作日记与商业思考。日记里有她解读过的文献，有消费行为实验全程，还有她问自己的领悟。她想让你置身自己之外，旁观那个花钱的自己。她想通过公开消费行为的逻辑拆解，让你明白「如何花钱会更幸福」。


「拍照」会让人感觉食物更好吃吗？「触摸」会让你更想把商品买下吗？「鸡皮疙瘩」说明你对商品可能产生什么感觉？「奢侈品」让人觉得爽的感受集中在哪些时刻爆发？「立 flag」这个和买东西有啥关系？


本专栏挑选了 9 个生活中常见的消费场景，用 9 个有趣的消费行为学实验拆解消费的逻辑。阅读过专栏后，也许你会发现，看起来形态各异的广告语、商铺陈设、销售话术，背后所运用的消费逻辑竟然不尽相同！


在这张专栏中，你会看到严谨、周密的语言娓娓道来我们心甘情愿付钱的道理。你会看到商业大佬常常在行业内部讨论的概念，但并不会觉得生僻难懂。你可以卸下有可能被「洗脑」的包袱，收下这门原本是在企业高层、大学讲堂一席难求的课堂上才听到的「课程」。


适合人群：


1. 常常与自己的购物欲望抗争的人；


2. 想要了解购物欲、拆解消费逻辑的人；


3. 喜欢逻辑性思考，并且喜欢阅读经济学相关内容的人。
