## 简介
中国只是农耕文明吗？明清时期「闭关锁国」是全部的历史真实吗？有人把中国文明等同于黄河文明，把西方文明等同于海洋文明，这符合历史情况吗？……本专栏力图用数据和实例还原海洋文明中国史，带你转变视角，了解一个完整的中国、一部完整的中国史。


中国只是农耕文明吗？游牧是不是中国？ 渔猎是不是中国？ 捕鱼采珠、远洋贸易，是不是中国？


摩洛哥旅行家伊本于 1342 年旅行到印度马拉巴海岸时，就在那里看到了中国海船。他写道：「进入卡利卡特及奎隆的中国船大若城市，船上种着药草和生姜。船上的中国高管和他们的妻子在船上拥有自己的房间，当时的中国人史世界上最富有的人。」


三十年前，有的人还把中国文明等同于黄河文明，把西方文明等同于海洋文明，再厚此薄彼。


而其实五百年前，一千年前，甚至一万年前，中国人早已拥抱蔚蓝。


著名历史学家李伯重认为：


「传统的明清闭关自守论，今天已经过时。只有摒弃了这种陈说的束缚，才可以实实在在地来看经济全球化的历史。」


只有超越黄河文明说，超越农耕文明说，超越专制集权大一统说，才能了解一个完整的中国。


换个角度看中国，展现一部不一样的中国史。
