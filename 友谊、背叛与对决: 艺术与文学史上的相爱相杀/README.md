## 简介
你一定爱看的艺术史故事


1504 年，52 岁的达·芬奇被一个横空出世的年轻画家「按在地上摩擦」——29 岁的米开朗琪罗决定接受前辈的挑战，与达·芬奇竞争创作佛罗伦萨旧宫的壁画。一场世纪艺术大战揭开序幕。


1952 年，萨特在杂志上刊登了一封两万字的对加缪的绝交信。存在主义最著名的作家友谊最终破裂——「***我亲爱的加缪：我们的友谊多艰，但我还是感到惋惜。如果您今天断绝了它，无疑是它应该被断绝。使我们接近的事多，使我们分离的事少。但是，这少仍嫌太多……***」


1871 年，16 岁的兰波给当时已小有名气的魏尔伦寄出了自己的诗，他们自此相识。他们是知己，是伴侣，一同私奔去了伦敦。1873，一场争吵过后，魏尔伦为了阻止兰波离开而开枪打伤了他，魏尔伦被判入狱，两人永生不复相见。


1886 年，已经功成名就的小说家左拉出版了一本小说《毕生杰作》。小说中的主人公克劳德·兰迪尔是个失败、焦虑、性生活放荡的艺术家，他的发小，印象派大师塞尚将之看成是对他的直接讽刺，并认为左拉在书中嘲弄了他多年来向左拉透露过的很多秘密。跨越三十年的美好友谊因这本书而终结……
