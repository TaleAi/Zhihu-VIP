## 简介
地球上每种生物都在为争夺更好的食物资源而「圈地」，在人类社会，因为「吃」不惜动武的事儿也不少，甚至不惜动用全国之力发动战争。本专栏意图围绕针对食物资源引发的战争故事，并引出背后更全面的时代背景、博物学知识，及人与自然的相处之道。


本专栏每一节将从一种常见的「食物」入手，牵扯出一段精彩的历史。


鳕鱼——地处偏远又刚刚独立的小国冰岛，为了鳕鱼，竟然挺起腰板对前世界霸主英国说了「不」，更蹊跷的是，在和英国的三次硬碰硬抗衡中，小小的冰岛居然连连得胜。是什么促使冰岛人选择和强敌「硬刚」？又是什么让冰岛笑到了最后？


鲱鱼——从渔夫手中的一把小刀开始，荷兰这个低洼之国，逐步巩固了海上贸易优势，摇身变成「海上马车夫」。这个世界性大国是如何崛起的？它与英国之间的几次战争又是因何而起？又是什么让这个如日中天的大国跌落神坛呢？


香料——对于迫切需要香料的中世纪欧洲来说，崛起的奥斯曼帝国切断了久远的商路，他们该如何获取遥远东方的香料？人类大航海时代的第一程是何等艰难？三百年间各大海洋帝国之间的战事为何愈演愈烈？


……


透过食物，让我们见证一段段绚烂的人类文明史。
