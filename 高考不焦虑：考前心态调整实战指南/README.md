## 简介
遇到考试，你焦虑吗？尤其还是高考！高考，考的就是个心态；心态好，高考就胜利了一大半。声工厂为你提供高考心态调整指南，让你告别焦虑，安心备考。


「没有经历过高考硝烟的课本知识不是好试题。」


诚然，寒窗十载，一朝奔赴考场，是实实在在的人生转折点。


重视高考是一种必然，但也不能因为太过重视，就给自己套上了枷锁，毕竟再惨烈的试卷也是纸做的，怕火、怕水，还怕手撕。


总有学生莫名的焦虑，一到考试就掉链子，见到监考老师，比偶遇暗恋对象都紧张，直接摔笔崩盘了；


有学生内心脆弱得像一根蛋卷，完全不用多少外力，心态就碎成了一地渣渣。


更严重者一听见考试铃声就要上厕所，甚至有的直接昏了过去。


**高考，考的就是个心态。心态好，高考就胜利了一大半。**


临场发挥自然是谬论，在高考的大道上，应该是厚积薄发，做足了准备，轻装上阵。


十年寒窗为的就是这一天半的书写，相对于积累知识而言，调整心态更应该是高考路上的护航灯塔。


你迷失方向没关系，让声工厂随时「嘀嘀」提醒你。
