## 简介
一名刑辩律师的日常是怎样的？一个杀人犯的作案动机可以有多诡秘？律师刘焱从业 8 年，经历了各类离奇案件，接触了形形色色的当事人。通过追溯他们的人生轨迹，刘焱看见了世间的错综复杂，并将在本专栏里，用独特的视角，真实的细节，带给你不为人知的带劲故事。


忍受家暴 60 多年的老太太，是如何在临终病床前宣告了自己离婚的消息？


同事眼中的老好人，为什么会因为被扰民失眠，就把女邻居绑架了三天？


妻子遭遇性侵，更大的伤害却在案发之后，丈夫怎样「为她而战」？


想「替天行道」的连环杀人犯，却在作案现场呕吐，他的偏执从何而来？


![img](https://pic1.zhimg.com/v2-30ac07c158a55e2dd4cbbdfbeb4501ae.png)

一位从业 8 年的刑辩律师，经办的卷宗能堆满一个十几平米的房间。在经历各类案件、与当事人交流的过程中，他收集大量资料，追溯他们的人生轨迹，也看到了人世间各种错综复杂的故事。


挖掘施暴者的经历，是为了规避暴行产生的诱因，而不是为暴行提供合理性。


相较于迫不及待地处死一个罪犯，我们更需要了解这个人，是怎么一步步走向犯罪的。


他们在想什么？为什么家庭、学校、亲人、朋友、社会，没能接住他？与其问「我们与恶的距离」，不如诚恳地重新审视，这你我共存的世界。


本专栏全部来自真实案件手记。


![img](https://pic1.zhimg.com/v2-70b7203ed07619e4f2ee5c519e1a48fb.png)

#### 专栏亮点


1、电影级质感的阅读体验。


2、颠覆认知，用另一个视角看同一个世界。


3、了解职业刑辩律师的真实日常。


4、专业法律知识科普，理解法律的精神。


#### 编辑推荐


律师刘焱在一次会谈时，当事人冷漠地问他：「说什么狗屁专家，研究这研究那，能研究出自个是怎么个活法吗？」


也许这正是本专栏 12 个故事想要探索的。


黑色与恐惧无法驱散我们的热情，每一个案件卷宗的梳理和记录，都在提醒我们保持对现实的积极和警惕，而非排斥与恐慌。


我们也许不喜欢自己所看到的，但这不是我们背过脸去的理由。去直面它，能够让我们更勇敢的去生活。
