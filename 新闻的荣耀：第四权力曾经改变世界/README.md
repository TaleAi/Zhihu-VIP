## 简介
不服从是人类与生俱来的美德。


在人类历史上，发生过很多次记者、编辑、专栏作家等人通过新闻媒体，曝光政治的腐败、战争的残暴、制度的腐朽、人性的黑暗的事件。


他们突破层层阻碍，顶住政治压力，甚至冒着生命危险，将真相揭露到世人面前。


本专栏将选取历史上的八大重要新闻事件：


韩国 N 号房事件，两名女大学生和媒体记者通过卧底调查，曝光了这个堪称女性炼狱的「熔炉」，三大主犯因此落入法网；


孙志刚事件，记者调查发现了这个打工者真正的死因，在社会上掀起了大讨论，最终成功结束了收容遣送制度；


……


随着时间推移，这些新闻报道不仅没有失去价值，反而经受住了岁月的考验，为人类提供镜鉴。


它们告诉我们，真实世界不是天然美好的，正是如你我这般普通人的共同关注和参与，推动第四权力共同改变世界，才让人类拥有更加光明的未来。
