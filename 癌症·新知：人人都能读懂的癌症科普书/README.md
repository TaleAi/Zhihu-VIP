## 简介
提起癌症，很多数人想到的总是「不治之症」，也因此害怕它，忌讳谈论它。可避而不谈无济于事，如果我们能对它多一分了解，才能真正离它更远一些。本专栏中，出版畅销书《癌症·真相》，用通俗俏皮的文字剖析癌症，深受喜爱的科普作家李治中（菠萝），将为我们介绍更为前沿的抗癌知识，通过最新的癌症研究成果解读：癌症是什么，如何防癌、科学面对抗癌，哪些癌症流言不靠谱，用科学终结恐慌。


近些年来，癌症似乎成了人们生活中躲不开，又避之不及的一个词。触目惊心的朋友圈谣言，五花八门的网络「科普文」，无不令人谈癌色变。尤其是随着治疗技术和药物研究的不断发展，对这个困扰已久的健康难题，我们有着越来越多的新问题：


国际前沿的免疫疗法有哪些？对什么癌症有效？


用靶向药物为什么会出现耐药？应该怎么办？


把癌症变成慢性病可能吗？


患癌后，应该首先问医生哪些问题？


年轻人预防癌症，需要做些什么？


本专栏中，李治中教授用通俗易懂又略带俏皮的语言，将这些关于癌症的前沿知识讲给大家听。他用人人都能读懂的方式，多角度阐述了关于癌症的最新研究成果，各种关于癌症的传言，以及普通人应该如何科学预防、面对癌症。


通过本专栏，我们希望：


患者和家属能更好地理解医生的推荐和选择；


医务工作者能更好地知道前沿的药物研究方向和成果；


普通人能了解癌症的本质和社会上癌症热点新闻背后的科学。


最后，希望大家能多癌症多一分了解，不再恐慌，不再盲从。
