## 简介
病娇、忠犬、强制，相爱相杀、神魂颠倒、痴缠虐恋……和纸片人谈绝美恋爱的乐趣，在这里全都能 Get 到！


云烟雾绕中，一扇金色的大门缓缓向你打开。


欢迎来到龙之九子世界！


龙长子囚牛，喜音乐，常立于琴头——


**便是温润公子，翩翩风流。**


龙二子睚眦，嗜杀喜斗，常常在刀环、剑柄吞口——


**其实睚眦必报下蠢萌又较真，单为你傻傻守候。**


龙三子嘲风，平生好险又好望，威严肃穆，辟邪安宅——


**却会在百万军马中冲你不羁一笑，鲜血淋漓也不皱眉头。**


龙四子貔貅，器宇轩昂，只进不出，招财进宝——


**傲娇爱吃醋的他财迷又抠门，却宠妻到令人发指。他赚钱，你败家就好**。


龙五子狻猊，平生喜静不喜动，好坐，又喜欢烟火——


**一枚冷漠霸总，表面安静孤冷，实则内心火热，强势又舔狗。**


龙六子饕餮，生来一张深渊巨口，贪婪成性——


**平安喜乐吃货心下，是扮猪吃虎纵横捭阖的腹黑心计。**


龙七子狴犴，形似虎，仗义执言，威严好讼，常蹲点狱门之上——


**众人敬畏的完美男神，却是一根弦的小书呆，被你调戏耍弄也没了脾气。**


龙八子负屃，身似龙，雅好斯文，盘绕在石碑头顶——


**温润如玉背后一个斯文败类，洁癖又龟毛，还总爱搞点「残暴」的游戏。**


龙九子椒图，性好僻静，忠于职守，看家大门上的铁环兽——


**忠诚又粘人，不过小心小奶狗被逼变小狼狗。**


……


各位小主们，准备好 Pick 哪位男主了吗？
