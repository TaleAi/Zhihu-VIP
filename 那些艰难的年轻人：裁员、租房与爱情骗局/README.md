## 简介
租房遇到黑中介，信用卡还不起，工作没几个月被裁员……哪个年轻人没被社会迎面泼过一盆冷水？


欢迎来到成年人的世界。


当你走出白色象牙塔，正式踏入社会，迎接你的是机遇，也是挑战。


他们透支未来半年工资才租到的「老破小」卧室，临时被通知搬家，没想到身上凭空多出五位数的债务，为了从黑中介手中要到钱，他们以命相搏；


灯火通明的写字楼里，为了高悬在头顶的末位淘汰，应届毕业生敲打着键盘，她的背后扛着遥不可及的 KPI，和不知道何时才能拿到的转正工资；


她刚毕业就要反哺家庭，父母要买车，弟弟要盖房，还有数万元助学贷款等待偿还，无奈之下只得独自一人劳务派遣到新加坡打工赚钱……


走入社会后，鲜花和掌声难得，墙壁却无处不在。


你或许开始学会和不喜欢的人虚与委蛇；享受一周 7 个工作日，及时删除自己的坏脾气，回复一句「好的」；为了一方公用的卫生间，和同一个屋檐的室友互撂狠话；等着为数不多的工资，填补账面上的欠债……


成长，充满阵痛，而这个社会的拍打，会让你更坚强。
