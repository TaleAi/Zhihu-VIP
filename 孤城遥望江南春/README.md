## 简介
这是一本没有宫斗、不涉及争宠的宫廷治愈文，古言·虐心·双向救赎。


周温和铃铛的相遇，注定是飞鸟与鱼的爱情。


他是少年天子，多智近妖，没被爱过，也不会爱人，习惯用权谋去夺取想要的一切，这其中也包括爱情。


而她，是江湖游侠，自由天真、敢爱敢恨，习惯了用真心换真心，容不得欺骗和背叛，一旦触及底线便是彻底决裂。


爱情于别人而言，是甜甜蜜蜜、欢欢喜喜，于他们却是势均力敌，感情与智商的博弈。


一枚行走的活算盘 VS 一把出鞘的利剑


这场爱情的游戏，走到最后，才知鹿死谁手……
