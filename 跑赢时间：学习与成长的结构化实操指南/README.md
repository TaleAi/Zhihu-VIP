## 简介
掌握结构化思维，突破成长边界。


你长期沉迷在各种社交平台带来的信息之海里，苦恼于自己知识碎片化的弊端，却无法脱身，不知道怎么建立自己的知识体系？

你刻苦努力，想要改变自己的现状，却发现自己总是没有学习与提升的时间，甚至连应该努力的方向都不知道……


Lachel 长期致力于系统化学习与心智成长方法的探索。他的实践已被数十万知乎用户认可与证明过，不是纸上谈兵，而是少有的，可以实际操作的干货。


结构化自己思考与学习方法、提升自己阅读与时间利用的效率、突破自己认知与成长的边界。


祝你可以赢得自己与时间的这场长跑！
