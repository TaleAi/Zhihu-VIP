## 简介
本专栏收录了世界科幻名家名作25篇。


本专栏收录的短篇故事中，有《冰与火之歌》作者乔治马丁流传世界的短篇名作《沙王》，这是他唯一一篇同时获得雨果奖和星云奖的作品。与此同时，本专栏还收录了日本科幻作家筒井康隆的名篇《立女》，他与星新一和小松左京并称 20 世纪日本推想小说三巨头。诸多殿堂级科幻名篇，尽在此中。
