## 6.寻找黑洞

80 年代初，我在英国进修，进修的单位叫作皇家爱丁堡天文台。大家知道爱丁堡是苏格兰的首府。英国的天文台有两个称为皇家，一个是皇家爱丁堡天文台，一个是皇家格林尼治天文台。爱丁堡天文台藏了很多的古书，我在那里第一次看到很多中国的古书，


特别有兴趣的是：在中国的古书里边，居然在清朝末年，有用古文写的微积分，很有意思。同时我也注意到在这个爱丁堡的北边有一个湖，叫作尼斯湖。说这个湖里边有鬼怪。这个鬼怪来讲的话，天文学有时候就把黑洞比作是鬼怪，所以我们想去看一看吧。


我英国的一个朋友就开车一块儿到了尼斯湖。这我说一下，英国人的英语用得很怪，在苏格兰来讲，这个湖不能叫 lake，一定叫 loch，所以尼斯湖就叫 Loch Ness。然后看着鬼怪就叫 monster。


我站在那上面一看，尼斯湖这个地方成天是云雾缭绕，大家知道英国阴天比较多。这个湖是一个狭长的一个带，非常的深。鬼怪没有看见，但是使我联想到了寻找黑洞，大家想一想，在地上看鬼怪这么难，这个鬼怪天文学就叫它「黑洞」，我们去找黑洞容易不容易啊？更不容易。那么在以前很长一段时间，我们找不到，所以对黑洞的研究不是那么热。这些年由于找到了黑洞，所以我们黑洞的研究特别的热。


那怎么能找到黑洞？根据我们以前介绍的，黑洞是完全看不见的，所以如果天上有一个孤零零的黑洞，我们是无法找到的！所以我们找黑洞的关键一定要找这个黑洞和一个普通的星结合成双星。


你说天上的双星多不多啊？相当多。大概说，我们肉眼看到的星有一半是双星。如果一个黑洞和一个双星结伴，这个黑洞就有可能辐射 X 射线。为什么呢？我们看一下这是一个双星，一个是黑洞，一个是普通的星。这个黑洞吞噬了普通星的物质，就要辐射 X 射线。这个 X 射线不是从黑洞里边发出来的，而是在黑洞吞噬物质的过程中吃得特别猛，在黑洞的视界面的外边，这个物质会剧烈地碰撞，这一碰撞就要产生高能粒子，这些粒子就会辐射 X 射线、伽马射线。


什么是 X 射线呢？就是我们普通大家到医院里去透射用的这就是 X 射线。但是天上的星发出来的 X 射线到达不了我们地面，原因很简单，我们的地球大气有一层保护层把 X 射线遮挡了，所以我们要看天上的星体发出的 X 射线就必须放卫星！


所以我们放了若干个专门探测天体 X 射线辐射的卫星，叫 X 射线卫星，因此我们寻找黑洞的步骤就是这样。


第一步用天上的 X 射线卫星找到辐射 X 射线的源，我们叫 X 射线源。找到以后我们再进一步普查，X 射线星会不会是一个双星？因为只有双星我们才感兴趣。如果说我们发现这个 X 射线源是一个双星，我们就进一步计算一下这两个双星每一个星的质量。计算出来以后，如果我们再发现一个星看得见，一个星看不见，而且那个看不见的星质量又相当的大。根据这一些，我们现在就判断了，看不见的那颗伴星就是黑洞。第一个黑洞就是这么找到的。


最有名的一颗黑洞叫作天鹅座 X-1。我们天上的星，为了查找方便，都画成一个区一个区，叫作一个一个的星座，其中在天鹅座我们观察它里边的 X 射线，第一个被发现的，X 射线比较强，所以命名叫作天鹅座 X-1。就是天鹅座 X-1 的照片。


 


你不是看见天鹅座 X-1 了吗？不是说黑洞看不见吗？这张照片我们看见的是双星的可见的伴星。我们看一下这两颗双星的物理数据。可见的星的数据就这样的，不可见的是这个数据。最后我们就判断出来了，这个看不见的、辐射 X 射线的天鹅座 X1 就包含了一颗黑洞，我们黑洞就是这么找到的。


大家知道，找到一颗就更容易找到更多的颗。所以这些年来我们在银河系里边就大量地寻找究竟有多少黑洞。首先我们先估算一下，你说我们银河系里有多少黑洞？我们说这黑洞是怎么产生的？是恒星爆发产生的。大家知道，这个恒星到了晚年绝大部分恒星都要经历一次爆发，但是并不是所有的恒星爆发都会产生黑洞，必须是叫作超新星爆发才能产生黑洞。那么超新星爆发多不多呢？比较少，大概说来，我们肉眼能够看到的，在银河系里边一般来说都得 100 年左右才能够发生一次。比方说，有名的一次超新星爆发是在宋朝 1054 年，那会儿就看到了一个超新星爆发，因为是在天关星的旁边，所以叫作天关客星。这颗星中国人在古代有了详细的记录，非常重要。


另外后来我们曾经提到过，丹麦有个天文学家叫作第谷，他当年看到过一次，所以就取名叫做第谷超新星。根据估算，最近这些年我们银河系里有可能再产生一次超新星爆发，我告诉大家，超新星爆发的时候，白天都能看到，所以大家拭目以待，好好等着，说不定这一代人，我们在银河系里会看到另一次超新星爆发。


那么是不是每一个超新星爆发都能够产生黑洞呢？不是！大约 100 个超新星爆发有一个能产生黑洞，所以我们就估算一下吧——我们银河系里边到底有多少黑洞呢？我们银河系到目前为止，已经生存了 100 亿年。所以大家想想看，100 年里边有一次超新星爆发，100 个超新星爆发里边有一个会产生黑洞，所以我们把这 100 亿除以 100 再除以 100 等于 100 万。所以我们估算了一下，银河系里边应该有 100 万个黑洞，


大家看这张图，我们观测到的、比较有名的黑洞都标在银河系的图里边，


我们到目前为止观测到了多少银河系里的黑洞呢？连万分之一都不到，所以含有大量的银河系的黑洞我们都没有看到。看到银河系的黑洞，我们发现，原来我们银河系的中心也有一个黑洞。大家知道，我们的太阳是处在银河系里边，我们人类当然也在银河系里边。可是因为银河系的中心这个星啊太密集了，物质也太多了，所以我们银河系的中心是直接看不到的，因为我们是在庐山中。那我们怎么来考察银河系中心呢？我们现在用射电望远镜、红外望远镜，透射力比较强的。但是即使这样，也不可能看到银河系的中心；但是我们可以看到银河系中心周围一些恒星的运动。这张图就是显示的银河系中心恒星的运动情况。我们发现这些恒星都是绕着银河系的中心在旋转，而且这个旋转的规律就是以前我们提到的开普勒的三定律。


根据恒星绕银河系旋转的规律，我们可以计算一下，中心一定有一个大质量的天体。计算结果居然发现，银河系中心这个天体的质量是多少呢？是 400 万个太阳质量。你们想想看，一个单星，质量能有这么大吗？所以最后推算出来，银河系的中心也是一个巨大的黑洞。


原来我们银河系里有这么多的黑洞，包括银河系的中心也是黑洞，你说银河系里有这么多黑洞，对我们人类的生存、对天体的演化会不会有影响呢？
