## 简介
知乎三绝之一，看虐文女主如何逆风翻盘，收获爱情


我穿越了，还是······穿书，来不及庆幸自己掌握了上帝视角，我就悲催地发现自己变成了书里的恶毒女配。


更要命的是，我穿越的时间节点正是小说里女配坏事做尽后开始走霉运，原女主触底反弹的转折点。绝境之下，我似乎只能苦修恶毒女配保命攻略，伺机跑路。


奈何我还有一堆拖后腿的「猪队友」——醉心权术的奸臣父亲，成事不足、败事有余的纨绔兄长，再加上原女配在小说前半段的作天作地······此时的我无疑是身陷囹圄，想跑路就算作者答应，恐怕女主也不肯。


而言情文里的标配——霸道腹黑男主、温柔缱绻男二，甚至是男三、男四······全是玛丽苏女主的裙下之臣。我一个没有金手指的炮灰女配，为了保命，只能时刻牢记切勿贪图美色痴心妄想，谨记八字真理——美男虽多，不是我的。


只是蝴蝶效应从来都不是恐吓，男主逐渐态度不明摇摆不定，本该钟情女主的男二，莫名开始针对试探我，只是不知这两人是想要我的情，还是我的命？心思晦暗的男一号，笑里藏刀的男二号，再加上原小说里都没有的新晋小奶狗男三号，一个比一个令人琢磨不透，而我究竟该选择谁？


不同世界形成价值观之间的碰撞，终究带来一发不可收拾的连锁反应。带着虐文里恶毒女配的青铜装备，我这个新时代的奋进女青年能否逆风翻盘？
