## 简介
六朝传奇，一世风华，还原大明奇后。以孙后的人生为主线，带领你一窥大明王朝从初建到国立鼎盛再由盛转衰的那段历史；透过人物背后，探究「帝王将相与历史大事件」背后的争议与谜团。


一个出身低门，十余岁进宫，在大明朝的汹涌风云里，历经了永乐、洪熙、宣德、正统、景泰、天顺六朝的女子，她是明宣宗朱瞻基的皇后，也明英宗朱祁镇的生母。


在历史的长河里，留下了很多女人的名字，但她的名字和故事却至今鲜为人知。虽然她没有长孙皇后的盛世流芳，没有武则天的丰功霸业，亦没有慈禧太后的崇高地位。但，她身上的传奇故事，同样精彩绝伦，她背后的大明王朝更是波澜壮阔。


百转千回的爱恨纠葛，步步惊心的宫斗大戏，惊心动魄的政权角逐，从贵妃到皇后到太后再到太皇太后，她几乎一生都在深宫中度过。


她，是幸运的，因为朱门宫阙内，她拥有过难能可贵的帝后绝恋；然而，她又是不幸的，即使历经六朝五帝，母仪天下的她，却不得不在国局政乱里背负了骂名。


在《大明风华》中她是大女主孙若微，在《女医·明妃传》中她又成了大反派孙太后。她究竟是温婉贤良，还是凶狠毒辣，是一代贤后，还是腹黑妖后？


让我们走进本专栏，为你呈现有血有肉的孙后，听听她传奇的一生。
