# 梁衡给孩子的科学启蒙：学科学简史、培养科学思维

### 作者：梁衡

## 简介
本专栏通过多学科、多视角的生动科普故事，以故事为线索渗透科学精神，将科学原理浓缩为科学模型示意图，立体感知科学的实证精神。一张清晰明了的科学史简表，带你把我科学发展的历史脉络。


每一个公式、每一条定理背后都隐藏着一段血与火、泪与汗的历史，这里有慷慨的悲歌，也有胜利的喜悦。


要论人物形象、故事情节，一部自然科学史绝不比社会科学逊色。


科学启蒙有什么难处？表现在兴趣偏差、思维局限、思辨缺乏、积累低效、浅层学习等方面，只学到科学知识，但形不成科学思维。


无论读者年龄大小、职业差异，都可以从科学的发展史和科学家的成长史中汲取科学的营养，感悟和领会科学精神。本专栏以阿基米德、祖冲之、牛顿等科学家的故事为线索，从「世界本原」到「人类基因」层层深入，融合科学与历史讲述数理化原理，增强孩子对科学的兴趣。
