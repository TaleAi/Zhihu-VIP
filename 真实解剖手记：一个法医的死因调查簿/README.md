## 简介
「尸体不可怕，活人才可怕。」


在罪案现场，警察手中拿的是枪，而法医拿的是勘查箱。


他们用组织剪、止血钳……从一片血腥之中，寻找案件的关键信息，与尸骨「对话」，以此揭露真相。 


法医更擅长用技术破案。


每一次技术上微小的的革新，都能让一批曾经的悬案告破。


![img](https://pic2.zhimg.com/v2-865891ffc80562c86e03de5cd28986ad.png)

法医廖小刀，在他的新实验室建成后，连续半个月每晚都要扎 50 多人的手指。依靠「DNA 人海战术」，破获了当地拉锯 10 年的「天字第一号案」；


  




如果说血液、头发、指甲、伤口……是受害者留下的「无声遗言」，


法医就是那个读懂密码的人。


同样是一滴血，20 年前只能检测出血型，如今可以直接锁定一个人。地砖缝里 13 年前的「滴落血痕」，就足以让法医刘八百比对出嫌疑人。


![img](https://pic4.zhimg.com/v2-b7bf1c22ef872a3086a7466d08e131ae.png)

更多时候，法医们都在等待，就像普通人等待一份化验报告那样。


建立资料库，核查、比对……


然后找出真相。


  




**这也是看法医故事最令人紧张的时刻：**


「他们相信，凶手还会冒头的。 


   那一天，到底还是来了。」


![img](https://pic1.zhimg.com/v2-70a57724ab6c041703dfdc25465c7851.png)

 


#### 为什么看


技术可以通往真相，但无法预测人心。


每一具尸首的背后，都是活人在算计。


每一条生命的逝去，都因为一个讽刺而冷漠的世界。


  




刘八百曾遇到一个「伪装的奸杀案」，受害者生前是个很美好的姑娘。


看过这个故事的人忍不住叹息：


**「养的太好了，太乖了，一点不懂人性的善恶，如果她懂一点人性的复杂。」**


![img](https://pic3.zhimg.com/v2-e97684d633a760479272c4f02ba28db0.png)

法医二人组碎碎念警示：


理解人性的幽微，保护自己免受伤害，也警醒自己不要糊涂。


**任何一点微小的恶念，都应该被锁住。**

**了解**物证学、生物学、病理学……知识


**感受**案件侦破过程中，经验与技术的交锋
