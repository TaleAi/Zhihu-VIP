## 78.从《说文解字》看盖提尔问题
**导语**


在讲述盖提尔问题的时候，我们举过一个子路的例子，他说「要么颜回有一辆马车，要么曾子在楚国」，结果歪打正着，蒙对了。本集将沿着这个思路继续讨论，即虽然这个命题在逻辑上是成立的，但是如果按照《说文解字》中对于「或」这个字的解释，这却是一个没有意义的命题。本集节目将包含以下要点：1.《说文解字》中「或」的含义；2. 该如何使用「或」这个字。


  



各位听友大家好，我是徐英瑾，非常开心和大家继续来讨论用得上的哲学这个话题。前面一次节目，我们已经把中国传统文化的正名论和西方的融贯论做了一些比较，今天我们仍然要沿着这个思路来讨论一下，如何从《说文解字》的角度重新看待盖提尔问题。


《说文解字》我们都知道，是东汉学者许慎所写的一本类似于字典一样的书，但它和今天所说的字典又有点不同，它对于很多字和词意的解释是掺入了所谓的儒家观点，所以我更喜欢把《说文解字》定义成一本带有儒家色彩的字典。


关于盖提尔问题，我们前面已经说过了，盖提尔问题有问题一和问题二，今天这次节目限于时间之所限，我们就只想谈谈问题二。现在我再把问题二重复一下，忘记盖提尔问题的听友们可以就此复习一下。


子路有很好的证据以证明颜回拥有了一辆新马车，这个证据就是他想起来昨天颜回驾驶着马车带着自己去兜风。而且子路还与曾子是好朋友，但是他的确不知道曾子现在在哪里。但是，如果子路能够熟练使用逻辑推理规则的话，那么根据前面所说的这些信息，他就能够得到下面的推论了：要么颜回有一辆马车，要么曾子在楚国。


请注意，子路是非常有把握的，他相信自己关于颜回有一辆马车的记忆没有任何错误，在这种情况下，曾子管他在南极还是在楚国，都与整个命题的真假是不会产生影响的，所以他说，「要么颜回有一辆马车，要么曾子在楚国」这个信念肯定是真的。


但是，我们知道盖提尔问题的实质是什么呀？是歪打正着。这个信念最后的确是真的，但真的方式有点古怪。子路的记忆的确没问题，昨天他的确坐上颜回的马车去兜风了，但是子路不知道的是，也就在刚刚，颜回的马车被鲁国的一个贵族给没收了，所以子路以为颜回现在还有一辆马车，但现在颜回没有了。而且非常凑巧的是，曾子现在的确在楚国，这件事是真的，尽管子路他说曾子在楚国，这一点仅仅是出于他自己的瞎蒙，但他蒙对了。


所以，我们在这种情况下就发现，「要么颜回有一辆马车，要么曾子在楚国」这样一个命题它仍然是真的，但它是以一种非常奇怪的方式成真的，也就是子路认为前者是真的，后者是假的，因为前者是真的，所以整个命题还是真的，但实际的情况是，前者是假的，后者是真的，因为后者是真的，所以整个命题还是真了。换言之，子路以错误的理由拥有了一个真信念。在这样一个情况下，我们会不会认为子路知道「要么颜回有一辆马车，要么曾子在楚国」呢？大家认为他只是蒙对的，所以说就是不知道。


关于这个问题，《说文解字》的许慎会怎么看呢？或者说得更大一点，站在中国文化的角度上，会怎么看这样一个非常烧脑的逻辑案例呢？请听我下面慢慢掰扯。


#### **《说文解字》中「或」的含义**


不难看出，在西方哲学的领域里面之所以会有盖提尔问题，它是预设了逻辑算子的运算规则。我们前面已经说过了，「或」这样一个算子的运算规则就是什么呢？A 或者 B 里面只要有一个真，整个句子也就真了。如果我们说得再复杂一点，A 或 B 或 C 或 D 或 E 或 F 或 G，其中只要有一个是真的，那就真了。这就是他们关于「或」的一个想法。但是，这个想法是不是符合《说文解字》里面对于「或」的定义呢？


我自己看了看《说文解字》，《说文解字》里对于「或」这个字的定义挺有意思的，叫「邦也。从口从戈，以守一。一，地也。域，或又从土」，这话听上去玄玄乎乎的，什么意思呢？「或」这个字的构成，大家想想看，里面是不是有个口字？它是不是还有个戈？戈就是兵器。所以它有一个口的成分，有一个戈的成分，它合在一起表示什么呢？是守一方疆土的意思，所以许慎就说，这是「邦」的意思。


这就非常好玩了，原来「或」这个词，它本来的含义是指一方疆土，这和我们今天讨论逻辑意义上的「或」，好像是一点点关系都没有，这两者之间又是怎么发生关联的呢？


清代的一个很重要的训诂学家段玉裁给出了一个解释，这个解释是什么呢？叫「以凡人各有所守。皆得謂之或。各守其守，不能不相疑。故孔子曰：或之者，疑之也」，这段话到底是什么意思？我稍微来解释一下。


「或」这个词它的原始含义是什么？就是防守一方、防御一方，各个防御者所得到的这块防御土地本身就叫作或。但是你可以设想一下，不同的防御者他们所得到的这块叫作「或」的土地，彼此之间都是有所疏远的，正因为有所疏远，所以这些防御者之间就会彼此生疑，怀疑别人可能会不利于自己，或者至少怀疑自己有困难的时候不会来帮自己。所以，从「或」这个字的含义里面，它就引申出了另外一个含义，就是怀疑。


这挺有意思，本来是讲守卫，讲土地，现在变出了怀疑。讲到怀疑这一步，又怎么变成今天所说的「或」？


大家想想看，你做道选择题，A 和 B 显然不是正确答案，你就会想了，是 C 还是 D？这句话也可以换一个方式说，你怀疑是 C，也怀疑或许是 D。所以，怀疑的含义和或的含义是密切相关的，「或」本身就意味着一种怀疑。「或 A 或 B」，就意味着你要在 A 和 B 之间选一个，但是你对 A 也不是很相信，你对 B 也不是很相信。如果你相信其中一个的话，你就不会用「或」了，你就会直接说这是什么了。


讲到这一步，有一些背景性的文化知识。《说文解字》虽然是在东汉初年所完成的一部带有儒家色彩的字典，但实际上它里面所解释的这些字的含义很可能在周代就有了。那么在周代的社会环境下，中国当时是分封制，周天子分封天下，姬姓诸侯各守一方，所以在分封制的情况下，因为当时交通地理方面的不便，各个诸侯之间显然会产生相互的猜忌，而这种相互的猜忌就会产生彼此的怀疑，这也就是「或」这个词的语义得以产生的一个历史背景。


#### **子路这个命题没有意义**


如果我们按照「或」的这个含义，重新来看待盖提尔问题的描述的话，我们就会觉得这个问题它本身是不会产生的。


原始的案例是什么呀？原始的案例是，子路他脑子里有一个信念，这个信念是基于他自己的记忆的，也就是昨天颜回弄了辆马车带他去兜风了，所以这辆马车是颜回的。对于这一点子路有没有怀疑呢？没有怀疑，子路是非常相信自己的记忆是可靠的。他也相信颜回这个人是很老实的，是绝对不会拿了别人的一辆马车来冒充是自己的，他不是这号人。


这时候，子路脑子里乱蒙出了一个想法，这个想法就是，他的好朋友曾子现在在楚国，但对这一点子路有任何主观上的确信吗？子路根本就没确信，他就是胡诌一句，他也知道自己在胡诌。在这种情况下，这两个子命题，一是颜回有一辆马车，第二个子命题就是曾子现在在楚国，这两个命题并不存在着一个必须要在两者之间进行取舍的问题，对于子路来说，他就会认为第一个是成立的，第二个就是胡诌的，没根据的。


这和我们前面所说的，你要做一道题，A 和 B 显然是不对的，这 C 和 D 哪个答案是对呢？我不知道，在 C 和 D 之间选一选。和这种情况是完全不同，因为在 C 和 D 之间选择一个正确答案的过程，是 C 也有点根据，D 也有点根据，但我又不知道 C 和 D 之间哪一个根据更加充分，所以我犹豫了。


但是对于子路来说，他遇到了这种选择的情况吗？我觉得他没有遇到。所以，我倾向于认为，子路如果他是一个按照正常的思维方式来进行思考的话，他或许就不会说出这样一个很奇怪的话，就是「要么颜回有一辆马车，要么曾子现在在楚国」，他说不出这样的话。


有人说，按照现代逻辑的要求，这样说是没问题的啊？但问题是，如果按照现代逻辑的要求说话是没问题的话，我们就可以在日常生活中造出大量的这样用「或」连接出来的句子，它们一方面的确是符合现代逻辑对于「或」的规定的，但另外一方面，你说出这些命题你都会觉得没意思，为什么呢？它们在日常生活中没用。


比如，你肯定确信 2+2=4，但是月球背面有一个坑，嫦娥在里面洗澡，你知道不知道呢？你想这肯定是胡扯吧。但是按照「或」的这个原则，你肯定能说出这样的话，就是：要么 2+2=4，要么月球背面有个坑，坑里面嫦娥在洗澡。你觉得说这种话有意思吗？你说出这种话，你是指望别人怎么来和你接话茬呢？人家是不是觉得你有病呢？


更重要的是，这样的用「或」所连接起来的古怪的句式，会大大消耗你自己的心力、你的时间，所以没事就想这种事，它是一种病。


#### **该如何使用「或」这个词**


说到这里，爱动脑筋的听友可能就要怼我了，他们会说，在上期节目里面你批评了儒家的正名论，觉得儒家的正名论就是拿过去的这些规范来套今天，你觉得这不对，因为生活在改变，社会在改变，我们得进步。但今天为什么好像是自打嘴巴了？说许慎说得好，许慎对于「或」这个字的解释非常有道理，我们要用周代对于「或」的解释来解决当代美国哲学家提出的盖提尔问题？现在讲的思路和上一次讲的是相反的？


那我在这里就要稍微解释一下了。我是反对食古不化的，老觉得先人讲的话就是对，这个观点显然太迂腐，但是我也反对为反对而反对，先人讲的有道理的话你也得听，你得具体看他讲的哪些是符合现在情况的。


那么我今天就要指出，周代对于「或」这个字的理解，或者说是许慎对于「或」这个字的解读，从某种意义上说还是有道理的。尽管周代的那种社会生活今天我们早就没有了，今天我们交通的条件、我们通信的条件，并不会使得两个相距很远的人很容易就生疑了。甚至不要说是在两个不同的邦国，就是在地球的另外一边，视频也可以对话。只要通讯手段跟得上，我们并不一定就会产生相互之间的猜忌。


但是，有一点我们和古人是一样的，哪一点一样？就是我们解决问题的一些基本的步骤是一样的。这个解决问题的步骤是什么呢？也是我以前所讲过的，就是由人工智能专家、经济学家赫伯特·西蒙所提出的目标-手段径路，或者说目标-手段研究方法。


举个例子，你现在要达成什么目的，你先把目的确定下来，比如你现在要吃饱，然后看看你现在的状态，那现在的状态是没吃饱，你就发现，没有吃饱的现状和吃饱的这个目标之间有差值，那怎么样把这两者之间的空隙给填满呢？你就得想办法了，使得你的不饱的状态能变成饱的状态。


于是，你就在诸种可能的选择方式里面进行挑选，比如你吃狗不理包子也可以饱，吃馄饨也可以饱吃，吃肯德基也可以饱，所以或 A 或 B 或 C 或 D 或 E 或 F 或 G。这些列举都有一个前提，这个前提是什么？你列的这些事情都是和你当下的任务有关的，而且也适合你当下的购买能力，或者与你当下的肠胃的适应能力是有关的。


我随便举一个例子，吃金枪鱼寿司显然能饱，但你手头就 12 块钱，那金枪鱼寿司肯定是买不起的，对不对？所以这和你的财力有关。


另外一种可能性，你不差钱，但是你朋友在旁边提醒你了：我知道兄台你是不差钱，但是你吃海鲜过敏，你上次吃金枪鱼寿司的样子你还记忆犹新吧？浑身出了疹子，狼狈不堪。你如果还想如此狼狈的话，请你吃。你一听，唉呦，慌了，我可不想这么狼狈，那就吃别的东西。


所以，我们所说的能够被列到「或……或……或……」概念之下的那些选项，一方面是和我们要解决的问题的性质相关，它一定要解决一个问题，同时也要和我们的客观情况、我们的主体的供应能力都有关联，你随随便便说一些不着边际的话，那就完全不行。


如果不理解这一点的话，就会造成很多的笑话。比如这样一个笑话，西晋的第二个皇帝晋惠帝司马衷知道灾民吃不起饭了，他就问了一个问题，「何不食肉糜」，为什么不吃肉饼子？他就忘记了，站在灾民的立场上，吃肉饼子这件事情完全不在他们的考虑范围之内，连米都找不到，他们怎么会有钱去买到肉饼？完全就是胡扯。这也就是说，不是在逻辑上这个命题有意义，在实践中就有意义了。


由此，我们在反观许慎对于「或」这个字的解释，虽然他一开始讨论的是土地、防御这些概念，但是许慎的讨论里面他一直预设了这一点，或者说至少在段玉裁对于许慎的解释里面预设了这一点，也就是各个土地防御者之间是有某些相关性的。什么叫相关性？也就是说，不同的土地防御者，他们虽然彼此相距遥远，但是他们仍然有共同的利益关切，而这种共同的利益关切才使得他们相互猜忌。这也就是说，他们仍然是在一个共同的讨论语境中被相互比较的。


由此可见，如果我们要在对知识进行表述的时候，我们其实是可以使用「或」这个词的，但是「或」这个词本身要慎用。


第一，「或」这个词本身就带有疑问的含义，而「知道」这个词本身就带有肯定的含义，所以你说「我知道正确的答案无非就是 A 或者 B」，你姑且可以这么说，但是，「或」这个词的疑问的含义，会冲淡「知道」这个词所带来的肯定的含义，所以这样的句子说出来多多少少有点奇怪。


第二，如果我们不得不使用「或」这个字，我们一定要保证由「或」所连接起来的这两个子命题是彼此相关的，而且它们都是在同样一个问题处理语境当中被提出来的，而且你多多少少都有一些证据，来证明这两个子命题当中的任何一个都是有其根据的，而绝不能够抱着一种极不负责任的态度，胡诌一个你自己都不相信的子命题，比如说曾子现在在巴塞罗那，曾子现在在布宜诺斯艾利斯，这种话就不要说了。
