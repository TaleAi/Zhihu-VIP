## 97.先天语法：语言是人类与生俱来的一种天赋？
**导语**


如果认真观察 0 到 3 岁的小朋友，我们会发现他们学习语言的速度是非常快的，很多复杂的表达只要听几次就能学会，这是为什么呢？语言学家乔姆斯基就认为，这是因为我们一出生就有一套「先天语法」在我们的脑子里了。这种说法讲得通吗？这就是我们今天要讨论的话题。本集节目将包含以下要点：1. 维特根斯坦的语法理论；2. 乔姆斯基的先天语法理论。


  



各位听友大家好，我是徐英瑾，很高兴和大家继续来讨论用得上的哲学这个话题。


我们今天所讨论的这个话题和语言学有一点联系。传统的语言哲学有个缺点，它们不太关心语言学在干什么，这给我的感觉就像艺术哲学不关心艺术家在干什么一样，好像有点名不正言不顺，所以我今天来谈谈语言学，我想谈谈乔姆斯基的观点。


我为什么要谈论乔姆斯基的观点？听了我这几回关于语言哲学的节目大家都应该知道，我本人是个维特根斯坦的支持者，我认为维特根斯坦的哲学是非常对的。维特根斯坦的晚期哲学特别强调一个很有意思的概念——语法，但语法这个概念听上去就是一个语言学的概念，并不是一个哲学概念，那凭什么哲学家要用语法这个概念？他说的语法和乔姆斯基所说的「先天的语法」到底有什么区别？这就是我这次要讲的重点问题。


**维特根斯坦：语法是语言中隐藏的搭配规则**


维特根斯坦要讲的语法，实际上不是语言学里面所说的语法，并不是说主语后面是主谓宾的结构，他想说的语法是什么意思？他是指有一些表达式只要说出来大家就会觉得很荒谬。


比如你要说 「红色非常悦耳」，当然可以在诗歌里面这么用，但如果在日常生活中说「我要一种悦耳的红色」，人家就会说「悦耳」只能指声音，「红色」是指颜色，这两者之间是不能胡乱搭配的。我们在日常语言当中有很多隐蔽的默契，这些词和那些词它们只能够这样搭配，否则这个搭配就不行，这些隐蔽的搭配的规则或者是隐蔽的搭配默契，就被维特根斯坦称之为一种语法。


他也尝试着对我们的语言当中的语法进行深入的研究。我们就拿颜色说事，有一些组合就是不行的，比如说你可以设想一种闪亮的银色，你能设想一种闪亮的灰色吗？这就有点难，因为灰色如果要让它变得闪亮的话，十有八九就是银的感觉了，画过画的人大概明白我在说什么。这也就说明，不同的颜色词之间的搭配有着非常隐蔽的默契，这需要我们在日常生活中多观察，这样我们的表达式才能够精准。


维特根斯坦恰恰指出，这不是语言学所能够处理的问题，不是语法的研究能够处理的问题，比如你说「红色是一种悦耳的颜色」，它的语法从纯粹的形式上讲是通的，但是实际上意思是不通的，维特根斯坦所说的语法实际上已经牵涉到语义搭配了，而不是纯粹的形式。


**乔姆斯基的「先天语法」理论**


我简单地讲了维特根斯坦以后，我就讲讲他的对立面，就是乔姆斯基同志。


乔姆斯基是美国著名的语言学家，又是政论家，他 1928 年出生，到现在还健在。我自己也和乔姆斯基先生在一次国际学术会议上对谈过，实际上我并不赞成他的观点，我认为他的观点有点是故意去找茬的，但不管怎么说，和一个非常重要的语言学家当面对谈过，也是我人生中一次很重要的经历。


乔姆斯基同时也是 20 世纪被引用最多的知识分子，一方面是他语言学方面的著作很多，一部分著作也牵涉到哲学，但他主要是个语言学家，同时他又是一个左翼的公共知识分子，写了大量的文章讨论美国的政治，我估计很多引用和那部分的书是有关的，因为那部分的书读的人就特别的多。


我不想掩盖自己的观点，我既不喜欢乔姆斯基的语言学观点，也不喜欢他的左翼的政治观点，而且我认为这两种观点的错误都有一个共通的地方，就是他是个理想主义者。因为他是个理想主义者，所以他要用一种数学家的精确，来刻画出我们语法背后的一种所谓的先天形式，也正因为他是一个理想主义者，他对于政治的结构、对于人类的公平和正义也抱着一种理想的观点，认为美国这里不对那里不对，他要站在理想主义的角度来进行讨论。


我自己在两个方面都不是这样的人。在关于人类语言的本性问题上我是个进化论主义者，我认为我们的语言是在进化的长河中所产生的一种副产品，这种副产品本身就是为了应付环境中的种种挑战，里面根本就没有什么理想，就是让我们凑合着活下去的东西。


至于对于政府的批评，我更主张的是要采取卡尔·波普的观点，卡尔·波普的一个观点是：你如果抱怨这件事不行，你就想一个方法把这件事给解决了，如果抱怨那件事不行，就想个方法把那件事给解决了。他称之为「零敲碎打的社会工程」。比如，你觉得这个水管不好了你就修水管，下水道不通了就通下水道，不要有这种理想主义的豪情，要把一切推倒了重来，没什么意思，凑合着过，也就这么回事。所以，「都挺好」「凑合着过」是我的一个信条。


这个信条既贯彻到了我对语言本性的观察上，也贯彻到了我本人对于政治问题的观察上，所以我不是很喜欢乔姆斯基这样的理想主义风格。但是我也必须承认，乔姆斯基很伟大、很聪明、很了不起，所以我还是要谈一谈乔姆斯基的观点是什么。


乔姆斯基对于语言的观点，多多少少和柏拉图是有关系的。柏拉图虽然没有全面讨论过语言的问题，但他认为人的心智是包含着某种先天的理性的，就是咱们没有出娘胎之前，我们就有某种先天的理性了。


他举了个例子，苏格拉底教一个小奴隶做一道几何题，小奴隶是文盲字都不识，但是就听苏格拉底和他讲，小奴隶基本上是在没有任何预先的几何学知识的前提下，就听懂了苏格拉底给他的那道证明题的意思，然后一步步就把证明题做出来了。凭什么没有什么文化的小奴隶也能够做出这样一道题目？柏拉图的解释就是，他心里面已经预先有理性的种子了。


接着这个话题，乔姆斯基就问了：0 到 3 岁的小孩，通过很少量的语料的刺激，就能够很快地学会母语的语法，这件事情听上去是比较神秘的啊？为什么比较神秘？如果你认为他是通过归纳、概括和经验观察来学会语法的话，一个婴儿他周围也就是爸爸、妈妈、姥姥、爷爷，就这么几个人，他们能说的话的数量你用录音记录下来也就这么一点，根本就是他的母语所能够拥有的表达式当中的一个很小的子集。在这样的一个情况下，你必须要对这件事做解释，到底是谁教这些小孩子说这些话的呢？


我自己也有个体会，因为我自己的女儿也是 4 岁了，我觉得她现在的句法结构也相当复杂了，经常会说从句了。像有一些很复杂的逻辑连接词，比如说「除非怎么样……」，「妈妈，我是不会再去背英文了，除非你再奖励我什么」。或者更简单的，就是「如果……那么……」，「如果妈妈你再多给我点奖励的话，那么我的表现会更好」，这话小孩都会说。这非常奇怪，这个话爸爸妈妈并没有教他很多，「如果……那么……」该怎么用，「除非」该怎么用，可能就听了一耳朵了，怎么听了一耳朵就会了？那也太聪明了。


乔姆斯基的解释就是，每个人心目中都有一个所谓的先天语法，先天语法是通过基因写死在里面的。当然，乔姆斯基并没有给出真实的基因科学的证据说明这个语法到底在哪里，他就是说应该是有这么一个东西，有了这样一个东西以后才能够解释，为什么在这么少的语料的刺激下小孩能够一点就通，因为智慧的种子已经在他心里面了，当然有点阳光有点雨露就发芽、茁壮成长了。


在乔姆斯基看来，一个小孩子学习语言的过程，它实质是个什么样的过程？就是怎么样在所谓的先天语法和经验的语法之间搭建一个桥梁。


这话到底是什么意思？比如我们就拿猩猩和人之间的差别来说，很多人尝试教猩猩语言，给它各种各样的符号——因为猩猩本身的咽喉结构和人类不一样，直接要求猩猩从嘴巴里发出人的声音是不可能的，除非你给猩猩做手术，科学家就用一个退而求其次的方法，给猩猩很多的符号——有的符号代表苹果，有的符号代表「我」，有的符号代表香蕉，有的符号代表吃，然后猩猩把这几个符号都学会了。


猩猩要表达一个意思 「我要吃香蕉」，猩猩就把这几个符号堆在一起给管理员看，「我要吃香蕉」，就这意思。但是猩猩搞不清楚主谓宾的区别，猩猩会把这几个符号之间的次序任意排列，第一次次序是什么？「我要吃香蕉」，第二次次序是「香蕉要吃我」，第三次是「我吃香蕉要」，第四次「要吃我香蕉」，次序都不一样。大概猩猩的脑子里就以为所以这些东西摆在一起就行了，至于哪个先放哪个后放这很重要吗？它说不重要。


我们人类一看，这猩猩完全不懂语法结构里面的主宾关系，你这样随便放每个意思都不一样，所以我们人类就知道这个区分是很重要的。乔姆斯基就说，小朋友他的心智天然的要比猩猩厉害，天然的就已经知道主谓宾的区别了，否则你怎么能够解释小朋友不会像猩猩那样犯愚蠢的错误，很快就能够做出对于成年人的语言行为的正确的模仿呢？他心里已经有这个种子了。


但是为什么还需要学习，你这样说每个人都无师自通了？那是因为他需要知道，他心中先天的语法规则和人类经验的语法规则之间是怎么对应的。


比如举个例子，我们的语言都有主谓宾的区别，但主谓宾怎么放这件事就比较复杂了，因为不同的语言放法是不一样的。比如说「我要吃香蕉」，这是中文的讲法，如果日本人来讲的话，日本人的语法次序是什么？是「我香蕉吃要」。那「我不要吃香蕉」，我们中国人是这么说的，日本人还要麻烦，把「不」放在最后，于是就是「我香蕉吃要否」，就是这个次序。


但这并不意味着日本人不知道主谓宾，或他们把主谓宾颠倒了，请注意，日本人没有把主谓宾颠倒，他们是把主谓宾的什么东西颠倒了？空间位置颠倒了，主谓宾本身扮演的角色它没有颠倒。这就类比说，有个导演就喜欢让与女二号先出场，女二号是一个引子她先出场，女一号再慢慢出来，在片子 15 分钟以后再出来，也可以这么拍。


一个小朋友，他如果生活在中国人的家庭，就知道动词后面是跟着宾语的，他生在日本的家庭就倒过来了，就要把宾语放到动词的前面，必须要这样放。因为小朋友自己是不知道他会投胎在哪个家庭环境的，所以他在 0 到 3 岁包括以后的语言学习过程，主要就是让他先天的语法和经验的自然语言产生一个比较好的相互沟通。他的先天语法告诉他，肯定已经有宾语了，肯定已经有主语了，但是他不知道主语、宾语到底该怎么放，于是他跟大人来学，所以他在学的时候脑子里实际上已经有一套关于先天语法的观念了，因此他要学的仅仅是经验语法当中具体是怎么放的。


这种观点听上去是颇有说明力的，乔姆斯基就问了：如果不是这样来解释的话，你怎么解释小朋友能那么快学会自己的母语呢？而且如果不这么解释的话，你怎么来解释另外一个现象，即人类之间的语言都是可以互相通译的？这是因为人和人之间有一个基本的共通的地方。


**人的脑子里真的有一套先验的语法吗？**


我认为乔姆斯基的理论是有一定合理之处的，但是我仍然认为他的理论有毛病。我为什么对乔姆斯基的理论有部分的同情，但还是抱有一个很大的怀疑？我部分的同情是我觉得乔姆斯基的确触到了问题的要害，看来我们语言的学习本身是需要一定的先天基础的，你让人类去学一种语言，不管怎么说都学得会，你让一只猴子学会英语、法语、德语、日语，这是完全不可能的事情，这说明的确我们在出生之前已经有这样先天的基础了，这一切都没有问题。


但是我对乔姆斯基比较怀疑的地方是什么，就是人的脑子里真的有一套已经做好的先验的语法吗？


按照乔姆斯基的观点，这个语法已经做死了，只不过这个语法哪个地方开哪个地方关，是要再通过后天的学习以后你再决定，所以他给我们的一个比喻是什么？有一根很长的水管连到整个小区里面了，小区里面就伸出了很多的水龙头，哪个水龙头开是取决于你这房子是不是卖出去了。比如东边那一家真的是卖出去了，水龙头就开了，水表就开始走起来了；但是如果没有卖出去或者卖出去了以后没人来住，水龙头就一直关着。但是，一个负责任的小区物业管理人员，你得保证所有设施都能够处在随时可用的状态，不能说这房子现在还没人住，我们就把这里的水管全破坏掉了，不让水龙头可用，不能这么干，弄不好明天人家就卖出去了，随时有人会拎包入住。


那么，这个比喻和语言有什么关系？就是按照乔布斯基的观点，我们的语言在先天语法的层面上对于各种语法成分的接收能力已经事先被做好了，至于这些能力是不是会被现实化，则要看机缘。类似于我这小区的水管的配套系统是否会被激活给业主所使用，取决于一些后天的机缘。


我个人对这个比喻是有点小担心的，我的担心是什么？我担心的是，我们人类的语言的复杂程度和它的多样性。有些语言语法就是简单，有些语言的语法比较难，咱们汉语的语法相对来说是比较简单的，但我们汉语别的地方很难，外国人在学汉语的时候就会和中国朋友说，你们的语法很容易搞定，你们的汉字很难搞定，两三千个常用汉字，我们记都记不下来。所以我们语言的复杂程度可能就往那个方向发展了，比如往汉字的方向发展了，并不是往语法的方向发展的。


当然，西方的像德语、英语、法语、俄语，复杂程度主要体现在它的语法的构成上，它们的写法是不难的，因为基本上是字母文字拼来拼去。当然，也有一些民族的语言兼具了这两种复杂性，典型的是日语，既包含了大量的汉字，所以汉字的麻烦它都有，同时它的平假名、片假名又是字母文字，所以字母文字的麻烦也都有。


所以，不同的语言有不同的难学的方向，这个方向这里来一下，那里来一下，就像不小心倒在纸头上的墨汁一样，它往哪个方向流好像都是随机的。在这样的情况下，难道任何一个小朋友心里面的语法装置的复杂程度，就已经能够把天下这么多复杂语言往不同方向伸展的那些可能性都预先做好了吗？大家是不是觉得这种理论有点小小的过分？好像有点把我们小时候的智力做了一个过分的夸张。


所以我倾向于认为，所有的语法复杂性背后的基本的真相，可能是数学递归性的复杂性。「数学递归性」这个词听上去稍微比较专业一点，什么叫递归结构？就是套嵌结构，就是我们小时候说的那个故事：山里有个老和尚，老和尚说个故事，故事说山里有个老和尚，老和尚说个故事……就这一句句套进去，这就叫递归。


就是说你能够想到的递归结构有多复杂，实际上就决定了你构造出来的语法结构是有多复杂的，也许咱们人类神经的结构已经复杂到能够搞出比较复杂的递归结构了，因此也就使得我们的语法由此诞生了，就是这么一回事。也许这样的解解释要比乔姆斯基的解释更加精简一点。至于递归结构是怎么样转化为一个具体的语言结构的，我个人认为这是一个经验层面上的问题，这就不是先天层面上的问题了。


还有一个我对于乔姆斯基理论的吐槽，就是他对于语法的问题关注得过多了，语言的问题不仅有语法，还有语义。语义是什么？就是维特根斯坦所说的语词之间的意义搭配问题，有些搭配就是对劲的，有些搭配就是不对劲的，尽管不对劲的搭配语法上好像也对，这个问题就很奥妙了，但乔姆斯基没办法处理这个问题，所以我认为他的语言学有很多的盲点。


正因为如此，我认为要在乔姆斯基之外再引入别的语言学理论和他对抗，这可能就是我以后的节目所要谈到的内容了。


**如何用得上？**


今天这一期节目所能够带给大家生活中的启示是什么，我相信大家已经听明白了，就是 0 到 3 岁的确是学习语言的关键时刻，如果 0 到 3 岁时小朋友的语言心智没有被很好开启的话，可能会对他以后的一生产生不太好的影响。现在的小朋友是生活在一个相对来说技术比较发达的时候，有很多智能的语音音箱、点读笔等设备，这些设备能够用相对规范的语言让小孩接受很多的语言信息，这对于培养婴儿正确的语法感觉应该是很有帮助的。
