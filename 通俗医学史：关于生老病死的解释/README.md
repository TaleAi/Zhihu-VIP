## 简介
每个人都会碰上生老病死，医学与我们每个人息息相关，是生存还是死亡，这的确是个问题。如何解题呢？听听科普作家吴京平怎么说。


当今人类，正处在有史以来最好的时代——健康和长寿成为一种常态，人类的平均寿命比百年前大幅增加了一倍多。现代医学在其中所起到的关键性作用，毋庸置疑。


可是，当我们对于抽血化验、B 超 CT、打针吃药、外科手术等等这些都早已习以为常的时候，大多数人可能都已经遗忘了人类曾经在病魔面前是多么的渺小而脆弱。


2003 年的一场 SARS 疫情令所有人至今心有余悸，而其实它的死亡率连古代大瘟疫的万分之一都不到。


医学为全体人类做出了居功至伟的巨大贡献，而很少有人记得，现代医学的发展和科学的进步一样，充满曲折和艰辛。


本专栏希望建立一条路径，让你像学习和研究历史一样，去全面认知医学发展的前世今生，去了解现代医学到底从何而来。


**你会听到：**


* 东、西方医学发展的共性，以及两者的分野
* 医学发展所经历的种种磨难，和它所攻克的许多重要难关
* 那些为推动医学发展做出卓越贡献的人和他们的故事

在这场人类对抗自身命运的伟大博弈中，你会发现，这是我们认识人类发展的另外一个不可或缺的角度。


**吴京平**老师将用他最擅长的评书风为你开讲，通俗而诙谐的语言令你对节目内容的理解毫无压力。


本专栏不会出现超出常人认知范围的医学专业知识，请安心「食用」。
