## 简介
这是一个双向治愈的言情短篇合集专栏。


在不可逆的大背景下，不论帝王将相，还是身处钢筋水泥都市中的普通人……或多或少都会受过各式各样的创伤。


不论是身体上还是心灵上，都存在着不同的情感障碍，也在命运洪流中不断追寻唯一灵魂契合之人。


而她们，是人生各有缺憾的不完美女主，与男主一朝相遇，纵使前路荆棘遍布，也要化解磨难、相依相偎。


当有缺憾的灵魂在某个契机下相遇，通过不断碰撞，孤寂的魂灵学会相互取暖，


各种相识于微末的特殊恋人，便成就一场绝美恋爱——


所以那就，一起穿越荆棘走花路吧。


「梁宫风月」： 


她是冷血无情的迟钝女杀手钟绾。


被沈小王爷用了三年时间，包装成一个贤淑美人送入宫内。


宫墙之内，邂逅了美人皮相、修罗手段的小陛下——虞子束。


她与他的白月光，何其相似。


权力交锋，面对爱情的抉择，身为棋子的她如何斡旋其中。


「玫瑰失梦者」：


职场小白姜晓晓，醉酒强吻的外卖小哥，竟是自己的新搭档。


为了栏目，姜晓晓曲线救国，决意先攻克高冷男神徐溪川。


却意外发现了男神背后的秘密……


她几乎落荒而逃。


然而教堂前，男神漆黑的长睫下雾气丛生，「姜晓晓，不如我们试一试？」
