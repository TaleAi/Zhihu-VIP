## 简介
系统呈现完整的西方哲学史，一听就懂的极简哲学史。


柏拉图的性爱论是什么？理想国真的理想吗？耶稣基督到底是谁？书杰，用最通俗的语言呈现最系统的西方哲学史。深入浅出，娓娓道来。你平日里听到的零散的知识将得到系统整理。


100 个问题 50 位哲学家，系统呈现完整的西方哲学史，探寻几千年来人类的智慧奥秘。苏格拉底、柏拉图、亚里士多德、笛卡尔、斯宾诺莎、莱布尼茨、洛克、贝克莱、休谟、康德、黑格尔等。 


  




**【适合谁听】**


1.哲学初学者：零基础，没有系统训练过，这套课程帮你快速入门；

3.企业管理者：商业的终极指向便是哲学层面；


4.学生：对世界充满好奇，想开启智慧。对于考研者，这套课程有助于你清晰梳理哲学史；


5.爱智慧的人：想成为一个有思想、有趣味的人；

7.胎教和育儿：文化熏陶从小做起；


8.失眠者：书杰的声音如涓涓细流，伴你入眠。


 


**【你将获得】**


1.系统的知识：帮你从 0 基础梳理西方哲学的大框架；


2.清晰的逻辑：梳理哲学史的脉络，训练逻辑思维；


3.眼界与境界：知晓人类最伟大的哲学家和哲学思想，提升境界；


4.安宁：为浮躁的生活，找寻一份安宁；
