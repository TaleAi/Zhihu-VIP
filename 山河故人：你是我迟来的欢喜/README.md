**《胡不归》**


她是古府千金古奈奈，亦是前世漠北公主。


昔日恋人早认不出面目全非的她。


这一世换她主动好了，她制造偶遇，表明心迹，捧上自己一颗真心，可都被他踩进泥泞中。


她才知道，她捂不热他的心。


她放弃了，只要他安好。


她离开后，那位权高位重的首辅千里奔赴，红了眼，「古奈奈，我来迟了，你可还愿和我走？」


  



**《我有读心术》**


结婚前一天，叶随随有了读心术。


未婚夫人面兽心，居然想抢家产，夺兵权，还要弄死她？ 


这婚结不得。


叶随随决定除掉这个祸害，顺便嫁给救过自己的萧世子。


借助读心术的神奇，叶随随假意和前未婚夫交好，套取有用消息，没想到被人撞见。


这位世子爷怎么又吃醋了？


「好哥哥，你听我狡辩……啊不，听我解释。」
