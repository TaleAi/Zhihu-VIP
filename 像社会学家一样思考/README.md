## 简介
「996」、双 11、网络谣言、饭圈经济，这些你熟悉的生活场景，背后潜藏着哪些社会问题？社会学是认识世界、思考问题的工具，它帮助你看清社会运作和发展的基本规律，反思习以为常的生活状态。马克思、涂尔干、韦伯、齐美尔……以沉淀百年的经典社会学理论作地基，从「秩序」与「人性」的核心出发，清华大学社会学教授带你重新审视我们熟悉的世界。
