## 50.维生素 C 能美白去痘印吗?

> **维生素 C 能美白去痘印吗?**「莫叔，维生素 C 是不是能美白呢?」**坏血病**


1497 年 7 月 9 日到 1498 年 5 月 30 日，葡萄牙航海家达伽马绕过非洲到达印度，他的 160 个船员中，有 100 多人死于坏血病。


奇怪，人类历史已经有几百万年了，有文字记载的历史也有几千年了，为什么几百年前才知道坏血病呢?


原来，最早的坏血病就是出现在几百年前远航的水手中，因为缺乏新鲜的水果蔬菜，从而导致维生素 C 的严重缺乏。


**维生素 C**


维生素 C 对人体是如此重要，来看看详细介绍:


1)维生素 C(英语:Vitamin C，又称「L- 抗坏血酸」)是高等灵长类动物与其他少数生物的必需营养素;


2)抗坏血酸在大多数生物体内可借助新陈代谢制造出来，但是人类是最显著的例外;


3)缺乏维生素 C 会造成坏血病;


4)在生物体内，维生素 C 是一种抗氧化剂，保护身体免于自由基的威胁，维生素 C 同时也是一种辅酶。


人类这么高度发达的生物体居然不能自己制造维生素 C，简直太遗憾了!你身旁的阿猫阿狗都能自己合成维生素 C，也就是说，它们不会得坏血病，可以不需要吃新鲜的水果蔬菜。


不过，换个角度来看，也许是我们的幸福，我们可以吃这么多美味的瓜果蔬菜......


那么，怎么知道我们的身体是不是缺维 C 呢?


坏血病开始症状是四肢无力、精神消退、烦躁不安，做任何工作都易疲惫，皮肤易红肿。病人会觉得肌肉疼痛、精神抑郁。然后脸部肿胀，牙龈出血，牙齿脱落，伴随着口臭。皮肤下大片出血(看来像是严重的打伤)。最后是严重疲惫，腹泻呼 吸困难，肝肾衰竭导致死亡。


皮肤易红肿、肌肉疼痛、牙龈出血、牙齿脱落......这些都是严重缺乏的症状，只要你正常饮食，是绝对不会出现这些症状的。因为，你每天的食物中多少会有一 些维生素 C。


**来源**


动物为来源的食物中维生素 C 的含量较少，看下面的数据:


1)鸡蛋里不含维生素 C;


2)每 100g 牛乳中只含有 1.0 毫克;


3)每 100g 人类母乳中含有 5.0 毫克;


4) 每 100g 生鸡肝肉含有 17.9 毫克，煎炸后，含量减少到 2.7 毫克。


经过煎、炒、烧烤、炖、煮之后，动物来源的维 C 很多被破坏了，含量急剧下降，我们对维 C 的摄入主要是来自于新鲜瓜果蔬菜。

对于女孩子来说，每天 75 毫克的维 C 就够了。也就是说，你只要吃 150 克橙子，如果橙子够大的话，一个就够了。


明白了吧，只要你正常饮食，瓜果蔬菜不缺，维 C 也是不缺的。如果你还额外补充维 C，那么要当心了，摄入过量会出现中毒反应的，每天最高摄入不要超过 2000 毫克。


那么，如果多补充一些维 C，会不会有传说中的美白、去痘印、抗氧化的功能呢?**美白去痘印?**


大猩猩是我们人类的近亲，也不能自己制造维 C，需要从饮食中摄入。大猩猩主要的食物是瓜果植物，每天食量惊人，摄入的维生素 C 也是相当的多，是我们维 C 摄入量的 N 倍。


如果维 C 有美白皮肤的作用，那么大猩猩不应该是黑色的，你见过有肤色较白的猩猩吗?显然没有。


你可能要说了，动物和人肯定还是有区别的，好，那么我们来找找是否有维 C 美白的证据。


在百科词条上，并没有任何文字提到维生素 C 对皮肤有美白去痘印的作用。


在网上搜索，也并没有找到直接明显的证据说维生素 C 能美白去痘印。 继续搜索，看到了一篇文章，原文很长，大概意思是:维生素 C 在表皮和真皮中都有。


表皮中的维 C 含量更高一些。紫外线下，维 C 在表皮和真皮中的含量都下 降了。口服维生素 C 能增加皮肤中维 C 的含量，但是血液中维生素 C 达到饱和后，皮肤中的维生素 C 就不再增加了。


Vitamin C is a normal skin constituent that is found at high levels in both the dermis and epidermis...（资料来源:http://lpi.oregonstate.edu/mic/health-disease/skin-health/vitamin-C）


根据这段文字，大家是不是能想到些什么?为什么在我们皮肤中有一定含量的维 C，没有发挥出抑制黑色素增加的功效，从而自动地给我们美白呢?


道理很简单，你皮肤里的黑色素是有大用处的，就是为了吸收进入皮肤的紫外线，来保护深层的细胞组织。皮肤里的维 C 发挥了别的效用。


那么，是不是又想到了些什么?如果你强行从皮肤外面往皮肤深层递送维 C， 就能达到美白效果吗?


答案是否定的，因为，皮肤里的维 C 不是用来对付黑色素的!


这是维 C 不能美白的最主要原因。


再看另一则资料，原文大概意思是:角质层会阻挡维生素 C 对皮肤的渗透，动物实验表明，经皮肤维生素 C 的吸收受 pH 值的影响，20% 浓度的维生素 C 已经被用于透皮吸收研究(吸收很少)。维生素 C 的稳定性很差，很容易受到光、热的影响而降解。


而另一种维生素 C 的衍生物抗坏血酸磷酸盐就要稳定得多，但是透过皮肤吸收得更少了。在皮肤上使用 0.6%~10% 的维生素 C 目前认为是安全的，没有不 利影响的报告。


也就是说，涂在皮肤上的维生素 C，经皮肤吸收的很少;而如果加大维生素 C 的浓度，就有可能对皮肤产生刺激。这也是维 C 很难发挥美白的另一个原因。


  

**护肤品里的维 C**


看到这里，可能有些人还是充满了疑惑，为什么有人使用含维 C 的护肤品后，有一点美白的作用呢?


我们可以简单做个推理，如果维 C 有明确的美白作用，那么医院和药店里出售 的维 C 药物会增加一个功能:美白。


而如果很多人吃了这种宣称有「美白」作用的维 C 药物，发现不能美白，会发生什么?很显然，很多人会愤怒，会去投诉，去告这个药厂，拿没有美白作用的药来愚弄大众。


所以说你明白了吧，为什么作为药物出售的维 C 没有标明有美白作用?就是因为维 C 真的没有明确的美白作用。


作为护肤品来说，美白作用是商家宣传出来的，如果你用了有一点美白作用，是你的运气。如果没有美白作用，你也不会去投诉这个厂家。其实你心里也只是有个期望而已，你也知道美白很难的。
