## 简介
说好的我是女配，怎么男主、男二、男三，一个个都要来宠我？


家境优渥的白雅南因病早亡后，穿越到了一个陌生的世界。


然而这个世界一开始，便是 hard 模式。父母虐待、叔叔变态，没钱还没爱，好在凭借前世父母的教导与见闻，白雅南的日子好不容易改善了些。


没想到，一朝风云变，她成了被抱错的豪门苦命大小姐。


也因为这个过于狗血的情节，她发现自己其实是穿越进了一本言情小说，而她就是其中下场凄惨的恶毒女配……


牢记女配不能跟女主斗的终极原则，一心咸鱼的她为了躲避纷扰而选择出国留学，奈何男主、男二、男三，这些原本剧情里应该避她唯恐不及的人，竟然纷纷要来呵护她，更有人布了极大的局，只想把她拴在身旁。


再次回国时，那些往事与年少时的悸动，又纷至沓来。


她会抛弃心中的疑虑，奔向一直凝视着她的他吗？
