## 简介
写给那些多少有点小缺陷的女孩们——你很可爱，你值得被爱。



一、朋友之名


林俏暗恋了许嘉衡八年，她以朋友的名义陪伴对方。


还不等她勇敢地迈出那一步，许嘉衡却忽然说自己有了喜欢的人。


林俏偷偷哭了一夜，第二天神色如常地帮许嘉衡制定追求攻略。


餐厅，游乐园，密室逃脱……她甚至贴心到陪着许嘉衡把攻略模拟了一遍。


许嘉衡面上笑着道谢，林俏心里哭着道别。


她一个人买了很多酒，喝到一半却忽然接到了许嘉衡的电话。


「明天出来吃饭。」


林俏心里正难受，有点负气道：「你明天不去执行攻略？」


许嘉衡声音有点委屈，「你不来我怎么执行？」


 


二、我爱吃肉


国庆回家，帮妈妈跑腿的胡佳佳对卖肉的小哥陈近一见钟情。


「你好，要点什么？」


胡佳佳抿了抿嘴唇，心里默默接了句「要你行吗」？


从这以后，胡佳佳每天往超市跑三趟。


「你每天买这么多肉，吃得了吗？」


向来不哎吃肉的胡佳佳昧着良心，「我爱吃肉，一天三顿一顿都不能少。」


「吃太多也不好，」陈近撑着柜台微微歪头。


「有什么不好？」


「吃腻了，以后你不来买肉了怎么办？」


 


三、同桌别来无恙


席佳上学的时候就知道周宜也难搞，没想到出国喝了几年洋墨水更难搞了。


方案不通过，加班盯着你改。


外出见客户，非要跟去监督。


前男友纠缠，指责影响工作。


家里打电话催婚，这你也要管？


周宜也表示可以管，遂打扮地人模人样与席佳一同回家，假装席佳男友。


谈话间周宜也说自己从高中就喜欢席佳，暗恋多年不敢开口，痴情模样另席佳父母侧目，肉麻的表达令席佳心惊。


会面结束，席佳为周宜也的表现大加赞赏，周宜也认真地看着席佳，「本色出演，情之所至罢了。」


席佳落荒而逃。


第二天席佳的工作汇报里加了一张纸条，「入戏太深，不如假戏真做。」
