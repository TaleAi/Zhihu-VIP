《原来他也喜欢我》

舒雪答应为赵扬帆补课，却不慎被他发现偷看腹肌男视频。眼看在男神心中形象越来越变态，舒雪干脆躺平摆烂。她仰慕过赵扬帆这件事，是一个秘密，关乎她青春时代的自卑与伤感。


就在赵扬帆即将成为她的意难平时，赵扬帆突然主动站在了她面前。原来，赵扬帆心中也藏着一个关于舒雪的秘密。
