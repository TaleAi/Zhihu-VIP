## 简介
想和有钱人谈恋爱吗？夜幕降临，不甘寂寞的猎食者出洞了。


***跨越阶级的逆袭爽文：《穷鬼女孩致富路》***


爸爸坐牢，妈妈改嫁，她成了可怜的小白菜，难道要穷一辈子吗？毕业后，她开启打怪升级，搞定难缠婆婆，嫁给了有钱的拆迁户，你以为这就结束了？别人有钱不如自己有钱，她再一次花心思搞起了事业……


十年过去，她成了首屈一指的有钱人，昔日唾弃她的人，现在，谁不是跪在身边，摇尾乞食。


***好想嫁个有钱人：《富二代选妃记》***


普通女孩嫁入豪门有多难？请先过了未来婆婆这一关。


小镇青年李雪，为了嫁富二代同窗，几经谋划过招，败下阵来，不能直捣黄龙？曲线救国也不错，等我生下孩子，看你还能奈我何，还不是得乖乖八抬大轿抬我进门……


正在她得意时，意外发生了。


***男孩子在外面保护好自己：《美女老总的诱惑》***


她事业成功，身材凹凸，妖娆多姿，老公高岩却让自己提不起兴趣，于是她把眼光放到了外面，和弟弟谈恋爱简直不要太美妙，上进、积极、身体好~


公开出轨？自己在家里当牛做马，哪里受得了这么大的绿帽，高岩忍不住出手了……


**本专栏，揭露富人爱情水面下的套路与算计，洞悉成年人情爱世界的残酷、规则、定律。**


**千万不要陷入、迷失在物欲洪流里奥。**
