# 所爱隔山海：娘子！你怎么这么能跑！

## 简介
家国之下，坚持所爱有太多艰难，但他们从未放弃。


1、《皆可弃》：


两年前，我是皇上最疼爱的小郡主，整日吵着要嫁给他，最爱看他无奈却宠溺着我的神态。


两年后，我是被送来和亲的六王妃，六王子为我放弃王位，嬉皮笑脸地同我斗嘴，叫我身处异乡，却不寂寞。


他们都那样好，让我以为自己真是什么幸运的人。


可惜，我错了。


  



2、《芙蓉帐暖》 ：


她带着恶毒庶女的配置意欲为祸人间一场，不讲道德不讲伦理，总是说着最疯的话做着最傻的事，可是，最后走的时候，却把所有人的心都偷走了。


  



3、《穿进色气修仙文》：


上课看小说被抓之后，我穿进了手头的色气修仙文里。


在这里，男主温文尔雅，男二阳光可爱，看到我，他们都顾不上女主了……


纷纷冲过来，


急着要帮我补课！


救命！我不想在这里学习啊！


  



4、《冷宫三年》：


恶女后宫生存图鉴。


我捧高踩低，人情练达，踩着所有爱我者的心血往上走，只为了完成一件事。


是的，我对不起你们，我有愧，但无悔。


  



5、《绿帽打工人》：


公主爱上太傅，太傅是我老公。


他俩两情相悦，我反而成了第三者，在一场绑架中意外去世。


醒来之后，我成了一名光荣的打工人。


嗯，这次，轮到你追我了。


  



6、《徐公甚美》：


我是京城中以能打著称的一朵霸王花。


你是刚回来貌美心狠演技高的小白脸。


你跟狐朋狗友说，我能泡到她，你们信不信？


现在我要问问你：我能打死你，你，信不信？
