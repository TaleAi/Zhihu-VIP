## 简介
一段在犀牛角尖舞蹈的青春


提到南非，不免会让人想到「动物世界」中的野生动物。当地政府为了确保动物种群数量的自我平衡，建立了许多大型保护区，大鱼河自然保护区就是其中之一。


2017 年，一则关于南非大鱼河自然保护区的新闻登上了中国的热搜：一头珍稀的黑犀牛乘着直升机搬离丛林前往新家，这只重约 998 公斤的犀牛被蒙起眼睛，四肢绑在一起用直升机吊起。 让人不免在心疼「牛哥」的同时，也开始好奇这片保护区里「脑洞极大」的研究者们究竟是什么样的人，他们每天都做着哪些工作呢。


作为一名极地生态学研究者，知友「Orca 是只鲸」在荷兰留学期间，就在南非大鱼河自然保护区生活过数月。在这和动物们相伴的 100 多个日夜，他和搭档尤伯特经历了一次次惊心动魄的冒险：差点被「杀手」黑犀牛「追尾」、每天与定时到访抢地盘的狒狒们斗智斗勇，当然还要与南非丛林中的莫名病痛抗争。在大鱼河自然保护区，和动物们相处的每一天都是充实而又令人着迷。


除了与动物为伴，他们还认识了在大鱼河自然保护区的人们：开朗乐观的管理员布拉德、唯一的玩具只是一把勺子的黑人小女孩、管理区的候选接班人来来图。「Orca 是只鲸」用他诙谐幽默的文字记录下了当地人民热情朴实而又真实一面，向你展现出一个你在纪录片里看不到的南非。


读完这个专栏，也许你会对这个早已熟悉的世界多一些认知。
