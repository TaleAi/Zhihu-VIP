## 简介
本课程是中国科学院动物研究所鸟类生态学博士亲自探访新几内亚岛后的分享，揭秘珍稀鸟类的生活习性、观鸟策略，将为鸟类爱好者和热爱生活、热爱美的人们带来一场完美的听觉盛宴。


你是否有这样的经历：


走在路上，你的目光会不自觉的追随天上的飞鸟而去，注视着它们在碧落长空划出美丽的云线，直到那小小的身影消失在天际。


2018 年的《泰晤士报》说出了人类喜爱观鸟的原因：观察鸟类，可以放松我们紧绷的神经，对现代社会的人类来说更是如此。而且，人类了解世界节律、理解生命张力，自古便从观鸟中来。


新几内亚岛是当之无愧的观鸟胜地，当地鸟类多达 600 余种，其中不乏集珍贵与观赏性于一体的鸟类品种。几个世纪以来，新几内亚岛吸引着无数科学家来此探索。19 世纪，英国鸟类学家、鸟类科学绘画艺术家约翰•古尔德就前往那里，精心绘制了 3000 幅鸟类科学绘画，并挑选了 300 余幅作品放进了自己的《天堂飞鸟古尔德手绘鸟类图谱》中。


中国科学院动物研究所鸟类生态学博士朱磊在亲自探访了新几内亚岛之后，将为我们揭秘珍稀鸟类的生活习性、观鸟策略，并带来数幅古尔德手绘鸟类图谱高清大图，以及珍贵鸟类的野外录音。
