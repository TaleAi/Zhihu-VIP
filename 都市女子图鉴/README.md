## 简介
都市繁华、欲望膨胀，身处其中的女子也难免经历各色诱惑、各种磨难，如何出淤泥而不染、脱颖而出，且看女子们的百般计策。


班主任在班会上说，「社会是个大染缸，女孩子要洁身自好……」


同学们齐刷刷回头看了我一眼。


「经得起诱惑，架得住勾引，切记不可贪图便宜走捷径……」


同学们回头又看了我一眼。


——《金主太正经》


  



我是个野模，最初做这行，只是受邀走一场内衣秀。


演出按场数计费，一场有个万把块，我忍不住心动了。


走了一场，就会有第二场，第三场......


可自那天老板要求我穿着透明内衣走压台时，我的生活一切都乱了。


——《野玫瑰》
