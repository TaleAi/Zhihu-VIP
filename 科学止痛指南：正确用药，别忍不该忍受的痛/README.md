## 简介
让疼痛远离生活，让科学战胜疼痛。


你是否经历过突然的头痛、肌肉酸痛或关节痛，「痛得倒不至于去医院，但也不知道如何缓解」的经历？


你是否偶尔痛经、宿醉头痛，「知道这不是什么毛病，所以忍忍就算了」？


你是否还有亲属或者朋友因为癌症而被迫忍受剧烈疼痛，想帮忙，却不知如何是好？


如果是这样，那就来听听专业的麻醉科医生的建议，帮你科学使用镇痛药物，既不受罪，也不用担心大脑伤害。


本课程以通俗的语言结合专业的知识，介绍疼痛的基本内容，了解镇痛药物的使用建议，以及包括急性或慢性疼痛的专业知识。


课程分为六大模块，分别是：疼痛的本质和原因、炎性疼痛的克星——非甾体抗炎药介绍、常见疼痛及日常药物、镇痛药的副作用和成瘾性、癌性疼痛的原因及治疗、以及慢性疼痛与疼痛科简介。科普镇痛药物的「治标」和「治本」、了解常见疼痛的应对策略。更进一步的丰富疼痛的相关知识，轻松掌握应对疼痛的理论和方法。
