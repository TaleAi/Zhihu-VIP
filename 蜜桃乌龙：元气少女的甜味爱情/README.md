## 简介
全糖去冰甜宠文，和元气少女一起感受别样的乌龙爱情。


**- 和破产老板的合租日常** **-**


我老板，一个年轻帅气、英俊多金的优质单身男人。


而我，是他秘书。


是不是觉得我俩可能会发生点儿啥？


不瞒你说，入职时我也是这样想的。


然而，刚工作三个月，公司破产了。


我失业了，老板变成了穷光蛋。　


机缘巧合下，老板不得已与我合租。


 


**- 关于重名这件事** **-**


我被渣男骗了，还是用微信小号骗的我。


我咽不下这口气，决定也用小号去骗他大号。


他今年刚考上 P 大研究生，于是我混进新生群，成功加到群昵称本名为「钟博」的渣男。


我每天嘘寒问暖、投其所好、茶里茶气地与他保持联系。


就在我充分准备，打算让他在校园彻底社死时，他朋友圈发了张照片。


我看着照片里的陌生帅哥，一脸懵逼地问：这是你？


他：嗯。


我火速翻回新生群。


然后发现——


这个群里，有两个钟博。
