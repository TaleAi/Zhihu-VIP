# 执子之手 将子拖走

### 存档日期：2022-05-22

## 简介
她是穿越来的呆萌公主，他是权倾朝野的腹黑王爷。她深居后宫，装傻度日，他看破不说破，陪着她演了这许多年。乱世之下，只要他在，就能心安。执子之手，将子拖走，子说不走，很好，关门放狗。


我是个公主，而且是个呆公主。


生在皇宫，我的相貌就和地里那萝卜白菜那样普通；一年四季里，我的表情总是呆滞，眼睛里老是含着迷茫，可其实，我知道这个宫里的所有秘密，


包括她的，他的，还有他们的。


我就这么悠哉惬意地在宫里存活着，我坚信我会一直这么傻下去，直到，我拿到了宫中某个被掩盖已久的藏宝图，从此被迫卷入三国纷争。


面对心怀不轨的人，在这乱世下自保尚且艰难，奈何身边还有一个最腹黑的王爷表哥。


王爷曰：执子之手，将子拖走。子说不走，很好，关门放狗。
