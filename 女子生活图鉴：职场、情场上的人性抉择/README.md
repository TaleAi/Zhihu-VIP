都市女子，为了生活打拼，希望从职场中获得成就，从情场中得到抚慰，但往往事与愿违。


都市女子，或野心勃勃，奋发图强；或虚荣势力，自甘堕落；或为爱痴狂，或单身快乐……不同抉择下的百态人生。
