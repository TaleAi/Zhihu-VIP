为了摆脱旧爱，她临时找了个新欢演戏，演着演着，新欢就变成了新婚。


世间最厉害的小偷是谁？她偷了金銮殿上，九五之尊的心。


……


她们的生活始终熠熠生辉，却不只有爱情。


爱很好，但我更好。
