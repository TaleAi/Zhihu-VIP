## 简介
美少女巧对生活难题，大聪明力争永不受气


我爸兄弟三人，他排行老二，只有我爸生的是女儿，所以我和我妈在这个重男轻女的爷爷奶奶家，时常被区别对待。


但我爸也是最有钱的，所以其他亲戚一句废话，也不敢在我们面前说。


直到我爸去世，这个平衡就打破了。


是个瘪三，就敢来我家蹬鼻子上脸。


——《给你脸了》


  



合租的女室友说她要回老家一趟，大概要走一个礼拜。


可是在她离开后的第二天，纵享独居生活的我突然发现……


出租屋里还有另一个人！


——《一次合租》


  



姐姐发现姐夫出轨，哭哭啼啼。


姐夫说姐姐晦气，怒极的时候还死死掐着姐姐的脖子，说就是因为姐姐，让他错过了最好的时机，所以投资失败了。


姐姐说，我也不知道怎么办，不行那个钱就不要了，转头对妈妈说，我没用，我现在只想离婚


我翻白眼，好家伙，十万，说不要就不要？


那必须不行。


——《姐夫在吗》
