## 简介
本系列是对演化生物学中的性择理论的介绍和演绎，并将这些理论应用到现实生活和文学作品中所获得的一些思考，文章诙谐幽默，让读者能够从另一个生物学的角度来看待原本熟悉的事物。



人是如何选择配偶的？


是天生的吸引？是生理的需要？是自然的选择？


为什么男人会选择貌美的女人？怎么样的外表才证明拥有好基因？


男朋友的基因掺了水要怎么办？


白雪公主为什么不嫁给小矮人？


**从根本上来说，自然科学和人文科学所表达的是完全不同的。它们所用的术语和方法是不同的，但同时这两者又是互为补充的，而且都是人类大脑的产物。如果我们能将自然科学的探索和分析与人文科学内省的创造力相结合，那么人类的存在将会具有无限的生产力，也会拥有更加耐人寻味的意义。**


本专栏将运用演化生物学中的性择理论，来讲讲关于人类的择偶那些有趣的事情。
