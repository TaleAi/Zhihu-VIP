## 简介
《西游记》陪伴了我们整个青春，承载了我们最深的记忆和情怀。西游记为什么好看，因为书里的妖怪神仙都颠覆了前有模式，他们会跌倒，会肚子痛，会哭，会过生日，一切都显得那么人情化，市井化。


一自开天辟地，两议便有吾身，曾教三界费精神，四方神道怕，五岳鬼兵嗔，六合乾坤混扰，七冥北斗难分，八方世界有谁尊，九天难捕我，十万总魔君。


你所知道的西游故事，在整个西游记体系里只占很小一部分，而西游记本身，是从中华丰富的神怪大世界中生长出来的，让我们一起走入这个更广阔而又神秘的世界，去了解神仙妖怪不为人知的另一面吧。
