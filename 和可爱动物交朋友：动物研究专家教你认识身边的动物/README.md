## 简介
本专栏将丰富多样的动物科普知识以讲故事的方式，用趣味的语言表达，辅以小朋友易于接受的童趣配音，呈现给孩子们一部可爱的儿童版动物世界。动物研究专家从详实丰富的科普知识，到趣味盎然的互动环节，让小朋友在欢乐中启蒙，在学习中成长。


认识小动物，一定要走进大森林吗？


或许，我们都忽略了身边经常能遇到的小动物。


爱偷吃灯油和奶酪的小老鼠、爱出松果的小松鼠、农民伯伯的好帮手老黄牛、时常搬家旅行的小蜗牛......


动物们作为自然界中的鲜活生命，与人类共同享用着地球资源。它们也和人类一样，有着自己的生存策略，有着自己的喜好。探索动物世界，让我们对生命有更加深刻的认识。


本专栏满足小朋友们的好奇心和求知欲，使小朋友对自然有更全面深刻的认识，感受到人类和动物和谐共处的意义。
