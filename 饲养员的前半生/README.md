## 简介
我曾经是名饲养员，这是我的故事，有人有动物。


**《秃头黑熊》**


一只来自马戏团的黑熊，它刚来动物园的时候浑身是伤。


为了让它好起来，我自掏腰包买了各种肉。


可没想到的是，有人为了一己私欲，竟想「熊口夺食」。


  



**《猴王》**


很多人可能认为猴子这种动物，天生萌萌的，很可爱。  

实际上并不是，据我多年的经验来看，  

猴子完全复制了人类的阴暗面，却不用受法律惩治。


  



**《一只胖狼》**


被狼群孤立的胖狼，竟在角落里刨坑，  

这时我才意识到事情的严重性，  

它可能觉得自己要死了，  

我该怎么帮它呢？


......


这个专栏里有温情感人的拯救，也有残忍心寒的压迫。


看似是动物们的故事，实际也是人性的故事。
