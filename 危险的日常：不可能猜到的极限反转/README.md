## 简介
Ta 的生活一如往常，没有变化，直到有一天，ta 一个寻常的动作却招来了「杀身大祸」……


和你我一样，ta 也是这个城市里一个普通的个体，工作学习，上班下班。但是，就在某一天，ta 却在平凡的日常里，因为一个再寻常不过的举动，落入万分危险的境地。Ta 的人生，也因此走向了截然不同的生活轨道……


《断指》|  一场外卖引发的血案


他和往常一样在网络平台叫了一份外卖，常点的店铺，喜欢的套餐，一切都没什么不同，但可怕的是这次他在饭里吃出了半根手指。警察的嗅觉让他意识到这截断指背后隐藏着一个案件，但每次他追踪过去，凶手就好像提前预知了一样逃避于无形之中……这个人到底是谁？他犯下了什么案子？为什么他要把断指放到外卖里送到警局？


《活体解剖》| 一个帖子带来的迷案


他打开贴吧，在一个恶作剧的帖子里搞笑般附上了自己的联系方式。没想到却从此被一个身份不明的人骚扰，对方执着地要求他解剖自己。他没有把这个恶作剧放在心上，对方却变本加厉，甚至出言威胁，他想找出这个人，却发现自己的行动仿佛都被对方掌控……他能拨开迷雾，找到这个隐藏在背后的始作俑者吗？


日常生活中到底暗藏多少杀机？这个专栏，一起来看他们在平凡的生活中经历的惊悚时刻。
