## 简介
本课程由动物学博士、《中国国家地理》杂志社旗下《博物》杂志编辑何长欢主讲，带你重看《神奇动物：格林德沃之罪》，寻找生活中的「神奇动物」。


上一部《神奇动物在哪里》引起了全球网友对电影中神奇动物的原型的持续不断的热议，而《神奇动物：格林德沃之罪》上映后，不论是善良可爱的护树罗锅、隐形兽，还是凶猛诡异的独角兽、马形水怪等，更是让影迷们好奇起来：


嗅嗅，看它的嘴还有它的小瓜子，到底是什么动物来着？


还有护树罗锅，害羞、粘人，有自己的世界的它，怎么那么像小版的格鲁特？


凶猛的独角兽，但总是哪里透着一股可爱？


隐形兽，它的脸，还有那一身飘逸的毛发，好似在《疯狂动物城》的某个柜台后面见过。


魔法世界中的神奇动物身上到底隐藏着什么秘密？我们在现实中又是否可以找到它们的身影？让我们和动物学博士一起走进纽特的皮箱，与这些神奇动物为伴，探寻它们的来历、现状、特点以及有趣的生活习惯。


皮箱已经打开，快来快来！
