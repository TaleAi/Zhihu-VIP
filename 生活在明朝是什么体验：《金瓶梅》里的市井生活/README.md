## 简介
本课程，主讲人占芳以史家的目光，从文化视角，对《金瓶梅》进行解读。美白是中国女人千年不变的追求？古人真的把银子放袖子里？中国人为什么不搞分餐制？主讲人将从《金瓶梅》中的服饰、饮食、习俗等方面入手，为你还原一个生动的明代市井生活。


《金瓶梅》号称「天下第一奇书」，这部奇书被人贴上「伟大文学作品」、「淫书」、「世相百科」等诸多标签。


400 年来，金学家在解读，文学家也在解读，林林总总，欲说还休。


金学家的解读，考训繁复而枯橾；文学家的解读，分析感性而飘忽。


本课程，主讲人占芳选择以史家的目光，文学的笔触，从文化视角，对《金瓶梅》进行全新的解读：


美白，是中国女人千年不变的追求？


银子那么沉，古人真像古装剧里那样放在袖子里？


如果穿越回明代，想穿貂皮大衣得多有钱？


生日到底从哪儿来，古人生日怎么过？


占芳为你解构「天下第一奇书」，从服饰、饮食、习俗等方面为你展现一个生动的明代市井生活。


欣赏古典名著就像爬山，在山顶一览众山小固然壮阔，但从山脚到山顶， 每一步的风景，一样别具风味。


本课程，就是想用不同的方式，打开新的一扇窗。
