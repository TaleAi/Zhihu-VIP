## 简介
本课程由资深撰稿人东郭的郭主讲，本课程不讲理论套话，只聊实际操作中需要注意的细节，教你如何通过需求清单、尽职调查以及聊天总体感觉这三重滤网，在相亲中锁定适合你的人。将相亲将变成你步入美好婚姻的最好途径之一。


觉得相亲很可怕？不知道为什么总是和一些奇怪的人见面相亲？


相亲的时候不知道聊些什么？相亲时候感觉不错，结果对方之后渺无音讯？


如果你有类似这样的问题，那么本课程将会很适合你。我们并不会让你成为相亲专家，我们只想让你掌握相亲的核心技能，通过相亲找到适合你的人，一起走入婚姻。 


在课程中，我们将教你如何通过需求清单、尽职调查以及聊天总体感觉这三重滤网，锁定适合你的人。同时我们将通过总共 27 个小主题，来分析大部分和相亲相关的话题，学习了本课程之后，你再次面对相亲时，不再会手足无措，相亲将变成你步入美好婚姻的最好途径。
