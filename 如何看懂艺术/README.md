## 简介
艺术经理人、艺术品收藏家翁昕，带你踏上欧洲第一站，意大利，为你讲述像小说一样好看的艺术史。


本专栏从欣赏西方艺术的第一站 ——意大利开始，由翁昕带领读者重回伟大艺术的诞生地，亲临罗马、梵蒂冈、庞贝、米兰四大古城，游走于大大小小的美术馆、教堂与屹立千年的名胜古迹，零距离欣赏艺术名作。本书从古希腊、古罗马时期讲起，经过漫长的中世纪，到文艺复兴的黄金时代落幕，跨越千年历史，细数达芬奇、米开朗琪罗、拉斐尔、卡拉瓦乔等众多艺术大师的创作生涯，及其旷世杰作背后的精彩故事。
