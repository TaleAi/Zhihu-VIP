## 简介
10 个残疾人生存故事，带你走进一个你想象不到的残酷江湖。


有一类弱势群体，全中国有 8500 万，由于出行不便，他们很少在公众场合露面，他们就是残疾人群体。


残疾人的江湖世界，是人性考验的实验场。一个天生就弱势的群体，你是伸手救助，还是趁机占便宜？不同的人选择截然不同。


一个母亲为了给智障的女儿追凶，不惜一切代价找到了那个强奸她女儿的凶手。原来，凶手就在身边；


一个哑巴姑娘，被父母遗弃给奶奶。缺爱的她因为痴迷偷盗团伙头目的虎豹哥哥，误入团伙参与偷盗。期间她多次间接帮助了团伙犯罪；


一个残疾家庭出身的姑娘，考上大学、参与工作，与同为智障家庭出身的丈夫结婚，可因为二人的家庭阴影，导致婚姻的诸多摩擦；


敏感而心思细腻的残疾人，更能感受人间的辛酸苦辣。


也许，他们更有机会窥见这个世界的真相和人性的善恶。


本专栏里多个故事皆为当事人亲自口述后整理，为保护隐私，已作化名处理。
