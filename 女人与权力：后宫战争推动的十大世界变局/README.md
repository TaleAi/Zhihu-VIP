## 简介
权力的游戏并非只是男人的专利。历史的惊涛中，也有女人的身影。宫廷战争如何改变世界？王室内部有哪些令人咂舌的秘密？一起穿过高墙，一探究竟。


宫斗剧源于历史的蓝本，但历史上真实的宫廷战争却远比宫斗剧更惨烈。因为它无关风月，只关乎权力。在权力的竞逐场上，不止有男人的金戈铁马，也有女人的善舞长袖。


为实现个人抱负不惜戕害亲生儿子的拜占庭女皇伊琳娜，最终让东欧与西欧，难以挽回地走向分裂。这道裂痕直到今天，还清晰可见。


莫卧儿帝国皇帝为娶努尔贾汗，不惜谋杀她的丈夫，但最终也使她成为了被印度百姓广为传颂的「打虎」皇后，她留下的建筑也成为举世闻名的「泰姬陵」的样板。


出身卑微的乌克兰奴隶罗克赛拉娜，通过一系列的斗智斗勇，华丽转身为土耳其宫廷中的翘楚，却在无意中葬送了奥斯曼帝国的国运。 


催动每一次世界变局的，在名载史册的的男人之外，还有那些身居宫闱的女性。她们构成了左右时代命运的另一条暗线。究竟谁在书写历史，谁在描摹世界的面貌？读懂这十场宫廷战争，你将会有全新的答案。


  




**你将获得：**


1. 环球旅行家的独特人文洞察；


2. 世界宫廷内幕的深度揭秘；


3. 突破思维惯性的历史解读；


4. 身临其境的十次穿越漫游。
