## 简介
爱情的世界没有妥协，婚姻里容不下第三个人。


《亲生子之谜》


陈芸无意中发现自己孩子的血型和他们夫妻的血型不合，她带孩子去做了亲子鉴定，却发现十月怀胎的孩子和丈夫是亲生，跟自己却毫无关系。


层层疑雾，重重谜团，陈芸突然发现自己那为人称颂的模范家庭、完美丈夫，其实在下一盘很大的棋。而自己，从一开始，就只是颗棋子……


《结婚前夕》


三室一厅的房子、四十万的彩礼、二十万的家用车……我东拼西凑，几乎倾家荡产凑齐了彩礼。


可假怀孕的女友、泼妇骂街的准岳母、高利贷登门追债……等待我的竟是无休无止的欺骗。
