一觉醒来，我被病娇的青梅竹马囚禁了。


他切断了我跟外界的一切联系，帮我把工作辞了，还说会一直养我。


还有这种好事？


我当即痛心疾首地表示：你怎么不早说啊？


早说的话，我还当个屁的社畜啊！


虽说是他囚禁我，但比他更害怕分离的人，其实是我。


几次分离又重逢，我原以为只是巧合，后来才发现，是他蓄谋已久。


我为他的表白和毫无遮掩的爱心动。


可当年与他形影不离的美术生姜妙，后来陪伴他时间更长的女强人邱织月。


出现在他身边的每一个人，似乎都比我更优秀。


还有，当年临近高考，他不辞而别，突然出国的真相，究竟是什么？


我真的能打开心结，帮他走出病娇背后的心理困境，和他终成眷属吗？
