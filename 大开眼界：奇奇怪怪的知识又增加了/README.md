## 简介
《异类》《引爆点》作者纽约客怪才格拉德威尔经典作品，罗永浩个人演讲首推图书！


女性为什么老爱染发？乳房 X 光摄影为什么等于是白做了？避孕药的发明有何意义？驯狗之前为什么先得训人？


《大开眼界》将一点点揭开微妙世界的秘密，如果只站在你的角度看这个世界，始终无法真正「看见」；如果换上另一个人的眼睛、用上另一个人的头脑……会大有不同！


作者**格拉德威尔**，他是《福布斯》杂志 10 大最具影响力的商业领袖之一，被《时代》周刊评为全球最有影响力的 100 位人物之一，他将用非凡的笔，点破我们漏掉的视野，挖掘世事隐藏的奇迹。


在这个专栏里，我们将与「天才型小人物」亲密接触，看厨具界王牌推销员如何施展口才魔法，从仅靠触摸驯服凶猛小狗的「神童」训狗师身上顿悟「人」才是关键，并从西红柿酱的口味、避孕药的发明、染发剂的营销故事中找到卓越人士的成功诀窍；


我们会用「另一双眼睛」，看透街头游民、乳房 X 光摄影、航天飞机爆炸等问题，更好地博览及微观世事，洞察事情背后的内在逻辑；


而我们评断人的能力也将面临质疑，最终发现，其实每个人都具备「大器晚成」的潜能，关键在于我们愿意给自己多少从错误中学习的机会。
