## 简介
世界历史就是一部战争史。


没有一个国家或者民族是以纯粹的平和走到今日的。


战争是政治的延伸，一些迷惑的军事操作背后，往往隐藏着不为人知的大国博弈。


战争胜负的天平，不仅仅衡量双方的武器装备、奇兵良谋，更受控于瞬息万变的战机、 背水一战的士气……


本专栏将带你看透战争背后的因缘际会：


* 长津湖战役有多惨烈？
* 上甘岭战役中无一人牺牲的班组是如何创造奇迹的？
* 清朝时，中国广州的一个村落如何对抗了一整支法国军队？
* 阿富汗为什么是「帝国坟场」？
* 墨西哥毒枭为什么可以嚣张到对抗军队？

本专栏邀请众多军事战争领域的优质创作者，带你看那些匪夷所思的战争纪实，解析大国背后的博弈操作，分析战争对一个国家乃至世界格局的影响。
