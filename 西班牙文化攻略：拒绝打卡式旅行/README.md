## 简介
斗牛、海鲜饭、大航海，提到西班牙你会想到什么？想去西班牙，要怎样做攻略才不会白去一趟？西语专家、西班牙最高学府双料硕士李沛姿，将用 12 节音频课带你领略异彩纷呈的西班牙文化。


古罗马、角斗士、航海家、征服者......西班牙比你想象的更加多元，面对西班牙充满异域风情而又绚丽多彩的历史、艺术、人物和宫殿、教堂、城堡等文化精华，如何去接近西语世界的人，消除隔阂感、陌生感和茫然感，了解这些地方骨子里的精髓呢？


西语专家、西班牙最高学府双料硕士李沛姿为你描述其无与伦比的人文、艺术价值，用 12 期音频节目，带你领略异彩纷呈的西班牙。希望下次你去西班牙的时候，不再只是去网红景点拍照打卡，而是能切身感受历史进程中的西班牙底蕴，找到旅行的意义。


**课程特色：**


* 实操性强：知识干货，贴近最真实的西班牙
* 极简入门：用最短的课时走进西班牙
* 探索未知：全面了解西班牙的艺术与历史文化
* 生动有趣：每期 10 几分钟，拒绝枯燥乏味的理论

**你将获得：**


* 最全面的知识储备：为你展示旅游攻略所不能提供的景点知识与百科
* 最丰富的冷门信息合集：发掘冷门而又精彩的西班牙人文景点、人文信息等
* 全新的文化解读：带你走近不一样的西班牙

**适合人群：**


* 想要深入了解西班牙绚丽多彩的艺术文化的小伙伴们
* 热爱足球的巴萨、皇马球迷们
* 想去西班牙旅行的旅行爱好者
