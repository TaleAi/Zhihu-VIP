## 简介
10 个案子，带你了解司法进程中的曲折之处。坚持正义和真相，是每个人的良心底线，也是司法运行的基石。


「妻子被杀害，抛尸江中，我成了凶手。我喊冤没人相信。为了得到清白，我奔走二十多年。」


「8 岁时，我亲眼看到妈妈躺在血泊里，而爸爸被指控为凶手。爸爸坐牢 17 年，我一直坚信爸爸是被冤枉的。」


「一个少年被绑架杀害，我的儿子成了杀人嫌犯。性命攸关之际，我的妻子却向警方做了伪证，儿子直接被判无期徒刑。」


......


我们永远也无法保证自己的人生不会遭遇「黑天鹅」，可我们也永远不能在命运面前败下阵来。


正义可能会迟到，却从来不会缺席。
