## 简介
每一个普通女孩，最终都不要迷失自我。


本专栏的故事，讲述的也都是普通女性的故事。「我只想做我自己」并不是一个口号；面临婚姻、事业，一重重选择和被选择；经历了那么多困难，付出那么多代价；一点一点地成长，并且从中吸取教训。


**生而为人，去做自己。**


为了孩子看病花了一百万的母亲，快要撑不下去了；


丈夫残疾后，面对一众献爱的男人，她清楚他们的目的；


面对巨婴丈夫，浪费了十多年青春，决定提出离婚；


……


本专栏将分为三个部分，「婚姻」「事业」「自我」，为你立体展示 30 岁+女性的真实生活。她们每一步都走得很难，可依然咬牙去追求更好的生活。
