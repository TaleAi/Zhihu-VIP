# 爆裂煤城：九位小人物的宿命舞台

### 分类：奇幻玄幻 · 悬疑推理

### 存档日期：2022-05-22

## 简介
故事发生在某座以煤为生的小城，九位小人物的宿命在这座小城舞台上轮番上演：有患癌青年因绝望而踏上犯罪的道路；有归乡青年逃离一线路途的奇遇；有 90 后哭丧者的成长残酷史；有失业矿工预谋抢劫煤矿；有山村前支书杀人；还有失意青年相约旅途自杀，一车一路，终得因果……


**身患绝症后，我们决定合伙犯罪——《不为死》**他已经很有条理，由简到难地规划好了一切。1、去步行街偷行人的钱包和手机。2、抢一辆跑车。3、纵火。4、杀人。我把打印纸重新放回茶几，用质疑的口吻问他：「真要这样吗？」


**有一种青年丧，叫回乡丧——《年三十儿》**他想过把这些白底黑字的数据用快捷键一扫而空，那样就换个方式生活，没有理想，没有追求，眼里只剩金钱和权欲，坦然一点，自由一些，但事实上，梦这个玩意儿在偏执狂内心是比毒药还要上瘾的东西，难以舍弃，他不清楚这是生而为人的优点，还是缺点，只好在其中纠结，痛想欲生。


**子弹辗转的夜，支离破碎的人——《弹无虚杀》**从审讯室出来，赵焕握紧拳头捶了捶的脑门，点了根烟，法医报告是昨天凌晨提交过来的，死者成大海，男，四十六岁，职业村长兼矿长，死亡时间是二零零七年五月十九日上午九点四十分，即使解剖尸体后发现死者又是高血压又是潜在性肺癌，但致死原因仍旧是中弹流血过多身亡，凶器是一把猎枪，一次性可装填五发子弹，上膛一次皆可射出，在死者身上法医取下三枚弹屑，嵌在死者胸，肺，肝处。 


**论如何抢劫一座煤矿的技巧方法——《劫煤记》**那天是个黄昏，我们拼凑了拼凑兜里的钱，正好够去小吃街喝四碗肉丸吃两提小笼包，大家坐在那二十来公分高的长凳上就开始掰扯，可能此情此景，没有酒都能尽兴，心里那点犯罪欲望强烈燃烧，聊着聊着就扯到了非法赚钱的渠道上，孟伟提议不如去贩毒，他舅舅黑社会，主要营生就是以买卖大麻为主，可惜几个人都怕染瘾，别到时候钱没赚到，倒是把自己给毁可。牛博说不如去盗墓刨坟，专找元代以上的墓下手，连个尿壶都能买个十几万，不做亏心事，不怕鬼上门，谁都怕鬼，更别去找鬼，还偷鬼的东西，很容易把自己也搞成鬼，这个建议也不科学。余军指了指不远处掉漆的舞厅招牌说：「不如我们开按摩店吧！」异口同声地滚蛋，抑扬顿挫。


**我是个 90 后，也是个哭魂人——《哭魂艺**术》  

我指指身后那些穿着青色长褂的王八蛋们说：「我在门口等来着，他们把我锁外面了。」  

「没人会把你锁在外面，只是你不愿意进来，所以才觉得把你锁外面了，你为什么不推一推门，人生嘛，你得推推，才知道前方的路通不通。」


**生擒劫匪，智斗杀人狂——《我的自杀旅途》**你自杀属于生活所迫，我杀你是生活向往。


**杀死城市的水——《乌流》**凌晨十二点半的排污管道口准时爆发，从原本细小的水柱变成了粗壮般的巨蟒，张着獠牙，撒着毒液，游进了赤河当中，她不知道这条毒蛇有多长，或许根本无法测量，日以夜继的，每分每秒的，奔跑在赤河汹涌的水流中，流过每一个乡镇，流过每一个县区。


**开学那天的荒诞杀人事件**这里是警官学校，2016 年 9 月 10 日开学首天，新生宿舍楼就死掉了一个学生，我们一致认为他是被谋杀的，系副主任用他多年教书的经验命令大二的学生会成员封锁了现场，因为找不到合适的警戒线替代物，警察赶到之前，整间宿舍缠满了我新买的一大卷卫生纸，我泣不成声，因为这卫生纸是给移动硬盘准备的，那些女神知道后一定再也不让我下载她的片了。


**怀念夏天的日子，可以撩起女同学的裙子**要有多少酒才能撂倒这充满诱惑的黑夜，窗外霓虹灯闪烁，街道上呼啸的引擎声，广场舞的音乐，谁也不想拒绝。
