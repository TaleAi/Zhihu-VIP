## 简介
婚姻真是爱情的坟墓吗？爱情一定在结婚后才减退吗？
不一定！也许，有的爱情，半路就已经脱轨……而每个脱轨背后，都是故事。
一段感情，无疾而终，他背叛你，你报复他。
一份承诺，转头成空，她爱上你，你爱上另一个她。 
这是一个讲述爱情与现实、背叛与复仇的故事合集，数十位主人公的经历、情感，一一为你讲述。


《渣男的代价》


生产当天我收到老公和其他女人亲热的视频，又意外得知这个渣男想骗走父母留给我的遗产。劳心劳力操持这个家，结果换来的是背叛和欺骗？渣男不仁就别怪我不义了！


  



《我不是樊胜美》


几年前家里拆迁，爸妈把房子都给了哥哥，现在父母病了，却只找我养老。我卖房给父母治病，结果却发现钱是用来给哥哥还赌债。我狠心拒绝，却被父母卖给隔壁老光棍……


  



《我生下了小三的孩子》


结婚四年，我意外发现我怀胎十月生下的儿子，竟是小三和老公的亲生儿子。我竟被骗做了小三和渣男的代孕……为了找出设局害我的小三，我忍受着内心的绝望和煎熬，不动声色的和老公周旋，只为了寻找事情的真相，还自己一个公道。
