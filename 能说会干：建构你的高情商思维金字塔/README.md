## 简介
本课程由沟通大神仲佳伟和文娅联合主讲，教你如何轻松化解沟通难题，快速成为沟通高手。课程中老师会从清晰表达、强化关系、化解冲突等 4 个模块出发，共 21 项高效沟通技巧，让你学到最实用的管理沟通课。


你是否有沟通方面的烦恼？


你是否是刚跳槽到新公司，希望通过高效沟通，完成工作的职场新人？


你是否是想要通过沟通解决职场难题，快速赢得领导赏识的骨干员工？


你是否是希望通过沟通高效管理下属，并且有效争取资源的管理者？


如果是这样，那么本课程将是你的福音。本课程由沟通大神仲佳伟和文娅联合主讲，教你如何轻松化解沟通难题，快速成为沟通高手。


两位老师将用自己 10 年的一线职场经验，历经 1000+ 小时打磨，手把手教你成为高情商的沟通大神。


两位老师基于独创的「高情商金字塔模型」，从沟通的底层逻辑出发，还原多种沟通场景，让你从沟通小白成长为沟通高手。


本次课程，将为你提供 1 个科学公式， 3 大沟通模块， 21 个沟通法则，让你拿来就用，脱口而出。


不管是什么行业、什么职位、内向者还是外向者，都可以快速掌握沟通技巧，解决沟通难题，获得人人都羡慕的高情商。
