## 简介
如果真相是血淋淋的，你还敢于直面吗？


「我受邀参加读者的婚礼，没想到那是一场『阴阳媒』。」


「读者把房间装修成荒野血屋，只为写出恐怖故事，我却看到失踪女孩的手提包 。」


「女孩坚信自己被鬼附身，她怀疑是『神婆』奶奶暗中做鬼。」


……


如此惊悚怪异的事件，我听悬疑作家陆如书讲过很多。


在他秘密的笔记本中，记录无数个恐怖、灵异、惊悚甚至是难以解释的采风素材。


如果不是他亲口保证，我不会相信这是真的，就像他说如果不是亲身经历，他也不会相信真相如此离奇。


陆如书要把这些年诡异的采风经历写作成书，让我们一起看看阳光照不到的角落里，包藏何种祸心。
