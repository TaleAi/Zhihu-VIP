## 简介
明月寒江，朱颜映雪，古虐合集。


**《归字谣》**


他问：「公主有什么想说的吗？」阿若想了想，回：「希望下辈子不要再遇到你了。」


**《昭昭我心》**


泽乔撑起伞，白雪落到绘着缠枝花的伞面上：「你只对我一个人温柔便好，旁的妖凶一些也无碍。」


黎昭纠正他：「我没有对你温柔。」


**《枉宫梅》**


即使黄土白骨，山河荒芜，我都不会再丢下你了。


那是他对我的承诺，句句我都记得。


**《瑶山有渊》**


「暝嫣，在你眼中，朕是不是从来都是个笑话？」


  怎回事笑话呢？他曾是我最羡慕的人，如不是落入权谋的污泥中，如今或许他会在边疆肆意地策马奔腾，娶一心仪女子，幸福一生。


  我想救他，可我本身就身处炼狱。我想赶他走，又想拉他下来，让他同我一起堕落。
