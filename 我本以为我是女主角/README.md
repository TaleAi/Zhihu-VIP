# 我本以为我是女主角

### 作者：喵太郎

### 分类：奇幻玄幻 · 现实情感

### 状态：已完结，共 18 节

### 存档日期：2022-05-22

## 简介
如果世界是一本书，那么女主就该是我的模样。


我从小肤白貌美，家世显赫，万众瞩目。


直到楚楚可怜的金楚涵出现，将我顺风顺水的生活彻底打碎。


青梅竹马的恋人对我冷嘲热讽，却转头丢弃自尊向她摇尾乞怜。


宠爱我的父母将我养的猫丢走，却温声询问她晚餐是否合口味。


就连从小到大对我百依百顺的哥哥，当我被她污蔑抄袭的时候，也只是站在她身边，用陌生无比的语气对我说道：「你怎么这么蛮横，不懂事。」


我无法理解地看着周围的一切，仿佛一夜之间从女主变成了炮灰。


就在这时，我听到了一个来自「系统」的声音。


原来她不过是占据了别人身体的外来者，靠着攻略男人的积分恣意玩弄别人的生活！


她与系统得意洋洋地说着那些自以为隐蔽的恶毒计划，那些我本来要经历的可怕下场，殊不知我全都听得到！


我需要隐忍，与此同时一个复仇计划在我心中酝酿着。


就算拼尽一切，我也要将背叛我的人全部送进地狱！


饵已放好，局已布下。


如果这是永无休止的轮回，那就看看究竟是我还是所谓的神能笑到最后！
