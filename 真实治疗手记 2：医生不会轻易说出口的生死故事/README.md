## 简介
医院是人生的「困境聚集地」。平时你不会去思考的问题，在这里却必须⾯对，并且给出⼀个答案。我们会在这里看见，自己⽇常⽣活中那些艰难时刻：因为每位被记录的主人公，都是和你我⼀样的普通⼈，只不过，他们正在经历⼈⽣中最艰难的⼀役。


有个段子说：


年轻人不要轻易觉得人生到达了低谷，因为你的下降空间还有很多。


当你觉得熬不下去的时候，移步本地三甲医院急诊室呆一晚，会对人生有全新的认识。


 


许多人不愿触碰医院的故事，这里有太多悲情，太多无能为力，


然而在医生眼里，


生老病死关头，有很多远比情绪更深刻、更硬核、更真实的内容：


![img](None)
重庆的一个 ICU 病房里，


有人渴望活着，有人祈求死去，


而他们每人手里，都捏着一只小黄鸭； 


一个自杀未遂的 18 岁东北姑娘，


住进精神科病房之后，


医生们决定，配合她备战高考，一天一套黄冈密卷；


服侍送别了五位渐冻症至亲，


老奶奶说：「从今以后我在外面忙，再也不用回头看，没有人再等我。」； 


![img](None)
 「医⽣天⽣就是讲故事的⾼⼿。」


他们目睹了痛苦和死亡，见证了穷困患者的痛苦，见证了濒死患者的绝望和孤独。


医学是一种回应他人痛苦的努力。


当医生去讲述自己治疗过的一个个人、经历过的一件件事，也是在整理关于生命终极问题的瞬间。

### 为什么看


医院里来来往往的人，都有自己想要的。


读懂这些，


你可以看见，经历过某种绝望的人，怎样面对命运的不公？


你会重新理解，面对生命，什么叫做「乐此不疲，义无反顾」。


你能从中找到，⾃⼰当下所需的答案：我们该以什么样的姿态去⽣存？ 


  



### 专栏亮点


1、感受到美剧般的阅读体验。


2、看见全国顶尖科室医生们的真实日常。


3、医生视角，带你看见生死场上的人间百态。


4、医生如何思考决策？理解医学的精神。
