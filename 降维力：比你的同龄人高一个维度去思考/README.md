## 简介
打破思维界限，打开人生。


他们为什么能边打游戏边成为学霸？我和学霸到底差在哪里？


都说鸡汤没营养，我怎么知道自己读的是不是鸡汤？


书读了很多，但好像又什么都没读过？我的记忆是丢失了吗？


早六晚九，依然考研失败，你是不是在行为艺术式学习？


疫情来临，如何提升在家办公效率？线上办公时代会到来吗？


……


更多时候，我们的迷茫与焦虑，我们的困惑和不解，我们经历的种种失败，并不是我们本身的能力与别人有多么大的差距。而是我们的思维方式出了问题。


如果我们能跳出原有的思维的禁锢，比别人高一个维度去思考。很多问题就能迎刃而解。而这里，就是想给你一个有趣的思维工具箱。
