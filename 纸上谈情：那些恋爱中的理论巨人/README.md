## 简介
爱情，爱情，真伤脑筋！就算有一身本领，却也不能解决恋爱中的难题。


爱情三十六计已经熟练掌握，情话七十二变已烂熟于胸。


可我还是追男撩妹的矮子。


我不过是个擅长纸上谈情的理论巨人罢了。


甜甜的爱情什么时候会降临在我身上？


苦大仇深的虐恋也行啊。


我就不信一身理论知识对付不了「甜宠」「虐恋」「男二上位」「追妻火葬场」「移情别恋带球跑」……了？


本专栏的故事有甜的虐的穿越的，还有沙雕搞笑意难平的。
