## 简介
换个角度看美苏冷战这段历史


美苏冷战，除了宏观层面上的大对抗，背后还有很多鲜为人知的「硬核」对峙。


甚至可以说，冷战的大部分时间里，真正处在对抗最最最前沿的，不是双方军队，而是情报机构。


那时，各国间谍们早就不满足窃取情报、策反对方人员这类基本操作。


为了获得敌对阵营武器装备的一手资料，他们经常会把对方手中的实物直接打包带走，各种硬核「骚操作」频出：


* 以色列间谍在瑞士和法国眼皮子底下，偷走了重达两吨的法制幻影战斗机图纸。
* 苏联间谍不仅成功偷到了美国的响尾蛇导弹，还光明正大用航空快递，把重达一百七八十斤的导弹寄回了苏联。
* 更绝的是，美国中情局在苏联的严密监控下，偷走了一条重达 2700 多吨的苏联潜艇。

······ 


本专栏带大家走进冷战时期的精彩情报战，通过这些构成、影响了战争走势的硬核对峙，让我们从微观层面，对这段历史有更加立体生动的了解。
