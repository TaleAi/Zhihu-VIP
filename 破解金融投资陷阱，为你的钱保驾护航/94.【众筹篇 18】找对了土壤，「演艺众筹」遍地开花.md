## 94.【众筹篇 18】找对了土壤，「演艺众筹」遍地开花
2018 年的春节档，我们大陆电影的票房让人刮目相看，仅仅是春节假期，《捉妖记 2》、《唐人街探案 2》、《红海行动》、《西游记——女儿国》等几部片子累计票房就达到了一百亿人民币。太震惊了！这其中呢少不了演艺众筹的功劳，像《捉妖记 2》，《唐人街探案 2》这些影片在宣传发行阶段就提前就通过众筹的方式预热，在影片未上映之前就已经有票房预售收入产生了。演艺众筹，在国内的演艺行业发展得越来越完善，参与的支持者也越来越多。演艺众筹属于众筹当中的奖励式众筹。这几年国家在政策法规方面是扶持文化行业发展的，在演艺影视行业经常有一些大的 IP 产生， IP 指的是版权，有很多的支持人或者是粉丝，出于对这些影视 IP 的热捧，也愿意参与到演艺众筹中来，演艺众筹打破了传统的消费者观影方式，艺术消费方式，使消费者能充分参与到演艺作品的制作发行过程当中，参与感更强。


演艺众筹的操作模式是筹资人，也就是项目发起人，首先要在众筹平台上注册，然后向平台提交详细的项目说明资料，介绍项目的进展、项目内容，对支持者的回报，以及必要的风险提示。演艺众筹的项目推介材料中，一般都会有演艺作品展示或者是拍摄宣传视频，做一些美工设计这种体现美感的方式，上传给众筹平台进行审核，众筹平台对项目资料审核通过以后，项目上线，已经在众筹平台上注册的投资人会寻找自己感兴趣的演艺众筹项目投资。项目发起人要设定准备筹集的目标金额，还要确定筹资期限，在投资期限之内，如果筹集的目标金额实现的话，那么这个项目就算募集成功，如果在规定的时间内，项目没有募集到目标金额，一般情况下项目就会终止，已经支付的资金会退款给投资人，这种情况演绎众筹项目就是筹资失败了。


演绎众筹的项目，有的是通过综合性的众筹平台进行的，比如国内的综合众筹平台点名时间，还有的是通过专门做演绎众筹的平台，比如国内的淘梦网，淘梦网是属于垂直型的众筹平台，由北京淘梦网络科技有限责任公司运营，只做演艺众筹，淘梦网成立于 2012 年，现在已经发展成为国内最大的微电影众筹平台。淘梦网上的微电影是按照内容细分的，有动画题材、爱情题材、动作题材、恐怖悬疑等等十多种类型。通过淘梦网已经成功实现演绎众筹的项目有上百个，成功率接近 50%，成功率还是蛮高的。


有的项目发起人为了加大宣传，也会通过 0 元募集的方式，也就是不募集资金，只是为了展示，将项目放在淘梦网上，希望通过支持者的留言、点评、讨论产生互动，对作品进行检验，这种项目的发起人会给支持者提供签名，合影，或者是提供一些群演的名额。有很多喜爱电影的支持者是很愿意参与点评的。淘梦网现在已经与优酷网、爱奇艺、腾讯视频这些主流的视频网站达成合作，还会将平台上的一些项目推荐给国内的电视节电影大赛等等，加大项目的宣传力度。有的演艺项目通过这些宣传渠道为大众所熟知，反过来会更助推筹资的成功，很可能就成为一个非常好的演艺作品。现在，通过演艺众筹发掘了很多优秀的演艺作品，比如《大鱼海棠》、《大圣归来》这些优秀的影片。演艺众筹帮助很多项目发起人实现了艺术梦想，促进艺术传播和艺术普及。项目发起人和支持者都受益，这就是演艺众筹的魅力！
