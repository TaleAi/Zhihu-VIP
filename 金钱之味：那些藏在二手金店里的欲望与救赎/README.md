## 简介
有时人性在金钱面前无处遁形。


我是「开源金店」店主，一个干黄金回收的。我们这一行很杂，不光收金子，也收银元，收烟酒，收购物卡这些杂七杂八的。


该专栏主要由「我」来讲述那些在我金店里发生的故事。


这些故事皆因一个字而起——「钱」


所谓「天下熙熙皆为利来，天下攘攘皆为利往」，所以我在店里看尽了人性之复杂。


**葵花茅台——夫妻本是同林鸟**


他们白手起家，相濡以沫，最后却走到了你死我活的境地。她是我见过的最淡定的被出轨者，面对丈夫的层层套路，她见招拆招。酒是假的，情是假的，到底什么才是真的？


**黄金手镯——寻找消失的妹妹**


母亲去世后留给她一只金镯子，叮嘱她一定要找到妹妹。她在风月场所守身如玉，却在街头以最肮脏的姿态离去。「店主，帮你……」我知道她希望我帮她找到妹妹，但这个妹妹仿佛人间消失一般。


……


金钱从来不是作恶的源头，那些隐藏在背后的欲望和救赎才是真相。


温馨提示：虚构故事，带你见识人性。
