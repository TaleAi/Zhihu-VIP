## 简介
王子一定会和公主在一起吗？女巫真的很邪恶吗？巨龙总是大反派吗？童话中的结局是否已经注定？被小公主坑了的巨龙、被女巫救了的青年，想种田却被迫当上英雄的淳朴弟弟……当一切童话反转，你将认识不一样的王子、公主和巨龙！


万物王国的公主希望像哥哥们一样在战场上保卫国家，而那些哥哥在上战场前都要杀死一只猛兽，带回猛兽身体的一部分，以此证明自己的勇气和能力。为了获得国王的认可，5 岁的公主决定挑战巨龙，以此证明自己……


——《公主：巨龙，在？借个头》


一个青年献出双眼，从女巫那里换取解药，将失明的公主治愈。按照约定，公主会嫁给治愈她的人。但当公主看到青年是个瞎子时，不仅反悔，还把青年当作骗子打入大牢。然而喜欢青年的女巫拯救了他，故事开始朝着大家意想不到的情况发展……


——《爱的选择》


一个年轻人在森林中遇险，被骑着马的少女搭救。少女是森林女神，身边有一只鹦鹉。后来年轻人和少女相爱，可是一场意外的旱灾降临，开始吞噬森林，少女的容貌也飞快衰老，生命力不断减退。少女看出了年轻人的恐惧，便编造理由骗走了年轻人。后来少女死去，年轻人回来了，带着那个虚构出来的物品……


——《NSZD》
