## 简介
作家盛文强，从古代志怪、野史、方志中打捞出海盗碎片，拼接重构成《中国海盗奇谭》，让你重温海盗精神、领略万里海疆风云。


  



![img](https://pic2.zhimg.com/v2-b5cce70ff9f31aee4190f4e9e3fc3bc1.jpg)

这张画看起来，是不是很劲爆！一位海盗和一群官兵抡刀互砍。官兵这边已经倒下了一个，剩下的几位直往后缩，脸上写满了恐惧。海盗呢？神情自若，从容不迫。马上，画上就该倒下一片了。


这么猛的人，到底是谁啊？ta 的名气那可大了去了。


所有的『最凶残海盗』排行榜，必然占据一席之地；


曾经把清、葡、英三方海军耍得团团转，杀得片甲不留；


在西方世界家喻户晓，《加勒比海盗》也安排了角色；


是《文明 6》的 11 位中华伟人之一，和孔子、屈原、李白并列。


最为传奇的是，ta 是个女人，叫郑一嫂。做海盗前，是个妓女。


这就让人惊讶了：  

史上最凶残的海盗，居然是个中国女人？  

她比绝大多数男人活得更传奇，怎么没写在历史书里？  

中华伟人何其多，《文明 6》怎么偏偏选中了她？


**被埋没的「中国海贼王」，可不止郑一嫂一个。**


明朝大海盗王直，最早把火炮传入日本，并在萨摩洲称王开府；


郑成功的父亲郑芝龙，会五国语言、能和「佛郎机人」高谈阔论，广阔的东海是他家的游泳池；


张保仔，从海盗头子的男宠到朝廷二品大员，最后得以善终。


他们曾在漫长的海岸线巡游，乘潮水上下。他们一度强大到不可战胜，组织和装备远比朝廷水师精良，甚至令西方殖民者也望风披靡。他们身上的叛逆、个性、自由、放荡不羁，乃至契约精神，与西方同行相比，毫不逊色。


可是，中国没有留下一本以海盗为主题的专著。正史上也没有他们的位置。中国海盗故事，一直都隐而不彰。


作家盛文强，从古代志怪、野史、方志等文献中打捞出海盗故事的碎片，拼接重构成《中国海盗奇谭》。


  



![img](https://pic4.zhimg.com/v2-3b0d9b946a210d628eccc5a0082bf690.jpg)

在这套课中，


你将走进海盗们的日常，全景扫描生活细节的方方面面：


**海盗们怎么拿到赎金，并分配赎金的？**


**为什么海盗船忌讳带女人？**


**为什么海盗喜欢蒙上一只眼睛？**


你将旁观海贼王们的传奇，在波浪翻涌的海里，与他们同呼吸共命运；


你还会在一面海盗旗前伫立良久，因为它暗含着中国海盗兴衰的秘密。


**土地的尽头，是雄心与梦想的开端；风云变幻的海上，要担负起新奇的考验。**


请打开《中国海盗奇谭》，敲开历史的彩蛋，告别繁复的日常，自由、不羁、冒险、进取，你值得拥有。
