## 简介
在这个睡前深夜，你将跟随作者，走入他们大脑深处的羊肠九曲。


一个正常人的大脑，平均有 860 亿个神经元，这些人却专门拿来撰写世间「惊奇」。他们仿佛一群不受控的神秘生物，用手里一支笔，把「人性」植入一场场思想实验，一个个心之妄念。  




这里有让人倒吸一口凉气的「惊」，故事中塞满令人毛骨悚然的鬼型怪状：


多重人格分裂，「假性」家暴者，引发一起悬疑叵测的致死虐恋；


有北宋年间的文人，和「女鬼」书信往来，意外拯救自杀的自己；


有人在一家路边小馆，端起一碗黄焖鸡，却开启一场生死交替，欲望轮回；  




此处还有让人叹为观止的「奇」，世界里填充了出离想象的天马行空：


有人拿到了 10 亿美元，却被一只蜗牛追杀，疲于奔命；


地球上出现一个没有边界的循环空间，人类在这里无止境坠落；


还有某个村落，它的祭祀仪式是一场编程，先人的名字就是参透秘密的代码；


……


看惊奇故事，加入一场让人血脉喷张的惊声尖叫，走进一个匪夷所思的异世界。


在这个睡前的黑夜，亦刿心怵目，亦触物兴怀，深吸一口气，故事开始了。
