## 简介
爱意和杀意一生双念，放下了可以成全彼此，放不下就是阿鼻地狱。本专栏为您娓娓道来那些和爱意、杀意、恨意、歉意相关的人间故事。



偏僻的养老院住进了一对相互扶持的兄弟，一人痴傻，一人照料，本来是人间的善事。可夜深人静时，在他们居住的深深的走廊尽头，总是传出渗人的声响，养老院的老人们为什么隔段时间就会消失无踪……


深海上波涛汹涌，风雨将至。大船向前行进，船头的剑客和逃难的旅客不约而同说起了扬州河上的绝色歌女。死去的歌女，深宅的大户，善妒的夫人，他们之间到底有什么恩怨情仇？


被深囚于地底的骄横女孩，因辜负了自己的恋人，而被对方父母买凶困住，只为求她一句歉意。可到底是什么原因，让女孩宁可生生世世被囚禁在铁笼，也不愿说出对不起三个字？


……


每一个故事讲述着每一个人的过去，也许还一起埋葬了他们的未来。看似离奇的经历和无法言说的复杂情感纠葛在一起，也许是最纯粹的爱意，也许是最纯粹的恨意，也许是扭曲的嫉妒，也许是最无私的奉献。也许今天您看到的只是故事，也许明天您就能在生活里找到他们的影子。人世间的爱和恨啊，永远没有尽头，理智和情感的挣扎是我们每个人都会面临的巨大挑战。


祝您阅读愉快，也愿这些故事永远都只是故事而已。
