## 简介
二十个现代爱情故事，有校园有都市，有悲剧有喜剧，有偏轻盈的青春幻想，也有偏现实的沉重引力。主角有在校学生有都市白领，有校园男孩女孩们的酸酸甜甜，也有在北上广打拼的小镇青年的无奈，尽显爱情世界的多张面孔。一切爱情都为时不晚，因为有爱情，世界才变得鲜活灵动起来。


男学渣暗恋女学霸，但他不奢望平庸的自己能参与她本该灿烂的一生，只愿祝福她一生光明顺畅；


过气选秀男明星为翻红雇枪手写睡前故事，打造暖心博主形象，却与女枪手暗生情愫；


父亲葬礼当天，爱上父亲第三者的拖油瓶，他和她秘密交往十年，却从不敢让母亲知道这份交往的存在，最终免不了分道扬镳，你我各有余生，但与彼此毫无关系；


天才长发舞蹈少女突然宣布退圈，十年后，爱慕者找到她，提出一个奇怪的要求，希望她留长头发，跳一支舞……


从校园到婚纱，从青春到白发，爱情都是永恒的话题。少年时青涩的暗恋、婚姻里的同床异梦、人生中不可言说的遗憾……只要你愿意相信，那么一切爱情都为时不晚。
