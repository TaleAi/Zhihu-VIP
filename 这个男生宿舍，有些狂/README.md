## 简介
如果大学提前把生活费花完怎么办？


你心中的大学是什么样子的？


是和高中一样努力学习？还是没人管束后放纵自己？还是追着学校里漂亮女生到处跑？


我来给你介绍下我的男生宿舍——


这几个男生一个比一个奇怪，有人脾气不好，有人性格猥琐，还有人像傻子一样天天乐呵呵。


但就是这三个怂货，帮我一步步追到校花。


我们没有生活费只能蹭吃蹭喝，谁承想蹭到了校花桌子上。更想不到的是，校花还是省散打冠军。


这样的校花怎么追？她真的打人！
