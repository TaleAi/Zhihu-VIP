## 简介
劲爆的宫廷秘闻，离奇的传奇故事，这里是慈禧的江湖，盗贼的江湖，也是你的江湖。


关于慈禧墓有着各种传奇故事，这些故事能颠覆你的想象力。


传说，慈禧死后竟然三进两出棺材。


此后，慈禧陵居然出现了神秘妖艳、时时哀鸣的九尾狐，飞沙走石呼啸而来的大蛇，让人后脊梁骨发冷的眼睛。


有了布老虎、藏宝图、盗墓贼、绝世珍宝，还有辱尸传说。


当然，这里也有劲爆的宫廷传说，离奇的传奇故事。


这些故事多数都是人们想象和编造出来的，但趣味十足，让我们一同走进这充满迷雾的慈禧墓。


注：本专栏内容纯属虚构
