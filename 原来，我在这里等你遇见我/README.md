## 简介
温暖治愈的浪漫军旅小说，一场始于意外，终于幸福的圆满爱情。


那年夏天，他站在急救室里对个小护士高声大吼时，她把他当成军人里脾气最暴躁的那一类；那年夏天，他站在急救室大厅看她桀骜的高声「我是民兵」，她成了他想要征服的「兵」。


那年夏天， 知了伏在树上蛰蛰而鸣，背着红书包的小学生上学去又上学回，门口卖冰棍的大妈来来回回叫卖三声。 而对温昕来说，这个夏天里最重要的事，是一个叫厉铭辰的军人就这样悄无声息地入侵了她的生活。


他建起爱的城池，自此给了她三月的温暖。一切都始于那个夏天，那个 43° 高温的早晨。他和她的旅程，始于意外，终点幸福。


「未来，我在这里，等你娶我。」
