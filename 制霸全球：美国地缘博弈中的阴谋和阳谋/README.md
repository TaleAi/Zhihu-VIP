## 简介
地缘冲突，只不过是棋手们的棋局博弈。


「二战」后，美国为了维持全球霸权，频繁给他国挖坑，用经济、军事等手段影响一些国家或者地区，这背后的地缘博弈错综复杂，不仅关乎他国命运，也影响未来全球大格局。


美国在地缘冲突和博弈中，阴（谋）阳（谋）并济，挥动手中大棒，不断改变这些国家的命运走向。


而远离冲突的美国，自然也是地缘冲突中的最大获益者。


本专栏将挖掘地缘冲突背后隐藏的秘密，带你看清美国背后的霸权逻辑：


* 小布什为什么要打伊拉克？布什家族背后的利益集团都是谁？
* 阿富汗穷得掉渣？错！阿富汗的矿产才是美国死扛二十年的信念支撑！
* 科索沃战争，美国人是如何坑惨欧元的？
* 卡扎菲死了十几年，利比亚的日子好过了吗？
* 俄乌战争，美国如何搅黄欧俄一体化？普京到底赚了还是赔了？
* 美国控制全球海上咽喉，中国想要破局，为什么麻烦不断？

本专栏将从历史、地缘、军事、经济等多角度入手，用全新的角度去分析各国的得失，以及美国隐匿其中的霸权逻辑。
