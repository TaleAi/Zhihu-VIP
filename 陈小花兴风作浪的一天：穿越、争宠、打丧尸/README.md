## 简介
一觉醒来，本 GG 穿越成了皇妃，皇上竟然还丢下我跟一个和尚跑了；死后我找到自己的女朋友：说好了你爱的是我的灵魂呢？我的 Siri 失控了，但是我备忘录里的羞耻心事还没删干净啊喂……反转多，会发糖，但不腻；很搞笑，有时候也会让你掉眼泪。如果你累了，不妨歇一歇，看看这个专栏里的故事。


生活的难题是无尽的，不解决内心的烦恼，谁也无法抵达通关之后的梦幻岛，不如让专栏主角陈小花替你试一试：


如果你如愿当上了皇妃，如果皇上和一个和尚私奔出轨；


如果你被困在同一天，如果代价是你必须每天历经死亡；


如果你在潦倒生活中骗保成功，如果你发现自己是真的挂了；


如果古代就有了丧尸，如果丧尸们的英雄是个人类少年。


「如果」之后还有「如果」，「愿望」之后还有「失望」。但面对生活，总是有人愿意回报以最纯粹的热忱。他们是冷冰冰的机器少女，无法投胎的鬼魂，率领丧尸的少年……是这些在崩坏年代的幻想世界中，拼命找寻生活出口的脑洞故事。


反转多，但不至于烧脑。会发糖，但不腻。很搞笑，但有时候也会让你掉眼泪。


如果你累了，不妨歇一歇，看看这个专栏里的故事。


你不一定能找到你的影子，但一定能找到一些奇奇怪怪的力量。


出发去梦幻岛吧，「陈小花」们！
