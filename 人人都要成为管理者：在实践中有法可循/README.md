## 简介
了解管理实质，轻松掌握管理的科学方法。


 不仅领导要学习如何管理员工，员工同样要学习如何「管理」领导，还有同事、客户还有供应商，他们需要你的「管理」，管理好了他们，你的工作才能够事半功倍，游刃有余，而不是杂乱无章，疲于奔命。


本课程一共分为 21 讲，由拥有丰富的管理教学经验的何川老师为你讲点人人都能用得上的管理知识。


本课程，何川老师围绕管理常识、管理目标、管理资源、管理行动 4 大模块，深入剖析管理的科学知识，帮你充实自己的管理知识储备。


何川老师将管理行动分为 4 种分类、 1 个关键、 5 个步骤、 3 个动作和 3 个要点，让你在实际应用中有章法可循，提升自己的管理能力。


同时，教你如何找到属于自己团队的管理目标，找到正确的前进方向。突破传统固有的与上级相处观念，传授「管理」上级的方法技巧，实现在职场上真正的成长。
