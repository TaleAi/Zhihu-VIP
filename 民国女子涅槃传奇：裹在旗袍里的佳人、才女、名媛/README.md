## 简介
解读民国奇女子进退得宜的人生智慧，讲述名媛大小姐高情商社交手段。


人生不易，女人的一生彷佛一直在升级打怪，


有些人需要面对问题百出的婚姻、难以教育的孩子、沉重的现实压力。


人这漫长的一生总会遇上一些无奈，内心的梦想被遗忘，生活仅剩一地鸡毛。自感平凡，对未来生出无奈之感…


如果人生有 100 种活法，给你轮番体验，你会如何选择？


读民国女子传记，在他人经历中找到属于你自己的人生答案。她们的故事中都有你的影子，也有你前行的方向。


不读林徽因，你不会知道，这位才女为何会吸引当时最优秀的三个男人；


不读董竹君，你不会知道，这个被卖到青楼的女人怎样成为督军夫人收获事业与爱情；


......


在本专栏你将感受民国代表型女子的跌宕人生、名媛大小姐的爱恨情仇，掌握高效实用的管「男」宝典、捍卫婚姻和爱情的正确打开方式。
