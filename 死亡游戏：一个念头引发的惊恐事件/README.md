## 简介
你可以拥有花不完的钱、至高无上的权利、永恒的爱情，前提是参加一场无底线的游戏，你会来吗？15 篇悬疑故事，15 段惊魂传说，用经典的设定，跨越生死，看穿人性。


「天堂公司发布一款重大游戏，随机邀请幸运儿参加，最终胜利者将获得 4 亿财产继承权！」


游戏中，他享受着金钱的狂欢。几天后，他收到了女友临终前的警告。


「陈墨，如果你看到这封信，说明我已经死了，切记，不要碰天堂游戏！


 ——《天堂的游戏》


  



为了挽回失败人生，他参加了一个真心话大冒险游戏，胜利者将拥有一切。


当他亲手掐死最后一个对手时。


「恭喜你，游戏的唯一终结者，发起人曹芝欣刚刚被你杀了。」


 ——《真心话大冒险》


  


勘探队员陈婉玉于东海孤岛给未婚夫发来一条奇怪信息：吴子明，勿要登岛！他不顾反对组建救援队登上孤岛。


在一棵大「树」上，悬挂的诸多尸体当中，他找到了活着的陈婉玉。


陈婉玉见到他的第一眼说。


「快杀了我！」


——《绝望孤岛》
