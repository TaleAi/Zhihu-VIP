## 简介
本科课程由北京市盈科（深圳）律师事务所合伙人陈雪松主讲，帮你切入广阔的诉讼逆转市场，不再把财富和机会拒之门外。帮助当事人在二审、再审中绝地反击。


有律所合伙人感慨过，擅长诉讼逆转的律师才是行业里「闷声发大财」的存在。每年看着所里律师们因为「很难搞」而不愿意接受这类型案件，流失了多少案源、多少代理费啊。


而这些流失掉的财富和机会去了哪儿呢？当然是擅长诉讼逆转的律师手里。与其追热点、找蓝海，为什么不把目光放在已经有成熟市场、自己主动拒之门外的业务上呢。


诉讼逆转主要针对已有初步结果，且对己方当事人不利的案件进行权利救济，主要针对的就是二审、再审案件。


二审、再审案件因其「立案难、改判难、案件复杂」的特点，一直以来都是律师执业的「痛点」。在二审、再审案件中，大部分当事人均为一审的败诉方，案件案情也相对复杂，法律适用及法律解释也比较有难度。


这让很多律师都望而却步。


但实际上，一时强弱在于力，千秋胜负在于理。一审败诉没关系，只要掌握了有力的事实和法律依据，在二审和再审过程中，仍然有机会反败为胜，实现诉讼逆转。律师能否在庭审中，利用证据帮助法官推导出真实的案件全貌，直接影响到再审案件结果。


诉讼逆转，无疑是最能体现律师价值的。无论是对当事人、对自己的成长都是其他案件所不能比拟。


为了帮助广大律师能展示自我价值，切入广阔的诉讼逆转市场，不再把财富和机会拒之门外——本课程邀请到盈科所合伙人陈雪松律师，开设诉讼逆转专项课课程，帮你在庭审中，反败为胜，实现绝地反击。
