## 简介
在这城市里，我相信一定会有那么一个人，想着同样的事情，怀着相似的频率，在某站寂寞的出口，安排好了与我相遇。


在这城市里，我相信一定会有那么一个人，想着同样的事情，怀着相似的频率，在某站寂寞的出口，安排好了与我相遇。 


于是我们相遇，相爱，十指紧扣，坐看年华老去。 


有人说，相濡以沫，不过是两人分享两段不同的寂寞。而我庆幸，那个人是你。 


因为我的恋爱，要么，不开始；要么，一辈子…… 


像塞纳河静静流淌巴黎城，颜珏和厉铮，在一起就是自然而然的幸福。
