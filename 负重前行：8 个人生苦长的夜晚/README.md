# 负重前行：8 个人生苦长的夜晚

### 作者：船长

### 状态：已完结，共 9 节

### 存档日期：2022-05-21

## 简介
孩子身患重病，亲人生离死别，恋情一拍两散，生活中的苦痛那么多，谁没有想一了百了的时刻，一咬牙就负重前行挺过去了？8 个平凡的人剥开自己的壳，向你展示那些「人生苦长」。


她只想做个母亲，经历了多次撕心裂肺的流产，家庭破碎了，工作到头了，拼命生出的宝宝却患了脑积水，希望在哪里？


她年老退休，丈夫离异，女儿北漂，父母不在，每天寂寞得发慌，一辈子的棺材本又被金融公司骗光，这日子一眼望到头，可什么时候是个头？


他是个大老粗，靠种地把孩子们拉扯大，都说养儿能防老，可是他每天和儿媳妇吵得天昏地暗，尊严扫地，他不想做个老年废物……


每一段苦于长久的人生，从生到死皆是苦痛。他们就像负能量永动机，挣扎在你不知道的角落，有些悲剧无法避免，有些难熬无法言说，在无数个夜晚，他们辗转反侧，沉默流泪，忍受着那些爱别离、怨憎会与求不得。


但是即使生老病死都会到来，生活也要继续，他们依然负重前行，如果你能感受到痛苦，那就没道理感受不到快乐。


**停下脚步看看，人生确实不易，可也不是都那么难。**
