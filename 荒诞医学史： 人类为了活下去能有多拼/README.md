## 简介
这是一部有趣又有料的世界医学发展史：吃黏土预防瘟疫、「水银浴」治疗梅毒、用黄鼠狼的睾丸避孕、搅拌脑浆治疗精神病……每一种误入歧途的治疗方式，都源于人类强烈的求生欲，我们愿意吃下砒霜，跳进滚沸的油锅，这一切，都是为了生存。


「独门暗器」连绵不断，「辣手飞镖」层出不穷，试问，谁才是暗黑医术界的「C 位神功」？


宫斗必备马钱子，竟能帮人夺冠马拉松。


打开脑颅，搅拌脑浆，治好了精神病，却也让人眼神呆滞永「享」安宁。


一个人的截肢手术三个人 game over，烧得通红的烙铁只想让人大喊救命。


……


你没有看错，上述都是盛极一时的终极治疗「神技」，部分方法在今天依旧盛行不衰，「拥趸」无数。为什么会这样？没有别的原因，一切都是为了生存。


本书将医学史上疾病治疗中的谬误悉数整理，内容广博，时间跨度从古到今，5 大门类、近 40 种荒诞医术，系统、全面、专业，个个有据可查，是目前市面上全面权威的医学史趣味读物，搭配近 300 幅图片，为读者呈现一部血泪杂糅的另类医学史。
