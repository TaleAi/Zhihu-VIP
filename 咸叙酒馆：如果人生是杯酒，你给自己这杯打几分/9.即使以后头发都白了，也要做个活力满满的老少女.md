## 9.即使以后头发都白了，也要做个活力满满的老少女
迪娅日记：



> 客人莉莉今天来店里的时候，给我展示了她新入手的一块陆地冲浪板——西班牙的 YOW，31 英寸 COXOS  橙轮，5000 元不到的价格。



> 我不懂滑板，但是觉得这块板有点好看，就是价格有点贵。莉莉笑着告诉我，玩陆地冲浪的，都会戏称有了这块板，就能「YOW（药）到病除」——因为这个品牌的中文发音接近「药」，板子又有点贵，很多喜欢的板友除非是入手了这块板，不然会「思念成疾」。



> 当然，这是戏谑我知道，不过不可否认的是，自从一点一滴重新捡起以往热爱的那些事物，莉莉真的整个人的精神状态都不一样了。她眼里闪烁着快乐的光，脸颊也变得红润起来，活力满满，完全看不出来她已经是两个孩子的母亲了。



> 莉莉这么跟我说：我现在是真明白了，不管什么岁数，也不管我们身上被贴了多少标签，妻子也好母亲也罢，最关键的还是得有自己的生活。做好自己，才能更好地扮演生命中不同的角色。



> 莉莉的话，是她这段日子的总结。我觉得挺有道理的，毕竟，只有努力让自己发光，生活才会充满阳光。


**1.照顾家庭，看着孩子一天天成长，我很高兴，但是不知道为什么，我总觉得有点迷茫**


莉莉有两个孩子，老大是女儿，叫诺诺，今年 6 岁了；老二是儿子，叫言言，今年 3 岁。从孩子的名字就能看出，莉莉和老公周涛的感情很稳定，很好。


可以说，莉莉有个很美满的家庭。


自从诺诺出生，莉莉就辞职在家，专心照顾孩子和家庭。很多人都觉得，家庭主妇没有职场压力，日子会轻松很多，其实并不是，尤其是当你有两个孩子需要照顾的时候。


莉莉每天都要早上六点半起床，先去菜场买菜，给老公、孩子做早饭；然后叫诺诺起床，给她穿好衣服，催促着她吃完早饭后，让老公上班途中顺带送她去上幼儿园；接着就要给言言穿衣，照顾他吃饭。往往这些事情弄完，都要早上八点多了，这才轮到莉莉自己吃早饭，还得收拾碗筷。


接下来，按部就班，先是洗衣服，收拾家里，再是按照前一天晚上想好的菜谱，准备言言的午饭食材，这些事情做完就已经快要到中午了。


言言还小，他一个人玩的时候，总是会喊「妈妈，妈妈，快来帮我」或者「快来看我」。总之，言言喊妈妈的声音无处不在，莉莉不管做什么事情，都要中断无数次。


十一点左右，莉莉开始准备言言的午饭，接下来又是照顾他吃饭，然后哄他午睡。或许是因为早上起得比较早，往往孩子还没睡着，莉莉自己先睡着了。


等孩子睡着以后，莉莉轻轻起床，有时要接着做上午没做完的家务，有时要忙着收拾衣服等。总之，家务活是一个很奇怪的存在，永远都做不完。


等言言醒了，天气好的时候，莉莉要带着他去小区里走走，或者去小公园逛逛，有时还要带他去超市买些东西。


不过说起来，每天的这一段时间，是莉莉一天里最期待的时光了。尤其是去公园，言言在草地上玩，她坐在长椅上，手里握着一杯路过便利店时买的咖啡，微风和煦，阳光温暖，这一刻莉莉才感觉到自己总算能稍微放松一下了——就像是中场休息。接下来，她还要继续做晚饭，收拾厨房，给两个孩子洗澡，然后陪他们玩，再哄他们各自睡觉。


往往等自己能睡觉的时候，已经是晚上 11 点多了。


有时节假日，婆婆一家还有小姑子一家也会来吃饭，莉莉就更忙碌了。去饭店吃饭当然轻松，可是婆婆肠胃不怎么好，饭店的菜她吃了容易拉肚子，于是就只能是莉莉做了。


遇到这样的日子，莉莉得连续忙上两三天，早早就开始做准备。


当然，也不是说周涛不体贴她，只是周涛的工作很忙。他在一家外企做到了管理层，工作要跟着国外总公司的时间走，经常晚上八九点还要在家开电话会议。


生活可真是无数个琐碎细节拼凑起来的啊！关于这一点，莉莉深有体会。有时她会无缘无故地想到万花筒，从万花筒里望出去，虽然眼前是一片五彩斑斓，但是不管如何旋转和拼凑，最后也只是一个循环的圆——和她的生活一样。


这样的生活，莉莉过了六年了。


自己到底是从什么时候开始，感觉这样的生活有点不对劲？是从镜子里发现自己左眼角上的鱼尾纹加深了，还是发现自己的脸上开始露出疲态？


莉莉一直都没有找到答案，直到有一天，她在买菜回家的途中，遇到了从前的同事——暂且叫她小 C 吧。莉莉觉得，或许自己终于明白答案是什么了。


**2.遇到曾经的同事，看着她精致的模样，莉莉忽然很感伤**


莉莉以前在一家很大的保险公司工作，她是做内勤的；当时的小 C 是个小小的业务员，因为销售业绩不是很好，一直被部门领导批评。


印象中，小 C 瘦瘦的，素面朝天，说话时眼睛里总有一种怯怯的神情。有时客户问到的是组训没培训的内容，小 C 就会露出慌乱尴尬的表情，拿着手机说稍等，我去确定一下。


可想而知，小 C 的销售业绩自然不会好到哪里去。不过，小 C 也有个很特别的地方，她是一个韧性很足的姑娘，即使业绩总是在部门里排倒数，也从没听到过她说半句泄气的话。


莉莉万万没想到，6 年后再一次见到小 C 的时候，她的改变居然这样大。她不但化了很精致的妆容，还穿着干练的黑西服，里面配了一件乳白色内搭，脖子系着一条丝巾——那是爱马仕的 Les Becanes 自行车，还是很难抢的蓝色，莉莉一眼就认出来了。


这一款丝巾，茜茜公主的扮演者罗密·施耐德曾经戴过，这是复刻版。莉莉一直很喜欢这个图案。事实上，莉莉也在 APP、官网上抢购过，不过没抢到。后来，莉莉想想感觉自己似乎也没机会穿搭，于是就放弃了。毕竟她做了家庭主妇以后，感觉穿什么衣服都已经不重要了，衣服上还特别容易有孩子们打翻的汤啊，菜啊，或者吃东西掉在她身上留下的菜渍、油渍等。


小 C 一反过去怯怯的样子，很主动地喊住了莉莉，还和她聊了一会儿，言辞的表达上和莉莉印象中的小 C 完全不同了。


莉莉得知，小 C 现在已经是部门领导了，她负责的片区业绩很不错。后来，小 C 因为赶着去公司，匆匆结束了聊天，临走前还说如果有业务需求，可以找她。


莉莉自然明白，小 C 的最终目的是最后一句话。不过讲真，看着小 C 活力满满的样子、匆匆的步伐，她忽然有点失落，眼神落在自己手里拎着的一把鸡毛菜和鸡蛋、莴苣、手抓饼上。


时间不早了，容不得她再沉浸在自己的情绪里，莉莉也匆匆走向家里，她的目标是要尽快准备好早饭。


后来，和往常一样，在送走了老公和诺诺，带着言言吃完早饭，安顿他看电视节目后，莉莉开始收拾屋子。也不知道是不是潜意识里想起了从前，她在整理东西的时候，特意打开了收纳间的柜子。


一时间，莉莉有点窒息。


金色的拉丁练功服、黑色的拉丁鞋、陆地冲浪板……她打开的根本不是柜子，而是曾经的自己。


莉莉鼻子有点酸，她拿起拉丁练功服轻轻抚摸，脑海中回想起当年的自己为了考拉丁舞的一级教师证，每天下班后都泡在舞蹈工作室练习舞步，恰恰、伦巴、桑巴、牛仔、斗牛每一种都少不了。


那时，她身体每一块肌肉都是紧绷的，整个人不论什么时候，都站得特别直，看起来特别有气质。


说起来，她费了很大功夫考上的一级教师证，其实也从来没用过。但莉莉也不在意，她考证不是为了带学生赚外快，而是想用这种方式让自己好好练习。


但是现在呢，莉莉叹了一口气，捏了一把自己腰上的肉——早就不复当年的紧致了。莉莉失意地将练功服放回柜子里，又拿出了划痕累累的陆地冲浪板。


这块板子还是那会儿莉莉花了一个月薪水买的，当时玩陆地冲浪板的人还没现在那么多，它还是一个很小众的体育运动。莉莉当年也是对照着视频，和玩「陆冲」的朋友们摸索着练习。pumping、反手 carving、正手 carving、横刹 slide90 度、反手 slide180 度……这些动作啊，她练得好辛苦，甚至练反手 slide180 度接 pivot180 度的时候，还因为发力不对，没有控制好力度，整个人狠狠从板子上摔了下去——脚踝骨折。没办法，有人说，滑板的尽头是骨折，这话倒是一点毛病都没有。


那时，莉莉已经和周涛在一起了。他见莉莉因为玩陆冲搞得进医院了，很是心疼，让她可别再玩这个了。可是莉莉哪里能忍得住呢，虽然骨折很疼，她也因此一瘸一拐了一个多月，但是在身体恢复后，她又重新站在了板子上。


这些可都是美好的回忆啊。莉莉回忆这些的时候，嘴角都是上扬的。


莉莉在把板子放进柜子的时候，又看到收纳柜的角落里，还放着她之前种花种草用的小铲子、给欧月打农药的深蓝色小喷壶，还有黑色的遮阳布——专门为一些不耐晒的花草准备的。夏天阳光太晒，很多植物例如铁线莲，夏天会很难度过，绿叶和花会被晒焦，这时就需要遮阳布，丑是丑了点，但起码植物不会被晒焦甚至晒死。


还记得那时，莉莉种的欧月爬藤爬得可好了，整个院子的小半面篱笆上都是大朵大朵的花。


黄金庆典、夏洛特女郎、玛格丽特王妃、威廉莫里斯、亚伯拉罕达比，每一款都是莉莉的心头大爱。


种花其实也是很费功夫的，每天早晨起来后，莉莉都要花一个小时时间查看自己的花花草草。哪些长了红蜘蛛要打药了，哪些要有蚜虫了得先喷药水，哪些需要打顶才能开更多的花，这些都要花时间去做。


莉莉每次在小花园里劳作好，站起来巡视的时候，都有一种成就感。她喜欢自己的院子一年四季都鲜花盛开，就像她的生活——永远生机勃勃，永远充满了活力。


可是，到底是从什么时候开始，自己的生活变得如此狭隘，只剩下厨房、孩子和丈夫了呢？莉莉闭上眼睛，怎么都想不起来这一切是怎么发生的——好像是自然而然发生的吧。


自从有了孩子，莉莉自然而然地放弃了跳舞，放弃了滑板，也放弃了高跟鞋，放弃了侍弄花草的乐趣。


如果说，最初是因为怀孕，不适合再玩这些运动，也不适合再穿高跟鞋、给植物喷农药，那么在孩子生下来以后，她的时间就不是自己的了，此时的莉莉又多了一层标签——母亲。


妻子和母亲的身份标签，让莉莉放弃了自己的喜好，将全部的时间都奉献给了家庭。


事实上，这也由不得她，孩子还小，需要母亲的陪伴；丈夫工作太忙，需要妻子的扶持；而莉莉就只能做家庭里那个被需要，也是被牺牲的人吧。


莉莉失魂落魄地看着柜子里的一切，阳光从她身后射进来，给它们加上了一层极不真切的滤镜效果。


那一刻，莉莉忽然感觉自己像是在做梦——下一秒，她就听到客厅里传来「扑通」一声响，接下来是言言的大哭。


莉莉就像是梦醒了，立刻急急忙忙跑去了客厅。


还好不是什么大事，言言这个调皮鬼，一边看节目一边在沙发上爬来爬去，结果从沙发上掉下来，摔疼了屁股在哭。


莉莉连声叹息，将言言从地板上抱起来，搂在怀里不住安慰。言言被莉莉哄了一阵后，哭声渐渐停止，他凑到莉莉脸上亲了她一口，说妈妈最好了。


瞧着言言眼角还带着泪水的模样，耳边听着他奶声奶气的话，莉莉整个人都融化了，或许这就是她愿意放弃自己曾经生活的原因。


一时间，莉莉心里五味杂陈。


**3.成为最好的自己，一切都还来得及**


后来这一整天，莉莉都感觉自己过得有点浑浑噩噩的。晚上十点半，两个孩子都睡了以后，莉莉换上了出门的衣服。


周涛惊讶地看着她，但因为在开电话会议不方便说话，于是，他发了信息问：「怎么了，你要去哪里？」


莉莉回复说：「感觉有点睡不着，想去咸叙酒馆喝杯酒。」


周涛虽然很想知道莉莉是不是有心事，但他实在分身乏术，于是只得给莉莉回短信说：「早点回来，有什么事情也是可以给我说的。」


莉莉回复了个笑脸，出门了。


深夜的风吹在脸上，有一丝让人清醒的凉爽，莉莉也不走远，直接走到家附近的咸叙酒馆，推门进去。


咸叙酒馆是莉莉和周涛都很喜欢的一家酒馆。平时，如果两个孩子偶尔在爷爷奶奶家过夜，他俩总是会到这里喝上一杯。


这天，咸叙酒馆的人不多，莉莉推门进去的时候，迪娅正低着头擦杯子。这个动作，可算是迪娅的经典动作了，莉莉几乎每次去都能见到迪娅在拿着口布擦杯子。


没来由地，莉莉忽然想到：迪娅擦杯子，这是她喜欢做的事情吗？——可是，不管喜欢不喜欢，都得做啊。说到底，人生哪有那么多喜欢的事情，又恰好能有机会、有能力去做！


这么一想，莉莉似乎有点轻松了，她坐在吧台前，要了一杯鸡尾酒。


「怎么现在来啊，你老公呢？不和你一起来啊。」迪娅将酒放在桌子上，推给莉莉。


「他还在开会呢，现在可能结束了吧。」莉莉接过酒，喝了一小口说，「他来不了，两个孩子还在家呢。」


「呀，那你倒是难得啊。」迪娅笑了，「怎么，是出来放风吗？」


「可不啊。」莉莉也笑了，可是笑着笑着，神情就落寞了，瞧着酒杯出神。


迪娅看得出莉莉有心事，不过她没问，她知道很多时候客人想说自然会说的。果然，在喝了半杯酒后，莉莉开始给她说那些事，包括她的拉丁练功服、她的陆地冲浪板、她的小花园，还有她的高跟鞋。


「迪娅，我下午看到这些的时候，真的感觉自己当年的种种快乐，就像是发生在上个世纪那么遥远。」莉莉闷闷地说。


「怎么会呢？」迪娅被她的话逗乐了，「只要你想，随时可以重新开始啊。」


「可是我没有那么多时间啊。你看，两个孩子耗去了几乎我全部的精力，还有家庭，还有那么多的琐事。」


「所有事情，其实都有解决的办法，就看你从哪个方面去看了。」迪娅温和地说，「比如说孩子，周末的时候你可以拜托你父母或者他父母照看一天，而这一天，你可以尽情做自己啊。别去想周涛吃什么，他是个成年人了，不需要你如此无微不至的照顾，甚至你可以让他照顾你。」


「照顾我？」莉莉吃惊地抬起头，「我才不指望，他可能只会泡面。」


迪娅笑了，说：「还会点外卖哦。真的，你试试看，寻求身边人的帮助来给自己点空间，不管你想成为从前的自己，还是更好的自己，其实都来得及。」


「是吗？」莉莉小口小口地喝着鸡尾酒，若有所思。到最后，莉莉想着的是：那么，试试看？


很明显，莉莉的尝试是有效果的。


迪娅看得到，后来莉莉每次来酒馆，脸上的快乐是发自内心的。当然，孩子和家庭也给她快乐，但是多一份快乐不好吗？


尤其是，莉莉重新跳起了拉丁舞，甚至还接受一家舞蹈培训机构的邀请，成了拉丁舞老师。


不过因为时间有限，所以她一天只上两个小时的课。但即使这样，莉莉也很满足了。


「迪娅你知道吗，我昨天晚上九点回家的时候，还很着急，我想诺诺和言言都在家，周涛在开会，家里一定闹得不可开交吧。结果没想到，回到家后，我看到诺诺带着言言在画画，两个人还在房间里看动画片，而周涛在客厅安安静静开会，一点都不像我想象的那样鸡飞狗跳。」


莉莉笑着说：「原来啊，很多事情都能有另外一个结局，就看我们如何冲破自己的狭隘，努力去寻找另一种可能。」


迪娅点头，微笑着说：「是啊，其实生活从来不曾为难过你，让你觉得为难的事，多数都是自己造成的吧！」


理性饮酒小贴士：

*我们从莉莉的人生故事里，看得到她和周涛的生活其实和我们大多数人一样，平稳而踏实，没有生活习惯上的不良嗜好。尤其是莉莉，她还很喜欢运动，这其实是一种很平衡的生活状态。这样的生活状态，无疑是有利于身心健康的。*


*那么，如何维持平稳的生活方式呢？很简单，不吸烟，健康饮食，大量运动，保持健康的体重以及不过量饮酒；当然还有其他因素，例如充足的睡眠，避免使用违禁药物和滥用某些处方药。*


*对大多数人来说，坚持上述几个关键的因素，不但能收获平衡的生活状态，还能降低患上慢性（或非传染性）疾病的概率。*


*有一点我们要特别注意，如果你有饮酒的习惯，那么一定要避免大量饮酒。因为不论是长期大量饮酒还是酗酒，都会增加患病风险，这显然有悖于平衡生活状态的形成和维持。*
