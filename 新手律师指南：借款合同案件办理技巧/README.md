## 简介
本课程由中闻律师所资深律师李斌主讲，李斌律师根据实际案例和自己过往的实务经验，总结出了一系列实用的案件办理技巧，帮助新手律师快速系统的提升借款合同案件办理能力。


对律师来说，借款合同案件是一项必修的内功。


据统计，中国裁判文书网访问量排名前十的案由分别是民间借贷纠纷、买卖合同纠纷、离婚纠纷、机动车交通事故责任纠纷、金融借款合同纠纷、劳动争议、物业服务合同纠纷、盗窃和借款合同纠纷。


借款两个字的频繁出现，也证明着借款合同案件在律师业务中的重要性。


借款合同案件在实务中难点多、纠纷多，非常考验律师的综合素质。面对这种高频案件，律师如果不尽快提升自我，无疑会错过很多案源、甚至很多潜在机会。


为了加强律师的专业交流，促进和提高借款合同案件中，律师的实务水平——


本课程邀请中闻律师所资深律师李斌开设干货超多的借款合同案件律师实务课程，课程中，主讲人将结合案例和自己过往的实务经验，分享他的经验总结。帮助新手律师快速掌握借款合同案件技巧。
