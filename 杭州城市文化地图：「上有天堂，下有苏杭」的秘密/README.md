## 简介
江南忆，最忆是杭州。这是一场高曝光的江南文化博览会，华少用他独具辨识度的声线，带来一场环节紧凑，景点逻辑清晰的杭州 Talk Show，让你了解让杭州成为「天堂」的秘密。


你知道灵隐寺的冷泉亭与杭州两大「老市长」的故事嘛？


你知道「断桥残雪」因故得名吗？


你知道西泠桥畔埋葬的「中国版茶花女」吗？


从世上最美的路出发，途径 900 年沧桑的南宋御街、千年古刹灵隐寺，在胡雪岩的故居里找寻红顶商人的身影，在虎跑梦泉与西泠印社的角落中感知弘一大师的风采，在云松书舍中窥见金庸先生创造的江湖一角。让主持人华少带你认识一个可爱、充满灵气的诗情杭州。
