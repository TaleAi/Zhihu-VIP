## 38.KOL 养成第一步：打造个人 IP，塑造一个可描述的形象
2019 年，当自媒体和网红越来越受资本的青睐时，承载这些网红和自媒体的平台也红的发紫。如今，小红书这家已经拥有 2.5 亿注册用户、估值 30 亿美元的独角兽企业，自然也成了各大品牌商家、网红、KOL 的必争之地。


**如何成为一名合格的 KOL**？
------------------


要成为一名合格的、有话语权的 KOL。首先要打造一个个人 IP，塑造一个可描述的形象。如果你今天发美食，明天发旅行，后天又变成了微商，大后天又发布了一些穿搭秘籍，后来又说起了实事新闻……经常更换发布内容的领域，是无法被系统明确定位的，你运营的再好，被系统降低评分是一定的。那你也就离一名有话语权的 KOL 越来越远，再发布什么内容也不容易被曝光。


当然有些领域本身关联强度就高那么也是可以的。例如美食与旅行，美妆与穿搭等等，聪明的薯宝宝们稍微动脑子想一想，就能判定是不是合理的。


**1.基于小红书定位的个人 IP 定位**


基于小红书平台可以有以下两个角度的定位：


一是从行业领袖、专家、个人 IP、公司 IP 角度定位，分享一些专业度高的内容。


二是从第三方的角度，也就是普通用户的角度去定位，侧重写第三方视角下的真实感受和体验。


**2.打造个人 IP：呈现一个具体形象，彰显你在某个领域的专业度**


想要内容被更多人看到，同时又要关注你，你必须要有一个自己的 IP 形象——你必须呈现出来一个可描述的形象，让不认识你的人在看到你的 IP 后立刻就能想到你。只是当你打造 IP 形象并经营了一段时间之后，后期想要再去改变恐怕来不及，所以你只能提前做好准备，向着 KOL 的定位方向出发。


**3.两种方法筛选出适合发笔记的内容**


有三个方法帮助你来筛选发什么内容：


* 追踪法

追踪法就是你打开小红书的页面，默认搜索以及搜索页面打开的热门榜单，直接看下图：


![img](https://pic1.zhimg.com/v2-3e47f299659eba6c2ca9cd94966698e1.webp)

由于这些热搜是系统自动+人工干预生成的。所以我们看到的这些话题内容其实正是官方默认推荐的内容。跟着话题走准没错，只是强扭的瓜不甜，你自身的定位一定要能匹配上话题才行，不然就别发。


**2.埋伏法**


什么是埋伏法？


顾名思义，提前埋伏布局。例如，当一部国外大片上映之前，可以提前整理好相关的笔记。待到真相应的时候，当平台热搜出现电影名字时，你就可以第一时间发出笔记。这样你的笔记就很容易获得了大量的关注和点赞。


根据我这个思路，薯宝宝们可以提前「埋伏」到一些关键词，这就好比考试提前押题一样。虽然有难度却也是一种可行的方法。一旦命中，你就可以随时登上热搜。
