## 简介
名著里有哪个女性角色让你难忘？


你也许看过很多名著，但你可能没有想过，故事里她们面临的很多问题，其实跟现代女性是相似的：


**《金瓶梅》**


放纵声色的潘金莲、白富美李瓶儿、逆袭上位的庞春梅，从西门庆的后院出发，她们走向了截然不同的人生道路。


但她们的人生选择，却反映出了自身人格形成背后的心理需求与心理倾向。


**《红楼梦》**


凤姐和晴雯，都是一开始就有一手好牌的人。


但在对「领导」的选择，以及对「同僚」的态度上，显示出了她们的实际情商，也一定程度上预示了她们的结局。


**《红玫瑰与白玫瑰》**


红玫瑰王娇蕊风情万种，白玫瑰孟烟鹂拘谨守旧。男人喜欢红玫瑰，可还是选择跟白玫瑰结婚。


表面上看，这只是一个男人的选择。但实际上，爱商高低的区分，才真正导致了爱情的开始与消逝。


……


**从她们身上，也许我们可以看见自己。**
