## 简介
假如你是历史先贤，会发生什么不一样的事？


诸葛亮、刘备、武则天......


这些历史名人相比你都耳熟能详。


他们在历史上留下了浓墨重彩的一笔。


有丰功伟绩，也有饱受诟病的地方。


于是，很多人会说：当年如果 XXX 怎么干就好了。


现在，我们把这个机会给你，如果你穿越到历史先贤身上，看看你会怎么干。
