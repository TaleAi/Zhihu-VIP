# 恰恰好在一起：你喜欢的人，正在暗恋你

### 作者：佩奇酱

### 分类：现代言情

### 状态：已完结，共 15 节

### 存档日期：2022-05-25

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1422227407662751744


## 简介
偷眼看你时目光对撞，擦身而过时听到你的呼吸，我的主动都得到了回应，最好的爱情，就是我们恰恰好相爱，恰恰好在一起。


《我只是为了热搜》


为了制造绯闻，


她深夜敲开了圈内大佬的门，


本来只想拍个照，


没想到对方让她进了门……


  




《不露声色》


结婚时，她以为这场婚姻是交易，


直到打扫书房，她无意看到丈夫写的情书。


这些情书和自己年少收到的一摸一样，


只是字迹不同……


  




《捂不热》


对大少爷一见钟情之后，


她追了整整一年，


终于获得了大少爷的直白拒绝，


没想到，她刚一放弃，


对方竟反过来追她了。


  




《小哭包》


于茉社死了，


开会时和姐妹吐糟老板，


聊天记录被全部投影到大屏幕上。


只是老板凶完竟然送巧克力给她，


还天天换着花样送。


救命，老板疑似暗恋我该怎么办！


只好以其人之道还其人之身。


  




《万有引力》


梦见月老送我红线，


说带上它你爱的人也会爱上你。


醒来后桌子上居然真有条红绳，


我立刻带上它去追男神。


只是，梦里的月老，真能保佑得了现实中的爱情吗？
