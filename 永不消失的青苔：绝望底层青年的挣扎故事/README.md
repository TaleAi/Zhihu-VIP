## 简介
碌碌无为的青年、逃离不掉的原生家庭、绝望堕落在毒品中……你不曾体验也无法想象的人生，就真实的发生在这些底层人物的身上。这是边缘、无奈的人生故事，只因他们不甘，但也无能为力。


* 父亲离世，辍学去打工，遇到异装的男人、出轨的客人、自杀的同事，他们有怎样的无奈与挣扎？
* 小镇恋人，来不及憧憬美好的人生，失意与毒品如何交织在爱情与生活中？
* 第一任丈夫车祸、第二任丈夫被杀，面对「克夫」的流言，她如何带着三个孩子寻找下一个男人？
* ....

这些人与我们生在同样的年代，却有着不同的命运，演绎出低层边缘的现实群像。


生活的苦难，防不住。死，有千条路。


熬，是他们生存要诀里的第一项。
