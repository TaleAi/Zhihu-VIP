## 简介
在黄河源头，医生和活佛如何与包虫病斗智斗勇。


闪米特是一位独木舟探险家。这一次，他的计划是从黄河源头开始，用独木舟漂流整个黄河干流，直至黄河出海口。在青海黄河源草原上，一种非常危险的寄生虫疾病——包虫病正在肆虐。极高的患病率和极高的致死率让援助藏区的医生们心力憔悴。然而，落后的生活环境、延续已久的生活习惯让这种疾病在草原上迅速蔓延，现代医学虽然可以医治疾病，但是需要获得牧民对于科学的信任。


死亡还在继续，感染越发扩大，牧民们对于现代医学的不信任，以及对固有生活方式的坚持，让悲剧越来越多地发生。在经受了长时间的挫折之后，聪明的医生们不再用教条的科学去改变现实，而是用一种更巧妙的方式去引导牧民。现代科学与牧民的信仰并不一定就是矛盾的两端，为什么不能让矛盾统一呢？他们与宗教人士一起，让牧民们一边就医，一边祈祷诵经，并对牧民们传达一种双保险的精神。邀请活佛在宣传佛法的同时，要求信众们来医院检查。甚至，活佛还会亲自带信众来医院医治。科学与宗教相互协助，挽救牧民的生命。


那些医生们的智慧不仅仅在医术，更是一种人生智慧。我们生活在一个复杂的现实世界，在这个世界中，推倒一面墙的成本太高，难度太大，而且可能会带来意料之外的灾难，而结果往往事倍功半。那些在藏区同包虫病抗争的医生们，既有仁心仁术，又会灵活务实，这种不去争论对错，而是关注促成良好的结果的成熟，不仅仅感动了探险家闪米特，也感动了编者。


于是，我们把探险家这篇文笔平实、认真、悲悯的记录整理出版，以飨读者。同时，也为了纪念那些生命。
