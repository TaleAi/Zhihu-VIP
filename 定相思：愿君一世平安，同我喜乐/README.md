## 简介
这是一个或甜蜜温馨，或啼笑皆非，或温暖安宁的古言故事专栏。


长相思，不负相思，定相思。


本专栏中，收录各种金风玉露一相逢，不胜相思的古风言情故事。


《宿敌》：我嫁给了我的死对头，因为我哥拐跑了他的未婚妻。为了报复，他就去向皇上请旨，要我嫁给他。出嫁那天，我哥终于带着我未来的嫂子回来了。我哥号啕大哭，说我这个祸害终于有人收了，还是宿狄收，真是一石二鸟。真的，要不是他身边站着温柔似水的未来嫂子，我就骂街了。


怎么办，和死对头成亲的日子，好难熬！


……


一世平安，与你享受烟火气中的喜乐，


便已胜却人间无数。


  



《风时絮》


我曾经是一名暗卫，专职杀人。


我杀过很多人，但只救过一个姑娘。


姑娘用刀比我好看，挥舞起来如追星赶月，长的也很好看。


现在，我专职杀猪。


十八文一斤，童叟无欺。


  



《海棠疏雨》


我的仇人回京逼宫造反的时候，我刚收拾好东西准备跑路。


别问，问就是相爱相杀的狗血故事。


万万没想到对方居然在门口堵我。


一想到我杀他爹流放他全家的各种仇怨，我慌得一匹。


想死又怕死怎么办？在线等，挺急的。
