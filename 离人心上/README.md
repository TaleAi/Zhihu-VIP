## 简介
【热播剧《离人心上》同名小说，同步更新中】，幼年事故致失眠的九公主初月，在皇宫中昼伏夜出孤独长大，阴错阳差之下，她嫁给了以凶厉出名的将军薛曜，却不曾想这个满身杀伐之气的男人，竟慢慢治愈她的失眠。从此她的漫漫长夜不再孤独，本是一场阴谋的婚姻，却让两人在不觉中赋予了真情……


Part 01 


南桑国的后宫一直有个传闻，公主徐初月身患异疾，白天缠绵病榻，夜晚却四处游走，流言愈演愈烈，更有甚者还称在宫中看到了动物精怪，宫中皆称因为公主不祥，招致后宫不宁。


初月就这样一晃长到了十八岁，伴随她成长的，除了这些传闻还有她心爱的各类风月游历话本，其中以一本《关山纪事》的游历小说尤为喜爱。这天，被西风侯嫌弃退婚的初月在夜晚迷迷糊糊的睡去，梦到自己被刺客刺死，苏醒后的初月不由分说的拉起贴身侍女桃夭仓皇逃命。


原来，南桑国有颗占卜未来的「生辰石」，初月小时候不慎让生辰石进入体内，从此便获得了梦知未来的能力，但只要她改变未来，就会遭到生辰石反噬，变成十二生肖中的其中一种。国师东识告诉她，她只有 12 次预知未来的机会，机会用完，她的生命也会走到尽头。为了活下去，初月只能昼伏夜出的生活着。 


Part 02 


这次梦到自己被杀，是初月第九次做梦！


在出逃的过程中，她与偷潜入皇宫调查哥哥死因的大将军薛曜相遇，薛曜在刺客手中救下初月，初月受伤昏迷。一方面为了调查初月和刺客的关系，一方面为了确认初月是否识破了自己的身份，薛曜便向皇上求亲，以婚事将初月困于王府中。就这样还在昏迷中的初月被薛曜抱着完成了婚礼，成了王妃。 


嫁入府中的初月苏醒知道木已成舟，便和薛曜约法三章，二人两不相犯谁知恰逢赶上初月遭到生辰石反噬，洞房当晚，新娘子变成了老鼠，把薛府上下捣了个鸡犬不宁，形迹可疑的初月让薛曜警钟大作，薛曜步步紧逼的试探初月，最 后都被初月无形化解，薛曜自以为统帅三军战功赫赫，没想到却在初月这儿屡屡碰壁。  


Part 03 


外有虎视眈眈的强盗西风侯，内有温柔可人初月的弟弟星辰，这二人的对自己妻子的觊觎，让薛曜对初月的感情慢慢变得不可言说，而初月也在薛曜一次次的保护中，感受到了别样的温暖，终于有一天，两人决定，要像初月最爱的话本《关山纪事》一样，走遍书里描写的名流大川。


然而事与愿违，在阴谋的巨轮下，薛曜哥哥的死因真相也被一步步揭开，原来薛曜的哥哥薛暮早就爱上了这个日夜颠倒的公主，薛暮为了保护初月默默的付出了生命。


薛暮的死，是阴谋密网里的一个导火索!不久，初月便梦到薛曜被一箭穿心，为了保护薛曜，初月最后一次利用生辰石预言改变了现实，从此长睡不醒。阴谋破，恶人除。薛曜带着沉睡的初月踏遍名川大江，在最后一站，薛曜烧掉了初月最爱的《关山纪事》，在见证了两人爱情的悬崖边正准备纵身一跃时，初月的双手攀绕上薛曜的脖子，一双盈盈的眼眸盯着薛曜.....
