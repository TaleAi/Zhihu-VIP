## 简介
人类文明的进化史，是一部战争史


四渡赤水，国共战争中最艰苦，也最惊险的战役之一，3 万对阵 40 万，面对层层围追堵截，红军却能屡屡料敌于先，从重重大军的包围缝隙中穿插过去……


淞沪会战，中日战争史上规模最大、战斗最惨烈的一场战役，国军仅将军就阵亡了十四人，中将军长吴克仁与其率领的 67 军，从上到下全部阵亡，国军最精锐的德械师、最优秀的黄埔系将领被一批批送往前线，几乎全部拼光，整个战役的国军总伤亡更是高达 29 万以上……


上甘岭战役，抗美援朝的大背景下，又一场信念战胜炮弹的典型。志愿军在兵员数量、武器装备、后勤补给等都处于绝对劣势，仅仅占据着一些地理优势的情况下硬是坚守了超过 40 天，最终击溃了被认为不可能被击溃的敌人，这不只是一场战术上的胜利，更是在精神上彻底挫败了一向骄横狂妄的敌人……


宁死不退，血战到底，无数中华儿女抛头颅洒热血，只为脚下这片深爱的土地。革命先烈们，今天的中华盛世……如您所愿！
