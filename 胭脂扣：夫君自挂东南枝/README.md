## 简介
他们是天造地设的冤家，是不被祝福的绝配，红尘中滚滚劫灰落尽，唯他们并肩立于山巅。这里有女皇x宦官，富人x寒门，还有九重天上的魔x神。九世追妻路漫漫，疯批、病娇、清冷禁欲、温润如玉，总有一个人设会戳中笑点与泪点。



**《明月照百里》**从十六岁开始，李明月嫁了三次人，守了三次寡，次次不得善终。十九岁这年，为报兄长之仇，李明月爬上司礼监大太监殷百里的床，与他狼狈为奸，将整个朝堂搅得天翻地覆。


**《氓》**当你的女朋友或者心上人被强暴了，你会怎么做？是一别两宽，还是共赴白首。


**《今日师尊入魔否》**为了逼师尊——九重天的唯一上神入魔，我将他虐了又虐，终于成功使他堕魔。


**《但为君故》**唐缘笙是临安城富商唐家的唯一千金，自小貌丑无颜，蠢笨如猪，但善于察觉人的情绪变化。其父唐老爷为招赘，救助了一寒门秀才林子安，以期他将来高中，娶唐缘笙为妻。「先生眼盲心不盲，还是一朵可远观，也可被我亵玩的高岭之花。」
