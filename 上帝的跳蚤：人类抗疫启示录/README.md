## 简介
「人类在过去的 200 年间靠着对科学的信任和追求，一个又一个地战胜了困扰我们几千年的疾病，这是人类文明的光明大道。」


人类的历史是一部与疾病斗争的历史。


本专栏以「微小的生命如何在巨人的世界中赢得竞争」为切入点，不仅提供了一个研究历史的新视角，更向大众普及了现代医学中非常重要的预防医学、微生物学、疫苗学的基础知识。


鼠疫、天花、流感、艾滋病......这些横扫过人类的瘟疫，在带走了上亿人生命的同时，也改变着我们文明的走向。


这些惊心动魄的描述和感人肺腑的故事使人欲罢不能，既有造化弄人的悲哀，也有对人类开创科学新纪元的欢欣鼓舞。
