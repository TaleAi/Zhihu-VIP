## 简介
想提高成绩却无从下手，没有学习方法？花了很长时间复习，成绩却没有明显提升？学习遇到了瓶颈期，再也无法有突破？无法专注、精力不济，学习效率低？刘媛媛教你一套长时间高效学习的策略和方法。


***以下这些场景你是否经历过？***


**专注力低下：**一上课就想睡觉，一下课就来精神；


**偏科严重：**考完一科「心飞扬」，考完另外一科「透心凉」；


**不会记笔记：**一字不拉地记下老师的话，我成了全班最勤奋的「 仔 」；


**不会用错题本：**总结错题=把错题都摘抄在一个本子上……


**对于这些通往高分路上的拦路虎，你是不是束手无策？**


刘媛媛将分享自己的学习方法，手把手教你独门记忆法、自学心法、保持专注、攻克不擅长的科目、高效行动指南等学习的技术。


这套方法，不仅适用于考试，更适用于自学其他领域的知识，让你快速实现跨越式提升。
