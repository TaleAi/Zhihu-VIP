## 简介
我深爱群星，因此无惧黑夜。
——莎拉·威廉斯


这是献给每一颗好奇心的一场星际之旅、时空之旅。


从我们仰头就可以看到的太阳、星星与月亮，到更远的星系乃至宇宙边缘，这段旅程将覆盖 930 亿光年的空间，横跨近 140 亿年的时间。


从熟悉的太阳、月亮讲起，直到恒星、星系，再到黑洞 、暗物质、大爆炸理论……作者精心编排了一条富有趣味的路线，带我们领略那些我们好奇的关于宇宙的一切，同时体验一段人类探索星空和宇宙奥秘的「浓缩版」认知历程。


让我们一同踏上探索宇宙的旅途吧，希望你也爱上这片繁星点点的夜空。
