## 简介
本专栏详细介绍了潘建伟院士领导的中国科学技术大学量子信息科研团队在近几年取得的科研进展，将十余篇刊发在《自然》、《科学》等国际知名学术期刊上的科研成果通过生动有趣的漫画形式展现出来，内容包括量子通信、量子密码、量子计算、量子模拟、量子纠缠等。


你是否想看看量子世界的故事呢？


「光子」患有多动症，整日上蹿下跳；


墨子号卫星成了「段子手」；


量子论大师玻尔偶尔「小人得志」 ；


一代宗师爱因斯坦成天「又哭又闹」 ；


原子在「恋爱」，分子闹「分手」；


 纠缠中的两个光子摆着同一个 pose，方向死活对不上。


本专栏通过生动有趣的漫画形式展现量子世界，适合对量子领域感兴趣的读者阅读。搞笑的漫画段子科普漫画大咖 sheldon「精心烹煮」，为读者奉上了一碗美味的科普「螺蛳粉」。
