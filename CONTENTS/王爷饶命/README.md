## 简介
当女扮男装的王爷恋爱后，她该怎么让对方知道她不是要跟他「断袖分桃」？


我，许氏君灿，一出生爹就死了。


只因为我是个女儿，所以我爹活生生地气死了。


世人皆知，晋王府男丁稀少，向来是一脉单传，而到了我这一代，竟直接成了绝唱！


我娘为了维持自己的荣华富贵，竟胆大包天对外宣称我是个儿子！


于是，我只有抛弃红妆爱武装。


夏天光了膀子练功，冬天下着大雪耍棍。


娇娇女变成了糙汉子，我誓要成为京城中最纨的绔！


喝花酒、调戏民女、抢人家水果、夺他人烧饼，终于，我成为了人见人怕的小魔头！


皇上堂兄疼我宠我护我，却也气得想打我，一朝我被他踢去了京兆尹衙门，被王荀直接领导，捱的第一顿揍就是来自于他！


好啊，你这鳏夫，我记你仇了！


这头刚给他捣乱，那头我就深陷在小叶子师爷的美貌当中。


京兆尹里故事多，一条画了翅膀的飞蛇图腾，引出了一桩桩命案，王荀经验老道，却也需要我这样混不吝坐镇。


日益相处下来，我们三人也积累下来了深厚的友谊。


可我不能一直隐瞒自己的身份。


面对着什么都不知晓一心想要给我指亲的皇兄，我陷入了深思。


走，还是留，这是个难题。


一方面惦记着王荀为我舍身取义，另一方面又舍不得小叶子的绝色容颜。


哎呀，人生在世，真是艰难险阻，我到底该何去何从？
