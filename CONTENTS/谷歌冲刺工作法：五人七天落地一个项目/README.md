# 谷歌冲刺工作法：五人七天落地一个项目

## 简介
分步拆解五天七人工作步骤，消除效率和成本困扰。


市场变化加速，时机稍纵即逝。面对时间紧、预算低、人力少的情况，如何快速落地一个项目？


如果你正为效率和成本困扰，这是上手设计冲刺的好时机。


本专栏特邀 ARK 创新咨询首席创新官滕磊老师，带来一套来自谷歌项目冲刺的标准流程：五天、七人，即可完成一次产品迭代，实现一个项目落地。


小团队作业、短时高效，精准解决难题、验证创意。


周一到周五，从描述问题到测试目标用户，分步拆解每个工作步骤，零基础上手。


同时以谷歌冲刺方法为基础，糅合导师实战经验进行本地化阐述，揭示常见错误，让团队更好掌握实操流程和思维逻辑，指导项目更好、更快落地。
