## 简介
家是我们最温馨的幸福港湾。然而在家庭生活中，我们也会经历很多事，遇到一些困难，发生一些误会矛盾，让亲情备受挑战。风波面前，曾经最亲密无间的家人，该如何化解矛盾，再次抵达幸福的彼岸？


为方便乡下的侄子上学，强势的婆婆要把他户口迁入她和老公名下，做城里人。此时她又意外发现，小侄子跟老公的关系，绝不止叔侄那么简单。而迁户口背后，隐藏着婆婆和老公更大的阴谋。最温柔体贴的枕边人，变成一心算计的陌路人，她该怎么办？


她最大的心愿，就是父亲能去帮她开一次家长会。可父亲从来是沉默寡言的，冷漠的，日日早出晚归，别说开家长会，多年来话都没跟她说几句。直到一次意外事故，她才重新发现，那些隐藏在无声里，她不曾看见的默默奉献，是父亲对她最深沉的爱。


······


6 段发生在家人间的爱恨交织的情感故事，让我们看到在复杂矛盾中，亲密无间的家人勇于面对问题，努力解决问题的决心，探索家的意义，发现幸福的真谛。
