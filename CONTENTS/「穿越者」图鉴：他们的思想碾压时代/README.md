## 简介
当代青年人，往往戏谑地称超越了时代的古人为「穿越者」。


中国历史中，总有那么一些人，他们的所作所为、思考处理问题的角度和方式、对事物发展的预见，都明显异于同时代的人。


时代楷模王莽
------


土地公有？废除奴隶制？货币改革？


打眼一看，都是现代制度的烙印啊！


商战鼻祖管仲
------


商业立国？市场调控？用投资拉动经济发展，降低失业率？


国企和商战，居然能追溯到两千多年前……


真科学家墨子
------


十进制？杠杆定理？时间与空间？


搞了半天，大科学家竟是你！


  



本专栏将讲述诸多超越时代的中国人物故事。


他们有些取得了辉煌的成就，光耀千古；有些则不被时代所理解，郁郁而终。


让我们跟随他们的步履，感受各个时代中的剧变！
