## 简介
你叫不醒装睡的人，也留不住要走的心。


你知道思念一个人的滋味吗？


就像喝了一大杯冰水，然后用很长很长的时间流成热泪。


我去翻你的过往，并不是嫉妒，我只是难过，有种深情，你从未给过我。


算了，就这样吧，相濡以沫，不如相忘于江湖。


  



**《布达拉宫的爱情》**


我拿着癌症通知单从诊室出来，碰见了我的丈夫扶着我的妹妹从妇产科出来。


他是那么小心翼翼，看向她的目光是那么温柔似水。


而我，只是一个抢了妹妹幸福的恶毒女人。
