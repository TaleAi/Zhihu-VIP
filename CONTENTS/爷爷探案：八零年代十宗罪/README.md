## 简介
在上个世纪，摄像头、DNA 检测等技术尚不完备的情况下，若遇到十分狡猾的凶手，怎样才能破案？本专栏讲述我的警察爷爷，在上个世纪 80 年代亲自侦破的十宗烧脑大案，每个故事能带给你的阅读快感都不输最优秀的侦探小说！


一个深夜，三人被关在一个封闭上锁的粮仓中，第二天醒来，其中一人被锤杀，另外两人似乎并无作案动机，真凶究竟是谁？


一所学校的黑板上出现神秘板书，字迹显示是一名失踪老师所写，调查过程中，又有一名学校职工惨死在废弃教学楼中，现场留下的符号有何意义？


改革开放初期，小富起来的一家人正热闹地准备年夜饭，第二天却全家惨遭灭门。每个尸体的手都不翼而飞，凶手为何这样做？


密室谋杀、不可能犯罪、灵异现象、地方志怪……一起跟随曾经的虎扑推理大神陶球霸的笔，勘破悬案！
