## 简介
秋水为神玉为骨，芙蓉如面柳如眉。


大宅宫墙里的女子，或是温良恭顺，或是刁蛮跋扈，她们处心积虑、步步为营。


愿当咸鱼的皇后，穿越而去的沙雕女，还有为国为民的女将。


本专栏的古代女子，不做那附庸的菟丝花，只做反套路的黑莲花。


凭何女子要为男子争风吃醋，凭何她们就做不得那一世枭雄？


如她愿意，便为利往，褪去一身柔情化作最狠的刀。


无边媚色都成了入骨尖刀，一刀一刀，尽数刻在他人心上。
