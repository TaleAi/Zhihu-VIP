## 简介
初冬的第一场大雪过后，一个女人被残暴地挖去心脏，而她只是第一位受害者……


2019 年初冬，宣阳市的第一场大雪过后，一个叫赵洋的女人遇害，心脏被人挖走。


宣西区公安分局牵头调查此案。然而，耗费大量人力之后，他们依然查不到关于凶手的半点线索。


不久之后，第二名被害人出现。死于同样的作案手法。专案组压力骤增。


不得已，女警李蓉在领导的推荐下，去请因过失被内部处分的派出所刑警徐海。


一个一丝不苟的严肃女警，和一个吊儿郎当的「神探」，磕磕碰碰地开始合作。


徐海拥有敏锐的观察和分析能力，在他的帮助下，凶手终于露出模糊的身影，专案组也发现了对方用被害人鲜血写成的数字。这让他们意识到，对方一共要杀五个人。


随着调查展开，一位明星企业家卷入此案。紧接着，一名酒吧老板娘自首，称该市存在一个诱骗强奸女性团伙。


很快，两起大案的嫌疑对象汇聚在同一个神秘人物身上。


然而，事实真如人们所见？当身处其中的每个人都不得不直面幽暗过往，真相才浮出水面。
