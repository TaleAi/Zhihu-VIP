## 简介
本专栏由北京公立三甲医院的皮肤科主治医生徐宏俊主讲，为大家科普梅毒的相关知识，帮助大家了解、认识、治疗与预防梅毒，以及哪些情况下需要尽快去医院检测才能更快治愈或者排除梅毒，消解大家对性传播疾病的恐惧。


你是否了解曾被大力防治的「花柳病」的古往今来？ 


你是否检索过许多语焉不详的资料，却仍然对梅毒一头雾水？


担心是否被传染上梅毒，接下来应该怎么办？ 


梅毒是否真的会影响人的职业抉择，甚至一生？


患上梅毒，会不会危及生命？ 


其实，梅毒如今已经成为最常见的性传播疾病，而且呈现出逐年增加，快速增长的趋势。年轻人正是性活动最频繁的年龄，也是皮肤科门诊最常见的梅毒患者群体。


一旦发生高危性行为后，很多人惶恐不可终日，甚至频繁去医院检测梅毒指标，但是也有不少人忽视了最常见的典型梅毒体征，延误了梅毒的治疗，传染给性伴侣，给家庭带来了不稳定因素，亦或者因为梅毒影响了未来的职业选择。


为此，本专栏请到了首都医科大学附属北京友谊医院皮肤科主治医生徐宏俊，来为大家科普梅毒的相关知识，帮助大家了解、认识、治疗与预防梅毒，以及哪些情况下需要尽快去医院检测才能更快治愈或者排除梅毒，消解大家对性传播疾病的恐惧。
