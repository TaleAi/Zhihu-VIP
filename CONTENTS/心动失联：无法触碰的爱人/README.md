## 简介
被时间抓走的爱人，再见你会拥抱我吗？


**- 月亮来信 -**


不小心双击了去世男神的头像：


我拍了拍「时砚礼」的肩并说哥哥来我怀里吧。


五分钟后，男神回过来一句：你老公不介意吗？


  



**- 过期心动 -**


男朋友准备向我求婚那天，我意外上了死对头的车，下车后竟被告知已经过去十年。


我深爱的男朋友，早已变成了成熟陌生的大叔。


他捏着我的下巴，神情阴狠：「既然选择跟他私奔，为什么还要回来？」


  



**- 今夜降落 -**


我和机长老公新婚燕尔，正浓情蜜意时，他在一次飞行途中和整个航班一起消失。


七年后的某个深夜，他裹着一身寒霜出现在我面前：「老婆，飞机已安全降落。」


  



**- 百年情书 -**


连日暴雨冲开桎梏，我从棺材里爬了出来，距离我被封进棺材已经过去一百年。


我问遇见的每个人：「你有没有见过一个叫顾廷斯的男人？」
