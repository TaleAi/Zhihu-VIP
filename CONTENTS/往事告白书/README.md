## 简介
我未曾珍惜的，我不再拥有。


年轻的时候，年少轻狂不知所谓。


那些求不得、爱别离、意难平，是大家的共同回忆。


你还好吗？


  



**「请你别再纠缠我」**


我就是手贱，非要下载那个聊天软件。


如果重新再来一次，我绝对不想认识她。那段时间活得像狗一样狼狈，我亲手毁了自己几年经营、马上步入婚姻的恋爱，把生活搞得一地鸡毛。


  



**「我爱他只在武汉」**


我和李贺，在武汉，四年，抗过了疫情，斗过了父母，却最后败给了时间。


我很难想象，爱了四年，经历过生死的两个人，走过逆境却走不过坦途。


我宁愿我们的故事，就停留在武汉的那个冬天……
