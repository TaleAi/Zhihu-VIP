## 简介
爱情——人生为数不多的英雄之旅。


我听说过一些不真实的爱情。


少女在父母被害后，遇见了来自未来的「丈夫」。那一天，两个人抓住了凶手，可她背着重伤的少年，却找不到一间开门的医院。这是她毕生的力量，只能用来跟一个人告别。


一年年的高考让她崩溃。那一天，每年表白的男孩作为凶手出现在新闻上，她才惊觉自己因为一场事故，永远活在了「十八岁」。有时候你会痛恨自己长大，因为最后一个爱你的人已经不在身边。


他们本来都是一个个普通人，但因为有了另一半的出现，他们注定不再平凡。


哪怕他们所做的只是一件微不足道的事情，在爱人眼中，他也是一个会发射 biubiubiu 冲击波的英雄。


因为这就是爱情的力量，是一趟人生为数不多的英雄之旅。


越绝望地挣扎，越要单纯地相爱。
