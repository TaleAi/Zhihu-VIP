# 犯罪心理分析：他因何杀人

## 简介
一个轰动社会的杀人犯都经历了怎样的人生？ 他们因何而杀人？中国政法大学犯罪心理学在读博士十方迦南将为你解读惊天大案背后的犯罪心理。


曾经的白衣天使，把救人的工具变成了杀人的武器，200 多个普通人因他而丧命，而他为何到临死前都未曾认罪？


刚满 18 岁的青年杀死同学后，便一发不可收拾，死者足以装满国际象棋盘，杀人上瘾的他暗藏着怎样的秘密？


年仅 10 岁的男孩，却是十恶不赦的杀人犯，他的童年经历了什么，让他走上这样的道路？


声名显赫的大学教授，自以为计划天衣无缝，但依然逃不出法网，他为什么要除掉一生挚爱之人？


在普通人看来，这些凶手十恶不赦，得到法律的惩罚大快人心。


但中国政法大学犯罪心理学在读博士十方迦南将告诉你：每个案子都事出有因。


这些案件不仅对当事人产生了影响，还深深被社会所铭记，凶手挑战了人伦底线、造成了人们的恐慌、引起的社会讨论。他们都有一个共同特征——极其扭曲的心理。


拥有大量的犯罪心理研究经验、参与过众多大案要案犯罪现场勘查的十方迦南将用专业的犯罪心理分析，从科学的角度，解读这些惊天大案。


还原每一个悲剧中隐藏的真相和启示。
