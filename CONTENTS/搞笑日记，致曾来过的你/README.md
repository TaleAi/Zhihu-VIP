《宿主摆烂后我觉醒了》


我是个系统，但宿主摆烂了。


他不走剧情，天天作死，不是尝尝毒药就是围观乱斗。


我：「宿主你不努力真的会死的。」


宿主神态安详：「正好，我想死。」


我：「你乐观点。」


宿主：「我想安乐死。」


为了我的业绩，我决定亲自下场帮他走剧情。


好消息，宿主有欲望了。


坏消息，他的欲望是我。
