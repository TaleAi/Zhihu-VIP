## 简介
荆棘之下，野蛮生长。


**不忠的背叛**


相爱多年的老公变心了。


他似乎吃定了能瞒住我。


一边和我柔情蜜意，一边和外面的女孩暧昧，甚至有了孩子。


发现端倪后，我按兵不动，默默搜集证据，漂亮地反击回去。


离婚后，我过上了梦寐以求的自由生活。


风雨过后，往后余生，我要活出自己。

解锁本专栏，看更多在困境中勇敢活出自我的女孩，如何逆风成长。
