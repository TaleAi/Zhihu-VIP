## 简介
本专栏由来自北京三甲医院的神经科主治医师晓晴老师主讲，帮助大家认识头痛、判断头痛、预防头痛、初步的治疗头痛，以及告诉大家，哪些情况下，需要毫不犹豫的冲向医院，才能保命。


你是否患有头痛？


你是否曾因大姨妈来临，而头痛难忍？


你是否曾因吃了一大口冰淇淋而大脑抽痛？


你是否曾因连续熬夜，而导致大脑隐隐作痛？


你又是否仅仅是因为一些无关痛痒的原因，比如烟味或香水味呛得头痛？


其实，你的头痛病不孤单，据流行病学调查表明，我国头痛患病率为 985.2/10 万，30 岁以下发病者逐年增长，而且每位城市患者平均每年要花费 1098.08 元治疗头痛。


但是，不要以为头痛的人很多，就可以对头痛掉以轻心，相反，头痛原因复杂，而且一些头痛十分危险，如有肿瘤病史的患者，或有神经功能缺失的患者，如果有头痛症状，不及时前往医院就诊是非常危险的。


为此，本专栏请到了来自北京三甲医院的神经科主治医师晓晴老师，来为大家讲解头痛问题，帮助大家认识头痛、判断头痛、预防头痛、初步的治疗头痛，以及告诉大家，哪些情况下，需要毫不犹豫的冲向医院，才能保命。
