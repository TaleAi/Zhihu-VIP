## 简介
为出名要去干死总统的疯批刺客、徒手偷窃机密导弹的奇葩间谍、几乎制定了美国「抢劫业标准」的强迫症悍匪·····在历史的洪流中，他们或许鲜为人知，但他们，曾经差点改变了世界。本专栏里，擅长用有趣的故事科普历史的小约翰，将为你讲述这些硬核小人物的奇葩事迹。


电影的影响能有多大？如果说影史上影响最大的电影，一定要了解一下由马丁·斯科塞斯拍摄的剧情片《出租车司机》。在这部史诗级电影的无数观众里，有一个名叫约翰·辛克利的人，看完电影后受到剧情的启发，决定去刺杀美国总统。


在西德做间谍是什么体验？冷战时期，一个潜伏在西德的苏联间谍曼弗雷德·雷明格，成功偷到了美国的新式对空武器：响尾蛇导弹。更骚的是，他光明正大用航空快递，把偷回来的这个三米来长，一百七八十斤的大家伙，寄回了苏联。


强迫症抢劫银行会怎样？赫尔曼·拉姆作为银行界的「大贤至圣先师」，同时也是个典型的强迫症患者。他还有一个 buff：运气极差。最后一次抢银行，强迫症发作的拉姆准备了 N 套预案，本以为万无一失，最终却输给了倒霉的命运。


······


本专栏带你重回冷战时期，看各路硬核狠人如何用一己之力，搅动历史，撼动世界。
