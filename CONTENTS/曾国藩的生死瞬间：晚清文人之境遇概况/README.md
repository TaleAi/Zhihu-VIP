## 简介
一代名臣的生死瞬间，晚清文人的境遇缩影。


「养活一团春意思，撑起两根穷骨头。」


一代名臣曾国藩为何要自杀，却又为何没自杀成。


本专栏特邀鞠易白老师回溯曾国藩一生中的几个生死关头，细数他跌宕起伏的一生。


以曾国藩的日常生活为切入点，深度拆解往来书信、奏折和日记，把冰冷的史书记载延展到有血有肉的生活中。


通过曾国藩的所作所为、所思所想，探讨曾国藩与皇帝、与官僚同事、与门生、与兄弟父母以及与自己的关系，展现晚晴时期文人的行为处事和当时的君臣关系以及中央和地方的关系。
