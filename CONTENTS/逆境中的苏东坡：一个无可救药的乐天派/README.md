## 简介
苏东坡一生坎坷，被一贬再贬，但他却活成了中国人的精神丰碑，被誉为「完美的中国人」。在逆境中，他如何消化负面情绪，重新营造新生活？被命运的大浪拍入谷底后，他又是如何翻盘逆袭，重新站上北宋的浪潮之巅？跟随本专栏一起，探究苏轼乐天精神背后的故事。


乍一看，苏东坡活得无比潇洒：


顶级吃货、皇帝秘书、社交达人、网文大神、酿酒大师、大文豪、大书法家、新锐画家、瑜伽修行者……各种名头，数不胜数。


但其实他的后半生，不是被贬谪，就是在被贬谪的路上。


然而苏东坡最大的才华，就是无论命运怎样毫不留情地碾压他，他都能用自己独特的逆境哲学，让自己放下过去，享受当下。


在人生的逆境中，他于文学上开创出宋词的「豪放派」 ，书法上挥毫写下被誉为「天下第三行书」 的《寒食帖》，美食上独创 66 道中国名菜「东坡肉」「烤生蚝」「烤羊脊」等，政治上修建利民工程、设计风景名胜「苏堤」「三潭印月」等。


怪不得同时代人王安石感叹道：「不知更几百年，方有此人物！」


近代知名作家林语堂也因此评价：「苏东坡是个秉性难改的乐天派。」


本专栏包含 25+首苏东坡经典诗词的解读，从「审美、思想、情感、意义」4 个角度，为你深度讲解苏东坡的逆境哲学，为身处逆境的你赋能，让你重获直面现实、翻盘逆袭的勇气。
