## 简介
阿鱼西施与绝症少年的奇幻爱情之旅。


我只是个普通的杀鱼店老板娘，除了失去记忆这点。


有一天，我收到了一张拍摄于七十年前的老照片，照片上是自己赤身裸体被铁链锁着的模样。寄送照片的是一个身患绝症的年轻男子，他绑架了我，声称只有我可以协助他前往广西密林寻找救治绝症的方法。


与此同时，我们在路上还遭遇了其他不明势力的追杀，不得已，我只好假装信任对方，陪着这个绝症少年踏上了广西密林之旅，一路经历各种凶险，终于解开了自己的身世之谜……
