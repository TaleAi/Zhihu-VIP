## 简介
在古代，很多人们习以为常的事情，在今天被视为异类，古人都有哪些怪诞行为，他们为什么这么做？本专栏将带你一探究竟。


古今中外的浩瀚史海中，有着包罗万象的科技文明，从现代人的视角去看，有许多现象令人匪夷所思。


在人类的历史中，这些看似奇葩另类的事物，却有其存在的必然性和合理性。


烤蜘蛛、鸵鸟的舌头，这些看似神奇的东西，在古代确实美味，他们为什么会吃这些？


人生病了怎么办？放血和灌肠，即时治死了很多人依然坚持，古人为什么这么热衷于如此危险的疗法？


中国人讲入土为安，但在某些地方人们习惯把死去的亲人定期挖出来清洗干净「团聚」。


看完本专栏的奇葩历史故事以后不禁掩卷沉思，先祖们真不容易，基因流传至今，吾辈不要轻易放弃生命啊。
