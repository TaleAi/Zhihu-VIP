## 简介
我肯定和你走，没船票我和你游。



爱情是一种赌博，一种冒险，一种世界上最不理性的投资。


所以有人说，面对爱情，要保持冷静，谁爱得多，谁就输了。


可爱情，同时也是一种意外，一种本能，一种不守秩序的闯入。


电光石火间，我就这么孤独地爱上了你。


于是，我暂时忘记了自己是谁，只记得奔向你，不问归期。
