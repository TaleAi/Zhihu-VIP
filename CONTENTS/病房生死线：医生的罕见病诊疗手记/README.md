## 简介
身患不知名的绝症是一种什么样的体验？


**医院里的故事总是耐人寻味：**


* 有人浑身上下长满肿瘤，做了十五年手术都没治好。
* 有人抽出的血液泛白，不知身患何病。
* 有人身上的皮肤像树一样变硬，大夏天也要裹着棉被……

这些病人辗转于不同的科室，有着各不相同的病症，唯一的相同之处是：不同寻常的病症及病因。


他们的就医之路都非常艰辛，甚至有人走了十几年的弯路才真正确认了自己的疾病。


  



**在疾病的痛苦下，人性的复杂和光辉也要被「看见」：**


* 她的目标是十年后的高考，但连医书上都没写自己能活多少年。
* 饿了八年，他最大的心愿是吃饭。
* 一个骨科医生决定放下手术刀，因为他的病人都在不停截掉自己的腿……

本专栏根据一线医生在罕见病及特殊病症治疗上的一系列见闻写就真实的诊断经历，解剖生死与人性，还原医生真实工作里不为人知的细节。
