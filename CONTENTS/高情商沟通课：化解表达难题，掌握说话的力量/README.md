## 简介
本课程由陈慕妤老师主讲，帮你成为沟通达人。15 项高效沟通技巧，帮你在饭局酒桌上如鱼得水，跟人聊天时话题不断，把女友哄得开开心心。善于说话，善于沟通，就是用用最小的成本，更快地得到成功。


 为什么谈话的时候，有的人让你感觉分分钟都在尬聊；而和有的人却能让你相见恨晚？ 


为什么在酒局饭桌中，有的人总能如鱼得水成为话题焦点；有的人永远只是可有可无的陪衬？ 


为什么在恋爱中，有的哥们总能把女友哄得开开心心；有的哥们总莫名其妙惹女友生气？ 


以上那些总能做得很好的人，都有一个共同点：深谙说话的艺术。 


学会说话，是一项最基本的、也最容易从中获得好处的技能。它能帮助你用最小的成本，更快地得到成功。 


说话是一门艺术，说话更是一种习惯，说习惯了也就变成一门艺术。 


欢迎参与本次课程，十五节课学会自如表达的技巧，助你掌握说话的力量。
