# 社会第一课：从「学生思维」到「职场思维」

## 简介
你的人生中有没有过被「上了一课」的感觉？学霸的一场分享，过来人的一次推心置腹，老板的一句提点，突然让你恍然大悟，发现之前一直在做无效的努力。



工作之后，你被社会上的「第一课」是什么呢？


* 是从学习成绩不错变成工作吊车尾的落差？
* 还是遇到事成为背锅侠的错愕？
* 抑或是一年到头打杂只能看别人晋升的艳羡？
* 甚至是全套 Office 技巧在手却得不到重用的落寞？

你遇到的问题，很多并不是因为能力不足，而是你的思维方式没有完成转变。


人的行为都是由思维方式决定的，**是否能用「职场思维」思考事情、解决问题，将决定你在职场能走多远。**


在本专栏中，你将看到不同领域、不同职业、不同背景的资深职场人们，**为你解读如何以「职场思维」做事，帮助你快速完成身份转变，成为职场达人。**


本专栏内容包括但不限于：


* 初入职场，如何处理好人际关系不做背锅侠？
* 为什么同样的工作，同事的效率和成果都比你更好？
* 如何摆脱明明做了对的事情却被批评的命运？
* ……

不想被社会上课？那就赶快主动出击吧！与过来人们一起击破困难，屏退困惑。


期待你的华丽转身。
