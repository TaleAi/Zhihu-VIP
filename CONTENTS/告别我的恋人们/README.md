## 简介
后来的我们，是否有人获得了快乐？


我们曾是彼此的月亮与太阳，最终却不知落在谁人心上。


《你是我的星光》


她是娱乐圈的一股清流，怼天怼地怼空气。


而她之所以如此胆大，除了过硬的实力，还因为她背后有个强大的金主。


可这个金主，从来没有露面。


他只是在背后默默地守护她，把一切最好的东西奉送到她面前。


但直到她站上云巅，才知道他的沉默，既是选择，也是被迫。


那时的他，如此孱弱，如此卑微。


这一次，轮到她守护他。
