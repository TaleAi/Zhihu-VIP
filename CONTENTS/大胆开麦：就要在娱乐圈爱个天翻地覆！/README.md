## 简介
这是一个娱乐圈故事合集。永远捧不红的资源咖，糊穿地心但站得够高的千年美女，哥本哈根喂鹌鹑的三金影帝，只想下班的顶流小狼狗……他们好苏好甜，好好看啊！大胆开麦，颜性恋就是永远的神！



**- 泪痣蛊人 -**


商业联姻，陪我试婚纱的是老公的弟弟，一个糊出天际的小爱豆，有趣。


他半推半就，「嫂子，我哥是不是没见过这样的你，那就别让他看见，嗯？」


我和他哥哥真就杠上了，哥哥包养女明星，我反手就给弟弟喂资源。


陪着弟弟进组，本来只是寻欢作乐，可最后我这个名门淑女竟做起了吻替。


一个是朝气蓬勃的小奶狗，一个是禁欲系原配，那么问题来了，我该选谁？


  



**- 温柔豢养 -**


落魄影帝顾浔为了翻红，偷偷养起了「小鬼」。


本以为小鬼好糊弄，没想到招来的是个辣手摧花的厉鬼。


她爽了，顾浔继续火；她烦了，顾浔遭反噬。


好郎也怕烈鬼缠，顾浔对女鬼越看越顺眼，想和她一生一世，女鬼却逃了。


前世记忆涌来，女鬼身份重见天日，这二人之间竟有有如此羁绊。


前世她是一人之下的将军，他是艳绝长安的戏子。


人鬼殊途，爱如何继续？


……


准备好了吗？


在娱乐圈爱个天翻地覆吧！
