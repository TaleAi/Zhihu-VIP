## 简介
 10 年公文写作经验的浓缩，党政机关负责人亲授； 9大常用公文题材，逐个攻破实战难点；没有高端深奥的理论知识，只有学完即用的提升技巧；从日常工作中提炼出通用模版，即学即用。


在机关工作，免不了要公文写作，


有的人宁愿组织十次活动，也不愿意写一篇稿子，


一提到整理材料就头疼，


领导布置下的写稿任务，迟迟下不了笔，


不是言不达意，就是发挥过度，


不少机关人员都因为公文写作苦不堪言。


![](https://pic2.zhimg.com/v2-34f5d447154b2441769ed07394d601e0.jpg)


在这个越来越重视原创、版权的时代，


文字创作已经不是只靠「复制粘贴」就能成功的一件事了，公文写作更是如此。


那么如何在公文写作中避免平庸，提升高度？


如何让公文立意深邃高远，不再「空谈对策」？


如何写出让领导满意，同事欣赏的公文？


![](https://pic1.zhimg.com/v2-4250bbf87e0ac320482281320b03b9df.jpg)


本门课将从立意、选材、结构、标题、内容、金句、公文气质 7 个方面入手，


全方位讲解公文写作的方法，


让你掌握新环境下的公文写作技巧，


再通过分析具体场景，


不管是个人总结、会议纪要、调研报告还是领导讲话稿，


都能让你写出的公文拥有灵魂，独放异彩。


![](https://pic4.zhimg.com/v2-5ac1df9a464e0e7ca85719164e0ef3d7.png)
