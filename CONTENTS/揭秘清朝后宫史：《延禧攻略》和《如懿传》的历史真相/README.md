## 简介
热播宫斗剧《延禧攻略》和《如懿传》中主人公的真实命运究竟为何？本专栏将带你一起了解真正的人物历史，揭秘清朝后宫故事，在历史真相中体会相处智慧，掌握社交生存术。


或许看过《延禧攻略》和《如懿传》这两部热播清宫剧的人都知道，两者剧情相差甚大：


《延禧攻略》中，富察皇后温柔贤良，是乾隆心中的「白月光」；而《如懿传》中，富察皇后却善于权术，陷害嫔妃，残害皇帝子嗣，而且并非是乾隆真爱。


那么，历史上真实的富察皇后，到底是怎样的一个人呢？ 


乾隆皇帝的真爱，到底是富察皇后还是如懿呢？


本专栏由历史学硕士文元讲师为你还原历史上真实的富察皇后和如懿。同时，他也将详细讲解关于上一届宫斗冠军甄嬛的那些故事。不仅如此，你还可以了解历史中关于香妃、还珠格格、孝庄以及乾隆的种种身世之谜，一起倾听清朝后宫的历史真相。
