## 13.大学宿舍不是个好待的地方
1


何然觉得自己快要被逼疯了！


凌晨三点，上铺的呼噜声打得震天，带着耳塞依旧如雷一般传到了她的耳朵里面，何然翻来覆去无论如何都睡不着，瞪着两只漆黑的熊猫眼看着头顶上的床板。


这种日子真的是没法过了！


何然从小到大从没有上过寄宿学校，因为无论小学、初中，还是高中，都在自己住的那个镇子上，父母又管得严，何然那个时候真的特别羡慕班级里面的寄宿生。


不为别的，只因为他们不用每天听父母的唠叨，也不用每天背着厚厚的书往家赶，第二天又要背回学校去。


在那时候的何然的眼里，寄宿生有着一样她怎么也得不到的东西，那就是——自由。


所以她在高考之后报大学志愿的时候咬紧了牙没有听从父母的安排，报了一个外省的学校。


收到录取通知书的时候她高兴得欢天喜地，已经大学毕业的表姐却突然在她耳边幽幽地吐了一句。


「别太高兴，大学宿舍可不是一个好呆的地方。」


何然当时虽然有那么一点疑惑，不过即将离家闯荡新世界的她很快就忘记了表姐说过的话，开学的时候她美滋滋地踏上行程，想象着自己的舍友会是什么样子。


四人宿舍的话，应该是一个文静端庄，一个成熟大方，一个可爱明媚，加上自己的开朗活泼，这大学四年，一定会结下珍贵的姐妹情才是。


但是何然的推断只对了一半。


2


宿舍里面的四个人的确是四种性格，但并非是自己之前以为的那样。


小 A 住在何然的上铺，是一个很胖嗓门很大的山东姑娘，喜欢动漫，喜欢打游戏，喜欢吃美食。


小 B 在何然的对面，是班级里面的班长，有一些微胖，不算很高，有一张蛮可爱的脸，说话总是一本正经，是一个典型的狮子座喜欢领导别人的人格。


小 C 在小 B 的上铺，是一个货真价实的美女，柔柔弱弱的，总是买各种各样的保养品，平时也很注意自己的饮食和健康，身旁围绕着一群追求者。


何然就简单了，除了上课就是宅在宿舍里看剧，或者出去找找兼职，没什么特别的爱好。


最开始的时候，几个人虽然兴趣爱好都不太一样，但是相处的还算是融洽，平时也会打打闹闹开几个玩笑话，在外人看起来也是亲亲密密的四个姐妹花。


只是时间久了，隐藏在欢笑下面的东西便一点点地露出来了。


最先让何然无力吐槽的，就是小 A 每晚的呼噜声。


那叫一个惊天地泣鬼神，那叫一个风云巨变天地失色！


当然，上述不太可能，但毫不夸张地说，小 A 的呼噜声真的非常可怕，曾一度将何然从睡梦中惊醒。


虽然睡眠质量很不好，但是打呼噜这个问题是没办法控制的，所以何然只是默默地自己去买了耳塞，虽然晚上还是会偶尔被惊醒，但也总算是能睡着了。


只是并不是所有人都有如何然一般的忍耐力。


最先爆发的是小 C。


小 C 注重保养，每晚睡的也是最早，十点左右就上了床，但每晚十点的时候小 A 还现在游戏里出不来，键盘敲得啪啪响，一声高一声低地咒骂着游戏里坑爹的队友。


小 C 说过几次她想睡觉了的这个问题，但是小 A 是从来不听的，加上小 B 也习惯晚睡，不到十一点不洗漱，每次洗漱的时候一定会弄得跟打仗一样，叮叮当当一阵乱响。


所以，小 C 也只能认了，白天趁没人的时候跟何然吐槽，还让何然看她的黑眼圈又重了多少。


只是夜里的呼噜声还在继续，于是何然和小 C 眼睛下面的黑眼圈再也没消失过。


终于有一天，呼噜声震天响起的时候，小 C 忍不住了。


她喊了一声：「小 A 你小点声！让不让别人睡觉了！」


小 A 的呼噜声瞬间停了，不过也就停了一会儿，紧接着又响了起来，甚至比之前更甚，睡在下铺的何然觉得自己的床都在随着那呼噜声一晃一晃。


于是整个晚上，就如同二重奏一样的，小 C 和小 A 的声音从头到尾，都没有停止过。


矛盾的种子就是从那一天起被埋下的。


小 A 是独生女，爹妈都把她宠成了上帝，把她养成了一副「全世界都应该宠着我惯着我」的性格。


对于夜里小 C 无法睡觉的这个问题，小 A 丝毫没觉得自己有什么错，错的都是小 C，每晚不让她睡觉，这简直就是无法忍的大事。


于是小 A 开始了她的报复计划，晚上十一点定下闹钟，每隔半个小时响一次，持续到凌晨一点为止，第二天五点继续。


小 A 和小 B 睡眠质量极好，夜里闹钟的声音压根就吵不醒她们，可怜的是何然和小 C，硬生生地被逼出了神经衰弱。


3


小 A 的故事到这里结束，现在我们来说说小 B。


小 B 是班长，狮子座女生，她喜欢在任何事上领导别人，包括平时生活中鸡毛蒜皮的小事。


何然习惯用价格亲民好用的护肤品，小 B 跟她说廉价的护肤品如何如何不好，语气就如同在班里开会一样的正式严肃。


还没等何然辩驳几句，小 B 就把她的护肤品扔进了垃圾桶。


何然说某某某电影还蛮好看，小 B 突然说那个电影一点也不好看，完全就是反面教材毁三观，看那种电影的人心态都不正常，还过来给何然推荐她自己认为很好但河然认为没什么看头的电影让何然看。


大概是习惯了否定别人，无论别人说什么做什么喜欢什么，小 B 都喜欢出来插几句，神情语气都是不容别人拒绝的。


时间长了，寝室里也没人愿意同她说话了。


但如果只是这样还好。


小 B 虽说是班长，但是她是一个很自私的班长。


平日里接到的通知她会挑选对她有利地下发到班级群，例如学院组织的并没有什么用的讲座，例如给预备党员的她投票，再例如学生会检查卫生她通知让大家去把卫生收拾好。


除了这些能够让她在老师面前扮演好学生的事情，其余的小 B 都不怎么放在心上，每天回了宿舍就上床看剧看小说，吃饭的时候就订餐叫外卖。


小 B 对别人要求很严格，但对自己却并不是。


她的脏衣服一攒攒一盆，里面内衣内裤袜子外套什么都有，气味刺鼻，夏天的时候招了一阳台的苍蝇。


宿舍里面的其他人忍无可忍说出声的时候，小 B 才会很不乐意端着盆子送去洗衣机。


小 B 从未刷过鞋子，一堆鞋子全扔在床底下，时间长了，她的床铺都弥漫着臭烘烘的味道。


小 A 有轻微的洁癖，她无数次对小 B 说注意卫生的问题，但每次小 B 都能找到无数借口，时间长了，也就没人再对她提了。


大家都是大学生，都是成年人，都不愿意互相说来说去，好在大学只有四年，忍忍也就过去了。


何然是这样想的。


只是总有一些事是无论如何也忍不下去的。


4


小 C 外形青春靓丽，性格也蛮好，在班级里和每个人都相处得不错。


但是她有一个让人无法忍受的缺点，那就是手脚不干净。


小 C 喜欢占便宜，喜欢占各种便宜。


谁的东西好用，她看到了，就借走了，然后就……没有然后了。


外人看到的也不过就是上面的这一点，但是和小 C 日夜相处的何然却清清楚楚明明白白地知道自己的这个室友有多过分。


大一一整年，小 C 没有买过香皂，没有买过洗衣粉，也没有买过洗发水沐浴露，她一直用的，都是何然的。


何然在的时候，小 C 借着用，何然不在的时候，小 C 就完全把那些东西当成了自己的，随便用。


刚开始何然不好意思说小 C，但后来忍无可忍说出来了，小 C 也买了自己的东西了。


但是她那些东西用了一整年都没用完，反倒是何然的东西一个月就没，还得再买。


小 C 的这个毛病也不仅仅局限在日用品这一类。


宿舍里面大家的零食总是丢，刚开始大家还觉得是被自己吃掉了或者掉在哪里了，直到有次小 B 买的俄罗斯糖果丢了半盒，然后无意间在小 C 的书里发现了包装精美的俄罗斯糖纸。


发现小 C 的这一点之后，陆陆续续地又发现了很多以前都没注意到的事。


小 A 的芦荟胶丢了半学期，半学期后在小 C 的桌子上发现了。


小 C 去洗澡，拿回来装着换洗衣服的袋子忘记收了，小 B 不小心碰掉了，从袋子里滚出来了自己丢了好几个月的沐浴露。


何然从家里带来的蜂蜜一直放在柜子上没怎么吃，有一天她突然想起来想喝蜂蜜水的时候，才发现那蜂蜜不知何时只剩了一个瓶底。


这样类似的事情数不胜数。


后来大家都无法忍受了，于是决定一起排挤小 C。


刚好这个时候小 C 交了一个男朋友，于是她沐浴露也有了，洗发水也有了，零食也有了。


于是大家提起的心也就放下一半了，最起码不用担心自己的东西哪一天突然没了。


后来何然也交了男朋友，再加上没有课的时候去校外做兼职，日子变得很忙很忙，寝室里的事情似乎一下子离她远了不少。


何然的男朋友高大帅气，性格也很好，把何然宠的像个孩子，小 C 曾经不止一次的说过「你男朋友真好啊」这样的话。


何然心里甜丝丝害羞地捂脸，没有把小 C 说的那句话太过放在心上。


只是她忘记了小 C 喜欢偷别人东西的这一点。


5


天气晴朗的某日，何然汗流浃背地在街上发着传单，猛然间看到不远处的小 C，正一脸甜笑地挎着一个男生的胳膊。


那男生一脸不耐烦地抽回胳膊走了，小 C 不依不饶地又贴了过去。


那男生，正是何然的男朋友。


当夜，一向好脾气从不会发火的何然听着上铺的呼噜声和小 C 不满地嘟囔，第一次飙起了脏话，伴随着她起来摔碎了一个杯子。


「都特么小点声，老老实实睡觉能死啊！」


于是，寝室里第一次迎来了安静的没有半点声音的夜晚。


清晨醒来时一屋子的人看着一地的杯子碎片，都选择了不出声，对夜里的事情闭口不谈，收拾了碎片后各自忙活各自的问题去了。


留下何然一个人在宿舍，捂着胸口感觉难受得快要炸开。


何然终于理解来大学前表姐说的那句话了。


大学宿舍可不是一个好呆的地方。


真的，不好呆。


后来的日子和以前一样，小 C 还是和小 A 不对付，小 A 还是明里暗里针对小 C，小 B 还是只关心自己那一亩三分地时不时地说道说道别人，何然也和从前一样的好脾气。


某日又是被呼噜声吵了一整晚的小 C 和何然一起出门上课的时候疯狂大骂差点崩溃。


「真不明白那头猪打呼噜声音堪比打雷是怎么做到的！睡眠质量真的好到无敌了！」


「也不是哦。」


何然笑了笑，「打呼噜也是一种病呢，听说严重的时候会导致呼吸不过来，会猝死呢。」


小 C 似乎很惊讶，但随即她便不再说话了。


何然满意地笑了。


6


紧随而来的是一群同学去政教处把小 B 告了。


原因是班级有一些同学选修没修满，学校规定大三结束前必须修满所有学分，大四不能修选修了。


但是那些同学左等右等，大三马上就要结束了，都没能等到选课的通知。


是因为小 B 从始至终都没有把选课这个事放在心里，因为她选修课早就已经修满了，所以接到通知的时候直接略过了，也忘记了班级里还有没有修满学分的人。


小 B 自然而然被导员骂了，也自然而然地失去了民心。


随后有关小 B 自私自利不讲卫生的流言一夜之间冒了出来，所有人看小 B 的目光都是带着刺的。


心高气傲的小 B 自然无法接受，她特意开了一场班会想要挽回自己的形象和地位，但是那场班会，除了何然，无一人去听。


小 B 神情难掩愤怒，她无法忍受别人不遵从她的命令，后来她终于想到了这段时间以来班级流传的谣言，不由得看向了从始至终一直沉默的何然。


「你告诉我，我的事情，是谁说出去的。」


何然很是犹豫，半晌才缓缓开口。


「我不知道是谁，但是我曾看到过小 C 跟同学们在一起说笑话，大家都笑得很大声。」


小 B 一下子明白了什么，她低下头，默不作声。


不得不说小 B 还是很有能力的，不过三天的时间，关于小 C 做小三勾引有妇之夫的流言疯狂地席卷了整个年级。


不仅如此，还有她平时如何喜欢占便宜偷东西的流言也迅速地传了出来，有理有据，以至于跟在小 C 屁股后面的那些追求者一夜之间全不见了，气得小 C 在宿舍里疯狂大骂那个造谣的人。


于是一场迟到了很久的争吵爆发开来。


何然沉默地从热闹的宿舍中退了出去，那一晚她没有回宿舍，而是同男朋友去了宾馆。


7


再次回到宿舍的时候，房间里面的气氛很沉闷。


何然小心翼翼地说：「不如我们出去吃顿饭吧，都这么长时间了，我们也好久没有在一起喝过酒了。」


的确，上次大家聚在一起吃吃喝喝笑笑闹闹还是在大一的时候，一转眼时间过得飞快，三年都快过去了。


其他三个人最开始都是拒绝的，但不知道她们又想到了什么，不约而同地点了头。


小 C 要了度数高的白酒，她举着杯子敬小 A，说这段时间她做的不对，希望小 A 能够原谅她，还说大家以后好好相处好好过，继续做好姐妹。


小 A 很得意小 C 终于向自己服了软，她喝了小 C 敬的酒，还反过来劝小 C 手脚不老实的毛病也最好改改，不然早晚有一天会人赃并获被全世界唾弃。


小 B 也抓紧了机会敬小 C，明里暗里地说了不少带刺的话，小 C 自然也不甘心，反过来也不停地向小 B 灌着酒。


待一顿饭吃完，四个人里面唯一还清醒的，也就只剩下了一直专注吃菜的何然。


四个人回了宿舍各自睡下，第二天一早发现小 A 死了，死于打呼噜导致的睡眠呼吸暂停。


小 A 打呼噜太过严重本就是病，睡前又喝了很多酒，睡眠中窒息也并不是不可能。


小 A 的父母却不甘心，哭闹着说他们的女儿打呼噜很多年了，怎么会突然就死了，一定是有人故意谋杀的。


的确是被谋杀的，因为法医在小 A 的胃里发现了份量不清的安眠药残留物，而警察在调查的时候，又在小 C 的枕头下面发现了剩下的没用完的安眠药。


小 C 因涉嫌故意杀人罪被带走的时候全校都在看着她，她一边被警察押着一边说她没有想杀人，她只是放了一点安眠药想让小 A 睡觉不打呼噜罢了。


但没有人听她的。


这个所有人眼里的女神，现在是个杀人犯了。


8


因为寝室里发生了这种事情，身为班长的小 B 自然要背负很大的责任。


再加上之前她信息通知不到位的错误和流传的那些谣言，校方把她叫过去谈了话，把她的班长职位拿掉了，又取消了她的预备党员资格。


变成了普通学生的小 B 在宿舍里发呆，失去了发号施令管理别人的权利，她突然之间不知道自己该干什么。


宿舍里只剩下了两个人，夜里再没有了恼人的鼾声，也不用再担心东西会丢，何然很满意现在的现状，自来到大学以后，她第一次睡了一个好觉。


在出去吃饭回来后的那天晚上，小 C 的确没有在小 A 的水杯里放多少安眠药，那些份量也许会让小 A 睡个好觉，但不足以让小 A 在睡梦中窒息身亡。


多出来的安眠药，是何然放的。


要怪就怪小 C 吧，千不该万不该，不该觊觎她的男朋友。


还有那些谣言，大部分，都是她散播出去的。


表姐说的真对，大学宿舍，真的不是好呆的地方。


不过不好呆的不是她，而是别人。


暑假回到家的时候，小几岁的妹妹高考刚刚结束，满脸兴奋加好奇地向何然打听大学生活好不好，何然只是笑了笑，回了一句。


「好啊，当然好，不过前提是要和同宿舍的人关系好。」


「如果关系不好呢？或者有不好相处的同学呢？」


何然微笑，想起了同宿舍的小 A 小 B 和小 C。


「很简单的，能忍就忍，忍不了了，就毁了她们。」
