# 谈恋爱不如搞事业：权力巅峰的女人们

## 简介
她们既要事业，也要爱情。


做个争宠的后宫嫔妃有什么意思？自己变强大，做个跟皇帝平起平坐、指点江山的女强人，才有意思。她们是一群不按套路出牌的宫廷女人，是霸气外漏、大权在握的皇后、贵妃和女王爷，没兴趣谈情说爱，只想兢兢业业搞事业。至于感情，随缘随缘。


**迷花不事君**


我是穆州只手遮天的皇后，也是朝野得而诛之的佞贼。琴棋书画不堪所托，杀人放火倒十分精通。


为了收复西北六城，我逼宫老皇帝，鸩杀长公主，灭了后宫的流言，缴了朝臣的舌头。我眼中是杀红的血，唇角是恣睢的笑，手中是滔天的权势和三十万大军的兵符。


哪怕这一切代价是极大的，身侧的小皇帝怨恨我，青梅竹马的小将军要手刃我。


只因我曾尝尽乱世之苦，我要这天下人，不再吃这乱世苦；我要这芸芸众生，享个太平盛世。


 


**诸相非相**


我屠染染，王府最低等的侍妾，带着景亲王宁怿一起穿越了。


这穿越穿得着实妙，染染我翻身农奴把歌唱，成了大越女国一人之下的堂堂晋王；而宁怿，穿成了我的侍妾。


穿越后我大权在握，迎娶高富帅，走上人生巅峰；更是三妻四妾、左拥右抱，将宁怿气到泣血连连。


可这女王爷不好当，新婚夜的莫名大火，朝堂上的明枪暗箭，同宁怿间讳莫如深的秘密，都令我屡次身陷险境。这鬼地方，怎么人人都想干掉我？
