## 简介
因为年轻，我们始终对人生充满疑惑，对世界缺少理解，跟台湾民谣之父胡德夫一起，重读《飞鸟集》，解答不同阶段必然会遇到的人生必答题。


《飞鸟集》（Stray Birds）是有着「诗圣」美誉的拉宾德拉纳特·泰戈尔发表于 1916 年的经典作品，通过对自然的入微观察，表达对人生的深刻体悟，用宁静的心感受世界的美丽与脆弱。


325 首清丽小诗，晶莹剔透，字字珠玑。每一篇都是一个美妙的故事，每一句都是饱含智慧的生命答语。


世界以痛吻我，要我报之以歌。


使生如夏花之绚烂，死如秋叶之静美。


当你没胃口时，不要抱怨食物。


让睁眼看着玫瑰花的人也看看它的刺。


我们把世界看错了，反说它欺骗我们。


尘土受到损辱，却以她的花朵来报答。


我不能选择那最好的，是那最好的选择我。


最好的东西不是独来的，它伴了所有的东西同来。


你看不见你自己，你所看见的只是你的影子。


「台湾民谣之父」胡德夫，带来《飞鸟集》的**最高诠释版本**，**独家翻译、朗诵、解读**，并亲自**作曲配乐、钢琴弹奏、现场演唱**。


历经六十多载风雨人生，他通过自己的感触和对泰戈尔诗作的感受力，道出福至心灵的处世哲学，解答不同阶段必然会遇到的人生必答题。


忙碌困顿的你，不妨来听听诗中的美好与哲思。


愿这些生命中的暖意和微光，陪伴你每一个好梦。


**【你将获得】**


* 胡德夫亲自创作配乐，钢琴弹奏，感受诗与音乐的绝美融合
* 现场演绎歌曲，聆听最有生命厚度的魅力嗓音
* 独家翻译《飞鸟集》，来看看和郑振铎、冯唐的版本有什么区别
* 胡德夫倾情解读，用六十余载的人生历练，道出诗中的哲思
* 一本让你豁然开朗的人生解忧手册，为你指点迷津，找到生活的最优解
