## 简介
如何打破职场焦虑，找回激情？赵力老师用自己的真实经历为你讲述如何跳出职场焦虑，发掘自我潜意识里的需求，保持新鲜感。


不少职场人每天工作做起来很顺手，但逐渐消失了对工作的激情，没有了成就感。


安分的人，开始沉寂，而不安分的人害怕被磨平了性子，却又不知道如何突破，便感到焦虑。


那么我们应该如何打破这种焦虑，找回激情？如何才能有所作为？


赵力老师用自己的真实经历为你讲述如何跳出职场焦虑，看看她是如何摆脱职场焦虑，从行业小白晋升到业务总监。


她将教你如何走出舒适区、焦虑期，发掘自我潜意识里的需求，保持新鲜感；修炼迈向成功所需具备的素质；结合自身和家庭，做好职业和人生规划。
