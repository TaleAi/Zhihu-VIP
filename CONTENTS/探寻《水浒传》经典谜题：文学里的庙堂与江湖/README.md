## 简介
《水浒传》已经成为我们中国人难以割舍的文化记忆。林冲、鲁智深、武松、西门庆、潘金莲，这些名字你一定从小耳熟能详。本专栏特邀文史学者刘勃老师，带我们打破文学与历史的界限，探寻《水浒传》里不曾被阅尽的文学与历史的世界。


《水浒传》是中国文学绕不过去的经典之作，然而却留下许多经典谜题：


为什么说「少不看《水浒》，老不看《三国》」？


为什么这样「三观不正」的小说，却被奉为「四大名著」之一？


为什么明代大才子金圣叹要「腰斩」《水浒传》？


……


你也许从没看过《水浒传》，你也许嫌弃它血腥残忍，你也许对它敬而远之，但林冲、鲁智深、武松、西门庆、潘金莲，这些名字你一定从小耳熟能详。


不知不觉中，《水浒传》已经成为了我们难以割舍的文化记忆。


但是我们真的读懂《水浒传》了吗？


你以为林冲是个好丈夫，其实他是个最典型的渣男；


你以为李逵是个杀人魔，其实他反而代表着童心；


你以为潘金莲道德堕落，其实她才是最有魅力的女汉子……


本专栏特邀文史学者刘勃老师，带我们打破文学与历史的界限，探寻《水浒传》里不曾被阅尽的文学与历史的世界。
