# 旧剑斩桃花：魔头的替身女友是大佬

## 简介
我，武林第一剑客唯一的徒弟，偏偏剑法不精，是个废柴。要不是意外射伤大魔头花轻尘······的马，我可能永远不知道，自己竟是个大佬。魔头把我当成了他白月光的替身，将我扯入一场接一场迷一般的江湖阴谋中······


出门做点小生意的工夫，我，江湖中籍籍无名的废柴剑客，被武林大魔头碰了瓷。


是我的错，谁叫我真误伤了人家的马，那我……赔钱吧。


大魔头却说：赔钱就完事儿了吗？


我：不然呢？


大魔头：你跟我前女友挺像的。


啊这！有关系吗？！


就这样，我被大魔头囚禁了，成了他的白月光前女友——武林第一美女东方燕的替身。


被囚禁的鬼日子，我不仅咽下血泪赔了银钱，还得舍命做他前女友的替身，又被他拉进了几大武林世家的权力争斗。


谁知道，嚯，这几大世家的人竟都认识我，貌似还跟我挺熟……


乱了乱了，我这么牛的吗？为啥我自己都不知道？
