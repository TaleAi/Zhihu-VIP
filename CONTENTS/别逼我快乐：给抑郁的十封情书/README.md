# 别逼我快乐：给抑郁的十封情书

### 状态：已完结，共 10 节

### 存档日期：2022-06-02

## 简介
抑郁症患者的世界是什么样子的？
丘吉尔曾说，心中的抑郁就像只黑狗，一有机会就咬住我不放。
对于抑郁症患者来说，不仅要时刻提防那只挥之不去，卷土重来的黑狗，还要应付现实世界中种种的难题。所以「人生苦短，快乐一点」这种话并没有实质性的帮助。
隔着网络，隔着千山万水，这个专栏也许不能提供具体帮助，但希望这十个关于抑郁症的故事能温暖你治愈你，带你寻到出口。


每当新闻里那些年轻鲜活的生命突然消逝，报道称其生前患有抑郁症时，总会激起大家对抑郁症的热烈讨论。「怎么这么多人抑郁啊！」「Ta 那时候的行为是在求救吗？」「为什么在 Ta 生前大家不能多一些体谅和关心呢？」仿佛都自动站在了 Ta 那边，但明明 Ta 曾是那样的孤立无援。


如何自救或救他，更需要专业人士的帮助，但至少我们可以让他们的世界多一点阳光和希望。


作者城闭喧长久以来一直在网络上帮助抑郁症患者，他说起初只是耐心聆听一段段经历，想给大家一个倾诉的地方，后来想给大家更多的精神支持，于是写下了一个个暖心的故事。


这个专栏里收录了十个关于抑郁症的故事，主人公有深受校园暴力困扰的高中生，有沉迷于网络游戏的待业青年，有郁郁不得志的女作家，还有勇敢逃离原生家庭的女孩，他们虽然都被抑郁症困扰，但最终都找到了一种方式，学会跟自己和解，与现实和解。


希望你也早日找到这种和解的方式。


温馨提示：如果觉得自己有抑郁症的倾向，还请尽早寻求专业帮助～
