## 简介
现代大都市当中，年轻的精英阶层面对新时代时，所产生的婚姻、爱情与社会的矛盾是什么？
一对夫妇，印证了「月子之仇，不共戴天」，另一对夫妇，坚持「哲学与诗性」，进行了婚姻实验……


房产公司高管柳之峤在生下孩子三个月之后跳楼自杀，摔成了重伤。


导演赵珑珑感觉奇怪，想要追查她这样的职场女性为何会走上这一步。


两年前，赵珑珑的丈夫郑山河去过柳之峤的婚礼，在婚宴上，有人大打出手，他就看出来柳之峤和她的丈夫彭小辉和婆婆之间奇怪的关系。


柳之峤比彭小辉的职位和收入都要高得多；凤凰男出身的彭小辉也甘愿下班后在家做饭、做家务。但在妻子的企业高管内斗得最严重的时候，彭小辉设计让柳之峤怀孕，并迫使柳之峤被火线解雇；在她临产的时候，彭小辉叫来了自己妈妈前来干预……


另一对夫妇，郑山河是红三代，赵珑珑是富二代。媒体高管郑山河一直追求新闻理想，为了能客观报导医患冲突，顶住压力，被免职；一直庇护他的上级媒体总编，东窗事发，坐牢了，满怀理想的他，该何去何从……


两对夫妇，在现实的碰撞中发现了一个真理：爱是使每个人自我认识和升华的必经之路。
