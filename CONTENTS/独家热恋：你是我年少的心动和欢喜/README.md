**冬日爱恋**


去医院拔智齿，医生是前男友，我肿着腮帮子嘴硬说不疼。


重逢后，得知当年误会了他，我勇敢二次告白，却发现他已经暗中筹谋婚礼。


原来这么多年，念念不忘的不止我一个。


**非正式相亲**


相亲不小心相到顶头上司，还顺便帮他当了一次挡箭牌。


他认为进一步联系就是默认交往，我硬着头皮跟他说清楚，分道扬镳。


恋爱没谈，倒是分了一次手。


同事女朋友误会我抢她男朋友，正好被上司撞见，为我撑腰。


我终于发现，原来我早已喜欢上他；而他，比我更早。
