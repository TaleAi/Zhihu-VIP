## 简介
人间皆苦，你的偏爱就是救赎



爱情有千万种模样，乍见倾心，久别重逢，甜涩暗恋，求而不得…… 


这个专栏，带你看遍爱情的酸与甜，甘与苦。


**不美丽罪名**


我是年级公认的丑女。


愚人节，我的暗恋日记被示众，成了全班的笑料。


课间，我从厕所回来，藏在书包里的日记被班里的混混翻了出来。


他站在讲台前，当着全班的面一本正经地朗读——


最让我难堪的是，我的暗恋对象也在。


他安静地坐着，一言不发地旁观这场闹剧。


很久后才淡淡出声，「恶心。」
