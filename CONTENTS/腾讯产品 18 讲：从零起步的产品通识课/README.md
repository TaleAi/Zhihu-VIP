## 简介
QQ、微信、王者荣耀、微信支付、公众号、小程序 ……腾讯为什么能做出这么多备受用户喜爱的产品？这是一套腾讯独家的产品经理养成方法论，也是腾讯做产品一直坚持的产品精神，毫无保留，全面呈现给正在学习这个专栏的每一个「你」。加入专栏学习，与行业万千产品经理交换经验，成为更棒的产品经理；也将产品方法，应用到你的人生，将「你」这款产品，打造成爆款，成为更棒的人！


QQ、微信、王者荣耀 ……22 年来，腾讯做过上千款产品，其中不乏一些国民级应用。


做产品，腾讯认同什么、坚持什么？腾讯产品经理们都是如何做产品的？有哪些方法和工具？


腾讯大学代表腾讯官方，首次推出《腾讯产品 18 讲》。


力邀内部数位打造过经典产品、并在方法沉淀上颇有建树的专家，以及业务一线的资深产品骨干；访谈 20 个经典产品的核心团队，提取过往培养产品经理的课程中的精华；历时 2 年，迭代提纲 16 版，修改逐字稿 30 万字；将产品经理们对做产品的思考整理出来，高度提炼成「找-比-试」三步法，全面呈现产品从 0 到 1 的完整流程和方法。


**课程亮点**
--------


**●** **权威性：**腾讯官方推出的沉淀腾讯产品思考的课程，帮你获取最干货的产品知识，降低筛选成本；


**●** **完整性：**全面呈现产品从 0 到 1 的完整流程和方法，帮你一站式建立产品认知体系，避免只见树木不见森林； 


**●** **实战性：**数十款腾讯各类产品的经典案例复盘，帮你回到具体工作场景中，好理解、易上手、能应用。
