## 简介
职场社会学，用社会学解决「跳槽也解决不了」的难题。职场最重要的是做事与做人。社会学绝不只是理论，在做事与做人上社会学往往能发挥意想不到的效用。希望职场社会学，能帮你在面对人生一些选择时，优雅且不惶恐。


无论从事什么工作，无论是初入职场还是马上奔三，我们都可能会在工作中遇到各种各样差不多的问题：


* 怎样适应陌生的工作环境？
* 怎么面对批评？
* 怎么控制我们的工作情绪？
* 为什么越努力越焦虑？

这些问题的背后其实都有**社会学原理**的影子，了解社会学原理，你会更简单更正确地处理这些问题。

**适合人群**


1. 即将走进职场的你
2. 为职场「人际」「社交」所困扰的你
3. 25/30/35 遇到职场瓶颈的你
4. 对「人的行为」背后的规律感兴趣的你

**不了解社会学的人往往会认为社会学理论有余实用不足，但实际上，每个生活在人类社会中的个体，都需要足够多的社会学知识来面对复杂社会的各种现象。社会学的功能并不只是描述和解释，它还可以在生活、学习、职场、人际关系、婚恋爱情中发挥实际效用。** 


本次课程就是用**社会学的经典理论**来帮你解决职场中经常遇到的事务上、人际上难以处理的问题。职场中两件事非常重要：一个是做事，一个是做人。学会做事做人，是你在任何公司、任何时候都可以修炼的一种职业素养，处理好工作中的人际难题，会为你的职业发展增添筹码。


希望本次课程可以给你带来帮助，为你在日后面对职场和人生中的一些选择时，能优雅且不惶恐。
