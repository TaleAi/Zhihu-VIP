## 简介
生活如死水，但神明的奇迹，永远在降临。


奇幻、科幻、克苏鲁神话、末日逃生、时空轮回等小众元素故事集。  




《旧日英雄》少年总有英雄梦，但当少年落入末日试炼中，生与死的较量，恐惧与真相的探索，伙伴的背叛和牺牲，最终的胜利真的是胜利吗？也许一切都只是一场没有赢家的选拔。  




《他的弦外之音》当你最爱的人成了杀人犯，若时空可以重启，未来可以挽救，你愿意牺牲什么，去换取最完美的结局？  




《肿瘤》当你在梦里醒来，发现自己正被最亲的人追杀，当你在现实里醒来，发现早已遗失的记忆正在慢慢恢复，在那些记忆中，藏着让你无法面对的死亡真相。
