# 诗人之死：中国诗人的绮梦、爱欲和国忧

## 简介
在没有英雄的时代，让我们诗意地活着。


**本专栏是一部中国诗人版的「人类群星闪耀时」。**


南京大学中国古代文学博士聂飞琼，为你讲述十二位中国诗人将死之际的境遇，书写那些令人扼腕的诗人结局，描绘诗人们在各自时代中的人生选择、对生死的超然态度和对家国的深沉情怀。


**《夏完淳：此生，我每一刻都尽力了》**


夏完淳死时，只有十七岁。十四岁那年，甲申之变，明亡，他就随父亲抗清。亡国孤臣，怀着一腔忠愤，却在短短三年中，尝遍世间离别辛酸滋味。父亲和老师先后投水而死，而他自己也最终被俘。


最终，他被带到洪承畴面前，亲自断绝了自己最后的生路。


**《钱谦益：不要温和地走入那良夜》**


1664 年，钱谦益以 84 岁的高寿去世，留下一生故事，与后人评说。


在国家生死存亡之时，他「大节有亏」，由此成为明末清初遗民中的大名人。他活得「不彻底」，既不是忠臣，也不是彻底的「贰臣」，在汉人中不讨好，而他的新效忠对象也不念他的「功绩」。钱谦益的人生，有一种荒诞的悲剧性，他的人生经历和选择，也反映出人性的复杂性。


**《李煜：做个才子真绝代，可怜薄命作君王》**


李煜死于自己四十二岁生日的那天。牵机药下肚，全身抽搐，头脚相抵，死状凄惨。他曾是醉心文艺、爱好佛理的六皇子，却在国破家亡后沦为阶下囚。这一生短暂，从皇帝到阶下囚**，**正是这种巨大的人生落差，使得李煜后来的作品如用鲜血写成。


《**陆游：丢失的故国和丢失的爱情，让我想念了一生》**


「遗憾」是陆游一生的关键词。


陆游的绝笔之作《示儿》广为人知，「王师北定中原日，家祭无忘告乃翁」。


同样广为人知的，是他一生未改的爱国情怀。而陆游的另一面同样鲜活而重要：除了丢失的故国的「情结」，他一生未放下的另一个「情结」，就是丢失的爱情。


**《王国维：看得透世间所有，放不下心中执念** 》


1927 年，王国维自沉于颐和园昆明湖，留下遗言称「五十之年，止欠一死。经此世变，义无再辱」，震动文化界。对其死因，众说纷纭，有人说他是为了殉清，有人说他是为了殉道，有人说他早已厌倦世事，无求生之志。
