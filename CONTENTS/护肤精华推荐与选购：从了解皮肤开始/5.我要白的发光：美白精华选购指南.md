## 5.我要白的发光：美白精华选购指南
中国有句俗话叫「一白遮百丑」，皮肤白确实可以让我们颜值更有优势，那如何让自己皮肤变白，甚至白的发光呢？今天就来跟大家聊一聊美白的机理和有效成分，以及美白精华如何选择。


**黑色素的代谢：从形成到脱落**
-----------------


传统意义上的皮肤美白，其实就是围绕着抑制黑色素的形成，转运和加速黑色素的脱落而展开的。因紫外线照射引起的黑色素的产生，运输和脱落大概可以用下面这张图来解释：


![img](https://pic1.zhimg.com/v2-1f1a61df7fc59c908a5778b543d9a4d3.webp)

概括起来，大概就是 5 个步骤：


* 第一步，紫外线照射激活表皮细胞产生炎性因子；
* 第二步，炎性因子激活络氨酸酶并刺激黑色素细胞开始生产黑素小体；
* 第三步，通过树突结构，黑色素细胞将合成的黑素小体传递给周围相邻的角质细胞；
* 第四步，逐渐被角质细胞携带并转移到皮肤表面；
* 第五步，表面肤色变深或形成色斑，最后随着角质更新脱落分解。

**黑色素的形成：酪氨酸酶的激活及酪氨酸的多步氧化过程**
-----------------------------


而黑色素的形成与酪氨酸酶的激活和酪氨酸多步氧化有密切关系，这个过程如下图所示：


![img](https://pic1.zhimg.com/v2-dba08a3f8328835faa5d5e30e7765a32.webp)

可以看到，在酪氨酸酶激活后，酪氨酸形成多巴，而多巴在氧化之后继续形成多巴醌，也就是初级黑色素，然后在继续形成黑色素被皮肤角质细胞代谢调。


一般黑色素的代谢周期是 28 天。也就是说，如果被紫外线晒黑之后不用任何美白产品，也不再被任何紫外线照射，那皮肤自己变白大概是 28 天。但我们很难保证在黑色素代谢的 28 天里不会产生新的黑色素，所以一般还是会用美白产品。


最常见的是使用美白精华来抑制新的黑色素产生或者运输到角质细胞，从而预防皮肤进一步变黑，让皮肤能在一个代谢周期内白回来；还有个办法一般针对已经形成的黑色素，就是加速黑色素的脱落，让黑色素能更快的被代谢掉，这样皮肤就能更早的白回来。


另外，因为黑色素的形成需要多步氧化，因此抗氧化对美白也很重要，可以有效防止黑色素的合成。 


**常见的有效美白成分**
-------------


美白成分有很多种，我根据上面提到的黑色素产生的原因、形成的过程以及脱落代谢的过程，列出了不同美白成分的总结归纳：


**（1）保护黑色素细胞因为紫外照射产生炎性因子**


代表成分：甘草酸二钾


**（2）通过抑制酪氨酸酶活性以及黑色素细胞激活**


代表成分：熊果苷，苯乙基间苯二酚(俗称 377)，曲酸，传明酸(又称凝血酸和氨钾环酸)，甲氧基水杨酸钾（俗称 4MSK），


**（3）还原黑色素中间产物多巴，阻碍黑色素的氧化过程**


代表成分：维生素 C 及其衍生物，白藜芦醇，虾青素，谷胱甘肽，多酚类提取物


**（4）阻碍生成的黑色素运输到表皮角质细胞**


代表成分：烟酰胺


**（5）加速黑色素从角质细胞脱落**


代表成分：果酸类，水杨酸，维 A 类，羟乙基哌嗪乙烷磺酸


可以看到的是，大部分美白成分会通过抑制酪氨酸酶的活性或者阻止酪氨酸氧化来阻碍黑色素的形成，来到预防皮肤变黑，让皮肤保持美白的效果；对已经形成的黑色素，大部分通过酸类的成分剥脱角质来加速黑色素的脱落。


家喻户晓的美白成分烟酰胺是少数通过阻碍已经形成的黑色素运输到表皮来美白的，所以有很多人反应停用之后会出现返黑，就是因为黑色素其实已经产生了，只是暂时阻止了转运到角质层而已。


这里再挑几个经典的美白成分做一个详细的介绍：


**熊果苷：**


结构类似酪氨酸，竞争酪氨酸酶活性位点，抑制酪氨酸酶活性，阻止黑色素合成。


熊果苷分成α型和β型，α型熊果苷美白效果更强，但是刺激性也更强，β型稍微温和一些。


**苯乙基间苯二酚（俗称 377）：**


跟熊果苷的作用机理一样，通过抑制酪氨酸酶的活性位点阻止黑色素合成，具有高度的美白功效，但在中国法规要求中用量不能超过 0.5%。


**甲氧基水杨酸钾（俗称 4MSK**）：


资生堂投入了多年研发出来的经典美白专利成分，不仅能通过竞争位点来抑制酪氨酸酶的活性，作为水杨酸的衍生物，还能调节角质细胞代谢，加速黑色素脱落。除了美白，4MSK 对改善色斑和雀斑也有一定的作用。


**维生素 C 和衍生物：**


维 C 是很好的抗氧化剂，这也意味着它很不稳定，容易被氧化。直接添加进精华很可能直接被氧化的一点不剩，上脸时也就一点用也没有了。


为了让产品里的维 C「稳定住」，品牌通常有三个方案：使用特殊的配方工艺，使用稳定的维生素 C 衍生物，以及使用能够保持活性的粉状维生素 C。


**烟酰胺：**


烟酰胺又称作维生素 B3, 作为美白成分主要的作用就是通过抑制黑色素转运到角质细胞来达到美白效果，另外也有加速角质细胞代谢加快黑色素脱落的作用。


其实，作为一个护肤的万能成分，烟酰胺还有抗炎减少自由基伤害的作用。价格上，烟酰胺大概是美白成分里面最便宜的了，由于含有杂质烟酸，一般添加量不超过 5%，以防止烟酸残留照成的皮肤刺激性。


**果酸：**


果酸是常见的一种剥脱角质的成分，很多人用果酸焕肤作为祛痘祛粉刺闭口的成分，而其实对于到达角质层的黑色素，也有加速脱落的过程，但果酸一般不会作为主打，而是作为辅助成分添加到美白配方当中。


**美白精华推荐**
==========


一瓶好的美白精华虽然有主打的美白成分，但是很少会添加单一的美白成分，一般都是通过添加不同通路的美白成分，多维度狙击黑色素来达到美白的效果。


**1. 薇诺娜熊果苷美白保湿精华**
-------------------


参考价：288 元/50g


![img](https://pic4.zhimg.com/v2-40a9ef4edaf0a77d7d7acb8c19d8295c.webp)

**美白成分搭配：熊果苷+烟酰胺+ 甘草酸铵+七叶树皂苷**


薇诺娜这款美白精华是针对敏感肌使用的，添加的应该是β-熊果苷，刺激性更小一些。


同时，精华内添加了烟酰胺、甘草酸铵以及七叶树皂苷，烟酰胺阻碍黑色素转运，甘草酸铵作为甘草酸盐，作用类似甘草酸二钾，通过保护黑色素细胞产生炎性因子起到美白作用，而七叶树皂苷主要是通过缓解红血丝达到去红美白的效果。


总的来说，薇诺娜这款美白保湿精华是对敏感肌比较友好的一款，对美白有需求又担心刺激的敏感肌可以入。 


**2. The Ordinary 维 C 熊果苷精华**
-----------------------------


参考价：124 元/30ml


![img](https://pic3.zhimg.com/v2-0eb2247ef785a749721a3dcfcee83675.webp)

**美白成分搭配：8% 维 C+2% α熊果苷**


添加的是高浓度的原型维 C 和α熊果苷，溶剂是 1，3-丙二醇，无水配方可以保持维 C 的稳定性，但肤感可能比较油腻，并且会有一定的刺激性，适合追求性价比和美白猛药效果的非敏感肌，敏感肌慎用。


原料溶度添加高，性价比高一直是 The ordinary 的特点。这款精华更追求美白效果，跟薇诺娜的熊果苷美白保湿精华选择温和的美白成分刚好相反。 


**3. 城野医生美白淡斑精华**
-----------------


参考价格：450 元/18g


![img](https://pic3.zhimg.com/v2-eab877e8b06253b413d0d2622329fd87.webp)

**美白成分搭配：苯乙基间苯二酚（377）+ 大豆异黄酮+视黄醇棕榈酸酯+四己基癸醇抗坏血酸酯+红没药醇+ 海岸松树皮/芽提取物（2016 老版配方**）


城野医生用的这款 377 原料是纳米包裹体包裹的，能渗透到表皮细胞深层美白。 除了 377 美白成分，还搭配了大豆异黄酮，这是一种类雌激素的成分，有抗氧化作用，也能抑制酪氨酸酶蛋白激酶活性。


视黄醇棕榈酸酯可以通过加速细胞代谢从而加速黑色素细胞脱落，当然这个成分也有抗衰老的作用。


四己基癸醇抗坏血酸酯也是一种维生素 C 衍生物，有还原多巴美白的作用，而红没药醇的美白原理类似甘草酸二钾，主要是减少炎性因子从而抑制黑色素细胞激活。海岸松树皮/芽提取物是一种天然抗氧化剂，也是通过抗氧化还原多巴实现美白的作用。


整体而言，城野医生 377 这款美白精华除了 377 的强大美白作用，其他成分也覆盖了美白的多道通路。但是因为 377 是油溶性的美白成分，所以这款精华肤感上可能偏油腻，更适合偏干性肤质。


**4. 卡尼尔专业全效瞬白美白精华**
--------------------


参考价格：98 元/30ml


![img](https://pic2.zhimg.com/v2-f088dd5377b11862f67bc9f2bcd8f544.webp)

**美白成分搭配：烟酰胺+苯乙基间苯二酚（377）+生育酚乙酸酯+维 C 乙基醚+羟乙基哌嗪乙烷磺酸+水杨酸+柠檬酸**


卡尼尔这款美白精华号称「平价中的战斗机」，价格不到 100 元，美白成分却添加了 7 种，其中就包括 377 这个成分，虽然配方里不是很靠前，但是 377 溶度在 0.5% 就能发挥作用，所以也不需要太高溶度。 


另外，美白成分搭配了抑制黑色素转运的烟酰胺以及稳定性非常好的维 C 乙基醚。维 C 乙基醚是维 C 衍生物中渗透性和转化率比较高的，可以进入表皮深层的黑色素细胞还原黑色素的合成。另外添加了羟乙基哌嗪乙烷磺酸（简称 HEPES）软化角质，加速角质细胞脱落，帮助美白成分渗透吸收。


同时，水杨酸也是类似的作用，能加速角质细胞新陈代谢，从而加速黑色素的脱落。但是这款精华因为添加了酒精，并且没有其他的舒敏成分，对敏感肌可能不是很友好，可能更加适合油皮美白。

参考价：259 元/30ml


![img](https://pic1.zhimg.com/v2-b037028f685ab4905a518f779f0c0c94.webp)

欧莱雅的光子瓶在美白成分搭配上有点类似卡尼尔，也是烟酰胺+377+抗氧化成分+三种不同的酸。只不过抗氧化的维 C 乙基醚换成了鞣花酸，鞣花酸是植物多酚类化合物，协同维 E 抗氧化，同时也能抑制酪氨酸酶的活性。


另外，虽然配方也含酒精，但还添加了泛醇和水解亚麻籽提取物作为舒敏成分，缓解酒精可能带来的皮肤不适，所以轻微的敏感肌也可以使用。


**6. 资生堂光透耀白祛斑焕颜精华液**
---------------------


参考价格：880 元/30ml


![img](https://pic2.zhimg.com/v2-f7cc99f6057922b85ae5bffc7a5e3635.webp)

**美白成分搭配：4MSK+生育酚乙酸酯+维 C 乙基醚+甘草酸二钾+东京樱花叶提取物+柠檬酸**


资生堂这款樱花瓶美白精华主打的是资生堂独家研发的美白成分甲氧基水杨酸钾（简称 4MSK）, 能抑制酪氨酸酶的活性，同时能和柠檬酸协同加速角质代谢，促进黑色素的脱落。

这款雅诗兰黛的「打斑弹」精华主打抗坏血酸葡糖苷（AA2G）和三种发酵提取物 Ferments（酵母提取物，藻提取物和糖蜜提取物），简称 AF3 组合，号称 28 天能淡斑。

AA2G、白藜芦醇和生育酚乙酸酯还是属于传统的抗氧化的美白思路，另外甘草酸二钾属于抗炎通路的美白，最后水杨酸和柠檬酸加速角质代谢和黑色素脱落。

**9. 修丽可臻白焕采精华液**

参考价：980 元/30ml

在成分圈子里，富勒烯是被鄙视的，因为这玩意看着像智商税。


传统的抗氧化剂，要么牺牲自己，要么促进细胞抗氧化，而富勒烯的机理比较玄乎：一个富勒烯分子里面有 60 个碳原子，这些碳原子组成了一个巨大的π键，这个大π键就像海绵一样，能够吸附和淬灭自由基，所以富勒烯又有个名字「radical sponge」，也就是自由基海绵。


![img](https://pic1.zhimg.com/v2-eefa2ec08890d870c60341582a1abafd.webp)

市面上添加了富勒烯的产品不少，卖的最好的算是 HAA 富勒烯精华，成分中油溶性和水溶性的富勒烯各 1%，起到抗氧化的作用。


油水双相的质地肤感不错，坚持使用，能带来比较不错的美白效果。
