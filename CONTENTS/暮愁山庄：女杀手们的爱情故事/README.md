## 简介
暮愁山庄，江湖上最讳莫如深的一处地方。传说，暮愁山庄专门培养女杀手，誓死杀尽天下负心汉。女杀手出身不同，长相各样，性格迥异，进入山庄的理由千差万别。但她们都是人，一个个活生生的女人，她们都会遇见一生难忘的男人。即便，最后她们杀了那些男人。



**《苏霜：毒者，杀心》**


她是江湖中杀人不眨眼的毒娘子。  

他是江湖第一纨绔，云家二公子。  

出生以来，所有人都告诉苏霜，她是杀手，唯有心狠，才能立于不败之地。  

唯有云蔺寅对她说，你不是个狠心的人，何必逼自己心狠。  

苏霜望着茫茫大雪，心想若有来世，她想做个人。


**《段季尤：与宿敌相爱相杀》**


段季尤被称为五十年难得一遇的习武奇才，却遇上了百年难得一遇的天才宿敌柳霄。  

师门同出，段季尤日夜习武，孜孜不倦，却一而再再而三地败给柳霄。  

一直以来，段季尤视柳霄为敌手、心魔、人生道路上的绊脚石。  

可相爱相杀之后，怎么两人的关系更亲密了？


**《萧一笑：女扮男装主仆情》**


杀手领进门，修行在个人。  

萧一笑绝对是杀手师父最头疼的弟子。别的杀手十岁就已经拎着人头满山跑了，萧一笑都十六岁了却连鸡都不敢杀。  

为了完成第一单杀人任务，萧一笑女扮男装潜入武林第一世家的玄阴城，成为少城主的贴身仆从……


**《姬仙愁：背叛与复仇》**


武林大会上，她是江湖第一美人，他是江湖第一剑客。  

人人都说他们是金童玉女，神仙绝配。  

可当她被陷害成凶手之后，他却要以正义之名，将她诛杀。


**《天音：幸好我还没有爱上你》**


她知道山庄里所有人的秘密，她的，她们的，包括庄主的。因为她是庄主身边的丫头。  

她知道朝堂里许多人的秘密，他的，他们的，包括太子的。因为她是备受敬仰的国师。  

皇子们纷纷为了至尊之位拉她入伙，威逼利诱，甚至色诱。  

深处皇室旋涡她将明哲保身，还是不断在危险的边缘试探？
