## 简介
本课程由北京师范大学宁欣教授从历史角度为大家讲解唐诗背后更广阔的的时代背景，让大家不仅仅停留在欣赏诗词的优美词句、美丽意境、诗人情怀等文学层面，发现唐诗新的历史面向，从而更深入、更全面地理解唐诗。


 电视剧《庆余年》中，范闲醉酒之后一夜背诵出上百首诗歌，征服了所有人。诗歌为什么有如此穿透千古的魔力？


我们知道，每首诗有一句最出彩、最打动人心、流传最广的句子，被称为「诗眼」。


其实除了「诗眼」，很多唐诗还有一种「历史眼」。通过历史眼，可以看到唐诗背后更广阔的的世界。


比如《卖炭翁》，大家可能都学过，都概括过中心思想，但它仅仅是表达诗人对当时黑暗现实的揭露、对底层劳动人民的同情吗？


著名唐史学者宁欣教授会告诉你，事情绝不是宦官抢东西那么简单，它至少透露了当时社会两个重大变化。 


宫里应有尽有，宦官出宫抢的为何是区区一车炭？宦官从什么时候开始干宫廷采购的活儿呢？


过去读诗，诗人、史家、读者往往都站在文学或审美的层面，很少注意到诗歌给我们透露出的历史信息。


宁欣教授讲述的唐诗背后的历史，通过 50 节精品课，精心讲解 27 首脍炙人口的唐诗，让大家既领略到唐诗的美，又能捕捉到之前被忽视的历史面向，真正吃透这些唐诗。
