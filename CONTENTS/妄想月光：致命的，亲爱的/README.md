## 简介
最危险的病毒是爱与谎言。它们在我们未曾预料的时候就已经悄悄摆好了局，所有人既是猎物也是猎人，当被迫深陷迷途中时，等待主角的究竟是救赎，还是堕落？


《继承者们》


我是 3 亿财产的唯一继承人，然而当我准备顺理成章接收财产时，现实却给了我狠狠一巴掌。我爸不知从哪冒出了四个私生子女，来分割财产。我好心邀请他们参加聚会，没想到却被他们谋杀……


《无罪谋杀》


故事的最开始，是我在高架桥上的那一跃。


「少女与母亲争吵激动投江」的新闻瞬间引爆整个网络。


被救起后的那段日子，来采访的记者不计其数，但其中最特殊，却是梁川，或者说，梁警官。


当他站在我面前，看似平和地向我问出那句：「你真的是无心投江？」时，我知道，我遇上了麻烦。


《谁是受害者》


我结婚还不到两个月，我的小姑子就想置我于死地。


硫酸毁容、浴缸水淹……我九死一生。


面对我遭遇的危险，我的丈夫却说她只是顽劣。


为将这个阴毒的女孩子赶出去，我被逼调查她的底细，却发现了惊天谋杀大案。
