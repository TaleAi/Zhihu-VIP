女主：我已经是一个成熟且清醒的我了！


当女主不再恋爱脑，千年舔狗举起屠刀：「你不再无情了，真无趣。」


替身文里的早逝白月光，一心只想当天下第一：「男人只会影响我拔剑的速度！」


不敌天降的小青梅，在算盘上十指如飞：「爱情哪有赚钱香！富婆应有尽有！」


权谋文中的咸鱼公主整日躺尸：「远离男人就是远离纷争，高品质摆烂从戒色开始！」
