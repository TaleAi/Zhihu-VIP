## 简介
全网阅读点击量超过 20 亿的《人匠》作者——无色方糖，带来他奇幻惊艳的 13 个脑洞故事，站内累计点赞 51,314。一个个穿梭于时间、记忆、末日的故事主角，在无色方糖充满张力的想象中热烈而真切地活着。你将在故事里，透过他崎岖绚烂的脑回路，走入一个磅礴的幻觉宫殿。


我的超能力是读心，而读心的方法是看到人头顶上显示表达内心的字。这时，一个穿着白衣的女孩从身边经过，脑袋上赫然顶着「死」字，我要对她一探究竟；


夏寒一觉醒来发现走不出自己的教室，无论以任何角度从窗户、门走出都会回到原点，惊慌失措时还在这个诡异的空间发现一个如死尸般的男人，一场趋近死亡的奔逃开始了；


这世上有黑无常，白无常。我却遇到了青无常，那日她告诫我三天换一次皮囊，六天走一趟黄泉，我的魂魄开始附着在不同人身上。我究竟为什么被青无常找上，我又将面对怎样的结局；


小说家方唐，创造了主角小明和小暖。小明的超能力是让所有人讨厌自己，小暖有感受不到疼痛、气味等任何感官的弱感症。一天相爱的两人竟然躲开作者设定，携手走向死亡，最令方唐震惊的是，自己也是其他人笔下的小说人物；


在极度老龄化的世界，对死刑取而代之的，是对杀人犯的 null 处理，即返厂设置，他们会毫无负担地重新开始。杀害我女友的人被 null 了，我无法忍受失去女友的痛苦，提出 null 掉自己的申请，却在这一过程中发现了埋藏已久的真相；
