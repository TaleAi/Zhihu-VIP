## 简介
有什么让人背脊发凉的恐怖故事？


这是一系列发生在普通人身边的诡异故事：


朋友圈里有人直播自杀，我想救下她，但在她的重重指令下，我发现了越来越多不对劲的地方；


厕所对面的宿舍传来阵阵恶臭，里面竟然有个死人，学校里诡异事件频发，这里到底有什么猫腻？


……


表面上看，这些都是惊悚恐怖的故事，但深入内里，我们却可以看见故事下面深埋的人性内核。


希望你可以发现：不管眼前的世界多么可怖吓人，总有些东西值得我们为之活下去。
