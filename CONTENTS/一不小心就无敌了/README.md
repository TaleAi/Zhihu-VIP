简单的口嗨，极致的快乐


人生苦短，爽就完事。开挂、金手指、逆风翻盘······这个专栏，我们将打脸进行到底。


**假千金的阴谋**


养母为救我被车撞死，临终前告诉我，她并非我的亲生母亲。


我这才知道，我居然是有钱人家的女儿。


有一个女孩，代替我在那个家里生活了 17 年。


我知道，养母的死并非意外，凶手的目标其实是我。


而真正的凶手，似乎藏在我的家人背后，若隐若现。


我发誓，要让他们付出代价。


他们想要的，我要一点点毁灭给他们看。


**前男友的暴富梦，破碎了**


为了拆迁款，男朋友家里逼我结婚。


让我签婚前财产也就算了，没想到却嫌贫爱富，觉得我配不上他儿子。


男朋友反手把我踹了，和青梅订婚。 


可我爸有地啊，他们家那边拆不拆迁，我爸说了算……
