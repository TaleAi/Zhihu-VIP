## 简介
本专栏是以男性视角为主的情感故事，讲述了各个阶层不同地位的男主们，面对人生抉择和情感挫折时，灵活机智、化被动为主动，重新走出了不一样的人生。


《荒岛》


一场意外，我与小二十岁的娇妻以及和她有私情的助理，流落在了荒岛之中。


绝境面前，兽欲横行，一张张虚伪的面孔剥落之后，流露出来的是比这暴雨还要冰冷的人心！


道德与伦理的界限正在崩坏，这是我逐渐步入疯狂之旅的故事。


《上门女婿》


一场醉酒后，女神居然让我做了上门女婿。


本想低调做个隐世赘婿，奈何时势逼人，我只能挑起危机重担，独战巨鳄。


此时，却发现杀父弑母仇人竟然就在眼前，是正面决斗，还是徐徐图之，我将何去何从？
