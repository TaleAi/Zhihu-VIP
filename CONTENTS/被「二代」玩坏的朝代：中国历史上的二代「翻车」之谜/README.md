## 简介
中国历史上的二代权力更替或多或少会发生一些「翻车」事故，它们或导致王朝直接覆灭，或影响了后代的历史走向和格局，是这些二代统治者太傻或是过于精明，还是他们的一代没有把后事安排好呢？又或是这背后有着不能说的秘密？本专栏挑选了几个二代权力更替的典型，并以故事为主，论述为辅的形式，为你揭开二代权力更替为何总是“翻车”之谜……



【先秦篇】从二代魔咒开启者到有事叔叔帮你扛


大禹的儿子夏启建立了中国第一个王朝——夏，然而在他死后，他的儿子太康沉湎于声色酒食，不理政事，致使东夷部落入侵，导致夏朝政权中断了四十年之久，由此他也成为中国历史上第一个开启「二代危机」魔咒的统治者，之后商朝的太甲虽不是二代君主，但在前期因为和太康同样的问题被丞相伊尹流放；而西周二代君主周成王继位后，三监之乱差点颠覆周朝，不过幸好他的叔叔周公旦及时站了出来……


* 夏朝：三大爱好引发的灭国案——二代魔咒开启者的日常
* 商周：从流放到叔叔扛一切——从商王太甲流放到周公旦摄政

【亡国篇】富不过三代——二世而亡的三个大一统王朝  

秦始皇统一六国后建立了中国第一个封建及大一统王朝——秦朝，但仅仅只有十四年，秦王朝便分崩离析，由此开创了二世而亡国的先例，后来步秦王朝后尘的，还有结束三国鼎立的西晋，统一南北朝的隋朝，它们的寿命都没有超过五十年，和汉、唐同样都是大一统王朝，但这些王朝怎么到了第二代就被「作」没了呢？


* 秦朝：人生在世如白驹过隙——最知名的败家子，二世而亡的开启者胡亥的一生
* 西晋：所有人都是狼人——西晋八王之乱
* 隋朝：我梦江南好，征辽亦偶然——「文青」隋炀帝杨广的一生

【政变篇】盛世前总要流个血——西汉吕氏专政、玄武门之变、靖难之役


刘邦去世后，刘盈即位为帝，吕雉开始独掌大权，不仅开启了后宫干政的先例，还大封诸吕为王，打破了刘邦当年立下的「非刘姓不王」的规矩，吕雉是如何从一位贤妻良母走向权利巅峰的呢？对于后来的「文景之治」又有何影响？


毋庸置疑，唐太宗李世民和明成祖朱棣都是雄才大略，有所作为的明君，然而他们登上皇位并非名正言顺，而是靠的「逆取」，他们一个是弑兄杀弟，一个是把自己的侄子搞下台，身为二代继承者，是他们背后有许多难言之隐，不得不这么做，还是本身就对皇位有着某种渴望呢？


* 汉朝：这个女人不寻常——第一个临朝称制的女性吕雉的一生
* 李世民玄武门之变：积善三年，知之者少，为恶一日，闻于天下
* 明朝靖难之役：爷爷刚去世就要搞死他所有儿子
