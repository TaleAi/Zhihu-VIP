## 简介
趣解诸子百家之道家，没有乏味的名词考证，没有无中生有的脑补，而是从道家的经典中提炼智慧，助你了解世道格局，掌握处理事务决策的逻辑，完善自我，变成更优秀的人。


2500 年前在中国，有这样一群人，他们交相辉映，**奠定了中国学术思想的基础，完成了中国思想文化的大体，**先秦诸子的群星闪耀，思想流派的百家争鸣，**中国历史上从没有哪个时代，在思想上爆发出如此「集中」、「有干货」的智慧，这是洞悉人心的学问，也是解决问题的学问。**


**但是，诸子百家却让很多人望而却步：**首先，诸子百家是个庞大的哲学体系，数得上名字的一共有 189 家，4324 篇著作，体系之庞大，内容之繁多，似乎怎么也读不完；其次，相隔千年的文言文晦涩难懂， 很多人乍一看，就已经退避三舍了。


萨苏将在本专栏中为你讲解诸子百家之道家，简明易懂、妙趣横生。


**为你讲诉最真实的故事：**杨朱「避世」的由来；老子创造《道德经》背后的渊源；仿佛一幕幕话剧，我们为你情景再现。


**教你拿来就用的智慧：**老子会告诉你如何适应不完美的世界；庄子会教给你如何豁达、潇洒的过一生以及道家「不搀和主义」背后的人生智慧。


**让我们为你拨丝抽茧，提升你的思维与认知，五千年的中国文明，总有一点可以帮到你。**
