## 简介
本专栏将围绕买车这一件事带来购车前、中、后期科普分享。


对许多消费者来说，汽车选购是一件谨慎又困难的事：


* 不管销售报指导价还是配置型号都没有概念；
* 想拿到更多优惠，又不知道到底有多少还价空间；
* 购车之后，才发现这不是自己的刚需；
* 看车、买车、上户手续一波三折，搞到身心俱疲；
* 好不容易选定，最后发现自己的购车经销商不具备售后服务功能。

汽车是大额消费商品，品牌多、车型多，市场竞争激烈，商家宣传也五花八门，所以作为消费者往往会在购车时面临很多问题和困惑，不知道自己购车后会不会后悔，也不知道自己拿到的价格是否合适，后期的汽车养护更是没有头绪。


其实，买车这件事可以很简单。


即使你是初次购车或是对车辆选购完全不了解，也可以轻松理解专栏内的购车诀窍并落地。专栏围绕买车这一件事，从确立购车意向到提车落地再到养车、用车带来全方位的科普。内容并不会只谈理论不顾实践，作者会结合自己出身一线经销商的从业经历，尽可能从现实出发，给出具有参考性的购车建议和技巧。
