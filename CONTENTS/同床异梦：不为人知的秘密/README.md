《不为人知的秘密》


和他在一起三年，我们终于修成正果。


就在这时，她以好友的身份闯进我的生活，还同我分享她和自己初恋的难忘经历。


殊不知，她口中的初恋，正是和我订了婚的未婚夫……


《合租室友》


办公室里都知道我跟他不对付，针锋相对，连户外团建都不敢让我们在同一个组。


但是他们不知道，下班之后，我们是住在同一屋檐下的合租室友。


《心有白月光》


家里给我安排了一场相亲。


我却在那天闹了乌龙，认错人而不自知。


后来我才知道，我错认的那个相亲对象，当天被他的白月光放了鸽子。


我误打误撞，坐在了他的对面……
