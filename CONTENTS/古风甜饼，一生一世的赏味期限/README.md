## 简介
本专栏集中收录古言小甜文。莺飞草长的双向奔赴，纤云弄巧的眉目传情，斗智斗勇的欢喜冤家……总之，玻璃渣已经全部挑到《此情明灭：故人曾扑灭红鸾光焰》专栏里啦，这里只管让你甜到齁！


**《衔春燕》**


公主从来都不知道，在与她成亲前，驸马就已经喜欢上她了。


驸马也从来不知道，在他出征之后，公主后知后觉地爱上了他。


**《相思缚》**


我一个平平无奇女大学生，穿越到熬夜追的书里，被病娇王爷逮个正着！他花样百出想要玩死我，还好我主角光环强大，保住了小命。但是......这个王爷对我越来越暧昧是怎么回事？


**《脸皮厚才是追妻正道》**


作为一人之下万人之上的丞相的千金，左仙仙从没遇到过难题，除了那个莫名其妙出现的「采花贼」，结果这采花贼还是个身份尊贵的小世子。没想到世子脸皮厚的刀枪不入，还要身体力行「烈女怕缠郎」？你你你你别过来啊——！1


**《重生什么的小场面，不虚》**


相府嫡女陈子衿前世痴心错付，死于非命，醒来却发现重生在及笈之前。改变命运，拯救家族，正面硬刚什么的不在话下！可那个前世没什么交集的小侯爷，干嘛总在自己面前乱晃？


**《小贼王又被素质教育了》**


长安城有个做黑白两道生意的清水居，女主阿鲤最有可能成为下一任清水居主人，于是最有可能成为下一任锦衣卫统领的男主楚清河，二人在各自师父的授意下刻意笼络接近对方，二人却渐渐互相吸引……
