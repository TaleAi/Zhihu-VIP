## 简介
以前的老家，黄昏后大家总会围在炉边，点一根蜡烛，每人传递。传到谁，谁就讲一个故事。蜡烛不息，故事不止……


第一夜：黄泉白水村


相恋三年的男友不肯结婚，还带了我的闺蜜回了老家。我气不过，根据男友身份证的地址，找到了男友的老家白水村，却未曾想到白水村竟然迷惑重重。总是说一半留一半的村长，热情到吓人的老婆婆，总是出来扰乱我的傻子……男友身死，我好不容易死里逃生，终于找到了送我来的出租车离开村子。可是，我这才发现最后的真相……


第二夜：女寨


我和男友回他南疆故乡，他不带我去见父母，却找了一个旅馆。这旅馆奇奇怪怪，女孩们都身穿异族服饰，却很暴露，所有人见怪不怪，大家还一起围着篝火开心调笑、喝酒。我以为只是地方风俗，却在第二天醒来后才发现自己的遭遇……


……


蜡烛不息，故事不止。


后续的夜晚，拿好蜡烛，敬请期待。
