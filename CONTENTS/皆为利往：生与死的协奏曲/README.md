# 皆为利往：生与死的协奏曲

## 简介
人性的背面，无非一个利字。


利字当头，生门面前，奏响生与死的协奏曲。  

聚焦人性的背面，透过故事讲述人性之殇。


  



**「多出一人」**


同学的生日宴当晚，我的好友死于非命，见到尸体的那一刻，时间竟然回到了前一天晚上。我试图救她，却一次次见证她的死亡。我陷入了时间循环，我想逃离这个循环。也开始怀疑，怀疑身边的每个人。可现实却一次次告诉我，真相还未浮现。就在我感到绝望时，我最不愿相信的事实，却一点点在眼前展开……


  



**「特殊爱好」**  

试管婴儿第四次失败后，被婆婆大骂到大出血，晕倒在地，婆婆扬长而去，又被追着到月子中心大闹。面对众人指责，她饭菜扔在地上让我爬下来吃，而老公在旁边一言不发。忍无可忍，我在民政局将老公不行的秘密说了出去，却造来了老公的家暴和婆婆的再一次辱骂：「伺候不了男人，是你无能！」


  



  

 **「真相的代价」**  

放假回家，我和父母欢欢喜喜地前往度假山村旅游。但是我却被人推下悬崖，一身重伤……而我的继母也受到了同样的迫害。嫌疑人指向我的父亲，但是他居然失踪了。
