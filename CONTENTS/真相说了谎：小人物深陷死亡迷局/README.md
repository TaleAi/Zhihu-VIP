## 简介
本专栏的悬疑故事着重呈现小人物们与「罪恶」之间的较量。阴险狡诈的偷拍狂、「杀人偷脸」的阴谋者......小人物们深陷迷局，自身难保。他们只有一个疑问：是死，还是活下去？


**- 螳螂**、**蝉与黄雀 -**


这几天每到凌晨十二点，我都会准时收到一封死人的邮件，里面还附有一个偷拍录像。


录像告诉我：我和有夫之妇做那件事的那晚，床底下竟然藏着第三个人......


  



**- 完美替代品 -**


我快递被偷，让大爷给我看监控，画面一出，我冷汗狂冒。


有另一个「我」在视频里咧嘴笑着，那嘴型像在问我：「好，玩，吗？」


当诡异的事情发生，我从来没有想到藏在我身后想要捅我一刀的，是我最爱的人。
