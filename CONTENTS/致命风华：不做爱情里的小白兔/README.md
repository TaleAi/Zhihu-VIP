## 简介
专栏里的故事女主，都有着「致命」魅力，面对爱情，她们敢爱敢恨，主动出击，从不卑微。


夜萧是莫安市的一霸。除了人狠话不多，她办事其实没有什么毛病，就是情感上特别渣。身边的男人那是常换常新。但是有一天，夜萧踢到了铁板，她在渣了一个看起来很纯良的男人之后，对方哭唧唧地把她给埋了。夜萧当场就变成了鬼。


——《今夜天气好，适合埋前任》


我，天下第一美人，后宫三妃之首。先是被人造谣我的宫里有鬼，搞得皇帝冷落我，接着又在我想翻身的时候送了个男人来我宫里，想污蔑我私通侍卫。我心生一计，假如把这男人变成太监……但看到对方也是无辜被卷入，我决定先给他点甜头。


——《男宠邪魅狂狷》


拿到医院那张乳腺癌确诊通知单的时候，汤盈盈的世界变成了灰色。偏偏学弟苏记那个胖家伙来缠着她。她一时没有忍住，见对方胸部竟然比她的还大，恶向胆边生，对他的胸部伸出毒手，结果第二天就被对方宣传得尽人皆知，害得她背上了和对方「热恋中」的名头……


——《学姐别摸我》
