## 简介
帮你先人一步，成为精英新青年。


青年人初入社会，懂得的道理太少。


你以为升职加薪只是工作能力强？


你以为买不起房只是房价太高、自己不够努力？


你以为女神对你爱答不理，是因为自己不够优秀？


其实在成长的道路上，当你对每一段人生进行总结反思，对每一个看起来自然而然却又难以理解的现象进行分解、剖析的时候，都能得出很多新的、有价值的东西。
