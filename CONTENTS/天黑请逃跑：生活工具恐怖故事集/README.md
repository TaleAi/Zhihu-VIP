## 简介
在现代生活中，我们有各式各样不得不依仗的社会工具，我们受益也受制于它。而在这个专栏，作者在深夜，给这些社会工具赋予了谲怪的灵魂，让它们对使用者展开戏谑。6 个现代生活工具的恐怖故事，将让你在毛骨悚然之外，重新思考人与物之间的依附、牵制与共生。


作为现代人，我们的生活几乎离不开各类社会工具。合同是行使社会契约的必需品；手机上各式各样的 app 成为我们的第二外脑和身体延伸；断网后没有信号和 wifi 的我们仿佛失去了全世界……


但如果有一天，它们从工具，变成了诡异幻象的始作俑者，会是怎样的情景？


  




有人用 Swi\*ch 游戏机不断刷新高分纪录，在刷分结束后获得一串字母，组合起来竟然是「JiuMing」；


有人用外卖 app 点餐，在餐点中发现残破的肢体，想向平台投诉时，却发现这家外面店不存在，自己同时涌起了无法言状的欲望；


有人被告诫，要在深夜马路烧掉一份「合同」，然而合同拥有者并未听取建议，而后遭遇了一场阴阳之旅；


有人发现了电视里的隐藏频道，每集都是视角独特的室内戏，直到有一天电视里的人对着镜头方向拿起遥控器……原来自己才是在电视里的人……


  




当夜幕降临，现代生活工具开始蹊跷作怪。


故事中的「TA」正在经历一场身临其境的幻觉，又或是，真实？
