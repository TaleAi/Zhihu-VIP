## 简介
《海错图》是一部怎样的书？为何嘉庆、宣统等皇帝对其爱不释手？这本书究竟有何种魅力？知名科普达人张辰亮带你揭秘《海错图》中的神秘水世界。


人们对海洋的好奇由来已久，就算是在没有高科技的清朝也不例外。清朝画家聂璜就是一位「什么都想知道」的博物种子选手。相传当年聂璜想一探海洋究竟，却怎么也寻找不到满意的海洋生物书籍，于是愤然起行，走南闯北，四处拜访渔人、老叟，最后终于绘制出一本中国古代海洋奇书——《海错图》，而这也成为了这位「博物学家」唯一的传世珍宝。


《海错图》完成后，聂璜就此销声匿迹，史书中再也没有他的下落。直到图谱完成 37 年后，太监苏培盛将其带入宫中，《海错图》才重现江湖。经雍正重新装订，它一度成为乾隆帝的「枕边书」，嘉庆、宣统等皇帝也对其爱不释手。


那么这部图谱究竟有何魅力？书中绘有哪些光怪陆离的海洋生物？其中所述孰真孰假？这些动植物从哪里来，未来又会到哪里去？


可以说，有关中国古代海洋生物与海洋文化的一切答案，都在这部《海错图》里。本节课，知名科普达人张辰亮将与「博物前辈」聂璜进行一场穿越时空的对谈，让我们跟随张辰亮，一同揭秘《海错图》中的神秘水世界。
