## 简介
八年后再次见面，新闻记者余生发现，她当年的同桌，嚣张跋扈的校园问题学生乔牧，竟然成了一名救民于水火的优秀消防员。


【霸道酷拽消防员 vs 口是心非女记者】


她还记得，当年初次见面，乔牧对着她的笑脸说：「笑得这么假，你累不累啊！」


起初，她害怕他，讨厌他。后来，她和他的八卦传得到处都是，人人都说，乔牧爱惨了余生。


隐秘地相爱，又隐秘地分手——余生提出分手的时候，没想到会和他再相逢。


工厂爆炸现场，负责现场采访的余生笑着和满身灰尘的乔牧打招呼，可乔牧却黑沉着脸：


「余生，别来这招，老子现在不吃这一套。」


真的吗？余生不信。她想再试一次。追夫火葬场没关系呀，意气风发的正能量好青年，谁能不爱呢？
