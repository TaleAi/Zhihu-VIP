## 简介
最亲密的关系是夫妻，最遥远的距离是人心。婚前情有独钟，非君不嫁。婚后是否依旧浓情蜜意，还是反目成仇，泥潭深陷？


婚前我们你侬我侬，甜言蜜语。


婚前我们山盟海誓，矢志不渝。


然而当我们一脚踏入婚姻以后，一切却都变了原本的样子。


我们会要一个可爱的孩子吗？有了老大我们又会要二胎吗？


婚后激情褪去，你会出轨吗，会爱上别的女人吗？


为了房贷、车贷我们日日煎熬，这样的生活是我们想要的吗？


拥有自己的工作我有错吗？为什么必须是我放弃自己事业呢？


原来婚姻是这样的啊。


你还爱我吗？
