## 简介
本课程用庖丁解牛的方法，从君主、科举、台谏、财政、政治、福利、城市管理制度和市井文化等八大方面，细致解构宋朝统治的底层结构。并带着观众一起，从民本角度，体验一把活在宋朝的感觉，从而客观辩证的重新看待宋朝留下的真是足迹和教科书给我们的刻板印象。


各位看官，不知道你是否怀疑过，为何电视剧里看到的精致优雅的宋代生活，和历史书上教我们的那个「积贫积弱」的宋朝是如此不同？


为何那个时候的士大夫出言无状，拿生命去大放厥词的时候，宋朝的贴吧管理员们却连个「和谐」的权限都没有？


为何拥有了互联网自由，在家中即可「云」游世界的我们，却还是写不出「山色空濛雨亦奇」那样皎然出尘的游记？


在这一期课程中，我们不从「天下兴亡」那样宏大的君主视角，而是从城市居民的角度，来为大家解析宋朝的真实和刻板印象中的矛盾，为大家剥开近来产生的种种疑问。


相信大家通过本次课程的学习，可以建立唯物史观和辩证的思维模型，并用这个模型去解析任何你感兴趣的一段历史；并从宋朝这个专题的学习中，找到可以应用于当下制度修正和文化建设的启发。
