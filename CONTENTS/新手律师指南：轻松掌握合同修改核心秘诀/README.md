## 简介
本课程由上海段和段律师事务所知识产权部合伙人刘春泉律师主讲，帮你轻松掌握合同修改秘诀，拟出完美无缺的好合同。


草拟或修改合同是律师们的一项基本功，这项基本功既需要良好的文字功底，又需要缜密的法律思维，更需要充分的沟通与理解能力，甚至还要求律师懂得一定心理学和生意经。


所以哪怕执业多年的律师，在草拟或者修改合同时还是如履薄冰，慎重有加。毕竟每一份合同不完全相同，每一次商机对客户都至关重要。了解客户痛点，预知法律风险环节，文本意思表示完整，约定清晰，法律条款完善，关注交易架构，规范合作流程……这些可能都只有有丰富经验的律师才能真正做到。


在商业发达的今天，遇到交易标的与众不同的合同，还要律师临时学习行业相关知识，了解准入资质、资格等。对于战斗在一线的律师，要时刻准备着学习新知识。


有人说，改合同的律师收的是卖白菜的钱，操的是卖白粉的心。为了帮助广大学员轻松掌握合同修改的方法，拟出完美无缺的好合同，本课程特邀请上海段和段律师事务所知识产权部合伙人刘春泉律师，带来合同修改课程，帮你迅速掌握合同修改秘诀。
