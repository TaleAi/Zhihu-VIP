## 简介
本专栏由室内设计师王英俊结合个人经验与理解，带来切实有效的小户型改造攻略，解决小户型装修中的常见问题。


小户型通常被大家称为刚需户型，使用人群多为一些刚结婚的年轻人，或者有学区需求的父母等。该户型内的各个空间都相对紧凑，因此，空间的收纳和改造也要结合自己的需求详加考虑。


通常，一些更大的户型，比如大平层或者别墅可以在空间的使用规划上面更加「任性」一些，而对小户型来说，家庭成员的多样化需求的满足方面更容易受到一定的空间限制，所以，装修过程中对小户型的理解和挖掘就尤其重要。


室内设计师王英俊将结合自己的职业经验，以及个人对户型的理解，给大家分享小户型装修中应当树立的正确认知和切实有效的改造攻略，帮助大家解决空间和思维上关于小户型装修的疑问。
