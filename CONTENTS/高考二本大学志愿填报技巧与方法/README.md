## 简介
二本分数线的考生不知道如何选择大学和专业？本专栏将为你介绍二本大学志愿填报技巧与方法，教你如何从海量大学中挑选出最有潜力那个，选对适合自己未来职业发展的专业，重新拿回主动权。


高考分数处在一个二本的中等分数，面对海量的大学，你是否不知道如何选择？


二本大学的社会认可度高吗？要不要复读？


如何才能在二本院校里选择好的专业？有哪些专业最有潜力呢？


二本的大学生怎样合理规划未来的考研路径呢？


......


本专栏将从宏观角度为你分析二本院校，教你如何判断二本院校的潜力，选对大学、选到适合自己的专业。


此外，还将为你剖析二本大学生的职业发展路径，包括考研深造或者是国内直接找工作、考公务员事业单位编制、出国升学这一系列的未来职业生涯规划。助你圆梦，前程锦绣。
