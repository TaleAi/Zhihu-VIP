## 简介
本专栏的作者是湘雅博士、三甲医院外科医生倩SUR。内容主要分为是否该学医，想学医该怎么做，以及学医的乐趣三方面，希望能给已经学医或者还在犹豫要不要步入医学殿堂的学弟妹们一些好的职业规划的建议，帮助大家在学医的路上少走弯路。


你是否想学医，却被家人阻拦？


你是否常被网上「劝人学医，天打雷劈」的说法全退？


你是否想学医却不知道如何选学校，以及未来如何发展？


本专栏的作者是湘雅博士、三甲医院外科医生倩 SUR。医学界素有南湘雅，北协和的说法。


说起来，作者算得上是医学世家出身，因为早在一九三几年的抗日年代，作者的爷爷就考上了湘雅，并和那个年代的医学前辈们，在战火纷飞，物资紧缺的情况下，在世界顶级杂志上发表文章；成为全球 TE8 病毒分离的第一人；从零开始研制出国产盘尼西林等各种战时药物……


本专栏请到倩 SUR，就是希望她能够利用自己亲身的学习工作经历，以及对医学行业的内行了解，为大家答疑解惑。本专栏将告诉你，你是否适合学医，学医之后要如何学习，如何自我提升，后续的求学和工作要如何规划，以及学医中的各种乐趣。给已经学医或者还在犹豫要不要步入医学殿堂的你一些专业内行的职业规划的建议，帮助你在学医的路上少走弯路。
