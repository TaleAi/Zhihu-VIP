# 我的初恋是纸片人

## 简介
穿成虐文女主怎么办？在线等，急。


因为睡前熬夜看虐文，我被手机砸进了狗血霸道总裁虐文里，成为里面的倒霉女主。作为一个精神正常的普通人，我表示这种脑瘫总裁虐恋一千章的活，加钱都不干！毕竟活着比较重要。


可惜不仅是女二夏慧想让我死，背后似乎还有一只黑手，差点让我离开这个美丽的世界。逃避不仅没用，人可能还没了。你能推我下游泳池，我也会装楚楚可怜反将一军，你敢造谣我师生恋，我就敢跟你对簿公堂，有一说一，打脸黑心莲就是爽。


或许是女主光环，我怎么都没想到，因为被夏慧下药差点失身，在酒店救了自己的人竟然会是室友的哥哥，同门师兄傅琛。于是果断抱住这个在原书中唯一搞赢过男主的大腿，立志要把虐文变甜文。直到那次我被傅琛强吻后搂在怀里，在耳边轻声说：「眠眠，我想让你做我一个人的小姑娘，你愿不愿意。」我才知道，完了，我要和纸片人谈恋爱了。


然而，天不遂人愿，就在我踹走原男主，出任 CEO，迎娶高富帅，走上女主角巅峰之时，却有一个更大的阴谋等着我......
