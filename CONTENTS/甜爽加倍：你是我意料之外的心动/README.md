**卿卿我心**


出国两年后回来，男朋友身边有了一个跟我十分相像的女生。


他说她是我的替代品，说他只爱我。


可每次替身凌楚楚遇到危险和麻烦，周旭都会抛下我去照顾她。


后来我毫不犹豫地离开，遇到了命中注定的他。


也才知道，原来在我看不到的地方，一直有人默默守护着我。


**深夜三人行**


闺蜜给我介绍了一个猛男弟弟。


深夜，我登门拜访，结果敲错成了男朋友的家门，当场上演前任和新欢的修罗场。


前男友跟猛男弟弟每天都在为了我争风吃醋，怎么办？
