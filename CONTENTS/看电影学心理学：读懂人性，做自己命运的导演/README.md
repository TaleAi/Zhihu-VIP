## 简介
本专栏精选十部精彩的高分电影，在每一节中，将会运用心理学知识为你讲述电影人物的人格、选择、困境和命运，每一部电影中讲述的心理学知识都能够应用在你的生活情景中，为你分析自己的原生家庭、童年创伤、伴侣关系等问题提供有益的帮助。


同样看电影，别人能发表精彩的影评获得一片喝彩，而你只记得情节和结局。


当你在生活中遇到心理上的问题想要解决，上网一查不是广告就是「我有个朋友」。


你对心理学有兴趣，可专业的书籍太难看不懂。


资深内容策划星恢带你用心理学的知识解读电影人物，在光影中获得心灵的成长。


高分电影之所以能够被大众接纳和喜爱，是因为它讲述的故事符合人性，触动心灵；


我们之所以会随着银幕上的人物哭哭笑笑，是因为在他们身上，看到了自己。


戏如人生，电影人物的命运他们无法选择，而我们要在看戏过后，从电影人物的身上汲取能量，走出心灵的死胡同，走出悲惨剧情，安排属于自己的戏码。
