## 简介
这个世间有许多说不清楚的事情，看不清的生物，他们就在我们身边，与我们的生活有很多碰撞。现在我带你一起，看看这人间妖鬼事。


我们家世代居住在沂河岸边。


据说，沂河的源头是沂源县，就在泰山脚下。


有传说，这泰山下面压着的就是阴曹地府。要是没泰山压着，那阴曹地府里的鬼怪早就祸害人间来了。


奈何桥下就是忘川河，忘川河尽头就是沂河的出水口。


可能因为沂河发源诡异，所以世代以来，沂河两岸各种诡异事件频发，特别是发水的时候。


每次发水，河水中总能看到三四米长的大鲤鱼，十多米长的大蟒蛇，如小船一般的大王八。


另外，各种死去的尸首更是不计其数。还有不长脑袋的摆渡人、水面上奔跑的小孩儿、岸边排长队的灯火、居住在狐仙庙里的小狐狸仙、从阴曹地府流淌出来的铁棺、排着队视察人间的夜游神……


本专栏记录的，就是这些说不清道不明的诡异故事……
