## 简介
本课程由美女律师罗艺主讲，教你最实用的职场 7 大法律常识。课程中罗艺律师会从三方协议的秘密、协议与合同的区别、违约和解除的处理方式、实习生最低工资标准等 7 个方面出发，为你的实习工作助力，不怕被坑。


如果你正要毕业，你一定忙着搞论文、找工作。


手忙脚乱之下的你是否遗漏了一些必须要了解的知识呢?


学校发的三方协议,你知道具体要怎么使用吗？


实习期要注意什么？劳动合同应该怎么签订才不会吃亏？


如果你是毕业生，如果你有以上的困扰，那就千万不能错过啦！本课程由高颜值技术派女律师罗艺老师主讲，帮你了解应届毕业生必须知道的法律常识。


罗艺老师具有多年一线的法律指导与讲学经验，擅长将法律条文转变成浅显生动的语言，同时结合鲜活案例，你不需要啃大部头的法律条文，也不用到处请教学长学姐，罗艺老师给你一套「拿来即用」的法律常识傍身，不用担心在「小白」阶段轻易被坑。


本课程中，罗艺老师着重讲解了三方协议的秘密、协议与合同的区别、违约和解除的处理方式、个人所得税的缴纳、实习生最低工资标准、实习的意外事故处理、试用期与实习期的区别等 7 个模块，涵盖应届毕业生所需的绝大多数法律常识，让你实习、工作都能轻装上阵。
