《论语》记载了孔子及其弟子的言行，孔子弟子三千，从平民到权相，七十二贤者留名于世而为世人所知，而他们的故事里，也蕴含着职场的若干道理——


颜回为什么能成为领导最爱的下属？子贡资质一般，为啥能赚大钱？子路武功高强，神通广大，却为何受人排挤被杀？子张没钱没背景，为啥能在乱世中当好公务员？
