## 简介
有效沟通有技巧，随学随用。


同样是说话，为什么有的人可以让大家如沐春风，有的人能让大家立马黑下脸色。


沟通，是生活、工作中必须掌握的技能。


人人都知道发生问题要沟通，但到底怎么样才能有效沟通。


我们抱着想要沟通的心，却时常事与愿违。


* 有时候告诉别人自己的感受，别人反而更加不能理解你；
* 年轻人想要愉快地和父母沟通，却不小心踩了雷区；
* 与伴侣到底是讲道理还是讲感情，怎么能明确核心诉求，适当妥协。

本专栏通过几个精选的实际案例，告诉大家沟通在实践中应该如何操作，搭配真实案例，更加通俗易懂，简单上手。
