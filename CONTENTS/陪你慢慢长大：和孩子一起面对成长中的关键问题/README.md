## 简介
不愿分享、不肯写作业、吵着要买玩具、注意力不集中……面对养育挑战，这里有知乎母婴盐究员、「儿童教育」话题优秀答主、教育行业资深从业人员们多年思考、亲身实践后的养育感悟 。


孩子的成长过程，就是一次次对父母的认知和格局发起挑战的过程。


比如：


* 早教越早开始越好吗？
* 有没有必要让孩子上双语幼儿园？
* 对于普通妈妈，幼儿英语启蒙怎么做？
* 怎么才能教育出一个开朗、自信的孩子？
* 小朋友注意力不集中怎么办？如何培养孩子的专注力？

**……**


父母的每一个选择，可能都对孩子影响深远。


这也是很多父母最焦虑的地方。


我是不是错过了孩子成长的关键期？是不是忽略了孩子身上的独特优势？


该专栏从孩子成长中的关键问题切入，提供行业专家、资深从业人员，以及在养育孩子过程中遇到各种问题，摸爬滚打总结出经验的爸爸妈妈们多年的养育心得，分享切实可操作的方法，培养孩子的专注、自律、高情商等多种能力，奠定孩子受益一生的强大内核。


父母成长了，才能帮孩子成长。
