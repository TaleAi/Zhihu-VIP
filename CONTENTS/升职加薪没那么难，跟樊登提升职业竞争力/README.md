## 简介
樊登老师推荐的 10 本职场必读书单。


职场套路深，我要回农村？


别急，这有职场「套路」修炼大法，全方面帮你提升职场洞察力和竞争力。


这些「套路」来自樊登老师精心选择的 10 本书：


* 帮你找到职业方向：《匠人精神》、《干法》、《如何衡量你的人生》、《穷查理宝典》
* 帮你提升工作效能：《刻意练习》、《 OKR 工作法》、《深度工作》
* 帮你提升通用能力：《关键冲突》、《即兴演讲》、《可复制的领导力》

**讲师：樊登**
---------


樊登读书的发起人，前央视节目主持人，倡导「全民阅读」的先行者，旨在为 3 亿国人打造有用的知识平台。


从央视主持，到高校老师、行业讲师，再到企业家，樊登职场的每一步都从容不迫，又精彩绝伦。在这个过程中，通过对管理者心态与思维的透彻了解，和对数百万人职场痛点的精准把握，他得出了职场核心竞争力的组成要素和获取方法。现在，他用 20 堂课，把他的思考倾囊相授。


**适用人群**
--------


1. 初入职场，想得到更多成长建议的职场新人；


2. 正遭遇职业瓶颈，却不知道如何突破的职场「老」人；


3. 想要快速实现晋升，得到发展机会的有志青年；


4. 想要管理好 90 后、00 后员工的管理者。


**你将收获**
--------


1. 从心态到技能、职场素质全面系统提升；


2. 10 位企业界、理论界大咖的亲身示范和全面指导；


3. 深入理解个人成长关键节点，洞悉行业组织运行原理；


4. 掌握事业、家庭、心灵三大领域的资源配置与精进方法。
