## 简介
黎明前的黑暗


凡是过往，皆为序章。


在第二次世界大战后，世界的新格局建立，但这并非一朝一夕、平地青云之事。


在这个过程中，许多地区或多或少经历了混乱、贫穷、张狂与悲痛，才建立了今日的繁荣与从容。


而有些地方，至今仍然未能获得平静。


毒品、黑社会、赌博等社会问题以及军政、宗教冲突甚至恐怖主义等都在这些地方交替发生过。


这些问题是当时社会的一部分，有其存在的合理性，有些如今已然退出历史舞台，有些仍在上演。


本专栏将抛开现在的价值观点，以当时当地的视角去回顾这些地区曾经的过眼繁华与黑白混淆，了解黑暗，才能正视光明。
