## 简介
穿越历史的封尘，巫臣迷恋在人群中望见的夏姬，一眼万年；辩机会不会懊悔自己贪恋红尘，爱上高阳公主；霍小玉还会为了一个负心汉苦等一生吗……本专栏挑选了几位典型的历史女子，故事加论述的形式，为你解开历史上的真实爱情传奇故事之谜。


夏姬、高阳公主、王幼玉、霍小玉、颜令宾、穆萨公主、严蕊……


本专栏挑选了几位典型的历史女子，故事加论述的形式，为你解开历史上的真实爱情传奇故事之谜。


**一、倾世绝恋三十载**


随着郑国的送亲队伍簇拥着新娘的马车缓缓驶来，人群开始骚动起来。年轻的楚国大夫耳边也传来了令他刺耳的闲话：「来了来了，这就是那个和亲兄长私通又把他害死的小荡妇，快些看看到底是有多妖艳。」……


**二、宁负如来不负卿**


「孩子，可是不愿意出家？」男孩倔强得回道：「是！我父母双亡，叔叔觊觎祖产，才出此恶招！」道岳微微点头：「佛门本不该强人所难，可你的叔叔已经为你办下度牒，就算你要还俗，也需三年以后，否则，以你现在的出家人身份，你是无法参加科举，也做不了官的！」……


**三、未知何日是佳期**


只见不远处的树荫后，一位一袭白衣的女子怀抱琵琶缓缓走来，虽是低着头，但其清丽的容颜和脱俗的气度已经让在座诸人为之侧目，连见惯了京城佳丽的夏宫酉都不由得眼前一亮。


**四、似是故人去复来**


三叠唱罢，反复吟唱「开门复竹动，疑是故人」这一句，余音散去良久，才悠悠自语道：「究竟是什么样的男子，能写下这样的句子。」忽然旁边一个清脆的声音响起：「原来貌似孤傲清高的霍小玉是在想男人。」霍小玉立时羞红了脸，低头娇羞的说道：「青蓝姐姐莫要胡说，我只是仰慕他的才华，喜爱他的诗文罢了。」


**五、今生只为一人歌**


挹翠楼的别院里，一个相貌清丽的少女正独坐妆台，凝视着镜中的自己。十六岁，正是豆蔻年华，十二年时间里，精通了琴棋书画诗词歌赋，写出的诗文连长安的才子们都赞不绝口。今晚，自己更是马上就要一步登上平康里的至高处，可为什么，自己却一点都不开心呢？


**六、西出阳关无故人**


相隔三十里的陈村有一个受伤退伍回乡的老兵，已经六十多岁了，据说当年是战神吕布身边最精锐的陷阵营里的，具体做什么的，没人知道。老兵最喜欢的事，就是和人诉说当年的威风事迹，吹嘘如何跟着战无不胜的吕布横扫千军，每次末了被人问上一句：「那吕布这么厉害怎么还败了？死得那么惨？」老兵立刻便住了嘴，面色涨得酱紫，每每拂袖而去，剩下背后的一堂哄笑。


**七、梨花落处君莫问**


女子缓缓抬起头，虽然伤痕累累，面无血色，但仍然难掩秀丽的容颜，女子脸上没有半分的恐惧和畏缩，反而眼神犀利得直直看着高高在上的官员，冷笑一声，嘶哑得回道：「朱熹，枉你还是文坛领袖，一代大家，却用如此下作的手段威逼小女陷害唐大人，你不知羞耻二字怎么写，小女可还知道义字怎么写的！」


**八、风起于末逆水寒**


花船大都停在热闹的地方，方便客人往来，每艘花船也都有自己的特点，或灯火璀璨，或插满鲜花，可谓争奇斗艳。可偏偏就有这么一艘花船，船身素净不说，还远远停在一个幽静之处，这艘花船上，此时刚刚上去几位富家公子，侍女立在花厅说道：「几位公子想必已经知道我们姑娘的规矩，请吧。」说着便伸手示意一旁的书案，书案上整齐地摆放着笔墨纸砚。


**九、相思不断笕桥东**


刘粹刚开心得说道：「太好了，希望下次还能相见。」说罢，将胸前的钢笔取下，双手递给许希麟：「这支笔陪伴我多年，助我考了多次第一，今天送给你，请你收下。」说罢，不等错愕的许希麟有反应，将笔塞到她手中，转身离去。胡庄如也连忙掏出一枚弹壳做的哨子塞给刚才搭话的女孩便头也不回得跑了。
