## 简介
有哪些令人吃惊的历史真相？历史上有哪些细思恐极的事件？历史上有哪些仿佛开过挂的人？


历史由一个人人物，和围绕这些人物发生的事件组成。


但对于普通人来说，我们对历史的理解，可能还停留在教科书，或者是某个时间点时的热搜，关于背后的「为什么」，还需深挖。


这个专栏，将带你了解国家风云的背后真相，以及搅动风云的各位主角。


在变幻莫测的当下，用历史的眼光，审视洪流下的真相。
