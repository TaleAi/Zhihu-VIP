## 简介
桃花笺，只深红一色，情不可言尽。


古有薛涛恋上比她小 11 岁的元稹，情意难平；今有糖中玻璃渣专业户「十两相思」，写尽缱绻销魂姐弟恋。


**— 龙犯桃花 —**


我是一条发育不良的金龙，被貌美禁欲的天帝陛下带回天庭养了起来。


他待我极好，让我做了四象之首，掌东乙青木。


但我却很是不高兴，毕竟我是龙，性本淫；有美人如斯，却吃不着摸不着。我性子被养得愈发古怪，俗称，上火。


为了败火，我只能把他偷出来，藏起来，囚禁起来，一晃百年。


按我的计划，朝夕相处，同榻而眠，他该非我不可才是，谁知……我是个病娇，他也是，不该以常理揣度。


**— 朝 暮 —**


司朝醒来发现自己身上伏着一位如玉少年，貌美且多情。


等等……她好像穿进了书中成了可怜的恶毒女配，而少年恰好是书中最大反派，禁欲又腹黑的拂暮。


书中司朝本是天之骄子，却受尽陷害，被拂暮诬成了魔界走狗。


恶鬼相争，必有一伤。


司朝以为她和拂暮会不死不休，谁知这少年一声声「姐姐」地叫她，命都给她，忍千般嫉恨，求一丝垂怜……


**解锁本专栏，看腹黑貌美小狼狗叫一声「姐姐」，杀伤力有多猛。**
