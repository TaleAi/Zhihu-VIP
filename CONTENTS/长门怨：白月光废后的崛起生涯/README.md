## 简介
他们，汉文一朝最年轻的帝后，因一场误会而分离。她被打入冷宫，决意此生再也不与这个曾经深爱的男人相见。五年后，一纸诏书将她纳入掖庭为奴，一场偶然的相遇，将他们的命运再次紧紧地锁在了一起。


***宠一个人，不过给她想要的，就足够了。***


***而爱一个人，总会有诸多隐忍，百般顾忌和……万分不舍。***


废后长孙秋水在冷宫中隐忍五年，终于等来一纸诏书，她从冷宫废后，被迁居到了掖庭作奴。


眼见新人起，眼见宫门落，眼见她曾经的丈夫、一代君王刘昶眉眼如霜，心硬如铁，她以为他们这一生再也回不到从前，却不想为了让她回来，刘昶也想尽了办法。这对汉文一朝最年轻的帝后，因为误会而分离，又因为彼此深爱而重新走到一起。


他们彼此试探，小心翼翼地揭开过往，却万没有料到，后宫之势风云诡谲，那场导致他们之间分崩离析的误会，虽是奸人之计，却也在秋水的心中留下深深的伤痕。


她怨他，原来在他的心中，社稷永远置于她之前。


他失措，思念她度日如年，以为转过身去背对他们的从前就可放下与她的旧情，却在重逢的一刻，心中溃败如斯。


「秋水，我要你回来，无论付出什么样的代价，我要你知道，这世上，终有一个人在等你，等你回来。」
