## 简介
本专栏将由吴志根博士结合自己的科研与招聘研究生经历，为纠结于读研选择的本科生与苦手于论文写作的研究生解答一系列关乎切身利益的问题，成功读研，顺利毕业。


在职场竞争日益激烈、「内卷」已成常态的时代背景下，读研成为了许多人眼中的一条堪比「黄金之路」的选择。


从教育部的数据也可以看出来，近年来，考研人数逐年持续增加，至 2021 年已达 377 万。随之而来的便是与考研和读研有关的一系列问题：


* 读研究竟有什么意义？
* 非全日制或双非学校读研是否有用？
* 研究生如何做好科研和撰写毕业论文？
* 硕士论文如何成功通过盲审，提升答辩成功率，以及应对抽检？

本专栏由吴志根博士结合自己读研、申博、答辩毕业、招聘研究生等的亲身经历，解读以上一系列关乎研究生切身利益的问题，旨在为面临选择时陷入迷茫的你提供前进的指引，抓住读研成功的关键因素。
