# 狱警往事 3：囚牢里的红与黑

### 存档日期：2022-06-02

## 简介
这是作者北邙做狱警的第 7 年。2014 年，他大学一毕业就进入了监狱系统，调侃说「犯人都是有期徒刑，而自己是无期徒刑」。7 年过去，他早已不是当初的毛头小子，聊起接触过的犯人和案件，他说那是冰冷牢笼下涌动的红与黑。以下是北邙的讲述。


我看见，毒贩得知老家的房子即将拆迁，若无其事地跟来探监的姐姐提了一句「去把阁楼里的东西处理掉」。回到家，姐姐在阁楼床下发现了一只行李箱，打开后，里面竟然是一大一小两具被保鲜膜包裹着的尸体，小的那个是还未发育完全婴儿；


我看见，一个中年女子入狱。她不堪忍受家暴，在晚饭中下了毒，杀死了丈夫和孩子。她说其实她没有想杀死孩子。虽然孩子被丈夫教唆，对她仇视又冷漠，眼睁睁看着她被打，还嫌她吵，但她觉得那毕竟是自己的儿子。于是，她趁着孩子被送去奶奶家的时候动手，谁知儿子太闹腾被送回来了。面对做好的毒晚饭，她想，如果阻止儿子吃的话，一定会被发现，自己被丈夫活活打死。于是她就站在厨房门口，看着丈夫和年仅 9 岁的儿子，一口口吃下她做的毒饭菜；


我看见，老病残监区来了一个健康的犯人。他是一名惯犯，涉嫌参与了 8 起工伤诈骗案，自称初中学历，却比很多医生都更熟悉人体构造，总能恰到好处地弄伤自己，从而骗保，涉案金额高达数百万。在押期间，他把自己的头颈折断了 7 次；
