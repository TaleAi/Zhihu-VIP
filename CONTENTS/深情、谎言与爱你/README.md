## 简介
爱情不是必需品。


少女的初次心动是团艳丽烈焰，炽热且干净、持久也脆弱。


冷落是冰、拒绝是沙、轻视是泥、欺骗背叛则是风暴，终将扑灭会它。


亲情生而携带，友情后天培养。


她们也终会知道，爱情并不是必需品。


磨砺过后，便是涅槃重生。而到时，要好好的再见。


没有一帆风顺的恋爱，没有天生一对的爱人，没有爱情不经磋磨，炙热感情路总有各种荆棘摩擦。


爱情需要经营、需要两个人的认真对待和回应、也禁不起无数次的摩擦、轻视和欺骗。渡过，便是洗涤过后的两个人。


没有渡过，那也要好好再见。
