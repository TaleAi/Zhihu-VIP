## 简介
手机里未点开的陌生红包，充满未知的诡异佛牌，深夜收到的诡异快递……上一秒还是阳光明媚的日常生活，下一秒便误入暗无天日的致命困境。恐怖的另一面徐徐展开，无处可逃。


你在手机里抢过红包吗？有些红包，抢了可能致命。校园的平静被一个陌生红包打破。但凡抢了红包，就要完成荒谬可怕的任务，否则，下场就是死。参与者们由起初的不屑，变成预言成真后的惊恐，直至最后的绝望。这个陌生的「黄泉引路人」，究竟是谁？
