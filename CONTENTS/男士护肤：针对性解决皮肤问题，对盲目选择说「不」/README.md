## 简介
为男士打破认知误区，重建正确护肤观。


作为一位对自己极有要求的男士，你是否徘徊在「护肤」这个领域的大门前踌躇不已？


或许你有亟待解决的肌肤问题，诸如痘痘、黑头、毛孔、色斑；或许你希望自己永远能以整洁干净帅气的面貌示人，但却不知道肤质的提升能给人的精气神和视觉上带来多大的改善，不收拾一下都不知道自己原来这么帅。


但是，对护肤的误解让你不敢做出尝试，你以为护肤就是涂脂抹粉，是女性专属，是「娘炮」才会做的事；又或者你已经踏出了第一步，却受困于专业知识的不足和广告营销的迷惑，肌肤问题没有如期望一般得到解决，信心也大受打击。


护肤是一门科学，需要用对待科学的态度和方法去学习它。本专栏没有长篇大论难以理解的专业术语，只有简单易懂的粗浅基础以及精准简要的解决方案，旨在为各位男生打破这些有关护肤的认知误区，重建正确的护肤观，同时破除各种扰人视听的虚假宣传和刻意误导，教你如何科学有针对性的解决自己的肌肤问题，满足自己的护肤诉求。
