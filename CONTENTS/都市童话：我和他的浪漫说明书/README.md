## 简介
熟男熟女的都市爱情，烟火气十足的甜蜜童话～


擅长写都市爱情的「听晨雪」带着新专栏来啦。呆萌小学妹暗恋高冷学霸，清贫女孩倒追毒舌富二代……谁说飞鸟和鱼没结果？跨越山川河流和时光的海，他们终于在一起。


  



**山川为枝**


我暗恋祁川整整九年。


却因为他一句「兔子不吃窝边草」，把秘密深藏心底，选择相亲结婚。


结婚前，男朋友发现我暗恋的秘密，指责我把他当「备胎」。


我决定服软，却发现他早就移情别恋，还想逼我主动提分手好独占婚房。


此时的我，已经做好了反击准备，却在人生最狼狈的时刻，再次遇到祁川。


再见祁川，我仍控制不住地心动。


彻底沦陷前，我逃了。


准备在一个十八线城市安定下来时，祁川将我堵在窗边：


「南枝，我把家给你准备好了，你怎么能跑了呢？」


原来当年那句话，只是保护我免遭同学围堵的戏言，我却因此和他错过九年。


 


**坠落光海**


一觉醒来，小区封了。


没来得及囤菜的我，望着比脸还干净的冰箱，傻眼了。


苦哈哈吃了五天白米饭后，我打起了前男友的主意。


是夜月黑风高，我鬼鬼祟祟溜进他家。


在他家双开门冰箱扒拉得正欢，他穿着睡衣幽灵般出现在厨房门口……


本想在他家蹭吃蹭喝，却奇妙地跟他复合了。


他妈再次作妖逼迫我俩分手，我跟他妈爆发激烈冲突，让他二选一。


他选了我，还深情告白：「冉冉，我留在这个城市，只是因为这里有你。」
