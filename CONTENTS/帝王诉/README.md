## 简介
当后宫争宠时，皇帝在想什么？


帝王将相，既是遥不可及的天潢贵胄，却也是极为寻常的普通人。也为人子、为人夫、为人父。


有过喜怒哀乐，也有过悲欢离合。


有过缠绵情爱，也有过壮志豪情。


情爱，从来都不该是判断他们的唯一标准。


功业，从来都不该因为感情而被抹煞。


天子从来都不只有爱恨情仇，也该有他们应付的责任与承担。


可以文武定国安邦，也可以执手秉烛夜话。


他们是帝王，也是——常人。
