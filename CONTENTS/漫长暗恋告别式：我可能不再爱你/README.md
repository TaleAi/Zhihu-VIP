## 简介
你身边有比较虐心的爱情故事吗？
或许是一场无疾而终的暗恋，或许是一场来不及轰轰烈烈就散场的恋情。
兜兜转转，终是遗憾错过，将曾经的心动留在青春的时光里。


专栏收录了各种酸涩虐心，最后遗憾错过的爱情故事，关于青春里那些让人心动的暗恋、暧昧、初恋和热恋。


**恋人未满**


卑微暗恋他八年后，我终于如愿以偿，跟他走到了一起。


可我们的生活里，处处有他不曾放下的前女友的影子。


直到发现，他与前女友再次纠缠不清，我终于决绝提了分手。


喜欢一个人可以卑微，但不能卑贱。


后来他后悔，拼命挽回，只是那时，我已经有了自己真正喜欢的人。


······


曾经我们全情投入，认真喜欢着那个人。


当爱意在漫长的等待中一次次被失望磨灭，也许，是时候告别了。


愿每个女孩都能收获一段甜甜蜜蜜的美好爱情～
