## 简介
情绪，会影响我们的状态，改变我们的决策。我们无时无刻不在跟情绪共处，但是如何掌控情绪，我们常常感到很无力。


* 面对工作压力，不知道如何排解，白天很疲惫，晚上睡不着；
* 面对感情，常常因为价值观差异、鸡毛蒜皮，和伴侣争吵。谈个恋爱，比单身还累；
* 一个人的时候，也没办法静下心来。不断自我贬低，让自己找不到个人价值……

如果你想掌控情绪，不妨思考一下，自己与世界的关系。


**我们的困扰，往往在于，与世界的关系太紧密或太疏离。恰到好处的亲密，才是我们活出自我的关键。**


本专栏作者曾旻老师，是北京师范大学心理学硕士，2017 中国心理学最具影响力 50 人，对于情绪调节与心理创伤有深入研究。在这个专栏里，他将会从「原生家庭」「亲密关系」「人际交往」和「自我成长历程」，分享如何做情绪的主人，应对工作生活中，困扰已久的情绪问题。
