## 3.第 3 讲：武则天：一代女皇的成长之路
大家好，我是铁扇。上期的王者英雄录，我给大家讲了讲上官婉儿的故事，那么上官婉儿生命中最重要的那个女人武则天，究竟又是个什么样的人呢？这武则天大家都知道是中国历史上唯一的一位女皇帝，也是女强人的代名词，她的名气大到很多幼儿园的小朋友都有所耳闻。不过说到这儿，您是否了解她作为一个手无缚鸡之力的弱女子，是怎么冲破层层的禁锢和枷锁，一步步走上的权利巅峰的呢？她到底有什么过人之处呢？为了权力和地位，这个女人能狠到什么程度呢？这期节目，铁扇就来带大家全面的认识一个有血有肉的武则天。


武则天祖籍在山西文水，她老爸是个成功的生意人，所以武则天也算是个富二代。不过可惜那个时候商人的地位很低，而武则天祖上呢虽然也有人做过官，但他们家族并不算是世族大家，也就是普通的官家子弟，这个地位就属于咱们现在的中产阶级水平。天生丽质的武则天从小就是校花级别的女神，要是放在现在很有可能就会在街上被哪个导演选中，去演一部大片儿，或者包装一下参加个选秀节目什么的，然后一炮而红。就是这么个感觉到美女。但是在古代，家世不错长得又漂亮的姑娘，就很有可能会成为皇帝后宫团的一员了。这武则天也不例外，她十四岁那年就进了宫，跟着一群同样很漂亮的小姐姐们，管 40 岁的唐太宗李世民叫老公。唉。。。。有钱真好，当皇帝真好，起码这个方面是真的好。呵呵，言归正传啊。


本来吧，这对武则天来说是件好事，毕竟成了皇上的女人，这就已经是于鲤鱼跃龙门啦，一人得道鸡犬升天啦。身份一下就不一样了，要是混的更好说不定还能光宗耀祖呢。好事儿，对吧。可是武则天才刚一进宫就发现自己被坑了，为什么这么说呢？因为她在后宫团里的身份只是个才人，当时后宫的这个职称里边儿最大的是皇后，皇后下面有四个妃子，妃子下面有九个嫔，然后是九个婕妤，九个美人，美人之后才是才人。瞧见没，咱甭说是女团 C 位出道了，这么说吧，如果说后宫就是一台歌舞晚会，那么皇后站的最中央就是 c 位。剩下的按级别往后排，那么武则天的位置基本上就是看不见正脸的小透明。这么说您心里有概念了吧。


这要是光是级别低也就算了，咱可以一步步慢慢来嘛，这是大家都明白一个道理。可是在后宫里，如果没有过硬的后台，还想要往上爬，那靠的只有皇上的独家宠爱了。不过问题也就来了，唐太宗这哥们儿偏偏就不喜欢武则天这个类型的。你比方说，有的人就是喜欢安吉丽娜朱莉那一款的性感大飒蜜，也有的，喜欢像倚天屠龙记里小昭那样的甜美小可爱，更有的口味独特审美清奇的喜欢韩国整形蛇精脸。反正是各花入各眼，喜欢什么样的都有，这都无可厚非。


可唐太宗不喜欢武则天的原因，还不是外表相貌这么表面的问题。而是在内心以及性格上排斥她，据说，有一次李世民在家庭聚会上问大家：「朕最近新得了一匹宝马，不过这马是好马，可就是它这性子太烈，实在是不好驯服，爱妃们，你们能不能给朕出出主意啊？」


武则天一听，心想，这可是我吸引皇上注意的好机会啊，于是起身就回答道：「嗨，这有什么难的，这马要是不听话干脆直接就杀了夹火烧吃吧，省得皇上您心烦。」听到了吧，从这话里咱们也能看出来，武则天的性情确实是相当刚烈了，要不说人家能成为一代女皇呢，跟一般的正常女人就是不一样。可是李世民并不喜欢她这个回答，首先，咱们老李对刚烈御姐这个类型本来就不感冒，人家喜欢的是柔弱娇嫩的软妹子。其次，武则天的这一番话，让老李觉得戾气太重，不愿意亲近。就这样，因为一句话。这武则天就在后宫团呆了 12 年。这么多年期间，别说升职了，说不好听的，根据正史记载，他俩也没孩子，后人就猜测这李世民可能压根儿都没碰过她。更有可能都忘了有这么一号人物存在了。


那么为一个有野心有抱负的女强人，武则天自然是不甘心就此在宫中默默无闻的老去，毕竟李世民已经是快六十的人了，可是自己还是一枝花正当年的年龄呢。终于，机会来了。在李世民病重的时候，武则天被派去和李世民的儿子，当时还是太子的唐高宗李治一起照顾李世民。这个李治和他老爸不一样，不知道是不是因为他天性优柔寡断，还是有什么其他的倾向。这咱也不知道，也没处问去。反正这小子他还就喜欢武则天这种强势的刚烈御姐。而武则天心里也明白这个太子是帮助自己翻身的重要砝码，于是乎，就这么一来二去，李治和武则天俩人就看对眼了。


后来李世民去世了，李治成功的继承了皇位，成为了唐高宗。虽然老爸死了，可是毕竟武则天还是李治名义上的后妈，这俩人要是就这么光明正大的在一起，那实在是有点说不过去。


于是李治就先把武则天送到了尼姑庵去避避风头，等他皇位坐稳了，又赶紧把武则天接回了宫里。那武则天这次回宫算是彻底的扬眉吐气了，要说这武则天对李治是不是真爱，不好讲，但能确定的是李治对武则天那可是上天摘星下海揽月，含嘴里怕化捧手里怕飞，那叫一个万般宠溺，给她爱的都没边儿了。再加上武则天善于耍手段用计谋，也是把李治忽悠的一瘸一拐的，没多久，武则天就成功晋升为了皇后。这回，终于从后排站到了 C 位上。


关于武则天当皇后的这个过程，坊间有一个传闻，据说武则天亲手掐死了自己刚出生的大女儿，然后嫁祸给了当时的皇后，这才成功上位。如果这件事情是真的，那么武则天为了权势能够做到这样的地步，确实是称得上是丧心病狂了。


按理说，做到皇后的位置在当时对于一个女人来说已经是极致了。你看啊，咱翻过来，换位思考的想一想，如果说女儿国的国王，非得死气白咧的把铁扇我拽过去当皇后，那我就觉得挺好，是吧，独得皇帝恩宠，我是要啥有啥，还能整天开开心心的修理后宫那些小崽子们，反正身份我最大，这绝对光宗耀祖啊。是吧?还要啥手表？还要啥自行车？对不对？


可是她武则天就没有铁扇我这么懂得知足常乐。她并不满足于此，她的最终目标是像一个男人一样指点江山，管理国家大事。是人都知道，自古以来后宫干政那都是皇家的大忌，可是偏偏唐高宗李治就是个另类，他优柔寡断，又怂又蔫儿不理朝政也就算了，还三天两头这儿疼那儿痒的屁事儿多，就不上朝，也没法上朝。而这对于武则天来说就是机会啊，因为她自己就很有能力啊，经常帮着唐高宗出谋划策，处理政务。还都让她处理的井井有条，这边儿唐高宗一看这个情况也是乐得清闲，久而久之，武则天参与的事情越来越多，所以在朝中的地位也就越来越高。


到后来，唐高宗干脆就带着武则天一起上朝，这夫妻俩就并排坐在大殿之上，而在历史上，这一幕也被称为二圣临朝。这武则天则被封为了天后，这就是要跟皇上平起平坐的意思。话说都到了这种地步了，差不多得了。是吧。这已经是到极致了吧？嗯嗯，并没有，武则天最终到底是怎样登上的帝王之位呢？关于这些，咱们下期再接着聊。
