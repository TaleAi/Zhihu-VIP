## 简介
帮你掌控下级关系，学会用不同的方法应对不同的下属，和下属一起成长。


和下级沟通，比和老板沟通难太多了。来了工作任务，下属根本不配合，上面领导全指责，完全成了「夹心饼干」。


那么，新官上任，怎么才能让下属信服、领导信任？


**作为领导者，运用好沟通技巧，不仅可以了解情况、沟通思想、交换意见、提高认识、解决问题，还可以畅通言路、集思广益、凝聚人心、增进感情。**


因此，领导者要想成功领导团队，就必须掌握好与下属沟通这一基本领导艺术。


本课程由聚商圈创始人王晓芳重磅打造，通过 23 节职场关系课，帮你掌控下级关系，学会用不同的方法应对不同的下属，和下属一起成长。


另外，本课程将解锁一个个与下属相处沟通的难题，运用经典案例，还原职场真实问题与困境，挖掘事后的本质与解决方案，打通你与下属之间关系沟通通道，成就自己。
