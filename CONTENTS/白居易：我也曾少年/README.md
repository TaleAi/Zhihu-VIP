## 简介
元和十年，白居易「死」在了本该风生水起的年纪，那之后的他，如同一具行尸走肉…… 如此生活，三十年……有人说，人生就是马不停蹄地相遇和失去，这是一本关于成长的书，而书中白居易，又何尝不是另一个我们？


「有时候，我会问自己，到底为了什么努力？」 多少年后，年已古稀的白居易，经常会眯着眼，认真地思考这个问题。


少年的他，有亲情、有爱情、有友情，还有向全世界宣战的勇气。


晚年的他，在纸醉金迷里颓废沉沦，至亲自尽、挚友早逝、挚爱失踪，就连原本那个经世济民的梦想，也在现实面前支离破碎。


元和十年，白居易「死」在了本该风生水起的年纪，那之后的他，如同一具行尸走肉…… 如此生活，三十年……


有人说，人生就是马不停蹄地相遇和失去，这是一本关于成长的书，而书中白居易，又何尝不是另一个我们？


我曾在知乎社区中，写下白居易与湘灵的爱情故事，有幸获赞超过  

160K。在这个专栏中，我查阅了《白香山集》、《白居易年谱》等史料与论著，力求为你还原一个更完整，也更真实的白居易。
