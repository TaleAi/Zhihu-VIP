## 简介
本专栏按照初唐、盛唐、中晚唐的时间顺序，精选具有代表性诗人的 50 篇唐诗，搭配优美赏析，带你领略大唐风采。


**在《唐诗》中，你可以看见这样的气韵浑厚、质朴苍正：**


烽火照西京，心中自不平。


牙璋辞凤阙，铁骑绕龙城。


雪暗凋旗画，风多杂鼓声。


宁为百夫长，胜作一书生。


**也能看到盛唐塞外的风光：**


五月天山雪，无花只有寒。


笛中闻《折柳》，春色未曾看。


晓战随金鼓，宵眠抱玉鞍。


愿将腰下剑，直为斩楼兰。


**还能看到晚唐的凄景悲凉：**


风急天高猿啸哀，渚清沙白鸟飞回。


无边落木萧萧下，不尽长江滚滚来。


万里悲秋常作客，百年多病独登台。


艰难苦恨繁霜鬓，潦倒新停浊酒杯。


聆听经典，感受诗意。让我们跟随卫东老师的声音领略唐诗的风采。


系列专栏特色


* 卫东倾情朗读：由中央人民广播电台播音指导、「金话筒」奖获得者卫东老师独家倾情朗读。
* 严谨易懂的深度解析：在原文之外，还配有深度解析。内容严谨、易于理解。在体会古典原文美感之外，了解古典文学知识、写作背景、诗人生平、常识典故、诗歌意象等基本文学素养。
