## 简介
听了会自杀的《黑色星期天》、看了会做噩梦的《雨中女郎》、全世界的人梦中都出现过一个同样的男人……这些莫名其妙又上头的都市传说，背后的真相是什么？


**《迪奥的世界》**


传说这幅画中小男孩的眼睛是魔鬼的眼睛。如果看着这双眼睛叫出小男孩的名字「迪奥」，就会离奇死亡。👇

**《雨中女郎》**

**梦男**


据说世界上很多人的梦里都出现过这个男人，但不知道他是谁。👇


![img](https://pic3.zhimg.com/v2-0d2779c7588514cbecf16df865848133.jpg)

这些都市怪谈是怎么大范围传播的？


作者深挖全网报道、文献，甚至辗转联系到当事人，用客观严谨的态度，梳理这些传说广泛流行的逻辑。


这些传说背后的真相可能朴实而平淡，但这种在信息洪流中保持独立思考和探索本质的精神，值得我们珍惜和学习。
