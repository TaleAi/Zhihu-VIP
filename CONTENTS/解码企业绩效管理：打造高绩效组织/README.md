## 简介
公司绩效、部门绩效和员工绩效，一网打尽。


对于企业经营管理者而言，需要学习了解企业整体绩效管理背后的逻辑和思路，这是企业管理不可或缺的重要环节。


具体来说，对于新晋管理者，需要学习了解员工绩效管理的流程、思路和方法；


对于部门主管，要能够了解自己的部门绩效如何与公司的经营目标挂钩起来；


对于公司高层，则要懂得如何规划自己公司的战略，并将其形成有效的公司级目标。


本课程特邀人力资源管理专家邱艺老师，用她多年总结的经验与方法，结合实际案例，帮助企业管理者了解绩效管理的全景方法，懂得其背后的逻辑和思路。


另外，本课程也从公司绩效、部门绩效和员工绩效这三个层面的逻辑、方法和工具出发，力图帮助企业中高管、人力资源管理者学习掌握日常工作中所能运用到的绩效管理技巧。
