## 简介
本课程运用 25 讲，通过三大模块，为你提炼 25 招《孙子兵法》中拿来就用的技巧，给你一套在残酷职场竞争中运筹帷幄的计谋宝典、决胜千里的实用指南。


兵法，本就是生存之法。


职场如战场，想要在事业上成为强者，就必须有一套斗争法则，《孙子兵法》是当今最有用的生存指南、是真正的「职场生存宝典」。


本课程用 25 讲，为你提炼 25 招《孙子兵法》中拿来就用的技巧，给你一套在残酷职场竞争中运筹帷幄的计谋宝典、决胜千里的实用指南。


课程分为三大模块：


第一模块「解决难题」：总结你在职场上最常遇到的痛点问题，教你在职场上如何“知己知彼”、一步摸清对手底牌、一招抓住对手弱点；


第二模块是「看透人心」：你会学到如何使用反间计一步洞察对手的内心；巧打烟雾弹、隐藏自己的真实目的让对手摸不着脉；看破对手虚实、运用「诡诈之术」打对手一个措手不及；


第三模块「终身练习」：通过孙子「九变」思想，帮你逃离自己的舒适区、让你拥有居安思危、随机应变的战略洞察力。


助你在工作中赢得未来，生活中增加智谋，用最短的时间，让《孙子兵法》成为你职场上的「圣经」。
