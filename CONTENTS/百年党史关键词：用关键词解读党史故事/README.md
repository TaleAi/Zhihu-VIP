## 简介
心中有信仰，脚下才能有力量。中国共产党自成立开始，就义无反顾地肩负起了实现中华民族伟大复兴的光荣使命，团结带领全国人民走进新时代，迈向新的未来。这一百年，由无数个「关键词」串联，记录着中国发展变化的足迹。


一部百年中国共产党党史，就是一部矢志不渝的初心奋斗史。


中国共产党的成立，给中国人民带来了光明和希望，领导中国革命走向了光明，开创了建设中国特色社会主义道路，为实现中华民族伟大复兴指明了方向。


从「民主与科学」、「外争国权，内惩国贼」到「供给侧结构性改革」、「精准扶贫」，实践证明，中国共产党的领导是历史的选择、人民的选择。走到再辉煌的未来，也不能忘记过去走过的路，不能忘记当初为什么出发。


本专栏从党的百年历史中，精选出了一百个关键词进行解读。不仅对每个关键词本身进行了介绍，而且对其产生背景和重要意义进行了详细阐述，同时通过「知识链接」提供了与关键词有关的党史故事、文献资料、领导人论述等，力图通过「点—线—面」相结合的形式，用一个关键词带出整个历史脉络。
