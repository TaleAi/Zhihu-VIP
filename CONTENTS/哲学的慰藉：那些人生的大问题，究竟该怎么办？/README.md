# 哲学的慰藉：那些人生的大问题，究竟该怎么办？

### 作者：(英)阿兰·德波顿

### 分类：社科 · 成长

### 状态：已完结，共 44 节

### 存档日期：2022-05-24

## 简介
哲学不是躲在象牙塔里的文字游戏，而是帮助人们解决心灵伤痛的良方。用六位哲学家的大智慧，解决六种人生的疑难杂症！


英伦才子阿兰·德波顿向苏格拉底、蒙田、叔本华、尼采等六位伟大哲学家取经，用他们的个人体验与思想学说，为现实生活的郁闷提供慰藉。


◇如果不受人欢迎，该怎么办？


◇如果经济拮据，无法享受多采多姿的生活，该如何调适自己的心态？


◇遭遇挫折却无力回天，该如何走出怀疑自我的阴影？


◇被认为身体、文化或是智力具有缺陷，该如何不受他人目光影响？


◇当爱情的破灭使我们怨怼自伤，要怎么平抚心碎的苦难？


◇当你在人生历程中深陷泥淖，该如何脱离困顿？
