**- 网恋翻车日记 -**


我在游戏里娶了个老婆，技术菜，讲话甜，还特有钱的那种。


有天她突然打字和我说：老公……其实我是男的……


我笑摸她的狗头：好巧，其实我是个女的


  



**- 租了个真男友 -**


为应付爸妈催婚，我租了个男人回家吃饭。


对方配合度很好，我爸妈也很满意，可后来平台却告诉我当天的「男友」有事没来。


而那租来的男人，竟是曾经被我拒绝的小男孩！


  



**- 你是我独一无二的偏爱 -**


我和前男友谈感情，他却觉得我只图他的北京户口。


终于在恋爱三周年那天，终于提出了分手，逃离那个整日 PUA 我的男人。


失意间，我请假去外头玩了几天，却不料在另一个城市邂逅了位可爱小奶狗。


  



**- 夫君简直太抠门 -**


依从父亲遗愿嫁入豪门，谁料夫君却是一个傲娇抠门精？


忍无可忍提出和离，却收到长长的账单，要求我付清钱款后才能离开。


无法，我只能开始打工还钱的日子。


  



**- 当胆小鬼遇上捉妖师 -**


元清歌依照师门传下的画去封印即将出现的鬼王，


没成想在万鬼窟刚见到她，对方却抱着他哭喊着：道长救命！


  



**- 白月光竟是我自己 -**


受心上人之托，我偷摸潜入他白月光的婚宴，企图带走新娘。


却不料屋内穿着红嫁衣的姑娘竟是个大老爷们，


对方告诉我这是场阴婚，他受人之托来为新娘避祸
