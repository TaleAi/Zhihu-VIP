## 简介
影视剧中关于战争的浮夸场景，并不是古代冷兵器的真实面貌。本专栏考古学专家韦正结合自身丰富的理论与实践经验，解码古代战争中关于冷兵器的奥秘，通过声音和有趣味小故事传播真实的历史。


严刀枪剑戟，是历史剧中的常见景象，这些武器是古代战争的符号。


然而，对于这些古人智慧的结晶，你又真正了解多少呢？


有多少兵器上藏着不为人知的秘密？


有多少屡屡出现在影视作品中的描写是不准确的？


又有多少兵种和军团有着我们根本想象不到的绝技？


......


将古代兵器作为了解历史的一个窗口，只有了解真实的历史，才能更好地面对未来。


对于现代的孩子来说，他们最先接触到的历史和冷兵器，并不是通过一些正确、科学的渠道。本专栏考古学家韦正用科学的思维讲解古代历史中的冷兵器，让孩子们了解真实的历史。
