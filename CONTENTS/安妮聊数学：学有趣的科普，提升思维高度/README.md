## 简介
了解数学，我们的生活也将有所改变。


「学数学有什么用，反正离校后也会还给老师。」


真的是这样吗？数学真的那么高深莫测吗？困扰我们的数学问题真的那么复杂吗？


* 数学是我们创造出来的，还是我们发现的？
* 「米」这个计量单位有那么好吗？
* 无穷究竟有多大？
* 我们为什么会被统计数据「绑架」？
* 为什么买彩票中大奖的人会很少？
* 菠萝也懂数学吗？
* 为什么我们要早些开始缴纳养老金？
* 哥德巴赫猜想、毕达哥拉斯定理、费马大定理、「A、B 两点之间线段很短」……这些数学定理真的那么难吗？

数学，无处不在。


它是一种语言，不仅能够帮助我们研究数字、图形以及掌管宇宙运行的某些定律，为我们理解周遭事物、解构和预测某些现象提供一种方法，而且它还在贸易、商业、生物学、化学和物理等方面发挥着无可替代的作用。


对数学有所理解，我们也将拓宽思维，对事物有更深刻的思考。正如数学家克莱因所说：



> 数学是一种理性的精神，使人类的思维得以运用到最完善的程度。


本专栏的作者安妮·鲁尼，著述颇丰，且涉猎题材广泛。她的书很受欢迎，曾荣获安万特科学图书奖。让她带你从全新的角度认识数学，回归自然，发现数学的趣味性，了解数学的本质。
