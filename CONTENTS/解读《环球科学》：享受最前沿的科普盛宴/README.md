## 简介
知名科普作家汪诘将用自己独特的风格为你深入浅出地解读《环球科学》，给你带来全球科学前沿的精华动态和知识，权威性与趣味性兼具。每天 30 分钟，一起了解全球科学前沿，让你在听故事中理解科学背后的科学。



希特勒为什么没能造出原子弹？  
中微子能带来下一场物理学革命吗？  
生命是否真的起源于海洋？  
意识的分界线到底在哪里？  
未来货币和区块链技术有什么关系？  
所有的疑问，  
都将在这套课程中得到解答。 


![](https://pic4.zhimg.com/v2-7ee0d1391cf6b57516d99ac5d769a817.jpg)


本课程来源于《环球科学》  
但绝不是照念文章，
而是作者把文章吃透消化之后，  
再补充大量的背景知识，  
以一种符合普通人口味的方式，  
通俗易懂，深入浅出地为您讲解。 


《环球科学》是世界上历史最悠久的科普杂志《科学美国人》的中文版，  
158 位诺奖得主为其撰稿，  
爱因斯坦、爱迪生、比尔·盖茨都曾是其读者和作者，  
这本杂志影响了一代又一代的科学家和科技人，  
每篇文章都是科普精品。  
![](https://pic4.zhimg.com/v2-4c694f99d414d49bbb4ffd22f04c3a9f.jpg)


在本课程中谈到的科学，  
包含了「科学知识」与「科学精神」  
它并不是简单的知识点堆砌，  
而是凝结了作者的智力创作。  
帮您在浩如烟海的科学知识中，  
筛选出最优质的内容，  
让您在潜移默化中领悟科学精神，  
并且获得认知升级。
