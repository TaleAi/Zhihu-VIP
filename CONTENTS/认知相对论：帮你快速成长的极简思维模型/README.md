## 简介
一起成为一个有智慧的人，在达尔文时代找到生存的价值。


**开动脑筋，想想下面三个问题的答案：**


1.我们为何对必须要做的事情难以成瘾？


2.某地风景优美，空气清新，被认为是疗养胜地。可是该地区肺癌发病率却比其他地区高很多倍，为什么？


3.五个囚犯先后从 100 颗绿豆中抓绿豆。抓得最多和最少的人将被处死，谁的存活概率最大？


这三个问题的答案，分别对应着自我实现与目的论，相关与因果和不完全信息博弈。


**现在，把这三个问题换种问法，你会发现它们的答案对你的重要性：**


1.我想对学习上瘾，我该怎么做？


2.大数据时代，如何不被表象迷惑，通过数据快速得出正确结论？


3.在信息不充分的情况下，如何做才能让自己的利益最大化？


你想知道答案吗？本专栏将会带你一起探索。
