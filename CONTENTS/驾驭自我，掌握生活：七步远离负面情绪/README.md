## 简介
远离负面情绪，重新掌控自我。


你也在经历这样的痛苦吗？


* 乱发脾气：经常因为小事生气，总是对伴侣和家人发泄情绪，冷静下来后十分懊悔.....
* 焦虑拖延：工作繁多，十分焦虑，却进入不了状态，直到 deadline 到来.....
* 烦躁抑郁：生活中一旦遇到不合心意的事，就感到特别烦躁.....

别再让自己被负面情绪控制了，你需要学会驾驭情绪。


资深心理咨询师清流老师将教给你一套科学有效的情绪管理法，帮助你真正成为管理情绪的高手，驾驭自我，掌握生活。


通过这套科学的方法，你将会：


及时觉察到自己的情绪问题，在负面情绪进一步发酵之前轻松消解；


快速回归理性，学会客观分析给自己带来情绪困扰的原因；


合理地应对自己出现的情绪问题，减少给朋友和家人带来的伤害；


每个人都需要学会情绪管理，更好地 hold 住自己的工作和生活。


  




课程特色：


* 方法科学：这套情绪管理法基于认知行为疗法，它是心理学界主流的处理情绪问题的方法。
* 可靠有效：经过无数次科学实验和调查验证，被证明可靠有效，在临床咨询中得到了进一步验证。
* 简单易学：课程聚焦于解决关键问题，每个步骤都有程序化的操作方式，容易上手。
