## 简介
无厘头的、浪漫的、奇幻的、科幻的、脑洞大开的...... 关于欲望、勇气、爱情、时间、或是超能力，欢迎来到宇宙怪谈故事集。


【神珠】  

 杨为生的房间凭空出现了一颗可以传送的神奇珠子。这颗珠子究竟从何而来？又究竟会给杨为生带来什么呢？这究竟是潘多拉的魔盒，亦或者是天使的馈赠？


【天才发明家的对决】  

 发明家阿杰和阿云一直在进行发明竞赛，却始终不分胜负，这天，他们迎来了他们的第五百零一场对决——一个有着奇怪需求的客人。


【启示日】  

 二零二四年，一个名叫陈威的人觉醒了「吃啥补啥」的超能力，似乎可以突破人类极限的他，却因为时间限制而苦恼…


【爱在街角下雨天】  

 一九九四年十二月二十二日，我出生在南方的一个小乡村里，待我些许记事开始，便总会在街角下雨天的时候，遇见一个孤独的女孩，她为什么永远会在雨天出现？她在等着谁吗？


一起登上故事的飞船，向奇幻世界出发吧！
