## 简介
以热血的故事，带大家回到那个艰辛的年代。


新中国成立以前，内忧外患，百业凋敝，民不聊生。


当战争降临之际，冲在战场最前面的，毫无疑问是军人。


为了救亡图存，他们抛头颅、洒热血，为了振兴中华，他们前仆后继、英勇就义，用自己的鲜血乃至生命谱写了豪迈悲壮的人生篇章！


  



用大刀对抗日寇枪林弹雨，用血肉之躯抗衡飞机大炮，无数场抗日战争的一个缩影


——《赵登禹：让日寇闻风丧胆的白日噩梦》


琼崖革命第一人，用生命孕育了当地的红色精神


——《冯平：没有军衔的将军》
