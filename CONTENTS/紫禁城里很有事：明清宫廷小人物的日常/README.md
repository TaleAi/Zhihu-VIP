## 简介
紫禁城里的人是怎么过日子的？跟宫斗剧里的差别大吗？本专栏里，作者王一樵通过还原太监、宫女、侍卫这些常在小说、影视剧里出现的，宫廷小人物的故事，带我们了解紫禁城里鲜为人知的日常生活。


除了《甄嬛传》《步步惊心》里的后宫争宠、前朝夺权，住在紫禁城里的人们，日常生活是怎样的？ 


宫廷内为什么有那么多太监、宫女投井自杀？


宫廷流行的衣着时尚如何在市井传开？


清朝的快递员过着怎样的生活？


如何把自己的亲戚带进皇宫？


宫女、太监、工匠、紫禁城守门兵丁、驿递马夫、皇宫仆役、不得宠秀女······这些生活在紫禁城里的小人物，他们才是本专栏的主角。


作者王一樵查阅明清内阁大库档案、明清两朝实录及清代养心殿造办处、军机处、内务府等机构一手档案资料，通过这些小人物的日常，为我们展现了一幅充满生活气息的紫禁城图景，也让我们透过这些小人物，看见更为丰富的历史、社会、经济、民间信仰与日常生活图景。
