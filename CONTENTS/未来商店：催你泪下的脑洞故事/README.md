# 未来商店：催你泪下的脑洞故事

### 作者：木水火

### 分类：脑洞 · 奇幻玄幻

### 状态：已完结，共 6 节

### 存档日期：2022-06-16

## 简介
带你解开身边「习以为常」的谜团。


**「不知道你们有没有发现：在你的城市里会有一种奇怪的店，它可能开在校门口的小吃街上，门脸上挂着塑料牌子，上面写着『芳芳服装店』——**


**玻璃门上，挂着一把大红色自行车锁；透过门往里面一望，店里挂着一些女式衣服，一看就过时几十年了。**


**经常有路人这样议论：『芳芳服装店』？这是我妈小时候那年代的商店名字吧！这地方租金那么贵，卖这种过时衣服也就算了，店家还不赶紧开门搞促销？大门紧锁算怎么回事呢？太诡异了。**


**不好意思，这些吐槽我都猜到了。**


**我就是开在某城某角落的『芳芳服装店』的老板。**


**你别只相信看到的事情。在表象背后，往往是你猜不到的惊奇真相！**


**比如说，我做的买卖一般人想都不敢想；我的货，可都是这个世界不该存在的东西，不信你就进来看看。不过白天可别来，等天黑了，我们再坐下来，慢慢聊。」**


来自 2060 年的药剂师——木水火，因为特殊的原因穿越到了 2020 年，开了一家看上去像是过时女装店的「未来商店」，专门售卖来自未来的奇特物品。


这位「人形哆啦 A 梦」，为生存在 2020 年的人解决了许多匪夷所思的困境：


被偷窥的女士希望偷窥者忘记自己赤裸的身体，赶来向木水火求救；一个程序员的猫主子走丢了，他怀疑是流浪猫拐走了它，就来寻求能听懂「猫话」的药；年轻夫妇觉得养孩子又吵又麻烦，希望获得一个「方便婴儿」，像泡面那么方便的那种；一个运气超差的极品倒霉蛋，毕生最大的愿望就是逆天改命，于是来求购改善运气的药物……


面对这些「奇葩要求」，这位 2060 来客该如何应对？


而他自己的来历究竟如何？他心底的夙愿又是什么？
