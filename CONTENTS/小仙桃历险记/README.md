## 简介
可可爱爱仙女遇到心机冷面魔王。


三年前，他将我关在仙障中，替未婚妻受了天雷。


三年后，重逢，他红着眼道：「桃桃，你就没有想和我说的话？」


我看了看远处那个威名四震的上古仙君，莞尔一笑：「有啊，作为昔日同窗，元律仙君不如也来认识下我的男朋友。」
