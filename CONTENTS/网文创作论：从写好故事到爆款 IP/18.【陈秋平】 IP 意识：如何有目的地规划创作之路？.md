## 18.【陈秋平】 IP 意识：如何有目的地规划创作之路？
IP 这个词其实是中国人发明的，它的意思呢，是创意产业里的无形资产，或者叫做知识产权，也可以理解为在创意产业当中，比如，游戏、小说、影视，它的品牌效应，那么所谓创造 IP，其实就是提高一个作品它的知名度，它的产权价值。


我们经常所说的大 IP 指的就是特别有影响力、特别具有商业价值的文学品牌或者是知识产权。


那么创造有价值的 IP，就是把一个创意作品，它的知名度、它的商业价值、它的影响力给提高。 


那么怎么在小说里去开发和创造它应有的 IP，下面我们给大家几点建议。


一，潜心开发具有 IP 价值的创意和项目。我们作为一个小说的原创者，其实我们更多的是一个文学创作者。我们很少用商业的价值去评估我们要写的一个小说或者小说当中的人物。但是到了现在到了今天，我们如果把我们小说的它的文学价值和商业价值要是认识透的话，我们发现一个小说在创作之前，在构思的这个阶段，可能你就要考虑到它的商业价值，那么它的商业价值是哪些？有可能是小说本身的人物形象。有可能是小说的模型，有可能是交给小说的故事，以及它所代表的许多文化符号和它的其他影响力。


比如说，我们在写一个小说的时候，我能不能够把小说的人物符号化、图形化、个性化。以至于我写完了小说以后，我可以把这个小说卖给其他的领域来进行二度创造，比如说，我可以把它卖给漫画，卖给动画片，甚至未来可能做成某些装饰品，或者印在 T 恤衫上，或者我这个故事不仅仅只有人物，可能还有人物的关系，人物的对峙，还有它所生活的环境，它所处的年代，那些东西它本身也有商业价值，也有可扩散可传播的价值，这些价值的总和就是它的 IP 价值。 


如果我们在写一部小说之前能够兼顾其他，我们可以提高这个作品当中它的传播价值和商业价值，其实就是在创意阶段找到了它的 IP 价值。在这样的前提下去写小说，我们可以指望未来的小说，它的商业价值可以发挥到最大。


二，努力写好具有高度可读性的原创小说作品。毕竟我们是小说作者，我们不是漫画家，我们不是影视编剧，所以我们对 IP 价值的最大的贡献是对这个作品的故事的贡献，当然包括他的人物的贡献。所以努力写好一部好的小说，尤其是具有相当高度的可读性和独特性，这样的小说是最终得到 IP 开发得到 IP 扩散的一个非常重要的前提。 


换句话说，做好自己最擅长的事儿，把小说本身写好。


第三，深入研究互联网传播，寻求高效率的作品推广。这什么意思呢？因为我们处在一个非常伟大的互联网时代，一个作品假如借助了互联网，尤其是移动互联网的翅膀，它可以得到前所未有的传播，受到非常巨大的收益。反过来说 IP 它也是在网络上来流传来形成的，并且在网络上去提高它的价值。如果说我们要提高我们作品的 IP 价值，其实就必须提高它的网络传播和影响。你要把这个作品传得很远，你就得了解传播学，你就得去深入地研究新出现的这些传播模式。 


比方说现在新出来的一些平台，像抖音，或者今日头条，它的算法是什么？它是怎么样去推荐一部新的作品的，因为现在的推荐越来越智能化，越来越计算机化，模型化，它会根据很多元素、很多参数去找到你这个作品的最佳的定位人群。比如说它会根据你所提炼出来的关键词，你的标签去找到关心这些标签的那些潜在的读者，它也可以根据那些读者所感兴趣的其他作品再去传染，再去病毒性的去传染，去传播你这个作品。这些算法在互联网的很多平台上已经有了，我们必须去学习，必须去研究。如果我们研究的好，我们就可以用那些因素，那些元素放到我们的创作当中去，让我们的作品具有更加的传播能力，从而提高作品的潜在的 IP 价值。


第四，创办作品的自媒体，仔细地经营自己的一亩三分地。我们现在广大的小说作家，他的作品的出口主要是纸媒的刊物，出版社和网络文学网站。但是你再把你的作品拿到别人的平台，别人的出版社和媒体媒介上去的时候，你一方面是借助了它的传播平台，提高了你的传播速度和传播广度，同时你也受制于它，它也成为了你的某一种束缚。 


换句话说，如果你遇到一个非常好的平台，或者说你正好在这个平台上施展得很好，那么，这个平台对于你来讲就是一个非常有效的传播。但是如果这个平台封杀你，或者这个平台出了什么其他的问题，你以前的铺垫就可能会荡然无存。所以我们建议小说作家不仅在公众平台上找到你你自己的传播渠道和平台，更要同时在自己的身上去下功夫，换句话说，你要碰见和经营你自己的自媒体，你的作品既可以在其他平台发，也可以在自己的平台发，如果你在自己的平台上发，你会拿到最好的无形资产，你会收到最稳定的关注度，甚至可以培养你最忠诚的粉丝。 


第五，加强粉丝互动，创造良好的网络社群生态。我们知道小说的作家以前是很高大上的，许多读者想跟作家去交往，往往只能给作家寄信。这些信发到了出版社或者是杂志社，在由杂志社转到作家的手上来。而且面对浩大的这种来信，多数情况作家是不回复的。那么这种不回复，这种高冷，有可能在现代的传播条件下，就变成了你的 IP 成长的一个约束，一个束缚。现在因为我们有了互联网，移动互联网，有了网络文学平台，所以你可以迅速地就看见了，哪怕是只言片语的粉丝和读者的那些反馈，那些评论，甚至你不排除有一些非常忠诚的粉丝，会对你的作品提出要求，提出对你的作品的批评。 


如果你在现代生活当中，你是一个非常热情的作家，是有助于跟你的粉丝和你的读者来互动的。这种有效的互动，这种营造的非常良好的这种社群的生态，对于扩大你的知名度，对于保住你的那些忠诚的粉丝，从而锻造出非常有商业价值的 IP，这都是有贡献，都是有好处的。


第六，勇敢地走向前台，树立作者自己的个人品牌。在传统文学当中，我们对作家是仰视的，虽然他们有至高无上的那些光环桂冠，但是我们觉得他们远得像一尊神，甚至觉得他是好像在纪念碑上的人，他不是一个真人，但是当代的作家他可能是完全不一样的。 


当代的真正有名的有影响力有话语权的作家，他本身就应该是一个品牌，本身就应该是一个领袖，是一个网红，是一个偶像。有许多作家总愿意躲在文字后面做一个安静的美男子，但是事实上这是对你的 IP 的成长，IP 的开发是有局限性的。所以我们建议的是一个当代的作家，一个当代的小说作者，就要勇敢的走到前台，正面的去和粉丝互动，去和各种媒体打交道，去找任何一个可能的机会去推销你自己，打造你自己作为作家的品牌，而不仅仅是作品的品牌。 


如果做到了这些，我们的小说作家就可以创造出非常有社会价值和商业价值的 IP。
