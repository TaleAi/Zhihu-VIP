## 简介
不可能犯罪推理大神「陶球霸」出山了！地方志怪、恐怖传说、灵异现象、密室杀人……看似不可思议的情形背后，隐藏着计划缜密的犯罪阴谋，人性的阴暗与卑劣暴露无遗。试问你与凶手易境而处，能否经受住考验？



1925 年，我曾祖父从日本医科大学毕业后回到南京。我们家在比国公使馆附近有一个西药堂，打算让我曾祖父去坐堂。但留学日本期间，他被横沟正史带偏了节奏，特别着迷那些稀奇古怪的案子。那时社会乱，南京周边相当一部分三教九流遇到麻烦，不愿找警察就来找他。他和记者萧芝仪搭档破获了不少案子，不过一些因涉及地方权贵，或太阴暗残忍，无法见报。


（本专栏基于真实故事改编）


**- 无头新娘案 -**


在一座荒僻的旅馆内，一对新婚夫妻几分钟之前还在恩爱说笑，几分钟后，丈夫突然在雨夜走出旅馆，头也不回地走进旅馆对面苍茫的千里野人山中；而妻子则在密室内自尽，死前穿着华丽的古董嫁衣，妆容骇人。更诡异的是，这件古董嫁衣的上一任主人也是自缢身亡，死后还被人割掉了头颅……当时旅馆没有任何异常发生，夫妇平时感情也很好，没有人猜出事情真相，直到十年后案件才被破解。


**- 红十字迷案 -**


八个海员驾驶一艘渔船出海，想要寻找明代沉没的西洋宝船，不料寻船之事一无所获，反倒被海盗打劫。当他们好不容易回到海岸时，却发现出航时的八人竟然少了一人。那人尸身在船的底仓木箱内被找到，已被斩去头颅与左臂，掏空五脏六腑，右手掌心隐隐有一个血红的十字。


死者之死，疑与一艘清代失踪于海上的幽灵船雷神舟有关。当这艘渔船重新起航，打算返回上海母港后，竟多次在深夜遇上那艘可怕的幽灵船雷神舟，雷神舟的破帆上正好画着一个大大的红色十字！


**- 人骨祭奉案 -**


法租界一间废弃的澡堂地下室内，发现了一具衣着华丽的女尸。该尸体不仅面部被划烂，双手被斩下，手指剔肉成骨，摆成特殊形状，周围还有有符纸灰烬。该谋杀案颇具邪教仪式感，与当时苏南一带农村流行的圣母邪教似有关系。


一案未平，一案又起，我曾祖父刚受探长委托调查此案，又一具女尸出现，死亡方式与第一案如出一辙。此后几乎每一天，都出现一名死者。最后共有五人身死，经查实，五人都是在南京颇有身份地位的人。
