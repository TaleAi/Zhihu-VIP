## 简介
本专栏由自北京三甲医院口腔科医生郝泽良，为大家介绍电动牙刷的发展、功能、使用与选购，帮助消费者全方位了解电动牙刷，选择到适合自己的那一款。


电动牙刷是懒人不想动手刷牙的「杰作」吗？


电动牙刷比普通牙刷贵了那么多，刷牙效果又能好多少？


市面上琳琅满目的电动牙刷，究竟如何选择？


电动牙刷最早是为手部运动障碍的残障人士发明的，但是，电动牙刷可不仅仅是解放手部而已，除了清洁效率远高于手动牙刷以外，它还具有很多的优势。不过，电动牙刷的各种附加功能到底是商家的噱头还是「一分价钱一分货」，这需要我们一探究竟。 


为此，本专栏邀请到来自北京三甲医院口腔科医生郝泽良，为大家带来电动牙刷的发展、功能、使用、选购相关的介绍，希望大家对电动牙刷有个全方位的了解，针对不同的牙齿和牙龈情况选择适合自己的电动牙刷。
