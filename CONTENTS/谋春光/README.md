## 简介

「王妃！上天怜见啊！王爷回来了！」


「王爷怀中还抱着一个昏迷的姑娘！」


「王爷和那个姑娘一起晕了！」

自认有上帝视角，主动为男女主铺路，却引起了原书中被恶毒女配疯狂虐待的病娇疯批谢浸池的注意！


我逃，他追，我插翅难飞。


而随着男二谢浸池的步步紧逼，原书中不曾存在的人物竟然一一解锁，剧情彻底不受我的控制了——
