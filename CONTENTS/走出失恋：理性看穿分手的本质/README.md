## 简介
认清分手，才能明白爱情。分手后，「我还是忘不了她/他」、「我不敢再谈恋爱了」……我们对爱情投入了过高的期待，所以分手成为挥之不去的「梦魇」。本专栏将从社会学的视角，围绕分手的本质、分手的原因、分手的方式等多个角度为大家解读，发现亲密关系的本质。


当爱情中独立和自信说烂了大街，分手成为新时代爱情带给我们的最大困扰。


「男朋友和我说分手，我觉得自己活不下去了。」


「分手半年了，我还是忘不了他。」


「我已经不爱他了，可我不敢和他说分手，怎么办？」


「自从分手后，我好像不敢再谈恋爱了。」


……


我们对爱情投入了过高的期待，所以分手成为挥之不去的「梦魇」。


但是，当我们从内心世界走出来，站在人类所构建的社会结构及文化角度看待爱情时，我们会发现：分手和失恋，没那么可怕；爱一个人，也没那么困难。


本专栏中，我们将从社会学的视角，围绕分手的本质、分手的原因、分手的方式等多个角度为大家解读，发现亲密关系的本质。


认清分手，才能明白爱情。
