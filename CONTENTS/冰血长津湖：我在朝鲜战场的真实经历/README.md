## 简介
在战争这个宏大的命题下，我们同样需要看到普通人的故事。那些牺牲在冰雪中的战士，不应该被遗忘。回忆历史，不是为了重蹈覆辙；回忆战争，更是为了珍惜来之不易的和平。 


本专栏是由朝鲜幸存老兵亲历口述的真实历史。


长津湖战役是世界军事史上最惨烈的一场战役。在朝鲜长津湖畔零下 40 度的冰雪中，中国士兵冻死、饿死 3 万余人，比战死的多一倍。


这些老兵都是战场上的普通军人，他们的讲述，让那些冷冰冰的数字变得立体，变成一个个活生生，有温度的人。


你儿时记忆中的某位前辈，或许就是他们中的一员。
