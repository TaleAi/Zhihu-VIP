**- 醉笑陪君三千场 -**


三百三十天，离恨天最高。四百四十病，相思病最苦。


周烬将我从离恨天救赎，偏偏又赐了我一场难捱的相思。


本以为飞上枝头也变不成凤凰，可与周烬离心离德，深陷阴谋的漩涡的时候，我才发现，原来自己是涅槃的凤凰，要用前半生的烈火煎熬换后半生的塞外风雪。


而那月下踏梨花而来的神仙，也终将为我折腰万万次……


  



**- 暴君 -**


一朝穿进起点大男主成长爽文中，正逢退婚宋渊。


想到宋渊其人表里不一，自己将来身首异处，我只能连夜翻墙去表明心意。


栽进鸡窝，被狗咬，也不能阻挡我抱紧宋渊大腿的决心！


凭借着顽强的心，奇厚的脸皮，我终于成为了宋渊名义上的「妹妹」，实际上的二十四孝舔狗。


可万万没想到，宋渊终于快要对我软化的时候，书中跟我一起被砍头的暴君要娶我……


「好哥哥，你听我解释。」
