## 简介
我望着月亮，却只看见你。


就像是有人爱吃榴莲蛋糕，有人爱喝苦咖啡，哪怕你觉得自己平凡无奇，觉得自己不够优秀，也会有人找到你，看到你如同太阳般闪耀的光芒。太阳出来的那一刻，一切都黯然失色。


你是我的可遇不可求，你是我在梦里才敢肖想的秘密。


  



**《历劫》**


我的未婚夫爱上一个凡间女子，还要抽我的仙根助她成仙。


为了成全他们，我退婚了。


他却连腾云都不会了，赤脚跑来问我：「十里红妆，八抬大轿，够不够我赎罪。」


我望着他身后的女子，笑笑：「这当妾的好福气，明宴君还是留给别人吧。」
