## 7.令人眼前一亮的公关照：打造自己的「社交名片」
大家好，我是丛雪老师。在今天的分享开始之前，我们先来听一则小故事。


我的朋友小 C，在某国有大型银行担任部门主管，事业一直做得顺风顺水。因为她个人和带领团队的出色表现，领导安排她在下周的交流会上代表公司向同行业人员分享管理经验。得知消息的小 C，感到荣幸的同时又带着几分忐忑，由于她是第一次以这样的形式出现在公众视野，格外重视，开始积极准备起来。这时，小 C 接到了公关部同事的电话，他们正在制作交流会的宣传海报，希望小 C 提供一张自己的公关照。这可让小 C 犯了难，她翻遍了照片库存，满眼的生活照，最正式的就数去年拍摄的婚纱照了，可是没有一张适合放在宣传海报上呀！最后，由于时间紧迫，小 C 只能截取了一张生活照的头像交差，效果可想而知，看到海报的同行们怎么也无法把小 C 同心目中专业的行业标杆形象划上等号。


 


相信许多朋友与小 C 一样，在此之前，完全没有意识到一张足够令人眼前一亮的公关照对于职场人士来说究竟有多重要。在现今这个读图时代，只需要几秒钟，人们就能通过照片中的细节读取有关于你的各项背景信息，是职业经理人还是时尚设计师，性格内敛还是开放，工作风格谨慎或激进……一张小小的照片，浓缩的却是个人乃至整个企业的形象，可见，公关照就是你的形象「名片」，在职场与社交场合中发挥着举足轻重的作用。


 


从专业摄影角度来说，俗称的「公关照」其实应该叫做「商务人像摄影」，它既不等同于凸显个人特色的个人写真，也区别于四平八稳的普通证件照，它所塑造的是专业感和精英感，需要着重表现出拍摄对象自身的职业素养和气质内涵，表现出强大的气场和影响力。


 


那么，如何打造出完美公关照呢？


1. 着装要凸显拍摄对象的职业和身份


提到公关照的着装，许多朋友想当然地认为穿西装就好了。其实，公关照的着装远没有这么简单，首先，从图案和颜色的选择上来讲，务必从简。请尽量选择纯色系或者简单图案的，例如暗条纹，过于花哨的装饰和繁复的设计不适宜出现在公关照中，会使人的目光从拍摄者身上转移，不利于有效的信息传递。其次，着装务必要凸显拍摄对象的职业和身份，这是在拍摄公关照时最为重要的着装原则。如果是从事法律、金融等对着装要求比较严谨的行业，最好穿着黑色或深灰色套装，比如西装+铅笔裙、西装+阔腿裤、西装外套内搭无袖连衣裙都是不错的选择，面料要挺阔平整，可以佩戴手表、胸针和其他小体积饰品作为点缀。


![img](https://pic3.zhimg.com/v2-40e6c534663ed405319b34427dd8fb1e.webp)

如果是从事教师、销售、服务等表现亲和力的行业，可以选择浅色系套装，衬衫连衣裙或者白衬衫与彩色下装的搭配，配饰同样遵循简约原则。


![img](https://pic3.zhimg.com/v2-996628cb14e71b61fdf0065270a19a9d.webp)

如果是从事广告、设计、时尚等创意性行业，对服装的要求则相对宽松，可以更多凸显个人风格，例如有设计感的廓形衬衫、西装背心、连体裤、小礼服等，配饰上可以结合帽子、彩色丝巾以及存在感强的珠宝。


![img](https://pic2.zhimg.com/v2-624f7e93b3da8c74fe4ffc6febd1b902.webp)

还有一些从事特殊职业的朋友，例如厨师，你完全可以穿着大厨的衣服出现；例如户外远足达人，将一整套户外专业行头穿进影棚，这是多么令人期待的画面。


最后，需要提醒大家的是，尽量自备服装。我知道许多摄影机构会为拍摄者提供公关照着装，但通常只有为数不多的几件非常大众化的廉价套装，没有任何设计感和特色，高级质感就更加谈不上了，千万不要为了图一时方便，用廉价套装毁掉自己的重要照片，而且为了避免成为千篇一律的复制品，我的建议是最好自备服装，只有穿着真正属于自己的着装才能代表你的内在气质和外化身份，更好地呈现出基于真实职场角色的优质照片。


 


2. 淡雅妆容是最优选择，通过加强局部更好体现不同风格


在公关照中出现浓妆艳抹是一大禁忌，会使整体印象显得不够庄重和专业。以表现良好肌肤质感与光泽的淡雅妆容才是最优选择。其中，底妆是最为关键的步骤，要在匀净肤色的基础上尽量保持清透自然，不要涂得过厚造成假面感，建议选用保湿度高的水凝质感的粉底液或气垫粉饼，并在妆前做好保湿功课，避免干纹、卡粉等尴尬状况的出现。


如果想体现雷厉风行的干练形象，可以将眉形适当加宽，并强调眉峰的位置，加强硬朗的英气。


如果想体现温柔和蔼的亲和形象，可将腮红的涂法由斜扫变为在颧骨处划圈，唇部则采用西柚色、豆沙色等低调色系，表现柔美的女性气息。


  



3. 在日常发型的基础上，强调蓬松空气感


公关照是基于日常职场当中的真实形象而诞生的，所以在发型方面不建议做大的变动，以通勤中个人最常采用的发型为主。无论是长发还是短发，都要避免油腻、过于贴头皮、没有及时补染造成的明显深浅分界等现象，这些都会让你的形象看起来一点也不高级。可以借助吹风机、卷发棒、发蜡等产品使头顶部尽量蓬松立体，发尾可以略带微卷，营造自然的空气感。比起死板固定的发型，空气感发型更能体现出拍摄者自信潇洒的活力状态。


4. 每个女人都应该学习的上镜秘笈


因为工作关系，我会接触到许多职业摄影师，在跟他们交流时发现，大家普遍认为普通人面对镁光灯和镜头时，往往会显得手足无措，手不知道往哪里摆，身体和面部也极度僵硬，如同棍子一般直直地立在那里，十分尴尬。其实，掌握了以下上镜秘笈，在镜头前的你也可以如模特般收放自如。


1） 避免直线：头部和身体不要呈一条直线，假如头部正对镜头，身体可以转向一侧，如果身体正对镜头，头部可以稍稍偏转或微侧，可以让画面更有动感，人物也更加生动，避免僵硬刻板印象。同样的道理，手臂和腿也不要完全垂直正对镜头，需要强调身体的曲线感。


关于手臂拍摄姿势的原则就是手臂不要靠着身体，抬起手臂更显瘦。手部的常见动作包括双手环抱胸前、手叉腰、不对称放于身前、轻抚额头、撑下巴、双手插兜等，如果佩戴了饰品，也可以用手扶着它们，看起来更加自然。


  



![img](https://pic4.zhimg.com/v2-cc11c3cc160d83afa5b5c3a2f5020c38.webp)

双腿则可以通过曲直交差构成一定角度让姿态变得丰富灵动。


  



2） 拍摄坐姿时不能放松：假如你像平时坐沙发一样将整个身体深陷进椅子里，实实地坐进去，你的大腿和臀部脂肪就会溢出，整个人看起来粗壮无比。正确的坐姿是浅坐，臀部搭着椅子前部的边缘即可，浑身的肌肉要绷紧，不能放松，背部挺起，收紧小腹，同时，如果是拍全身照的话，双腿尽量向斜前方伸出，绷紧脚背，会让你的腿看起来更加修长。


![img](https://pic2.zhimg.com/v2-9ae6a91d51294b87635544ec42dad66e.webp)

3） 会说话的眼神为照片赋予灵魂：由于公关照明显的社交需求属性，拍摄者需要通过照片传递出真实的情感和信息，与看到照片的人们有更加直接的沟通，因此，照片中的眼神是非常重要的，如果片中人有一双会说话的眼睛，势必会吸引到更多的人。在拍摄时，请务必直视镜头，并让你的眼神坚定有力，炯炯发光，最简单的办法就是盯住镜头的中心点，让眼神聚焦，不要游移散漫。


![img](https://pic3.zhimg.com/v2-1d120a26ec7bf317f40a2d81bc3980b5.webp)

 


4） 微笑是最具感染力的表情：当你不知道采用什么表情时，请一定记得要微笑，这是迅速拉近距离的最好办法，用在照片中也同样奏效。可以用舌头顶住门牙内侧，嘴巴微张，这时的嘴型是最美的。除非你有信心能控制好自己的表情，否则不要轻易大笑，非常容易暴露面部缺陷。


  



5） 大女人气场，一件披肩外套就搞定：想要在照片中迅速营造大女人气场，有一个非常简便的方法，不要直接穿着外套，而是将它披在肩上。无论是大衣、西装、皮衣，只要外套有明显的肩线，比较硬挺，都可以采取这个模式。


![img](https://pic4.zhimg.com/v2-a67886ab8a28f62505c8896d28af7a5d.webp)

 


6） 飘动的发丝更显自信：中长发的女性在拍照时，要充分利用好头发这个武器。看看这些发丝飘动的画面，比起死板的固定位置，更能衬托出拍摄者的自信。想要制造发丝飘舞的照片，可以用吹风机对着拍摄者的脖颈部位轻吹，切忌对着额头部位，那会将前额的头发吹乱，风力也不要过大，否则会有大量发丝遮挡脸部。


5. 想让照片变生动，添加小道具吧


如果你想赋予一张公关照更丰富的内涵，让它看起来蕴含着精彩的故事，添加小道具是个不错的选择。尤其对于普通人来说，当你手里拿着杯子，一定会比空手时更加放松自然，也更明白你的手应该放在哪合适。在通常的人像摄影中，鲜花、咖啡杯、英文报纸都是常见的手中道具，在公关照中，还可以尝试添加与职业背景相关的小道具。拿着粉底刷的小姐姐应该是个靠谱的造型师，手持教鞭的温柔女孩肯定是位和蔼可亲的老师，比起端端正正的照片，有意思的小道具们让你更有了解这位拍摄者的动力。


  



6. 单色纯净背景让视线聚焦在人物本身


公关照通常是在影棚内拍摄，不适宜选取过于复杂和花哨的背景，会扰乱照片所传递的信息。单色纯净背景是首选，可以让视线聚焦在人物本身，黑色、灰色背景比白背景更富有厚实的高级质感，是我最为推荐的。


![img](https://pic2.zhimg.com/v2-fe91a5f435fcfad6b98da3451bf1a513.webp)

当然，选择在办公室内拍摄，营造场景感也是个不错的想法，但选取的位置是有讲究的，最好是在像落地窗边这样背景色单一、光线充足的地方，如果有办公桌出境，放置的办公用品不要太多，少而精为妙。


7. 善用光线塑造精致面庞


棚内打光至少需要三个光源，也就是主光、辐光和背景光，如果能增加发型光、眼神光等能令人物表现出更丰富的细节，面容将会表现地更加精致。因为公关照需要清晰地反映拍摄者的面容，建议使用偏软偏柔的光线。通过以下两张照片中硬光与软光的对比，可以清晰地看到明暗对比特别强烈的硬光会使脸上出现明显的暗影，而偏软的光线，能较好地增强皮肤的漫散射，弱化在硬光时人物脸部棱角过于分明、肌肤瑕疵容易暴露的问题。


在公关照的拍摄中，还有一类非常值得推荐，那就是黑白照片，因为摒除了色彩的干扰，能够让观察者集中注意力，获得更加震撼的视觉效果，照片的优雅氛围也得到了极大的烘托。


  



8. 摄影机构的选择


虽然你并不需要亲自操刀拍摄公关照，但为了打造出完美照片，你非常有必要了解公关照所涉及的方方面面，只有这样，你才有可能从「乱花渐欲迷人眼」的摄影机构中筛选出心仪的那家。在摄影机构的选择过程中，首先，你需要通过观察样片了解他们的基本摄影风格和能力，是否与你心目中理想的效果一致。其次，你可以通过与摄影师聊天，确认他的工作方式，他是想充分了解你的职业背景、性格特点以便确定如何拍摄，还是直接一股脑的扔出报价，把你作为流水线上的又一个复制品。总之，认真对待，多花一点时间，会让你的公关照成品效果更有保证。


  


团队公关照的拍摄要素


通过以上要点，我们了解到如何打造出一张个人的完美公关照。此外，社交需求绝不仅仅局限于个体，还有多人参与的团队公关照在职场中也很常见，例如你所在的企业或部门集体拍摄照片，用于公司周年庆纪念或是团队宣传使用。除了刚才讲到的要点之外，团队公关照还存在着区别于个人照的其他要素。


 


1. 统一的着装规则


Dress Code 是晚宴邀请函上的常见信息，对于团队公关照，Dress Code 同样适用，团队中的每个成员必须遵守着装的统一规则，以确保画面呈现出一支具有凝聚力的团队形象。如果公司有统一定制的制服最好不过了，即便没有制服也没关系，可以通过其他规则维系一致性。色彩一致是最常见的要求，例如黑白两色或是白色+彩色，或者是每个成员包含统一元素，例如佩戴相同的领结，穿着同款西装外套等。


![img](https://pic1.zhimg.com/v2-d84f76c77df6d763698ea3cafd448ff5.webp)

2. 成员排列错落有致


团体照的站位和姿态是很讲究的，首先，身份地位高的人比如企业高管人员应出现在画面中央最容易被看到的位置，也就是平时我们所说的 C 位，新手职员或实习生则出现在画面两侧。其次，如果人员较多，直直地站成一排看起来就过于死板，通常会呈现出错落有致的排列，例如前后错落或高低错落，有坐有站，增加梯子等道具。


另外，如果公司氛围比较自由轻松，也可以考虑更具创意性的拍摄方式，比如所有成员组成具有代表性的字母或图案；或者分小组站位，并增加更具活力的道具或手势。
