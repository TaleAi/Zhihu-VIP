## 153.流言蜚语：应该如何管理公司员工之间的流言蜚语？
大家好，我是姚余梁。


欢迎来到我的生活中的管理学节目。


今天我给大家讲一讲，公司里面怎么样管理公司员工之间的流言蜚语。


**一、流言蜚语的定义与负面影响**
------------------


流言蜚语，英语叫作 gossip。当公司发展到一定的程度，人多到一定的程度以后，流言蜚语是不可避免的。这是一个社会现象，只要在人多的地方，人和人之间有交流，那么就一定会出现：A 和 B 说话的时候，他们谈到 C。那么当在 C 的背后谈到 C 的时候，我们就把这个叫作流言蜚语。


当然，我这个流言蜚语可能跟我们传统的定义有些许不同。我们的流言蜚语是贬义词，在这里面是一个中性词汇。


在一个员工的背后谈论这个员工的时候，这就叫作流言蜚语。


用更正经的、更正统的管理学理念来说，这叫「feedback」，就是对一个员工的反馈。但这个反馈并不是我直接给到员工，而是我告诉别人，我认为这个员工他什么做得好，什么做得不好。


首先我要说的是，我的讨论范围是正常的公司内人和人之间的交流，这个人和人之间的交流就是在某一个人背后进行交流，这是在公司内经常发生的事情。


当我们在背后讨论某一个人的时候，这种情况下你会发现，这里面经常会有一个负面的、偏颇的差异在里头，为什么？


比如说我是一个经理，我告诉另一个经理，关于我们共同的一个员工，我对他的一些想法：他不是很细心或者很没有创造性。这个时候，我可能就把我自己的负面的一种感官、一种偏颇会带进去。


当听的人听到我的负面的这种评价以后，他们可能把我想说的好的评价，或者「在某种情况下他做得不是很好」，把这个「某种情况」给拿掉了，就把负面的东西记住了。然后等它再传播的时候就变成了百分之百负面的了。所以这种流言蜚语在公司内传播，它是有一定的伤害性的。


**二、如何管理不同情况下的流言蜚语**
--------------------


今天我来讲一讲，关于流言蜚语的三种情况，我们怎么样能够合理地管理流言蜚语。


第一种情况，如果我们作为一个管理人员，A 员工说 B 员工做得不好，B 员工说 A 员工做得不好，那么这种情况应该怎么处理？作为一个经理，你是要选一边站，听从 A 的，然后告诉 B，你做得不对，你要改正；还是听从 B 的，告诉 A 你要改正，你做得不对。


一般来说，如果是这种情况发生了，那么直接告诉 A 或者告诉 B，你们做得不对，这其实不是一种很好的处理方式。最好的办法是把他们叫到一起，把这些信息仔细讨论了以后，让两个人之间的误解得到消化和解决。


当然有的时候可能不是误解，那么这时候作为一个经理要果断地做一个决策。我认为 A 说的就是对的，那么 B 你不同意没有关系，我已经做了决定，理解要执行，不理解也要执行，这件事情就这么定了。这也是一种办法。


第二种处理方法，要鼓励 A 和 B 进行直接交流。你们两个自己去一块吃个午饭或者去喝个茶，把你们自己之间发生的事情交流一下，不要老给我找麻烦。你们这些问题没有敌我矛盾，都是一些误解，可能在工作中 A 没有理解 B 的好的地方，B 没有理解 A 好的地方。你们两个互相交流一下，尤其是能到工作之外的场景，比如说喝茶的时候，仔细谈一谈，仔细交流一下，形成一个在工作之外的这种联系。


那么这样的话，交流就会更加通畅，他们就会形成一个比较好的工作关系。如果能够做到这一点，当然这是最好的解决办法。如果做不到的话，那我们只能三个人坐下来，回到第一点说的那种情形，我们三人一起找到一个解决方案了。


第三点，如果是你对某一个人有意见了，这时候你要向另一个人把这个意见提出来的话，因为考虑到你所说的可能会流传出去（你告诉了 B 关于 C 的情况，那么 B 可能再告诉别人，所以对 C 的不利的这些信息可能会流传出去），可以说这是责任重大。


那么在告诉 B 之前，你一定要把自己当时假设什么，为什么在这种假设情况下我觉得 C 做得不是很好，也跟 B 说清楚了，让 B 也帮你判断一下这些假设是不是对的，不是结论。


结论是我认为他做东西不是很细，但是我要把这些假设也告诉 B。可能我给他就两个小时时间，那么他做得不是很细。那么 B 可能会告诉你，两个小时时间太短了，你给他两天时间，可能做得会很好。所以你需要把你所有的信息，从假设到结论都跟 B 讲一讲，让 B 也跟你分析一下。


这样你才能做一个更加合理、更加客观的判断，让流言蜚语不是简单地传出去，对 C 造成很大的困惑和影响。这是对别人负责，也是对你自己负责的一个态度！ 


**三、流言蜚语的积极作用**
---------------


说了这么多，我们都说流言蜚语是负面的，实际上，我们可以从正面来考虑一下。我们能不能从正面利用一下流言蜚语，让别的员工或别的同事更喜欢我，或者我间接地鼓励他们把自己工作做好？


比如，我跟 B 说 C 的时候，如果我说的是负面的话，那么我一定要小心。那我如果特别喜欢 C 的话，我能不能故意把我的表扬、我的赞扬跟 B 说，然后 B 传出去，传到 C 的耳朵里头。这样的话形成一个正面的流言蜚语，当 C 听到以后，他肯定对我的好感也会增加，我们公司的工作环境也会更加和谐，这是一件好的事情。
