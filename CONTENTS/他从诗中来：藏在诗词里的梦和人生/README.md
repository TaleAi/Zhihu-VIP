# 他从诗中来：藏在诗词里的梦和人生

### 分类：文学

### 状态：已完结，共 10 节

### 存档日期：2022-06-02

## 简介
是名篇，也是他的全部人生。


提到名作名篇，不擅诗词的人也能来上两句；


提到名句赏析，不擅文学的人也能点评一二——


「这句代表诗人凄苦的命运」「这句写他未尽的梦想」……


但他的命运是怎样的？他的梦又是什么呢？


基本没有人能说得出。


毕竟，时间和历史能如大浪淘沙，将熠熠生辉的部分留在纸上——他是诗仙、他是爱国诗人。


却也能局限住我们对他们的印象——比起活生生的人，他们更像一个符号，一个标签。


但其实，每一首诗里，都有一段或温暖、或凄惨的人生故事；每一个名句，可能都是爱而不可得、志所未能酬的悲鸣。


本专栏以诗词为线索，串起诗人以「报国梦」为中心的人生故事，从少年的立志入世，到青年的热血奋斗，再到老年的颠沛流离，从全新的视角读懂诗人的人生、梦想，以及笔下诗词的内核。


* 绯闻满天飞的杜牧，究竟是有才任性还是心里有苦但他不说？
* 两次当上门女婿的李白，究竟是爱吃软饭还是志在千里？
* 被恩人拉黑封杀还被骂进史书里的李商隐，究竟是情深似海还是人品有问题？

人寿有限，纸寿千年。


让我们在故事里与教科书上的老熟人重逢。
