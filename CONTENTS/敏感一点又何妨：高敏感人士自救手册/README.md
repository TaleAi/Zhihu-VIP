## 简介
敏感是一种非常稀有的特质，请好好利用它。


间歇性 emo，很容易被其他人的情绪影响；难以忍受噪音、强光，强烈的气味等；对疼痛的耐受很低……如果你身上有这些特质，那么你很有可能是高敏感人群。


高敏感并非玄学，而是有科学基础的一项心理学理论。高敏感人群简称 HSP，指的是具有高感觉加工敏感性的人群。在实际生活中，高敏感人群遇到的问题会比其他人群更多：


* 大学住校或与他人合租，很容易受到刺激，影响休息。
* 对刺激过度反应，逃避社交，和朋友渐行渐远……
* 焦虑型依恋，很难开启长期稳定的亲密关系。

在遇到上述问题时，高敏感人群往往意识不到自己的敏感从何而来，经常错误地将它归因并使自己更加痛苦。


所以高敏感人群急需一本自救手册，来帮助自己认识到问题所在，不再轻易「破防」、「emo」。


本专栏将会带你认识自己的敏感特质，在解决上述问题的同时，帮你接纳并改善自己的敏感特质，并正确地寻求理解和帮助。
