# 总编辑的媒体思维与技巧：融媒时代的敲门砖

### 作者：老王说

### 分类：职人

### 状态：已完结，共 24 节

## 简介
面对技术与管理的融合的关键时期，一个老媒体人将为你讲述媒介的基本原理、基本要素和基本原则，颠覆你的固有认知，升级你的思维方式，从生产力和生产关系两个变量构成的坐标系中去寻求未来自身的发展。


眼下的媒体正在发生百年之巨变，无论新旧/老幼，不管报纸电视还是垂直抑或 APP，面对巨变都是皱着眉头。


老媒体，报纸/电台/电视台面对人们手中死攥着不放的手机小屏幕，面对 5G 的发展，眉头紧锁，扼腕叹息，抑或悔恨难当。


新媒体，面对纪律管束，面对从未有过的严厉，正在体会着成为主流/主舆论阵地欣喜后的压抑困惑，紧张甚至恐慌。


如今的时代，需要的是新媒体的技术优势以及对于未来发展的把控，同时也离不开老媒体对于管理体制的熟悉以及对于内容操作的老练。


作为媒体人，我们需要做的是将两者融合，让媒体在这个有技术和社会体制交织在一起的国度更好地生存与发展。


老王，一个老媒体人经历了老的辉煌，探索过新的崛起，三十年基本上就在媒体这条路上执着走着。面对技术与管理的融合的关键时期，他将在本专栏中，告知那些准备进入融媒的新老人：媒体的基本原理不会改变；内容生产的基本要素不会改变；面对镜头，低头写作的基本原则不会改变，政府对于媒体的管理本来不会改变；遵守底线纪律编辑基本原则不会改变。


也想告诉融媒新老人：技术一定改变媒体传播方式；镜头一定改变媒体报道的思维习惯；视觉必将颠覆深刻思想的固有模式；运营会使得我们盈利方式根本转换。


作为媒体人，你要知道未来是一个文理融合的产业，必须从生产力和生产关系两个变量构成的坐标系中去寻求未来自身的发展。
