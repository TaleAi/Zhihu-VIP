# 哭泣金三角：走私、孤儿与瘾君子

### 作者：黑叶

### 分类：真实故事

### 状态：已更新至第 9 节, 共 10 节

### 存档日期：2022-06-06

## 简介
金三角是个绝望之地。在这里，以贩养吸的夫妻失去挚爱，瘾君子的孩子失去至亲，无助的孕妇身如浮萍，搏命的毒枭命在旦夕，似乎每一个人，都可能无声无息地消失。我在这里当老师，这是我目睹的一切。


我叫黑叶，在金三角落后地区志愿支教。


瘾君子、贩毒产业、枪支走私、赌博杀人无处不在，深深浸透到每个人的生活中。


我的日常工作不止是讲课，还包括随身带着匕首和枪支，防止疯魔的毒贩和瘾君子靠近学校，拿着麻袋去镇上给大家乞讨粮油米面，出门吃饭前检查有没有罂粟壳。


一对中餐馆夫妻，公开往米线里加「料」，笑我不懂什么是「好吃」；


一个缅甸富二代姑娘，被家族抛弃深山，只因为她未婚先孕了中国人的孩子；


一个大毒枭，原是知青同盟军，参加越战后他和战友们决定一起出逃缅甸，介入鸦片买卖……


这里有很多孩子，父母悄然消失，每天与毒品零距离接触，针头就藏在家里某个角落，他们对命运一无所知，好好上学是一种奢侈品。


**曾有个问题：「有没有哪一刻你觉得世界不应该是这样？」**


**我想说，每一刻。**
