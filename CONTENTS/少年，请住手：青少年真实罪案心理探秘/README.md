## 简介
本专栏记述了九个青少年暴力犯罪事件，探究少年犯罪的原因与心理，故事情节由真实事件改编，发人深省。


「我们与恶的距离」是什么？没有杀人和杀了人的少年，差别在哪里？


令人冒冷汗的是，所有少年在杀人前心理变化，你在年少时或多或少都曾经历过： 


不想上学，特别想炸了学校；


好奇小猫为什么会发出「呼噜」声，所以想解剖看看；


想知道蓝色油漆滴到眼睛里，会不会变成蓝眼睛；


一个同学背后说你坏话，你恨不得他人间蒸发；


成绩不好，爸爸妈妈一直骂你，你再也不想看到他们……


如果杀人是最终引爆了火药桶，从这些极端案件中，我们可以窥探这个「火药桶」里到底装了什么。
