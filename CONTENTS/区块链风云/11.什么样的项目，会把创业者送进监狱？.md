## 11.什么样的项目，会把创业者送进监狱？
1.
==


他没想到一切进展得如此顺利，尤其是这家叫 Swiftly 的公司，太上道了，居然直接免费且开源，简直就是在向全行业宣战。可以预料的是，在接下来的时间里，新脑科技会继续流失客户，直到破产。


隔岸观火的少年坐在电脑前，浏览起刚刚下载的 Swiftly 源码，心底升起一股手刃仇敌的快感。他相信，这家名为 Swiftly 的公司之所以敢免费，背后一定有它运作的门道。而对于一个技术人员来说，了解一项互联网产品，最快且最直接的方式，就是阅读它的源代码。


可看着看着，他本来轻松的面孔忽然变得凝重。


因为他看到了一串熟悉的字符。


怎么会呢？它怎么会出现在这里呢？


他拿起鼠标，将那串字符复制，在全部源码中进行搜索，发现……


它竟然出现了一百多次？！


这意味着写这段代码的人，至少曾深度参与过这个项目。


by.PengJian


他越看越愤怒，看得他青筋暴起，胸中燃起一股战斗的火焰。


这是我哥哥的代码，这是我哥哥的代码。


他摔碎了手里的鼠标。 


2.
==


冷板凳上，创业三人组饥肠辘辘，坐在福通地产的会议室里发呆。


窗外，晚霞铺满天空，将太阳烘托得一片猩红。屋内没有开灯，夕阳下，几人的身影格外落寞。何小婉焦躁地跺了跺脚：「白思成到底还来不来啊？」


不怪她着急，毕竟已从上午九点等到了傍晚，她又饿又困又无聊，便又拿莫飞撒气：「都怪你，整天磨磨叽叽的，把正事给耽误了。你说，要是早早联系人家，他一满意，当场过合同签字盖章，不就啥事都没有了？非等到现在，人家不想见我们了，咱们再热脸贴冷屁股，傻了吧？」


莫飞瞪了她一眼，没吭声，拿出手机开始刷朋友圈。


「跟你说话呢！」何小婉咄咄逼人，「今天拿不下白思成，明儿一早，UST 就撤资了。告诉你啊，咱可还欠着人家投资款呢！到时候你拿什么还？」


莫飞关掉微信，打开消消乐，开始玩游戏。


「青雁也不接我电话。」何阳看看手机，「是不是发生什么事了？」


「能有什么事情啊，明显就是人家生气了！不想搭理咱们！」何小婉激动起来，「那白思成是谁？全国有名的富二代！多少人上着杆子想跟他攀交情？可你瞧瞧莫飞，整天不慌不忙的，死盯着那几个芝麻绿豆大点的订单不放，都火烧眉毛了，才着急忙慌地找白思成——哦，合着把人当备胎了？」


莫飞戴上耳机，调大了音量，又拿出一瓶二锅头，刚要拧开，就被何小婉一把夺过，摔到墙角。


「你这一天天的！就知道喝酒！早晚喝死你啊！」何小婉气得眼泪都快出来了，「咱们创业本来就不富裕，折腾了两三年，也没见到什么钱，眼瞅着好不容易有点希望了，又没把握住，搞不齐还得背上一身债……」她看向何阳，用带着哭腔的声音抱怨，「哥，你说，万一 UST 真管我们要钱，咱上哪儿凑去啊？！」


「行了！别叨叨了！」莫飞终于忍无可忍，他摘下耳机，「大不了我去给 UST 打工抵债！不会把你拖下水的！」


「你把我当什么人了？」何小婉抹了把眼泪，「咱们在一起这么久了，我是那种人吗？」


「那你到底想怎么样？」莫飞望着一地的酒瓶碎片，欲言又止，「不是我不想跟白思成合作，是他那个生意……咱们不能碰！」


「不能碰？怎么不能碰？人家一个富二代，还能把我们带进监狱不成？」


「没错。」天已经彻底黑了，夜幕降临，会议室内一片昏暗。酒味弥漫开来，醉意有点上头，莫飞闭上眼睛，在黑漆漆的空间里，缓缓开口：「他那个生意，搞不好……真的是会进监狱的。」


3.
==


挂钟嘀嗒嘀嗒地走着，素来张狂的白思成，今天却像一只温顺的小绵羊，闷不吭声地低着头，心随着钟锤摇摆的节奏忽上忽下。对面，穿着中山装的老者坐在由小叶紫檀打造的办公桌前，不怒自威。


桌上放着一份盖着红章的通知。


「这是怎么回事？」老者用夹着雪茄的右手，狠狠敲了敲那份通知。


《关于防范代币发行融资风险的通知》[[1]](#ref_1) 


中国人民银行网站发布公告称，近期，国内通过发行代币形式包括首次代币发行（ICO）进行融资的活动大量涌现，投机炒作盛行，涉嫌从事非法金融活动，严重扰乱了经济金融秩序。


中国人民银行、中央网信办、工业和信息化部、工商总局、银监会、证监会、保监会等七部委要求，自此公告发布之日起，各类代币发行融资活动应当立即停止。


央行将规范管理比特币交易平台，即日起，关停所有比特币对法币（人民币）的交易。 


这份通知以明文形式，彻底封停了中国境内的有关数字货币的运营活动，目的是为了加强央行对数字货币的监管，维护国家金融制度的权威性，并将代币发行定性为一种未经批准的非法公开融资活动。


「哑巴了？」老者将通知砸在白思成的身上，他提高了声音，仿佛要榨干屋内的空气，「央行的文件，你竟熟视无睹，还顶风作案？」


「爹，你听我讲。」白思成鼓起一点勇气，争辩道，「比特币和区块链真的是未来，你要相信我的判断力。」


「狗屁判断力！你瞅瞅都跌成什么样子了？」白老太爷指指电脑屏幕，白思成缩着脑袋，哪儿敢去看。新闻一出，他就预感到市场会受挫，只是没想到暴风雨来得如此猛烈：消息发布当日，主流币币值直线下跌，比特币跌幅超百分之三十，莱特币大跌一半，其他山寨币更是直接归零。


冷气呼呼地吹着，可他脑门儿上的汗就没有停过。


「你所投资的交易所，在中国境内吗？」


「不在。交易所都设在境外，往来的钱经过几家公司周转，绝不会让人查到福通地产。」


「保险吗？」


「应……应该不会有问题。」白思成犹犹豫豫，三缄其口，不敢打包票，毕竟世上没有不透风的墙。但话又说回来了，谁会吃饱了没事跟他对着干？


「你还是太年轻。」白老太爷磕了磕雪茄的烟灰，望向屏风，「罗小姐……」


屏风后走出一个穿旗袍的女子，玲珑曲线尽显东方韵致。她经过白思成身边时，留下一缕清香。


「这是那几家交易所的资料。」女子双手奉上一份厚厚的文件，「想必少爷当初是为了规避备案[[2]](#ref_2) ，才把交易所设在了境外。交易所的账目虽经过周转，但最终还是流回到他的私人账户，并被有心人记录了下来。一周前，网上有人高价兜售这些资料，我就把它们全买回来了。」


又是这个莫名其妙的女人？她到底想干吗？白思成心里犯起嘀咕。


几个月前，这个来路不明的妖艳女人开始频繁跟白家接触，既不谈钱，也不图利，令他琢磨不透。一开始，他以为她是在打自己主意，但约过她几次，都被其冷冷拒绝。


难不成……这娘们儿是在打我爹的主意？要给我当后妈？


白思成狠狠打了个冷战。


白老太爷拿起文件，忽看到里面夹带的一张新闻网站截图：「这是？」


「一周前，朝阳民警发现了一具男尸。经调查，此人生前痴迷于投资数字货币，并在某交易所中压下了全部资产，结果庄家跑路，他欠下巨额债务，最终自杀。而那个交易所的投资人……」


她话说了一半，扭头看向白思成。


「你……」白思成的额头青筋暴起，这女人是在把火往他身上引。


「还有别人知道吗？」白老太爷举起手里的资料。


「都在这儿了。」她答道，「卖方答应我，会销毁这些副本的原件。」


「虽然我不清楚你到底在打什么算盘，但我相信……你一定另有所图。」白老太爷将文件放进碎纸机内碾碎，「说说你的计划吧！」


「我也认同区块链是未来的方向，从比特币到以太坊[[3]](#ref_3) ，它的生态一直在进化，早一点进入这个行业，就早一点占据先发优势。但数字货币不是一个合适的切入点，尤其是在中国。」说着，她笑起来，笑声高深莫测，「而我今天来，是想跟二位谈一谈——AI 城市的订单。」罗琳说。


4.
==


四百年前，荷兰人将西班牙人驱逐出境，独立建国。之后，西班牙封锁了港口，对荷兰进行经济制裁。荷兰人无奈，只好自己造船，发展航运业务，并成立了荷兰东印度公司，在大航海时代中慢慢崛起。


但公司运营的成本极大，政府负担不起。为募集资金，东印度公司开始向市民贩售股份：每次出海前，荷兰市民可以花钱购买一部分公司股份，等船队回来，就能换取等额甚至超额的战利品。


这个「某家公司将它的股份向公众出售」的过程，就演变成了今天的 IPO[[4]](#ref_4) 。


2008 年，神秘人中本聪基于区块链网络，开发了比特币，之后，他销声匿迹，将比特币开源。


2011 年，谷歌公司的工程师李启威受比特币启发，在其基础上进行改进，设计出莱特币。因为比特币居高不下的热度，自称「比特币第二」的莱特币也飙涨了数千倍之多！


之后，李启威在最高点清仓，抛空了手中所有的莱特币，套现离场，被誉为「逃顶之王」[[5]](#ref_5) 。


莱特币推出后，无数人效仿李启威，推出自己的数字货币。这些基于开源代码改造出来的数字货币，被投入到交易所中，供买家公开交易、追涨杀跌，这个过程就叫作 ICO。


「白思成的目的，根本不是控制比特币，而是想自己 ICO，取代比特币，最终，操纵整个数字货币市场。」莫飞拿出田大壮的手机，点开上面的交易所 App，「这是我朋友的手机，他在玩一款我从未听说过的数字货币。为了追涨，他将店铺抵押了出去。上个月，这个数字货币崩盘，创始人跑路，我朋友血本无归。最终，他从天台上跳了下去……


「此后，我开始研究这些数字货币，我发现，它们名目众多，但大部分都架设在境外服务器，通过专有的 App 或 VPN[[6]](#ref_6) 来连接，项目资料纯属虚构，创始人信息也真真假假，毫无踪迹可循。


「于是，我只好把目光放在了为这些数字货币提供交易服务的平台上。我反编译了田大壮手机里的交易所 App，才发现，虽然它连接的服务器已经关闭，但因为这部手机的安卓系统版本太老，导致 App 里清除缓存的 API[[7]](#ref_7)  并没有生效，于是，我提取了缓存里的连接日志，通过分析，得到了几个 IP 地址。这些 IP 地址遍布全球各地，但其中一个……」莫飞在手机浏览器里输入了一串 IP 地址，经过一番加载，出来的结果让何氏兄妹大吃一惊。


浏览器居然打开了福通地产的官方网站！


波澜壮阔的川西大渡河，汹涌的河水掀起冲天巨浪，气势如万马奔腾，河岸两侧，高高矮矮地林立着数十座水电站，里面巨大的电子设备轰轰隆隆，日夜不息。


「为了进一步了解白思成，我曾经用新脑搜寻过他的资料。我发现，很多文档都来自于一个未被解析的网站，域名直接以 IP 地址的方式呈现，于是我查了下，果不其然……」


「我懂了。」何阳说，「白思成直接把水电站的矿机，当福通地产的网站服务器用了。」


「懂什么啊？我还是没明白。」何小婉纳闷儿，「白思成跟你朋友的死到底有什么关系？」


「给你讲个故事吧。」莫飞说，「网上，曾经出现过一个叫太空链的区块链项目，他们号称要把区块链的节点发射到宇宙上，调动全人类的力量探索太空。听起来很牛吧？你是不是也觉得，能做这个项目的公司，创始人不得埃隆·马斯克[[8]](#ref_8) 这种级别的巨头？可笑的是，这个公司的老板是个一九九二年出生的年轻人，他做太空链时，还不到二十六岁。因为概念酷炫，又有几个商业大佬为他站台，于是，大家纷纷认购了这家公司发行的代币——也就是 ICO——你可以把这理解为购买这家公司的原始股。最终，他通过太空链这个项目，募集了，」莫飞把两根食指交叉，「十个亿。」


「那之后呢？」


「之后？太空链被曝出诸多造假，他们发行的『太空币』跌到破发 ，说好的『宇宙卫星』也没有如期升空。等到众人反应过来，才发现创始人早已卷走了那十个亿，消失了。[[9]](#ref_9) 」莫飞握着田大壮的手机，朦胧中，仿佛又看到了那张憨厚的脸，「这算什么？很多人把这定义为集资，但在我看来，这就是诈骗。」


资本市场中有一个博傻理论，指人们之所以完全不在乎某个东西的真实价值而愿意花高价购买，是因为他们觉得会有一个更大的笨蛋，愿意花更高的价格，从自己这里把它买走。


只要自己不是最大的笨蛋，那就一定能够成为赢家，无非是赚多赚少的问题。人性最大的弱点就是贪婪，在彼此吞食的黑暗森林里，每个人都认为自己是百发百中的猎手，自信地提着枪，寻找发光的猎物。


殊不知，数字货币的本质就是投机，而但凡是投机，普通人的赢面都太小：庄家自己 ICO，发行个一文不值的数字货币，之后找操盘手控盘，同交易所联手拉升币价，吸引散户跟风入场。币价飙涨后，庄家选择在合适的时机退场，同交易所分钱跑路。散户则搭上毕生积蓄，白白成了被收割的韭菜。


5.
==


「原来是这样……」听完故事，何小婉后怕地拍拍胸脯，「好险啊！幸好我们没答应白思成，万一真跟他混了，搞不好要进监狱啊！」


「可不是嘛。」莫飞长叹一声，这几年来，他每天都过得小心谨慎，如履薄冰，就是怕自己有朝一日，再次因为钱而冲昏头脑，走上歪路。


这是一个号召「全民创业，万众创新」的社会，更是一个「人有多大胆，地有多大产」的时代，在媒体包装和资本助推下，有些人渐渐模糊了利益和道德的标准，而「创新」，就代表自己要去做一些前人未做过的事情……


可这事情是对是错，谁都说不好。


「创新」是一种「冒险」，但「冒险」和「违法」，只有一步之遥。任何形式的创新都有可能会被滥用，当人们不惜代价地试图从中获利时，不经意就会越过法律和监管的框架。


他以前吃过不懂法的亏，现在，他不能再带着团队冒这个险。


天色已黑，莫飞望了望窗外：「走吧。看来他不会见我们了。」


「走？去哪儿啊？」何小婉问。


「回公司。把咱们的家具电脑归拢归拢，看看哪些能卖，哪些能退，换点钱，把 UST 的投资款还回去！」月光下，莫飞似乎恢复了一点精神，「对了，小婉，你还有孵化器的联系方式吧？问问现在什么价格，咱们再搬进去，这次创业失败了，没关系，咱从头再来！」


这是众人最不愿意看到的结果，UST 之所以给新脑投资，就是希望通过他们拿下白思成的合约，而此时，创业三人组决定不蹚数字货币这浑水。毕竟法律就像一把利剑，始终悬在每一个创业者的头上。


「谁说你们失败了？」


忽然，会议室的灯亮了起来，众人四下看去，只见一个熟悉的身影。他还穿着那身笔挺的西装，坐在会议室的最后一排，礼貌地朝众人笑笑。他摘下墨镜，湖蓝色的瞳孔在灯光下闪闪发亮。


「安先生，您怎么来了？」


6.
==


「抱歉。」想到明天就是投资保护期的截止日，莫飞顿时明白了安先生的来意，他用略带愧疚的口吻说道，「让您失望了，我们会把所有的钱都拿来赔偿 UST，如果还不行……」莫飞摊开空空的两只手，耸耸肩膀，「那只能给你们打工还债了。」


「不。」安先生笑笑，「四百万美金的投资尾款，已经打入你们的账户了。」


「啊？」莫飞愣住。


「回去再查吧。」安先生说，「但你现在得跟我走一趟。你有一个老朋友，想要见一见你。」


看着二人离去的背影，何小婉一脸疑惑：「哥，这到底是怎么回事啊？看来，他们好像很早以前就认识？」


何阳点了一根烟，摇了摇头：「不清楚。」


他真的不清楚。虽然一同创业三年了，但关于这个合作伙伴的过去，他知之甚少。或许网上能查到一点蛛丝马迹？可他并不关心，只要莫飞不说，他就不想去问，在他看来，这是合作伙伴间应有的一种默契。


有人说，合伙创业跟经营感情一样，有些事情，看破不说破，日子还能过下去——只要大家清楚底线在哪里。


以前何阳不想触碰那根底线，担心之后连朋友都没得做了。


但此刻，莫飞的昨天和自己的今天，忽然产生了交集，这令他不得不好奇，这个神秘的创业伙伴，到底什么来路。


7.


楼下停着一辆黑色轿车，安先生打开车门，把莫飞请入后座。


「去哪儿？」莫飞问。


「机场，出差。」安先生往驾驶座走去，「你带身份证了吧？」


「开什么玩笑？太突然了吧！」莫飞刚要跳车，这时，旁边伸过来一只女人的手。


「好久不见。」她说。


这声音让莫飞愣住，他呆滞地扭过脸，望着身旁的人，鼻子竟酸了起来。


你为什么北漂？又为什么创业？


那个人曾经这么问他。可当时的他太年轻，年轻到以为生命里的一切，只要努力就能把控。可回想这些年漂泊的时光，眩晕间，他才知道自己错了，现实残酷到让他又一次身不由己地陷入回忆的旋涡之中。


而忘记一个人需要多久？漫长的时间够不够？


他以为只要不去触碰，就不会难过。可此刻，往事像海啸一样席卷而来，跌跌撞撞，兜兜转转，曾以为已经逃离的一切，刹那间又浮现在了眼前。莫飞望着她，一句话也说不出，愣了很久，这张不听使唤的嘴，才缓缓念出被尘封已久的名字：「林……林姿？」
