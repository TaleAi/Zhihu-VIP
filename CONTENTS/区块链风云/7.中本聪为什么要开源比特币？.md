## 7.中本聪为什么要开源比特币？
1.
==


「报警？」少年警觉起来，「为什么要报警？」


「少跟我装蒜！门刚是你背着我关上的吧？要不要去物业看看监控录像？」


「我不知道！」少年一脸无辜。


「那我再问你，你怎么连上我们公司的 WiFi 的？」


「不是说了嘛，穷举法啊……」


「真当我傻啊？你就算能破解密码，也得先知道网络名称吧！」何小婉抖抖桌上的纸片，「莫飞那个傻缺，把 WiFi 名设置得这么复杂，我都没记住，你怎么可能一眼就认出来？！」


「这个也简单！」少年大脑开始飞速运转，「我是根据信号的强度来判断的。」他指指电脑上的无线网络列表，「信号满格的，肯定就是你们公司的网络嘛。」


「你确定信号满格？」何小婉敲了敲少年的笔记本屏幕，「你仔细看，我们公司的 WiFi 只有一格信号！」


少年看了一眼自己连接的 WiFi，惊了。


真的只有一格信号！


「都怪莫飞那个混蛋，整天迷迷糊糊的，买了个破路由器，信号极差，刷个微博都卡，连对面洗脚城的 WiFi 都比我们的信号强！难不成，你连进了洗脚城的网络？」何小婉又举起手机，威胁道，「快告诉我，你到底是谁，来这儿有什么目的？不然我真的报警了啊！」


「好好好，你先冷静一下……」少年缴械投降，「我承认，我入侵了你的手机。」


「手机？！」何小婉忙把手机扔到桌上，仿佛自己拿的是一个炸弹，「你对它做了什么？」


「其实是这样的……」少年从兜里拿出一张纸，正是何小婉刚刚发布招聘启事时看的那张传单，「这上面有个二维码，你扫描了它，对吧？于是你进入了我做的网站，刊登了招聘启事。而这个二维码上面有后门，你一扫，你手机里的网络配置就自动发送给我了。」


「那网站是假的？」何小婉怒不可遏，气得牙痒痒，「你到底什么来头，敢戏弄老娘？难道是竞争对手派来的奸细？！」


她拿回手机，再一次打算报警。


「姐姐！能不能别总想着叫警察啊，听我解释好不好？」少年把何小婉按在沙发上，死扣住她的手腕，「你还记得去年，你跟你哥一起参加了孵化器组织的创业大赛的事情吗？」他试图带领何小婉回忆过去，「当时我也在，我做了个在线听歌的网站，但评委们都不喜欢，而选手互评的时候，只有你们给我投了一票。」


「我确实记得那个比赛……」何小婉敲着脑袋想了想，「但实在想不起来给你投过票啊……」


「想不起来没关系！那不重要！」他大手一挥，「总之，我今天来是跟你们交朋友的！」 


2.
==


「交朋友？」何小婉半信半疑地看着他，「不行！」她又戒备起来，一把将少年推开，拿回手机，「不管你说的是真是假，有什么话，咱们到警察局再说！」


何小婉刚拨通了报警电话，可这时，办公室的灯忽然……灭了。


「停电了？」何小婉望望窗外，对面洗脚城的霓虹招牌仍在闪烁，搞得办公室里忽明忽暗的。她正纳闷儿，却又传来了电磁门上锁的声音。


「糟了！是那小子搞的鬼！」


何小婉忙追过去，可那破门又一次紧紧地锁了起来。


「混蛋！」她按了按失灵的开关，用力砸门，并朝黑漆漆的走廊骂道，「你给我滚回来！」


在骂声中，少年快步下了楼。他边走，边揉着胳膊，有点疼，似乎是在刚刚的争执中伤到了。他拉起右臂的衣袖，露出一道长长的疤痕。仲夏之夜燥热难耐，他瞅了瞅那道疤痕，又将衣袖放了下来。


少年拿出手机，浏览着刚从这家公司窃取来的资料，看了一会儿，他满意地将手机塞回裤兜。音乐响起，他背着单肩包，双手插兜，吹着口哨，拐入一条幽暗的胡同。


办公室的灯又亮了起来。何小婉站在门口，四下望去，少年已消失无踪。一切如初，只是桌上多了一张纸片。她走过去，拿起纸片，上面歪歪扭扭地写着两个英文字母：


PT。 


3.
==


北京，昌平郊外的一栋别墅。


王杰森翻着手里的文件：「竟然只有三百多万的用户？这么少？」


「除去僵尸和非活跃用户，估计真实数据不足一百万。」艾伦补充，「事实上，一聊 SDK 在早年时的表现还不错，只可惜张克亮这个人格局太小，只想赚快钱。在微信和微博的攻击下，逐渐失掉了市场，打烂了一手好牌。」


「看来他不值得我们花时间。」王杰森掩饰不住失望之情，「那白思成呢？有消息吗？」


「仍然没有消息。上一次航班动态是在成都双流国际机场，此后就一直没有更新。但据内部人士透露，他最近投资了几家数字货币交易所。」


「数字货币交易所？比特币？」


「不止，莱特币和一些主流的山寨币也包括在内。」艾伦打开手机，指着上面的一个 App，「这是其中一家，提供上百倍的杠杆借贷[[1]](#ref_1) 。我找人查过，背后的股东正是白思成。」


「他为什么要投资这个？为了洗钱，还是炒币？」


「Boss……」艾伦欲言又止，「我可以问个问题吗？」


「你说。」


「为什么您完全不碰比特币？」艾伦说，「您的朋友们，那些科技公司的创始人、基金公司的控盘手、财阀的公子哥……似乎全世界的人都在追逐比特币的热潮，但您似乎一点也不感兴趣。为什么？」


「是个有趣的问题。」王杰森望向艾伦，忽然问道，「你想赚钱吗？」


「不想。」艾伦斩钉截铁地回答，「您知道……我跟着您，从来不为钱。」


「因为你不缺钱。」王杰森笑笑，「去年，美国一家初创公司，以非常低的价格向你父亲出售了少量股份，相当于白送。半年后，那公司上市，你们家持有的股值翻了好几十倍，所以，以你们家的关系，这种锅里抢饭的机会太多了，你当然不缺钱。」王杰森停顿一下，接着说道，「投资比特币，本质上与投资股票、黄金和期货没有区别，都是利用信息不对称追涨杀跌。我十六岁时就厌倦了这种野蛮的金钱游戏。多年来，我们之所以扎根 IT 行业，就是想实现一个真正能改变世界的梦想……」


说着说着，王杰森的神色黯淡下来，「很多人都想改变世界，但我们没有做到，中本聪做到了。他推出比特币时，我花了很长时间研究它对未来的影响。最后，经过各种沙盘推演，我得出了一个沮丧的结论——比特币的世界，我无法控制。」


「为什么？」艾伦不解，「像白思成那样，不断地买下世界上的数字货币交易所和矿厂，不就等于控制了比特币吗？」


「不。」王杰森否定道，「2008 年，中本聪发布了比特币项目白皮书，建立了第一个区块链应用，之后他就退出了比特币的运营，并开源了源代码。从那以后，这个人就消失了。你觉得这是为什么呢？」


艾伦摸着下巴，若有所思：「这是因为比特币使用了『去中心化』的架构。如果创始人仍在任，代码又不透明，则没有人会相信比特币是去中心的。」


「你很聪明，很快就想到了。」王杰森对此表示赞赏，「比特币开源后，世界各地的程序员便自发地成立了比特币社区，推广和维护比特币网络。所以，如今比特币市场的繁荣，完全可以说是社区自治的成果。艾伦，」王杰森话锋一转，「你认为，互联网公司最重要的是什么？」


「最重要？」艾伦歪着脑袋沉思了下，「速度吗？毕竟，互联网强调快速操作和快速迭代。」


「速度是一部分。但比速度更重要的是——稳定。」


「稳定？」艾伦想了想，点点头，「对，要是服务器宕机了，就什么也做不了了。」


「互联网需要稳定，比特币网络也需要稳定，只有当每个人都遵守游戏规则、共享算力、乖乖挖矿时，大家才能一起赚钱。如果有人试图控制比特币网络，他会怎么样呢？」王杰森从兜里拿出一张美元，「它是什么？这是钱，但本质上它只是一张纸。为什么纸张可以消费？因为它有国家和政府的信用支持，保证了这张纸能够买到它的等价物。而为什么比特币可以兑换法币？因为比特币社区信任它的去中心化，以及不可改变的特性，才使它有了流通性。」


「我明白您的意思了！」艾伦恍然大悟，「如果有人控制比特币网络，想成为去中心化网络的中心，那他将破坏自治社区的稳定。一旦他做到了，比特币就不值钱了！」


「所以我说，没人能控制比特币。」王杰森躺回沙发上，「OK，今天的课就上到这里。」他拿起刚刚那份调查资料，从中间掉出一张表格，「这是什么？」


「这是一聊科技的 ToB 业务[[2]](#ref_2) 清单。为了增加现金流，张克亮将他们 App 的通信协议分离，提供给各企业进行收费定制。目前，已有十几家企业嵌入了这份服务。」


「查过吗，有多少用户？」


「没有详查，估算加起来几百万吧。」


「几百万吗？」王杰森握着报告，陷入了沉思，「看来，有必要让我们的张总尝点甜头啦！」


4.
==


北京，金融街。


一栋狭长的塔形建筑，上面没有挂任何铭牌，灰暗的外墙使它完美地融入夜色。不知道是第几层的第几个办公室的第几间房，里面正在进行一场热闹的圆桌会议。


UST 的中国区代理人罗琳端坐在圆桌外侧，她的对面，是三块巨大的电子屏幕。圆桌的中间是一个半圆形的玻璃罩，里面放着一块电子芯片，在玻璃罩的反光下，它显得格外神秘。


「我不想再催了。」右侧屏幕中的人用日语不耐烦地催促道，「DARK 什么时候可以部署？」


「我建议再等等。」左边屏幕中，穿白大褂的老者显得不慌不忙，「DARK 还有很多不完善的地方，尤其是防注入和对旁路攻击的防御，还需要花时间修正。」


「没有时间了！」右边屏幕中的人狠狠地拍拍桌子，似乎连屏幕都在颤抖，「再拖下去，UST 会被拖垮的。」他怒气难消，「供应商今年又提高了售价，对手的相关业务也已经上马，我们现在处于极度危险中！」


「您说得是。」白大褂缓缓附和，「但为了确保 DARK 的可靠性，我们要找到巨大的流量做测试。如果匆忙部署，或许会面临难以预料的后果……」


「那是你的工作！我只知道，如果不采取措施，股价会再次下跌！」右侧屏幕切换成了 UST 的股价走势图，「上周收盘时，我们的收盘价又下跌了近五个点。你算算，UST 今年的市值缩水了多少？！」


白大褂看着惨绿的股价，一声不吭。


「股价只是小问题，」中间屏幕里的年轻人说话了，「英特尔[[3]](#ref_3) 在全球电脑芯片市场的占有率超过一半，如果他们停止向 UST 供应芯片，后果将不堪设想，我们不能被人掐住咽喉。」他推了推眼镜，冷峻的脸上露出不容置疑的神情，「部署自主知识产权的芯片势在必行。」


见他开口，罗琳知道自己不能再保持沉默了。她连忙起身，双手贴裤缝地弯腰道：「好的，宫崎先生。」她看向那块芯片，立誓说，「下周……下周我就把测试方案拿给您。」


「谢谢。」宫崎耀一礼貌地笑笑，「UST 的未来就靠你了。」


 画面暗掉，灯光亮起。


寂静的会议室里，罗琳缓缓拿起电话。


「Mr.安，」她对安先生嘱咐道，「准备准备，跟莫飞见面吧。」


4.
==


莫飞用工卡在门锁上一扫，玻璃门自动弹开。凌晨一点，新脑的办公室里，电脑屏幕映射出来的光芒格外刺眼。「咋不开灯？」莫飞按下墙壁上的开关，屋内骤然亮起。


「我在探查网络攻击的源头，如今服务器只是暂时恢复，我们需要做好抵御下一波攻击的准备。」何阳手夹着烟，盯着屏幕，揉了揉猩红的眼睛，「在 DDoS 中，攻击者和防御者消耗的流量成本是不匹配的。假如别人花一块钱来攻击，那我们至少要花三块钱来防御，所以对我们来讲，这就是在烧钱。」


莫飞透过烟雾看去，屏幕上，是何阳测绘的网络攻击线路图，代表着流量动向的曲线从世界各地延伸而来，像一枚枚势如破竹的导弹，汇集到新脑位于北京大兴的服务器上，并将它轰个粉碎。


「节点太多了，但我相信它们一定有个源头。」说着，何阳反问道，「这大半夜的，你怎么也跑公司来了？」


「心烦。」莫飞把手中的塑料袋放在桌上，那是他刚买的盒饭，「没吃饭呢吧？」他递给何阳一双筷子，之后，又掏出一部手机，「你能帮我把它解锁吗？我想知道里面都有什么东西。」


「我试一下。」何阳匆匆扒拉了几口饭，接过手机，连上电脑。


「应该是老版本的安卓系统，我记得这个版本有个穷举法破解锁屏密码的 Bug（漏洞）……」他喃喃自语，在键盘上键入一串命令，不一会儿，屏幕上飞快地变换着数串字符，「需要点时间，等等吧。」 


「谢了。」莫飞说。


突如其来的这个谢字，让何阳无比别扭。他才反应过来，这是他第一次听到莫飞向自己道谢。


他回想起二人相识的过程，那时的莫飞意气风发，有一家规模不小的创业公司，但忽然被合伙人挤对出局，失去了一切。而自己，作为清华大学的优秀生，毕业后在一家跨国公司做开发，因心高气傲屡次跟领导起冲突，最终愤然离职，蜗居在孵化器里捣鼓各种小项目。


两个落魄的人一拍即合，踏上了联手创业的路。


但何阳常常觉得，自己并不了解这位朝夕相处的伙伴。


吃完饭，莫飞拿出何阳的烟，点上，用力地抽了一口。风把烟灰带向夜空，同繁星汇聚成一条奇妙的银河。


「那女的不错。」莫飞忽然说。


「嗯？」


「我说青雁。」


「是，」何阳握着筷子，不解道，「可她为什么要跟着白思成呢？」


「有啥想不通的。有钱又风流的男人，哪个女人不喜欢？」


「也是。」何阳苦涩地笑笑，「何况人家还要统治世界。」


「哈哈，统治世界。」莫飞也乐了，仿佛听到了世界上最好笑的笑话，可笑着笑着，他又皱起了眉头，「老何，你觉得白思成的那个大计划……靠谱吗？」
