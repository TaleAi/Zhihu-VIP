## 简介
世界信息过分繁杂，用数学思维清理一下。


什么是思维黑客？


信息爆炸时代下，保持清醒的人。


深入了解规则，不用已知方案解决问题的人。


  



他们拥有自己的一套认知体系，而这套逻辑离不开被我们忽视的数学。


数学不是一种知识，而是一种能力。


它是逻辑思维、形象思维、空间抽象思维的总和。


它实现了实用和理性的平衡。


这个专栏将从数学思维角度，给予大家更加完备，更加科学，更加理性，更加全面的思维方法论。
