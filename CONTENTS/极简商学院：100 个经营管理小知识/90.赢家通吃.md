## 90.赢家通吃
做思考达人，走精英之路。大家好，上一回我们讲了，寿司馆是靠寿司招揽顾客，靠酒水赚钱。今天我们来说说，赢家通吃。


业务的类型多种多样。比如所谓的「分散性业务」，就是一种竞争变量很多的业务，即便只有一家紧贴本地居民的店铺，只要能把它做好，就能收获足够的利润。房地产业就属于这种类型。


但有些业务必须做到世界第一或全球一流，否则就难以盈利。我们可以把它的特征归纳成本节的篇题——Winner takes all。推进此类业务时，「成为市场的龙头老大」就是战略层面的首要任务。


下面来为大家介绍两种典型的「赢家通吃型」业务。


一、网络型 IT 业务。


最典型的就是 SNS（社交网络）了。这类业务的关键机制是「网络经济」。用户越多，其他用户能享受到的便利也会越多。


二、存在「默认标准」的业务。


曾几何时，微软凭借电脑操作系统 Windows 叱咤风云，而 Windows 就是一种「默认标准」。因为 Windows 的用户多，所以许多应用程序都是以「在 Windows 平台上运行」为前提进行设计的。于是 Windows 就会变得越来越方便，赢得更多用户的青睐，形成良性循环。


这两种业务的共通点是「网络经济」与「IT 平台」。


所谓平台，就是供应商们提供产品或服务的共同基础。知名度高的平台有 Baidu（搜索平台）、Kindle（电子书平台）、iPhone 的 App Store（应用程序平台）等等。再通俗一点说，当越来越多的出租车司机脱离出租车公司这种传统方式，改用滴滴等平台接单赚钱、亦或是自媒体人用以发声的微信公众号和微博平台，放眼我们的生活，平台已经无处不在。一旦构筑起方便易用的平台，将用户「圈住」，就很容易吸引到更多的用户与互补品（注：相互提供便利的关系，如 DVD 机与 DVD 光盘），于是平台的地位便会坚若磐石，带来极高的收益性。


关键词：网络经济、默认标准、平台互补品。
