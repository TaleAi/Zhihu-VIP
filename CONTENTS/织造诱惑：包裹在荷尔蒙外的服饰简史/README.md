## 简介
纺织、魅力与荷尔蒙，这三个常见的元素交织在长达几万年的人类文明史中。其间，艺术的面貌经历了翻天覆地的变化，但是服饰与性的关系在无尽的时代变迁中，始终没有改变过。就读于世界三大服装学院 POLIMODA 的笔者，将结合国际服装设计理念和对传统文化的感悟，向读者讲述人类「织造诱惑」的历史和趣事。


在大多数人的眼里，服装只具有保暖与美化形象的功能，若是一定要与两性联系到一起，大家最多也就只能想到情趣内衣，是这样的吗？除了这些冷知识之外，本专栏还可以教你如何选择合适自己的服装和配饰，让你尽量少在穿衣打扮上走弯路。


  



服装的起源，始于生殖崇拜。服装的审美和性文化的发展分不开，由始至终，服装最初和最重要的作用都是强调两性差异，吸引异性，最终的目的，是性。


——《从史前到维多利亚：服装如何成为最有个「性」的文化化石》


服装中的性暗语都是由当代的文化，当代社会风情民俗，当代的政治现状所决定的，所以每一个服装中的性暗语，都带着历史的烙印。比如：压襟。压襟是流行于明清妇女，用来压住衣襟防止走光用的珠串等小物件。


——《服装中的暗语，带着历史的烙印》


现代社会的性感，是私人订制的，是根据你的社会角色来辅助定义的。你现在生活在哪里？你现在做的是什么工作？你的教育程度是什么？你的性感所要吸引的人群是哪些？了解了这些，你才知道应该如何穿出你独一无二的性感。


**——**《穿出你独一无二的魅力：现代服装设计如何重新定义性感》


执着的宋美龄一生都只穿旗袍，无论做了颜色多么鲜艳的旗袍，真正穿出门的，总是暗沉沉的那几件，从这里我们可以窥见，宋美龄所选择的服饰单一，颜色并不引人注目，未发出求偶信息，可见她与蒋介石之间的感情生活应该是很和谐的。本篇带你一起去看看历史名人爱穿的衣服所折射出来的，不为人知的秘密。


——《我们能从历史名人的服饰中所窥见什么？》
