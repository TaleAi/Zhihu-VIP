## 简介
历史上号称无解的经典阳谋，到底难在哪里？


「阳谋」是与「阴谋」相对的一种兵家策略。


靠着对人性和大势的洞察，阳谋高手通过把对手所有可能要走的路都堵死，倒逼对手走进自己的局。


一般情况下，阳谋是强势，甚至是占据绝对优势的一方才能发动的技能。由于本就大局在握，不怕对手掀起的任何风浪，更不屑于耍阴谋或是直接碾压对手，所以阳谋就成为了部分强者优雅而又不失逼格地灭掉对手的最佳选择。


历史上诸如汉武帝刘彻、魏武帝曹操、唐太宗李世民等略带傲娇属性的牛人都是阳谋的 VIP 中 P 级的忠实用户。


* **刘彻**削弱诸侯国实力的方法是「分蛋糕」，即诸侯王的孩子不论嫡庶，都有继承权。诸侯王如果不执行政策，要造反，儿子肯定不干，因为造反成功黄袍加身的是父亲，造反失败身首异处的是自己，太不划算。
* **曹操**挟持汉献帝，想以汉献帝的名义一统天下。各诸侯一旦选择不奉诏，便成了真正的「汉贼」，不仅会被群殴，还会失掉人心。
* **李世民**通过科举制，既增强了统治者的力量（让平民通过一层层的竞争，选出最尖端的人才为自己所用），又削弱了被统治者（世族豪强）的力量。
* **……**
