## 简介
甜文专业户「小柒」花式撒糖～


看小甜文里的各路宠妻狂魔，如何演绎宠妻的一百种方式。


《谋妻》


凤宁晚是京城最贤淑的夫人，过门第二日就给夫君陈钰纳了四个妾。一年过去，四个肚子，整整齐齐，安静如鸡。她忽然意识到：陈钰好像……不行……


作为一个合格的细作，她致力于唆使夫君宠幸小妾，其实她有什么坏心思呢，只不过想完成任务罢了。


可陈钰偏偏盯上了这个不太聪明的细作，高冷禁欲的丞相大人，开始了他苦心孤诣的谋算，让一个傻女人爱上自己，实在太难。


《天定良缘》


原以为是天作之合，郎情妾意，一场离奇的偶遇，秦娇撞破了未婚夫温文尔雅的假面，从此，简行知在她面前，彻底暴露出恶劣本性，三番五次的捉弄，温柔细语的哄骗，让秦娇这个脸皮薄、守规矩的大家闺秀乱了方寸。


简行知宠她，到底有何目的？面对所有人的袒护，秦娇发誓要挖出真相，揭开简行知虚伪的面具。


直到谜底层层剥开，秦娇最终发现，简行知「阴险狡诈」的外表之下，是一颗疼她入骨的心。
