## 简介
「前云南边防部队禁毒人员」田浩讲述边境隐秘故事，演绎缉毒战里承受不了的真实。数年实战缉毒的纪录，细节呈现边境英雄的真实人生，与命运过招，和魔鬼交易，同生死诀别。


云南边境，一场缉毒行动正在紧锣密鼓地进行。


一名抓捕目标却在精良的武警边防部队眼皮子底下跑了。部队成员黎耀祖不顾队长阻拦，丢下全身装备奋起直追。后来才发现，这位抓捕对象居然是自己人，是我方安插在毒枭集团内部的秘密线人。


这位线人死后，黎耀祖自告奋勇接手了他的卧底工作，改换姓名，继续未竞的任务，开始了跟梁氏贩毒集团的牛鬼蛇神们的斗智斗勇。


经历了一轮又一轮的作局与破局，流过血，流过泪，发过誓，发过狠，最终，他能否力挽狂澜？此时，一场看似风平浪静，实则暗流涌动的缉毒大战才刚刚开始……

**编辑推荐**


缉毒是一项长期、复杂、艰苦、危险的行动。在「缉毒」与「贩毒」的生死较量中，往往存在着许多不可预见性。警察、卧底受伤或牺牲，兄弟、亲人反目成仇。人性的背叛与堕落，亲情友情与欲望的交割，林林总总的人性真相，有着强大的震撼力和警示力。


作者田浩，服役云南边防队，从事禁毒工作数年。故事根植于真枪实弹的缉毒现场，来自作者多年来的缉毒经验。对专业实战场景的描述，告诉你什么才是真正的毒战现场。
