**《停电了，我们去南方》**


末日来临，我和领导赵发财并几个新认识的同伴，组成了一个小队。


冬天到了，我们深知自己挨不过去，有人提议去南方过冬，


生存物资有限，只有赵发财知道物资在哪里，由此引发了一场大乱……


  



**《逼仄之城》**


未来世界，人格分离&融合技术被发明了出来，不同的人格可以共享身体。


我在人格管理局工作，我心爱的姑娘即将失去自己身体的自主权，


为了和她在一起，我杀了一个人。

末日来临，我变成了一个丧尸，老詹姆是我的丧尸朋友。
