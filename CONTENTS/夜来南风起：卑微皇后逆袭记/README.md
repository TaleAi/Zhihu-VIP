## 简介
网络千万点击作品，一场守得云开见月明的冷虐痴恋。看性格孤直、不受帝宠的卑微皇后，如何在尔虞我诈的深宫内步步经营，逆风翻盘。


出身江南布衣之家的邹阿南，身世坎坷，因祖父曾有恩于太后水星，自小被接入宫中生活。


阿南与皇帝成灏青梅竹马，倾心于他。成灏却只把她当作值得敬爱的姐姐，深爱另有其人：太后宠臣之女沈清欢。


帝王之路上，成灏与太后矛盾渐深，阿南利用矛盾，成功离间了他和沈清欢，入主中宫成为皇后。


两人的婚后生活并不如意，成灏始终心系白月光沈清欢，与阿南在感情上有嫌隙。而随着邹家的强势崛起，成灏对阿南的防备之心也日益严重。


阿南怀孕后，成灏对她的忌惮达到了顶点，甚至打算在她诞下皇子后对她下手……


与此同时，权谋纷争，圣朝的前途也笼罩着一层阴云。


母族的荣耀、子女的前程、个人的荣辱……阿南该如何面对这危机四伏的深宫人生？


帝后的感情会如何发展？又能否携手解决前朝后宫的内忧外患？
