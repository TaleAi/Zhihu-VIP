## 简介
我们生活中的所有关系问题，归根结底，是沟通的问题。我是心理咨询师胡慎之，在这门私家课中，我将结合 12000+ 小时的个案咨询经验与 20 年的心理学研究历程，带你重新认识人际关系的 3 大问题：我和我自己的关系、我和他人的关系、我和社会的关系。


为什么你总遇不到「对的人」，在情场上屡战屡败？


为什么你和父母越来越「无话可说」，成为了精神上的「陌生人」？


为什么你总是习惯性地讨好，还是换不来同事的友好？


为什么你已经绞尽脑汁避免尴尬，还是经常把天「聊死」？


高情商的人能正确地认知自己、他人和社会；而一段好的关系里，你可以顺畅表达自己的观点需求，懂得设身处地换位思考，从而建立相互滋养的合作共赢。


好的恋爱婚姻正是关系滋养和共赢的体现。美国心理学家 Gail Goodman ，研究了许多新婚夫妻后发现，一对幸福的新婚夫妻，其中一个人肯定是关系沟通的高手，TA 会关怀和体贴对方，营造出亲密的氛围。


我们的朋友圈中，也一定存在这样几个人，他们对局面有掌控，对未来有余地，对他人有宽容，对自己有约束；他们以相处舒服且相互促进为终极目标，是人群中最受欢迎的人。


在职场上，良好的关系维护也尤为重要。工作上稳步上升的人，除了能力出众，还能正确处理正面与负面的评价，形成自我认知和外界认知的统一；他们能管理好情绪，与不喜欢的同事也能和谐共事，与意见不同的人争执的时候，能够礼貌而坚定。


本门私家课将把工作、生活中最常见的几十种人际关系，一一分析给你听。作为你的私人心理咨询师，我将手把手教你如何成为一个高情商的人，让你在社交技巧、情感魅力、家庭关系、职场相处上，全面获得提升。
