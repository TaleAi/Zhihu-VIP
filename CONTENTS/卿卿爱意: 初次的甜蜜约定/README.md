***「竹马撩人」***


我不小心睡了我的竹马，兼我的大学班主任。


一个自小品学兼优，容貌超绝，却片叶不沾身、从小狗到大的男人。


我猜他睁眼看见我的时候，一定在想怎么处决我吧。



***「偷了隔壁弟弟的菜之后」***


疫情期间，偷了帅哥舍友的剩菜被抓个正着后，我索性厚着脸皮开始蹭吃蹭喝，谁让他温顺实惠，好吃不贵呢？可我没想到，有天他将我诱入房内，从身后圈住我，低语道：姐姐吃了我这么多东西，不会真以为不用付一点代价吧？
