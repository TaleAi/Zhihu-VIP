## 简介
作者与编辑，明星与资方，职员与上司，销售和客户……


都市里存在着各种各样的共生关系，人们被金钱和生活的压力紧密连接在一起。


为了职业理想也为了安稳的生活，共生者们不得不将最大的信任和最长的社交时间交付给对方，这样的生态，让他们被迫建立了独特的「亲密关系」。


工作压力，网络暴力，原生家庭，催婚、高昂的房租和社恐、躁郁等问题的困扰下，共生者之间的那一点点亲密或将成为都市人最后的的救命稻草。


当爱情爆发在共生者之间，是不是我们枯燥的生活，也可以变得没那么苦涩。
