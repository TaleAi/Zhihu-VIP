## 简介
这是一部中国都市犯罪档案，香港、深圳、丽江、南京……你曾生活过的城市有哪些震惊世人的滔天大案？那些血腥的第一现场、那些人性泯灭的杀人恶魔、那些失去挚爱的受害人家属，他们不该被遗忘。


**《深圳六魔女》**


上世纪九十年代，深圳发生了一起震惊全国的特大连环抢劫案，多名司机失踪，从出租车司机到开豪车的老板，连人带车的凭空消失，几天后司机尸体会出现，车则被销赃。


经过一年的时间，该团伙终于在警方的侦破下落网，人们惊异地发现，主谋竟是六个美丽的女孩。


**《血色丽江》**


丽江河内发现被分尸的尸体，经过警方检验调查得知死者是某工程的负责人，生意上没有仇家，感情上极为混乱，有家庭孩子但女人不断。


警方调查到，他的情妇之一是个因男友介绍在夜总会陪酒的女大学生。经调查，死者曾性虐过该女生，并且有过纠纷。展开调查抓捕后，女学生承认了罪行，但事情却远没有这么简单。


**《香港德福花园五尸命案》**


九零年代的香港，某富商报警，说自己的妻子失踪了。警方来到富商太太的朋友家，发现朋友家门口布局很奇怪，挂着疑似玄学的东西。推门一看，加上富商太太的五具尸体早已凉透，警方初步认为是集体自杀事件。德福花园人心惶惶，人们纷纷传言，五个女人是 19 年前死于非命的建筑工人在为自己寻找鬼新娘！此时，生前为三个富豪太太作法的李育辉道士突然决定离港，案情有了新的转机。
