## 简介
国际争锋总是冷战与热战交替。


在当今和平年代，我们提起国际争锋，总是更关注外交。


但在历史的浪潮中，国际争锋总在冷战与热战中交替。


国家博弈的进程中，谁拥有更强的军事实力，就拥有更多的话语权。


本专栏将带你看透国际争锋中的冷战与热战：


* 水门桥战役对当时的局势意味着什么？
* 冷战时期，为什么美苏要造几千颗核弹？
* 阿根廷为何会由发达国家变为发展中国家？
* 美国为什么就是不放过俄罗斯？
* 阿富汗为什么是「帝国坟场」？

本专栏邀请众多军事战争领域的优质创作者，带你看那些匪夷所思的战争纪实，解析大国背后的博弈操作，分析战争对一个国家乃至世界格局的影响。
