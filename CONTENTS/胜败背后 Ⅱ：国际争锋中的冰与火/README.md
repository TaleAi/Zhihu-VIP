国际争锋总是冷战与热战交替。


在当今和平年代，我们提起国际争锋，总是更关注外交。


但在历史的浪潮中，国际争锋总在冷战与热战中交替。


国家博弈的进程中，谁拥有更强的军事实力，就拥有更多的话语权。


本专栏将带你看透国际争锋中的冷战与热战：
