## 简介
作文是语文学习中非常重要的一环——想要在语文考试中取得高分，作文写得好是不可或缺的条件；然而现实中，不少学生对写作怀有恶感和恐惧，一写文章就头疼。人大附小名师作文课为莘莘学子提供了一份高质量的写作指引，有针对性的提高写作能力，提升考场作文分数。


作文是语文试卷中占据分数最大的一块，写好作文，整个语文成绩就会上升一个台阶，然而，写作文却是很多孩子的大难题。


拿到作文题目，脑袋空空不知道写什么好。


搜肠刮肚，想到写什么了，不知道怎么下笔。


动笔写了，要不肚子里没词、语言贫乏，要不就是信马游缰、越写越散。


然而，无论是上述的哪一种问题，本课程都能很好地解决。


![img](https://pic4.zhimg.com/v2-543f2108fff2d4baff851d08ecbe1aba.jpg)

这门专栏特邀资深语文名师，紧扣小学语文国家课程大纲，采用层层推进的教学方法：


第一层，精选名家名篇，逐字逐句进行解读。


第二层，基于名家名篇，总结出切实可行的写作方法。


第三层，课后练习，将小学作文最常见的题目留作课后练习，让孩子练遍常见作文题目，考试有底，心里不慌。


三层方法层层递进，有效解决孩子的写作问题，让写作文不再是问题。
