## 简介
被困的你，甘心成为被驯服的那只象吗？


身陷囹圄，心念迷离时，你还会记得自己最初想要的是什么吗？


**《午夜的婚礼》**


欠我钱一直没还的大学同学突然邀请我去她的婚礼，顺便当面还我钱。


我答应了，但很快就有点后悔。


因为她婚礼的时间实在有点劝退：


深夜 12 点。


婚礼上，种种怪象让我疑惑不解。


婚礼后，我更是频频看到幻象，惊恐不已。


为了让自己的生活回到正轨，我重回一切开始的地方——那场婚礼的现场。


但现实只让我看到了更黑暗的深渊。
