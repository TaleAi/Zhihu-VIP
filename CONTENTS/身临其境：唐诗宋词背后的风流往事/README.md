## 简介
读过那么多唐诗宋词，你对诗人的真实人生了解有多少？从小死记硬背学诗词，诗歌背后的真意你都了解吗？……本专栏用 200 多首经典诗词的创作情境还原，带你穿越唐宋，如临其境；15 位唐宋名家的人生「大电影」，走进千古风流人物；600 年唐宋家国事，给你一份行走在诗词中的历史百科。


**【读过那么多唐诗宋词，你对诗人的真实人生了解有多少】**


忧国忧民的诗圣杜甫，也曾是意气风发的「高富帅」？一心要做「帝王师」的诗仙李白，还有个打打杀杀的江湖侠客梦？古今第一全才的苏东坡，一辈子起起落落都是因为那张嘴？……


**【从小死记硬背学诗词，诗歌背后的真意你都了解吗】**


《滕王阁序》原来是王勃杀人入狱、死里逃生后的命运嗟叹？《长恨歌》为何是白居易自身悲情苦恋的真实写照？《夜雨寄北》是李商隐是写给友人的问候之语，还是怀念亡妻的深情之作？……


**【你知道如何将诗词和历史如何结合学习，事半功倍吗】**


安史之乱中，李白、杜甫、王维谁最遭殃？永贞革新如何让刘禹锡、柳宗元的人生深陷泥潭？李清照颠沛流离的后半生和靖康之难有哪些千丝万缕的联系？…..


本专栏用 **200 多首经典诗词的创作情境还原，带你穿越唐宋，如临其境；15 位唐宋名家的人生「大电影」，走进千古风流人物；600 年唐宋家国事，给你一份行走在诗词中的历史百科。**


诗人生平、经典古诗词与历史事件的完美融合，让孩子听故事、学诗词，打破学不懂、记不住的魔咒，让家长和孩子一起博古通今，共享美妙亲子时光，让老人晚年的精神世界更丰富，多一份陪伴和消遣，成为「爱诗歌、懂文学、通历史」的书香家庭。
