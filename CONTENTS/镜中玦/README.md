## 简介
这是一篇仙侠玄幻长篇言情。


石曦本是一块石头，落入了镜墟，遇上黑狐不孤，和白蛇小龙，并在不孤的「孵化」下化成人形。


石曦身有女娲之力，力量被封印，失去了记忆，为了解开封印找回记忆，三人离开了镜墟。


他们在人间遇上妖鬼杀人，在救人途中又与神秘人长隐相识。


经过一些坎坷磨难，三人终于发现妖鬼背后竟是——天帝。


为了躲避天帝追杀，三人在长隐的帮助下逃到妖界，但最终还是被天帝发现三人的踪迹，在最后关头，石曦被带回天界，不孤假死，小龙下落不明……


她该如何逆转局面？
