## 简介
穿成被养废的嫡长女该怎么破？
古中医世家的大小姐，一觉醒来发现自己穿越了？！


说是嫡长女吧，其实是个爹不疼娘不爱的可怜虫，徒留个名头罢了，原主被退婚寻了短见才有了她，夭寿啊，她穿过来就是来收拾烂摊子的吗？真是倒霉透顶。算了算了，来都来了，那就既来之则安之吧。伪善继母、白莲继妹、狠心渣爹接踵而至，哦吼？宅斗啊，宅斗我拿手，嘿嘿。


「那个，你刚才说什么来着？」慕容久久回过神来。


「大小姐，您又被退婚了……」宁儿被气得暴走。


「嗯，我知道啊，不是三天前就退了嘛？今天又退的那门子婚？」慕容久久一副闲事不理的样子。


「您果然忘了。」宁儿无语问苍天。


最终，一开局就被退了两次婚的她，能否利用自己的医术和宅斗技巧，活出属于自己的精彩人生，收获幸福呢？
