## 简介
本专栏为你提供阅读向导，重启你的文学阅读快感，充分感受文学的治愈性，建立文学阅读思维。从专家视角入手，不以枯燥的文本读物来讲文学，影视与歌曲等思维表达形式，都可以作为文学的表达。


文学诗歌巨大的精神资源宝库，阅读世界名著是享用文学不可分割的一部分。


世界名著欣赏不来，质疑自己没文化？


无法理解书中细节伏笔，甚至感觉情节拖沓？


一看书就犯困走神，完全 get 不到月阅读的快感？


......


这个世界从来不缺少文学的盛宴，真正稀缺的是懂得品尝的味蕾，用细嚼慢咽的方式谈一谈「文学的甜头」，这个世界的滋味又多了一点。


本专栏用通俗直白的语言打开文学大门，以「说人话」的方式将文学批评、文学理论与思维相结合，拓宽你的事业，让文学为你服务。
