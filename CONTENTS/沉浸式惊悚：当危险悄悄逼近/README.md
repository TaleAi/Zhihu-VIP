## 简介
多个互相独立、风格迥异的悬疑故事。


● 妻子最近变得非常古怪，我们共同使用网络账号的 APP 也不断向我推送一些与杀人相关的信息，种种线索告诉我，最亲近的人有想杀害我的念头…… 


● 我竟然用一元钱在拍卖网站上拍得了一栋别墅的十年使用权。当我兴致冲冲去到那里，却发现一些怪异的事随之出现……


● 独居女性的门前被贴上某种符纸，几天后她便失踪了，我追踪蛛丝马迹来到某个山村，发现她已经和一个死人冥婚了……


● 和未婚妻去本地某著名景点拍婚纱照，照片中却出现了「鬼影」。后来我才知道，在日据时代，这里曾是万人坑，而鬼影的诅咒似乎在逐渐逼近……
