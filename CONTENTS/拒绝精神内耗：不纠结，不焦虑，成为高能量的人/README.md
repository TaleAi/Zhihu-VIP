## 简介
告别精神内耗，掌控情绪，驾驭人生。


「为什么我内耗那么严重啊？」


不要苦恼，不只是你，人人皆内耗，没有人可以避免内耗的影响。


因为内耗，我们做事往往瞻前顾后，工作效率很低；


因为内耗，想得太多，社交时小心翼翼，生怕得罪人；


因为内耗，脑力劳作旺盛，明明没有干什么，就是很累……


很多时候，阻碍我们成事的，不是方法论不足，而是胡思乱想耗尽了电量。哪里还有精力支棱起来？


内在不冲突，是成事的前提。


那为什么有的人内耗少，想法干脆，做事果断呢？


本专栏的作者何圣君，实战心理学研究者，同时也是反内耗的践行者。他曾经也是一个对未来迷茫，心态不那么稳定的拖延症患者。当关注内耗，研究成因并践行解决方法后，他也成了一个每天 5 点起床写作，持续出版多部著作，通过文字影响了 1000 万人的人。


通过本专栏，你将学习到他的研究结果：


* 到底什么是内耗？
* 内耗包含了什么样的心理问题和表现？
* 对不同的内耗行为，我们如何才能有效解决它？

真正排除不必要干扰，掌控自己的生活。
