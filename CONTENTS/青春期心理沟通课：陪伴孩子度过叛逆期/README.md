## 简介

当孩子进入青春期，许多家庭也进入了「敏感期」，那么青春期的孩子究竟是怎样的呢？下面这几组词或许都可以形容他们：


天真、多变、缺乏安全感；


好奇、易怒、敏感；


不合群、冲动、喜欢寻根问底；


叛逆、冒险、固执；


标新立异、理想化、不切实际；


关注未来、积极主动、勇于尝试……


其实，不论好坏，上述这些关键词出现在青春期的孩子身上是再正常不过的事了，但青春期中还有许多重要问题值得我们注意：亲子关系危机、校园霸凌、同辈压力、早恋、身份认同、厌学、离家出走……作为家有青春期孩子的家长或是正在经历青春期的孩子本人，如何面对这些挑战至关重要。
