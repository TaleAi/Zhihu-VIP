## 简介
本专栏以《西游记》与《封神演义》为蓝本，将两者结合在一起展开分析解读，用趣味易懂的文字，为你揭露「所有的神仙佛菩萨妖魔鬼怪，都是人的化身」这一现实。


《封神演义》成书晚于《西游记》，不知是有意还是无意，作者将《封神演义》的故事写成了《西游记》的前传。


读《西游记》，我们经常一头雾水，对神仙的关系、妖精的由来完全不知道是怎么回事。甚至会看到特别多矛盾的地方，无法理解。读《封神演义》也难免有一些不好理解的地方。如果将两本书放在一起，理出一个逻辑来，再读原著，则豁然开朗。


本专栏基于三个政治学的基本设定：人性本恶论、资源稀缺论、权力政治论，以《西游记》与《封神演义》为蓝本，将两者结合在一起展开分析解读，用趣味易懂的文字，为你揭露「所有的神仙佛菩萨妖魔鬼怪，都是人的化身，也因此具备基本的人性」这一现实。
