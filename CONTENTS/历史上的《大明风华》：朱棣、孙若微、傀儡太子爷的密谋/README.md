## 简介
明史学者向敬之为你讲解电视剧《大明风华》背后的历史故事。那是大明王朝最跌宕起伏的时期，横跨五十年，四代人，五个帝王，权力、阴谋、忠臣、爱情，道不尽的历史沧桑。


电视剧《大明风华》在湖南卫视黄金时段热播，以朱瞻基与孙若微情感纠葛为主线，贯穿朱棣、朱高炽、朱瞻基与朱高煦、朱高煫之间的权力之争，再现了朱棣发动靖难一役后，永乐至正统近 50 年历史。这些扣人心弦的剧情背后，真是这样惊心动魄的历史吗？


知名明清史学者、独立书评人向敬之，在这里为您讲述这一段不忍细看的历史背后恩怨，以及《大明风华》没有讲述的故事。


有人说：永乐皇帝是权谋、怨恨和无赖行径的操弄高手。


他真的是这样的人吗？「好圣孙」又是怎样出炉的？


体肥多病的朱高炽，这是朱瞻基的傀儡、龙椅上的活死人吗？


朱瞻基与孙若微的原型，是仇人相爱，还是青梅竹马？


明英宗接手了一个鼎盛的大明王朝，为何遭遇断崖式厄运？


……一切的一切，求索历史真相，探究盛衰缘由。


明史专家说：向敬之写的明史，细腻，爆发着细节的力量。
