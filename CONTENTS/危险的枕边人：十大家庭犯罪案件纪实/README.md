本专栏记录了十个引起震惊和轰动的家庭凶杀案件，为你带来历史上十个夫妻离心以致家破人亡的案件纪实。

世间只有两样东西不能直视：一个是太阳，一个是人心！


作为伴侣，要关注枕边人，如果人心已变，就要及时远离，决不能被劣质婚姻拖下水。


作为犯罪人，别忘了，善恶到头终有报，你凝视着深渊，深渊也在凝视着你。


这十个真实案例，都是因感情琐事、家庭矛盾引发的惨案。这些犯罪人有的已经伏法，有的还在狱中，他们不仅给受害人带来痛苦，也葬送了自己的一生。让我们再次剖析这些案件，发现罪恶根源，反思现在的自己，从而更加热爱生活，珍惜生命，希望悲剧不再上演。
