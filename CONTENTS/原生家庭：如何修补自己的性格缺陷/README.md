## 简介
本专栏结合大量原生家庭中存在的问题案例，分析父母的不当行为如何对子女造成伤害并影响了子女成年后的生活，同时提出对策，帮助有类似经历的人摆脱原生家庭带来的困扰。


幸福的人一生被童年治愈，不幸的人一生都在治愈童年。原生家庭对每个人的影响，或积极或负面，但都深入骨髓，且伴随你的一生。


针对原生家庭中各种各样的问题，苏珊·福沃德博士通过工作中接触到的大量真实素材，分析了各类「有毒父母」的所作所为，以及这些行为如何伤害了子女并特续影响子女成年后的生活。


但是，本专栏主旨并不在于控诉这样的「恶劣」父母，而在于传授具体的对策，使那些受过或仍在承受父母伤害的人们获得勇气，帮助他们从与父母的负面关系模式中解脱，恢复自信和力量，得到自由和幸福。


过去已经发生，但未来可以改变。
