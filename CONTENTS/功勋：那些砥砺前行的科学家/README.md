## 简介
如果人类历史有一块功勋碑，他们的名字都应该刻在上面。


相较于大众更关注的明星八卦，和科学家有关的事情，好像只会出现在课本的只言片语上。


可能最被熟知的那些人会有个人传记，但更多的贡献者们，只是在默默付出，甚至有些还会长期被「黑」，被大众误解。


本专栏着眼于那些在历史进程中，为人类文明、科学进步所付出过的科学家们，讲述他们的人生故事、还原他们为世界做出的贡献。


希望所有人都能知道，「今日的世界，之所以能成为现在的模样」，离不开这些砥砺前行的科学家们。
