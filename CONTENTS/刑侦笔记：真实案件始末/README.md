## 简介
通过这些案子，探讨隐藏在迷雾背后的生存、社会、家庭和人性的复杂。


作为一名刑警，参与过许多案件，每个案件就好像一副拼图，刑警要力求还原图的本身，这些图的背后，是一个个复杂的人心和人性。


每一件案子都承载着一条条曾经鲜活的生命。


那些看似匪夷所思的案件背后，都是某些人曾经最真实的人性。


还原案犯在作案时的心态，作案前的心路历程。


讲述在办案过程中，那些曾经有过的情绪——欣喜、愤怒、无力和绝望……
