## 简介
加速社会中，如何追寻持久的恋爱关系？


爱情是一场浪漫的游戏，也是困扰每个时代万千男女的难题。


加速变化的当下社会，为什么「脱单」越来越难？


爱情为何会走向悲剧的结尾，要如何谈一场「不分手的恋爱」？


婚姻能给我们带来什么，现在的年轻人适合步入婚姻吗？


……


爱情是人们永恒的追寻，让无数有情人「痛并快乐着」。


《苔丝》《呼啸山庄》《霍乱时期的爱情》《雪国》……爱情也是作家笔下永不褪色的故事，让万千读者或喜或忧。


复旦大学中文系教授梁永安带你走进经典文学作品，探求问题答案，看见爱情本色。


正如梁教授所说：「爱情确实是可以改变人的命运，使我们在纷乱的社会生活里面找到自己，让生命水落石出。」
