## 16.【情绪】表达情绪能够改善失眠？
情绪，是指我们内心的感受。爱、恨、情、愁，喜、怒、哀、乐，都是对情绪的描写。比起思维来，情绪更加接近人的内心，对人身心健康影响更大。情绪本身就是一股能量。这个能量如果不释放出来，就像「炸弹」一样，会有爆炸的危险。如果把「情绪炸弹」拆除了，身心就安全了。所以我们一定要把情绪，尤其是负面情绪，释放出来。如果情绪释放不出来，就容易导致很多问题。会出什么问题呢？中国传统医学认为情绪可以导致脏器损害，如：怒伤肝，喜伤心，思伤脾，悲伤肺，恐伤肾。现代心理学认为，情绪的压抑可以表现为躯体化或者攻击性。


躯体化表现为各种躯体不适症状，最常见的是疼痛，而使用现有的各种医学检查都查不出明确的器质性疾病的原因。攻击性常表现在对自己的攻击或者对外界的攻击。对自己的攻击可以是身体上的，如划伤皮肤、以头撞墙等；也可以是心理上的自我攻击，如无力感、无望感、自责等，严重的会发展为抑郁症。对外界的攻击形式更多，如批评、谩骂、抱怨、打架等等。比如现在常见的「路怒症」。别人开车变道时，没有开转向灯，「路怒症」的人就受不了，非要超车到人家前面猛踩刹车，把人家逼停。这就是因为长时间的情绪压抑，没地方释放，借机发泄出来。


那么，我们应如何管理自己的情绪呢？主要可以从「觉知」、「表达」、「不住」、「破执」、「无别」等几个方面开展。


第一点，我们先来说一说觉知


管理情绪就像《孙子兵法》所讲的那样，首先要做到「知己知彼」，才能「百战不殆」。首先要知道「敌人」是谁，你才能去跟他打仗。管理情绪也是这样，你必须要知道你的情绪，你才可以去管理。所以，情绪管理的第一步就是要先觉知到自己的情绪是什么样的。如果连自己的情绪都感受不到，如何去管理它呢？但是很多人平时觉知不到自己的情绪怎么办呢？那就借助正念呼吸练习。在正念呼吸练习时，人的内心比较安静，这时候容易听到内心的声音，用心细细去感受你的情绪。感受到自己的情绪以后，给情绪进行命名，是愤怒？委屈？悲伤？还是恐惧？一旦感受到并给情绪进行了命名，那么就对情绪有了觉知。


接下来我们谈一谈第二点，表达


当我们觉知到情绪以后，要学会把情绪表达出来，也就是释放情绪。我们该如何去表达呢？主要有四个途径。分别是：向当事人表达、向他人表达、向环境表达、向自我表达。


1、向当事人表达


谁让你产生了这个情绪，你就对他说出来。尤其是在家庭关系里面，比如夫妻关系、亲子关系，鼓励大家要把自己真实的情绪感受说出来。如果平时不表达出来，堆积之后就会大爆发，后果就非常严重了。很多夫妻之所以离婚，就是平时没有很好的进行情感的沟通。表达的方式是以「我」开头，而不是以「你」开头。可以说「我感觉很委屈」，而不能说「你让我感到委屈」。如果以「你」开头，就容易让对方感受到指责，从而影响后续的沟通。鼓励孩子表达自己的感受更加的重要，我们要允许孩子学会说「不」。但是，我们现在子女教育大多是「听话教育」，家长都希望孩子「听话」。很多时候，「听话」就是在压抑孩子的情绪，导致孩子出现很多心理问题。作为父母，我们要鼓励孩子说出自己真实的感受，允许子女说「不」。


2、向他人表达


如果不方便向当事人表达，就可以采用向他人表达的方式。比如我们在单位受到领导的批评，你很委屈、愤怒，但又不能向领导说出来，这时候就可以找家人、朋友诉说出来。从心理学上来讲，如果把你的痛苦告诉你的朋友，你的痛苦就能减少一半。如果你家人、朋友里面没有这样的倾诉对象怎么办？那就可以去找心理治疗师。心理治疗起效的一个重要的因素，就是治疗师帮助来访者表达了压抑的情绪。


3、向环境表达


就是把情绪发泄到环境当中。最常用的向环境表达方式是运动、呐喊等方法。当心情不好时，做一些运动，出点汗，会让人感到畅快淋漓、身心轻松。呐喊也是非常好的向环境表达情绪的方法，比如「喊山」、「喊海」等方式，都可以释放内心压抑的情绪。有些地方有「宣泄室」，可以在里面尽情的踢打喊叫，也是不错的向环境表达情绪的方式。


4、向自我表达


就是自己看到自己的情绪，也就是我们前面提到的觉知情绪的过程。觉知本身，就是一种情绪的表达。觉知力提高以后，可以随时随刻看到自己的情绪。如果在情绪刚刚开始起来的时候就能觉察，情绪就不会继续恶化，也就所说的「不怕情起、就怕觉迟」。觉察太迟钝了，就容易导致更严重的情绪暴发。一滴水，任何人都可以挡得住；当一滴水变成一江水的时候，人就挡不住了，情绪的洪水就会暴发。


其实，情绪本身是没有好坏之分的。之所以有好坏之分，就是因为这个情绪有没有被看到。情绪好比是一股能量，当这个能量发射出去以后，被对方看到了、接纳了，那么这个能量就是「白色能量」；如果你发出的能量对方看不到，或者看到以后不接纳，这个能量就是「黑色能量」。我们在与人相处时，有一个非常重要的沟通的技巧就是「共情」，也就是看到对方的情绪。你看到了对方的情绪，并能保持接纳，这就是最好的礼物。所以，心理学上经常讲「看见就是爱」、「抱持就会变」，就是说我们要学会看见别人发射过来的那股情绪能量，并能接纳。抱持就是接纳的意思，一旦能量被接纳了，黑色能量就自动变成白色能量，也就是「抱持就会变」的道理。


第三点是不住


不住就是不执着的意思。我们觉知到情绪以后，不要执着在情绪上面不放，就好比看着一条河里的水一样，河水来了它又流走了。我们的情绪也是这样，没有一个情绪能够一直存在的，它都像河水一样流过来又流走了。如果它流不走，那最主要的原因是你和情绪较劲。对于情绪，掌握了「八字方针」就可以不住了。「八字方针」就是：「不迎、不随、不拒、不抗」。意思是：情绪来的时候，不迎接它；情绪走的时候，不跟随它；不拒绝情绪来；情绪来了也不和它抗争。


第四点是破执


破执，就是破除思维的执着。思维的执着，其实就是我们所坚持的一些不合理的思维规则、信条等，并把这些观念当成了「真理」一样信奉。从认知心理学角度来看，导致我们情绪反应的不是事情本身，而是我们对事情的认识态度。这就是认知心理学的 「ABC 原理」。A 是事件，B 是信念、想法，C 是结果。A 事件不会直接导致 C 结果，而是因为 B 信念决定了 C 结果。


举个例子，假设你一个人在家里待着，突然家里的门「咣当」响了一声。这是事件 A。这个事件会导致你什么样的反应呢？起决定作用的是你如何看这个响声，也就是你的想法 B 在起作用。如果你的想法 B 是认为有小偷进家里了，那么你的结果 C 可能是害怕、紧张；如果你的想法 B 是认为风吹动了门才导致响声，那么你的结果 C 可能并没有情绪波动。


所以同样的一个事情，你的反应不一样。决定你反应的，就是你的思维规则、想法、信念等。如果你能够找到思维规则，并打破这种思维规则，那么你的情绪反应也就自然改变了。如何找呢？第一，凡是你观念当中含有「应该」、「必须」等词汇的时候，基本上都是你的思维规则。失眠者经常持有的关于睡眠的思维规则有：「应该睡 8 个小时」，「应该沾枕头就着」，「应该一觉到天亮」。第二，「当……时，才……」的句式结构，也提示着思维规则的存在。比如：「当有钱了，我才幸福」，「当学习成绩好了，才会有幸福的人生」。


第五点是无别


无别，也就是没有分别心。分别心是人类痛苦的根源。我国明代哲学家王阳明著名的「四句教」：「无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物」，首先指出人心的本体没有善恶，也就是没有分别心。「格物致知」的修行，其实就是让人回到没有分别心的「心之体」。没有了分别心，人也就没有了烦恼和痛苦。没有睡和醒的分别心，就没有失眠的痛苦。我们之所以失眠了痛苦，是因为你认为睡了就是休息，醒着就是消耗。这个分别心一来，你就有失眠的痛苦了。


**这节课主要介绍了情绪管理的方法。很多人失眠，往往都是因为情绪不好所诱发的，比如和家人吵架、和同事闹矛盾之后开始失眠。如果掌握了情绪管理的方法，就不会因为这些事情而诱发失眠了。同时，当情绪表达出来以后，失眠也自然得到了改善。**
