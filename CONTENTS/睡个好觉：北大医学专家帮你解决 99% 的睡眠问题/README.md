## 简介
睡眠占据了我们生命 1/3 的时间。失眠会让记忆力减退、精神不振、免疫力下降，长期失眠甚至会引发高血压等其他疾病，对工作生活和健康影响巨大。「睡不好」并非无法解决。北大医院睡眠专家将从行为、药物、心理等方面，结合 14 年临床经验，科学专业，解决你的睡眠问题。
