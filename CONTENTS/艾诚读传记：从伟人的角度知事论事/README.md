## 简介
双语主持人艾诚对话千余世界顶级人物，阅读世界人物传记。本专栏从传记人物的财富法则、人生历程、处事之道、思维密码、格局胸怀、魅力灵魂六大部分为你全方位解读，帮你扫除盲点，让你快速深入窥透历史人物。


你为什么要读人物传记？


读人物传记可以帮你窥见世界的复杂：英雄名人的事迹，更多并不是为了领略其智慧，而是感慨其环境之复杂，共鸣其决策之艰难；


读人物传记可以帮你认识残酷法则：读传记后我们才会体会到不能左右的切肤之痛，种种算计的辛酸残酷；


读人物传记可以帮你找到生活小确幸：不平凡的人生背后，牺牲了各式各样看似平凡或者普通的幸福。也许平凡的人生，未尝不是我们的幸运；


读人物传记可以帮你从容面对悲喜无常：他们内心的纠结、无助、彷徨等人间清味，与我们并无二致。无论悲或喜，我们不是一个人。


英雄们的成功必然与他们所处时代自我的把握有关，我们在对比中可以从他们身上找到「平行力量」，可以从中思考人生的极限。


读传记，在精不在多。如何在琳琅满目、古往今来的传记中找到适合自己的那本呢？本专栏带你走进 50 位中外名人的故事，站在伟人的肩膀上，从伟人的角度知事论事。
