## 14.Google Docs：最高效的免费办公工具
今天我们一起来学习如何像 Google 精英一样，高效地协同办公。这里介绍一个 Google 最常用的工具：Docs 。


不用我说，你可能也知道，Google 是一个特别不能接受做事低效率的公司，相信也是基于类似的原因，他们开发了 Google Docs 这个神奇的东西。它让我在 Google 的工作效率有了一个质的飞跃。这也是我个人在 Google 的众多工具中最喜爱的一种，即使离开 Google 很多年，也是念念不忘。


在正式讲之前，我先给你描绘两个场景吧，在后面，我会讲到 Docs 是如何解决场景中这些问题的。


我们先来看看第一个场景，一般公司是怎么在项目组中，进行内容同步和整合的。


通常是这样子：公司里几个人共同负责一个项目方案的 PPT ，每个人做完便会发给其中的项目负责人汇总，可是刚汇总完， 有人又发了一个，说是有一个小的调整，于是项目负责人又重新汇总了一次，可这时另外一个人也说自己的版本有一个地方需要调整，于是项目负责人又得重复之前的工作。汇总若干次之后，也许他已经不确定自己手上的那份汇总的 PPT 是不是最新的了。无奈之下，他只能再把这份 160 页的 PPT 发给所有人，每个人检查自己的那部分是不是最新的。


另外，再跟你描述一个场景：


你在写一封重要的邮件，酝酿了 30 分钟，仔细斟酌了正文中的每一个词之后，小心翼翼点了发送。但几乎同时，你意识到忘了一件很重要的事——加附件！


好了，上面这两个场景听着是不是很熟悉呢？ 相信在公司里，你也见过，或者经历过。因为，我们大部分人都习惯于先创建一个 Office 的文档，然后再输入内容，最后通过邮件发送给别人。但是，就像上述场景描述的那样，在这个过程中你可能会出现各种错误，还有，不同版本的 Office 文档也很让人头疼。


所以，这种传统的办公软件，在这个快速变化、高度协同的社会已经不能满足大家的办公需要了。为了解决这个低效协作的问题， Google 开发了一套在线办公软件，叫做「Google Docs」。


它类似于微软的 Office ，相当于在线的 Word 、Excel 、PPT ，同样的操作界面、基本一致的功能，这样任何 Docs 的新用户都能很快上手。但是它跟 Office 相比，有 4 大亮点：


**第一个亮点是支持多人实时协作编辑。**也就是很多人同时在线编辑同一个文档，而且能看到谁在阅读、谁在编辑，以及编辑哪个部分的内容。比如说每周周五写周报的时候，当你点进去周报链接的时候，从旁边状态栏的名字，你就会看到有谁在和你一起写周报。


说到这，开头我们提到的第一个场景问题，如果用 Google Docs 来协作的话，效率就会大大提高。项目负责人可以在 Docs 上创建一个文档，大家就能随时在线修改自己负责的部分。而且系统会实时保存，项目负责人也能看到谁在什么时候作了什么修改。最有意思的是，当你能实时看到其他成员正在编辑、修改时，虽然你和你的印度同事、澳洲同事远隔万里，但这时你就会感觉他们就像坐在你身边一样，这就是 Docs 的美妙之处。


**第二个亮点是，能和 Office 软件无缝对接，能够导入、导出 Office 文档，没有任何不兼容性。**


比如，如果你打算周末去郊区徒步，山里也没有网络，但你又希望晚上的时候再看看项目的文档，怎么办？ 不用担心，下班的时候把你的在线文档下载保存到电脑或者 U 盘上，直接用 Office 软件打开就可以了。等周一到公司，再把你修改过的内容上传到 Google Docs 上就可以了，非常方便。


**第三个亮点是，云盘储存。**Google Docs 文件不是像 Office 一样保存在本地，而是保存在 Google Drive 中，也就是 Google 的在线云盘 ；只要有网络，你可以在任何一台电脑或手机上，登录后就能进行编辑 Docs 。


除此之外，Docs 还有个神奇的功能，自动备份。你可以想想看，这一点是不是很有用呢？假如你的笔记本电脑丢了，Docs 上的文档却没事，因为它们都存在网上，而不是在你的电脑上。


**最后一个亮点是，非常方便分享。**因为每个文档都有一个专属的链接。在编写邮件时，直接放上这个链接就可以跟他人分享。如果邮件发送后，发现这个文档还有一些要修改的，你还可以继续修改文档，丝毫不受影响。这也就解决了第二个场景中的问题。


除此以外，不得不提的一个优点是：Google Docs 是完全免费的。你只需要注册个 Google 的账号，有网络就可以开始用了。


**那么，如何使用 Docs 进行高效工作呢？**


在这里，我给你举一个我自己的例子吧。在汶川地震以后，我们文化委员会开始在全球范围进行募集资金，这个项目中我们团队成员分别来自不同的团队，每个人白天都有自己的本职工作，没那么多时间开碰头会，也不容易凑大家都合适的时间，偶尔开会也都是在夜里，所以 Docs 就满足了大家同步信息和沟通的需求。当时，我们用 Docs 完成了以下几个工作：


第一，Docs 存放了我们项目重要的文档。我们新建了一个「汶川地震资金募集」的文件夹，跟这个项目相关的所有文件，都可以及时更新到这个文件夹里面，以便其他人及时读取。这样一来，所有报告都放在了一个地方，我们可以随时随地访问相关内容，根本不需要发邮件给项目相关的十几个人来要文件。


第二，我们用 Docs 中的 Excel 表格，创建了一个公开的待办事宜清单，也就是我们常说的 to do list 。上面列出了每个人的具体任务、时间点。 每个人都可以随时往清单中添加、更新任务，同时查看大家的工作进展。对于部分公共信息，如果谁觉得其他人写的有误，可以自行更正，这些修改都是有记录的，别人会看到什么时候，谁修改了什么。


第三，我们用 Docs 中的问卷来进行数据的收集。在募集捐款的时候，我们会通过各种场合，给全球的员工发送捐款链接，他们只要填写了相关信息，在 Docs 的后台，就会自动收集到捐款人的姓名、部门、国家、捐款金额等信息，我们每天给员工的集资汇报用的就是 Docs 自动收集上来的信息，因为提前已经设置好了模式，你连报告格式都不用整理，节省了很多时间。等募集结束的时候，我们用 Docs 自带的功能做了数据的视觉呈现，哪个国家捐了多少，谁捐的最多等信息一目了然。


Google 在新员工培训的时候，第一天就会教大家用 Google Docs ，很多人反映，一旦习惯这种工作方式，都不愿意回到常规的方式了。最后八卦一下，整个知乎也在用 Google Docs，好不好用，你在知乎上问问他们就知道了。


**本节小结：**


本节我为你讲述了 Google 高效协同工作的秘密武器 —— Docs。它能实现在线、实时进行 Word、Excel、PPT 的编辑，让你和你的团队协同效率更高，同时发邮件时，再不用担心忘记加附件了，只要登录，看到的永远都是最新的文件。
