## 简介
本专栏由猿某人为大家带来空调选购、安装和使用的详细指南，帮助大家选到最满意的空调，避免空调使用中的种种误区，收获舒适室内环境。


空调是现代家庭必备的大家电之一，一个家庭通常可能都不止会安装一台。然而，市面上的空调品牌繁多，型号庞杂，大多数人难以看懂，面对复杂的参数指标一头雾水，对机器内部更是不够了解，经常是只能按照销量和价格排名去选择，结果买到的都是低配高价的产品。


 本专栏由知乎空调类问题专业答主猿某人出品，从空调的结构，原理，类型出发，结合 200 多台次的拆机和多台空调上墙实测经验，让大家可以分辨真正值得关注的指标和功能，解决选购和使用空调的常见问题，打消关于购买与安装的各种顾虑。


同时，专栏内也给出了目前市面上不错的各种类型空调，各品牌热门型号点评，让想要购买空调的用户们可以按方抓药，挑选到最满意的空调。
