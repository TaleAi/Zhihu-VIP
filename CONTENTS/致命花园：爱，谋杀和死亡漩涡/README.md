## 简介
有了富贵逼人的房子，就能拥有与之对应的美好人生吗？最奢华的小区里，连续发生了最耸人听闻的谋杀事件。房价会应声跌落，可罪恶，从不轻易消失……


永安花园曾经是城中最负盛名的小区，这里有最漂亮的别墅、最齐全的设施、最茂盛的绿化，当然，还有最富贵逼人的居民。


可就在这个昂贵富有的小区里，却接连发生了数起耸人听闻的死亡事件。


纵情声色的富家太太、闻名全国的奥数名师、衣锦还乡的名校博士、穷困潦倒的厨房小工……全都卷入到了这场死亡漩涡之中。


永安花园也从一个富丽堂皇的小区，一路跌到门庭冷落、无人接盘，逐渐变成人员混乱、肮脏不堪的群租房，最后沦落成为了廉价的日租公寓，可这场死亡漩涡却从未停止转动……
