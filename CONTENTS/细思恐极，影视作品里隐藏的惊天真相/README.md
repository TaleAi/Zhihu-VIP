## 简介
小时候常看的动画片，其实都是披着童真外衣的科幻惊悚大片，而那些经典影视作品，也都隐藏着大量未解之谜。


大头儿子和小头爸爸长得一点也不像，难道围裙妈妈出轨了？是的，我有证据。


《喜羊羊与灰太狼》隐藏着一个非常大的终极阴谋，这还要从青青草原上那群诡异的生物讲起。


《哆啦 A 梦》里的世修，你还记得吗？虽说他是大雄的玄孙，可他肩负着一个未完成的使命......


《蜡笔小新》中小新不小心点火引爆了屋里泄露的瓦斯，导致房子爆炸，小新一家人无奈，最后搬家了，你是不是也有这样的疑问：整个房子都炸了，位于房子中心的小新一家人一点伤都没有？


这些经典，小时候，你看得津津有味，可长大后，再翻看，细思极恐，让人心里打颤。
