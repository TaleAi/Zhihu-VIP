* 上一秒刚发工资，下一秒就被某呗「洗劫一空」；
* 除了满足自己的基本生活需求外，不敢再有其他「烧钱」的爱好；
* 羡慕自己周围的同事买车买房，然而自己却凑不出首付；
* ……

宁波银行的蛋黄超人将针对年轻人量身定做理财指南。本专栏内容基本涵盖市面上的所有理财方式，教你利用手机即可实现简单理财；深入浅出，结合理论与案例，教你运用简单理财技巧告别「月光」。
