## 简介
恋人抬头望，一轮纸月亮。


爱，不是上位的工具，不是站上制高点的正义，不是恃骄行凶疯狂打脸的依凭。


爱情是我们不再提起的词汇，它深沉痛苦又充满挣扎。


这个专栏，专注讲述那些冲突之内的爱情故事。作者用其独特的成熟细腻的笔法开始讲述那些非对等关系里的挣扎与绝望，他们是如何放弃一段感情又如何选择自己的人生，通过动人的细节描写逐步向读者揭开那些不够正义的三观背后的故事——


她们是人，我们是人，我们不是最好的，但我们至少可以是诚恳的，坚定的，怀抱着痛苦的爱走在人生的道路上。
