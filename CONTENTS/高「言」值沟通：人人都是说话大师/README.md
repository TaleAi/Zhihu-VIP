## 简介
不管是商务谈判还是私人聊天，情感交流或者熟人闲聊，面对面交流亦或微信沟通，都有很多技巧，告别无技巧，从心学会沟通。


**交流沟通是人的本能，但是你会聊天吗？**


有些人一张嘴就脸红，说话结结巴巴；有些人只会说「天气不错，吃饭了吗」老三句；


还有些人，说话滔滔不绝，听众呵欠不断。


如果你是想要恋爱的男生，好容易遇到心仪女孩，开口就把别人吓跑；


如果你是一个职场人士，一张口就怼天、怼地、怼空气，把人得罪光了，还以为自己满腔正气；


**这些都是因为说话聊天的技巧太差。**


所以，无论是商务谈判还是私人聊天，无论是情感交流或者熟人闲聊，无论是面对面交流亦或微信沟通，**都有很多「言值」技巧，赶紧放下那些老套的话术，学一学真正行之有效的新方法吧。**
