## 简介
十大「禁地」，众说纷纭，被诅咒和怪谈萦绕至今。
刚刚参加工作不久的「我」，跟采访队走南闯北，亲身经历了「十大禁地」中的种种诡异现象。人鬼同居的封门村、飞禽不敢穿越的罗布泊、鄱阳湖的「魔鬼洲」，每一个地方，都有一段不为人知的诡异往事。


「进了封门村，如果有陌生人叼着烟找你借火，说什么也别应，他们是在借阳寿！」封门村被誉为「中国第一鬼村」，在这个小村落中发生了许多灵异事件。「我」不信邪，决定跟随采访队一起探秘；


位于中国首都的北新桥海眼，据说是一个「锁龙井」，井下的铁链，困着一条千年老龙，据说谁要是动了这个泉眼，整个城市都会被大水淹没……


「禁地」二字，诡秘奇谲，缭绕在传说的雾气中，令人摸不着头脑。真假虚实，跟随本专栏，一探方可知究竟。
