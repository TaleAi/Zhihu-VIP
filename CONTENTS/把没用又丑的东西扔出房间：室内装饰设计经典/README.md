## 简介
本专栏是「美国室内装饰第一人」艾尔西•德•沃尔夫的经典之作，被后来者奉为室内装饰设计的宝典。她对房屋内各个角落、各种家具装修实例都有很多经验之谈，对墙面装饰、色彩应用、各类房间布局、各种细节设置等作了具体阐释。她的装饰思想影响了好几代美国室内装饰设计师，直至今天，仍然有不少的室内设计师会从艾尔西那里得到创作的灵感。


本专栏是「美国室内装饰第一人」艾尔西•德•沃尔夫的经典之作，至今都被装饰设计师奉为后维多利亚时期室内装饰设计之宝典。


**「请把那些没用而又丑陋的东西扔出你的房间」**


作者艾尔西•德•沃尔夫用这句话，教导着美国中上阶层家庭在装饰中如何避免毫无意义的奢侈浪费，并彰显低调、倡导简洁，对陈腐观念嗤之以鼻的设计原则，为许多美国家庭装饰注入了精致却精简的新鲜血液。


这个专栏，从家居装饰的发展史及装饰家居的总体思路出发，并以具体的装饰案例向你介绍「后维多利亚时期」室内装饰：


* 现代家居发展史
* 家居装饰的思维方式
* 世界著名室内装饰案例
* 会客厅、起居室、卧室等空间的装饰布置原则
* 家居装饰中关于色彩、布局、比例、光线、通风和舒适度等问题
