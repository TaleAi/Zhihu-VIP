## 简介
三位心理学专家解读，50 节心理沟通课，带你走近青春期孩子的内心世界；聚焦痛点问题，通过典型案例分析，为孩子和家长提供专业的解决方案。


青春期是每个孩子人生中最严峻的考验，一不留神可能会出现心理风暴。


面对青春期的孩子，你是否困惑：


为什么孩子总是一言不合就翻脸？  

为什么孩子不愿意沟通，甚至容易产生极端行为？  

为什么孩子爱说「不要你管」？


那么，我们该怎么做？


本专栏将由三位心理学专家为你解读青春期常见问题，带你走进青春期孩子的内心世界；通过典型案例分析，为孩子与家长提供专业的解决方案；教你化叛逆为动力，陪孩子共度青春。
