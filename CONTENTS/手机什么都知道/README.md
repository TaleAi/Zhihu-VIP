## 简介
你有多干净？肯定的回答？如果允许别人随意查看你的手机，你的态度还会这么坚定吗？


无论你贫穷还是富有，生活在如今这个社会，你一定都会有一部手机。


聊天记录、转账记录、订单记录、行车记录……在不知不觉的过程中，手机拥有了你的全部行为资料，它就像一个以你为目标而做出来的巫毒娃娃，不需要提前经过你的同意，手机就知晓了你的一切——而且不承诺会为你保密。


在本专栏中，你可以看到围绕「手机」发生的故事，每一桩看似隐秘的勾当，都被手机记录了下来：


顾客在深夜订购了一份外卖，却在菜品中发现了一颗人类牙齿，顾客、骑手、商家陷入了摆脱责任的拉锯战，最终出乎他们预料的是，这份外卖竟然与一条人命有关……


买家网购了一块移动硬盘，却被客服要求立刻寄回，买家出于好奇查看了硬盘中的内容，竟然发现了一段拍摄于命案现场的视频……


乘客叫了网约车前往机场，准备登机的时候却发现订单没有结束，此时司机正沿着原路返回，目的地就是乘客的家。乘客立刻放弃登机，因为在他刚刚离开的家里，藏着一个绝对不能被发现的秘密……


如同《完美陌生人》的僵局一般，手机成为了击碎我们面具的突破口，每一段扣人心弦的故事也将由此展开。


阅后请关机，手机什么都知道。
