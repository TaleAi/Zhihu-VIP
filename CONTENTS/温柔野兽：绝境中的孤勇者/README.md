## 简介
对抗命运的女性故事。


她们残酷而温柔，正直且隐忍，幸福却扭曲。


她们曾独自面对过世界的冷漠、不公与屈辱。


所以，她们拿起了手中的武器，对着黑暗发出怒吼，向着命运发起反击。


  



她们，从未屈服。
