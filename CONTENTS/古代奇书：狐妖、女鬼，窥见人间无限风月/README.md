## 简介
狐妖、女鬼，历来是中国知识分子寄托无限幻想的符号。从《聊斋志异》到《阅微草堂笔记》，它们仿佛暗夜中的一只眼，窥探人间无数传奇。本专栏借由经典故事源本，以「反套路」方式新编，为你重叙一个个精彩的鬼话妖谭。


兰溪水鬼：兰溪镇的人以打鱼为生，但他们却从来不去最近的兰溪捕鱼。因为，传说兰溪有水鬼，会将人溺死当自己的替身。赌局输了的许言被迫前往兰溪一探水鬼之谜，却不慎掉入水中，他看见无数「沉尸」站立在水底。一阵窒息后，一个「沉尸」向他游来……由此，他揭开了一个关于自己生生世世的秘密。


真假阿绣：我是翠屏山的一只狐妖，从小样样都争不过一母同胞的妹妹阿绣。妹妹为练功走火入魔死去，我开始不断做关于她的噩梦。不堪忍受的我逃离翠屏山，却在盖州看见了另一个阿绣。阿绣和我之间，到底发生了什么？


古庙夜谭：乱世末年，京师中传言叛军即将围城，几个官宦子弟偷偷逃了出去。他们夜宿古庙，聊起了两年前的京师大疫。一个公子讲了那次瘟疫中，遇到一个诡异的老人吃人肉的故事。第二个则说了有人偷挖病死的人身上的珠宝，结果第二天被发现死在自家床上。第三个人则说了自己看见过一个人不断吐血肉的离奇见闻……聊着聊着，他们发现各自的故事，似乎都指向一个看不见的恐怖真相。而此时庙外却传来一阵脚步声……


……


「反套路」鬼话妖谭，集风月与离奇于一身，带你窥见千年来中国人内心那不甘而狂野的 B 面。 


而多少不能被圣贤书记录的人间痴缠、欲望纠葛，


都在这些鬼狐妖怪的故事中，一一诉说。
