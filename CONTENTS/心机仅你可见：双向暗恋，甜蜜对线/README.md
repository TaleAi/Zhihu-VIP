双病娇、男弱女强......爱情里的各路反常规较量，在这里上演。


这是一个现代言情故事合集，有不走寻常路的青梅竹马小甜文，也有反转再反转的双病娇悬疑文······你永远猜不透，甜蜜爱情里刚好遇见的那个人，究竟是浪漫邂逅，还是阴谋的开始。


**特别恋爱**


身患重病的我，只剩半年生命，为了离医院近一点，我便宜租下了一个郊区的房子，没想到诡异的事情发生了。


这个房子，闹鬼！而且我还能看见房子里的这只帅鬼！


但是毕竟要死了，我也不带怕的，甚至有点好奇，这只鬼是什么人？他怎么知道我的名字？他为什么会出现在这里？


**撒谎金鱼**


凌晨一点，我发现，有个人每天半夜三四点反复试我家的密码锁。


这说明，在我熟睡之际，他就蹲守在距我一门之隔的地方，只为无声无息地闯入，也许要杀了我，也许要毁灭我。


此时的我寒毛卓竖，如坠冰窖，像是被恶鬼扼住了咽喉，在这个空间里一秒钟都待不下去。


收拾衣服，推门想跑，迎面对上一个男人。


——原来他，早已在此等候多时。
