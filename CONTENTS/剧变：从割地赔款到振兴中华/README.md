## 简介
身在中华，与有荣焉。


1895 年，实力位于世界前列的中国北洋舰队惨败于日军之手，清廷无奈签署丧权辱国的《马关条约》。自此，中国开始了长达半个世纪的「恐日症」。直到 1945 年抗战胜利，中国人才终于不再生活在日本侵略的阴霾中。


1949 年，新中国成立。不久后，中国人民志愿军即开赴朝鲜，铁血将士以意志为枪，打出了新中国的铮铮铁骨。


1978 年，改革开放的「春风」吹进千万家，复兴之路渐行渐顺，封闭、贫穷、落后的中国不再，开放、富强、文明的中国正在谱写世界舞台上的新篇章。


19 世纪末至 20 世纪末，中国遭受了千年未有之重创，也完成了举世瞩目的崛起，这是近千年来最值得书写的时代，也是离我们最近的历史。


在本专栏中，拉费耳伯爵将为你讲述这一历史时期中，中国所遇到的前所未有之变局。
