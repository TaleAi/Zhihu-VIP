## 简介
邂逅一阙好词就像走进了一场真实的穿越，曾经颠簸的历史和心史都留在了词人独特的感受里，是自我的消解、是故国的思念，是人生的感怀，是回忆的美好。


「人生若只如初见，何事秋风悲画扇」，他仍是旷世名主，她可以继续做她的绝代佳人；


「大江东去，浪淘尽，千古风流人物」，苏轼那些曾经仰望的英雄梦随着江水一并东去了；


「红酥手。黄滕酒。满城春色宫墙柳」，陆游说不出口的还是变成了遗憾。


115 阙词让你重回古人的词情画卷，40 位词人带你倾听古人的浅歌低吟，你总能找到疗愈你的那一首。
