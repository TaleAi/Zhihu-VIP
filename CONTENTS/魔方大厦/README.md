**- 009 怪物清除计划 -**


我本该是个孤儿。


直到一天，我溺水消失的继兄，带着巨额财富回到家。


他领养了我。


表面上，我们是一对相处和谐的兄妹。


实际上，我隐瞒了一个秘密：他曾经的死，是因为我。


然而我逐渐发现，他也隐瞒着无数秘密:


他的地下室里，藏着我从小到大的照片。


我的父亲及无数人类，都是被他所害。


……


原来，他是个非人的怪物。


而许多年前，我与这个怪物，定下了一个契约。
