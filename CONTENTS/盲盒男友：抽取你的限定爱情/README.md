**-  钓系猫咪男友：重生也不嫁你 -**


殷乾不懂我为什么退婚，就像我也不懂他前世怎么能为了稳固权势，狠心将我们亲生女儿送给他的宠妃虐杀。


重生一回，我只想保全家族，殷乾却依旧步步紧逼。


我恨之入骨，转头与他的劲敌殷九河联盟，签订契约，做他侧妃。


却没想到一份假婚约，得到份真感情。


  



**- 高岭之花男友：今天和尚撩到了吗 -**


世人都说我是妖女，我遇见和尚莲生那天刚屠完山。


他面色悲悯，薄唇一张一合：「姑娘，回头是岸。」


我不太理解这句话，甩甩手上的血，不爽道：「秃子，找死啊。」


后来，他教会了我许多，可惜我没机会说出想说的话。
