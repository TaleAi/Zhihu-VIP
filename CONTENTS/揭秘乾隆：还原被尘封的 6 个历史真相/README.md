## 简介
开创客观全面解读历史事件的新角度，为你揭秘乾隆时期最关键的历史事件，让你系统了解清代国家政治运转和经济运作的规则。


影视剧《延禧攻略》和《如懿传》的热播，一次次将乾隆帝的后宫韵事搬上舞台。然而，真实的乾隆时代是如何的呢？到底是「盛世」还是「衰世」？涉及这样的历史评说，常常让人哑然。


乾隆帝和美国第一任总统同卒于 1799 年，前者是一位封建帝王，而后者已然是一个现代化国家的开创者。该如何鉴定世界史中那些看似时光错位的世事变化？


老生常谈的「文字狱」、乾隆盖章和御题的背后，又有什么样的力量角逐？


韩非子言：法莫如显，而术不欲见。


为了给大家解读乾隆时代一些典型事件背后的动机、力量的博弈以及帝王心术，我们邀请了中国人民大学清史研究所的副所长，刘文鹏教授，从专业的角度，以几个重要历史事件为例，给大家揭示一个真实的乾隆时代。
