## 简介
《职场沉浮记》是半虚构短篇小说合集，取材自某几家创业互联网公司的真实事件，略作剧情化改编。讲述四个年轻人走出校园，步入社会，在有黑有白的职场中经历考验、迷茫、逐渐找到目标和做事原则的故事。

**无数真实的职场问题，留给职场新人做选择。**


**吴霏霏、刘润芳 Sophie、王乌龟、王志远。四个普通却不平凡的年轻人，在真潇洒互联网公司，开始了一系列抉择。四个年轻人逐渐在职场中成长，建立新的规则。**  




吴霏霏：多年海外留学归国的独立女性，性格直爽，风趣幽默。一毕业就入职了真潇洒直播互联网公司，成为一个明星直播运营。她创意满分，做事又快又好，但是过于「对事不对人」，经常得罪心胸狭窄的上司周敏。一次因为 C 姓男星合作事件，他们的矛盾激化了，吴霏霏愤然离职。


当吴霏霏来到新公司，却发现了另一个周敏，甚至是周敏 plus，这次她的选择是……


  




刘润芳（Sophie）：就像电影里那种为梦想奋斗的女孩一样，农村出身的她，靠自己的努力读了名牌大学和研究生，内心却依然保留着一份互联网女工的单纯。她是真潇洒直播公司的一个直播运营，吴霏霏的好友和搭档，周敏的得力干将。刘润芳很少出错，活好话少，责任心强，做出了很多在公司为之瞩目的 Case。有一天，刘润芳突然发现自己一切的努力都不足以让自己升职加薪。


当她开始浏览新的机会，她才发现，原来世界是这样……


  




王乌龟：互联网公司的营销策划，一个不会说拒绝的人，看起来人很好，但是其实很压抑，经常默默在家里「诅咒」不靠谱的同事。有一天，他忍无可忍在公司掀了桌子，没想到竟然因为这个意外的行为获得了公司的重视，升职加薪。


他当上领导之后，一直留意是否身边会出现第二个「王乌龟」，关照曾今的自己，建立了新的职场规则……


  




志远：冷静低调的高材生，在真潇洒互联网公司做产品经理，对于公司的投融资计划和现金流水非常清晰。他一直观察周围的同事，在业务层面有一套自己方法论，有时候会用语言惊艳上司。


在他兢兢业业为公司卖命 3 年之际，却意外被公司举报关进了拘留所……他终于他看清了公司的面目，很快离职，并辗转于其他知名公司和创业公司之间……
