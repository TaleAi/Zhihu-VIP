## 简介
文明是因高低而冲突？还是持平等而互鉴？堪称当今的时代之问、世界之问。文扬老师将从文明史观的角度和大家聊聊，真实历史上人类社会的领地扩张、经济增长、科技发展、战争与和平。


随着中国重新成为世界大国，并再次回到世界历史的中心，关于中华文明的问题，也开始像中国经济问题一样，引起了世界广泛关注，很多人开始用文明来解释他们所谓的「中国问题」。


但在西方中心论的立场下，中华文明的唯一性和独特性没有得到充分诠释。大多数人对于中华文明的了解，除了表面上的概念和人云亦云的说法之外并不能认识更多。


文明到底是一个什么故事？中华文明与其他文明的主要区别在哪里？为什么会产生这种区别？


文扬老师将从文明史观出发，为大家解读以下问题：


1. 中华文明是原生文明之一，为什么大约同时期的原生文明没有延续下来？


2. 这些文明覆灭的原因是什么？


3. 为什么中华文明没有毁于蛮族入侵？蛮族入侵中国后发生了什么？


4. 当今世界的各大文明如何划分？


5. 这些文明又是如何产生出来的？为什么只有中华文明一直延续至今？延续至今的中华文明将如何实现复兴？
