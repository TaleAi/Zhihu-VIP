## 简介
她本是大将军独女，为一个不受宠的皇子一句「喜欢」，便倾尽一生，眼睁睁看着自己的父亲被杀，家族被灭；她本有经世之才，从不掩饰自己的野心和张扬，却将对他的爱掩饰得极好，封入心间，严丝合缝……她们多有不甘，却也敢爱敢恨，天下男子多权情难两顾，此情应付余长恨，此生不过是霜雪满头，若有来生，相扶白首。


《心间砂》


她本是沐大将军独女，因他一句喜欢，助他登上皇位，却又被当成影子藏在这深宫许多年，再次见面，她静静躺在棺椁，纵使面色苍白如纸，亦不减丽色，眉心朱砂赤红，仿若一颗玲珑的相思豆，那是初见时，他亲手为她点上。


《江山为嫁》


她是巨富商贾的女儿，求学时遇上郑国侯不受宠的小儿子。虽然他眼盲，但他太过好看，她一眼就喜欢上他。书院不收身体残疾的学子，她就让他扮作她书童，可以旁听。两人相处期间，她顽劣捣蛋欺负他，他对她总是冷冰的，却对另一个小姑娘极尽温柔，还送她定情信物，可那又怎样，最终，她还是成了他唯一的皇后，只是此生不过是霜雪满头，从未有过机会，相扶白首。


《卿卿为君》


那个叫沐卿卿的女子已经死了十年，他却还是会想要为她而大开杀戒。他以为自己不喜欢她的，她那么有野心那么张扬，简直令他恐慌……


却也羡慕。


母妃弥留时为他向父皇求了亲，他说服自己成亲是为孝道。他对自己说，他从不喜欢那成天嚷嚷着想要做帝君的女孩子。


可她真的是比他更适合做帝君，她那么无情那么狠心，对别人对自己都是。她居然从未泄露过一星半点的心意。又或者泄露过的，他却因为害怕所以视而不见。


《渊王妃》


她是南月国的小公主，集万千宠爱于一身，却逃不过和亲的命运，命运也罢，她还是爱上了自己夫君，渊王周世尧，直到有一天，他从外面带回来一个受伤的姑娘……
