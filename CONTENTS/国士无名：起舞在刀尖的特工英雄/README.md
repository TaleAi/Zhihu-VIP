## 简介
信仰无敌，在最黑暗的地方等待黎明。


他们大部分没有留下名字，只有一个代号。


他们可能是一个人或者是一群人，在常人无法想象的刀光剑影、血雨腥风之中，传输情报、策反、窃听、破译……


 他们是历史上诸多重大事件的参与者和见证人，也是在重要节点扭转局面的大功臣，日军和蒋介石严防死守，没有挡住他们的的大智大勇。


这些隐蔽战线的先辈们，他们在暗无天日中提着信仰的萤火之灯，与狼共舞。


右手捏紧的铅笔，写下的是左右战局的只言片语；


左手握着的手枪，最后一颗子弹永远留给自己。


相比于驰骋疆场、壮怀激烈的将士们，他们默默无闻、备尝艰辛，在隐蔽战线上，孤独而无比坚定地书写自己的丰碑。


* 黑白通吃的「资本家」，如何在国民党眼皮子地下为延安输血？
* 蒋介石的金牌速记员，如何将最高机密送到毛泽东的手上？
* 潜伏在国军核心的「高级内线」，如何用一计让李仙洲 5 万大军全军覆没？
* 麻将桌上的阔太太，如何策反国民党海空军官？
* 400 反间谍小组，如何折服杀人如麻的日本宪兵队长？
* 曾救下蒋介石年轻军官，为何主动找共产党弃暗投明？

本专题将带你深入虎穴，拨开历史的烟尘，去探寻我党真实的地下工作者的传奇潜伏经历。
