## 简介
一首情诗，一场爱；十位文人，十段爱情故事。生命之上，诗情漫天。这是独属于中国文人的浪漫，也是我们每个人心底对「情」的期待与向往。


由古至今，情关最是难闯。无数文人雅士写诗词寄相思，情与诗是中国文人独有的浪漫。


**一首情诗，一场爱**


**十位文人，十段爱情故事**


从年少相知相爱，却未能走到相守一生，错过的感情终究追不回。古人以情写诗寄相思，每一个文人的深情背后，都有一段鲜活的故事与人生，写尽爱情的聚散悲欢。


**情圣白居易：** 一生与 33 个女人谈情说爱，失恋却长达 62 年，「他有多不正经，就有多深情。」


**人见人爱苏东坡：**豪放但专情，「一生妻子三个，个个都是真爱。」


**大女主李清照：**除了白富美的头衔，私下还是个赌徒和酒鬼，「容貌与才情皆出色，但做女神她从不认真。」
