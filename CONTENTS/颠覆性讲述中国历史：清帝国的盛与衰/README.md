## 简介
中国传统历史为什么不能像西方一样步入现代化？


大清帝国，近三百年历史，让人五味杂陈。


既有入关之初，强制汉人剃发易服的精神伤害和各种疯狂屠杀，也有积极融入汉文化的励精图治；


既在国家统一和领土层面有很大贡献，又因专制和闭关锁国而导致各种落后挨打、丧权辱国……


从极盛到极衰，有太多的问号与谜团等待解答：


顺治皇帝怎样推行汉化？雍正进行了哪些吏治和经济改革？


康熙和乾隆有哪些「文治武功」？为什么说「康乾盛世」是「瘸腿的盛世」？


为什么马嘎尔尼访华后欧洲人对中国的看法大变？没有鸦片就不会有鸦片战争吗？


太平天国、义和团运动是怎么闹起来的？中日甲午战争中有哪些不为人知的真相？


……
