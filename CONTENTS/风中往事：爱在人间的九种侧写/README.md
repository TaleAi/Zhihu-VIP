## 简介
人世间的诸多遗憾，最终化作被吹散在风中的往事。


这是一个关于女孩保护女孩的故事。


北方女孩宋小月转学到南方沿海小城，在备受欺凌的日子里遇见了坏学生赵霁云。


赵霁云在宋小月身上看到了曾经的自己，某种意义上说，她拯救宋小月也是拯救曾经的自己，为此她默默付出了牺牲，毁了自己接下来的人生。


经年以后重逢，她们身份调换，赵霁云成了需要被拯救的那个人，宋小月会怎么做？


——《毒草》


  



廖贱狗为艺术而生，却生在了落后农村，得到了傻子的名声。


父母双亡以后，去县城讨生活，遇到婉儿。两人心意相通，却一直没有挑明。


一天，廖贱狗画画的天赋被葛老发现，被带去上海。一场误会让廖贱狗和婉儿相爱却不再相见，又过了十多年，经历了诸多苦难，他们最后能不能如愿在一起？


——《画作》


  



李兰花得到了丈夫阵亡的消息，是由丈夫的战友陈清亮送来的。


从那以后，陈清亮住在了李兰花所在的村庄，暗自照顾起李兰花来。村里的闲言碎语，和对寡妇的欺凌，都被陈清亮一一还击化解。


时日渐久，陈清亮意识到自己心里对李兰花有了感情，这份感情让他感到羞耻和愤怒，对不起为了保护自己而丧命的李兰花丈夫。


为了逃避这份感情，陈清亮决定再次应召入伍。


——《阵亡》


  



我父亲死了，在我开人生第一次签售会那天。


丧葬期间，我回忆起与父亲生活中的点点滴滴，回忆起和他无法调和的矛盾，回忆起对他的埋怨……但每当这样想，脑海中总会跃起另一片回忆对抗这个说法。


到后来，所有的回忆都变作一句懊恼的话：「要是买小一码就好了。」


因为父亲车祸身亡那天，正穿着我送他的手工皮鞋，皮鞋偏大一码，我贪便宜没有换。被车撞后，父亲爬起身来，找撞飞到灌木丛的皮鞋，找了好久才找到，穿上脚后坐在路边，慢慢倒下了。


直到念遗嘱时，我才终于理解一个父亲对孩子笨拙的爱。


——《皮鞋》


自闭症农村男人广生在深圳做建筑工人。


一天在地铁站迷路，急得大哭，周围人拍照取笑。民警来后了解到，他将要迟到，假如今天日结的工资拿不到，在大学的妹妹将没有伙食费。


在帮助下，他终于抵达了工地，但他没有意识到，一场早有蓄谋的意外事故正在等着他。


——《高楼》


  



他们用力地爱、认真地活，却被生活弃之如草芥。


于是那些曾经的爱而不得，也最终化作风中的往事，散去了……
