## 简介
卖假药、逃婚、入狱......社会底层人物手记。


社会底层的生存现状是怎样的？


彝族女孩，初中选择去广东打工，她会遇到怎样的爱情？


城市角落，那个女人在等待什么样的男人「宠幸」？


出狱后的四叔，如何在三年内积累 7 位数资产？却又仓皇而逃......


你可能不曾经历这样的人生，但却能在这里看见人间。


你也许无法体会他们的处境，却能感受小人物的颠沛流离。


每一种命运都是面对世界的妥协与抗争；


每一个个体都在世间寻找属于自己的舞台和角落。


一个人物，一种人生，看小人物如何颠覆命运活出不一样的人生。
