## 简介
恋爱一时爽，分手「火葬场」。


恋爱时，Ta 是人间的四月天，十里春风不如 Ta；


分手时：


「原来他是个渣男啊。」


「原来他不是潜力股，就是普通民工。」


「原来谈了这么多年，他是故意不结婚。」


……


是的，爱情的本质，往往都是在失恋悲痛时恍然大悟的。


与其这样，不如提前看清。
