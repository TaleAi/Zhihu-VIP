大自然中有各种鸟儿，你有没有想过这些问题：


五脏俱全的小麻雀，为什么飞不高？


小鸡如果生气，战斗力比恐龙还大吗？


传说中，猫头鹰也会送信，是真的假的？


......


其实，鸟类作为恐龙的后裔，拥有很多奇特有趣的秘密。了解鸟类飞行的奥秘，发现隐藏在他们炫彩羽毛与美妙歌喉背后的故事，和大自然中的鸟儿做朋友。


本专栏和青少年对话，唤起大家的好奇心，将野外得到的自然享受和快乐分享给更多青少年。让更多的人懂得尊重自然的道理所在，学会欣赏自然，自觉地爱护我们的地球，欣赏鸟类世界的魅力。
