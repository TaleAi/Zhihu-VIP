兄友弟恭：沉没的人生


父亲猝然离世，给家里带来了五十万。靠着这笔钱，哥哥陆友如愿开起了餐饮店。我本以为家里的日子还能支撑，然而陆友逐渐沉迷赌博，卖了店面，抵押了房，甚至借了高利贷。陆友沉迷赌博，最后竟跑到赌场偷筹码，被看守赌场的徐兵发现后，让我把陆友带回去。怎料当我到徐兵家里时，徐兵正要掐死陆友，情急之下，我本能的捅死了徐兵，背上了命案。为了不被黑道报复，我只好以徐兵的身份逃亡外地，假装徐兵还活着……
