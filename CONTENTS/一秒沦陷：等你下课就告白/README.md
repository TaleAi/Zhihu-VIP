## 简介
跨越山海，为你而来


《椒吻》


有个喜欢辣椒学的男友是啥体验？


我只是偷偷欣赏了一下其他男生。


他就让我吃辣串串。


还恶狠狠地对我说：「没吃完，不准走。」


我含泪吃下，到处找水。


他说找水没用。


「这样才管用。」


瞬间，冰凉的唇覆上我的唇。


甜甜辣辣，像是我们的青春


从高中到大学，一路有你。


《打脸恶毒姐姐》


车祸后我拥有了读心术，才知道身边对我恶意最大的人，竟是姐姐。


姐姐自诩才女，一直将我视作草包。


她爱慕著名作家沈郁之，把我误认为情敌。


而沈郁之，其实是我的笔名。


她反复打压挑事，想让我出丑，反而给了我秀才华的机会。


我一次次证明实力，她都拒绝认清现实，还屡次坑害我。


最终害人终害己，她受到了法律制裁。
