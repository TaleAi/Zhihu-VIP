 《理性男的失控婚姻》


在餐厅去了一趟厕所再出来，我居然误入了平行空间。


那个时空的我跟现在的个性和人生简直南辕北辙。


平行空间中，我是个人气时尚主播，拉开衣柜清一色的都是吊带露背小短裙。


更诡异的是，我居然还跟自己暗恋许久的校草结婚了，还有了个娃。


但看着对我冷冰冰的丈夫、对我小心翼翼地奶娃我有点头痛，我怎么感觉，自己在这里混得好像也不太好……


 


《冷都男的爱情陷阱》


平静的婚姻突然被一份亲子鉴定打破了平静，我的女儿居然不是我丈夫许高远的亲生女儿，而我，被迫离婚净身出户。


搬家当天，我遇到了一个自称是我女儿亲生父亲的男人。


我没想到才从一段婚姻逃离的我，转身却掉入了顾修为我准备的爱情陷阱之中……


 


《闷骚男的离婚手札》


在我老公第十次和我提出离婚的时候，他死了。


事后在整理遗物的时候发现了他没有送给我的 48 封情书。


明明爱我为什么还要执意和我离婚？


一觉醒来，我发现自己重回一年前。


这次我要搞懂陆延提出离婚的真相，挽救我们的婚姻！
