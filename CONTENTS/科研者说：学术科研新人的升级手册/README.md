## 简介
针对想从事科研工作的本科生、已就读的硕士博士研究生谈谈科研生活该怎么展开。


科研到底是做什么的呢？


相信初入大学，甚至准备考研的同学们都会心中暗想这一个问题。其实，再牛的科学家，也有着辛苦的研究起步阶段；再好的论文，也是积累在阅读、实验、撰写、打磨的过程后，从实验室中诞生出来的，科研的基础莫过于此。


遗憾的是，其中的艰辛和门道往往在科研一段时间之后才能知道，不少关键点需要在反复试错和煎熬后才能掌握，实验室的选择也充满了不确定性。更不要说科研本身有很大难度，读论文的艰辛、实验设计和进行的困苦都让很多人焦虑。


科研工作，并非像本科学习一样有明确的目标和确定的成果，失败反而是科研的常态。


「Don't Panic」，这个专栏就是为这些问题而来的！


本专栏将从如何阅读论文和选择选题入手，综合介绍科研的基础准备部分；之后，介绍如何和导师以及同门同辈相处交流，拓展到如何做好学术交流；接着，介绍本硕博等等不同科研阶段的的特点与发展，为大家的选择和目标提供一些基础；最后，我们会回到科研本身，介绍一下论文的写作和成果的呈现。


Keep calm and carry on！希望本专栏能给大家的科研之路一些帮助。
