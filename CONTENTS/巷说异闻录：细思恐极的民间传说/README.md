## 简介
你听过哪些至今难忘的恐怖民间传说？本专栏里，作者檀信介结合真实的时代背景，联系各地民间传说、风俗习惯，带来了 8 个荒诞诡谲的志怪故事。这些脑洞大开的奇闻异事背后，折射的是不同时代下的世间百态和人情冷暖，说的是异闻，聊的却是人性。


抗战时期，三个精通「秘术」的游方道士卷入一场离奇命案，多年后，命案真相浮出水面，秘术杀人的背后，竟然还掩藏着一场重庆与南京间的惊天金融谍战。


乱世间，一队逃荒难民误入一处世外桃源，却发现偌大的村庄里只有一对男女。难民们发现，这对男女只能形影不离，否则······而村子之所以成为世外桃源，正是因为背后这恐怖的真相。


······


檀信介带来的这 8 个细思恐极、结局难猜的异闻怪谈，情节曲折动人，集志怪、悬疑、推理等多种趣味阅读元素于一体，通过各种荒诞离奇的民俗怪谈，为我们再现了不同时代里的人间百态和人情冷暖，读来回味无穷。


这些故事里有对残酷战争、女性歧视的反思，也有对江湖骗局的揭露，留给我们的，除了故事本身的离奇和诡谲，更多的是对人性的思考和对人心的拷问。
