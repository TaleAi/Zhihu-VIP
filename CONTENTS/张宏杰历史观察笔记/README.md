## 简介
新一代史书必读经典，知名历史学者张宏杰作品精选。


  



**朱元璋：**


他建立的大明朝，是中国历史上昏君最多的一代，也是忠臣辈出的一代。


在历代皇帝中，他也是对贪污腐败最深恶痛绝的一个。


而大明王朝最终以中国历史上最腐败的王朝之一被列入史书。


这位中国历史上出身最贫贱的「乞丐皇帝」到底有着怎样的经历？造就了这个中国历史上最矛盾复杂的朝代？


  



**魏忠贤：**


在宫中飞黄腾达需要有三个条件，一是识文断字，二是富于心机，三是有强烈的野心。


而这三条，魏忠贤无一具备。


说到学识，没上过一天学，进宫多年，大字不识一个；


说到心机，人们对他的评价是憨，常被宫中奸猾的太监戏耍，得个「傻子」的外号。


这个大字不识一个的农民，为命运赌博而进入了深宫的十几年后，只钻营到了一个伙食管理员的位子；


而这个位子，阴差阳错，是在东宫太子身边……


  

**曾国藩：**


他详细记录自己一生的收入与支出，迷信风水与天命；


他一生遭遇五次重大挫折，终从「愤青」走向「老奸巨猾」；


他与左宗棠纠结一生的情谊，堪称男生版的「我的天才女友」；


他是当代成功学最爱的「圣人」，正直、清廉、智慧，将一个「平庸」之人可能达到的成就最大化；


他也是一个矛盾、挣扎的深嵌体制核心的职业官员。


透过曾国藩，你可以窥见中国传统官场，微妙而复杂的「潜规则」与中国特色体制的独特生存哲学……
