## 简介
但凡发生，可能不被发现，但不可能毫无痕迹。探寻「无声言证」，正是痕检师存在的意义。


作者刘神隐是一位痕迹检验工程师，从业 7 年，他是站在警察背后的男人，为警察提供痕迹检验专业领域的帮助，频繁出入各种极端现场，专门处理无解的疑难杂案。


别人一辈子遇见的离奇现场，没有他一年见得多。这个系列记录了他参与破获的真实案件，大多围绕现场勘察的铁律「触物留痕」展开。


接触的命案多了，刘神隐渐渐发现，痕迹和凶手之间还有一段路最幽暗最、漫长——通往人心之路。
