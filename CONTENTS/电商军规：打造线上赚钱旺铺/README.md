## 简介
本门课注重战略规划和实际运营技术相结合，归纳总结了上千家电商企业的经验与教训，为你呈现现象背后的本质；电商专家亲授店铺年销量过亿的电商运营高招，助力你的整个电商运营生涯，使你实现从运营专员到运营总监的能力跃迁，使你的店铺成为赚钱旺铺。


千辛万苦开的店铺，却没有生机？


含辛茹苦达到品类前几，不知道如何维持？


做店铺就像是打战，对于电商人而言，方法即武器，正误即生死。


唯有不断在实战中纠错，给出立竿见影的攻坚策略，才能真正帮助电商人实现自我成长。


![](https://pic2.zhimg.com/v2-36bea286d72f0768da94bea27823edc3.jpg)


鬼脚七专注电商领域，已有十年之久。在阿里期间是淘宝和天猫的搜索技术和产品的负责人。十年间，经手上千家电商企业，听取了无数经验和教训。基于此，鬼脚七将它们进行归纳总结，透过现象抓住本质，成就本门课 25 条电商铁血军规。


贾真老师是一位杰出的电商讲师，也是一位优秀的电商企业家，有着丰富的实战经验。他白手起家，从一个淘宝小店开始，到如今天猫店铺年销售过亿。他从纷繁复杂的电商套路中发掘规律，奉献出本门课中的 56 条电商运营高招。


本门课将电商的「战略规划」和「运营技术」完美结合，更注重如何引发你的思考，帮助你建立独立的思考模式，而不是简简单单地操作复制；深度剖析电商的本质，即便时代更替，依旧适用。
