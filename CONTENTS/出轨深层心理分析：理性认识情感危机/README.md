## 简介
什么原因导致了出轨？背叛是欲望还是性格导致的？这节课带你认识出轨方的想法，帮你理性分析出轨的心理模型。


出轨是婚姻的最大敌人，很多人怎么也想不明白，昔日恩恩爱爱、和谐温馨的家庭生活，怎么突然就坠入深渊。感情土崩瓦解，信任分崩离析，TA 的背叛让自己的整个世界都变了样。


想问问 TA 为什么出轨，TA 却避而不谈；想让 TA 彻底忘掉第三者，最后却变成了强人所难；想再重拾往日的美好，却变成了尴尬和难堪；想修复裂痕，TA 却说回不到从前。


想要应对出轨后的问题，你必须了解清楚出轨的心理原因。理清出轨心理，也能防范于未然，将出轨的苗条及早扼杀，悬崖勒马，有时也是经营好自己婚姻的必要选择。


本课程由知名心理咨询师图南老师主讲，内容包括了解婚外情的本质、出轨后修复感情的办法、如何重建双方的信任以及挽回婚姻等方面。帮你认清出轨的本质，了解出轨的心理模型，从而对症下药，修复家庭裂痕或是防范婚姻雷区，让你经营出幸福美满的理想婚恋关系。
