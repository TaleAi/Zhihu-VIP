## 简介
在我们元昱国，女人比天大。我元九想娶个男人怎么了？我乐意的话，可以娶了整个宜春院的小公子。他裴筠煊是这辈子爱错了人。我怎可为了一个男人，放弃整个森林！


我叫元九，元昱国首富之女，元家下一任继承人，荆州城最有名的恶女。调戏良家妇男，打架斗殴，买下整个宜春院，只要能用钱解决的事，我都做过。


我认为，在我们女皇帝坐天下的元昱国，这很正常。女人挣钱养家，男人嘛，在家抄抄《男德》便好。


但是……但是裴筠煊怎么就这么不听话呢！


非要嫁到我家就算了，还逼我让我发誓这辈子只娶他一个。


他身为江湖赫赫有名的暗影阁阁主，明知我元九风流成性，不给我介绍手下的小公子也就罢了，还非要搞这种奇奇怪怪的一妻一夫制？


我呸，《男德》给我抄两千遍！我元九，接着欣赏我的小公子去了。


他怎么这么烦呢？


可他，怎么又这么甜呢……
