**回首顾佳期**


羊水破那天，我被救护车拉到医院。


接产男医生掰开我双腿：「一个人来的？」


我瞧了眼男医生那双跟我暗恋对象一模一样的眼睛：「等等！」


吓得连滚带爬从产床上翻下去：「其实我现在也不是很想生，我还能忍忍。」


……


**口嗨闺蜜的疯批哥哥**


闺蜜突然给我发了一个男人不露脸游泳的视频。


我没忍住口嗨了一下：


「细狗……大树挂小辣椒，嗯……把视频撤回去，别玷污我的眼睛。」


晚上，她的疯批哥哥不顾闺蜜的阻拦，踹开我家大门。


一把扛起瑟瑟发抖的我关进房间。


闺蜜在房间外拍门求情，他在屋内威胁我当他媳妇。
