## 简介
《虚构的犹太民族》轰动世界之后，备受推崇的历史学家施罗默·桑德回应犹太复国主义者的质疑，解密以色列国建立与扩张的真相。


何为「以色列地」？它既不是当代作为民族国家的以色列国，也不是历史上新月沃地出现的犹太王国。


它是不同历史时期的犹太复国主义不断累加、增添的历史概念，它的边界不断浮动，为当代以色列国扩张领土提供了历史合法性。


作为一个应由犹太人民统治的、变动着的领土空间，「以色列地」是如何被虚构出来的？为了回答这个问题，桑德从历史证据入手，详述自《旧约》时期的犹太民族流亡神话到现代以色列国的建立和扩张，解构了犹太人对以色列地的「历史权利」概念以及相关的民族主义叙事。


揭开这块土地神秘面纱的不是上帝之死，而是上帝的民族化，它把土地变成了新犹太民族能随心所欲地践踏和建设的一片土壤。
