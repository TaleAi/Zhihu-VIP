## 简介
姐弟恋、忘年交、替身之恋、复仇绝恋…… 专一纯爱与畸病之恋，有时仅有一线之隔。本专栏收集那些在爱恋中迷失自我，痴情已入膏肓的男男女女，有的人有幸在迷失中偶遇新生的希望，而有些则在执念里荒废了自己的一生。


《容我枯萎在时光中》


她大他六岁，收养弃猫一般，把他「捡」回了家。


八年，她像照顾弟弟一样看着他长大，


直到她宣布订婚，他再也按捺不住，横刀抢婚，终于娶了她，


她以为那是幸福的开始，等待她的却是他的变心和一纸绝症通知单……


  



《我拒绝了白富美校花的求婚》


开学不到一周，校花怎么就瞄准了平平无奇的我？


上下课路上故意和我「偶遇」，图书馆里和我旁边的人换位子，


操场上的摆满蜡烛向我告白，


拒绝之后，她竟然一病不起，入院治疗，


她的父亲竟然也找上门来，恳请我做她的男朋友？


这位美女，我究竟哪一点被你看上了？？
