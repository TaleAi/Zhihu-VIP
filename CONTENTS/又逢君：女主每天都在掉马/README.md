谈恋爱吗？顺便一起抢个皇位？


古言甜文钉子户「李富贵儿」个人专栏来啦！不论是娇弱貌美质子还是竹马少年将军，我全都要！谁能拒绝一个理智清醒的姐姐主动出击，带你拿下各色美男呢？


**每天都在救质子**


被折辱的质子正抬头看着我，同为质子，我劝他识时务者为俊杰，他却骂我是软骨头。


我的日常就是，救他，救他，还是救他！


而他却以为我是个对他见色起意的死断袖，我说我连那玩意都没有，他恍然大悟以为我被阉了。


可我是顶替了六弟的身份来当质子的公主啊。


**竹马将军逼我当皇帝**


储君之争愈烈，我只好诈死逃离。


谁让我是女扮男装的三皇子呢。


却被少年将军顾北斗当成替身给逮住了。


他看穿了三皇子的诈死，以武力相逼让我假扮三皇子，去争夺储君之位，把真正的三皇子给引出来。


我假扮我自己？


后来我才发现他从前暗中对我的那些情愫。


他要助我夺得皇位成为真正的君主，那我便倾尽全力应君一场！
