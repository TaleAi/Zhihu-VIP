## 简介
本课程由从事妇产科临床工作 35 年的范玲医生主讲，为你解惑怀孕期间，不规则出血、下腹隐隐作痛、孕酮低、宫外孕等诸多状况，为你创造安心孕期。


建档查白带回家后见红，怎么办？


腹痛+持续不断褐色分泌物+孕酮低，是怎么回事？

这些问题，北京妇产医院产科主任范玲都能为你解答。


课程中，范玲大夫将为你讲解孕期腹痛、不规则出血、粉红色分泌物、下腹隐隐作痛、孕酮低、HCG 持续不翻倍、急性腹痛、宫外孕等诸多孕期常见问题的出现原因和护理方法。


帮你安心度过孕期，生出健康宝宝。
