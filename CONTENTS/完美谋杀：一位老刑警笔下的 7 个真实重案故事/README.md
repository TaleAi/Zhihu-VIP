## 简介
现在的刑侦技术到底有多先进？刑侦人员如何攻破一场场看似天衣无缝的谋杀案？本专栏中，警察唐风（作者笔名）将为你讲述自己办案生涯中最难忘的 7 个谋杀案，真实呈现他如何用丰富的侦查经验、发达的现代刑侦技术，抽丝剥茧找到真相。唐风曾破获大案、重案数起，虽然如今已离开办案一线，从事刑事科学技术研究，可回想起来，令他至今难忘，现在想想依然毛骨悚然的，还是本专栏收录的这 7 个堪称完美的犯罪案件。


被害人在众目睽睽下离奇失踪、出租屋内莫名惊现无名尸体、下水道里意外发现人体组织，你以为这是哪部 TVB 刑侦剧的剧情，事实上，它们都是一位资深刑侦人员唐风经手过的真实犯罪案件。


这些堪称完美的犯罪案件，要么疑犯有无懈可击的不在场证明，要么犯罪现场是个天衣无缝的密室，要么凶手明明近在咫尺，却没有证据抓捕他。作者用刑警的经验和直觉、发达的现代刑侦技术，一点点找出真相，也得以窥到犯罪背后人性的阴暗面和复杂性。


没有编造的离奇，没有刻意的悬疑，本专栏所有内容都来自于作者真实的办案经历，可以说是一部真实版的《重案六组》。
