## 简介
家庭关系对孩子的成长具有很大影响。本门课将由婚姻家庭咨询师歆瑶为你讲诉家庭关系，让我们带着尊重、包容，察觉家庭对我们的影响，处理好家庭的关系，塑造孩子健康人格。


如果孩子独立性差，也许是因为我们与孩子的关系过于紧密，对孩子过度照顾，让孩子形成黏连的共生状态。


如果孩子时常陷入焦虑、愤怒或者厌倦的情绪，也许是因为缺少父亲的陪伴，让孩子承担更多不属于孩子的责任和压力。


当父母关系不稳定时，孩子觉得被忽视、不被认可，有可能用利用离家出走、网络游戏的方式来退出三角关系，保护自己。


每个人都渴望着被爱、被关注、被认可、被尊重。在孩子的成长过程中，需要的不仅是生理营养，更需要注意心理营养。


我们应该采取怎样的方式与孩子建立良好的关系？如何终止对孩子造成的无意识伤害？


本门课将由婚姻家庭咨询师歆瑶为你讲诉家庭关系，让我们带着尊重、包容，察觉家庭对我们的影响，更有智慧的经营好亲密关系，塑造孩子健全人格。
