## 简介
孩子有点熊、很执拗，不太自信，怕打针，甚至有强迫症、抑郁症等表现，家长该如何正确应对？请家长朋友们跟随本专栏一起，学习艺术治疗的理念与方法，认识孩子们在日常生活中常见的行为表现和心理情绪，帮助有心理障碍的孩子走出心理困境。


听到「艺术治疗」，你会想到什么？


你是不是在想，我家孩子就是有点熊，有些执拗，或者不太自信，又不是得了病，不需要「治疗」。


再或者，艺术，不就是用来欣赏的吗？它能治疗什么？


简单来说，艺术治疗，是一门运用了「视觉艺术」的心理介入方法，它与音乐治疗、舞动治疗等表达性艺术形式有所不同。


这里的「视觉艺术」，既包括相对常见的形式，比如绘画、雕塑、剪贴画，也包括工艺美术，比如木工、纸艺、编织、布艺，等等。


在艺术治疗师提供的安全空间里，来访者通过艺术媒介、创作过程，由此产生的艺术作品，对自我感受和想法进行探索，从而达到调和情感冲突，缓解焦虑情绪，培养自我意识，提升自尊、自信，发展社交技能等目的。


请家长朋友们跟随本专栏一起，学习艺术治疗的理念与方法，认识孩子们在日常生活中常见的行为表现和心理情绪，帮助有心理障碍的孩子走出心理困境。
