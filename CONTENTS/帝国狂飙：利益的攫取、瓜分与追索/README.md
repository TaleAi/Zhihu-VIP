## 简介
铁幕开合，狂人登场。


人类历史是被一桩桩传奇色彩浓厚、戏剧冲突强烈的重大事件所影响和推动的，在这些争端背后，隐藏着不同阵营的国家之间惊心动魄的博弈和斗争——


* 被印度人视为民族英雄的甘地，为何在印度独立后，遭到暗杀？
* 狂人卡扎菲悍然制造洛克比空难，而后拒不承认，此举遭到国际社会的围剿——是什么给了卡扎菲这样的底气？
* 1943 年，新四军一支 600 人的军团叛变，投靠日伪。与此同时，一场我方精心策划的情报战悄悄开启。执行者，正是这支潜入日伪的军团。
