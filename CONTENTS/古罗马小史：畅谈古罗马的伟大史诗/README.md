## 简介
本课程由《中国通史大师课》出品人谭瑞岗主讲，在专业的历史事实基础上，用精心设计的场景和音效，为你生动呈现古罗马一幕幕的史诗级场面。同时，课程绝不仅仅满足于听个故事，主讲人会将故事背后的发展逻辑为你娓娓道来，为你呈上一场知识的盛宴。


古罗马人是如何赶跑国王确立共和制的？为什么说「条条大道通罗马」？罗马人如何打赢了称霸西方的决定性战争？埃及艳后到底爱安东尼还是凯撒？罗马帝国与基督教到底有何关联？


皇皇一部罗马史，千头万绪，从何读起？本节目精选了古罗马历史上那些重要转折的高光时刻，以一系列精彩讲述的故事，让你在短时间内对古罗马历史不但知其然，更知其所以然。


我们都听过「光荣属于希腊，伟大属于罗马」，可是，为什么说罗马堪称伟大？


本课程用 13 节小课，在专业的历史事实基础上，用精心设计的场景和音效，为你生动呈现古罗马一幕幕的史诗级场面。
