## 简介
科学谈爱，理智谈情。


男女之间很多的感情矛盾，其实都是择偶策略的不同，与其说是性别差异，不如说是人性问题。


* 为什么说，我们到 6 岁的时候，就基本确定这辈子，会爱上什么样的人？
* 为什么女性出轨倾向于比配偶更强的人，而男性则相反？
* 为什么我们不会爱上自己的血亲？

那么这张困缚着所有人，紧密而错综的爱情之网，到底是如何编织成的?


本专栏从心理学、社会学、生物学、人类学等多个角度，带领读者从好奇心出发，探索人类爱情背后的进化逻辑。
