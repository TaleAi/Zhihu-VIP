## 简介
这次的内容讲述了 SARS、埃博拉、H1N1、寨卡病毒等近十几年的「杀手」病毒，你会知道，每一次天灾，其实都是人祸，人类从没有正确认知「防疫」的重要性，也从未正视病毒与微生物一直是这个星球的主人。


鼠疫、天花、麻风、疟疾……人类和病毒的战争到了 20 世纪，我们战胜了一个又一个传染疾病，但每当我们认为人类已经赢得最终胜利时，病毒却以新的身份出现，人间失序。


人类其实从未战胜过 SARS


埃博拉病毒每隔两三年就会回到地球


H1N1 的「祖先」病毒曾经感染了世界 1/3 的人口


寨卡病毒只针对未出生的宝宝下手


你是否想过，如果这个世界爆发一次传染性高，致死率也高的病毒，人类将面临什么？


你是否想过，现在有多少种已爆发病毒，但人类从未搞清它的起源、更没有治疗手段？


你又是否想过，如果病毒变异升级再次席卷全球，人类是否有办法抵御？


也许人类，从未认识到「防疫」的重要性，也从未也从未正视病毒病毒与微生物一直是这个星球的主人。


现在，是时候了。
