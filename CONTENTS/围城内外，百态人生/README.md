## 简介
人生如戏，我必须是主角！


有人认为爱情是性，是清晨六点的吻，是孩子。


但其实，爱情是明明想触碰却又伸回手。 


我喜欢你，可能是见色起意，但也许是日久生情也不一定。


爱情最上头的那几秒，是暧昧开始的时刻。


就像我们在一起，可我只想吻你。破镜重圆后的爱情不一定会有好的结果，可是，如果他站在你面前的时候你只想抱抱他，那么就试试也无妨。


霸道总裁、深情小奶狗、斯文医生、傲娇教授……在本专栏你会看到无数都市男女破镜重圆、追妻火葬场、一见钟情的爱情小甜饼，你想要的我都有！
