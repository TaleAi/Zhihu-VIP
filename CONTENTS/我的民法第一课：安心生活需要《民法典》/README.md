## 简介
本专栏由知乎人气答主猴子老湿主讲。按照一个人的年龄从小到大，根据出生、入学、恋爱、结婚、置业、工作等不同场景，分别讲述一个人一生中可能遇到的民法风险及规避风险的方法。帮你建立对《民法典》的基本认识，获得与日常生活相关的法律风险提示和规避风险建议，了解与自身密切相关的权利，提升法律素养和人生格局。


《民法典》号称社会生活百科全书，对老百姓的生活无所不包。


《民法典》是公民权利宣言，可以帮助公民避险维权。


《民法典》也是一部被官方评价为「固根本、稳预期、利长远」的法律，无论是提升人生格局，还是了解天下大势，都是不可或缺的。


因为本专栏要求到长期从事民法研究猴子老湿为大家详细讲解《民法典》。


本专栏有三大特色：


一、用户向。与其他《民法典》普法读物不同，本专栏主要采用法律风险点提示的方式进行科普。从普通读者的高频次需求和日常生活出发，利用不同人生场景，提示普通人可能遇到的不同法律风险。针对不同的法律风险，提供了规避风险的建议，力求贴心、实用。


二、场景化。本专栏的主要脉络，是从《民法典》与人的一生的关系、影响入手进行科普，主要分为少年（含幼年）时期、成年之时、恋爱时光、买房买车、结婚生子、创业上班等人生的几个阶段，通过这几个阶段可能遇到的问题，来对《民法典》进行科普，帮大家找到这些问题的应对方式。


三、真实有趣。本专栏的作者长期从事民法工作，接手过各种各样的真实案例。因此，在撰写本专栏时，作者将会加入大量真实案例，也帮助大家消化知识的同时，也增加内容的可读性。你可以理解为，这既是一个科普专栏，又是一个民生八卦专栏。
