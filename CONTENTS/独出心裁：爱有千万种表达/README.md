## 简介
彩虹糖与小甜饼，向日葵与镁光灯。


她们各有各的精彩人生，或温柔救赎，或光芒万丈，或众星捧月，或寒江孤影，在爱与被爱里成长，在人间这一场惊鸿宴里熠熠生辉。


爱有千万种表达。


总有一种能戳动你心弦。


《你我皆黑马》


我是史上第一个女状元，刚殿试完，就嗝屁穿到了一个准备高考的学渣身上。


所有人都不看好我。


逆袭成为高考状元后，台下继妹和后妈脸色一会青一会白。我环视他们，微笑:


「乾坤已定，我为黑马。」


《猫猫》


精神病院门口捡了个帅哥，他坚定地认为自己是一只猫猫，要抱抱，要贴贴，洗澡时还要扒我浴室门。


可怜巴巴喊：「主人，放我进去。」


猫怕水，他怕我被花洒淹死。


后来，我发现他竟是裴氏那个神秘的大佬。
