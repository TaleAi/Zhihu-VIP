## 简介
何以「立国」？以血肉铸尊严。


我们都知道抗美援朝战争是新中国的立国之战。


但中国究竟是如何打赢这场旷世大战的？我们却所知寥寥。


* 抗美援朝诸次战役中，敌我双方的兵力、装备究竟如何？
* 长津湖战役中，我军战士如何在零下 30 度的环境中作战？
* 第五次战役时，我军为何被截断在三十八度线以南？
* 铁原阻击战中，我军为何能以弱胜强？
* 冷枪冷炮运动中，我军如何以劣势装备还击美军？

本专栏中，「梁山微木说历史」将用地图以及动态地图的形式，再现抗美援朝诸次战役的具体经过，全面展示我军战士是如何堂堂正正打赢的。
