**《朕是猫》**


远未来，经过了星舰战争后，人类得以稳定地生活在宇宙中。


小美是一名刚毕业的护士，被分配了一个重要任务


——去照顾立下了卓越军功的老猫虎威七世。


小美的举动让虎威七世想到了老主人，也就是传奇名将郑维韩将军。


虎威七世向小美讲述了它的经历——从宠物店逃出成为夜猫，受尽欺辱后进入星舰，在星际战场发挥自己的能力。


  



**《寄生之魔》**


***—— 2003 年度第十五届银河奖最佳新人奖***


一种名为「魔菟丝子」的转基因植物，正在消灭地球上的人类。


生物学博士尉迟，是该植物发明者的学生，无奈地肩负起收拾烂摊子的重任。


而尉迟博士的私人生活，也是个烂摊子……


表白失败，还被一只会「吃人」的外星小猫纠缠。


  



《**山海间**》


***—— 2003 年度第十五届银河奖读者提名奖***


一支载着二十万余人的舰队来到了一颗美丽的星球，这里完全适合人类居住。


在初探之下，人类以为这里没有智慧生物，便放心地进行勘探工作。


但其实在这颗星球上，生活着人类的古代传说中的异兽，它们被禁止与人类进行接触。


记忆的无端消失引起了舰长的注意，种种迹象让他开始怀疑，这个星球上真的有其他智慧文明。
