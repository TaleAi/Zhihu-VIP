## 简介
家长为什么需要先做好自己再教育孩子？孩子逆反怎么办？孩子学习成绩不稳定怎么办？只有幸福的教育才可以培养出幸福的人，北京四中璞瑅学校校长徐加胜带你一起寻找幸福教育的秘密。


你是否也曾经这样：  

 天天着急，拼学区房，拼家教；  

 天天焦虑，拼课外班，拼兴趣班；  

 最常说的就是「为你好」，却很少问自己的方法对不对；  

 你总感觉是孩子不听话，却很少从自己身上找原因； 


  




![img](https://pic4.zhimg.com/v2-40e0ad5762b638c0750cb225e3df1f27.jpg)

其实，最佳的亲子关系是温暖；  

 孩子的好习惯比好成绩重要的多；  

 孩子的核心竞争力不是分数而是拥有一个好的性格； 


  




![img](https://pic2.zhimg.com/v2-60189e7d4d531a59cdbb8449f87214f9.jpg)

徐加胜，中国最有实战经验的教育家之一；  

 他面对过成千上万的孩子；  

 他帮助过数以千计的家长；  

 对于家长的焦虑，  

 他有着拿来即用立竿见影的解决方案。  

 作为家长的你，  

 真的有必要认真听一听这套课程。
