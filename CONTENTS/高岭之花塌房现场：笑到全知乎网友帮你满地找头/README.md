**《七王爷有一个秘密》**


七王爷朱以之有一个秘密，那就是他会读心术。


帝王的谎言，闺秀们的算计，母亲的腹诽，他都听了个一清二楚。


像他这样洞察世界的人，本该孤独过一生。


直到一次游园会上，他听到了那个女孩的心声——


*「我刚刚听到他放了个屁。」*

**《为什么美男他总生气》**


醒来时我的面前站着一个绝世大美男。


我的鼻血蜿蜒而下。


下一秒，他拔出手枪，打死了我。


我没了。


我又醒了。


一次次的轮回中，美男反复掏枪，我死去活来。


但美少女永远不会认输。


我一定能在他手下过上混吃等死的好日子！


  



**《郝池真是大》**


曾经有个姓甄的祖先得罪了某个神明，现在神明降下诅咒，让他的后代厄运缠身。


很不幸，我，就姓甄。


意外连续发生，生命惨遭威胁，我，危！

「你女扮男装去当兵吧！」
