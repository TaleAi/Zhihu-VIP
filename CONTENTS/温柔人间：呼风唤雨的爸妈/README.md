## 简介
TA 们是「家」本身。


《**妖精**》


我六岁那年，有了继母。


继母姓罗，身边还有个女儿。


我奶奶从开始就不同意爸这桩婚事，她对罗阿姨极度反感，嫌她太漂亮、太时髦：「妖精妖怪的，一看就不是好女人！」


**《脸孔》**


他跟我说：「米拉，你晚上害怕时就摇铃，铃一响，爸爸就来陪你说话。」


时隔多年，千山万水之外，我终于找到了爸爸。


他昏睡不醒，我只得又摇响了这个铃铛，可是爸爸爽约了，他始终不肯醒来。


《**你养我长大，我陪你变老**》


但她是个了不起的女人。


他醒来后，看见地上的死鸡，一个猛子扑上去，紧紧搂在怀里，对我和石明却视若无睹。


我扯着多喜的胳膊哭着说：「爸，我是佳慧，你的毛丫，你看看，你怎么不认识我了？」


多喜却一个劲的摩挲着那只死鸡，任凭我怎么呼喊，他都置之不理。
