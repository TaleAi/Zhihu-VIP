## 简介
今日份小甜饼：豁出去了，好不容易双向一次！！！


嫁给唐先生


杜草草头一次在邻居奶奶撮合下相亲，就直接被唐先生的帅气折服。此后她又在 2 场相亲局里都偶然遇上了唐豁，心动的她决定主动出击，但唐豁属实难追，任她百般纠缠，他始终不为所动……


  



我女朋友一不高兴就爬树 


刘嘉琪是理工学院的高冷女神，偏偏主动追求王珂，交往后王珂才发现刘嘉琪是个活宝，不仅性格活泼沙雕，还一言不合就当众爬树，时间久了，王珂有点嫌弃刘嘉琪爬树的毛病丢人，总想着分手，可又舍不得


  



男友不分手


顾铎高三时，爸妈领养了一女孩，就是钟宁。钟宁很懂事，顾铎却处处为难她，把她当小跟班使唤。两人瞒着父母偷偷谈起恋爱，直到后来顾母让钟宁劝顾铎找女朋友，钟宁开始内疚，觉得对不起顾家的恩情。


  



他一定是为了我的钱


田小欢中了大奖后，失散多年一直想搞到手的男神池放找上门来，开口就是借钱。可怜他现在家里破产，工作上也是没有起色，立刻决定为男神一掷千金。两人的爱情是怎么水到渠成的？


  



时年情歌：薄荷


夏俐莉学生时代学渣一枚，陈斯旬是转学来的学霸，两人成了同桌，暧昧不断滋长，并约定考同一所大学。夏俐莉怕拖累陈斯旬改了他的志愿，并说了狠话。两人从此没了联系。直到夏俐莉 26 岁时，两人纠葛再起~
