## 简介
后妈是豪门赶出来的病美人，灵堂里孱弱艳丽的清冷少年，是她留给我的遗产。


夏蔓枝有个被豪门赶出来的病美人后妈，灵堂里孱弱艳丽的清冷少年，是后妈留给她的【遗产】


十七岁的夏蔓枝干了三件大事：葬礼上扬了她亲爹的骨灰，打十八份工养异父异母的漂亮拖油瓶夏洲，酒吧兼职时成功得罪了「太子爷」言楚及他的小弟们


然后她睡了一觉，发现自己来到了七年后


二十四岁的夏蔓枝，贫民女变富家千金，还嫁给言楚成了豪门太太，婆婆不喜老公不爱


继弟夏洲变成了宋鸿，连夜撬墙角送离婚协议书


但——眼前的宋鸿，真的是【夏洲】么？


层层反转，究竟谁说了谎
