## 简介
把血迹，从笔端烙印在纸上。


「血过眉，眼飙泪，今夜我陪你化成灰。」直到故事的末尾，我们一层一层地剥离出血中带泪的秘密。


《七号楼的死者》


我是个保安，某天值夜班的时候，在小区里发现了一具全身赤裸的尸体。


但问题是，警察认为人是我杀的。


《烈日之下》


我的猫上吊死了。很奇怪吧，我也很奇怪。我报警了。


警察勘察后，明确说明，你家来过人。


警察还说，你是写过真实案件改编故事的《饲养真凶》作者吧，不排除犯人刑满释伺机报复你。


《特殊病人》


某天有位诡异的来访者找到我，跟我说他的妻子复活了，来找他报仇。我对他的病状进行观察，治疗，却发现他并没撒谎，似乎真真切切的相信着妻子真的来复仇。


他的左手不受控制，只要他一个人待在房间里，就有可能产生自残的行为，我就曾看过，他把自己掐得濒死。


他不是简单的精神分裂，或者抑郁自残，所以大多数人都以为这真的是灵异事件。


但随着对他的深入调查，我才发现，他其实只是病了。
