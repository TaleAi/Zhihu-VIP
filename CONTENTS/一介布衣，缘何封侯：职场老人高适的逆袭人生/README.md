## 简介
公元 749 年，高适 50 岁了。在人生的前 50 年，高适见证了盛世大唐的崛起，但出身显赫的他，一直未能获得一官半职。

在平均寿命只有三十岁的唐朝，高适的仕途还有希望吗？


公元 749 年，50 岁的高适已经写出了千古名句「莫愁前路无知己，天下谁人不识君」，成为了盛唐时期的著名诗人，与李白、杜甫相交为友。


但他既没有考试天分，屡中不第，也不受当时的政坛待见，求职 20 多年一直没拿到一个 offer。


好在这一年，高适遇到了人生第一个贵人，通过「内推」得到了他人生第一份正经工作，在 50 岁高龄，成为了一名光荣的职场新人。


从一介布衣到普通办事科员，高适花了 50 年；但从一个小小的县尉到威风八面的节度使，高适不过用了 7 年。升官速度之快，实在让人咋舌。


同时，高适，也是唐朝诗人里唯一封侯的人。


但同样是在这 7 年，盛世远去，安史之乱席卷整个帝国，君王无度，名将惨败，故乡毁灭，旧交李白投敌，好友王昌龄遇害。


高适的身上，究竟发生了什么？为什么他能从一个大器晚成的白衣成为唐朝诗人里官做得最大的那个人？


这个专栏，北师大文学博士 Zpuzzle 为你讲述高适的传奇故事。
