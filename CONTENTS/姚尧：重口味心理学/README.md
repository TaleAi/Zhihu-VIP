经常觉得门没锁  

 反复一段重复的梦境

 人类各种怪癖的背后，隐藏着怎样的终极秘密？


不论你是心理学的爱好者，还是想从心理学中找到解决办法，亦或者只是有个砰砰跳的好奇心，这门「重口味心理学」都一定能满足你。
