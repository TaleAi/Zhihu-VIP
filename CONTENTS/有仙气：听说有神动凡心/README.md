《上神以为要挨打》


「少帝君回来了!」


梨容翘起二郎腿，「回就回呗，咋，我还得给他跳个舞欢迎一下？」


「你不跑吗？」


梨容撇嘴，「为什么要跑，我可是上武神，我打天打地打空气，这四海八荒谁能打得过我？」


「可是少帝君就是凡间那个小太子啊。」


梨容笑容凝固，慢条斯理地站起来，拍了拍衣服，不屑地哼笑一声，「就这？」


表情突变，拔腿就跑。
