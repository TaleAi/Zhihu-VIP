## 简介
在我们生活总是面临各式各样烦心的「破事」，让我们来找寻「破事」背后爱的秘密。


《寻父记》：


我爸患有阿尔兹海默症，在本月第 12 次寻找走丢的他后，我意外发现爸爸藏在心底多年的秘密。


《我的后妈》：


我后妈第一次对我「欺负」的时候，是刚嫁给我爸的第一次，从此我开始了和她斗智斗勇的每一天，她一打我，我就大喊后妈虐待小孩啦，这一招屡试不爽，直到高考那年，我爸去世了，所有人都以为她会离我而去的时候，她却告诉我：「别拿后妈不当妈。」
