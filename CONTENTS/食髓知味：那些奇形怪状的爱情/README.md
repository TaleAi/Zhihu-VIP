## 简介
当非寻常的怪癖遇到了真爱，故事将会如何发展？


《我以为的先婚后爱》


我眼尖地看到我老公，喝了一口别的女生端过来的饮料。  

我老公有洁癖，从没有亲过我的嘴唇，甚至从没碰过我的杯子。  

婚前，我们约好，相互扶持，相互帮助，尽量相爱。如果过不下去，就和平分开。  

这四年，我们确实做到了相互扶持。只是，他尽了力，也没能爱上我。


《我想起来了，我爱他》


我经历了一场车祸，脑震荡，把我未婚夫给忘了。  

不但忘了，我还对他声色俱厉，说我只喜欢我前男友，让他滚出我的生活。  

他在我楼下等的时候，我朝他泼了一盆冷水。  

然后，在他结婚那天，我想起来了。


《博士站姐实录！不是追星，是暗恋》


我上了新闻。电视机画面是我应援齐梦屿的画面。主持人一脸严肃地评论说：「花季少女放弃学业，甘当职业站姐，在机场追星扰乱治安。用网络上的话说，这是病，得治。」  

我撇撇嘴，他知道个屁。  

那些人是职业站姐。还有的小女孩是来追星的。可我是去追男人的！


《私人厨师与吻的甜度》


我的老公是我舔来的。真的是舔来的。我第一次见许故遥，就在他脸上舔了一口。  

我再怎么解释他也不相信，我只是觉得他的脸味道不错。我有异食癖，什么东西都想舔上一口。  

「那你舔过别人吗？」我回想了一下，好像还真没有。
