## 简介
李碧华说：「什么叫多余？夏天的棉袄，冬天的蒲扇，还有等我已经心冷后你的殷勤。」那些握不住的沙，攀不起的高岭之花，我们不要了。爱自己才是终身浪漫的开始。


**- 不二救赎 -**


我嫁人了。


对方比我大十岁，除了不爱我，各方面条件都很优秀。


婚礼进行时，我听到有人议论，说我曾经是新郎弟弟的舔狗，现在为了钱，换了个年纪大的舔。


我嗤之以鼻，他们懂什么？


我舔的从来都不是弟弟。


所有殷勤都是为了我身边这个人。


甚至怀上孩子的那一夜，也是因为我看错了人。


  



**- 我在破晓时分遇见你 -**


我男朋友出轨了。


我骂了他两句，他竟然跳楼了……


所有人都怀疑我是凶手，他的全家更是让我给他儿子抵命。


流言如猛兽，铺天盖地的谴责让我一度迷失，我不禁也在怀疑，我会不会真的是杀人凶手？
