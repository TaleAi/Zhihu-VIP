## 简介
这是一个完全甜宠恩爱的古代言情专栏。摒弃后宫争权的套路，没有阴谋诡计，没有争风吃醋，没有猜疑嫌隙，只有皇上和皇后甜到发腻的恋爱趣事。


这个皇后，长得不漂亮，还没生个皇子，怎么还这么让朕添堵？要是这样下去，朕就只好……  

——只好想办法把她哄开心。


选妃大典，秀女个个绝色，皇上小心翼翼，不时侧头看皇后脸色。皇后脸色很不好看。  

——不选了，皇后不高兴，没法选。


皇上生病，端起药碗，舌头稍微碰下药汤，一脸苦相：「有没有点甜的东西中和一下？」  

——「皇后，你亲朕一下。」


一对恩爱的皇上与皇后间发生的一系列趣事。


文笔清新幽默，风格轻快温暖，摒弃后宫争权的套路，将描述重点放在两个人如何相遇相知相守，彼此信任和依赖，共同经历种种考验。


这是许多人心中所向往的爱情，没有所谓阴谋诡计，没有各种争风吃醋，没有许多猜疑嫌隙，只是简简单单相守一生，虽显平淡，却更加珍贵。
