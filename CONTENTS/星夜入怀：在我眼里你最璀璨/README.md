## 简介
让你身临其境每一个脸红心跳的瞬间。


在这个专栏里，你可以邂逅无数个都市童话，遇见不同的自己。


或是体验甜蜜的爱恋，或是感受只愿得一人心的温柔，或是享受快慰人心的甜爽时分。


生命中遇到的人如同流星，但总会有一颗专属于你与我。


那颗星在你眼中最为璀璨，像是薄雾中为你亮起一盏街灯。


或迟或晚，它总会降临。


若生活不如你的心意，我们亦可以成为自己的星。

帅气教官从宿舍楼下经过，我收衣服的手一抖，袜子掉进了他饭盒。


我红着脸下去捡。


谁承想后来会被他圈在怀里，「顾老师，接吻怎么接，能不能请教请教？」
