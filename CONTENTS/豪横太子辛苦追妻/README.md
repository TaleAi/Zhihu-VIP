## 简介
一纸退婚书，豪横太子本想做个渣男，全身而退，却没想到，从此走上了一条漫漫追妻路。


白萃薇及笄那天，太子来退婚。


这消息太突然了，前不久白萃薇还熬了几夜赶制棉袄送到边疆，就怕太子冻着。


白萃薇没有死缠烂打，她只有一个要求：面子没了，里子一定要有。


锦绣阁、金玉铺……太子赐的这几处产业，可保白家这一辈的富贵。


白萃薇以为往后能过上富贵闲人的日子，没想到少了太子爷的庇护，不仅自己被色欲熏心的登徒子盯上，白家人在官场上也屡遭小人陷害。


白萃薇想在太子门下谋个职位。


她请太子吃饭。


席间，白萃薇故作乖巧姿态，太子一时没忍住，强吻了她！


？？？


更奇怪的是，卫小侯爷追求白萃薇时，太子就如打翻了醋坛子，各种别扭。


好马也吃回头草？


「你当初为什么要退婚？」


在太子面前，白萃薇终于问出了这个困扰她已久的问题……
