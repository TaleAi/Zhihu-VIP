## 简介
《西游记》里有诸多未解之谜。
例如：作为佛子的金蝉子为什么被贬下凡，历十世转世之苦？
例如：孙悟空为什么大闹天空，葬送花果山四万八千小妖？
例如：红孩儿真是牛魔王之子吗？铁扇公主的芭蕉扇和兜率宫太上老君的芭蕉扇是不是同一把扇子？
反套路，重悬疑！
误入西游世界，从原著的角度再次解读取经路上妖怪以及各路神仙的终极目的。



误入西游世界，我成了西天路上最没有存在感的沙和尚，上有疑是恢复前世修为和记忆的唐和尚，下有真假善恶莫辨的孙悟空，还有处处给我打禅机的猪八戒……


而我，却为着《大乘真经》版权费操碎了心！
