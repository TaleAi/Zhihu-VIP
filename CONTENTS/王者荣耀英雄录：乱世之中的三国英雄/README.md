## 简介
本课程主要讲解王者荣耀中女性英雄的历史背景，这些传奇人物的背后有着怎样不为人知的动人故事，本课程为你还原一个比游戏更加精彩的历史真相，让你在轻松愉快的氛围里感受历史的独特魅力。


王者荣耀作为一款经久不衰的国产手游，它火热的背后原因在于有着源远流长的中华文化作为底蕴，每一个游戏角色都是根据历史或传说人物的原型而设计的，这也是英雄们独特的魅力所在。


经常玩王者荣耀的朋友对其中的英雄人物一定都很熟悉，可是对于历史上的原型，你又有多少了解呢？


李白的剑术究竟如何？兰陵王为何要以面具示人？上官婉儿究竟是何方神圣？她手里为何总是握着一根毛笔？干将莫邪到底是一个人还是两个人？雌雄双剑背后隐藏着怎样的故事？


在王者荣耀故事系列课程中，铁扇老师将用风趣幽默的方式，妙趣横生的语言和引人入胜的故事，对那些耳熟能详的历史人物和事件进行解读。从娇柔抚媚的乱世红颜到开疆辟土的传奇帝王；从吟诗作画的骚人墨客到英勇善战的三国群雄，去发掘每个历史人物背后鲜为人知的一面。


本课程主要讲解古代将相们的历史故事，他们或草根出身，或名门望族，在英雄不问出处的乱世里，他们成长为传奇的路上有着怎样的故事，其名垂青史的功绩里闪耀着何等的人性光辉。这节课为你还原一个比游戏更加精彩的历史真相，让你在轻松愉快的氛围里感受历史的独特魅力。
