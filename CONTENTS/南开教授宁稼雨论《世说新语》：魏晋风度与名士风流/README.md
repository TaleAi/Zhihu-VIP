## 简介
《世说新语》为南朝刘义庆编著，是魏晋名士「言」与「行」的真实写照。莼鲈之思、楚楚可怜、拾人牙慧、标新立异，都是我们熟悉的成语故事与名人轶事。在本专栏，感受《世说新语》中的人物之美、自然之美、语言之美，告别枯燥「啃」古籍的方式，轻松了解历史文化。


鲁迅认为：「《世说新语》是一部名士的教科书。」


傅雷曾说：「你现在手头没有散文的书，《世说新语》大可一读。」


在中国历史上，没有哪个朝代和魏晋一样，诞生了那么多名士，容貌秀美，神姿高彻，才智凌厉，境界超然。读《世说新语》，解读魏晋，领略一个时代的文化高峰。


在名士的故事中，找到现代人的精神寄托。超然物外的态度、宽宏雅量的气度、自在洒脱的性情，都值得我们学习。


本专栏南开大学中文系教授宁稼雨，带你从《世说新语》入手，全面了解魏晋风骨，从政治、思想、文学、语言等方面读懂千古经典，带你领略最专业、最丰富的魏晋文化。
