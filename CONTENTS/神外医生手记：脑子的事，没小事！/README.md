## 简介
这里是最「危险」的科室之一。


神经外科是看什么的？


简单点说，看脑子的。


复杂点说，主治由于外伤导致的脑部、脊髓等神经系统的疾病，例如：脑出血出血量危及生命，车祸致脑部外伤，或脑部有肿瘤压迫需手术治疗等。


在这个科室发生的事情，多半和「危急」有着千丝万缕的关系。


这里有惊险万分的抢救：


* 车祸头骨破碎的人，是如何被救活的？
* 突然脑动脉瘤破裂出血，如何和「死神」抢时间？

也有医患之间的温情和纷争：


* 家属自愿放弃治疗，又为何带人砸上医院？
* 伤情并不严重的患者，却被警察告知人死了？

在本专栏中，神外医生、知乎答主 神经木木杨 将为你讲述他从医生涯中难以忘怀的八个真实故事。
