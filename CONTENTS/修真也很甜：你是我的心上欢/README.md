## 简介
岁月漫漫，修仙本是清苦无情，但我想到你就脸红，见你不用腮红。



所有的爱恋不是突如其来，所有的缘分都是他披荆斩棘，伤痕累累地走到你面前，只为递给你一朵盛开的鲜花。


《师父行行好》


我去和师兄表白，结果脸盲说给了师父听。


社死想逃！


可很久以后我才知道，所有的阴差阳错，都是师父步步为营的结果。


原来才认识十年的师父，已默默守护了我百年。


《小饕餮的食谱》


我是凶兽饕餮，万物都是我的食谱，但我只想吃上仙折浔。


为了吃到这口上仙肉，我屁颠屁颠地缠着他。


最后我才知道……这都是套路！


心机上仙追妻的套路！


《清纯小狐妖》


都说四大凶兽，穷奇最恶，张口老子，闭口打架。


我瑟瑟发抖，就害怕哪一天成为他的盘中餐。


可他却小心翼翼地，费尽心思地，用心头血把我养大。


只因我是他的红鸾命定……
