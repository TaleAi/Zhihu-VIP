## 简介
本专栏从零构建古希腊哲学框架，带你读懂先哲经典，掌握哲学的底层逻辑。在哲学世界，一探智者的思想光芒和人生伟绩，以人为镜，运用先哲智慧重新构建人生观，拥有独立思考的能力和批判性思维，提升做事格局。


生活中的难题，都能在哲学中找到答案。


职场中有必要和同事交心吗？


买房是现代人生活幸福的标准吗？


真正的幸福到底是什么？


......


低头实干的同时不忘抬头仰望星空，人类思想星空中闪耀着不灭的理性光，借以照亮你人生的前行方向。哲学帮你解脱现实迷茫，清醒处世。


古希腊哲学更是西方哲学的源头，甚至可以说是整部西方哲学史，就是对柏拉图哲学的注释。


本专栏赵林教授有 30 余年哲学授课经验，用朴素清晰的语言讲述古希腊先哲对人类恒久命题的深刻思考，一个个令人生畏的哲学名词，变得不再陌生遥远。结合当代人生婚困惑和难题，给你幸福生活良方，修炼处世智慧，提升人生格局。
