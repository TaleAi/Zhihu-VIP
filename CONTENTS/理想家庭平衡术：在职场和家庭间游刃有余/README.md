把握平衡家庭和事业的秘诀，有效改善亲子关系。


平衡家庭和职场，是每一个职场女性兼母亲都要面对的问题：


每天疲于奔波，吃不好、睡不好，无法全力以赴地工作，更不能陪伴孩子健康成长。


是时候跳出困境，用更高的角度、更广的视角直面经典两难。


本专栏特邀张雪梅老师从心理学角度重新审视你的家庭，依据 4 种情感依恋模式了解孩子成长的情绪变化。


通过真实案例剖析生活难题，从时间规划到生活小技巧，快速把控孩子成长过程中的情绪变化，打造最理想的家庭关系核心认知，面对困境不慌张。
