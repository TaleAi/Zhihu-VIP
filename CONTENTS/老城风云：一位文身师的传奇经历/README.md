# 老城风云：一位文身师的传奇经历

## 简介
在这个专栏里，你将看到文身行当的规矩和讲究，济南老城在 20 世纪八九十年代的社会变迁，还有那些让人或心惊或温暖的奇人奇事。


我以文身立命的时候，是 20 世纪八九十年代那会儿，济南城。


那个时代正处在社会快速变化时期，可谓风起云涌、波谲云诡，明潮之下亦有暗潮，暗潮之中又有人性和正邪间的拉扯，产生了诸多当今所无法想象的骇人之事，或黑暗血腥，或正义温暖，或扑朔迷离......


这个专栏将由文身师「我」，来为你讲述那些奇人奇事。


**龙虎之争**


1992 年我刚出狱，就被卷入了一场龙虎之争中。当时在济南城有两大黑势力，一边就是城东刘凯元，另一边是城西胡海。就在我的接风宴上，两股势力的较量一触即发，后续更是引发了一场血腥残暴的入室枪击案，就此拉开济南全面反黑的序幕……


**厨肉**


1998 年我脱离黑社会，终于在洪楼开了属于自己的文身店，那时虽然生意惨淡，但结识了一些聊得来的朋友，其中就有平成饭店的老板兼主厨苏百杨，他做的那道溜肉片简直美味绝伦。一次偶然，我发现苏白杨有食生肉的习惯，还发现了平成饭店火爆的秘密……


**雨燕**


1999 年我的文身生意有了些起色。这一年我认识了一个在 KTV 做皮肉生意的女孩。她请我在她左边锁骨处文了一只燕子，她说想像燕子一样自由。因我之前为她解围，她便经常来我店里，一来二去渐渐熟络。那天她对我说，攒够钱年后就不干了，然后像燕子一样冲进雨里，谁知再也没回来……


后来的事呢？


别急，走进专栏慢慢看。
