## 简介
本课程融合了讲者多年企业管理与咨询经验，为所有管理者输出一门可复制的业务管理课。


业务完不成，团队管不好？布置任务时没有问题，执行时却让人生气？组织看起来人才济济，能用的寥寥无几？团队表面和和气气，暗地里互不搭理？


本课程从业务管理者身份出发，围绕业务管理流程，手把手教你定目标，带团队，拿结果。


让你识人心，懂人性，追过程，拿结果，从心出发带团队，以终为始出业绩。
