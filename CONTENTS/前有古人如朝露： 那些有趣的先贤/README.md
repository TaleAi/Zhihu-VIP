## 简介
这个专栏会介绍几位有意思的古人，他们有过奇幻的经历，他们做过非主流的抉择，他们是族谱中的先人……他们也是历史中不灭的蜡烛，燃烧着人性的火光。


我们将刷新对这些人的认识。


他是我们小时候看过的动画片主人公，他叫「大英雄狄青」，但青年的他吃过牢饭；


他是提出「格物致知」的人，是历史课的考点，也是被我们调侃说有兄弟在卖十三香的王阳明；


他是神话里哪吒的托塔父亲，是初唐名将，也是爱情典范《红拂夜奔》的主人公李靖。


……


他们是我们的先人，是另一个时代的我们，是过去的朋友，是我们应该重新认识的人。
