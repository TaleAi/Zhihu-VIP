## 简介
为何说隋炀帝与明君李世民只有一线之隔？隋炀帝到底有什么不为人知的秘密？历史学者于赓哲将为你颠覆性解读中国历史上诋毁争议最大的皇帝，巧妙地结合历史与现实，严谨不失幽默，轻松不乏深刻，为你揭开隋炀帝的人生真相。


在中国历史上，有两个「甲兵强盛」而又二世而亡的著名王朝，首为秦王朝，次为隋王朝。隋炀帝杨广是中国古代隋王朝第二代皇帝，亦是一个亡国之君。隋炀帝和秦始皇有相似的结局和业绩，但同时也有相似的长期不公正的评价。


「虐用民力，骄奢淫逸」是以往人们对隋炀帝最多的评价，但为什么是西方人最推崇的中国帝王？


  



![img](https://pic2.zhimg.com/v2-4276eaf9cb08257ba1eb42317691850c.jpg)

著名历史学者于赓哲认为隋炀帝的形象被严重地丑化、脸谱化。隋炀帝短暂的帝王人生，生动而又不可思议，但在历史学者眼中，大唐盛世的荣光里，隐约可见隋炀帝的背影。我们有必要正本清源，客观地考察隋炀帝的生平、功绩与过失。


于赓哲以「大历史」的视野展开隋炀帝的真实人生。隋炀帝是中国古代社会历史上建树最多的皇帝之一，是一个复合型人物，其业绩不下于秦始皇。隋炀帝在位期间其伟业足可扬名后世。但其罪也彰，隋炀帝亦有暴而淫的一面，致使民不聊生。


「历史是最好的教科书。因为历史中有我们的人生，有我们的世界，有我们的自己。」


于赓哲用易懂，有趣的语言，带我们拨开迷雾，揭开隋炀帝的人生真相。且提出以隋为鉴，从隋炀帝的失败中汲取经验教训。比如如何处理战略与执行，把握自我个性等。
