## 简介
本专栏介绍了十位春秋时期的著名刺客，从他们的生平事迹，揭开这个群雄逐鹿的纷乱年代的面纱一角。


十步杀一人，千里不留行。事了拂衣去，深藏身与名。


提到「刺客」，你首先会想到什么？是易水河畔「风萧萧兮易水寒，壮士一去不复还」的荆轲，还是王者草丛里的「兰陵王」？


刺客是人类历史中一种由来已久的特殊职业，常由于政治、私怨等原因，负责对某个目标人物实施谋杀或暗杀。他们选择成为刺客的原因五花八门，为却私仇，为图钱财，为报家国，等等，不一而足。


春秋时期便有这么一批刺客，与现代的刺客不同，他们行刺的理由不仅仅是为了钱财。他们既有国士之风，又有侠客之义，更有甚者有忧国忧民之心。在成为刺客之前，他们都是些隐于市的普通人，从未经受过什么专业训练——打渔为生的要离，当过屠夫的专诸，主业是乐师的高渐离……当然，比之市井平民距离权力中心更近的军政大臣，也不乏「剑走偏锋」之辈。


春秋战国是个充满魅力的传奇时代，我们不能穿越时光去见证这些波澜壮阔的历史画面，但是透过这些刺客，我们也能领略到那个时代的几分风采。
