## 简介
时至今日，社会对女性仍然有着许多刻板印象与偏见，希望女性按部就班地成长、上学、工作、结婚、生子，成为听话的女孩、温顺的伴侣、伟大的妈妈……因为这些偏见，她们被困于种种角色中，牺牲了自我和人生。她们备受煎熬，但她们也正逐渐从各种偏见中醒来。试图挣脱被定义、被禁锢的人生，活出真正的自我。


女性大约从出生开始，就遭遇性别偏见。


有人出生在重男轻女的家庭，被迫沦为「扶弟魔」；


有人遭遇职场性别歧视，能力出众却不被重用；


有人因大龄未婚，被当成婚恋市场上的「打折货」处理；


有人迫于无奈做了全职妈妈，牺牲事业，却没能得到丈夫的肯定和尊重……


这些扎心的故事，每天都在真实上演。


  



怀胎十月的人是我，冒着生命危险生产的人是我，每天每夜带孩子的人也是我……


生不生，最大的决定权难道不应该在我吗？


——《一个女人决定放弃她的孩子》


家境贫寒，妈妈让我网贷供弟弟上大学，被暴力催收后，母亲和弟弟不帮我还款，还要跟我断绝关系……


——《妈妈让我网贷供弟弟上大学》 


婆亡公病，无奈做家庭主妇的第四年，老公却说，「你整天围着灶台、老人和小孩转，不再是以前那个温柔的文艺女生了，我跟你没话聊……」


——《全职妈妈的不浪漫罪名》


  



身为女性，在这个时代，活出自我需要更大的勇气，路途也更艰辛曲折。


但无论如何，值得为之冒险。


愿每一位身边的女孩，终将醒来。
