## 简介
本专栏目前分为 《地府之前》 《妖怪客栈》 两部小说，构架于同一世界观内，讲述各路正统与非正统妖魔鬼怪的万般无奈与前尘往事。


一场大战之后，鬼族归顺，妖魔四散，幽冥灭亡而九重天独断专行。从地府到深山，从妖魔到鬼怪，命途多舛、生活不易。


妖魔鬼怪并不可怕，他们其实是一群懒得搭理人类又自视甚高的幽冥物种。他们曾经很团结，直到鬼族起了幺蛾子，背叛幽冥，投靠了九重天的老神仙并与昆仑神兵联手发动大战。自此妖魔四散，鬼族被收编。


但故事并没有因此结束，冥府的消亡带来了地府的时代，恩怨仍在延续、战争一触即发…...


  



**《地府之前》**


陛下终于下旨，说要诛我九族。


我实在太高兴了，在屋内忍不住想要大叫。


这次任务结束，我就可以回到地府，继续做我的押魂使。天知道为了完成阎王给的这劳什子任务，我在人间已经待了多久。


十八年！十八年啊朋友们！


 


**《妖怪客栈》**


荒山野岭有家客栈，住着一群不着调也不合群的妖怪。


那里有妖打尖儿，有怪住店，每到夜里，群魔乱舞。他们有着各自难以开口的秘密和放不下的过往。


千百年来，掌柜和伙计送走了许许多多稀奇古怪的客人，却从未想过也许有一天，他们也终将离去。


人有人的武林，妖有妖的江湖。
