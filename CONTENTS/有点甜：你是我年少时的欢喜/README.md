## 简介
你是年少的欢喜，喜欢的少年是你。


《今晚月色好美》


  



分手后，校草公开了和我的恋情。


我发了条朋友圈：「谢邀，爱过。」


他秒回：「？昨晚，你可不是这么说的。」


  



《哥哥》


  



我惹到了学校里的女大佬，决定找男大佬帮我。


赖野锢住我的腰，把我拉进怀里，贴在我耳边，低低地说：「叫声哥哥，我就罩着你。」


哥哥多没劲啊。


我搂住他，喊了声：「老公。」


他脸红了。


  



《招惹》


  



篮球场，宋擒隔扣了我哥。


还放狠话：「离我女朋友远一点，这次是警告，下次废了你。」


我哥敛眉，朝我走来：「这就是你交的男朋友？」


我腿一软，差点跪到地上。


「哥，我错了。」


不远处，宋擒的表情像是吃了屎。


「你叫他什么？」


哦，我忘了，他俩是死对头。
