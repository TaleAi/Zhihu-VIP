## 简介
手机能有什么坏心思呢？手机当然没有，有坏心思的是人哦！本专栏采用新型对话体的形式，用更直白易懂的方式，写了一个个有趣而不失深刻的故事，帮你了解多面的人性。


一群人被莫名拉入一个群里，瞎聊中竟发现许愿能成真。


群里一个单身汉，希望脱单暴富，成功了；


一个加班狗，希望老板大腿骨折住院，成真了；


就在大家觉得一切妙不可言时，群里一个神经质出现了，于是一切都变了。


——《话不可乱说》


  



有个群，群主说他有让别人遗忘任何事情的能力，只需要支付身体的一部分，比如头发、指甲......


一个赌徒，希望他的债主忘了赌徒欠的钱，成功了；


一个教授，希望学生忘了教授曾猥亵过她，成功了；


……


就在大家准备好好利用下这个能力时，一个叫「愚者」的人下了个订单，结果每个人的身体越变越糟糕。


——《遗忘的代价》
