## 简介
唐朝是个开放的朝代，万国来朝；唐朝又有它保守的一面，日落后就要严格宵禁。真实的大唐究竟是什么样的？当地人民是怎么过日子的？「时尚大唐」中，长居西安（大唐长安）的作家王一凡带你深入大唐朝，从吃穿住行到消遣娱乐，从升职加薪到婚恋嫁娶，为你生动还原当地人民最真实的生活状态。


「定居在唐朝是种什么体验？」


没有手机，没有 WIFI，一千多年前唐朝人怎么过日子？定居在大唐长安，是不是就是白天在外干干活，晚上点灯唠唠嗑？唐朝人的真实生活，其实比我们想象的有趣：


吃生鱼片是日本人的专利？一千多年前，会吃的大唐人就已经尝过了生鱼片的美味……


开车？坐轿？还是骑马威风！在唐朝，拥有一匹好马，就跟今天开着辆宝马心情是一样的……


没有电影的日子里，大唐人爱上了「看戏」。不是去电影院看，唐朝人看戏，首选的场所是寺院……


没地儿购物？「西市」这个唐朝超级 mall 里卖的都是名牌货，引领着大唐服饰的时尚最前沿……


不过，定居在大唐朝也不是样样都随心所欲：买房不仅要有钱，还得有身份；厕所里没有卫生纸，只能用竹签做做清洁；就连大晚上吃个宵夜也不行，别忘了这里可是有宵禁的……


本专栏中，长居西安（大唐长安）的作家王一凡，为你提供了一套「定居大唐」的完备方案。王一凡多次走访西安的各处文化古迹，翻阅大量古籍专著，从饮食起居、工作求学、服饰潮流、风俗娱乐等各个方面，为你还原了时尚大唐里普通百姓的日常生活。


最后，留个悬念，你知道天南海北的大唐诗人们，都用啥话念唐诗吗？答案就藏在本专栏中～
