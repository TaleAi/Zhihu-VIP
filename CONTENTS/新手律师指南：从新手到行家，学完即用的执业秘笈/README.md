## 简介
本课程由执业十余年资深律师李斌主讲，从 7 个板块出发，指导青年律师进阶成长经验，涵盖律师执业多年的切身经验，分享律师相关的办案经验与成长经验。课程框架清晰，干货满满！


进入律所，你可能是法学象牙塔青涩出炉的学生，也可能是从其他行业转行做律师，掌握着另外一种工作的基本的方法论，本想大展宏图，却发现困顿于最基础繁杂的工作之中。


「读不懂老板的心思」、「工作时间被大量的基础性工作占据」、「活多、钱少、累心」更是日常。而只有日常拼命修炼，才能为日后成长为优秀的执业律师打下牢固基础。


其实如果你平日留心观察，你就会发现，真正优秀出彩的律师，尽管他们在性格特点、从业领域、知识结构、履历背景等诸多方面存在较大的差异，但他们身上具备一些共同特点：


一、「脑瓜子」灵


二、「笔杆子」硬


三、「嘴皮子」活 


总之，作为律师新人，想要快速成长，就必须充分利用好职业生涯的起步阶段，把握住培养良好工作习惯的关键时期。


为了助力新手律师在入行初期快速成长，本课程邀请中闻律师所资深律师李斌，开设干货超多的新手律师执业秘笈课程，分享他积累十几年的实务技巧与经验，助你从 0 到 1 迈出执业逆袭第一步。
