## 43.《妙手情真》：从小丑身上学会轻松玩乐的智慧
妙手情真：从小丑身上学会轻松玩乐的智慧


 


1.


亲爱的听众朋友们，欢迎您收听今天的节目。


不知道音频那一段的您，会不会有这样一种跟我类似的感觉？


对我来说，我明白，生命的意义很大程度上要促进自己和这个世界的融合，多与他人连接，让生命流动起来。


但在实际执行的过程中，不知道是不是性格中有拘谨、被动的一面，有时候，主动的展示自己，去与他人保持积极正向的链结，对我来说，还是有些困难。


面对陌生人，我不是那个主动会热情打招呼的那一个；而点头之交点头之后，还真不知道要多说点儿什么。


其实，我也是从今天跟大家分享的电影《妙手情真》中，学习到了不少轻松玩乐的智慧，了解到更多跟别人链接和融合的方法。


 


2. 


这部《妙手情真》，英文名叫 Patch Adams。这是一个人名，它取材于真实的故事，Patch Adams 确有其人。


讲起他的这个绰号 Patch，还有一番典故。


Patch 在英文里有修补之意。Patch Adams 本人，原名叫做 Hunter Adams。少年时的 Hunter 一度深陷抑郁症难以自拔。他的父亲参加过朝鲜战争，一生都很不快乐。少年时的 Hunter 一直以为是自己不够好，所以父亲才不开心。直到父亲去世，他才明白父亲的不开心源自战争创伤。但明白这一点并没有对 Hunter 的抑郁症起到什么积极作用，他一年搬了 7 次家，实在不得已，他自己申请去到了精神病院。


没想到，他一生的转折点也由此而来。


在精神病院，他遇到了知名实业家 Arthur Mendelson，这位毕生耗尽脑力，追求创新的老人，也是自愿住进精神病院。


一见面，老人抓住年轻的 Hunter，伸出四个手指问他，这是几？


谁会面对四根手指，回答出其他的答案呢？


按照常理，大家一定会认为，这位老人果然是疯了，但 Hunter 很好奇，他深夜走进老人的房间，想要问个究竟。


老人并不理会他，只是埋头密密麻麻演算着公式。Hunter 看着老人装咖啡的纸杯漏了，随手拿起一小条胶布补上了漏洞。


老人大概是觉得孺子可教，然后伸出四只手指，让 Hunter 学会 look beyond things。也就是说，当你看问题的时候，一定试着要超越问题本身，这样你会发现更大的图景，有时候会看得更清楚明白。


是的，如果伸出四根手指，放在你的面前，如果可以把焦点集中在手指后方，你会慢慢发现，由于左右眼之间有距离，双眼对焦产生偏差，四根手指像是变成了八根。是的，这是一种新的看问题的视角。


年轻的 Hunter 问智慧的老人，您在我身上看到了什么呢？


老人看了看被修补好的纸杯，吐出一个字：patch，没准儿，你善于修补。


年轻的 Hunter 陷入沉思。


在精神病院的日子，他被周围的病人吸引。同屋的病号有幻想症，总是觉得屋子里进来了小松鼠，而小松鼠在蓄谋攻击自己。晚上，因为惧怕小松鼠，室友坐在床上，虽然咫尺之遥，但根本不敢去上厕所。


Hunter 突发奇想，也虚拟出枪炮，一一击毙进攻的那些其实并不存在的「小松鼠」们，一场酣畅淋漓的「战斗」过后，Hunter 终于「掩护」着，让同屋开心地跑去厕所。这个过程紧张刺激，又那么好玩，而年轻的 Hunter，不，年轻的 Patch Adams 找到了一生的事业——当他尽力想要帮助别人时，他完全忘记了自己的痛苦，他想要成为一名医生。


 


3.


Patch，缝缝补补。


医生，治病救人。


两者确实有共同之处。


如果可以将破碎的生命能量重新融合，让生命能量可以流动起来，这是不是就起到了治疗的作用？


Patch Adams 进入医学院读书。没想到，在这里，他的感悟和现实发生了一系列的碰撞。


 


碰撞一，医生到底要不要跟病人有情感链接？


医生如果太代入情感，在某种程度上来说，可能会促进了解病患的疾苦，但病人的痛苦也有可能因此深深的困扰医生，影响医生做出理智的判断和感觉。


但如果医生不去与病患链接，拒绝代入情感，那他也许可以根据事实做出理性的判断和处理，但他算是治愈了这个病患吗？身心一体，医生的职责到底是消除症状，还是重建健康？


Patch 积极的跟病患建立链接。教授查房，大家都在用医学术语分析病患所有的状况，分析病情可能的走向——如果病情再严重，病人有可能要截肢。但有谁去了解过躺在病床上的病患的感受呢？越听越害怕？觉得这些听不懂的术语像手术刀一样冰冷？


Patch 不同，他的提问让周围人侧目。他的问题是，这位病人叫什么名字？


大家都感觉奇怪，教授也是看了看病例，才知道病患叫作 Margareta。


Patch 朝病人打招呼：你好啊，Margarette。


有时候，是不是我们太过于重视实际的解决问题，而总会忘记最基本的问题。比如，去了解这个人的感受，在情感上跟她建立起链结？


 


碰撞二，病人到底需要的是什么？


作为病人，到底需要的是医生带来的轻松玩乐，还是危机时刻可以救命的技术？


医学院的规定，不到大三，学生们不允许接触病患。


但 Patch 并不这样想。他溜进医院，试着早早的接触病人，跟病号产生链接。


在儿科病房，他看到所有的孩子都安静躺在床上，一个个的小脸儿并不开心。


Patch 灵机一动，把橡胶吸水球轻轻一剪，戴在鼻头上，一秒钟变成红鼻子小丑。孩子们马上被吸引住注意力，接下来，Patch 使尽浑身解数，开始扮小丑！孩子们大笑不止，整个阴郁的病房像是变成了轻松玩乐的马戏团，每个孩子别提多开心！


Patch 这样实践着他的理念，尽可能多的让病人感觉开心、搞笑，至少可以让他们忘记眼前的痛苦。但这样的观点，真的有用吗？


电影安排了一场 Patch 和室友的争论。两个人的成绩都很出色，但医生世家出身的室友却看不上 Patch 四处扮小丑，逗乐病人的举动。室友认为，医院不是马戏团，在病人需要救命的时刻，他们不需要小丑，他们需要的是真正能治病救人的技术。


这个观念听上去有道理。但轻松和玩乐对于病人到底能有多大的好处呢？


在 Patch 看来，在病人生病的时候，往往是这个人最为脆弱不堪的时候，如果可以真的在这个时候打开心扉，与病人紧密相连，不仅仅提供治疗，也带来希望，或是提供一些建议，这也许就是最好的帮助。


 


4.


轻松玩乐和与别人产生链接。你看这就是 Patch 修补健康的方式。换句话说，如果我们希望人生充满活力，生命充满流动，这两点也是多么有效！


轻松玩乐是属于小丑的智慧，这意味着不管你是出糗、难堪，或者是做了多么出格的事儿，在小丑的逻辑里，这通通没有问题。而不再惧怕于他人看待我们的眼光，生命的紧张一经消除，身心放松，意识清明，在这个时候往往是人们有着最大创造力的时候，生命的活力在这种情况下往往会被激发出来，没有恐惧、没有担心，只有当下一刻的轻松、好玩，想一想都觉得好像嘴角忍不住的向上翘了起来呢。


与别人链接，说起来困难，但其实也容易。只需要一点，静下来，去了解别人的想法，去帮别人实现他想要做的事情，真的没那么困难。Patch 会从树上倒挂下来，向人行道上的老太太问好。老太太虽然吓了一小跳，但想了想这个陌生人这么有趣，还是忍不住回头打招呼。你看，有时候，跟别人打交道，也并不是那么难。只要可以多付出真心，真的没什么困难。


 


5.


今天的小练习，就让我们不妨实践一下从 Patch 哪里学习来的智慧吧。


先说说我自己练习的结果吧。


我每天早上会非常早到办公室，有时候趁着屋里没人，我会安静的做个动功。这是一套藏传佛教的锻炼身体的方法，经常锻炼，非常有益。


有一天早上，我在练习时，保洁阿姨刚好路过，她特别好奇，认真询问。我不由得动了念头，干脆教会她做了这一套操。第二天，另一个保洁阿姨也感兴趣，赶来也学了一遍。第三天，两位阿姨见到我亲热熟络了很多，她们跟我闲聊起家乡小吃的做法，还带了自己种的藿香叶子给我，嘱咐我做饭时放进去会很好吃。


一来二去，我们从陌生人变成了像朋友一样可以交流的人。我想，也许电影里 Patch 讲述的与人链接的智慧，真的没有那么难。


亲爱的听众朋友们，今天的小练习，就让我们试着去和身边的人努力去建立链接吧。也许是对小区里的人打个招呼，对常常碰面的人 say hi，也许只是随手可以帮助一下别人。这些看似不起眼的小事儿，会因为你的善意和好心而闪光。


或者，你也可以想一想，看看你的朋友们、同事们、同学们有没有什么一直想要实现的小愿望，也许你力所能及刚好能够实现，也许你只是碰巧记得对方的生日，多祝福那么一句。赠人玫瑰，手留余香。我想，做这件事的你，一定会感受到心口里有一份暖暖的感觉。真的不妨试一试啊！


 


好了，这期的节目就到这里， 我们下期再会。
