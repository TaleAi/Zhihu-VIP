## 简介
从心理学角度发掘电影的魅力，收获自我心智的成长。使用电影中蒙太奇技巧，发掘生活中的正能量的方法。唤醒身体和内心的力量， 化解压力，收获强大的内心。从根本上告别拖延，身心一体摆脱焦虑和失眠的困扰。


**对人生充满困惑，看不到希望怎么办?**


《阿甘正传》


告诉我们很多事情的成功是有其偶然因素的，它并不会因你的处心积虑而更接近终点, 却会在你不经意的时候，给你 的努力以一个最恰当的褒赏。


**生活重复单调，无力打破循环的怪圈怎么办?**


《土抜鼠日》


告诉我们真心听从爱的召唤，愿意通过改变 自己、改变自己面对生活的态度，才能真正 理解生命的宝贵，一切才真的改变。


**总是无法消除自己的拖延症, 情绪「死机」 了怎么办？**


《当幸福来敲门》


在电影中，我们看到了别样的人生，另类的活法，那里有你想成为的样子吗？


43 部电影，带你读懂自己的心理学密码，收获 43 个心理学奥秘。


跟随柠杨老师走进电影中的世界，用心理学冥想方法摆脱负面情绪，你的人生也将是一部精彩大片。
