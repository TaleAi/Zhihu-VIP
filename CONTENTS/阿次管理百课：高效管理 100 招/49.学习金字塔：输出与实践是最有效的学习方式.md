## 49.学习金字塔：输出与实践是最有效的学习方式
大家都知道在埃及有个很有名的胡夫金字塔，在学习领域也有一个很有名的理论叫做学习金字塔。这个理论的提出者是美国的一个学者，叫爱德加·戴尔，他用一个数字来显示，采用不同的学习方式，学习的人在两周以后还能记住内容有多少？


这个理论说的一共七种学习手段：


**第一种叫做听讲**，比如听演讲，听讲座这种，两周以后的留存率只有 5%。


**第二种叫做阅读**。阅读两周以后，留下来的只有 10%。


**第三种叫做视听**，就是看音频视频这种刺激你的，那大概两之后能留下 20%。


**第四种叫做演示**,你看到这个演示的过程，你能记住的可以达到 30%。


**第五种叫做讨论,**如果你亲自参与讨论了，那你大概能够记住的有 50%。


**第六种叫做实践**，也就说，把你听到的，学到的东西拿到实践中去用一遍，那这时候你能记住 75%。


**第七种叫做教别人**。教别人，效果最好，你可以获得 90%。


你会发现从听讲到最后教别人，留存率的数字从上到下的变化是从小到大，如同一个金字塔一样，所以这叫做学习金字塔。那这个学习金字塔对我们在培养人当中有什么样的意义呢？


理论上人们把前面四种，也就是听讲，阅读，视听和演示称之为被动学习，也就说你是个被动的接收者。而后面三种比如说讨论，实践和教别人称之为主动学习。如果用人力资源在培养当中的一个术语，前面叫做 Training（中文是培训），后面叫做 Learning（中文是学习），培训是被动的意思，学习是主动，显然主动的效果远远大于被动的效果。


为什么会有这样的一个原因呢？从心理学的观点来看，被动是由外而内的，是你要去改变他，让他去接受一个东西，他本身很可能会有抵触的。而主动是表明他是由内而外的改变。他自己想去了解一些东西，这必须是他感兴趣的，否则他不会主动，因此它的效果会好很多。


我有一个亲身经历，有一次一个朋友请我去分享「怎么打造个人 IP」？IP 全称 Intellectual Property 翻译成中文就是知识产权的意思，现在一般代指专属品牌的意思。其实我以前从来没有考虑这个问题。那他之所以请我去分享这个观点，是因为说我在 IP 的打造方面好像做的还可以。为了完成这样的一次演讲，我就去思考 IP 是怎么打造的？怎么叫才叫打造一个好的 IP？当他让我去跟别人分享的时候，其实按照金字塔原理来讲，就是你要去教别人怎么做这件事。所以为了教别人做这件事，我就开始思考。


当然思考是有很多的模式和套路可以采用的，比如我用一个最简单的模型叫做黄金圈法则。所谓黄金圈法则，就是任何事情你用三个词可以讲清楚，第一个叫 What（是什么），第二叫 Why（为什么），第三个叫 How（怎么做）。


用在这个例子上就是说，为了教别人，那我要去思考:


IP 就是一个定位，IP 就是你在人家心智模式中占据的那个位置，让大家都知道你。当然我们用品牌来形容就很清楚，就是一个品牌、一个标签，一个定义，人们看到你的名字的第一反应是什么，在你的领域第一个能想到你的名字，这就叫做 IP。


我们说在这个世界上，在这个 VUCA 时代，人们往往只能记住第一，不能记住第二。这个第一就是一个 IP 的概念，就像你一定知道，世界上最高的楼是迪拜塔，但你很可能记不住排第二的是哪个？所以你对世界冠军记得很清楚，但是世界冠军后面的拿银牌的这个人往往就被人家所忽视。就像每次奥运会，中国拿首金的人一定是要刮出一阵旋风出来的。那道理就是这样，所以这是 Why？因为在注意力经济时代，人们的注意力是有限的，人们只能为那个唯一或者第一所吸引住，就像你只会知道世界最高峰是珠穆朗玛峰，但你一定记不住，排第二的山峰是哪个，知道乔戈里峰的人寥寥无几。


其实为了去分享，我就思考重点是怎么打造呢？我自己总结了四个步骤：


第一你的定位是什么？IP 是要和个人职业定位有关的，如同做事情要以终为始那样，所以想清楚你的最终定位是关键，打算做讲师的就要定位在名师上，打算做企业的就要把自己定位在企业家上，此时讲课就不是重点了。定位到最后就是你的标签是什么？


第二用什么样的渠道？不同渠道的受众是不一样的，效率也不一样，抖音和微信、QQ 和直播也不一样，做严肃理论派和时尚流行派的方法不一样，所使用的渠道当然也不一样。


第三怎么去传播？你是用写作还是用演讲的方式？善于写的人不一定善于讲，反之亦然，所以使用你擅长的传播方式很重要。这是传播的概念。


第四如何去维护。如何去确保你有持续的输出？所以你的学习能力很重要等等。所以这就是 How。


好了，当我为了去分享这样的东西，我一定会去做这样思考，那你有没有发现，做这样的思考恰恰就是你最佳的学习手段。这远比我听人家讲一遍，什么是 IP 的打造，给我的收获更大。这是很典型的例子，叫 In teaching others we learn the most。中文是在教别人的时候我们学的最多，这也非常符合中国人说的**「台上一分钟，台下十年功」**。许多外企，为了让管理者更快的成长，于是就有了内部讲师的要求，让管理者成为讲师，目的不只是为了去教别人，而是这个过程真的会让管理者去学很多很多的东西，而这种学习属于主动学习，属于有目的的学习，效果非常好。所以这就是为什么我们说这个**知识本身不重要，重要的是你掌握知识的方式和手段。**


1、不同的学习方式，效果不一样，最好的方法是教别人；


2、 教别人知识的最简单逻辑是黄金圈法则：What、Why、How；


3、企业中要让管理者成为老师，对于学员和管理者本身都是很好的事情。
