## 简介
8 个短篇互动文字故事，均以第二人称「你」作为视角，通过选择 A、B 不同的选项，你将跟随文字走向不同的结局。这一次，你，不仅仅是阅读者，更是主角宿命的创造者。


女孩总是在梦里梦到同一个男孩，直到有一天梦里的男孩问女孩：「你愿意在现实里见到我吗？」女孩同意了。第二天女孩参加早就定好的相亲，发现相亲对象跟梦里的男孩长得一模一样。


女孩被索要电话号码，你会选择 **A 拒绝给电话号码**，还是 **B 把电话号码给他**？


**选择 A：**拒绝给电话号码后，发现男孩点的饭菜都是女孩喜欢的，她越发好奇，问男孩之前认识自己吗？男孩说「这是你第 50 次问我这个问题」，接着女孩就被男孩用匕首捅死了。


**选择 B：**男孩说自己被拒绝了五十多次，女孩笑着说不相信，这顿饭快乐地结束了。后来偶然在新闻里看到一个连环杀人案，死者都是二十多岁的女性，而且长得一模一样，而通缉犯就是相亲见到的男孩。女孩回家，发现男孩在家等着自己。


你将进行第二次选择，**A 拿起棒球棍砸向男孩**，**B 选择听男孩解释。**


**这一次，你又将作何选择？女孩究竟为什么被杀死？女孩的最终命运将走向何处？**


 


你将在这里看到 8 个像这样反转再反转的故事，也将通过一次次选择，找到 8 个主人公背后的秘密。而你，就是他们命运背后看不见的手。 


  



**故事开始了，请做出你的选择。**
