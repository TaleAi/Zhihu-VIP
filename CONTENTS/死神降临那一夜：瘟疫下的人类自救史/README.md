## 简介
瘟疫改变了世界，改变了人的生活方式。人类是如何从死神的阴影下幸存的？让我们回顾历史上「杀人无数」的瘟疫，在病毒的肆虐中见证生命的脆弱与顽强。在这里，细数令人闻风丧胆的致命流行病，用科学的眼光解剖一次次改变历史的病毒。


**瘟疫，看似与健康的人类很遥远，但其实，它与你我如影随形。**


病菌对人类的威胁由来已久，而当一种新病原在人群中出现，易感染人并引起严重病况的时候，往往会暴发大流行，造成死伤惨重。十四世纪欧洲的黑死病，就是代表性的例子。


传染病与人类的历史也是息息相关的。传染病造成的人口大量损失，可能会扭转正在进行中的战争局势，也会带来政治、社会、宗教的变革。


我们将会详实地探讨鼠疫、天花、大流感、疟疾等等疾病在全世界范围的肆虐情形，揭示出一个个显微镜下的病毒，描绘出一幕幕传染病史的图画，梳理出一桩桩人类对抗病毒而做出的努力。


**【引发欧洲历史转折的大瘟疫：黑死病】**


它是怎样改变欧洲历史的？鼠疫杆菌是如何进化得有极强的传染性和致死率的？宗教也受到了重创，人们在黑色死亡瘟疫中感受到了前所未有的绝望。


![img](https://pic4.zhimg.com/v2-fccb5b665ab2aa6d6affc2fa9e47f98a.png)

**【历史上杀人最多的传染病：西班牙大流感】**


大流感是如何影响正在打响的第一次世界大战的？在这里你会看到似曾相识，却并不高明的防疫手段。


![img](https://pic2.zhimg.com/v2-c0c0386881cb7e82db2fcf7be1cd291b.png)

**【人造的瘟疫：生物武器是怎么来的】**


古代的人虽然不清楚传染病是由什么引起的，但知道它是可以在人和人之间传播的，并使它变成了一种武器：最早的有记载的生物武器大概在公元前 1200-1500 年前，赫梯人驱赶了患有咯血热的人进入敌人的领地，造成了当地的瘟疫流行。人类使用这些致病微生物当作武器的历史已然悠久，而在生物技术突飞猛进的今天，我们又应该如何应对它们呢？


![img](https://pic4.zhimg.com/v2-e0fa2f1009876e90090294ea8cf370ea.png)

**【世界历史的幕后推手：天花】**


公元前 430 年的雅典、公元 1 世纪东汉时期的中国、公元 2 世纪的罗马、1519 年的美洲大陆、1644 年的明朝覆灭、1775 年的美国独立战争、1849 年的美国内战······在这些朝代更替或者战争中，**天花**都起到了「幕后黑手」的作用。


![img](https://pic2.zhimg.com/v2-04fc5dfb492b140612f8d7943d37c46d.png)

**【人类千万年来的「老朋友」：疟疾】**


镰刀形贫血症一度被医学研究者认为是一种「先天性缺陷」，然而很奇特的是在撒哈拉以南的非洲人群中分布着大量的这个基因，按理说早该在漫长的进化中被淘汰，为何还会保留下来呢？那就要从人类千万年来的老朋友——**疟疾**说起了······


![img](https://pic2.zhimg.com/v2-aabe425d9342e380956eb70323666ea0.png)

**【人类疫情防卫战：公共卫生】**


从古罗马时期的公共澡堂、下水道，到现代医学的疫苗、各种疾控系统，公共卫生的变得越来越重要，它成为了人民健康生活的守护者。**人类疫情的防卫战**中很重要的一环，**公共卫生**是怎样进化到今天的状态的？


![img](https://pic1.zhimg.com/v2-47ef030ce256a6790719f6a81bf2dda7.png)

**【烈性杀人魔：埃博拉病毒】**


**埃博拉病毒**早在 1976 年就导致过非洲地区整个村子的覆灭；在 2013 年的一次爆发中，埃博拉杀死了大约 11000 人；而就在 2019 年也爆发过疫情。一般过于烈性的病毒会导致宿主死亡而停止传播，病毒也就消失了，究竟是什么导致埃博拉能够卷土重来这么多次呢？


![img](https://pic2.zhimg.com/v2-abf050623e315d76f19b4ad5f8d3c5cf.png)

**【他的咳嗽中带着无限的优雅：痨病】**


得了**痨病**的人面色苍白，柔弱的样子被人们认为是「上流的」「敏感的」——就像豌豆公主或者是「林妹妹」一样。有人形容患病的肖邦「他的咳嗽中带着无限的优雅」。


![img](https://pic3.zhimg.com/v2-2c31f1eec7e69d40f70a2b3cbb5b71c2.png)

**【不速之客：冠状病毒】**


**冠状病毒**家族一直以来默默无闻，但却在 2003 年非典时期变得极其致命。这次，深度剖析冠状病毒这个大家族。


![img](https://pic2.zhimg.com/v2-0e49c0311d1b89e0ad220aebf4096c02.png)

**【药物和疫苗的开发】**


病毒的变异很快，这导致人类总是落后病毒一步。不过在人类和病毒的「军备竞赛」中，人类也变得越来越强，能通过各种方式很快的制造出相应的对抗手段。


![img](https://pic2.zhimg.com/v2-5f9cee8a19dfa58da8603486de7f1707.png)

**【瘟疫医生及治疗设备手段】**


生活在过去的人们使用过哪些方式来对抗瘟疫？相关医用设备、防护服装、治疗手段，哪些是有用的？哪些是没用的？哪些是用了反而更糟糕的？


![img](https://pic3.zhimg.com/v2-0fc0f66933d244d0e7ed9962f6df1659.png)

**【瘟疫与艺术的转变】**


当未知的疑云笼罩人心，恐惧就伴随着死亡而来。在人们不知道瘟疫到底是由什么造成的时期，瘟疫相关的**艺术**作品也就从单纯的宗教式的歌功颂德变成了描绘人间炼狱的镜子。


![img](https://pic3.zhimg.com/v2-5cfae12d7718a1fbdabf07c6d633aa26.png)
