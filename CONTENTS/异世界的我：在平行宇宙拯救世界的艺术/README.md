## 简介
你是否相信平行宇宙里还有另一个你？


我失恋了，从接到前男友结婚请柬的那一刻开始的。


我心里有万般不甘，我好想时空倒转，回到汶川地震那一天。那一天，是他跟请柬上这个绿茶女定情的日子。我记得所有人都在操场上，我在他身旁又作又闹，而她，却拿着吉他给他唱了一首歌……


要脸么？！这是老娘的男朋友啊……


等等！我睁开眼，发现自己竟然在大学寝室？？我，穿越成功了？


...


我 13 岁，是个不起眼的普通学渣。最近我被一个骗子女跟踪，她非说自己是未来的我，还说我害死了她妈。


她妈就是我妈，我校金牌奥数教练，一个终日给学生辅导竞赛题，却从未记住过我生日的女人。


「咱妈将死于一场车祸，在她带队去市里参加数学竞赛的路上。」


「你还愣着干嘛！还不快救。」


「这是我第 1000 次穿越，前 999 次救妈计划，全都失败了……」


...


各位可爱的小读者们，准备好与另一个你，相遇了吗？
