在这个专栏，也许你会找到这些问题的答案。


从起步到兴盛，在不到 30 年的时间里，互联网就彻底改变了我们的生活方式。除了衣食住行的生理需求外，我们的精神需求乃至思考方式也发生了天翻地覆的变化。


**网络社交**替代了现实的语言交流，但人类之间的社交法则没有改变，无论是网恋还是网暴，无论是朋友圈人设塑造还是网红直播打赏，**传播学可以解释一切人与人、人与社会之间的关系，教我们突破网络的桎梏，看到人类社会的本质**。


如果你是一名爱思考的重度网络冲浪者，对朋友圈人设、网络谣言、手机焦虑等问题产生过困惑，那么本专栏不仅可以为你提供一份网络冲浪指南，还能帮助你透过与生活息息相关的日常话题，洞察热点事件背后的原因，用社会科学研究的态度思考，并敏锐地捕捉信息。


如果你是一名互联网从业者，在本专栏你将收获用传播学的视角，来探索虚拟世界里人与人关系的方法，这可能会帮助你在营销设计或者文案创作上得到新的启发，获得独立思考的方法。
