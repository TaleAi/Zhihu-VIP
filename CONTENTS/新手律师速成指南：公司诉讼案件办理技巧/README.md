## 简介
本课程由中闻律师所的资深律师李斌律师主讲，为年轻律师系统讲解形式特殊，专业难度高的公司诉讼案件诉讼技巧。帮新手律师提升专业水准，轻松迎战公司诉讼难题。


由于经济环境的不断变化，国家经济实力增强，相关政策法规的出台，使得公司如雨后春笋般成立、生长，涉及公司法的各方面纠纷也不断增多。


公司诉讼案件还有一个特点，就是标的额大，少则几十上百万多则数十亿的交易，案件所涉标的额远大于普通民事案件。


案件量大、标的额大，这对律师来说，是利好也是挑战。


但公司诉讼案件在专业性上难度很大，许多律师在专业上很难达标。


为了快速提升律师的相关能力，本课程邀请到邀请到中闻律师所的资深律师李斌律师主讲本课程，为年轻律师系统讲解公司法诉讼的特殊性和高难度，提升办理公司诉讼案件的水准。
