## 简介
青春有我


活了二十多年，我从未想到自己竟然会被变态缠上。


当我收到一条「我喜欢你」的短信的时候，我还窃喜了一下，我的妈呀，我被暗恋了？


但后来，短信的内容就完全不可描述了。


拿着手机看着满屏的骚言骚语的时候，我悟了，这根本不是什么都市爱恋，是我被变态盯上了。


——《死变态，我来教你做人》


  



「姐姐，好久不见。」


「你谁啊？！」


「我也想喝酒。」


「小孩子喝什么酒！」


「我都成年了……」


——《弟弟，别逗姐姐了》


  



我是一个被驾校教练虐了还不敢反抗的人。


我本人是个性格很弱的女生，上学的时候被欺负了都不会还手的那种。


最多心里默念一句袁华儿的台词——「我迟早会回来的」。


因为他，我这个暑假别说拿驾照了，我连做个人都不配了。


——《教练，离我远点儿》
