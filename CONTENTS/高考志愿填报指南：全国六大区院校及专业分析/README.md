## 简介
本专栏将从多方面入手，具体讲解在填报志愿时的各类注意事项，指导考生及家长详尽了解各种报考知识，填报志愿不再犯难。


夜以继日的刻苦学习是高考成功的基础，而科学合理的填报高考志愿，则是高考成功的保证。七分考，三分报，每一个考生都需要用 12 年时间努力拼搏，再用 7 天的时间填报志愿，而这个决定将影响他未来几十年的人生走向。


高考，考得好更要报得好，如何用填报志愿的手段为成绩加分以及在填报志愿的时候不踏入雷区，这是一项系统工程。


本专栏将从我国大学的分布区域、招生录取的专业类型以及志愿填报的规则、方法、技巧等多方面入手，具体讲解在填报志愿时，我们必须掌握的各大院校信息，各种专业的特点，指导考生及家长详尽了解各种报考知识，以志愿填报宝典的形式，分享各方面信息，惠及千万考生。
