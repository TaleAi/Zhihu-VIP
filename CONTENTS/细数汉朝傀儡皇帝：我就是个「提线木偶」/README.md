## 简介
汉朝后期发生了戚宦之争与党锢之祸，导致朝臣专权，皇帝无能，由此也产生了许多「提线木偶」般的傀儡皇帝。帝位之下，他们的命运各不相同。让我们走进本专栏，看看他们的一生。


皇帝，贵为一朝之君，权倾天下，他们也会被人操控吗？


是的，在皇帝—权力—大臣的政治格局中,权力就像是一台天平的指针。指针偏左,可能会出现独裁皇帝;指针偏右,多半要产生傀儡皇帝。而傀儡皇帝就是那些空有皇帝之名，却没有实权的皇帝。


作为中国历史上第一个疆土辽阔，武力强悍而又四方来贺的大一统王朝，汉朝常常被人追思。前有文景之治，汉武盛世。后有光武中兴，文化统一、科技发达。但就是在这样一个朝代的后期却发生了戚宦之争与党锢之祸，导致朝臣专权，皇帝无能。由此也产生了许多「提线木偶」般的傀儡皇帝。


汉惠帝刘盈，本身也是很有才华，但是性格懦弱，一生都被牢牢掌握在自己的母亲吕后手中。


汉平帝刘衎，即位时不过九岁，当时外戚专权，大司马王莽独掌朝政，生怕小皇帝成长起来，还早早毒死了小皇帝。


孺子刘婴，两岁即位就开始了悲剧的一生。


......


这些傀儡皇帝被人挟持的有之，被人废黜的有之，被人砍头的有之……帝位之下，他们的命运各不相同。让我们走进本专栏，看看被操控之下他们的一生。
