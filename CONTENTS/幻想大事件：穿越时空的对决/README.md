**《白龙之叛》**


我叫敖烈，名字已经没几个人记得。


世人只唤我作白龙马。


师徒本来一行五人，可说起取经人，从来只有四个。


他们说的对，我心有异，度不得劫难，封不得佛。


我恨他们，没来由的那种。


**《又打白骨精》**


「齐天大圣，帮我杀妖怪！」


城市马路旁，一个小学生叫住了我。


我搔搔头，眨眨火眼金睛。


……


后来，我赢了，也输了。


但世间总归会有齐天大圣在的。
