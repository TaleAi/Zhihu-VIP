## 简介
以牙还牙，百倍奉还！



遇到令人崩溃的怎么办？当然是给他一拳啊！


如果有得选，生活中的每个人都想成为体面人，但生活往往并未提供这种可能。


一次又一次的吃闷亏、被欺辱之后，年轻人终于长大，他们学会了反击。


本专栏带来一组故事，选取最真实的案例，直击 20-30 岁青年女性的生活状态，从高中走向大学，从校园迈入社会，他们无论是在生活还是职场上，都有最多的问题和困惑，也面临着最多被「体面」困住的挑战。


抛弃体面，他们用成人的法则解决问题。


管那么多呢？干就完了！
