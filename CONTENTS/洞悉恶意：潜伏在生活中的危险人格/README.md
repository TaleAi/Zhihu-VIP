## 简介
识别危险「人格」，有效避免和远离伤害


你是否想知道：  

*惊世案件的犯罪人究竟有着什么样的性格？  

为什么有的人一门心思就想报复社会？  

什么样的人更容易犯罪？*


  

你是否也担心**我们离恶的距离其实很近**：  

*只是正常社交，却被熟人伤害；  

只是喝了一瓶饮料，却被莫名其妙地毒害；  

小孩只是去上个学，就被无情夺去生命... ...*  




既然犯罪离我们那么近，就需要我们深入理解犯罪心理，**识别危险「人格」，才能有效避免和远离伤害**。


  

本专栏由中国政法大学犯罪心理学博士（在读）十方迦南精心挑选 8 大惊世案件，从全新科学视角，带你重新审视案件全过程，推敲罪犯人格特质，从而避开潜在风险和套路。
