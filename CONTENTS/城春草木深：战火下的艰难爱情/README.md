## 简介
艰难岁月里，吾定当，不负家国不负卿


这是一个讲述战争爱情的故事专栏，故事里的她们，是战争中的平凡女性。


苦难深重的城市里，她们的爱情在硝烟中遭遇困境。


努力追寻爱情的同时，她们也在为抗战，为救国不懈努力。


专栏呈现给我们的，不仅是战火中的动人爱情，还有硝烟里女性的觉醒与成长。


**锦瑟｜长沙保卫战**


民国 27 年的长沙，我被军阀顾秀山抢来做了四姨太。


其实，我是来监视他的军统特工。


为了完成任务，我努力和姨太太团相处，


最后却发现，她们竟然全员卧底？！


而我的丈夫，也一早就知道我的特工身份？！


硝烟起，嗟险阻，叹飘零。命运浮沉，湖湘如何坚守？


**未有期｜重庆大轰炸**


我和赵渝宁青梅竹马，少有婚约。


一场猝不及防的轰炸，他逃婚从军抗日。


我沿着他的足迹一路出川。


见过他激战过的战壕，遇过他帮助过的战友，也遭遇过鬼子散兵。


乱世人海中，我该怎么把他找回？


日寇驱尽，战事又起，回家的路究竟还有多远？
