# 上枝头：盲女被权臣囚禁以后

### 作者：吧唧

### 分类：古代言情

### 状态：已完结，共 16 节

### 存档日期：2022-06-12

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1403729470973460480


## 简介
我只想做枝头自由的鸟，他却要剪掉我的脚。


被有着变态占有欲的权臣囚在了府中怎么办？当然是跑！


但作为一个无依无靠的青楼盲女，我连跑都跑不了，只能投其所好，哄他欢心，编织出让他步步沦陷的温柔网，再逃出他的掌控。


然而，随着时间的推移，我却慢慢被卷入更大的阴谋。


他控制我，却又保护我；寸步不让，却肯为我而死。


他到底是天性残暴，还是有难言之隐？


温润哥哥、刁蛮妹妹，他们到底是受害者，还是披着羊皮的狼？


我的家人、朋友，又是为何而死？


随着真相一点点浮出水面，我才惊觉，原来我所看到的一切，远远不止我想象的那么简单……
