## 简介
生活中的错位，大多数人都能一眼看出来，但是在这个专栏里，这些错位又显得这么合理，恰好发生。殊不知，暗地里有人正在实施狠毒的计划，因为鱼儿刚刚上钩。


一个儿子失踪多年的家庭，一个年龄相仿的男孩，双方在派出所相认了。本是一件开心事，但是大家都明白，对方在说谎。


——《你是谁》


  



我爸爸在家里被杀，警察竟然怀疑我是凶手，因为刀上有我的指纹，以及我恰好没有不在场的证明。


难道我杀了我爸？


——《你怕黑吗》
