## 简介
十年前，我目睹初恋女友被人凌虐后跳楼身亡；十年后，我重回悲剧发生之前，只为可以再次见到你；致命的穿越之旅、被献祭的少女、死而复生的爱人……纵使这是一场有去无回的旅行，我只想不顾一切地去见你。


2017 年，人造虫洞被发明，人类实现时空穿越。刑警孙愚接到一个特殊的军事行动任务：暗杀人造虫洞之父周海峰，以阻止各国争端，维系和平。


然而，周海峰失踪多年，最后一次出现是在 2007 年，孙愚需要使用穿越通道回到过去完成暗杀任务；


此次任务极其凶险，先前派去的三人均已殉职，孙愚被特批，完成任务后可以小范围改变历史——阻止十年前初恋女友的悲剧……


穿越回到悲剧发生之前，暗杀计划和拯救爱人需同时完成，这是一场几乎不能完成的极限任务，但终点是你，我便没有了回头的余地。


被献祭的少女、死而复生的爱人、致命的时空穿越之旅、错综复杂的权力游戏……这是一个暗黑、烧脑的爱情故事，也是一次关于过去的救赎，人生有很多「来不及」，如果可以，我一定会毫不犹豫奔向你。
