## 简介
了解千古良相魏征的传奇人生，品味铁骨诤臣的官场生存之道。


曾是一介死囚，却能逆流而上，成为帝王身边最「得力」的大臣，他是如何做到的？


殚精竭虑辅佐君王，实施仁政，直言敢谏，为贞观之治打下盛世基础……他究竟是谁呢？


没错，他就是唐朝著名诤臣——魏征，生前是政治家、文学家、史学家，死后被奉为「凌烟阁二十四功臣」之一。


作为臣子的魏征，甚至还和唐太宗李世民，传出了「明君贤臣」的传奇佳话。


魏征去世后，唐太宗李世民悲痛不已，留下一段千古名言：


「以铜为镜，可以正衣冠；以古为镜，可以知兴替；以人为镜，可以明得失。今魏征殂逝，遂亡一镜矣。」


本专栏带你了解千古良相魏征的传奇人生，品味铁骨诤臣的官场生存之道，深度剖析魏征的为臣之道与官场高升的谏言技巧，让你掌握古代「偶像」实现人生逆袭的秘诀。
