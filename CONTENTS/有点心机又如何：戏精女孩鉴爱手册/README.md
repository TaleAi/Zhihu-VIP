**《温床入夜》**


和我相恋五年的男友因为一段短视频爆火成了网络达人。


随即而来的是一份分手大礼，他转手就把小三带进了我们刚刚一起花钱买的新房子。


就在我对未来充满无限向往期待着他会以什么样的方式向我求婚的时候，他亲手把我送进地狱。


我听见他用那张曾经说着爱我的嘴，冷冷开口。


「现在的你配不上我。」


「继续和你在一起是我的恩赐。」


「开个价。」


那一瞬间我才明白，我的五年喂了狗，不折不扣的狗。


  



**《持色诱鱼》**


我被人挂在了表白墙上。


匿名的人扬言要揭穿所谓校园女神的真面目，说我是渣女整容怪，是绿茶小三，勾引她的忠犬男友，顺带附上生活照、聊天记录，一番荡妇羞辱极尽恶毒。


聊天记录是伪造的，生活照是我上公共课的时候偷拍的，衣服发型都是我，只是上面的脸扭曲变形看起来像极了整容网红脸。


我往下翻着评论，心里逐渐沉到了谷底。


［我就说池鱼不是什么好女孩，之前我加她微信压根不搭理还以为多高冷。］


［经管那位院花传闻多好看，生活照不过如此，丑八怪整容脸。］


［有一说一，身材不错脸干嘛打码求视频。］


……


一时之间好像世界都对我极尽羞辱，手机里微信的消息疯狂 99 加，大半都是来八卦的，其中还不乏落井下石的人。


室友们看向手机又看向我，大家表情尴尬神色各异。


我撇了撇镜子里的一张脸，巴掌大冷白皮又精致又自然状似不在意地笑了笑，眼眶却已经红了。


欲加之罪何患无辞。


只是我也不是什么怕事的。
