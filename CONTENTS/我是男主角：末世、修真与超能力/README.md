## 简介
本专栏将从末世、修真、超能力关键词出发，讲述以男孩子为绝对主角的故事。在这里，你将见证主人公的超凡实力，英雄的成长之路，以及别出心裁的脑洞，拍案惊奇的设定。


【末日】降临，是时候展现我真正的实力了，在这个弱肉强食的世界里，我要变得更强。


【修真】很难，尤其是在腥风血雨的江湖中，躲得过明枪暗箭，躲不过歹毒人心。


【超能力】是把双刃剑，有时它让人自带主角光环，有时它把你推进深渊。


**我来自英国伦敦，我叫傅满洲**


傅满洲是盎格鲁萨克逊人和东方人的后代，他的父亲在利用了犹如「东方精灵」一般的母亲，并造出了傅满洲后，便用计杀死了她。傅满洲长大后，因武力值太高，而被父亲刮目相看。面对弑母的父亲，他会做出怎样的抉择？这是一场东方血统和蓝血之间的较量。


**我不欲成仙**


穷小子石铁柱机缘巧合拜入玉宁观，成为清虚真人门下最受宠爱的徒弟。石铁柱与大师兄一并下山降服狐妖，没想到却被狐妖耍得团团转。本来貌似简单的降妖之路陡增许多波澜：这狐妖变化多端，又擅揣度人心，怎样确保抓到他？师兄对抢走师父关注和未来掌门之位的师弟，又是真的别无二心吗？清虚真人的宠爱下，又藏着什么道不明的秘密？傻小子石铁柱，最终能化险为夷吗？


**世外之时**


我叫郑霖，是个普普通通的医学狗，一觉醒来却发现自己穿越到了一个陌生的宿舍，同宿舍的六个人皆是从不同的末世穿越而来，有的是机器人世界，有的是丧尸世界，有的是修真世界……而且，这六个人都是所在世界的位面之子，拥有高超技能，我呢，却是个毫无能力的普通人。是谁把我们召唤到这里，我们又该如何回到各自的世界？
