# 鲜衣怒马少年时：唐宋诗人的诗酒江湖

## 简介
大女主体质李清照、全能天才辛弃疾、 千古情圣杜甫……三十位诗词大神、三百首诗词经典，专栏另类解读名篇背后的精彩故事，重逢唐宋诗酒江湖，再遇才子佳人，一起品读千古名篇，发现不同角度的诗意与文学之美。


唐诗宋词是汉字之美的巅峰。 一首首名篇背后有怎样的故事？一个个大咖如何快意恩仇、神采飞扬？换个角度读诗词，会发现不同的诗意。 


超级愤青骆宾王、千古情圣杜甫、逆袭牛人高适、完美男神王维、 文艺大叔孟浩然、花间浪子温庭筠、大女主体质李清照、全能天才辛弃疾…… 三十位诗词大神，涉及近三百首诗词经典。


全书考据精到，融诗、史于一体，串起唐宋文坛一个个耀眼的瞬间， 从中可以窥见一部起伏跌宕的热血历史。 这本读起来非常过瘾的「诗词小说」，是一个以前没见过的文学江湖。


 唐诗宋词骨灰级爱好者「少年怒马」的致敬之作， 随便翻一页，读就对了。
