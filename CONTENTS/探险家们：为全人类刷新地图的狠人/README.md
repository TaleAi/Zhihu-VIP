## 简介
人类的本能是探索未知。


每个时代，都有将生死置之度外的狠人，去挑战人类活动的极限。


他们，有的震撼人心；有的奇葩无厘头；有的让人咬牙切齿……


第一位环球旅行的中国人，用脚步丈量世界，用行动征服各国元首，一个可以登上教科书的中国青年榜样，却被历史的尘埃掩盖。


最离谱的探险家，在北极用大便做刀、在希特勒眼皮子下越狱，拿了奥斯卡奖，安稳活了 71 年。


最让人气愤的探险家，以探险之名，行间谍之实，屠杀中国百姓。


请带上勇气、果敢、必胜的决心和一双慧眼，本专栏将带你一起探索未知的世界。
