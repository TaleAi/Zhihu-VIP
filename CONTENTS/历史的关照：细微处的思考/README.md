## 简介
本专栏是王丹誉多年来，在中央报刊发表历史随笔 300 多篇中精选的 70 篇精品力作，分为读书治学、修身养性、官德善政、清廉公正、制度得失等五部分，结合历史随笔的写作对中国传统文化进行深入研究，受到学术界乃至社会各界的广泛关注。


人们往往喜欢从历史的教训的角度来反观历史，很少从关照的视角来亲切地解读和认识历史。


然而，每位历史人物、每个历史事件、每位历史学家，甚至是历史教训本身，对于我们来说都是历史的关照。


位高权重的李林甫为何「压力山大」?「终南捷径」为何比买官卖官，跑官要官更可耻?太守刘宠为何收了百姓的钱，却留下千秋美名?唐玄宗「夜开禁门」为何会引发天下大乱……其中，许多是千古未解之谜，有的则成为史学公案。


无论如何，后人在读史、研究的过程中，都会受到前人和史册的多方关照。本专栏，丹誉老师带来 75 篇历史随笔，与你一同深入研究传统文化与历史人物。
