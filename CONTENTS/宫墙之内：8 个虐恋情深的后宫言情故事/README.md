## 简介
为什么后宫中嫔妃们一定要争宠？


后宫争宠也许是为了得到帝王的爱，也许是为了攀上权力的宝座，而有的不过是人在深宫，身不由己。


如果你爱看古代言情、宫斗戏码、虐恋情深，这个专栏太适合你了。


别担心，这里不只有虐文，还有小甜文，不过也有糖中混着玻璃渣的，小心食用。


这个专栏收录了 8 个后宫言情故事，每个故事的背后都是一个宫廷女人的一生，或欢喜，或悲凉。


《独占》：在边陲长大的她进宫后，为了赶走不爱的皇帝，跟他打了一个赌。在这场赌局里，她渐渐芳心暗许，两情相悦本该幸福一生，无奈命运弄人。


《玉树后庭花》：姐姐死后，她虽然心有所属，还是奉旨嫁给了她的姐夫，当朝的太子殿下。她挂念着自己的心上人，也顾忌姐姐生前与太子的琴瑟和鸣。殊不知，所有人都不过是爱而不得。


……


一场场精心策划的阴谋，一段段至死方休的爱恨，在宫墙之内上演。
