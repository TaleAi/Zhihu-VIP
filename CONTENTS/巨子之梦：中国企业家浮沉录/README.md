## 简介
起与落，舍与得，家与国。


1978 年，中国改革开放开始；1992 年，中国市场经济体制改革目标树立。


自那时至今，30 年以来，中国出现了一种新的社会身份——**企业家**。其中佼佼者，成为知名人士、财富英雄、青年偶像。


褚时健，带领玉溪卷烟厂成为亚洲最大烟草企业，却因为时代的原因身陷囹圄，背上贪污罪名。家遭变故、妻子入狱、女儿自杀，他竟在古稀之年变换赛道，重新创业，创立褚橙。


王石，把万科带大，带到国内第一的高度，却甘于职业经理人的身份，不求占有。当王石开始寄情山水的时候，他见识到了自然之伟大、人类之渺小。他决定再放弃一些东西。


当联想失去往日光彩，举步维艰的时候，柳传志扶持亲女儿与侄女儿，分别成长为滴滴和字节跳动的核心高管，柳家依然是商界高门，但近年来被舆论反噬的柳传志和柳家的命运究竟何去何从？


曹德旺，中国最复杂、最矛盾的企业家。他是如何为美国人带来工作机会，如何瓦解美国工人成立工会的计划的？他让子女坚持中国国籍，又砸大钱做慈善的用意是什么？


世人皆知格力的董明珠，却忘了奠基人是朱江洪，也忘了是朱江洪选择了董明珠，并把格力托付给了董明珠。如今，董明珠也到了要找接班人的时候，她能有朱江洪那样的大智慧吗？


……


中国商业巨子的产生、发展与壮大，伴随着中国经济的腾飞，他们的兴衰，也体现了中国改革开放的成败。


是非功过，如何评说？
