## 简介
关于青春、爱情、亲情、人性，浮在海面上的是孤岛，海面之下的是人心。


你以为的帅气爱人，他潇洒体贴，温柔多金，把你捧在手心？


——其实，他只是在觊觎你的身体，等待你心甘情愿地掏出自己的心；


你以为的叛逆少女，她大大咧咧，活泼可爱，和你许下了青涩的约定？


——其实，她的生活早已一滩泥泞，爱上你是她做过最勇敢的事情；


你以为的模范老公，他陪你熬过七年之痒，因为爱你放弃了生育，成为丁克？


——其实，他和同事酒后乱性惹了风流债，小三带着孩子来逼婚......


你以为的，只是你以为。


那些猜不出结局的反转故事，带你体验过山车般的刺激人性。
