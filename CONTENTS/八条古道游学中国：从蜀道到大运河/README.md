## 简介
山川河流、日月风雨、祖国大地、人文历史，我们身边的一切，都和地理密切相关。本专栏从「道路」切入，这些路的「年龄」都很大，跨越千年，蕴有丰富的人文历史内涵。让我们一起边走边讲，沿途风景，历史故事，切身感知人文地理。


中国的整体地貌是什么样子？


一些独特的城市是怎么发展起来的？


各个地域的文化又是怎么形成的？


本专栏涉及八条不可错过的「道路」，讲师耿朔大都亲自沿路走过、深入调查：8 条古道、60+景点、3000 年历史，还有沿路的城市、文化、古工程。他将结合自己的亲身经历、真实感受，为大家娓娓道来，附上亲自拍摄的照片与独家手绘地图，一目了然，展现完整、真实的古道图景。


这些道路，有的至今仍然可以行走，驴友时常出没，我们无需出门，便能一起感受「力崩山摧壮士死」的天险蜀道，看看世界上最长、直接打通南北的人工运河......切身感知人文地理。
