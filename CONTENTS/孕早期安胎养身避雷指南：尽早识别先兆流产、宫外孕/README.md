## 简介
本课程由上海市第一妇婴保健院产科副主任医师赵莉主讲，教你应对孕早期可能遇到的各类安全健康问题，宝宝健康，妈妈安全。


孕早期对各位新妈妈来说是最重要的极端。


一方面很多新妈妈们缺乏孕期的经验。


另一方面，孕早期虽然不显怀，但是孕吐往往很严重。


随着工业环境的恶化，很多新妈妈容易出现孕酮低，血 HCG 也低的状况。


这些问题该如何处理呢？


 


⽤ 本课程由从事妇产科临床工作 20 余年的赵莉大夫主讲，教你尽早识别先兆流产、宫外孕等诸多孕早期问题，生出健康宝宝。
