## 简介
穿越这件事，真的会上瘾！


叶满这孩子，打小就聪明。


所以她被系统选中穿书，成了里头的虐心女主。


叶满咬牙切齿，「为什么偏偏是我？」


系统：「嘻嘻嘻，缘分咯！」


他还说，只要走完剧情完成任务，她就能回去。


叶满一心想着回去，却没料到自己会爱上那个狂霸酷炫拽的男主。


等她精疲力竭完成任务，系统又说，男主的真实身份是主神，她要是想救心上人，就要继续穿梭别的世界，收集主神碎片，从而唤醒他。


叶满：「……」


就这样，叶满在话痨系统的胁迫下，开始了 N 个世界的穿梭之旅。


第一个世界：她是胆小的探险少女，丛林里的紫眸神秘人；


第二个世界：她是女扮男装背锅妹，他是冷淡强硬狱霸；


第三个世界：她是变不成天鹅的真·丑小鸭，他是阴鸷莫测的漂亮少年；


第四个世界：她是怯懦的亡国公主，他是苦大仇深的勇猛将军；


第五个世界：她是表面高冷内心沙雕的美人师父，他是前期废柴后期开挂的小徒弟。


……


在这个过程中，叶满不仅收获了爱情，还收获了个儿子。


「统统？叶统？叫爸爸！」


「滚！」


「叫不叫？不叫我让我男人收拾你！」


「爸爸！」


「乖！」


……


且看沙雕叶满如何克服万难，勾搭上全宇宙最厉害的男人，走上人生巅峰！
