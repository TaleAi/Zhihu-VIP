## 简介
七个不同侧面，还原一个真实立体的李鸿章。


李鸿章，是中国近代史上极具争议色彩的人物之一。


他谤誉一身，对他的评价往往形成针锋相对的两极：


有人说他委曲求全，跟洋人低头，是「卖国贼」、「伴食宰相」；


也有人说他据理力争，一时妥协纯属被逼无奈，是「国士无双」、「中兴名臣」。


的确，两极分化的评价，源于他本身的功过两立：


他有着不小的历史功绩，领导洋务运动，组建中国历史上第一支海军北洋水师，建立最大军火工厂江南制造总局，还有派遣留学生，改革科举制等等；


同时，他却也代表清政府签下一系列丧权辱国的不平等条约，如《烟台条约》《中俄密约》《中法新约》《马关条约》等。


那么，我们现在究竟应当如何客观看待李鸿章？


本专栏用七个不同的侧面，为你解读一个立体的李鸿章，展现时代浪尖上的历史人物，感受在大时代浪潮中的个人命运起伏。
