## 简介
事在四方，要在中央


权力，对于某些人来讲，是这个世界上最让人为之着迷的事物。


要掌握权力的柄杖，必先经历血腥残忍的磨砺。


而此过程，或出卖自我、或出卖他人、或横尸遍野、或血流成河。


从古至今，世界都是被极少数的权力掌控者所左右，大众只是他们手中的棋子。


他们有的是野心勃勃的政客，有的是操奇计赢的巨商，有的是刀口舔血的黑帮头目，有的是亡命凶悍的恐怖分子。


他们不断搅动地区的局势、左右世界的走向，甚至影响人类的命运。


本专栏将挑选当代世界最具代表性的权力执柄者，选取他们过往中的标志性事件，看他们如何翻手为云、覆手为雨，如何左右时局、主宰天下。
