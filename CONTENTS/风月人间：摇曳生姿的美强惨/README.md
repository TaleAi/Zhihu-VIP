## 简介
风情万种乱红尘，提刀斩尽负心人。


平生最可笑的事，便是爱上了你，结识了她，听你们许下白头，转身将我推下深渊。


我从深渊爬上来，变成了杀人不眨眼的恶鬼。


《媚鱼》


我死之前，皇上的白月光回来了。


他纵容她扯坏我的衣裳，毒杀我的小猫。


我心疼得睡不着觉，整晚流泪。


我曾是皇上最爱的贵妃。


他许我凤冠霞帔、母仪天下。


后来他挑起别人的盖头，要我跪在殿外，听他们一夜欢好。


他问我：「你不嫉妒么？」


我没告诉他，我已经病入膏肓，快死了。


《藏娇》


我与贵妃同月有孕。


沈放赏她安胎药，却骗我喝下堕子汤。


我拖着血衣，摔碎他为我求来的送子观音。


重活一世，我当众烧毁订婚书。


祝他和方琬宜郎情妾意、百年好合。


沈放却说，非我不娶。


我笑笑，转身嫁给前世早死的太子爷。


《玉蛮》


太子的白月光因我而死。


他与我退婚，反手抄我家满门为她陪葬。


他剥掉我的衣裳，将我扔进官妓所。


他说是我爹杀了他的心上人。


所以，父债女偿。


后来真相大白。


他在乞丐窝里找到我，发疯似的向我忏悔，说全是他的错。


他求我再看他一眼。


可是，我已经瞎了啊。
