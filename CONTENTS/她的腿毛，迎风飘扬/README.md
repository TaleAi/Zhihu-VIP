## 简介
热情！奔放！火辣！滚烫！在俏皮与嬉笑怒骂中反抗对于传统女性形象的标准定位。


这里有勇敢追爱的都市女孩，也有试图拯救他人的牡丹小妹，更有惨遭变态杀人狂跟踪的无辜少女。她们的抗争故事，勇敢，正义，爽快，被照耀在阳光之下。


但不是每一个人都能像他们一样站在值得同情的立场上。


这里也有没有生产力，不断忍耐家暴的妻子；确诊抑郁，准备坐吃山空的富女；甚至是勾引有妇之夫，争当同妻的傻女孩。她们同样需要抗争自己的命运。


渴望更好的生活的心情，是没有高低贵贱之分的。
