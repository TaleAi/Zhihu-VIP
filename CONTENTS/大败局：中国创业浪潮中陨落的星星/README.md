## 简介
这是一个中国著名企业失败故事集。健力宝、巨人集团、三九药业……那些你能叫得出的闻名一时的品牌，为何最后陨落？本专栏带你解读著名企业盛极而衰的失败基因。


从改革开放至今，一代又一代的企业在商业浪潮中披襟斩棘，创造独属于自己的神话。


可大浪淘沙中，也有一个又一个的公司，从光芒万丈的顶端突然陨落，成为令人唏嘘的传说。


曾经的健力宝，从一个地方市的小酒厂，一步步走向奥运会，走出国门，成为当之无悔的中国第一饮料品牌。为何却突然销声匿迹，只能成为大多数人年少时代的回忆。


曾经的巨人集团，其创始人史玉柱堪称 90 年代创业潮中开疆拓土的一代宗师。为何却转瞬而下，只能成为那个年代无数热血学子的心中剧痛。 


股市第一庄家德隆，如何布下一场场资本迷局，最后惨遭反噬？


和联想关系匪浅的顺驰，如何从一匹黑马变为人人喊打的害群之马？


曾经的中药业领头兵三九，如何从一骑绝尘到溃不成军？


……


再也没有比真实更惊心动魄的故事；


再也没有比商场更剑拔弩张的江湖。 


众位看官，且以史为鉴，


在其烈火烹油之时，可看它鲜花着锦。


在其灰飞烟灭之时，却看清这场大梦方觉。
