## 简介
一群中国神探，工作作风机智勇猛，也有不为人知的铁血柔情。
他们依靠缜密的逻辑分析，先进的科技手段，揭开一个个复杂、离奇的人性犯罪。


咸阳检察长杀妻，现场伪造成入室抢劫强奸。在检察长凶手的「庞大」势力以及高智商犯罪下，险些逃脱法律制裁，造就一场悬案；


千岛湖流传着恐怖传说，让一艘游船 32 名游客，全部死于非命。面对当地村民的阻拦，陈世贤通过法医技巧抽丝剥茧，找到了被人掩埋至深的真相；


石家庄连环爆炸案，犯罪分子威胁警察，在城市中不同地点安放炸弹，时间紧迫，必须尽快将犯罪嫌疑人绳之以法，不然整个石家庄将沦为火海……


这些曲折、复杂甚至离奇的特大案件背后，是谁让凶手显形，让罪恶曝光？


本专栏从中国神探的独家视角，向读者展开一个个传奇大案要案的侦破过程。他们是新中国首批痕迹鉴定专家，是对指纹有惊人记忆力的公安八虎之一，是公安部物证鉴定中心主任法医师，是解读现场第一人……


本专栏是中国神探们近 20 年来特大案件的侦破实录。
