## 简介
神仙也有凡人心，在无尽的寿命之中，他们也会产生无法克制的欲望，无法割舍的执念。这个专栏讲述的，就是一群法力滔天的仙君，被逐一拿下的故事。小甜文，大女主，请你来吃糖吧！


《两世欢》


为了嫁给病娇天帝，我实在是费尽心机，趁着他下凡渡劫，抓紧时间给他批发了一车绿帽子。好不容易抱得美人归，他却要为了我的一句戏言去攻打魔界？！


《驯夫记》


战神是个相当禁欲的男人，但我魔界圣女也不是吃素的！打倒他，囚禁他，得到他……一番努力之后，被捆在墙角的他终于松口：夫人，请慢慢调教我吧！


《情人劫》


和我有一夜露水情缘的仙君要杀了我，狼狈逃窜回家后，温润如玉的大师兄向我求婚了。一边是追妻修罗场，一边是男二想上位，唉，都怪我魅力太盛，这该怎么选？
