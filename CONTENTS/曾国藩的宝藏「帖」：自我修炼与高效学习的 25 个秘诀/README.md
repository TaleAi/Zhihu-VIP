## 简介
《曾国藩家书》蕴藏着曾国藩一生那种真实、诚挚、强大的精神力量，本专栏将以白话文的形式直接为你诵读《曾国藩家书》，通过曾国藩本人的经历和故事，详细生动的解析和提炼家书中的智慧，让你零距离、零障碍、读懂、读透曾国藩家书。


近 20 年来，「曾国藩热」席卷了华人文化圈，成功学、为人处事、政治、军事、冰鉴 、家书家训，尤其是《曾国藩家书》，满大街充斥着各种版本，**但市面对于这部家书的解读更多的是「教条格言」式的曲解。**


曾国藩是中国近代「内圣外王」第一人，据说他有十三套学问，流传至今的只有这一套《曾国藩家书》，但这本唯一流传下来的著作，**背后却隐藏着曾国藩一辈子最干货、最实用的智慧。**


但多数人真的拿起一本《曾国藩家书》开始读时，**第一感觉是：根本看不懂，更别提去领悟其中的智慧。**


本专栏将为你去繁化简，从 1500 封半文言文的书信中精挑细选出最有价值的内容，提炼出当下的你最需要的自我修炼、为人处事、治家教子等多方面的智慧和秘诀。
