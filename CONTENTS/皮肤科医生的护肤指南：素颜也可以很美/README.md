## 简介
了解自己的肌肤类型，才能更好地养好皮肤。本专栏带来皮肤保养的科学方法论，选择适合自己肤质的护肤品，学会正确清洁和补水保湿，保持肌肤的透亮和光泽。科学护肤、科学医美，让肌肤重回年轻。


我们每个人，都可能会陷入这样的护肤陷阱：


你每天都在认真护肤，但皮肤还是变差；


被美妆博主推荐「种草」，买了一堆不适合自己的护肤品；


被推荐刷酸祛痘，使用之后皮肤却变成了敏感肌；


在美容机构买套餐，花了大钱却没效果；


......


每个人的肤质不一样，护肤需求也就不一样，有些冤枉钱根本不必花。


本专栏针对不同肤质提供具体养护方案，5 大皮肤类型的全面护肤技巧，美白祛痘抗衰老一网打尽，满足不同人群的护肤和医美需求，拒绝无效的美容「智商税」。
