## 简介
死都不怕，为什么要当汉奸？


人是会变的。


英雄好汉会堕落变成国贼，流氓废柴也会有变身英豪的那一刻。


让他们走上与最初规划的方向完全不同道路的，到底是什么？


时代环境、主观意识还是命运之神的安排？



> **汉奸汪精卫曾是民国「顶流偶像」，在狱中写下爱国诗篇，最终却被粉丝拽着走向万劫不复之路……**



> **出身辽东将门，多年镇守边疆，吴三桂真的是冲冠一怒为红颜，才致使千万生灵涂炭？**



> **近代史上最突出的二代废柴，除了吃喝玩乐，样样稀松。不，这不全面，张学良并不是单单投胎技巧高超。他，只是英雄得不够明显……**


当时局每天都在变，人的信仰，该何去何从？
