## 简介
所谓妖魔鬼怪，鬼者，人死后所化。我们看志怪小说，关于降妖除魔的，常会出现百鬼夜行一词。你是否好奇过这「百鬼」是哪些鬼。在这里，100 节课，满足你的好奇心，帮你揭秘形色各异的鬼怪，感受不一样的中国传说。


你知道无常，孟婆，夜叉吗？


这些都是大家熟悉的传说中的鬼怪。


但是除此之外，你知道我们国家还有许许多多其他种类的鬼吗？


比如：


蓬头鬼，它可是个十分有爱心的鬼，专业保护动物几百年。


又比如痴鬼，种类很多，有痴迷于情的痴情鬼，有痴迷于酒色财气的吝啬鬼、滥赌鬼、更有痴迷于琴棋书画、花鸟鱼虫的高雅鬼。


还有穷鬼，我们平时说某某是「穷鬼」，没想到还真有那么一种鬼。他身材羸弱矮小，性喜穿破衣烂衫，喝稀饭，是一个会让人变穷的鬼。


中华文化上下五千年，其中包含了许多妖魔鬼怪的传说，它们种类繁多五花八门，多出现于中国古代志怪小说当中，中国妖怪全录便是记载中国亡灵的图书，里面记载着 100 种妖魔鬼怪。


在这门课中，一颉老师将带你看遍各式妖魔鬼怪，了解它们的来历和排名，一起步入神秘的世界。
