# 拍出触动人心的作品：撬开艺术家的大脑

### 作者：王晶

### 分类：艺术

### 状态：已完结，共 14 节

### 存档日期：2022-06-16

## 简介
我们从如何向摄影作者入门，到大师是如何思考与创作作品，又谈到艺术家的头脑与方法，通过三章内容给你立体、全景的知识！让你从这里，知道路在何方。


你一定会遇到这样的情况：


偶尔拍出一张很漂亮的作品，别人都夸赞你，但却再也重复不出来了。为了学习摄影，读了许多摄影史，也读了不少画册，可还是摸不着头脑。  

  

为了拍得更好而去艺术学院深造，是不是很有必要？参加了一些课程，每次似乎都很有收获，但又好像没学到什么。


世上有那么多好东西。一幅画、一张照片，居然能那么彻底地打动人。你是否也想创造出优美的、动人的作品？  

  

有的作品看起来很简单。翻看艺术家的访谈，却发现说法都不相同，让人摸不着头脑。硬着头皮去模仿，却发现无穷无尽的技术、思路问题，简直无法解决。模仿不出来，就深深怀疑自己；模仿出来，却发现下一步不知道怎么走了。


为了解决这一系列不连续的问题，我们从如何向摄影作者入门，到大师是如何思考与创作作品，又谈到艺术家的头脑与方法，通过三章内容给你立体、全景的知识！  

  

让你从这里，知道路在何方。
