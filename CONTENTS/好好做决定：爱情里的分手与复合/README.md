## 简介
本专栏将结合心理学理论，从科学视角带你深入剖析有关分手复合的那些事，帮助你走出爱情迷宫，摆脱情绪困境，在情感中不再纠结，做回自己。


人的一生那么长，难免会在爱情的迷宫里看不清方向。有的人深陷在错误的爱情里无法自拔，明明分开才是最好的选择，却在痛苦和纠结中错付了时光；有的人总是忍不住羡慕别人，最终把原本美好的感情推向虚妄；还有的人，在一段感情中分分合合，磨灭了耐心和希望，只剩下无尽的纠缠和绝望……


面对爱情冲突，是分离还是继续，这是每个人都会面对的人生选题。在本专栏中，心理科普作者暗涌将以科学的视角，借助经典案例带你深入剖析「爱情中的分与合」，帮助你走出爱情迷宫，摆脱情绪困境，找到心中的答案。
