## 简介
这是一个囊括了穿越、传奇、宫斗的古风专栏，4 个不同风格的古言故事，带你进入不一样的奇妙世界。


《今天我当上太后了吗？》


话本无数女配的人设，是恶毒无脑妖艳贱货，我不一样，我不妖艳，我只贱。


京都十亿少女的美梦，是成为皇后，我不一样，我只想当太后，升职加薪死老公。


后宫十个嫔妃的日常，是养娃唠嗑疼小皇后，我不一样，我只想发起第一届宫斗。


《浣纱溪旁见卿卿》


一觉睡醒发现轻薄了太子，没掉脑袋却成了太子妃？


太子冷你，厌你，不见你，所爱非你，一回府却要你跟他生个娃？


爱你的人为你赴汤蹈火，他爱的人却令你命悬一线？


江山、红颜、情动、薄幸，孰是孰非，何去何从？


《松香泪墨》


我出生卑贱，母微父贫，头二十年总是在羡慕别人的人生，但我从来都觉得，我命中注定是个主角。


这倒因为不是二十岁之后，我那个啃窝窝头、想女人的爹变成了权倾朝野的奸臣；


亦不是因为，我那个功勋之后、连我女扮男装也爱惨了我的贵族丈夫；


而是因为我天赋异禀，哭的时候流的不是泪，而是松香墨水……


《念君有慈》


顾念慈进宫那年十四岁，是先帝威胁阿爹的人质。


盛宠如咒，她失去了阿爹，眼看整个顾家就要在被权势吞噬。


顾念慈为保顾家将皇子宋凜扶上皇位，不料在脱身之时被宋凜百般阻挠。


后妃三千，宋凜却偏偏要留她一个太后。
