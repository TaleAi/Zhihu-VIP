## 简介
曾经有一家黄焖鸡店，那里连接着阴阳两界。吃下一碗黄焖鸡，就能撞见一场天人永隔。在这个专栏中，你将再次见到 8 份让人食指大动的传统佳肴，然而每一个珍馐背后，都是一个让人惶恐不安的惊异故事。

品饕餮美味，看人间幽冥。


吃一碗香臭四溢的螺蛳粉，吃的方法不对，可能会折寿；


点了份卤鸭舌，吃得津津有味，细细品尝却发现这肉的质感分外奇怪；


如果在不对的时间，一个人吃鸳鸯锅，你将永远不知道，和你拼桌的那位，到底是什么东西……


涮羊肉、鸭血粉丝汤、佛跳墙、煲仔饭……这一次，知名惊悚故事作者川戈再次对美食下手，呈上一个个细思极恐的人间诡事。


你会发现，在虚构的妖魔鬼怪背后，最令人胆寒的，终究是浮世的凉薄。


 


拿起碗筷，即刻开餐！
