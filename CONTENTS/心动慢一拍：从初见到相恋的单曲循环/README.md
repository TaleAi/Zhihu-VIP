## 简介
生活里发生什么样的故事情节，才会让你的心脏漏跳了一下？是甜甜的初见、是酸酸的暧昧，还是完美的奔赴、无奈的成全？那是我们心动的一瞬间，看见了你我的未来。


《老板，CPDD》


游戏陪练喜欢上了素未谋面的老板，正当我要表白这天，他却脱单了。


我悲痛欲绝，火速找了份工作，想借此来驱赶失恋的痛。


然鹅，在我决定拥抱新生活的时候，我忽然发现，那个拽咖同事居然就是我游戏里的老板。


《老弟，学着点》


我跟男朋友作为公司里的上下级，一直在隐藏着我们的关系。大家热衷于给我介绍对象，我一一婉拒，直到有一天部门聚餐，我一手带出来的徒弟三杯酒下肚，忽然跟我告白了。


正不知道该如何拒绝时，男朋友在众人的注目之下笑眯眯向我走来。在众人的惊异与起哄下，他牵起我的手，猝不及防地在我唇上印下一吻，然后对着我已经傻眼的徒弟说：「学着点。」
