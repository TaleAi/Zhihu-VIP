## 简介
历史上一次次的血雨腥风


一部晚清史，我们会记住不少大人物的名字：曾国藩、李鸿章、左宗棠、张之洞……我们也会记住一些奋战在一线的将官名字：关天培、陈化成、邓世昌、刘步蟾……但有一个人，他在我们的教科书上只一闪而过，他是一名提督，但发生在他身上的故事，却足以让我们掩卷长思......


袁世凯，我相信每个人对他的名字都挺熟悉的，但同时，又对他挺陌生的。在整个民国的前半段历史上，他可谓是个知名人物。但他到底做了些什么？他后来怎么样了？甚至，他是个好人还是坏人......


在第二次世界大战中，轴心国阵营，其他国家都是提前无条件投降的，只有一个国家是一直打到了所谓的「首都保卫战」。这个国家就是纳粹德国。也正是因为这场玉石俱焚的垂死挣扎，出现了一个「投降两次」的故事......


我们都知道俄罗斯方块，它是一款游戏，但又不仅仅是一款游戏。当一款游戏流行到几乎人人都有接触的程度，那么，背后就肯定有故事......


到目前为止，切尔诺贝利核事故是人类历史上最惨重的一次核事故。但在这场事故的背后，我们看到的悲剧，又岂止是灾难本身……
