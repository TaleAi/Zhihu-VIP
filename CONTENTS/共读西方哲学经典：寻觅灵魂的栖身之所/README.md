## 简介
共读经典，徜徉哲学的海洋，解开人生的真谛。


社会快速发展、生活水平不断提高的今天，依然有很多令人不解的事：


对日复一日的生活感到厌倦，不明白人生的意义；忙碌半辈子，回首往事却只觉得空虚；明明物质条件不错，但仍觉得不快乐……


也许你应该了解一点哲学。


一定意义上说，「任何一门学问的终点，都一定是哲学」。


本专栏特邀黄向军老师共读五十部西方哲学经典，在哲学浩海中领悟智慧、感受成长，品味人生，聆听自己内心深处的声音。


《理想国》《从黎明到衰落》《伦理学》……徜徉经典，这里有你想知道的一切：关于生活的本源、幸福的真谛还有你我的存在。


答案都藏在哲学世界中，等待你的探索。
