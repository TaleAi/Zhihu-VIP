## 简介
一个关于权力与爱情的古代言情专栏，带你看男女主势均力敌的爱情。


「万箭穿心」：传说万箭穿心是箭流剑的至高招数。此招一出，方圆一里之内无活口。杀手苏年太想练成这一招了，用它干完一大票，在京城买那座梦寐以求的小院，过上美滋滋的退休生活。可惜师父说她练不成，因为，她不够无情。苏年只能继续接单赚钱，没想到这一天，她在刺杀目标静王时，却被人截胡误伤，堂堂一个杀手，却落到了刺杀目标手里……


「女逆贼」：我不是一个反套路女主，却遇上一个反套路男主。太子皇帝什么的，不都是一脸王霸之气的么，那么这个黏人、话痨、爱生气的小可爱到底是谁？可是，就在我放弃了从小暗恋的班主，想要和这个小可爱好好生活的时候，却发现他早已不是我认识的那个人。权力这种东西，会让一个人的改变有多大？


「菱花镜」：为了和他在一起，我堂堂一介公主，亲自下场搞掉了「白莲花」女二，在他刺杀哥哥之际，尽力保住他的命。可我没有办法阻止这个腐朽王朝的衰落。改朝换代，成王败寇，我无话可说。只要，只要他不发现那个秘密，在这凋零的天下中，我们就还会有未来。


……


女杀手 x 病娇王爷，无情歌女 x 奶狗皇帝，黑莲花公主 x 逆贼世子……


各种古言顶配，带你看一场场势均力敌的虐恋情深。
