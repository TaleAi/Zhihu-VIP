## 1.IP 时代来临：知识产权的前世今生
一、生活中的知识产权


  



当我们的人生步伐刚迈进 2022 年，一部网剧《开端》，强势霸占了网络热搜榜。这部无限流网剧对我们这些在信息轰炸下，早已审美疲劳的人们来说，无疑是一股清流。


《开端》的剧情很简单，一句话就可以说清楚。


一辆公交车即将爆炸，一男一女在时间循环中不断尝试阻止爆炸，最终成功拯救了所有乘客以及前来支援的警察，在这个过程中顺便谈了场恋爱。


新颖的故事情节深深地抓住了观众的眼球。因此，这部剧产生了很多「二创」网络梗，诸如「锅姨」「高压锅 PTSD」「恋爱循环」「这人能处，有 XX 他真 XX」等等。


其实近年来，像《开端》这样的优秀网剧层出不穷。既有孕育出「带你去爬山」热梗的《隐秘的角落》，也有胡歌主演的《琅琊榜》，还有制作精良的《知否知否应是绿肥红瘦》等。虽然这些电视剧题材差异很大，但它们都是 IP 剧，共同见证了一个 IP 时代。


什么是 IP？


IP 不是互联网协议地址吗？刑侦剧里的警察总是通过电脑的 IP 地址抓住犯罪嫌疑人。其实不然，此 IP 非彼 IP。我们这里说的 IP 是英文「Intellectual Property」的首字母缩写，直接翻译过来就是「知识产权」。


人们之所以把《开端》叫做 IP 剧，因为它是由小说《开端》改编而来。同样，《隐秘的角落》改编自作家紫金陈的小说《坏小孩》。《琅琊榜》和《知否知否应是绿肥红瘦》也是同名小说改编作品。从严格意义上来说，是这些小说，享有了最初的「知识产权」（IP），而电视剧是衍生品。


那么 IP 跟我们普通人有什么关系？它不是只对文化产业有影响吗？                                      


其实，IP 早已渗透到我们生活的每一个「隐秘的角落」。


当我们在做核酸检测时，检测费用里面就包含了一笔专利费，支付给了核酸检测试纸技术；


如果看见一个叫做 LV 的楼盘，你只会莞尔一笑，因为我们知道 LV 是奢侈品牌，和房地产没有关系。但是你会不自觉地将它们联想在一起；


麻辣火锅的「人生伴侣」——可口可乐，它的配方就是商业秘密。 


所有这些都涉及到了知识产权的相关内容，而且实实在在对我们的日常生活产生影响。而如果你是脑力工作者，可能已经或者将会是知识产权所有人。因为你拥有的知识或许正变成你的财产。


 


二、有知识，不等于有知识产权


 


每个人都有知识，但并不是每个人都有知识产权。


有这么一则笑话。一个穷书生想要成家，因此到相亲市场上转了一圈。一天下来，他发现征婚启事都要求男性有不错的经济条件。家徒四壁的穷书生捏捏了空空的口袋，计上心来。他找到一个媒婆，拍着胸脯对媒婆说：「我很富有，尽管给我介绍最优秀的相亲对象。」媒婆盯着他满是补丁的衣服问道：「具体多富有呢？」书生自信地回答：「富可敌国，因为我满腹经纶，拥有无穷的知识。」 


如果你是媒婆，会信书生的「鬼话」吗？


毫无疑问，答案是否定的。显然，肚子里有多少墨水和手里握有多少钱是两码事。否则，所有上过学的人都将是富翁。


回到知识和知识产权的关系上来，我们每个人都可以运用知识来挣钱，换句话说从事智力劳动，最典型的就是技术研发人员。在这个过程中可能会产生智力成果，其中符合法定形式的智力成果才是知识产权。

与传统意义上的财产相比，知识产权是无形的。所谓的无形主要体现在两个方面。


第一，知识产权的形态是无形的。


一说起财产，我们马上想到的是房子和车子，它们是有形的，既看得见也摸得着。通常我们讲，房子是固定在土地上的不动产，车子是可以动的动产，而无形的知识产权是和不动产和动产所并列的一个法律概念，这三者既是法律意义上的财产，也是经济学意义上的财产。


看到这里，或许你会有疑问，像游戏装备这样的数据资产属于哪一类？它好像既不属于知识产权、也不属于不动产、更不是动产。


应该说，数据资产既具有无形性，也具有非物质性，但究其本质，数据资产是和知识产权所平行的一种概念，是新技术产生的新法律概念。《民法典》第一百二十七条确定了数据是受法律保护的，但至于其属性，法律上尚未明确，法律已经滞后于社会和技术的发展，法律人也正在对其进行法律上的构建。


第二，知识产权的形成脱离劳动本身。


古话说「书中自有黄金屋」，讲的是通过读书获得的知识，能够帮助读书人从事可以获得财富的创造性劳动，但如果从知识产权的角度来看这句古话，那么这本书无需学习，其本身就是一座拥有知识产权的「黄金屋」。


怎么理解这一点呢？


这是因为知识产权是智力劳动的成果，它已经脱离了劳动本身。一位作家写了一本书，当它一旦完稿，就不仅存在于作家脑中，也在现实世界中获得了知识产权。就像金庸先生所著的「飞雪连天射白鹿，笑书神侠倚碧鸳」，这些小说完成之后，就已经脱离了金庸先生的思想而拥有了自己的 IP。


我们还需要注意知识产权的另一个特点，那就是知识产权的价值区别于我们为智力劳动支付的费用。智力劳动本身是和体力劳动相对应的概念，在这种场景之下，劳动者所投入的智力劳动本身是有价值的，提供的费用是劳动者智力劳动过程的报酬而非智力成果的对价，智力劳动成果是和财产相对应的概念。


 


三、俯瞰知识产权


 


萧伯纳说：「你有一个苹果，我有一个苹果，彼此交换一下，我们仍然是各有一个苹果；但你有一种思想，我有一种思想，彼此交换，我们就都有了两种思想，甚至更多。」


正如这位大文豪所言，由于无形性的特征，知识产权可以被简单地复制与分享，而时代和科技的发展也给知识产权的保护带来了更多挑战，因此为了保护知识产权，我们将对它做了分类，主要包括三类权利：**著作权及与它相关的邻接权、专利权、商标权。**


对每一种权利，法律都为其建立了相应的制度。著作权相关制度，专利法律制度、商标法律制度、反不正当竞争法律制度等，都是我们的知识产权制度的一环。


需要我们注意的是，中国目前还没有建立所谓的「知识产权法典」， 我还记得在一部电影中，有人以科普的口吻说道：「根据《中华人民共和国知识产权法》……」这是很多人常见的误区。实际上，这些保护制度的存在形式都是单行法。例如《中华人民共和国著作权法》《中华人民共和国专利法》《中华人民共和国商标法》《中华人民共和国反不正当竞争法》等等。


因此，如果你想要就某一个具体问题查找知识产权法的相关规定，就应该找


对应的法律法规。


让我们往知识产权的大门里再迈一步。在知识产权的法律部门里，著作权、专利权和商标权又有何不同？


从对象上来看，著作权的适用对象是作品，专利权的适用对象是技术方案，商标权的适用对象是标识。三种权利的关系用一个例子就可以理解。


我女儿写了一本小说，她对小说就享有著作权，而她又在小说中写了一个技术方案，是「基于机器学习的智能化实现发型设计系统」，对于该方案本身她又可以去申请专利，专利被批准后就能受到专利法的保护，从此，她发现了商机，根据她的技术方案开始制造产品，并注册了一个商品名称开始销售，这个名称一旦获得商标注册之后，她对产品的名称就享有了商标权，这个商标便体现了产品的商业信誉。


 


四、作为平衡器的知识产权制度


 


如前所述，知识产权制度并不是一开始就有的。十七世纪，法国学者卡普佐夫最早厘定了知识产权这个概念，将其概括成一切来自知识活动领域的权利。知识产权这一概念，在 1967 年《成立世界知识产权组织公约》签订后，才被世界各国广泛认可。在知识产权制度建立之前，我国古代的四大发明显然具有巨大的商业价值，但是发明人并没有从中获得任何的收益。


若回顾著作权的历史，它最早叫做版权，这里的「版」指的是「出版」，它侧重保护的是出版商的利益，而随着印刷成本的下降、传播技术的高度成熟，复制作品与盗版变得越来越容易了，在这种情况下，法律就需要去平衡传播者和文化艺术科学创作者的利益。


知识产权制度是一个平衡器，它需要平衡创作者、传播者和社会公众这三者的利益，在不同的领域中，相同的平衡思路也可能带来截然相反的社会效果。


我们以徐峥主演的电影《我不是药神》来举例。


在电影中，情节围绕一种名为「格列宁」的白血病治疗药物展开。「格列宁」对慢性粒细胞性白血病的治疗有奇效，但是专利药售价高达 4 万元一瓶，而印度的仿制药疗效一致，但售价只有 5000 元一瓶。徐峥饰演的程勇就在为病患走私这些印度仿制药。


随着程勇最终被抓，这个走私行为在法律和道德上面临双重审判。在法律上，走私属于犯罪行为，因此程勇被判刑。在道德上，走私仿制药挽救了很多吃不起专利药患者的生命，应该受到褒扬。


其实，这部电影的故事有真实的原型，而「格列宁」正是瑞士制药公司「诺华」（Novartis）研制的格列卫（Gleevec）。


情与法的争执是看待这一电影的视角之一，而如果聚焦药物本身，就是知识产权制度该不该倾向保护技术创造者利益的争论。如果技术创造者的利益被优先保护，于产品使用者而言，就会出现电影中专利药售价 4 万元一瓶的场景；于技术本身的进步而言，则可能会阻碍对专利的仿制，基于专利开发出更为便宜的药品。


「格列宁」售价如此高昂的一个主要原因就是投入药物研发的成本高昂。一种新药从无到有，必然面临极高的不确定性，用于试错的成本自然很高。从商业利益考量，既要填补巨额研发成本，又要赚取利润，自然就出现了高定价。


专利制度保障「格列宁」生产者的利益，使得这种制药技术被垄断。一方面，使得定价降不下来，另一方面，又影响了这种技术的迭代进程。


正如电影所展现的，印度的专利保护制度约束力较差，印度的制药公司就竞相仿制「格列宁」。仿制药之所以只售价 5000 元一瓶，是因为印度制药公司不需要考虑「格列宁」的研发成本，只要覆盖迭代成本以及制造成本就可以了。同时，这种制药技术本身也变得普及，更有可能获得进一步的迭代升级。


但同样是选择倾向保护创造者，这一思路在文学艺术作品的保护中便会产生不同的社会效果。


技术的发展导致网络的盗播越来越容易，打击盗播盗版的难度也越来越大，因此像《开端》这一类热播剧在互联网领域想要发展，就要对作者的知识产权加大保护力度，让作者能在文学艺术作品的创作中获得收益与正向激励，作者才能在一个更好的保护环境中创作出更好的作品，让咱们这些观众获得更好的视听体验。


由于盗播盗版现象严重，那么作者无法从自己的作品中获得应有的尊重与收益，整体的知识产权保护环境便会让作者失去了创作的热情，这显然是不利于文化事业的繁荣。这几年的电视剧越来越水，根本原因是需要延长电视剧的时长来获得更大的播出费，实质是饮鸩止渴。近几年来，我国未能推出经典的电视剧，知识产权保护不足是重要原因。


所以，知识产权制度就是一台精密的平衡器，需要不断去平衡各方利益来达到促进社会进步的社会效果。


 


五、学习和尊重知识产权法是我们每个人的责任


 


我们谈知识产权，并不是一件遥远的事，它与我们的日常生活息息相关。


作为一个商业从业者，当我们遇到一个好作品的时候，就能意识到它的商业价值。正如科幻小说《三体》如果拍摄成电影，就自带流量和票房。


作为一个消费者，当我们读了一本好书之后，就会找找这个作者有没有其他作品。当我们购买运动装备的时候，就会去安踏、耐克、鸿星尔克的专卖店，因为我们知道这些品牌质量有保障。这些就是作者的著作权和商家的商标权赋予我们的生活便利。


作为一名创作者，当我们在抖音上传一段自制的故事短视频，就会加上自己的水印，防止别人盗用，因为他们会白嫖我们的劳动成果。


而作为一名专注知识产权的法律从业者，我曾以律师的身份为当事人代理知识产权纠纷案件，也曾以法官的身份在国徽下为原、被告做出庄严的判决，现在将以学者的身份研究知识产权法，为法理分析和普法工作贡献绵薄之力。


无论你是不是法律从业者，无论你身处什么样的行业，知识产权都与你紧密相关，因为每个人都有自己的智力创造，这就是我们的财产。


学习知识产权法，既是为了保护我们自己的知识产权，也是为了让知识产权法发挥它应有的效力。这是我希望这个课程能够达成的目标。


在本节课结束之前，我再提醒一下，您正在看的这篇文章也是我生产的智力成果，我现在生产的是版权，希望大家能够尊重我的版权，让我能在这次授课中获得尊重。


让尊重知识产权成为我们的习惯，让我们在余下的课程一起学习知识产权法吧！
