## 简介
「天坑专业」就业难，对自己专业也不喜欢，我到底要不要选择跨专业考研？跨考上岸难度有多大？跨考会在复试时被歧视吗？考研不易，跨考更不易，若是缺乏理性的思考和深入的调研，跨考某种程度上就只成了「碰运气」。本专栏邀请到从事高效硕士招生工作的纸谈风月，对各专业现状及前景进行深度分析，手把手带你完成跨考目标选择、跨考复习备考、复试调剂等环节，助你实现专业的完美切换。


在 2021 届研究生招考中，有超过 30% 的考生选择了跨专业考研，而 2019 届时这个数字还是 20%。跨专业，正成为越来越多考研生的选择。


这种趋势，一方面是由于产业结构更迭，很多原本还不错的专业逐渐边缘化，变得性价比越来越低；另一方面，现在的同学们越来越敢于去追求自己想要的职业与生活。


但是，在这三成跨考考生中，不乏凭着一时冲动决定逃离本专业的、随大流追求热门专业的、对目标专业未经调研就报考的，总之，大多数同学在决定跨考时，缺乏理性的思考和深入的调研，跨考某种程度上成了「碰运气」。


考研不易，跨考更不易，不应因为自己的草率白白浪费一年宝贵的时光。


本专栏将帮助跨考生如何评估自己是否适合跨考，各专业方向现状、前景及考研难度分析，跨考信息收集及人际关系，专业课资料真题获取方式，各科复习指南，复试、调剂，如何克服跨考生的劣势等。总之，关于跨考，你能想到的困惑，在这个专栏都能得到解答，且听我细细道来。
