## 简介
旧社会流传下来的骗术可以说是五花八门。你可知道五花是哪五花？八门究竟又是何种八门？众说纷纭间，唯蜂、麻、燕、鹊四大门流传最广。却不知除此之外，尚有册、火、飘、风、惊、爵、疲，要八大行。世人认为耳听为虚，眼见为实。对骗子来说，你看到的，不一定是真的。


**我爷爷是个金盆洗手的骗子，熟知蜂麻燕雀四大骗门的各类套路，和骗术秘辛：**


蜂门擅长天衣无缝的天仙局：因拆迁获得百万巨款的青年，以为自己马上能中天价彩票，殊不知逐渐堕入网中；


麻门也叫「马」，单人行骗，骗完就走。一位高学历单身女青年，发誓只嫁优质男，最终被「麻门」盯上，因为贪念，下场凄惨；


燕门，藏匿于高端商品、高端住宅之间，经过高级服务训练，懂得揣摩男人心思。富商每天重复同样的生活，自觉婚姻缺乏激情，直到买学区房时认识了售楼小姐。一步步，他被引入深渊；


鹊门也叫缺门，与蜂门不同，鹊门都是生活在一起的家人，极具排他性，外人难以接触。做局之时，先瞄目标，查看家室，心要黑，手要辣……


所谓：骗中有骗，局中有局。人心难测，无欲则刚。


以下是骗门的隐秘故事。
