## 38.望穿黑洞的「慧眼」：硬 X 射线调制望远镜
你一定听说过「黑洞」吧！茫茫宇宙中，黑洞是一种特殊的天体。它的质量和密度都非常大。根据万有引力定律，一个物体质量越大，对其他物体的引力就越强。那么，黑洞的引力到底有多大呢？它的引力大到，连光都无法从它身旁逃脱呢！光都被吸进了黑洞，没有光发出来，黑洞当然就变得漆黑一片喽！


如此「内向」的天体，我们该怎么样去认识它呢？这可不是件容易的事情，直到今天，人类对黑洞的认识还非常有限。但黑洞越是神秘，就越是激发起了人类探索它的好奇心。科学家们发现，黑洞也不是什么都不发射的。黑洞强大的引力之手把物质拉向自己的同时，物质的速度会越来越快，温度也会越来越高，最后就会发射出一种高能射线——硬 X 射线。通过探测和接收这种射线，就可以间接地了解黑洞，揭开它的神秘面纱了。


那么，什么是硬 X 射线呢？你可能听说过 X 射线，也就是 X 光。它的穿透能力很强，所以在医院里，经常用它来为人体内部的骨骼或器官拍照。我们告诉你，光的本质是电磁波，所以它在传播的时候，也像水波一样会形成有规律的高低起伏，这种起伏叫作波峰和波谷。


光有不同的颜色，这是因为每一种光的波峰和波谷交替的频率不同。不同颜色的光有各自的频率范围，我们眼睛看不见的 X 射线的频率也是分布在一个区间。正如太阳光可以分为红、橙、黄、绿、蓝、靛、紫七种颜色一样，X 射线也可以大体分为两部分，频率较低的叫做「软 X 射线」，能量比较小；频率较高、能量较大的，就是「硬 X 射线」了。怎么样，黑洞确实不一般吧？连发出的 X 射线，都是高能量的「硬 X 射线」呢！


虽然知道了黑洞会发出硬 X 射线，但探测和接收这种射线，也不是件容易的事。因为地球引力的作用，我们生活的地表被稠密的大气层包裹着。虽然 X 射线穿透能力很强，但还是无法穿透大气层，所以在地面上，我们是无法收到来自遥远宇宙的 X 射线的。其实，我们应该感谢大气层的这种阻挡作用。虽然少量的 X 射线可以用来检查身体，但大量的 X 射线对人体是有害的。如果哪天大气层消失了，我们人类可能就要像宇航员那样，穿着厚厚的宇航服才能生存了。


既然在地面观测行不通，科学家和工程师们想出了另一个办法，那就是把接收硬 X 射线的望远镜安装在人造卫星上，然后发射到远离大气层的太空中观测。在那里，X 射线还未被阻挡，因此望远镜可以成功地接收。人造卫星在收集了硬 X 射线的数据之后，再将数据传回地面，我们就可以看到宇宙深处的世界了。


如果把宇宙想象成一个舞台，黑洞、恒星等天体就是舞台上的演员，这些演员在上亿年的时间里，演绎着一幕幕精彩的舞台剧，而太空中人造卫星所在的位置，就是清晰地观看这些演出的绝佳座位。在这些座位上，已经坐了不少的外国观众，其中就包括美国著名的天文望远镜——哈勃太空望远镜。


作为一个航天大国，我们中国当然也不想错过如此精彩的演出。于是，就在今年的 6 月 15 日，我国在酒泉卫星发射中心发射了硬 X 射线调制望远镜卫星。它还有一个好听的名字，叫做「慧眼」。


之所以取这个名字，第一个原因，是它的美好的寓意，希望它真的能「慧眼如炬」，穿透宇宙的迷雾；第二个原因，则是为了纪念著名的女科学家何泽慧院士，她是我国天体物理学的奠基人之一，也是著名物理学家钱三强的妻子，被誉为「中国的居里夫人」。有了「慧眼」，中国终于有了属于自己的第一手宇宙 X 射线数据，黑洞的秘密，离我们更近了。


作为一架高科技天文望远镜，「慧眼」的本事可不一般呢！它不但能「望」得很远，而且还能「看」得很清楚，即使是宇宙这个大舞台角落的细节，它也能够捕捉到。「慧眼」之所以有这么大的本领，要归功于科学家和工程师们赋予它的三项本领。第一项本领，叫做「扫描巡天」，指的是望远镜对整个银河系进行扫描，去寻找黑洞。此时，「慧眼」就像草原上的一名机警的猎手，不断地扫视整个草原，去寻找猎物。


第二项本领，叫做「定向观测」，指的是将望远镜指向某个天体，进行专门的观测。这就相当于，猎手发现了一个猎物，于是将目光紧紧地锁定它。第三项本领，叫做「伽马射线暴监测」，它就像猎手听到某处突然有猎物的明显动静，于是立即转过头去观察它。有了这三项本领，「慧眼」的观测就非常灵活了，仿佛真的获得了智慧呢！


现在，你已经知道，我们国家为了对黑洞等特殊天体进行观测，发射了硬 X 射线调制望远镜。虽然它能观测到黑洞发出的硬 X 射线，获得关于黑洞的第一手信息，但理解这些信息，却需要人类非常高超的智慧和不懈的努力。认识宇宙这个伟大的事业，是需要一代代人接力去完成的，你愿意接过这个接力棒吗？
