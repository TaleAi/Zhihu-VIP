## 简介
勤护肤不等于会护肤，只有找到最适合自己皮肤的状态的方法才是好方法。本专栏由闫妍博士主讲，她从事制定护肤品国家标准的工作，也在《Beauty 小姐》节目中担任行业权威专家主持人，拥有丰富的护肤理论及实践经验。


大多数为了拥有好的皮肤状态的人，做了很多努力：


为了提亮肤色，各种美白产品买买买；


刷酸换肤，耗时耗力又不能保证安全；


美容院护肤，花钱多，没多久又复发；


每天用心护肤，依然出油起皮长斑……


花钱多，耗时间，太盲目，没效果……


本专栏为你介绍护肤常识、注意事项，只为找到最适合你的护肤法，让皮肤轻松「换新」。不仅为你介绍高效的护肤方法，找到核心要点解决护肤问题；还让你掌握不同阶段抗衰关键节点，轻松做到比同龄人年轻。
