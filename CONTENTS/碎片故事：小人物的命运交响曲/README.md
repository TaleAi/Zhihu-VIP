## 简介
这是一本小人物故事集，没有名人事迹，只记录着有血有肉的鲜活人生。


这是一本小人物故事集，它不讲述什么名人事迹，不吹嘘什么成功秘笈，不会告诉你什么矫情的鸡汤。


它只是把文字当镜头，记录着一个个有血有肉的人，记录着他们和生活作战的过程。


从山村里跑出来的站街女，被人在街边痛殴后，她为什么对着太阳发笑？


被辅导员恶搞发配去支教的大学生，在脱离苦海的最后一天，他为什么要痛哭？


那个在路边日复一日等待的婆婆，她到底在期待着什么？


……


什么是生活？


生活就是一座大山，有的人站在山顶，看着星光灿烂，有的人被压在山底，拼尽了全力，也不过能喘一口气。


读下去吧，朋友们，也许在故事的最后，你能看到那个一直陪伴自己的人，你能共情那些执拗却无可奈何的反击，甚至，你能看到过去或者未来的自己，他们都在向你招手微笑。


这也是作者希望的，这些破碎的故事，真的能够击中你的心。
