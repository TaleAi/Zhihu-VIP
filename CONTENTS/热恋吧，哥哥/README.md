## 简介
勇敢一点，让暗恋变成热恋！


我们躲在无人的角落，看着他光芒万丈，却不知道其实勇敢跨出那一步，暗恋就能变成热恋。


**《佳佳有家》**


分手六年后，我坐在他的急诊室。


「我怀孕了，孩子是你的。」


他脸色铁青，「什么孩子要怀六年？」


一时间气氛尴尬到极致。


「不认？」


「你觉得我会接盘吗？」他反问我。


我沉默几秒，「行，那我去给他找个爹。」


九个月后。


他恶狠狠地拽着主刀医生，「兄弟，算老子求你，给她划好看点，她爱美。」 


**《念念不妄》**


医学院的姜妄腿断了。


吓跑了所有女朋友。


「往好处想，只剩我一个，你不会那么虚。」我试图安慰。


他踹翻我喂的饭，并赐我一个字，「滚。」


后来，我坐在天台抽根烟……


听说——


他从轮椅翻下来，爬了两里地。
