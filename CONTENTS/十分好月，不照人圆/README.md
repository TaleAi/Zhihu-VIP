## 简介
在梦里，故人还是相亲相爱的模样。


月亮总会缺了又圆，但分开的人，是否还有重聚的一天？


**《一枕槐安》**


上一世，允王周淮设计我爹杀人，让我娘给我下药，拿我当他心上人娴妃的替身。


在天子去世后，他更是用我换出他那个即将殉葬的心上人，把我封在帝王陵寝中活活饿死。


之后，他们俩用着我的名字、我的身份，过得好不快活。


重生后，他一如上一世那般诓骗我。


但我已不是当日的我。


我假装信他，表面上却学着上一世娴妃的模样，装可怜扮柔弱，暗中一步步筹划复仇。


我要让他心愿落空，我要让她被爱人抛弃。


我要负我的人，一个也跑不掉。


而那个我曾辜负过的人，这一次，我们会拥有不一样的结局。
