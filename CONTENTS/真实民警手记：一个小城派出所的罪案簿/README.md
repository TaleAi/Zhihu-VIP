## 简介
每一起案件发生后，警方都会从当事人身边的熟人着手，作为盘查的起点。想来是不是很耐人寻味？我们每个人、每天，都会和许多人打个照面、聊句闲话。他们却是我们身边，最熟悉的陌生人。


大学毕业后，蒋述回到生活了 20 多年的家乡，当了派出所民警。


在这座特产豆腐和酥瓜的南方小城里，他突然发现：自己熟悉的小镇其实一点也不平静。


他经手的案子里，受害者和嫌疑人大多和自己有着千丝万缕的联系，那些熟悉的面孔背后，多得是他不知道的秘密。 


![img](https://pic2.zhimg.com/v2-4e1da082b2a142f47212118564160bd0.png)

作为一个基层民警，每天会遇到什么人？


辖区常去的网吧，老板竟然是「公斤级」大毒枭


自己同小区的老邻居，是技艺高超的盗墓贼


不苟言笑的所长，还是把女「毒鬼子」当朋友，被小年轻们起哄的老大哥……


  




每个基层警察，都是一部关于「活着」的百科全书。


他们经年累月地，经历着社会底层真实发生的人生悲喜。


既是看客，又参与其中。


每一个惊险故事背后，都是血与泪、绝望和希望交织的人生。


世间的大幕缓缓拉开，再也没有一页能让人轻松翻阅而过。
