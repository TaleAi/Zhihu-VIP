## 简介
从前的日色变得慢，车，马，邮件都慢，一生只够爱一个人。


在信息流通缓慢的年代里，书信是人与人之间沟通最重要的媒介之一。


如今，那个时代已经远去，那些人、那些事件，已经成为我们解读历史的坐标；然而那些纸张、那些字，却穿越时空的限制，在我们面前缓缓舒展，讲述着一个个令人唏嘘的故事。


· 「醒来觉得，甚是爱你。」


朱生豪与宋清如的书信里，藏的不只有连绵的情谊，还有乱世下一个理想主义者的殒灭。


· 「他日抗日战争胜利后，你为世界名将，乘舰过吴淞口时，如有波涛如山，那就是我来见你了。」


郭汝瑰的遗书里，不只有他视死如归的革命情怀，更有他作为共产党人潜伏国民党高层深切的报国热情。


爱人的思绪、家国的情怀，甚或是世界变局的瞬间、历史走向的脚印，都在这方寸之间，得以永存。
