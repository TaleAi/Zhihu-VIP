## 简介
一家直抵灵魂深处的深夜小酒馆


作为一家酒馆，「咸叙酒馆」显得有那么几分不同，它低调、温暖、闲适，安静地伫立在夜色中，迎接一位又一位属于它的客人。这里的老板娘迪娅从不劝客人喝酒，只是在浅酌几杯的休闲时光中，听他们讲述自己身上发生的点滴故事。迪娅将这些故事记录在了她的日记中，或许，其中也有让你感到似曾相识的人生片段？


2022 年，知乎盐选会员联合帝亚吉欧，共同呈现「咸叙酒馆」，以「盐选」品质，讲述十个与酒相关的真实故事。感受过这些杯中时刻，你会发现，原来在这个充满烟火气的人世间，最耐读、最好看、最能让人产生共情和共鸣的，是我们普通人的鲜活日常。没有大悲大喜，也没有生离死别，却是我们每个人都可能遇到或正在遇到的。


这里有故事也有酒，那么，你愿意来「咸叙酒馆」坐坐吗？
