## 简介
从前，书信很慢，车马很远，一生只爱一个人。如今，爱情与你我，甲之蜜糖，乙之砒霜。缘分天定，一眼万年，怨偶佳成，日久生情。也许，柴米油盐，老夫老妻在玻璃渣中寻找一点甜，最终相携一生，但更多的是，经残垣断壁，如今满墙夏花初开。


求而不得的爱情，也许是门不当户不对。


也许是身份的羁绊。


也许是他未婚，她却要成为别人的新娘。


也许是他爱而不说，她求而不得。


爱情，只有敢爱，它才能开出最美的花。
