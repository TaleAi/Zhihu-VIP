## 简介
大器可以晚成。


在历史长河中，有些人含着金钥匙出生，赢在起跑线，也有人开局一手烂牌，毫无出息相。


为混口饭吃，他们当过猪倌、工匠、城管、狱警、农民、赘婿，尝遍人生百味，饱受折辱和白眼，甚至经历过了险象环生。


靠咬牙坚持、奋斗不息、审慎权衡或选择，他们最终等来了机遇，扛到了拐点。有的少年得志，有的中年起飞，也有的大器晚成，留下了许多精彩的故事和曲折的传说。


本专栏聚焦那些成功逆袭的历史奇人，复盘他们不可复制却充满现代启示的一生——


**看千古第一赘婿韦皋打脸老丈人，历经二十年漂泊后荣归故里，成为唐一代蜀地最杰出的封疆大吏；**


**看丙吉从一个普普通通的小狱警，逆袭成为汉武帝国的一代名相；**


**看青年时代在边疆放牧的牛羊大户马援，跟随刘秀打仗时发明沙盘实现对敌降维打击，管金融时重铸五铢钱助力「光武中兴」，越南叛乱时成为「马革裹尸」的伏波将军……**


他们只需要一个舞台，金刀在手，富贵不愁。
