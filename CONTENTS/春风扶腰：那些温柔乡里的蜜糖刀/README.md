## 简介
这里的男人都是大魔王，这里的女人都是降魔女。这是一个任你惊天伟业也逃不出爱情紧箍咒的故事合集。春风扶腰，桃花入眼。这里除了有催泪的情，还有扎心的刀。


《太后又失踪了》


她只不过进宫看看姐姐，就被那摄政王扶上了太后之位。傀儡太后不好做，她索性堕落给他看，败了名声丢了德行，带球跑不说还拐走了小皇帝。摄政王不干了，窃国他不怕，偷走他的心，算怎么档子事？


《女将军在上》


他说，没想到杀人不眨眼的女魔头也有这么撩人的时候？


她说，没想到吃人不吐骨头的摄政王也有这么纯情的模样？


她一个落魄女将军和大权在手的王爷厮混在一起，不过是各取所需。


只是，庙堂刀尖下滚过的人，没有谁是无辜的。
