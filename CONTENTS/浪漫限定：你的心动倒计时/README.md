## 简介
你是我专属的浪漫。


撒糖专业户「小柒崽子」新专栏甜蜜上新。


所有的为时已晚，其实都是恰逢其时。无论何时，你都是我唯一的心动。


**重逢**


同学聚会上，他功成名就，带着漂亮的女朋友来的。 


她说：「你当初要是没放手，这会儿傅氏集团的太太就是你了。」 


他坐在对面，拥着女朋友，神色冷淡：「都是过去的事。」


我一直在期待一场体面的重逢。 然而为时已晚。 


在他注视下，我默默捂住了手腕上的疤痕，牵强地笑笑：「是，我们两个……早没关系了。」


**少来得子**


我失忆了。 


某天，突然有个帅男人带着孩子找上门。 


一脸幽怨：「你不要我，连孩子都不要了？」 


我过上了起早贪黑，养家糊口的日子。


直到后来，我在街边的财经杂志上看见了男人的脸。 


他已婚，妻子是知名设计师。 


回家后，我恶狠狠地揪住他的领子，「社畜的便宜你都占？该死的资本家！」 


他把我光鲜亮丽的照片贴出来，温柔一笑，「不好意思，亲爱的，你也是该死的资本家。」
