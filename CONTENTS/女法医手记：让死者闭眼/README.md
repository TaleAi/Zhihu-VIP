# 女法医手记：让死者闭眼

## 简介
女法医记录的十三起不为公众所知的杀人案件。它们或过于血腥暴力，或社会影响太坏，不能见诸报端。女法医淑心奔波在各个案发现场，聆听「尸语」，用高超的法医学知识破案，让死者闭眼。


我叫淑心，是个法医。女的。


你一定在影视剧里见过许多女法医。其实那大半是假的，做我们这行的，女人极少，可以用凤毛麟角来形容。


一个原因是，女人大多胆子小，常年近距离接触腐尸、浮尸、碎尸，细致入微地分析刀口、枪伤、硬物打击伤，一件件一层层地剖开和检验人体内脏，对于女人脆弱的神经而言确实过于残忍了些。


另外，法医这行没有休息时间，一天二十四小时准备，有案情随时出发，有时候刑警们恩威并施地或命令或请求尽快拿到检验结果，法医就要加班加点，点灯熬油，弄得自己家里头老公怒孩子怨，根本就无法胜任妻子或母亲的角色。


我做法医之后，很快就学会了不再和人握手。因为我发现，有些人和我握手后，会找个借口溜去卫生间，就着肥皂在水龙头下狠命地搓洗他们和她们肮脏的爪子。


而我的手每天在消毒药水里泡到起皱，也许是世界上带菌最少的手。不过既然人家有顾忌，我也就不再去惹他们不高兴。


* **在我的手记中，记录了我亲身参与的案件，一共十三起：**

共有相拥而死、校园鬼影、尸体痉挛、畸胎、尘归尘土归土、硫酸伤人、空中目击、十六年冤狱、潜水惊魂、心理痕迹、黑太阳、死亡艺术、黑水迷局。


这十三起案件或者由于过于血腥暴力，或者社会影响太坏，不能见诸报端，所以不为公众所知。但毕竟都是公安系统侦破的经典案例，而法医也在其中起到关键作用，案件不乏警世喻人的意义，我不揣浅陋，在此记录下来，与你分享。


  




*\*案件全部来自于我真实的职业经历。*  

 *\*但出于敏感和保护相关人员隐私的理由，文中涉及到的地名、人名我做了模糊化处理。*
