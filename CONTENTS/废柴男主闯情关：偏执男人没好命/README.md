**《皎皎明月》**


自古以来，皇宫就盛产白莲花，且品种良多。


江玄明成长过程中最重要的阶段，被一朵中级白莲教养了几年，


青出于蓝而胜于蓝，潜移默化里，他早就是宫中开得最好的一朵盛世白莲。


不过他这朵白莲很有脾气，一般不开，或者说只在直女许芷面前开。


众所周知，但凡一个人沾了个「直」字，那么她身边必定不止一朵白莲。


  



**《长梨别》**


季长梨，你的名字不是不好听，是寓意不太好。


季长梨，我种了四季花，也亲手扎了秋千架，你怎么不夸夸我。


季长梨，接吻应当闭上眼睛。


季长梨，我们不过是错过了两年，为什么你一辈子都不属于我了？


季长梨，你成亲那日的装扮很好看。


季长梨，我会在奈何桥上等你一百年，等你来骂我。


  



**《迟迟春》**


新年的烟火中，我许下最后一个愿望。


心爱的姑娘啊，愿你一生平安，愿你万事顺遂，


愿你和真正心爱之人岁岁年年。


  



**《何枝可依》**


明月应该高悬于苍穹，但总有人想把她拉进淤泥中，


试图用她早已变得微弱的光芒来拯救自己的一颗肮脏的心，


可是也会有人重新把她捧到天上去，好让她再次变得皎洁。
