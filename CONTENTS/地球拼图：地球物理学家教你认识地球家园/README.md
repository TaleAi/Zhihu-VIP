## 简介
本专栏地球物理学家通过专业、趣味性的讲解，带孩子们了解我们赖以生存的地球，由 52 块地球拼图开启小朋友们对地球的探索之旅。在真实、有趣的科学学习环境中，让孩子们从小听科学、学科学、懂科学。


我们生活赖以生存的地球家园，你对它有多少了解呢？


海洋是如何诞生的？一场大雨为什么能冲垮大山？


地球也有体温吗？地震到底是怎么发生的？


......


从风云变幻到气象万千，从千山万水到沙漠戈壁，从江南夜色到塞外风雪，从小桥流水到大漠孤鹰，地球家园用无私的爱孕育着文明。


本专栏由国内一线科学家为孩子们讲解科学知识。通过科学家的声音，将科学观念和知识传达给小朋友们。在这个信息漫天飞舞的时代，能够让孩子们听到有价值、有趣的科学知识，为他们营造一个客观、真实、趣味的学习科学的环境，让孩子们从小热爱科学、发展科学。
