# 江湖不可能这么萌

## 简介
奇妙的江湖物语，有糖有刀；高萌的搞笑故事，不止可爱！这里有故事和酒，你人来了就好。


警告！这个江湖太奇怪了！ 请慎行！


发现小萌新：谁说江湖险恶了，狐狸怎么不知道？


多么痛的领悟：江湖套路多，「过气的大侠」太年轻！


厉不厉害：公子不在江湖，江湖总有公子的传说！


情为何物：人在江湖飘，糖从天上来，饕餮与鱼的爱情很突然，甜甜甜！


道家哲学说，有人存在的地方就是江湖。


那么遇到一个过气的大侠算是踏入江湖了吗？


和小妖怪一起捉大妖怪是不是就在行侠仗义了？嫁给了一个凶巴巴的武林盟主是不是就可以在江湖狐假虎威了？


成为一个职业酷炫的杀花人，是不是就达成了闯荡江湖终极成就？


可是不对呀，这跟你们说的江湖不一样啊……


从前慢，一生只够爱一个人，一辈子就像一团小火，勤勤恳恳不离不弃地炖着一锅牛腩。


这里是你从未见过的奇妙江湖，每天都发生着奇妙的故事，等你来，饮上一杯浊酒，看这江湖悠悠。
