恋爱虽然麻烦，但更讨厌孤独~


当你的世界发生了前所未有的变化，当一个完全预料不到的对象闯入你的世界，你会怎么做？


这个专栏，看看他们的选择。


**《我和虎鲸有个交易》**


水族馆的虎鲸可以化成人形。


于是我跟他做了一个交易。


他帮我杀掉虐待我的混混楚越，我当他一个月的宠物。


后来，楚越死在水族馆，阿鲸带着我入深海。


深海的一切都很新奇，可我身为人，无法离开地面。


阿鲸带着我重回岸边，没想到上岸后，我竟然看到了已经死去的楚越。


这究竟是怎么回事？


**《人鬼情未了》**


变成阿飘后，我依旧住在之前租的房子里。


一个漂亮弟弟成了我的室友，而且，他好像可以看见我！


他好奇我的死因。


我困惑他的身世。


在结伴捉鬼的旅途中，我们俩越走越近。


但人鬼毕竟殊途，为了让他过正常的人生，我狠心拒绝了喜欢的他。


然而，这一切，真的会如我所愿吗？
