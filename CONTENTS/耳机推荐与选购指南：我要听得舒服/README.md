## 简介
耳机市场鱼龙混杂，如何选到适合自己的那款？


近几年，由于国产智能手机的迅猛发展，并且市面上大多高端智能手机已经不再标配耳机，蓝牙耳机已然成为了年轻人几乎必备的数码产品之一，头戴式、挂脖式、骨传导、TWS 真无线等不同形态的耳机百花齐放，各厂商在耳机的技术上不断创新，提升用户的听觉体验。


那么，如何在鱼龙混杂的蓝牙耳机市场中挑出最适合自己的那一款呢？


耳机说明书上的各种参数、专业名称各代表什么，对用户体验又有怎样的影响？


本专栏会对大家购买蓝牙耳机时最关注的几个因素（蓝牙版本、降噪、音质、续航、舒适度等）进行详细讲解，同时把蓝牙耳机分成多个维度（形态、类型、价位以及品牌等），对其性价比分析排行，并对收录的具体产品型号进行评价，最终帮助你出挑选最适合自己的蓝牙耳机。
