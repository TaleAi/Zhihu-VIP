## 简介
这是一篇长篇犯罪小说。一场天才心理侧写师和反社会高智商杀手间的生死角逐。连环杀人案件昭示着杀手的重新现世，心理侧写师白秋安临危受命探查真相。但当光芒万丈的侧写师转瞬成为嫌疑人，他该如何拯救自己，揭开一切阴谋之下的真相？


社会的稳定遵循着四角平衡原则。


栅栏里的羊群被狼猎杀，猎犬站在狼的对立面。


除了三者之外，还有——牧羊人。


牧羊人看似是善意的主导者，但他们善于利用资源，擅长伪装，


他们可以是维持秩序的「主人」，也可以是潜在的「恶狼」。


这是天才心理侧写师白秋安提出的「牧羊人理论」，他是国内最年轻心理学教授，有着与生俱来的「超能力」，能够通过事件共情别人的想法。


但他却同时患有严重的躁郁症，还是双重人格。


在连环杀人案打破城市宁静之后，白秋安临危受命，协助警方追查凶手。奇怪的是，每一场杀人案，似乎都和八年前已经消失的高智商反社会杀手「冰蝶」有关。而每一场案件的嫌疑人，都对白秋安表达出不同寻常的崇拜，他们说：


「白秋安，其实你和我们是一类人。」


案件的迷雾和越来越频繁出现的子人格快将白秋安逼至崩溃。某一次醒来后，他发现自己居然在一处新案发现场，手持尖刀，旁边就是倒在血泊之中的受害人……


似乎是转瞬之间，他从高高在上的心理侧写师，变成所有人怀疑的那个不可说的杀手——「冰蝶」。


 


父亲死前的诅咒回荡在他耳边：「秋安，我们是天生的犯罪者。」


心理咨询师女友的话语出现：「秋安，你想没想过，你的身体里，谁才是子人格？」


他，是那个潜藏在羊群中的「牧羊人」吗？
