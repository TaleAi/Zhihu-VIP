第二个我再详细的讲一下给大家分析一下在每一个阶段除了刚才说的融资金额释放比例以外你还应该注意哪些点主要呢我是从条款以及董事会席位以及其他各方面给你做个提醒那我们首先来说，种子轮和天使轮，种子轮和天使轮刚才我也讲了，基本上就是一种比较早期的一种融资方式，在中国也好，硅谷也好，基本上个人参与者为主，投资者主要是个人，这种情况下的话那我的建议那就是你给他的股份最好是普通票不要优先股，大家都知道优先股有很多很多权利，优先股优先退出啊，赎回啊各种权利你给他普通股，因为什么呢因为他做为天使本来也是你共同创业的一部分事实上很多天使投资人跟创始人之间都有很强的关系纽带，所以我建议你给他普通股第二呢一般来讲种子轮和天使轮，不太可能给董事会席位，当然也有例外因为为什么呢因为人也比较少，她也比较信任就让你就是执行董事兼总经理创始人就是董事长总经理也不需要其他的一些董事会因为你比较小嘛。这是 preA 之前的天使轮和种子轮，preA 到 A 以后公司已经逐渐长大这个时候我觉得你可能比较难的去阻碍投资人拿到一个董事会席位而且你也比较难的让她不再享有所谓的优先股的权利了，为什么呢因为 preA 到 A 的投资机构很有可能他已经是机构投资人机构投资人他要保护自己他更系统化他有律师团对他更加的要求这些条款。 


那么这个时候你要注意的是什么呢在我看来的话要注意几点：一点就是你尽量给他的优先股的条款要相对要，不要那么多，而且有些条款你要控制住中，为什么呢，因为 preA 和 A 的条款会影响 B 轮和 C 轮的条款，我举个例子优先股里面有个优先退出的倍数的条款我建议你定百分之一百或者一倍，这个条款什么意思呢，就是说他退出的时候因为清偿或者是并购退出的时候首先应该拿到他投资金额的一个倍数然后再按股权比例来分配所得的剩余收益那如果你刚开始定了一百五或者两百或者两倍讲那么后来 B 轮 C 轮投资机构他可能会参照执行所以我在想你 preA 和 A 一旦有优先条款的时候你相当于立了一个标准这个标准一定要对你有利，我是站在你创始人角度去想这个问题这个时候一般来讲他要个一个董事会席位董事会席位的状态当时应该是公司两席投资人一席，那么这种情况下有可能你会请你公司中的另外一个你信得过的同事成为一个董事你自己成为董事长然后再加一个投资人董事的席位，三席那么 B 轮以后呢？以后你要注意另外一个问题首先 B 轮以后他肯定会有优先条款同时的话因为它的比例大概还有百分之十所以他也可能会有一个董事会席位那个时候你的董事会席位可能会变成五席，投资人两席，公司三席，你会吸纳另外一位你信得过的同事或者是外面的人员成为你的董事会成员这样的话你还是控制董事会因为三比二嘛，这个时候你要注意一点什么呢就是很有可能那个时候 B 轮时候可能会有一些战略投资人进来当一般我的建议是 b 轮 C 轮以后才考虑战略投资人，但是事实上我看到的情况就会有相应的战略投资人进来因为种种原因，那这个时候你要做到是业务上一定尽量要保持独立英文叫 independence 独立是个好东西，独立会增加你的估值不独立会减少你的估值所以你在这里战略投资人进来的时候不要签太多的排他条款，这个是在这一时候要注意的事情，那么我们就说 C 轮，C 轮以后基本上属于比较靠后的这个时候，你要注意什么呢？因为你经过那么多轮的轮次当然中间，可能你不一定会那么多人次你的股份比例，你和你团队的股份比例一定是大大的被稀释了那这个时候你要注意什么，很有可能你团队的股份包括你和你的团队加起来可能已经不是单一最大股东哦，那这个时候对你会怎样的影响在各种协议条款里面你不是最单一最大股东，而且你肯定也不一定有百分之五十的时候你要保护自己也要有些条款要保护自己相对在有些条款中要体现出这个变化那么最后一个是 C 轮以后 C 轮以后有一种情况是我看到那些公司最后有一个股东不是你不是团队外面的股东战略投资人或其他股东已经超过了百分之五十这个时候我觉得你应该注意的是另外一个问题出现了就是你以前觉得公司所创始的我理所当然是创始人理所当然是经营管理团队成员理所当然是我是 CEO，但是如果一旦有一方部分超过百分之五十对不起这个公司已经从名义上来讲，经营团队应该有超过百分之五十以上的股东来进行任命那这个时候我之前就遇到过这种情况我上一家公司的时候，某家公司巨头公司就已经事实上超过了这个股权比例。


  



那这个时候经营层怎么办？CEO 怎么办？一个办法就是需要大股东作出一个承诺在多少年的时间之内 guarantee 保证我是 CEO ，这是一个 CEO 的 guanrantee 条款这个条款异常的重要这个是我们在 C 轮以后当有一方股份比例超过百分之五十以后我管理团队怎么保证自己继续能执行这个公司的管理层这么一个条款的保护，以上我大概分析了从很早期的种子期到最后 C 轮。C 轮以上大家要注意。这点从条款董事会席位从业务的谈判点给大家一些建议，这一讲大概就讲到这里，下一讲我们将开始讲如何去找投资人，谢谢。
