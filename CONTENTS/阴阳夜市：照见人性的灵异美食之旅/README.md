## 简介
魑魅魍魉，神仙奇兽，最终照见的是人心。本专栏讲述在一条阴阳两界边缘的街市上，发生的灵异、惊悚、爆笑、温馨故事，悬念重重背后，是人性的修罗场。


夜深了，在八香街深处，一家烧烤店按时开张。名为「暗夜烧烤」，吃啥补啥，它的老板，正是阴间的帅气冥君。


神仙太白，为了爱情丢失了心。


烧烤摊的常客胖子老沈，为了还够冥府的钱去投胎，每夜必来吃串。


墨龙敖放，从小活在龙王父亲，「别人家孩子」的阴影中。


女孩小夏，摆脱了鬼魂缠绕，成了店小二。


众人在八香街，救凡人，逞恶鬼，爆笑欢脱，却也有不少令人唏嘘之事。


深爱的妻子，其实早已阴阳两隔……


医院的少女，得了莫名的硬化症……


每天点一百单外卖的公司，却空无一人……


八香街的小摊越来越多，蜜桃酒，爆米花，麻辣烫，龙骨灰奶茶，活死人刨冰……每道「美食」之后，都是一个回环婉转的阴阳故事，有惊悚，有温馨，有欢笑，有泪水。


阴阳夜市的食客和店主，无不有一段纠缠阴阳的故事。跟随本专栏，在神鬼之后，看见人性。
