## 简介
车里的秘密，比想象中危险。


我喜欢的女神小姐姐因为乱停车，车胎被人放气，我被叫去帮忙换备胎，补胎时却发现漏气的轮胎被人塞入了诅咒符纸「三天内必死」，而当晚小姐姐就死了，凶器上留有我的指纹，我成了第一嫌疑人......


我老公有路怒症，开车路上经常骂东骂西，因为这个也没少跟人闹矛盾。有一天我老公停车突然发现后视摄像头有一个可怕的鬼影，原来是车尾摄像头被贴了一张贞子照片……


连续好几天，我每天早上准备开车去上班的时候，都发现我车的油箱莫名其妙变满了。可明明前一天我下班开回来的时候，油箱都已经被消耗了一些……
