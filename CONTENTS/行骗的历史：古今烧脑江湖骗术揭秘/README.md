## 简介
历史上有哪些著名的骗局？


在这个专栏里，记录了中国古今江湖骗术与反骗历史故事。


从宋朝开始，跨越元明清三朝，再到民国，再到 80 年代，浩浩汤汤绵延了足足 900 多年。


如宋代的人口拐卖案、明代的淫僧求子骗、清代的绝世神偷盗宝案、民国的假古董案、80 年代的特异功能等等......


人在俗世，难逃七情六欲，而「骗」字由此而生。


然而在不断迭代的骗局中，一直暗藏反骗之人，**揭示骗术之秘，授以防骗之法**。


本专栏是以现实元素为背景的非虚构写作。
