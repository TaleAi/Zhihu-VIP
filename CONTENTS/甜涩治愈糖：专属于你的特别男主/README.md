## 简介
本专栏是偏甜虐的现代言情合集，包含数个治愈系悠长低回的爱情故事。


世间情动，总有人是你落入梦境里的妄想。


既是妄想，那就吃了这盒糖，做一个关于他、关于爱情的梦吧。


希望这里有一个，今晚你想做的梦。


**《星星不会躲》**


四年前初遇，他在沙滩上拿着烟火棒对我笑呀笑，奶乎乎的样子让我忍不住想靠近。


后来，我们有过多么可爱的曾经啊，他像是灰暗生活里闪过的一道光。


可无意间，我发现这道光，原来是一把他藏在背后的匕首。


刺中我的时候，稳、准、狠。


四年后，他说要给我一个解释。


我这才知道，原来这段路他一直在奔向我……


**《你是绚烂白昼，也是沉寂黑夜》**


吐槽老板被发现，我打好辞职信准备开溜。


谁知道老板却突然开始撩我。


我打听到，老板是个记仇的狠人。


所以做好了他撩到我，再甩掉我的准备。


谁知那天他喝醉了，把我壁咚在墙上：


「记得吗？我说，以后我有钱了，不管你在哪儿，我一定会找到你，把你牢牢地禁锢在我身边。」


他指尖凉凉的触感，划过我的脑海。


原来是他。
