## 简介
当妄想冲破囚笼化为利剑，那些尘封的童话，也将被刀刃划破，分崩离析……


一睁眼，我成了童话故事中的恶毒反派。


  



**- 白雪的野心 -**


自从得知白雪公主是个男人后，头顶就好像落下了一道无形的网，包裹得我喘不过气。


诡谲迷雾之下，是不知所踪的猎人、是死去多时的王子、是破碎到不可拼接的魔镜……


当一切真相浮出水面，我被他彻底桎梏。


男人几近痴迷地欣赏着我满脸的惊异，随后吐出疯狂病态的话语。


「母后，我之所以会吃下您为我准备的毒苹果，是因为我拒绝不了您对我深沉而致命的爱啊。」


  



**- 人鱼的陷阱 -**


第二次醒来，我又进到了美人鱼的世界，尽情扮演着故事中的恶人。


让我始料不及的是，小美人鱼并非钟情于王子，也没有表面上那般纯真无邪。


看着我与王子成婚，扭曲逐渐在他心底蔓延，因嫉妒而衍生出可怕的占有欲，不断腐蚀着名为理智的那根弦。


我是否还会沉浸在他善良虚伪的表象中？


  



**- 与小红帽的末世逃亡之旅 -**


第三个故事起，一切都脱节于我记忆中的童话。


小红帽当着我的面，杀死了他已经变异的外婆，这镇上到处都是行尸走肉。


最后的最后，尸潮将我们围住。


他在一片血光嘶吼中紧紧抱住了我，耳边回荡起那人扫过舌尖的缱绻甜蜜。


「如果我们终将被这些怪物的利齿所穿透，那不如就由我把你吃掉吧。」


  



**- 睡美人的獠牙 -**


当夜幕高挂起一轮血月，那沉寂在古堡中的睡美人也睁开了猩红的双眸。


荆棘捆住了我的双腿，下一秒，尖锐獠牙刺破了我的肌肤。


随之响起的是他温柔低醇的声音：


「没有死亡，没有苦痛。待在我为你打造的囚笼里当只乖巧听话的金丝雀，不好吗？」


……


破碎不堪的童话、崩坏诡谲的故事，以及他们疯狂病态的爱意。


是接受他们的扭曲，被他们囚禁豢养，还是义无反顾地逃脱他们的掌心？
