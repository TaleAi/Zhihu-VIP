**沟通是一个技巧，幽默则是最高级的情商，并且能够后天习得。**


**人们往往以为：幽默=讲段子？**  **幽默=油嘴滑舌？**  **幽默=自黑？**  **幽默=讽刺？**

**幽默感不是天生的，也不是只属于那些妙语连珠、逗得别人捧腹大笑的人；没有幽默感的人，难道只能一辈子低头做个「闷葫芦」？**
