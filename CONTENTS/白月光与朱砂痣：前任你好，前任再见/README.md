## 简介
视奸前任、时不时撩拨前任、报复前任……6 个剑拔弩张的前任故事，6 段患得患失的情感经历，折射出亲密关系带给我们的不确定感和失控感。


在张爱玲的笔下，每一个男子都有过两个女人，娶了红玫瑰，久而久之，红的变了墙上的一抹蚊子血，白的还是床前明月光；娶了白玫瑰，白的便是衣服上沾的一粒饭黏子，红的却是心口上的一颗朱砂痣。


在很多人心目中，前任是白月光，现任是米饭粒；前任是朱砂痣，现任是蚊子血。


正因为如此，「前任」是一个在亲密关系中很敏感的话题，甚至被很多人当作「雷区」「禁区」，也很容易引发一些相爱相杀的故事：


**·** 他每次和女朋友亲热时，女朋友的微博都会收到来自陌生人的私信，有时是一段辱骂，有时是一张他的裸照，有一次甚至发来了一张产检报告。女朋友凭着女人的第六感，怀疑他的前女友在房间装了监控。他决定把前女友约出来，让现任去撕前任……


**·** 综合条件普通的她，遇到了一个完美男朋友。男朋友对她呵护备至，两人很快走进婚姻殿堂。直到有天她去接在同学聚会上喝多的他，看到了另外一个女人，才知道，原来自己不过是丈夫求而不得的替身罢了……


**·** 前男友半夜 11 点多给她发了一条微信：「想你了」，3 秒钟后又马上撤回。她其实已经看到了，手停在对话界面良久，那句「我也是」不知道该不该发送出去……


每一个故事中，可能都有你我的影子，有我们的不甘，有我们的嫉妒，有我们的愤怒，也有我们的眼泪。
