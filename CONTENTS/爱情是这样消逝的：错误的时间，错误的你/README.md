## 简介
8 段「情不知所终」的残酷爱情，8 个普通人跌宕起伏的人生经历，写尽他们爱的真诚、执拗、痛苦、煎熬，却无怨无悔。即使爱情无果，我们拥有的记忆，也弥足珍贵，足够记录成一部「罗曼蒂克消亡史」。


遇见她/他之前，我既不懂美好的爱情，也看不清苦涩的现实。


大二失恋后我颓废了，暗恋上门口网吧的女网管，她却喜欢长期网吧包夜的飞哥，我们三个的感情注定无果，走出网吧那一刻，就是梦醒时分；


18 岁高考落榜，我走上了工厂流水线，是洗头妹柠檬还是工友阿玲？在这里，我同时爱上了两个女孩，却一个都无法带走；


我的发小是一个小破游戏主播，自从认识了一个女「徒弟」，他拼命爱她，她不断出轨，为了搞一波快钱给她盖房子，他决定去东南亚赌场卖命……


有些爱情，生来不够成熟，没有生存空间；有些爱情，即使始于相濡以沫，最终也会相忘江湖；有些爱情，有人爱而不得，有人弃如敝履；有些爱情，明明开始就知道结局，总有人一边大口吞咽泪水，一边爱得卑微。


**因为爱情是种易逝的奢侈品，所以拥有过就会珍惜，在这错误的时间，遇到错误的你，我却甘之如饴。**


**我从未后悔爱过你。**
