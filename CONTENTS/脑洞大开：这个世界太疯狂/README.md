## 简介
一本正经的荒唐。


未婚夫带我回他老家散心。


他父亲招待我的第一餐，


是带着鲜血的人眼珠。


1


意外流产后，我的未婚夫张存带我回他家散心。


那是南方某小岛上的渔村，风景绝美，但是闭塞少为人知。


张存小时候在那里长大。后来父母离婚，他跟随母亲一起来了大城市生活。这也是二十多年来，他第一次回到这里。


渔村在一处远离陆地的小岛上，我们需要坐皮艇才能到达。


上岸的时候已经是傍晚了，夏日晚风，明明应该是很惬意的，可我却有一种说不出的不适。


领我们上岛的老伯说，我俩正赶上村中的祭祀盛宴，所有人都在祠堂祈求来年风调雨顺，我俩也应该去敬神。


破旧的祠堂里，密密麻麻的人群跪倒在地，用难以听懂的方言低声祷告着什么。


香灰的味道和鱼腥味混合在一起，浑浊的让人难以呼吸。


出身神婆世家的我，立马察觉出了其中的不对劲。


诡异神像前，摆着两颗巨大的红烛，火光摇曳，闪烁着血红色的光芒。


与寻常祭祀的贡品不同，这里的贡品都是牲畜被掏空眼睛的脑袋。


我定睛一看，发现神像的眼睛里竟然也没有瞳孔。


无瞳之眼是大凶，这里供奉的是邪神！
