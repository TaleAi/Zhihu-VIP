# 春秋战国的兴亡过手：盘点那些曾经称霸过的诸侯国

### 状态：已完结，共 8 节

### 存档日期：2022-05-21

## 简介
春秋战国是中国历史上一段大分裂时期，也是军事、科技、外交、思想、人才、最为活跃的时期，在这段五百多年的动荡时期中，各个诸侯国无论规模大小，都在极力地施展一切手段，以图在这个乱世中谋得一席之地，本专栏以故事形式和历史分析相结合，为你盘点那些曾经在春秋战国时期一度称霸却又最后走向衰亡的国家，为你揭示那段不为人知的有关历史兴亡的秘密……



【天下诸侯，莫非郑党——第一个敢与周王室正面硬刚的国家】


无论是血缘关系还是地理位置，这个国家离周王室都是最近的，但偏偏这位周王室的「近邻」，却给春秋的乱世开了一个「好头」：兄弟相残、号称这辈子不见亲生母亲、干涉他国内政、第一个在春秋时期称霸的诸侯国……当然最让周王室颜面扫地的，是和这个国家互换人质，最后自己还被这个国家打败，从而彻底丧失了对各诸侯的控制权……总的来说让周王室真正失去统治权的，正是这个国家——郑国。


  



【仁义道德能当饭吃吗？——最后的贵族之国】


宋国既不是周王室血亲，也不是有功之臣，而是殷商王朝王室后裔，并被周王室特准使用天子礼乐来祭奉商朝宗祀，这个身份特殊的诸侯国在春秋战国时期虽然逐步褪去了殷商的那股原始、血腥、野蛮的气息，但却保留了更多的夏商先民习俗，君王将相亦是如此，然而这种过时的「上古遗风」在不按套路出牌的春秋战国时期，真的能当饭吃吗？


  



【从独霸中原到三家分晋——一个超级大国的分崩离析】


晋国长期以来一直以中原超级大国自居，它也承担着大国所在的使命：护卫周平王东迁，在齐桓公后接过霸主的接力棒，阻止楚国的北上，可以说一部春秋历史就是晋国和楚国互相争霸的历史，晋国也是中原文明最为杰出的代表，文化、军事、工商农业都十分发达，这也是它独霸中原多年的原因之一，然而就是这么一个超级大国，却免不了最终被瓜分的命运，三家分晋，标志着春秋时期的结束，同样也标志着战国时代的开始，这样的分国事件为何没有发生在其他国家，而这背后又有什么样的故事呢？


  



【从「傻讲道德」到「世风日下」——吴国兴亡记】


吴国历来有「让国」的美德，从先祖太伯到吴王寿梦的几个儿子，都秉持着这种「美德」，然而在吴王阖闾看来，这种所谓的「美德」就是一种「傻」的体现，他继位后，在伍子胥和孙武的帮助下一度攻入楚国都城郢都，然而伍子胥的「鞭尸掘坟」却让吴国的「美德」一夜间荡然无存，一时间「世风日下」成为了批评吴国最多的声音，可吴国依旧还是成为了春秋中后期最强大的诸侯国之一，从「傻讲道德」到「世风日下」，吴国究竟经历了怎样的转变和起伏呢？


  



【争！争！争！杀！杀！杀！——鼎盛一时的越国为何突然销声匿迹】


我们都知道越王勾践卧薪尝胆的故事，勾践忍辱负重，经过十余年的艰苦奋斗，终于消灭了吴国，成为了春秋末期最后一位霸主，然而就是这样的一个强国，却在战国时期突然销声匿迹，史书对此也记载甚少，而这一切的源头，都要从越王勾践之后那一段「黑历史」说起……


  



【大争之世什么最贵？人才！——最出人才但又不重视人才的矛盾国家】


魏国地处于华夏文明的摇篮之地，也是战国时期最大的人才培养基地，商鞅、吴起、张仪、公孙衍、范睢、尉缭子……这些耳熟能详的名字都源于魏国，然而他们却没有一个人终生留在魏国效力，相反，他们的归宿和真正的大放异彩都在其他国家，进而改变了整个战国的历史和格局，是魏国不受欢迎吗？还是这其中另有隐情呢？追根溯源，还需要从一个简单的故事说起……


  



【远超秦国，却为何没有统一天下？——事不关己，高高挂起的国家】


从姜子牙立国建邦，到春秋时期齐桓公九合诸侯一匡天下，从战国初期的田氏代齐，到后来的桂陵、马陵之战，在春秋战国的历史舞台上，齐国可谓是绝对的「角儿」，然而这个将魏国从霸主位置上拉下马，将秦国打的直喊自己西番之臣，将燕国差点洗劫一空，甚至一度称帝的东方强国为何最后没有一统天下？相反当列国遭到秦国猛攻向它求援时，都是一副事不关己的模样，这究竟是什么原因呢？


  



【我荆蛮也，不与中国之号谥——被中原歧视的「非主流」国家】


一直以来，楚国都被中原国家视之为蛮夷，从来就没有被中原的诸侯国正眼相待过，等级也是最末等的「子爵」，然而楚国进取中原的心却从未改变，从问鼎中原到成为战国七雄，楚国的实力一直不容小觑，甚至在战国后期还一度击败过强大的秦国，创造了战国中后期山东六国对秦作战的最大胜利，然而它终归有些地方，和中原国家相比显得不伦不类，这也成了导致它衰亡的原因。
