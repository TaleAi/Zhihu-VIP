## 简介
让我们谈一场轻松简单的恋爱吧！


忙碌庸常的生活之余，有时候，我们真的很需要双向奔赴的甜甜爱情。

**《靠翻车撩到美食 up 主》**


我关注了一个美食 up 主，每天看他的视频，学做饭。


up 主做的菜，营养又好看。


但手残如我，每次都翻车。


我把火烧厨房的翻车图贴在评论区，收到的点赞甚至比 up 主还要多。


如是一周，up 小窗我。


「妹妹，给条活路。别学我做饭了，行吗？」


靠翻车撩到美食 up 主。这个逻辑好像有点奇怪？


但不重要了。因为我发现，up 主是个小哥哥。


而且，他长得好好看。


**《驯服冤种竹马的十八种方式》**


从小到大，陈勐最喜欢跟我攀比。这次秀的，是他新收养的流浪猫。


「俞灿，我有猫，你没有，你不嫉妒吗？」


我当然嫉妒。


是夜，我穿成了陈勐的猫。


陈勐厚着脸皮，搂着小猫，既要亲亲，又要抱抱。


于是我干脆利落地挥起爪子，对他哈气，「滚开吧你！」


很久之后，陈勐才跟我坦白。


「这猫是我从寺庙里求姻缘，顺便抱回来的。」
