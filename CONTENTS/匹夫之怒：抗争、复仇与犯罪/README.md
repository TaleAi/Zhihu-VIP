## 简介
直击重案现场


他们都是生活中的小人物，是他人眼中不起眼的老实人。


可当「老实人」被逼上绝境时，他们用自己的方式进行抗争。


这是关于小人物的抗争、悲剧与犯罪的故事。


* *贵州五兄弟追凶万里，风餐露宿，为弟报仇；*
* *北京母亲向妙龄女子泼硫酸，背后牵扯出另一桩命案；*
* *弟弟惨死长白山无人区，姐姐梦中破案。*

在法制与公义照射不到的阴暗面，还有许多因压迫、愤怒和无奈所产生的悲剧。


本专栏通过纪实的手法，还原案件现场，探讨悲剧发生的深层社会原因，警醒世人，避免重蹈犯罪悲剧。
