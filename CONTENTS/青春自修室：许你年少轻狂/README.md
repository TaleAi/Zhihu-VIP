## 简介
青春本来就短暂，年少怎么就不能轻狂？愿我们永远年轻，永远热泪盈眶。


他说好和我一起上大学的，却临阵放弃了。


他捧着我的脸说，「对不起，我爱上了你姐姐。」


都说爱情必须要轰轰烈烈，那我们为什么会如此潦草的离散？


——《唯一》


  



我喜欢了三年的男生，和我报了同一所大学。


第一次吃饭时，他却让我帮忙介绍我室友给他认识。


他说，看室友穿的那双球鞋，一定是个喜欢篮球的女孩，他就想找一个懂篮球的女朋友。


可是他忘了，我为了他还做了校篮球队经理，知道三分线、知道擦板球、知道投球命中率……


——《亲爱的路人》
