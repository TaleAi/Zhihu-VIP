## 简介
爱情总是会让人迷失心智，让一代代帝王被蒙蔽双眼，拉下神坛，最后推进深渊。让我们看看这古代的妖妃、公主们是如何霍乱江山，翻手为云覆手为雨。


自古深情留不住，唯有套路得人心。历史上有名的「红颜祸水」，哪一个不是让男人们牵肠挂肚？


但爱情总是会让人迷失心智，一个男人爱慕一个女子，可能会做出许多疯狂的举动。


\* 周幽王为博美人一笑，赏千金，烽火戏诸侯。自古夸赞美人都说「一笑倾城」，这一笑反倒是灭城，不是倾倒众生，反倒是倾塌城池。


\* 当一个掌权者的决策取决于一个女人的表情，历史的悲剧就离他们不远了。但凡有忠臣上奏的时候，妹喜就百般阻挠，眉头紧锁；当有奸佞之人前来阿谀奉承，妹喜就会赏金赏银赏美人。真是完美诠释「黑白颠倒是什么样子」。
