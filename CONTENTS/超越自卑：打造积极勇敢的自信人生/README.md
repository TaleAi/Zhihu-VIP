## 简介
自卑的人往往会陷入自我怀疑的苦恼，如果你渴望改变自我，想要掌控自己的人生，那么请跟着这节课一起来学习克服自卑的方法，完成从自卑到自信的蜕变。


自卑的人往往会陷入这些苦恼：「TA 那么好，我实在配不上」，「我不行，我会把事情搞砸的」，「TA 是不是不喜欢我啊」。


陷入自我怀疑一直是我们前进路上的绊脚石，缺乏自信更容易让我们处于不利境地。


在领导面前没法儿自信地汇报成绩，别人在台前风光，自己只能在台后任劳任怨；


有好感的异性不敢主动接近，错过可能很美好的感情，留下一生的遗憾；


人群中没有存在感，很难融入一个圈子，错失获得优质人脉的机会；


在客户面前局促不安，不知道该聊些什么，别人发展平步青云，自己还在原地踏步。


如果你渴望改变自我，想要掌控自己的人生，那么请跟着这节课一起来学习克服自卑的方法，告别自卑的人生底色。


本课程由张现利老师主讲，从认识自卑的原因、突破自卑的障碍、建立自信的方法和维持自信的秘诀等方面进行讲授，以专业的心理学知识和多年的心理咨询经验，帮助你消除负面情绪，做自己的主人，完成从自卑到自信的蜕变。
