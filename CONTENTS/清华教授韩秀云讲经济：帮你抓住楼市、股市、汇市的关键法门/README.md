## 简介
清华的MBA课，教你看懂经济，掌握先机


你最近是不是经常听到这些词：中美贸易战、消费降级、经济寒冬、公司裁员、股市大跌……


以为这些钮和你无关吗?


但你知道:


公司不断裁员,所在的行业命运将走向哪里?


使用的进口化妆品,即将涨价还是降价?


房价会不会降,买房还能不能当成一笔投资?


物价不断上涨,为什么钱越来越不值钱?


这些问题的本质其实就是经济学的问题!


不懂经济学?不清楚经济形势?你就无法做出正确的判断!


懂一点经济学,你就可以了解国家政策的背景,结合自身需求,定位发展方向和目标；把握经运转检势,多抓住一些机遇；合理理财,做到投资利益最大化，实现财富自由。


经济学研究的正是你所关心的其实,经济学并不难,并且你一定要学！不要被「经济学」三个字吓住。


本课程中，韩老师将通过一个个妙趣横生的故事,将经济学化繁复为精妙。即使没有经济学的基础也能让你了解经济的变化,让你抓住机会的时候,别忘了规避风险。
