## 简介
嘘，天黑请噤声！


古老的风俗传说与老一辈积累的经验，加上对未知和死亡的恐惧，让我们构建了一道道民间禁忌的枷锁。可怕的不是触碰这些禁忌，而是不知不觉间，我们成为了「禁忌」。


**-黑猫撞棺-**


出马仙三姑奶，说我命犯邪煞，二十六岁之前，不可参与任何和死人有关的事情。不听劝告的我，在二十五岁这年，参与了一次诡异的抬棺。黑猫撞棺，死者开目，传言「惊尸」者将遭遇不详，无一生还。


**-火葬场惊魂夜-**


作为一名遗体火化师，我上班第一天就要在午夜烧掉我的前任火头儿老夏。然而怪事接连发生，老夏尸体离奇失踪，被禁止打开的火化炉里传来敲打声，来去无声的红衣女子，敛尸房里的猫脸老太太，当一系列禁忌之门被打开，我能做的只有亡命狂奔。


……


本专栏包含 8 个邪门的民间禁忌灵异故事，故事主人公在禁忌边缘疯狂伸脚脚的同时，也在一次次触碰着人性的底线。最后提醒一句：你回头看，背后有人。
