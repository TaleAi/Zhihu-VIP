## 简介
贩毒，是十恶不赦的极端；毒枭，是穷凶极恶的尽头。


臭名昭著的鸦片战争，源头竟然只是一个毒贩头子的只言片语……


建立毒品王国金三角的人，其实是一支来自中国的队伍……


震惊世界的湄公河惨案，背后是贩毒集团精心编造的弥天大谎……


民国时期毒品猖獗，与上海青帮和时任总统之间的「生意」脱不开关系……


在毒品诞生之后的漫长历史中，无数年轻人因为沾染毒品而毁掉一生，只是为了片刻虚假的欢愉，以及对「功成名就」毒枭的效仿追随，却不知那些所谓的功绩，曾经破坏了多少家庭与希望。


专栏中记录了那些历史上最嚣张恐怖的毒枭，剖析他们如何用骇人听闻的手段完成贩毒事业的原始积累，并一步一步打造属于他们的犯罪帝国。


在鲜血和枪火的映衬中，我们可以看清每一个毒枭脚下的万千枯骨，正是他们的经历不断提醒着世人——毒品比战争更加恐怖。
