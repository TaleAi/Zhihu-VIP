**《皇后每天都想辞职》**


狗皇帝逼我进宫当皇后，帮他追贵妃。


我便与他约定，叫他每来一次我的未央宫，便要送我一套新衣裳抵作借宿费。


直到他追上贵妃，便同我和离，封我当宰相。


没多久，皇帝哀求我：


「不要贵妃行吗？以后宫里就我们俩，好不好？」
