## 简介
以科幻为结界，让脑洞飞一会儿。


「在这年头，当一个丧尸，比当一个人好多了。」


起初，他并不明白这话是什么意思。


直到那一天，他失去了自己的孩子，失去了自己的记忆。


在实验室里醒来。


——《我是丧尸》


王木投篮，百发百中。是真的百发百中。


他的教练为此发疯，有生之年一定要看到他投丢一个球。


但做不到。王木依然百发百中，所有投丢的球，教练一眼就能看出是他故意。这样的选手，让比赛变得没有悬念，进而，失去意义。


这样的天选之子，真是好事吗？


——《天选之子》
