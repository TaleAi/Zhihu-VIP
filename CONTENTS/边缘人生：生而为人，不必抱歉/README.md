## 简介
瓷娃娃、失独父母、欢场姑娘、跨性别者……请你摘下滤镜，看看他们眼中不一样的人世间。


在我们不熟悉的社会角落里，散落着一些为生活摸爬滚打的「边缘人」。


**有的人与主流价值观格格不入：**


在夜总会卖笑坐台的姑娘，有的不想成为流水线女工，有的想要圆播音梦，还有人只是想要找个结婚对象永居海外……


**有的人即使「活着」，就已经竭尽全力：**


身患罕见病的「瓷娃娃」男孩常年卧家，没有机会上学，不敢恋爱，只拥有头顶一方天空；


女孩为了搭上赴日本纺织厂打工的致富列车，父母倾家荡产，最终却沦为暗无天日的跑黑女工……


**有的人内心世界一片黑白，无法与现实生活和解：**


那个喜欢穿裙子、给自己起名叫「茉莉」的男孩，卖身供小妹读书，却遭到了父母唾弃，他不后悔，只是迷茫；


失独父母失去了热爱生活的能力，每一个睁眼的夜晚都在反复回想，一个无病无痛的儿子就这样平白没了……


当生活的美颜滤镜消失，社会就变得赤裸而真实，有些经历说来轻巧，却是一个人 100% 的苦难。


但是他们不屈不挠，用自己的方式，追逐着物质与精神的幸福。


来到这苦难的人世间，生而为人，不必抱歉。
