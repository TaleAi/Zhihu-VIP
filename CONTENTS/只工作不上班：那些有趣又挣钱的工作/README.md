## 简介
工作与兴趣真的不能两全吗？


随着互联网发展，带来的不仅是科技的日新月异，更是信息的高速传输。


我们发现身边出现了一个又一个新兴的行业；


这些行业也创造出了一个又一个崭新的职业。


或许对于年轻人来说，发展最大的意义，就是让人生的每个转折路口，都多了一些新的方向。


本专栏通过从业者的现身说法，向当下年轻人介绍这些新出现的，既能挣钱又非常有趣的小众职业：


宠物托管师：撸猫/狗赚钱两不误？


球鞋鉴定师：超爱球鞋，不但不让你花钱，还帮你赚钱？


游戏捏脸师：不爱游戏只爱捏脸，还能挣钱？


改娃师：娃圈最「神秘」的大神级存在？


……


从职业前景到入行方法的详细讲述，让年轻人实现「只工作不上班」的愿望。
