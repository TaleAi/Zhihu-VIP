## 12.【彩蛋】NFT 下半场：续写造富神话，还是跌落神坛？
**NFT 下半场：续写造富神话，还是跌落神坛**？


区块链一天，人间一年。去年的 NFT 掀起了各种造富神话，有些人提前入局成为项目方，有些人提前掏口袋成为早期投资者，有些人观望着研究着，有些人还在了解 NFT 到地是什么......不管你的节奏是什么样的，对不起，NFT 已经走完了一个波段周期。 


早在 2017 年的时候，便有了像 Crypto Kitties 这样的 NFT 项目，但 NFT 真正火爆起来是 2021 年三月推特联合创始人 Jack Dorsey 将其的第一条推文作为 NFT 售卖，并最终以二百九十多万美金成交。随之而来的是各类 NFT 名项目出圈。而 NFT 头部交易平台 OpenSea 的交易量也从 2020 年两千多万美金涨到了 2021 年全年交易量 140 多亿美金，整整翻了 646 倍！


![img](https://pic3.zhimg.com/v2-5b0ca648481b9dd923d0efbb53f913fe.webp)

不过时隔一年后，NFT 市场仿佛开始凉凉了。首先是那个著名的 Jack Dorsey 的首推 NFT——当初以二百九十多万美金买下它的男人叫做 Sina Estavi，数字货币交易所 Bridge Oracle 的创始人——在购买了它一年后，不知是觉得没用，还是想溢价转手，总之，在不久前，他将此 NFT 在 OpenSea 挂出售卖了。本来要价四千八百万美金（是的，近 20 倍溢价，你没有看错），然后截止到目前最高的报价是：6800 美金 （是的，你没看错，原价的近四百分之一）——这可能才是理想与现实差距的完美诠释。不得不说，该 NFT 一年前的成交价和现在二手市场的报价，仿佛有点标记一个周期的开始与结束的意思。当然这个 NFT 还没有成交，现在说它价格下滑还为时过早，而且也不能因为一个 NFT 的价格变化来判断整个 NFT 市场的走势。 


不过 OpenSea 的成交量近期也出现了显著的下滑，从进入 2022 年一月底最高日交易量两亿多美金，现在下降到日交易量七千多万美金，下滑了近 70%。即便如此，按照现在七千多万日交易量计算，如果保持此水平，今年 OpenSea 的年交易量仍然可以达到两百多亿美金，超过去年一百四十多亿美金。但从近期的走势来开，NFT 交易量仿佛和我大 A 股保持高度协同。


![img](https://pic2.zhimg.com/v2-498a19b4fc56542a9c5366dc1b9d99a4.webp)

对于成熟的交易市场，比如，A 股, 一段时间的下跌是周期性的；但是对于 NFT 这个新兴的市场来讲，期本身的价值内核体系还未完全形成，这样的下跌很有可能成为接下来的一种走势，有可能预示着 NFT 在经历了一段时间的投机热以后，开始逐步冷却下来，进入下半场。 


为什么 NFT 近期市场转冷？是否意味着 NFT 进入下半场？如果 NFT 进入下半场，它会何去何从，是会继续续写造富神话，甚至创造新的造富神话，抑或跌落神坛，抑或脱虚向实，回归价值？


**一、演化中的 NFT** 


为了拉平讨论的起点，还是需要浅补一下 NFT 是什么。相信很多人已经耳熟能详，NFT 是 Non Fungible Token 简称，意思是非同质化代币。NFT 的本质其实是基于公有区块链发行的一种智能合约，最常见的是基于以太坊的 ERC721 标准合约。这种合约规定了所发行出来的事物是非同质化的、不同于代币等是同质化的；NFT 是不可复制，独一无二的。因为这种合约的性质，使得发行出来的事物具有了资产属性，所以 NFT 也渐渐由一种合约发行物，变成了虚拟资产的代名词。


其实比起理解 NFT 的字面意思，或者进一步了解其发行机制，从分析 NFT 发展趋势来看，尤其是判断是否进入下半场这个层面来讲，更重要的是了解 NFT 过去、现在和未来发挥的作用和意义。


尽管现在 NFT 肩负很多「美誉」：规模万亿虚拟经济的基础，元宇宙火了之后被视为通往元宇宙的钥匙，等等；NFT 从发展初期到现在仍然没有逃出数字货币交易的生态圈，甚至有可能 NFT 是为币圈带来增量的工具。很显然挖矿受限、公链迟迟不能解决可扩展性等让币圈在比特币上一轮在大放水中疯狂上涨后，处于缺乏炒作热点的素材。而还未被充分炒作过的 NFT 可以填补缺乏热点的空白；同时发行 NFT 都离不开公链，无形中可以带动主流公链数字货币的需求和价格上涨，进一步盘活市场。也许在大部分「局外人」认真研究 NFT 的真正价值的时候，「局内人」最关心的其实是用户量和交易量。 


不过随着越来越多 NFT 名项目、元宇宙和区块链游戏中的 NFT 进入市场，NFT 逐渐显现出除了纯交易外，它所具有的虚拟资产本身的价值。事实上，人们可能真的会为好的虚拟 IP, 或有交互用途的虚拟资产买单。NFT 初步具有了带动虚拟经济发展，形成虚拟资产工具的作用。 


目前，大部分 NFT 虚拟资产看起来仍然是缺乏实际应用价值的虚拟作品，贴近它的中文翻译「数字藏品」；但是也有不少人预测，未来 NFT 会发展到「万物皆可 NFT」的阶段，随着元宇宙的发展，构建虚拟场景和交互的技术的发展，人们的虚拟活动将越来越多，届时可以被发掘的虚拟资产也有更多更丰富，NFT 作为被验证过的可以实现将虚拟世界元素资产化的工具，会最大化发挥它的作用，届时可能才会释放出真正的潜力。 


**二、NFT 前半场的造富神**话


受疫情影响，各行各业都出现不同程度的衰减的情况下，2020 年比特币的上涨和 2021 年层出不穷、动辄就是上百万美金成交的 NFT 项目，恐怕是近两年最撩人心弦的财富神话了。


从前文提到的推特联合创始人 Jack Dorsey 的百万首推 NFT 开始，NFT 的造富神话大概可以分为四种类型。 


第一：明星带货类。


不可否认早期的 NFT 项目会大火，和明星入局有着不可分割的关系。比如单个价值上百万的「无聊类人猿游艇俱乐部」NFT, 在篮球明星库里、美国晚节节目主持人 Jimmy Fallon 购买并将其作为头像后，大受关注。


![img](https://pic2.zhimg.com/v2-94f99ed39e737c68febaf4f5726b2559.webp)

![img](https://pic2.zhimg.com/v2-8f392e1cc3b7d3b90bfe135abfe163c0.webp)

除了购买、投资、做头像之外，很多明星也发布了他们的 NFT 项目，比如著名饶舌歌手发布了」a journey with the dogg」的 NFT 项目，将其早期的一些会议制作成为视频 NFT 进行发布。其中每一个 NFT 都为 Snoop Dogg 带来了超过十万美金的收入。


![img](https://pic3.zhimg.com/v2-5fed8c8ab870809024e3170c12abc88a.webp)

我国的明星也有很多参与或投资 NFT 的，如周杰伦与好友共同创办的潮牌 PHANTACi 发售 NFT 项目 Phanta Bear（幻影熊），单价 0.26ETH（以太坊），发售当天火爆非凡，1 万件幻影熊不到一小时售罄，总价达 6200 万元。余文乐、鹿晗等人都花钱购买了 Crypto Punks 等 NFT 作品，并作为自己海外社交媒体头像，等等。


这些明星带货的行为，让 NFT 受到了更多的关注，无疑对 NFT 的市场价格也产生了或多或少的影响。有外媒评论说道，购买有明星购买过的 NFT 项目，让普通的用户会产生一种和明星获得同等身份的感受。 


第二、符合社区审美的 PFP 项目


在诸多的 NFT 项目中，有一类较为常见又出现过较多成功案例的 NFT 项目叫做 PFP 项目，PFP 指的是「Profile Picture Projects」 或是「Picture for Proof」。指的就是那些看起来可以用作社媒头像的 NFT 项目。


其中最著名的有上文提到的 Crypto Punks, 无聊类人猿游艇俱乐部，还有以太坊曾经排名第一总价值达到九百万美金的 Hashmasks, 给一个 12 岁小男孩带来 40 万美金收入的 Weird Whales, 还有目前 OpenSea 上正在火热销售的 MoonBirds 等。

从审美或者艺术的角度上看，这些 NFT 可能并没有那么高深复杂，但也许是符合社区的口味，总之购买者愿意付费的 NFT 就是好的 NFT, 这些被称为 PFP 类的 NFT 项目在整个 NFT 市场中都有着不俗的表现。 


第三、艺术类


被翻译为「数字藏品」的 NFT 无疑是艺术家、设计师展露才华，并且获取财富的新途径。NFT 过去一年的造富神话中，也不乏艺术类项目，比如艺术家 Beeple 的 NFT 作品《每一天：第一个 5000 天》在佳士得拍卖行拍出天价 6000 万美元，约合人民币 4.5 亿元。Beeple 成为在世艺术家中第三高价的艺术家，这场拍卖也打开了虚拟世界和 NFT 艺术的新篇章。


![img](https://pic2.zhimg.com/v2-01a4631929386f8a6df7331720c3092c.webp)

第四、Gamefi 类


作为在游戏中提取出来的虚拟资产 NFT, Gamefi (去中心化可以边玩边赚的游戏）NFT 也是 NFT 中创收较高的一大类 NFT, 比如可以买卖的小怪兽、Axie Infinity 中的 Axie, 再比如虚拟房地产开发类游戏 Decentraland 中的地皮 NFT, 也在不久前掀起了一阵 NFT 炒房热。


**三、NFT 当下的隐**患


2021 年的 NFT 确实有很多高光时刻，输出了很多令人垂涎欲滴的财富故事。然而正如本文开头提到的，从 2022 年 3 月底开始，NFT 整体市场交易开始面临下滑。而 NFT 本身的发展也**逐渐显现出各种隐患：价值体系还未形成、同质化项目、欺诈和资产安全问题以及普通**人难获取收益等问题。 


NFT 自面世以来，其内核价值一直保守争议。虽然成功售卖的 NFT 名项目价格在不断刷新人们的认知，但是缺乏实际需求和使用价值的 NFT 多少有些像「皇帝的新衣」，其本身价值不明，但因为有人说它好，而且说它好的很多是牛人，大众便也只能随声附和，不便成为那个戳破真相的人。而在数字货币领域，这种靠「共识价值」而非「实际价值」，进行击鼓传花的游戏，获取财富的例子并不再少数。游戏的操控者们也深谙此道。然后没有实际价值支撑的事物终归是不具备可持续性的。Jack Dorsey 现在在市场上二次售卖报价惨遭滑铁卢，也许就在释放这样的信号：缺乏实际价值，价格虚高的 NFT 也许要面临泡沫破裂。 


NFT 作为一种虚拟资产上链的工具，其实并无好坏。可以区分好坏，影响项目价值的是 NFT 承载的内容。而目前看来，NFT 承载的内容开始出现同质化现象，比如大量的 PFP 项目。虽然 OpenSea 最新的 PFP 项目，Moonbirds 像素猫头鹰的总交易量已经突破三亿美金，但是这些购买者们，用户们，是否能够有足够的激情一直买卖着卡通式的头像类 NFT 作品，这一点是值得存疑的。在 NFT 具备实际应用价值的场景还不足的情况下，在用户逐渐对同质化的 NFT 项目开始出现审美疲劳的时候，NFT 项目和交易市场如何保持活力，目前是无解的。 


另一方面鉴于 NFT 进入门槛较低，项目方背景鱼龙混杂，不乏很多具有欺诈性质的项目。这些项目发行简单的 NFT 后，用左手倒右手的方式，抬高作品在二级市场的价格，利用价格的涨幅来吸引用户，寻找接盘侠。而最终的接盘侠可能不得不接受花大价钱买了个垃圾的事实......除了项目欺诈行为之外，NFT 作为链上资产也面临安全的技术问题。前不久周杰伦价值 50 万美金的 NFT 藏品被盗就是一个例子。


此外，如果缺乏对社区审美的把握，缺乏在社区营销的能力，普通人发行 NFT 售卖成功并赚取可观收益的概率其实并不高。不仅如此，普通用户还将面临着发行 NFT 价格不菲的上链手续费，以及进入 NFT 市场后成为韭菜的风险。对于整体 NFT 市场而言，韭菜割得太狠将逐渐导致用户的流失、交易量的下降，使得整体是市场进入衰退。 


**四、NFT 的下半**场


一艘全速前进的大船想要很快改变航向是很难的，一艘很大的船即使有船身破损开始进水，要沉下去也要很久很久。


目前年交易量超过百亿美金规模的 NFT 市场，即便有隐患，但在造富效应的光环下，在社区用户的支撑下，仍然在不断发展。虽然有一些隐患信号，但很难判断是否意味着 NFT 即将进入下半场。 


但是维持现状的 NFT 如果不解决其价值体系的建立、项目多样性、加强对欺诈性项目的监管，解决资产安全等问题，恐怕现有交易市场将逐步迎来发展瓶颈甚至市场萎缩。那么除去在交易市场创造财富效应的作用外，NFT 怎么可以获得更加具有可持续性的发展呢？NFT 的下半场何去何从？


其实作为虚拟世界资产上链的重要工具，或者目前来看是唯一的工具，NFT 的真正价值和其可持续发展的潜力是和未来虚拟经济，或者说是 web3.0 以及元宇宙的发展紧密相关的。


NFT 是元宇宙中元素资产化、元宇宙实现商业化的重要途径，是以「拥有权」为基础的 Web3.0 重要的确权工具。 


随着元宇宙场景和应用的发展，以及 web3.0 价值互联网技术的成熟，NFT 将在真正有价值的场景中发挥其重要作用。目前来看，NFT 的下半场将逐渐回归其工具的作用，并在以下四类场景中发挥重要用途。 


第一、Gamefi、元宇宙交互中的重要资产交换工具


从 Roblox 到 Axie Infinity, 受制于实体经济增量放缓，年轻人缺乏工作机会，可以边赚边玩、发挥去中心化创作能力的 Gamefi、元宇宙游戏等迎来了发展机遇。而在这些项目中，主要的价值交换需要依靠 NFT 工具来实现资产上链和交易。相比于传统的静态 NFT 项目，这样具有交互性的、可以激发用户创作潜力的、具有经济激励模型项目中的 NFT 在内容和用户使用方面更加具有实际价值，会具备更长期的发展潜力。 


第二、真正的数字艺术品


相比于现在以 PFP 为主的 NFT 项目，未来将会有更多艺术家、设计师、甚至是人工智能创作力量进入虚拟创作空间，涉及出真正具备美学价值、收藏价值的艺术品，继而借助 NFT 工具实现价值的交换。 


第三、数字孪生体 NFT


数字孪生是指利用物理和科技等继承手段，通过对现实事物数据的采集、完成在数字世界的映射，形成和现实世界一摸一样的孪生体。数字孪生体对与模拟现实生产、数据测试、设计优化、决策优化等具有重要意义，在工业产业有巨大的价值空间。未来从业人员可以凭借自己的行业经验和设计能力构建数字孪生体，并以 NFT 的形式进行售卖，将数字孪生形成产业链的一部分，并带动相关行业的发展。 


第四、数据 NFT


我们已经置身于数字时代，随着虚拟经济的发展、数据的价值将被不断挖掘，未来我们将进入更告诉发展的数据经济时代。区块链以及 NFT 的存在，可以实现数据的确权和资产化，用户本身可以将自己的数据变为资产，通过发行和售卖数据 NFT 进行数据变现。这在生物医疗、零售等行业都会发挥作用。 


总体来看，和大部分新生事物的发展一样，在发展初期都会存在一定的泡沫；处于发展初期的 NFT, 现阶段也存在很多投机行为。但是随着市场自身对建议的追求和对供需关系的纠正，随着监管的强化，比如我国已经开始率先政治 NFT 投机行为，前不久中国互联网金融协会、中国银行协会、中国证券业协会三大协会联合呼吁会员单位共同遏制 NFT 金融化证券化倾向，NFT 很大概率会逐渐退去泡沫，进入能够发挥真正价值的工具时代，为下一代互联网、甚至元宇宙的发展提供基石。
