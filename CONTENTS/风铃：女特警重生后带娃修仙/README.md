## 简介
看「带球跑」妖女，如何降妖除魔，收获真爱。


风铃本是末世对抗丧尸的一名女特警，死后重生，竟是穿越到了修仙世界。


穿越的对象，不是自带光环的女主，而是人人唾弃的合欢宗妖女。


她一觉醒来发现自己睡了正派天才少年，本想拍拍屁股走人，过上自由洒脱的生活，谁知几个月后发现自己怀孕了……


这个娃不仅性格随了爹，古板较真，还跟他天才爹一样，也是个天赋异禀的修仙奇才。


为了不耽误孩子的大好前程，风铃决定带娃修仙。


阴差阳错，再次遇到男主时，她避之不及，却发现男主早已将她遗忘。


而她穿越到这个修仙世界也是另有玄机。


等到她揭开层层迷雾，才发现，她一直所渴望得到的，原来就在身边。
