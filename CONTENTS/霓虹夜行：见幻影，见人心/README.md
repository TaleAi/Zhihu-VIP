# 霓虹夜行：见幻影，见人心

### 状态：已完结，共 30 节

## 简介
这是一个脑洞惊悚故事集。一步一鬼，见幻影才能见人心……


欢迎来到「霓虹都市」。


有人说，比黑夜更可怕的，是在黑夜中闭眼夜行。


人和鬼一样看不见前路，鬼似人一般看不见归处……


  



**— 还魂外卖 —**


给我送外卖的骑手，是个白瘦幼的美女。


她连着给我送了两天外卖，里面都有令人恶心的玩偶，是巧合？是恶作剧？


公司里似乎也有女生暗恋我，每天早上给我送早餐，但就是不肯出现见我一面。


这突如其来的桃花运是天降良缘，还是宿命般的诅咒？


  



**— 孤楼怪夜 —**


我的同事离职后，我接手了他的烂摊子，每天吃睡在办公楼里，成了公司的「隐形人」。


两个月后，我一直暗恋的女同事突然说要跟我一起加班通宵，


而那个晚上我亲眼看见——她敷着面膜失去了呼吸。


可第二天，她却「死而复生」，同时所有的死亡证明都消失不见！


这一切的谜底，都藏在这栋办公楼里！


……


**百鬼夜行，生人避让**


**霓虹闪烁下的，是幻，是影，**


**还是人心中的恶欲？**
