## 简介
你了解唐朝吗？你了解唐诗吗？陈子昂、李白、杜甫、白居易、李商隐……唐朝，中国古代最强盛的王朝之一，开放包容。唐诗，则是唐朝的文化地标，是中国文学史发展中的一座高峰。本专栏精选 50 首唐诗佳作，逐句赏析，品味语句背后的情丝与哲理。


「床前明月光，疑是地上霜」，唐诗赋予孩童关于世界最初的想象；


「近乡情更怯，不敢问来人」，唐诗让种种人生际遇有了恰当的表达；


「绿蚁新焙酒，红泥小火炉」，唐诗让平淡无奇的生活有了诗意的点缀；
