## 简介
真爱，会在黑暗中打下一束光。


即使生活在阴沟里，也有人仰望星空。


**《隐秘的痂》| 萧笑**


她面对继父的威慑与觊觎，在隐秘煎熬中长大，但她遇到了相似境遇的他。终有一天，她鼓足勇气决定战胜黑暗。


**《破晓》| 周嘉言**


她发现了福利院的肮脏交易，但求救无门。如何逃出生天，拯救自己与同伴？危急之际，她找到了靠谱的盟友。


……


这个专栏，我们希望通过她们的经历，告诉每一个人：


懵懂无助的女孩，总有一日会变成坚强的大人，回来摧毁那个禁锢她的世界。 


也希望女孩们知道，就算面对那么多的苦难与困境，我们也永远可以有冲破黑暗的勇气和机会。


我们还是可以站起来，笑得漂亮。
