## 简介
近几年兴起国学热，可究竟什么是「国学」？天文、服饰、交通、地理、饮食……均可算作国学的一部分。就让我们跟随窦可阳老师一起，带着现代的眼光，从日常生活的「日用之学」出发，上至天文历法、下至衣食住行，绘制一幅古代的社会文化地图，真正了解古人的生活观念。


古人是如何「仰观天文，俯察地理」的？


服饰、发饰为何在古人看来「兹事体大」？


……


近几年，复兴起一股国学热潮，然而自五四运动「打倒孔家店」以来，我们和传统文化断裂已久，这也让我们对传统充斥着模糊不清的刻板印象。


究竟什么是「国学」呢？国学包罗万象，是中国古代数千年知识和文化的总结。天文、服饰、交通、地理、饮食……均可算作国学的一部分。不仅治国平天下的大格局属于国学，衣食住行的日常琐碎，也是国学。


就让我们跟随吉林大学文学院副教授窦可阳老师一起，带着现代的眼光，从日常生活的「日用之学」出发，上至天文历法、下至衣食住行，绘制一幅古代的社会文化地图，真正了解古人的生活观念。
