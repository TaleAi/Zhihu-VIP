## 简介
本专栏由数十起耸人听闻的案件组成，在侦破案件的过程中，拨开迷雾，展示人性的复杂。


腐烂女尸穿着崭新的红色连衣裙深夜点外卖，隔壁单元的下水道里发现了人体组织和毛发，迷雾背后究竟隐藏着怎样令人毛骨悚然的真相？


村里的人几次想移走一棵百年槐树，均发生了一系列流血事件，一时间，村里人心惶惶，议论纷纷。几天后，种植槐树的高龄老人被发现吊死在槐树上，是灵异事件还是故意杀人？


海上飘来一具无名男尸，妙龄少妇前来寻夫认尸。半夜捞尸的过程中，众人却在海面上听到了已经死去的男人声音，女子的丈夫并没有死，死去的男人又是谁？


无名小岛上的一间农舍里，摆放着七口大缸，六口大缸中均装满红色的动物血液，经过一系列推理，第七口缸中即将盛满人血，凶手为何故布疑云？
