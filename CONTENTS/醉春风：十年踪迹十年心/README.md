## 简介
十年前的那颗心，今日依然在胸膛跳动。


梦里，我们还是相亲相爱的。


但一睁眼，倏忽已过十年。


**《更三愿》**


我和爸爸一起穿越，他成了皇帝，我成了公主。


按说生活应该很是惬意。


但我们所思所想，只希望找到我早逝的妈妈。


一次宴会，我们惊讶地发现，妈妈成了宰相的夫人。


兜兜转转，我也跟宰相的儿子牵扯到了一起。


所有人都被卷入了帝位之争。


已经分开太久的一家人，是否还有团聚的一天？
