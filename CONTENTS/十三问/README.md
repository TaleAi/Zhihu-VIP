「你有 13 次机会，解开谎言的线头」
刑侦支队长被迫陷入一场破案游戏。
每一局只有 13 次提问机会。【一人剧本杀】请你入局！


这是个有点疯狂的游戏。

【逃出粉红色房间的姐妹】


十八岁的高三少女醉酒溺水身亡。


对于她，三位相关人给出了三段故事：


姐姐口中，


她是和自己相依为命的妹妹，历经常人不敢想象的苦难，终于等到救赎。


班主任口中，


她是仗着美貌不努力、自甘堕落的坏学生。


校外好友却一口咬定：


她绝不可能是意外身亡！


  



你必须从这三段故事里，辨别谎言，寻找真相关键词，在十三次提问机会以内，还原整个事实。


游戏开始了。
