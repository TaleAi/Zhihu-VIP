## 简介
人性七宗罪，贪婪位居其二。想更有钱，想更漂亮，想有更漂亮的另一半，想更年轻……小心别作恶，毕竟，人在做，天在看。



我与老公非常恩爱，但新婚不久后，我的生活受到了严重的干扰——我怀疑我们的床正对着的墙里，藏着一个人。随后我得知，丈夫所购买的这套婚房是凶宅，当年就是这堵墙里藏着受害者的尸体。我惊恐之余做出了一个决定：我要砸开这堵墙一探究竟。


——《墙里有人》


嫁给丈夫的前夜，我睡眼朦胧中看到一个奇怪的影子出现在床头，好像是穿着婚纱的小梦。可小梦已经失踪几年了。她是爹从山沟里捡到的弃婴，得有软骨病，非常漂亮，好像芭比娃娃。爹曾经带着她去做些肮脏的勾当，用赚来的钱供我上学，走出大山。


——《玩偶》


生完孩子，我凑巧跟婚前的白月光陆南联系上了，他提出了见面，但严重走样的身材和脸让我自卑不已。下班回家后，老公说他给我买了一副面膜。郁闷之中，我用上了面膜，发现自己变得很美，在和陆南的见面中擦出了火花，然而我的身体开始出现不适。


——《人皮面膜》
