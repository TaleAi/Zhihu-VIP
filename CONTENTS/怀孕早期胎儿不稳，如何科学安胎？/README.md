## 简介
本课程由上海市第一妇婴保健院妇产科医生董一诺主讲，帮你避开安胎雷区，科学健康安胎。


相信刚怀孕的你，有很多疑惑和担心：


宫外孕、孕酮总是低、见红、肚子痛、查不到胎心？


怀孕以后，孕妈担心最多、顾虑最多的就要属「流产和保胎了」。


但是「保胎」也是门技术活，盲目保胎反而会伤害宝宝。


本课程特邀上海市第一妇婴保健院董一诺医生教你科学的安胎方法。


孕产妇营养、孕产妇运动、孕期常见风险避雷、孕产期常见问题答疑。将专业的知识深入浅出，通俗易懂地讲给你听。
