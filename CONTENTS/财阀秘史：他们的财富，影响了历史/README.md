## 简介
看不见的地方，总是容易藏污纳垢的


百年家族，豪门财阀，盛衰兴废，权力更迭……


他们有的靠着战争发家，有的靠着卖国兴盛，也有的从监狱萌芽。


他们带着时代的印记，乘着发展的巨轮，从遥远的过去驶向现在、未来。


他们有人在艰苦的年代创业，有人在和平的年代夺产，也有人在财富中沉溺走向犯罪，更有人因为金钱而丧命。


除了堆砌的金玉，那些金玉之下的隐秘，金玉背后的故事，每一个都带着传奇的色彩……


* 从「红顶商人」到「地下皇帝」，揭秘三星「血汗工厂」背后，为了权力的手足厮杀；
* 从「地下浪人」到「海上霸主」，靠侵华发家的日本三菱集团，如何成为日本政府手中最锋利的妖刀？

这个专栏，我们将透过一个人，或者几个人，为你还原那些巨无霸家族背后的故事，以及那故事背后的一段段历史，真实展现各个世家豪门的兴衰隐秘。
