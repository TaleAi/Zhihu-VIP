## 简介
教你简单实用的人际交往策略，帮助你打破社交壁垒，成为高情商社交达人。


对于社交，你是否也存在以下这些困扰：


害怕与人沟通，从不主动去社交，不懂拒绝，疲于应对无效社交？


社交场合也能聊一聊，却没办法更近一步？


微信好友不少，但关键时刻找不到人？


……


美国人际关系大师卡耐基曾说：「一个人的成功，15% 取决于他的专业能力，85% 取决于他的人际关系。」


人际关系质量决定了你的生活质量，而你的社交能力，则决定了你的人际关系。


人际关系处理不好的人，容易自我否定，人缘差、没朋友，事业也受阻碍；而人际关系好，则能够自信从容，人缘好、人脉广，事业进步快。


别担心，我们教给你一套简单易学的人际交往策略，帮助你突破社交障碍，学会与陌生人建立关系、与熟人深入关系，从而建立起优质的人脉圈。
