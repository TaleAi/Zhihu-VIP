## 简介
如果你是要去和亲的公主，你会怎么做？一个爹妈全没、又需要母债女偿的和亲公主，皇帝们忙着争夺地盘，太后们忙着公报私仇，一向佛系的她决定换个梦想：既然不能安心养老，那不如把这些人都一锅端了！


我，和亲公主，欠钱、远嫁、头很秃。原本想着醉生梦死才比较符合公主的身份，却不想宫廷难混，人人都披着笑面，背地里藏着阴谋。都说和亲公主，难逃一死，要么殉国，要么殉夫。我去问天下第一的说书先生，和亲公主怎样才能活下去？他说：公主就要对自己狠一点，你得干事业。


万幸我自小学的是兵法权谋，裙子一掀，袖子一撸，不就是干事业吗？九千岁招摇而来，谆谆教导：古往今来，成功的公主背后，一定有个狂拽炫酷的大帅比。那就扑他个满怀，我要这江山如画，开万世太平。
