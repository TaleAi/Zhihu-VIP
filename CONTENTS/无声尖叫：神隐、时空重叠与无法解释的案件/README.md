## 简介
真相让人浑身发凉。


监控的出现，并没有消灭神秘。


直到今天，世界各处依然发生着难以解释的案件。


那些隐秘的线索以一种现代的方式被记录下来，被反复观看，被反复推演。


因为那些追求真相的人，依然相信着：


「除去所有不可能的，剩下的即使再不可思议，那也都是真相。」


本专栏，将带你一起，在资料和图像的海洋中，一步步去还原那些无法被解释却又真实发生的神秘案件。
