## 简介
本专栏里，悬疑作家无颜为我们塑造了一个逗趣、热心肠，还有点儿侦探头脑的牌商陈海，他专做买卖泰国佛牌的生意，懂点儿佛教文化，对鬼神之事也颇有研究，从业这些年，亲历过大大小小无数件「灵异事件」。这些荒诞离奇的诡异事件，究竟是真有鬼怪作祟，还是有人精心策划的阴谋诡计？陈海将带你一点点找到真相。除了猎奇爽感，这些故事希望带给我们的，更多的是对复杂人性的思考，如同陈海说的：「有时候，比鬼怪更可怕的，是活人的心。」


我是个牌商，专做买卖泰国佛牌的生意，职业原因，懂点儿鬼神之事，偶尔也帮人解决些棘手的「灵异」事儿。干这行十来年，稀奇古怪的事情我遇见不少：


**来买佛牌的女客户说，老公去世后让她怀上了「鬼胎」**


睡在老公尸体隔壁的第二个月，黄三怀孕了。婆婆坚称自己儿子还活着，固执地将尸体藏在家中，此时她又凭空怀上孩子，再加上睡梦中反复出现的恐怖场景······一连串诡异事件令黄三怀疑，究竟是她真被死去老公阴灵缠身、身怀鬼胎，还是有人在捣鬼，编了个恐怖陷阱等她跳？


**出租房里藏着个「贞子」，还抢走了租客的孩子**


搬进新租房子不久的某个深夜，陈玲亲眼见到房间里莫名多了个女人。这女人状若怪物，匍匐前进，抢走她的孩子后迅速消失在房间尽头。她该怎么办？这神秘女人究竟是人是鬼，藏身何处？而这间租金便宜到令人咋舌的房子里，究竟藏着什么秘密？  


······


这一桩桩看似古怪离奇，似有鬼怪在祸害人的离奇事件，真相究竟如何？
