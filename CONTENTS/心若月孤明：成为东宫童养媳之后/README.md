# 心若月孤明：成为东宫童养媳之后

## 简介
太子未定，先定了个太子妃。
可惜，我这个太子妃，貌似有点克夫。


作为长公主和执掌大权的泠水侯的女儿，我从出生开始，就赢在了起点。


五岁时，皇帝舅舅大手一挥，我成了史上年龄最小的太子妃。


但离谱的是，我成为太子妃之后，适龄的皇子们，一个接一个地死了……死了……


十八岁那年，我和八岁的最新任太子终于成婚。我把他当崽带大，照顾他起居，哄着他入睡，保他全须全尾地长到成年。


原以为太子成年后，我这个太子妃就可以享清福了。谁知道，风波骤起：


太子的生母成事不足败事有余，卷入诸多皇子的被害案，太子也因此被废；


年轻的皇后想维护自己和嫡幼子的地位，把太子当成眼中钉；


我的妹妹不想嫁入皇家，设计陷害我……


谜案之中，太子的态度也让我琢磨不透。他极度害怕我的离去，却又一直向我隐瞒真相。


父母、手足、朋友，在这重重迷雾之中，我能看清他们的真心吗？


而被我呵护长大的这个少年，又会和我走到什么样的境地？
