还有些老物件，因为长年累月地沾染煞气，能别碰就别碰，譬如刽子手的鬼头刀，不仅刀柄刀刃有讲究，就连不用时盖着的那块红布，也暗藏玄机。


……


这个专栏将讲述 8 个让你心惊胆战的民间异闻故事，尤其适合午夜观看。


精彩的故事会让人不断回味，而隐含其中的真善美丑更是让人唏嘘。
