## 简介
比起「阿尔茨海默病」这个名字，大家也许对「老年痴呆」这个称呼更加熟知，然而却不知道因为这个贬义的名称，延误了多少患者的治疗。或出于不重视，或羞于承认，这个老年人的高发病症就诊率竟只有三成。究竟什么是阿尔茨海默病，预警症状有哪些，当出现征兆又该如何就诊，制订治疗方案，以及在后续照顾患者的过程中将如何处理。这个专栏将给你一份科学的解答。


对于阿尔茨海默病患者来说，生活不应只有遗忘和失去。就像综艺节目《忘不了餐厅》里可爱的老人们一样，他们可以服务客人，带给大家欢声笑语。


而我们需要做的除了给他们多一点体谅，多一点温柔，还需要科学的照顾方式。


作者 TianyinL 目前在香港大学担任社会工作与社会行政学系研究助理教授，对阿尔茨海默病有着深入的研究，她将在工作中接触到的诸多案例写进了专栏中：


当记忆变差，丢三落四，可不一定就是阿尔茨海默病，那它的征兆到底有哪些；


不肯就医，你该怎么劝服，又该如何简单测试一下家人是否真的患病了；


多疑、吵闹、囤积垃圾，当患者出现奇怪的行为举止，该如何应对；


如何打造一个适合阿尔茨海默病患者生活的居住环境；


……


除此之外，她还介绍了阿尔茨海默病目前在全世界范围内的社区互助、新药研究进展。


希望这份写给阿尔茨海默病照顾者的实用手册，能真正带给大家一些帮助。
