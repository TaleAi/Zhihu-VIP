## 简介
都市婚姻情感博弈故事集。


爱情快餐的大环境，「一生一世一双人」，大概已经成为遥远的爱情神话。


古语云，「夫妻本是同林鸟，大难临头各自飞」。


现在，不用等到大难临头，男女就已想着各自飞了。我们曾经，自豪地宣称嫁给爱情，忠贞不渝几近信仰；而现在，手里不拿把算盘，都不好意思踏入婚姻。


现代人的婚姻如此脆弱。所有夫妻关系，如果深究，都是黑洞，都有外人不能理解的部分。


已婚夫妇们，当无人旁观时，他们究竟是谁？


《最终的胜利者》


长相和出身都极为普通的我，大一军训时对同班花花公子般帅气多金男同学一见钟情，颇有心机地忍住没有表白，渐渐与其成为好朋友，冷眼看他在情海爱欲里沉浮多年。他是被母爱控制的妈宝男，多段恋情都没有得到母亲的承认。直到在他感情最为脆弱之际，我趁虚而入……迅速恋爱、结婚，完成自己的华丽转身。而他的母亲去世后，我所有的设计经营，更另他一辈子都无法离开我们这个婚姻。


 《完美而脆弱的中年夫妇》


一对夫妇过着波澜不惊的生活，即将迎来十周年结婚纪念日。直到一天深夜，一条突如其来的信息打乱了这个家庭的平衡……这对夫妇表面上光鲜亮丽，而实则千疮百孔的婚姻还能继续维系吗？


《嫁给富一代之后》


自小丧父的我，机缘巧合认识了大她十来岁的多金男性，对我极为宠爱。我难以自已，在爱情与金钱的双重影响下迈入了婚姻，却在婚后慢慢发现，与有钱老男人在一起的婚姻生活，有着难以言说的痛苦之处。
