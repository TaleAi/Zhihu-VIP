## 简介
这是那个叫苏景宁的女孩的一生。


斜拔玉钗灯影畔，剔开红焰救飞蛾。


苏景宁说，如果她可以回到十五岁那一年，她一定要告诉自己，不要走上那条漫天大雪的街，不要遇见那个红衣铠甲的将军。


如果你问她这一生过得好不好，她会告诉你，开始是好的，只是她没有福气，好不到最后。
