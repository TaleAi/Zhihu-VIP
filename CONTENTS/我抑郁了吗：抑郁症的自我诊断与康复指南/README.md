## 简介
本课程专为身边的人得了抑郁症，不知该如何帮助TA的人、想要全面了解抑郁症的人、自己得了抑郁症，想获得亲友帮助的人、抑郁症相关工作者等提供专业抑郁科普。



每天情绪低落很难受，不知道自己出了什么问题？对网上鱼龙混杂的量表无从下手？决定去看医生或心理咨询师？但不知道就诊流程，担心抗抑郁药物有副作用？


本课程专为抑郁症患者提供抑郁科普，结合上千名患者案例、社群朋辈咨询，内容全面覆盖抑郁症相关内容。专业医疗、心理团队数轮精心打磨，大范围调研+社群答疑采集病友最想了解的问题。


致力于让患者和家属了解抑郁症所有相关知识、详细的就医治疗指南、日常自我调节与保养方法和对抗抑郁症的应对策略。
