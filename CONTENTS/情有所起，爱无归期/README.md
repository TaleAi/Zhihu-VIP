## 简介
关于『爱情』，我们一直诉说不尽。


少年的爱，青涩懵懂，心动时，如微风拂过夏夜的原野，芳草碧连天。


青年的爱，热情奔放，情浓时，便如飞蛾扑火，奋不顾身。


中年的爱，沉重如山，有七年之痒的坎坷，亦有柴米油盐的桎梏。


老年的爱，浪漫优雅，轰轰烈烈前半生，相濡以沫这一世。


本专栏将收录不同年龄、不同身份的恋人，从暗恋到初恋、从婚前到婚后的每一个故事。
