## 简介
本课程将给你：科学靠谱的性教育知识、行之有效的性教导方法、破除性的教育误区，和孩子沟通无碍、增强自信，让你从容谈性。


作为成人，你可能在困惑：


「别人触碰我身体，我为什么很抗拒？」「经常自 wei，所以很愧疚怎么办？」「被人说女汉子/娘娘腔,心里不舒服。」


作为家长，你可能有这样的难题：


「孩子总问我：妈妈我从哪里来？」「社会负面新闻多，担心孩子被坏人伤害。」「有一天发现，孩子在偷偷看色情信息。」


你不知道的是，这些都是因为你「缺乏性教育知识」。不管是了解自己，还是教育孩子，不管是青春期，还是成年后，如何开口谈性成了一个难以启齿的问题。性心理的发展，性生理的成长，性价值观的形成，这些都是成长中必须经历的课题。
