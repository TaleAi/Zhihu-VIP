江湖传言：剑术的最高境界，乃是一步一杀。


费劲下山前，师父将他叫到座前，语重心长地教育他：「小费啊，你知道要练成一步一杀，最难的地方在哪里吗？」


「宝剑？秘籍？」


「不对，是要保证你每步之内都有敌手。」


「那可得当武林公敌了。」
