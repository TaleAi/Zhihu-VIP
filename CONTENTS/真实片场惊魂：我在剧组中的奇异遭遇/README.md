## 简介
镜头所记录下的恐怖，只是真实现场的十万分之一。
我叫乔飞，是个不得志的小导演。我曾游历过数个国家、服务过数十个剧组，至今能让我记忆犹新的却不是镜头前的故事和明星，而是那些真实发生在拍摄现场的命悬一线的瞬间。


战争片：在拍摄一场大夜戏的战争场面时，剧组的人听到了真枪发射子弹的声音。很快，大家就意识到那是流窜于边境线上的毒贩团伙，而且正在飞速接近我们……


纪录片：我和同学去古巴偷拍当地的巫毒教女巫，等到回来的时候，却总是感觉女巫的诅咒一直在跟着我们……


恐怖片：在泰国拍完了一部鬼片，杀青那天晚上，组里的剧照师在相机中发现了「多余」的演员……


犯罪片：在我们拍摄以「人祸」为主题的悬疑短片时，所有人都没有注意到，一场「天灾」正降临在整个剧组之上……


我叫乔飞，干过制片、做过编剧、兼过摄影、当过导演。


毕业后的几年间，我的足迹涵盖越南、古巴、美国、日本、泰国等地，参与过战争、历史、犯罪、记录等各个类型的剧组，经历过无数次生死奔逃，也感受过后怕到脊背发凉的体验。


这些经历中有的是天灾人祸，也有的是灵异传说，同样还少不了娱乐圈中的绯闻轶事。


待我把这些真实经历一一道来，你听了可别跑……
