## 简介
当生存变得困难，边缘人物，是如何走向「犯罪」之路的？


吃「死人饭」、贩毒、帮派、捡尸、杀猪盘、命案……这是一个关于社会边缘人物的犯罪手记的专栏。


恶寒深水里，有人捞钻头、捞尸，这个「水鬼」如何通过吃「死人饭」，一次性捞出百万资产？


戒网瘾学校中，男孩子的帮派、命案，轮番上演，有人挺住了，有人以自杀出逃；


被困东南亚，穷小子网上「养猪」，搭上了致富快车，摧毁了一个个家庭后，警察却找上门来；


偏远山村，被拐卖来的女人生下的男孩，试图改变命运，却误入了人贩子的世界，他的下一代还会重复这样的悲哀吗？


这些取材自真实事件的故事，也许可以带你看到不一样的世界和人生。
