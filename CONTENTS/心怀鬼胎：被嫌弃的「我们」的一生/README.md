## 简介
世间之罪奇多，唯爱最痛。
智者不入爱河，愚者溺水而亡。


家具设计师以人为椅，在恐怖的真人扶手中，寻求畸形的童年温暖；


死期将至的植物人，神游疗养院，却发现骇人真相；


满嘴跑火车的神经病，无意中发现了一个连环杀人犯，她家的房子不仅住着阳间人，还住着阴间魂。


……


依旧是无数个「我们」为爱犯罪的故事，怪奇艳情疯癫。


「我们」无处不在。困兽无处不在。
