## 简介
这是一个或唯美浪漫、或扣人心弦，有虐有甜的古言专栏。


本专栏收录了多个古都爱情故事，既有美丽风流的佳人才子，也有倾国倾城的祸国美人，还有快意江湖和诡谲朝堂，为你创造一个又一个充满「中国美」的古言世界。


落花时节，古城之中，与君相逢。






《求药》


姑苏首富沈家七娘沈知意，聪慧秀美，绣工独绝，可惜才高惹人妒，沈知意十五岁那年，被庶出的姐姐毒瞎了一双眼睛。人人都避着沈知意，唯独其教书先生贺行云待她真心，不仅真的教她诗书，还带她走过姑苏城的沿河街巷，成为沈知意漆黑世界里唯一的光亮。可后来才发现，贺行云其实是沈老爷诓来的药人，用他的一条命能治好沈知意的眼睛；他也不是什么教书先生，而是当朝太子殿下的庶长子，沈家不过是权贵手里的一枚棋子。到底谁是真心？谁是假意？
