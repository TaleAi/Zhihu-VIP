## 简介
人生如戏全凭演技，可怕的是你拿错了脚本。


神秘人发过来的妻子艳照，彻底毁了我的生活。


当我抽丝剥茧般寻找妻子出轨的证据，做出一系列两败俱伤的复仇计划。


愈发冲动的行为背后，是我对妻子背叛的痛苦，还是人性本来就多疑？


——《神秘来电》


  



离奇的凶杀案，所有的线索都充满疑点。


死者的邻居，死者的哥哥，死者的儿子，仿佛都在期待她快点死去，仿佛她的存在就是一种罪过。


胸口前的伤，不是致命伤，但她还是死了，为什么？


——《凶手不止一个》
