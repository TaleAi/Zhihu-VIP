## 简介
一起算算，上帝掷出的骰子是几个点？


现代科学起源于 16 和 17 世纪，从哥白尼的日心说、开普勒的行星三定律、到伽利略和牛顿的物理学，一切都和数学物理息息相关。


那时的科学家们努力探求那些我们身边可观测的世界，研究太阳是否真的由阿波罗驾车升起，1+1 为什么等于 2，地球是方还是圆，太阳光究竟有几种颜色。


随着时代的进步，他们开始越来越多地研究不可观测，难以描述的现象；


他们甚至大胆地对地球以外的世界进行预测：宇宙到底有几个维度？黑洞是否真的存在？人们能进行时间旅行吗？


在纷繁复杂的世界中，唯有数学和物理，构成了科学世界的基本规则。


通过对数学和物理的研究，人类越发接近「上帝」：


* 身处三维空间的我们，如何理解四维？
* 宇宙中没有氧气，太阳是如何「燃烧」的？
* 双缝干涉实验恐怖吗？恐怖在哪？
* 我们所处的银河系正在进入银心的黑洞吗？

本专栏邀请众多物理、数学、天文学等领域的优秀答主，与你一起追寻世界至繁背后的至简。
