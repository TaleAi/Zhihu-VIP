## 简介
8 个情节紧凑、扣人心弦的短篇故事，忽而俏皮，忽而细思恐极。擅长情感悬疑故事的尸姐作品，带你搭乘精彩故事的过山车。


我是个普通得不能再普通的宅男，邋遢、懒散，不善交际。但在众人眼中，我却浑身优点。


因为我有个秘密，谁都不知道的秘密：我有个替身……


——《替身》


毫无疑问，我已经死了。 


是被一个抢银行的劫匪头头一枪打死的。 


 那天我只是出门买个烤红薯而已，红薯拿在手里，还没来得及咬上一口。 


没想到死了之后，还有糟心事：我无法离开，只能跟劫匪共处一室……


——《这原本应该是一个悲伤的故事》


我死了。等等，我死了？  

我根本不知道自己为什么要从那么高的楼上跳下来。  

我要寻找自己跳楼的理由。


——《我忘了自己自杀的理由》
