## 简介
自由学者姚尧，全面注释评论《资治通鉴》，以更高视角，立体多维、前后延展地理解战国变法，重构史观，辩证解读史书误区，助你更清晰地认知现实，提升自我。


都说历史是一个任人打扮的小姑娘，有的人读史，只能看到打扮好的小姑娘，有的人，更希望看到素颜小姑娘的真面目。


事实上，历史真相是无法完全欢迎的，姚尧也不能。但姚尧能给你一个更高的视角，让你立体多维、前后延展的读历史。这就是我们所说的「姚尧精读」。


作为「帝王史书」的《资治通鉴》，以其详实的内容和超长的时间跨度历来为众多历史爱好者青睐，但如果仅仅是只读该书，很容易被司马光带入「先有论点，再有论据」的史观中去。而姚尧的精读，则把更多互相印证的史料和透过现象看到的本质呈现在我们面前。


《通鉴》开篇的“三家分晋”中，关键事件「三家灭智」，很多人会归咎于智宣子选拔继承者不力，但姚尧却用 1972 年出土的《吴问》告诉我们：两千多年前、三家分晋前 50 年的孙武早就预言，生产关系的先进性决定了生产力水平高低，也决定了谁的国力更强盛。即便是应用到今天的中美竞合，也同样适用。这就是《姚尧精读资治通鉴》带给我们的更高视角。


作为现代首部详尽注释、评论《资治通鉴》的巨著，姚尧将倾注 30 年精力来完成它。这也是为什么我们说：


读史必读《资治通鉴》，读《资治通鉴》必读《姚尧精读资治通鉴》。
