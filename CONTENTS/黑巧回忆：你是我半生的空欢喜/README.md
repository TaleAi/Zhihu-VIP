《记昨日书》


她是狠戾重情的资本大佬——沈泽捧在心尖，视作生命的人。


他们年少就相依为命，沈泽曾承诺等他有钱了要将世界上最好的一切全捧在宋蕊面前。宋蕊很好哄就信了，她等啊等，等了十几年，等到三年前意外怀孕，沈泽没求婚，孩子生下来还没求婚，后来孩子没了，她才知道，沈泽正在和一位高官的孙女谈恋爱。


「等我有钱了，我带你把你最喜欢的衣服鞋子都买下来……」


「等我有钱了，我带你吃最好吃的一切东西……」


「等我有钱了，我们就结婚好不好？」


后来沈泽有钱了，可他承诺给宋蕊的一切都给了另一个姑娘。
