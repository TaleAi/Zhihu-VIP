## 简介
末日生存堡垒。


疫情爆发，城市封锁，还好我已经囤了足够多的物资！


在招募安安和陈林之后，我们组成了末日生存小队。一边研究丧尸，一边对抗其他幸存者。


即便我们已经十分小心谨慎，但还是遭遇了重重危机。


小区里的清道夫、超市的武装集团、还有不断变异的丧尸……


我们始终没有放弃，一路搜集到的碎片线索，也在最后拼出了真正的答案。
