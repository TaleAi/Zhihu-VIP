## 简介
改变英语思维，让你把英语用起来。本期专栏将会为你带来 18 个在「用」中「学」的独创英语工具，一整套化繁为简的应用学习模板。


都是成年人了，还在用「应试思维」学英语?


没时间学:想学英语，但是工作忙，总觉得没有时间；不能持续:学一段时间就丢到一边，再学又要从头开始；用不起来:学了太多单词、语法，用的时候根本套不进去；中式英语:经常犯 Chinglish 的毛病，不知道怎么建立英语思维。


丢掉「祖传」的应试思维吧，现在的你，是时候从「应试思维」转向「应用思维」了。
