他是黑色利益链上的重要人物。


2018 年 1 月，警方得到线索，杨文回国了。


侦办民警迅速赶赴其老家，在成都双流机场的出境口岸，把准备再次赴美的杨文抓获。


而此时的杨文，即将名校毕业，并且已经拿到全球最大的电子商务公司的 Offer。


![img](https://pic4.zhimg.com/v2-db7203d907a233d3b7940445e27202e3.webp)

这个曾经的学霸，「别人家的好孩子」，成了阶下囚。


警方对其检查时，发现他身上没有文身。


通过审讯，杨文交代他不是「夯先生」本人，而是他的合伙人。


真正的「夯先生」他从来都没有在现实中见过，也不知道对方的真实姓名。


2016 年上半年，杨文在 91 网上购买了「夯先生」的视频后，成为其粉丝。


随后，被「夯先生」拉到了铁粉群里。


在聊天过程中两人熟悉起来，「夯先生」得知小杨在美国并有绿卡，就提出让他帮忙销售视频，自己主攻「业务」。


小杨痛快答应。


刚开始是兼职帮忙，销售出一个视频拿几十元提成。


后来俩人业务越做越大，小杨变成了合伙人，跟「夯先生」之间二八分账。


除了视频外，小杨还售卖「夯先生」出品的 PUA 教程。


他曾打听过，这种行为在美国不违法，随之愈发胆大，把所有内容上传至百度网盘。


每部视频售价 200 元，买五部送五部，全套视频统一 4000 元。


老客户还会加微信，可以通过微信直接购买。


这些视频仅在小杨的渠道里，就产生了 60 多万元的销售额。


小杨的网盘里，淫秽视频的数量多达 113 个，图片 55 张。


警方查找到他给「夯先生」的汇款账号，发现收款人名为刘 X 辉。


可是，这个刘 X 辉身上也没有文身，也不是「夯先生」本人。


经过调查，刘 X 辉是南京一家法资公司的工程师，也是「夯先生」的粉丝之一。


他负责把小杨汇来的人民币，转成美元，打到「夯先生」指定的一个美国账户里。


狡猾的「夯先生」以为，这样做就很难有人查到他头上。


但警方利用技术手段，很快找到了他。


2018 年 1 月 31 日，丽水警方出现在了上海一所民房内。


![img](https://pic1.zhimg.com/v2-7270e169bf68eca197446503d696181a.webp)

现场，警方厉声质问这个身高一米八，带有文身的男子：


「夯先生是谁？你清不清楚？」


「我们为什么来找你？你清不清楚？」


男人虽然嘴上说着，我不知道你们在说什么。


**但，警方发现了他的双腿在发抖。**


警方在其家中查获了大量还没剪辑过的淫秽视频，以及数个偷拍设备和硬盘。


最终，这个本名为王 X 雷的男子，承认了一切。


他就是「夯先生」本人。


直到这一刻，王 X 雷的妻子才知道，这个温柔体贴的丈夫，竟然是个极度活跃、精力充沛的时间管理大师。


王 X 雷被戴上手铐，瘫坐在走廊里痛哭流涕。


他跟妻子告别，但一切悔之已晚。


![img](https://pica.zhimg.com/v2-c3e7a58f164a8c1db58249f9daebaebe.webp)

![img](https://picx.zhimg.com/v2-b1168572b72a6f9cd0cde828f912ffb3.webp)

4 月份，多家权威媒体对成功抓获「夯先生」的新闻做了报道，在社会上引起了很大反响。


此后，随着案件的破获，越来越多分散在全国各地的「大神」——「秦先生」、「小鲜肉」、「乌索普」、「信哥」、「仓本 C 仔」、「呆哥」、「黄药师」、「大闸蟹」一一落网。


其中，「秦先生」的落网源于一位「前女友」的举报。


在报警前，她就在微信上跟前男友陈宇摊牌：你就是 91 网上的秦先生！


没想到，陈宇回复的是：对，是我。被我 xx 感到荣幸吗？


![img](https://pic1.zhimg.com/v2-cb81242dabff1b3fc5d246ea94fe95e7.webp)

而「仓本 C 仔」被抓捕，则源自于一桩狗血的黑吃黑事件。


「仓本 C 仔」很有头脑，在 91 网站时，他就发现有差异化，才能赚更多钱。


他花钱请人假扮日本 AV 女友，做高自己的身价。


由于视频清晰，尺度大，加上可以表演与普通恋爱之间的差异，果然，「仓本 C 仔」很快被奉为大神。


火了之后，「仓本 C 仔」与精通网络技术的李松合作，新建了一个网站，也带有 91 字样，收取 VIP 充值费用，并形成了八人团队。


至于刚说的黑吃黑事件，是「仓本 C 仔」与一家中国台湾淫秽网站合作时闹起了矛盾，愈演愈烈，无法收场。最终，中国台湾网站一气之下就把「仓本 C 仔」的个人信息在网上曝光了。


丽水警方根据姓名地址等信息锁定了他，并发出网上通缉，最终在他老家的一个酒店里将其抓获。


![img](https://pica.zhimg.com/v2-41596c75f191b4b18bb2108dbcd20829.webp)

![img](https://pic4.zhimg.com/v2-9bd23950a5ecab45a3e8de54ae25ef78.webp)

「91 大神」们被抓获后，法庭陆续做出了审判。


其中：


* 「仓本 C 仔」，被判处有期徒刑六年，并处罚金人民币三十万元。
* 他的搭档李松，被判处有期徒刑十三年，并处罚金人民币三百万元。
* 「夯先生」，被判处有期徒刑十一年，并处罚金人民币四十万元。
* 他的搭档杨文，被判处有期徒刑六年，并处罚金人民币十五万元。
* 「秦先生」，被判处有期徒刑十一年，并处罚金人民币三十万元。

但是，侥幸之人仍在继续，罪恶仍在滋生。

**\* For girls \***


**a.防止被偷拍**


网络上，与情色相关或擦边的内容，一直都是「流量密码」。


巨大流量，就能产出巨额的利润。


虚拟产品，成本低、利润率极大，可以让生产制作者躺着赚钱。


只要传播不停止，就能持续产生利益。


利欲熏心之人，不惜铤而走险。


几乎没有成本的偷拍，成了一个庞大的黑色产业链。


地铁偷拍、公交偷拍、试衣间偷拍、酒店偷拍，就连厕所和浴池都有偷拍。


**在这个庞大的产业链中，女性永远是最大受害者。**


在无法精准识别坏人的当下，你要学会，如何保护自己，防止被偷拍。


入住酒店、民宿，进入出租房的第一时刻，检查房间内是否存在隐藏式摄像头。


进入公共浴室、更衣间、公厕时，检查是否有不常见的多余物品。


乘坐公共交通工具的时候，注意姿态，是否有裙底隐私被拍摄到的可能性。


![img](https://pic1.zhimg.com/v2-71114d10c327f3ab17be224e2966ee32.webp)

**b.识别虚假人设**


熟悉 PUA 这个词的人都知道，PUA 的第一步就是先引起目标的好奇心，根据目标的喜好，塑造相符合的「人设」。


我们来看，哪些场合会存在虚假人设？


A 相亲时、B 酒吧等公共场所、C 学校图书馆、D 公司聚餐、E 网络


比较后会发现，公共场所、网络，非熟人环境。


**在这些环境里认识的人，是真正的陌生人。**


你对 TA 的一切一无所知，TA 的一切都可以是伪装出来的。


当一个陌生异性主动找你搭讪，或在非熟人环境里让你产生以下反应的，


都需谨慎：


「这个人的长相正好卡在我的审美点上……」


「小说里的高富帅终于被我遇上了……」


「他真会撩，跟我异性同学同事比，他好会啊……」


**没有恋爱经验的人，更容易被套路。**


在网络交友时，更需谨慎而行。


网络交友类性侵犯罪，时刻潜伏。自「净网」专项行动开展以来，全国公安机关共破获性侵案件（含积案）2.86 万起，抓获违法犯罪嫌疑人 3.22 万名！


在照片可以 P、车可以租、名片都可以随便印的时代，造假成本很低很低。


常人都有防骗之心，却无防范之法。


简单地相处和聊天，并不能判断一个人身份的真伪。


识别「高富帅」真伪的小 tip:


在工商资料查询网站「天眼查」里，输入姓名或公司名。


倘若他或他家人真的是经商之人，不论是股东还是高管，都能被查询到。


**c.冲动前考虑好代价**


此系列案中，不少受害者是知道男方在用手机拍摄的，还有人在被拍摄后提出：给我看看。


殊不知，手机里的内容被删，也能被修复。手机之外，还有别的摄像设备。


![img](https://pic1.zhimg.com/v2-350c8f6a25c71e807beb299351a2fc8e.webp)

冲动前考虑好代价，是成年人必备的行为准则之一。


除此之外，做好安全措施更是必备的，艾滋病、性病，会对人的一生健康造成影响。


做出亲密行为前，三思。
