## 简介
地理对历史的影响到底有多大？


* 为什么各种「斯坦」的首都，都离得那么近？
* 土耳其的国土大部分位于亚洲，为什么坚称自己是欧洲国家？
* 朝鲜地区一直离我们非常近，为什么历史上却没有一个朝代吞并过它？

……


以往的历史，都是文化、政治、艺术的历史，但与生存和发展息息相关的地理因素，却少有讨论。


但其实，我们的语言、我们的历史都和地理息息相关，也因为地理这个因素，才有了文明的相似、文明的交融以及文明的分裂。


本专栏中，作者安森垚通过极强的跨学科思维能力，以及丰厚的地理、历史知识积累，聚焦于亚洲，从具体的地理问题切入，用新的视角为我们梳理亚洲各国历史，以及国家、民族之间的交融与冲突。
