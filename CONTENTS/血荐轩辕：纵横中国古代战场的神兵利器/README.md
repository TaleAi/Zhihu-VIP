## 简介
「兵者，国之大事」，战争与军事是古代乃至现代最为核心的国家事务之一。


在冷兵器时代的中国，刀枪剑戟「各领风骚」。兵器的演变在一定程度上决定了战争的走向，关系到王朝的兴亡盛衰、国家的聚散离合，兵器也见证着中国民族的战争、融合、进步和壮大。


在这里你可以看到：


* 为什么剑在中国的地位这么高？
* 青龙偃月刀是真的吗？
* 私藏盔甲等于造反？
* 赛约河之战，蒙古弯刀是如何征服欧洲的？

本专栏带你走进「进击」的冷兵器，看遍中国兵器演变历程和兵器对战争的影响。
