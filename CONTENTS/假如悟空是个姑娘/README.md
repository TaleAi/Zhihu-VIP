## 简介
假如悟空是个姑娘，这漫漫取经路将会怎样？


师徒四人，经历九九八十一难，西天取经的故事想必大家都耳熟能详。


但如果悟空是个姑娘，情况就大不一样了。


为花果山的徒子徒孙们生小猴子；


「教育」有色心没色胆的八戒；


与嫦娥王母情同姐妹，把酒言欢。


......


西行之路从此有了新的变化，玉帝、如来、唐僧、二郎神面对红颜悟空也有了不同的态度。


更加奇幻的经历，毫无边际的脑洞，「猴姐」悟空将带你进入一个神奇的世界。
