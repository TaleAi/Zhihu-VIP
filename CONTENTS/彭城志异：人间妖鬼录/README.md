## 简介
我家世代居住在沂河岸边的彭城，这里平日岁月静好，但也许因为沂河发源诡异，我自幼便遭遇过许多邪物和怪事。


深夜旷野上阎王爷派来的红花轿、木箱里被砍成两半的会动的女尸、祖宅里身上长满眼睛的婴儿、专偷孩子的木腿老太太、为人妻为人母的猫妖、娶亲的黄皮子……


这些玄之又玄的奇诡之物，仿佛暗夜中的一只眼，窥探着人间的罪恶与温情。


1.《尸追》


小时候，我和同学玩捉迷藏，碰见一具死去多日的女尸。之后这具女尸折磨了我半辈子，无论我到哪里，她都会追去，为此我跑遍了半个中国，可还是没能摆脱她。


姥姥给了我一张黄符，让我贴身带着，二十八岁之前不要拿掉，如果丢了就再来找她。可我姥姥在我十几岁的时候就去世了，后面的十年我几乎每日都生活在恐惧之中。


直到我长大成人，做了母亲，才了解了当年女尸的真相。原来，她来找我，并不是要害我，而是来报恩的。


2.《红花轿》


我们那一直有一个传说，如果半夜走路看到红花轿，那是阎王爷娶媳妇了，要赶紧逃。但是逃也没有用，他会找到你家里去，阎王爷看中的人，你跑得了吗？


大人都说这个事情是假的，但是我母亲告诉我，她真就遇到过红花轿。


后来，我最好的朋友也被红花轿接走了，她怕路上寂寞，还差点拉上了我。
