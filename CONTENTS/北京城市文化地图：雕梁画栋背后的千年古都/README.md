## 简介
赏城市兴衰，读历史沉浮。老戏骨王刚首次献声旅游音频节目，倾情演绎「和大人」，带你领略北京的万种风韵，阅尽雕梁画栋背后的千年古都。


八大胡同上演过怎样的传奇故事？


京城「雅游之地」的琉璃厂有过怎样的盛世？


「开国第一宴」的主厨师傅怎样用大锅炒出来美味佳肴？


老戏骨王刚将以戏说的方式讲述皇城根下的故事，倾情演绎《梦断紫禁城》中和珅片断，追忆两次进入恭王府拍戏的幕后花絮。


用声音记录一座城，用心体味街巷里弄，城市文化地图，声音版的「千里江山图」。本专栏带你走过故宫、王府井，看看什刹海与法源寺。一起读懂北京。
