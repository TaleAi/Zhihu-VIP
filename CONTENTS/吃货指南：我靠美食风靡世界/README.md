## 简介
美食和爱，治愈人生。
专栏收录各种跟美食相关的小故事，爱情亲情友情，温暖感人。



**《我在仙界搞吃播》**


我貔貅，人间吃货，是个没得灵魂的干饭人，他饕餮，Aka 吃播机器，能上九天吞月，下五洋啃鳖。


我们的目标是……天界吃播第一人！我们的口号是，吃饭睡觉喝饮料，喝完再吃又睡觉。


**《我在后宫卖奶茶》**


穿越后，我本以为我可以靠现代人的智慧征服这个世界。


没想到皇帝在宫宴之上来了段 rap，「大金链子小手表，蜜雪冰城就是好！」


更没想到国师来我店里，非常轻松地点了个单，「杨枝甘露，三分甜，加冰。」


最没想到，把他们教成这样的人，是我？
