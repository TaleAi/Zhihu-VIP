**- 闪落女孩 -**


女孩江小岛，出生在偏远地区单亲家庭。


她 18 岁那年，母亲贪财被骗，母亲先后两次把她卖掉。


生死一线之际，她获得了一种不定期被动发作的瞬间移动能力。


她把自己的能力命名为「闪落」。


不管上一秒在做什么，下一秒她都有可能凭空消失。


在普通人眼中，她无所不在等同于不存在。


她没有家，没有朋友，没有活下去的意义。


她想要结束这一切，就在第 60 次闪落。


  



**- 深空恐惧 -**


作为一名护工，我负责照顾一位编号为 1023 的植物人客户。


在一次新型疗法中，沉睡已久的 1023 醒过来了。


醒过来的 1023 口中发出没有人能理解的声音，并表现出异常的攻击性；


他很快就学会了说话，但所说的话反而更令人不安。


我确信 1023 体内的意识并不是他本人而是一个来自深空的外星意识……
