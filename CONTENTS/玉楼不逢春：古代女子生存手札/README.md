## 简介
那些生活在大宅宫墙里的女子，或是温良恭顺，或是刁蛮跋扈。为了地位永固或是国仇家恨，她们处心积虑、步步为营，在各自的故事里演绎着这世间的爱恨情仇。本专栏记录了古代女子在爱情和权力、家国与竹马间艰难抉择，最终活出精彩人生的故事。


《玉楼春》


他一眼瞧中了她那双像极了那个女人的眼睛……可他不知道，她就是那个被他灭了满门的大家闺秀；她是玉春楼的花魁，也是曾经备受宠爱的千金小姐，她能深情款款的同他周旋床第间，也能风轻云淡的拿出匕首插入他的心脏……


到最后，爱与恨早已经分不清界限……


《清平乐》


我是大梁国刁蛮跋扈、睚眦必报的长公主，年十七，名平月，号长乐。


大梁上下人尽皆知，长公主卿平月，与权倾朝野的摄政王符离处处针锋相对，为的就是有一天夺了他的兵权，再要了他的人。


没错，本公主不仅无法无天，还毫不避嫌地耽于本朝摄政王之美色。


于是，符离为了自保，决定拿我去和亲。


我：你想得美。
