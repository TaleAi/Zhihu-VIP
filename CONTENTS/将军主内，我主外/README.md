## 简介
将军和下堂妻灵魂互换的爆笑日常。


威武大将军萧情对正妻宋微微百般厌恶，打死也想不到有一天他会和他最讨厌的宋薇薇灵魂互换。


随即而来的是——


温柔的青梅竹马对他恶语相向。


懂事的小妾们每天想方设法给他下套。


下毒、落水、滑胎……但凡是人能想到的屎盆子都扣到了他的脑袋上。


宋微微不仅落井下石，还嗑着瓜子看他的笑话：「来人！杖责三十！以儆效尤！」


宋微微在当大将军的道路上一去不复返，倒霉催的萧情在对付小白花的道路上越勇越挫。
