## 简介
本专栏包含了 7 篇对语文课文的新编故事，给课文中细思恐极的情节插上脑洞的翅膀，打开另一个多彩的中文世界。



还记得每篇课文结束后，你看到那一句「背诵全文」时的恐慌吗？


还记得在诗圣杜甫的插画上的恶搞涂鸦吗？


你肯定记得，甚至记得对课文的胡思乱想：一个不用背诵全文、神奇有趣的 B 面脑洞语文。


于是，语文课本就成了另一个神奇的世界。


《口技》中的表演是斩妖除魔的少年侠事：「屏风中妖风四起，凄厉声不绝于耳，又有剑客怒叱声，火光爆炸声，利剑入体声，妖魔惨叫声。」；


《逍遥游》的鲲与鹏是真实存在的，「而鲲之大小，全在我宁家筹师的一念之间」；


《木兰辞》里花木兰为什么打仗这么厉害，是不是因为她是「兵王」穿越附身的原因？


还有《荆轲刺秦》、《牛郎织女》、《核舟记》……


都是你熟悉的课文，也是惊奇的好玩故事。
