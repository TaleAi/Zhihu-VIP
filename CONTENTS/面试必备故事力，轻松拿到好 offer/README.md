## 简介
有感染力的讲述，多多少少都带着「故事感」。而在面试这个关乎职业生涯走向的重要环节中，「故事力」的巧妙运用，可以大大提高你的胜算。如何做一个有故事力的应聘者，在各种面试中过关斩将？Huber 博士将在 5 小节课程中带你用「故事力」轻松应对各种面试场景。


-面试一到自我介绍环节就抓耳挠腮，不知说些什么，草草收场？


-面试官的各种刁钻敏感问题，如何优雅地避开雷区？


-为什么十分的实力，在面试中费尽力气也才展示了三四分？


同样是职场新人，「面霸」们所向披靡，百发百中，你却经常在面试环节铩羽而归——你有没有想过，你和「面霸」的差距到底在哪里？


* 记忆点：候选人众多的时候，留下一句有记忆点的话，就能让自己脱颖而出；
* 业务力：识别面试官的考察关卡，最大效率地秀出肌肉；
* 信任感：用有限的表现时间，让面试官迅速得出你很「靠谱」的结论。

Huber 老师将在这门课程中，结合 20+ 年的管理与培训经验，教你面试必备的故事力。让你：


* 用有逻辑和说服力的语言展示自己
* 让面试官对你的实力和人品产生信任感
* 有效把握各种重要面试机会
