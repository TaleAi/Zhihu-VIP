## 简介
如果将嫦娥奔月改编成言情小说，会是怎样的故事？


甜文女主嫦娥最近有点烦。


大婚前夕，未婚夫后羿把她给绿了，对象还是她最好的闺蜜。


再后来，一夜醉酒，她把后羿也绿了。对象……还是只奶！兔！精！


兔子看上去奶 fufu 的，内里却是个狂拽酷的霸道总裁，还总黏着她，甩都甩不掉。


为了挣钱养兔子，嫦娥不得不……接了个寻宝的活。


这活凶险万分，幸好一路上有奶兔精为她保驾护航。


等等，这兔子法力低微，为啥能凭一己之力斩杀各路妖魔鬼怪？


这不对劲。


兔子还一个劲儿装：「嘤嘤嘤，人家是一只平平无奇的奶兔精啦，绝对没有什么特殊身份哦。」


鬼才信！


还有她的渣前任后羿，为啥要扮成另一个人，装作不认识她，也来抢夺宝贝？


直到真相渐渐浮出水面，嫦娥才知道，原来她和兔子的缘分，早在很久之前就已经注定。


一切不过是他蓄谋已久，情根深种。
