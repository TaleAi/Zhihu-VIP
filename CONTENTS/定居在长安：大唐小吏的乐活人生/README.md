## 简介
影视剧《长安十二时辰》为观众打开了一幅盛唐居民日常生活的瑰丽画卷，也引发了大家的热议：唐朝人的真实生活究竟是什么样的？本专栏中，作家王一凡塑造了一个居住在大唐长安的公务员「唐小吏」，通过他的视角，从饮食起居、工作求学、婚恋习俗、消遣娱乐等各个方面，生动还原唐朝人的日常生活，带你了解大唐盛世的开放文化、发达经济，是如何影响人们的思想与生活方式的。


「定居在大唐长安是种什么体验？」


大多数人印象里的大唐，那可是一个万国来朝的空前盛世，首都长安这座「国际化大都市」，更是世界闻名。生活在这里一定很美好吧。


唐朝人的真实生活真的很美好吗？看看长安「本地人」唐小吏是如何过日子的：


大早上去如厕，却发现厕所里没有卫生纸，清洁只能靠竹签；


想好好休息两天睡睡懒觉？很遗憾：公务员的双休？不存在的；


兴致勃勃揣着块银子去购物，店主却这样回答：本朝拒收银两；


新时代的女性想高歌「单身快乐」？也不行！「剩女」太多，地方官可要遭殃了；


似乎也不太「美好」？接着往下看：


我的爱情我做主！一千年前的唐小吏已经谈了场「自由恋爱」；


长夜漫漫太无聊？别担心，大唐夜生活，丰富到你无法想象；


好友相聚怎么安排？挑家「洋酒吧」，点杯「绿蚁酒」，看胡姬偏偏起舞；


作者王一凡长居西安（大唐长安），她多次走访当地的各处文化古迹，翻阅大量古籍专著，力求为我们呈现唐朝人最真实的生活状态。


希望这些历史书中不曾细说的唐朝「冷知识」，能让你对大唐风貌有耳目一新的了解～
