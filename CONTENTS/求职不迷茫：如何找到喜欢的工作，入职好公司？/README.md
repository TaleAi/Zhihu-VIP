# 求职不迷茫：如何找到喜欢的工作，入职好公司？

### 状态：已完结，共 16 节

### 存档日期：2023-04-23

## 简介
对于职场新人，第一份工作极大影响着职业发展的方向。如果工作了很多年才发觉方向错误，付出的精力和时间成本必然是巨大的。本专栏集合 16 节求职必修课，通过专业咨询师的指导、案例故事，帮助求职经验欠佳的新人了解自己，斩获好工作。


找工作，难免会踩坑。虽然成长也在于此，但也不是所有坑都要踩一遍。比如：


➤ 刚毕业时太迷茫，单纯觉得这份工作来钱快就躬身入局，多年后想改赛道转行，已经非常吃力。


➤ 明明知道了自己喜欢干什么，适合干什么，但因为没什么工作经验，不能写出高含金量的简历，让 HR 看到自己的价值；


➤ 面对严肃的面试官，不敢自荐，回答问题也抓不住重点，导致重要问题一笔带过，不重要问题长篇大论，从而错失了入职好公司的机会。


找工作是门技术活儿，与其在多次踩坑后涨经验，不如一开始就科学认知、规避风险，尽早明确自己所属的人才类型，逐步对自己进行塑造和培养。通过被印证可靠的写简历、面试的方法论，进入一个优质的平台，发挥自我的价值。


本专栏的作者莱茵，是资深职业咨询师，也是中国人民大学、南开大学、对外经贸大学等 985、211 名校职业生涯规划讲师。她将从「职业生涯规划」「简历制作」「求职面试」三个方面，结合从业以来的诸多咨询案例，帮你定位自己喜欢的工作，入职好公司。
