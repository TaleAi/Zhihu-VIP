## 简介
工作和生活中，我们总是会遇到一些难相处的人，让人感叹，真不知道 TA 在想什么。或许，你该学点心理学了，75 条趣味心理学知识，帮你获得与人高效沟通的 5 大方法。快速提高自己的社交能力，和沟通能力。


在日常的工作和生活中，我们经常会遇到一些难相处的人：


挑剔计较、虚伪、两面三刀、矫情、心机深沉、得理不让人、没有边界感、过分敏感……


面对这些难相处的人，我们究竟是要积极相处还是唯恐避之不及？


生活经验告诉我们：


良好的人际关系真的非常重要，


它能为我们的工作和生活提供很大的帮助。


那么，我们如何与不同性格的人都能相处的来呢？


如果你能吃透对方的内心，看到对方话语中隐藏的诉求与渴望，


那么，不管是什么类型的人，你都能轻松与他相处！


本课程中，主讲人少毅将为你讲解 75 条实用的趣味心理学知识，帮你：


情商提升：读懂对方内心的真实想法。


沟通策略：听懂对方话语中的深层需求。


和谐关系：与不同性格的人轻松相处。
