中国古代是个等级制度非常严格的时代，这其中呢，「三教九流」就是一个典型。


三教即儒释道，九流则分为上九流、中九流和下九流。


你知道古代的各种职业都处于什么等级吗？


人们常说「士农工商」。但其实在这里商人虽然排在了最末尾，却也是在上九流的最末端，所以放在整个古代社会来说商人的地位其实并不低。


而古装剧里的师爷，那个站在审案主官县令或州府长官的身边，时而与之窃窃私语的学究式的人物，那个为主官出谋划策的人则处于下九流的地位。


此外，还有媒婆、隐士、娼妓等也都被划为三六九等。
