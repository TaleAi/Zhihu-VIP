**- 最后的祭品 -**


我是一名对神话和民间传说颇有研究的作家，非常擅长写有些恐怖的奇幻小说。


有一天编辑约我吃饭，上车后才发现不是他，是几个我不认识的男人。


领头的人在车上表明了身份，原来他是警察，约我是想拜托我做一件事。


他告诉我，半年前有一家四口神秘失踪，只剩下二十二岁的孙子。


  



- **白噪音 -**


雷雨夜后，胖子在楼下屋子的角落里发现了精神失常的女人阿青。


房子闲置多年，其主人早已搬去国外销声匿迹，


但在女人阿青的记忆里，雷雨那夜被屋子「吃掉」的，


除了租房给自己的神秘房东，还有她的老公赵楠，


一个根本不存在的男人......


  



**- 模因 -**


这个世界病了。


2023 年，一种病毒突然出现。


或者这并不是病毒，更准确的说法，它叫做「模因」。


一开始，受到感染的是猫、狗、鸟、宠物，后来是高山、深海、房屋、树木，乃至你手中的汉堡、身上的衬衫，所有被感染的对象都显示出了一种「生命」的特性，


而本就具有生命的动物，则表现出了更高的智力，但却并不遵循人类的沟通逻辑……


唯一的病毒绝缘体，是人类本身。


可惜的是，人类本身并不知道这一点。
