## 简介
重新品读西游记，与书中的人物对话，一窥人物的内心世界，或许你会发现，某个神佛像你曾经的领导，某只妖怪像你身边的朋友，某个凡人的遭遇，像你自己经历过的某件小事。



比如很多人说《西游记》是一本非常暗黑的小说，里边充斥着吃人、阴谋、血腥、情色，绝不像电视剧拍得那么有童趣；


还有人说《西游记》是一本玄妙的小说，在取经故事表象下，另外隐藏着一条道家修炼修真的法门；


还有人说《西游记》写的是佛道相争，隐射政治；也有人说把《西游记》倒过来读，才能品味出它的真味道……


你相信哪一种说法？


长久以来，大家都热衷于揣测《西游记》隐藏着什么样的目的，却忽略了把它仅仅当一本小说细细品读一番，看看它刻画了哪些人物，营造了什么样的环境。


《西游记》以孙悟空开篇，以唐僧师徒取经为主体，营造了一个包含了神佛、凡人、精怪的世界。这个世界分四大部洲，形形色色的角色被取经这件事搅在一起，让我们得以窥探全貌。


所谓读书，都是读别人的书，取自己的经。


学问之路，好比西行之途，有幸与诸位同行。
