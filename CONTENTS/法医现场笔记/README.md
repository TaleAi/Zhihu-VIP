## 简介
法医这门职业，常常显得冷酷又神秘。《法医现场笔记》中，死者代言人将通过法医的第一视角，带我们回到他经历过的一个个刑侦现场，为我们讲讲现场的故事，用专业的法医知识抽丝剥茧，找出案件的真相。书中并无过于戏剧化的夸张内容，所有案例和知识都基于专业法医的专业知识。


「人这一死啊，很多的人情冷暖就都出来了。」


可对于法医来说，最重要之处，在于从病理上对死因作出判断，从而给警方搭建证据链提供最有效的支持。也因于此，法医这门职业，常常显得冷酷又神秘。


专栏作者 @死者代言人，曾在知乎解答了诸多关于法医的专业内容。这一次，他经过慎重考虑，终于决定为我们来讲讲故事，用故事，带我们回到他经历过的一个个刑侦现场。


《法医现场笔记》中，有他第一次去现场的经历，也有他遇到过的各种离奇案件：河中浮尸头顶上不得其解的钝器伤，家具城楼顶凭空出现的高度腐烂的尸体， 有人争着抢着来当杀人犯的车祸现场……从法医的第一视角，我们会读到抽丝剥茧找出真相的过程。书中并无过于戏剧化的夸张内容，所有案例和知识都基于专业法医的专业知识。


更重要的是，生死面前，我们也从法医的角度，看到了更多人情冷暖。


作者 @死者代言人，曾出版过知乎电子书《我是法医》。毕业于四川大学华西医学中心。曾经为了进入法医这一行，留过级、蹲过班，成功解锁「年级最老」成就。从事着为死者代言，替逝者发声的工作。可以接受各种重口味，然而却最怕恐怖片。
