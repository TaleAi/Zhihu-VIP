## 简介
学习是一种能力，聪明是可以学会的。


我们总是轻易地将和「学习」相关的能力，定义为一种聪明人才有的天赋，所以很自然地接受了——


我记性不够好，所以背书才会背了后面忘前面；


我性格太跳，所以无法沉下心来持续学习；


甚至——


我太笨了，所以别人一看就会的东西，我怎么也学不会……


但其实，大部分的学习远不到考验天赋的程度，我们所认为的「聪明」都是可以被学会的——


放弃「完美思维」，我们可以小步快跑，让学习从量变积累到质变；


利用「固定时间学习法」，我们可以成为能坐得住、沉下心学习的人；


练习「记忆技巧」，我们也可以「过目不忘」。


在本专栏中，我们将「学习」这种能力，拆解成思维方式、学习习惯、学习技巧 3 个版块，带着大家逐个击破，掌握高效的学习方法，成为会学习的「聪明人」。
