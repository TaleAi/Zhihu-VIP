## 简介
家人让你感到绝望压抑？怎样从不健康的家庭关系中逃离？明明是家人，为什么却像「恐怖分子」？专业心理咨询师将在本专栏内介绍 7 种不同类型的人格障碍特征，以及如何应对这 7 种类型的家人。每天进步一点点，让我们从正视问题开始。


家，在主流印象中一直是温暖、爱和天堂的代名词。但事实上，不正确的亲密关系，甚至有偏激人格特征的家庭成员，给家人带来的伤害甚至灾难也是超出人们想象的。


**人格障碍是心理问题的「硬核」。它是比我们通常所知道的心理问题（比如抑郁症、强迫症等）严重得多的情况。**


人格障碍患者有异于常人的思维方式、情感模式和行事风格，僵化固定，且持续时间长，难以改变，会影响到当事人的人际关系和社会功能。


大部分人格障碍都不认为自己有问题，所以不会成为心理治疗的来访者。


但他们是人际关系中的「恐怖分子」， 成为来访者的常常是他们的家人，尤其是孩子。这些孩子投胎时选择了 hard 模式，几乎是和披着人皮的怪兽一起长大。


本专栏会主要讨论几种常见的人格障碍的类型，以及与这几种类型的家人如何相处的原则。


#### **适合谁**


感到家庭让自己疲惫甚至绝望的人


希望摆脱家庭给自己带来困扰甚至创伤的人


想要重新认识家庭关系，重新定义和发现自己的人


#### **你将获得**


1）调整自己对家庭的合理预期：和有人格障碍的家人相处很难，但并非不可能。


2）了解自己可以选择怎样的策略：开始看到问题，本身就是进步。


3）意识到你不是一个人：很多人面临和你一样的问题，有些人成功了，有些人失败了，更多的人正在路上。
