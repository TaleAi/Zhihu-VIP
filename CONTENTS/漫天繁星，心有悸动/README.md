——《锦鲤女神》


520 那天我准备跟上司表白，他却一天连排三场相亲局。


被扎心的我还得接人，买花，订餐厅。


情场失意，钱包鼓起。


我意外获得锦鲤体质。


买彩票，中一个亿。追尾帅哥，帅哥对我一见钟情。吃十块钱海螺，开出海螺珠。就连喝瓶矿泉水都再来一瓶。


天啦撸！


好像突然没那么难过了。


离职，抽最贵的华子，把烟吐上司脸上，骂他是个小趴菜！


做一个独美的富婆！


  



——《成为三个帅哥的团宠》


喝酸奶都要舔三遍盖儿的我意外获得巨额财产。


前提是要在财产主人的三个孙子中选择一位成为我的合法丈夫。


据说——


大哥优雅矜贵。


二哥长相对胃。


三哥刚满十八岁。


见面之后发现，二哥居然还是我的怨种前男友，为了不让前任和我共享财产，我早上和大哥吃早餐，晚上和三哥看电影，行程排得满满登登。


前男友还吃醋了，不知道是因为财产，还是因为从没忘记过我。


  



——《男神，恋爱吧！》


渣男友不想努力了，傍上褶子都可以夹苍蝇的富婆，喊人甜心 baby。


分手的时候还给我列出了一张清单，让我归还交往期间的费用。


饭钱 AA 就算了，就连上周他买了一瓶农夫山泉，我就喝了两口，还要收我五毛。


我受不了直接打钱分手，生气发了个抖音吐槽后，视频还火了。


我大学时候暗恋的男神突然出现在评论区。


「过来，哥刚发了年终奖，可劲造！」


我才知道，曾经以为的暗恋无果，其实是两个人因为腼腆不敢捅破才错过。


他也一直关注着我的社交软件，准备等我结婚就取关。


还好，我及时醒悟，我们没有错过。
