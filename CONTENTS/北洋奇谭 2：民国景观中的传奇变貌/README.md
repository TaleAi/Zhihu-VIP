## 简介
柔光之下，你很难看到民国社会的真正面貌，尤其是底层的混乱和无须，既危险惊悚，又荒诞离奇。穿越到真正的民国，用现代文明的滤镜重新审慎，窥看其中的传奇变貌。


这是魔宙的历史非虚构专栏。讲述北洋时期前后的近代中国真实历史故事，从清代奇趣话题到民国传奇人物，每个故事都是一次与历史的奇遇。 


在全新《北洋奇谭》第二季中，有更加令人惊觉荒诞离奇和慷慨激昂的传奇异事：


* 民国性学大学搞了一场「色情征文大赛」，为了怼他，鲁迅写了个黄段子
* 「他好我也好」，那些靠袒胸露乳的女性吸引眼球广告
* 前卫、性解放、政治讽刺……与民国政府「硬刚」的传奇杂志
* 为了美国梦而移民，那些被囚禁在「天使岛」的底层人民是如何受尽屈辱的？
* 最早留美学生、华盛顿监狱待得最久的囚犯，却推动了美国司法进步

……


在此之前，你从未如此打量民国时代。
