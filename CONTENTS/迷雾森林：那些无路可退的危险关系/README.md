## 简介
这是一个现代都市惊悚故事集。妻子的日记中密密麻麻写着「杀死他」，他是谁？网约车司机因后排座位的异味屡屡被投诉，他发现那不是普通臭味，而是血腥味。男模带过很多女孩回出租屋，但没有一个女孩走出那个房间。
迷雾重重，人心诡谲，且看危险关系的结局如何。


欢迎来到「迷雾森林」。


有人说，雾比黑暗更可怕，黑暗尚有光可以穿透，而迷雾只能等待它消弭。


当迷雾散去，你触到的就是真相吗？


**— 致命妻子 —**


我的妻子是个编剧，她为了创造出精彩剧情，把我当做试验品。


我要配合她演医生、律师、洗脚城的客人、家暴的丈夫。


后来，她变本加厉将我囚禁家中，想杀我。


我恐惧之下杀了她，把她藏在冰箱里。


当我慌慌张张逃出家门，却遇到一个人告诉我，我妻子在一年前就死了。


**— 请给个好评 —**


我从部队退役后开上了网约车，上夜班。


工作很顺利，我一晚能拉 50 多单，清一色的五星好评。


但一周后，我开始连续收到差评，投诉我车里的味道比猪下水还难闻。


我趴到后排仔细一闻，发现是座椅里发出的气味，有点像臭脚丫子，有点像豆豉。


也有点像血腥味。


……


**迷雾森林无鬼，是人心中有鬼。**
