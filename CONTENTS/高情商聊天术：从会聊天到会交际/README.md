99% 的人际问题出在不会说话上，本专栏教你做一个真正会聊天的人！

跟陌生人聊天，如何找到令对方兴奋的话题？


跟客户谈判，如何引起对方的重视？


说服领导时，怎么做到有理有据不卑不亢？


安慰朋友时，怎样说效果更好？


……


这个专栏，语言表达高手张超帮你提高情商，教你真正会聊天。听说两种能力训练+3 大场景模拟对话+10 年表达经验锤炼。影响、说服、沟通、谈判，谈笑间搞定！
