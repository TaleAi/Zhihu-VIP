## 简介
午夜时分，楼上总是传来奇怪的声音；被活埋进棺的男人，突然直起了身子；半夜，扎纸人的铺子，突然来了一位诡异的客人……让人心跳加速的生死游戏，下一个轮到谁了？


我们镇子发生了怪事，最开始是一户人家做丧事，半夜亲属守夜，亲眼看见棺材两边的纸人活了。后来怪事就多了，半夜总有人看见街上有纸人在走路，有时候还突然扭头，吓坏了好几个人。


——《纸人》


我不知道自己躺了多久，我的头顶是密封的，这里比我想象中的还要狭小，闻着有股淡淡的雄黄酒味儿，这味道我十分的熟悉，或者说记忆深刻。我奶奶去世的时候，棺材外面洒的就是雄黄酒，目的是为了防止一些食腐动物啃食尸体。


——《棺中二十四时》


楼上又开始了，那种奇怪的声音，像是被堵住嘴巴发出的声音，还有脚步不断来回的碰撞声……


——《楼上惊魂》
