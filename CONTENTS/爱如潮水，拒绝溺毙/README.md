## 简介
就想说，你到底有没有做好撩汉的觉悟啊！！！


搭讪帅哥，扫了我的微信二维码，页面上却显示我们已经是好友。


他手机上的备注是 AAA 潮鞋特卖。


那一瞬间，我想了很多……


  ——《AAA 潮鞋特卖请求添加您为好友》


  



世界上有这样恶心的一种人，他们借着朋友之名，行着情侣之实。


他们之间互相关心、嘘寒问暖、谈天说地、无所不聊，他们出现在彼此生命中的每一个时刻，是彼此生命中不可或缺的人，但就是不在一起。


我就有一个这样至阴至贱的前男友，他还有一个至阴至贱的「好朋友」。


他称她为女、弟、弟。


  ——《女弟弟》


  



撞我的司机是个帅哥。


他一路把我护送进了手术室。我想说，行了，可以了，知道你的诚意了，不会讹你钱的，手术室就不用陪着了。


但是帅哥换了一身行头，手握两把亮闪闪的手术刀，眼神里充满着自信和坚毅，他说：「放心，死不了，我撞的，我知道。」


  ——《飞来的爱情》
