# 高中生：三步打通理科思维

## 简介
很多高中同学，都面临「理科学习难以开窍、理科思维没有打通」的问题。在数理科目上花费大量时间，反复刷题，然而收效甚微。然而到底什么是理科思维？如何培养理科思维呢？


本专栏将会全面拆解理科思维，带你进入理科学习之路。


概要的说，打通理科思维，要从树立「理科三观」开始：


第一步：形成理科学习观


* 为什么记忆刷题没有效果？
* 一次搞定的学习是怎么样的？
* 如何做到基于通透理解的学习？
* 如何建构起融会贯通举一反三的知识体系？

第二步：形成理科思维观


* 为什么缺乏效果我都还不得不刷题、找不到出路？
* 要实现深度理解、通透掌握需要怎样的思维能力、思维方式？
* 如何建立这样的思维能力和思维方式？

第三步：形成理科教育观


* 理科教育就是讲课刷题应试吗？
* 为什么学校更容易鼓励大家刷题、而难以发展高度的思考能力理性精神？
* 真正能培养出理科思维的教育是怎样的？
* 如何获取和创造这样的教育？

这三观，也就构成了本书的 3 大部分。


作者简介：


徐强，前微软工程师、知乎 30 万关注者。理科教育 4.0 体系创始人、知识森林学习软件开发者。


多数人认为，透彻理解、融会贯通的学习是一种天赋，是少数学神的专利。而他认为，这是每个同学都可以掌握的能力。你也可以不刷题，实现全科优秀。


为此，他研发了以通透理解为核心的学习方法和科技，让普通的同学也可以走上学神之路，享受洞察真相的乐趣。
