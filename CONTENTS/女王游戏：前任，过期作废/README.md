## 简介
当灿烂人生突然转了个弯，一手好牌骤然变成了必输局，是否就再无翻身可能？当从万众瞩目的人生赢家变成了艰苦挣扎的小透明，她的人生被迫重启，怎样才能改变命运？且看她女王归来，在黑暗中破出最亮的光。


乔伊奋斗八年，从十八线龙套位居一线影后，还创立了自己的公司伊城传媒，终成人生赢家，可此时她却发现一同打拼的丈夫早已有了异心。在被撞破后，他竟对她下了毒手。


宋乔伊是同在娱乐圈挣扎的小透明，19 岁意外怀孕生下宝宝后，她只能选择做娱乐圈某男星的名义上的妻子，以保护宝宝不被戳脊梁骨，可这些年因为她的懦弱，母子二人深受男星的家暴。


一场意外，乔伊成了宋乔伊。


女王归来，人生改写。
