## 简介
一个国富民穷的「盛世」，一个历史倒退的「进步时代」。被翻拍影视剧数量最多、收视率最高的传奇帝王大 IP 乾隆皇帝，他真实的政治生涯究竟有多传奇？影视剧中关于乾隆的文治武功和风流韵事都是真的吗？为什么说乾隆治世下的封建中国既是盛世巅峰，也是中国近代化道路上困难重重的罪魁祸首？


为什么说雍正是个心机老爹，利用自己的儿子上位？


康熙为何一眼相中乾隆的八字？在他年仅 12 岁时就决定传位于他？


天性善良、身强力壮的乾隆皇帝，与孝贤皇后之间真的像影视剧中写的那么恩爱吗？


借着皇后病逝的机会，他又冷血地肃清多少政治隐患？


大清王朝唯一一位死后配享太庙的汉臣张廷玉，如何与皇帝展开「宫心计」巧妙周旋？


国库充盈，四海臣服，GDP 占全球三分之一的乾隆盛世，为何在西方人眼中却是贫穷与落后得「触目惊心」？


本专栏从康熙帝与幼年乾隆的相遇写起，回顾了乾隆皇帝毁誉参半的一生，揭露了乾隆盛世武功背后残忍与腐坏的真相。


他创造了空前的政治稳定，养活了数量空前的人口，奠定了中国今天的版图。同时也给中华民族精神上造成了永久性的创伤，精心塑造出了最适合统治和奴役的、精神上贫乏空洞的饥饿盛世。
