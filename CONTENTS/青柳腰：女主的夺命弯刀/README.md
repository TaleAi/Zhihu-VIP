## 简介
这里的女主曾桎梏于泥潭，但她们有信仰与力量。若命运不公，就跟它斗到底。


**- 青柳玉姿 -**


成为定国公府世子爷的通房之前，我曾是扬州盐商养在府里的瘦马。


于他们而言，我仅是权贵手中的玩物。


但他们不知，青柳细腰，也是一把温柔弯刀。


  



**- 芙蓉乱 -**


汉军攻陷洛阳，我那九五至尊的丈夫跪在叛军脚下瑟瑟发抖，如待宰的羔羊。


「皇后在椒房殿，别杀朕……」


我嫁给他五年，生下女儿河清公主，危急关头，他义无反顾地将我献了出去。


  



**- 烬燃 -**


我上大学时勤工俭学，在 ktv 做兼职，学校疯传我是干夜场的，如他们所愿，多年后我在当地夜总会做包厢营销。


同学聚会上她们冷嘲热讽，饭店偶遇的那位传闻中不苟言笑的叶大律师，却西装革履半蹲在我面前——


「嫣嫣，拜托，今晚给我留个厢……」
