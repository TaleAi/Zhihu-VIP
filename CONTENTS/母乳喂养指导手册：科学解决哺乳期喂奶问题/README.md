## 简介
少奶、涨奶、回奶、转奶、断奶……妇产护士教你解决宝宝奶水问题。


我们都知道，母乳是妈妈给新生宝宝的一份礼物，母乳喂养的妈妈也十分令人钦佩。


她们刚刚经历完十级疼痛指数的生产痛，下一秒又要面临一系列令人痛到无法呼吸的「开奶、催奶、涨奶、堵奶」等难题，而且最终每位母乳妈妈都将面临「回奶、断奶」的不舍和无奈。


同时，这一过程中，很多宝妈也会为一些问题感到苦恼：


比如，很多新手妈妈认为，哺乳必然会导致妈妈发胖？产后想要奶水多一定要多喝催乳汤？哺乳就是产后乳房下垂的「罪魁祸首」吗……


看得出，妈妈们真的是身心都受到了巨大的压力。但也许你真的不必那么辛苦，只是你的方法可能不对。


本课程特邀原上海仁济医院妇产科护士长蔡红，将她 20 年的产妇和新生儿护理经验提炼成科学易操作的方法，给你一份最直接有效的母乳喂养指导，不仅能让你的母乳喂养更加轻松，还能让它成为你和宝宝之间最温馨、舒适的情感交流。
