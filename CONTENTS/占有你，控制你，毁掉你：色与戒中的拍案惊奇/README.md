## 简介
这是一个拍案惊奇式的两性故事专栏。出轨、小三、年下姐弟恋、师生、孩子非亲生……专栏中包含了当代种种男女纠葛，每一种都会引发强烈共鸣。


男和女，仅仅两个性别，他们之间却可以发生无数纠缠不清，狗血淋头的故事。 


**怀上姐夫孩子「逼宫」亲姐：**刘松茂家里的房子拆迁后，他变有钱也变膨胀了，俗话说，男人有钱就变坏，他和公司的女孩搞了起来。在家人的帮助下，妻子张青在斗退小三，却没想到引狼入室，丈夫又和亲妹妹搞在了一起，妹妹也怀孕且逼宫自己。这场不可调和的狗血矛盾，到底要以怎样的代价为结局？


**妻子的「烟灰」，成了他心中的魔障：**陈焕良回家发现床上有一丝烟灰，质问妻子，妻子不愿意说。他对自己漂亮的老婆产生了无限的怀疑。在和妻子争吵、冷战之中，他意外染上毒品。妻子提出离婚，女儿也一并离开，他在戒毒所里还在纠结那一丝烟灰，慢慢恨意越来越浓。他出了戒毒所之后，买了枪杀了他怀疑和妻子有染的男人和他老婆，后来，他变成了要杀妻子全家的狂魔…… 


**拯救丈夫却遭逼杀，谁能阻止那把带血的屠刀：**韩嫣与夏海林是一对北漂情侣，两人结婚前夕，夏海林意外沦为植物人。为拯救男友与爱情，韩嫣辞职做网红。2016 年，夏海林奇迹般康复，与韩嫣结婚。婚后半年，韩嫣怀孕了。孰料胎检时，她却被查出是艾滋病毒携带者。原来，韩嫣在做网红的过程中，有一名中年男人威胁她就范，结果染上了艾滋。丈夫本来想和韩嫣离婚，但想到她的牺牲没忍心，却在后来各种软暴力折磨她，最终韩嫣自杀。家人悲痛过度，又向夏海林举起了带血的屠刀。  


……


看了这些故事，才能明白一句话：


生活远比戏剧来的狗血。


无数爱恨交加，悲欢离合的故事，来源只有一个——生活。
