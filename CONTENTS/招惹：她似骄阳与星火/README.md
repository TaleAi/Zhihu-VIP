## 简介
骄阳似火，她似日月星辰。


《时光的幻想》：


室友的婚礼上，因为婚闹，


我和我打伤的伴郎去了前男友工作的医院。


多年不见他的爱意好像没减少...


《和影帝隐婚后，我掉马了》：


和影帝隐婚后，为了低调，我对外只称已婚，


但同学嘲我老公一定很丑，见不得人，所以才拿不出手，


后来，她们扒出了我的影帝老公，还有我的无数马甲，


看着我名下的公司、房产和身家，他们沉默了......
