## 简介
一段好的感情，是一起进步，共同成长～


*理性学霸和痞坏校草的甜酸爱情。*


程栀和张越，是一对没有血缘关系的继兄继妹。


程栀清醒理智，对人生规划清晰。


张越随性张扬，对未来毫无计划。


他们像一对矛盾体，偏偏走到了一起。


同住一个屋檐下，程栀眼睁睁看着张越掉落小霸王的面具，露出傻瓜美人的本体。


在冷静自持的程栀面前，小霸王成了爱哭又没有安全感的黏人精。


如果生活是部言情小说，他们应该拿反了剧本。


性格的差异，让这段感情甜酸参半。


两人都在努力为对方改变，最终却还是以分手收场。


多年以后，留学归来的程栀，与褪去青涩，成熟稳重的张越重逢。


这一次，他们都变成了更好的自己。


程栀教会张越成长，张越教会程栀相信爱。


生活肯定还存在着各种问题，只要有爱，他们就有勇气再次牵手。


愿每一段真挚感情，都有一个圆满的结局～
