## 简介
10 个发生在极限狭小舞台上的悬疑故事，极少人数之间的惊悚博弈，不计正邪，只论生死。


本专栏里的故事均发生在封闭空间，张力拉满，杀死尿点。


深夜，代驾司机接到了一个订单。上车之后，种种迹象表明这辆车是凶案现场，而坐在后座的男人，就是凶手。行驶在寂静无人的公路上，距离凶手毁尸灭迹的目的地越来越近。代驾司机必须抓紧时间，利用身边一切可以找到的东西，逃出生天……


某护士的日常工作，是照顾几个重症患者的饮食起居，其中有一个植物人，需要靠葡萄糖和稀食来维持生命。这天，整个楼层只有护士自己值夜班，在她去库房取药品的时候，猛然意识到自己犯了大错：在整整一周的时间，她给那位植物人注射的都是毫无养分的生理盐水。但是，他为什么还活着呢？


几个好友约定去玩大型恐怖主题密室，可是当 NPC 引导着众人，剧情走到一半时，一名独自执行游戏任务的玩家当场丧命。所有人陷入极度惊恐，密室大门被反锁，这里似乎真的「闹鬼」了。时间紧迫，众人决定利用密室的建筑结构，逃往临近的大楼，但计划再次失败……
