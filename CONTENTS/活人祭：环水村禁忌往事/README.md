## 简介
我们的村子叫环水村，顾名思义，村子三面环水，后面靠山，处在一片「三不管」的地带，再加之交通不便，我们环水村数十年来几乎与外界隔绝。


村里新建了一座桥，但是最后一根桥墩却怎么也立不起来。


村长请来一位风水先生，他一进村，便说我们村子是白虎乱抬头。


要是想立起这个桥墩，必须得牺牲一个人……
