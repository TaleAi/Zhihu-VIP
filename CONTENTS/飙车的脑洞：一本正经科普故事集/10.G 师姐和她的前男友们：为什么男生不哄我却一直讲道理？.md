## 10.G 师姐和她的前男友们：为什么男生不哄我却一直讲道理？
同实验室的小 C 有个很要好的 G 师姐，腹有诗书、颜如舜华，但爱情总是不能修成正果。


![img](https://pic1.zhimg.com/v2-647973a8027d913dde2223f9c43840c0.webp)

有次 G 师姐又失恋，拉我和小 C 出去游山玩水。吃东西的时候，她忽然问我们：「你俩天天一个实验室，没谈恋爱？」


小 C 嫌弃地看我：「他？」


我瞪了小 C 一眼：「我怎么了？」


小 C 问我：「现在实验室离心机有多少？」


考我？我自信满满：「5 台艾本德 5424R，2 台赛多利斯 G16-C；另外还有一台日立 CR-G3。老板还订购了 1 台『贝克曼』马上……」


小 C 突然指了指我身后说：「劳驾帮我拿点饮料呗？」


我埋怨了一句：「你又不是半身不遂……」然后很不情愿地起身，拿了一整瓶 2.5L 的可乐放她面前。


小 C 叹了口气，对一名服务生招手：「小哥哥，麻烦帮我倒杯饮料。」


服务生和颜悦色地跑过来拧开可乐倒了 2/3 杯，还加上冰块和柠檬片，再插上一根吸管递给小 C。


小 C 拿起杯子指指我，对 G 师姐说：「喏，跟这种人形 AI？怎么可能……」


什么 AI？


我一脸蒙圈地眨巴眼睛，而 G 师姐已经笑得花枝乱颤。然后她说：「我初恋男友跟你倒是很像。」


「也这么聪明？那怎么还舍得分手？」我脱口而出。


小 C 同学抬头瞪了我一眼。


G 师姐叹了口气：「那时候只要一吵架，他总是挑我逻辑上的漏洞，自己却死不认错。我也是年轻不懂退让。最后大吵一场就分了，彼此再无消息。」


「怪不得你后来找的都是商界精英。」小 C 说。


G 师姐叹了口气：「商界精英们嘛……好是好，但也不知道感觉上缺了什么，所以老是分。」


我忍不住说了一句：「缺真实感。」


G 师姐瞪大眼睛好奇地看着我：「这话怎么讲？」


我抬头说：「我只是根据我的观察进行推测，对不对你自己感觉？」


G 师姐点点头：「嗯，你说。」


我：「商界精英们跟你谈恋爱，看似面面俱到、无微不至，但这种感觉就像是在高级西餐厅里吃烤串，没内味。因为他们在你面前这种表现是刻意地，并不是完全发自内心。」


小 C 表示不屑：「拉倒吧！天底下善解人意的男人都是虚情假意？」


我回答她：「不是虚情假意，但肯定有刻意成分。由于胎儿期受性激素影响，男性大脑和女性大脑发育侧重点不同，导致大脑结构不同，思维模式就不同，所以两性沟通过程很容易引起误会。这种情况下却能相处融洽，一定要有妥协和修饰的成分。商界精英都是谈判沟通方面的老手，谈恋爱也很容易带入这方面的职业习惯。但这种取悦女性的表现却不是最自然、最真实的表现。师姐慢慢感觉到了这种不真实、觉得不安心，所以就会选择分手。」


小 C 还想反驳：「谁说男女大脑结构不同？我记得以前看过一篇文章，神经科学家莉丝·艾里奥特（Lise Eliot）说过，男女的大脑结构并没有什么显著差异，都是后天成长环境惹的祸。」


我摇头：「我也看过她的文章，有些道理但结论有些片面。如果男女大脑真的没有差异，那为啥儿童自闭症患者多数为男孩子？青春期发生严重抑郁症的则女性居多？而且，女性进入老年期更易发生阿尔兹海默症，而老年男性却更易得帕金森病，这又如何解释？」


G 师姐一看我俩要争起来了，赶紧举起酒杯和我俩分别碰了碰：「这里的红酒不错，外面喝不到的，尝尝。」


我们仨喝完酒，G 师姐又问：「你刚才说我那些男朋友追我的时候，都是职业习惯而非真实表现？」


我点头：「因为正常来讲，男性的大脑跟女性比起来，灰质发达而白质不发达。」


小 C：「这又怎么了？」


我：「灰质的工作方式是『区块链』式的，所以男人专注于某件事的时候，大脑皮层局部灰质区域就会活跃，并带动周边区域一起兴奋。这样，男人对一个问题能思考得很深。但是，灰质里的神经信号传导路线非常曲折，所以信号传播距离很短。因此，与激活区域隔得远的大脑皮层，就很难被调动起来。」


![img](https://pic4.zhimg.com/v2-7258e778b470d67a7e331b41242603cd.webp)

G 师姐：「这会产生什么结果？」


我：「就是你说你初恋男友那种情况，思考问题逻辑有余、情感不足。」


小 C：「就因为男人是用灰质思考问题的？」


我：「其实男人也不是完全用灰质思考问题，但基本会优先用大脑顶叶的灰质。」


G 师姐：「顶叶？」


我点点头：「男性大脑顶叶和枕叶比较发达，所以靠枕叶处理视觉信号后，分析信号的思考功能则由顶叶的灰质来主导。顶叶负责逻辑分析，因此男性思考问题的逻辑性一般都不错。但这块区域并不太参与情感信息处理。顶叶的神经电信号又很难传递给真正适于情感信息处理的额叶，所以男性思考问题时容易忽略情感因素。」


G 师姐说了一句：「顶叶兴奋很难带动额叶，是因为男性的大脑白质不发达吗？」


我点头：「没错！」


小 C：「那商界男精英们在恋爱中的优秀表现，又怎么解释？」


我：「多年在商场中摸爬滚打训练出来的经验嘛！」


小 C：「什么意思？他们的白质通过商场打拼变多了？」


我指着盘子里的剁椒松花蛋问她：「鸭蛋做成皮蛋的过程，里面的蛋白会变多吗？」


小 C：「怎么可能？说人话！」


我：「好的。商界男精英们不是大脑白质变多了，而是他们通过训练，利用顶叶灰质中的逻辑思维回路，部分弥补了情感表达的不足。」


G 师姐夹起一块松花蛋递给我：「为什么是部分弥补？」


我一看有人赏识，我忍不住兴奋继续说：「男性商人们和客户打交道的时候，全靠临场发挥？我觉得不可能。优秀的男性商人总是习惯提前做情况分析——先对客户的需求和基本情况获得一定的认知。根据这些分析，他们心里会形成好几种可能适合的沟通洽谈方案。再根据对方支付能力、痛点、性格，根据现场看到的情况、结合以往成交案例的经验套路进行演绎，不断切换洽谈方案『看人下菜碟』，就能很大几率达成交易。」


G 师姐点点头：「没错我也发现了，他们和客户打交道的时候，确实每个人都有固定的几种套路。」


我得意地笑了笑：「这就是了。所以，他们这种恰到好处的表现，其实全靠经验。本质上，男性商界精英们在沟通过程中，还是顶叶那块儿的逻辑分析居多，而情感建立往往就比较生硬，不喝酒情绪表达就上不来。」


G 师姐：「但为什么一定要喝酒？」


小 C 插话说：「酒精能促进神经兴奋嘛，估计能让额叶更容易激活。喝咖啡也能有同样作用。」


![img](https://pic2.zhimg.com/v2-c101f48fae3502d54fb589927a331983.webp)

我冲着小 C 点点头：「就是这个意思！而女性商业精英的工作方式就不太一样了。她们的客户分析能力也许不是很强，但颞叶和顶叶功能强大，临场走情感路线拉家常，就能很好地打动客户，成功率相当高！」


G 师姐拿起酒杯和我碰了一下：「你这个见解很不错，我以后做对外联络的时候可以尝试下。」


小 C 问了一句：「照你的意思，这些商界精英把商业谈判的套路也用到谈恋爱里？」


我拿起一根孜然羊排：「没错，犯罪心理学上有个理论，惯犯总喜欢用得手过的方式反复作案……」我看了看她俩，觉得这个比喻用在 G 师姐前男友身上似乎不妥，于是补充说：「正所谓『一招鲜，吃遍天』。孜然能烤羊肉串，当然也能拿来烤羊排。既然难搞的客户能拿下，和师姐这种大美女谈恋爱的时候，他们自然也会用到这些套路。」


G 师姐笑了：「所以我是羊排？」


小 C 一听也乐了，看我怎么圆场。


我赶紧说：「不是不是！我就形容一下而已……」


G 师姐看我一脸尴尬，赶紧替我找台阶下：「那既然这个套路在商场上很吃得开，为何到我这里就不适用了呢？」


我赶紧说：「因为商业谈判是短时沟通，谈恋爱是长期沟通，套路反复在同一个人身上用多了，就容易被人看穿。」


G 师姐说：「可是你今天说着话之前，我并没有觉得他们在套路我呀！」


我：「其实你已经看穿了，只是这种看穿是在潜意识里的，所以你不知道。之前我说了，我还是部分认同莉丝·艾里奥特的理论，成长环境对脑发育会起到很大影响。师姐你是理科出身，后天学习环境促进了逻辑思维能力不断提升。因此，商界精英那些套路人的沟通模式，一开始也许能提起你的兴趣，久了就很容易被你的逻辑分析能力反向识破。加上女性额叶本来就发达，面部情绪识别能力很强，所以很容易辨识对方的言语行为是否发自内心。这些商界精英的重复套路慢慢在师姐面前丧失了真实感，就会让你潜意识觉得不可靠、没归属了。」


小 C 撇了撇嘴：「好男人真难找！」


服务生此时把小 C 点的酸汤面端过来了，我赶忙接住放她面前：「也不能这么说。酸汤面也许很对你胃口，但却不一定是师姐的最爱。所以套路并不是什么坏事，关键是得用到正确的受用对象那里。正所谓『王八看绿豆』，对上眼儿就成。」


小 C 又逗我：「你说师姐是王八？还是绿豆？」说完她和 G 师姐俩人嘻嘻哈哈笑成一团。


我有些窘迫地说：「……我是说，师姐要找到适合她的人，商界精英也许不是最佳人选。」


G 师姐安静下来，用手撑着下巴看着我：「那你觉得，什么样的人合适我？」


说实话，她安静下来妩媚的样子太动人了！要不是敬重她是师姐，我就想说那个人是我。


但我终于低头冷静了下来，然后说：「还是要从咱们这个圈子里找比较合适。虽然理工男们不懂人情世故、太真太直了，但总还是有一些情商略微高的。至少，有些师兄的研究课题中会涉及到神经科学，因此是具有情感因素这一方面认知的。这样的男人，就有机会和你在真诚的基础上建立起相互理解。而师姐你比起以前，心理也越来越成熟了。这样的话，两人就能顺顺当当一起过日子了。」


G 师姐若有所思地看着我：「嗯，我心里有人选了。」


我很欣慰地点点头：「成了的话，哪天带来一起聚呗？」


G 师姐满口答应，然后举杯和我碰了碰。


然而再后来，G 师姐并没再交任何一个男朋友，而是专心一意忙工作去了。即使偶尔有空，她也只是找我一起去吃东西而已。 


她到底在等谁？对我来说，始终是个谜。
