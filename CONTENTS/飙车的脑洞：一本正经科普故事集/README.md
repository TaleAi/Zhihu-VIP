## 简介
这是一趟从实验室发往脑洞世界的科普故事列车。


毕业之后，我，一个生物狗，进入了一家体制内的生物研发中心。


工作枯燥且单调，守着凯氏定氮仪、96 孔板、离心机，以及各种反应器，往往数小时就过去了。这个时候我的思绪就像脱缰的野马般不受控制，实验结果还没出来，我脑海里的小剧场已经上演了无数小故事。


为什么男人喜欢盯着女人的胸部看？  

 ——大师姐的秘密里，藏着人类繁衍进化与审美形成的故事；


为什么婴儿那么爱哭，人类的幼崽还可以存活下来？  

 ——宝宝「看人下菜碟」的哭法，有婴儿操控成人的秘密；


学渣和学霸的区别在哪里？  

 ——注意力的秘密，藏在不同的大脑分区里。


同事们在繁忙的实验之余，偶尔会来找我聊天。一开始我们聊的都是生长曲线、库存种子质量和国外最新文献报道，自从我和他们讲了我开脑洞想出来的小故事以后，来找我的人越来越多。


脑洞越来越大，我开始在知乎上用故事的方式，回答科普向的问题。没想到，当我开始飙「科普段子」时，却收到了很多好评。


原来，对于科普这项看似严肃的事情，也可以有这么「不正经」的表达方式。


大家安静啦，实验室里要发车了哦，请系好安全带！
