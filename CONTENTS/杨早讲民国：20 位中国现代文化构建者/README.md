## 简介
中国社会科学院文学研究所杨早老师，将讲述 20 位民国先生。通过其经典作品和事迹，展现他们的人生轨迹、思想精神、学术追求；揭秘他们如何影响着中国文化的发展和我们今天的精神生活。


鲁迅、胡适、陈寅恪、梁启超、蔡元培、辜鸿铭……这些名字似乎都再熟悉不过。「不在沉默中爆发，就在沉默中灭亡」、「少年强则国强」、「思想自由，兼容并包」，先生的言语和思想划破历史长空，至今仍然熠熠生辉。


但仔细想来，他们又是「熟悉的陌生人」。你是否了解他们跌宕起伏的人生，他们的成长环境和经历如何塑造其人格？你是否知晓他们对中国近现代文化所作出的贡献，他们如何开创和构建了国人的精神世界？


中国社会科学院文学研究所的杨早老师，将从文学之「文人」、政治之「士人」、传媒之「报人」、治学之「学人」、教育之「舌人」五个维度，介绍 20 位具有代表性的民国先生，带我们回顾其生涯、领略其风骨、探析其影响。他们值得我们每个人的虔心学习与致敬。
