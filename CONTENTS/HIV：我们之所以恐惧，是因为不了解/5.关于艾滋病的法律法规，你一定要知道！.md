## 5.关于艾滋病的法律法规，你一定要知道！
2018 年，某男子在社交平台上炫耀称：「我已成功将艾滋传染给一名大二女生,过程中成功把她的下体弄破然后摘掉避孕套,这次不中就天理难容了。」一时间，相关话题引起热议。若该男子所说属实，其实已经触犯了法律。


被检测出感染艾滋病病毒，对一个人来说犹如晴天霹雳，如果因此报复社会,这无疑是一种缺乏道德底线、缺少法律意识的犯罪行为。


相对于恶意传播者，被传播的感染者是无助受伤的。安全意识薄弱的小 C 就被「故意」传播了，因为自己一时的放纵，因为性伴的一句不想戴套，小 C 就感染了艾滋病病毒。


小 C 是个在校研究生，前段时间和网恋一周的网友见面了。网友的主动让小 C 不知所措，当网友提出想不戴套的时候，小 C 没有拒绝，他被当时的欢愉冲昏了头脑。


分开后，网友对小 C 的态度开始变得冷漠，直至完全联系不到这个人。没过多久，还处于失恋情绪中的小 C 开始持续性腹泻发烧，吃药不见好就去了医院就诊，检查结果让小 C 很绝望，他感染了艾滋病病毒。


小 C 把自己的经历分享在了社交平台上，在为很多安全意识薄弱的朋友敲响了警钟的同时，很多网友也纷纷表示小 C 可能遭遇了恶意传播，应当报警追究那个人的刑事责任。


的确，如果和小 C 发生性关系的那个人知道自己感染了艾滋病病毒或者已被确诊为感染者，那么他已经违反了艾滋病相关的法律法规。 



> 《艾滋病防治条例》第六十二条 **艾滋病病毒感染者或者艾滋病病人故意传播艾滋病的，依法承担民事赔偿责任；构成犯罪的，依法追究刑事责任。**



> 《中华人民共和国刑法》第三百六十条 **【传播性病罪】明知自己患有梅毒、淋病等严重性病卖淫、嫖娼的，处五年以下有期徒刑、拘役或者管制，并处罚金。**



> 《中华人民共和国刑法》第二百三十四条 **【故意伤害罪】故意伤害他人身体的，处三年以下有期徒刑、拘役或者管制。犯前款罪，致人重伤的，处三年以上十年以下有期徒刑**；致人死亡或者以特别残忍手段致人重伤造成严重残疾的，处十年以上有期徒刑、无期徒刑或者死刑。本法另有规定的，依照规定。



> 《最高人民法院、最高人民检察院关于办理组织、强迫、引诱、容留、介绍卖淫刑事案件适用法律若干问题的解释》第十二条 **明知自己患有艾滋病或者感染艾滋病病毒而卖淫、嫖娼的，依照刑法第三百六十条的规定，以传播性病罪定罪，从重处罚。具有下列情形之一，致使他人感染艾滋病病毒的，认定为刑法第九十五条第三项「其他对于人身健康有重大伤害」所指的「重伤」，依照刑法第二百三十四条第二款的规定，以故意伤害罪定罪处罚：(一)** **明知自己感染艾滋病病毒而卖淫、嫖娼的；(二)** **明知自己感染艾滋病病毒，故意不采取防范措施而与他人发生性关系的。**


**「**故意传播艾滋病」是指明知自己感染了艾滋病，故意不采取防范措施发生某些高危行为，那么**「明知」**该如何判定呢？


根据最高人民法院和最高人民检察院司法解释：


**（1）有证据证明曾到医院或者其他医疗机构就医或者检查，被诊断为患有疾病的；**


**（2）根据本人的知识和经验，能够知道自己患有疾病的；**


**（3）通过其他方法能够证明行为人是「明知」的。具有上述情形任何一种，都应认定为「明知」。**


根据以上的法律法规，相信大家应该很清楚故意传播艾滋病这种行为的严重性。当然，感染艾滋病病毒后，不仅要遵守以上的法规，还要履行以下义务，尽到自身的责任。



> 根据《艾滋病防治条例》第三十八条，艾滋病病毒感染者和艾滋病病人应当履行下列义务：**(一)** **接受疾病预防控制机构或者出入境检验检疫机构的流行病学调查和指导；(二)** **将感染或者发病的事实及时告知与其有性关系者；(三)** **就医时，将感染或者发病的事实如实告知接诊医生；（四)** **采取必要的防护措施，防止感染他人。艾滋病病毒感染者和艾滋病病人不得以任何方式故意传播艾滋病。**


在我们后台的私信中，也有不少网友对**「艾滋病病毒感染者还能结婚生孩子吗？」「能够办理健康证吗？」**这类问题很关心。


首先**对于结婚，答案是肯定的。**


我国婚姻法规定，**只有「患有医学上不适合结婚的病种」的人才不能结婚**，艾滋病并不是我国法律禁止结婚的病种，而且艾滋病病毒感染者也享有婚姻、就业等权利。


因此，**在结婚之前就感染了艾滋病病毒的一方一定要让对方知道自己是艾滋病病毒感染者或艾滋病患者**，只要双方愿意，并保证采取防止对方被传染的相应措施，则可以结婚。


那么，**如果婚后感染了艾滋病病毒，需要告诉配偶吗？**


尽管《艾滋病防治条例》中明确规定要保护艾滋病人的隐私以及合法权益，也**建议感染者主动告知自己的配偶。毕竟，如果隐瞒感染艾滋病病毒的情况，必然会对他人特别是性伴侣的生命健康带来潜在的风险和威胁。**



> 云南省于 2021 年 3 月施行的《云南省艾滋病防治条例》第二十条 **艾滋病检测机构应当指定专门人员将确证的艾滋病检测阳性结果及其治疗服务信息告知本人，对限制民事行为能力或者无民事行为能力人应当告知其监护人。**



> 《云南省艾滋病防治条例》第五十七条 **感染者和病人有下列情形之一的，依法承担民事责任。构成犯罪的，依法追究刑事责任：（一）不及时将感染艾滋病病毒的事实告知其配偶、有性关系者等存在暴露风险的人群；（二）明知自己感染艾滋病病毒仍进行卖淫嫖娼，或者不采取防范措施与他人发生性关系的，或者不接受医疗机构采取的母婴阻断措施的，或者与他人共用针具吸毒的；（三）明知他人感染艾滋病病毒而隐瞒情况，介绍其卖淫嫖娼或者与他人发生性行为的；（四）以其他途径恶意传播艾滋病病毒的。**


关于艾滋病病毒感染者是否能够生育子女的问题，答案也是肯定的。


**无论是夫妻双方均为艾滋病病毒阳性或者有一方为阳性，想生育健康的孩子，大前提是要进行药物干预。**在医生的指导下，服用抗病毒治疗药物，将病毒载量控制在检测下限，最大程度避免将病毒传染给配偶或者孩子。具体情况可参考《中国艾滋病诊疗指南（2021 年版）》中「HIV 母婴垂直传播阻断及单阳家庭生育」部分。


**但如果不及时接受抗病毒治疗，可能会造成悲剧的发生。**曾有一名女性艾滋病病毒感染者在首次被检测出阳性后，并没有把自己的艾滋病病毒检测结果告诉性伴侣，并且结婚怀孕了。在住院时用化名登记后，这位女性感染者又再次被检测出艾滋病病毒阳性，结果她本人和胎儿都没有存活，唯一庆幸的是她的丈夫没有被感染。


虽然艾滋病属于传染病，但要知道艾滋病病毒并不会通过日常接触传播，所以**「艾滋病病毒感染者到底能不能办理健康证」**成了大家比较关心的问题。


像**从事接触直接入口食品工作的食品生产经营人员、从事饮用水生产的经营人员、化妆品生产人员、消毒产品的生产分装或质量检验的人员等**，这些人员是需要出具的无指定传染性疾病的医学证明，办理健康证的。


![img](https://pica.zhimg.com/v2-047fc5552cbd0078017a7ebd057c4ee2.webp)

健康证的检查项目一般包括：**常规内外科体格检查、肝功能一项(ALT)、大便培养(痢疾、伤寒杆菌)、胸部透视等，并没有要求查艾滋病病毒。**实际工作中，**有部分地区是检测艾滋病病毒，但不影响健康证的发放。**


所以说，艾滋病病毒感染者或艾滋病病人也是可以合法办理健康证的。


同时，法律规定艾滋病病毒感染者和艾滋病病人可依法享有以下权利：


**(一)** **任何单位和个人不得歧视艾滋病病毒感染者、艾滋病病人及其家属；**


**(二)** **艾滋病病毒感染者、艾滋病病人及其家属享有的婚姻、就业、就医、入学等合法权益受法律保护；**


**(三)** **未经本人或者其监护人同意，任何单位和个人不得公开艾滋病病毒感染者、艾滋病病人及其家属的有关信息；**


**(四)** **医疗机构不得因就诊的病人是艾滋病病毒感染者或者艾滋病病人，推诿或者拒对其其他疾病进行治疗；**


**(五)** **国家实行艾滋病自愿咨询和检测制度。县级以上地方人民政府卫生主管部门指定的医疗卫生机构，应当按照国家有关规定，为自愿接受艾滋病咨询、检测的人员免费提供咨询和初筛检测。**


总而言之，大家一定要提高安全意识，保护好自己。**如果发生了高危性行为，请不要忘记还有暴露后预防（PEP）能够帮你降低感染艾滋病病毒的风险。**如果很不幸**恶意传播的事件**发生在自己身上，一定要拿起法律武器，**通过以上的法规维护自己的生命健康权。**
