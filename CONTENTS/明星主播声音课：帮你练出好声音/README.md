## 简介
本课程由电视台明星主播王品月和于浩主讲。课程将划分为认识声音、常用表达技巧、发声练习 3 大类别，理论与实战相结合，剖析受欢迎的好声音具有的特点，帮你练就好声音。帮你无论是私下交流，还是公开演讲，都能挥洒自如，发出悦耳动听，让人好感倍增的声音。


无论是声音不好听，普通话不标准，深受方言困扰，还是公开场合说话紧张，喉咙发紧，声音颤抖，其实这些问题，都不是天生的，通过后天训练，都可以改变。


本课程共 21 讲，由全国小学语文教材课文配音员王品月老师和北京人民广播电台新闻主播于浩老师主讲。


课程中将围绕认识声音、常用表达技巧、以及发声练习 3 大类别，揭晓你不知道的声音秘密，了解好听的声音具备的条件，把握说话的情绪，帮你将自己的声音训练的更有魅力。


给你一套拿来即用的科学发音技巧，无论是方言、平翘舌、前后鼻音还是说话声音难听，生活中 80% 的发音问题都可以帮你解决。通过练习，使你收获完美的发音方式，在交流或演讲时快速抓住他人印象。


同时，老师们还会带你还原 14 种交流场景，实战演练出最有魅力的声音。
