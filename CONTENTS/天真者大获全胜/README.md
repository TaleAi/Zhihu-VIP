## 简介

对于小孩子来说，世界上只有两种事，一种是正确的事，一种是不该做的事。

如果世事如此，他们还要沿着自己真正渴望的方向走下去吗？


当然了，提一口少年意气，他们永远天真，永远大获全胜。
