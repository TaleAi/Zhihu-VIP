## 简介
自控很重要，为什么你做不到？自控失败不是因为你不够努力，而是因为方法错了。我们往往以为，自控靠的是意志力。实际上，自控力能力强大的人根本不是这样。真正有效的自控，是依靠习惯的力量，自动实现自律。


你是不是尝试过提高自己的自控能力？


用了各种自我管理 App，没有用；


强迫自己不看手机不刷抖音，做不到；


工作学习时间表从早上排到深夜，依旧拖延。


自控很重要，为什么你就是做不到？


这门课将会让你彻底告别逼迫自己的错误自控，系统+方法+实践三管齐下。


通过学习，帮你根据自己的目标建立一套自控习惯；


告别不由自主地刷手机、拖延和自控失败的懊悔；


你将收获一种全新的生活方式。
