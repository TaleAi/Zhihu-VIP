# 又逢春：为了攻略暴君我重生了三次

### 作者：阿幸

### 分类：古代言情

### 状态：已完结，共 17 节

### 存档日期：2022-06-02

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1463882172721106945


## 简介
宁遥穿越到一个陌生的世界里，匹配到一个奇奇怪怪的系统。
想要活下去，只能按照系统的指示，攻略病娇暴君，唤醒他心底的善意。



为了攻略他，她死了两次。


第一次，他头也不回地离开了，直至她死也没有回头再看上她一眼。


第二次，他跪倒在地上，眼底猩红，身子直颤。


他抱着她，像抱着一个再也醒不过来的美梦。


......


她见过他跌在泥塘里，举步维艰艰难求生；也见过他意气风发高高在上，挥手间便定人生死。


她还见过他哭红了眼睛，求她再也不要离开他。


 


殷绥是个货真价实的暴君，暴戾恣睢、冷心冷情，视人命为草芥。


他半生伶仃，身世坎坷，备受欺凌。


他不懂什么是情什么是爱，甚至不相信这个世界上有纯粹的感情。


直到宁遥出现。


她教会他信任，教会他温柔，赋予他哀痛也赐予他喜乐。
