## 简介
本门课既有政府机关公文写作中的困境和谜团，又有企事业单位公文写作中的陷阱和技巧。「笔杆子」结合大量具体案例，生动的教你各种公文写作方法，使你轻松驾驭不同场合，同时学会如何看书，如何收集材料，参透公文写作的门道。


一些工作中不可或缺的日常活动，比如：工作总结、会议纪要、公关稿、发邮件等，是否让你感到头疼，不知如何下笔？


上级有重要工作安排时，你总是从网上查找模板，却难以判断好坏？


在职场里摸打滚爬这么多年，你是否摸清公文写作的门道？


如何才能写出好公文，使你成为领导和同事心中的「笔杆子」？


![](https://pic1.zhimg.com/v2-b5f0e24c914562be55dc655f39352108.jpg)


本门课将从 「基础篇」开始带你揭开公文写作的面纱，明晰什么才是一篇好公文，公文的素材需要如何准备，公文的格式应该注意哪些以及公文的写作逻辑等内容。


在「实战篇」中将教会你如何驾驭不同场合的公文写作，如何写好个人工作总结、单位工作总结、讲话稿、邮件、会议纪要、调研报告等。


在「技巧篇」中将教你给公文画龙点睛，如何巧妙使用新词汇、金句，标点的正确使用方法，主题句的提炼法，错别字如何避免等。


![](https://pic2.zhimg.com/v2-58568356df543db2e3594b5a931956e1.jpg)


26 讲内容全部由小马哥和大姚撰稿，老娄转述，既有政府机关公文写作中的困境和谜团，又有企事业单位公文写作中的陷阱和技巧。通过本门课系统学习，你将有意想不到的收获。


![](https://pic4.zhimg.com/v2-5ff90b0b9797acf35753f9beefbe71b5.png)
