## 简介
分手快乐，祝我快乐。


**不知重逢是何年**


半夜接到诈骗电话，说我涉及重要案件，让我去警局。


我平静地问：「是我杀人的事吗？」


那头沉默半晌：「你杀谁了？」


「我老公，把他放冰柜里了。」


……


洗澡时我和闺蜜吐槽骗子不专业，还和我唠嗑。


门被踹开。


随后我被一群壮汉摁倒在地！


 


**朝暮不相逢**


冷战后，男朋友第一次低头。


却是为了抢走我的名额，和小青梅双双保研。


「你不想出国，不就是为了和我谈恋爱吗？你就当帮帮我。」


为了逼我同意，他不惜用分手威胁我。


后来，我和别人在一起，他却哭了。
