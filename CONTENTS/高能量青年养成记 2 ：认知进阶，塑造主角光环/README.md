## 简介
聚焦培养更优质的自我认知，科学解决心理问题。


接纳自己，是认知进阶的第一步，为「成为更好自我」打下基础。


很多心理问题的起因，在于归因偏差，比如一味地认为自己不好，认为自己个性有问题等。究其本质，是认知方式的问题。


比如在看待性格缺陷这个问题时，将讨好、共情能力差、缺爱等特质定义为性格上的「劣势」……真的是这样吗？


在这里，我们不接受这样的定义。


正是因为我们缺乏了正确看待他们的视角，一味排斥，才渐渐让自己陷入到无意义的痛苦内耗中。


本专栏将会深入本质，从来源去理解这些特质，让用户能够更直接和清晰地了解自己的痛苦从何而来，并提供如何解决的方法。


拥有正确的认知，才能修炼出真正强大的内心。


希望阅读这些内容后，读者能够实现认知和工具上的双重进阶，成为自己人生的主角。
