## 简介
用「图文+配套视频」的形式，和你用「扯闲篇儿」的方式，聊聊有关电影的一切。


发现没有？


不管是在热闹的露天大排档，  

  还是在 CBD 街角的咖啡馆，  

  

  有人聚在一起交谈的地方，  

  电影，永远是个热门话题。


我们是那么热爱讨论电影。


对电影的讨论，往往不会局限于某一部影片本身。  

  -如何评价一个演员演技的好坏？  

  -什么样的脸才算「电影脸」？  

  -文艺片好在哪儿，为什么我看了只觉得闷？


关于电影，我们有太多奇奇怪怪的问题。


本专栏，就是一场关于电影的「漫谈」：普及一些电影知识，回答一些热门问题。  

  希望下次再看电影，会有那么一些不一样的感触。
