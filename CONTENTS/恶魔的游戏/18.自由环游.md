## 18.自由环游
84


尽管最高的「晨星」依然耸立，那里灯火通明，似乎伸手可触群星，但 A 区所谓的威严在这一天已经被按在地上狠狠摩擦。


倒塌一小时后，Eva 系统内留下了一封来自匿名者的公告信，全科技司也陷入了沸腾。


被叫起来加班，正在排查问题的科技司大牛们，发现 Eva 被黑，而且被篡改后的 Eva 无法被快速修复，既紧张，又带着棋逢对手的兴奋。


这封信后来被顶层列为特级机密，但在科技司内部却成为了隐秘流传下去的传说。


公告信的内容是这样的：


科技司的各位，你们利用技术服务全区，本该值得尊敬。


我一直认为，技术应该更公正地服务，而不是沦为少数人的统治工具。


所以现在出现在你们面前的，是全新的 Eva2.0，这个系统只有一个目标，公正。


不用惊慌，它是无害的，它所有的运转法则，都以缩小各区差异系数为目标。


但光有系统是不够的，它需要被执行。


所以温馨提示，如果它不在线上正常运转的话，A 区会爆炸哦。（不信你们戳这里检查看看哈哈哈）


这个系统一定程度上是开放的，你们可以维护与改进，只要是为了促进这个目标，Eva2.0 很欢迎你们。


欢迎你们加入，为了更公正的全区而奋斗。


当然，你们也可以尝试着来挑战我。那群老家伙肯定会逼着你们来挑战我。


不过相信我，大概率你们不会成功（鬼脸）。


发件者是，「一个普通人，跟你们一样希望全区变得更美好」。


简而言之，就是既声情并茂，又嘚瑟，非常嘚瑟的一封信。


科技司几乎炸了。不管是上头的要求，还是身为计算机强者的尊严，都让他们在接下来几年里一波接一波地去挑战这个口出狂言的无名黑客。


但全部败下阵来。


他们不得不承认，这个猖狂的小子技术实力是一骑绝尘，而且其中体现的忧区忧民的情绪，确实闪耀着人文的光芒。


科技司内部，有人崇拜他，有人又爱又恨，有人把打败小白作为自己的人生目标……


当然，这是后话了。


小白打点完这些后就跑路了，他说此时不跑待何时，等调查到他头上他早没影了。


「453，记得你们还差我好多顿饭呐！等以后我找你们！」


然后他撒脚丫子跑了，深藏功与名。


这一刻我突然被小白折服了，他真的是顶尖的高手，还低调得不行。


饭，确实没吃上，不过也没调查到小白头上。


科技司迅速将系统问题上报，在顶层形成轩然大波。


突如其来的爆炸，突然被掉包的系统，这些足以被认为，是经过精心密谋的有计划有组织的行动。


而另外一头，真正的老 J 出现了。老 J 带着原来假老 J 的部下，气势汹汹地要谈判。


顶层直接把所有的事，当成是老 J 策划的。老 J 也不反驳，于是这口锅就被他们接住了。


最重要的是，卡隆也没有透露科技司的人在军务中心的任何细节。


在顶层看来，C 区现在掌握着让半个 A 区爆炸的关键，暂时不能轻举妄动。而老 J 那边，什么也不做，就多了一项足够重要的筹码。


A 区短暂地陷入混乱。


而后，A 区顶层代表与老 J 进行了会面。


能达成会面除了能够毁灭 A 区的爆炸外，还有两个重要的因素。


一方面是老 J。


靠着老 J 留下的人格魅力，黑衣男召集了几个头部小弟，告诉了他们这么多年他们都被欺骗的秘密。他们同仇敌忾迅速反扑，扭转局面将老 J 搭救了出来，而假老 J 则被监禁。


在一番拷问下，假老 J 吐露了实情。


他只是以前老 J 身边的一个手下。在一次激战中老 J 受了重伤，可能陷入永久昏迷，而他为了不让失去老 J 的帮派陷入混乱，整容成了老 J 的模样取而代之。


老 J 没有醒来，他顶替老 J 的身份，模仿老 J 的风格推进所有的事务。


他遵循着自以为的老 J 的「遗志」，要让各区获得公正，要让 A 区有一天为之震颤，反过来被踩在脚下。


只是逐渐越走越远，变得不择手段，某种程度上迷失了自己的初心。


另一方面是顶层的某种变化。


赵冶等人几乎完全被夺权，现首席被迫退位，前首席和乔辰他们经过一系列运作，完成了顶层内部的清洗。


我见到了前首席和老 J 会面的场景。


这是我第一次看到真正的老 J。他和假老 J 面容很像，但又年轻得多。


躺在床上那二十来年，并没有给他的外貌增加太多的风霜，他看起来还是那么年轻。


虽然刚醒来不久还有些孱弱，需要长期的复健，但骨子里那种意气风发依稀可见。


满头银发的前首席，一步一步地走向老 J，面色微动，眼里一瞬间划过无数情绪。


惊讶、怅然、激动……最后只是克制住，一言不发。


老 J 也是如此。


两个人站在各自的身份立场上，一言不发地严肃颔首，眼中却仿佛隔着山海。


那一瞬间我忽然明白了，老 J，就是前首席口中那个人！


他说的那个极其有意思的人、天才，站在那里就会让人移不开眼，仿佛能让周遭的一切为之燃烧。那个看得清，却执意走其他路的人。


两个年少时关系密切的好友，因为信念的不同走上两条路，一别就是几十年。


而当他们再次碰面的时候，又仿佛变成从前鲜衣怒马的少年。


那时，一切还没有破裂，他们还可以勾肩搭背，畅想着遥远的未来。


只是这次，两人站在了不同的立场。


谈判进行了长达 8 小时。


我和祁时在外面等待，等待着谈判的结局，也等待着我们的结局。


走出来的时候，每个人脸色都阴沉沉的，看不清到底谁胜谁负。


后来听说谈判过程很艰难，一度剑拔弩张。


A 区顶层那批家伙上来就是恐吓、威胁，称 A 区尊严受到了践踏，必须让这群贱民付出代价。老 J 手下听了恨不得立刻就开打，让这群自以为了不起的人也流血，也感受痛苦。


「如你们所见，现在有了一个更公正的系统，我们有机会把全区建设得更好。」


「这场仗打不打，在于你们的选择。如果想要世界变得混乱，那就来吧，但 A 区会先倒下。」


老 J 软硬兼施，也丝毫不在怕的。


毕竟现在的 C 区也不是手无缚鸡之力，军备、人力都足够与 A 区抗衡一段时间。


A 区那群人气得吹胡子瞪眼。


打不是不行，他们本来绝对相信 A 区会赢。但现在，一个随时能让 A 区爆炸的黑客还在 C 区他们手里呐，真打起来怎么样还真不好说。


就这样，双方各怀心思，掰扯了 8 小时。既是脑力战，也是体力战。


最后还是前首席出头拍板，愿意遵循 Eva2.0，与老 J 他们一起建设 B 区与 C 区。这对顶层来说是权宜之计的决策，对 C 区也同样如此。


出来后，前首席跟我到一边谈话。


「老师，老 J 就是你说过的那个人吗？」


前首席脸上浮现神秘的微笑，语气淡然，「没错。很久以前他，产生了某些不被认可的观点。后来他被流放到 C 区，我们也分道扬镳。」


「他可能坚持要通过自我建设完成独立，让 C 区有一天脱离 A 区的掌控，让人们变成能掌控自己的命运。」


「呵呵，谁对谁错呢，到今天这一步……」


「为什么会选择和谈呢，战争，对 A 区也是一种选择。只是也许并不是利益最大化的选择。」我问道。


前首席投过来一个赞许的眼神，「你也说了。人类，永远都会选择利益最大化的那条路。」


「还有一点，出于私心，或许我也想看看，那个所谓的更公正的世界是什么样的吧。我已经老了，守着这一切也没有什么意义。」


他神秘一笑，目光有些狡黠，「但没有你，就不会有今天这一切。」


我一惊，心里战鼓齐鸣。


「年轻人啊，世界都是你们的。」他突然开始感慨起岁月来。


「去看看你哥吧，乔辰为了你可急坏了。啧啧，我就没见过这小子十三岁以后有过这么严肃的时候。」


我侧身，看到一边沉默矗立着的乔辰。


那双总是蕴含着笑意的眸子，此刻平静得像冬日冰封的湖面。一向运筹帷幄、轻松自如的气质，变得凌厉了几分，也许是多日没睡好的缘故。


我走向他，看着他眼睛下面发青的眼圈，突然就有些委屈。


「哥……」我颤颤地喊了一声。


然后就被乔辰拥入了怀里，很紧很紧，又带着失而复得的颤抖。


和那次被挟持后安全碰面相似又不同。


那时乔辰和我之间，还横亘着长期分离造成的陌生与不信任。但这个怀抱是颤抖的，异常亲密，像是真正的亲人。


我突然觉得有些难过和委屈，声音也软了几分，「对不起……」


「下次出事要先跟我说，不许招呼不打一声就跑！」


「有事我会不帮你吗，我是你的哥哥啊！」


「不许跟我脱离关系，永远不可以，听见没有！」


乔辰声音低沉，带着恼怒和威慑，有点不完全像他，却让我的鼻子有些酸。


也许不知不觉，乔辰真的把我当做了他的亲人。


但确实是亲人，这个世界唯一的亲人。


「不会了，再也不会了……」我也抱住了乔辰。


这兄妹情深的场面，都能让某些人眼里闪烁出泪花。唯独祁时板着脸，一开始还算配合地站在三米外，见时间有点久，越靠越近。


最后走到我旁边，死死盯着我跟乔辰，似乎在说：「抱够了吧？」


被看得有些尴尬，我讪讪地放开。


乔辰冷冷地给了祁时一个白眼。这个保镖，有点没眼力见，还是得找机会开了他。


85


顶层向全区发布了公告，说 C 区部分精神失常的暴乱者安排了爆炸事件，拿 A 区的人命作为威胁。不过 C 区其他有识之士提前告知预警，在军务司当机立断的疏散下，没有出现人员伤亡。


公告里还说 A 区会彰显自己的公正，今后将向 B 区、C 区投入更多资源，共同建设一个更强大的全区。


简单来说就是安抚下 A 区民众的心，告诉他们此次危机解决有 C 区的功劳。


不过主要目的还是往自己脸上贴金，把变革当做是顶层自上而下的授意，出于人文关怀角度的「施舍」。


实际上，这是在随时可能爆炸的威胁，以及老 J 那边虎视眈眈的军力下，综合制衡的结果。


小白的 Eva2.0，老 J 的威慑，这些单独都不足以构成威胁。但它们融合时，就发生了奇妙的化学反应，缺一不可。


直播事件水落石出，但并没有对外公告。


关于视频本身传达的观点，是激进还是达到判区，已经无法有一个明确的官方定论。在 A 区表现出投入 B 区 C 区建设的意愿后，我在视频里「说」的那些话，反而成了某种正确。


经过了爆炸、Eva 重置这一系列动荡后，目前达成了一种诡异的平衡。


总之，我跟祁时算是安全了。


当然，这背后也有乔辰在顶着。就算是真的判区，乔辰也会想办法瞒天过海。


再次见到好久没碰面的姐妹团时，苏昕嗷嗷叫着扑了过来，给了个熊抱。


「呜呜呜思思，我差点以为再也见不到你了。」


「你怎么这么虎呀，你都可以不叫乔思，改名叫乔虎虎了！」


听得我苦笑不得。


小姐妹们再度聚在一起唠嗑八卦，对我这段时间的经历她们啧啧称奇。


真假老 J 的部分有所保留，但我到了 C4，进了老 J 他们的地盘又逃出来阻止爆炸，这是顶层基本都知道的内情。


尤其是苏昕，她逮着我跟祁时，使劲问我们被关起来后的细节，问为什么我们不逃出来。


当听到祁时为了我被虐待时，她们眼泪都快飙出来了，泪眼汪汪地听着。


「不过呢，最后我找到了祁时，把对方头头狠狠骂了一顿。最后我们找到了机会，逃了出来。」我昂起脑袋，有些小骄傲。


「不是这样的。」


「那会你看到我，明明就哭着扑上来……」祁时表情无辜，小嘴叭叭地开始说。


吓得我火速上去捂住了他的嘴，用眼神剜他。


该死，要是把我们这样那样的事说出来，你人没了！


祁时瞪着我，嘴巴还在一张一合，听起来好像是：「明明自己做的，怎么不想承认了？」


他说话的时候，嘴唇开合碰到我的手心，痒痒的，像羽毛挠在心上。


不过我的挣扎徒劳无功，很快被坏笑的苏昕她们拉到一边。


她们两个人压着我，另外的人则眼睛发亮地让祁时赶紧说。


祁时睁着眼，似乎在回忆被假老 J 威胁时我们的惊天一吻。


他张嘴，我紧张得心脏都快跳出来了。


「不过，那是我们之间的回忆，不能分享给你们。」祁时转而这么一说，慵懒地靠进沙发里。


呼，我的心脏放了下来。


而小姐妹们则发出嘘声。


我们聊到最近这些变化，韩橙橙冷笑，一如既往的姐就是女王的架势。


韩橙橙有些狐疑地瞥我，「乔思，你是不是又偷偷摸摸做了什么？我怎么感觉每次有你出现的地方，都会搅得一团乱呢？」


我讪笑：「额，你的错觉吧？我只是一个手无缚鸡之力的普通人而已啦。」


苏昕撇嘴：「你哪里是，就算你是，祁时也不是！」


我不敢说，真的不敢说。如果韩橙橙知道始作俑者的话，怕是会跟我分道扬镳。


尽管屡次帮我，但她一直守着那道底线。以 A 区利益为先，以自身利益为先。


在不威胁自身利益的情况下，一些力所能及的帮助没问题，但要成为颠覆这套系统的敌对一方，估计她会立刻冷脸。


不过我还是十分感动地扑过去拥抱：「呜呜橙橙，谢谢你！我的好橙橙……」


她嫌弃地吐槽「别把我衣服弄皱了」，嘴角却忍不住扬起弧度。


「真是任性啊这些人。」


「A 区不会坐以待毙的，他们会继续想办法破除新 Eva 的规则，到时候 C 区会迎接疯狂的报复。」


没错，A 区不会坐以待毙。


但小白放言，A 区出现下一个能匹敌他的顶尖高手，至少需要三十年。


也许有些夸张，或许是二十年、十五年、十年……


但那已经留下了一定缓冲，火种已经开始燃烧，世界会发生改变。


一个更公正的全区，正在冉冉升起。


而我和祁时，轻装简行地踏上了去荒漠的旅程。


临走前，乔辰问我是不是真的要走。


「是的哥，在 A 区或者 C 区都太久了，我想去看看其他地方是什么样子。」


乔辰拍了拍我的头，「记住，这里是你的家，随时累了都可以回来。」


「就算不累也要回来。」他笑，一双桃花眼玩味又温柔。


我重重点头。


「不过。」他脸上笑意消失，冷冷地瞥祁时，「你真的要跟这个家伙一起走？」


祁时正在另一边往包里塞吃的，感受到有不太友善目光落在自己身上，冷冷地回瞪过来。


「嗯嗯，他在可以保护我嘛。经历了这么多，要不是祁时我早就没命了。」


乔辰眯眼，「做保镖可以，不过你千万不要想其他的。听见没？」


我讪讪点头。


emmm，要是乔辰知道了他妹已经和对方芳心暗许，还发生那么多事，会不会气得脸色铁青。


不过不管他愿不愿意，这个大舅子是已经落到他头上了= =


越野不疾不徐地行驶着，风从车窗灌进来，令人神清气爽。


像是第一次逃亡的时候。那时我们在路上飙车，疾风透过车窗灌进车内，耳边全是汽车疾驰时呼啸的风声，头发也被风吹得凌乱飞舞。


那时我感慨，如果不是在逃亡而是度假该多好。


这次，是真正的度假。


「你为什么不留在这里？一切都解决了，可以好好享受生活了。」祁时侧脸，淡淡地看了我一眼。


「因为你喜欢自由，不是么？」


我目光灼灼地回看他。


不是被遗弃的那种自由。


不是被所有人遗忘，仿佛生活在被遗弃的世界的自由。


而是真正触摸这个世界，感受它的热烈与真实、快乐与痛苦，然后决定以什么态度生活的自由。


现在，我们做出了自己的选择。


祁时勾起嘴角，「我喜欢自由？那你呢，你为什么过来？」


我知道他又想如法炮制，引诱我说出那些话。


但这次我没有别别扭扭，而是咧开了一个大胆的笑容：「因为我喜欢你呀。」


祁时一愣，似乎没想到我说得这么轻易，一时间竟没有接上话。他乖乖看路开车，嘴角不自觉地上扬。


啧啧，看把孩子高兴的。


天空碧蓝如洗。我伸出手向窗外探，感受着来自旷野的风，感受着速度与自由。


也许这并不是结局，也许多少年后会发生血雨腥风，但这一刻，我们走在路上，毫无顾忌。


从现在开始，是真正的自由环游。


和祁时一起，度过之后的每一天！
