## 简介
7 个悬疑感十足的揭秘故事，带你看看人，看看鬼，有时人比鬼更可怕，希望你别被吓到。


我是一个鬼，但，我和其余的鬼都有些不同。因为，我是被自己娘子杀死的。


孟婆安慰我说：「没事的，以前也来过一个年轻人，他可比你惨多了，他叫武大郎。」


我一个生气没忍住，不小心踹翻了她的锅。


——《昔有贾恶》


  



「我们村，是个鬼村。村里有一口枯井，像你们这些城里人估计没有见过井。」老人说得很慢，但显然已经找到了适合他叙述的节奏。


「闹鬼的事，其实已经有几十年了。算一算，到今年刚好四十年整。」


——《鬼村》


  



古灵觉得就只有母亲扎的这个稻草人最精致最像真的人。瘦瘦的稻草人站在开阔的麦田里，双手高高举起，古灵看着它，感觉那是个很绝望的姿势。


日子一天天过去，母亲总是会搂着古灵微笑着说：「不要怕，爸爸就快回家了，这次回来，他就不会再离开我们了。」


——《稻草人》
