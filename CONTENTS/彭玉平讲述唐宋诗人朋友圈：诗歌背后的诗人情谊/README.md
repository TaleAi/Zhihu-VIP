## 简介
李白和杜甫如何相识相知，成为彼此惺惺相惜的知音？白居易和元稹之间穿越千年的感情，为何还能打动我们现代人？……本专栏特邀中山大学中文系教授、百家讲坛主讲人彭玉平老师，沿着盛唐至南宋近 600 多年的文学史进程，以知人论世的方法，用诗词名篇还原唐宋诗人的朋友圈，带你梦回唐宋，感受诗人与诗情。


**穿越回唐宋第一等大诗人的朋友圈，你会发现——**


大名鼎鼎的「诗仙」李白，刚来长安时却饱受白眼，出道艰难；


还要多亏贺知章在唐玄宗面前大力举荐（吹捧），才得以成功从翰林供奉 C 位出道，自此一炮而红，「谪仙人」诗名誉满长安。


而南宋知名诗人范成大，身为「钢铁直男」，却尤其热衷于社交；


他邀请江湖词人姜夔，到自己的石湖大别墅共赏玉雪坡前暗香袭人的梅花花海，拉着小手月下闻笛。就是这样触景生情，姜夔才能创作出千古名作《暗香》。


……


以诗词名篇，真实还原唐宋诗人的朋友圈，一个个情景剧等待你的发掘。


本专栏特邀**中山大学中文系教授、百家讲坛主讲人彭玉平老师**，沿着盛唐至南宋近 600 多年的文学史进程，以知人论世的方法，带大家挖掘诗词背后诗人不为人所知的丰富心灵史和时代的风云变幻。


老师的讲解贴近日常生活，解读方式深入浅出、轻松有趣，让那些唐宋诗词名家不再是高高在上、正襟危坐的诗人，而是如你我一样有着平常喜怒哀乐、如朋友般亲切的普通人。
