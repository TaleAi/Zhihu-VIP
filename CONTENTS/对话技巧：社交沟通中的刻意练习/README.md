## 简介
匠心迭代的对话训练，深入浅出展现普通对话的专业化升级；植入经典心理学跨学科研究成果，通过对内在「自动导航」的升级换代推动「对话方式」的彻底改变。


恰到好处的沟通表达，永远都是一个职场人必备的职业素养。学会说话，可以带来意想不到的资源和机会。


把自己的心意适时、准确地表达出来，还要让别人有效地理解，愉快地接受，并不是一件容易的事。


沟通并不是只是从「话术」上解决问题，一味的使用「套路」。


本专栏将话术、情商，以及内在的心智模式成长整合在一套训练方法中，通过改写你内心的「剧本」，让你在人生舞台上的台词发生焕然一新的变化。


本专栏分为七个部分，将常见问题划分为七个沟通情境，情境里不仅有问题的解决方法、常用的话术，还有对自我和他人的心理过程分析，让你对「沟通」这件事有全新的专业化视角，发现自己说话习惯背后的秘密，找到自己困惑已久的沟通堵点，让自己获得全新的蜕变。
