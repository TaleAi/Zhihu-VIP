## 简介
人间清醒的大女孩们，追爱勇敢，挑逗无限。


她们，在生活中勇往直前，在感情中敢爱敢恨，在事业上风生水起，这就是不甘平凡的大女主。


《我欲「乘风」》


男朋友让我做她的人体模特，不穿衣服那种，美其名曰为了艺术，被我毫不留情的拒绝了。


没想到我闺蜜不仅代替我做了模特，还取代了我这个女朋友。


多年后，俩人不幸福，纷纷找上我。


自找的麻烦，姐不买单，姐只想跟成熟的男人谈一场势均力敌的恋爱，以结婚为前提。


《恋恋不忘》


深夜一点，他发来信息：我记得你说要报答我还做数吗？


我秒回：当然。


他说：你来吧，我需要你。


我懵了，半夜一点，他需要我，哪方面？


不管了，沈医生需要我，那一定得赴汤蹈火，在所不辞。


我立刻爬起来喷了点香水，就去对面找他了。
