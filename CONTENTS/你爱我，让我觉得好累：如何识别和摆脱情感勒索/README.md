## 简介
作为临床心理治疗师，伯恩斯坦以其独到的视角、通俗的语言，深入浅出地分析了反社会型、表演型、自恋型、强迫型和偏执型五种情感勒索者的特点。学完本课，你将会有足够的武器来对付消耗你情感能量的人，轻松自如地进行心理调适，摆脱不健康的人际交往，对软暴力说「不」。


最使人疲惫的不是繁重的工作，而是糟糕的人际关系。因此，你在感到心力交瘁又不知道原因时，就需要退一步思考一下是不是遭遇了情感软暴力，遇到了「勒索者」。


尽管情感勒索者并不是敲诈我们钱财的不法分子，但他们却在不断消耗我们的精力、活力、情感以及注意力来满足自身需求：


◎ 有人为了追求刺激，无视任何社会规则，恃强凌弱，藐视权威；


◎ 有人爱哗众取宠，渴望时刻得到别人的关注和认可，他们的人生就是一场演出；


◎ 有人孤芳自赏，沉浸在自恋的泡沫中不可自拔，无视他人的感受和需求；


◎ 有人过分追求完美，再小的细节都不放过，刻板而僵化；


◎ 有人总是疑心重重，怀疑爱人、朋友和家人的忠诚度，且崇尚「阴谋论」。


作为一位在临床心理领域从业多年的治疗师，伯恩斯坦以其独到的视角、通俗的语言深入浅出地分析了情感勒索者的特点，全面描述了反社会型、表演型、自恋型、强迫型和偏执型五种情感勒索者。


学完本课，你将会有足够的武器来对付消耗你情感能量的人，轻松自如地进行心理调适，摆脱不健康的人际交往，对软暴力说「不」。
