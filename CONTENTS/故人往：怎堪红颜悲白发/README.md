## 简介
当时的人，如今在何方？


他们曾经分享过同样的月亮。


但多年后再忆起，他们是否还记得当时的心情？


《帝女谋》


父皇称帝后，封我为最尊贵的镇国公主，地位仅次于太子。


但我所图，绝不止于此。


为了能入朝堂议政，我女扮男装，做了古往今来第一位女皇子。


后来，我又做了第一位女皇帝。


而代价，便是孤寡一生，身边亲近之人皆离我而去。


临终之际，我回想一生，只觉得亏欠了一人，至死都不曾放下。
