## 简介
本专栏从巧辩之道、辩证之法、王道思想，讲述生活处理方法。专栏应用诙谐的讲授方法，细致讲述 100+历史名人案例，深度解读圣贤处世之道。无论你是什么身份、什么职业，都能在其中，得到不一样的体会。


很多人觉得《资治通鉴》读不完，阅读的过程中困难重重；


很多人觉得《资治通鉴》看不会，复杂的年代，晦涩的文字；


很多人觉得《资治通鉴》看完没收获，读一遍，知道了人物和事件，仅此而已？


生活中的困惑一样没少？


比如：


为什么总是因为选择而纠结？


选工作、选伴侣、选生活的城市、选孩子的学校。


为什么忙忙碌碌，却依然感觉空虚？


工作、应酬、家庭、整日疲惫不堪。


心力憔悴，却不知归向何处。


说到底都是不经世故的问题。


本专栏带你精读《资治通鉴》，20 天共读+10 天经典人物番外，历时 1500+小时匠心打造，细致讲述 100+历史名人案例，智慧与故事结合，有趣又有料。
