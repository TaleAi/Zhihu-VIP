**- 美人豪横 -**


继姐看上了我的未婚夫，成亲那日，继母偷天换日，让继姐上了花轿，而我则被五花大绑，卖进了青楼。


危急之中，清冷王爷救下了我，还对我百般呵护。


我本以为他是我的救赎，直到他的未婚妻出现，我才知道，我不过是他手中的一颗棋子，我所面对的，还是个必死局。


可我不认命，就算是为了复仇，我也一定要活下来。


  



**- 冒牌皇后 -**


我与情敌一同进宫，她成了贵人，我却成了听床宫女，夜夜跪在龙床边，记录妃子受幸，地位卑贱，前途渺茫。


直到一个风雨夜，皇后死在了龙床上。


皇帝将我拉到床边，说：「自今日起，你就是皇后。」


从此以后，我开启了小心翼翼，生怕被皇帝掐死的皇后之路。


直到有一天，我发现，皇上，好像也是假皇上。


  



**- 重生后太子妃不装了 -**


我本是谨小慎微的大家闺秀，上辈子，好不容易搞死了一众小妖精，熬成了太子妃。


结果一夜之间，国破了，家亡了，我自己也被迫跳城楼了。


重生之后，我性情大变，既然怎么着都是个死，那还忍什么？让什么？当什么贤良淑德的太子妃？


我就要做个坏女人，在有限的时间里，无限地膈应那些讨厌我的人。


没想到的是，本该人人嫌恶的我，反而遇见了一个男子，愿意为了我，以性命守住这个风雨飘摇的国家。
