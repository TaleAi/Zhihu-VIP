## 简介
穿越成为虐文女配怎么破？没有金手指，功能为虐女主的穿书女配沈玲玲，按原小说走向，本该被虐身虐心，最后还落个凌迟处死的悲催结局。不想穿越后放飞自我的她，莫名开启了开挂人生，从炮灰女变成了书中几大男主男配的白月光。只是不知道，这霸道腹黑的男一号、笑里藏刀的男二号，是真想跟她谈恋爱，还是只想利用她搞事业，甚至要她的命？她这个带着女配装备的新时代青年，能否逆风翻盘，把虐文改写成爽文？


穿进这本玛丽苏虐文的第一天起，我就十万分清楚自己的人设和下场：


恶毒女二，男主最大舔狗，增进男女主感情的工具人，技能是作死，最后被男主亲手干掉。


既来之，则安之。手握剧本，苦修恶毒女配保命攻略的我，始终默念十字口诀：远离男主女，方可得平安。


可这剧情，似乎越来越不按套路出牌：


冰块脸的腹黑男主，不跟女主虐恋情深，总跑到我这装深沉，卖深情人设，算怎么回事？


女主的万年备胎男二，不去搭理她，反倒成天往我跟前凑，打的什么算盘？


两大帅哥的强势追求下，我还没来得及从炮灰华丽转身，又因他们莫名卷入了一场场阴谋中，被绑架、被投河、被刺杀，九死一生。


求求你们了，别爱我，我只是个莫得感情的苟命女配。


然而，在这本剧情崩坏，完全不按原文节奏发展的虐文里，我这个只有青铜装备的恶毒女配，真能平平安安苟到大结局吗？
