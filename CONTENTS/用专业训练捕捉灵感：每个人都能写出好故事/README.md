## 简介
本专栏像是带你参观「演员的摄影棚」，不仅能清晰直观地看到深度写作通用的技巧，而且还将这些技巧抽丝剥茧，从写作者自身的写作历程、知名作家的创作剖析，具体到方法层面上的技巧，三个方面从内而外进行讲解。学会用作家的视角去观察，用作家的思维去思考，用作家的习惯去生活，以及像作家一样去写作，当然，还要不断激发自己创作的欲望。


写作并不是一件难以企及的事情， **足够的生活感悟和阅读体验 ，加上一定的写作技巧，就可能成为一名作家** 。


对于想要表达的人来说，走近写作，走近世界和「自我」，好的作品自然而然就形成了。就是在讲「如何走近」。


**什么是创意写作 ？**


创意写作兴起于二战后，为解决退伍军人就业问题。大多数老兵没有一技之长，而且文化基础较差，但是他们亲身经历过战争，比作家更有切身体会，他们的写作更真实更震撼人心，可以由此记录真实表达感触，于是创意写作课程应运而生。


**文艺创作是有基本规律的 ，美术、舞蹈、音乐如此，写作也是一样。**


创意写作的核心观念是，写作是可以练习的，作家并非天才「下笔如有神」，而是也可以通过后天培养。


这个过程，其实就是一种 **内在的文化思维训练** 。


本专栏像是带你参观「演员的摄影棚」，不仅能清晰直观地看到深度写作通用的技巧，而且还将这些技巧抽丝剥茧，结合讲者自身的写作体会、知名作家的创作历程进行讲解，帮助你学会  

**用作家的视角去观察 ，用作家的思维去思考，用作家的习惯去生活，以及像作家一样去写作，当然，还要不断激发自己创作的欲望。**


拥有了这样的思维和视角，你就可以在下一次构思一篇文章、一句文案、一页日记、一段短视频……的时候，享受成为一个创作者的乐趣，感受到表达所带来的快乐。


**你将获得**


1、专业训练和技巧，面对写作，你可以胸有成竹的表示「我会」。


2、增加观察世界的视角，丰富生活体验，让日常的训练激发出自身的天赋。


3、更加丰富的阅读体验和文学品位。


4、让写作帮助自己更好的观察、记录生活，产生更精彩的创意。


**适合谁 ？**


想提高自身品味和审美能力的普通人


喜爱阅读，想要更好的观察、记录生活的人


创作型职场人：文案、策划、自媒体创作者


专业编剧、作家以及职业写作者
