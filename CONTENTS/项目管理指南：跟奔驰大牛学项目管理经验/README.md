## 简介
本门课程将全面讲解从项目计划到项目收尾的整个操作流程，全方位为你呈现一个项目管理的全貌，课程结合实际工作场景，配合大量的实际案例，使你熟练掌握项目管理流程，实现工期、成本、质量最优化。


项目管理从业者很容易遭遇：


不了解项目前提和时间规划的制定原则；  

没有清晰的预算制定和控制流程；  

项目管理工具使用混淆。  

......


掌握项目管理的基本方法，能明显提升项目的完成质量与执行的过程体验；


借助合适的工具和正确的方法，能梳理清楚项目的繁琐细节，实现效率最大化。


  



![img](https://pic4.zhimg.com/v2-3e33c29726c579af8d6436007025dcc0.jpg)

高雁鹏有 5 年的奔驰项目管理经验、曾管理过数百人的项目团队。他将多年的项目管理经验，结合大量的项目实践案例和系统知识，打造本门课程。


本门课程将全方位为你呈现一个项目管理的全貌，使你熟练掌握项目管理流程，实现工期、成本、质量最优化。
