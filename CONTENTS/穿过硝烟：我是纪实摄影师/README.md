# 穿过硝烟：我是纪实摄影师

### 作者：韩冲

### 存档日期：2022-06-03

## 简介
在中东的每一个决定性时刻，战地摄影师隐身于相机背后的深思。


「2011 年 8 月底，我去了中东常驻，9 月中旬，前往战乱中的利比亚采访。我 24 岁的生日就是在利比亚度过的。那一天我们几个拍摄完一次示威活动，去中餐馆吃了一顿，算是庆生。」 


提起战地记者，眼前闪现的，可能是背着相机冲锋陷阵，在枪林弹雨中，一边东奔西跑，一边举起相机，记录血与火的场景。 


然而韩冲的《穿过硝烟：我是纪实摄影师》开篇，竟是如此平淡，沉静。 


去中东意味着什么？韩冲说，基本上买保险，保险公司都说不在保险范围内，但「每天晚上听着如鞭炮声一样的密集枪声，看着高射机枪的曳光弹带着光亮划过天空，久而久之便习惯了。」 


在中东战场——这个特殊的新闻现场——他必须靠近、再靠近，以一种极具侵略性的方式拍摄照片。问起在利比亚拍摄死去的卡扎菲是什么感受？他只说，无尽唏嘘。 


而在穿过这些硝烟之后，在死海拍摄孩子的时候，他才意识到「没有战斗机飞过的时刻，才是拍摄照片的最好时候。」 


在《穿过硝烟：我是纪实摄影师》里，韩冲一共挑出了 36 张照片，每张照片背后，都有一个故事：在我们想去却没敢去的地方，面对我们想见却可能没机会见到的人，纪实摄影师在每一个决定性时刻，隐身于相机背后的每一次深思。 


作者韩冲，毕业于中国传媒大学，记者。曾于 2011 年-2015 年期间，在中东地区常驻，在埃及、利比亚、伊拉克、叙利亚、以色列、巴勒斯坦等国进行采访。
