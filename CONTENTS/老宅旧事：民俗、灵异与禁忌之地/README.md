# 老宅旧事：民俗、灵异与禁忌之地

## 简介
离奇命案、灵异传说……老房子总是伴随着各种各样的诡秘传闻，若是将事件原本的真相抽丝剥茧出来，无一不见证着人心险恶、世态炎凉。


我在地产行业工作了十年，这里收录着我经手过的十二套诡异恐怖的房产，我要讲的不仅仅是房子的故事，更多的还是那些隐藏在砖瓦背后的、让人脊背生寒的人心……


**「钉子户」的家里，停着一口老缸，户主说自己全家人的性命都被这口缸收走了；**


**某大学校区，女生宿舍的 509 号房间被彻底封死，据说是因为发生过一起血案，之后又闹出了不少惊悚的灵异事**件；


**乡间别墅，富商一家人常年遭受不明事件的骚扰：突然出现在窗口的白毛老妪、无休无止的诡异声响、擅自闯入的蒙面人……长达数年的昼夜折磨，背后却是一桩耸人听闻的惨剧。**


这些老房子的故事，你可以理解为是怪谈，也可以想做是悬案。而我更想只把它们当作是悬而未决的故事来听、来讲，不必把每一件事都考证得顺理成章，也没必要让所有悬念都阐述得合情合理。


毕竟——


「人类居住在幽暗的海洋中一个名为无知的小岛上，这海洋浩淼无垠、蕴藏无穷秘密，但我们并不应该航行过远，探究太深。」——霍华德·菲利普·洛夫克拉夫特
