## 简介
本专栏汇集 30 个亲历者的生死困局。困局中让人尖叫的，从来不是事件本身，而是光怪陆离的人性。


仅仅为了逃张门票，18 岁男孩翻入动物园的虎园，开始了一场「武松打虎」式搏斗；


一个古玩爱好者，交易中被盗墓贼打晕，醒来眼前一片漆黑，原来自己被活埋在了墓穴里；


半夜时分，高速行驶的火车从一个女子身上碾过，而一分钟前，她还在该火车的卧铺车厢之中。难以置信，她是从车厢里「漏」了下来……


有人绝境逢生，也有人惨遭毒手，在绝境里，更能看到人性的贪、嗔、痴、爱、恨、恶、欲，善恶抉择仅在一念之间。 


本专栏选择这些困局经历，不只是想让你看个奇闻异事，更希望能让你感受人在绝境时的求生力和突破困局的勇气。 


本专栏皆为真实故事，为保护当事人已做化名处理。
