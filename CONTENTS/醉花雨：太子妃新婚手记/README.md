## 简介
太子和太子妃的新婚日常，小虐，微甜。


我是太子妃，但太子不爱我。他爱的，是我的长姐。


得知长姐被指婚给四皇子的时候，他大闹了一场。一纸赐婚再次到了丞相府，将我指婚给了太子，也许是因为我长得太像长姐了。


我们成婚那日，他喝得大醉，醉到忘记来揭我的盖头。我自己卸下繁重的头饰，洗去脸上过分浓艳的脂粉。


我的新婚生活开始了，看起来并不顺利……


守着一份没有感情的婚姻，我恪守着作为太子妃的本分。


朝堂上波诡云谲，后宫也不得安宁。


为了他，我犯险进入深山，落得一身伤，但不愿因此换来他的深情。


谁知，他对我说，你怎知我不心悦你。


我的新婚生活没过多久，看起来出现转机了……
