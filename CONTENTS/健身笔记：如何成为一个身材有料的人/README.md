## 简介
知乎人气健身科普作者叔贵写给健身小白的入门指南。


面对互联网上大量的健身信息，刚开始健身的你是否感到迷茫——作为健身「小白」，应该如何开始锻炼，从哪里入手？坚持运动了很久，为什么一直瘦不下来？「三分练，七分吃」是真的吗？拉伸到底能不能瘦小腿……对于这些常常令新手感到困惑的问题，本书都给出了解答。


此外，书中还提供了适合新手的身体各部位的训练动作，以及适用于家庭、健身房和办公室的拉伸、减脂、增肌等训练计划，并对日常的营养策略做了详细阐释，教你如何根据自身的热量需求安排健康的一日三餐。


当年，刚走进健身房的作者叔贵，曾踩过很多「雷」，掉过很多「坑」。他把这些全部记录下来，与大家分享。


希望刚开始健身的你，能少走一些弯路。
