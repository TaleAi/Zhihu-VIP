## 简介
成事，你需要的是一套有效的能量感系统。


做事情坚持不下去，真的是因为缺乏自律吗？


意志力是一切拖延症和半途而废的解药吗？


状态不好时，我们该做些什么来走出低沉的状态？


每年年初都信心满满地给自己列下今年要实现的大小目标，到头来为什么一个都没完成？


那些厉害的人为什么永远活力满满？我却总是大脑空转，明明没做什么还累得不行……


  

以上种种困境，美国加利福尼亚州立大学助理教授「刀熊说说」为你提供的解决方案是——**打造一套专属于自己的「能量系统」。**


专栏中，「刀熊」以心理学、组织行为学、社会学的研究为依托，介绍如何通过**闭合任务回路**、**打造小胜**、**立刻行动**、**减少能量漏出**等具体方法增加个人能量感，形成个人成事系统，赶走拖延症，增加自我效能。


希望你能够走出不想做事、无法坚持、自我苛责、状态起伏不定的低迷状态。


然后，去实现更大的梦想。
