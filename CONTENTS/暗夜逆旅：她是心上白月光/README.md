## 简介
朝暮与岁月并往，与君同行至天光。


直球配傲娇、拯救美强惨、病娇黑心莲……


故事里的她，是黑夜里照亮他的一盏灯火。


**《养一只九千岁》**


被弟弟害死之后，她重生了。


这一次，她决心不给任何人活路。


只是，前世权倾朝野的九千岁，此时为何会对她摇尾乞怜？


**《师尊变猫以后》**


全山门的人都知道，师尊是最冷情的人。


但是，某天，我发现清冷臭屁的师尊，竟然是我省吃俭用养了两年的猫。


……


原来从始至终，他们既是双向奔赴，也是互相救赎。
