## 简介
日本和食造就了日本料理精致而健康的饮食理念，它代表着日本独有的价值观、生活样式和社会传统。那么，它到底是一种什么样的味道和文化？日本文化学者、和食专家徐静波带你走进和食的前世今生，在和食中探索日本的人文历史和民间故事，感受日本的感性文化和生活美学，体会和食的精致健康。


一个世纪以来，日本料理店追求做料理需「全心全意，使用最好的食材，做出符合季节的料理，漂亮地盛放在美丽的食器里」。 


![](https://pic3.zhimg.com/v2-22b719e457732653584fb1ce3e13f041.jpg)


听说，来到日本，你会吃上三碗鳗鱼饭，再与拉面来一场邂逅； 


你会舍不得一口吃掉寿司； 


即便你从来不喝酒，也会爱上日本清酒； 


那里的每一杯茶都能品出唐风宋韵。 


![](https://pic3.zhimg.com/v2-b5ddef1b928d03413bc8849ce616701c.jpg)


是什么样的味道，成就了这个城市？ 


是什么样的味道，让人如此沉迷？ 


是什么样的文化，造就了这样的味道？ 


又是什么样的人群，造就了这样的文化？ 


有着多年旅日经验的资深和食专家徐静波将以食为先，带你走进和食的前世今生，一起领略日本的人文、历史，寻觅和食的「精味时鲜」，倾听那些熟悉又陌生的日本人平凡真实的故事。
