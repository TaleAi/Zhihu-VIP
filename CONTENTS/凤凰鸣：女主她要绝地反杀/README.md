**1.** **今天选秀教母作妖了吗**


她一路打拼，成了被无数人骂的选秀教母，一手带出中国最能塌房的男团。一次车祸，她穿越到了古代，被提示要支持正确的人角逐皇位，才有可能回到现代。


于是，她在古代大搞男团选秀，疯狂捞金。丫鬟叫翠果，助理叫宋之，楼名干脆改叫翊坤楼。信女愿荤素搭配，换男团生意大发！


她赚得盆满钵满，又网罗出情报小王子、杀人小天才等若干人才，打造了一批造反预备军。


但她很久之后才发现，暗恋她的那个 C 位，似乎有着不同寻常的故事……


**2.** **今天琵琶精报恩成功了吗**


她是游荡在人间的琵琶精，一念成神，一念成魔。


四百年前，因为他一捧心头血，她有了形体，有了魂魄。为了报恩，她失去大半法力，被仇人抽筋拔骨，却在临死前意识到，她认错了人。


那一夜，她死去，又重生成仇人的女儿。背叛之苦、拔骨之痛，她一一奉还。


只是，心心念念了四百年的你啊，究竟在哪里？


**3. 今天小宫女宠冠六宫了吗**


她是卑微低贱的宫女，却也是皇帝求而不得的心上人。十三岁时她救下了一只黑猫，从此也爱上了黑猫的主人。


黑猫的主人死了，她的心也跟着死了。她活在世上的目标，便成了替心爱的人报仇。


她从暗卫变成宫女，用暗香蛊惑皇帝。猎人设下了圈套，猎物乖乖走入。她永远也不会知道，所有的不加防备，原来是因为用情至深。


爱到了最后，究竟谁是猎物，谁是猎人？
