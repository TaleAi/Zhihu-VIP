## 简介
每个人都向往财富自由，但你真的知道如何赚钱吗？掌握财务思维，才能真正理解财富的秘密，在赚钱、理财、创富的每个环节理对思路、选对方向、投对项目。掌握财富背后的逻辑，在通往财富自由的路上升级打怪，实现财富自由。


想想看，你是不是经常面临以下这些情形：


被业绩指标压的喘不过气，在公司都不敢大声说话；


喜欢的东西舍不得买，问价讨生活的日子一眼看不到头；


工资一直没起色，老婆也越来越没有好脸色；


公司又要来新人，手里的饭碗稳不稳也难说；


上有父母养老，下有孩子教育，存不住钱就经不住事；


......


财务思维是从大量经济学知识中整理出来的一套工具箱，是能够支撑起我们整个人生的思维方式。本专栏教你如何掌握财富背后的逻辑，避开暗坑，在前进的道路上所向披靡，最终实现高效创富。
