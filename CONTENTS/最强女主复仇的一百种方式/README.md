## 简介
一直以为防火防盗防闺蜜是流行语，直到我真的遭遇闺蜜背叛，才去认真的想该怎么办？

是喝得酩酊大醉，大哭一场，就此结束？还是发到网上，人尽皆知的毁掉他们？亦或是以情治情，找下一个更好，更乖的男人？


被闺蜜抢走未婚夫，并且被他们联手挖心致死，还被夺走全部财产，顶替身份，她是最惨原配，没有之一！ 


结果她重生了，重生成为一个只会吃喝玩乐，包养小白脸的纨绔大小姐，她要怎么做？


当然是扒掉隐藏马甲，做最强女主，虐最狠的渣渣！


她复仇的正痛快着，却被全国最尊贵的男人，来了个壁咚：「还要嫁最好的男人，谈最甜的恋爱！」且看《最强女主复仇的一百种方式》。
