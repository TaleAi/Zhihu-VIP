## 简介
病宠成瘾，命悬一吻。


**- 夭言惑色 -**


传闻皇后与先皇宠妃莲夭是同一个人。


有人当朝质疑，新皇微微一笑，唇边漾起小梨涡，「是她又如何？朕心悦母妃已久。」


满朝哗然，众臣惊惧。夜里，他跪在胭脂盒上，捏耳朵向她承认错误：「是朕不好，说了实话，明日朕就跟那帮老头说是开玩笑。夭夭，你困不困，朕哄你睡觉了好不好？」


  



**- 扯落他的佛珠 -**


百鬼云集的破寺，落魄小殿下遇上高冷白玉佛。


小殿下：「臻观，有鬼。」他转身灭了百鬼。


小殿下：「臻观，我饿了。」他沉默，为她洗手做羹汤。


小殿下：「臻观，我脚酸。」他叹了口气，俯身将她背起。


她越来越得寸进尺：「臻观，我唇干，你帮我润润？」


他淡淡瞥她一眼，「小殿下，请自重。」


算了，禁欲佛子撩不动，她决定转移目标。


「臻观，我要去选驸马，你不用跟着了。」


话未说完，他将她抵在佛堂下，嗓音低哑：「听话，驸马一事，贫僧代劳了。」


「臻观师父，请自......」她的话淹没在绵长的吻里，「重。」
