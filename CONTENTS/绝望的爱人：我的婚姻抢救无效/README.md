# 绝望的爱人：我的婚姻抢救无效

## 简介
每对夫妻都有一千次想要「掐死」对方的冲动，是什么摧毁了爱情，又摧毁了婚姻？婚内出轨、家庭暴力、七年之痒、经济危机……婚姻让人们满怀期待地开始，有时候又会收获遗憾的结局。在这十个婚姻故事里，他们发现，没有对方也可以。


婚姻就像一艘永不返航的游轮，我们带着全部身家登船，是为了与伴侣同行，快乐环游世界，却没想到路太长了，这么难，这么慢。


同行伴侣会争吵，航行会迷失方向，生活中处处暗礁，双方竭力修补，船只却会沉底。


一个小家庭，如何面对经济困难、原生家庭的强势参与、事业与顾家的冲突？


曾经我们以为，当婚姻成为生活主旋律，有些爱情会成为隔夜凉茶；可谁曾想，婚姻也会抢救无效，走向死亡？


让我们聆听他们的故事，看懂十段婚姻里的难舍难分，也理解这份重要的亲密关系。


无论坚持，还是放弃，希望每一个人都发自内心地承认：婚姻曾是快乐的。
