## 简介
家门外有个盒子，没想到这是死人寄给你的东西，一旦签收，就再也丢不掉了……


陌生快递，收了也许有性命之忧——我收到了一盒金首饰。无论我把它扔多远，最后都会回到我房间；


我贪便宜，租了一套廉价的房子，打听才知道，是套凶宅，每一任租户都死了……


欢迎来到午夜场。你不经意的一个举动，已经出卖了你。你被一双陌生的眼睛盯上了，有人正在暗处观察你，在午夜等着你。
