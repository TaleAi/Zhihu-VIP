## 简介
科研不单纯，学术有内幕，ViaX 盐趣在知乎为你揭开学术科研骗局背后的惊天真相。


**巴甫洛夫说：科学不会抛弃真诚追求它的人。** 


科学家是真诚的，而科学的核心是诚实。 


是这样吗？ 


**本该如此，很可惜现实的真相从来不单纯：** 


**科学家，那些世界上最聪明的人，也可能为了自己的野心，编织巨大的谎言。** 


  

 你不信？ 


那些写进教科书、挂在名人堂里的科学巨匠，也有不为人知的黑历史 


·牛顿排挤同僚，修改结果，只是为了保护自己的自尊心。 


·「微生物之父」巴斯德为了获得想要的结果，实验近乎「草菅人命」。 


  




**但这不能都怪科学家，那些触目惊心的学术造假背后，还有更耸人听闻的内幕。** 


·「韩国之光」黄锡禹，怎样一步步被推上巅峰，又如何跌落神坛、被万人唾弃？ 


·「血检魔女」霍姆斯，怎样忽悠众多位高权重的老男人心悦诚服，让前国务卿都执迷不悟？ 


·「未来之星」舍恩造假案的背后，诺奖得主和《科学》、《自然》杂志怎样「推波助澜」？ 


  

 个人的野心、民众的狂热、巨大的利益，让原本纯净的实验室不再纯净。 


**ViaX 盐趣，在知乎为你解密学术骗局背后的残酷真相**。
