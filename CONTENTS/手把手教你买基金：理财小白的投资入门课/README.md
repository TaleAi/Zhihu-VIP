是炒股、存款，还是买基金？从风险与收益的平衡角度来说，基金投资是更适合普通大众的理财方式。本专栏从基金投资的前期规划和知识储备讲起，介绍了基金的基本常识和投资原则，帮助你一步步选出符合自己偏好的基金。

**从风险与收益的平衡角度来说，基金投资是更适合普通大众的理财方式**。


可是，基金种类和产品众多，该如何选，又该怎么投呢？


本专栏从基金投资的前期规划和知识储备讲起，介绍了基金的基本常识和投资原则，以及基金的评价标准和选择方法，帮你一步步选出符合自己偏好的基金，这是每个入门新手都要掌握的基本技能。
