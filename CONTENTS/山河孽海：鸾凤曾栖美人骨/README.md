## 简介
刺杀、夺嫡、新政、乱世开场。


作者：任多鱼


荣国公幺女谢韫十四岁入宫，未能得到皇帝的青睐，反而与后宫诸妃关系和睦。她目睹了一位奴妃替罪而死的现场，自己也困在深宫的樊笼中，直到执拗的养子向她伸出手，困兽与幼狼抱团取暖。


渐渐地，她发现后宫嫔妃各有故事，华丽的凤凰袍剥开，是封建制度的尸山血海，萌生觉醒思想的年轻人无声死去。刺杀、夺嫡、新政、战争，乱世即将开场，而为他人命运感伤的谢韫，也在伦理与感情的抉择中，迎来了自己在宫廷中度过的第十年。
