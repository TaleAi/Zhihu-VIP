## 简介
「爱情」是江湖里面的一面镜子，照的见人性善恶，辨得出良知优劣。声工厂为你解读金庸《射雕英雄传》中，那一件件浪漫而始终如一的爱情故事。


有人的地方就有江湖，有江湖的地方就有恩怨情仇。


**「爱情」是江湖里面的一面镜子，照的见人性善恶，辨得出良知优劣。**


不一样的爱情有不一样的心酸和喜乐。


**金庸《射雕》的江湖世界里，「你侬我侬」的爱不是一句空谈：**


或许它不是花前月色下的卿卿我我，或许它不是柴米油盐间的闲话家常；


**但它带给我们的，是对爱情一往无前、生死相随的态度，促使我们心间骤然而起一份最真的感动，久久回荡，难以平息……**


一个人的特质是什么，是寻而有迹，是溯源有根的内在修为和外在言行，这或许也是郭靖收获黄蓉珍贵爱情的关键所在。


「情不知其所起，一往而深」，走遍千山、历尽沧桑才明了，命中指引皆是浮屠，唯有这份始终不改的初心才是爱情的真谛。


声工厂为你解读金庸《射雕英雄传》中，那一件件浪漫而始终如一的爱情故事。
