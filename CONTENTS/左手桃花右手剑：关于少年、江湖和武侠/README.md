## 简介
如果中文世界里的武侠共用一个江湖，来过金庸走过古龙，郭靖与李寻欢行路上相遇，杨过和楚留香在山间把酒相谈。武侠的世界太过庞大，容得下每一个人在里面潇洒走一回，值得被大书特书一笔：英雄、杀手、刀客、剑侠……写下的是快意恩仇，听闻的是刀光剑影，印证的是我们真实的人生。


所以，从江湖新人成为大侠的人，他们都经历了什么？


如果，把你抛进江湖的世界里，你想让自己成为什么样的大侠？


在开局时，你会选择：


A 身怀灭门大仇、不努力就不能报仇的的家族独苗；


B 天资绝顶但克死生母，被身为江湖一哥的父亲抛弃的孤儿；


C 书香门第，从小叛逆就要舞刀弄枪打打杀杀的混不吝；


D 别说了，我是穿越来的；


然后，你会有个师父，他可能是：


A 名门正派的掌门人；


B 大隐于市的绝世高手；


C 傻子；


D 和我一样穿越来的；


师父说，我毕生绝学，你学懂一种便可行走江湖无人可拦，你会选择：


A 七绝斩；


B 螺旋丸；


C 军体拳；


D 全都要；


……


你渐渐熟练师父传授你的武功，你的全拳越来越有力，你的刀越来越快，在你的眼里，生活的村子已经不能承载你的胸怀，你打算用脚步去丈量天下，去爱去恨去生去死。


你在柳下拜别师父，临走前师父说：「江湖建议第一条，保持做自己的赤子心。」


欢迎来到武侠世界，总有一个江湖需要你。
