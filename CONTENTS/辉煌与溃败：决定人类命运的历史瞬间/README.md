## 简介
人类社会，总是周而复始地疮好忘痛。


课本之外的世界历史，是如何写就的？


那些隐藏在大变局中的历史的细节，那些用鲜血铸就的帝国神话，那些将人类命运引向暗黑终点的瘟疫、暴乱、入侵的根源是什么？


本专栏为大家呈现一部显微镜下的世界历史——


* 1918 的西班牙大流感如何从欧洲扩展至全球，演变成一场世纪瘟疫？
* 德意志帝国的修昔底德困局如何将其卷入战争车轮，成为两次世界大战的始作俑者？
* 在科技、文化水平都不高的情况下，蒙古人是如何征服半个地球的？
* 面积与中国差不多大的欧洲大陆，为什么没有走向统一，而是分裂成如今这般格局？

我们阅读历史，是为了更深刻地理解我们当下的生存处境、全球各文明各社会的脆弱性。


我们阅读历史，也便理解了我们正身处的「危险世界」在社会历史内部有其根源。
