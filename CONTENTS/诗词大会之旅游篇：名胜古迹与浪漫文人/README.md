## 简介
名胜古迹，似乎现如今只剩一个风景的空壳，而失掉了它历史的厚重感与文学的浪漫情怀。我们的旅游究竟在看些什么？随本专栏一起，吟诗畅游好风光，去感受名胜古迹里暗藏的古代诗词韵味和诗人们的浪漫故事吧。


德国作家黑塞说，「旅游就是艳遇」。


既然是「遇」，自然是遇而不可求。想想我们是多么幸运，因为我们有那么多的美景可赏，有那么多的人可遇。


如今，五一、十一、春节，长假多了起来，旅游成了时髦而且是可以轻易实现的事儿。每逢佳节，祖国大好河山的名胜古迹上，就遍布人群。


的确，我们去过越来越多的地方，但是面对美景，我们只会说江南多么秀丽，大西北多么壮阔，除此之外，再也抒发不出什么别样的感情。


旅游，变成了只是随着人群走动，拍照留念。我们用来发现美的眼睛，只能看见风景的表面。


**殊不知，每一处风景都有它的妙处，也有可以跨越时空流传下来的诗词作品，更有无数名人趣事可以被后人吟咏。**


假如你也想体味每一处风景的真正妙处，那就随本专栏一起，吟诗畅游好风光，去感受名胜古迹里暗藏的古代诗词韵味和诗人们的浪漫故事吧。
