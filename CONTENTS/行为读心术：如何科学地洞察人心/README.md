## 简介
提炼北大心理学课程的精华，为你呈现一套系统、科学、实用的心理学知识体系；让你拥有截然不同的心理学思维，帮你摆脱困惑迷茫。


如果你是一个懂心理学的人，可能你会：


当不喜欢自己的性格时，会选择接受并保持内在和谐而不是内心矛盾；  

 当遇到挑战时，如何觉得自己不行会对自己说「我正在进步」，而不是「我不行」；  

 当向另一半提要求时，会告诉对方「可以多关心我一点吗？」，而不是「不要总是对我不闻不问」；  

 当孩子情绪有问题时，会想办法改变自己，而不是责备并改变孩子；  

 ......


心理学其实与我们息息相关，是人人都应该学习的一门学科。


本专栏将为你讲解 80 个实用心理学知识点，让你成为生活中的「心理学家」，读懂自我与他人，看懂群体现象和群体中的自己，升级思维和认知，看到最本质的问题。
