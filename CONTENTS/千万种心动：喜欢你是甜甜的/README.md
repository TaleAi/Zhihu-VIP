《女主的醒悟》——阿靖


一觉醒来，我穿书了。


穿成了一本古早虐文里受虐的女主。


挖肾，流产，误会……凡是在古早虐文里出现过的虐梗，都将会一一出现在「我」身上。


苍天啊……这个剧本也太惨了。 


  



《我有一只可爱鬼》——否极泰来


如果有一只鬼跟在你身后，你会害怕吗？


别人我不知道，但我不会。


因为他信誓旦旦对我说，他是全天下最温顺的鬼[了](#5-21-22)。


可我看着眼前这只害羞不经撩，一紧张走路还顺拐的鬼陷入了沉思……


鬼也会脸红的吗？
