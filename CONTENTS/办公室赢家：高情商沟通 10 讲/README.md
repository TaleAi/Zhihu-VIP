## 简介
为什么职场上加班最多能力最强的人反而有可能被老板炒鱿鱼？为什么职场上说错一句话，可能就会失去老板的重用？为什么性格孤僻不合群的人，最容易被同事排挤和边缘化？职场如战场，工作、人际、利益等关系错综复杂，要想游刃有余地应对，必须提高自身情商。这门课程我们选取了职场中最有代表性的 10 个场景，采用「真实案例 + 应对方法 + 认知升级 」模式，把职场牛人绝不会说的秘诀都告诉你。


**本次课程亮点**


**一、真实案例解读，一听就懂**


如何快速与同事拉近距离？怎样从细节中赢得领导的信任与重点培养？如何团结同事，让其欣然为自己解围？性格内向的人如何克服自身短板，成为谈话高手……课程中用到的案例均为真实案例，选取日常工作中最有代表性的沟通场景，代入感强，不仅感同身受，而且听后即用。


**二、职场沟通禁忌与误区，一点就透**


过度分享自己的私生活、给同事贴标签、下定义、过度透露工作进度、
宣泄对同事和上司的不满……兆民老师为你指出职场最常见的雷与坑儿，教你绕道而行。


**三、应对不同情况的话术与策略，一目了然**


聚焦职场，选择最有代表性的使用场景：如何避免尬聊？怎打造让人舒适的「谈话磁场」？替同事背了黑锅该怎么办……不同场景给出了不同应对话术，并总结成为以不变应万变的策略，教你巧妙应对职场难题，化解尴尬与矛盾。


**四、实操性强的方法论，一家之言**


本着「场景 + 话术 + 理论」 三位一体的原则，兆民老师打造了可迁移的职场沟通场景、可模仿的应对话术、可复用的理论方法，让你不仅知道什么场景说什么话、怎么说，更让你明确为什么这样应对。


**五、让努力变得卓有成效，一击中的**


为什么有的人在职场混了很多年，看上去勤勤恳恳，可就是得不到领导的重视，更谈不上被提拔呢？因为，他们忘记了在互动中更好地服务于组织。本课程将有针对性地教你如何在职场表现自己及与领导互动，让努力变得富有成效。


![](https://pic4.zhimg.com/v2-35f9009eccc5968539217cd678093a43.png)


**适合谁听：**


1. 只会干活不懂沟通的职场老实人
2. 想快速赢得领导赏识的员工
3. 经常因人际关系烦恼的人群
4. 刚毕业进入职场新人
5. 刚跳槽到新公司的职场人士
6. 所有渴望职场逆袭的人士
