## 简介
不为人知的时代真相们。


李世民开启「贞观之治」，堪称中国第一明君，而他所掩盖的丑史逐渐被人遗忘。


柴荣英武、睿智、仁德，肩负着结束乱世的使命，可他的生命突然在征程中停止。


锦衣卫作为一个特务组织，不仅替皇帝监控大臣，还会用最极端、最残酷的手段杀人。


清军入关后，为了让汉人臣服，制造了一次又一次屠城，而屠城的执行者也是汉人。


清朝统治时期，中国开始落后于世界，而导致这一切的原因，只是清朝想消灭郑成功。


翻阅史料，还原细节，在细节中找到「魔鬼」。
