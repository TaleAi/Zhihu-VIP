# 都市追风者：永远在路上

## 简介
你得过什么奇怪的病？假如你得了一时治不好又死不了的病会怎样？本专栏讲述一个特殊群体「追风者」的故事：身患奇怪的过敏症，无法在同一个地方待三个月以上，否则会诱发激烈的过敏反应甚至死亡。他们将如何面对注定漂泊的宿命？在不断迁徙的路上，又会发生什么样的故事？


追风者们的命运决定了他们会「一直在路上」。他们不会从事长期的工作，捞偏门的居多，因此生活更加自由，遭遇更加奇诡。 


在成都，接连多名追风者穿着熊猫人偶自杀，包括我的女朋友，这一切竟是一名记者出于「好心」做的局……


在桂林，深夜时分，追风者「辉子」在公园解救了一个落水女孩，身上有无数血手印，难道这里真的闹鬼？


在湘西，我拉了一个醉汉上百公里前往贵州，途中我发现他是个赶尸人，更可怕的还在后头……


追风者就像一群没有脚的鸟，永远无法停下，永远享受着别离。


我是陈双，二十五岁，这是我们的故事……
