**是你的环环 | 忠犬少年**


分手后我的狗变成了人，他不仅帅气温柔，还试图上位成为我的新一任男朋友。  

他睁着一双湿漉漉的狗狗眼看着我，「主人，我是你的环环啊！」  

好家伙，我养的边牧就叫环环。  

这算什么？大变活人吗？


**一百天情人 | 温柔教授**


替朋友参加一个百日情人的游戏，去了后发现配对对象是自己的大学教授。  

游戏规则是两人要维持一百天情侣关系，以这个身份参加未来一百天里一系列的情侣活动。  

游戏结束那天，刚好是大学的毕业典礼。  

他一把捞过想跑的我。  

「好不容易等你毕业了，还想逃？」


……
