# 神经质少女：别杀我，你就不会发现自己死了

## 简介
女孩们永远感到恐惧。夜色里打在她身上的目光，爱人嫌弃的神态，房间里、旧物上可能隐藏的摄像和秘密……一旦开始关注这些若有若无的细节和恶意，再神经大条的女孩也会变得疯狂。而这个专栏，讲述得就是那些疯狂的故事——


《匿名信》


*我的妹妹啊，她天生没有眼睛，她一直在看着我呢。*


我收到了匿名快递，里面是一幅画，画着一个女人牵着两个女孩。


女人，是我的妈妈，高个女孩，是我，穿着红裙子矮个女孩，是我的妹妹。画面上，她没有眼睛。


我的妹妹孔朵早就死了。


而我，已经逃出来 7 年了。 


  



《房间来客》


*他深深地吸了一口枕头。永远在一起，谁来和我永远在一起？*


家里不对劲。有人来过。


三天前我就发现了，有时明明记得出门时没锁门，但回家时却发现门被锁了，房间里叠好在床头的衣服也会突然变了个顺序。


而这次确认下来，是因为厨房里摆着一碟切好的苹果。


是谁？他在挑衅吗？


他会满足于仅仅是——挑衅——吗？


  



《家教之恶》


*我什么都没做。这就是我的恶。*


我来到做家教的人家，主人家正在祭拜，香烟缭绕中，我看到遗像上的照片。


那不是别人，正是我今天要教的男孩。


男孩的妈妈哭着向我解释，死去的人，是男孩多年前失踪的哥哥。


「他们真是长得一模一样啊。」


「只除了这里，太阳穴上的痣。」


女人一边说，一边摘下遗像，用黑色的马克笔，在照片里点上了一颗黑痣。
