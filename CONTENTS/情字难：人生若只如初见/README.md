## 简介
爱人之前，先爱自己



看清楚深爱背后的谎言与背叛，她们仍旧具有反击的能力与再爱的勇气。不管生活有多糟糕，愿女孩们都能明白，这个世界上最重要的人，还是自己。


**念念复年年**


嫁给宋随的第四年，我确诊了癌症，他的白月光离婚回国了。


我冷眼看着他和白月光卿卿我我。


结婚纪念日那天，他丢下我去找白月光。


计划顺利进行。


等我死后，他的报应才刚刚开始。


**舟难渡**


男朋友是不得志的歌手，终于有一天爆火，所有人都在嗑他和当红女明星的 cp。


说他们相识于微末，说他们曾相互救赎。


说最难熬的那段日子，他们相互陪伴。


那我呢。


我陪着他的八年，就不作数了吗？
