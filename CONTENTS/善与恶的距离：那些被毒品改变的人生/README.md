## 简介
人间与地狱，可能只有一纸之隔


人之初，到底性本恶还是性本善，至今争论不休。


能确定的是，人的理智让我们大多数人都能向善而不是向恶。


但如果，理智失去了呢？


善与恶、人间和地狱，距离还远吗？


作为戒毒所的管理人员，作者陈敏除了关注宏观侧面的禁毒与打击毒贩，更真切地感受到毒品对于个体、家庭、社会的真实影响，这也是这个专栏呈现的初衷。


在这个专栏里，你将看到——


曾经豪气万丈的卧底刑警，为了破案，被迫吸毒上瘾，从此英雄变狗熊，堕入无间地狱。


一个在家被父母嫌弃，在校园被同学欺凌，在社会被人用毒品控制，并强迫卖身还债的女孩，她说自己最大的愿望是下辈子不做人。


一名当过社区禁毒志愿者的女医生，怀疑丈夫与人有染，因爱生恨，愤怒之下把毒品当杀人工具，并利用专业知识将尸体肢解。


让我们跟随这些被毒品改变的人生，真实感受理智背后的人性深渊。
