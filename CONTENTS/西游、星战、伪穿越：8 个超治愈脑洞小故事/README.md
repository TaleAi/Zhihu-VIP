## 简介
西游记里唐僧最牵挂的初恋，来自未来的神经病怪人，抗击外星人的高中生……奇幻、脑洞、贫嘴，8 个超治愈的甜甜小故事，看到你笑，暖到你哭，点击专栏开启「郁闷」星人自救计划，一起来唠会儿吧。


带着动物园去西天取经的唐僧，他也有自己心爱的姑娘；


无忧无虑的高中生遇见了来自未来的神经病；


捡来的小孙女得了绝症，捡垃圾的老大爷决定打黑拳捞一笔；


脑海中有了爷爷的记忆，突然觉得老爸可爱了起来……艾玛，咋还乱辈了呢？


或大或小的脑洞，发生在我们成长过的年代里，发生在我们道听途说的故事里，发生在我们打小追更的西游记里，更发生在我们共有的记忆里。


奇幻，脑洞，贫嘴，乐子，都是曾经我们一起唠过的嗑。


既然我们都在不可避免的长大，变丧，变平凡，如果有幸，在某个失眠的夜晚，你无意间点开了这个专栏，希望能带给你短暂的休息和片刻的治愈，比心。
