## 简介
封闭空间，总是能够引起人们的恐惧心理。大雪封路的小镇，茫茫大海中的孤岛，甚至，是一个棺材。当故事里的女主身处其中，面对邪恶的反派力量，她们如何能逃出生天，实现反杀？


在这个专栏里，都是关于女孩的极限逃脱故事。


面对各种各样的「囚禁」，她们有的展现出超高智商，有的被无奈的现实逼成坏人，更有的，是在可怖的环境里，不得不亲手沾染鲜血。


贯穿着的，是女孩们面对困境，身上迸发出的生命力。


这些生命力，将带领着读者，参与体验一段段或离奇或真实的人生。


生活是无趣枯燥的，现实是充满了压力的。


唯有故事里的「体验」，能让我们短暂地离开自己的世界，去过一次别人的生活。


生活在别处，生活可以在别处。


很高兴你来到这个，我为你准备的「别处」。
