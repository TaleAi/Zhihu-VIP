## 22.化妆穿搭：复合定位法，实现副业收入翻倍
上节课我们讲到【旅游领队】个人能力探究，边玩边赚，无压力不囤货。我从时间角度出发，详细给大家拆解了，当我们掌握一项新技能的时候，可以如何提高自己的能力。那么今天我们来到第 21 节课：【化妆穿搭】复合定位法，实现副业收入翻倍。


小乐是我副业赚钱实操营的学员，第一节课结束后，答疑环节，她向我提了一个问题：


Angie 老师，我非常喜欢穿搭，我身边的朋友对我的评价是很普通的衣服，经过我的搭配后就变得很好看了。但是我又很纠结，因为我更擅长的是化妆相关的周边，比如说化妆技巧分享，而且我有很多性价比非常高的化妆品资源。所以我内心很纠结，请问老师，我该做如何取舍呢。


看完小乐的「纠结」，我顿时乐了，在我看来，小乐不单止不用纠结，还要偷开心。


因为小乐完全不需要取舍，只要把两个标签进行复合定位就好了。


因为这两个标签都是和美相关的，完全不需要作取舍，共同宣传就行了。


所以今天的课程重点就是我会先通过小乐的例子，来让你发现你自己适合复合定位的副业类型，然后再和你分享，如何把身上的两个标签进行复合定位，从而轻松实现副业收入翻倍。


接下来，我们来到第一个知识点：盘点适合进行复合定位的标签类型。


小乐的例子，提到的是化妆和穿搭可以进行复合定位，这都属于美学范畴。


复合定位第的第一个思路是，在同一类的大的范畴下，找到可以进行复合定位的两小类。


凡是向小乐咨询化妆相关技巧和购买化妆品的用户，小乐可以问多一句：在穿搭上有需要吗？


这样多问一句，就可以直接进行再一次的营销和成交了，为用户提供更多的价值。


第二个思路是，表面上看起来非同一类的范畴，但服务的终极目标一致的情况下，可以进行复合定位。


举个例子，减脂美食+健身减脂。


比如因为爱做美食的小杰，在研究美食的过程中，发现瘦和吃有很大的关系，于是开始研究减脂餐。


那在研究减肥餐之后，发现瘦不单止和吃有关，还和健身有关。小杰就又开始自己健身，并研究如何健身减脂。最后把两个标签结合在了一起。


大家可以跟着这样类型的老师学习健身减脂的同时，还和老师学习如何制作减脂餐，如果老师销售减脂餐，还可以和老师进行减脂餐的购买。


第三个思路是，从变现方式出发，拥有多标签复合定位。


举个例子，擅长时间管理能力，同时是时间管理一对一咨询师，也是时间管理课程讲师。


最后，我们第二个知识点：复合定位的标签类型该如何进行捆绑变现。


首先，第一个方法是不断打造自己的个人品牌，成为多个标签个人 IP。


先来解释一下什么叫做个人 IP，简单来说，像我这类型的人就是个人 IP。


以我的副业是讲师为例子，我把自己擅长的时间管理能力，以课程的形式进行呈现，课程是我的产品，讲师是我交付产品的其中一个赚钱身份，这就是 IP。


成为 IP 非常重要的一个维度是，从能力的角度出发，把自己的定位确定下来，并且把展示自己需要的一些能力，像是写作、演讲、文案能力一个个击破掌握。


IP 定位可以是大众的这种阅读、演讲、写作技能类的提升，也可以是从小到大的一些爱好，比如从小喜欢画画、表演等，也可以是一些先天的优势，比如声音很好听，或者是长得好看并且很喜欢护肤、穿搭等。


以我的故事为例，2015 年初，在渐渐适应了所在外企的工作节奏后，我每天多出了一大段空闲时间来。


起初这些时间几乎是完全被浪费的，经历了一段迷茫期后，在完成工作之余，我开始了大量阅读、学习的阶段，几乎是每天阅读 1~2 本书的节奏。书读得多了后，发现自己渐渐有了分享的渴望，最初的分享方式是对自己学习后的阶段性梳理：分享给自己看。


在这一步，我的兴趣爱好没有给自己带来任何的收入，但却为之后的副业赚钱打下了坚实的基础。


如果你目前的状态是工作之余有比较多的时间，而你又暂时没有跳槽的计划，建议不要再把时间花在无聊、逛淘宝以及和同事闲聊上，而是开始探索自己的兴趣。


在这里，我还要提醒大家一点，刚开始不用太介意一定要学习固定的一类内容，如果你花过多的时间在思考自己的定位上，反而容易导致迟迟无法行动起来。


除此之外，你还可以选一些工作上也能用到的技能，比如说你是销售，在空余时间里，可以花在提升自己的表达沟通能力上。对工作本身的提升有帮助之余，未来开展副业也会有很大的帮助。


经过一段时间的积累，可以直接尝试以咨询师、私教、讲师等身份进行副业的变现。


我的朋友 G，正职工作是一名财务，她特别喜欢跑步，业余时间几乎都用在研究跑步和到处去跑马拉松上，前段时间还自学拿到了营养证书，现在副业在教别人如何科学健身和饮食，从而达到塑身效果。


每一次见到她，都觉得她的状态特别好，也特别羡慕她，把副业做好的同时就已经是做好了精力管理本身。


我的另一个学员有一个高中老师 Y，在约了我一对一咨询后，就以咨询师这个赚钱的身份定位开启了副业赚钱之旅。


她通过写作，以文字的形式让更多的人知道自己。


随后学习了生涯规划，成为了生涯规划师，一开始给别人做公益咨询积累经验，随着能力越来越强，她开始收费，一个人最高可以达到收取五位数的咨询费。


知名度越来越高的同时，Y 创办自己的训练营和社群，也多次受邀到多个地方开展演讲。


对于大部分人来说，也许在一些领域已经积累了很完善的框架和很丰富的教学内容，但是写和演讲的能力不足，无法很好的通过这些渠道分享出去，但对于 Y 来说，写和演讲的能力早就在职业生涯当中锻炼了出来，一旦对自己所具备的知识进行重新梳理和整合，表达出来的效果将大大提高。


我虽然不是老师，但毕业后一直从事销售相关的工作，表达能力自然不差，再加上一直都很喜欢学习，自身的职业经验也很丰富，这个组合非常契合我最开始探索副业时的状态。


其次，第二类是把自己拥有的多标签能力，用在服务 IP 作为开启副业身份的方式。


挖掘出自身的能力之后，发现自己并不具备教授的能力，比如说掌握了穿搭的技巧之后，并不想走上教授他人如何穿搭这样的副业赚钱之旅上，而是想直接通过协助别人打造形象的角度出发来开启自己的服务，这个就叫做服务 IP 的类别。


类似的方式我还可以举非常多的例子。


比如说 IP 打磨出了自己的课程之后，发现自己不具备运营课程社群的能力，你协助此类 IP 进行社群的运营。


再比如，每个 IP 一定需要制作 PPT 来进行课程的讲授，但是 IP 并不具备制作精美 PPT 的能力，而你又具备了制作 PPT 的能力，于是你通过为 IP 制作 PPT 来赚取副业收入。


我身边通过这类身份赚取到副业的人也有非常多。


并不是人人都想曝光自己，当你觉得自己能力还不够或者内心想做幕后工作者的时候，服务 IP 是非常好的副业变现方式。


我在之后的课程里，也会讲解大量的案例，方便大家直接对号入座，找到最适合自己的副业赚钱路径。


以上就是我们本节课的主要内容了，一共有两个方面：


第一个知识点：盘点适合进行复合定位的标签类型。


第二个知识点：复合定位的标签类型该如何进行捆绑变现。
