## 简介
我买的婚房成了烂尾楼。从此以后，厄运连连，绝境求生。



我本来买房打算结婚的，结果买了个烂尾楼，从此以后，厄运连连。


和挚爱分手，闹得家徒四壁，自己的征信也烂了，银行卡也被冻了，成了见不得光的老鼠。父母为了帮我还贷，打工还受了重伤，性命垂危，我却连医药费都拿不出。我恨这该死而狡猾的开发商李建光！


我去了李建光的家，发现那么多业主被烂尾楼拖得半死，他把自己摘得干干净净，居然还买了一堆纯金的铸件生肖大摇大摆的放在家里。我本打算拿走李建光的纯金生肖，先把家人的医药费填上，却卷入了一系列案子之中。


业主赵雪莲的纵火、巨财产产转移、商业大鳄的陷害、不明身份大佬的追杀、刑警苏清河的抓捕、被通缉——


这一系列的事件让我喘不过气来。无论如何，我都必须查明真相，绝境求生！
