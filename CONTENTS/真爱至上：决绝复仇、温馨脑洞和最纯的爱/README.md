## 简介
爱，是我们的特殊食物。


为了保护自己的女儿，一个父亲，踏上复仇之旅。勾心斗角，玩弄人心。道阻且坚，却也挡不住，守护女儿的心。


梦里梦见的手机号，醒来要不要打？我不仅打了，还加上了微信。奇奇怪怪的爱情，就此开始。


我的老婆，是修真者，正在渡天劫。修真者有啥忌口不？吃啥比较补？修真者的保姆丈夫，在线发问。


……


在这个专栏里，有极致爽文、决绝复仇，也有温暖治愈的浪漫脑洞，每篇都有新鲜花样，却始终贯穿着「爱」。


对家人的爱，对恋人的爱，对职业的爱……


爱，是我们的特殊食物，


吃下去后，不会长胖，却让我们更有勇气，面对生活，


一日三餐，日出日落，闲暇时候，来这里，给自己加个餐吧。
