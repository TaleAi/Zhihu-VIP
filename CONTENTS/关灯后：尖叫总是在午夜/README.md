## 简介
关灯之后，各种诡异的事情都出现了，奇奇怪怪的。


我死去的丈夫陈远头七那天复活了！


大清早的他递给我一碗粥，我一口而尽，凝视着他那张脸。


他没有消失，我也肉疼，这不是梦。


可我一个星期前，还亲自看他的遗体进入火炉。


——《头七复活》
