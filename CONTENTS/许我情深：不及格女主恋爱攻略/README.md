1.《我当美强惨女配那些年》：万虚境百年一现世，再过三个月，正好一个百年，这一次，晏九不要再弄丢她了。她是魔神为万虚境造的炉鼎，每百年向万虚境献祭一次灵力，并丧失全部记忆，她总会忘记他，于是他以血肉筑造血器，只求毁了万虚境。「这一次，就换你来记住我吧。」


2.《做一个合格的公主》：窝囊公主被窝囊父皇拱手送给了摄政王，但这个摄政王好像没有传闻中那么铁血无情，只是有点奇怪的癖好，比如总喜欢搂着她睡但啥也不干，还喜欢在无人时擦拭一身将军甲胄和一把破的黄金伞骨。只忽然有一天，他穿上了他的甲胄，执起黄金伞，倾身询问：「本王把江山送你如何？」完了，他终于要造我父皇的反了！诶？但为什么是我称帝？


3.《是风动》：我从小就知道，我成为皇后只是早晚的事。我合该荣宠一生，当个悠哉悠哉的皇后，与我的家族相辅相成。但我最担心的事情，还是发生了——那就是我的夫君他太能干了，以致于我阿娘到现在都还搞不明白，她当年给我安排的分明是妖妃剧本，我夫君现在怎么就成了明君......


4.《枝影扶疏》：生在一个女尊国家，我死那天，全京城的人都在欢呼。锣鼓喧天，鞭炮齐鸣，不知道的人还以为全城都在办喜事嫁儿子呢。但他们高兴得太早了，因为我重生了。正得意的时候，我却发现自己重生到了前夫被我打断腿的第二天......
