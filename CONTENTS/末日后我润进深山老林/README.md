## 简介
我带着小奶狗一家在末日建桃花源！


我是一名快递女司机，末日后被丧尸围攻死在送货的路上。重生后，我果断卖房，在老家荒废的山村盘下仓库囤了大量物资。跑路前我救下同事老张一家三口一同带进山村。进山后，外界丧尸爆发，天气异常。四人不想坐吃山空，于是对深山进行探索，意外发现深山里物资丰富，别有洞天。期间我和老张的儿子张斯年发现了仓库地窖通往后山的隧道，在一次外出寻找石油后，意识到了山村并不安全，于是四人举家搬迁进入深山山洞中，过上了与世隔绝的桃源生活。
