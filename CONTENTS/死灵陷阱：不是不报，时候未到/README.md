## 简介
深夜的敲门声，窗前的流浪者，陌生的枕边人……那些惊悚瞬间的背后，隐藏着怎样的真相？一组超多反转的现世报故事集，为你的盛夏送上凉意。


《善良的房东大叔》


一室一厅的房子，地铁口，出门三公里就是商圈，只要三百一个月。


这样的房子，明显有鬼，你敢租吗？


我倒是有胆子租，可是现在，我也忍不住要怀疑：


我还有命住吗？


  



《深夜的敲门声》


深夜，门外传来敲门声，三长两短，听得我寒毛直竖。


我听到他说：开门，我想回家。


可是当我真的推开门，却发现，门外，空无一人。


  



《神秘访客》


我怀疑我女朋友不是我女朋友。


她在深夜给备注成「老公」的人打电话，不允许我碰她甚至是拉手，更在我试图出门时安排保安阻拦……


我被一个自称是我女朋友的人软禁起来了。


而现在，变态杀人狂，正在敲我的门。
