## 简介
谁能穿破迷雾，拯救我于水火？


**《消失的女孩》**


最近，我一直在做同一个梦。


梦里，我的尸体被人拉着腿在地上拖行，瞳孔涣散。


我想，我是被脏东西缠住了。


让我没想到的是，这个脏东西还不满足于只出现在梦中。


卫生间镜子中，她扭曲咆哮，企图脱离镜子扑向我。


我去找我的术士朋友帮忙，却在调查途中，在一段不该出现的录像里，看到了一个和我长得一模一样的女孩……
