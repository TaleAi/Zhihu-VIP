## 简介
魏晋文化是中国文化史上的一次文艺复兴。



魏晋南北朝是中国历史上非常有特点的一个时期。


从政治上来看，这是一个分裂的、混乱的、人民很痛苦的时期。但是从文化史、精神史、思想史的角度看，这却是辉煌灿烂的一个时期。


这个专栏，我们从惊才绝艳的代表人物入手，介绍魏晋时期精彩纷呈的文化现象：清谈、啸、酒……从这些存在于史书记载，流传于泱泱中华的文化现象入手，也许你可以一窥魏晋文化之盛。


* 屠场里的风流名士王衍

作为西晋年间最有名的清谈名士之一，王衍亲眼见证了清谈之风的兴起与兴盛，最后，在宁平城外的屠场中，这位狂热的清谈爱好者，却用对清谈的谴责结束了荒唐可笑的一生。随同他一起死亡的，还有被他祸害过的国家。


* 沧海一声啸：大晋废物阮籍

作为精通啸的行为艺术家，阮籍的一生，都浓缩在啸里。在魏晋南北朝这个中国历史上罕见的大乱世，啸是他抨击虚伪的卫道士的武器，也是他逃离污浊的现实世界的凭借。但当享乐与诡计成为政治的主流，阮籍和他的啸一起，沉没在历史的沧海里。
