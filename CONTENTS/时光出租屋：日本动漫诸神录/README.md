## 简介
来吧，让我们重新亲近道别已久的漫画往事，了解值得怀念的漫画家传奇，探究历史际遇的奇妙机缘。


曾让我们偷偷放在课桌下的漫画名著，迫不及待放学收看的动画经典，似乎已离长大后的自己越来越远。


但时光没有令这个世界褪色，与之相关的热血、有趣故事，反而被挖掘出得越来越多：


富樫义博：一个原本敬业精神极佳的好作者，是怎么作的孽，成了知名的「坑王之王」？


藤子不二雄：《哆啦 A 梦》传奇作者二人组，一人几乎成为曹洞宗住持，一人差点被母亲逼得放弃漫画梦。


梶原一骑：声名狼藉的流氓、浪子，女儿被黑道绑架撕票，最终却逆袭为日漫大师。


 


知名日本文化研究者王新禧以一部《时光出租屋：日本动漫诸神录》，从战后日本新漫画诞生的时期，讲述到了近百年后的今天。


他历数了那些广为人知的漫画家们一咏三叹的传奇经历，观览了那些万人空巷的动漫背后不为人知的商业秘密。


他揭示出的，东邻日本动漫产业取得如此巨大成就的诸多成因，更是值得我们学习、借鉴。
