## 简介
围绕都市男女展开的「中式」灵异故事。


事出反常必为妖。可有多少妖物祸端，是由人心滋生的。


本专栏都是男欢女爱，人心不甘，从而引出妖物祸端的故事。


有些人为达目的不择手段，为了逃避责任和罪过，寄希望于妖物、巫术、邪术，继而害人害己，惨遭报应。


某些篇章有些许重口，请大家注意绕行。


  



**《狐魅人妻》**


结婚后，发现老公林瀚云有病，他供了个狐仙像，把我献给狐仙，想借此来治他的病。


可他的一个死党突然死了，怀着孕的老婆找上了门，我才知道供狐仙，还要生小狐狸。


在我的逼迫下，林瀚云告诉我，他们当年虐杀一窝狐狸的事情，还威胁我，如果不生下小狐狸，我会死得比他更惨……


  



**《柳仙破巫》**


小姑子未婚先孕，婆婆劝我把身份证和社保借小姑子住院，到时生下孩子出生证上就写我的名字，算我和老公生的，免得小姑子名声上不好听。


我不同意，便回了娘家，但在回去后，我下面出血不止，昏迷后送医院，说是子宫肌瘤，后期也可能会不孕，可我半年前体检都没有事。


我躺在病床上，老公和婆婆来看我，就又劝我，反正都可能不孕了，还不如直接养小姑子的孩子。


我当然不同意，婆婆阴恻恻地走了，当晚我在病床上痛得死去活来，可却查不出病因。


旁边一个守着孕妇的老太太，说看我这样子就是被咒了。我妈托关系找了个神婆帮我看，那神婆拿一根柳条，在我手腕上扎了一个环，那痛就消失了。


我还吐了很多像牙刷毛一样的东西出来，神婆说我这样子是被人用巫术扎咒了，她也解决不了，要想解决的话，就得去城北一棵千年大柳树下，请柳仙出手。


我找到小姑子，她一时得意告诉我是婆婆要咒死我，而我老公都是知道的，于是我开始了报复……
