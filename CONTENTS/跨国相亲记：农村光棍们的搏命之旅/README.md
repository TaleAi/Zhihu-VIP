## 简介
本专栏采访跨国婚介李向东和数位当事人，记录下一幕幕险象环生的荒诞往事。


在中原腹地的广袤农村，到处弥漫着大龄青年的婚恋恐慌。


在村里，30 岁是男人的一个婚姻大坎儿，如果解决不了单身问题，可能会光棍一生。


被婚恋市场淘汰，又被媒人放弃后，出国相亲成了这些 30+ 光棍们的救命稻草。


为了出国拢媳妇，很多家庭砸锅卖铁。


年过五旬、从未出过国的农村老汉李向东，从中嗅到商机，干起了跨国婚介这门生意。


**他带着同村光棍，万里之遥，豁出命奔赴一个个陌生的国度：巴基斯坦、尼泊尔……**


![img](https://pica.zhimg.com/v2-fa5043136c1d866f778d708240c30998.png?source=6a64a727)

⬆️ 巴基斯坦不少地方非常贫穷，毒品泛滥，遍布卖血的黑市，很多姑娘生不如死，普遍有远嫁的意愿，因此成功率极高。而尼泊尔是世界上人口贩卖最猖獗的国家，尤其是尼泊尔和印度边境地区，人贩子们几乎就是明目张胆地抢女孩。在传统尼泊尔人眼里，女性没有任何地位和权利。


![img](https://pic2.zhimg.com/v2-f232670122267b4559da26a6ef395baa.png?source=6a64a727)

⬆️ 尼泊尔边境偷猎猖獗，偷猎者大都是身背血债的亡命之徒，无人区稀释了法律的力量，是罪恶最好的掩护。如果不是熟门熟路的老司机，很容易出事。


![img](https://pic3.zhimg.com/v2-ca36a80ee90fc00074c5831ffda8736e.png?source=6a64a727)

⬆️ 这个白楼原本是援建中巴经济走廊的技术人员的落脚地，跨国中介在使用的空档期，将其打造成相亲团的临时落脚点。街上很多拿枪的保安，大部分是外国商人在当地雇的，打死人是常事，警察不会管。


**风雨险恶，李向东带着相亲团跑了几年，每一次都在押财搏命：**


* 农村小伙自作聪明误信骗婚黑中介，从此在巴基斯坦人间蒸发；
* 相亲团途中遇边境偷猎者火拼，一行人差点命丧他乡；
* 被边境中介诓骗，花了十几万领回来的巴基斯坦寡妇，却没有生育能力；
* 贪小便宜，领回来的三个尼泊尔女孩，都检查出艾滋病；
* 被毒辣的边境中介当作销货、出逃的幌子，差点沾上走私的罪名。

在村里，李向东是如雷贯耳的能人狠人，是财资雄厚的村镇豪强。到了边境，李向东明白，他只是一个在人家嘴里掏食的小角色，在跨国相亲这条利益链条上，他自始至终都只是可有可无的底层中介。


真正能赚大钱的营生在村子外面，但乡村之外的江湖，他此生已无法插足。
