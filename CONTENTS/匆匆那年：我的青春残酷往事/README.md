# 匆匆那年：我的青春残酷往事

### 存档日期：2022-06-12

## 简介
喋血街头的小混混、被校园贷逼到工厂流水线上的学生、费尽心机留在大城市的室友、论文造假差点被发现的研究生……7 段跌跌撞撞的青春，7 条艰难的成长之路。


我的室友刘义守是农村考上来的小伙，在我们满脑子都是女神的时候，他却在下一盘大棋。凭着对院领导的百般殷勤，他一步步当上了学生会主席，毕业后还在院领导推荐下拿到了一家大型国有企业的签约名额，顺利留在了省城。不过，在毕业酒会上，曾经和他出双入对的班花，揭开了一个他一直在躲避的秘密。


李小军带着东拼西凑的初稿去见张教授，张教授三重否定后，让李小军当面撕了论文。心情苦闷的李小军，再次打开了陪聊网站，选了一名文艺女青年，并单方面将她的观点写成了一篇论文，大受张教授夸赞。论文答辩现场，他陡然发现新来的答辩评委似乎就是那个陪聊的文艺女青年。


我大学最好的朋友王辉，曾经穿着安踏，用着充话费送的魅蓝手机。追女神时，虚荣心作祟，用校园贷购买了苹果手机，穿上了阿迪耐克。他的借款也从 3000 滚到了 50000，随之而来的是无尽的威胁和恐吓。每月生活费一千的他，为了还清校园贷，四处找人借钱，却又陷在网络赌博中。


大飞年轻时，港片像潮水一般涌进来。他最喜欢的是山鸡，所以学着他的模样，染了一头黄毛，招摇过市。在录像厅的一次扫黄中，大飞认识了高三学生朱蔷薇。朱蔷薇说她喜欢《红玫瑰与白玫瑰》，大飞假装恍然大悟，说：「哦，原来是玫瑰花啊！」两种不同的人终究要过不同的人生。朱蔷薇考上了大学，阿飞砍伤了人躲在外地避风头。朱蔷薇再见到阿飞，是在一家医院里。


……


这 7 个故事的主人公，在这些残酷往事中理解了人性，看清了社会规则，体会了爱而不得的美好情感……


希望我们也能从以上故事中获得成长。
