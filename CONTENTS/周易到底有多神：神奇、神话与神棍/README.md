## 简介
不是教你「术」：纯方法论；不是教你「道」：大而空泛，而是用独特的视角，将易经的理论结合我们的实际工作和生活，深入浅出地化之于无形，实用性强，把玩亦妙。


孔子说，读懂了《易经》，人生就不会犯大错误；曾国潘说，《易经》是一个人安身立命的根本；郭沫若说，《易经》是一座神秘的殿堂。


张瑞敏把《易经》思想融入海尔文化，把《易经》智慧运用于海尔管理；马云不止一次公开推荐易经，并建议职场人都该云《易经》里拜拜师；日本「经营之神」松下幸之助，要求公司员工熟读背诵、灵活运用《易经》。


那么，《易经》到底好在哪？


《易经》被誉为群经之始，它是中华文化的总源头，它可以指导天下所有人做人做事，帮助人们提升道德、提高能力，趋吉避凶，成就顺达人生。


读懂《易经》，你能学到职场进阶之道；学到夫妻和睦之法；学到扭转危局之术。可以说，人生大智慧、大格局，都藏在《易经》中。


文化学者刘天然凝聚二十年所学，借助易经的智慧为您开释解惑，指点迷津。听刘天然嬉笑怒骂，妙解周易，讲易经智慧，测人事之变，知祸福倚伏，成就顺达人生。
