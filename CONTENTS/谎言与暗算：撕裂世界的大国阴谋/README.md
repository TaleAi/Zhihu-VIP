## 简介
幕后黑手，悄然将人类推入深渊。


波云诡谲的 20 世纪，人类的过往是一部泣血之书——


将欧洲迅速卷入衰落的「一战」、硝烟弥漫全球的「二战」、战后数次将人类推向毁灭边缘的「冷战」……似乎无论如何抗争，世界终会走向割裂破败的结局。


这一切的始作俑者，是几个永远在蠢蠢欲动的大国，是一些沉浸在蜜语甜言中的腐朽政权，是几位傲慢践踏他国利益的领袖们，他们在铁幕背后磨刀霍霍，他们成为撕裂世界的始作俑者——


**30 年代的乌克兰，如何因为苏联的「五年计划」，举国遭受饥荒之苦？**


**1939 年，德国用「闪电战」奇袭波兰，16 天后苏联从东侧入侵波兰。1943 年，苏联卡廷森林里发现两万具惨死的波兰军官的尸体。举世震惊的「卡廷惨案」的幕后黑手，究竟是谁？**


**从 1942 年美国陆军部制定的「曼哈顿计划」到日本广岛上空的蘑菇云，原子弹是如何诞生的，又是如何一次次将人类推向毁灭的边缘？**


**韩国 70 年代最著名的导演申相玉，为什么会被朝鲜领袖金日成绑架，又如何借口参加电影节逃回南韩？**


**1968 年，汉城「1.21 刺杀案」发生后，逃过一劫的韩国总统朴正熙发誓要派出南韩最精锐的特种部队，向朝鲜领袖金日成复仇，一场杀戮的角逐悄然上演……**
