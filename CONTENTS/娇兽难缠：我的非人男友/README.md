## 简介
万花丛中过，烈女怕「缠狼」，《娇兽难缠》为你讲述人类女主与「非人男友」的爱情故事。


**1.锢神**


暹巫一族世代养蛊，信奉蛊神晏奚，他拥有强大的力量，俊美且温柔，对信奉他的族众充满了悲悯——除了他的圣女阿巫。他亲手养大阿巫，却将她囚禁在神殿，若她踏出一步，便要接受来自于蛊神的惩罚。阿巫想要逃离晏奚，并为此尝试了不同的方法弑神，然而皆以失败告终，直到晏奚承诺——待阿巫十六岁那日，他会将自己的心送给她……


**2.缠蟒**


古代巴蜀之地信奉蛇神，每隔一百年，巴族便会将族中最美丽的少女献祭给蛇神，弯奴便是那个被选中的新娘。她被自己的族人送到了蛇神身边，发现所谓的蛇神，竟是一条上半身能够分化为人形的巨蟒，还阴差阳错为它取了一个名字，斯傩。斯傩将弯奴视为珍宝，待她温和纵容，却仍旧讨不到伴侣的欢心……
