## 简介
传统的三百六十行在飞速发展的今天被迅速颠覆，越来越多新奇特的职业悄然出现。从事这些职业是一种怎样的体验？本专栏将挖掘那些我们身边的特殊职业从业者，讲述他们的职业故事，深扒职业潜规则，撩开神秘的职业面纱。


男人没有子宫，注定无法理解女人怀孕生产的痛苦？  

 ——男助产士不同意


忍痛挨刀子整容的女人们，背后都有怎样的苦楚和动机？  

 ——整形医生知道


开豪车的深夜买醉人，有什么样的无奈和心酸？  

 ——代驾员略知一二


为了拉到融资，创业者都吹过什么样的牛皮？  

 ——投资人见过太多


**选择一种工作，就是选择一种理解人生的方式。**  

 **不一样的职业，就有不一样的故事打开方式。**


到离风口更近的地方去，  

 到离财富更近的地方去，  

 你，会看见人性的另一面。


我们经营一间有故事的「小酒馆」，  

 职业各异的人在此短暂停留，  

 述说与世界自在相处的方式。


来，喝一杯。
