## 简介
夜幕降临，校园里的活人变成了丧尸。这是残忍的游戏，黑暗中蠢蠢欲动的猎食者，隐匿在幸存者中的「犹大」，人们一个个死去，幸存者挣扎着等待黎明。
然而天亮之后，故事才刚刚开始……



薄暮冰轮代表作《彩蛋游戏》1-3 部全集。


普通的男大学生林觉在广场接到彩蛋，随后被卷入接连不断的生存游戏。


校园内的丧尸、热带雨林的昆虫，不断变强的对手、潜伏在队伍之中的「犹大」……内忧外患之下，一场场厮杀不断来临，危机之中，人性的阴暗面被暴露无遗。


游戏过程中，林觉遇上了学长宋寒章，一个异常冷静而且判断力出众的临床医学大四生。


在宋寒章的提点下，林觉逐渐成长起来，从一个生存菜鸟逐渐变成了值得信赖的伙伴；而宋寒章也被林觉的信任打动，两人在这个充满背叛的游戏中建立起了坚实的友谊。 


直到终章之夜来临，时空被扭转，来自不同年代的队伍齐聚一堂，为了争夺活下去的唯一名额，展开了一场逃杀游戏……
