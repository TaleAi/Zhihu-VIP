# 激发孩子主动性：从「要我做」到「我要做」

### 状态：已完结，共 21 节

## 简介
培养内动力，让孩子独立自律。


有内动力的孩子和一般孩子有什么区别？


从性格急躁，磨蹭拖延到有条有理，独立自律。


从害怕困难，情绪低落到迎难而上，乐观积极……


内动力能让孩子拥有自觉性，从「要我做」到「我要做」。


它是孩子与生俱来的能力，但很多家长在与孩子相处的过程中毫无意识地把它破坏了。


那怎么判断孩子的内在动力情况呢？可以观察下孩子有没有这样的行为：


做作业学习对他来说非常难过、学习的过程中很容易放弃，碰到困难很容易哭、做事情总是三分钟热度……


如果有这样的行为，那就要注意是不是孩子的内动力被破坏了。


那内动力被破坏了还可以重建吗？杨杰老师说「当然可以。只要你看到了那一条感觉的魔法线，就可以修复孩子的内在动力」。


本课程中，家庭教育专家杨杰老师将通过 4 大模块，给家长 6 个行为指南，4 个实操工具和 3 项保障措施，帮助家长系统地掌握一套培养内在动力的方法，帮孩子重新拥有积极向上的力量。
