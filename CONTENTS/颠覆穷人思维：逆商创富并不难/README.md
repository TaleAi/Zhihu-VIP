## 简介
你靠什么挣钱？靠时间和体力赚钱还是靠资源和谋略赚钱？本专栏将分享大咖商业实战经验，解读逆商思维，巧妙融合心理学、处世哲学以及管理艺术，颠覆穷人思维，成功逆商创富。


如何快速开单，站稳市场脚跟？


借势营销，该如何零成本设计？


疫情期间的三个电商红利，你抓住了多少？


大多数的普通人，靠时间和体力赚钱，有商业头脑的人，靠资源和谋略赚钱。拥有逆商思维，研究如何用小资源撬动大财富的学问，了解企业扭亏为盈的手段。


本专栏将从企业活法、营销三度、投资内参、人情练达等方面分享多位大咖的商业实战经验，带你培养高级逆商思维。
