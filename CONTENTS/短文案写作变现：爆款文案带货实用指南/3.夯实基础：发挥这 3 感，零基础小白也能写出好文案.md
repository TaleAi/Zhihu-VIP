## 3.夯实基础：发挥这 3 感，零基础小白也能写出好文案
很多人一听到文案二字，总觉得写文案难，遣词造句要很高大上，需要花大量时间和工夫，却不一定有等比的效果。


那么，零基础新人如何靠文案变现呢？在学习具体的变现方式之前，我们首先要打好文案基本功，了解什么是好文案，懂得分析市场和用户，能够把握一些基本的营销心理，做好变现准备。


今天这节课主要讲两部分内容：


第一部分，我会让你了解什么样的文案才是好文案，这个过程中，你会发现人人都有写好文案的天赋，新手小白也不例外。


第二部分，我会给到你新手作者写好文案的三个心法。助你写出抓人眼球，直击要害，打动人心的好文案。


**一、什么是好文案**
------------


首先我们来看第一部分，什么是好文案。好文案一定是能打动人心的。而想要打动对方，根本不需要写作能力，只要会说话就行，更需要的是共情力。其次，现实生活中，听起来悦耳、华丽、优美的语言不一定打动人心，反而是一些简单、通俗的话更能触动人。


很多人一提到文案，就觉得门槛高、不好写，是做营销推广和专业文案才能写的。其实文案，文案可大可小，大到宣传一座城市，小到写 100 字的微信群消息或一句话标题，都是文案。


每个人在日常生活工作中产生的，用来传递信息、影响他人情感和行为的文字，都是文案。


专门从事文案工作的人，反而因为受经验和模式的限制，写出呆板麻木的文案，缺少打动人心的灵气。实际上，只要有心，普通人也能写出打动人心的好文案。很多走心的好文案就是一句人人都能写出的普普通通的话。


例如今年很火的情感治愈类的节目《朋友请听好》的文案「太阳不出来，也是在那的」，句子简短、质朴、温暖，虽然没有直接表达节目卖点，但蕴含了丰富的情感，表达了陪伴和情感治愈的主题，达到「无招胜有招」的境界。


![img](https://pic3.zhimg.com/v2-fb3be19182b57d5db7b90880c3acb5b1.webp)

只要掌握 3 个心法，没有文案经验也能写出打动人心的好文案。


**二、把握「三感」，写出高转化率文案**
---------------------


这就来到我们今天要讲的第二部分，这 3 个心法概括起来就是三感：对象感、代入感、共鸣感。而这是影响他人，促成行动的关键。如果你在写文案的时候能够时刻记住这 3 感，你也能写出走心的 10W+爆款文案，带来高转化率。


先来看几个例子。


第一个是 AIRisim 的广告，这是优衣库下面的一个子品牌。我们看这个文案：


极致舒适 AIRism


升级后的 AIRism 全新登场！极致柔顺，吸放湿效果更强！


![img](https://pic2.zhimg.com/v2-43e008864772c6d1c7c3aebacf7db8dc.webp)

我们感受一下，当你看到这个文案，想要点击进去吗？在感受下看到极致柔顺，极致舒适，吸放湿效果这些词，你有什么样的感觉？


第二个是淘宝上一家店铺的首页广告：早秋灵动上新，一眼就爱上！感受下「灵动上新」「一眼就爱上」这些词，真的一眼就爱上了吗？


![img](https://pic4.zhimg.com/v2-f421daa8123b29e944ab651795b645da.webp)

第三个是某个房地产项目的软文标题：漏窗花影，曲水流觞，小镇中式合院示范区去风雅盛放。


漏窗花影、曲水流觞八个字确实挺美，也很有意境，风雅盛放也是这种调调，但是很难打动人心，好像跟我没什么关系。


市面上 80% 的文案都是这样的。虽然看上去文字考究，还故意弄得很高大上。但实际上很无感，这就是自嗨，没有对象感、代入感，更不用说引发共鸣。


具有这三感的文章是什么样的呢？如何写出对象感、代入感和共鸣感呢？下面我们一个一个来讲解。


**第一，对象感**
----------


所谓对象感，就是心中要装着说话的对象，说人话。如果不是正式发布会、媒体公关稿，使用的场景是线上转化，建议文案都尽量做到口语化，简单直白，这样更有亲和力，受众感知更好，更利于转化。


![img](https://pic3.zhimg.com/v2-9ae190064a6a2bf0b8e721de1dbe37f2.webp)

像「极致柔顺，吸放湿效果更强！」这样的语句，就是不说人话，如果换成下面这样「面料表侧棉成分，单穿舒适也时尚」、「透气性更出众，不易产生闷热感」，就会好很多。后者明显更有对象感。


![img](https://pic3.zhimg.com/v2-7edb08f51d0e507877519979f390d3f0.webp)

要增强对象感，你可以这么做：


第一， 想象你文案的对象，是个活生生的人，TA 就坐在你面前，你在跟对方面对面交谈。


文案的本质就是「人与人」的沟通，只有将沟通的对象当做一个活生生的人，去对话、去交流、去沟通，用对方一眼就能看懂的文字，才有可能打动 TA。


你所想的对象越具体，就越知道你的沟通方式，越容易产生现场感强的语言。


第二，像跟闺蜜或哥们说话一样写文案，营造「对话感」。


把自己当做目标对象的闺蜜或哥们，写文案时，要像跟闺蜜或哥们说话、聊天一样。


举个例子，同样推荐一款护肤品，文案 A 是这样写的：高浓度波色因，卓效抗氧，一瓶彰显，饱满高光肌。


这依然是不说人话，没有对象感的文案。


再看 B 文案：内含抗老黑科技波色因，含量超过 30%，用一瓶相当于普通的三瓶，让你看起来年轻 2 岁。


后者就像是跟朋友聊天对话的感觉，易于对方理解，也更容易打动对方。


第三，先说后写。


如果你一时半会还无法写出对象感，你可以先说后写，一边想象你的文案对象，一边用手机 APP「讯飞语记」把你想和对方说的话录下来，一边录一边就转化成了文字，这就成了文案的初稿，在这个基础上修改，就能很好地避免不说人话的问题。而且，用说的，也利于我们发散思维，带来更多灵感。


**第二，代入感**
----------


什么是代入感？是指用户看到你的文案会产生一种自己就是里面人物的感觉。吸引人的文案，一定是有代入感的。读者读到会觉得，这件事情我也正在经历，或曾经经历过，哪怕没有经历未来也有可能经历，或能够理解。


我们想要触动受众，就要努力和受众拉近距离，让他代入自己，感同身受，这是引发共鸣的基础。


例如 AIRism 内衣，主打透气性的这组广告文案，就很有代入感：


![img](https://pic4.zhimg.com/v2-d4c98be4d2c8926cd81789f91ca721ba.webp)

高温天通勤汗流浃背，办公楼冷气比人，刚出的汗，冷气一吹让人透心凉。人人都有过这种体会，特别难受，紧接着就是引导文案是：左滑帮她缓解汗湿。多数人应该会忍不住左滑一下，其实就是购买链接。如果你刚好正在经历这种情形，购买详情页的介绍让你觉得衣服超好又超值，大概率会购买。


![img](https://pic3.zhimg.com/v2-cb2560b5a9c92ef52bdd0a63a2b94472.webp)

同样的，还有这组：在几千行代码中寻 BUG 丝毫不慌，拥挤闷热的早高峰，令他「头秃」。左滑帮 Eric 环节闷热。


如果是每天挤早高峰的程序员，会特别有代入感，感觉说的就是自己。不过更有代入感和对象感的写法，是用第一人称「我」或第二人称「你」。目前这两版文案是第三人称。


例如步履不停的广告文案用的就是第二人称「你」：「你写 PPT 的时候，阿拉斯加的鳕鱼正在跃出水面；你研究报表的时候，白马雪山的金丝猴刚好爬上树尖；你挤进地铁的时候，西藏的山鹰一直盘旋在云端；你在会议中吵架的时候，尼泊尔的背包客端起酒杯在火堆旁。有一些穿高跟鞋走不到的路，有一些喷着香水闻不到的空气，有一些在写字楼里永远遇不见的人。出去走走才会发现，外面有不一样的世界，不一样的你。」


这段文案就非常具有代入感，有很强的画面感，特别能引起大部分朝九晚五的白领们的共鸣，一度被广泛引用甚至抄袭。


要写出代入感，AIRism 和步履不停的案例，正好代表了两个方法。


**第一，描述受众熟悉的场景。**


一个瞬间增强代入感的方法，就是不要告知，而要描述。描述具体的场景，让读者读到这些文字的时候，瞬间唤起自己曾经也有过相同或类似的经历的记忆，从而产生强烈的认同感。


所谓场景，就是生活中的一个个真实存在的、具体的画面。选取一个熟悉的生活场景，用简单、易懂的文字，三言两句勾唤起熟悉的场景记忆。


AIRism 的两组海报文案就是这样。


**第二，** **利用感官和细节写出画面感。**


上一段步履不停的广告就是一个非常具有画面感的代表。我们再来看一个案例，这是芝华士父亲节的广告文案。


![img](https://pic3.zhimg.com/v2-7caf38fdcd824367674820eace8978ba.webp)

![img](https://pic3.zhimg.com/v2-cea1f5a4af9d69938493ae3fec3b1460.webp)

大家可以感受一下，这是一段非常经典的文案，至今都被文案圈作为经典范本。里面有大量的描绘和父亲的各种细节，有很多画面感的东西。


在文字表达中，我们能调动的感官感受越多，所描述的事物就越具体可感，也越有代入感。当我们尽可能调用到视觉、听觉、触觉，甚至味觉来进行写作，有细节描述，写出画面感，就更容易让受众产生代入感。


注意不要用一些看不懂的专业术语，也尽量不要用复杂的长句，多用短句。文案越贴近生活，越真实，越有代入感，也越走心。


**第三，共鸣感**
----------


共鸣是交流应该追求的更高境界，也是文案的终极目的。


什么是共鸣？心理学的解释是：由别人的某种情绪或思想感情，引起相同的情绪和思想感情。物理学的解释是：物体因共振而发声。我们物理课上都学过，两个频率相同的音叉相互靠近，其中一个振动发声时，另一个也会发声。放在文案写作上，共鸣的意思应该是这两种解释的合体。我们不但希望引发共同情感、情绪还不够，还希望用户能发声，看完之后愿意去转发分享，让更多的人看见。


对受众而言，如果看到一篇文案中有他生活的映射，比如场景或故事，或者引发情感共鸣的金句，让他产生「的确如此」「说得太多了」「你说的就是我」的感觉，他就会特别有共鸣，因为你说出他感觉得到却说不出来的那种感受，他就会对你产生强烈的认同感，进而喜欢你这个人以及你推荐的产品


这里分享一个制造情感共鸣的方法，就是替人发声。也就是制造身份认同，替某一特定人群发声。帮用户说出他们想说的那句话。


去年毕业季江小白推出「做大人久了」主题海报，致我们终将逝去的青春，对于毕业已久进入社会工作的人，特别容易引起共鸣，引发广泛传播。


![img](https://pic2.zhimg.com/v2-a23a258845b4440ed6bb4fe30a2277a1.webp)

「一台游戏机、一次通关


就让自己成为最受欢迎的小孩


即使敲出几万字稿件、上百种方案


也未必获得一份肯定


做大人久了顶不住的」


这则文案没有华丽的文字，就是朴实的真情流露，这种自嘲、怀旧、自我同情等万千感慨引发众多共鸣。


再比如：


![img](https://pic4.zhimg.com/v2-c8ba7ab15e3d43ceed283bac89c99a85.webp)

「没能集齐 108 张卡片


集齐了互换卡片的三五兄弟


互换名片的所谓『朋友』


交换的不过是卡片上的头衔


做大人久了三五老友」


这则文案唤起人们对于大学好友的情感，对比社会上真心朋友越来越少的慨叹，引发很多人同感。


其实写出共鸣感的文案并不难，关键是要在心理装着你说话的对象，说他们想听的话，替他们说出心声。通常情况下，如果不是专业文案工作者，我们面向的往往都是和我们差不多的人，一个好用的方法就是代入你自己，说出你的感受和心声，就是一篇走心的文案。


**总结**
------


我们总结一下：首先我们讲了什么是好文案，好文案一定是打动人心的，没有写作基础和文采，并不妨碍我们写出好文案。关键是把握 3 感：


一，对象感，写文案的时候心里要装着文案的对象，想象你是和对方面对面沟通聊天，说人话，像跟闺蜜或哥们说话一样。


二，代入感，让用户在看文案时产生一种自己就是里面人物的感觉，有两种方法可可以增强代入感。第一，场景化描述，描写受众熟悉的场景，唤起感同身受的体验。第二，利用感官和细节写出画面感，让人身临其境或增强感知。


三，共鸣感，换位思考，代入自己，替受众说出他们想说但没有说、说不出的话。用替人发声的方式达到情感共鸣和强烈认同，从而促使人产生行动的效果。


**作业**
------


为了帮助你学习致用，我给大家留了一个作业：假如你想给朋友推荐最近看到的一本好书，你觉得特别适合他读，你要怎么说才能打动他并且让他迫不及待要去读呢？试着自己写一段 200 字左右的文案。
