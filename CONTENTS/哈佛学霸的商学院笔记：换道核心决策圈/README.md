## 简介
从普通人思维到哈佛精英。


从普通人思维到哈佛精英需要几步？


「方案能完成吗？」，其实是「方案能快速复制吗？」；「项目能赚多少钱？」，其实可以换成「项目的投入产出比是多少？」；「产品优势是什么」，改为「产品壁垒是什么」会更恰当。


**其实，商业世界的核心是一套语言体系。**


如果你不会，你将无法进入这个社会的获利圈和决策圈。


本专栏特邀哈佛学长赵海柏老师基于哈佛商学院的授课方法，还原真实的哈佛课堂，零距离分享顶尖职场精英们的学习笔记。


蒂芙尼珠宝的客户选择 、海底捞的师徒制、巴菲特的投资策略、海尔的内部革新……二十五个经典商业案例，构建包括管理、创业、战略、投资、政策系统在内的商业知识体系。


还有终身获益的人生管理法则，通过内观冥想带你获得心理安全，有效平衡工作与生活。
