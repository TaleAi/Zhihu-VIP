## 简介
不管你是什么身份，从事什么样的工作，都免不了跟财务和数据打交道。懂点儿财务常识，不仅可以帮我们解决工作中的实际问题，还能帮助我们更顺畅地获得职场进阶。


这个专栏，我们邀请了跨国公司全球首席财务官，浙江大学客座教授，畅销书《从总账到总监：CFO 的财务笔记》作者钱自严。


作为一个资深财务人，他将通过大量实战案例，配合各类原创图表，深入浅出地科普财务常识，帮助我们解决工作、学习中与财务相关的实际问题。


并且从财务的角度看职场，帮助我们规划职业道路，获得职场晋升：


* 如何通过财报信息，准备一场关键面试？
* 工作中如何将财务数据业务化，找到绩效管理的关键点？
* 怎样用证券化的方式管理自己的经历，提升自己的职场价值？
* 人工智能都会写诗的时代，财务这门学科还会有价值吗？

······


不管你是不是专业的财务从业人员，都可以从中获得认知提升和职场进阶的有益参考。


愿你有所收获。
