## 简介
50 部国学经典，七大核心构成。北京大学哲学系专家黄向军老师结合大量实际生活案例，轻松幽默讲述国学经典，300 节专业音频解读和图文展示，3000 分钟核心理念和知识点，覆盖 50 本好书精髓，深入浅出地带你遨游于经典的海洋，让你收获一套完整的知识体系。


作为中国人，你有多久没有读过一本国学经典了?


主持人董卿熟读《诗经》、《楚辞》，不断提升文化涵养，顺利完成生子后的自我成长和突破，创办了《朗读者》；


演员胡歌，在经历车祸毁容时，通过阅读大量经典，孔子、孟子、老子、庄子……都看了个遍，重新找回生活的意义；


企业家马云总会随身携带一本《道德经》，他说：「2000 年来知识大爆炸，但智慧还是那些智慧，古圣人完全能够解读今天的人心。」


大道至简，国学是人类对大自然最简单质朴的认知和总结。学习国学不仅身心得到休整，思考得到释放，更能达到高层次的人生状态。


黄向军老师从事中国传统文化研究近 20 年，他结合大量实际生活案例，轻松幽默的讲述 50 部国学经典，包含儒家理念、道家精神、禅宗智慧、诗词歌赋等七大核心，帮助你打开思考的大门，破解属于自己的生命策略。
