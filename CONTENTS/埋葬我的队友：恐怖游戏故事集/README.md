# 埋葬我的队友：恐怖游戏故事集

## 简介
喂，想和我一起玩游戏吗？捉迷藏、抢椅子、纸飞机、翻花绳、跳房子，十个熟悉的童年游戏，十种不同的惊异玩法……但请注意！每种游戏都有一个小小的陷阱，如果不小心输了的话，你的命，就是我的啦~


**- 捉迷藏 -**


0 点前，关掉灯，点上一只蜡烛，在蜡烛前摆放几样玩具，然后播放自己录好的录音。


「心慌慌，捉迷藏，人躲鬼，鬼进房。」


「躲好我了，来找我了，抓到我了，带走我了。」


「1、2、3……10，躲好了吗？我来找你了！」


「找不到我，赢家吃糖。」


最最最重要的一点：在录音说出「赢家吃糖」之前，不管发生什么，


一定一定，不要走出躲藏的地方！！！


  



  



**- 猜拳 -**


你孤独吗？


一个人的晚上，是否连最简单的猜拳游戏都找不到人一起玩？


没关系，其实猜拳可以一个人玩的。


将光源——手电也好，手机也罢，平放在镜子下方，让光线从下至上打到你的脸上，看着镜子中的自己，默念十遍：


「谁像我？谁像你？谁是我？谁是你？你像我，我是你。」


然后与镜子里的影子玩剪刀石头布的游戏。


只要赢了，就能获得朋友哦。


至于输了的话……没关系，只要变得和它不像就行啦~


  



  



**- 抢椅子 -**


你玩过抢椅子吗？


这是个国民度很高的聚会必备游戏，只要随意改变其中一两个关键元素，这个本来欢乐的游戏就会变得……诡异起来。


比如，夜深人静的时候，在地上摆放死者的遗物，并围绕它点燃一圈蜡烛，在蜡烛中心摆放一圈椅子。


蜡烛与椅子数量一致，而玩家则比椅子多一人。


玩家围绕椅子转圈，每当蜡烛熄灭一根，便抢一次椅子，没抢到的人淘汰退到墙角。


当蜡烛全部熄灭，最后的赢家坐在椅子上，就能看到去世的死者……


唯一要注意的是，千万别拿错遗物，否则你见到的，可能会是另一种东西呢~


  



  



恐怖游戏，今日开玩，


你，敢来吗？
