## 简介
皇帝贵为九五之尊，难道也会被人戴绿帽子吗？你还别说，这还真有。有那么一群后妃，她们耐不住空闺寂寞，竟挑战皇权，给皇帝戴上了绿帽子。


都说古代皇帝三宫六院，妻妾成群。那后宫的女人，就只能作为男人的附属，这在当时被当作天经地义的事。那么皇帝贵为九五之尊，难道也会被人戴绿帽子吗？


你还别说，这还真有。有那么一群后妃，她们耐不住空闺寂寞，竟挑战皇权，给皇帝戴上了绿帽子。


那么是谁有这样胆量呢?


就比如说北齐武成帝高湛皇后，胡氏，出身名门，长相明艳，但她丈夫不爱他，经常和自己嫂子搞在一起，胡氏也不甘示弱，这绿帽子就成批地给高湛戴了上去！


还有如花般美貌的尹德妃，到唐高祖晚年才备受宠爱，正值大好年华，自然看不上这个老男人，不光要给他戴绿帽，还要毒杀他的儿子！


一代女皇武则天就更不用说了，有才华有手段，把老子儿子都玩得团团转。


那么这些如此「勇敢」的女子她们的结局如何？那些被「绿」了的皇帝又作何反应呢？这其中详细，请听洪宇老师为你慢慢说来。
