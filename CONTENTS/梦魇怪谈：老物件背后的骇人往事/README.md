## 简介
我叫秦海峰，是潘家园的一个文玩店老板，文玩店叫八宝园，取义「八方来财」。别看我店面不大，但我经常走南闯北地进货，踏遍了祖国的大好河山，也因此碰见不少圈子里的故事——不乏有灵异诡事，请君小酌一杯，且听我娓娓道来。


**古都疑云**


世代居住在河南开封市的人都知道，开封地下埋有八朝古都，但是没有门径可以打开。说这话的都该烂舌头，明明在黄河泉眼下有一个入口——然后，市里来了领导，从黄河泉眼下打捞出一具不腐的古尸……


**铁将军冢**


故事发生在湖北省鹤峰市北部，一片荒野之中，有一个无名冢。因为道路施工，需要开发这片荒地，经手的开发商准备挪树，顺便把无名冢也给挪了。附近的土家人不让，说是无名冢有灵，能防灾害。动工那天，突然下起了暴雨，不得不停工，土家人说无名冢显灵了。


**古井悬尸**


老家的荒宅中，有一口古井，这口古井中曾死过主角的祖奶奶，因为不满八国联军进京受辱，所以跳井自杀。原本很正气的故事，后来成了后世人的谈资，被越传越邪乎。
