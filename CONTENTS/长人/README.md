## 简介
人为恶，鬼作祟。


张老三家生了一个长人，刚出生就死了。


张老三试图用邪术让死婴活过来。


我调查此事发现——埋葬死婴的坟墓空荡荡的，周围的坟墓也被掏空。


接着村里便出现了在地里游走的吃人的怪物。


我本以为这一切都是张老三做的孽。


尤其是当张老三的六个兄弟全遭到残害时……


可我顺着追查下去时却发现，一切似乎跟我那软弱怯懦的父亲有关。
