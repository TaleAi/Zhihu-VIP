## 15.天下武功唯快不破：提速 N 倍的快捷键
高手们给人的第一感觉就是快如闪的速度和不假思考的精准操作，键盘哒哒作响，屏幕来回闪动，观者眼花缭乱！而小白们总是会伸长脖子，眼光在屏幕上来回寻找，鼠标箭头也会随着眼光游离不定。


有人统计过，好莱坞大片里的黑客和电脑高手从来都是键盘党，甭管什么操作，都是啪啪啪敲几下键盘来搞定。为什么会这么设定呢？


作为电脑的输入设备，键盘和鼠标各有不同的定位。键盘讲究的是精确，追求的是速度。鼠标则是个定位设备，通过移动光标在屏幕范围内去寻找所要的目标范围。


大多数常用软件的菜单都有快捷键，用鼠标点击能完成的事儿都能通过几个键迅速搞定。简单地说，会不会用快捷键，用得熟不熟，就是区分高手和小白的标志。


有人会说，快捷键那么多都记住多费脑子啊！难不成和考试一样要把它们都背下来?其实不然！快捷键虽然多，但是掌握了规律并不难记，很多都是和操作过程的英文缩写相关，而且最重要的是一旦你经常使用形成了肌肉记忆，就再也不会忘记。就像打字、骑车、游泳、驾驶汽车一样，成为一种本能反应。你的大脑会自动完成，把资源和关注点分配给更重要的事，所以根本不是费脑子而节省你的脑力资源。


如果你学习了前面的章节知道如何利用搜索引擎，得到任何软件的快捷键列表都很容易，但很快就会发现快捷键太多了。办公软件系列、图形处理软件，任何功能齐全一点的大型软件都有几百个甚至更多的快捷键。如果记不住，快捷键就根本快不起来，所以我们要用正确的方法来记忆和运用。


首先你要找到最容易记忆，运用最频繁的快捷键，然后在实际操作中通过反复运用形成肌肉记忆 ，然后再从这些键出发扩大范围，去关联更多的快捷键。


我们先来看看一些能够大幅提高效率的通用快捷键，需要说明的是这些快捷键都是基于 windows 操作系统的软件。


1.掌控一切的 Ctrl 键


我们先来介绍一下 Ctrl 键，它位于键盘的左下角，来自英文单词「control」——控制、管理、支配，好霸气的名字！通过这个按键的组合，你确实能够控制很多东西，看看最常用的几个：


![img](https://pic2.zhimg.com/v2-2fbe3e339945afe4a93cdf4fde4682e3.webp)

这几个按键几乎人每个人都知道，包括操作系统在内的所有软件中，基本都是一样的功能。不过很少有人思考过，为什么要安排这几个按键，仅仅是因为英文首字母？有人说 Ctrl+V 的 V 是英文单词「viscidity」粘性的首字母，这个单词对非英语专业的人有点陌生。更容易理解的思路是：粘贴需要先复制，当然放在复制旁边最方便啊！


最常用的操作分配给最方便的键是有道理的。其他几个组合按键 Ctrl+N 新建、Ctrl+O 打开、Ctrl+P 打印，就不怎么常用。正巧它们分布在键盘上右手按键范围下，必须放开拿着鼠标的右手才能按到。


2.替换鼠标点击的 Alt 键


还有一个按键也很重要，Alt 键来自英文「Alternate」——交换、替换，它经常用来打开软件菜单。


![img](https://pic2.zhimg.com/v2-244dff0832fdec36488d80c26ca21cdb.webp)

通常情况下，在软件中按一下 Alt 键，菜单栏就会高亮显示，这时再敲击对应菜单括号中的英文字母，就会弹出对应的子菜单。同样，敲击这些子菜单括号中的英文字母，也会打开对应的下级子菜单或功能窗口。


例如打开上图中的「曝光度」，你可以用鼠标去慢慢点选，但是如果用键盘操作，只需要称按 Alt+I 再接着敲 J 和 E，就能打开对应功能窗口，响应时间不会超过一秒。


像 Photoshop 这种功能比较多的软件，给一些常用功能都设置了快捷键，如图中所示。这些快捷键有的是 Ctrl 键和字母的组合，有些是 Ctrl 键和 Alt 键一起再和字母组合，熟练之后可以节省大量的时间。


3.跳出窗口的 Win 键


在 Ctrl 键和 Alt 键之间有一按键很特别，样子像个窗户，这就是大名鼎鼎的 Win 键。一般人很容易忽略它，其实用它可以实现很多快捷操作。


![img](https://pic4.zhimg.com/v2-85e5e1059b10f0ea1fa7161bffca8cda.webp)

上表对于不同版本的 Windows 略有差别，不过多数情况下差不多。Win10 下的 Win 键还有更多的组合。比如 Win +↑(上方向键) 使当前使用中的窗口最大化，Win +↓ (下方向键)使当前使用中的最大化窗口正常显示。Win + ←/→(左/右方向键) 使当前使用中的窗口贴向屏幕左/右侧，占用 50% 显示器的面积。


如果需要开多窗口对比参照，这个组合操作非常人性化，节省很多用鼠标拖拽的时间。


上面这三个按键可以说是快捷键的基本，尤其是 Ctrl 键和 Alt 键，几乎所有的软件都用它位来组合。而且不同的软件之间也有一些关联，大家可以找找其中的内在规律。


下面我们来看看使用范围非常广泛的办公有哪些快捷键需要掌握。


* 办公软件常用快捷键

上面只是几个最常用的快捷键及组合，还有一些非常个性化的用法需要你自己去挖掘。用好搜索引擎以及软件的帮助文件，相信你会大有所获。


![img](https://pic2.zhimg.com/v2-5826fb4bd636c2adb2e370224e2556de.webp)

还是那个道理，先掌握易记的，高频使用的快捷键，这些从投入学习的时间和实用价值上来说「性价比」最高。


接下来再根据你的工作性质来确定下一步的学习目标。比如你工作中用 Excel 特别多，那么抽点时间，搜一搜「Excel 快捷键」，把结果先复制到文档，然后打开电子表格一个个试用一下。在试用的过程中回想自己工作，哪些是最常使用的，哪些是能够提高 N 倍效率的，标红做出标记，并按重要性和频率进行排序，保存好这份文档。开始工作的时候，打开这份文档，一边用一边看，随时调整，随时修改。用不了多久，这些快捷键就会在你脑海中牢牢生根了。同样的方法也适用于 Word/PPT，以及其他软件。

有的人在网上把常用快捷键做了壁纸，有人在网上售卖专用的键盘快捷键贴纸和薄膜，这些都是帮助记忆的好方法，可以自己试试效果。


快捷键不只是能够让你省下大量喝咖啡伸伸懒腰的时间，还能够让你在工作中把关注力集中到你要解决的问题上，操作这种事情交给肌肉记忆好了。除此之外还能够树立你的专业职场形象，就像文章开头描述的一样，一个拿着鼠标到处找目标点击的形象是无法和电脑高手划上等号的。下次领导到你电脑跟前和你沟通工作要你调用资料的时候，用快捷键操作惊艳一下他吧！
