## 简介
如何体现一个人在职场上的价值，最直观最根本的就是工作效率，职场小白工作效率低下，没有思路，不得章法，结果是常常被安排做一些技术含量不高的简单工作。而职场精英，则能够迅速分析工作目标，绕过无效工作，直指问题核心，领先于竞争者。


作为职场新人，你肯定想：


快速了解各种效率工具和办公软件，以提高工作效率，避免经常加班；


做出上档次的 PPT、漂亮的文档和表格，让领导和同事对你另眼相看；


拥有职场思维方式，知道工作该如何分解，分解后又该如何落实跟进；


懂得如何利用时间来提升自我，开拓更多市场；


……


本专栏通过职场思维、效率工具、办公软件、时间管理以及构建知识体系这五个方面，以点带面，在浩如烟海的职场知识中挑选出职场新人最急需掌握的问题进行讲解，帮助刚入职场的新人们学会使用合适的工具软件来提升工作效率、减少重复劳动，学会运用时间管理和知识管理的方法来掌控自己的工作和生活，实现自我快速成长。
