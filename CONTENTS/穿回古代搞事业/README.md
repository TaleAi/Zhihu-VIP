## 简介
一觉醒来穿越到了古代，说真的宫斗什么的太没劲了，搞事业多有意思呀。古代的商业帝国，我来了！


女刑警掉到青楼淸倌沐婉儿身上，穿越后的女主刚苏醒就发现有嫖客想对她用强。


沐婉儿不但把事主遣返回府游街示众，还对广大民众进行了普法教育，同时转头就把自己所在的青楼进行了合同制改造，对姑娘们进行职业再教育，将其打造成为集文化娱乐、化妆品研发和餐饮行业的连锁品牌。


芙蓉楼火了，日进斗金。沐婉儿是护国将军之女，亲娘当年救了将军一命，珠胎暗结，担心名声有损，便将她抛弃在青楼门口。现在却看中了这块肥肉，大张旗鼓想要认回她，其实是想用沐婉儿的财产倒贴浪荡子弟弟……
