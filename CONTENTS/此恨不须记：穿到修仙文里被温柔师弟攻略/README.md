# 此恨不须记：穿到修仙文里被温柔师弟攻略

### 状态：已完结，共 16 节

### 存档日期：2022-06-03

## 简介
如果穿到修仙文里，成了一个试图勾引男主的恶毒女配，怎么办？ 


一朝穿成修仙文里的恶毒女配，原主勾引男主不成功，阴差阳错搞成了师弟。


她原本打算春风一度便作罢，谁知师弟温柔又深情，如今她想轰轰烈烈爱一场。


可师弟似乎也不简单呐。


七百年前被镇压的魔尊即将出世，他为何跟师弟有着一模一样的面容？


师弟本是降妖伏魔的修士，为何一夕疯魔，斩杀同伴无数，成为正道众矢之的？


不按套路出牌的剧情谜团重重，她这个失去金手指的女配，该如何挽救心爱之人？


眼看着正邪大战即将爆发，她又该何去何从？


她不过想简简单单谈场恋爱，怎么就这么难？！


等一切尘埃落定，蓦然回首，却发现，这个故事本没有主角，身处其中之人，各有各的爱恨辗转，有人遗憾一生，有人挥别过去。


梨花又雪，一去经年，只要能重逢，过去的绵绵长恨何必铭记。
