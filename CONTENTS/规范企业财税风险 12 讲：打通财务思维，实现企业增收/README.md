## 简介
本课程由包颖贤老师主讲，教你如何打开财务思维，实现企业增收。理论讲解+案例解析+实战演练，三位一体授课模式。财务平衡三角、金三系统等财务工具，帮助提升税务利润。帮你 0 基础也能打开财税任通二脉。


 如果把经营企业比作驾驶一架飞机，那么，财务就是我们机舱内的仪表盘。 


作为「飞机」的机长，老板们在驾驶飞机的过程中，有没有遇到以下问题： 


面对新的财政政策，看不懂、不知如何应用和落地。 


财务报表看不懂，无法从中找出企业经营管理的问题和风险。 


对企业的财务人员不满意，不知如何运用财务手段降低企业财税风险。 


税务稽查找上门,公司总陷入想办法补窟窿的被动局面。 


遇到上述情况，作为老板应该怎么办？ 


可能你还不知道：在整个企业中，老板才是真正的财务总监！ 


只有老板对公司的财务情况了如指掌，才能带领企业朝着正确的航线去航行！ 


本课程由包颖贤老师主讲，理论讲解+案例解析+实战演练，三位一体授课模式。 


财务平衡三角、金三系统等财务工具，帮助提升税务利润。 


引用大量案例，老师语言风趣幽默，0 基础学起来也非常轻松。 


5 大模块+12 个关键问题，从认知到实践，帮你打通财务思维，实现企业增收！
