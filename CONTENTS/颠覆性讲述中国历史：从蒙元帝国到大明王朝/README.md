## 简介
揭开历史谜团，呈现中国文化基因。


元朝堪称中国历史上武力值最高的朝代，在亚欧大陆上所向披靡，却在短短几十年后分崩离析。


究其缘由，是汉化不彻底，还是没有完整的制度朝纲，抑或是无节制收税、大规模赏赐、大规模印钞？


最终导致社会矛盾全面爆发，虽有贤相「脱脱」，却也不能力挽狂澜。


历史由此进入大明王朝。


明朝三百年，有过洪武之治、永乐盛世、仁宣之治，甚至隆庆新政、万历中兴，似乎辉煌过，但万历怠政、魏宦与党争、海禁与倭寇、东厂与西卫，加之天灾、内乱、外患，最终导致明王朝在黑暗中轰然崩塌。


明朝给后人留下了太多问号：朱元璋为什么杀功臣、废宰相、传位朱允炆？朱棣算千古一帝还是残忍君王？郑和下西洋的真实目的是什么……


本专栏致力于解答您心中关于元代与明代历史的种种问题，探寻历史真相，以全球视野观照中华历史，横向比较中外人物事件，生动呈现中国文化基因及其独特的发展轨迹。
