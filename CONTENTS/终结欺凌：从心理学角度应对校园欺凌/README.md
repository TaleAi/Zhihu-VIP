## 简介
面对校园欺凌的孩子往往会产生恐惧心理，每当回想起这段经历，即使多年后也会痛彻心扉，进而影响一生的心理成长。本专栏帮助你认识欺凌，直面欺凌，教你正确预防和处理欺凌行为的发生。


孩子在校园、在社会被欺凌，家长往往既愤怒又悲伤，无论是打在孩子身上的每一拳还是在网络上对孩子的恶意中伤与攻击，都会成为家长心头上拂之不去的阴云。


家长无法对孩子做到 24 小时的实时保护，又找不到更好的方法来制止孩子在校或在网络上受到欺凌。


而作为老师或教育工作，又该如何识别和防范那些发生在校园里隐蔽的欺凌行为，如何安抚受欺凌的学生，及时制止欺凌者，又该如何安抚家长的情绪，处理家长的投诉，以及营造一个安全的校园环境？ 


本专栏心理学专家从专业心理学的角度，分析了欺凌者、被欺凌者和旁观者的心理特点，提供了切实有效的应对校园欺凌的策略，从而引导全社会营造对欺凌零容忍的文化，让孩子在安全的环境下快乐成长。
