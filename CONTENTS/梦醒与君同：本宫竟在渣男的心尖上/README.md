# 梦醒与君同：本宫竟在渣男的心尖上

## 简介
不受宠乌鸦嘴公主 vs. 面冷腹黑渣驸马，是命中注定，还是前世纠葛？


我自幼起就是个不受宠的公主，娘亲走得早，野爹把我当棋下，偏生京中权势最为滔天的宰相秦珏似是待我有些不同。


他虽生了张好面容，行事却狠辣无情，瞧着就是冷心之人。更何况，我在「梦」中看到这男人太渣，我的下场太惨。


这情缘，借我一万个狗胆我也不敢牵。


只要本宫没心，他就不能虐我，想绿本宫，本宫先绿得他脑袋冒光！


可我左等右等，秦珏却和传闻及梦里完全不同。


敬我爱我宠我不说，更是从不让别的女人近身。


我处处讨他嫌，他依旧深情款款：微臣甘做公主裙下之臣。


我送他俩字——无耻。


与其跟渣男谈恋爱，不如专心搞事业，我一步一步走到长公主之位，不再任人摆布。


而助我走到这步的秦珏与我道：微臣只愿公主事事顺心。


我实在想不通，本宫怎么就住在了渣男的心尖上。


这份孽缘，我突然不知道该不该断了……
