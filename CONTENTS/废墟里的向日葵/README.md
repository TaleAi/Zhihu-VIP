**- 米粒和珍珠 -**


周珍珠一出生就被重男轻女的奶奶丢了，养父母一开始对她不错，有了亲生儿子后，也遗弃了她。


她被好心的爸妈捡回去，当成亲生女儿养。


送她学画画，她获奖后，亲生父母找上门，想将她带回。


珍珠得知她被丢了不久，生父母就生下了妹妹，她拒绝回去，还是跟爸妈在一起。


但是为了方便她读书，她的户口被迁回了生父母家。


在哥哥周榆的帮助下，她顺利考上高中，大学考上了中央美院，成了一个大师的关门弟子。


她谈了一个渣男男友，周榆连夜驱车 1500 公里，接她回家。


而她在这之后，也知道哥哥这么多年都不结婚的原因……


后来，他们资助了很多贫困孩子，希望他们也能从平凡的米粒，变为珍贵的珍珠。
