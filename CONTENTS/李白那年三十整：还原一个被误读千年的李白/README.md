# 李白那年三十整：还原一个被误读千年的李白

### 作者：周白之白

### 状态：已完结，共 24 节

### 存档日期：2022-05-24

## 简介
让我们从李白三十岁那年出发，走进李白的一生。以李白为入口，进入唐诗的世界、了解盛唐的社会。


他渴望成功又不愿低头，求职期望过高频频遭拒，获得工作会沾沾自喜，事业遇挫时借酒浇愁，结交姑娘后会写诗吹嘘，和朋友醉酒后会吹牛口嗨……


或许，「诗仙」李白也只是一个不想被困于原地、一如你我的普通人。


在唐朝开元盛世下，李白看似风光潇洒、人人称赞，实则因得不到重用而无比苦闷，到处访道求仙来排解自己壮志难酬的苦闷。


他的潇洒不羁是真的，他渴望走入仕途封侯拜相是真的，他内心的苦闷惆怅也是真的，他就是这么一个处于各种矛盾心理状态中的大诗人、大才子。


本专栏将为你还原一个被误读了千年，诗仙光环下，那个真实得你不太想承认的李白。


文中还附有李白有关历代字画配图，介绍了七十余首古诗的创作背景和心境，帮助读者理解李白的诗文和精神气质。


你会发现，李白受欢迎的背后，是我们想做而做不到的自己。
