## 简介

「还乡」：五年了，我终于回到这个小城，算是衣锦还乡。五年前我出走时，未婚夫和从小到大的好闺蜜勾搭，并且还是在我为未婚夫流产后才得知。这次回来，闺蜜成了曾经的我，渣男又勾搭上一个白富美，把已经为他流产三次的她抛弃。闺蜜被家人嫌弃，躺在医院无人问津，我冷眼旁观着这一切，掐灭了手中的烟，走上前。好戏，这就要上场了。


「风波」：中风多年的母亲不幸被车撞死，悲痛之余我也接受了这个现实，觉得对她和父亲都是解脱。好强的母亲病后常年卧床，饱受折磨，性格越来越暴躁，父亲精心伺候，任劳任怨。我感动于老一辈人深沉的爱，无意中却发现他与前来帮忙葬礼的桂芬婶早有私情。我虽震惊，却也劝自己谅解他，没想到奇怪的事情接二连三。在读法学研究生的我感觉敏锐，顺藤摸瓜，发现惊人真相，母亲被车撞死的地方居然距我家两百多公里。一个中风多年的病人怎么会跑那么远？更多蛛丝马迹露了出来，我穷追不舍，证据确凿，父亲跪在了我的面前。
