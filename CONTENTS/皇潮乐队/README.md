皇上每日宵旰勤政，为国操劳，但他心里却一直有一个音乐梦想。


为此，皇上特意搜罗三位音乐奇才，分别是：白鹿馆扫地小妹、蛮族刺客马头琴手以及钦天监玄学鼓手，在皇宫组建了地下乐队。


从此，皇上过上了白天当皇帝、晚上当艺术总监的繁忙生活，为出道鞠躬尽瘁，为队友焦头烂额，与太后斗智斗勇……


另一边，稳坐大唐第一唱跳爱豆的隆禄王爷，发现了他那全科制霸、宫斗满级的皇兄有向乐坛进军的征兆……宿敌见面，分外眼红，掀起乐坛一场腥风血雨，鸡飞狗跳的皇宫日常拉开序幕！


后宫、宫斗、脑洞、爱情、狗血、爱豆、娱乐圈、唱跳、蹴鞠、历史……《皇潮乐队》都有，魔性到底，笑到打鸣，要你好看！
