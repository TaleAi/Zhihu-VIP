## 简介
这个世界是残酷的：狼吃羊，羊吃草，草侵占着土地。
李家臣早就知道了。


李家臣参加初中同学会。聚会当天，他亲眼看着女神的头爆炸了，原来没有在最后时间赶上地铁的乘客，都会神秘死亡。


地铁开动起来，李家臣和其他人收到一条短信：


**欢迎参加闯关图游戏。请各位在 5 分钟内，使用你们无神的眼睛找到二维码，并进行扫描，扫描完成即可通关游戏。**


大家不论怎么找，都找不到车厢内的二维码。


最后李家臣破解了谜题，并且在酒九的帮助下，完成了扫描。


结果一车人，完成扫描的只有 7 人。


然而，突如其来的残酷死亡游戏并没有结束，很快第二个游戏接踵而至。


到底是什么人在组织着游戏？这一切的目的又是什么？


李家臣和酒九一边努力闯关，活过死亡游戏，一边寻找幕后的真相......
