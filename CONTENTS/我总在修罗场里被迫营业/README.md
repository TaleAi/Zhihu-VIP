## 简介
一心只想苟命的女主总是莫名卷入修罗场……


重生、穿越、穿书……活在已知剧情里，女主她明明只想规避风险，好好苟命，却总是莫名卷入修罗场，纠缠在男主男配甚至不知名男 N 号之间，陷入未知的危险。


  



**折腰**


上辈子，我被暗恋的师尊一箭穿心。


重活一世，本想离他远远的，可一向冷淡的师尊却执意要娶我。


为苟命，我只能暂且答应嫁给他，没承想大婚当日，我又被魔尊劫走。


也因此发现，魔尊竟然跟师尊共享了同一张脸。


纠缠在二人之间，我渐渐发现，不管前世还是今生，一切似乎都没那么简单······


  



**太子他失忆后火葬场了**


我是太子妃，但太子并不爱我，在白月光回来之后，我们便商量和离。


没想到他却突然失忆了，还将我当成了爱人。


太子：「说爱我，命都给你。」


我冷笑一声：「我正巧想要你的命。」
