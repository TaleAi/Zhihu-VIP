## 简介
这是一个普通又不普通的人经历的一系列匪夷所思事件集。在鬼事奇闻中，看见不一样的人世间。


林为溪，一个普通人。但他却有和一般人不一样的地方。


从旧时生活的小山村，到进入大学进入工作，他遇见了一系列普通人所经历不到的「鬼事」：


「长生活葬阵」：我读历史系的时候，遇见一个很奇怪的教授。某天去找他签一个非常平常的单子。单子拿来后，我发现他签的不是名字，而是一句咒语。从这句签名开始，我的校园生活陷入了突如其来的诡异之中，先是宿舍师兄突然惨死，后来我去找这位教授，他却突然说已经等了我三十年，可我明明才二十岁。他说，我就是那第十八个人……经过探查，我才明白，这一系列怪事，其实和一个古老的邪术相关。其前身，就是消失已久的殉葬……


「表嫂小秋的生死局」：表哥在 25 岁那年终于结婚了。25 岁在村里已经是大龄青年，他终于结婚而且还找的是个漂亮老婆，全家人都为他高兴。尤其我还看到，那位叫小秋的表嫂看我哥的眼神充满爱意，羡煞旁人。可是没想到，一向对小辈慈爱的三姥爷看到小秋却变了脸色，说这女人不能进门。大家觉得他老糊涂煞风景，表哥表嫂按时完婚了。没想到仅仅半年后，就出事了……


「我们还会再见」：高三那年，我喜欢上了班里的一个女生。她是从其他镇上的学校转过来的。有人建议我去镇上据说很灵的塔寺拜一拜，可不知为何，塔却莫名其妙的倒了。后来，我的愿望显灵了，女生在我的同学录上写了一句话：我们还会再见。但是高考后，她却人间蒸发了……


……


以玄学看世间诸事，以鬼事写人情几何。


跟着主人公林为溪，看见不一样的人世间。
