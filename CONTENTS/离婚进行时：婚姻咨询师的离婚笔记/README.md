## 简介
一个正在离婚的婚姻咨询师，接手各种情感故事，一边尝试治愈他人，一边努力分辨人世间普通的你我，力图在两性关系中窥得婚姻与爱情的真相。
她用第二重身份——作家，写下自己的经历、思考和见闻，分享关于婚姻的体会。


人们普遍愿意追逐美好、温暖的爱情故事，一朝缔结，唯求白首不分离。而婚姻的本质复杂曲折，两个不同的人生交汇，携卷着各自的人生经历、童年创伤、原生家庭……等各种错综复杂的因素，要想白头到老，着实道路艰难。


80% 的婚姻，其实都存在这样的日常：他们每天都在思考，这个婚到底是离，还是不离？没有在深夜思考过离婚的婚姻，不足以谈人生。
