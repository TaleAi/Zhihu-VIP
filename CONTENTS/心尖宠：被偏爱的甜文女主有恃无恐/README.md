## 简介
男主无限宠溺 ing，甜到打滚，美滋滋……


**1**


被六王爷推入水底后，我死了。


但我也活了，成了丞相大人的宠妻。


不能让丞相发现这具身体换了个人。


我小心翼翼，还是被周围人看出了反常。


但丞相没有杀掉我这个冒牌货。


相反，他看我的眼神更加温润。


他说：「我最近看不惯六王府，打算想个法子玩玩他们，夫人有什么好的建议吗？」


**2**


我，一个老实干游戏主播的社畜，被一个弟弟盯上了，还是我的青梅竹马。


「姐姐，帮我开个家长会。」


「姐姐，帮我打打游戏，我打不过他们……」


终于有一天，他良心发现了，「姐姐，叔叔阿姨催婚，不然我帮你个忙吧。」


我嗤之以鼻，「你怎么帮？」


「娶你。」


**3**


我穿越成了恶毒女配，还是一本霸道总裁文。


系统给我的任务是让女主黑化。


看着两个半大小子，一个是男主，一个女主，我不禁陷入了沉思。


系统你认真的吗？


为了完成任务，我找上了男主的叔叔厉邢渊。


可谁知道某天，半人高的男主站在我面前，甜甜一笑，「婶婶好！」


……


人类高质量男性花式宠妻。


太上头、太好嗑了！
