## 简介
以 XX 为开头写一篇古言小说。


各种新奇设定的古言小说合集。故事中的女孩子正值芳华，有人嫁给了不喜欢的公子，有人是京城第一美人，有人看上了美强惨男配，她们在各自的故事里演绎着一段段让人唏嘘的古代情缘。


《公子陷落记》


我嫁给了不爱的公子，事实上，我爱的是他丰神俊朗的叔叔。大婚之夜，公子也向我坦诚，他爱的是他温柔娇俏的婶婶。于是，为了心中所爱，我俩开始合谋。但是，叔叔和婶婶这对貌似恩爱的夫妻后面，好像藏着一个巨大的谜团……


《丑女大翻身》


我是丑名昭著的长公主，被父皇赐婚给重权在握的驻疆大将，成了监督他的一枚棋子。婚后，将军跟我相敬如「冰」，可我却渐渐发现，将军宠爱的小情人们，怎么个个都那么像我？将军出征归来，撞破我丑名背后的庐山真面目，一切似乎都不一样了，他开始对我频频示好，而我也越来越觉得，自己似乎早就认识他·····
