## 简介
真实的凶杀，常伴随的是恐怖，变态，残忍。刑警、法医首次披露重大悬疑案件案发细节，带你置身刑警、法医专业视角，重回奇案现场，感受真实破案现场，揭穿命案背后的人性黑洞。


真实的凶杀，常伴随的是恐怖，变态，残忍，这些案件究竟是怎么被侦破的？


在北京有很多出名的警察与法医，他们炼就了一双火眼金睛，犯罪分子一经他眼，插翅难逃。


本专栏将为你讲述他们的破案故事，无论犯罪分子如何狡猾与凶残，办案的刑警总能在蛛丝马迹中找到线索，将凶手绳之以法，将公正还给那些被杀害的冤魂。


\* 诡异之夜，公共厕所惊现女尸，血肉模糊的头颅、墙上的鞋印、手臂的划痕、污秽的血衣......刑警在现场竟然轻松「捡」到了一名嫌疑人，是百密一疏还是另有蹊跷？凶手已定，刑警、法医却做出逆天反转，到底原因为何？


\* 恐怖之夜，回龙观惊现焚尸案：焚烧焦化的颈部、白化的骨骼、神奇的「烛芯效应」......现场法医披露惊悚案发细节，挑战心理极限！


\* 惊悚之夜，村民被人劈开头颅，脑浆和鲜血迸流的伤口、飞溅的血迹......侦查过程中，被害人竟然抱住办案人员？


战胜恶魔的第一步，是承认恶魔的存在，并找到恶魔之所以成为恶魔的密码。


希望这一个个真实而相似的案件，能让你体会到那追缉凶犯过程中，究竟会发生怎样的博弈。


你，准备好了吗？


***注：为保护当事人和刑侦人员、避免泄露侦破手段，本专辑所涉案件细节均进行了一些处理。***
