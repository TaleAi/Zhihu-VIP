## 简介
作为一个亡国之君，李煜为什么能够超越时代、留名千古？根本原因还是在于——词写得好。他善诗词，精书法，工绘画，通音律，是写一手好词的千古词帝。就让我们跟随本专栏一起，将李煜的诗词融入他几经辗转的人生世界。


「无言独上西楼，月如钩。寂寞梧桐深院锁清秋。」


相信大家都对这句诗词印象深刻，它的作者就是被后世誉为「千古词帝」的南唐后主——李煜。


**作为一个亡国之君，李煜为什么能够超越时代、留名千古？**


**根本原因还是在于——词写得好。**他善诗词，精书法，工绘画，通音律，是写一手好词的千古词帝。


清代著名词人纳兰性德非常推崇李后主的诗词：「花间之词如古玉器，贵重而不适用；宋词适用，而少质重。李后主兼有其美，更饶烟水迷离之致。」


王国维也在《人间词话》中称赞道：「词至李后主，眼界始大，感慨遂深，遂变伶工词而为士大夫之词。」


在诗词上，李煜有我们无法超越的艺术成就；在生活上，李煜有我们无法经历的快乐和无法承受的痛苦；在人性上，李煜有痴情和多情的矛盾性。就让我们跟随本专栏一起，将李煜的诗词融入其几经辗转的人生世界。
