## 简介
百万级销售大神的晋升之道，教你提升销售思维，决胜人生。


读了个大学还去做销售，真的很丢人吗？


在销售领域中，哪些行业能体面的赚大钱？


只有会忽悠、厚脸皮的人才能做销售吗？


新手入行销售，就要烈日下奔波、饭桌上陪酒？


销售的职业方向是什么，只能做一辈子苦工作？


那些通过销售获得百万年薪的人，是怎么做到的？


你见过依靠「做销售」改变命运的人么？


前阵子有个新闻惊艳了众生：北大女生毕业后竟然去卖房！这么好的学历都被糟蹋了。但是她糟蹋的结果怎么样呢？一年内的提成就拿到了 200 多万，还不算底薪与奖金！换个版本，你认为她毕业后按照常规进入到某银行或是企业，需要加班多久才能赚到 200 万？


如今在疫情的侵袭下，中年人失业，大学生无业。人们纷纷跳入到直播的红海当中，梦想着成为独立的职业人。可是当你打开页面你就会发现，主播们即便才艺傍身、美若天仙，最后比拼的仍然是拥有销售色彩的带货能力。


教父说：一秒钟看透本质的人与一生看不透本质的人，注定是两种人生。


那么，拥有销售思维的人与不懂销售的人，也必然是两种结局。销售不是一份工作，而是一种思维。营销自我，决胜人生。
