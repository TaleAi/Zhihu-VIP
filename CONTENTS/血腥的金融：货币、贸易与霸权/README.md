## 简介
蝴蝶震翅，天下大变。


金钱和权力的交易，人的权衡与主观干涉，市场的风险与波动，都可能成为影响全球经济的「蝴蝶」。


金融市场的斗争，或许比真刀真枪更加血腥。


一刀一次杀一人，而金融市场小小的波动，一次便影响千万人。


* 布雷顿森林体系解体后，美元成为信用货币体系时代最具实力的硬通货，面对美元贬值，美国如何以美元霸权拉全球下水？
* 「二战」后日本高速增长 40 多年，从一片废墟发展成为世界第二大经济体，又因何陷入失落的三十年？
* 20 世纪末，奋斗了三四十年发展起来的泰国，如何被一个拥有数十亿美元的人在两周内击溃，并由此引发席卷亚洲的金融危机？

本专栏将解读诸个金融危机背后的因缘与博弈
