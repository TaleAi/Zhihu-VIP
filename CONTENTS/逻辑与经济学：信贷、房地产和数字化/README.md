## 简介
从专业视角看懂当下经济环境。


2008 年之后房地产开启了快速的增值。


3 次大型宽松货币周期促进了 3 次大的泡沫膨胀。


然而物极必反，再叠加疫情、战争对经济的影响；


房地产的未来发展趋势也愈发清晰起来，不确定的只是软着陆还是硬着陆。


而房地产只是金融玩法其中之一。


* *元宇宙、NFT 到底是不是骗局？*
* *P2P、长租房金融模式等金融创新衍生品是如何转移风险的？*
* *全球经济衰退一触即发，动荡时代，普通人要如何明哲保身？*
* *美股短时间熔断 4 次，欧洲、日本等经济体纷纷陷入困局，货币政策又应该如何选择？*

本专栏将从金融内在不稳定性出发，回顾房地产泡沫的崛起，解读金融扩张下资本的狂欢，讨论当下热门经济现象的本质，看金融创新的暴雷与实际应对。
