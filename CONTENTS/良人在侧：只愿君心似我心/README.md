《望川》


我死了你就能娶她了，你难道不高兴吗？


我把玩着鸩酒，作势要喝。


姬凡冲上前，眼尾猩红，低声笑道：「公主敢死，我就敢杀了你与他的孩子。」


他嫉妒我与贺恒川有孩子不是一天两天了，这事儿他还真干得出来。


我覆上他的手，轻声细语地哄道：「我放夫君自由，愿夫君与良人百年好合，子孙满堂。」


  



《鱼没了》


何大妖能有什么坏心思呢，不就是想谈点让人快乐的恋爱。


她不喜负责任，但爱绝对保真。


好不容易想赌一把，柴米油盐酱醋茶地和一个人生活，那人却又不要她了。


还真是风水轮流转呢。


别说对不起。


不原谅。
