## 简介
本课程针对第一次买商业险的小白，教大家掌握购买保险的基本原则，为自己和家人做财富和健康的高杠杆保障。
 



商业险买对了是省钱，买错就是花冤枉钱。但保险公司那么多，每个业务员都说自己的保险好，该怎么办？意外险、寿险、重疾险、年金保险，各保什么？哪种适合自己？如何配置？为什么保险不仅仅可以对冲风险，还可以进行财富的重新分配？


本课程主讲人——保险专家侯会军，具有十年保险从业经验，能深入浅出地教你从零开始配置保险，一次搞懂所有商业险种，了解每种保险的特点和理赔方式，从购买到规划，从读懂条款到提前知晓理赔步骤，让你和家人的保险买得放心明白。
