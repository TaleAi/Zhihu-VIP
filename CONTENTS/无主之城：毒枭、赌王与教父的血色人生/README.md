## 简介
本专栏描写了十位游走在法律边缘的黑道人物，情节紧张刺激、扣人心弦。


毒枭、赌王、黑帮大佬……


他们游走在法律边缘，凭借狠辣获得巨大财富，他们的「成功」背后，更重要是时代的造就。


墨西哥、哥伦比亚毒品泛滥，是地理环境与拉美十年贫困造成；吴锡豪、林梧桐跨洋到香港、马来西亚，是 60 年代偷渡移民潮的代表；社会从贫困中醒来，经济急速发展的时候，来钱快的行业会迅速崛起，在有钱就是硬道理的片面观念下，犯罪滋生。


但他们中，也有人从一开始便走上合法的发家道路。


繁华、迷失、财富、深渊……一场血雨腥风的故事。
