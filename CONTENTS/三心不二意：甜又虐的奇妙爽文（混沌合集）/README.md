在后宫，我只是一个傻傻的忠臣之女，只想跟我的好姐妹吃喝玩乐到老，并不想待在这这动不动就要灭人九族的皇帝身边！但是，为什么，这个男人频繁出现在我附近，还要骂我蠢？而我的好姐妹，又背着我在悄悄地谋划什么？在这个吃人不偿命的后宫，我会收获真心真情吗？
