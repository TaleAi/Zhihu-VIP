## 简介
诗歌是孩子从小学到高中学习生涯中越不过的坎，但是很多孩子从背诵诗歌，却懵懵懂懂不解其意。本课程主讲人郎净老师，不仅是华东师范大学文学博士，复旦大学中国史博士后，更是一个会教孩子的妈妈。背诗不如欣赏诗，学诗不如感受诗。本课程将为孩子打开诗歌的世界，帮孩子轻松越过诗歌高山。


 每一个妈妈都希望自己的孩子，腹有诗书，气质如兰。 


于是，孩子被迫背诵了成千上百的诗歌，却无法理解诗歌描绘的花鸟鱼虫，秀丽山川。 


孩子厌烦，家长无奈。 


其实，孩子是天生的诗人，比大人更会感受诗。 


只在于，你有没有为孩子打开诗歌的大门。 


他们的想象力可以无边无际，他们可以和世间万物对话，感受诗歌里的宇宙乾坤。 


在这里，他们能体会「人闲桂花落」是闲人才能听到的自然之声；「听取蛙声一片」是夏日独有的有趣景致；「绿蚁新醅酒，红泥小火炉」里蕴含的是诗人想见到朋友的急切…… 


能和星星共舞，和月亮对话。 


背诗不如欣赏诗，学诗不如感受诗。 


本课程将为孩子打开发现的眼睛。 


华东师范大学文学博士，复旦大学中国史博士后，更是孩子妈的郎净老师让学诗歌像听故事一样有趣，更有启发式教学，引爆观察力，想象力，让孩子在玩中快速成长！ 


为孩子打开发现的眼睛，孩子天生比你更懂诗！
