## 简介
本专栏由 SUNNY CHAN 老师教大家如何应对关于室内环境污染的检测与控制方面的各种常见问题，最大程度降低装修给住户带来的健康隐患，打造健康舒适的家。 


新房装修完成，家具进场之后，甲醛、苯系物、TVOC 等造成的室内空气污染会在相当长一段时间里会保持较高的水平，立即入住的话，可能会带来很多健康隐患。 


那么，怎样才能相对准确的评估新家的空气环境实际污染水平？如果遇到严重污染的状况，应该怎样去正确应对，才能避免对家人的身体健康造成不必要的风险呢？


人居环境安全十年从业者 SUNNY CHAN 老师教大家如何应对关于室内环境污染的检测与控制方面的各种常见问题：


* 如何寻找专业检测机构，选择合理的检测时机；
* 在进行了科学的准备工作基础上，如何完成一次有意义的检测；
* 当地缺乏专业机构，如何找到别的替代办法去完成一次相对靠谱的评估；

了解这些问题后，你能够对污染源有初步的认识和评判，对有问题的装修或家具进行合理有效的维权；遇到污染水平较为严重而无法及时入住的糟糕局面，也可以尽快找到降低污染的有效措施，让新家真正变得安全起来，把装修和家具给家人带来的健康隐患降到最低。


没有什么比家人的健康更重要，让我们一起来打造一个安全的港湾吧。
