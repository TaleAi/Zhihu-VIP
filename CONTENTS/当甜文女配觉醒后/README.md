## 简介
杨岁发现，自己的生活居然是一本校园甜文，而她，是衬托女主的悲催女配。


 十八岁以前，杨岁的生活只是在泥潭中挣扎，卑微地看不见光。


十八岁以后，自我意识觉醒，她才看到命运的不公。


身处校园甜文小说世界，生活没有甜味，只有作为主角对照组的苦悲。


林音音和周裴才是这个世界的主宰，只要他们动动手指，整个世界都要围着他们转。


可是为什么？杨岁不甘心对这样的命运妥协。就算全世界都对他们偏爱，她也不想失去自我做那块垫脚石。


她要抗争，她要拥有自己的人生，她想自己决定成为什么样的人！


人活着的意义，从来不是为了衬托谁，而是为了成为更好的自己。


——「这个世界，总会有一束光是为我而来的，我也可以闪闪发光。」
