## 简介
神秘事件、平行时空、脑力与良知的角力，为您献上一幕幕精彩剧情。


作为一个心理医生，我遇到一个奇怪的病人，声称自己杀人后对方却一次次复活。


随着了解深入，各种陈年谜团以及证据相继出现，似乎印证着这并不是一场笑话。


而背后的「真相」，源自多年前的一场平行时空事件，我也似乎看到了曾经挚爱复活的契机。


然而，事情向着令人意想不到的方向发展了。


——《双缝》  

  

因斗殴被拘留的第四天，我在警局接到了妻子的死讯，死亡时间正是我被捕当天。


妻子的出轨对象成为了第一犯罪嫌疑人，随后警方发现情人在酒店畏罪自杀，案件落幕


就在这时，一个警察出现在我家，他声称我才是这个案件的真凶……


——《脱身》  

  

我醒来，发现自己身在一个由五个房间组成的十字密室，一声枪响带走了一条人命，我进入了一场逃杀游戏。


主办方声称：在场众人都有形或无形中参与过一起谋杀案，凶手就在我们之中，120 分钟之内，我们必须找到凶手，否则就会死去。


随着线索展开，我们曾经的罪恶也逐一浮现。就在死线降临之时，尸体却突然复活，声称自己才是凶手。


——《十字密室》  

 


作家希野死了，定案自杀，死亡方式和正在写的小说主人公死亡方式一模一样，而知道这份草稿的只有我们同为悬疑小说家的六个人。


这不是自杀，是一场伪装得极好的谋杀。


我们三人组展开调查，更多人接连死去，草稿也始终不见踪迹。


当我们来到一个海岛，真相徐徐揭开……


——《杀死那个小说家》
