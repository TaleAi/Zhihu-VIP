## 简介
自在的生活，首先是疼爱自己，认识自己，有了强大的内心，才能掌控和这个世界的关系，外表优雅，内心坚定，温暖从容，这才是从幼稚到成熟的标志。
享受自在生活，有方法，心理学就像是一套工具，帮你更加了解自己，了解内心世界的运行法则。本课程用简单有趣的心理学知识，给你有效的方法。
