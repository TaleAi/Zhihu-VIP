## 简介
历史的遗憾总是太多，但假如……


天时、地利、人和，是历史人物成大事的三要素，缺一不可。


历史上，很多英雄人物都因为某一要素的缺乏，或溃败逃亡，或含恨而终。


后人研究那些名人的失败，因为放在历史的尺度去看，英雄们身上令人羡慕的才干和能力，因为各种机缘巧合，最终未能成就于历史。这也正是我们学史的意义：以史为鉴，以正自身。


遗憾太多太多，假如，有一道光照向他们，奇迹发生了：


* 如果被矫诏赐死的公子扶苏，灵魂穿越到了胡亥身上，能不能挽救秦朝的灭亡？
* 如果把历史上所有的亡国之君安排到知乎上学习一个月，再让他们回到各自的历史时空，那么他们可以避免自己国家的灭亡吗？
* 如果历史上的岳飞和一名抗战时期的八路军战士互换灵魂，会发生什么故事？

作者将带领我们一起，解析历史名人的时代背景、突出才能、人物弧线，来向我们讲述一个不一样的历史。
