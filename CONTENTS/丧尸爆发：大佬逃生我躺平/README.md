## 简介
末日来临，丧尸病毒爆发，重活一世，主角们开始了躺平囤货的快乐生活。


继承遗产后，我摆脱了那些觊觎我财产的亲戚跑到一个陌生城市的郊区，买了一栋有巨大地下室的别墅。我将别墅一顿改装，又采购了大量物资想过蜗居生活。  

没曾想竟遇上了极端天气，末日降临，丧尸成群。正好，我搬去了别墅的地下室开始正式躺平。
