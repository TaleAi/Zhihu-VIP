哈佛、耶鲁高分毕业生李柘远分享他的 100+ 精进方法和独家学习经验，陪你一起终身学习。


如何记忆，才能牢固掌握知识点？ 如何复习，才能夯实完整知识体系？ 如何自学，才能实现终身跨界成长？ 


这些问题，《学习高手》里都有答案。 


本专栏是一本写给广大学生和家长的，培养高效学习能力的方法工具书。


作者作为哈佛、耶鲁高分毕业生，结合先进的理论知识、经典的学习经验，系统分享了超实用的学习法。


本专栏分为 5 大模块、30 节大课、100+ 方法，帮助你提高学习成绩，打开人生格局。
