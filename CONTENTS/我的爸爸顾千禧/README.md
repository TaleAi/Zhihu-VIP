## 简介
一对青梅竹马，从年少到成熟，共享生命欢乐，共面命运不公。


顾千禧好像很擅长当爹。


怎么形容呢。


他三岁的时候，就从「人贩子」手中救下了我，成了我的「爸爸」。


乌鸦反哺，羊羔跪乳的内种爸爸。


我爸特别不幸。


他生下来后爹不疼娘不爱，谁也不愿要他。


但是不幸的他又有个特别大的优点。


他有钱。


我爸太有钱了，提起这个，我总是激动得想哭。


他从「人贩子」手里救下了我，以为我从此能跟有钱的他相依为命。


他没想到我是个白眼狼，一门心思想找别人当爸……
