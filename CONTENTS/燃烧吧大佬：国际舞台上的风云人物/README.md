## 简介
每逢世界变局，必有大佬出现。


在不同历史时期能脱颖而出的风云人物，必然是博弈和权术高手。


* 前特工普京当选总统，「真·男人」带领俄罗斯硬刚西方列强；
* 默克尔带头修建「北溪二号」，为团结欧盟操碎了心，同时也被称「中国的老朋友」；
* 文在寅为老友「报仇」竞选总统，如今卸任的他是否能打破「青瓦台诅咒」？
* 车臣领导人为什么对俄罗斯「言听计从」？

虽说各位大佬手握重权，但也做不到「一言堂」。


历史变局中的所有重大决策，都是多方博弈的产物，大佬们话语权的多少，只取决于手上有多少「可交易的筹码」。


这个专栏带你了解历史舞台上的风云人物，以及他们在重大事件中的博弈过程。
