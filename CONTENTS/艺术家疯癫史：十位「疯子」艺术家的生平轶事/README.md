## 简介
本专栏为你讲述十位「疯子」艺术家的事迹生平，带你走近这些不寻常的天才，体会「天才在左，疯子在右」的艺术世界。


这世上有难以看懂的艺术，更有难以看懂的艺术家。


「圆点女王」草间弥生、「哭之笑之」八大山人、「看透男人」弗里达、「割耳赠友」的梵高……这些流传久远的画家们把自己的生活活成了无法容于世人的行为艺术。


「我一天到晚都在制造这些令人生畏的形象，借此来克制我的情感，也只有通过这种方式我慢慢把恐惧变成了我熟悉的事物。」


「我生命中遭遇过两次巨大的灾难。一次是被车撞了，另一次是遇见我的丈夫。」


他们是在艺术殿堂留下无数瑰宝的名家，也是「世人笑他太疯癫」的「神经病」，然而，他们的癫狂背后是多少辛酸与无言？


过往种种，最终皆成名人堂内一句词，画纸之上一抹色。
