## 简介
当爱情被阴谋裹挟，当婚姻被利益绑架，当每段感情故事都遇到了狗血的展开和反转的结局，总有一个人你爱不到，总有一个故事你忘不了。


本以为是异国他乡的艳遇，却差点人财两空，甜言蜜语原来是布满鲜花的陷阱？


交友网站匹配到百分百心动女郎，确认关系后却离奇失踪，留下一连串疑团......


在通往成功的野心之路上，「真爱」到底是绊脚石还是助推器？


相亲到底能不能相来真爱？拥有一段体面的婚姻算不算人生赢家？


本专栏内每个故事都有啼笑皆非的反转结局，或心痛，或释然一笑。


暗恋，情感诈骗，爱错人，求不得，忘不了......


这些都是我们为「爱情」受过的苦。
