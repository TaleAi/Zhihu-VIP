## 简介
准备好了吗？以《周游记》为蓝图的脑洞工程开始了！挖，挖，挖！


嘿，快点准备好，我们要出发咯！以全世界为行程，绝佳的风景和美食，一切都是计划中的完美……等等，我们的旅行才没有这么简单。通往奇妙脑洞旅行的大门里可没有落入俗套的安排，只有意想不到的情节，各种大跌眼镜的操作。在无数个突破想象的脑洞故事里，带你去感受世界的不同，欣赏各种荒诞与浪漫、勇气和温柔。在这个疲惫的时代，让精神得以休憩，让故事唤醒内心深处那渐渐沉睡的少年——周游在路上，才发现，无论岁月如何改变，心中的少年始终未变。


善凡居然穿越到了动物园，而且还变成了一只鬣狗……


作为一名绑匪，阿海大概从未想到过，自己也有被绑架的一天！而且对方还是条美人鱼。


脑洞大开的故事，忽然出现的杰伦，哎呦，真的不错哦！
