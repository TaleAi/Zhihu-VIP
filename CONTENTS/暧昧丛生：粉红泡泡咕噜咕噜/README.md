**《南和初初》**


我听见我哥喊一个好看的男人「老公」。


我刚准备把暗恋喂狗，却不想每个人都叫他「老公」。


我大脑这才转过了弯，原来此「龚」非彼「公」。


讪讪一笑后，男人却不依不饶：「小朋友，你的小脑瓜子里都在想什么？」


  



**《吃货老板》**  

我是个美食博主，朝九晚五的工作只是我生活里的调味品。


只是我没想到,我上班停车的时候竟把老板的法拉利刮花了。


这就从「调味品」上升到「命根子」了……


我瑟瑟发抖：「老板，要不你把我卖了吧，我还能给你数钱。」


但没想到老板只是冷冷地看了眼车被刮花的地方，语气不甚在意。


「你每天给我带一份爱心午餐。」
