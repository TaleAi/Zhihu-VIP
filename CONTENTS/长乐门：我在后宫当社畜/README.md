## 简介
当穿越遇到重生，当社畜遇到腹黑 boss，当单身汪一见钟情最帅男配，会发生怎样的故事呢？


毕业后囿于工作困境的业余 rapper 林苏，一朝醒来，成了某大女主小说中女主林芷……的妹妹。


此时正逢女主与青梅竹马私奔，她爹要让林苏代嫁入宫。


为避免走上原主入宫即失宠，窥见大秘密，被处以极刑的悲惨结局，林苏决定接替原主，将后宫当做古代职场，以领导皇上为最高旨意，艰苦奋斗。


宠妃让作诗？来首 freestyle。


太后要贺寿？一首 rap 送给她。


只是，原文里对女主一往情深的皇上奇奇怪怪，一入宫便逼我做宠妃。


原文里同为炮灰的少年将军多次救我狗命，女主试图收买他未果，他却为我出生入死。


一朝入宫，险些不得善终。


当命运的剧场在我面前拉开帷幕，原文里两条人命的消亡真相渐渐揭开，皇权倾覆下的人生举步维艰，我又能否真的如自己所愿，在后宫的社畜人生中成功升职加薪，出任 CEO，迎娶小奶狗，走上人生巅峰呢？
