## 简介
当金钱与爱情酿成大难题……


金钱，对爱情、婚姻来说，是一把双刃剑。


它能让爱人们免于陷入「贫贱夫妻百事哀」，能带来更好的物质生活、成为家庭稳固的保障……


但在更多时候，它是人性的镜子。


镜中仿佛有个幽灵，在不经意间召唤你我，摧毁那些意志薄弱的灵魂……


**专栏《情难比金坚》，集合优质作者笔下最现实、最直指人心的故事，为你呈现人生中 15+ 个「金钱 X 爱情」故事。**


「大学生还没收入，约会该不该 AA ？」 


「妻子给丈夫名下的房子还大笔房贷，心里会有芥蒂吗？」 


「黄昏恋再婚老夫妻的遗产怎么分、分给谁？」


……


故事，将引领你观照生活。


**而问题的答案来自日常，却终将归于每个人心底。**
