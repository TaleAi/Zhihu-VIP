## 简介
一觉醒来，我睡在男神的床上。


陶长安，一个普普通通的大四女生，却在某一天，睡到了男神的床上。


「当时的情况很复杂，但请相信我，我和男神是清白的！」她试图抗辩。


一段时间后，她又睡到了男神的床上。


对外温雅和煦，对内腹黑狡诈的男神咬着她的指尖：「现在还是清白的吗？」


陶长安：「呜——」
