**《朋友已满》**


男友打来电话说分手。


我愣了愣，回了句：「我们不是一直都是朋友吗？」


对面安静一秒，随即炸锅：


「嫂子威武！」


「老大，你怎么舔了这么多年，还没舔到啊？」


许惊辰轻咳一声：「抱歉失陪，我先处理一下家事。」


  



**《怨种闺蜜和我的十八个男模》**


怨种闺蜜应聘模特经纪人，夸下海口说资源丰富。


公司让她带人参加 500 进 18 的模特大赛。


她赶鸭子上架，派我出战，并承诺一轮游，快速把我淘汰。


可是，到了现场我才发现：这 TMD 是男模比赛！


  



**《嘴瓢少女》**


救命！穿拖鞋去图书馆学习。


写完作业探脚找鞋，把对面鞋勾过来了。


小学弟一脸震惊地看着我。


我纠结说「对不起」还是「骚瑞」。


结果说成了「骚不起」……
