# 我爱你，就像雾里看花

### 存档日期：2022-06-02

## 简介
这里的女人或是身处情感的弱势关系中，或是戴上了生活的痛苦面具，是得过且过还是自我救赎？看她们逆着风飞出深渊，飞向真善美的高空。


***冷暴力与家暴受害者，如何联手完成自我救赎。 ——《以牙还牙》***


丈夫陈征被拍到与女模在酒店亲密，事后还无耻地让我帮他善后。


家丑之事惊动了父母，他们打电话来施压，我只得迅速料理了试图博上位的女人。


我在拳击馆打拳发泄，无意间发现拳馆员工张芹被丈夫殴打，以她的身手没理由忍气吞声，我想肯定她也有难言之隐。


我们两个被伤害的女人，决定一起快刀斩乱麻。


***我们愿穷一身之力，为你守住这方圆三里的幸福。——《方圆三里》***


我被监视了，怀疑自己陷入了楚门的世界中。


种种疑点一一被印证，甚至连父母和家人都是被扮演的。


我决定逃离这里。


直到一个自称是我未婚夫的神秘人出现，出逃的过程中，事实一直在反转。


我怀疑过的坏人，都被印证是好人，那面前带我出逃的男人，是坏人吗？


***人心和情感，哪个经得住金钱的试探？ ——《一夜暴富》***


婚期前一个月，男友王洋因拆迁暴富，可我被甩了，我单方面被王洋全面拉黑。


我找王洋算账，原来他早已和同事余娇苟且在一起。


但余娇恶人先告状，诋毁我见钱眼开贪图拆迁款，还与王洋的外甥暧昧不清，被王洋发现后才分手……


忍不了了，那我将计就计，顺势把谣言坐实，看我拨弄风云易如反掌。
