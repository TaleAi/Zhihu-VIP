## 简介
这是一个大权在握的摄政王，追求武力值爆表的心上人，却惨被抛弃的故事。


早年间，王爷捡来一个小姑娘，他勤勤恳恳地一心搞养成，不料养得太过优秀，竟把她养成了自己的政敌。


在朝堂大获全胜的第二天晚上，他就收到了心上人病逝的消息；人死灯灭，但他依旧恼她的背叛，力排众议要将她的坟迁出王陵，也因此发现了她的金蝉脱壳之计。


边关战事吃紧，摄政王亲征大尤，却在那里重逢了假死的心上人。


她终于又回来了，可是她变了太多。


和她一起出现的，还有一个与她长得一模一样，但所图甚大的邪教女子。


朝堂上，羽翼渐丰的皇帝小动作不断；


江湖里，邪教不断进逼京城，酿成多起悬案；


王府中，真假心上人轮番出现，扰乱了摄政王的一池春心。


真真假假，摄政王患得患失。


王爷将心向明月，奈何明月照沟渠。
