## 简介
哪些国宝级文物至今仍然流落海外？


直到今天，依然有很多珍贵文物流落海外，或馆藏于某外国博物馆，或被某些外国私人收藏家收藏，成为我们深重的遗憾。


从出土到流失海外，它们经历了怎样鲜为人知的历史；为了寻回这些珍惜国宝，我们做出了哪些努力，取得了什么成果？


* 圆明园十二兽首如何流失海外，目前有哪些已回国？
* 《盗墓笔记》原型子弹库帛书是如何被外国人骗走的？
* 美国人华尔纳真的是用胶水揭走了敦煌壁画？

本专栏，我们邀请了 2021 新知答主，浙江大学文物与博物馆学硕士「项木咄」，带我们回溯国宝从出土到流失海外的全过程，以及在寻回国宝的道路上，我们做出了哪些努力。


作者还将为我们科普文物背后的历史价值，以及随着考古技术的不断发展，文物的发掘和保护都取得了哪些进步。


希望通过阅读本专栏，可以让我们从珍贵文物的角度，对历史有更生动鲜活的认识；也让我们对考古学这门学科有更深刻的认识，满足猎奇感的同时，对考古获得耳目一新的认识。


祝你阅读愉快～
