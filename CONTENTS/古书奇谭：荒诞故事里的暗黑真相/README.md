## 简介
从「三言二拍」到《聊斋》，古书中不乏荒诞离奇、悬念迭起的精彩故事。本专栏借由这些经典故事为原型，脑洞大开进行改编，通过一环接一环逻辑缜密的推理，将这些令人心惊肉跳的恐怖故事，合理演绎成一桩桩有预谋的迷案，抽丝剥茧带你寻找原故事之外的另类真相。


**算死**


开封府的孙衙司一天无事，被偶遇的算命先生下了个「今晚子时，必死无疑」的卦。气愤不已的他被息事宁人的同事好友劝下，想着撑过今天，明天去算命先生面前讨个说法。但就在当晚子时，孙衙司竟然主动跳进了河里，尸体都没有找到！孙衙司之死，背后究竟有什么故事？


**空心柳**


杭州钱塘县的柳林里，惊现一具无头尸。死者是一个当地富商的独子，家属和官衙重金悬赏，找寻人头和凶手。一个月后，一个叫黄大保的穷人提着颗腐烂的人头来领赏，而种种证据都表明，这领赏的黄大保，是本案唯一的嫌疑人。令人不解的是，黄大保的不在场证明完美到无懈可击。
