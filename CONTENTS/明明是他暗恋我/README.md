## 简介
你喜欢的人，刚好也在偷偷喜欢着你。


「哥哥的床大不大，一个人睡怕不怕？」


我跟风录了这个视频，然后，男神很快给我发了消息：


「床大，不怕，下楼。」


我跑去窗前看了看，他居然真在楼下。


然后……


男神把我带去他家，美其名曰：看床。


——《股掌之上》


我拿错了快递，打开后，里面是一箱子 big size 的情侣用品。


见面还快递时，发现对方居然是我暗恋已久的小学弟。


小学弟说她没有女朋友，我信了。


可是——


他忽然出现的室友却搭着他肩膀笑：「晚上一起把这箱子用掉啊？」


用……掉……


——《星河暗恋记》
