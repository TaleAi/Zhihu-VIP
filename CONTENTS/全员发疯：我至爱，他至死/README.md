## 简介
我至死都渴望有人不顾后果、不计回报地爱我，热烈、疯狂、精彩。


学生哪儿有不疯的，社畜哪儿有不疯的，现代人哪儿有不疯的！


尖叫、扭曲、阴暗爬行！我早起一拳打爆这个屑世界！


发疯、摆烂，躺平、但内心深处依旧热爱生活，渴望热烈乃至极端的感情。


爱欲于人如逆风执炬，发疯也是。


  



《六天六夜》


我被绑架了。


将我囚禁的男人每天都会来与我玩扮演游戏。


有时他假扮影卫，而我是他誓死守护的公主。


有时他假扮变态杀人狂，而我是他心心念念的猎物。


我原以为这只是男人的人格分裂，直到隔壁传来一张纸条——


「小心，他们中有一个人格是假的，他想杀你。」
