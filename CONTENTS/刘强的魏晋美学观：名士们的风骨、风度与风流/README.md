## 简介
提起魏晋，你会想到什么？是王羲之父子的字、顾恺之的画，还是刘伶的酒、嵇康的广陵散？魏晋，是打开中华文化最为绚丽迷人的窗口之一。本专栏特邀刘强老师深度解析《世说新语》，带来一场「魏晋风度」与「竹林精神」的多元回溯，遇见魏晋时代的风骨、风度与风流。


汉末魏晋六朝究竟是一个怎样的时代？


政治混乱，社会痛苦，并由此带来了精神上的极致自由和解放。


求自然、求自我、求自由，是魏晋风度的「三自追求」。


这个时代好比西欧十六世纪的文艺复兴，前无古人，光芒万丈。


而那些特立独行的魏晋名士，更是千古风流、姿态万千。


本专栏特邀同济大学人文学院教授刘强老师带来一场「魏晋风度」与「竹林精神」的多元回溯。


曹操、孔融、王粲、嵇康……透过《世说新语》，对魏晋知识分子深度摹画，一窥十大生活美学主题的流光溢彩。


美容、服药、饮酒、任诞、隐逸、品鉴、清议、清谈、奢侈、艺术，遇见魏晋时代的风骨、风度与风流。
