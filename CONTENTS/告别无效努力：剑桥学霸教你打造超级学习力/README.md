## 简介
初中作为一个学渣，如何后来居上，实现人生逆袭？三年高中数学，如何在三周内完全学会？辅导 7 位学生考上剑桥大学，究竟有着怎样的学习诀窍？这门课，Frank 将为你展现成为学霸所具有的独特的高效学习方式。


这是一个超级学渣的故事，他在国内读到初中，便因无法适应应试教育而辍学在家；  

 这也是一个不断寻求变化而人生逆袭的故事，他顶着巨大压力出国留学，寻求更好的自我教育模式；  

 这更是一个证明学习方法远高于学习时间的故事，他用七天学完高中三年的数学，用高效学习法辅导 7 位学生考上剑桥大学。


故事的主人公 Frank，将在本课程让你认识一条全新的个体成才之路。 


本课程由剑桥学霸 Frank 主讲，主要介绍自己在学习方面所运用的另辟蹊径的思考模式和学习方法，用这套方法将极大提高自己的学习效率，杜绝无效努力。解决知识看完就忘、学习精力不集中、考试时知识用不上等问题，破除花费巨额时间，却收获寥寥的学习困惑，教你收获超级学习力，做高效能的优秀学习者。
