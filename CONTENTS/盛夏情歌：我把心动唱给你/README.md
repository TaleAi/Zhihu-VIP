## 简介
心动寄放在月亮上，抬头看，我在想你。


青春里，我们会记得彼此经历的每一件小事。他走过的路，她说过的话，哪怕是一个不经意的调侃，都始终铭记于心。


酸甜向恋爱日常，关于那些我不敢说出口，却发现他也刚好喜欢我的爱情故事……


  



**我的男神是花痴**


我发现了男神的微博小号。


表面高冷无情，背地里却是个花痴。


暗恋他多年，这还是我第一次发现他的另一面。


看完微博，我怅然若失。


他有喜欢的人了。


而且那个人，我好像认识。


  



**恋爱直播**


我参加了一个恋爱直播节目。


节目里，我遇到了前阵子才被我折磨过的乙方弟弟。


听说配对成功，能有六十六万的恋爱基金。


为了这笔奖金，我盯上了这位乙方弟弟，并承诺拿到钱后跟他平分。


一人三十三万，好聚又好散。


可是演着演着，他好像当真了。
