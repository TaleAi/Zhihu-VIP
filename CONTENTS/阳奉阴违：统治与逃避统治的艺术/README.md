## 简介
「阳奉阴违」「制度套利」，这样的社会潜规则其实普遍存在于古今中外的各个国家中。「统治」与「被统治」者间不是简单的管理与服从关系，而有着种种荒谬的利益博弈与认知错位，这正是「上有政策，下有对策」等「社会潜规则」的底层运作机理。



* 古代偏远山区的人们真的会知道皇帝已经换了吗？
* 被殖民地人民是怎么开始说殖民国语言的？

「二十四史非史也，二十四姓之家谱而已」，传统历史高头讲章中，普通人真正的日常生活向来不被重视。


但皇宫贵胄这些「不普通人」的日常，以往相关的知识其实也少得可怜：


* 为什么后宫中嫔妃们一定要争宠？
* 为什么「和亲」在古代会被作为求和的外交手段？

还有太多这样似是而非的问题常常被人们忽视，但普遍存在于古今中外的各国历史中。


更重要的是，要理解这些人类文明普遍现象的深层原因，仅仅中国一国的历史就无法解释清楚了：


* 古代空地那么多，为什么农民宁可被地主剥削，也不去开垦荒地？
* 中外的古人为什么都喜欢打猎？

除了对中国古今的情况烂熟于心外，专栏作者莱茵行宫伯爵也精熟于东南亚、拉美等后发国家的政治、经济、文化及历史。


它们与中国有着相似的历史进程，也有着各种相似的历史现象与社会潜规则。


莱茵行宫伯爵以其对中外各国历史文化的全方面了解，发掘出了这些历史现象背后出人意料的共性——社会上下层之间的关系错综复杂，在各国统治者与被统治者之间，都有着种种荒谬的利益博弈与认知错位。


洞察没有历史的人民，由小历史见大历史，由百姓小故事讨论时代大问题，由此认识政策与人性的复杂博弈，了解权力运作与应对的真实机制。
