## 20.如何成为前辈愿意帮带的潜力新人
有人问我，新人进步最快的方式是什么呢?


我觉得学运营就像学武功一样，有各种的门派，书、线上课线下课音频课都可以学一点，但是最重要的是身边要有个能帮带你的人，十分十分重要！


我的职场非常幸运的一点是，不管在携程还是在阿里，都有师兄带我，这是一种公司文化。他们会教给我很细节的东西，在过程中一遍遍的打磨。可是当时的我也有年轻气盛的一面，有时觉得师兄或主管太不近人情，为什么一点点东西都要改，为什么那么严格。现在当我对我的团队和学生展示细节狂魔的一面时，我才发现，这是最珍贵最难得的，也是让对方最累心的。前辈本可以睁一只眼闭一只眼就过了，或者是自己改了，不与你争论，但是她给你指出问题在哪里，指导你把一颗原石打磨成珍宝的时候，那是最有价值最锻炼人的部分。是任何课程都比不上的。


那作为一名小白，怎么做才能成为前辈愿意帮带，扶持，给资源的潜力新人呢？


我用我的助理团队里的典型人物来举例。我现在有一个兼职的助理团队，我只给他们支付了微薄的奖金，但他们愿意跟着我一年以上，他们身边的人能感受到她肉眼可见的进步，就是师徒帮带的魅力。


其中进步最快的一位是一个商会工作的同学，叫小甜甜。商会是很传统的架构和工作内容，跟互联网几乎不搭边。第一次让她做的活动物料，我差点崩溃。但在跟我磨合了几个月之后，上手特别快，不仅把圈子工作做得好，本职工作也被运营加持了，而且，她还通过持续输出，成为圈内外小有名气的知识网红。我们其他的几个助理，现在也都慢慢有了自己的个人 IP。


那这些同学身上有什么特质呢？为什么有的同学却得不到这样的帮带呢？从他们身上，我看到这样一些闪光点，给大家分享一下。


1. 有用户思维，为用户创造价值


要胜任运营这个岗位，用户思维太重要了。简单的说，就是能 get 到用户的需求，能戳到用户的痛点，不要自 high。在运营手段上，不要炫技用一些生硬伤害用户的套路，多走心一些，让用户觉得舒服，就会顺着你设计的路径，一步步往下走，最终达成目的目的。


我发现很多人即便不是运营出身，但只要有运营思维，做的事儿大差不离不会偏离太远。比那种塞了很多知识，但是用户并不买单的人好很多。


运营和销售最大的区别，就是运营是先为用户创造价值，以用户为出发点考虑，提供价值，再获得回报。服务前置，获益后置。而销售思维的人，常常为了把东西卖给用户而去硬推产品。


这点上，几位助理都做得很好。在做社群工作时，能隔着屏幕揣摩背后那位用户的情绪，知道用户需要什么，在用户没开口之前，就提供了服务，让别人觉得特别爽。


2. 有灵气，遇事儿愿意往前多想一步。


我在《我在阿里做运营》一书里也有写到，大家都喜欢有灵性的新人，那灵性怎么体现呢？我觉得就是凡事往前多想一步，举一反三，前辈给你说一个步，你能往后推三五步；并且下次再遇到同样的场景时，同样的问题不要再犯，不要让人多次在一个点上提醒你，消耗前辈的精力，久而久之，前辈就会觉得时间耗在你身上没有价值，转而关注其他的苗子。


3. 长存感恩之心，坚持做个长期主义


运营这个事，挺考验耐性的，你如果一来就想从和前辈的互动中得到直接的好处，可能会得不偿失。就想武侠剧里，高人常常会想考验下这个人是否能吃苦，值得长期投入时间精力去培养。


可能一开始会分配一些枯燥琐碎的事儿给你去做，也有可能一开始不告诉你正确答案，暗中观察你在完成过程中的特点。总之，你要经得住考验。把小事做好了，做出差异性了，自然有更大的项目和资源给你。而一开始就拈轻怕重的话，好事可能根本就轮不到你。


比如我有一个助理叫梦奇。她之前在很传统的行业，她来上过我的线下运营私塾课，但对她没有特别的印象，后来她约我说想当助理。一开始的时候我还是有点怀疑，心想，先丢给她一件事儿去试吧，做不好的话自己会打退堂鼓。但是我发现她特别珍惜机会，在言几又和我们运营圈子计划的联合活动中，我在主要负责，我发现她把每个细节都把关的很好，运营思维和运营节奏也特别不错。她没有专门接触过运营，但她愿意花时间去琢磨，为了大家的学习体验和学习效果更好，她甚至自己会想出一些玩法，远远超过我所交代的事情。通过一场场的活动，她被我看见，被更多的人看见。现在，她也顺利转行，开启了自己的理想人生运营之路。


真正的收益，往往在日积月累中就出现了。他们会发现被我虐过之后，出去很多公司都会伸出橄榄枝。而在成长之后，即便有一天不在团队内了，也会心存感恩之心。


当他们在外面的圈子里表现出很好的运营功底时，别人表扬他们时，他们会说我是她的运营启蒙老师。这时候，作为老师，是最有成就感的。


4. 有打造个人 IP 的意识，会链接，善于借势


前期你向前辈学习，或者给他当志愿者换取共事的机会，是属于前辈在给你输血，如果有一天你成长了，能反哺给前辈，这样的人是更容易引起前辈关注的。


毕竟谁都希望自己的徒弟好， 自己的徒弟好了就是行走的口碑广告。打杂的好找，但是好徒弟难找呀。


比如， 我在选我的助理时，有个硬性规定，朋友圈好友数 1000 人以上。因为我觉得这代表她有意识的在扩大人脉，在接触社交电商、私域流量这块。而不是什么都靠平台给流量。而我也会在过程中让他们尽量达到朋友圈好友数 3000，再加上一些自我优势的探索和朋友圈技巧，个人 IP 的雏形基本就有了。


还是举助理小甜甜的案例，她在上了运营私塾课之后，发现有的技能在互联网公司才能用上， 而自己在一家传统商家，怎样才能活学活用呢。于是她选了朋友圈运营这样一个很垂直但受众面非常大的切口作为她努力的方向。


她开始在朋友圈输出#懂点运营#这个模块，从 day1 到现在的 day100 多天，每天坚持，把从我身上学到的结合她的案例以听得懂的方式讲故事一样讲出来。这个系列吸引了很多人关注，很多人都说看上瘾了。一点点看着她转型成功，并在新领域获得变现能力。很多人就感兴趣了，向她请教。她再把我的课程推荐给身边的人，大家上了觉得课程很不错，又反过来感激她推荐，也变相提高了自己的势能。


前期我带了很多流量给她，现在她能接住我给的流量，而且反哺给我，甚至再去混很多圈子，帮我筛选带来新的用户，我们之前形成良好的循环互动，自然是多赢的结果。


可能你要问了，我前面提到的同学都是有灵气有一定工作经验的，如果你现在底子特别差，没有什么能给前辈做置换的。那应该怎么办？


这种情况下最好的方式就是为知识付费，争取一个让前辈看到你的机会。哪怕在一个小小的点上闪光，也有可能被注意到。比如我还有一位志愿者想去教育公司的用户运营。她默默在做的一件事就是把我的音频课和书上的知识点用思维导图的形式画出来，在各个群分发。这个点让我注意到她了， 给她推荐了很多机会。


请记住，人生是一个别针换别墅的杠杆游戏。你不可能一小子换个大的，但是可以从踮起脚够得着的地方去一点一点撬动。


三、总结提升


今天主要为大家讲解了如何成为前辈愿意帮带的潜力新人。主要为大家介绍了潜力新人一般是具备的那几点特质，分别是


1、有用户思维，为用户创造价值


2、有灵气，遇事儿愿意往前多想一步


3、长存感恩之心，坚持做个长期主义者 


4、有打造个人 IP 的意识，会链接，善于借势
