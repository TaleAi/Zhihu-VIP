## 简介
想进入互联网大厂运营岗位？最重要的是掌握运营的基础逻辑。本专栏带来一线大厂经验，抛弃枯燥的理论和专有名词，结合具体案例讲解运营的玩法和方式，教你掌握运营底层能力。掌握运营逻辑，获得进入互联网大厂的通行证。


互联网领域有一个共识：「百度看技术，腾讯靠产品，阿里拼运营」，阿里的运营一直被人们广为赞叹。


在互联网下半场，「运营驱动」成为共识，各个行业如饥似渴的寻找「好的运营」。


运营更是一个人的底层能力，如何把一手普通的牌打赢，考验的就是运营能力。


让产品被更多人看到是运营，发一条朋友圈销售更多货物是运营，打造自己的个人品牌也是运营。


可是大多数运营人只知道埋头做事，陷入到每天事情的打杂中，能力停滞不前，没有上升空间。本专栏阿里运营人小马鱼为你揭秘鲜为人知的阿里运营秘籍，带你了解运营的底层逻辑，构建系统的运营体系。
