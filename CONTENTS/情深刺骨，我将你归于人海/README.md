## 简介
烟花一瞬间，却困我一世仰望。


《论白月光的杀伤力》


我穿回十七岁，看到男友深爱白月光的模样。


他们在路灯下相拥，飘雪中接吻，课堂上偷偷用笔戳闹对方。


他的少女笑容开朗、熠熠生辉。


我僵在坏路灯下，哭得多难堪。


《星辰交错》


异地恋第五年，我没忍住考研二战的孤寂，连夜跑去了邓源学校找他。


我找到他的时候，他正跟学妹讨论课题。


饭桌上，他跟服务员去选食材。


而他的学妹跟我说：「你总不能阻止他奔向更好的人吧？」


几个月后，我二战上岸，邓源来找我，问我为什么不报他的学校。


我说：「你总不能阻止我考上更好的学校吧？」
