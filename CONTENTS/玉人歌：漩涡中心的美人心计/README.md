## 简介
「心机」女主逆天改命，活出自我。


专栏收录了各种「心机深重」的美人的故事，她们中有用尽十八般计策勇敢追爱的豪门贵女，也有在敌营中忍辱偷生，用心计复仇的亡国公主······在男尊女卑的古代社会，她们是一股与众不同的泥石流，特立独行，敢爱敢恨，用智慧谋略对抗命运，活出自我。


**迟归**


我是京城男子闻风丧胆的大龄恶女。


为能嫁出去，我不惜敲晕亲妹，顶替她嫁给了我爹的宿敌。


没承想原该剑拔弩张的婚后生活，却意料之外地和谐。


直到冬猎那天，我拉开弓箭，向我「亲爱的夫君」直直射去······


 


**已临晚**


我曾亲手把刀捅进最心爱之人的胸膛。


多年后，他的亲弟打着为兄长复仇的名义，用最残暴的法子禁锢我，折磨我，占据我。


而我忍辱偷生、抽丝剥茧发现的真相，却比他的手段更为残酷。


原来复仇的人，从来不只他一个。


 


解锁本专栏，看风起云涌的朝堂下、暗潮涌动的后宫内，命运坎坷的倾国美人，如何用「心计」逆天改命，活出自我。
