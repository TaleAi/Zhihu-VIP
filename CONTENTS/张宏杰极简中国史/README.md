**他是百万畅销书作家——**


著有热门图书：《简读中国史》、《曾国藩的正面与侧面》、《饥饿的盛世》、《大明王朝的七张面孔》等销量超过 200 万册！央视《百家讲坛》观众最喜爱学者，一档《成败论乾隆》栏目夺得 2013 年收视冠军。 


**学术大咖、畅销作家、商业精英都是他的读者**



> 莫言：他擅长把熟悉的历史用陌生又新鲜的笔触表述出来。

> 马伯庸：他的作品既真实不虚，又突出人物心路，非反复穿模咀嚼不能为之。


**他是张宏杰，被称为「最会讲历史」的人。**师从葛剑雄、姚大力等历史学巨擘；复旦大学历史学博士、清华大学博士后、中国人民大学历史学院研究员；央视《百家讲坛》主讲人；历史类畅销书作家、中国作家协会会员。


张宏杰讲中国史，给你一个全新完整的中国历史概念，跳出通史套路，抓住核心脉络，摆脱错误常识，发现历史另一面，多学科解读方式，把握中国历史独特性。用史家眼光、文学笔法，让历史比小说更精彩，揭露历史游戏规则，读懂中国史的变与不变，让你不用记不用背，快速了解中国历史的必备指南。
