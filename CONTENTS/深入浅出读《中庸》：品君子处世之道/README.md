## 简介
本课程由北京大学中文系教授王岳川主讲，带大家一起走入儒家精华的经典之作《中庸》，领悟国学智慧。


  



《中庸》是儒家经典的《四书》之一


**中庸之道**是中国思想史上


重要的**哲学观念和践行标准**


史论结合，纵横交错，


通过开掘个体身心修为之维，


将外在事物和天地大道相合相生，


显示了中国哲学思辨性和践行性。


  



现实生活中有些人把中庸混同于平庸


相反，它是一种力求完美、力争上游的进取哲学


那中庸真正的含义是什么？阐明了哪些观点？


为什么会被很多人误解误读？


这本书中究竟告诉了世人


哪些为人处世之道、君子之道乃至学习方法？


  



**朱熹形容说:「《中庸》亦难读,看三书后,方宜读之。」**


《中庸》是「四书」最后的结晶


是四书之中最难理解的一本书


让我们跟随王教授一起


深入浅出地学习《中庸》


从中寻绎古代先贤的睿智


感悟中国人精神的真正特性


**学习立身处世保持和平、中正的态度**


  



专辑特色
----


最难懂的儒家经典，却是最易懂的讲述


最古老的圣贤智慧，却是最当代化的解读


最被误解的儒家经典，拔除迷雾还它本来面目



你将获得：
-----


1、如何为人处世以及获得幸福的大智慧


2、领悟中庸之道，从焦虑中解脱出来，疗愈心灵


3、提升管理水平和领导能力，成为圣明的领导者


4、提升格局观和人生境界
