## 简介
1990 年在山西铁路上五吨天价羊绒突然人间蒸发，铁路公安随即赶往现场展开调查。放羊杆子内涵玄机的放羊老人、用棉花擦铁轨的老幼妇孺、无意透漏惊天秘密的男孩、神秘的村子，五吨天价羊绒的失窃到底跟谁有关？本专栏将带你感受真实的破案现场，揭穿五吨天价羊绒失窃背后的惊人真相！


**五吨天价羊绒在火车上突然人间蒸发**


有位港商一下子向内地订购了二十吨羊绒，为了尽快推动内外贸易，各部门一路绿灯，全力支持。然而，就是这样一笔标志性的生意，竟然硬生生地出问题了。羊绒从西部运出，到天津加工后发货香港，二十吨羊绒装上车，到天津站居然只剩十五吨。也就是说，一丢，就丢了五吨羊绒！五吨羊绒，说没就没，到底是预谋盗窃，还是坚守自盗？


**放羊老人竟是江洋大盗？**


火车经过时，牧羊人等第一节车厢快要冲出他们的控制范围，最头上的那个「苏武」忽然把手一挥，所有「苏武」把杆子往地上斜着一插，还拧上两下，那节杖奇特的杆头便把焦炭从车上钩了下来。眼看焦炭落到了铁路两侧，列车飞驰而过，落日余晖中从离铁路四五十米的一排黄土小丘后面突然冲出一群推小车的老乡来，上百号人熙熙攘攘。团伙作案？这群人究竟是谁？


**路遇小男孩，无意探知铁路旁边村落的秘密**


铁路神探老刘故意向小男孩问路，并送玉葫芦给孩子作为酬谢。随后装作轻松的样子，问小男孩：「小家伙，你会扒火车吗？」。这一句话把助手吓了一跳。难道主任要教唆这么小的孩子扒火车？


主任想的是童言无忌，也许能有线索。但老刘并没有得到消息，小男孩与他叔叔们都不会扒火车。告别时，小男孩走了两步忽然回头，一本正经地对老刘说道：「我姐姐会。」


**一个八岁的小女孩居然会扒火车？神秘的村子究竟还隐藏着什么秘密？**


**老幼妇孺一人一大块丝绵，帮铁道部擦铁轨？究竟是好心还是有预谋？**
