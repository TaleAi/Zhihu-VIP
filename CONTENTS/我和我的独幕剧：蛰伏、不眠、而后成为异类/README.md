## 简介
每个人都是钢铁丛林里的异类！每个异类心里都有一只蛰伏的巨兽……


**- 别墅剧本杀派对 -**


隔壁的女邻居邀请我大年初一去她家里玩剧本杀，


但是我并不是她的邻居，我只是借住在她隔壁的陌生人而已……


  



**- 作家之死 -**


我是一名过气的作家，结婚生子，按部就班是我的日常生活，


直到一天，一位圈内知名大作家的编辑找到我，告诉我大作家的死并不是意外，而是谋杀。


她请求我为大作家撰写人物传记，而案件的真相就藏在传记里。


  



**- 在恐怖本里撩 DM -**


好友迷上了剧本杀，天天去，也去不腻，


女友不放心他自己去，于是他便拉我作陪，


那天拼场的玩家挑了个恐怖本，来开本的 DM 是个漂亮姐姐。
