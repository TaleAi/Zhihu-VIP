## 简介
21 世纪的竞争不是企业与企业之间的竞争，而是供应链和供应链之间的竞争。

在消费电子领域，苹果的成功就是供应链的成功；

在汽车行业，丰田凭借供应链管理实效高效运作；

在服装行业， ZARA 因供应链优势迅速开拓市场。


**华为总裁任正非说：供应链只有一个，关系着公司的生命，一旦供应链问题解决了，也就基本解决了公司的管理问题。**


**本课程主讲——辛童深耕供应链管理 20 年，曾供职苹果、华为、ABB 、联想等一线公司。在本课程中，她将透过华为成长和发展的脉络，分享华为应对各种供应链管理问题的解决办法。**
