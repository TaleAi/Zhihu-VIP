## 42.商业模式第一关：风口的猪
今天我们讲商业模式第一关，风口品类。


有一次我跟小米的创始人王川聊天，他说他做小米合伙人受雷军的启发特别大。雷军有一次让王川一起投资一个公司，叫 UC Web 。 UC web 雷军说这个公司特别好，王川看了半天说看不懂，也不知道怎么赚钱的，王川就没有投。


结果是什么？ 王川错失了一个 10 亿美元的机会。


雷总靠投 UC Web 还是赚了不少钱的。如果当时王川听雷军的建议, 投 UC Web 最起码能赚十个亿。王川后来就反思就是这句话，我是猪！ 


什么意思就是人一定不要跟封口作对的人做生意做商业一定要找准风口，雷总经常讲叫天道不酬勤，酬什么，酬风口。其实雷军创立小米讲的最多的一句话也叫顺势而为。顺势而为就是顺用户的大事。


这是我们今天讲的最重要的一个关键点，也是知识点叫风口的猪。


什么叫风口的猪？就是我们要找准大风口。比如说前段时间区块卖很火，好多人扑上去，它是风口么？其实很多时候不一定。其实好多人摔得头破血流。选风口的猪是有诀窍的，我们想跟大家讲三个核心的干货点，怎么样选好风口的猪？


第一， 又肥又大。


什么叫肥大？就是肥就是厚，利润高；大就是份额大市场大。


比如说餐饮他就是又肥又大的生意。一定不是来自于虚的判断，而是来自于用户真实的感受。如果我们做一个婚纱摄影，那我们怎么样找到风口呢？用户对婚纱摄影哪个需求量最大，这就又肥又大。我们会发现比如在天猫里面搜索很大的两个字叫旅拍，婚纱摄影加旅拍就是一个很大的风口，我们有个学员叫伯爵，只作旅拍，其他都不做，一年能做20多个亿，这就叫选对了风口。第一个词记住，又富又大。


第二高频，就是用户的消费，用户的使用是高频的。


打车市场崛起的时候，其实先融到资的是专车，后来叫代驾，最后被看好的才是打车。 但是我们知道现在打车最值钱，滴滴最值钱，百亿美元级别的公司。一个很大的风口是什么？高频，因为打车是很高频的行为。再比如团购，团购后来为什么成就了美团。有那么多团购，美团为什么起来。一个最重要的核心外卖，外卖最高频，高频会降维打击低频，所以来讲各位选好高频的特别重要。有的生意来讲，一上来选择很低频的，有的生意可能一辈子只用一次，这个要么他很肥大。比方婚纱摄影一辈子只有一次，但是它很肥大，但这就挺好的。当然有的产品可能客单价又很低，又不怎么赚钱，可能用户的使用频率又很低，那就很麻烦，所以各位高频很重要。


第三个干货点叫用户的转移点。


现在社会来讲就是用户的转移成本是很低的，就是用户经常会发生大的转移，我们要抓住用户这种转移点。最近几年有好多新的品牌的崛起，比如知识付费。知识付费其实是从过去买书的转移到音频。比如过去我们经常是APP，我们能浏览核心在APP上，所以说做好流量分发的平台它就很值钱。但是现在我们看到用户转移到超级APP，比如微信是个超级APP，比如抖音是个超级 APP ，这种超级 APP 让这种流量分发的平台就开始变得不值钱了，这叫用户转移点。 


再给大家举个例子，比如说社交电商，你会发现过去我们电商买东西都是在京东天猫，但现在来讲你会发现比我们拼多多。两三年之间一下子干到几千亿的规模，是过去淘宝七八年才干成的规模，其实是因为用户通过社交、通过拼团的方式买东西，也变成了一个非常大的用户转移点。要抓住真正的用户转移点，当然讲有可能是伪的。比如说区块链，我们不要追概念，我们要追用户真正的转移点，比如说拼多多是吧？你看我们好多三四线人真的用它。但是区块链你会发现都是专家在用，它其实大部分人并没有真正的用它，所以我讲有的概念是虚的，这叫用户转移点。 


所以我讲抓住风口有时候不容易。记住三个关键词，肥大高频用户转移点，也希望你不管在做任何事情的时候，首先考虑一下我离风口是远还是近，一定要想办法做风口的猪。
