## 简介
爱情像张纸，秘密是团火，纸永远包不住火。
有些人只能遇到一次，有的爱可能也只有一次。
爱情不是游戏，每个人都玩不起。


相亲中意外遇到心动女孩，却意外发现她曾经不堪的过去。


我依然爱她，但是我们却不能在一起了。


金钱，就是一面爱情的照妖镜，是人是妖无从遁形。金钱与爱情，永远都是一道必选题，也是一道单选题。


——《爱情不是游戏》


作为被欺骗者，我恨那个隐婚的男人，他打破了我对爱情的美好向往；


作为正房，那个女人肯定也恨透了他拈花惹草的老公。


我要报复男人，她要报复老公和我。


爱情不是游戏，无法重来……


——《非诚勿扰》


我到丽江旅游，老天安排了一个女孩和我拼房。


她年轻漂亮，气质高雅，我很快就迷上了她；


可告白的时候我才知道，原来她藏着一个秘密……


——《遇见》
