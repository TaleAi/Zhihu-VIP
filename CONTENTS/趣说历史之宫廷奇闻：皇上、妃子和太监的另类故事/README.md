## 简介
本课程为趣说历史系列课程的第四弹，主要讲述宫中帝王、妃子和宦官们的那些鲜为人知的另类故事。离奇又有点魔幻的故事，为你揭开宫廷深院里别样的世界。


每个学不好历史的人，背后或许都有一个让人昏昏欲睡的历史老师。在很多人眼中，历史只是枯燥无味的事件记录，或是一些老生常谈的兴衰道理，实在让人无法提起兴趣。


但其实，历史作为几千年人类智慧的结晶，其中饱藏着各种各样我们生活的映射和社会的缩影。光怪陆离的传奇故事、曲折离奇的爱恨情仇、尔虞我诈的权利纷争，都充满着人性的色彩和光辉。


那如何愉快不枯燥的了解历史呢？这门课程来帮你。


趣说历史课程由知名历史杂谈讲师「铁扇」老师主讲，以风趣幽默的叙述、妙趣横生的语言和引人入胜的故事，形象生动的讲述中华上下五千年的奇人和趣事。从娇柔抚媚的乱世红颜，到开疆辟土的传奇帝王;从吟诗作画的骚人墨客，到英勇善战的三国群雄。发掘每个历史人物背后鲜为人知的一面，还原一个比正史更加精彩的历史真相，让你在轻松愉快的氛围里感受历史的魅力。


本课程为趣说历史系列课程的第四弹，主要讲述宫中帝王、妃子和宦官们的那些鲜为人知的另类故事。大多数人想象中的深宫，即是《甄嬛传》的描绘，勾心斗角，拉帮结派，充斥着不是你死就是我亡的争斗。不可否认这是宫中状况的一部分，但其实在这些争斗之外，也有着一些皇帝的荒诞趣事，娘娘不同寻常的情感历程，以及太监的坎坷升职记。离奇又有点魔幻的故事，为你揭开宫廷深院里别样的世界。
