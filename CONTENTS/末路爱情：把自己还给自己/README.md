## 简介
我们要永远成为独一无二，而不是退而求其次。


爱情有很多种，最好的是：彼此深爱，则好好成全。走到末路，则果断放手。有些人注定只能陪你一阵子。


《意外的孩子》


在我怀孕之际，丈夫带回了一个八岁的孩子，让我给她当妈。


我笑了，我是没有孩子吗？


想要收养她？可以，除非我们离婚。


《白月光和白米饭》


我被他的一句十年暗恋拨动了心弦，我以为我是他的白月光，可是等到在一起才发现，原来他心里还有个朱砂痣。


白月光瞬间退位成了白米饭。


既然如此，你也就是姐的一个过客了。


男人，多的是。


比如我身边这一个。
