## 简介
面具之下，难辨真假。


生活让每个人都戴上了面具，面具的背后，藏着不同的善恶美丑。有人选择与人伪善，有人选择笑里藏刀，有的人选择伸张正义，有点人选择了掩埋真相。也许你的身边，正藏着无尽的黑暗，防人之心不可无，可面具下的他（她）该如何防？

我在黄金地段租了个房，房东 给我免租，我以为是凶宅，直到被装进冰柜才意识到贪便宜要付出代价。


  

**「我怀疑女友是逃犯」**  

我在工地捡了个女朋友，警察上门后她赶紧躲起来，我怀疑她是逃犯，到底要不要说？


  

**「夺命外卖」**  

独居女孩横死家中，我是唯一指定的外卖骑手，嫌疑最大。我真没杀人，到底谁是凶手？
