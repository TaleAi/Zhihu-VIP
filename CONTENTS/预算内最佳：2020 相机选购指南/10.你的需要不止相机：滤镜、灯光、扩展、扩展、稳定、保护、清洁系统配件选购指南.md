## 10.你的需要不止相机：滤镜、灯光、扩展、扩展、稳定、保护、清洁系统配件选购指南
对于摄影来说，要获得更好的图像效果，并不是有相机和镜头就足够了，实际上，相机配件也扮演着十分重要的角色。


相信大家都有这样的体验，线上购买相机时，各种包含配件的套餐让你眼花缭乱；线下购买相机时，销售人员也会安利推荐各种配件：「要不要来个高速存储卡，可以获得更好的读取速度哦」，「要不要来个三脚架，可以拍到更加专业的照片哦」。


是的，相机配件也是摄影器材中很重要的一环，有的能够提升我们的拍摄体验，有的能够保护我们的相机，有的能够实现更加复杂的创意。


那么，相机配件到底起到了什么作用，而我们自己又需要什么样的配件。我们来一起看看。


**滤镜系统**
--------


滤镜本质上就是放置在镜头前的一块玻璃，通过采用的不同的制作工艺实现不同的效果。


**UV 镜**


UV（ultraviolet）指的是紫外线，UV 镜功能就是过滤紫外线，由于紫外线也会引起胶片的感光，所以胶片时代的 UV 镜是很重要的，但是到了数码时代，UV 镜的作用几乎只剩下了保护镜片了，这个道理其实跟手机贴膜差不多。


优点：


1. 比如前镜片落灰，擦拭的时候总是要小心翼翼，防止刮花，如果安装了 UV， 就可以放心大胆地擦拭；
2. 如果镜头不慎掉落在地，由于 UV 镜在前，会吸收大部分的冲击力，一般情况下，基本上都是 UV 镜碎裂，而镜片无恙；
3. 如果你竟然去海边，沙漠，或者大风的地方，沙尘和水汽也会影响镜片，有了 UV 之后也可以对镜头进行保护。

但是也有几点需要注意：


1. 在暗光或者拍摄夜景的时候，由于多了 UV 一层玻璃，会产生鬼影和光斑，这时候建议取下 UV 镜进行拍摄；
2. 不建议购买太过便宜的 UV，比如几十块钱的，有可能会影响画质；
3. 没有必要追求顶级昂贵的 UV 镜，比如 B+W，一方面是因为实际上的画质提升也许肉眼根本看不出来，再加上市场是假货横行，一般来说，一些国产的具有多层镀膜的 UV 镜也是不错的选择，这些 UV 镜的价格基本上在百元左右。

 


**ND 镜**


风光摄影中，经常会用到「慢门拍摄」这种手法，顾名思义，就是使用很长的曝光时间，但是如果在白天，光线很强，即使你把光圈和感光度调到最低，有可能快门速度还是很快，而且光圈过小的时候，有可能会产生波的干涉，导致画面的锐度下降，从而影响画质。


在视频拍摄中，如果你使用 Log 模式拍摄，很多 Log 模式都是 ISO800 起跳的，而且你还想使用大光圈拍摄，为了获得 1/50 秒具有电影感的视频的话，画面肯定是过曝的，这时候，你就需要 ND 镜的帮忙。


ND 镜叫做中灰镜，或者减光镜。那么 ND 镜应该怎么选？首先来了解下 ND 镜的等级。


ND 镜的等级一般是用「ND」+2 的次方值表示，比如 ND2，ND8，ND64，ND1000（可以理解为 ND1024）.


每减一档光，曝光时长需要增加一倍，比如原先不加 ND 镜的曝光时间是一秒。


* ND2，表示此时需要曝光时间是 2 秒
* ND8，表示曝光时间时长需要增加三档，即 1×2^3=8s
* 以此类推。

一般来说，风光摄影，建议 ND8 或 ND16，ND500（512）或 ND1000（两组各选 1 片）。


ND8 或 ND16 适合较弱的光线环境降低快门速度，ND500（512）或 ND1000 适合在光线较为明亮的白天使用。


ND 镜又分为固定值 ND 和可变 ND 镜。一般来说，固定值 ND 的画质是要好于可变 ND 镜的，但是可变 ND 镜胜在方便，可以快速调整灰度值。


目前来看，市面上在售的可变 ND，便宜的画质实在惨不忍睹，画质上能说的过去的，价格又实在是太贵。所以还是不建议选择可变 ND。 


而滤镜又分为方片滤镜和圆形滤镜，那么，选方形滤镜还是圆形滤镜？


一句话，如果你的镜头不是灯泡头，选圆形滤镜，毕竟圆形滤镜装不到灯泡头上。


灯泡型镜头：多出现在广角镜头的设计上，比如佳能 EF 11-24/4L USM、尼康 AF-S 14-24/2.8G ED、适马 12-24/4 DG HSM A016，三阳 14 F2.8。由于外组镜片向外凸出严重，像个灯泡，故得此名。


![img](https://pic2.zhimg.com/v2-32934ba215fcbbd4860315dd4237fb9d.webp)

方片滤镜，就是在镜头上增加一个底座，可以插入不同的滤镜，可以兼容更多的镜头和滤镜，所以也称之为方片系统，更加的进阶和专业，当然花费也是不菲的。


![img](https://pic2.zhimg.com/v2-1d6e435535c9c71912babed0764d3ec6.webp)

 


**GND 镜**


与 ND 镜类似，具有减光的功能。区别在于 ND 是全画面的降低光线亮度，GND 只是降低一部分画面的亮度，这里是「G」是指 Graduated，叫中密度灰镜，也叫渐变镜。通常是一边为透明的，然后渐渐过渡到暗灰色。 


比如拍摄落日场景，天空较亮，而地面较暗，使用 GND 可以降低天空的曝光，从而获得更好的画面效果。


由于圆形的 GND 的过渡位置是固定的，使用 GND 还是推荐上文提到的方片系统，可以自行调整镜片的位置。


 


**CPL 镜**


CPL 全称 Circular-Polarizing Lens，即圆形偏光镜，这里的偏，指的是「偏振」。


光是一种波，自然光能够朝着不同的方向振动，这种现象叫做偏振，而 CPL 镜的作用就是过滤掉特定方向振动的光线。从而实现消除反光，加深色彩饱和度，较少大气中雾化的影响等功能。


由于可以消除反光，比如玻璃或者水面的反光，所以 CPL 镜是唯一一个无法通过后期手法代替的滤镜。但是由于 CPL 过滤了一部分的光线，所以会降低一档左右的曝光，而且偏振镜片对于色彩饱和度的影响，并不是适合所有的拍摄题材。比如如果你是一个人像摄影师，那么 CPL 对你来说作用就不大。


很多 CPL 镜都会有暗角的问题出现，建议购买时选取较薄的滤镜。


 


**柔焦镜**


在很多情况下，照片并不是越清晰越好，比如拍摄人像时，画面朦胧一些，可以获得更加唯美梦幻的感觉。


![img](https://pic4.zhimg.com/v2-185a859d653c50c1d0acfcf015c49633.webp)

柔光镜由透明玻璃做成，玻璃的表面被腐蚀成凹凸不平，当光线经过镜片表面时就会被散射，从而使被摄物的轮廓柔化、反差降低，产生一种「柔光」效果。


不过「柔焦效果」也可以通过以下的方法进行替代。 


前期方法：


1. 镜头上套点东西：丝袜，塑料纸；
2. 涂点东西，凡士林，大宝都行，边缘多一点，中心的用纸擦一擦（可以涂在 uv 上），注意尽量涂少一些，这里建议直接涂在一些廉价 UV 上，因为镜片并不太好清洗。

后期方法：


1. 新建一个高斯模糊的图层，柔光叠加；
2. 有很多柔焦滤镜的插件，比如 Nik 套件中的 color efex。

柔焦镜片的价格并不太贵，如果经常需要拍摄此类风格的照片，购买一个也无妨，几十块钱左右。


 


**星光镜**


使用星光镜，可以获得类似于星芒的效果，以发光点为中心，分 2 线、4 线、6 线、8 线、16 线等。


原理是在无色光学玻璃表面有规则地蚀刻一系列平行线条，产生光的衍射作用后而形成光芒四射的效果。


![img](https://pic1.zhimg.com/v2-1a46f73b880366ea99ed028ab57fdedf.webp)

除了拍摄星光和灯光，也可以拍摄珠宝，波光粼粼的海面，来表现闪耀的光芒。


  



**红外滤镜**


我们目前使用的数码相机，在 CMOS 之前会加上一块红外/紫外滤镜，所以市面上卖的红外滤镜，其实作用不大，如果真的需要拍摄红外照片，建议进行改机，即把感光元件前的红外滤镜去掉，但是也就意味着你无法拍摄正常颜色的照片了，建议使用备机或者不用的机器进行改机操作。


 


**色彩滤镜**


本质上就是不同颜色的镜片，由于数码时代，色彩调整是一件很简单的事情，并不建议购买。


**灯光系统**
--------


摄影是用光的艺术，当自然光线不足以满足我们的需求的时候，就需要我们去改变或者创造光线。


**反光板**


反光板也许是大家最常见的摄影配件之一了，现在市面上的反光板基本上都是「五合一」的。具体介绍如下：


![img](https://pic3.zhimg.com/v2-8c45455ef50f7c7d5d456c04c607b205.webp)

**热靴闪光灯**


闪光灯可以直接插在相机的热靴底座上，但是直闪的效果往往很生硬，还是强烈的建议你买一对离机引闪器，这样就能获得更好的光线方向。


![img](https://pic1.zhimg.com/v2-b48dd6d5512bea591f238cc6bdddc01c.webp)

除了原厂的闪光灯之外，也有一些口碑不错的国产闪光系统，比如神牛，永诺之类的品牌，具有很好的性价比。当然也有那种固定的的闪光灯，可以获得更稳定的色彩和功率输出，适合室内固定场所使用。


 


**LED 便携灯**


近些年出了一些 LED 便携照明设备，作为一个常亮光源，可以调整色温和亮度，在拍摄静物或者小范围补光时，或者拍摄夜景人像时，也能获得不错的效果。


![img](https://pic2.zhimg.com/v2-6da9b8388c778bc3df894afdb25d045f.webp)

价格在百元到千元不等，购买时注意功率，续航和颜色调整功能即可。


  


**柔光罩**


为了获得更好的柔光效果，通常我们会采用柔光罩或者柔光伞来增加「光线的面积」，从而使光线更加的柔和，不论是棚拍还是外拍都可以使用。如果是外拍尽量选择容易折叠，轻便，方便携带的。


![img](https://pic4.zhimg.com/v2-f4c57adfd84b7ca5fb273984da9d5049.webp)

**扩展系统**
--------


**握持系统**


现在的微单机型，机身小巧，但是带来的问题就是握持感不佳，手大点的人会感觉小拇指无处安放，所以需要一些配件来增强握持。手柄是一种常见的扩展系统，除了有增强握持的功能之外，还可以在手柄中安装电池，增加相机的续航力；有的还具有快门和调整波轮，方便竖拍。


有些镜头很大很重，可以通过增加手柄来平衡重量，获得更好的拍摄体验。


![img](https://pic4.zhimg.com/v2-4445f513ecad00f5d70241ffc3ad7c3d.webp)

**兔笼**


兔笼并不是拿来养兔子的，如果你是一个视频工作者，那么兔笼也许是不可或缺的配件。


![img](https://pic3.zhimg.com/v2-6947b414bffa3cbedaaaf28d32df6cf9.webp)

兔笼的作用主要有两个：


1. 保护相机，类似于给机身包裹了一圈机械外骨骼。
2. 扩展其它外设配件：拍摄视频时，有可能你需要外接监视器，麦克风，三脚架，稳定器，遮光罩，补光灯等设备，但是如果把这些设备固定在相机身上呢，这时候就需要屠龙的帮助了，他身边具有很多标准螺纹接口，可以快速地外接各种各样的配件。

常见的品牌有 SmallRig。


**稳定系统**
--------


**三脚架**


居家防身，旅行拍摄的利器，你要没个三脚架，都不好意思说自己是搞摄影的。


记住，不要买太便宜的三脚架。


注意事项：


1. 三脚架的管径越大，承重力也越大，越稳定；
2. 球形云台的球径越大，承重力也越大，越稳定；
3. 球形云台除了便携，除此之外在三维云台上没有优势；
4. 脚架的节数越多，收纳长度就越短，越便携，但稳定性也会随之下降。

有一句经典的话：买了三个三脚架之后，你才知道自己需要什么样的三脚架。


常见的品牌有：捷信，曼富图，百诺，马小路等。 


 


**稳定器**


通常是拍摄视频时使用， 其实大部分的稳定器的稳定效果都差不多，购买时需要注意两个点，一个是稳定器自身的重量，一个是最大承载量，如果自重较大，那么长时间拍摄有可能就会很累，如果最大承载量不足，那么在一定场景下就无法进行拍摄。


常见的品牌有大疆，智云，飞宇等。


![img](https://pic4.zhimg.com/v2-4ef590d0c81fdd5f3422869a9de80bc0.webp)

购买时需要注意以下几点：


1. 买新不买旧，新的往往具有更多的功能和性价比；
2. 根据自己的相机重量选购合适的稳定器，比如你的相机只是小型微单，那么就没有必要选择大型的稳定器；
3. 大型的稳定器建议构面具有提壶模式的，这样比较省力；
4. 外接能力是否完全，外接位置是否合理，建议购买稳定器前，可以先去实体店试用。

**保护系统**
--------


**遮光罩**


遮光罩严格意义上是镜头配件，顾名思义，可以防止眩光的进入，除此之外，遮光罩最大的作用就是保护镜头，当镜头不慎跌落时，首先受到冲击的是遮光罩，从而避免镜头被摔坏。


经常见到很多人遮光罩反装，这样既起不到遮光的作用，也无法保护镜头而且还会遮挡手动对焦环，所以说反装遮光罩是一种比较「蠢」的做法，建议每个镜头都要购买，并且正确使用遮光罩。


![img](https://pic4.zhimg.com/v2-7155c61c36efa7494f00c70324738622.webp)

当然如果你想拍摄眩光的画面效果，或者使用机顶闪光灯是遮光罩遮挡光线时，可以暂时取下遮光罩，但是平时拍摄时可以一直安装上。


  



**相机包**


其实普通的背包也是可以放相机的，为什么还需要相机包？


相机包相对于普通的背包多出了以下特点:


1. 内胆保护层，防止你在走路时磕磕碰碰，损坏器材；
2. 可以调整的内部空间，如果你不只是一个器材，那么相机包可以通过一定的方式规划内部空间，从而放入各种不同的器材，器材之间也不会产生磕碰；
3. 快取，很多相机包支持从侧面或者背面取出相机；
4. 防水，防盗等功能。

![img](https://pic2.zhimg.com/v2-84d23e3060265a1a6e94c4ec37cd5dd2.webp)

如果是城市通勤，近途拍摄，选择单肩的摄影包即可；如果需要远出旅行，推荐双肩摄影包，会比较舒适。


常见的品牌有：国家地理、乐摄宝等。


  



**防水/防沙罩**


在雨天，海边拍摄的时候，一定要注意防水，这时候可以使用防水罩；


同样的，由于沙漠或者沙尘很大的地方起风的时候，细沙很容易钻进相机内部，有可能损坏相机，所以去这些地点时，最好也要配备相应的保护套。


**清洁系统**
--------


相机是精密的器件，所以要注意清洁和保养。


常见的清洁产品有：


**气吹：**吹落镜片，相机，反光板上的灰尘，除了常见的气吹之外，也可以购买罐装的高压气吹，有更好的吹力。注意尽量不要用嘴吹，最大的风险就是会吹出口水，更加难以清洁；


![img](https://pic1.zhimg.com/v2-639733901c3b3f0c5772d52fc291b1bb.webp)

**毛刷：**扫去灰尘，也可以使用干净的毛笔代替；


**镜头纸/布：**用来擦拭镜片上的污渍和灰尘，建议气吹后使用，**尽量购买较贵的镜头纸，**比如蔡司的擦镜纸，便宜的纸会有掉渣的风险；


**传感器清洁棒：**用来擦除 CMOS 上的污渍，注意，清洁棒是一次用品，不可以多次擦拭。


![img](https://pic3.zhimg.com/v2-710b0e5d6dd2c092fd507cc6b73e42e2.webp)

配件的选购实际上也是根据需求来的，如果不怎么拍摄视频，那么稳定器，监视器对你来说作用就不大，如果你只喜欢人像摄影，那么渐变镜对你也没有什么意义。


但是对相机的保护和清洁的配件，不论是从事哪种类型的拍摄都需要准备的。如果具有多个器材，那么购买时，也尽量考虑配件的通用性。
