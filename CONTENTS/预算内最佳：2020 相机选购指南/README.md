## 简介
2020 相机选购指南，轻松入手心仪相机。


提到摄影，许多人或许会第一时间想到很多关键词：镜头，相机，曝光，锐度，快门，后期，美女，风景……但这些似乎都不足以概括真正的摄影。 


摄影是什么？ 是你站在哪里，你看到什么，你怎样观察事物，你关心什么， 你的想法，你的感觉以及我们的生活方式。 


而这一切都可以用照片去表达，这就是摄影存在的意义与魅力。


影像的记录和表达，离不开相机。而市面上的相机形形色色，哪一台能够符合你的需求？那么多的品牌型号，如何用仅有的预算，挑选出最适合自己的设备？选择相机时有哪些原则？相机参数又到底代表什么？


现在，这些问题将在这里得到解答。本专栏将为你带来 2020 最新的相机的导购与科普指南，帮助你解决相机选购中遇到的种种困惑，轻松入手心仪的相机。
