## 简介
如果你对图书编辑岗位感兴趣，希望本专栏能为你带来更多的深入的了解。如果你还在出版业的门口徘徊，希望本专栏能成为把你压进去的最后一根稻草……同时本专栏也能助你想明白图书编辑相关的事儿，关于工作细则、所需的能力、薪酬……


「学校内的教育与实际工作存在落差。」


「许多出版机构对于人才需求与职业培训机制不完善之间的矛盾。」


「很多编辑新人其实都还不会学习。」 


这些说多了以后好像成了套话烂话正确的废话，但它们依旧是需要正视的问题，本专栏希望能为解决这些问题做点什么。


本专栏作者陈瑶，是理论物理学硕士出身，「跨专业」进入出版行业，国有出版社编辑+民营出版公司编辑+独立图书策划人+作者，有着多重身份集一身的独家经历。


本专栏通过思维启发+具体方法，从理论到实践，涵盖编辑工作的种种，与你分享作者在从事编辑的过程中的思维体系、方式方法，助你更好地思考编辑这项工作。
