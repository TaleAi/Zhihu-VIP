## 简介
人类一切痛苦的根源，都源于缺乏边界感。


现代人越来越重视个人空间和社交距离，从某种程度上来说，现代人的边界意识在增强，更倾向在社交上保持界限，对某些越界行为时常会感到厌烦和无奈。


* 我明明不喜欢朋友这样对我，为什么我不敢拒绝 TA？
* 我总是很容易被别人的情绪和评价影响，经常怀疑自己是不是很差劲；
* 我很难过，因为我拒绝了别人的无理要求，反而被指责自私。

这些问题常常让人误以为是自己的错，是自己的能力不够好、不擅长社交、不会情绪管理……


但这其实更多是「边界混乱」的问题。


边界力，是现代社会最需要但又是最稀缺的一种底层能力。


俄国著名作家邦达列夫曾说：**「人类一切痛苦的根源，都源于缺乏边界感。」**


本课程由著名关系心理学家胡慎之系统地讲解「边界力」，让我们在做事时能清楚地知道自己想要什么；在与人相处时，能知道什么是自己该负责的、什么是别人要负责的，找好自己的角色定位，不受他人影响。
