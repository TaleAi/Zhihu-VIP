## 简介
对于许多大学生来说，大学代表人生新的起点和希望。然而对于一部分世界观初具雏形、不具备判断力的大学生来说，这意味着第一次面对一个小社会，靠近一个更真实的世界。它是青春的结束，也是直面成人世界残酷的开始。

该专栏将包含 8 个在大学期间发生的误入歧途的故事，第一次远离父母和老师建造的温室，第一次拥有所谓「自由」的大学生，在直面物欲横流的成人世界时，被不受控的欲望吞噬，丧失了判断力。


有人被嫉妒心冲昏头脑，对室友展开报复；


有出于虚荣同时跟好几个男生交往，用他们的钱打造自己富二代人设的女孩；


有质朴的农村男孩当上学生会主席后变得自大虚荣，开始自制奖状，贷款组织活动；


有一整个宿舍的男生因为法律意识淡薄，出于好玩假装自己是女生，在网上通过裸聊和诈骗骗钱……


  



这一个个故事在告诫我们，在大学这个「自由」的微型社会，当初生的欲望不受控制地野蛮生长，踏入深渊有时就在一念之间。


每一个拥有「自由」的人，都当引以为鉴。
