**《装腔作势》**


我开车撞了个小帅哥。


怕他讹我，我骗他车是租的，包是假的，人刚失业，身无分文。


他面无表情：「我也刚被房东赶出来。」


几次交集后，我竟跟这个穷小子搞起暧昧。


就在我心生愧疚，负罪感爆棚，向他坦白其实我是个富婆时……


我发现他是我顶头上司的儿子。
