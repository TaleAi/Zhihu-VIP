本专栏从猫的心理角度逐一分析猫的各类行为背后的秘密，不论你是新晋铲屎官，还是正在观望的爱猫人士，相信本专栏都会给你启发。


为什么要养猫？


猫，这个生活中常常见到却很难琢磨透的生物，外表软萌居多，却也有无毛猫一类另类别致的模样；性格高傲自大，顽劣狡黠的却也不在少数；行为奇诡多变可也不是无迹可寻……


历史长河和现代文明的碰撞，人们对猫有了新的理解和判断，有的人渐渐意识到猫某种程度就和人一样，它的行为匪夷所思只是人类没有掌握那把钥匙——猫的心理。


如果你同样想弄明白「为什么要养猫」这个问题，那你或许可以关注这个专栏。这个专栏主要从猫的心理角度逐一分析猫的各类行为背后的秘密，或许可以给你适当的参考或启发。
