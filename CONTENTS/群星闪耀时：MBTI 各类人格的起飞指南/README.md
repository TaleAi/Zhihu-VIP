## 简介
人类历史浩灿星海，每一个独立人格都闪闪发光~


本专栏针对 MBTI 性格测试，将 16 种人格详细拆解。对探索自我、亲密关系配对、未来职业发展等多个方面都进行了深刻的探讨。以每个人格所代表的名人为例，用 20+ 生动故事案例展现出每个人格的灵魂路径。


MBTI 性格是从 1921 年心理学家荣格提出人格理论后，不断演变发展而来的。它将人的性格分为 4 大方面、16 种类型。  

MBTI 性格共有四个维度，每个维度有两个方向，共计八个方面。分别是：  

**1.注意力方向：**   

E（Extrovert）外向， I（Introvert）内向   

**2.信息获取方式：**   

S（Sensing）真实感受， N（Intuition）直觉为主   

**3.判断风格：**   

T（Thinking）思考， F（Feeling）感受   

**4.生活方式偏好：**   

J（Judgment）判断， P（Perceiving）理解
