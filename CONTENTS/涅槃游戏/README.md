## 简介
游戏者身份确认，欢迎来到涅槃游戏！


128 个已死选手，被卷入诡异的游戏关卡。


系统引诱他们：


想活下来吗？想复活吗？那就来玩通关吧。


  



这里有扭曲可怕的怪物们，


有阴暗血腥的场景，


还有可能背后插刀的所谓「同伴」


……


是弃游消失，还是直面恐惧？


是游戏失败惨烈而死，还是咬牙拼取一线生机——


  



**欢迎来到涅槃游戏。**
