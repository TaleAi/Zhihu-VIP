## 简介
本课程将为你全方位的展示千古一帝唐太宗，波澜壮阔的一生。兄弟相残、逼父退位、开疆拓土、开创盛世……带你看清刀剑无情的密斗，透析人文与历史的交融沉思。


在世界历史长河中，中国的唐王朝一向享有很高的声誉。


唐朝的经济、文化发展对中国乃至周边国家都产生了非常重要的影响。


作为当时最强盛的国家，它离不开它的缔造者——李氏家族的苦心经营。


而这段辉煌历史的开始，就要从唐太宗李世民的贞观之治讲起。


为什么是李世民而不是其他人开启了这样一段传奇的时代？


这位大唐皇帝到底是一位凶狠专制的君主，还是位励精图治、任贤革新的明君？


本课程将为您揭晓答案，全面展现唐太宗李世民开疆拓土，贞观之治的雄伟一生，还有扑朔迷离，亲生兄弟相残的玄武门之变，一代帝王的绮丽后宫，从贤德的长孙皇后到坚毅的武则天。


让你在放松之余能有所收获，能了解李世民这位帝王跌宕起伏的一生，看清刀剑无情的密斗，透析人文与历史的交融沉思。
