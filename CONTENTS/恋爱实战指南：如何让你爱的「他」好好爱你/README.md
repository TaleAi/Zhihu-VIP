## 简介
爱情需要经营，让你爱的「他」好好爱你


爱情中，这些糟心事你遇到过吗？


* 想和「他」说话交交心，最后却吵得面红耳赤；
* 日常交流互动，不会「调情」，死板无趣；
* 你掏心掏肺热情付出，「他」却冷漠不在乎；
* 爱情总是甜蜜开始，不合适、没感觉结束；
* ......

为什么在爱情里，我们总是会因为一些小事情，不欢而散？也许问题在于我们不知道如何正确地跟自己的另一半沟通。


本课程邀请有 20 多年婚恋咨询经验的卢悦老师，手把手教你经营爱情，掌握沟通方法，轻松读懂你爱的「他」，让你越来越会爱，越爱越幸福！
