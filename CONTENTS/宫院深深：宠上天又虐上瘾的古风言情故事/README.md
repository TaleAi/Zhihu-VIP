## 简介
十个古风言情故事，或在深宫，或在深院，欲罢不能的是爱情，刻骨蚀魂的是人心。女主在宠与虐之间死去活来，爱恨纠葛，命运羁绊，前世今生，两小无猜……读后的淡淡怅然，久久不散。


《废后》


只因为在人群中多看了旧爱两眼，皇帝下令屠了旧爱全家。风灵雏淡淡说：废后吧。一步步，她走出绵长深宫。身后的那个男人拥有天下，但绝不可能拥有她。


《我与将军两白头》


我当年拜别师门，出山时曾遇到过一个算命的神棍。


我那会问的是姻缘，这神棍便算准了我将来嫁的是一只恶鬼。


我跟着温三将军，已经九个年头。他屠城不眨眼，孤独爱饮酒，将军披着一张人皮面具，底下是不能被任何人知道的一张脸。但是我爱他，这个毁了世界，也想要毁了自己的男人。


《王爷是个重口味》


「打劫了！」秦圆圆只好懒洋洋地配合：「劫财还是劫色？」天呐，劫匪也太好看了。那人声音清朗，像是泉水低吟，敲在秦圆圆的心上。她犹豫了一会儿才说：「要不你还是劫色吧！劫财我舍不得。」劫匪脸红了，重口味王爷慕容甫居然也有今天！秦圆圆果然是个狠角色。
