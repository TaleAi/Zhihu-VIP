## 简介
单身妈妈，选初恋还是前夫？


寒门贵子周建宇带着投资项目衣锦还乡，遇到多年未见的初恋情人林又青，她没有像他那样去大城市里扎根奋斗，貌似还在底层挣扎。


周建宇心生愧疚和怜悯，试图拯救她，却发现小丑竟是自己。林又青虽然留在小城市，但无论事业，经济还是精神层面都比他这个所谓的体面人富足。


此时的他与林又青重逢，她的从容、乐观和坚韧让他受到了很大的震撼，小城的变化和新面貌也给他带来了极大的冲击，他开始反思自己的追求是否值得以及人生真正的意义。


周建宇逐渐被美丽坚韧的林又青吸引，但一直发乎情止于礼。当他和前妻办完离婚手续再回来时，突然发现自己有了个强有力的竞争对手——林又青的前夫贺冬生。


女主林又青曾经热情如火地拥抱这个世界，却被打击得遍体鳞伤，她没有屈服，没有随波逐流或逃离，而是以女性特有的坚韧和勇敢，在逆境中开出花来，获得了事业的成功和真正的爱情。


最终，她会选择刻骨铭心的年少恋人，还是选择有男子汉气概和担当的前夫呢？
