## 简介
本课程讲述了五胡十六国时期，最具传奇色彩的人物之一慕容冲的一生。带你看慕容冲如何从一国皇子，到国破家亡，沦落为敌国皇帝的娈童，又是如何冲破重重磨难，拿起复仇之剑，复兴故国，成为一国之君的。


「北乔峰南慕容」，金庸的《天龙八部》塑造了一正一邪两位武林高手的形象。


作为大英雄乔峰的反面陪衬，慕容复最终落得一个可悲可叹的下场，不仅复兴大燕国无望，而且迷失心智，疯疯癫癫，结局让人唏嘘。


那么，这个让慕容家心心念念的大燕国究竟是一个什么样的国家？


慕容世家在历史上是一个怎样的存在？


由于历史课本对于「五胡十六国」轻描淡写一笔带过，让大部分人都对这一段历史的认识十分模糊。


实际上，慕容家族是那个时代最优秀的豪门世家，涌现出的英雄人物数不胜数，他们前后一共建立了五个大燕国：前燕、后燕、西燕、南燕、北燕。


自前燕慕容皝于公元 337 年建国，到公元 436 年，沿袭慕容氏国号的北燕为北魏所灭，慕容家的大燕国累计存在了一百年。


这一百年，也是慕容世家在历史上最为风光的一百年！


本专栏讲述了慕容家族里最具传奇色彩的人物慕容冲的一生。


一个亡了国的王子，12 岁那年就遭遇前秦天王苻坚的粗暴侵犯。


而这个人掌握着他和姐姐，甚至是整个家族的生死存亡。


年幼的姐弟俩只有强颜欢笑，屈身侍奉灭了他们国家的敌酋，换来家族的生存。


三年的恋童生涯，十年的隐忍不发，终于让他等来了复仇的机会。


当年的翩翩美少年，如何化身为复仇之神向仇人展开疯狂的清算？


如何登上皇位？


最终又如何走上了毁灭之路？
