在这些故事里，他们倾尽所有，寻仇猎爱，一搏输赢。


《玩家》


有人拿着一个视频，威胁我进行了一个荒唐又惊险的「游戏」。


我一一照办后，发现自己被挟裹着步入了深渊。


我以为对方只是要钱，可「他」想要的却是我的命。


而幕后黑手，居然是她……
