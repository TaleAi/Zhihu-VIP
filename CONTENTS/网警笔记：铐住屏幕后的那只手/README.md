聚焦「网警」职业的新型网络案件悬疑故事集！！


- **直播奇案 -**


我局接到报案，一名女子在直播时疑似吸毒昏迷，现场还有两名幼童，情况相当紧急，我局抽调干警连夜彻查，却发现直播现场竟然很久没人住过。


  



- **买凶谜案 -**  

我在日常网络巡视中，发现论坛竟然有人买凶杀人，并且已经有人回帖愿意接单，原本以为只是网友之间的玩笑，在我攻破对方的邮箱之后，发现凶手已经启程出发。 


  



**- 死亡游戏 -**  

近期有一个本地的微信群引起我的注意，群内都是 20-25 岁的无业年轻人，他们约定参加一个游戏，根据我的调查，他们参加的竟然是一个自杀游戏，所有的参与者，没有一个人能活下来，而我是最后一个。


  



- **网络暴徒 -**


近期社会上出现了一个舆论事件，我妹妹身为记者，发表了一些相对中性的文章，没想到因此引来众多网友的口诛笔伐，导致精神和身体受到双重折磨，患上严重的抑郁症，而我深入调查之后，发现事情远没有那么简单。
