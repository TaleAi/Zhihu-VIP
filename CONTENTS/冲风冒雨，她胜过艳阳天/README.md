## 简介
在黑暗中驰骋，在阳光下闪耀。


她在平凡的世界，长出一身傲骨，不畏惧艰难险阻，冲云破雾。


**《白雪成烟》**


男朋友砸三百万让我在综艺上做白月光的反面对照组。


白月光自强，我摆烂。


白月光坚韧，我绿茶。


白月光优秀，我无能。


最后，白月光皱眉很是担忧：「你这么傻白甜，出了节目被欺负怎么办？」


于是，男朋友既失去了我，也没得到白月光。


还赔了六千万。
