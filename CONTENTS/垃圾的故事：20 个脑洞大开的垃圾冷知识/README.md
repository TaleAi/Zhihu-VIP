## 简介
追溯国内外垃圾历史，勾勒世界垃圾收集分类和处理的清晰脉络，以可持续发展的角度，深入浅出地阐述了垃圾处理和利用的理念、先进技术和发展方向。


乱扔垃圾居然要剁手？古代的「垃圾王」居然是如此富有？


在不同的时代、不同的国家，人们都是如何处理垃圾的？


你是否对垃圾的前世今生感到好奇？


本专栏将通过对国内外垃圾历史的追溯，勾勒出世界垃圾收集分类和处理的清晰脉络，以可持续发展的角度，深入浅出地阐述了垃圾处理和利用的理念、先进技术和发展方向，对于了解和认识人类文明史具有积极意义。


以大量生动的故事和案例，结合简明易懂的介绍，使你在轻松有趣的内容中了解科技知识和生态环境保护理念，寓教于乐，课程内容力求完整准确，紧跟时代步伐。
