## 简介
从童年经典《葫芦娃》到热血动漫《海贼王》，动漫是每个人年少时期离不开的陪伴。可是你能确定，记忆中这些动画片，真的是你以为的那个样子吗？本专栏聚集了一群「动漫解析鬼才」，带你看见那些年追过的动漫里，你没有看懂的秘密。


你知道吗？小时候你以为看明白的那些动漫，其实有可能根本不是你理解的那个样子。


《葫芦娃》不是七个兄弟齐心协力战胜妖怪守护家园的故事，这个童年专属的动画片里，有可能隐藏着人类文明轮回的一个巨大秘密。你以为穿山甲真的是一朵小白莲花？妖怪们的山洞为何有这么多机关？爷爷的真实身份到底是什么？


《海贼王》中最后的 One Piece 到底会是什么？这个热血少年漫的世界里，又有哪些细思极恐的背后法则？


《千与千寻》中千寻是一个没有丝毫魔力的普通女孩，为什么可以一眼从那些猪中认出没有自己的父母？这个看似光怪陆离的妖怪世界中，到底埋藏着哪些现实世界的隐喻？


…… 


来吧，乘上这艘「海盗船」，重启动漫的热血世界。


一场可以刷新你三观的揭秘之旅，带你追寻那些年你没有看懂的秘密。
