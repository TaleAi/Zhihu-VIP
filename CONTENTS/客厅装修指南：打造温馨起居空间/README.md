## 简介
本专栏结合设计师多年研究与设计实践的精髓，完整涵盖了客厅装修的平面布局、装修避坑、色彩搭配和省钱诀窍，帮你有效避开客厅装修布置中的误区，抵达美好生活的彼岸。


客厅是家里的门面，也是大部分户型里面积最大的空间。但在很多人家里，这个空间却并没有被好好利用，或多或少存在以下的问题：


* 客厅没人用，家人各自呆在各自的房间里
* 客厅乱糟糟的，堆了很多东西
* 装修时觉得客厅太空，又无从下手
* 浪费了很多钱装了一面电视墙，好像又没有什么用处

本专栏将结合设计师多年研究与设计实践的精髓，教你从最底层的理念出发，平面家具布置、装修水电布局，再到家具的挑选和风格的搭配，循序渐进教你轻松设计出最适合自己家的客厅，一举破解困扰万千中国家庭的「客厅魔咒」。


对客厅的合理装修改造，能够有效解放出更多的空间，甚至盘活整个家的布置，让所有的房间都能变得更好用。经过优化的客厅，也能让家人重新在屋檐下相聚，使家从「群租房」回归心灵的居所。


专栏内完整涵盖了客厅装修的平面布局、装修避坑、色彩搭配和省钱诀窍，无论是新房装修、精装房改造还是已经入住之后的调整，都可以从中找到完整的解决方案，帮你有效避开客厅装修布置中的误区，抵达美好生活的彼岸。
