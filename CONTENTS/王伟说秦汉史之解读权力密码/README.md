## 简介
以经济和管理学的视角解读历史。


如果你觉得《权力的游戏》里的血腥斗争已经足够触目惊心，《纸牌屋》里的权术谋略已经登峰造极，那你应该看看真实的中国历史，它远比虚构的美剧更加精彩。


如果你想厘清复杂的人际关系，看清事情的发展脉络，那中国历史也是值得你一再参考的绝佳教科书。


但是，中国历史上下五千年，面对如此丰富的材料，最佳的切入点是什么？  

 这个最佳切入点就是「权力」：中国历史就是一部围绕权力展开的斗争史和博弈史。  

 而最佳切入的时代，就是「秦汉」。


先秦两汉，是中国权力格局的奠基期，也是中国社会变革、人才涌动的大时代。  

 当时的英雄豪杰，面对许多前所未有的权力难题：


如果你是皇帝和君主，该如何把权力集中在自己手里？  

 如果你是贵族和大臣，该如何在权力斗争中得以保全？  

 如果你只是一介布衣，该如何在权力格局变化时抓住机遇成就自己？


这些问题，秦汉历史中的一个个人物，都给出了自己的答卷。


百万畅销书作家王伟，用现代经济学和管理学视角，为你重新解读秦汉史，解开历史背后的权力密码。
