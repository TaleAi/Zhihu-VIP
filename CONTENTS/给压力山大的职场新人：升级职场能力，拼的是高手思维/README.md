## 简介
当遇到束手无策，且无法解决的时候，除了能力方面问题，思维模式也是决定性因素。本专栏将从「思维方式」落地到「职场专业能力」，给入职 1-3 年你一堂最该补充的职场必修课，帮助你解决职场关键问题，赢得时间和效率。


当你把绞尽脑汁、使出了洪荒之力完稿的方案交给老板时，被老板臭骂了一顿，你是否心生愤懑？


当你与同事沟通一个简单问题时，你的同事提出了无数个可能性的问题，你们的沟通问题徒增，你非常奇怪为什么问题会像滚雪球一样越来越多呢?


当遇到束手无策，且无法解决的时候，除了能力方面问题，思维模式也是决定性因素。因为，人与人之间的差异，归结到最终还是思维模式的差异，思维决定格局，格局决定了职场的高度，所以，职场精英均无比重视思维模式的培养和改变，希冀优化思维模式，解决职场关键问题，赢得时间和效率。


不幸的是，仍有一些职场人深陷思维泥沼不能自拔，唯有转换下思维，去看看事情都另外一面，去看看别人如何去思考，也许能跳出固有都思维模式，扭转困局。


本专栏来自最前线的实操体会，从「思维方式建设」到「职场专业能力」给入职 1-3 年的职场人一堂最该补充的必修课，在思维、效率、学习、汇报、写作、职场副业等细颗粒、全方位的帮助。


思维模式的升级虽非一朝一夕之事，但是，正因其能有改变人生的能量，那么就应该尝试改变。
