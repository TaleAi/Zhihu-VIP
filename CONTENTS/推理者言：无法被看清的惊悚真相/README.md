# 推理者言：无法被看清的惊悚真相

## 简介
这是一个非常规的悬疑推理故事专栏。这是一个你不看到最后，就不知道到底是「谁」在讲「故事」的故事合集。


推理者言，每一个故事都由一位推理者的视角展开。


但他们并不是常见的「侦探」或者「警察」，或者寻找真相的人。


他们，可能是这场事件中的任何人。 


……


旧工厂疑案：一对网红博主前往一个旧工厂探险，直播时却突然被杀。现场只有一位同行的女孩，警方怀疑女孩行凶。但奇怪的是，女孩却说自己刚到现场就晕了，根本没有记忆。杀人现场留下了一段异常诡异的视频，甚至有人说是「鬼杀人」。女孩也发了疯。于是，警方拜托了精神病院院长姜乃行来对女孩进行精神司法鉴定。可进行鉴定的过程中，姜乃行却发现了女孩身上不可思议的「真相」…… 


危险窥视：中年男人王浩的老婆出轨了。好友建议彷徨无助的王浩在家中安摄像头拍下证据，这样至少可以留住房子。但王浩却在摄像头中看见妻子杀死了她的「小三」。妻子跑来找他求助，他以为她回心转意，帮她处理了尸体，接下来两人就可以好好过日子，为了避免嫌疑两人决定暂时离婚，没想到离婚前夕，他发现「小三」复活了。他以为自己被这对奸夫淫妇设计，没想到最后的「真相」比这还刺激…… 


而身为读者的你，也不要轻易相信他们带你看到的东西。


因为，连你本身，可能都是这个故事中的一环……
