## 简介
渴望爱情，却又处理不好亲密关系，是当下年轻人的通病。


我们憧憬爱情，想要拥有幸福甜蜜的亲密关系，但爱情如同一团谜，离得越近的两个人反而无法看清问题所在。


所以，我们总是一边在爱情里受伤害，又一边在爱情里伤害别人。


* 爱情里，如何戒掉对 Ta 的过度依赖？
