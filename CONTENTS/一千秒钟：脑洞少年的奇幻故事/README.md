## 简介
我们总是惊讶于故事里的脑洞大开，却对现实中的疯狂习以为常。荒诞的故事里，其实是一个少年的成长瞬间。10 个故事，关于爱情、家人、磨难、逃离……让我们开始一次脑洞少年的奇幻漂流。


一个女孩和一只猫咪意外互换了身体，当她想要换回的时候，猫咪却不让她如愿。女孩竭尽全力帮猫咪解决难题，然而却陷入了一个又一个谜团……


一座生来就习惯了被崇拜的神像，突然有一天被国王下令送进焚烧厂，劫后余生，它偶然间碰到一个向他倾诉的女孩；


一个拥有知觉和情感的几何体，穿越森林、草原、城市、荒漠……它想要成为一个乐手，却历经波折。然而在漫长的旅途里，一直有另一个几何体在自己身后；


组成世界的每个卑微碎片，都有热血的瞬间；


平淡点滴和渺小角色，也可以拼成奇幻叙事诗；


浩瀚的宇宙里，有无数像你我一样的小星球。 


这个专栏想讲讲关于小人物的成长故事和热血瞬间，每个平凡人物都是时间长河里的一秒钟。
