## 简介
午夜时分，让我们一起看看那些在身边流传的小故事。这些事情，也许你也听到过，遇到过。神秘的事件，令人细思极恐的瞬间和这窗外的漆黑很配哦。


红砖墙，在很多地方都很常见。但你有没有思考过，那砖为什么会那么红，那么鲜艳呢？


夜间的网约车并不一定安全。乘客有顾虑，司机同样也有着顾虑。在手机面前，谁也不知道约了车的人，是谁？


他被一把血雨伞追杀，这背后有什么隐情？


 午夜洗澡，花洒中洒下来的，不一定是热水，也许是.....


当深夜来临,大地归于沉寂,有些故事却正在发生。这些深夜的故事，或神秘离奇、或诡异恐怖，却都暗藏人性的弱点，世间的百味。


让我们跟随落地老师独具创造力和感染力的声音进入到这奇异的世界中来吧。准备好让自己的神经绷紧了吗？白天的背后是深夜，平静的背后是惊奇，故事开始了。
