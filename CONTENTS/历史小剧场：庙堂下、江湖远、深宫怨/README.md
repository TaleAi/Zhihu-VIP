## 简介
课本里，三行小字皆历史，可古人的内心戏全都被省略了。如果给你个机会，穿越回古代，让你了解每个大人物意难平的内心独白，是不是很有趣。你会发现李清照不只是「凄凄惨惨戚戚」，她还在为自己的丈夫生气，「恨国不成钢」；苏东坡不只是在作诗，他还在为穷得要吃什么好而发愁......


看似书本上平平无奇的历史人物，背后都是当年的血雨腥风、尔虞我诈、颠沛流离，他们的事迹能流传至今，背后都是超乎常人的非常遭遇，但在史书中，落笔却只有几句。


**「吕后恐，不止所为。」**


背后是刘邦一举定天下，吕后曾在背后倾尽全力，可自己的儿子却即将被易储。


**「丁巳，金人胁上皇北行。」**


背后是靖康之耻，徽宗和宗室 3000 多人被押到金国，女人们受尽凌辱。


**「姊娣专宠，而悉无子。」**


背后是赵飞燕、赵合德姐妹将汉成帝玩弄于鼓掌，甚至让他断子绝孙。


本专栏分为上下两部分。


「梦回篇」以历史人物的第一视角，揭开大人物们的多副面孔，还原他们有血有肉的跌宕人生。


「穿越篇」以现代人身份穿越到古代场景中，再现历史群星闪耀时的遗憾与壮丽，复刻古人的时代风骨。
