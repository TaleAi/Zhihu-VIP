## 简介
本课程由北京大学中文系教授王岳川主讲，带大家一起走入儒家精华的经典之作《大学》，领悟国学智慧。


  



**所谓大学，即大人之学**


《大学》是整个儒家体系的基础，


  是初入儒学的「入德」之门，


  朱熹把它列为「四书」之首；


《大学》被誉为东方人之圣经，


  中国人言行的标杆，


  是古代培养精英骨干的教育原则和方法：


  穷究万物事理获得知识，


  掌握处理事务决策的逻辑，


  培养道德品性，完善自我人格。


  



让我们跟随北京大学王岳川教授一起原汁原味理解《大学》，从中了解世道格局，掌握处理事务决策的逻辑，完善自我，变成更优秀的人，变成更美好的自己。


  



你将获得：
-----


1. 全面了解儒家核心哲学观，拓宽丰富精神生活领域
2. 领悟做人的大智慧，成为智慧的父母和老师
3. 学会如何处理人际关系，掌握处理事务决策逻辑
4. 明白立身之本，构建全方位认知系统，提升认知格局
5. 提升管理水平和领导能力，成为圣明的领导者
