## 简介
佛学不仅是一门学科，更是一种观看世界的眼光和方式。本堂课将围绕 16 个佛系关键词，窥探佛学的智慧，从经典文学、艺术、电影，以及流行文化现象谈起，开启一场活泼有趣又信息量满满的另类佛学。


唐诗宋词的美，张爱玲的小说，动漫中的人设文化，日本人的国宝戏剧……我们熟悉的日常中，一直潜藏着佛学。


最亲切的佛学，远不止《西游记》和《红楼梦》，在推理、怪谈、玄幻、影视、动漫里乐活的你，其实逃脱不了佛学。


你是否好奇：


电影复仇者联盟中灭霸是不是一个佛系反派？


现代西方心理学和电影理论，跟佛系的轮回思想有什么关系？


古代的佛学思想，是如何演变成五花八门的流行词汇和文化产品？


本堂课将围绕 16 个佛系关键词，窥探佛学的智慧，从经典文学、艺术、电影，以及流行文化现象谈起，开启一场活泼有趣又信息量满满的另类佛学。
