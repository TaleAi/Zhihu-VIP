## 简介
这是一门 100% 针对实际工作需要的课程，拥有 20 年一线经验的李国威，教你一次吃透顶级品牌公关的秘籍。不仅与你分享品牌公关入门需要具备的知识储备、关键能力和职业素养，更结合实战经验，进行品牌公关多场景多纬度多层次探讨。「案例 + 常识+ 方法 + 技巧」四位一体，让经验不系统、身边没高人的从业者，也能学习全球最顶级的品牌公关经验和方法。

如果你是公关从业者或者打算从事公关类工作，本门课将会让你学有所获。


本门课由公关实战大咖李国威老师亲自授课，他不仅代表中国最资深的一批公关人，并且具有国际视野，业界成绩显赫，堪称行业翘楚。在本次课程中，他将毫无保留地为你分享其一流的目标管理能力、临场应变能力、资源调动能力和知识人脉储备，让你在耳濡目染中学到品牌公关的精髓。


本门课中具有大量开放、系统、扎实的内容，从优秀品牌公关的岗位职责、品牌规划、公共资源管理、活动管理，到危机公关管理、员工沟通与雇主品牌管理、公共事务管理、投资者关系管理、企业新媒体管理，再到品牌公关的高阶能力、企业社会责任和热点公关事件解读。这门兼具深度与广度的系统化品牌公关实战课程，都将让你视野与见识更上一层楼。


这是一门看中理论方法的公关课，更是一门针对实际工作需要的品牌公关实战。不仅提供了系统实用的岗位能力、高效落地的工作方法，更重要的是教会你如何拥有比同事同行更出色的关系管理、执行和领导力，让你及早实现能力飞跃与升职加薪。
