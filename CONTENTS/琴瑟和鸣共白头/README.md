## 简介
滚滚红尘，邀君共周游。



《赴月》


很久以后，我遇到沈隽衣。


彼时他是全京城最炙手可热的少年将军，是我拼尽全力也够不到的天上明月。


而我是宁王叛逃后抛下的姬夫人，是所有人眼中最为不堪的存在。


宴席之上，公主问我，倘若回到那年他落魄潦倒之时，是否后悔同他退婚？


我俯下身，叩首到底。


「不悔。」
