## 简介
一觉醒来，发现自己被丧尸包围怎么办？


丧尸来袭，一夜之间城市沦陷，女生顾心清晨醒来，发现自己被困在了房间里，门外是成群丧尸......


凭借闺蜜方晚萍留下的救命号码，顾心联络上号码主人余禾，在他的帮助下逃出生天。


来到秘密基地避难的顾心发现，基地成员好像都跟曾经的自己有千丝万缕的联系。


两年前顾心车祸失忆，如今她得知，此前自己身上发生了很多事儿，并且跟丧尸爆发有关。


可这跟车祸醒来后，闺蜜方晚萍告诉她的情况，截然不同。


她该相信谁？那段空白的记忆里究竟发生了什么事？失联已久的方晚萍又去了哪里？ 


丧尸究竟是如何爆发的？疯狂袭击人类的丧尸，是否真如基地成员告诉她的那样，是在复仇？它们为什么要复仇？ 


就在恢复记忆，一切谜题即将解开的关头，顾心不幸感染了丧尸病毒，她该如何抉择？
