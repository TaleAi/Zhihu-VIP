# 在美国上历史课：历史还能这么学

## 简介
为什么要学历史？如何学好历史？历史不仅是教科书上的时间线，而是一种批判性的思维模式。唯有「像历史学家一样思考」，才能有所突破。


历史不仅仅是一系列历史事件和历史日期，而是对过去的讨论。


如果只读教科书，只会从一个角度看历史并且以后会无条件地接受任何呈现给他们的信息；只听老师讲课，后果也是一样。


唯有「像历史学家一样思考」：在教科书之外阅读一手文献；考察两种不同教科书的记载；对不同立场的文献进行解读；查看不同类型的文献，进行文献评估；进入历史语境中阅读，才能掌握学习历史的方法，培养个人批判性阅读和思考的能力。


这个专栏展现的正是美国中学如何教授历史课，如何通过批判性思维的训练培养公民素养的教学过程。


在上篇中，作者对美国历史老师课堂教学过程的记录和分析，以白描的方式展示了老师在课堂上引导学生运用批判性思维的细节：


老师会给出一个历史议题，围绕其发放多角度多类型文献，训练学生批判性阅读，然后鉴定文献和评估文献，并且做对比，理解事件的起因及影响，对事件进行充分的讨论。


在下篇中，老师会让学生去自主探索历史项目，并进行项目写作和展示，作者对自己做过的几个历史项目研究也进行了详细的介绍。
