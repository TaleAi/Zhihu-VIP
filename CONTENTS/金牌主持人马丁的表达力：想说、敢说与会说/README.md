## 简介
结合常见场景，运用沟通模板，重塑你的表达力。 


家里闹矛盾时，如何几句话就让老公孩子乖乖听话，甜蜜幸福？


开会发言时，如何让每一次发言一语中的，在公司里说得上话？


公开讲话时，如何让人点赞、捧场，轻松吸引贵人相助？


人们似乎总是不太擅于表达，有事只会闷头干，但如今是一个需要既会干又会讲的时代。人人都应该重视说话，学会认真表达。


本专栏涵盖日常 30 多个表达场景，马丁讲师将通过幽默的授课方式，结合「沟通+表达+演说」，带你由内而外养成气场，放大你的表达影响力。
