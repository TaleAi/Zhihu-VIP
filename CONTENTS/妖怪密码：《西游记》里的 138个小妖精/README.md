你想不想听听它们的故事？

还有狐妖、蜂妖、逐香魔……

历经九九八十一难，方能笑看世间风云。在这门课程中，一颉老师将带你「畅游」西游世界。挖掘妖怪们的故事，它们的真身，它们的下场和其中潜藏的现世社会的缩影在这里，都会为你一一揭秘开来。
