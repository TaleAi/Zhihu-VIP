## 简介
考研伊始，众说纷纭，到底哪种说法是对的？


全日制非全日制、专硕学硕、保研考研、联考统考傻傻分不清楚？


没了解清楚考研难度就莽撞出击？


对热门专业的认知还停留在过去？


考研要知己知彼才能百战不殆，一些基本信息没有了解清楚，容易引发「蝴蝶效应」，影响考研结果。


与其盲目刷经验贴，不如先从理性角度对考研做全方位了解，这样更有利于结合自身实际，针对性的选择各类考研信息。


本专栏由中公考研政治研究院总结了 7 年的考研辅导实践经验，筛选了广大学子选择考研之前最关心的问题和普遍存在的疑惑，并从专业角度对考研常识、考研难度和考试内容以及专业选择进行了深入解读，帮助同学们在全面了解考研的基础上做出最适合自己的选择。
