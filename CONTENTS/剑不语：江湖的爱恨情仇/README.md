# 剑不语：江湖的爱恨情仇

## 简介
剑光之下的江湖，藏着多少爱恨情仇与不能说的秘密。


我是天下第一铸剑师，我见过很多的剑，也听到过很多关于剑的故事。


名不见经传的小人物与成名已久的侠客对决，他们天差地别，到底谁能取胜？


书生青梅竹马的恋人给地主当了小妾，他想要劫了花轿，为何又匍匐而去？


将军被奸臣陷害，死在自己所守护之城，他用命换回来的一个小兵，开始走向了复仇之路。


霸王舞刀，歌姬清唱，身份悬殊的爱情与国恨家仇又该如何收场。


成名已久的剑客，一生无后，勉强求了一个女儿，但却夜夜见到鬼差索命，他靠什么可以保护自己的女儿？


仇杀百年的两大门派，他们的门人却相爱在了一起，可是最终在生死擂台上面对面的，正是他们。


天下第一的杀手组织与疯狂的邪教，到底隐藏着什么样的秘密？


那把完美无缺的神剑，真的可以手动搓核弹吗？


……


在这个江湖上所发生的的一切，全都跟剑有关，但剑便是剑，从来没有说过一句话，哪怕是哑巴剑客最后自剑身上传出来的那三个字，似乎也不是剑说的。


剑不语，人在看。


欢迎走进这光怪陆离的江湖世界。
