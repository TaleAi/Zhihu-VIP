## 简介
每年的「双 11」，你花了多少时间和金钱贡献给购物 App？提前透支的消费模式，让我们渐渐被延绵不绝的商品挟持了生活。我们该如何从消费社会中「解绑」？本专栏将从具体案例出发，深入挖掘消费社会的成因和机理，为你揭开消费主义的幻象。


每年的「双 11」，你花了多少时间和金钱贡献给购物 App？


没有人能够脱离消费而生存，每个人或许都曾有过那么一些时刻，很想购入那些梦寐以求的东西：一件衣服，一双球鞋，一部手机，一个名牌包，一块手表，一台跑车……


提前透支的消费模式，让我们渐渐被延绵不绝的商品挟持了生活。


从商品的生产开始出现供过于求的状况那刻起，人类社会就从生产型社会转变成了消费社会。


贷款买奢侈包包的人，可能把这种消费能力误认为自身实力的一部分，因此超出能力的消费，让他觉得自己仿佛摆脱了贫民身份，一跃而上成为了有身份的社会上层。把自我外化成为物质，这便是消费主义的阴谋。


明明知道身无分文，却总是抵御不住商品的诱惑，我们该如何从消费社会中「解绑」？物欲横流的世界，我们如何在物质的标签下重新发现真实的自我？


本专栏将从具体案例出发，深入挖掘消费社会的成因和机理，为你揭开消费主义的幻象。
