## 简介
贫富、阶级、年龄、种族，现代电视剧的狗血剧情都是几百年前文学作品玩剩下的，从英国的乡野田园，到香港的中式风情，爱情在这里生根发芽。然而，爱情并不都是一帆风顺，千人千面下，爱让人为之焦躁、忧愁与痛苦，门当户对、跨国恋、处女情结、性格不合……问题层出不穷，在爱情之路跋涉的你，或许该学一学文人的智慧，跟着文学去谈恋爱。


如何不费吹灰之力，挽回变心的爱人？


如何应对男友的「处女情结」，保护好自己的身体？


如何找到爱情中的立足点，平衡面包与情感的关系？


追溯古今，通过阅读文学名著，我们更能明白爱情的意义与本质。


本课程将通过一系列外国文学作品的浮光掠影，深入 17-19 世纪的欧美世界，从门当户对的理念、跨国恋的辛酸、性格不合的矛盾、处女情结等爱情经典问题，来寻找解决问题的途径，借鉴世界爱情名著的智慧，古为今用、洋为中用，为你揭开爱情的真相。
