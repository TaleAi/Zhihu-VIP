## 简介
这是一本年下伪纯良病娇弟弟的恋爱手册。


阴差阳错，郁双穿进了修仙文里，在原著作者的指引下，踏上了拯救世界的漫漫之路。


她的任务，就是救下那个之后会成为毁天灭地的妖王，但此时还在被普通人欺负的小可怜反派，并成为他的白月光，然后杀了他。


前面的郁双照做了。


可最后那个——


郁双看着面前委屈红了眼的少年反派......糟糕，她下不了手了。


沉寂本是无父无母，在村子里饱受欺辱的可怜少年。


他本以为，他的人生也就这样了。


直到那天被人追进了山洞里，他在那里遇到了一个板蓝根精姐姐。


姐姐不仅救了他，还收留他住下，教他读书写字。


「姐姐，你真好，你是这么多年唯一对我好的人。」（茶言茶语）


所以，他发誓，姐姐永远也别想离开他。


沙雕又温柔的年上大姐姐，和病娇又「纯良」的反派妖王，会发展出什么样的故事呢？
