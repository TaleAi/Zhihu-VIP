## 简介
糟糕，甜蜜恋人还有另一幅危险面孔。


他温柔专情的外表下，藏着不为人知的另一面。甜蜜又烧脑的危险爱情里，我该怎么办？


《旧梦》


她与沈京墨相识相恋十年。两人订婚不久，她滚落山崖，醒来后一切天翻地覆。


身边人都说，她与沈京墨从未有过交集。更离谱的是，她的未婚夫另有其人。


可本该亲密无间的未婚夫，处处透着怪异。而与她素不相识的沈京墨，总让她似曾相识。


波诡云谲的环境里，她能否寻出谁是真正的爱人？


《待春闺》


上辈子她没过上一天舒心日子，好不容易当了皇后，却被先皇后的养子——当朝摄政王逼死在坤宁宫。


重生一回，她想逃离这鬼地方，又一次次被这狗男人捉了回来。


直到他露出蛛丝马迹，她才惊觉，难道……他……也重生了？
