## 简介
绿茶会有好下场吗？那些在爱情里心机用尽的女孩，最终能得到她们渴望的回报吗？这里有心机用尽却被正牌女友反复打脸的绿茶，也有一心上位却被看透了技巧的捞女，更有心机度爆表的红茶 VS 海王斗法大戏……一组都市情爱故事，向你展示她们的人生。


《实习生妹妹》


男人真的看不出绿茶的技巧吗？


夏初相信，男友不可能真有那么傻。


他只是装作不知道，享受着暧昧的同时，也践踏了两人感情。


这样的男人，他没必要留。


  



《我们只是朋友》


他的「多年好友」，成了她心中最讨厌的那根刺。


那些被「友谊」隐藏的暧昧，骗不了任何人。


除了身处局中的那个傻子。


  



《金丝雀的人间游戏》


重逢初恋，我做了他的情人。


一边是貌美如花的我，一边是端庄富贵的妻子。


这个坐享齐人之福的男人不会知道，风花雪月背后，我们究竟酝酿着什么。


  



《绿茶遇上海王》


「甘思思，你没有错，人人都想过好的生活。」


我对着镜子中的女孩一字一顿地反复说着，好像这样就有了勇气。


可是嫁入上流社会的道路，哪有那么简单呢？


  



《空姐海王的捕获》


作为一个空姐，我的战场，在高空之上的头等舱。


「Coffee，tea or me？」是我百玩不厌的戏码。


即便这一局里，那个最好的猎物，已经是同事的男友，我也要在所不惜，将他拿下。
