## 简介
影视剧中的战争，往往是于战场上见胜负，但真实的古代战争，胜负的决定性因素往往都在战争之外。


长城一直被誉为抵御外敌的防御利器，但真实的长城要么建筑于足以抵御外敌的陡峭山岭上，要么建筑于荒漠或茫茫草原之上。


前者本没必要建筑城墙就能防御，后者建成了也很难完全防住铁骑。


那长城到底防的是什么？


管中窥豹，由此可以看出，哪怕我们了解兵法，对著名战役品评得头头是道，在真实战争面前，我们仍有很多忽略的细节，而这些可能恰恰才是成败的关键。


在本专栏中，作者张英锋以其渊博的知识和对古代战争文化的透彻了解，从古人战争的后勤、侦查、行军、布阵、经济影响等方方面面，为我们提炼，总结出一时一地的战争胜负背后，更为深刻的历史规律与胜负决策关键。
