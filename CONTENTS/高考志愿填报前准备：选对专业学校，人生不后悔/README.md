选择比努力重要，大学、专业究竟如何选？


所有考生们，恭喜你们毕业啦！


寒窗苦读数十载，相信你在考完后的这一刻，舒了长长一口气。因为我们即将迈入一个全新的阶段。同时，我们仍面临着一个重要的选择——如何填报高考志愿。


这决定了很多年后，我们会不会在所选择的领域里发散光和热，是否感谢当初所做的选择。

在本专栏里，升学顾问满老师，会从 2 个方面，带你了解专业和学校。让你在对专业学校充分了解的情况下，做出不后悔的选择。
