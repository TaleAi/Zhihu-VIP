## 简介
抓住客户的痛点，说准产品的卖点，拿下订单。


做销售，你是否经常困惑：


怎么抓住客户的痛点，说准产品的卖点？如何搞定每一笔订单背后的关键人物？销售做了五年，如何突破转型之痛……


销售的最大秘密就是：销售永远不是卖产品，而是如何巧妙地出售你自己，把你的目的、观点植入别人的心中。


而这其中的关键就在于——拥有**「销售思维」**。


职场人也是「销售人」，每个人都需要把自己的主张「销售」给老板、客户、同事，搞定订单，实现职场的升职加薪。


「阿里铁军」是最早的互联网地推部队，曾经创下过一年 50 个亿的销售额。本课程特邀**阿里铁军首任校长李立恒老师**，为你讲解他们结构化的实战销售经验，帮助你全面提高销售业绩。
