## 简介
什么是复试？复试在考研中的地位如何？应该如何应对？初试结束后，考研的万里长征走向了最后一步——复试。
相比于初试，复试的考查形式对于同学们而言更加陌生。本专栏聚焦广大学子普遍存在的疑惑，从复试的考查趋势和政策、复试面试及笔试、考研调剂三个方面对考研复试进行了全面剖析，帮助同学们在初试结束之后，顺利开展复试的复习工作。


听说复试在考研总分中占比越来越多？复试面试环节没把握？初试没进面意味着考研结束？


考研复试是广大考研 er 都比较陌生的环节，因为考研复试需要大家用更少的时间，展示尽可能多的能力。如果对复试的基本信息不了解、复试备考不充分，很容易在群雄逐鹿的考场上败下阵来。


打破信息壁垒、打消对复试的刻板印象和认知误区，是在复试中抢占先机的第一步。明确了复试复习的基本方向，才能有的放矢地制定复习规划，把精力花在该花地地方。


中公考研政治研究院总结了 7 年的考研辅导实践经验，筛选了广大考研学子关于复试存在的各种疑问，帮你进行「复试大扫盲」。从专业角度进行深入剖析，帮助大家解复试发展趋势、政策以及复试各个环节考查侧重点。
