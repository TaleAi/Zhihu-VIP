## 简介
本专栏是由四川省第三人民医院刘峰副主任医师主讲，解读在医院工作中和开设诊所中所见到的，被大家误会最多的疾病之一————哮喘。帮助大家认识哮喘的病因、诊断、治疗和其中的误区，接受最正规有效的哮喘治疗。


你是否被长年累月的咳嗽困扰？


你是否因为哮喘而无法正常锻炼、工作、生活？


你是否经历过看着家里的老人小孩被哮喘折磨，自己却束手无策的困境？


你是否因为哮喘病而乱投医，或因为无法坚持长时间的治疗而放弃？


你是否因为激素的使用而问题缠身？


我国的哮喘发病率在全球算低的了，但死亡率却排在全球第一梯队。是因为我国医疗水平有问题吗？不！是因为我们对哮喘误会太多了，对哮喘的治疗误会太深了。 


今天起，请大家跟着刘胖子一起，从哮喘的发病原因、病理变化到治疗方案，一点一点来剖析这个折腾人的病，达到最终的治疗目的：提高疗效、减少复发、提高生活质量。
