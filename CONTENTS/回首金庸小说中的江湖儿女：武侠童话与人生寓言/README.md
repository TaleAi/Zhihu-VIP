## 简介
重返金庸的武侠世界，终于读透了那爱情的万般模样。


四海列国，千秋万载，只有一个金庸。


除了精彩绝伦的故事，异乎常人的想象力，磅礴恢宏的历史叙事，当我们读金庸时，我们还可以读些什么？


南京大学古代文学博士聂飞琼撰写的这个专栏，提供了别具一格的阅读金庸的方式。


如果你想知道金庸的武侠世界里，藏着哪些中国人独有的民族观；如果你想知道「写情圣手」金庸，是如何写爱情的；如果你想读懂金庸小说中大量的古诗词用典、那些掩藏在文本深层的中国传统文化密码。关于这些问题，《回首金庸小说中的江湖儿女》会为你一一解答。


本专栏分为两部分：


第一部分是人物篇，以 19 个小说中的主要人物为脉络，阐释人物的成长之路、爱情关系和命运转折之处的选择：


炽热如火的杨过和冷若冰霜的小龙女，一个着急入世，一个淡然出世，却超越年龄、身份、生死甚至是时间，上演了一场冰与火之恋。那么《神雕侠侣》真的只是一部「写情之书」吗？


为情舍命的女子在金庸笔下并不少见，岳灵珊可能是当中「最不讨喜的一个」，可细品岳灵珊的爱情之选，她又何尝不是从一而终，尽兴爱过的一生。


身为金庸小说中武功配置最高的张无忌，因对待仇敌和男女问题的摇摆不定而被多数读者形容为「软弱」，其实，他性格的隐喻深藏在他的身世里。说到底，《倚天》是一部写人性的书。


…


第二部分以作品为脉络，旨在帮助读者深度了解金庸的小说技法、立意高度：


为什么说《笑傲江湖》是一部常在西方文学里出现的「英雄之旅」的故事？


《天龙八部》中隐藏了什么样的佛学密码？


写情圣手金庸，是如何写爱情的？


金庸小说中映射了怎样的中国民族观？


…
