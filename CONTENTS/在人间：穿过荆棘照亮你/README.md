## 简介
努力生活，人间值得。


阴霾人生，他们倔强生长，给珍重在乎的人带去爱和希望。


**拐二代的救赎**


我是人贩子的孩子，我的血，是脏的。


拐来的母亲受虐待自杀后，父亲和奶奶又谋划着，要卖掉年幼的妹妹。


我不能再眼睁睁看着悲剧再次发生。


这一次，我要带着妹妹，逃出这罪恶的大山，去看看外面的世界。 


**姐姐的复仇**


妹妹遭遇校园霸凌，从楼上一跃而下，成了植物人。


查出那个一直霸凌、欺辱她的女生是城中某富豪的女儿后，


我应聘成为了这家的保姆。


我的复仇，要开始了。
