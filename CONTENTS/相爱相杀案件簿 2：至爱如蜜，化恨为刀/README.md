## 简介
有时候，亲密关系是把双刃剑。爱情里的蜜里调油，分开时或许会变成浓浓恨意和执念，甚至让人铤而走险，将犯罪之手，伸向深爱之人。《相爱相杀案件簿》第二季，有着多年一线刑侦工作经验的唐风，将继续为你讲述那些匪夷所思的情感犯罪，探寻罪案背后的血色人性。


密室逃脱中的玩偶，竟然是个死人。更诡异的是，这具尸体没有脸。被高度毁容的尸体身份扑朔迷离，而态度躲闪、闪烁其词的店家，频繁光顾密室的女玩家，都给这件奇案蒙上了一层更深的迷雾。


男生猝死密林，没有任何谋杀痕迹。警方怀疑，这并非简单的意外。调查线索将嫌疑人锁定在男生的四任女友身上。可这四个住在同一寝室的女生，都有完美的不在场证明。 


······


匪夷所思的案情背后，是不一样的爱恨情仇，和人与人之间复杂深刻的人性。


如果人生可以倒带，在情感纠葛中走向犯罪的人，会不会放下执念，拥有另一种结局？


祝你阅读愉快，也愿每个人都有一段健康而美满的情感～
