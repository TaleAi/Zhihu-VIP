## 简介

字对中国人来说有多重要？


中国人强调：「作字先作人，人正则字正。」


**一个人写的字，关乎其性格、能力、人品，甚至能透露出家风和教养。**


一切都能在字里体现，但只有了解书法，才能拥有好字。


1800 多年以来，楷书一直作为我们中国人的标准字体。楷书四大家，则不仅是字之楷模，更是人之楷模。


**颜真卿：**以文官之身，平定两次叛乱，面对刀山火海，也威武不屈，死后更是三军轰动，皇帝为他罢朝五天；


**柳公权：**历任七朝，是刚正不阿的谏官，面对喜怒无常的皇权，永远都宠辱不惊；


**欧阳询：**出身将帅之家，却沦为罪臣之子，因外貌奇丑屡屡被嘲，字却被赞为「唐人楷书第一」；


**赵孟頫：**为人品性纯正，为官嫉恶如仇，面对世人的不解和嘲弄，依旧坚定内心的追求。


就让我们跟随本专栏一起，领略「颜体」、「柳体」、「欧体」、「赵体」的魅力吧。
