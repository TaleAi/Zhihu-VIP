## 简介
秦质头一次见她，就栽了。


作者：另外半只


温媛是个海王，上到四十下到十八，没她拿不下的男人。


正辉煌着呢，迎面撞上来一小伙儿，腰窄肩宽，五官凛冽。


眼角还有一疤，死死撞她审美点上了。


温媛勾唇，偏头问身边的人，「站那发传单的，叫什么？」


「秦质，人挺穷的。」


人穷志就短，可温媛屈尊给他点过烟、擦过鞋，硬是没让小伙看她一眼。


温媛手一挥，行，老娘不干了。


可下一秒，男人就将人圈进怀，「都是我的人了，还想跑？」
