## 14.如何打造个人 IP
IP,其实是英文 Intellectual Property 的缩写，意思是知识产权。近年来，随着互联网、新媒体的介入，IP 的外延和内涵也发生了变化，个人 IP，简单说就是个人品牌。现在是个体崛起的时代，每个人都可以打造自己的影响力，拥有流量聚集能力。一旦形成了个人品牌，那么你的职业生涯可以说上了一个新台阶。


打造个人 IP 有多重要？


我给大家举个例子。张大奕大家应该都听过，她最开始是一名模特，后来成为淘宝店主，通过与众不同的穿搭以及个性化的品牌营销，成为国内全品类网红店主第一人。2016 年，张大奕在淘宝的一场直播，有 40 万人观看，2 小时卖货 2000 万，2017 年，她四家淘宝店的合计成交额将近 10 个亿。


从一名模特到年销 10 亿的网红，这其中离不开个人品牌的打造。


**那如何打造个人 IP？给大家几点建议。**


**1、找准自己的定位，给自己贴标签。**


每一个个人 IP 都有自己清晰的定位，如张大奕是淘宝第一网红，papi 酱是短视频网红，罗振宇是死磕自己、贩卖知识的自媒体人……此外还有理财达人、旅游达人、美妆博主等


你擅长什么？你想让大家对你有一个什么样的认知？打造个人 IP 的第一步，就是要找准自己的定位，并给自己贴上标签。


我的一个朋友，从企业高管辞职，专门为中小企业提供专业的营销策划课程和顾问服务。企业培训的市场非常大，人缘也是残次不齐。很多人初期为了进入市场，迅速积累客户，不论什么主题都会讲。而这位朋友坚持自己的定位。他说，他可以讲管理，也可以讲全员营销，但是他的优势是针对企业高管和公司老总。虽然一开始请这位朋友的人不多，但是，他每次都会用心讲好。两年下来，他在营销策划课程方面已经做到如火纯青，很多企业家都是提前半年才能约到他的时间。而他的出场策划费用，已经从最开始的几千，到现在的几万。年收入也超过两百万。


这位朋友的成功，在于清楚知道自己擅长什么而不是什么赚钱做什么。在职场中，要知道自己需要成长什么而不是什么受欢迎就学什么。其实个人 ip 打造，和小时候看过的一个故事是一样的，小猫钓鱼。小猫是要钓鱼，这是他最重要的事。但是他一会追蝴蝶，一会抓蜻蜓。本职工作，或者是擅长的东西不去花时间深化和操作。到最后就是什么也得不到。


在个人 ip 打造方面要明确一个问题，要一米宽一千米深，而不是一千米宽，只有一米深。


**2、明确用户画像，找到用户聚集地**


找准自己的定位，接下来你就需要明确自己想影响哪些人，给用户做个简单的画像。


比如你是一名医生，那你的群体自然是病人，病人的聚集平台就是类似丁香医生、春雨医生等 APP；再进一步，如果你是一名整形医生，那新氧平台的用户更聚焦，更精准。


如果你是一名自媒体人，那微信公众号、头条等就是你的阵地。如果你想影响你的客户，那朋友圈就是你的主阵地。


**3、日复一日，持续输出**


明确了定位，找到了用户阵地后，接下来你要做的，就是持续输出内容，内容形式包括文字、图片、音频、视频等不一而足。


这里需要特别注意的是，内容的输出是一个长期的，不间断的过程，只有这样，才能留住粉丝，形成品牌效应。


**4、不定期举办沙龙、分享会等线下活动**


个人 IP 一旦形成，除了持续输出内容外，还应该不定期举办一些线下活动，和粉丝近距离交流，进一步增强粉丝的粘性。


**5、持续有效的个人增值**


个人 ip 是个持续增值的过程，而增值的标准，就是你个人的能力和所能为他人带来的价值。持续学习，不断优化自己作为个体 ip 的价值，增强 ip 的实力，这样才能不断让自己的身份呈现螺旋式的上升，因为你价值的不断上升，你这个个体 ip 也会越来越值钱。


今天，和大家聊了聊如何打造个人 IP。自媒体时代，人人都可以成为 IP，关键是找到方法，并持之以恒，今天分享了打造个人 IP 的五部曲，希望对大家有启发。
