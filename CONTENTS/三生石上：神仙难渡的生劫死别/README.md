## 简介
生在仙界，亦身不由己！
生而为妖，也情劫难渡！
当神仙难做，妖邪难存，内心的爱恨翻涌却明知无法善终，是否还要放手一搏？
三生石上，你我的名字早已难以抹灭……



《古冢狐》


如果没有容止，阿狸大概会一直做个逍遥快活，嫉恶如仇的九尾狐仙。


她既不求历劫成神，也不求法力滔天，直到容止的出现，她发现自己所求只是和容止白头到老，永不分离。


可容止亲手斩下她的头颅，灭了她整个狐族，那时起，阿狸心中的火光熄灭了。


她联合魔尊，打上天庭，却发现了背后更大的隐秘。


《白骨夫人》  


她为爱成妖，只求相爱终老，没想到他只图她的血肉来长生不老，活活将她千刀万剐，弃尸荒野！那么，她只能以牙还牙，以血还血！
