## 简介
爱情，千奇百怪，可可爱爱。


非正常爱情研究中心，探索爱情的千般可能。


她们或是遇上了病娇、狗血、霸道的男神，进退之间跳起爱的华尔兹。


或是自己就是病娇、狗血、霸道的大女主，为小奶狗费尽心机。


爱情之中，没有弱者，只有赢家。


**日出桑榆**


丧尸末日，人人自危。我穿上最美的红裙，敲开了桑榆的门。


他是一个窥觑我的病娇，想要囚禁我已经很久了。


——这个世道，谁不想住在地下室好吃好喝？


来吧，看看谁是这场末日爱情中的赢家。


**霸道娇妻**


我和程昱是契约婚姻，他给我一笔钱，让我恪守程夫人的本分。


他给的实在太多了，以至于我不愿意失去这位阔绰的甲方爸爸。


我每天矜矜业业为他做饭，做家务，spa，按摩，财务管理……甚至为程昱追求白月光彻夜做策划案。


两年之期已到，契约婚姻失效。


程昱：再充一千万，续约。


我：恋爱是另外的价钱。
