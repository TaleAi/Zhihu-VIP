## 简介
几百年前，我们就已经知道地球是圆的，绘制出了世界地图。几百年后的今天，换一个角度看世界，我们仍可以收获别样的美。在这个信息和知识爆炸的时代，如何去给孩子们讲科学，讲什么样的内容，都值得我们去深思。


地球陆地的最低点是哪里？世界上哪个地方最能下雨？


世界上最大的沙漠是哪个沙漠？海底的最深处是马里亚纳海沟吗？


......


地理，是与我们的生活息息相关的一门科学。


小到出门看地图、旅行看天气，大到我们生活的地球环境变化、地质灾害、人文差异等，都与地理学科密不可分。了解地理是国民最基本的素质，和孩子一起了解地理小知识，既满足了孩子对地球的好奇心和求知欲，也让家长了解到关于地理科学的最新信息。


本专栏通过音频的方式讲解科学，使用生动形象的语言来讲述科学知识，可以让孩子利用一些细碎的时间片段学习科学知识，并从中获得快乐，感受到科学的神秘与美丽。
