## 简介
监狱是个盛产阴谋的地方，犯人们为了多挣减刑分，不惜一切手段弄掉竞争对手。服刑十年，能炼成百毒不侵的金钢之身极少，大多数犯人不是傻了就是变态了。


我是黑叶，曾被判过无期徒刑。


被逮捕的时候，我 30 岁，单纯、书生意气，15 年的牢狱让我看到太多的罪恶和黑暗。


我在监狱里当过学习管理员，积极改造委员会委员，能看到一般犯人看不到的秘密：


* 黑帮是如何在监狱里找人才的？
* 为什么狱内医院是犯人们的「红灯区」？
* 犯人们是如何策划「保外就医」的方式逃狱的？
* ......

除了这些秘密，让我久久不忘的是监狱里残酷的命运：


北大毕业的扒窃团伙头目，为了兄弟们被处以死刑；


砍人如佐罗的黑帮大佬在出狱前一天自杀谢罪；


犯盗窃文物罪的考古学家，却在监狱被妻子灭口；


一个穷六代做足努力逆天改命，最后还是在监狱度过一生；


…...


这段监狱经历让我深刻地体会到了：**所有的犯罪史，都是生存史。**


我把这些感受和经历，写成了 11 个故事。从这些故事里，希望从这社会的阴暗一角，帮你窥见暗网下的人性。
