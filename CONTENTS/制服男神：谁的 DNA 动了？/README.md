## 简介
穿制服的男人，总有一款是你的菜。


《绑架我的心》


我被绑架了，绑匪好凶。


我最喜欢这种不把我放在眼里的男人。


我给他加价，指挥他逃跑，他忍无可忍按住我。


「再不闭嘴我亲你了！」


《一网情深》


和网恋的特种兵男友分手，他直接找上门。


「这种话当面说比较好。」


没想到，因为疫情，我们被隔离在小区十四天。


半个月后，衣衫不整的两人被我妈堵在房间门口。


《心悦臣服》


我养了一个小奶狗，公司破产，背后却是他搞的鬼。


他一改往日乖顺的模样，叼着烟掐住我脖子。


「没钱了，姐姐是不是就不会离开我了？」


我摇头。


「抱歉，25 岁的男人是我的底线。」
