## 简介
平凡的人生，在某一刻突然「神展开」！因为你所在的世界，它可能是一本恐怖小说，记录着下一个要杀你的人；它可能是一部《山海经》，满地都是奇珍异兽；它还可能无限循环，有一天另一个你会来到你面前……


我是个声优，配男神音的剧本，没想到我家的橘猫听完怀孕了？！这个世界能让配音演员梦想成真；


我是个记者，在采访一位杀人犯，他却说自己是自卫，因为他在一部恐怖小说里看到未来对方会杀了他，没想到这是真的；


我和女朋友相爱多年准备结婚，突然遇到一个中年男人，说我是白素贞？他胡扯！我拉起女朋友要走，她却说「姐姐，我是小青」；


我是天下第一的剑客，武林盟主，可是我不会分毫武功，这江湖，被我稀里糊涂就统一了……


在这个奇妙的世界里，谁的人生都不一定平凡，下一秒你的故事就会神展开，手中的刀叉碗筷，家里的猫狗主子，个个都有不平凡的来历。


**Warning：要对这个世界保持警惕，脑洞随机出没，注意脚下安全。**
