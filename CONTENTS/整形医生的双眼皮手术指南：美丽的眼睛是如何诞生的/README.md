## 简介
本专栏由北京大学第三医院整形外科的副主任医师马建勋结合自己的临床经验，介绍双眼皮手术的标准流程以及它的术后护理要点，帮助求美者认识自己的眼睛，以及对自己是否需要接受这项手术做出准确的判断。


你是否因为想要一双更美的眼睛而考虑过双眼皮手术？ 


双眼皮手术后的眼睛是否会「千篇一律」，医生手术的标准是相同的吗？ 


怎样的一台双眼皮手术才能算作成功，做完双眼皮手术是否真的能让眼睛变美？


除了双眼皮手术，眼部整形还有哪些号称能让人「年轻十岁」的项目？


术前如何跟医生进行沟通，才能找到最适合自己的那双眼睛？


在医美整形行业热火朝天的今天，美貌似乎变成了可以「量产」的东西，而那些被称为「美」的法则，到底是商家的噱头，还是科学的真理？浪潮之中，需要我们擦亮眼睛，重新审视。


为此，本专栏邀请到来自北京大学第三医院整形外科的副主任医师马建勋，结合自己的临床经验，全方位为大家讲解双眼皮手术，希望想要通过眼部整形变美的你能从中得到一些启发，也期待这一传统的整形项目能够获得新的阐释。
