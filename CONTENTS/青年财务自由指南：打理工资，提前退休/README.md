## 简介
如何用实现财务自由？90 后作者也小谈告诉你他正在实践方法：通过工资理财，提高被动收入。


作者也小谈和大多数青年一样，居住过糟心的合租，体验过拥挤的地铁，遇见过无理取闹的甲方……


但是大家都没有「Say NO」的底气，因为生活中许多问题大都需要「钱」来解决。


财务自由，就给了你不想做什么，就不做什么的资本。


作者通过亲自实践验证，发现财务自由并不需要你有过人的胆识、惊人的运气和高深的智慧，通过工资理财，已经完成了自定的财务自由目标的 70%+。


如果你是对经济、金融、理财了解不多的青年人，只要做好规划，踏踏实实稳步积累，一样可以实现财务自由。 


本专栏按照一个人实现财务自由所需要的理财要素，分成四个主要部分： 


• 准备阶段；制订一个可量化的财务自由计划；


• 投资攻略；理解复利、被动收入，实现开源，获得简单有保障的投资策略；


• 高效储蓄；理性消费并不难，物质享受与少花钱并不冲突；  


• 理解保险；有效配置保险，为自己的财务自由之路和未来的生活保驾护航。  


本专栏不会推荐任何理财产品，只会告诉你利用「工资」去计划从开支预算到储蓄投资，如何实现财务自由。
