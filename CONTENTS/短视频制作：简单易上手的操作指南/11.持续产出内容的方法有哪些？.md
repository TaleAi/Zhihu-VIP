## 11.持续产出内容的方法有哪些？
如何持续产出优质短视频？


大家好，我们今天来上最后一课（ PS :这句让我想起了法国作家契科夫的《最后一课》），如何持续产出优质短视频。这个问题几乎是所有内容创作者共同的焦虑。有一个同行告诉我，他最开始做视频的时候才思泉涌，做了五六期后感觉身体被掏空。这是很正常的，首先人的知识沉淀永远有枯竭的时候，其次，个人或者小微团队毕竟势单力薄，不像二更，他有 300 人的团队，可以通过系统内资源的调配让创作者修生养息，您要是歇了，这视频项目也就黄了。事实上，我刚刚开始做视频的时候就感到了这种压力，幸好我遇到了高人指点，暂时性地规避了创作枯竭这个矛盾，这次就来讲授一下我的 4 种方法。


第一个方法把现有的文字内容变成自己的视频内容


大家有没有想过，为什么短视频会火？其中一个原因是，图文信息数量饱和，且高度同质化现象严重，这刚好说明，市场里其实并不缺内容，它们只不过以文字的状态保存着，如果你把这些内容转化为视频内容，那会是什么成色？


比如你要做 A 领域的视频，那你一定会去关注这个领域里的好文章，学习它的逻辑思路和典型案例，把它转化为你视频的文案，再从你的嘴里说出来，这个作品就是你的了。有人会说，这不是抄袭嘛，我想说的是，你把一段只能用眼睛识别的文字，变成了能用耳朵和眼睛接受的视频，这本身就是一种创造。


举几个例子，PAP I酱。她的内容大部分来自于网络上经典吐槽段子，她把这些段子用演技重新诠释了一遍，就能获得巨大的流量，而网络段子有多少，可以认为是无穷无尽的吧，所以她的创作来源是源源不断的。再举个例子，在今日头条上有一个视频号叫《不上班》，这个团队就是把创业投资圈的一些故事类的文章，用电影的手法再演绎一遍，他们的视频可能很难推送到普通人那里，但是在创投圈还是小有名气的。


我说了这两个例子，看似很简单，但你不要小看这件事儿，其实是很难的。下面我说一下这种做法对人有什么要求。


1、 你必须对这个内容的领域有所了解，尤其是做垂直类内容的短视频，这点更为重要。如果你对这个领域不了解，就抓不到原始内容的脉络，原始文章写得再精彩，被一个外行人转述后就变得平庸了。曾经有个人这样问我，周老师，你看我做哪个领域的视频内容有前途，汽车、金融、健身还是美食，我说，兄弟你是全才啊，哪个领域都成啊，他说不是的，这些领域他都不熟，就是想用我这种方法去做文字到视频的转化，然后我劝他，选一个你最熟悉的领域去做，从投入产出比上看，才是最可行的。


2、 你要有一些编剧的能力。一篇文章的行文方式和一个短视频文案不同，有些内容用语言很容易表达，但是用视频表达就需要设计一些镜头语言。否则你的视频看起来会像新闻联播，你喜欢看新闻联播吗？


第二个方法、制作属于自己的视频日志



```
   如果你不想转化别人的文字内容，就像做纯纯的原创视频，那可以考虑制作属于自己的视频日志。视频日志在国外叫做 Vlog，就是用摄像机记录你的生活，这个绝对不存在创作枯竭的可能。什么是视频日志？在电影阿凡达里，杰克苏里每天都要在摄像头前面做的那件事，就是视频日志。在这方面，YOUTUBE 上有一些红人做得比较好，比如 sniperwolf ，你看看她的视频可以对它有个感性的认识了。

    做视频日志最大的好处，就是成本非常低，视频在拍摄过程中永远处于直播状态，甚至可以这么讲，视频日志就是直播你的生活。但是，视频日志并不是谁都适合做的，下面我告诉大家做视频日志的一些基本条件。
```

1、 最重要的一点，你的日常生活必须足够精彩。这个简单的标准其实就刷掉了大部分人，因为大部分人的日常生活是单调的，是缺乏刺激和新鲜，这样的生活很难勾起别人观看的欲望。但如果有一些兴趣爱好，比如做饭、做手工、健身之类的爱好，就可以用摄像机把这个过程录制下来，经过剪辑之后放到网上试试水。如果你的兴趣是带有刺激性、风险性的，那就更靠谱儿的，受到关注的几率会成倍增加。比如 youtube 上有一个人叫 dirtbike ，把自己骑摩托车飙车的画面用运动摄像机记录下来，里面甚至有一些躲避警车追踪的刺激镜头，他的视频非常受欢迎，每个视频可以有一百多万的播放量。我不是鼓励大家去做这种危险的事情，而是告诉大家，惊险和刺激本身就是带来流量的武器，如果你的生活里本身就有这些东西，把它转化为视频日志会很受欢迎。


2、 第二点，你必须有一定的表演能力，把一件稀疏平常的事情表演得恰到好处。这方面，我比较推崇日本的 youtube 红人 hikakin，他能把一些日常生活中的小细节演绎得具有趣味性，比如品尝酸奶，品尝红酒、数码产品的开箱之类的情节。假如你不想冒着生命危险去做一些带有刺激性的视频，可以学一学他的套路。


第三个方法 从国外网站获取视频源，翻译配音



```
 由于 YouTube 的内容暂时无法正常进入国内，这条天然的屏障给了我们短视频制作者们一个好机会。把国外的视频拿过来做二次加工，这是持续输出内容的方法里最省事儿的一条路径。
 在这方面，做得比较好的账号就是《科技微讯》，他从来没有感受过创作枯竭，只感觉要做的视频太多，忙不过来。我在做视频的初期，也是从国外获取了大量的视频素材做翻译配音，让我获取了充沛的流量。

 做这个事情，至少需要 3 个条件。
```

1、 能够在国外网站找到视频源，并且是持续不断的视频源，这个并不难。
2、 有一定英文功底，能够听懂原始视频的意思，之后再加入你自己的一部分理解。
3、 会使用剪辑软件。
因为我自己做过这件事，所以非常清楚它的优点和缺点，优点就不多说了，我说一些缺点。
1、 对原创度的质疑
只要你做国外视频的翻译配音，这个问题将始终成为你的困扰。最近，今日头条对此发布的一份声明，这份声明里说，「在现有视频图像素材基础上，加入解说、剪辑、配乐等制作，使原视频素材成为一个较为丰富、有信息量、有自己编辑思路的视频作品。」才能被当做原创视频来对待。但是「丰富」、「有信息量」、「有编辑思路」这几个概念的边界很模糊。假如你的视频被用户频繁举报为抄袭，平台真的有可能认为你是在抄袭，这种事情的确发生过。倘若真是这样，会对你的账号造成致命的打击。


2、 版权的隐患


所谓天高皇帝远，youtube 上的作者不太可能千里迢迢来找你维权，因为 youtube 暂时进不来，但是 youtube 进不来，不意味着作者进不来。前不久，在 youtube 上拥有 500 万粉丝的脱口秀红人 mattsantoro 入驻今日头条并签约，在此前，我见到过一些人把他在 youtube 上的视频搬运到今日头条上，但是今后，至少这条搬运之路就断了。在未来，必定会有大量的国外网红入驻中国的短视频平台，因为中国是一个拥有世界五分之一人口的海量市场，谁不想染指呢？


第四个方法 向用户征集内容



```
  这种方法有一个时髦的说法，叫引入 UGC，就是向你的用户收集原创内容，然后做精细剪辑变成一部作品。比如 PAPI 酱曾经就向粉丝们征集模仿她的视频，再把这些素材编辑成了一个新的内容。文艺教主新世相做过一个项目「凌晨四点的北京」，他们先向用户做了一次 UGC 征集，就是征集用户如果在凌晨四点钟不睡觉的话在干什么，有一千多个人讲了他们的故事，新世相从中挑了 25 个人，把他们的故事做成一次视频直播内容，效果非常好。

 如果你的粉丝数量众多，活跃度非常高，忠诚度也不错的话，用这种方式可以缓解创作枯竭的。
```

总结一下这次的课程，如何持续产出优质短视频，三个方法。1、把现有文字转化为视频。2、用视频记录自己的生活。3、对国外视频做翻译配音。3 种方法给有利弊，您自己掂量吧
