## 简介
「急转直下」的心理悬疑短篇故事。



> 一个奇怪的念头掠过她的脑海——她突然很想拿起沉重的玻璃烟灰缸，砸向丈夫的头。


*——雪莉·杰克逊《可怕的念头》*


在这些故事中，命运总会在你意想不到的地方强行逆转：


**《苍幽公园》**


人烟罕至的苍幽公园，埋藏着各种都市传说，当我们五个人决定放学回家绕行苍幽公园时，谁都没有预料，我们会遭遇一个疯子的跟踪。他从远处的暗自观察，慢慢靠近，再到追逐着我们。但这绝非是他的恶趣味，他似乎有某个目的，与我们五人之中的其中一人，有着不可告人的绝密，腐叶丈厚的苍幽公园，秘密将被挖掘。


**《202 房间》**


一个女人去住宾馆，洗澡之后满是雾气的镜子上却出现了一个手印，和两个字「救命」。  

这女人死了，一切都像是自杀，但没这么简单，因为三年前，这个房间另一个女人也自杀过。


**《无间电梯》**


夏季，午夜。  

她刚加班完要回家时，却注意到一个打扮怪异的男人向她走来。惊慌失措的她，连忙冲进了电梯并按下按钮。  

然而，这架载着楼内最后几个人的电梯，却在 18 层戛然停止。门内对讲幽幽响起，神秘人讲述了一个女儿被杀害的故事。而在他口中，电梯内的几个人，恰好都是嫌疑人。
