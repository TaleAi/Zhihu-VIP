## 简介
不懂法律，生活处处是「坑」。了解法律常识，才能用智慧武装头、用法律洞察是非、用理性解决问题。在生活中遇到纠纷，用法律的手段维护自己的合法权益，遇到事情沉着冷静应对，你也可以成为自己的「法律顾问」。


日常生活中，常常会遇到一些不知如何处理的情况：


购物被坑、借钱不还、理财踩雷、交通医疗事故、网络诈骗，生活纷争难以避免；


家暴 、被出轨 、财产分割、争抚养权、房产纠纷，婚恋危机四处潜伏；


五险一金 、性骚扰、拖欠工资、意外工伤、合同纠纷，职场纠纷时常遇到；


......


法律人都是怎么思考问题的？律政剧里面的情节，可信度有多高？


法官、律师开庭时在想什么？真实的法庭如何做到公平？


打赢一场官司到底有几步？法庭辩论的画面和你想的完全不同？


从日常「避坑」到真实庭审的全流程，本专栏有观点、有案例、有知识、有办法，让绝对的「内行人」教你如何自保维权，让你感受法学这门古老学科的独特魅力，体验一趟特别的法律之旅。
