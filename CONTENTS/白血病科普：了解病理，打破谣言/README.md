## 简介
本专栏由一线癌症研究人员帅世民为大家带来白血病历史与现状、诊断与治疗的相关科普，帮助大家认识与正视白血病，了解关于白血病的最新研究进展。


号称「韩剧四大病」之首的白血病，到底是不是绝症？


「白血病」这个名字是怎么来的，这是一种「血液会变白」的病吗？


白血病是如何诊断与治疗的？


劣质装修材料造成的污染到底会不会诱发白血病？ 


白血病其实是一大类血液癌症的统称。如今，随着科技的发展和各类新型治疗方案的推出，很多白血病已不再是绝症。


为了帮助大家更好地了解白血病，本专栏将对白血病的相关知识进行系统性梳理，从白血病的发现史讲到白血病的现代解读，从白血病的分类讲到白血病的诊治，最后还将回答大家有关白血病的常见问题。


希望这一专栏可以让大家更好地理解白血病，消除对白血病的恐惧和误解，并且了解一些和白血病有关的最新的治疗方案和研究进展。
