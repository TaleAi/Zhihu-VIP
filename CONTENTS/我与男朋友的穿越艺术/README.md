# 我与男朋友的穿越艺术

### 作者：攸冥

### 分类：古代言情

### 状态：已完结，共 9 节

### 存档日期：2022-06-12

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1266021699440807936


## 简介
世界上最苦逼的事情不是穿越了。穿越怎么会苦逼呢？男一追着，男二求着。苦逼的是，你和男朋友一起穿越了。你变成进宫选秀的官家小姐，他……成了皇帝。


穿越是门艺术，穿得好，你成为逆转人生的终极赢家，穿得不好，你可能成为遭人恨的恶毒女配。我这趟穿越，穿得倒是别出心裁，因为我和男朋友一起穿了。


前一世，他是普通社畜，我是平凡女孩；这一世，他是皇帝，我是嫔妃。


穿越来的第一要务，在这个乱世活下去。于是他专心理政，我暗中相助。我想做能与他并肩而立的女子，而不是后宫宠妃。他说，他可以遣散后宫，绝不背叛我。


似乎，我们都已慢慢适应穿越后的生活。


然而，越来越复杂的局势，彼此之间越来越多的秘密，终究把我们推得越来越远。


之前恋爱中不愿触及的隐患也慢慢显露出来。


如今我是后宫权势最大的嫔妃，是可以先斩后奏的钦差大臣，是江湖情报组织的阁主，却不再是他身边唯一的女人。


他心心念念寻找的穿越之法，究竟能否把我们带回去？


即使能回到现代，我们又能过回从前的生活吗？


穿越，真是门艺术。
