**《名媛的谎言》**


梁歆是一位「假名媛」，靠着自己精致的打扮「钓」到了有钱的男朋友。有一天她忽然发现，自己只要说谎，身上就会长出密密麻麻的可怕脓疹。梁歆因此一度放弃做「伪装名媛」，但对奢靡生活的向往还是令她经不住诱惑、「重出江湖」，引诱了一名叫秦秋的富有男子。而这段新恋情的结局，却大大出乎梁歆的预料……


**《坏手机》**


普通青年侯炜，意外捡到一个高档手机。在此之前，他被同事排挤、被上司欺负，过着窝囊的生活。但令他万万没想到的是，捡到的这个手机，竟然可以帮他「除掉」任何他不想看到的人。面对诱惑与良知，侯炜陷入了两难境地……
