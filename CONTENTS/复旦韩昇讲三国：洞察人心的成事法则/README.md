## 简介
复旦大学历史系教授、著名学者韩昇，基于《三国志》，为你彻底还原真正的三国历史，让你看到历史上最优秀的一群领导者，是如何统御属下，应对困难，聚合资源，办成大事。读三国，学习最优秀的领导力。


白脸奸雄曹操，凭什么能做到麾下猛将如云、谋士如雨？


四世三公的富 N 代袁绍，手下能人无数，为什么会顷刻之间一败涂地？


血统扑朔迷离的刘备，凭什么能够力压曹操成为众人口中的「正统」？


鞠躬尽瘁的诸葛亮，为什么会落得壮志未酬身先死的下场？


如果一定要用一个词语来回答所有问题，那应该是——领导力。


本课程一共分为 22 集，韩昇教授将会颠覆你的认知，从不同的视角带你还原历史上真正的曹操、孙权、刘备、袁绍、诸葛亮，带你看这群领导者，如何因不同的领导方式，走出不同的人生轨迹和功业境界。


  



![img](https://pic1.zhimg.com/v2-afade138b109dab1fa6d51801a3c6889.png)

韩昇教授家学深厚，其父亲韩国磐，是当代中国魏晋隋唐史学科和中国经济史学科的奠基者之一，被誉为「一代宗师，史学泰斗」。


韩昇教授不但拥有数十年的历史研究经验，而且多年来热衷游历各地，他曾经说过：「我讲历史的信心在于，我讲的地方，都曾经去过。我闭着眼睛就可以感受到千年前战争发生的现场感，这种场景感，我也可以传达给你。」他解读起三国来，绝对是通俗易懂，有趣又有料。
