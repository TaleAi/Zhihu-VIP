## 简介
人潮汹涌的人世间，是否有过你多年来念念不忘的人，深夜亮起灯火万盏，是否有一盏属于你？
那个曾经明媚了岁月的人，如今在哪里？最后，你是嫁给了爱情，还是选择向现实妥协？
午夜梦回时，是否还会想起曾经他？
来这里，希望每一个故事都能感动你。


《友达以上，恋人未满》


在我婚礼当天，我遇到了我发小，也是我喜欢的人。


我曾经喜欢过他很久，一直等着他的告白，可是后来，他有了女朋友。


他对她介绍我是他好朋友，对我说，她是她女朋友。


他叫我刘雪，叫她婷婷。


帮我提行李，眼睛却一直看着她。


可是我还是喜欢他，后来又喜欢了许多年，直到 28 岁时妥协相亲。


我以为我们会到此为止，可在婚礼当天，他来了，告诉我他喜欢我，喜欢了很久。


《满分恋爱，归零离开》


在一起的 4 年，他不记得她的生日，给过她的所有惊喜都不是她所喜欢的，他没有送过她回过一次家，甚至，连他们第一次去开房的钱，都是她出的。


她喜欢的明星，他大肆诋毁。


她想做的工作，他无情嘲笑。


她想得到关心，他却从来都是自我感动。


可他不知道，她心里一直有一个计分器，喜欢的时候是 90 分，热恋是 100 分，而现在，已经变成了 0 分。


如今，她已经积攒够了失望，预谋离开。
