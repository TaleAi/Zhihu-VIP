# 面对问题，有点「孩怕」：那些育儿书上没有的答案

### 作者：Cecilia 等

### 分类：亲子

### 状态：已完结，共 8 节

### 存档日期：2022-06-15

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1434537827463401472


## 简介
就像看过许多道理，仍过不好这一生一样；尽管看过了很多育儿书，但面对孩子「提出的难题」，很多家长仍旧束手无策。


支持孩子探索世界一定是正确的吗？


如果我的儿子想穿裙子，我也该支持吗？


孩子重情义、会分享一定是正确的吗？


如果孩子把贵重物品送人了，我该怎么办？


学龄前的孩子自我意识逐渐萌发，经常会做出家长意想不到的行为；时代的进步，也让我们对育儿的要求从保障孩子「身体健康」到了关爱孩子「身心健康」；


所以，面对新时代儿童突然给出的「难题」，哪怕翻遍育儿书的父母，也很难给出正确答案：


我要如何做才能把握处理关键问题的度？


我要如何回答才能不伤害孩子的天真？


本专栏针对学龄前儿童的父母，筛选了知乎上关注度较高的问题，直接给出解决办法，让家长少走弯路。
