## 简介
少女动画、日式料理、草间弥生、妖怪文化，或是侘寂、御宅族、极简主义、低欲望社会……本专栏将以关键词作为文化标的，由点及面，让你在日本文化地图上巡礼，更好地深入背后的日本文化脉络。


关于日本，也许你听过「山河异域，风月同天」，也听过夏目漱石著名的那句「今晚月色真美」。


但如果在互联网耐心求索，你会发现前者并非鉴真所写，而是日本长屋王笔下，对中国唐代繁盛的佛教文化的倾慕。夏目漱石也并非真的讲过「今晚月色真美」，真正将「我爱你」（愛してます）描述成「月亮真美，也有星星」（月がきれいだな、星も、ある）的，是日本有名的爵士钢琴演奏家山下洋辅。


日本，这个一衣带水的近邻，与我们在文化上既近又远，既熟悉又陌生。如同上述的两则逸事一样，**暗藏着陌生而暧昧的文化密码**。


在今天，我们与日本之间，已经可以通过大量的直接到访，影像观察，包括无数的小说，散文，乃至论文文献阅读，很轻松的了解日本——算是资讯最充沛的年代，我们既可以看到黑泽明的《七武士》，又可以在大银幕上看重映的宫崎骏；我们既拥有各个版本的《源氏物语》，又能在在网络上看到最新版的动漫。可是，即使如此，我们就真的了解了日本吗？


我们说到的这些名人，这些现象，这些作品，就是真的日本？


文艺评论家加藤周一曾说过：「**文化**，**通过造型来表现自己**。**要想了解一种文化**，**就要了解它的形式**。」


了解日本更是如此。


从江户时代的浮世绘到流行文化中不容忽视的动漫，我们从日本众多的文化形式中反复取舍，谨慎地挑选出能够串联起日本古今的 **8 种形式**（浮世绘、动漫、电影、当代艺术、时尚、摄影、和食、茶道），力图为大家勾勒出一个真实而立体的日本世界。


**我们把这** **8 个领域拆分成了 58 个简明有趣的关键词**，**平均每个领域 7 个**。它们互为表里，既联系又独立，能帮助我们更好地描绘出背后的日本文化脉络。


来自北京大学、复旦大学、中央美术学院、东京艺术大学、北京外国语大学等学校的 8 位专家学者，将在自己擅长的领域，从江户时代聊到大众流行，为你解码日本文化中那些既熟悉又陌生，既和谐又矛盾的文化密码。


我们眼里的日本往往和真实的日本大相径庭，专栏通过对日本文化抽丝剥茧地解读，既提供了深入日本文化的轻巧入口，又层层深入，串联起一个真实而立体的日本世界。


接下来跟我们一起，从这些词语穿越进入日本文化的核心吧。
