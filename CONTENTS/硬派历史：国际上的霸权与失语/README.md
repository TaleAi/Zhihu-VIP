## 简介
国与国之间的关系，永远只看利益。


「落后就要挨打。贫穷就要挨饿。失语就要挨骂。」


翻翻世界历史，无数从古至今的惨痛案例，一遍又一遍印证了这几句话。


* 1962 年，美苏两个超级大国搞冷战，古巴成了双方争夺的对象，苏联横跨大半个地球跑到古巴这个小地方布置导弹，差点把整个世界都变成地狱。
* 1994 年，卢旺达大屠杀，生活在同一片土地上的两个民族，在和睦相处 400 多年后，被西方国家挑拨成近百年的世仇，用最原始野蛮的方式，一刀刀砍向对方。
* 萨达姆倒台后，留下一片权力真空。在中东地区，参与这场多边「战争」的玩家越聚越多：美国、土耳其、伊朗、俄罗斯……有权有枪的玩家乐此不疲，遭罪的却是那里的人民。
