## 简介
或许你唱过那首《清平乐》，或许你读过那本《水浒传》，或许你看过那部《知否知否应是绿肥红瘦》。这些经典的影视文学作品，它们无一例外讲的都是宋朝的故事。本专栏将介绍十位宋代词人，带你一起感受宋朝独特的文化魅力。


那是地理版图最小时的中国，却是文化发展最快时的中国。


那是军事力量最弱时的中国，也是经济实力最强时的中国。


那就是一千年前的宋朝。


然而真正能表现宋朝，让你了解宋朝的，往往不是那些后人创作出来的影视剧，而是跃然于纸上的长短句。你读过宋词吗？这是宋代文人的智慧精华，也是中华历史上永远闪烁的恒星。


本专栏将介绍十位宋代词人，不仅可以帮助你更好的理解宋词，也会让你得到更多关于宋词的历史知识。这些词人或是在史书中留下过浓墨重彩的一笔，或是流传于各种民间记录的野史传说。通过不同的视角，你会有不一样的体会。
