## 简介
紫金陈《隐秘的角落》出圈之前，国产侦探推理是怎样的？东野圭吾在日本推理小说占什么地位？本格派与社会派为何纠缠不休？剧本杀、狼人杀源自哪里？……本专栏特邀资深推理迷、北大文学博士卢冶老师，为你讲述推理小说的发展史，专业化解读带你揭秘推理套路，紧扣热点话题，助你成为懂推理、也懂生活的有趣青年。


推理小说的魅力，已经传承百年。


TA 们都爱读推理小说——法国总统戴高乐，是推理小说女王阿加莎·克里斯蒂的书迷；英国的玛丽王后，80 岁高龄时仍对推理小说充满兴趣；世界级诗人博尔赫斯，也要从推理小说里汲取营养。


推理小说历史悠久、积淀深厚。爱伦·坡，这位「暗黑文学鼻祖」，揭开了推理小说的序幕。而关于它，还有很多未知的故事等待我们发现：


为什么英国贵族会偏爱推理小说？日本为何盛产推理小说作家？


推理小说的烧脑属性，究竟是如何炼成的？其发展与世界历史的进程，有多密切的联系？


本格派与社会派各自是如何产生发展的，又各自擅长制造怎样的「诡计」？


在《隐秘的角落》让紫金陈出圈之前，国产侦探推理是怎样的格局？


还有柯南·道尔、京极夏彦、东野圭吾等推理作家，剧本杀、狼人杀等解谜游戏……


本专栏特邀资深推理迷、北大文学博士卢冶老师，带你建立侦探推理文学的历史坐标系，为你讲述推理小说的发展史，从它的诞生、到黄金年代、再到高峰期，让你站在社会、乃至思想的高度上，重新理解推理小说。


此外，本专栏力图用专业化的解读，带你成为深谙推理套路的聪明人；大量涉及知名影视剧、小说，和推理国的名人趣事等，紧扣热点话题，带你成为懂推理、也懂生活的有趣青年。
