## 简介
你知道吗？一份《纽约时报》所包含的信息，比一个生活在 17 世纪的普通英国人一辈子接收到的信息量还多。而我们大脑的「出厂设置」，和 17 世纪的人并没有什么区别。面对信息过载和社会剧变，我们的大脑承受了巨大的进化压力。

令人欣慰地是，人脑具有高度的可塑性，可以针对层出不穷的新需求，在越来越残酷的 hard 模式中不断地自我调整和升级。北大前沿交叉学科研究院讲师葛鉴桥，将在本门私家课中，带给你烧脑、实用且有趣的脑科学科普与用脑指南，助你「升级」自己的大脑。
