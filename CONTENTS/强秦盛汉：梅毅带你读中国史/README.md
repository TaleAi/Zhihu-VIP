## 简介
《百家讲坛》主讲人梅毅，认真打磨《梅毅带你读懂中国史·第一季》，告别死板教科书，给你从没听过的新知和独家史料。不管是表层史实下的底层人性、沿袭至今的处事之道，还是看待问题的大智慧与大格局，梅毅将通过 56 讲为你一一道来。


说到与我们中国人最仰望的伟大时代，强秦盛汉，肯定要算。


它和当今世界有得一比，是一个人才辈出，人人都有机会扬名立万的时代。


 李斯是如何从小公务员一跃成当朝丞相？


 韩信如何从淮阴街头小青年奋斗成大元帅？


 刘邦，一位副股级的泗水亭长，如何后来居上成正国级大汉朝开国皇帝？


《百家讲坛》主讲人梅毅老师将用现场感的讲述还原历史事件，从人性的角度解读有血有肉的历史人物，让学历史成为像听相声一样轻松有趣的事。


论英雄奇士，只秦汉两代就灿若群星；论文化集大成者，中国的诸子百家早早领先于世界；论处事格局，中国古代史堪称一部人间处世修炼教科书。本专栏带你走进中国史，给你只有从古书里才能获得的新知以及频频颠覆认知的真实史料。
