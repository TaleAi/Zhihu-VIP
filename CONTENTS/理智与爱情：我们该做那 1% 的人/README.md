## 简介
备胎算不算爱情，出轨有没有真心？如果爱情只剩下不堪的背叛，无处安放的心动该何去何从。
荒谬爱情的终点站，谁又是别人破碎情感的接盘侠。
所有人都在探索情感边缘的真相，在这些故事里，或许你能找到一二。


张小婵要我给她找个男人，她要证明生不出孩子，不是她的问题。用一场错误去验证错误，有可能得到正确的结果吗？  

——《不能生养的村姑》


他和我说，他老婆不是他真爱。而我怎么也没想到，本该是遇到真爱小哥哥的高糖剧，上来就来了个急刹车，把剧本生生改成了狗血的小三剧。


——《坦白局》


我比任何人都了解备胎，因为我就是一只女备胎。我抱过他的琴，披过他的外套，看到过他的眼泪，可我还是误判了一个男人会扶正备胎的可能性。


——《一只特立独行的备胎》


我是一个情感咨询专家，或者叫「鉴情师」，我坚信世界上 99% 的女人，都无法把婚姻和爱情的关系看透彻，而我应该是剩下的那 1% 中的 1% 。谁料医者不能自医，直到最后我也没有猜到我老公到底是何时出轨的。


——《鉴情师：原来我不是那 1% 》
