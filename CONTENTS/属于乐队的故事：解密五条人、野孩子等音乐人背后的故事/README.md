# 属于乐队的故事：解密五条人、野孩子等音乐人背后的故事

### 作者：叶三

### 分类：艺术

### 状态：已完结，共 8 节

### 存档日期：2022-06-16

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1294226142954491904


## 简介
五条人未成名前的纠结与迷茫，野孩子从西北到云南的漂泊之旅，独家真实成长故事。解密音乐背后的辛酸故事，展示当代独立音乐人的精神世界。


海丰的水土滋养了「五条人」的独特曲调，而西北的原野承载着「野孩子」的音乐才气。


对于艺术商业化这件事，张楚想知道自己是否是一个卑鄙的人。


在这些不为人知的故事里，他们道尽音乐的美丽与生活的疼痛。


音乐是将作者的亲情、友情，回忆、憧憬，将他们对生活的思考、对生命的感悟，全部串联起来。这本身就是一曲动人的歌谣。在他们的故事里，我们听得见那些曾经触动过我们、或一直打动着我们的乐声，看得到隐没其中的、或早已被我们遗忘的青春，忆得起那些难以追溯、不可重蹈的躁动、轻狂又美好的旧时光。
