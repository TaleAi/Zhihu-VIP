## 简介
幸福的人总是相似的，不幸的人各有各的不幸，留给他们的选择已经太少，更何谈治愈自己的人生？



我知道很多事情都不能选择，出生或者环境或者和他们相遇。但是我可以选择我做什么，我想要的人生，爬也要爬过去。如果命运刀尖相向，那我就拔刀相搏。这一辈子终究是我自己的人生，那就好好活过人样。
