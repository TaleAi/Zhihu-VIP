## 简介
成年人一旦心动，就是绝对占有。


有人说：「爱情是场没有酒精的酩酊大醉。」


这世间人人都想独善其身。


却不知，我们都在局中。


索性就正式与自己和解，奔赴一场热恋。


凭着一腔孤勇，在每一个夜里，我只会想你。


  



**《警察叔叔我错了》**


我斥巨资买通了闺蜜，让她换成我暗恋对象的头像。


供我狠狠吐槽。


当我骂道：「扒了你的警服，罚你跪着给我唱征服。」


那个头像回我了：「下楼，我看你怎么惩罚我。」


？？？


闺蜜发来消息：「给了钱，你倒是骂啊。」


救命！我骂到本尊了。
