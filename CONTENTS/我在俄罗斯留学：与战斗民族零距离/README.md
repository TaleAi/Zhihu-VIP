# 我在俄罗斯留学：与战斗民族零距离

## 简介
战斗民族是怎样炼成的？在俄罗斯留学是什么体验？俄罗斯人对中国人友好吗？我是一名在俄罗斯的留学生，在我的十个故事里，有俄罗斯当地警察、流氓混混、战后老兵、美丽女室友、嗜酒流浪汉……希望在这里，你能看到一个真实的俄罗斯。


俄罗斯，一个「艺术感」与「网感」并存的国度，这里有美艳绝伦的姑娘，有才情横溢的诗人，闻名世界的艺术，还有硬核的战斗民族。传闻中，俄罗斯人能单人徒手斗熊，开飞机穿过雾霾天气，跳湖面冰洞找刺激，一人干倒一瓶伏特加。


从世界争霸的两极之一，到经济沉睡的邻国老大哥，我们从各种作品中了解俄罗斯，却没有走出网络，踏上这片土地，感受俄罗斯生活中那些有血有肉的存在。


俄罗斯的警察受贿有多严重？外国留学生的安全感来自哪里？


俄罗斯的姑娘有多漂亮？有一个中国男朋友会幸福吗？


俄罗斯人能有多虎？什么叫战斗民族式的夫妻互殴？


我是一名留学生，生活在俄罗斯的一个大学城，这里有十个或悲伤或温暖的故事，我想为你讲述阿卡沙、根纳季耶维奇、瓦夏、玛利亚、尤里等人的故事，还你一个真实的俄罗斯。
