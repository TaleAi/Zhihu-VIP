## 简介
在观点繁复的世界里，成为清醒者


在每个人都困于信息茧房的时代里，特别是在互联网不需承担太多言语后果的浪潮里，人们开始变得各说各话、党同伐异。


在「杠」与「对杠」充斥的背景下，我们要怎么做出选择，才能过上更好的生活？ 


哲学家陈嘉映告诉我们：「与其费力说服，在智性上使人认同，不如用哲学思考，跳出是与非的立场，在不追求达成共识的前提下，学会与意见相左的人好好说话、好好相处。」


陈嘉映老师被誉为「中国最接近哲学家的人」，更为难得的是，他善于从抽象的哲学思辨中跳出来，从每个人的日常生活出发，思考哲学对于生命和人生观的观照，并用通俗易懂的方式，把道理平易近人地重新讲出来。


十几年来，「说理与共识」这个跟追求良好生活密切相关的问题，一直是哲学家陈嘉映思考的一大主题：「我们与其说需要共识，不如说需要学会，没有共识的人应该如何一起生存。」


他说：「我个人想要的是，认真思考，认真表述这些思考，召唤爱思考的人来一道思考。」


接下来，他会由浅入深地带领你拆解以下问题:


通过说理达成共识，为什么这么难?


积极说理和辩护者的消极说理各指什么?


什么样的说理方式更适合今天独特的时代背景?


共识真的是越多越好么?


.....


可以说，「说理」是活得更明白、更加理解世界的途径，哲学家陈嘉映会教你用哲学的脑袋「对抗」不一样的声音，形成对事件、对世界独特的见解。
