## 简介

从朋友到陌路，究竟是哪一步出了错。无法成为恋人的我们，也许曾经彼此凝望过。


在这个专栏里，你将看到各种「友达以上，恋人未满」的范本，在错过和遗失中缅怀青春。


《竹马不爱我》


他们青梅竹马长大，在他眼里，她是邻居家的假小子，在她眼里，他是优秀的暗恋对象。漫长的成长岁月里，她看着他与别的女孩牵手，亲吻，同居。就在她想要放弃时，竹马遇到了天降。这一回，竹马急了。


……


在爱情里，如果你一直在猜疑他是否喜欢你，是否在乎你，那他可能真的没有那么喜欢你。


当你尝试放过自己，不把他放在第一位，你会发现，原来不爱他也没那么难。


希望认真的女孩都能拥有一段双向奔赴的爱情。
