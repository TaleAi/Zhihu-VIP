## 5.笔记达人的酷炫笔记功能
清晰的笔记，代表聪明的头脑。


你好，欢迎来到在行一点的《思维管理笔记术》课程，我是 Stephanie 老师。今天是第一模块的第四课。


在上节课，我为你介绍 了如何用笔记进行信息管理，并介绍了印象笔记与信息管理有关的功能。只有在信息输入时就为下一步的思考打下基础，才能不断提高思维能力。在这过程中，坚持用正确的方法去记笔记是无法绕过的一条路。


按我个人经验，一旦你走上了这条路，就会发现笔记的作用强大到不仅可以提高思维能力，还会让你的生活发生很大变化。很多人甚至会不断探索笔记的功能，成为一个笔记达人。但这需要花非常多的时间和精力，所以，我决定这节课跟你分享几个我觉得超酷笔记功能，帮你节省一些时间，迅速了解当下最高效、最强大的笔记功能。不仅在信息管理方面能达到事半功倍的效果，在学习以及写作方面也能让你省时省力。


这几个功能是：


·既满足手写的需求，又能像电子笔记一样易于整理；


·笔记一键转思维导图；


·笔记达人必须知道的 markdown 功能。


下面我们就一起正式进入今天的学习吧！


一、既满足手写的需求，又能像电子笔记一样易于整理


我知道，很多人比较喜欢手写，事实上，手写记笔记也有好处。心理学家帕姆·穆勒（Pam Mueller）做过一个实验，研究手写记笔记和电脑打字记笔记的听讲效果，结果发现，用手记笔记的人，比起打字的人，对课上的内容记得更牢。


但很多习惯手写笔记的人一直都有这样一个困扰：记满了一个又一个笔记本，但不管是整理起来，还是查找起来都太耗时间了。


那有没有方法，能把手写笔记和电子笔记的优点完美结合呢？


当然有！方法就是：iPad+iPad pencil+Goodnotes。


实际上就是用 iPad 的手写笔，配合史上最适合在 iPad 上手写的 App——Goodnotes。把你的手写笔记电子化，方便搜索和整理。


![img](https://pic3.zhimg.com/v2-cef12f4a46e4cefd0d029097aae865e7.webp)

（这个就是手写神器——Goodnotes）


很多人都知道，印象笔记也有手写功能，但跟 Goodnotes 的手写体验比起来真的差太多了。但如果只是体验上的差别，我是不会推荐你的，更何况 Goodnotes 还是付费的，需要花几十块才能使用。


Goodnotes 有两个特别强大的功能，首先，它不仅能支持手写，而且能精准检索出手写的内容，这就完美解决了刚刚提到的手写内容难检索的问题。它的分类整理功能虽然没有到出奇的强大，但搜索手写内容的功能已经完全弥补了这一点。


另外，Goodnotes 还可以在导入的 PDF 和图片上，做手写标注。重点是，这个标注不是像其他手写笔记软件那样，只能画个下划线之类的，而是可以写字、画图的，简单来说，就是想写什么写什么。


![img](https://pic3.zhimg.com/v2-7a22ec37953f356d28e0e6c48d31f755.webp)

这个功能有多方便呢？我们上节课分享过，上课时可以拍下老师的 PPT 或者板书，然后统一保存到印象笔记中，但当你需要针对老师的 PPT 记些更个性化的笔记时，用印象笔记还是有些小别扭，不够智能，因为图片只是作为附件保存到了笔记中，再进行加工会有特别多的限制。


但如果你使用 Goodnotes 的这个功能，你就可以直接在老师的 PPT、板书，以及他分享给你的 PDF 上写写画画了。这是一个什么样的体验？是你在纸质书上空白处记笔记的体验！对于「手写控」们来说，是绝大多数 App 都无法提供的。


所以，如果你是个学生，建议你一定要试一下 iPad+iPad pencil+Goodnotes 这种笔记，它可以彻底改变你的学习习惯，甚至让你上课不用带纸质书了。如果身边有学生朋友，也不妨推荐给他们试一试。


除此之外，如果你特别钟爱印象笔记，其实印象笔记也有手写功能。打开印象笔记手机端，新建一个笔记，然后点击附件按钮，选择「手写」就可以了。文稿中的截图展示了这样做的效果。如果在笔记中需要快速画个图，又对图的要求不高时，这种方法倒是一个不错的选择。


![img](https://pic4.zhimg.com/v2-5f98b330e2a4869fb41cbb336fdf7984.webp)

二、笔记一键转思维导图


下面，我们继续讲一下另一个有趣功能，就是把笔记一键转成思维导图。


思维导图大家应该比较熟悉，它是一种可视化的图像思维工具，可以让思考更清晰。我知道很多人在一条一条记笔记的时候，总会觉得自己的笔记看起来有些乱，不够清晰。但这种情况几乎不会出现在思维导图上，所以很多人也很喜欢直接用思维导图记笔记。


如果你是一个没有做思维导图习惯的人，也不用担心，我来告诉你一个笔记转思维导图的方法，让你既不用改变你的笔记习惯，也能让你的笔记更清晰。


这个软件叫做「幕布」，是我试过最方便的笔记转导图的工具。虽然它没有像印象笔记功能那么强大，但它却是把笔记和思维导图结合得最好的一个软件。


它可以通过 Enter、Tab 和 Shift+Tab 三个快捷键，快速为内容划分层级，想分几级就分几级，特别适合用来写大纲。Enter 用来创建新主题，Tab 用来缩进一级，Shift+Tab 用来提升一级，操作十分简单。


比如，


![img](https://pic4.zhimg.com/v2-8047245d20cbe8b43dd388638c290010.webp)

记完之后，点击上方的「查看思维导图」按钮，就可以一键为笔记生成一个思维导图，购买高级版之后，还可以调整导图风格，让它更美观。


![img](https://pic3.zhimg.com/v2-a7109de4ba14b8fd20acdee632bc3f4b.webp)

你以为到这里就结束了吗？怎么可能，我们这节课可是要教你成为一个笔记达人呢！


还记得我们刚刚讲过的，把 PDF 导入 Goodnotes，然后用 iPad pencil 手写笔记吗？那记完之后呢？有没有更高阶的功能？比如把笔记转成思维导图？可惜的是，Goodnotes 无法实现，但另一款软件 MarginNote 可以做到。它也支持手写功能，只不过没有 Goodnotes 那么流畅！所以，如果你不是个手写控，而且有经常阅读 PDF 的需求，那你不要去花钱买 Goodnotes，而应该试试 MarginNote 与 iPad 结合的效果。

不仅如此，点击「大纲」按钮，你还可以看到根据标注生产的整个 PDF 的大纲，而且它还可以生为每个标注的内容生成一个学习卡。

如果说，幕布的转思维导图功能可以让你的笔记更加清晰，那 MarginNote 则为你提供了更全面的阅读和学习服务。


听到这里，你是不是对笔记有了不一样的认知呢？它可以完全改变我们的学习方式、阅读方式，以及思考方式。


三、笔记达人必须知道的 markdown 功能


但别着急，还没完。我还要为你介绍笔记的另一个高阶功能，叫做 markdown 功能。不经常接触笔记软件的人可能不知道 markdown 功能是什么。它是一种操作非常简单的标记语言，通过一些预设编码，让你不用再一遍又一遍地调整文本格式了，颇具「极客范」，能让你在记笔记或者写作时，省下不少时间。


比如，文稿中的这幅图是我在印象笔记里写的，完全没有格式，写完之后肯定要调格式，那就不得不先选中文字，然后点相应的按钮，如此循环往复。


![img](https://pic2.zhimg.com/v2-eec86f197ccec2a5f092d5f6476b0f6c.webp)

那 markdown 能有多省力呢？我们来看文稿中的图。这幅图就是我在印象笔记中建的一个 markdown 笔记，左边黑色背景的是 markdown 编辑框，右边白色背景的是显示框。


![img](https://pic3.zhimg.com/v2-85255214697d6566842ee9521a506553.webp)

![img](https://pic4.zhimg.com/v2-82ba2e578250102557045ec295736d1b.webp)

正常情况下，如果你想弄出右边的效果，写完之后得用鼠标选来选去，点来点去，很多格式有时候还不太会弄，要浪费很多时间。


但如果你用 markdown 功能，只是在文字前加一些符号，就可以达到右边的效果，时间大大缩短。比如，一个#是一级标题，两个#是二级标题，以此类推；「\*\*」是黑体，「\*」是项目符号 ；在文字前加「>」，可以达到引用的效果。


我在文稿的表格中列了几个最基本的 markdown 语法供你学习，只要记住这几个语法，能为你在记笔记、写文章时省下不少时间。


#一级标题


##二级标题


\*\*加粗\*\*


\*斜体\*


\*项目符号


下划线


~~删除线~~


>引用文本


除了表格中的这几个，markdown 还可以插入表格、图片、数学公式、网址，等等。具体的方法，在网上搜一下就可以找到。


但有人听到这里可能还有个问题：这些符号我在键盘，不知道怎么打进去怎么办？比如井号、星号。别担心，很多 markdown 编辑器都支持快捷键，比如在印象笔记中，用 Ctrl+B 就可以直接打出加粗的两个「\*\*」。


除了印象笔记，为知笔记、有道云笔记等也支持 markdown 功能。


四、总结


今天要和你分享的笔记功能就是这些了，但结束今天的课程之前，我有很重要的一点想要提醒你。


课程进行到现在，你已经了解了很多笔记工具，也知道这些工具可以发挥一些超酷的功能，为你节省很多时间。很多人会每一个都想试试，但是探索工具是会上瘾的，你会不断想知道有没有更厉害的功能，发现一个新的工具都去试一试。千万不要忘了你记笔记的初衷，是为了让你的工作和生活更高效，思考更清晰，更不是试验各个软件，成为工具的奴隶。


最后，来回顾一下今天的内容，我们主要讲了三个比较高阶的笔记功能。


第一个功能是，用 iPad 和 Goodnotes，让手写的笔记也能易于整理和存储，还可以导入 PDF、老师的板书图片，在上面做标注，摆脱沉重的纸质教材。


第二个功能是，用幕布可以一键把笔记转成思维导图，让笔记更加清晰，同时，marginnote 还可以将 PDF 上的标注转成思维导图、记忆卡片等，十分适合阅读书籍和学习。


第三个功能是，很多笔记软件都具备的 markdown，不用点来点去调格式，几个语法就能做出赏心悦目的效果。


功能很炫目，但涉及到的软件也很多，是不是感觉要犯选择困难症了呢？不用担心，在讲完接下来的两节课后，我在第二模块会告诉你如何选择适合你的笔记术。


学习了这么多炫酷的笔记功能，是不是信心暴增，那就将本课内容，发个朋友圈炫技一下吧。
