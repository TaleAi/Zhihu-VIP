# 恶之花：暗黑困境中的觉醒和救赎

## 简介
没有阳光眷顾的花，也要努力向阳而生。


**极致捧杀**


上一世，堂妹冒名顶替我上名牌大学，而我则被诬陷替她坐牢。


从此她的人生顺风顺水，光鲜亮丽；而我却灰头土脸，凄惨离世。


重活一世，我决定隐藏自己的优势自我保护，却将堂妹捧成一个「废材学霸」。


最后，再将她一举拽落低谷，打回学渣原形。


而我自己，终于夺回属于自己的人生，领取北大通知书，开启新的未来。


**语言利刃**


高考查分前一晚，妹妹无故跳楼身亡......


追查妹妹跳楼真相的我发现，是同班同学的污言秽语、恶意羞辱，迫使她自杀。


那一刻，我决定为妹妹讨回公道。


可是到最后我才发现，原来我自己竟也是「凶手之一」。


只有那个助我复仇的神秘女孩，才是真心为妹妹着想的人。


......


这个专栏里，在糟乱生活中面临各种困境的女孩们，有人被逼入绝境机智反击，有人既是受害者也是作恶者......万幸的是，危急时刻，她们能冲破桎梏，人性觉醒，获得救赎。


愿每个女孩都有勇气走出困境，自己做自己的光，收获美好的人生。
