## 简介
现代人不小心穿越到古代的步步惊心故事。


我穿越到了皇后的新婚之夜。见到皇上的一刹那忍不住震惊！


皇上有一张跟我前男友一模一样的脸。本以为只是长相巧合，没想到他一开口我就怒了，原来他根本就是我那忘恩负义的前男友！我们两个穿越成了皇上皇后。前尘旧恨纠葛在一起，更别提朝政复杂，后宫险恶……我和前任，何去何从？


——《宁为玉碎》  

我穿越到书香门第的陆家，成为一个丫鬟。家中等级森严，我几乎忘记自己曾是现代社会的一名大学生。大宅中小心翼翼求生，却因缘际会，与府中的二少爷相识了……


——《杨花落尽子规啼》
