## 简介
历史让我们延绵不息。


在很多人眼中，历史是静态的，是史书上的冷静旁观，是拥有「唯一解」的盖棺定论。


数千年的发展与变化、分崩与统一，被笼统地概括成标签，容易记忆，却也让我们离细节越来越远。


但历史真的这么简单吗？


* 为何李世民即位后没有杀手握兵权的重臣？
* 古代攻城战，士兵为什么情愿被派上去送死？
* 古代军队只认虎符不认人吗？皇帝也不认?

本专栏力图从长河里拾取碎片，从细节中剖析真实历史进程中，波谲云诡的交锋，带读者看看到底何以中国，以及是什么让我们生生不息。
