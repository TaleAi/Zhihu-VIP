## 简介
从道家的角度如何解读《红楼梦》？林光华老师将带着你走进人物内心，用曹雪芹赋予其中的道学观念，发现自我。


鲁迅先生说《红楼梦》：「单是命意，就因读者的眼光而有种种：经学家看见《易》，道学家看见淫，才子看见缠绵，革命家看见排满，流言家看见宫闱秘事。」


那么从道家的角度能看到怎样的《红楼梦》？


庄子讲人生如梦，宝玉的梦境又有哪些隐喻？


宝玉多情他爱的究竟是什么？又为什么说这是庄子的无情？


千面《红楼梦》中，究竟隐藏着哪些道家思想？对应现代社会又有哪些思考？


林光华老师将带着你从道家的角度看《红楼梦》，通过对《红楼梦》人物的解读，走进人物内心，用曹雪芹赋予其中的道学观念，发现自我，挣脱所谓成功标准的裹挟，不作强制包装价值观的傀儡。
