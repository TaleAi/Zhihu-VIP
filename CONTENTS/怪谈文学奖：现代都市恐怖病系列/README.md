## 简介
每个人都有着各种压力、担心和恐惧。这些压力被具象化为恐怖故事中的形象，便是现代都市恐怖病。「怪谈文学奖」精选现代都市恐怖病短篇，篇篇击中你的焦虑点，让你读后背脊发凉。


* **熔化：有些存在害怕人世间的光和热**

我家对门住着一个奇怪的女孩，大热天里，她家还开着空调暖风。她说之所以要待在这么热的地方，是为了防止被熔点低的人报仇。


* **失名症：因为一只猫，她忘记了自己的名字**

我在日本一家精神科医院工作，某日接诊了一个中国留学生，她说她忘记了自己的名字。她说她梦到了一只猫，那只猫说只要她拿一样东西给它，它就能实现她的一个愿望，于是她就拿名字做了交换。从那以后，她就忘记了名字……


* **鼠：做了那个实验后，我就算不睡也不觉得困了**

我是一名在日本学生物的留学生，最近一入睡就会梦到无数鼠爬满全身。那时的我正在用鼠做肿瘤实验，亲手杀死了许多鼠……


* **幸运儿：告诉你一种提升运气的方法**

私家侦探周奇受局长情人的指派，去跟踪调查局长。他发现局长竟然接连去了几家赌场，而且在每一家都赢了一大笔钱。另一个赌场员工发现局长总是赢后，便开始注意他是否在出老千，可什么把柄也没有发现。


* **小号：如果你不愿意做你自己，会有别人愿意**

我用来转发优惠链接的小号，突然有一天给我回了消息，让我不要再给他发垃圾信息了。我怀疑是被盗号了，结果发现已经被小号拉黑了。


* **视线：高中那个假期我染上偷窥癖，差点被看到的情景吓去世**

许哲是一名高中生，他在假期里经常用相机偷窥对面楼里的独居女子，观察她的生活。有一天，他发现取景框里出现了灰色的阴影，在女子屋里的墙上动，之后女子变得动作怪异起来，然后突然头转了 180 度看向了自己，他被吓得当场瘫倒在地。
