## 简介
本专栏专深入浅出地讲解人类对宇宙的科学探索，带你参观星系的形成奇迹，同时学习天体物理学家带来的理论小知识，培养孩子对宇宙的好奇心与科学思维，展开一场奇妙的未知世界之旅。


宇宙有无穷的奥秘等着人类探索，你对宇宙有什么好奇心呢？


宇宙曾经是一锅「汤」？宇宙有终结的一天吗？


比星系更大的单位是什么？组成宇宙的「积木块」是如何出现的？


......


当我们仰望星空时，进入对外太空的无限遐想。从渺小生物到浩瀚星空，宇宙有无尽值得探索的地方，其实我们日常生活中看到的宇宙奇观，都可以用科学的方法和视野来诠释。


本专栏由天体物理学专家为你带来专业的宇宙研究方法，通过天文仪器观察、物理理论分析、建立理论模型，讲述物质的诞生、恒星的形状、太阳系的形成等宇宙形成的相关问题。
