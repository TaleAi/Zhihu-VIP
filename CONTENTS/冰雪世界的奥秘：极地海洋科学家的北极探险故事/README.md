## 简介
对于现在的孩子们而言，也许他们接触自然的时间少了，各种电子产品的吸引力更强了。但是事实上，几乎所有的孩子对于神秘的未知世界，依然充满好奇，他们依然有无数个问题在等着爸爸妈妈去解答。


北极是世界上最冷的地方，你对北极有什么了解吗？


生活在地球最北边的居民是谁呢？北极真的有「小狐仙」吗？


太阳和月亮都跑哪去了呢？北极光是怎么形成的呢？


......


中国极地科学考察队员蔡明刚，作为最想带女儿去看北极熊的「超级奶爸」，给女儿和她的的小伙伴讲奇奇怪怪的故事，也讲特别令人神往的北极大冒险。也许就这样，在他们的天地里，会悄悄种了一颗科学的小种子。


面对能够给更多孩子科普的机会，本专栏将用最可爱的表达方式，为孩子们带来最权威的科学知识，为孩子们打开一扇冰雪之门，开启一次北极之旅，让孩子的冰雪世界闪耀科学之光。
