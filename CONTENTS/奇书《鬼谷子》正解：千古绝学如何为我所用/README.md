## 简介
以鬼谷子一生经历为载体，以他的门徒学成实战为案例，为你阐述鬼谷子在做人做事方面的思想，尤其是那些让人意想不到的辩诡之策、揣摩心理之术和长赢之道，助你以纵横家的气势、百战群雄的智慧立足职场，用缜密的思维，冷静的态度去分析现实、应对人生。


如今，鬼谷子似乎被贴上了「成功学、厚黑学、管理学、营销学、奸诈诡术、控制人心」的各种标签。


**这是真实的鬼谷子吗？你真的了解鬼谷子吗？**


史书中说他是长短纵横的谋略家；传奇中说他是神通广大的智者；宗教故事中说他是未卜先知的神仙；民间流传中说他是预测吉凶的命相家。


**鬼谷子到底是谁？为什么这么厉害？**


鬼谷子培养出来的徒弟都是惊世奇才，历史上正是由于他的出现，才有了纵横家的深谋，兵家的锐利，法家的霸道，儒家的刚柔并济，道家的待机而动。


鬼谷子深刻洞悉了人类的欲望，把他的一切智慧，都装在了《鬼谷子》这部旷世奇书之中。


本专栏将我们先从鬼谷子本人的传奇故事讲起，从他游历列国到归隐收徒，再到门下高徒苏秦张仪的双杰争锋，以鬼谷子一生经历为载体，以他的门徒学成实战为案例，用鬼谷子自己的语境，让你穿越回 2000 多年前鬼谷子隐居的山谷之中，身临其境般听他讲述纵横一派的核心思想智慧，从最基础的知识本源打好地基。


我们依托《鬼谷子》原著十二篇，一篇不落的详细解读，每一集都会先为你精选这一篇章中的原文，以白话文的形式直接为你翻译诵读，再结合详实的案例和结尾秘籍提炼，为你阐述鬼谷子在做人做事方面的思想，尤其是那些让人意想不到的辩诡之策、揣摩心理之术和长赢之道，让你零距离、零障碍、读懂、读透《鬼谷子》原著。
