## 16.TFNG解题（中）：True+False 命题原则
上节课，我们讲了 True 题的两种命题原则。一种是针对原文的一句话进行出题。还有一种是对原文相邻多句话的归纳总结。


**True 题第三种命题方式：逻辑推导不相邻信息**


这节课，我们先讲解 True 题的第三种命题方式，这种方式出题的概率非常小。我们展示一道剑桥的经典命题，那就是对不相邻句子进行归纳总结。请看音频文稿图 1.

这篇文章讲空中的交通控制。天上虽然没有马路红绿灯，但飞机也不能随便乱飞，也要遵守空中的交通规则。题目说 Class F airspace 就是 F 级的空域 is airspace 是一种空域。什么空域呢，which is below 365 米以下，而且不在机场附近，这叫做 F 级空域。我们先来定位，题目中 Class F 和 365 是明显的 keywords. 


先定位 Class F.请你找到文章最后一个自然段，G 段。图 2 最后部分 Uncontrolled airspace is designated Class F. 这句话说：不受控制的空域，就是 F 级。但是整个自然段里找不到 365 米，很多同学就选了 NG.


其实在文章 E 段，出现了 365. 我们请看原文 E 段这句话。


In general, 总的来说，from 365m above the ground and higher, 从地面 365 米以上甚至更高，the entire country is blanketed by controlled airspace. 整个国家都是被控制的空域。In certain areas, 在一些地区，mainly near airports, 主要是在机场附近，controlled airspace extends down to 215m above the ground... 控制的空域甚至往下拓展到地面 215米以上。


这个很好理解。飞机要起降，机场附近的空域控制更严格。


在 G 段又说，不受控制的空域叫做 F 级。现在，我们试着把原文这两处不相邻的信息连起来理解。


再来看题目。说F级的空域是 365 米以下， 365 米以上是被控制的，并且不在机场附近。在机场附近 215 米以上就被控制了，所以这道题选 True.


我们总结一下：True 题除了对原文一句话进行同意替换改写、对原文相邻句子用抽象单词进行归纳总结之外，还有第三种命题思路，就是对原文中不相邻的两处信息，进行逻辑推导。


**False 题两大命题原理**


True 题搞定了，接下来我们学习 False 题的两大命题原理。第一种很简单：原文和题干出现了一组反义词，有一组单词驳斥。比如原文说 often，经常的，题目说 seldom 很少；原文说 more than 更多，题目是 the same 一样多，驳斥原文；原文 major，主要的，题目说 minor，次要的。等等。


False 题的第二种命题规律。你找不到明显的反义词，但是这句话驳斥了原文，整个关系在驳斥。


单词型驳斥比较简单，找到一组反义词就搞定。请看音频文稿图 2.

还是空中飞行控制这篇雅思真题。题目说ATC空中的交通管制是在 GC crash 1956 年那次飞机失事之后开始的。题目很好定位，1956，和大写 ATC 定位，迅速找到原文 B 段首句。它说，rudimentary，意思是基础的，初级的，同义词 basic, preliminary. 初级的 ATC existed well before the GC disaster. GC 事故出现之前早就存在了。


原文 before 这个词，明显驳斥了题干中的 after，所以选 False. 


这种一组反义词的 False 题很直观很简单。如果出现两组反义词呢？那是 True 题，负负得正。会不会出现三组反义词呢？负负负得负。整个剑桥雅思真题没有见到过这种设计，你不用担心。

题目说：工业世界里边，每个人的用水量是高的。比什么高？比古罗马的用水要高。说现在工业世界里边，我们每个人用水比古代古罗马的哥们用水多。听起来感觉是应该这样，古人和现代人对水资源的占有率。


原文A段的最后一句话说：罗马帝国兴起时，九个体系，有一堆什么什么东西。后面说supply 提供了古罗马的居民 as much water per person .注意，as much as 一样多，每个人用水怎么样？as is provided in many parts of the industrial world today 和现代人用水一样多，也就是古罗马的水利体系好，古罗马的每个居民用水和现代工业时期用水一样多。
很明显，原文中 as much as 驳斥了 21 题题干中的 higher. 一样多，题干中比它高，那这道题选的是 NO. 所以这就是我们常见的False题的最经典的设计，就是一组单词驳斥，同学们学会了吗？通过大量练习去掌握。


**False 题第二种命题规律：关系型驳斥**


接下来，我们学习 False 题的第二种命题规律。你找不到明显的反义词，但是这句话驳斥了原文，整个关系在驳斥。


请看音频文稿中的图 4.


![图5-4]()加载中...

还是空中飞行控制这篇文章。题目说 The FAA was created as a result of the introduction of the jet engine. 这句话的意思是，FAA 被创立出来，as a result of 是因为喷气式引擎的introduction 出现。换成通顺的中文：因为喷气式引擎这种设备的引入，所以 FAA 被建立。


我们看原文 A 段第一句话：an accident 一次事故，that 后面定语从句修饰， occurred 发生在 GC1956 年上空的这个事故 resulted in 导致了什么呢，FAA 的建立，the establishment.


原文中明显是因为事故，所以我们建立了 FAA. 题目中说因为喷气式引擎，所以 FAA 被建立。


喷气式引擎和事故，其实不是反义词，是风马牛不相及的两个事。这道题选 False. 


原文表达：飞机因为撞了，出现事故了，所以我们管理天上的飞行规则。天上没有红绿灯，原来是随便飞，现在可不行了。飞机越来越多，飞行器越来越多容易被撞，所以我们得制定规则。由于事故，出现规则。


题目说，因为喷气式引擎的引入，所以出现的 FAA，这道题选择的是 False，是因为这个因果关系是驳斥的。这是 False 题的第二种命题规则，关系型驳斥。请注意，你可能找不到反义词。

这篇文章讲欧洲市民保护，它说 problems in Nordic countries 北欧国家那些问题 were excluded 被排除在外了，因为他们不属于欧洲经济共同体。


是不是如此？原文第一自然段最后一句话说: those confined to particular geographic areas 那些局限在特定的地理位置的地方，比如说北欧国家，因此就被 discarded 抛弃了。这个会议上，我们只讨论特定地理位置的国家的问题，北欧国家因此被抛弃。


原文中说，因为它的 geographic areas 由于它的地理问题，所以说，我们不讨论北欧国家的问题。题干说，因为你不属于欧洲经济共同体，所以我们不讨论。这道题答案是False.同样的原因，同样是因果关系。那这个关系型驳斥。因果关系是最常考的关系型驳斥。


接下来我们就要说到 NG 了。NG 题，很多同学理解为找不着。NG 就是句子当中有信息，在原文中找不着。


信息也分两类：第一类是前，就是我们可以理解一个陈述句，这个 true or false not given题干就是一个陈述句，它在描述 A 事物跟 B 事物之间的关系。


主语名词，宾语名词和中间的谓动词，这个句子主干，主谓宾最常见。有可能是 A 找不着，B 找不着，前后信息找不着。还有一种出题是前面找得着，后面找得着，那是关系不存在。这两种 NG，希望你认真去体会。好了，接下来我们来看一看 NG 的命题的真题的这个展示。
