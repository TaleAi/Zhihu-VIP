这节课我们学习雅思阅读的选择题。

**单选题核心原则：单选题是 TFNG 的变形**


好，我要说真经派的单选题核心原则了! 为什么我讲完 True , False , Not Given 之后，马上讲单选题呢？因为单选题就是 True, False , Not Given 的变形，考句子理解。ABCD 的正确选项加在题干中组成一句话，就是 True 的表达。选 ABCD，其实就在选哪个选项加题干是 True 的表达。剩下的三个选项只有两个特征：要么是 False、驳斥原文，要么是 Not Given，原文中找不到。


33 题我们再次定位到「嗅觉的重要性」，来自 introduction 的一句话。请你在 33 题的题干上画三个词，become、aware、importance. 意识到嗅觉的重要性，become aware of the importance. 这三个词等于哪三个词呢？等于最后一句话的 begin to，开始等于become，realise 意识到等于 aware of，意识到aware，那么essential role，essential等于 important，importance，重要的本质的。那么明显通过这三个词定位到这句话。


再看 ABCD 四个选项，原文一句话里边有同一个表达吗？A 选项，我们发现了一个新的嗅觉，最后一句话没有 new，也没有 new 的同义词。我们发现一个新嗅觉好像没提到。B 选项，我们经历到一个 powerful smell，powerful 在 introduction 的第一句话，明显不是这道题的出处。C 选项，our ability to smell is damaged，我们的嗅觉能力被破坏了，好像也没看到。D 选项，we are surrounded by odours，注意 odours、surrounded 这个词来自于 introduction 的第三行，也是在出题句子前面的 without，没有意识到嗅觉重要性的时候出现这个单词，考察的是什么时候意识到嗅觉重要性。所以我们可以明显排除 B、D选项， A 选项也可以排除，剩下的就是 C 选项。


为什么 C 选项正确？因为 C 里面有两个单词是考点，同义替换。C 选项的 ability，嗅觉的能力被破坏了，abilit y画下来。等于 introduction 最后一句话的 faculty. Faculty 除了指大学院系之外，也指能力，ability. 还有一个单词 damaged，等于原文的 impaired. 


如果我把 C 选项放在 33 题题干后面构成一句话，说 we become aware of，我们意识到嗅觉的重要性，当我们的嗅觉能力被破坏时，这句话就是 True 的表达。剩下三个选项，要么是 Not Given，要么是 False.


A 选项，当我们发现新嗅觉时，我们意识到重要性，错，False. B 选项，我们体验到很强烈味道时，False. D 选项，我们被味道包裹时，才认为嗅觉重要，也是 False.原文和题干对比，只有三种可能性：要么一致，那是正确选项，True；要么有反义词设计，那是False；既不一致又不相反，很可能风马牛不相及，那是 Not Given. 我再说一遍，正确选项是 True，干扰选项要么是 Not Given，要么是 False.


以后你做单选题时，我要求你不仅选出正确选项，而且请标注一下：这是 T1、T2 还是 T3呢？是对一句话的同义词改写呢？还是 True 的第二种命题方式：对原文的总结呢？还是第三种。 剩下三个选项也请你要标注，这是False题F1吗？是F1的设计干扰选项吗？还是F2 ？还是 NG1，还是 NG2，这样你可以透彻的发现单选题的命题思路。万变不离其宗，不离真经派的总纲。
