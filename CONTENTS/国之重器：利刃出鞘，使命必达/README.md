## 简介
科技创新铸就「国之重器」。


人类进化史也是一部「工具」制造史，从蛮荒时代的石刀骨针，到战争年代的飞机航母，再到和平时期的芯片卫星等，我们越来越依赖工具，尤其是高技术含量的「国之重器」。


随着「国之重器」精彩亮相，显露着中国科技的崛起，让国人自豪、让世界赞叹。


* 航母争霸，海上霸主，中国双航母南海游弋；
* 东风「快递」，覆盖全球，使命必达；
* 天鲲号，让「永暑岛」成为现实，守卫南海；
* 歼 - 20 翱翔蓝天，接力「81192」 保卫祖国；

古有玉玺，乃国之重器，受命于天，既寿永昌；今有国防，利刃出鞘，受命于科技创新，守护大国使命。
