## 简介
每一行都有认真的人，再平凡的人也有不平凡的故事。30 种形形色色的职业故事，带你看遍人生百态。


-女大学生毕业后被分配到刑庭，为了处理血腥暴力的卷宗，每天下班都窝在宿舍里看恐怖片壮胆；


-追梦失败后，她转战做明星经纪人，全心全意为他人做嫁衣，其实还是为了自己的经济收益；


-北漂青年来到税务公司当税务师，面对一堆「国税发」「国税函」「财税字」眼花缭乱的文号，焦头烂额；


-本科四年做兼职、玩乐队、谈恋爱、打游戏，荒废了学业没有毕业证，如何从卖山寨机的销售变成外资公司的白领？


在这里，有人选择坚持热爱，有人选择跳槽转行，有人看遍了人情冷暖，有人守在自己的心灵小天地。


再平凡的人也有不平凡的故事。在这里，遇见「一千零一种人生」。
