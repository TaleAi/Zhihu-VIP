## 简介
本课由钱耳朵财富学院创始人周勇主讲，主讲人将根据自己多年的一线投资经验，为你剖析富人的思考方式和投资法则，帮你找到最有「钱」程的投资理财方式和职业发展规划。


本课程由拥有 25 年投资经验和财经报道经验的周勇老师主讲。


 


课程将分三个阶段进行财商养成：


初阶：了解生活中的财富运行法则，以小见大，获得大师级洞察力。


进阶：解密航母级企业崛起的核心基因，学以致用，站在创始人高度考虑问题。


高阶：建立宏观经济战略视野，看投资走势，做有前景的理财和职业规划。


 


提高财商，为每一次决策嵌入财商基因；


刷新认知思考维度，更快突破财富困局；


武装前驱性眼光，综合检视商机、制度和时代，找到「钱」程最优解。
