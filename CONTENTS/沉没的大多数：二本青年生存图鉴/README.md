## 简介
第一学历自卑、职场歧视、圈层和阶级的壁垒……身为「二本生」真的就这么难吗？他们是大多数的年轻人，在现实残酷的环境下孤军奋战。这是以「二本学生」为主角的短篇故事集，这里有绝望的命运，心酸的过往，也有逆袭突围的胜利，和未来的无限可能。


* 背着「不孝」的骂名，干着一本生嫌弃的零工，为了钱，她能付出一切
* 跃出农门后，我受尽「只招 985、211」的职场歧视，被所有人称作「废物」
* 因为期待与不甘，让我从一名本科生变成了一名专科生

**中国二本院校的学生，折射了中国最为多数的普通年轻人的状况。**


他们是中国大多数年轻人，在现实又残酷的环境下，有人对未来摩拳擦掌，跃跃欲试；有人为摆脱「出身」的原罪和自卑，逆袭向上；也有人失去理想，接受了自己普通的一生。
