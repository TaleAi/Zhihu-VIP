## 简介
正值冬季，在东北一座普通的城市，五口之家正准备度过一个悠闲的星期天，却遭遇了突如其来的丧尸大爆发。没有超能力，没有异空间，不开金手指，演绎中国版的行尸走肉。



女主是和父母在家看娃的全职妈妈，老公跟朋友开着一家收入普通的小公司，儿子刚满 11 个月。


照常早上 7 点起床，突然发现整座城市已被丧尸围城。


居住在 18 楼的平凡夫妻，要怎么保障一切生存的基本需求，带着父母和幼小的孩子逃出生天?


当赖以生存的小区不再安全，小区里的幸存者们又如何凭着普通人的智慧和力量，寻找一个更安全的地方藏身和生存？


没有超人般的男主，没有脑残的美女，没有主角光环，只有手中的尖刀。


砍头，刺脑，躲藏，逃跑。


一起囤货，一起逃难，一起将丧尸爆头，且看末日生还者如何建立一座丧尸无法攻破的崭新家园！
