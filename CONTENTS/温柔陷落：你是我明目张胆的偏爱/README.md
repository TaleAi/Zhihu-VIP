## 简介
「在我这里，你永远第一。」


你是否因为不够自信，不敢向喜欢的人告白？


你是否因为太过普通，从未被人坚定地选择？


暗恋一个人时，宛如心里有鬼——怕他知道，怕他不知道，怕他知道装作不知道。


幸好，我喜欢的那个人，最终都会用实际行动告诉我：


「在我这里，你永远第一。」


  



**《我心里有鬼》**


我死后，竹马找了个和我很像的女人，将她养在我住过的房子里，给了她最极致的宠爱。


你以为这是替身梗？


不，他说要用这具身体复活我。


可我只是个穿书的短命白月光，她才是你这个偏执反派即将爱上的女主啊！
