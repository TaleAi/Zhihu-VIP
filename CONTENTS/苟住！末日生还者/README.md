**《苟住！末世囤货大作战》**


末日前七天，我重生了。  

带着重生前的记忆，我决心保护好父母，苟活到最后！我拿出所有存款，开启了疯狂采购模式……


**《苟住！丧尸不敌锦鲤》**


丧尸病毒爆发，末日来临，我和闺蜜恰巧提前囤了一大批物资，非常幸运苟住了。  

我们还顺带救了楼下一个老太太，三人相互扶持，在这末世中艰难地生存着。


**《苟住！裸辞遇丧尸》**


辞职第二天，我准备在家宅一个月不出门，采购到一半时，却意外听到保安大叔的儿子打电话跟他说丧尸病毒即将爆发。
