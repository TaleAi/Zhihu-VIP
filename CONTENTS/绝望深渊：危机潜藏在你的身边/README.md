你体会过真正的绝望吗？


是你无论多努力，却还是抵不过命运的把玩。


是你被当作小白鼠，让人肆意捉弄。


是你被拖入了利益的沼泽中，却无法挣脱束缚。


**生机？还是死路？**
------------

哥哥在前面开路，弟弟在后面跟随。


入洞后，他们本以为经过一条极窄的甬道，前面就会有一个宽敞的空间转身。


结果，他们遭遇了死亡岔路……


**网红酒店？还是隐秘人体实验室？**
-------------------


试想一下，当你拎着行李，风尘仆仆推开酒店大门，你本以为可以舒舒服服睡一觉。


殊不知，你的隔壁是毒气室和密室，酒店地下室是大型解剖室……


而这个酒店的住客，有来无回。
