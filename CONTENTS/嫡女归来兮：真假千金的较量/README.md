## 简介
被偷换的人生、已扭曲的亲情，前世她处处讨好、步步惊心，却惨淡收场。
睁眼回到悲剧起点，她端起嫡女的尊荣，露出了掩藏的刀锋。
前世她错盼恶人垂怜，这一世，她将恶人一脚一脚碾进深渊。

甚至死后她才得知，她的夫君一心想娶的竟然是那个冒牌货苏晴嫣。


重活一世，她誓要将曾经的一切统统讨要回来。


纵使机关算计，也要护住母亲、兄长和外祖一家！
