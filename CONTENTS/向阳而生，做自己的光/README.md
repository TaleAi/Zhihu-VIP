## 简介
如果过于黑暗，那就亲手凿开一道光。


**《绝对算计》**


我是富二代的女朋友，名义上的。


他和他见不得光的恋人，把我当成工具人，对我颐指气使。


就连他们的家人也有样学样，对我呼来喝去。


但我无所谓。


我只想让他们知道，所有命运的馈赠，早已在暗中标好了价格。


……


有人重活一次，赢回所有希望；


有人冲破亲情束缚，重获新生；


有人带着清醒后的豁然，决绝转身；


这个专栏，我们希望通过她们的经历，告诉每一个人：


女孩也有足够的力量，去破开黑暗，重见天光，自己做自己的光。
