# 再次心动：陪你走到最后的温柔爱意

## 简介
这是一个糖里拌着玻璃碴的甜虐言情合集。


故事里的爱情，开始或许不够美好，甚至虐身虐心，万幸的是过程中我们及时弥补，再次拥抱彼此。


**心上的玉**


林钰一直都觉得，男朋友对她好到没边。


肯冒着大雨为她横穿整座城市买花，


会背着醉酒的她穿过十条街道回家。


他说他爱她说得那么真切，她都信了。


直到那天，她看到了他和以前兄弟的聊天记录：


「我从没见过这么好骗的人，真的。」


「所以我哪怕对她好一点点，她就上赶着来舔我。」


**他是真的后悔了**


林子暮是陆臣卿的小跟班，这是谁都知道的事儿。


可是有一天，她不再跟着他了。


陆臣卿慌了，三番五次地主动招惹，林子暮却理都不理。


直到某天，猛然恢复了十三岁那年的记忆，陆臣卿才想起来。


原来不是林子暮喜欢陆臣卿，是陆臣卿喜欢林子暮。


 ······


猛戳本专栏，看各种破镜重圆、久别重逢的甜虐爱情，体会再次心动的感觉。
