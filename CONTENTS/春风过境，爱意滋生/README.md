人生百味，挑甜的体会！


最美好的事情是什么，是你喜欢的人刚好也喜欢着你。


遇见爱时，少年时懵懂，中年时胆怯，但无论何时那些心动的瞬间都是足以铭记一生。


《我死后的第十年》


「我们在一起吧，秦柔。」


成钰抱住了我，他的吻落在我的颈侧。


「我会试着忘掉她的。」


「她」是成钰死去多年的妻子。


他一定不会相信，那个死去十年的人正躺在他怀里。


我翻过去搂住了他。


「好啊。」
