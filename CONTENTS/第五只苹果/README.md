## 简介
为什么苹果总是出现在人类历史的关键节点上？善恶树上的苹果、砸在牛顿头上的苹果、毒死人工智能之父的苹果、乔布斯选用品牌标志的苹果……那改变人类的第五只苹果又会在何时出现？


未来社会，病毒肆虐，科恩提出「蜂巢系统」让人类避险，即让人类把肉体寄存在蜂巢内，灵魂得以在虚拟世界自由翱翔。


这项被称为「人类大迁徙」的举措持续了数年，近 20 亿人口移居虚拟世界。移居者被称为「蜂巢人」。科恩也借此成为了世界首富。


蜂巢系统是一套具有生命维持功能的虚拟体验设备，蜂室是蜂巢系统的基本组成单元，被置于蜂巢树中。


一天，蜂巢公司的创始人科恩死在了暗影镇贫民窟，枕边放了一枚毒苹果，死状与一个多世纪前人工智能之父图灵如出一辙。


神探孟云白奉命调查科恩一案，发现清洁女工艾米丽有重大嫌疑。艾米丽太太的丈夫死于「蜂巢」，她的儿子也流连于此，不肯返回现实。


随着调查的进一步深入，警方发现「科恩之死」只是更大阴谋的冰山一角……


这历史关键节点上出现的第五枚苹果，究竟代表着什么？
