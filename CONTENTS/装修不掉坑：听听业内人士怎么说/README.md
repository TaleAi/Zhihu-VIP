## 简介
按照装修流程，从报价单开始，教大家避坑，把遗憾降到最小，把费用降到最低！


装修这个行业，经常被业内人士称为「大行业小公司」，即行业虽大，五花八门的小公司却此起彼伏，整个行业没有一个透明的行业标准，也缺乏完整的管控体系，全靠商家的良心。


所以，经历过装修的人基本上都有这样的痛苦：


* 担心装修公司的价格不透明，等施工过半了才被告知资金不够了需要加钱，骑虎难下；
* 担心装修公司在施工过程中变来变去，导致最初确定的设计效果图跟实际装修完工的完全不同；
* 担心一线的工人不负责任，为了赶工期，敷衍了事，为以后的生活埋下很多「定时炸弹」；
* 担心建材没选对，全屋装修完后甲醛超标，带来健康隐患。

该专栏邀请知乎「装修」「室内设计」话题下的优秀答主，以及第三方评测机构，从制定装修预算，到看懂报价单，到硬装，到软装，针对大家在装修过程中都会遇到的问题，解疑答惑，同时也会提供一些消费决策上的建议。


装修房子是一个「痛并快乐着」的过程，希望这个专栏能帮大家减少一些装修过程中的「痛感」，更快乐地拥抱自己的新家。
