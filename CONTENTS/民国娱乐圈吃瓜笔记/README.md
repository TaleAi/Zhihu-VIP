## 简介
向来心是看客心，奈何人是剧中人。


民国多才子佳人，娱乐产业也曾经风靡一时。


当时诞生了一连串名演员、名编剧，各路权势也纷纷闻香而来，同样也留下了许多或曲折或动人的故事。


然而生活像一件爬满虱子的华袍，远看无比风光，但是扒开看，又是另一番景象了:


* 民国时期的娱乐圈是否也需要背靠资本？
* 大明星离婚是怎么打官司的？
* 民国时期想成为明星，要如何才能出道？

本专栏将会以原始的档案资料为蓝本，将民国娱乐圈背后的种种你不知道的事情讲给你听。


读者能通过这些鲜为人知的资料和记录，对民国这个特殊时期的社会制度、政治、经济、文化，有更立体的认识。
