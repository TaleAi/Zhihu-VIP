## 简介
这个年代居然还有租赁录像带的录像厅？



我叫小北，在上海最繁华的华山路上开了一家「闪灵」录像厅。


经常有客户上门，委托我调查匪夷所思的亲身经历。这些扑朔迷离、难以解释的事件，被我称为「闪灵事件」。


调查是免费的，但委托人必须同意我全程拍摄下来制作成录像带，在店里进行小规模秘密放映。


在这个专栏，我要讲述的离奇事件，都出自我的录像带。


· 存了一辈子的血汗钱，买到的学区房竟然是凶宅


· 来自平行时空的证据，她的出轨是一场「活见鬼」


· 老同学整容变美女，一夜风流后才知她已死三个月


摄影机就位，故事开始。
