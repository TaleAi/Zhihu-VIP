## 简介
一个关于女孩们的悬疑故事专栏，根据真实素材改编。


这是一群被「狩猎者」们盯上、陷入巨大危险中，最终决定反杀重生的女性们的故事。


她们可能是被跟踪的惶恐不安的单身女性独居者，也可能是被丈夫牢牢控制的绝望主妇，是网络暴力下濒临崩溃的上班族，是被校园霸凌欺辱的大学生，是不堪忍受职场性骚扰的女白领…… 


「无处不在的眼睛」：独居的单身女白领，感觉四周似乎有一双看不见的眼睛在无时无刻盯着她。几年前闺蜜被偷窥后自杀的惨剧仿佛又要开始应现在她身上，她陷入巨大的惶恐中，身边每一个人似乎都是那个可怖的偷窥者。某天晚下班，她被偷袭，一个男人救了她，开始走进她的生活。她以为自己的噩运会因此扭转。直到那一晚，两人即将开始一个甜蜜夜晚……


「爱我就去死」：第一次去男朋友家，他的母亲竟然让我给他们全家刷鞋子？为了证明「我是真的爱你」，女孩面对男友无礼的要求一次次妥协退让，殊不知掉入男友 pua 的陷阱中，直到得知男友以同样的方式，玩弄过不计其数的女性，她和好姐妹决定以其人之道还治其人之身。 


「是谁杀了他」：一觉醒来，丈夫死在自己身边，孩子不知所踪，原本幸福无比的家庭主妇成了「凶手」，为了洗清自己的罪名，她不得不一次次回忆宿醉那晚上到底发生了什么，正是这一连串苏醒的记忆，渐渐引出女人幸福生活的背后，不为人知的秘密……


到底是不是自己杀了丈夫？身上的伤痕又是谁造成的？童稚天真的孩子竟然从未存在过，「我」到底能不能走出迷雾？


……


因为自身的弱点，这些女性成为狩猎者的猎物，她们也许一开始软弱躲避退让，当退无可退时，她们终于觉醒，迸发出强大的生命力，直面困境，绝地反杀！


故事均取材于实际生活，根据新闻案件进行改编。


有时，生活远比故事艰险。
