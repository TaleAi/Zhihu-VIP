## 简介
谷底开花，海底望月，甜虐交织的爱情故事～


**炙吻**


季淮很讨厌他的小青梅。


嫌她黏人娇气，总是打扰我们约会，每次都不耐烦地赶她走。


如他所愿，在我们订婚的那天，小青梅真的走了。


可季淮却抛下了一切，疯了似的去找她。


春寒料峭，我茫然地站在夜雨里，看他们吻得缠绵放肆，浑身湿透。


暗处忽地伸出一只玉白的手。


季淮的小叔在我头顶撑起一把伞，黑眸清寂，淡声开口。


「想报复他吗？」


「那就和我在一起。」


······


对我心动吧。


这次，下次，次次。


我会一路撒着星星，奔向你。


怀抱鲜花，与你交换一场滚烫的浪漫热恋。
