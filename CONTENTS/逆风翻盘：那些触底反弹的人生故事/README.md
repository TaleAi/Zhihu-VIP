## 简介
即使是不被所有人看好的人生，也有逆风翻盘的可能。


中年失业，我背着妻子悄悄送外卖，却撞上了她的出轨现场。同时失去家庭和事业，我该何去何从？


大学毕业，心高气傲的我带着一腔热血准备创业。没想到遇到了传销，钱没挣到，反而被骗了找亲戚朋友借的二十万。有一天，我在出租屋上网打发时间，无意中点进了一个房地产网站。没想到，身边的一切就此改变。


当红女明星暴毙，我被要求狸猫换太子，成为她的替身。当我凭借自己努力，终于在娱乐圈站稳脚跟之后，却发现原来女明星的去世，隐藏着黑暗的秘密……


  



30 个大快人心的逆袭故事，穿过谎言、背叛与伤害，我的人生依旧精彩！
