## 简介
作家林遥为您演绎讲解《金瓶梅》中的人性与人心。


《金瓶梅》，一部千古禁书，到了现代为何被评为 18 岁后必读奇书？一部有「污点」的书，如何成为名著中的名著？


最重要的，大概就是因为它「有内容」：**洞察人心、写透人性。**


鲁迅曾评价《金瓶梅》「作者之于世情，盖诚极洞达」，认为同时代世情小说，无出其右者。


**潘金莲：**全书「最坏」的女人，也是最拼尊严的女人；


**李瓶儿：**口含金汤匙、手握女主剧本，一手好牌却一无所有；


**庞春梅：**为人仗义，有理想、有伙伴的「底层员工」最终成功。


**西门庆：**财商养成，靠女人做首富，抖机灵「钱生钱」；渣男本性，商人、恶霸、官僚于一体，三界通吃；领导艺术，竟把自己的跟班培养成「西门庆第二」。


《金瓶梅》写透人性善恶、世事炎凉。**只有看懂《金瓶梅》的恶，才明白如何守护善。**


所有人性、世情、欲望和生意经，


听超人气评书人林遥，为您娓娓道来。
