## 简介
本课程为后妃秘事系列第六弹，由超人气主播楚扬老师主讲。本期主要讲述的是三国时期后妃们的精彩故事。让我们一起溯古怀今，在历史中品味多样的人生。


现在人们看宫斗剧，都说自己活不过第一集，是不是古代的妃子们都要成天琢磨怎么耍阴谋诡计？


未必，你且看这历史长河中，既有那心狠手辣，千古毒妇吕雉；也有那沉鱼落雁，忍辱蛰伏敌国的西施；还有那杰出的政治家、女英雄妇好……


后妃作为封建君主专制制度下的特殊人物，她们的人生遭遇和一般的古代女子有相同也有不同。随着封建君主专制制度的消亡，她们也早已成为历史的陈迹。但历史是一面镜子，我们可以从她们的故事中窥见时代的变迁，岁月的积淀，也可以看到现代社会的缩影。


后妃秘事系列课程由超人气主播楚扬老师主讲，他将用抑扬顿挫的语调，引人入胜的故事带你探秘古代后妃们的传奇。从战火纷争的乱世到繁荣兴旺的盛世，从地位低微的妃子到母仪天下的皇后，她们的故事等你来倾听。


本课程为后妃秘事系列第六弹，主要讲述的是两晋时期后妃们的精彩故事。在这里，你将会看到聪明绰约，深谙男人心的武元皇后；红颜薄命，二八年华便为后的杜陵阳；何守寡大半生，历经六朝皇帝的何法倪……让我们一起溯古怀今，在历史中品味多样的人生。
