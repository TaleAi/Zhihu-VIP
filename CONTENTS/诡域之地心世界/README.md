## 简介
长篇科幻探险小说。一场绝无仅有的地心世界探险。


叶江山并不是「这个世界」的人。


原来十年前，他误闯入「百慕大三角」，从另外一个镜像世界来到了这个时空。因为念念不忘另一时空的未婚妻——尚小可，重返自我时空已经成了他的心结。


而他已经在这个世界生活十年，没有一丝一毫的迹象显示他还能重新回去。


叶江山就职于特调局，一个专门侦查超自然事件的神秘机构。保密级别——绝密。


这一天，特调局接到通知，国家安全局绝密资料被盗，行窃者竟然是一种昆虫——独角仙。而与资料一起丢失的还有红星桥史前文明遗址二号坑中曾发掘出的一枚奇怪「权杖」，据说这枚权杖涉及到人类基因的终极秘密——五重基因锁。


鉴于此事的诡异性与严重性，特调局立即展开了调查。


国家保密局的绝密资料诡异被盗，特调局介入调查，远赴大西北……


他们不知道的是，他们，即将迎来一场前所未有的地心世界冒险。


敦煌古城的密码？史前大洪水是否只是传说？神话故事究竟藏着什么秘密？人类真的被毁灭过吗？


而叶江山，是否能从这次空前的任务中，找到一丝自己世界的真相……
