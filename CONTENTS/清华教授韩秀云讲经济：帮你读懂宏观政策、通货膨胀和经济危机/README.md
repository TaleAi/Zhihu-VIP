## 简介
清华的 MBA 课，教你看懂经济，掌握先机


通货膨胀、金融危机下你该如何应对？


「五大率」与「一个周期」让你读懂日常宏观指标，看清经济趋势。


「五大经济学概念」，教你学会站在经济学角度思考问题。


不懂经济学？不清楚经济形势？你就无法做出正确的判断！


懂一点经济学，你就可以了解国家政策的背景，结合自身需求，定位发展方向和目标；准确把握经济运行态势，多抓住一些机遇；合理理财，做到投资利益最大化，实现财富自由。


经济学研究的，正是你所关心的。其实。经济学并不难。并且你一定要学，不要被「经济学」三个字吓住。


本课程中韩秀云教授将通过一个个妙趣横生的故事，将经济学理论化繁为简、深入浅出地呈现在你的面前。即使你没有经济学基础，也能通过课程快速了解经济的变化，让你站在一个更高的位置，看清产业格局背后的资本流向，能够在抓住机会的同时规避风险。
