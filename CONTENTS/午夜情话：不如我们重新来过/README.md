## 简介
有没有一个人，你想到就想死。


关于遗憾、关于错过、关于久别不重逢、关于彻骨的伤害与疼。


春天该很好，你若尚在场。


  



**「爱了很久的朋友」**


身边的人坚信我跟池樾曾经发生过点什么，但并没有，那段暗藏的甜蜜心事还欲说未说，他就不辞而别了。


  



**「我爱不起你了」**


收到顾云亭自杀消息的时候，我们已经分手三年了。


是他妈妈打视频电话告诉我的。


恋爱的时候，我曾加过他妈妈的微信，但我们没怎么说过话，分手后，我也忘记删她了。
