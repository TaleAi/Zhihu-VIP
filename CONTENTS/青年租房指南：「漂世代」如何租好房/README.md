# 青年租房指南：「漂世代」如何租好房

## 简介
「漂世代」实在是太难了！


年轻人初入社会，租房基本是必修课。


而加入「漂世代」，租房就成了年轻人「接触社会」的第一站。


一个暂时安稳、舒服的家，是所有「漂世代」的期待；


但大部分是不幸的：


* 或遇到黑中介和二房东的各种为难；
* 或遇到房东突然违约，被迫搬家；
* 或是长租公寓甲醛房，资金暴雷跑路；
* 或是一直在租得近、租得好、租得省，这个不可能三角来回摸索。

这个专栏，资深房产经纪人，与你一起直面租房焦虑，努力寻求「最优解」，在这个充满焦虑的时代，找到自己心仪的房子。


房子是租的，但生活是自己的。
