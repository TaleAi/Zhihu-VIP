## 简介
如果你是营销人，很可能遇到过这些情况：拉新拉不动，留存留不住，转化很困难，产品销量差。归根结底，这是因为你不具备用户思维，对用户不够了解。「42章经」创始人曲凯，为你带来理解用户思维的「独孤九剑」，通过九个系统性步骤找出用户想什么、用户要什么，最终实现业绩持续增长。


**作为一个运营和营销人，你是否遇到过：**


一周内换了 N 种方式做运营，拉新拉不动、留存率还越来越低


微信文案一篇接一篇地推广， PV 不高，转化率更是堪忧


设计了新产品，目标用户表示愿意买单，但真上市了业绩却惨不忍睹


**绝望的你，感觉自己掉进了一个无限循环的坑中.......问题到底出在哪儿？**


**原因在于，你不具备「用户思维」。**你从一开始就找错了「用户」，也并不了解「目标用户」的真实需求。击不中「痛点」，转化率提高和业务增长从何谈起？


只有找到「正确」的目标用户，清晰地了解他们的真实需求，你的每一个运营动作、每一句文案，才能撬动用户心智，提高转化率。


「42章经」创始人曲凯，给你一份把握「用户思维」的行动清单，照做即可。


**你将收获**


了解用户：一套系统的用户调研、用户访谈方法


实现转化：一套提高用户转化率的实用策略


业绩落地：一套实现业务增长的系统步骤


**适宜人群**


正在规划或者刚刚启程的创业者


互联网或者传统企业的市场营销从业者


希望实现用户增长的产品经理或运营人员
