说到《西游记》


很多人觉得就是师徒四人一路斩妖除魔取得真经的故事。


但其实西游背后所包含的文化内涵超乎很多人想象。


孙悟空的身世和中国文化有什么样的渊源？ 


八卦丹炉是怎么表达事物自身变化的阴阳系统？ 


猪八戒背后交织着佛道什么样的文化关系？


神仙等级如何体现中国道家内丹派的经典学说？


  



![img](https://pic4.zhimg.com/v2-c35cf214addd118b615ac57e8c24c01f.jpg)

《西游记》的故事大家都耳熟能详， 


但对于其中所蕴藏的文化底蕴，很多人都只看到了冰山一角。 


如果想真正理解《西游记》， 


最好方法还是走进原著。 


解谜的「玄机」就隐藏在其中， 


带着疑问再来品读， 


不仅能够帮助我们读懂西游， 


也能够拓展我们对中国文化、人生之道的理解。


  



![img](https://pic4.zhimg.com/v2-1e09b53a486d7cbd24bdb2966aea086d.jpg)

本套课程由百家讲坛知名主讲嘉宾韩田鹿所作。 


以一集原文朗读一集解读的形式， 


逐步解读西游世界背后不为人知的精神内涵。 


既观点独特，又推理严谨， 


让人脑洞大开，拍案叫绝， 


让经典有趣又有料。
