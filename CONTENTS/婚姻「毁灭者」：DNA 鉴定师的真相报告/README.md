## 简介
我是一个亲子鉴定师，从业十年，经手的亲子鉴定委托已经有一万余件了，「亲手拆散」的家庭也已经达到两千多个。从事这样的一份工作，见惯了各种奇葩人和奇葩事，见惯了各种人们在平时努力掩藏的阴暗心理，见惯了人们在失望乃至绝望的时候会做出如何惊世骇俗的事情来……我用我的眼、我的笔把这些人和事都记下来，把这些绝境之下的人性爆发真实地呈现出来，希望能引起一些思考，或者引以为鉴。


被软禁在家的富商妻子离奇失踪，屋内门窗紧闭、监控也显示没有任何人出入，室内只有一个掀翻的马桶，露着黑洞洞的下水道管口，难道真的是下水道吞噬了这位少妇？警方来亲子鉴定中心调查，我才知道了这起诡异失踪案背后惊人的真相！


民国时期，因为抗战分离的两姐妹，逐渐流落到祖国大地的两端，几十年后，父母、兄弟相继去世，两个老姐妹还没有放弃寻找彼此，他们的儿孙终于帮两位老人在互联网上牵上了线，并最终在鉴定中心「滴血认亲」！


福哥拿到「排除」的鉴定结果后，痛哭不已，等情绪稍稍稳定后，他说：「你知道吗，这是第二次了。」原来，福哥和妻子结婚后，妻子曾出轨、生下了别人的孩子，福哥原谅了妻子，在妻子生二胎后，福哥非常开心，结果妻子却说：这个孩子也不是你的。


年近花甲的钱老大爷急匆匆地来做鉴定，想知道小他二十岁的妻子生的孩子是不是他的。结果不是，钱老大爷暗中取到了所有怀疑对象的检材来做鉴定，却都被排除了。最后，孩子的亲生父亲竟是钱老大爷最想不到、和他同住一个屋檐下的男人！


带五个男人来做鉴定的富家女、可怕的斯德哥尔摩之爱、因为一个有钱人逐渐疏远的闺密……亲子鉴定中心，人性最阴暗的角落在此被照亮、婚姻最残酷的一面在此显露出来。


本专栏由著名亲子鉴定师、作家戴维根据亲身经历撰写，其在天涯连载的「DNA 鉴定师系列」早已成为天涯最热门经典的帖子，甫一出版即成为畅销书，让我们跟随他的笔触，看那些在鉴定中心被撕开的血淋淋的真相。
