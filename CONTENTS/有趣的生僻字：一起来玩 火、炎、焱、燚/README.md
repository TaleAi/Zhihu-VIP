70 堂课，一颉老师带你从各类古籍记载和神话传说中挖掘生僻字的起源和故事，带你重拾被冷落的汉字之美！


生僻字又又又又双叒叕火火火火炎焱燚啦！


中华文化五千年，留下来的文字瑰宝要被玩坏了！


为科普而来，保证一点都不水水水水水沝淼㵘！


最近，一首《生僻字》火遍了网络，身边的小伙伴们几乎都在网上听过那熟悉的旋律。


在欣赏音乐的同时，大家都在感叹「外语不认识也就算了，怎么自己中文也成问题了……」


如果你也这样想，这门「生僻字」课堂就是为你而来！


在这里，一颉老师将用引人入胜的故事带你进入神秘的「生僻字森林」中，探寻它们的独特故事。


从石刻字符、甲骨文、金文、篆书到隶书、楷书、草书、行书……


落笔成书写下五千年历史，我们的汉字，由古迄今，都是故事。


一起寻找最古老的文字，了解最有趣的生僻字。
