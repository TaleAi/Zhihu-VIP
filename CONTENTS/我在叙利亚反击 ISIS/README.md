## 简介
我的学生咕咕考取了一所不错的国外大学，却没有去留学，而是加入库尔德国际志愿兵，开始了在叙利亚抗击 ISIS 的军旅生涯。这几年，他见过太多生死，经历了许多惊悚时刻，甚至失联许久。本专栏记录的故事，来自咕咕的口述，相信它们将带你到炮火纷飞的战场一线。


如果被敌人俘虏，将遭受怎样的非人折磨？咕咕随身携带一枚手雷，为的是在被捕前炸死自己。


恐怖袭击后，为何不能清理尸体？很多人身上藏着炸弹和诡雷，一旦移动，很容易被引爆。


战争中，危险只来自于敌人吗？许多士兵患上严重的精神疾病，有人因为战友的一句话开枪自杀。


ISIS 最怕的部队是什么？库尔德女子自卫军，因为在他们的信仰中，死于女人之手，将不能上天堂。


当国际志愿兵，能赚多少钱？咕咕说他们一分钱不拿，是为正义而战。


咕咕第一次参战，他身经百战的战友老大哥，就死于一颗流弹。在战争中，生与死只是一瞬。


咕咕说：「只有死人能看到战争结束。」


希望你阅读完本专栏，了解到战争的残酷，更加热爱和平。
