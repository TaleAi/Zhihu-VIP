# 魏晋名士录：那些乱世中的风流才子们

### 作者：雪花如糖 等

### 分类：文学

### 状态：已完结，共 30 节

### 存档日期：2022-06-03

## 简介
专栏生动再现了每一位名士在动荡混乱的社会中的沉浮人生，并试图通过还原魏晋时期的社会风貌，来帮助读者深入理解名士们的命运抉择，同时又加入了适度的文学想象，把魏晋风流人物率真、任诞、从容、洒脱、追求个体自由的精神气质立体、饱满地传达出来，让我们更清晰地感受魏晋那段特殊历史时期的荒诞、痛苦而又令人着迷的文化气息。


说起名士，最有代表性的当属魏晋时期的风流雅士，他们纵情山水、豪放不羁、超然脱俗，也爱清谈饮酒。都说真名士自风流，而风流是有代价的。魏晋时期天下大乱，整个精英阶层的价值观被颠覆：文人清谈庄老玄学源于儒家信仰坍塌的无奈；纵情山水多因庙堂受挫，政治生涯一直郁郁不得志；而饮酒，也不过是他们麻痹自己、忘记现实痛苦的手段而已。


要想真正了解魏晋士族风流的精神气质，就绕不开这本被誉为中国「名士教科书」的《世说新语》，它从德行、言语、政事、文学等三十六个方面，全方位记录了魏晋贵族阶层的言行。作者雪花如糖在深入品读《世说新语》的基础上，选取出三十位具有代表性的风流名士，独辟蹊径，把散落在各章节中某个人物的记载摘录出来，将这些碎片化的文字有机地融为一体，串联成一个个丰富完整的人物故事。


目送归鸿，手挥五弦——似乎从来都来心游物外的嵇康，其实内在根本不是一位消极避世的隐士，他最终用自己的头颅捍卫了信仰。


掷果盈车，深情檀郎——美貌与才华并存、对待爱情忠贞不渝，看似完美的潘安却因攀附权贵、望尘而拜，把自己引向毁灭。 


木犹如此，人何以堪——在战场上的杀伐决断，如雄狮一般野心勃勃的桓温，却常被重情重义所牵绊，一生难登帝王之位。
