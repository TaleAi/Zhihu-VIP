## 简介
重复做同一个梦，预示着什么？梦里飞起来又掉下去，背后有什么心理暗示？为什么大家都会梦见类似的东西？比如掉牙、被追赶、考试……我们一生中 1/3 的时间都在睡眠中度过，但真正了解梦的人少之又少。心理学认为，梦，实际上是我们与潜意识沟通最直接的方式。懂梦的人，能从梦中得到启示。
