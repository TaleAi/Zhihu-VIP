## 简介
请你穿过失眠花园，与我一生热恋。


这是一个失眠可看的现言合集。


这座花园里，每一朵花都拥有自己的神祇。


**时时思慕，见你朝朝**


波斯菊的花语是勇敢而自由。


所以，当私密快递被偷，小偷还以此大做文章时，她不仅没有感到羞耻，反而勇敢面对，一心抓贼。


不过，这个鬼鬼祟祟的「小贼」，为什么会是校霸兼学霸？


好在一切都是误会。


但他俩却误打误撞，开启了甜甜的恋爱。


**她的反向时钟**


向日葵代表着「忠诚、有信念的爱」。


所以，时空流转，时钟倒回，她毅然选择再次爱上他，救他于噩梦般的未来。


为了让他免于被病魔折磨致死的命运，她愿意付出一切。


但年轻的他，这一次是否还会循着时间线，爱上她，离开她？


爱是自由意志的沉沦，也是可以违背任何科学定律的不二法门。
