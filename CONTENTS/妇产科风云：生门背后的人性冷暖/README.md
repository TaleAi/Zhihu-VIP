## 简介
怀孕、生产，是个人和家庭的大事，在经手数千个案例的医生眼里，却多看出一份人性幽微。本专栏的口述者是一名妇产科医生，在一家三甲妇幼保健院工作十多年，她所经历的故事可谓惊奇，有些甚至称得上惊悚。


结婚三年多没有怀孕，去医院检查，「她」竟发现自己是男儿身。


一名产妇执意要生儿子，多次打胎、去香港产检，最终命运向她开了一个大大的玩笑。


一名女孩在生完孩子三天后，竟离奇失踪了，一切似乎是因为她的「高富帅」男友。


高龄产妇难产，死在手术台上，她的婆婆和丈夫竟将立马遗体换了 5000 元。


我曾全力开导一个产后抑郁的产妇，因为我的家人有过一段类似的悲惨经历。


在妇产科中，有感人至深的爱情，也有背叛；有医生、患者共克难关的案例，也有凶险的疑难杂症，乃至死亡；有时女性还要背负着陈腐观念的重担。


本专栏由作者采访妇产科医生真实经历写成，让我们跟着作者的笔触，看到人性。
