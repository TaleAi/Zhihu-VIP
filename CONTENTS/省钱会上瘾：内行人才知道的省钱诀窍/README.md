## 简介
在「消费升级」、「精致生活」的影响下，如何才能存住钱？资深理财师刘笑维老师将从工资、五险一金、个人信用、现金管理、生活储备金、买房、买车、养老、海淘等多个角度，教你为生活开支节流的省钱大法。


你是否遇到这些问题：


虽然工资不低，但还是不够花？想省钱又想花钱，如何做好平衡？买房买车都要贷款，如何才能更加省钱？五险一金每年都交，可还是不清楚退休能拿多少钱？教育成本年年上涨，生活成本充满变数，储备金如何才能事半功倍？活钱放哪里收益更高？白条，花呗，信用卡怎么用才更划算？


挣钱不易，如何省钱便成为普通人理财的第一步。


刘笑维老师将从理论到实践，毫无保留地分享自己的生活经验，扫盲生活中与金钱相关的各种知识点，从衣、食、住、行全方位助你掌握省钱秘笈。本专栏能够帮助大家建立起合理消费观念，增加收入结余的同时、学会优化收支结构，降低生活中的财务风险。梳理自己与家庭的财务状况，科学制定理财计划，从而达成个人理财目标。
