## 简介
如果不知道职场的底层逻辑，很难突破职场瓶颈；如果不改变思维方式，可能就得不到领导的信任。本专栏教你学会汇报和总结，写出漂亮的策划案，了解职场沟通禁忌，成就自己「资深、靠谱、有能力」的职场终极标签。


在职场中，经常会遇到这几种情况：


不懂倾听，get 不到领导设定的目标，被领导和同事边缘化；


一番好心却因为表达不当，莫名得罪了身边的人；


不会汇报自己的成绩，辛苦一年，升职加薪的却是别人；


不会拒绝同事、朋友的要求，自己费力却不讨好；


......


现实生活中的职场很残酷，太多职场「老实人」人吃了不会沟通、不会汇报的大亏。


职场沟通与写作是进入职场后，跟同事和领导打交道说的每一句话、给同事和客户发出的每一条微信和消息。不同岗位，会面临不同的沟通难题。


本专栏从沟通技巧、职场写作、情绪管理等多个维度，把复杂理论化繁为简，全方位护航职场发展。同时学习处理职场情绪，让自己成为一个老板和同事眼中「最稳定」的存在，助你成为职场的「头脑担当」。
