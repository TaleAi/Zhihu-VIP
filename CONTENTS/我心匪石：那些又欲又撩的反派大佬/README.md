## 简介
反派他危险又撩人，我要怎么逃？


他为什么一天可以换几百句情话？他为什么笑一笑都勾着人？救命啊，反派他又欲又撩，我要怎么办？


**长辞谢山河**


小川总觉得自己那个禁欲脸的师父无情无欲，


可她第一次给他下药，他非但没端一副君子做派，还把她折腾得够呛。


谢长辞总觉得自己那个徒弟又蠢又废，极不讨喜，


可到头来护着她，守着她，半点危险也舍不得让她碰的，也是他。


**谁的白月光**


李自渊跟我在一起，纯粹是因为我清冷的性子像他前任。


我跟李自渊在一起，纯粹是因为他长得像我的白月光。


我们谁也没说破。


直到有一天，


他挽着前任的手臂，我拽着白月光的衣袖，


在大街上打了个照面。
