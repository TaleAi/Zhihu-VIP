## 简介
一百次走马观花，不如一次系统体验。故宫是皇家禁苑，也是中国最大的古代文化艺术博物馆。本专栏从故宫国宝切入，划分为陶瓷、青铜器、玉器、书法、绘画五大板块内容，带你一起理解中国艺术的底色，提升个人审美水平，重新唤醒你对美的感受力。


小小的明成化鸡缸杯为什么拍出 2.8 亿元的天价？


为什么打开《清明上河图》，看到一艘将要撞桥的船？


唐代的搨书人如何临摹王羲之《兰亭序》的精髓？


故宫荟萃了中国最顶级的艺术品。其中有不少稀世奇珍，或是历经劫难流传至今的孤本真迹，或是大师巨擘的铭心绝品，走马观花的游览，难以体会故宫珍藏国宝的魅力。


本专栏以历史时间为线索，以艺术品为经，精选逾百件国之重宝，贯穿自先秦到明清的中国历史，展示了中国之美的发展脉络，呈现了中国历代艺术审美风格的变迁，带给你一部来自故宫的中国艺术史。
