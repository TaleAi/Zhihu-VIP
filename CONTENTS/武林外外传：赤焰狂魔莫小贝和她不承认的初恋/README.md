## 简介
青春期的莫小贝和邱小冬，爬上了同福客栈的屋顶，这个屋顶不得了，哪对男女上去都是要出问题的。但小时候的莫小贝只对糖葫芦西凉河摸鱼感兴趣，长大后，她又成了身负血海深仇的衡山派掌门、全江湖通缉的赤焰女魔头；可邱小冬是老邱家的独苗，七侠镇后生里的翘楚，家庭好模样好，品学兼优性格好，还考取功名成了皇上跟前的红人……
自古红蓝出 CP，青梅女魔头遇上竹马状元郎，七侠镇红鸾又星动，来这里和同福客栈众人重逢~


全七侠镇的人都知道，莫小贝和邱小冬不是一路人。


一个是摸鱼逃学打先生的同福客栈小姑子，一个是品学兼优性格好的邱员外家俏后生，


但偏偏，这两位又是白马书院著名青梅竹马的红蓝 CP，


邱小冬的零用钱，只能给莫小贝买糖葫芦；莫小贝的赤焰掌，是邱小冬这个小鸡仔的冬暖炉。


有道是，郎骑竹马来，绕床弄青梅，同居七侠镇，两小无嫌猜；可谁知，灭门之仇不得不报，莫小贝南下，血洗江湖；光耀门楣重担在肩，邱小冬高中，初涉官场。


两人情路坎坷，女魔头和状元郎何时能成眷属，老白说，咱投票决定吧……


《武林外传》完结十年了，同福客栈打烊也十年了，但在这儿，老白真真儿娶到了佟掌柜，生了个大白胖小子，吕秀才带着芙妹去了京城，李大嘴厨艺倒是百年如一日……


只是众人都猜不到，佟掌柜给小姑子准备的嫁妆，啥时候能脱手，又会便宜了哪家公子哥？是竹马状元郎还是风流倜傥唐门少主？欲知后事，您专栏里瞧好！
