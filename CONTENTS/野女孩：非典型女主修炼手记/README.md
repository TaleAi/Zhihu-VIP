这是一个光芒万丈，一心搞事业的太后，和她的 N 个男人不得不说的故事。大司马想做她裙下宾；国相大人念她旧时情；一向心狠手辣的孟尚书也不知为何对她频频示好；连她一手扶上皇位的小皇帝都问她：母后，什么时候让朕也尝一尝，「销魂」的滋味。
