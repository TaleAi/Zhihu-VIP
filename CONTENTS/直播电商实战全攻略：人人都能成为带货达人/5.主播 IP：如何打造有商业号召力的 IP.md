## 5.主播 IP：如何打造有商业号召力的 IP
在直播行业中，前赴后继者众多，脱颖而出成为头部主播的毕竟是极少数。少部分主播经历过大浪淘沙后沉淀下来，成了能够支撑平台发展的腰部主播，而大部分主播不是被淘汰，就是在苦苦挣扎，以求生存。


在直播行业中，颜值和才艺毫无疑问是加分项，但一位优秀的带货主播不能仅仅依靠这些，还要具备一些关键能 力，如产品感知力、市场把控力、互动能力、控场能力和文案能力等。主播只有把这些能力综合起来，才能真 正具备号召力。 


遴选人才：行业对主播个人气质、素养和能力的基本要求 


下面通过一个案例说明直播行业对主播个人气质、素养和能力等方面的基本要求。 


在直播行业里，祖艾妈是一位非常特别的主播。2016 年，祖艾妈开始在淘宝上做直播。那时候，淘宝直播平台的功能还不完善，她没有办法实时分享产品链接。因此，她自己对着镜头倒数「5、4、3、2、1」宣布开始秒杀，然后按照粉丝的留言顺序安排发货。直到现在，她的直播间仍然如此。 


很多主播总是尽可能地迎合粉丝，但祖艾妈经常淡定地说:「想买就买，秒杀不退。」她既介绍产品的优点，也说产品的缺点，甚至在胸前挂个大牌子特意强调。在一些用户眼中，她一点都不可爱，脾气不好，说话难听。 


然而，就是这样一位主播，却做到了让粉丝一边抱怨她说话直白，一边一次次回到她的直播间买东西。我们可以看看祖艾妈的战绩：单场直播仅介绍 4 款产品，成交额却能达到几十万元；淘宝店月销售额高达几百万元；每场直播都有数万乃至数十万名粉丝进入直播间等着她开播。 


祖艾妈究竟有什么魅力，以至于她的粉丝如此死心塌地？我们不得不说她的气质。首先，祖艾妈的个人形象不错。第一眼看到她，你会觉得她很亲切。 她长着一双大眼睛，脸上总是带着笑意；说话的声音非常洪亮，带一点儿沙哑；看上去像邻家小女孩，很容易让人产生想要亲近她的感觉。高兴的时候，她会随着音乐节奏情不自禁地扭动身体；她跟工作人员打成一片，她称幕后的运营人员为「后台小哥哥」；她每场直播的造型都不一样，总是和当场直播所带货品相得益彰。 


祖艾妈的感染力非常强。有几次，她尚未详细了解产品底价就上货，并在直播过程中无奈地说:「不想知道底价，知道了也心痛！」她的这种表现就像是邻家女孩买东西的时候不问价，买完之后又后悔一样，这让很多粉丝都觉得她非常真实、不做作。当粉丝留言说已经收到货品并给出好评时，她会瞪大眼睛， 笑着问:「是不是很好？真的很好哦！」当粉丝催促她快点介绍产品时，她偶尔也会「抱怨」几句：「不要催了，冬天换衣服慢嘛！」


在直播行业中有一个非常有趣的现象，那就是颜值高的主播不一定受欢迎， 颜值不怎么高的主播也不一定就不受欢迎。直播行业对主播气质的基本要求是：让人感觉很舒服，想要情不自禁地支持主播。祖艾妈就是一个很好的例子，她的颜值并不出众，但气质不错——亲和力强、真实、不做作。 


主播必须具备的一个素养是尊重粉丝。粉丝是需要经营的，没有主播能够在缺乏粉丝支持的情况下获得长远的发展。同其他主播一样，祖艾妈经常在直播间里与粉丝聊天，从生活琐事到化妆，从孕期感受到育儿，一切可能引起粉丝共鸣的话题都可以成为她与粉丝聊天的主题。


在直播的过程中，对于粉丝提出的特殊要求，她会现场做出回复。只要发现问题，她就会直接告诉粉丝：「把货退回来，运费我承担，无条件退换。」直播结束后，时间再晚，她也会认认真真地逐一翻看粉丝的留言，并认真地回复。她尽可能多地记住粉丝的名字，并在直播过程中准确地叫出她们的名字。对每场直播有近十万观众的主播来说， 这并不是一件容易的事，但祖艾妈坚持了下来。 


主播要想高效地带货，光靠气质还不够，专业能力也要跟上。对于服装， 祖艾妈一向有自己的判断。当所有人都在做童装的时候，她却瞄准了亲子装。 她认为，童装虽然单价低、容易成交，但很难带来增量。她的店铺火了以后， 越来越多的商家开始走同样的路线，竞争变得激烈起来，她开始尝试自己做设计。


她到世界各地边看边买，了解流行趋势，然后自己做设计，推出独家定制款。随着用户越来越多，她逐渐扩展了经营品类，开始尝试卖女服、配饰等。 她总能敏锐地察觉时尚趋势，并迅速地做出反应。当粉丝反馈产品不够时尚时， 她会立即对新一批的产品进行改良。 


总之，主播不能仅仅依靠颜值和制造娱乐氛围来迎合消费者，而要依靠自身强大的业务能力，了解消费者真正想要的是什么。要想具备这些业务能力， 主播就要不断地学习专业知识，在开播之前认真做准备工作，并站在消费者的角度考虑问题。 


主播人设：主播也需要塑造独特而有吸引力的人设 


提起人设，人们总是会联想到很多演艺人士，不过人设的应用范围早就突破了演艺行业，例如，人设的相关概念在自媒体行业也十分火热。 


为什么带货主播也需要塑造人设呢？因为人设可以产生强大的吸引力，让粉丝感到他们面对的是一个有血有肉的人，而不是一个没有情感的销售机器。 在人设的影响下，主播与粉丝长时间的互动可以让粉丝对主播产生很强的信任与依赖。 


对主播来说，要想让粉丝很快地认识并记住自己，最有效的手段之一就是给自己贴上某种标签。人设实际上就是各种标签的组合。主播需要找到一些自身拥有的、具备一定的传播力且与自身定位匹配的标签。 


很多时候，这些标签是通过有趣的、能够抓住用户心理的内容而形成的。 如果主播持续输出内容，如短视频，就要在每一条内容中不断地重复这些标签，否则用户无法对主播产生一个稳定的印象，主播的人设也就无法建立。另外， 人设的独特性也是非常重要的。如果主播的人设没有亮点，用户就很难记住主播，这样的人设几乎没有价值。 


现在，在不同的直播平台上，各类主播层出不穷。虽然具备独特人设并不能保证主播获得成功，但没有任何人设的主播很难获得成功。我们不妨看看各个平台上的头部主播，他们中的很多人都是通过建立独特而有吸引力的人设吸引大量粉丝的。 


人设之所以具备强大的力量，主要是因为粉丝会把自己的情感寄托在主播所塑造的人设上，不断地将其美化，在心中构建一个符合自己预期的形象。建立人设的第一步是做好自身定位，明确目标用户的需求，找到一个或几个人格化的关键词，如可爱、理性、直爽等。主播要围绕这些关键词，在与粉丝互动的过程中逐步建立相应的人设。 


在自媒体行业中，很多过去只输出内容的账号已经高度拟人化，如微博上的「博物杂志」。该账号原本只有 2 万名粉丝，但依靠「反差萌」的人设，其粉丝人数已经突破了 1000 万。带货主播可以学习这些自媒体从业者的经验，更有效地打造独特的人设。 


对主播来说，塑造人设必须建立在明确自身定位的基础上。既然人设是主播的一个象征或缩影，那么它必须具有一定的代表性，要能让粉丝清晰地感知主播想要传达什么、主要服务于哪个群体。 


不过，这些都不是凭空想象出来的。主播必须经过深入的调查和分析，才能确立合理的定位。主播需要思考下列问题：自己想要走哪种风格或者路线？该风格或者路线能否使自己在竞争中脱颖而出？该风格或者路线是否符合目标 用户的需求？


例如，某主播主要销售母婴产品，其直播风格最好亲切、温柔一些，采用炫酷风格就很不适合。炫酷风格虽然可以让这位主播具备很强的差异性，但它与目标用户的需求不匹配，如果强行塑造这种人设，只会把粉丝推得更远。 


IP 化：IP 的重要作用与主播开发 IP 的四个要点 


从本质上来说，IP 营销就是通过对内容的创作和利用推广特定的产品。IP 营销的应用范围十分广泛，载体也十分丰富，包括动漫、文学和影视作品等。 IP 可以在多个方面发挥重要作用，因此很多主播都在积极地打造 IP，尝试开展 IP 营销。 


对主播来说，开发 IP 并不是一件容易的事情，很多细节都会影响最终的应用效果。下面介绍 IP 开发的四个要点，以帮助那些经验不足的主播少走一些弯路、避开一些陷阱。 


1、设计独特


主播 IP 的设计是一个关键，因为这会直接决定广大消费者对主播的第一印象。也就是说，主播能否以最快的速度抓住消费者的心，IP 的设计发挥着关键作用。例如，李佳琦的 IP 设计就十分成功，也许有很多人并不是李佳琦的忠实粉丝，但如果你跟他们提起李佳琦，他们的脑海中也会浮现出那个有着生动表情、高亢嗓音的主播形象。主播 IP 的设计成功与否首先体现在记忆点是否突出这一点上，毕竟观众只有先对主播产生深刻的印象，才有可能持续观看直播乃至下单。 


主播 IP 的设计要有一定的原创性，要尽量避免与其他主播 IP 过于相似， 否则就没有辨识度。经验不足的新主播不要急于求成，要多学习、参考头部主播所塑造的 IP，然后结合自身特点进行设计。 


2、巧妙融合 


设计出一个独特的 IP 只是第一步，主播还要花费很多时间和精力将这个 IP 巧妙地融入不同的业务场景。 


一位主播也许可以凭借独特、亮眼的形象引起用户的注意，或许还会引发小规模的自发讨论，但这些并不足以支撑其 IP，如果其他主播也采用相似的 IP 设计，这位主播很快就会因为竞争对手的不断出现而失去热度。 


为了使精心设计的 IP 能够持续地生存发展并获得较高的关注度，主播要尽可能提高 IP 的曝光度。直播间是主播的主要战场，因此将 IP 融入直播过程是必须做好的工作。另外，很多主播都会在各类社交媒体上注册账号，有些甚至还是热门的短视频博主，这些渠道和内容形式也是需要充分利用的。 


3、创意运营 


曝光度可以给主播 IP 带来更高的热度，但主播 IP 的日常运营工作必须做好，不能仅仅依靠大规模的商业宣传。大规模的商业宣传虽然覆盖面广、力度大，但投入非常高，而且有时候不太精准。而日常运营工作从某个角度来说相当于细水长流式的宣传推广，具有持续时间长、投入较低、精准度高等优点。 


主播 IP 的运营没有固定的套路，每一位主播的运营思路和手段都不一样。 但是，在运营的过程中都需要注意以下几点。 


(1)适当地借助时事热点。例如，在重要节假日期间或者发生重大事件时，运营团队可以将热点内容与主播 IP 结合起来，做一些有针对性的宣传。不过， 一定要注意内容与主播 IP 的匹配性，不能生硬地将两者捏合起来。 


(2)持续创新。引领潮流与紧追潮流是不同的，领导者和跟随者也是不同的。运营团队要积极尝试各种新的运营手段，使主播 IP 勇于创新的形象更加深入人心。 


4、注重长远发展 


一个优质的 IP 可以源源不断地为主播提供发展动力，因此直播行业才会如此重视对主播 IP 的打造和运营。不过，对主播 IP 的商业化开发也要适度，否则很容易过犹不及。例如，人们都知道所有广告的目的都是推销，但他们更喜欢看那些创意十足的广告，而不是一味推销产品、商业味道过重的广告。主播可以利用 IP 获得一定的回报，但从长远发展的角度来考虑的话，主播不能透支 IP 的生命力，而要细心地呵护它。
