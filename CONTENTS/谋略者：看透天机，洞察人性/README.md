## 简介
11 位历史上重要的谋略者，最强的智慧担当，在他们手中，成就了一朝一代的辉煌。读懂他们的人生智慧，看透人性的弱点，提升自己多元思考问题的能力。


不同时代的谋略智者，一次次令人拍案叫绝的计谋，充满着「跪感」的运筹帷幄：


周代姜子牙：清水直钩，愿者自来。看似佛系的操作，上钩了一只大鱼「姬昌」.....


战国商鞅：只用 3 步有明有暗的谋划，奠定了秦国灭掉东方六国的坚实基础。


三国诸葛亮：史上谋略者们的 C 位，一段《隆中对》指给了刘备 3 条明路，两人一拍即合，巅峰一时。


十六国王猛：辅佐苻坚，整合人心，使前秦帝国成为当时北中国第一强国，拥有了统一全中国的力量。


唐朝李勣：一句话刺激了李治的神经，助力武则天成帝。


明代刘伯温：辅佐朱元璋平天下，论安危，一句「三分天下诸葛亮，一统江山刘伯温」证明其地位。


......


谋略者所走的一步又一步，似有天助，实际却是人谋，将时局走向控制在意料之中。


而这背后，隐藏的是对人性的熟悉，直击了人性的弱点。


以史为鉴，或许能启迪当下，洞见做人谋事的智慧与灵感。
