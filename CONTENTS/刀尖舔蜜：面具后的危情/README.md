## 简介
爱情是一枚毒糖果，你与我却都奋不顾身。致人于死地的，有时不仅仅只有利器


一个变心的男人能有多狠？新欢被杀害，前夫认定我是凶手。他将我押上手术台，把我的脸换成了新欢的……  

——《前夫换了我的脸》


一次开心的泰国游，竟成了葬送我的地狱之路！男友化身恶魔，将我和女儿敲骨吸髓，甚至差一点，我被改造成了海豚人玩具……


——《在魔窟》


十二岁那年，我杀了自己的父亲，这个罪名，我结结实实地背了七年，而真正的凶手，却花着我父亲的钱逍遥快活。七年后，我拉满了复仇之弓……


——《我是杀人犯》
