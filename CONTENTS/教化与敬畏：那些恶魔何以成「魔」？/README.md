人性的沉沦和堕落，往往只在一念之间。


人，首先有动物性，同时也是社会的产物。


然而有些人，在成长过程中缺乏来自社会和家庭的教化，失去对生命的敬畏之心，一步步堕入犯罪的深渊。


挥起屠刀之前，他们都是普通人，一念之差，让他们变成杀人不眨眼的恶魔：


* 恶魔夫妇残害四十余人，竟然只为了几百块钱？

他们正是因为失去了教化和敬畏，才让恶念成了一根无法扑灭的导火索。
