## 简介
连环杀手，人类社会中的捕食者。


1 Wolfgang Schmidt 粉红巨人


活跃年代：1989-1991


他想变成女人，于是开始杀女人。异装癖杀人狂，德国最知名的连环杀手。


  



2 Egidius Schiffer 亚琛扼杀者


活跃年代：1983-1990


女孩在午夜时分离开迪厅，搭乘陌生人的顺风车回家。从此再也没有人见过她。一场长达 23 年的追捕拉开帷幕。


  



3 Arthur Gatter 法兰克福锤杀狂


活跃年代：1990


太阳升起，尖叫爆发——每隔几天，脑浆迸裂的尸体就会在黎明的大街上被发现。法兰克福陷入恐慌之中……


  



4 Martin Ney 黑面人


活跃年代：1992-2004


深夜，一名儿童离奇失踪。唯一的线索只有室友的证词：一个黑面人把他带走了……


德国历史上最残忍狡诈的恋童癖连环杀手。三个国家排队引渡的变态。


  



5 Volker Eckert 死亡卡车


活跃年代：1974-2006


2001 年，法国的尔米尼亚克，一个妓女被勒死、抛尸。同年十月，一名妓女在西班牙赫罗纳失踪。几个月后，又一名妓女在法国特洛瓦暴尸荒野。同样的案子也出现在意大利、德国、捷克……终于，有人意识到，这些案子可能是一个人干的……


  



6 Joachim Kroll 杜伊斯堡食人魔


活跃年代：1955-1976


1959 年，有人发现一具女尸。她死于奸杀，大腿上的肉被割掉。警方立刻意识到，这是一起食人杀人案。1976 年，警察终于找到凶手的时候，他正在做饭。炉灶上的锅里漂着一只人手……
