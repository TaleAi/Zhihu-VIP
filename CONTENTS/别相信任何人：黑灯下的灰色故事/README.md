## 简介
无限反转的灰色故事，看尽人性。


《弃子》


抗日战争时期，日本军部研发了一种催眠剂，注射进人的体内，就可以逼人招供。


国民党特工王飞不幸被捕，日军对其用药，但药物没有生效，导致王飞变成了傻子。在医院工作的女共产党党员刘萍，出于正义感，便开始照顾王飞，直到全国解放，刘萍通过截获的国民党信笺，才得知真相，原来，王飞只是一个被精心挑选的牺牲品……


《秘密》


农民周山右眼残疾，和乡下寡妇陈小红结了婚，婚后，陈小红出轨了，窝囊的周山去父亲的坟前哭诉，却发现墓室损坏，周山便把父亲的骨灰坛抱回了家，埋进了院子里……


《乞丐公子》


王员外的儿子失手杀了人，管家出主意，找了个与公子外貌相似的乞丐顶替公子，让公子化成乞丐，出门跑路，王员外因忧子心切，茶饭不思，重病身亡……王员外死后，化身为乞丐的公子希望跟乞丐换回身份，但，没那么容易了。
