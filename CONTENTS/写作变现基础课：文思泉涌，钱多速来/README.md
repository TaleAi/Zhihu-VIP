本课程由晓晓写作训练营创始人主讲，教你如何从零开始学写作，课程从理论和实践两大模块出发，解决了新手作者不知道写什么、没有写作思路、文章没有说服力、从标题、开头、提纲到结尾不知道怎么写等难题。共计总结了 65 项实用技巧，为你揭晓新手作者快速进阶的秘诀。

晓晓老师全职写作近 3 年，放弃过万月薪，靠写作月入 30000+，是多家知名平台签约作者，带领学员拿到了 4 位数稿费。
