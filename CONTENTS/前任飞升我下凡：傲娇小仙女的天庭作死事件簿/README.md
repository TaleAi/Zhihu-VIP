## 简介
本专栏集合了天界所有不作不会死的小仙女的故事。她们闯下大祸，被贬凡尘，天庭却处处是她们的传说。


他站在九天之上，冷眼望着坠下诛仙台的我，面无表情。


对，他是我师父，亲手把我推了下去，动作又快又狠厉，仿佛百年来的师徒之情就是一场空。


那一掌拍在心口，还挺疼。


我醒来后遇到的第一个男人是裴季凌。事实证明，这位，依然是个狗男人。


这天庭，实在没有我可留恋的东西，但落入凡尘前，我得先把仇报了。


----


我的前男友，是地狱阎王。


准确来说是阎王一职的继承人，小阎王伏宁。


上一世他负了我，这一世我得躲着他。三界之中通往幽冥界有很多条道路，我这次下来选择从人间进。


过了鬼门关就是黄泉路，黄泉路继续往前走就是恶狗岭、金鸡山和野鬼村，之后才算走进狗男人的阴曹地府。我心里默念着路线……


怎么好死不死，又碰到了他！


摆开架子跟伏宁再磕一世吧，反正一万年也没多长。
