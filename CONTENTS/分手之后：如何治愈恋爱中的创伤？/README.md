# 分手之后：如何治愈恋爱中的创伤？

### 状态：已完结，共 15 节

### 存档日期：2023-04-17

## 简介
分手后，我不再恨自己。


亲爱的小伙伴，我知道你分手了。


我还知道，你可能正处于茶饭不思、情绪崩溃、满脑子都是想挽回的状态，那么我告诉你，**你现在的状态很危险**，我必须要先给你按下一个暂停键。


⚠️ 不要顺着情绪本能去胡搅蛮缠，用卖惨、打扰、纠缠等行为强行让对方关注自己；


⚠️ 也不要病急乱投医，找一些市面上打着挽回旗号的情感机构，花大价钱「买套路」解决，大部分挽回机构中的老师没有任何专业资质，甚至有些是专业的销售人员出身，对他们来说，最重要的工作就是摸清你每一个阶段的情绪触发点，用各种话术一步一步留住你为自己的冲动一次又一次地付费。


**每一次分手，无论对方回不回来，你都应该以建立新的健康平等的长期关系为最终目的的。**


这段新的健康平等的关系，可以是跟那个你放不下的 ta，也可以跟一个新的爱人。


在这个专栏中，我不仅会和你分享分手后，治愈自己的实操步骤，还会带你**深入到「自我认识」「自我成长」的层面**，并以此为立足点，**学习长期关系的底层逻辑**，用一个全新的更好的自己去爱一个人，让一段关系获得新生。


我知道，和所爱的人分手可以说是一场酷刑，你可能正带着焦虑和抑郁在阅读这个专栏，觉得自己的生活已经支离破碎了，不知道怎么去重新捡起乱七八糟的生活。希望这个专栏可以陪着你走过这一段最黑暗的时光。


我希望通过这个专栏的学习可以让你：


1. 判断关系是否值得修复，对不值得的关系及时止损；
2. 掌握重建自我的正确方向和实操步骤，不走弯路；
3. 明白如何科学地爱一个人，让未来的关系健康可持续发展；
4. 处理好自己和自己的关系，随时有勇气重新出发。

为了共同达成这些目标，我也需要你的积极配合并且为此付出真正的努力，那就是：在学习本专栏分享的原则和策略的基础上，根据自己的实际情况进行独立思考，并且把思考结果真正付诸行动。


如果你准备好了，我们就开始吧。
