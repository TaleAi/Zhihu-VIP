## 简介
乔布斯说过很经典的一段话：很多人都以为设计就是外观，是设计产品看起来是什么样子的，其实，这些只是一部分。设计是 How it works，也就是，产品是如何运转的。


小到一个水杯，大到一个建筑；有形的如产品，无形的如 IOS 系统，一支球队，一个政权—— 都能找到设计思维的影子。


很多人希望提升自己的设计力时，不由自主的选择了「60 小时 PS 速成」或「128 分钟学会 UI」这样的工具课，结果事倍功半。


工具易学，思维难建。是时候让大家从设计的底层逻辑开始，重新学习和认识设计思维了。


在本套私家课中，拥有创始人、产品经理、设计师等多重背景的老司机马力，将从他十几年工作中总结出最能颠覆你观念的设计思维和理念方法。手把手教你如何赢在洞察，让你的思考力、判断力和决策力胜人一筹，给你一套「大设计思维力」。


**适合人群**


* 不甘平庸的产品、平面、交互和 UI 设计师
* 渴望精进的互联网产品经理和运营人员
* 希望把握产品调性的决策人
* 想转行做互联网设计的人
* 对 APP 上瘾的设计小白
* 想提升思维力的泛设计人群
* 对设计感兴趣的普通人

**圈内人推荐**


*设计不只是需要好看易用的界面，这只是一个产品应该具备的基本素质。除了基本的交互和视觉原则是否有方法论或可依赖的路径设计？我们应该从哪几个角度来审视自己所设计的产品？通过《设计必修课》你会得到自己的答案。*


*—— 罗子雄，设计师，前锤子科技 UI 设计总监*
