# 人类学家在田野

## 简介
如果说被误会最深的学科，人类学应该能排进前三。人类学家是不是都在某个偏僻的角落里和食人族斗智斗勇？观看原始部落奇怪的巫术仪式？人类学家的小本本里是不是记满了奇闻异事？在当代社会，我们还需要人类学家吗？

专栏通过 8 个人类学家的田野调查故事，深入浅出地介绍人类学研究的基本概念，在现代日常生活中的应用，消除对人类学家身份的误解。教你如何站在生活的另一面，观察我们熟悉的世界，感受文化的冲撞，学习如何用人类学视角看问题，体会到：当我们凝视另一个文化时，对方也在凝视我们。


田野调查的范围，从学校到写字楼，从医院到茶馆，甚至在研究对象的厨房里都能看到人类学家的身影。人类学家的研究课题和地点，如今已经渗透到了生活的方方面面，从认知进化到移民文化，从旅游发展到社交媒体，人类学家经常「一言不合」跨学科开辟新研究方向。 


我希望通过这个专栏中 8 个人类学家的田野调查故事，让你了解人类学研究的基本概念。 


故事中你能看到，有人被研究对象叫去「杀龙」； 


有人花了 15 年时间研究伦敦的性工作者； 


有人打破关系伦理，和自己伴侣的儿子结婚； 


有人在纽约调查亚裔为什么会和臭味联系起来； 


有人在中国多年研究新富有阶级的「应酬文化」； 


有人搬入纽约「上东区」，以「局内人」的角度研究美国精英阶层； 


也有人在观察过程中遇到被家暴的研究对象、斗殴致死的现场，被迫面临道德困境。 


从这些故事里，我们能揭下人类学（家）看似高冷的面纱，发现这个学科离我们的日常生活并不遥远。看他们如何通过撰写「民族志」反思，利用他者文化理解自我；如何在学术研究过程中，控制私人情感的剥离程度；如何在拥有多重身份的同时，进行向上研究。同时让你知道，人类学家也可以是不严肃、不正经、不客观的「逗逼」职业。
