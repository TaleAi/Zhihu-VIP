## 简介
现在开始，热恋吧！


**《第二次亲密接触》**


聂涛万万没想到，有一天相亲这事会落到自己的头上，他接过介绍人的照片，认出了她。


是十年前突然甩了自己跟富二代结婚的她，是消失了十年摇身一变成了公司对面小吃店老板娘的她。


往事涌上心头，聂涛不甘心，他要趁这次相亲，狠狠地羞辱她一番。


**《男友是长官：这片蔚蓝的天空有你》**


程真与杜斐的第一次见面，是在他们空军大队医院的泌尿男科。杜斐是医生，但程真，真的只是陪自己表弟来看病的。


程真与杜斐的第二次见面，在空军大队的体检处，杜斐是体检医生，不过她只关心程真一名飞行员的身体素质而已。


程真与杜斐的第三次见面，是在飞行大队的真人秀彩排后台，这次，杜斐再也不会让程真从自己手上溜走了。


**《现在开始我爱你》**


沈思颜搬进新出租屋的第二天早晨，就被邻居敲了门。


她打开门，只见门口站着前男友，著名校草、游戏大魔王陆峥。沈思颜厚着脸皮加回了陆峥微信，往事涌上心头。他们曾是大学校园最养眼的一对，当年只怪陆峥太不争气，沉迷游戏。


他们分别三年，沈思颜想要重新认识陆峥，她决定好好跟他，开一局恋爱游戏。


**《替身能有什么坏心眼呢？》**


我跟粱玺宸签了份协议，做他的「协议女友」，代替他心中的白月光姚芷烟。


不过姚女士最近在国外恋爱谈得不爽，要回国，养情伤。


作为替身的我，自动自觉退出游戏，搬离粱玺宸的家。


隔日我与姚女士相遇，她告诉我，从高中时代开始，粱玺宸喜欢的，就只有我一个。
