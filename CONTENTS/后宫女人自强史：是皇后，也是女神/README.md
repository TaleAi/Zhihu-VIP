## 简介
本课程通过讲述几位传奇后宫女性的传奇故事，向观众展示了古代女性超越时代的独立自主精神。


什么样的女人能被称为直男女神？


人类都是视觉动物，百分之八十的一见钟情都始于颜值，但爱情并不是靠颜值就能守住忠诚。像是某些社交 APP 上的网红，发个精修 8 度的自拍，收获一大堆好评，评论一片夸赞，但那些大都比较随意，顶多算是屌丝女神吧，能赢得男性尊重的女性，才算得上真正的直男女神。


从中国古代看来，大多数空有美貌的女子，那下场与评价都不是很好，而能流芳千古、受世人赞叹的女子，才说明了中国人真正的喜好。


结合历史看来，那些最后能深受皇帝宠爱，并且登上高位，还能受后世好评的，无一不是长相可人、多才多艺、贤良淑德、在后宫的残酷争斗中，她们有心计有手段却不会主动害人，有见识有才学却也懂得在丈夫面前示弱，给足面子，做得来解语花，又能给新鲜感，这样的女人谁会不爱？一个人能不能讨人喜欢，最终还是要取决于自身的谈吐与魅力，而不是靠长相的。
