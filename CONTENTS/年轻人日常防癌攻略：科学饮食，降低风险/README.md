## 简介
你的健康状况反映了你生活经历的总和，疾病是长期不良生活积累的后果，特别是癌症。大部分人被查出癌症时都已经到了晚期，因为大部分的癌症在早期是没有任何症状的——这份来自北京大学肿瘤医院主治医师的抗癌须知，能让你通过检视自己平时的饮食习惯，剔除会让你致癌的食物，帮你远离癌症。


你的健康状况反映了你生活经历的总和。


而癌症的发生，集中反映了我们的生活经历——我们在哪里出生，有没有接种疫苗，住在哪里，从事何种工作，那里的环境有没有污染，吃了什么，生了病能否得到有效的治疗，以及有没有遗留下病患？


**——别以为癌症离你很远。**


据 2018 年最新癌症报告：全球大约有 1810 万癌症新发病例，960  

万例癌症死亡，而且越来越呈现「年轻化」趋势。发展中国家，因生活水平低、环境与卫生条件差而引发「穷癌」高发，而发达国家则会因营养过度、生活优渥、生活方式不健康而频发「富癌」。


换句话说，一个人患癌症是他所处飞速发展的社会与生活习惯极度不匹配的结果。


也许你很难改变周遭的环境，但至少我们可以回顾反思一下：


——在从健康到癌症之间，存在哪些错误的饮食习惯？


* 隔夜后的蔬菜为什么会致癌？
* 从小吃到大的家乡菜，突然就致癌了？
* 如何科学地吃烧烤而不致癌？
* 吃什么食物防癌抗癌比较好？
* 癌症发生之前，有没有预警信号呢？

**请收下这份来自北京大学肿瘤医院主治医师的抗癌须知。**
