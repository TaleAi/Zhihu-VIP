## 简介
公元 1127 年，金朝南下攻取北宋首都东京，掳走宋徽宗、宋钦宗钦两位皇帝，北宋灭亡。由于这次事件发生在宋钦宗的靖康年间，因此得名「靖康之变」。


「靖康之变」是历史上一场国破家亡、天崩地裂的大悲剧，北宋这个繁华而富足的「世界中心」在一年之内被女真人轻而易举地毁灭。在宋朝 320 年的历史中，这段历史无疑最富有戏剧性：耻辱性的劫难、被洗劫一空的都城、妇女悲惨的遭遇、两位皇帝被掳走后的生活……在这个专栏中，知乎历史话题高赞答主、最会说历史的「刘老湿」带你重回靖康年间，梳理靖康事件的发生过程，为你讲述历史书不曾展现的事件细节:


事件发生的两年之前，听闻金兵南下的宋徽宗一时被吓破了胆，竟然突然决定传位太子，转身逃离东京。其长子赵桓仓促继位，从而吹响了大宋灭亡的号角。  




靖康之难爆发前夕，眼见女真人就要攻破开封城了，这时宋钦宗竟找来一位「郭大仙人」，并让这位仙人临时在城中组建了一只「六甲神兵」，随即大开城门。开封城就这样，以一种荒谬到令人发指的方式被攻破了……  




被俘后，一路北上的两位皇帝最终被带到金太宗面前。他们被要求脱掉汉族的袍服，换上女真贵族的服装。其余被俘的宋朝皇室们都被要求袒露上身，把羊皮衣服退到腰际，手里拿着毡条。随后,宋徽宗宋钦宗被引入殿内，按照金国的礼节，行表示臣服的牵羊礼。宋室女眷更是受到了惨绝人寰的凌辱……  




让我们跟着刘老湿，回到大宋 1127，见证那场悲剧的始末。
