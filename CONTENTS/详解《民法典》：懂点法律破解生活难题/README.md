## 简介
学习《民法典》，拥有法律思维，学会运用法律武器保护自身权益。


生活中，你是否被以下这些问题困扰过：


婚姻方面，出轨方净身出户，「忠诚协议」有法律效力吗？


个人方面，电话、资料、照片等个人隐私资料，被泄露了怎么办？


安全方面，对方言语轻薄、动手动脚，遇到性骚扰怎么办？


子女方面，熊孩子打赏网红上万元，能否要回赏金？


……


与其为未知的问题焦虑，面对当下的问题不知所措，不如直接从法律中找答案。


2021 年颁布的《中华人民共和国民法典》，被誉为「社会生活的百科全书」，你生活中遇到的种种法律难题，其实《民法典》早已给出解决方案。


但是《民法典》厚厚一本，上千条律例，普通人难读懂也没必要全读懂，掌握其中的法律常识，就已足够应付人生中大部分难题。


考虑到非法律专业人士的学习需求，我们特邀《民法典实用一本通》主编，带你巧学《民法典》，加强法律知识储备，帮助你拥有法律思维，学会运用法律武器保护自身权益，懂法守法，不吃哑巴亏。
