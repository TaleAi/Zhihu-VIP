## 简介
如何提升认知、逻辑、思考能力？


认知水平决定了你的时间分配；


认知水平决定了你面对选择时的取舍；


认知水平决定了你的社会关系布局；


你的认知能力将决定你一生的格局；


在这个「万物互联」的智能时代，


你原有的成功经验，不再有效，甚至是一种错误路径；


你原来仰仗的优势，不再重要，甚至是一种负资产；


你的真正对手不是别人，是这个迅速变化的时代；


你惯有逻辑思维方式，在未来都会被瞬间颠覆；


唯有不断刷新认知，自我迭代，才不会被淘汰。


**阅读本专栏你将获得：**


1. 掌握思维技能，加强你的思维逻辑能力；


2. 结合大量案例，让你认清思维本质；


3. 练习三个思维模型，一个下午就让大脑升级；


4. 多个沟通模型，有效训练你的表达能力；


5. 建立真正有效的知识体系。
