## 简介
你是盔甲也是软肋，愿你如期而至。


《梦到男上司》


我在公司午睡，又梦到了男神。


他看着我，欲言又止。


我给他展示了一下我的新睡衣。


男神的俊脸上先是冒出一个问号，随后又缓缓变成了一个感叹号！


突然间，我被人拍了拍肩膀弄醒。


睁开眼看见男神蹙着眉，耳根泛红，「能不能不要在工作时间想这种事？」
