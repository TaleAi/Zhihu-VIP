## 简介
有人为爱痴狂，付出一片真心，有人步步算计，把「爱情」做成一场局。有人把爱奉上神坛，有人只相信利益与欲望。在感情这场游戏里，究竟谁赢谁输？


闺蜜遭家暴离婚后住在我家，本来经常加班的丈夫突然开始准时回家，我怀疑他和关茜有一腿。闺蜜打离婚官司后，搬离我家，我过了 3 个月平静日子，突然在丈夫衬衫上发现根长发……


邵阳工作后，他的大学室友正和女友住在一起，热情邀请他来住合租屋次卧。另一间次卧住着陌生人辛雪菱，相处中，邵阳发现辛雪菱表面冷漠，内心温柔细腻，逐渐喜欢上她，事情却并没有那么简单……


李雅发现丈夫身上有陌生香水味，她知道自己仅靠美貌才得以上位，警惕之下对丈夫频频试探。偷偷跟踪丈夫后，李雅发现丈夫和前妻王梅在婆婆家见面，她顿时有了危机感，心生一计……


故事里的痴男怨女，是否有一丝让你想起自己？
