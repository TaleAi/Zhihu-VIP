## 简介
知乎优质脑洞悬疑作者施眠药，再次建立了一个充斥着诡异、超自然的怪诞世界。你将跟随主人公许泽，破译一个个超乎想象的离奇事件，找到异物的「躯干」「面部」「四肢」「人头」……当身体逐个归位，你将拼凑出故事的全貌。


在城市的钢筋水泥间，存在着一群被异化的人类，被称为「异物」。他们形似肉球，周身裹着恶臭的黏液，在暗夜中攒动。这些异物原本都是再正常不过的人类，却被他人和现实的「恶」裹挟，在痛苦和挣扎中逐渐异化，从被害者变成了寸草不生的施暴者。


在被异化的人类中，存在这样一个传说，异物们通过某种神秘的仪式重组肉身，再吞噬掉叫做许泽的男人，这些被异化的怪物便可以「成仙」，进入世界的另一个维度。


于是围绕「我」：许泽，一连串神秘的死亡事件发生了。


 


故事从深夜 QQ 上的呼救开始，许泽收到前女友的视频邀请。一并发来的还有一个文件，这是一段视频监控录像的动图，画面中一个男人一次又一次地把头向后扭曲 180 度。


跟随着这个诡异的线索，一具卡在墙壁中的女性尸体在调查中出现，随之引发了一连串耸动的连锁反应。


除了女尸，许泽身边名叫白斐的男人，似乎比这离奇出现的尸体更加神秘。许泽与他相识三年，却对这个面无表情的男人一无所知，只知道他隐藏着一个重大的秘密……


 故事将如何展开？一切线索将通往何处？世界的终极真理是否就在故事的尽头？
