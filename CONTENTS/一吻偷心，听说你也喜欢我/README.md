## 简介
无论重来多少次，我都会爱上你。


《他是工作狂》


前男友对我很敷衍，敷衍到只会用钱打发我。


「又生气了？」


「你的包到国内了。」


「快递到了，记得让人去搬。」


七夕节，我原本订了一家氛围感超好的酒店。


躺在床上，连姿势都摆好了。


结果收到一条短信：


「公司临时有会，宝，你先拆快递。」


淦！


这日子还过不过了！


《全瑕前男友被洗干净后送货上门》


我分手了。


我抱着前男友的猫，在抖音小号发视频。


「他妈的，傻猫，炸毛，5 元出，微瑕。」


下一秒，我看到前男友的妈妈评论我。


「出刚分手的儿子，全瑕，洗干净后倒贴送上门。」
