## 简介
不爱社交，不擅长社交似乎成为了宅男宅女身上一块大大的标签。我们不是天生不爱与人打交道，而是我们内心的焦虑与害羞，似乎并不是「努力」就可以解决的。如何摆脱社交困境，成为社交达人呢？


我们生来需要社交，但并不是所有人都擅长社交。


为什么有的人是社交天才，而有的人却总是充满障碍？


我们经常思考社交，思考与他人的关系，却不一定能立刻打开与他人沟通的大门。我们希望更好地社交，但心中有时会蒙上焦虑的阴影。


毕业于昆士兰大学的心理学博士王怡蕊分享了这样一个社交恐惧的案例：



> 2009 年博士期间，我遇到了 Jake。从小到大他的朋友不算少，几乎每个周末都会有聚会，喜欢户外运动，也有交往中的对象。  
> 
>   
> 
> 但是，他因为社交焦虑向我求助，说这已经严重影响到了他的日常生活。


**这样的男生也会恐惧社交？他的社交障碍表现在哪？**



> 他害怕和别人的眼神接触，不喜欢在人前说话，不敢表达自己的真实喜好和需求，特别担心别人觉得他怪异或者不喜欢他。  
> 
>   
> 
> 他尤其害怕自己会在人前出丑、引发别人的关注，比如不小心摔倒或者碰翻东西。  
> 
>   
> 
> Jake 还说，之所以朋友们没有发现他有社交恐惧，是因为他积累了很多掩饰和伪装自己的小技巧：  
> 
>   
> 
> 比如，不直视对方的眼睛，但是眼神从对方脸上扫过再移走；  
> 
>   
> 
> 不敢在人前说话，就多用肢体语言，比如点头、嗯哼地赞同对方；  
> 
>   
> 
> 害怕表达自己的想法引来他人的关注或者遭到反对，就变换着措辞附和对方，或者把对方的观点总结以后用自己的话再说一遍，给别人造成他也发表了观点的错觉；对于自己的喜好，他经常回答「随便」，询问大家的意见，从善如流。


**在朋友的心目中，他是一个性格随和、善于倾听、好相处的 nice guy。  

但只有他自己清楚自己心中的对于社交的焦虑和恐惧感。**


如果你也感受到了同样的压力，那么让我们一起，克服社交上的一些困难。并且做一些能够随手做起来的社交技能练习。通过几轮反复练习，来让自己的社交心态逐渐趋于平稳。只要坚持练习，你就会有明显的收获。


通过这个专栏，你将会理解社交恐惧是可以被克服的。社交能力和任何其他能力一样，是可以通过练习，有针对性地提高的。我们将在这里学习到，人们并不需要在拥有了完美的社交能力以后，才能正常的社交。让我们立刻开始吧！
