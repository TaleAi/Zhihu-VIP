## 简介
马家辉从历史、文学、艺术、匠人、科学、心理六大领域畅论百位大师名家，每周更新三期，用一年的时间还原大师们走过的人生，拆解他们，成就自己。


  



![img](https://pic4.zhimg.com/v2-5b912176f331bab540ecdf2ceec204b2.jpg)

  



这是一个八卦主导的年代


  



看八卦会很容易从道德高地去判断、去评论、去笑别人的不幸


  



往往八卦看久了，心态、境界都会变化而不自知


  



所以越年轻的时候，越要少听八卦、多读传记


  



读传记就是跨越时空和古往今来的人交朋友


  



交友以自大其身


 


  



![img](https://pic3.zhimg.com/v2-b8c88c9747f7b50618426d18044ec514.jpg)

  



马家辉带你结交历史、文学、艺术、匠人、科学、心理六大领域的百位大师名家


  



持续一年每周更新三期


  



直击历史现场，还原标志性人生


  



送给每位听众一部生活启示录


 


  



![img](https://pic1.zhimg.com/v2-6330fcde8f5be1f0d6076f6f78ad4136.jpg)

  



适合人群：


  



正值青春迷茫的你，如果听到了胡适给年轻人的三个中肯建议；


  



太在意别人眼光的你，如果领略到了安迪·沃霍尔的 So what 精神；


  



想转行跨界却犹豫不决的你，如果知道了富兰克林如何在不同职业间自由转换的秘诀；


  



对梦想坚持不下去的你，如果感受到了贝利是怎样一次次受挫又一次次奋起；


  



……


  



 一切都会不一样！


 


那些血肉丰满的故事、高低起伏的瞬间、拍案叫绝的畅快、扼腕叹息的沉思……


  



终将化作解决你自身问题的药引。


 


  



![img](https://pic4.zhimg.com/v2-b8c88c9747f7b50618426d18044ec514.jpg)

  



你将获得：


  



人格的教育：选择自己的终身偶像，让那些你崇拜的人，成为自己的人肉 GPS。


  



边界的拓展：透过别人的想法、故事、生命历程，来开拓视野、打破固化思维。


  



勇气的积累：学习大师们的成功，更重要的是透过挫败有所领悟。
