## 简介
在这个世界上，万物皆为欲而生。求知、求情、求富、求色……人们在各式各样的欲望之中徘徊。
人心歹，狗不吃！人心恶，鬼见愁！百鬼过道，恶者从，善者避！
在那些诡谲，又匪夷所思的事件背后，往往纠缠着至暗的秘密！你，是什么秘密？


**《蓝鲸游戏：结伴去地狱》**


我的狗患了抑郁症，它冲出阳台一跃而下，自杀了。


奇怪的是，这之后，小区里又接连发生了几起宠物狗自杀事件。


由此，我怀疑它们根本就是被人谋杀，可是我又拿不出关键证据。


**《被偷走的人生》**


大雨倾盆而下，车窗上惊现血手印，同行的几人都吓破了胆，纷纷抱怨不该来此地旅行。


可抱怨终归是抱怨，几人来到目的地后，竟发现整个村子的人都消失了。


而他们预定的别墅，也是诡异重重，看门的人，竟是【死尸】。


而后，突然出现的白衣小女孩、单只童鞋、发声的断电音响……


诡异频发，矛盾渐起，伙伴们之间互相猜忌，而别墅主人的秘密，也被层层剥开，显露出来！


**《剧本，杀》**


我对偶像爱的癫狂，因此做了很多出格的事情，但是窥私渐渐令我无法满足。


于是，我租赁了一个男人，来扮演我的偶像，久而久之，我的控制欲和占有欲越发膨胀。


那个租来的偶像，也被我牢牢囚禁。


然而，沉沦之后，我才发现，小丑竟是我自己！我的一切，都在他的剧本里。


**《Bug》**


A 班的那个孩子消失了，不是失踪，是真正的消失，不存在了。


所有人都不记得她，可是只有我知道。


因为，只有我能看见她。而她的故事，也被我一点点挖了出来。


**《问》**


我曾采访过一名年轻的女死刑犯。女生安静内向，但她杀害了寝室中的其他五人，并与尸体共度一夜。被发现时，女生已经精神恍惚。案件的起因是一名室友不堪压力选择自杀，而女生恰恰因为曾和自杀室友关系不错而被诟病。一时间流言四起，说自杀的女生是被她逼死的。案发当晚，女生发现流言竟是由寝室的其他五人编造流传，女生愤怒不堪，到兽药店买了毒鼠强，回到寝室后依次下入室友的水杯。


我印象最深的一句话是，少女说：我不需要以死亡自证清白，我要他们下去，亲自问问小欣，是不是我逼死了她？


**《采生折割》**


采生折割：是职业乞丐中最歹毒凶恶的一种。人为地制造一些残废或者「怪物」，以此为幌子博取世人的同情，借此获得路人施舍大量的钱财。


**《打生桩》**


打生桩，或称活人奠基，是东亚及东南亚民间在建筑前的习俗，属人祭的一种，日本称之为人柱，缅甸称为苗赛是指在建筑工程动工前，把若干人活埋生葬在工地内，其目的是祈祷工程顺利。
