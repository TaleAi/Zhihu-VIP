## 简介
冲出黑暗，打破传统，打碎桎梏，不被定义，她们是向命运开枪的女孩们。


《救赎之路：失踪的妈妈》


失踪人口最后都去了哪里？背后的真相让人触目惊心！知识女性，贫苦女性，在失踪面前人人平等，全都沦为了「资源」。我妈妈的失踪牵扯出一桩陈年谜案……


《车祸之后》


在金钱面前人性能够有多么扭曲？我爸爸车祸去世，赔偿下的钱竟然成了我们的催命符。舅舅，舅妈，外公，外婆……所有人的阴暗面都被钱映照了出来，而我妈被活活逼死。


《我的娘是大黄狗》


都说我是狗娘养的，因为是被大黄狗捡回来奶大的。我，大黄狗，我爷爷，我们三个在平凡偏远的山沟相依为命，一个人的到来打破我们所有的生活。大黄狗被打死，爷爷含冤而死，我所拥有的的一切幸福戛然而止。
