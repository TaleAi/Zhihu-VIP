## 简介
孩子抑郁了，父母比孩子还痛，父母该怎么办？作为人类的第二大疾病杀手，抑郁症在青少年中的发生率日益增高，而且青少年患抑郁症的人数每年都在递增。孩子抑郁并非小事，莫把抑郁当「逆反」。


对什么都提不起兴趣，每天只想在家里「躺尸」 ；


有时兴高采烈，但是顷刻间又跌入黑暗的深渊；


要么整宿整宿不睡觉，要么睡个昏天黑地；


在生活中话都懒得说，更何况跟人打交道或去上学；


......


抑郁症严重影响了青少年的身心健康，给家庭和社会带来沉重的经济负担和精神痛苦。


对于家有抑郁症孩子的父母来说，孩子的抑郁症也会牵动着其情绪波动，甚至让他们不知所措，痛不欲生。  


父母是孩子最重要的家人，本专栏心理学专家根据青少年成长的生理以心理特点，结合心理学的知识、量表，分析抑郁背后的成因，为父母正确认识抑郁症，学会与孩子共渡人生难关，提供了科学、有效的解决方法，以帮助孩子早日战胜抑郁症，重拾往日的快乐。
