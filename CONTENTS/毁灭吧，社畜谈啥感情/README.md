## 简介
爱情为什么会成为了奢侈品？


手术前，我才发现给我备皮的护士，是我的合租女室友！！！


啊太羞耻了，但气场上我绝对不能认输！


**——《我的同居女护士》**


  



我的身体给你，你的肾给我。这样的爱你喜欢吗？


**——《受害者》**


  



前世，我死在渣男和小三的婚礼上，小三是集团总裁千金，渣男为了钱途，眼睁睁看着我被小三虐死。再醒来，我回到到十八岁和渣男分手的那一年……


——**《我成了绿茶婊的小后妈》**


  



原来这些年渣男只把我当成白月光的替身？


这一次，我一定要这个渣男付出代价！


——**《白月光，心慌慌》**


  



去合作方公司开会，捧着手机走进洗手间，一抬头，就看到一个男人正在拉裤子拉链，脖子和脸中间夹着手机，正在听电话……


大写的尴尬！


——**《有钱大叔的表白最戳人心》**


  



刚拿驾照第一天上路，我就蹭了一辆卡宴。分明是他稳如老狗，反倒劝我「以后谁急谁先动」


三十来岁的男人，可真狡猾呀。


——**《对的时间，对的你》**
