# 殡仪馆纪事：生死之间

## 简介
面对他人的死亡，人们会表现出心底里隐秘的一面，或是悲痛，或是解脱，或是更为复杂的情愫。本专栏作者在殡仪馆工作多年，经常出入医院、养老院乃至非正常死亡现场，他看到过濒死之人的挣扎，也看到过活着的人们中上演的人间戏剧。


病人还在抢救，三家不同殡仪馆的人已经在外面为了这门生意开始明争暗斗，最终他们坐下来一起协商，怎么从病人家属身上榨出最多的油水……


为了向女友隐瞒我的职业，我拒绝在年底殡仪馆评比中当先进，可路人无心发出的一条朋友圈，让我苦心维系的爱情崩溃了……


一名老人还没去世就被儿女拉到了殡仪馆门口，一家人在大门口争吵起来，互相推诿，濒死的老人眼角流下两行清泪……


作为一名遗体接运工，作者常年在生死之间穿梭，由于这种经历，他的故事拥有震撼人心的力量，他的文字质朴却饱含意味。


死亡是每一个人都会面临的结局，理解死亡，才能理解生命。
