## 简介
房子给人庇护，人赋予房子生命，房子的故事就是人的故事。


有人卖房筹钱，只为给身患白血病的孙子攒笔手术钱；


有人移民国外，国内的房子迟迟不出手，只因挑剔完美的房客；


有人在房子卖掉后的午夜，悄悄潜进房里，想把留存在这里的回忆全部砸碎；


有人一房多卖，专坑老实人，把骗来的钱财挥霍一空，殊不知那是一对老夫妻一生的积蓄......


每一套房子都收纳着人世间的聚散悲欢，每一个房间都藏着人心的密码。
