## 简介
人的一生，大到天灾人祸，小到一句充满恶意的话，都有可能在我们内心留下创伤。被忽视的创伤，很可能造成心理障碍，如何自我疗愈呢？在创伤治疗领域研究超过 25 年的施琪嘉教授，将从脑科学、医学、心理学三方面，为你定制一套科学疗愈方案，帮助你走出受伤心态。


或许你走不出原生家庭的负面影响；


或许你的情路总是坎坷不顺，内心备受打击；


或许你在人际关系中遭遇过忽视、打压、霸凌；


或许你总感到压力与恐惧，总是敏感脆弱、否定自我；


或许你遭遇突发事件，面临无法承受的痛苦，濒临崩溃；


或许你曾被性骚扰，甚至是性侵犯，内心抹不去羞耻感。


**人的一生中，不可避免地会遭受大大小小的打击，在我们内心上留下创伤。**


然而**超过 80% 的人，都不懂得如何处理心理创伤。**


**持续的心理创伤，让我们一生都难得真正的平和、幸福。**


——————


**一个心里有伤的人，往往：**


· 陷入自我攻击：「我不行」、「我不好」、「我不配」、「都是我的错」


· 被糟糕的情绪笼罩：更多感受到不安、焦虑、无助、自卑，愤怒暴躁


· 敏感脆弱：害怕出错，害怕被否定，社交中不自信，战战兢兢


心理的创伤，不处理便会溃烂恶化，发展成更严重的心理障碍，甚至会「遗传」，受伤的大人，养出受伤的小孩。


**走出心理创伤，每个人都有自我疗愈的本能。**


——————

**课程说明**


1、这门课不适合谁听？


有自杀、自伤倾向；  

被确诊为重度抑郁症、严重 PTSD 疾患。  

若有以上几种特征，需要及时就医、并遵医嘱。可以配合课程做自我改善，但不可单独依靠这门课。


2、听完课后，会达到什么样的状态？


所有的改变都是有一个过程的，本课程或许不会让你立刻就能彻底摆脱心理创伤的负面影响，但是会让你更加理解与接纳有创伤的自己，同时能够学到系统有效的应对创伤的自助方法，坚持运用、练习，创伤的感受就会离自己越来越远。
