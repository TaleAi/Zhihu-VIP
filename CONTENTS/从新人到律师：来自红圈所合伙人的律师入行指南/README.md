# 从新人到律师：来自红圈所合伙人的律师入行指南

### 分类：职人

### 状态：已更新至第 9 节, 共 10 节

### 存档日期：2022-06-15

## 简介
你适合做律师吗？做律师真的有那么累吗？如何去知名大所做律师？本专栏邀请到中伦律师事务所合伙人，已执业 20 余年的薛海滨律师，针对在校学生，实习律师，执业律师等法律人群的种种困惑，用自己的经验进行解答。律师执业的路途并不容易，希望能给大家一些启发。


我看到知乎上的知友们，对于律师职业及律师行业有着各种各样的憧憬，同时也有着纷繁复杂的各类问题，比如：


我适合做律师吗？做律师真的有那么累吗？律师刚开始 2～3 年，难熬吗？法学专业很看重院校吗？法律职业方向还有哪些其它选择？


当然，也有知友问过我，如何去知名大所做律师？如何进红圈所诉讼团队？如何拿英国律师资质？涉外律师有何优势？国外律师工资高吗？这些非常具体的问题。


作为执业 20 余年的老律师，我愿意以自己的工作经验，以及职业生涯中的观察与思考来同大家一起探些问题，或许在探讨中，能解答彼此的困惑。同时我也邀请到了我的一些同事，他们都是年轻的法律人，希望他们的经验能给各位「新」法学毕业生、法律新人们一些新的角度去思考。


专栏中有许多我的经验与记忆的总结，难免带有主观色彩，故以下内容，如有偏见，绝非主观故意，有得罪之处，还希望同行在拍砖头时能秉承律师般的理性与专业水准。谢谢！
