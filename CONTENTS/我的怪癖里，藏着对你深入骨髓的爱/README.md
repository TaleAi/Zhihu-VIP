## 简介
「爱」是这个世界上最温暖，最治愈的情感。但是「爱」也能燃烧成最生动的恶魔。


当我的爱降临的时候，你怕了吗？


爱与恨，爱与欲望，爱与控制，相邻得住着。


那些不为常人所理解的，以爱为名的怪癖，


像是一面面镜子折射出人性的脆弱。


我爱你，因为我把你整成了，我最爱女人的模样。我仔细端详着她的脸，从大众审美角度来说，她的鼻子矮了一点，脑门不够丰满，牙齿咬合有点小问题，下巴前凸明显，长度过长。但是她有一双明眸，像无辜的小鹿一样惹人怜爱，眉眼间像极了我的妈妈。 


我爱你，所以把你身体的一部分给我吧。「美好的女孩就像器物，那些裂隙是她们爱过的证据。」我从身后环过男友的腰，握住他手上的龙泉瓷杯。「你看这些金色，流过杯子的裂隙和破口，又是如此完美。」完美么？直到我接过那条他用前女友们的趾骨做成的项链，我终于明白了「裂隙」的意思。


……


所以，我爱你，


你能负担得起，我的爱吗？


（本专栏均为虚构故事）
