## 简介
以艺术讲师王晶十余年来在法国、美国各大博物馆阅览绘画的经验，从作品的细节中，为你解读艺术史背后的暗黑细节与真相。


**美国的霸权，真的只靠拳头维持吗？**


——大都会博物馆，《一位青年的头像碎片》：


一个霸权，如果只靠暴力维持，很快就会走上绝路。其实，霸权还有另外一个形容方式，叫做「领导力」。


**20 世纪最著名的画家，究竟是天赋过人，还是抄袭过人？**


——毕加索，《一个古典雕塑的素描》：


毕加索有许多惊人之语，比如说：「我用了一生，才学会像个孩子一样绘画。」但他在孩子时的素描作品，竟然是抄袭而来？毕加索背后，究竟有什么样的故事？


**日本奈良兴福寺的哼哈二将，如何看出战火中重生的武僧之魂？**


——奈良兴福寺，《金刚力士（哼、哈）》：


如果你到奈良去，会看到两尊并不高大，却形似神鬼的木刻雕塑。他们究竟是谁，为什么会在兴福寺中站立？他们的肌肉，为何鼓鼓囊囊，看起来就像游戏《只狼》中的怪物？
