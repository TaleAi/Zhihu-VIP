年，区块链业务的附加值将达到 **万亿** 。


因此，关于加密数字货币和区块链技术，不论你是否看好，


▶以下问题你都需要了解：


**◎ 从金融的角度追本溯源 ，私营数字货币到底是下一轮金融危机的「解药」，还是一场注定破灭的「投机泡沫」？**


**◎ 经济学界如何看待加密数字货币 ？区块链技术当前存在哪些缺陷？未来有哪些实际场景的应用？**


**◎ 针对这个 「入侵者」，各国央行会如何监管应对？区块链的下一个十年，又会发生怎样的变化？**


▶知名央行数字货币专家，为你带来关于 **区块链与数字货币的实践应用与前景的最前沿研究**


◎ **姚前 ：**中国证券登记结算公司总经理，曾任中国人民银行数字货币研究所所长

◎ **刘瑾** ：清华大学金融科技研究院鑫苑房地产金融科技中心研究员


**▶阅读本专栏你将获得 ：**


**◎关于加密数字货币**


从现代密码学的演进脉络追溯数字货币的技术起源 探讨加密货币技术的优点与风险，

利用经济博弈论剖析区块链技术的激励相容设计，展望新的十年区块链技术的方向与前景………


从起源到未来，从认知到应用， 赋能区块链创业者与投资人， 全方位提升你对区块链与数字货币的认知。
