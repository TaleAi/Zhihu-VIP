## 简介
随着年龄的增长，我们都必须成为职场「社会人」。什么是职场？在不同环境、不同人群中，需要通过自己高效的工作能力和人际交往能力提升自己的竞争力。在职场中，只有具备了核心竞争力和沟通技能，才能减少在职场中的困惑。


你是否遇到过这样的问题：


刚毕业，去国企、外企还是民企？工作好几年，到底还该不该转行？


毕业几年，工资依旧在原地打转，问题出在哪儿？


想重新找工作，投出简历石沉大海是怎么回事？


遭遇职场「天花板」，如何追根溯源，突破瓶颈？


……


如果你即将进入职场，你应该提前评估和认识自己的能力，避免走弯路，赢在职场起跑线。如果你准备换工作，你应该学会在市场上寻求适合自己的机会。


无论你处在职场哪个阶段，都可以利用本专栏的方法指导，了解系统性的职业发展思考框架，找到适合自己的职业定位和发展目标，制定切实可行的职场提升操作方案，实现自我蜕变。
