## 简介
一朝穿越，发现自己竟是甜文炮灰女配？？？为了活命，只能拼命撩拨美强冰山男主。都说女追男，隔层纱，没想到这纱竟是撒哈拉的滚滚黄沙，本以为自己熟读甜文套路，没想到男主却是反套路撩妹王？？？


考研狗一朝不慎，穿越到要杀我的仙君床上？！装白莲花、绿茶婊、小作精，专注套路男神两千年，才发现我竟然一直被反套路！？


——《与我有一夜露水姻缘的仙君此时要杀了我！》


  



我穿越成了一只喜鹊，眼前七大姑八大姨已经变成让人流口水的烤鸟，眼看大祸临头，却被一个男子所救。他天天给我灌输灵力，终于让我变成女子。


我以为他对我百般宠溺是爱我，却不想我只是他心中白月光的替身，他为那人攻打天界，又为那人弃魔从仙。


得知真相我恨极了他的套路，可我早沦陷在他的温柔和溺爱中，欲罢不能……


——《我穿越成了一只喜鹊？？？》


  



沈燕一个普通大学生竟穿到大明星沈言之身上，近距离接触偶像哥哥徐赞，心砰砰跳！


「徐赞，我可以潜规则你吗？」


「好啊！我先脱还是你先脱？」


不是，沈言之惊呆了，节目里的真心话大冒险而已，他怎么还当真了？！就，赖上了自己了？


可是徐赞怎么和传闻中的人设不一样？他温柔体贴、霸道病娇，各种造作，还是个粘人麦芽糖……


怎么睡一觉起来沈言之就成为成为他女朋友了？


拉着沈言之去直播领证又是什么鬼？


嫁给偶像的她心里美滋滋，只是，怎么感觉好像有哪里不对呢？


——《言之必赞》
