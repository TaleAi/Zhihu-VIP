## 简介
漂泊在太平洋深处，33 名船员经历了长达八个月的「大逃杀」。船上没有摄像头，通讯设备悉数毁坏，他们作案手段凶残，伪造被劫假象，骗取死者亲属钱财，并企图销毁证据、掩盖实情。而事件的所有证据，都来自于当事人口供——到底谁在说谎？根据「鲁荣渔 2682 号惨案」创作的非虚构纪实报告文学。
