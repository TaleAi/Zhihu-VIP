## 简介
有了钱就会失去烦恼吗？其实不然。


没有人比那些不愁吃喝又谈不上财富自由的中产们更懂得「钱，买不来一切」的道理。


一毕业就结婚的「傻白甜」，


替初恋养儿子的「深情男」，


沉溺于黄昏恋的「老来俏」，


还有在围城之外虎视眈眈的「伏弟魔」、「凤凰男」……


财产不多，破事不少，一组发生在中产家庭的八卦故事，希望给你带来一些快乐。
