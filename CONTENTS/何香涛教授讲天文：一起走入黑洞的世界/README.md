## 简介
带你回顾了当代最著名的理论物理学家斯蒂芬•威廉•霍金曲折的一生，围绕霍金生前最主要的成就「黑洞理论」介绍了黑洞概念的提出、黑洞的形成和物理性质等，其中穿插了牛顿、爱因斯坦等物理学大家的传奇故事。


黑洞是什么？霍金为什么与爱因斯坦齐名？人类能通过虫洞实现星际穿越吗？


![img](https://pic4.zhimg.com/v2-11126b0add103fde07f8bd676df7b282.jpg)

对于科学家来说，黑洞一直是个未解之谜；


对于普通大众来说，黑洞是频频出现于科幻小说和大片中的热门话题。


如果一个人掉入黑洞，究竟是会被立即撕成碎片，还是会安然无恙毫发无损？


![img](https://pic2.zhimg.com/v2-f0d3cbcdb7cc23307e22208918c5c2a4.jpg)

黑洞概念的提出源于牛顿的万有引力定律。


1916 年，史瓦西从爱因斯坦广义相对论的场方程中推出了同样的黑洞半径公式。


但爱因斯坦本人却不承认黑洞的存在。直到今天，不承认黑洞存在的科学家也大有人在。


![img](https://pic4.zhimg.com/v2-22ad8c01d5fe9ad90459ff0e9f53e55a.jpg)

黑洞是否存在？为何这么神秘？背后有哪些科学故事？


**本专栏内容生动、图片丰富、叙述深入浅出，带你走入黑洞的科普世界。既可以作为各位天文爱好者的知识拓展课程，也是用于给广大青少年的天文学普及课程。**


让我们跟随天文学大家，一起进入黑洞的世界！


注：本专栏精选于北京师范大学天文系何香涛教授所著图书《追逐类星体》，该书围绕神秘天体的发现、特征、观测等讲述了一个又一个感人的故事。
