## 简介
远洋货轮在看不到边际的大洋中航行，船上的生活是怎样的？海盗、瘟疫、台风、偷渡客，危险无处不在！老船长忠叔，将他三十五年的航海经历，凝结成十篇专栏文章。在这些故事中，道德的冲突、人性的软弱、自然的残酷，无一不体现得淋漓尽致。


远洋货轮在看不到边际的大洋中航行，船上的生活是怎样的？


如果你问**忠叔——一位资深远洋货轮船长，他会告诉你这些：**


平静的时候真平静，甚至平静的有些无聊，无聊到需要几个 T 容量硬盘的电视剧打发时间。


但不平静的时候……船上的惊险远在你的想象之外。


**有时是遭遇天灾：**


·在东非喀麦隆的港口，死亡病毒埃博拉被悄悄带上货轮；


·在几内亚的海湾，货轮无意间闯入台风的正中心；


**有时则是遇到人祸：**


·在新加坡的狭窄海峡，南洋海盗尾随货轮，伺机而动；


·在通往纽约的航线上，一个偷渡客在船内引起轩然大波；


如果遇到这些，船长要怎么办？


老船长忠叔亲自口述，请女儿代笔，将他三十五年的航海经历，凝结成十篇专栏文章。在这些故事中，道德的冲突、人性的软弱、自然的残酷，无一不体现得淋漓尽致。


**忠叔说，船员有退路，但船长没有。出海不是探险，探险遇到危险可以选择放弃，而出海，不管前面是什么，都要抵达目的地。**


如果你对这些冒险故事感到好奇，想知道忠叔如何克服难题、化险为夷，那么请你走进这个专栏，与忠叔一起经历航行中的「惊涛海浪」。


**道人航海曾何劳，久将身世轻鸿毛。任凭风浪如山去，伫立船头一笑消。**
