## 简介
本课程由资深律师苏琦主讲，为你讲解基础税法及股权涉税相关的知识。课程中，苏琦律师将穿线讲解基础税种的原理、纳税义务人、税额计算方法、纳税义务产生时间四大部分，同时，对股权相关的税收进行了整理，力求给到投资人和相关服务人员一个框架性的认识。


针对一个交易，小到购销合同，大到并购重组，都与税收密切相关，尤其是大型交易需要了解纳税义务，税额计算，还需要了解征管政策，最新的税收动向等。


近年来，股权交易及投资市场非常的活跃，而与股权相关的交易中，税收更是必须在交易进行之前就必须要全盘考虑的重要问题，否则可能会直接影响交易的收益甚至导致交易的失败。


因此本课程中，苏琦律师将穿线讲解基础税种的相关知识，包括原理、纳税义务人、税额计算方法、纳税义务产生时间等，


同时，对股权相关的税收进行了整理，按照从投资取得股权到持有期间获得分红收益、再到转让退出的时间脉络，对每一道环节涉及到的税收以及相关筹划可能会涉及到的优惠政策进行了梳理，希望给到投资人和相关服务人员一个框架性的认识。


让大家在以后的交易以及为交易提供服务的过程中，关于税收问题有一个清晰的思路的思路。
