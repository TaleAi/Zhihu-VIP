## 简介
孩子厌学问题很头疼？听听厌学问题专家李旭怎么说。


调查显示，70% 的中小学生、80% 的大学生都有不同程度厌学情绪，且越来越呈现出低龄化趋势，比如从小学一年级就开始厌学。甚至，「好学生」突然厌学的现象也越来越多。


其实，如果在养育孩子的早期、厌学出现的早期、厌学加剧的早期，就发现厌学的真正原因，并尽早进行调整，孩子后续发生严重厌学的可能性就会大大降低。


**孩子动力不足，到底怎么了？**
-----------------


家庭治疗认为，个人的症状行为不仅仅属于个人，而是属于他所在的整个家庭系统。厌学也不例外。


积极心理学之父马丁塞利格曼提出的习得性无助，是厌学产生的主要原因。


然而，并不是所有的父母都能真正理解孩子厌学的原因，接纳孩子厌学的现实。


尤其是现在普遍重视「分数」和「排名」的教育体制，「父亲缺位」和「边界不清晰」等家庭功能不全的家庭氛围，非常容易挫伤孩子的学习动力和信心。


如何把「习得性无助」转化为「习得性乐观」，创造心流体验至关重要。


![img](https://pic4.zhimg.com/v2-4ecf4555861c7e7f6662036837ab7ed3.jpg)

身为厌学问题专家的李老师，是有医学和心理学双重背景的，他先后毕业于北京大学心理系、上海交通大学医学院，从事心理咨询工作已有 20 年，专业训练背景是精神分析和家庭治疗。


同时，他也是一个父亲，他的孩子成绩优异，正处于青春期。在长期一线临床工作中，李旭老师影响数十万家长与孩子，该课程经过近一年的总结和提炼，萃取出最实用的防厌学指南。


李老师结合多年家庭治疗经验总结出：


打破孩子习得性无助、创造心流体验，破解厌学难题、提高学习动力的金钥匙就握住在父母手中。


只要我们愿意付出理解、忍耐与爱的能量，成长的一切问题，都将迎刃而解。 


  




**这套课程有什么特别之处：**


1.实操性强：拒绝空泛道理， 20 多年实战经验与心理学理论结合，并集合了众多成功干预厌学案例的经验；


2.重拳合力：专业实操课程+全面的家庭功能评估测试+章节复盘与课后练习，全方位厌学干预解决方案；


3.聚焦家庭：以家庭治疗为理论根基，从源头上解决厌学问题，激发孩子内在学习动力，为家庭功能赋能 。
