## 简介
人生无常，唯意可平。


温柔坚韧、淡然从容的女主，更有深情难负、清冷绝艳的美强惨男主，即便身处不同的时空、相距遥遥的岁月，他们也能够彼此坚定、相互付出。


**《难珠》**公子为我取名难珠，却只唤我阿难。  

我伴公子身侧三载有余，在他红衣娶妻之际，我自城楼坠下，却听见有人声声悲戚，唤我珠珠儿。  

我便忽然想起当年背着公子一步步雪山求药之时，他也曾伏在我的肩上，落下一滴泪，唤我一声，珠珠儿。
