## 简介
「每个人都有属于自己的一片森林，也许我们从来不曾去过，但它一直在那里，总会在那里。迷失的人迷失了，相逢的人会再相逢。」看过村上春树小说的人，或多或少都被书中的话触动过。心里也曾泛出一丝探寻的意味：为什么村上春树尽管骨子里是日本式的，但其视角在很大程度上是西方化了的？村上小说中那些类似寓言、神话等离奇荒诞的情节都代表着什么？为什么说村上笔下的边缘人的感受和心理未必就是边缘感受呢？想读懂村上，你一定离不开他的翻译——跟随林少华老师，一同走进村上春树 の 小说世界。
