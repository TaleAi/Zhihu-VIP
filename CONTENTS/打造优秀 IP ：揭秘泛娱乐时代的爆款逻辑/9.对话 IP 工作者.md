## 9.对话 IP 工作者
在本期课程之前，我和多位一线 IP 工作者进行了交流，了解了他们目前所处的 IP 行业的一些状态，以及他们觉得这个行业需要解决的问题。概括而言，我们的交流主要涉及了以下三个问题：**第一，目前 IP 生产机构的情况；第二，IP 内容创作者的困境以及造成这种困境的原因；第三，IP 商业运营者的困境以及他们想要寻求的突破。**本期的所有内容都来自我与 IP 从业者的对谈，也是真正的 IP 工作者的心声。


之前我们讲到过美国与日本的 IP 生产，在中国的状况则显得更为独特，目前国内参与 IP 生产的商业机构大致可以分为两类，一类是大型互联网公司的文娱部门，在漫画、游戏、网络文学、影视等各个领域都有涉及，比如我们所熟知的有腾讯阅文以及阿里大文娱，另一类是垂直的 IP 制作公司，只专注于一方面的内容生产或者制作，比如做游戏的完美世界，漫画工作室夏天岛都是其中比较知名的几个公司。


这两类公司各有侧重。第一类公司因为有实力雄厚的互联网公司支持，在资金与渠道上占有优势，他们常常以提供平台为主，业余爱好者在其搭建的平台上进行创作，这样为他们发现具有潜质的作品提供可能，注重发现和运营。第二类公司一般是由专业的 IP 工作者组建，主要以内容的生产制作为主，具有较高的门槛，在内容上具有专业性，主要由公司内部生产内容。


从两类不同公司的特点来看，对于想要通过和前辈的学习，然后一步步从小虾米成为独当一面的大咖的创作者而言，在第二类公司能够得到很大的发展，积累经验，了解目前的市场状况、业内目前现有的技术以及未来的发展方向，对于个人能力提升很有帮助。而那些对于自己创作能力很有信心的创作者，也可以通过第一类公司所搭建的平台来进行个人 IP 的打造，只是在浩如烟海的作品之中，你要耐得住寂寞，并且适应互联网内容迭代的速度，在保证质量的前提下不断创作。


其次，在稍微了解了 IP 创作者与 IP 生产机构的关系后，我们来聊聊如今 IP 创作者在工作中遇到的一些问题。这些问题，主要体现在以下两方面：一方面，他们感到自己的「创作逻辑」和机构的「经营逻辑」之间有些难以调和；另一方面，在他们看来，现在的 IP 生产业界，存在着一种「人才错位现象」。


在第一方面，IP 创作者认为，在进行内容创作时，「内容」和「形式」是有很高关联度的。例如：同样一个故事，如果用漫画的形式呈现，注意力会集中在如何在纸张上运用分镜；如果用电影的形式呈现，就要综合地考虑画面、声音、表演等要素的配合。所以，用漫画来讲一个故事，和用电影来讲一个故事，其实根本就是两件事情。事实上，无论是美国式 IP 还是日本式 IP，其一开始的源头都是某种具有单一形式的「作品」而已，当「作品」本身壮大了，取得了越来越高的人气，才被开发、包装成 IP。而这意味着，**根据「创作逻辑」，在进行创作时，一开始时应该要将心力投放于创作「单一形式的作品」，至于 IP，应该是一件在人气积累超过一定阈值后，水到渠成的样态。换言之，IP 应该是一个「阶段性的终点」，而不应该成为「最初的起点」。**然而，由于现在 IP 已经成为了一个被确认下来的东西，所以制作方们出于「逐利」的本性，会倾向于按照「经营逻辑」办事。**具体来说，就是他们希望从一开始就打造 IP，从而可以在更多的社群和更大的人群中赚取利润。这样一来，创作者们被要求从一开始，就创作出符合「跨媒介」这一特征的故事内容**：它不但要能在影视故事上呈现，而且要能被开发成游戏，甚至还要有利于开发出各种周边产品。带着如此沉重的枷锁去进行内容创作，创作者能使用的题材以及他们的想象力空间被极大的压缩了。某种意义上，中国为什么很难出全新的爆款内容，和创作者其实是戴着镣铐跳舞紧密相关。鉴于以上情况，IP 的内容创作者大多会有希望 IP 生产机构「还内容于作品」的心声。


在第二方面，在 IP 内容创作者看来，他们认为「有前景」的东西，经常得不到经营者们的认可；而经营者要求他们做的东西，他们又往往觉得「无趣」。可以说，**内容创作者和经营者之间，对于什么才会成为一个「优质 IP」，有着相当不同的标准。**为什么会出现这种情况呢？IP 内容创作者做出了如下解释：就拿做游戏来说，无论是内容创作者还是商业经营者都知道，「好玩」是一个游戏的生命线。在经营者眼中，「好玩」是一套可以量化的标准，诸如「过往同类型游戏的销售额」、「当下同类型游戏的口碑度」这些数据，会被他们视为「好玩」的充分证据。然而，在内容创作者眼中，经营者的这套方式是有所偏颇的。他们坚持：「好玩」是一种超越数据的「感受」，所以，要知道什么游戏好玩，怎样才算好玩，就必须对「游戏」本身投入大量的时间精力。某种意义上，只有通过经常「打游戏」，进而「爱游戏」的人，才能真正理解什么是「好玩的游戏」。因此，对于打造一个 IP 的资源该投放在什么地方，要投放多少资源这些问题，IP 的内容创作者与经营者之间，经常会有意见相左的时候。和我对话的一位 IP 创作者，用一句话总结了这种状况：「会玩」的人「没钱」干事，「有钱」的人却「不会玩」。


听完了 IP 内容方面从业者的声音，最后，让我们也来听听运营方面的 IP 从业者有什么要说的话。首先，运营方面的 IP 从业者认为，IP 构建行业在本质上和其它所有商业行为一样，归根结底，为的都是逐利。这其实是无可厚非的。**真正要被非议的地方，是在于许多投资方或制作方用了不恰当的方式逐利，即：他们违背了「健康的 IP 逻辑」。所谓健康的 IP 逻辑，就是要让每一个新产品为品牌增添价值，而非通过开发圈钱型产品，攫取品牌固有价值。**像「青椒学院」里提到的《哈利波特》系列，就是一个十分健康的 IP。这个系列以优秀的小说为基础，每出一部电影，甚至每衍生出一个像《神奇动物在哪里》这样的新系列，都能为版权方带来巨大收益，同时也能让更多的人感受到那个魔法世界的魅力。实际上，像《指环王》、《星球大战》，都可以认为是一直用健康的 IP 逻辑运作着。然而可悲的是，我国许多 IP 生产机构却要反其道而行之。其中，对于金庸武侠小说的「魔改」就是明显例子。凭借着金庸先生这块金字招牌，当然也包括一些小鲜肉的号召力，在各大电视台上，播放着许多让真正金庸迷看了会破口大骂的电视剧。虽然这些电视剧的口碑很差，但是却还是能靠吃「金庸红利」斩获一定的收视率。只是长此以往，「金庸」这个品牌就要受到伤害了。


其次，和 IP 的内容创作者的意见不同，运营层面的 IP 从业者认为：**其实许多 IP 运营者所缺乏的并不是「娱乐素养」，而是「娱乐信仰」。也就是说，他们中的不少不是「不会玩」，而是「不敢玩」。**在如今「股份责任有限公司」的主流结构下，真正拥有决策权的并不是某个人，而是董事会。这样的直接后果，就是任何项目、任何决策最终都要对股东们负责。股东们最终要看的是什么呢？是业绩，是回报，所以办事的人无论地位高低，指导的核心思想都难免成了「如何在规定的时间内给出漂亮的 KPI」。一旦来自股东的压力骤增，即便明知是竭泽而渔，也只能做出一些违背健康 IP 逻辑的事情。客观而言，这并不能说品格低下，而只能认为是正常的人性。当然，也的确存在一些情操低下的行为。例如，有那么一些些游走于各个 IP 生产机构的职业经理人，他们明知一旦采取某些手段，虽然能提高短期 KPI，但却会长期伤害 IP 价值。然而，为了自己的履历好看，他们会强行将那些措施推行下去。之后，他们带着漂亮的数据跳槽到了别的公司，拿到更高的薪水和地位，而原来的公司，却要为一个烂尾的 IP 买单。


在听取了这些 IP 从业人员的经验总结和思想感悟后，也许大家就会对中国 IP 状况的内容有更深层次的理解了。是的，我们取得了很不错的成绩，但是距离一个健康、合理的业界生态，还有很长一段路要走。
