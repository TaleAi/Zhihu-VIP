## 5.美国式 IP：文化商品的标准化生产
「美国式 IP」和「日本式 IP」这两个概念的界定并不只是根据单一的地区划分，所谓的美国式 IP，指的是 「用美国主流方式打造的 IP」；同理，日本式的意思就是「用日本主流方式打造的 IP」。


美国打造 IP 的主流方式，是一种「标准化工厂模式」，在这种模式下，IP 通常会「以公司为名」；而日本打造 IP 的主流方式，却更加灵活小巧、形式各异，在这些方式下，IP 通常会「以作品为名」。然而，这种状况并非绝对的，美国同样存在着许多「以作品为名」的著名 IP，例如《星球大战》和《冰与火之歌》；而日本同样存在用「标准化工厂模式」打造的著名 IP，例如宫崎骏的吉卜力公司。因此，所谓的「美国式」和「日本式」，只是来自于「不完全归纳」的概念，我们说的只是一个大致状态，而不是一种绝对形态。我们分开讲述美国式 IP 和日本式 IP，为的是大家可以更为方便地接受不同的 IP 打造模式，而不是要为这两个国家的娱乐工业、出版行业进行绝对化定性。


明晰了概念之后，就让我们进入这一讲的内容：在这一讲，我们首先将目光聚焦于美国的「标准化工厂模式」，为此，我们将选取「漫威」作为此种模式的案例。


**漫威真的构建出了一个「宇宙」。而我们今天就要了解漫威是通过怎样的方法，构建出了自己的宇宙。**从漫威系作品的内容，以及漫威这个漫画公司的运营方式这两方面，我们能找得到问题的答案。


**·同一程式下的漫威宇宙**


在漫威作品的内容方面，让我们首先来回顾一下会能蹿红 IP 所拥有的三点特征：


第一， 有着核心程式简明但延伸内容丰富的世界观；


第二， 有着属性鲜明的角色；


第三， 有着群像剧路线的故事架构。


很显然，**漫威的作品几乎完美地符合以上三点特征。**在世界观方面，漫威的核心程式是「超越常规者对常规的守护」，美国队长、钢铁侠等人守护美国、黑豹守护自己的部落、绿巨人守护着「正常」的自己、X-MEN 守护地球、复仇者联盟守护整个宇宙，这些内容丰富的故事其实都是由同一程式所衍生的。


在角色方面，每一位超级英雄都有着区分度非常高的标志性造型，美队的盾牌、钢铁侠的战甲、索尔的锤子都让人过目不忘，某种意义上，如果为了「看个爽」的话，英雄的装备甚至比英雄自身更抓眼球。


在故事架构方面，漫威的作品本身就不是「一个」故事，而是「一系列」故事，如果我们将漫威视为一个整体的话，漫威作品天然就是群像剧，而这一点，在超级英雄共同登场的复仇者联盟中就有更为直接的体现。**而且，通过群像故事，我们可以看出漫威的作品有着高度的整体性，它是在同一个宇宙框架基础上进行创作的。**


**·漫威式创作法**


要回答这个问题，我们就必须提到斯坦李。对于漫威而言，斯坦李最重要的贡献还不是他创作出了「神奇四侠」、绿巨人、钢铁侠和蜘蛛侠这些经典角色，而是**他提出并最终确立了「漫威式创作法」。**这套方法可被具体描述如下：第一步，编辑和编剧讨论，得出一个故事大纲；第二步，画家接到大纲后，决定故事的步调与重心，画成指定页数的漫画。第三步，编剧拿到画家的铅笔稿后，以其作为指导，撰写最终的剧本、人物对话以及标题。第四步，画家进行描字、上色、墨线等工序。


可以说，这套方法是「漫威」这一品牌可以长盛不衰的秘诀所在。斯坦李之所以要大力推行方法，是因为他明白，**优秀的文艺创作的本质其实就是「创世」。**然而，并不是所有人都像他那样惊才绝艳，能凭一人之力架构、描绘并推动整个世界。在创作中，有的人长于「设定」、有的人长于「塑造人物」、有的人长于「讲故事」，但要同时具备、调和这几种能力的人，却寥寥无几。**也就是说，「创世」这种逆天的事，不但不适合普通人，甚至不适合「普通天才」来干。**于是，**在「漫威式创作法」中，斯坦李对「创世」进行了明确分工。**如此一来，有的人将精力集中在世界观搭建、有的人将精力集中在故事情节设计、有的人将精力集中在人物形象设计，然后，几者再相互组装磨合。在这样的标准化模式下，虽然很难出现质量极高的「神作」，但是却能保证源源不断地产出质量上乘的「佳作」。某种意义上，我们可以有这样一个结论：**要量产出好的 IP 内容，就是要化文艺为工艺。**


**·IP 至上的商业运作**


**在运营战略的层面，漫威是十分典型的「IP 至上主义」。在某些极端的时候，甚至能会为了「整体」而牺牲「个性」。**其中，将雷神索尔强行整合进复仇者联盟是一个十分典型的案例。如果按照纯粹的「文艺逻辑」，拥有「神格」的索尔与其他一大帮子超级英雄是不应该同台演出的，当真这样的话，用「关公战秦琼」都不足以说明这一状况，这可是「孙悟空战秦琼」。然而，即便冒着战斗力系统崩坏甚至世界观混乱的巨大风险，漫威还是这样做了。原因无他，相比起一个更「合理」的索尔，漫威更需要一个更「庞大」的复仇者联盟。毕竟，「妇联」的吸金和吸睛能力都远超某位单一的超级英雄。而「漫威」这个巨大品牌的吸引力，更非旗下任何一个或一群超级英雄可比。


**在「IP 至上主义」的战略下，漫威的实际操作手段可谓大刀阔斧。第一，漫威并不需要自己去「发掘」新的市场痛点，而是直接将已经被市场初步检验过的新鲜血液「纳入」到自身的「宇宙」中。**当一位新进的漫画家要进入漫威体系，漫威就会调整乃至重塑其风格，让这些新进的漫画家用漫威的世界观设定，用漫威的人物谱系，用斯坦李的创作方法，总而言之，就是要「旧瓶装新酒」，让新人匹配自己的宇宙**。**


**第二，漫威充分发挥了「跨媒介叙事理论」，而这方面最为极致的体现，就是「漫威电影宇宙」。**一方面，相较于漫画，电影拥有着更广大的受众，也有更高的社会接受度，可见，在「电影」这个正面战场上占据先手，极具战略价值。另一方面，相较于漫画，电影这种艺术形式其实更能展现漫威作品的魅力，虽然投拍一部电影，并为各位超级英雄制作相应的装备和特效必定成本高昂，但是当那些装备与技能真的在大银幕上活生生展现时，给予观众的震撼必定是巨大的。可以说，通过电影这个媒介，通过高投入而获得更高产出，极大地扩张了漫威这一 IP 的影响力。


**第三，漫威在「联动」方面也下足了功夫**，而这能体现在以下两方面。一方面，漫威非常注重旗下各作品的「内部联动」，我们能很轻易地在漫威的一部作品中，找到另一部作品的元素，事实上，对于社会上的绝大部分人来说，他们熟悉的只会是漫威的一部或几部作品，知道其中的一个或几个人物，而有了这种「内部联动」后，作品之间就能达到「相互引流」的效果了。


另一方面，漫威同样注重与自身品牌外的事物进行「外部联动」，主要是联合营销。在过去的 2018 年和刚刚开始的今年，漫威都和日本服装品牌优衣库联合推出了 T 恤和卫衣。对于漫威的粉丝而言，买上一件 T 恤表达对自己喜欢 IP 的支持并不是一件困难的事。同样的还有漫威和 VANS，可口可乐以及欧莱雅等品牌的联合款。通过与其他品牌联合推出带有漫威符号实体产品的方式，将漫威渗透到受众的生活之中，根据社会心理学中的「曝光效应」，人们对于曝光越高的事物好感越高。漫威通过充斥受众生活的方式不断强化着受众自己粉丝身份的认同。


**第四，漫威对粉丝体验的经营也堪称经典手笔。**一方面，根据「品牌社群」理论，漫威塑造自身品牌，笼络对自己有高认知度和认同度的受众为粉丝社群，并进行内部信息传递，使得粉丝接受信息并对品牌做出反馈。漫威著名的「推特运营」所践行的，其实就是这一理论。漫威公司通过运营推特账号，长期与粉丝互动，回复留言，并且向粉丝赠送礼物。漫威通过这种方式，笼络了数量极其庞大的，对其有极高认知度和喜爱度的用户。另一方面，根据詹金斯在《融合文化》中提出的「情感经济」理论，漫威深知，基于对自己的高度认知和喜爱，粉丝在进行消费时会让渡出部分思考主权，成为自己「完美」又「过度」的消费者，这样的消费者在消费时，会有「集邮式消费」的倾向。「集邮」的完成，不但会给漫威带来不菲的收益，而且又进一步加固了品牌与粉丝之间的联系。
