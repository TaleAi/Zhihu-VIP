## 7.IP 构建在中国：当代文化精品稀缺
在我看来，2019 年也许会成为一个值得纪念的年份。因为在这一年，那些此前虽然不被看好，但却凝聚了无数从业者努力的国产 IP，终于出现了一些良好的苗头。而这些良好的苗头，可以归纳为以下三方面：


**第一，** **具有代表性的类型作品接连出现；**


**第二，** **资本运作开始趋于理性；**


**第三，** **受众素质提高，版权意识和付费意愿都有所加强。**


先来看第一方面，根据瞭望智库在 2018 年发布的《2018 年度 IP 评价报告》中提到，2017 年到 2018 年的相关数据都显示，漫画、动画和游戏的原创 IP 位于排名前列的较少，头部 IP 稀缺。但**这一情况，却可能会在今年有所改观。**在影视领域，2019 年初的两个月中，《流浪地球》占据了各大主流媒体和自媒体的头条，作为制作水平毋庸置疑的科幻类型片，它让不少中国的电影迷看到了中国特效发展的曙光；同时，稍微档期靠前的《白蛇·缘起》也这样终于不是依靠「国漫崛起」的口号，而是真正依靠踏实地讲故事吸引了无数「自来水」的动画电影粉丝。至于游戏领域，先有《仙剑》系列珠玉在前，如今又有《王者荣耀》。国家广电总局重新发放游戏版号后，限制了游戏数量，但是却也变相对游戏质量提出了要求，更多成熟的游戏 IP 会陆续涌现。**可以预期，随着 IP 市场的成熟，漫画、游戏、动漫、影视等板块也将出现具有代表性的类型作品。**


然后是第二方面，资本的运作开始趋于理性了。在前几年，最为常见的模式就是网文 IP 加流量明星的组合，在宣传上具有很高的人气，IP 与明星的粉丝都愿意为此付费，但是由于这种组合前期投入较高，后期回报往往低于前期的投入，市场表现力并不如人意。于是在 2018 年的不少 IP 分析中，因为 IP 的市场效果不佳，而认为 IP 时代一去不复返的人不在少数。**难道真的是这样吗？事实上恰恰相反，市场效果不佳证明的只是旧套路的失效，只要能将资金投放到应该配置的地方，IP 的生命力依然是旺盛的。**就拿上面的例子来说，无论是《流浪地球》还是《白蛇·缘起》，都能够看出在制作方在视觉效果上做出了巨大努力，而这些努力，其实是需要有资金在背后支持的。这意味着，资方已经开始意识到：只要把好钢用在刀刃行，无论是成熟 IP 的跨媒介传播，还是具有内容原创性的新 IP，都一样能在商业上获得成功。


第三方面，得益于大家越来越成熟和坚定的版权意识。在国家大力打击盗版的大潮下，观众自己的观影、娱乐习惯也在一点点地发生改变。由于这是一个渐变的过程，所以身处过程中的我们，感受不会特别清晰。但是，只要回忆一下上世纪末遍布全国各地的小放映厅、曾经盘踞在各大视频网站上的盗版影视作品、那些你我可能都曾购买的盗版游戏光盘，然后，再和现在粉丝会用以「我欠星爷一张电影票」为口号走进电影院看周星驰的新作电影、为了能看到喜爱的动画作品而成为 B 站的付费会员、打游戏时会自然而然地想到在 STEAM 上下载这些现状进行比较，我们就能清晰地感到：「观看正版」、「保护版权方」终于成为了大部分观众的共识，付费享受文化产品也成为了自愿、自发行为。


·**新瓶旧酒，精品内容稀缺**


打造一个精品 IP，内容是基石。可惜的是，正是在「内容」这一环，我们遇到了瓶颈，这一情况表现在两个方面：


**一方面，我们打造 IP 时，更多考虑的是「经典」而不是「原创」。**即便是我们刚提过的《白蛇·缘起》，依托的就还是「白蛇传」这块金字招牌。而放眼向更多的 IP 看去，「三国」、「西游」这类的经典元素遍地开花，从网剧《军师联盟》到动画电影《大圣归来》，都在醒目地标记着「经典」如今仍旧盘踞 IP 行业的半壁江山。当然，我们并不否定「经典」的价值，但是对经典的过度依赖，却已充分说明我国 IP 行业的各方对「原创」依旧信心不足。


**另一方面，我们打造 IP 时，会更倾向于依靠「满足欲望」来提高流量，有深层次探讨的 IP 仍旧稀缺，对于引起情感共鸣这件事情，似乎大家只能想到一些负面的情感。**无论我们是否喜欢，「小时代」系列就是火了，买郭敬明「奢侈品罗列法」账的人只多不少；无论我们是否承认，「起点中文网」上的「爽文」就是当下阅读面最广，点击量最高的作品，大多数人对「成为龙傲天，一路开后宫，吊打高富帅」的套路就是乐此不疲。如此一来，要在我国的 IP 中找到完备的世界体系或深刻的思想内涵，真的难上加难了。


**·人才困境与「唯 kpi 论」**


从表面上看，我国的创作者呈现出了「才华有余但格局不足」的状态。而这种状态，是由以下两方面因素所导致的：一方面，到目前为止，我国大多数的 IP 内容创作者大多是「野路子」出身，所受的专业训练相对缺乏。在我国现行教育体制中，要接受关于「创作」的专门训练，就只能入读高校的中文系。然而，即便是各高校的中文系，对适应于 IP 这一新兴形式的专门创意训练也非常稀缺。可以说，创作者们天赋的才华是「骏马」，而通过专业训练得到的技巧就是「缰绳」，由于大多数创作者们的骏马缺少合适的缰绳，作品出现「脱缰」的现象就不足为怪了。另一方面，在 IP 这条产业链中，创作者的职业地位并不高，漫画家、作家的收入甚至可以说是微薄的。即便成名了，在不少思想保守的人眼里，他们依旧是「不务正业」的一群。在这种圈内圈外氛围的夹攻下，许多有天赋的创作者，可能等不到开花结果的一天，就撂挑子不干了。


让我们来谈谈运营者的问题。我们必须承认，活跃在一线的 IP 运营者们是深知市场规律和受众心态的。但是，「市场规律」和「受众心态」只是决定 IP 运营方式的「重要」因素而非「核心」因素，真正的核心因素，是「资本的逐利本性」。虽然说资本如今已经趋于理性，但他们依然存在以下两方面的问题：**一方面，他们追求「短平快」的投资模式，要求资金迅速回笼。**比如，「低龄向」IP 虽然在剧情架构以及思想深度上难有可取之处，但是作品的制作难度和成本都相对较低，它们一旦成功了，获取的回报定然十分可观。这一模式的存在，充分说明了为什么「喜羊羊」、「熊出没」这类「低龄向」的 IP 可以长盛不衰。出于「性价比」的考虑，资本投向这些方面本也无可厚非，但是在这种条件下还要求出现精品 IP，就有强人所难之嫌了。


**「唯 KPI 论」的倾向也一把双刃剑，迷信数据也就很容易被所谓的「关键数据」误导自身对全局的把握和判断。**用 KPI 对一个项目进行评价，实际是一种「以点看面」的机制。但是，这种机制的可行性却有一个前提，即它所在的体系中的各元素能少被、最好不被体系外的原因所左右，如果无法满足这个前提，「以点看面」就很容易沦为「以偏概全」。比如，刷票产业的存在，在粉丝群体中也并不是什么秘密。即使是限定了每个手机号的票数，粉丝也能找到境外的手机号码进行注册，在限定的投票时间内进行刷票。可见，常见的以转发评论点赞来确定热度的方式也并不一定可靠，由于有这些「水军」的存在，就让数据的准确性打了个问号。因此，如果投资者简单以数据作为商业价值的判断，那很可能产生错误的结论。


最后，让我们来看看受众的状况。因为 IP 一般具有跨媒介属性，我们所熟知的 IP 大多是由漫画或者小说改编成其他媒介形态的作品，但是，因为资本过于对粉丝过于依赖，所创作的作品也是粉丝作品，这导致了 IP 受众市场具有单一性，在更多 IP 的塑造上无法提供指导与借鉴意义。
