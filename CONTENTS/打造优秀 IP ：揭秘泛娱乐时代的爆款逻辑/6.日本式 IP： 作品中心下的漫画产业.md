## 6.日本式 IP： 作品中心下的漫画产业
**比起美国 IP 的标准化打造，日本式 IP 的格局相对较小，通常我们熟知的不是「公司」，而是「作品」本身。**即使是爱好日本动漫的朋友，他们平时讨论的大多是「龙珠」、「海贼王」和「进击的巨人」，而不是 「集英社」和「讲谈社」之类的出版机构。也就是说，**作品本身就是 IP。**


讲到这里，是不是会有人好奇**，日本的「作品即 IP」是怎么形成的？这样「作品即 IP」有什么影响？**


我们先来看第一个问题，历史学里有一个「层累理论」，**指的是：绝大多数事物的样态，不仅是被一个具有决定性的因素所「导致」，而是在漫长的历史过程中一点点「累积」而成的。**而日本现在这样的 IP 发展现状，也是有历史累积而来。


**·错失工业化发展良机**


日美两国的 IP 都由漫画起源，而**第二次世界大战是美国与日本 IP 发展的分水岭。**二战中，美国本土基本没有受到战争的影响，二战结束后，美国的工业依旧体制健全，运作良好，因此当地的出版公司和漫画公司可以凭借工业，不断壮大自身，同时他们也把漫画慢慢纳入到公司的「标准化流程」中。而也就是在「标准化流程」中，「作品」的地位被逐步瓦解蚕食了，就形成了我们现在知道与了解的美国式 IP。


与美国不同的是，二战中日本战败，整个国民经济都在崩溃的边缘，文艺行业和出版行业的发展也因此受到影响。在这样的大环境下，日本的漫画出版商们不但依赖不了任何稳定成熟的体系，甚至连维持自身的运营都困难。**在外无助力，内有重压的情况下，出版商们当然不可能考虑长远的「未来」，只能着眼于「当下」。**


那么，什么才能使得这些漫画出版商在「当下」活下去呢？它们能依仗的有两点：**第一，出版商们自身具有极高的灵活性，它们因势而动，以符合时代特征和阅读需求为前提，不断调整漫画本身的载体形式。第二，一些成名的漫画家已经有了一定的受众基础，出版商们要做的就是留住他们并不断出版这些拥有市场的作品。**


在不同的时代，日本漫画出版商都为了卖出作品绞尽脑汁。从上世纪 40 年代起，日本关西的一些小出版商们首先就开始发行了 **「赤本漫画」**，「赤本漫画」印在劣质的纸张上，故事以猎奇怪诞为主，吸引了大批的受众，属于连载单行本形式。但是「赤本漫画」以故事为核心，并不注重画面，也有漫画家不愿意只为了故事而创作，因而，在进入 50 年代后，他们高举「写实主义」的旗号，创办了月刊杂志《影》，创作**「剧画」，剧画也被人称作「视觉文学」，可以从这个名字上看出它与「赤本漫画」的区别**。


无论是「赤本漫画」还是「剧画」，都不是大投入大制作，成本很低，可以维持出版社的生存，但另一方面，过小的体量也限制了漫画故事的篇幅，让很多好的内容无法彻底展开。


50 年代末，经过了「赤本漫画」和「剧画」的井喷式混乱时代，日本漫画开始走向有序。同时，年轻的消费者也开始喜欢租赁漫画，而非购买漫画。为此，**「租书屋」开始出现**，出版社和经销商们聘请漫画家给出租屋画可供租赁，且故事体量也较大的**「贷本漫画」，这种模式成功降低了书商的经营风险和读者**的阅读门槛，达到了商业上的平衡。一时间，「贷本漫画」大行其道，许多本来画「剧画」的漫画家也投身其中。也在此时， **「赤本漫画」和「剧画」在「贷本漫画」中合流，画面和故事得到了整合。**


到了 60 年代，日本的经济复兴，人们不再纠结于过去几十日元的差价，租书屋也随之没落。在 50 年代末期出现的周刊们成为了新的舞台，比如我们比较熟知的《周刊少年 JUMP》就是在 1959 年 3 月成立。许多漫画家都能在这些周刊上刻画各自的故事，而「周刊」也不会强行去「整合」它们，而是让读者来决定哪个故事应当得到更多的重视。


于是，尽管故事的载体和运营方式发生变化，但是这种百花齐放的局面，就一直延续至今，优质的作者与内容也一直成为各家争夺的对象。


**·与出版商平起平坐的作者们**


这也就是我们之前提到的出版商需要靠挽留已经成名的创作者来留住自己的受众，这也是日本特有的一个现象，出版机构的权利是要让渡部分给创作者的。这种平衡的出现，就要提到一系列我们久闻大名的人物了。首当其冲的，就是漫画之神手冢治虫。


手冢最开始画「赤本漫画」出身，他的首部作品《新宝岛》由大阪当地一家小出版商发行，纸质粗糙制作简陋。但《新宝岛》引入电影拍摄技巧的独特画法在当时造成了极大的影响，使得手冢治虫在当时的大阪成为了小有名声的漫画家。后来手冢治虫曾经参与东映的动画电影制作，但是日本的电影业发展适逢电视业的兴起，东映照搬迪士尼的生产模式并没有获得理想的成绩。手冢治虫另辟蹊径，首创电视动画，并且成为第一个组建公司的漫画家，获得成功。**由于没有彻底和出版商绑定，手冢治虫的的独立性让他能够和出版商、发行商平起平坐，他的成功也奠定了「漫画家」和「作品」之于「出版商」的相对强势地位。**


这一地位一经奠定，不但没随着时间流逝而削弱，而且还进一步加强了：如果没有千叶彻弥和梶原一骑的拳击神作《明日之丈》，《周刊少年 MAGAZINE》根本无法和当时拥有手冢治虫、藤子不二雄等大神的《周刊少年 SUNDAY》分庭抗礼；如果不是因为出现了永井豪这样的绝世怪才，《周刊少年 JUMP》也难以异军突起，并在如今成为漫画周刊中王者一般的存在。**这就意味着，在漫画的出版商看来，漫画家和作品才是自己真正的生命线所在，给他们足够的空间和自由度，才是最能赢取利润的方式。**


由于上述两点，**在相当长的一段时期内，日本的漫画出版商们不但无法以自身为主确立一套「标准流程」，而且还要小心翼翼地处理好与漫画家的关系，向他们让渡出相当一部分话语权。**虽然随着日本经济的复苏，出版商的规模也在日益扩大，但是日本文化中因循守旧的特点却根深蒂固。**一旦日本人习惯了以某种方式运作一件事情，并且在这件事情上形成了相应的利益集团，那这件事几乎就注定要一直以这种方式运作下去了。**如此一来，这种「作品」，尤其是「大作」的地位足以和「公司」分庭抗礼的格局，就一直延续了下来。脱胎于漫画的日本式 IP，理所当然就会显示出「以作品名标识」的特点了。


**·制作委员会制度**


那么，日本式 IP 这种「作品本身成为 IP」的特点，会带来怎样的结果呢？这就是这一讲要处理的第二个问题。根据辩证法一分为二的方法，我们就分别谈谈**它带来什么负面与正面的影响。**


**在负面影响方面，日本式 IP 容易出现质量参差不齐的现象。这一现象，在漫画或轻小说作品进行「动画化」时，表现得尤其严重。**以「作品」而不是「公司」为基础的日本式 IP，根本无法像美国式 IP 那样集中统筹大量资源，严格按照标准流程打造跨媒介产品。而制作 2D 的 TV 动画，却是一个烧钱的项目。如此一来，单凭「作品」本身的号召力和作品原版权方的支持，很多时候，根本就无法将 TV 动画制作出来，即使强行上马，万一销量不佳，就会让作品原版权方赔的倾家荡产。**为了解决这一问题，具有「日本特色」的「制作委员会制度」粉墨登场了，**而初次采用这一制度的，正是大名鼎鼎的《新世界福音战士》。


所谓「制作委员会制度」，指的为了控制风险，日本 IP 的原版权方不会「独资」打造一部作品，相反，它们会拉上另一媒介的制作方、发行方、播放方乃至其它投资人一起，组成一个关于某个作品的「制作委员会」，投资的时候，大家按比例投资，项目有收益了，大家也按比例获利。虽然说这一制度有其合理性甚至必要性，而且在这一制度下，同样也诞生了许多脍炙人口的 IP，但是其缺点也是显而易见的。


**一方面，「制作委员会」并不是一个实体机构，它更像一个「代议机构」，也就是说，这个组织本身是没有统一意志的，它只是不同立场与意志博弈与争持的场所。**在「制作委员会」中的各方，都有着自身的诉求，都希望自身可以把持话语的走向，都希望自己所处的团体能以最小的代价获取最大的收益。如此一来，当项目开会运作时，环节内的良好运行，环节与环节之间的衔接都会大概率出现问题，用这种模式来整合资源，生产出「渣作」就成为大概率事件了。**另一方面，由于制作委员会中的各方都必须考虑「回本」问题，因此相比起开发「新用户」，保持「老用户」的粘着性，并将「老用户」的剩余价值开发殆尽才是他们更想做的事情。**日本式 IP 的老用户是谁呢？很显然就是所谓的「宅民」们。如此一来，日本式 IP 在打造时，往往就会不遗余力地塞入尽可能多的「废萌」或「废燃」元素，以进一步讨好宅民。对于那些本身就身处「宅圈」之外的受众，制作委员会中的各方只会「顺其自然」。毕竟，「讲好一个故事」乃至「创建好一个世界」都不能确定会带来更大收益，但是「将死宅们喜闻乐见的元素堆砌好」，至少能确定保本。某种意义上，这种做法颇为有悖于「社会责任」：当「宅民」们沉醉于「废萌」或「废燃」中时，他们就更不愿走入人群；而当社会大众看到满眼不知所云的「宅元素」，也会对 ACGN 文化圈敬而远之。如此一来，「宅民」们与「主流社会」就进一步脱节了。


然而，我们不能否认日本式 IP 的特点也会带来非常良性的结果，**那就是始终为最优秀的天才型作者留下了足够的空间。**试想一下，如果《周刊少年 JUMP》和漫威一样，存在一个「JUMP 漫画宇宙」，然后要求鸟山明、荒木飞吕彦、富坚义博、井上雄彦、尾田荣一郎、岸本齐史等作者必须按照「JUMP 创作法」进行创作，那将会是何等让人绝望的一派场景。正是因为没有做到彻底的标准化和同一化，我们看到了独一无二的《龙珠》、《JOJO 的奇妙冒险》、《HUNTER X HUNTER》、《灌篮高手》、《海贼王》和《火影忍者》。更有甚者，美国式 IP 虽然在整体思想层次上高于日本式 IP，搭建出了各种远比日本式 IP 要完善精巧的世界观体系，但是日本式 IP 在某些方面所展现的独创性和深度，却独一无二。日本式 IP 虽然无法给我们带来稳定的阅读、观看或游戏体验，但是那冷不丁出现的天才闪光，却总能让人激动不已。
