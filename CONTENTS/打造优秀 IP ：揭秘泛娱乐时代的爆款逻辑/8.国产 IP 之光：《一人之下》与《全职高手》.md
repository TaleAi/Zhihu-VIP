## 8.国产 IP 之光：《一人之下》与《全职高手》
虽然《全职高手》和《一人之下》讲了完全不同的故事，表达了完全不同的主题，但**它们其实都是「内容为王」的成功践行者**，也就是说，它们的成功也许有一定的运营因素在里面，但是最核心的竞争力，就在于作品本身内容实在过硬。那么，我们就结合「IP 构建内容篇」所说的几条内容构建法则，具体看看《全职高手》和《一人之下》的作品为何出色。


**·高度整合下的「二元世界」**


**在世界观搭建方面，《全职高手》的最突出优点在于「二元结构」；而《一人之下》的最突出优点则在于其核心概念的整合性和贯通性。**


在《全职高手》中，我们其实身处在一个「都市」与「游戏」犬牙交错的二元世界里，小说中的出场的每一个人物，既是「都市」中的「个人」，同时也是「游戏」中的「角色」。如此一来，《全职高手》就能同时满足读者对「现实世界合理性」的诉求和对「架构世界爽快感」的期盼了。某种意义上，《全职高手》这种处理方式和塔可夫斯基关于电影的「诗意逻辑」有暗合之处——老塔用电影交融着「梦境」与「现实」，而蝴蝶蓝则用荣耀交错了「游戏」和「生活」。


在《一人之下》里，漫画非常醒目地提出了「异人」这一群体以及「炁」这个概念，由这两者为基点，米二直接给出了这样一幅世界图景：古今中外，那些难以用现代科学解释的人或事，其实都不属于「日常世界」，而属于「异人世界」；而所谓的「异人」，归根结底就是一群拥有「炁」且善于用「炁」达到自身目标的人。乍一看去，这种世界设定有「无限流」的影子。但是，所谓的「无限流」，所做到的只是将科学、宗教、神话、传说、历史、现实、动漫、游戏拼凑成一个大杂烩，其中并无多少体系性可言。简而言之，「无限流」做的只是「罗列」而非「整合」。而「整合性」，恰好就是《一人之下》世界观的最明显特征。例如，在 TV 动画第二季的「罗天大醮篇」中，米二为我们展现了「龙虎山」、「全真教」、「武当派」、「东北萨满」这些真实存在的团体，以及「武侯门」、「贾家村」这样有历史渊源的架空组织，这些本来来自「不同世界观」的团体组织之所以可以「跨服聊天」，是因为它们都被整合进了「异人」这个范畴里；而龙虎山的道法、全家教的丹学、武当派的太极、东北萨满的召唤术、武侯门的奇门八卦以及贾家村的操物技，这些本来原理各异的术式，都被整合进了「炁」的框架内，被认为是「炁」的不同使用和表现方式。正是这种高度的「整合性」和「贯通性」，保证了作品走向的「稳定性」。


**·群像创作下的「独立人格」**


**在人物塑造方面，《全职高手》和《一人之下》可谓异曲同工：这两部作品中的人物，都有着各自「独立的人格」。**那么，「独立的人格」要如何体现呢？它的标准是这样的：**如果一部作品不是以主角的视角，而是以作品中其他人物的视角展开剧情，这部作品是否同样可以展开？如果可以，就说明这个作为「视角角色」的人物有着独立的人格，反之就是没有。**相对而言，这点在除了苏沐秋外，就没人「领便当」的《全职高手》中有更为直接的体现，所以，这里我们就主要以《全职高手》为例展开分析。


想象一下，在《全职高手》中，如果主角不是叶修而是韩文清，《全职高手》就是关于一个十年霸图，一如既往，书写「坚持」故事；如果主角是张佳乐，《全职高手》就是一个为了荣耀背负诅咒，却一直离冠军一步之遥，名为「悲情」的故事；如果主角是王杰希，《全职高手》就是一个为了战队褪尽繁华、然后在生涯晚期不遗余力提携后进的「传承」故事。这样的例子其实不胜枚举，而这其实也是为什么在 LOFTER 上，有那么多以《全职高手》为蓝本的同人作。正是因为《全职高手》里的每个人物都是「活生生」的，所以大家都愿意脑补属于他们各自的故事。当然，这一标准在《一人之下》中也同样适用。很显然，《一人之下》就算将主角换成了王也或者张灵玉，我们也能看到以「调和出世与入世」或「融合高洁与污秽」为主题的《一人之下》。事实上，要做到这点，对创作者的要求高到离谱：因为这就要求创作者不是首先想出一个故事，然后让不同的角色成为这个故事的零件；相反，**这要求创作者首先要理解到世界上本身就存在着形形色色的人，然后了解这许许多多人的真心与性情，最后把他们投放到自己设置好的世界观中，让他们自然地相遇。当他们相遇后，故事就会自然而然的发生。**


**·张弛有度的作品节奏**


在剧情架构方面，「群像剧」的创作方式只是《全职高手》和《一人之下》在剧情上成功的基础，而它们成功的关键，却是在于**他们在作品整体节奏的把握上做的十分到位。所谓「作品整体节奏」，指的就是「主线情节的展开」、「核心设定的披露」以及「人物心态的改变」这三者向读者呈现的程度与速度。如果说一部作品的整体节奏把握到位，实际上就是说其上述三方面内容的展现方式相互匹配，而且呈现的速度符合读者情绪流的走势。**


相比起主角一开始就是满级的《全职高手》，《一人之下》在这一点上体现得更加直接，所以这里我们主要就对《一人之下》进行分析。从作品开始，到「罗天大醮」开始之前，可算是《一人之下》的第一部。在这一部中，「宝儿姐的身世之谜与甲申之乱的关系」这个主线剧情，给出的只有宝儿姐自己的部分回忆；「异人世界争夺『八奇技』」这个核心设定，只展现了冰山一角和初级阶段；「张楚岚自身的成长」这个人物心态，也还只处于「大学生」层次。在这一部分，作为读者的我们，感觉是开始「进入节奏」。而到了「罗天大醮篇」，《一人之下》算是进入了第二部。在这一部中，和主线剧情有直接关联的人物陆续登场，主角也正是开始着手调查历史真相；八奇技也在「罗天大醮」的比武和及后的「全性夜袭龙虎山」中通过王也、风星潼、陆瑾等人之手展露真颜；张楚岚的心态也从「大学生」慢慢成长为了「战士」。到了这一部分，这种「主线剧情」、「核心设定」和「人物心态」协调展开的模式，在《一人之下》的后续剧情中一直延续着，而这就是为什么《一人之下》即便后来世界观越铺越大，出场人物越来越多，所做事情越来越复杂，我们都不会觉得混乱的原因。因为这一切都是循序渐进展开的，我们是步步为营，有条不紊地进入了米二笔下的「异人世界」。试想一下，如果在作品的最初期，当张楚岚还只有「大学生心态」的时候，就一下子要他进入主线剧情，或者要他面对核心设定的真相，不仅他要崩溃，连作为读者的我们，也难免会感到信息过度。


当然，《全职高手》在作品整体节奏上的把握也十分到位，在作品前中期，叶修要做的是「重返职业圈」，所以「荣耀」这个核心背景更多就呈现为「有趣的游戏」，叶修的心态也是「休息一年，然后回来」；而在作品中后期，叶修要做的是「夺冠」，所以「荣耀」就展现为「残酷的职业竞技」，叶修的心态就成了「将自己燃烧殆尽，留下纯白的灰」。通过上面的分析，我们看到了：**无论是蝴蝶蓝还是米二，能如此炉火纯青地驾驭作品整体节奏，是出于他们在讲故事时抱有的「节制」。**所谓「节制」，是对自身情绪的绝对控制以及对故事整体架构的绝对忠诚。什么时候该呈现什么信息，信息在呈现时该是个什么度，这些都要在创作者的严密操控之下。相反，有许多创作者一旦进入创作很顺手的状态、或者被自身创作的场景调动了情绪，就会忍不住在不该披露核心情报的地方将情报披露出来，又或者本不该说那么多，但却来了个滔滔不绝。所以说，在很多时候，创作其实是一件对自身要求高到近乎残忍的事。


以上，就是我对这两个 IP 进行的内容层面的分析，做一个简单总结就是，作为国产 IP 的优秀代表，《全职高手》和《一人之下》都贯彻了「内容为王」，这体现在：世界观架构方面，《全职高手》具有「二元结构」，《一人之下》则有很强的整合性和贯穿性；人物塑造方面，两者都塑造出了具有独立人格的人物；剧情架构方面，两者都十分到位地把握了作品的整体节奏。正因为有着这些优点，让这两个 IP 得以脱颖而出。
