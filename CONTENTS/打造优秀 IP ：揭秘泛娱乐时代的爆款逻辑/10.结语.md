## 10.结语
之前的课程中，我们审视了 IP 的方方面面：我们学习和了解了关于 IP 的内容与运营；分别介绍和分析了美国式 IP 和日本式 IP 的成功经验；并且对现阶段我们的 IP 产业进行了概括性了解。那么，知道了这些，对大家有什么用呢？或者直接问：学习关于 IP 的知识，意义何在呢？事实上，这是属于每一位朋友自己自己的问题，我能做的，只是向大家表达我的希望。


通过本专栏的学习，我希望大家在面对 IP 时，不再只有「含混的说辞」，而会有「清晰的思维」。如果说在学习这门课程前，一个 IP 对你而言，就只是一部小说、一部动漫或一部影视剧，你只是会单纯地点评情节好坏的话，那么学习了这门课程之后，你就应该带着「内容+运营」这个思维框架去审视 IP 了。例如，在内容层面，你除了可以品评情节，还可以分析它的世界观架构和角色塑造；在运营的层面，你可以去思考一部作可以用哪一种，或者哪几种媒介形态进行展现，如果作品内容需要进行跨媒介展现的话，又要注意哪些方面。这里因为时间有限，对于每个 IP 在不同媒介中进行展示，需要做哪些更多的工作，我们就不再一一展开。但总而言之，面对 IP，我希望大家不再仅停留在单一视角，而是从一开始，就知道它是个复合的事物。


另外，在这里，虽然我是为大家讲解 IP 的人，但在生活中，我与大家一样，都是 IP 的普通受众与爱好者。无论何时何地，我都希望未来能够看到更有趣的动漫作品和玩到更为有趣的游戏。但正如我们之前说到的，中国的 IP 行业还不成熟，还不完善。要让这个行业走向成熟和完善，不仅需要内容创作者，运营方的努力，我们作为普通受众也还有许多事情可以去做。比如，每个人在力所能及的情况下进行文化消费，通过正式渠道购买正版图书、正版游戏，去观看具有版权的影视作品；同时，由于互联网是一个有着很多声音的场域，参考他人意见，再选择是否观看、是否购买某一样文化产品，是一件很自然的事情。但是，正如一千个人眼中就有一千个哈姆雷特，所以，在对作品有所了解的前提下，也希望大家在做出判断时，不要被其他人传递出的审美倾向所束缚，而是培养起属于自己的审美意趣。


最后要说的是，生活中每一样事物背后，都会有很多意想不到的知识。今天我为大家讲述 IP 那些事儿，也希望能够抛砖引玉，让大家去思考我们生活中那些司空见惯的新兴事物背后，是否有着更为深刻的逻辑。


到这里，这个关于 IP 的故事就落下帷幕了，十分感谢各位朋友一路的陪伴和支持，希望在不久的将来，我们还能再次相见，给彼此分享各自的故事。
