## 简介
本课程由中山大学哲学系研究员肖力老师主讲，为你深入解读 IP 是怎样炼成的。本课程由 IP 发展超简史、构建基本法内容篇、国产 IP 之光等 8 个部分组成，将从文艺理论、传播学、产业经济学等 5 个学科交叉的角度进行阐述。


经常听人提到 IP ，但你是否真正知道 IP 的含义？


IP 发展有着怎样的前世今生？


国内外 IP 的发展情况如何？


一个优秀的 IP 不仅能给我们提供感官上的享受，更能够传递出一种情感共鸣与价值认同。


本课程就将深入为你解读一个优秀 IP 的形成。


本课程分为 IP 发展超简史、IP 构建基本法内容篇、国产 IP 之光等 8 个部分。中山大学哲学系研究员肖力老师将为你深入讲解 IP 的 3 个基本特征。


同时，从文艺理论、传播学、产业经济学、经营管理学和哲学 5 个学科交叉的角度，为你进一步讲述 IP 形成所必须的重点知识，并从国内和国外 2 个层面深入解读。


通过本课程的学习，你将了解 IP 产业的现状，创作和运营法则，以及产业模式等多个方面的知识。


无论你是 IP 创作者还是爱好者，本课程都将帮助你开启更多的知识探索。
