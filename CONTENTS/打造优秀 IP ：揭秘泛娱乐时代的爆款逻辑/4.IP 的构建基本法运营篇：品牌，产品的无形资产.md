## 4.IP 的构建基本法运营篇：品牌，产品的无形资产
IP 运营的基本战略，我们可以用一句话来概括，那就是：**构建整体品牌（即 IP 本身）比打造单个产品更为重要。**实际上，我们可以从「商业逻辑」和「哲学逻辑」两个层面理解这种「品牌」大于「产品」的思路。


首先，在「商业逻辑」中，通过类比「耐克」、「阿迪达斯」等运动品牌，我们可以很容易地认清以下两点：


第一，当我们在谈到品牌的时候，谈及的是标贴有这项标签的所有产品，而不仅仅是某一项或者某一类产品。产品在物理上具有确定性，可供发挥的空间有限；但品牌却具有衍生性，可以将不同事物涵括其中。


第二，品牌是产品所具有的无形资产，具有延展性，会随着传播而增益；产品具有消耗性，会随着时间的流逝而损耗。比如《蝙蝠侠：黑暗骑士》是一部十分著名的作品，但随着时间，这部电影中的内容与台词同样会被大多数人所遗忘，但是「蝙蝠侠」这个帅炸天的黑暗骑士形象，却已经深入人心。而只要人们依旧有着对「蝙蝠侠」的执念，那么关于蝙蝠侠的「作品」就会一部部地出现。这就是品牌的力量。成功的品牌直接可以让人们倾向于对产品作出正面评价，当多种产品在同一市场上竞争的时候，品牌所属的产品得以脱颖而出的几率会远高于其它产品。


其次，在「哲学逻辑」中，「品牌」大于「产品」其实是一件极其顺理成章的事，因为这一思路事实上就是「整体先于局部」这一命题在现象界的反映。这在「结构主义」中有所提到，我们在单集页面中有具体展开，感兴趣的朋友可以在单集页面中对此进行了解。


**·如何打造一个长期稳定的品牌**


在明确了「品牌」大于「产品」这一基本战略后，我们需要知道的就是需要用到怎样的具体战术去实现这一战略目标了。对此，我们可以将「如何打造一个能带来长期稳定效益的品牌」这个大问题分解成四个小问题。这四个小问题分别是：**第一，制作方会基于怎样立场决定 IP 的性质；第二，制作方会使用怎样的模型调配 IP 的资源；第三，制作方会使用怎样的方式进行 IP 的运营和传播；第四，制作方会使用怎样的活动维持 IP 的影响。**而这四个问题的答案，实际上就是 IP 商业运营中会用到的具体战术了。


**受众心理**


**制作方用于决定 IP 性质的基本招式，可归结为「直击痛点」。**对于 IP 建构而言，要抓的痛点是心理层面的，因此，对大众心理的研究、分析、把握和预判，就成了 IP 打造方必须要考虑的问题。例如：对相当一部分生活于社会底层的年轻人而言，「渴望改变命运」就是他们的心理诉求。那么，相应的 IP 打造方要做的，就是在这个痛点上做文章了。于是，我们看到了起点网上一部又一部的「逆袭流」爽文。那些年轻人将《斗罗大陆》中萧炎的著名台词「三十年河东，三十年河西，莫欺少年穷！」作为座右铭，日复一日地沉浸在小说带来的「逆袭」快感中。《斗罗大陆》看完了，「起点」就为他们提供《帝尊》、提供《天界打工皇帝》以及其它许多同类作品。同样，与男性逆袭文相对的是女性的玛丽苏文，满足了女性对于配偶以及情感的各种幻想。我们难以断言有多少这样的年轻人从这些小说和文章中获得了改变自身命运的力量，但却几乎可以肯定，类似日本轻小说《灰与幻想的格林姆迦尔》这种「真实细腻的异世界残酷物语」无法被「起点」捧为爆款。毕竟生活已经如此沉重，「起点」又怎么忍心再给年轻人们找不自在呢？


**跨媒介叙事**


**制作方用于调配打造 IP 资源的理论框架，就是跨媒介叙事理论。**最让原著迷们爱恨交织的事情叫做「你喜欢的作品要 xx 化」了，这个 xx 可以被替换为动画、真人、翻拍等等。其中，不少都是涉及媒介的变化，比如《青春猪头少年不会梦到兔女郎学姐》这部在去年大热的 TV 动画，其最原始形式是轻小说。而在今年暑期，「青豚」还将走上大银幕。这种不同媒介的展现方式，在 IP 的运营上并不少见。然而，并不是每一个轻小说作品或漫画作品的动画化或者真人化都能获得成功，这是因为，用于运营 IP 的资源是有限的，但每一种媒介却都拥有自己的特点与属性，所以要将有限资源进行最优化配置，就要找到每一种媒体平台的优点与缺点。提出「跨媒介叙事理论」的詹金斯曾经说过「其理想的形式是每一种媒体出色地各司其职，各尽其职。」比如说，电影是最具视觉冲击力的体验之一，如果要进行电影化的跨媒介展示，那么增加在视觉上的投入是必要的，即使再好的故事，5 毛特效的存在也会让 IP 的口碑大打折扣；又例如，如果要将动漫类作品游戏化，就必须考虑到游戏的「沉浸式娱乐体验」，因此，当把一个原本成熟的故事进行游戏化改造时，在不破坏故事与人物的情况下，通过将资源倾斜在「玩法设计」上，增强用户的参与体验，就十分重要了。


**品牌联动**


**制作方在进行 IP 的运营和传播时，会经常采用名为「联动」的手段，**而「联动」则又能被分为「品牌内各类型产品的联动」以及「跨 IP 的合作性联动」。一方面，所谓「品牌内各类型产品的联动」，指的是文化产品与实体商品的同步推出。在这方面，像《变形金刚》乃至《高达》系列这样的「萝卜片」就是很好的例证。我们可以将在电影院和电视台播放的节目视为厂商卖玩具宣传片，也可以将在实体商店中售卖的各种机器人玩具视为相关影视作品的附属周边。实际上，文化作品与实体商品无分主次，形成的是一种共同进退的联动关系。在这种联动关系中，真正得到强化的，就是这个品牌，即 IP 本身。


另一方面，所谓「跨 IP 的合作性联动」，指的是两个或两个以上的 IP 中的要素，在某些产品中共同出场，以求达到「相互借势」的目标。这种跨 IP 的联动，可算是日本各轻小说文库的惯用手法。这些轻小说文库会联合大型实体书店举行不同的「特典活动」，隶属于这些文库的轻小说作家，则要为这些活动写作相应的「联动小说」。举例而言，由角川文库出版的《为美好的世界献上祝福》和《RE0：从零开始的异世界生活》的联动是「相互借势」的一个成功案例。这两个轻小说 IP 本身在日本已经具有了相当高的人气。但是，由于两部作品的阅读群体并不完全重合，所以，为了能进一步「收割」彼此的读者，两部作品进行了「联动」，创作了一个让双方角色共同登场的故事。于是在这里，双方的读者都明白了一个道理：原来蓝色除了代表真爱，也能代表智障，反之亦然。如此一来，一加一是否能大于二虽未可知，但至少能起到等于二的效果。


**粉丝经济**


**制作方在维持 IP 的影响时，与 IP 粉丝的互动就成为了重中之重。**在 IP 的发展史那节课中，我们就已经知道了 IP 的成功与否，粉丝会在其中扮演举足轻重的角色。基于对 IP 的喜爱，粉丝会让渡部分思考主权，单向性信息传递具有极强的向心性和非理性因素，因而粉丝往往会产生一种集邮式的消费，为 IP 带来经营性的收入。


费斯克说过，「对于文化工业而言，粉丝已经成为一个额外的市场，他们不仅经常大量购买衍生产品，还提供宝贵而又免费的有关市场的偏好和反馈。」这个道理，制作方又怎么会不知道，因此，制作方会努力地将粉丝的热情以及行动纳入到自身的 IP 打造中，允许粉丝画同人漫画，写同人小说，或者进行周边创作，然后利用群体间自由互动以及和群体互动的方式让群体的身份得到不断的强化。


其中，我所热爱的型月世界的 FGO 就是如此运营的。在 FGO 中，玩家们可以按照自己的意愿运用 FATE 世界的元素，召唤和打造属于自己的从者，为了能打造出心仪的从者，玩家们投入了巨大的时间、精力乃至金钱。今年 1 月，FATE 中的 HF 剧场版登录中国大陆，虽然我不能算一名重症月球人，但是也积极参与了 FGO 在 B 站举行的抽奖活动，看自己是否能抽到一张电影票，当然，结果自然是光荣地成为了分母的一份子。参与抽奖活动，本身是我个人出于自己的情感做出的一个行为，而当我身处在一个全部都是粉丝抽票，大家有一个共同的想要观看电影的诉求的时候，我个人情感中对于 FATE 的喜爱就更为明显了。这也就是「互动仪式链」理论，指的是人的具体行动就是追求情感能量，在互动仪式市场中进行资源交换，并不断强化群体成员身份的过程，比如粉丝群，微博超级话题，这些运营手段都是常见的利用互动仪式链来达到巩固粉丝群体的方式。

结构主义


社会学流派，侧重对结构（交互关系）的认识，不甚讲求对本质的了解，大前提是科学与科学之间或多或少互通有无，提倡一种整体的科学，要透过表面的现象，寻求底层的关系，以期获得放诸四海皆准的结构。结构主义是二十世纪下半叶分析研究语言、文化与社会的流行方法。


结构主义者认为各种文化现象、社会现象乃至语言现象，它们其实是一套套系统，或者说一个个「结构」。每一个「结构」，都必须从整体上进行把握，是结构的机制建构出了结构的各个「要素」，而非各个要素根据某种机制组合成一个整体。同时，机制往往是隐匿的。
