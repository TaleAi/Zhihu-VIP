## 1.我们生活在 IP 的语境中
当我们走进电影院，会发现有些电影的原著是个漫画 IP；打开视频网站看电视剧，电视剧又可能是由热门的网络小说 IP 改编而成的，就连我们和朋友聊天，表示哈哈哈哈哈的时候，都会调出一个阿狸的表情包……仔细观察生活，不难发现，我们现在所有的娱乐、休闲似乎都离不开 IP。


事实上，我们正是生活在一个 IP 的语境之中，互联网下的文化传播正在悄悄地改变着我们的表达与对事物的认识，比如我们说「吃鸡」的时候不再是代表着饮食，而是一项游戏，我们说「真香」的时候，也不再是表达一种感官享受，而是打脸预警，相似的案例数不胜数，IP 已经成为了我们生活中的一部分。


**·IP 是可多维度开发的文化品牌**


IP 是英文 Intellectual Property 的缩写，翻译过来就是「知识产权」，是个法律概念。在讨论的流行文化产业范畴中，IP 可以被理解为一个「可进行多维度开发的文化品牌」，并且具备三个基本特征：第一，文艺作品的形态多样性；第二，运营手段的层次复合性；第三，具有社会生活中的影响深广性。


一个优秀的 IP 不仅提供感官上的享受，更能够传递出一种情感共鸣与价值认同， 同时它也影响着我们现在所处的社会，建构着我们的文化生活。


**·探索作品艺术与经营哲学**


研究 IP 是一个人文社科跨学科的尝试，它汇集了文化、消费、商业以及传播等多个要素。在这个专栏中，我们会从文艺理论、传播学、产业经济学、经营管理学、哲学等学科交叉的角度去对 IP 进行解读。


在接下来的课程中，我首先会和大家谈谈 IP 的前世今生，也就是 IP 是怎么来的，再和大家聊聊成功 IP 不可或缺的创作和运营法则，同时，接下来我也将以美国和日本两个大家熟悉的 IP 大国为例，分析成熟的 IP 产业的不同模式。


除此之外，我们也会将目光投向国内，谈谈目前国内的 IP 产业现状，为大家解读市场上公认的精品 IP，以及聆听来自 IP 工作者的声音。


无论你是对 IP 的背后逻辑充满好奇心的爱好者，还是一个想要拥有 IP 作品的创作者，希望这些有关作品艺术以及经营的哲学，可以帮助你开启更多的知识探索。本专栏共 10 节课程，愿与大家共同度过一段欢乐时光。
