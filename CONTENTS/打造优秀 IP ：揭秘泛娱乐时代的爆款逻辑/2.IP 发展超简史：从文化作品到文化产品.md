## 2.IP 发展超简史：从文化作品到文化产品
我们要知道 IP 并不是一个具体的「事物」，而是一个被建构出来的「概念」。讲述 「概念」的历史，其实是了解构建出这个概念的「背景」与「法则」，是如何成为所处生活的主流思考方式的。也就是说， IP 的发展史并其实是社会背景与人类思维的变化历程。 「社会背景」与「人类思维」的变化，塑造了并继续塑造着，定义了并将一直定义着 IP。


我们就拿一个大家非常熟悉的人物来举例：比如马克思，对于广大的 50 后、60 后甚至 70 后的中国人而言，「马克思」毫无疑问应该是一个与一切商业行为绝缘的政治符号。然而，在 2019 年的一部国产 TV 动画《领风者》中，马克思这一人物被二次元化。动画播出之后，奠基于其上的商业操作，如周边产品的销售，就会立即随之而来。也就是说，即便是「马克思」，也成为了一个 IP。为什么过去单纯的政治符号会在如今被运作成二次元商品呢？为什么那些难以想象会成为 IP 的事物也会 IP 化呢？很显然，那只能是因为「社会背景」与「人类思维」发生了变化。


所以，当我们想把握 IP 的历史时，我们要做的，是去了解「社会背景」与「人类思维」是怎样演变成我们今天这个样子的，以及它们的演变，是如何塑造和定义着 IP 的。


**·资本主义开启文化作品产品化**


IP 如今的概念离不开**资本主义制度奠定的社会背景基调。**随着资产阶级革命在欧洲取得胜利，以及西方国家在世界体系中愈发占据主导地位，资本主义制度已经成为了现代社会运行的「源代码」。**伴随着资本主义制度的传播，文艺作品也开始了漫长的「产品化过程」**：一开始，我们看到的只是中世纪时期活跃于佛罗伦萨的商业出版行为，但是随着印刷技术的发展以及出版行业的兴盛，到现如今，能否被「商业化」已经成为了一部作品是否得到社会认可的重要标准。超凡绝俗如大刘刘慈欣，都在为科幻作品可以「商业电影化」而奔走，销量与票房也成为判断作品是否优质的重要指标。可见，**在资本主义制度下，文艺作品需要作为一种「产品」来运作，「量化」和「逐利」不可避免地成为了「文艺产品」的两条运作铁则**。


为了更好地去理解「量化」和「逐利」对文艺作品的影响，我们来聊一聊国作家洛夫克拉夫特的《克苏鲁神话》。在洛夫克拉夫特那里，「克苏鲁神话」是一部探究深渊与未知的作品，在作品中，他力图展现的是一个超越人智的神话体系。然而，正是由于他想表达的事物偏离「主流」，在他有生之年，克苏鲁神话都只是一个小圈子的读物。


真正让克苏鲁神话走向大众的，是洛夫克拉夫特的追随者德雷斯。在洛夫克拉夫特去世之后，德雷斯所做的事就是让克苏鲁神话与当时的大众文化「和解」，以削弱这个神话体系的深渊感与未知性为代价，他将善恶二元论融入其中，将克苏鲁神话中的那些神秘、窒息、绝望的诡异氛围一扫而空，从而构筑出了出版商和读者都更乐于接受的完备架构与合理解释，然后再在此基础上写出新的故事、刻画新的形象。


通过这一改变，克苏鲁神话终于能「卖出去」甚至「大卖」，它的历史地位，从而被逐渐认可了，也有了如今我们所认识的「克苏鲁神话」IP。然而,稍显讽刺的，促使它的地位被承认的因素，显然不是作品本身的文艺性，而是冷冰冰的「发行数」、「购买量」以及「利润」。可见，在资本主义制度下，纯粹意义上的「文艺作品」已经难以生存，占据主流的，变成了「文化产品」。而**「文化产品」的存在，正是 IP 得以被构建的基础。**


**·互联网助文化产品去中心化**


互联网的出现以及流行也是让 IP 的概念发展至今的重要原因。互联网完全动摇了人类在过往时代所形成的许多认识，**奠基于互联网之上的互联网思维，让我们的思考方式完全发生了变化，这里可以用福柯在《词与物》中提到的知识型去解释，对于这个的解释具体可以在单集页面中进行查看。**


什么是互联网思维呢？，首先，互联网是一个跨越媒介、使得多种媒介信息可以共生，而且所有信息会以无限种方式交织与增值的「泛信息」平台。在这个「泛信息」平台上诞生的思维，可以被认为是一种「去中心化」的思维方式。因为，在「无限」的视野下，「中心」与「边缘」这些概念，也失去了意义。因此在这种思维下，无所谓权威、无所谓绝对，有的只是不断的诠释以及重构。


当「文化产品」被「去中心化」后，它就不再是一个「可被定义的事物」，而是一个「可供诠释的概念」，如此一来，每个人都能成为这个概念的创作者、传播者和接收者。打个比方，「去中心化」后的「文化产品」就像是一个舞台，所有的人，包括作者、读者和经营者，都能在这个舞台上跳自己的舞蹈。在这里，「舞台」不是 IP、每个人的「舞蹈」也不是 IP，只有结合了它们两者的这场「舞会」才是。


对于这点，最明显的例证莫过于日本游戏「东方 PROJECT」，虽然游戏制作人 ZUN 被认为是「创世神」，并奠定了「东方」的基础，然而东方之所以能被认为是一个体量巨大的 IP，依靠的却是许多高质量的二次创作。在这个意义上说，IP 的真正创作者早已经不能简单归结于文艺作品的作者本人，也不说是文化商品的贩卖者，所有被牵涉在这场塑造中的人，每个人都可以被认为是 IP 的创作者。也就是说，**当「文化产品」被用使用「互联网思维」去进行理解和建构之后，它的多维度开发才得以实现，才有了 IP 的诞生，这里的多维度，不仅仅是指媒介形态上，更是指对于传播学中对于创作者、接收者定义的颠覆。所以说，互联网的出现，以及互联网思维是我们现在认知的 IP 概念的重要由来。**


我们来总结一下这一讲中所提到的内容：首先，我们知道只有了解了与 IP 诞生、发展相关联的「社会背景」和「人类思维」的演变史，才算真正了解了 IP 的历史。接着，我们又分别知道了 IP 得以诞生和发展的「社会背景」与「人类思维」分别是「资本主义制度」和「互联网思维」。简单来说，资本主义制度是 IP 的基础，由于这种制度的存在，IP 才可能从纯粹的「文本」变为可被运作的「产品」；而互联网思维是 IP 观念得以以几何级数发展壮大的关键，由于有这种思维的存在，IP 这种「产品」的流通方式、传播方式与再生产方式有了颠覆性的革新，跨媒介使受众对于叙事的理解有了勾连的能力，才会让 IP 成为今天我们所见的模样。

知识型


话语构成或话语实践受制于「一组匿名的历史规则」，这些规则「在某一时期的时空中通常是确定的，而且对与一定的社会、经济、地理和语言领域来说，是陈述功能运作的条件」。在《词与物》中，福柯把这种规则系统称为「知识型」。


「知识型」是一个时期所有知识生产、辩护、传播与运用的标准，是那个时期所有知识分子的知识问题。
