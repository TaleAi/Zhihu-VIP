## 简介
看起来都是笑点，实际上全是考点！半小时漫画带你熟读唐诗三百首背后的故事，准确理解唐诗的原意、深意和诗意。


李白喊出「安能摧眉折腰事权贵，使我不得开心颜」是因为朝廷总不让他当大官，内心憋屈万众期待的诗圣；


杜甫，前半生是「会当凌绝顶，一览众山小」的狂放青年，后半生是「国破山河在，城春草木深」的愁苦大叔；


王维的「空山不见人，但闻人语响」能做到「诗中有画，画中有诗」，多少与他的豪华别墅和带薪休假有关……


现在，半小时漫画带你熟读唐诗三百首背后的故事，准确理解唐诗的原意、深意和诗意。边塞诗达人王之涣、唐诗界段子手柳宗元、精分诗人白居易……我们熟悉又陌生的著名诗人们都将在本书中悉数登场，抛开复杂的注释，丢掉公式化的赏析，用幽默的方式讲述严肃的唐诗知识，让你在哈哈大笑中重新爱上唐诗。
