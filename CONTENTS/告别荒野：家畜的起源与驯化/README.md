## 简介
家畜是推动人类文明进步的重要力量。


数万年前，许多野生动物都开始关注人类，它们有的跟随人类的足迹，最终走进人类的世界，用自己的力量改变了人类的进化方向，甚至影响人类的历史进程。


同时也改变了自身的命运，有的成为人类的牺牲品，有的则成为人类的朋友和助手，还有的则再次浪迹荒野。


毫无疑问，当动物离开荒野之后，成为「家畜」，它们的一生都取决于人类。


有的家畜以肉体为代价，有的以力气为代价，有的以自由为代价。


但它们同样换取了高额的生殖回报，它们的命运同人类的命运息息相关、难解难分。


* 人驯化了狗，还是狗驯化了人？
* 猫在人类家庭反客为主，天理何在？
* 羊除了被吃，难道就没有成为宠物的命？
* 牛只吃草就能长出健壮的肌肉，你为什么不行？
* 猪到底是战神，还是战五渣？

作者史钧曾出版有《疯狂人类进化史》等进化论科普作品。本专栏将带你去探索家畜与人类之间的进化逻辑，了解两者相爱相杀的陈年往事。
