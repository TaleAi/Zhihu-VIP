# 穿越到《灌篮高手》：陵南的逆袭

### 作者：樱木不哭

### 分类：现代言情

### 状态：已完结，共 25 节

### 存档日期：2022-06-16

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1274698258321575936


## 简介
我是《灌篮高手》的骨灰级粉丝，结果醒来发现自己居然穿越到了陵南队队员相田彦一身上！穿越都不给我一个流川枫或者樱木花道这样的角色吗？？？没有主角光环也就算了，身高怎么还生生矮了 5 公分！！！


「卧槽！这是哪儿？」


林峰一觉醒来，发现自己居然靠在电车的座位上睡着了。脑子里检索了一圈实在想不起来自己睡前干了些啥，但绝对不是在电车里啊！这会儿乘客不多，林峰很快发现无论是车厢内的标语、还是窗外建筑上的文字，竟然都是日文！


「我这是穿越了？」


林峰狠狠的掐了掐自己的大腿，清晰的痛感让他确定自己不是在做梦。坐在他身旁的男子正在翻阅一本体育杂志，林峰不经意歪头瞥了一眼。


《陵南崛起の巨人，鱼住纯》——大写加粗的文章标题非常显眼。


触电一样的感觉瞬间袭遍全身，林峰整个身体「噌」地一下从座位上弹起，他一把抓过杂志，满眼放光地盯着书页上的文字，「陵南高中拥有全县第一高度、202 公分的中锋鱼住纯，去年的失利激发了他的斗志，苦练一年之后……」


「卧槽！卧槽！我特么穿越到《灌篮高手》的世界里了？！」


前世的林峰就是一个铁杆高中生篮球迷，《灌篮高手》这部热血篮球漫画就是他的入坑引路人，反反复复看了有几十遍。只可惜现实中的他身高只有 1 米 7，也没什么运动天赋，在学校的篮球场上他根本就是个小透明。所以他常常在梦里幻想跟漫画中的球员们一起比赛、竞争的场景，没想到现在居然梦想成真，自己真的来到这个世界！


「这波……这波还不是成为高富帅，化身流川枫，走上人生巅峰吗？！」


「八嘎！」


身后一声暴喝打断了林峰的 YY，原来是邻座的男子，他劈手夺回了自己的杂志。


「谷咩纳塞、谷咩纳塞……」


林峰脸上一阵红一阵白，一边笑着道歉一边鞠躬后退。回到座位上坐定，心里暗惊：「我这微笑鞠躬道歉三连很熟练嘛，看车厢内的日语文字也没有障碍，应该完全适应了穿越后的身体，可怎么就是想不起来自己是谁呢？流川枫、仙道？还是别的谁？」


强忍内心的激动，林峰开始在身边的背包里翻找，希望能找到个身份证之类的东西搞清楚自己到底是谁。


「嚯！这小子挺有钱啊。」


林峰拉开书包的拉链，就发现里面放着一个小巧的精致软皮包，打开一看赫然是一部便携式数码摄像机。就算是日本这样的发达国家，在《灌篮高手》成书的 90 年代，一般老百姓家里能有这种设备的也是少数。


「这么有钱……难道真是流川枫？」


虽然《灌篮高手》中并没有详细讲述主要人物的家庭经济条件，但是在一些画面中男二号流川枫的鞋子、衣服、自行车、耳机都价格不菲，他的家庭经济条件不说大富大贵至少也是很优渥的，林峰有此联想也很合理。


这时，林峰又从书包内侧夹层里找出来一个蓝色塑料皮小本本，翻过来，本子的正面赫然印着两个金色大字——「陵南」！


「这是……陵南的学生证？」


发现自己是陵南的学生让林峰有一丝失望，不过很快他又重新兴奋起来，因为陵南的王牌球员仙道彰按原著的设定，实力更在男二号流川枫之上！


想到这里，林峰激动得手指都开始发抖了，他颤颤巍巍地翻开学生证的封面，满怀期待地在扉页上寻找着他内心期待的名字。


「啊——！」
