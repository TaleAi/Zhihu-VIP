## 简介
本课程，以帝王教科书《资治通鉴》为阅读蓝本，带你揭开历史的重重迷雾，看看司马光到底在教导宋朝帝王们，什么只可意会不可言传的道理，将枯燥历史背后的波谲云诡，掰开揉碎讲给你听。


你是否觉得，影视剧中生动的人物角色，到了史书上就变得枯燥乏味？


你是否曾想看宫斗权谋，人心叵测，却发现真实的历史清汤寡水？


你是否想要学习权谋智慧，纵横捭阖，翻遍史书却一无所获？


这或许是因为，上千年来，中国人含蓄内敛的传统思想，让史官们习惯了看破不说破，也习惯了故弄玄虚。


本课程，以帝王教科书《资治通鉴》为阅读蓝本，带你揭开历史的重重迷雾，看看司马光到底在教导宋朝帝王们，什么只可意会不可言传的道理，也带你看一看，史书上那看似寡淡无味的话语背后，隐藏着怎样的波谲云诡。


人心、欲望、阴谋、智慧、恩怨交织……拨开历史的迷雾，精彩才刚刚开始。
