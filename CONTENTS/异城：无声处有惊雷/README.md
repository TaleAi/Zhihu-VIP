## 简介
这是一个惊悚类的短篇合集专栏。每个故事发生在一个难测的「怪异之处」。


这世上有座城。


在你踏入这个地方的那一刻，就进入了一个诡谲之局中。


而最深的那座城，在人心之中。


「猴村」：市里突然出现一种怪异的传染病，陈梦冒死进入医院调查，却被上级卡住，还把她从热门的社会组调到了冷门的风俗组。她的不得志被同组的吴诗文看到，邀请她去她家乡散心并且可以拍摄选题素材。但进到这个据说猴子和人和平共处的朴素村落后，陈梦才发现自己入了一张巨大而古老的网中。


「蜕皮」：我们屯上来了一对外地母子，儿子是个憨厚的中年人，母子俩的关系让人看不清，似乎反了过来。我不小心看了那个母亲的眼睛，一双瞳孔竖起，布满褐纹，偶尔会刺出一线幽邃的金光。我吓了一跳，再看却没有异样。他们来了没多久，我们屯上却开始出事了。


「温泉镇」：苏瑾和男友去温泉小镇度假，放松一晚后，男友却失踪了。小镇不大，苏瑾很快就逛完却没有找到人，但只有一个地方她不被允许进入，就是镇中心的大楼。苏瑾还发现这个镇有个很奇怪的地方，她很少看见女孩。她从头走到尾，只见到一个面包店的学徒女孩，却根本不和她对上目光……
