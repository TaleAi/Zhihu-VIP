## 简介
人间喧嚣，爱意缠绵，勇敢女孩，无畏向前。


**《鉴茶记》**


继妹趁我不在，勾搭我的奶狗男友。


「姐姐这么老，不怕她先入土？弟弟，跟我在一起吧。」


小奶狗俊脸涨得通红，没有反驳。


我暗叹姐弟恋果然不靠谱。


谁知下一秒，他扯着向来软软的声音吼道：


「老子比她大！再多嘴，老子让你先入土！」


  



**《穿成黑料小花后我靠演技逆风翻盘》**


今穿古后，我又穿回现代。


身份是一个黑红小花，正在拍后宫戏。


导演对我嗤之以鼻，同行奚落排挤，全网都等着看我笑话。


我扶正发簪，掩住笑意。


那什么……


我刚在古代做了三十年的后妃哎。
