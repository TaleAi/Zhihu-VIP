## 简介
瑰琦绚丽的想象、曲折离奇的故事、丰满生动的人物形象，构成了唐传奇五光十色的奇幻世界。唐传奇中有虬髯客远遁中原的洒脱背影，女侠聂隐娘神出鬼没的瑰丽江湖，柳毅传书搭救小龙女的千古奇遇，霍小玉临终别离的悲愤诅咒，惊醒太守的南柯一梦，怅然书生荣华的一生……让我们打开《唐传奇》背后隐藏的时代秘密、剖析曲折离奇故事背后的含义、体悟唐传奇里的中国文化密码。


你也许会好奇芥川龙之介笔下杜子春的原型出处；


你似乎还钟情于舒淇在电影《聂隐娘》中饰演的那个带着野性的、风一样的女人；


你可能仍忘不了王小波《红拂夜奔》中那一句「李郎，谈谈你的长安城，我愿意听。」；


......


唐传奇，唐代流行的文言短篇小说。它远继神话传说和史传文学，近承魏晋南北朝志怪和志人小说，发展成为一种以史传笔法写奇闻异事的小说体式。瑰琦绚丽的想象、曲折离奇的故事、丰满生动的人物形象，构成了其五光十色的奇幻世界，成为诸多影视文学作品的灵感源头，一如陈凯歌《妖猫传》里那个盛唐的「云想衣裳花想容」。


同样是志怪小说，广为人知的《聊斋》所讲述的，是细腻的痴男怨女的故事，惆怅苦楚大于酣畅淋漓。现在换种口味，一起感受来自唐传奇的豪侠豁达，细品一份粗粝结实的烟火气。


在这个专辑中，韩教授将用他真挚磁性的嗓音，从爱情、文学、历史、侠义、神怪等多个角度来解读这些传奇故事。还原唐人生活的点滴，带你一起走进那个传奇隐秘的历史花园，追寻那个时代留下的美丽足印，畅想一千年前大唐的动人的情怀。
