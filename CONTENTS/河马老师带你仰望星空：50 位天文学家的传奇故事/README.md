## 简介
本课程穿越 6000 年的天文史，与 50 位天文学史上重要人物进行心与心的交流，还原出一个真实的天文学家群体，追随他们的脚步，感受他们曾征服过的星辰大海，做一个仰望星空的「前行者」。



自古以来，人类总是对天空有着无尽的好奇，对星空有着无穷的想象，天文学是可以让每一个人感兴趣的学科，再也没有任何一门别的学科，可是唤起这么神秘的想象，可以带来这么深奥的知识，可是蕴含这么丰富的人类探索自然的历史，可以串联起数不清的成功与失败的故事。


天文学的发展历史，不仅仅是干巴巴的科学公式的书写，更是人类逐渐站立起来认识世界的艰难过程。我们对宇宙的每一步认识，都是靠一个又一个活生生的人，去探索，去探索，再去探索，领悟到的。


从古代天文学家古巴比伦 Nabu-ri-mannu、毕达哥拉斯、柏拉图到天文学革命和近代天文学家哥白尼、伽利略、高斯，再到现代天文学家哈勃、爱丁顿、米歇尔小姐……


他们中有宫廷阴谋的野心家，也有忠诚的朋友；有最大程度的个人牺牲，也有陷害他人的叛徒；有健康长寿的学术泰斗，也有英年早逝的神童；有默默无闻的劳动者，也有世界明星般的领路人，他们不完美，但他们都有一个响亮的名字「仰望者」。人类不敢想象没有他们的世界。
