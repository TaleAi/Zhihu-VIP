## 简介
本专栏主要用风趣幽默的语言为你讲述古代那些冷知识，比如古人上班的时候去厕所需要打报告吗？古人辞职开民宿到底赚不赚钱？古代人会逃离 996 吗？让现实照进古代，看看曾经的前辈们是如何对付这些小 case 的！


**古代也有驾照考试？**


秦朝《除吏律》规定，所有驾车者要先经过考核，如果你四次还没过关，就会被取消资格。并且，还要罚四年徭役……


就是说，如果你考四次还没考过，那对不起，你得给国家当四年的免费苦力了。


**古代上朝时想上厕所怎么办？**


曾经明朝就有一位兄弟，在朝会上没憋住，尿了出来，皇帝很生气，你今天敢在朝堂上撒尿，明天就敢在龙椅上拉屎。你拿我朝堂当公共厕所呢？命人狠狠打了他一顿板子还不解气，又将他的官职一撸到底，直接从京官给贬到了地方。


**古代的「床上之狗」**


他爹刚去世的时候，刘建看到自己的那些小妈们年纪轻轻就守寡，于心不忍，大发慈悲，只好委屈自己，帮助老爹照顾那些小妈们。让他爹在天之灵得到了安慰。


最牛的是，这小子还比较有科学眼光，喜欢研究，热衷于做实验。有一次，他突然琢磨着，人如果跟狗合作的话，能生个啥玩意儿出来呢？我是不是就是这么出来的？


**古代是如何求雨的？**


由于这套仪式太过麻烦，所以我们就诞生了龙王庙，专门用来求雨。摆上祭祀用品开始求雨，如果下雨，那咱们合作愉快；假如不下雨，那就是给你脸你不要脸，特么的吃了我的给我吐出来，砸毁龙王庙……


**古代的冰箱**


有一个问题，因为祭祀一般都会持续好几天，毕竟咱也不知道老祖宗们啥时候赴宴，万一在这期间，肉都馊了，招一堆苍蝇，老祖宗们过来一瞅，太不像话，你这不成心恶心人么？


没准儿老祖宗一生气，就得把你带走。


**古代的商人**


小少妇见到老白，也仿佛找到了知己，对老白诉苦：「哎呀妈呀，你是不知道啊，我这身世那可叫一个悲惨，真的，听者无不伤心，闻者莫不落泪。」


她说：「你说，就我这长相，不说是天仙一样吧，那也是一个典型的小仙女。按说像咱这样的，找个啥样的没有啊？偏偏当初瞎了眼，嫁给一个商人。就我家那老头子，整天就知道挣钱。唉，我好空虚，我好寂寞……」
