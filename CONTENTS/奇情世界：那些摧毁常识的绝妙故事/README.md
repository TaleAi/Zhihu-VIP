## 简介
在世界的另一面，永远都有更奇妙的故事。


欢迎来到世界的背面。


这里是「故事会」。阴影中，讲故事的老人缓缓开口，奇诡命案，民间传说，阴阳怪谈，从他口中桩桩件件，缓缓道来。一人，一室，一世界。从古至今，故事百听不厌，你渐渐入迷。「鬼有什么可怕，带你看看人心」，老人笑道。


四兄弟出卖了做皮影的师父，师父死前咬牙切齿诅咒：要把四个孽徒做成活皮影。起初谁都没当回事，直到老大惨死桥边，死无全尸，手臂被人取走……接二连三的命案，让人不得不信，诅咒应验了……


农民夫妇守着一棵看似「长眼睛」的怪树，偏偏有个怪人来买树叶。一张叶子一角钱。这笔飞来横财让夫妇俩不知所措，随后深深生疑……


听故事的人已经在无数个奇情世界穿越过一遭，不知不觉，夜深露水凉。


老人渐渐隐入黑暗。


你，也想来一起听故事吗？下面是世界的另一面：「故事会」。
