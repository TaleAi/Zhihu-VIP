## 简介
唐朝末年，天下大乱，群雄逐鹿。北方沙陀族将领李克用讨伐黄巢起义、平定河东叛乱，创立了「连城犹拥晋山河」的基业。之后数年，他割据河东，与占据汴州的死敌朱温对峙。怎料李克用却在朱温建立后梁称帝的第二年即病逝，他只得将多年打下的基业留给儿子李存勖。


在一个暴雨之夜，一代枭雄李克用的生命走到了终点。死前他将自己的事业托付给了儿子李存勖，并留下了象征仇恨的三支箭，他命令李存勖一定要完成三件大事：讨伐契丹、攻克幽州，消灭自己的宿敌朱温。


然而李存勖接手的晋国却是一个风雨飘摇的、千疮百孔的势力。叔父李克宁对晋王之位虎视眈眈、宿敌朱温带领的后梁大兵正在围攻滁州。内忧外困之下，李存勖设计诱杀了叔父李克宁，震慑住李克用的样子们，然后趁势领兵出征，突袭梁军，解了滁州之围剿。


在一次被朱温手下杨师厚围剿的战役中，李存勖侥幸被手下五百轻骑救出。大难得逃的李存勖发现这五百人中竟有一个叫赵弘殷的半大孩子。他决定留这个孩子在身边，安置在自己的禁军中。13 年后，跟随李存勖南征北战的赵弘殷日益长大。他虽是晋军中足智多谋、骁勇善战的一员大将，但心中却有难以言说的一抹伤痛：心上人琬儿被杨师厚杀掉的仇还未报，自己在军中也被李克用的诸多养子们压制。


另一方面，李存勖在称霸之路上高歌猛进，几场关键大战后，他是否能亲自手刃仇人朱温？在一次次替琬儿报仇失败后，赵弘殷效力晋王的初心是否依旧不改？五代枭雄，鼓角战歌，一代人的命途终有变数，且听丧心病狂刘老湿为你带来的历史小说《鼓角英雄传之五代风云录》。
