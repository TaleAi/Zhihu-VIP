你在日复一日、重复混沌、浑浑噩噩地过生活吗？


你对社会和人性有足够的洞察，对自我的独特价值有追求吗？


如果你想在嘈杂的世界中保持通透，想参透人性弱点的本质，想清醒而自知地度过一生，那么，请你一定要读鲁迅。


著名文学家、教育家叶圣陶先生有言：「与其说鲁迅先生的精神不死，不如说鲁迅先生的精神正在发芽滋长，播散到大众的心里。」


什么是「鲁迅精神」？如何让鲁迅精神散播进我们的心中？


本专栏经过匠心打造，包含鲁迅先生 15+ 篇经典作品的分析解读，融合鲁迅一生跌宕起伏的风云故事，讲述他创作中永恒的人文价值。就让我们一起用 10 节专栏的时间，清醒地重塑人生最重要的十大命题。
