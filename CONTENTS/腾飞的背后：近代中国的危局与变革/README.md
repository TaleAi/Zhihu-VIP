## 简介
拂拭历史烟尘，讲述中国百年传奇与无畏


纵观近三百年历史，中国走出一个明显的 V 字形，一端是康乾盛世，一端是世界经济主引擎，中间则是晚清民国的衰败残景。


风雨如磐，大厦将倾，圆明园中燃起强盗的火把，数亿人民在铁蹄下呻吟，中国似乎将和其他古老文明一样接受奴役，只能在梦中回忆祖先的英名。


有人怀抱救国壮志，用生命唤起民族觉醒；


有人潜伏敌营十四年，惊心游走于森然刀锋；


有人冲破长空，以血肉之躯筑起碧霄长城……


千千万万倔强的脊梁，铺就中华崛起之路，英雄驱散黑暗，大地复现光明。
