## 简介
女军医在古代，治病救人平乱救天下。


天才医生苏云七，化身太子白月光替身，冥冥之中却似有只命运的翻云覆雨手，将她推向命在旦夕的萧王。  

哪料他，长枪策马，一笑破苍穹；  

她在侧，不离不弃，三针定江山。  

兵者：下兵伐战，中兵伐交，上兵伐谋；医者：下医医病，中医医人，上医救国。  

他与她携手生死，并肩濡沫，马踏天阙成霸业，披荆斩棘夺天下！
