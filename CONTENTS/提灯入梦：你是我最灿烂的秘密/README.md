## 简介
万千世界，总有一人披荆斩棘，为你而来。


小甜文选手张若妤新专栏来啦，天马行空的脑洞，缠绵悱恻的爱情，带给你前所未有的甜蜜体验～


**微信有诡**


我收到一条陌生消息：


「五分钟内强吻前男友，否则，他将会跳楼身亡。」


迫于无奈，我约前任见面。


他秒回：「姐姐，我可不吃回头草的。」


可是，如果不吃这回头草，他会死的。


我的微信里住了一个魔鬼，它安排我做各种诡异游戏。


而游戏失败的代价是……失去生命。


那时的我尚且不知，这场名为魇的游戏，究竟会让我们失去什么……


  



**昨日梦白鸽**


我给程隽当了三年的舔狗。


他太难舔，油盐不进，软硬不吃。


三年后我累了，选了他的同款机器人，性格设定：爱我爱得死去活来的超级舔狗。


可是，当我上班途中请假回家时，一开门，却看见了气喘吁吁的机器人。


我心有怀疑，可他说——


「刚刚在健身，幽幽喜欢摸腹肌，所以要多健身。」


救命，机器人怎么会这么懂我！
