## 简介
回忆里有糖，甜到流泪。


有些感情，需要藏在心底，默默地守候，才能长长久久的留存下来。


只要宣之以口，世间的纷杂便会践踏它。


沾满灰尘后，变了味道，却又不能怪它不再是它。


  



**《阿澈的日记》**


做了池珵六年舔狗后，他的白月光回来了。


他变得不怕黑，不喜欢穿黑色西装，不愿意陪我看日出日落。


于是我不爱他了。


他却将我抵在镜子前，卑微恳求我：「清清爱我啊！」


我小心翼翼抚摸着他受伤的眼角：「保护好他的眼睛，我就会爱你。」
