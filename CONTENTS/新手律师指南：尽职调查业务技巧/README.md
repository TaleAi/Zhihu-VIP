## 简介
本课程由《法律尽职调查完全手册》作者李俭律师主讲，课程中，李律师将根据自己多年的从业经验，配合 40 个典型案例，帮你迅速掌握尽调业务的各个环节。


做投资有三大基本功：数据、尽调、行研。


不管是刚入投行的新手，还是从业三五年的精英，这三样都是最为重要的技能。


这其中，又以尽调最为核心、最为实用，前能承接搜集的数据加以总结，后能开启行研分析。


而在律师的非诉业务中开展法律尽职调查越来越普遍，对于从事非诉法律业务的年轻律师来说，头三年的工作内容可能主要就是各种法律尽职调查，但是尽职调查却无统一标准。


尽职调查主要专注以下三个方面的内容：


开展尽职调查前的准备工作；


多查证、多访谈，确保第一手资料；


撰写尽职调查报告，表述要客观，具体。


年轻的律师没有太多实务经验，没有全面的专业知识，在具体实践中可能会遇到尽调事项不全面、尽调方法达不到预期效果、与项目公司沟通不顺畅、把握不到不同类型项目的尽调重点等难题，全面、系统的专业培训很有必要！


本课程邀请《法律尽职调查完全手册》作者李俭律师，带来法律尽职调查课程。


课程中，李律师将根据自己多年的从业经验，配合 40 个典型案例，帮你迅速掌握尽调业务的各个环节。
