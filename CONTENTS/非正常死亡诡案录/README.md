## 简介
多年前，我和水生、小袁遇到了几起让人毛骨悚然的案子。现在，我想把这些案子写下来，希望这些案子背后的东西对大家有所启迪。


《冥婚墓穴》


连续几天的大雨冲毁了一个结冥婚的墓穴，我们发现冥婚新娘竟是非正常死亡。顺着线索，我们寻找到了一个山中小村。在村镇中，我们揭开了这个新娘的身世及死亡原因。


《女孩的血泪》


合租的女研究生裸死房中。二十四小时后，她的尸体流下血泪。是什么样的冤屈让这个女孩死不瞑目？顺藤摸瓜我们找到了一个癖好小众怪异的群体。
