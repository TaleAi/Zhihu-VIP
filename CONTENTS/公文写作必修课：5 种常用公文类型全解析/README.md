## 简介
本专栏从公文写作的基本方法与常用公文写作入手，选取常用公文文种请示、工作报告、会议纪要、通知、函，结合实际案例分析，深度剖析公文写作的实用方法与实战技巧，帮助学员迅速提高公文写作的实战技能，成为公司或部门值得依赖的笔杆子。


好的公文千篇一律，不好的公文——


格式五花八门、行文规则混乱、结构缺乏逻辑、内容辞不达意、篇幅冗长啰嗦……


其实这些都是因为不了解公文写作的基本要求，未掌握公文写作的基本方法与技巧，以及缺乏实战训练。


「妙笔生花觅先机，言简意赅促效率」，优秀的文笔既能有效传达企业管理思想，提高管理效率，又能提升经理人的职业形象，获得职业发展的机会。


本专栏选取请示、工作报告、会议纪要、通知、函等 5 种常用公文类型，从公文写作的基础知识入手，结合丰富的案例加以讲解，帮助学员快速掌握公文写作的基本方法，纠正公文写作的常见错误，同时通过系统的讲解和结构化的方法，提高学员的结构化思维与语言文字组织能力。
