## 简介
本专栏用独特的日记形式、行云流水的文笔，娓娓道出一段缠绵悱恻的后宫爱情故事。


我进后宫后，不喜争宠，日子过得虽然平淡，却有些小趣味：


赏桐花，品美酒，岁月静好。


「怎生独饮？」一声询问，改变了我日后的闲散人生。


我莫名其妙地被晋升为桐妃，而后又被放逐出宫，去灵山寺为国祈福。


合伉俪，合相思，合恩爱：想与他有白首之约；


满陈酿，繁花叶，驰远山：为他走入市井江湖；


倾卷雨，揽风云，望山海：我的千里红尘有我也有他。


但原来，所有的冷落与生分都是蓄谋已久。


他一字一顿地说：白惊玉，你走吧。
