## 简介
她穿书了，穿成了一个炮灰小丫鬟，会爬床，并且不要命怀了主子孩子的那种。嗯，怎么活下去……是个问题。


穿书后，她衣衫不整地出现在王爷床上。还没反应过来，王妃就提着刀冲了进来。


很快她意识到：她不是女主，不是女配，是个分分钟领盒饭的炮灰小丫鬟。


更惨的是，她怀孕了。


怀孕后，她跟肚子里的娃成了书中全员的眼中钉。


作天作地都没能把这孩子作掉，她决定抱紧女主大腿，怒刷众人好感，苟命为上。


可是……重生人物，穿越人物为何接连出现？


扭曲的剧情线背后，究竟有什么角色在布网主宰。


随着剧情推进，怎么男主、男配、男 N 号看她的眼神，全都越来越奇怪？


原来她等来的不是爱情，是一次又一次的算计。


对此她表示：情爱勿扰，老娘只想独自美丽！
