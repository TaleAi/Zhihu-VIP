## 简介
小时候你一定幻想过，被窝外面站着可怕的怪物。也一定有一个故事、一个画面，成为了你童年噩梦的素材。本专栏作者小妖 UU 是悬疑恐怖杂志的资深作者，她的故事曾经成为了无数人的「童年阴影」，读完这些故事，或许深夜时你会祈祷：鬼不会攻击被窝里的人！


暴风雨夜，同学恶作剧将我反锁在学校实验室，深夜洪水爆发，他又折返来救我……醒来后我意识到，救我的实际是一只水鬼。


我生活在单亲家庭，父亲是个赌鬼，孤单的我为了博取注意，在冰箱里塞了一只死猫……有一天我发现，死猫变成了一个蜷缩的死小孩。


因住宿条件不好，某学校先后发生集体皮肤病和学生煤气中毒死亡事件。事故发生后，多名学生见证了灵异事件，我更是亲眼目睹了死去学生的鬼魂。


人们总认为孩子纯真可爱，但有时他们也会释放出最纯粹的恶意……


你是胆小鬼吗？读完本专栏再回答。
