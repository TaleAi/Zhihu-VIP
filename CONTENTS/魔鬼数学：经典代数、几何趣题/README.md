## 简介
学生时代，代数和几何是学习数学的重中之重，也是生活中有用的两大思维工具，我们时常需要借助数学思维的力量来处理问题。专栏《魔鬼数学》用多达 200 多道关于代数和几何的趣题，帮你从熟悉的生活应用场景入手，建立起抽象思维、归纳和推理能力、空间想象力等一系列数学思维，让你搞懂复杂数学问题的本质，爱上数学。


不久的过去，做数学题是一项很高雅的娱乐活动。在没有电脑和手机的时代，人们习惯用脑而不是用眼来寻找乐趣——就像曾经风靡一时的数独游戏。


但你往往会感到学习数学是困难的。特别是学生时代，代数几何成为让人烦恼的两座大山。但当你掌握数学思维，你会发现，数学不仅有用，而且并不难。


在专栏《魔鬼数学》中，作者会用 200 多道经典的代数几何趣题，以你熟悉的生活应用场景入手，帮你建立起数学思维，搞懂复杂数学问题的本质，爱上数学。


* 传达主流数学的魅力，揭开代数与空间的神秘面纱
* 从生活实例出发，深入浅出教你数学思维，启示你如何抽象思考
* 抽象思维、归纳和推理能力、空间想象力……建立完整的数学思维

这部充满睿智和趣味的数学集，与侦探游戏无异，从线索中寻求真相，并视之为打发无聊时光的高雅娱乐：

**要是你每天花几分钟，或许夜间窝在沙发里，做几道这样的题目，说不定会发现，其实这样才是真正适合你的生活消遣。**
