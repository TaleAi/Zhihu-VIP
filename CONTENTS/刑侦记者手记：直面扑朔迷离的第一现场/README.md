## 简介
凶杀案，听起来好像离我们日常生活很遥远，但其实很多它们就发生在我们身边， 记录凶残刑案，打开潘多拉的魔盒，笔者将通过一位刑侦记者的角度，为你讲述一桩桩神秘离奇又骇人听闻的杀人事件。


玫瑰渗出了鲜血，在杀人中热烈。两个变态连环杀手因为共同的癖好变成很微妙的「精神伴侣」，一起作案杀人。这其中隐藏着怎样的内情？


水泥墙里封锁的尸体，成了十五年的悬案，十五年后惨案再次发生，这背后有着怎样的恩怨？


宾馆秘闻，隔壁到底发生了什么？


神秘少女离奇失踪，警方在嫌疑人家里，发了七个箱子，里面装着七个人头，这系列的惨案，都源于校园欺凌......


双重人格是怎样形成的？隐藏在主人格之下的残暴人格，偷偷犯下了滔天的罪孽！


身患深度斯德哥尔摩综合征的妻子，与极度虐待狂的丈夫，失踪多年的未成年少女满身是血地走在街上，真相的探索由此而展开......


错综复杂的刑事案件，耸人听闻的犯罪手法，与刑侦队长梁震一起抽丝剥茧地寻找真相，探索人性的黑暗，感受生活的另一种真实。


本专栏由真实事件改编。
