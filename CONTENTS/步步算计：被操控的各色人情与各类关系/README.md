## 简介
不论何种人际关系，总存在各种博弈。有人输，有人赢，但总归是善恶有报终有时。


我生在棚户区、长在贫民窟，却遇到了跟我截然相反的人，那个人出生优渥、履历漂亮。


别人拼命一辈子想去到罗马，那个人他就生在罗马。在我偷钱买网红冰淇淋时，在被我弟追着揍，熟练地爬上树时……那个人住着别人进不去的大院，念着名校，晒着法国南部炙热的阳光，跟我的生活毫无交集。


按理说，1% 精英和 1% 底层永不相逢，可老天总爱开玩笑，在最窘迫的时刻里，我撞见了他。


或者说，我撞了他。


——《我偏要高攀》
