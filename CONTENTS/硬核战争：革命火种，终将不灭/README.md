## 简介
星星之火，可以燎原


1946 年，蒋介石撕毁双十协定，悍然对我中原解放区发起进攻。


一切随着中原的那场突围开始改变，缓缓拉开了时代的大幕......


在解放战争中，我们看到了运筹帷幄的林彪、刚猛劲道的彭德怀、出奇制胜的刘少奇、决胜千里的粟裕。


而那些跌宕起伏的故事，那些不被大多数所了解的战争，藏在文字里，却很少活跃在地图上。


本专栏以图纪事的方式，带你领略不一样的战场风云，看到不一样的故事。
