欢迎来到『幻想世界』！


她们是活在幻想里的人，但她们的故事，是真实存在的。


在这里，你可以跟小龙王谈恋爱，也可以跟着鲛人妹妹去造反，你可以进入一所全是妖精的学校，也可以到天庭去考公务员。

直到她长出了一对鹿角。
