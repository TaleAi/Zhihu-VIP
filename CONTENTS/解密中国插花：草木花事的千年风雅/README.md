## 简介
本专栏由《中国插花简史》的作者孙可和李响共同主讲，将中国数千年插花历史浓缩至 60 分钟：从中国插花的起源、发展，和日本花道的关系，到中国插花的精神内涵，再到如何通过传统插花复生古人生活，带你全面了解草木花事的千年风雅。


  



古人的生活在当下可以复制吗？


在中国传统生活美学复兴的今天，古代文人的四般雅事——**「插花、挂画、焚香、点茶」**再次回归人们的视野。然而，作为文人四雅之首，中式插花往往淹没于大行其道的日本花道和西洋花艺设计中。


事实上，在《诗经》、《楚辞》的时代，中国人已经开始自觉对花和植物进行筛选、分类、审美，并寄予情感；汉代出现了最早的关于插花的记载；隋唐时期诞生了第一部插花理论作品《花九锡》；宋代开始插花成为显学；明代袁宏道的《瓶史》被认为是插花艺术的最高峰……


**修剪枝蔓亦是修剪人心，塑花身亦在塑己身。**花中有世界，叶下有如来，了解中国插花的历史与技艺，不仅帮助你观照生命，观照生态，更可以观照生活，和谐身心。


  



你将收获：
-----


* 插花的缘起与祭祀、宗教有什么关系？
* 中国插花在上千年的发展历程中，产生了哪些变化？
* 古人重视花与器的关系，今天我们应如何看待、使用生活中的器物？
* 日本花道的诞生、演变受到中国插花怎样的影响？
* 中式插花的基础形式是什么？「三大主枝」在佛、道教中分别有什么内涵？
* 中国古代有哪些重要的插花典籍，以及涉及插花的绘画作品？
* 古代文人插花是什么样的？他们在什么场合插花？对当下有何参示？
* 怎样从「工具」、「花材」、「插法」、「欣赏」四个角度，创作一件中式插花作品？
