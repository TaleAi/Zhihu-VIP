## 简介
不完美的女生也值得拥有爱情。


很多人只想在爱情里，呈现出完美无瑕的自己。


这个专栏，我们偏偏想要描述那些不完美的女生的故事。


**《她与食物链》**


从农村出身，见惯了贫穷与丑恶，她想要获得很多很多的钱。


殊不知，她这份谨小慎微的贪婪也被人利用。


盘桓在两个男人间，她收获的只有一地鸡毛。


想要爱情与金钱的她，是否能如愿以偿？


欲望沉浮的大城市，又是否能承载她的未来与归处？


**《淑女》**


所有人都轻贱她，辱骂她。


她习以为常。


但那个男人，却把她当成正常的女孩子，尊重她，爱护她。


她以为自己得见天日，却只是跳进另一个男人缔造的深渊。


当一切真相揭开，她是否还会无所畏惧地走向他？
