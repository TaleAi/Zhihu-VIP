## 简介
爱是会骗人的。那些细枝末节的温情，那些边边角角的甜蜜，那些让你坚信的自己被爱的瞬间，其实也都有可能是假的。本专栏收录了一组在你侬我侬的甜蜜恋曲中藏满刀光剑影的言情故事，这里只有少数人是真正为爱而生的，而他们大多没什么好结局。


**《珍爱生命别钓鱼》**


老板让我好好带他的小公子，谁知小公子一心只想爬我床。


其实一直这样也好，只是我不明白，为什么他又要喜欢上别人？


我清理掉他身边的莺莺燕燕，却还是留不住他。


那，就请死在我手里吧。


  



**《棋活了》**


小皇帝的白月光贵妃死了，我杀的。


小皇帝的心又动了，我撩的。


小皇帝不想再做太后的棋了，我也是。


可是宫墙里的事，哪能有那么简单呢？


  



**《虐文男主的终极归宿》**


我穿进一篇虐文，成了试图勾引男主的恶毒女配。


我是个善良的人，男主他这么可爱，却要被虐文搞烂一生，这公平吗？


肯定不啊！于是我依照原计划勾引了他。


完全不是因为他长得帅！


但是等一下，这个黏在我身后的弟弟，好像有点不对头……
