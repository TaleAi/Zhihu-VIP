## 简介
我一直跟在爷爷身后帮别人迁坟。
迁坟中有着一种特殊的葬法，名二次葬，需开棺椁，捡尸骨，整新衣。
而我第一次遇到二次葬，就挖出一具栩栩如生的女尸⋯⋯


爷爷从王远胜口中得知消息，准确前往县城寻找怂恿王远胜活葬之人。


爷爷离去的这一晚，王远胜上吊身亡，而我却从细节中发现，王远胜是被人害死的。


这个时候，县城里的爷爷突然打电话回来，慌张的让我带着父亲逃，去县城找他，还没等我问清楚，电话挂断。


当我回家准备带着父亲走的时候，发现父亲不见……
