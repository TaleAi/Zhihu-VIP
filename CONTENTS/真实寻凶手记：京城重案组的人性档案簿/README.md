## 简介
刑警 8 年，他直面了太多罪恶。


2012 年 3 月，南城一名失足少女失踪——她跟一个男人走进小区就再没出来，之后的监控录像里，只出现了那个男人和他手上的拉杆箱；


五环高架桥下一个废车场里，刑警队、法医、和重案队的十几个便衣警察，在夜色中发现了一辆烧毁的出租车，和两具高度腐败的女尸；


一个夏天的清晨，城郊六环一个仓管员大爷遛着自家的金毛，却在垃圾堆里意外刨出了一具尸体，警察顺藤摸瓜，把一位心理学博士带进了审讯室；


……


在没见到下一个案子之前，赵赶鹅总以为这个世界上自己什么都见过了。各种人生悲剧的剧本，都在刑警支队后院那个密封的档案室里；


后来他才意识到，还真不能说警察做久了什么都见过。


离奇案件还在上演，从「阳光之下」到「阴影角落」，往往会有交集。


同一座城市里，这些人生活的轨迹，是怎样和我们擦肩而过的？

为什么看
----


寻凶手记的故事，是我们这个时代许多欲望、恐惧和焦虑心理的反映。


是用极端条件验证出来的，人的真相。


了解这些故事，也是在了解自己。


它们就真实发生在这个城市里，除了成为遗憾，更该留下警示。


赵赶鹅有一个很「刑警」的说法，把罪案的发生比作一把枪：基因是子弹，环境上膛，特定事件扣扳机。


而理性是枪支的「保险」。无论什么情况下，扳机都应该被牢牢锁住。


**多些对人性的敬畏，克制犯罪的欲望。**


**多「长点儿心」，更好地保护自己。**


最后提一句，挖掘施暴者的经历，是为了规避暴行产生的诱因，而不是为暴行提供任何合理性。
