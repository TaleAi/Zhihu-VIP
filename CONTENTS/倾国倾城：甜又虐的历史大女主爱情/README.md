残忍又专权的吕后，和被她残害的戚姬，都是对爱情心碎的女人。


黄月英女扮男装，和诸葛亮共破山庄奇案，因此结缘。

豪放如李白，帽子上也要染点绿，才学会真正的放下。


历史中的女主，面对的爱情像一道谜题：是真爱，还是利益？是选择爱情，还是选择生存？
