## 简介
这是一部长篇古代言情小说，《玉树后庭花》作者新作。



* 王爷男主：病娇也好，腹黑也罢，这一生做得最对的事是把你留在身边。
* 丫鬟女主：穿越成丫鬟就够苦命了，还要应付你这个死变态！
* 悲催男二：能不带上我吗，撩完就走，用完就丢，终究是错付了。
* 亲妈作者：穿越、虐恋、重生，你想看的这里都有。

天子脚下，晋王府中。


我穿越成了一个清扫佛堂的三等丫鬟，终日被困在一处四四方方的院落。


在现代身患绝症，此时能捡到一条命，已经感恩戴德。


这一世只想安安静静、平平安安地活下去。


可天不遂人愿，色欲熏心的大管家时常骚扰我。


那日，我假意逢迎：「您英姿飒爽，威武不凡，是天底下打着灯笼都难找的好男人！」


谁料，竟恰巧被老王妃和晋王撞见。


……


真是十张嘴都解释不清。


兜兜转转，当为了活下去不得不接近晋王时，他眼中尽是对我的鄙夷，「你的喜欢到底跟多少男人说过，你自己数得清吗？」


我不放弃，也不在乎别人说我品行不端，不择手段。


因为我只想活下去。


……


然而，再活一遭，没想到竟是认错了人，爱错了人。


赔了一生好光景。
