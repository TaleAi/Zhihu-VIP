## 简介
让朱元璋大开杀戒的南北榜案、让乾隆匪夷所思的甘肃冒赈案、让慈禧震怒的刺马案……这些历史疑案中，帝王心机深沉，不择手段，亲手掀起一场场腥风血雨。


**刺马案**


同治九年（1870 年），两江总督马新贻，被当街捅死。凶手被捕后，三番四次更改口供。凶手，究竟为何刺杀马新贻？背后的真正指使者，又是谁？真相扑朔迷离。据马新贻的儿子马毓桢私下回忆，对于自己的死，马新贻其实早就有了预感。这一场刺杀，跟慈禧对他的「训示」有关。


**南北榜案**


大明洪武三十年（1397 年），新的科举殿试结果公布后，破天荒地出现了一边倒的局面，上榜的全部是南方士子，北方士子非常愤怒，认为主考官作弊偏袒。朱元璋下令重新审阅试卷，但两批考官都认为结果没有问题。朱元璋明知考试结果公正无误，仍然下令将新科状元斩首弃市，是杀戮成性还是另有深意？


**甘肃冒赈案**


乾隆四十六年（1781 年），甘肃上下全体官员，串通一气，以赈灾之名，骗皇帝的钱。这桩贪污案的吊诡之处在于，全省没有一个官员是干净的，也没有一个官员举报。由于案发数年才被发现，涉事官员早已辗转到全国各地任职，真要查起来，扯出萝卜带出泥，可能引发巨大的官场动荡。但乾隆勃然大怒，下旨一查到底。


……


这些帝王们，为了巩固自己的权力，制造冤狱、寻找替死鬼……充满了倾轧与猜忌、阴谋与杀机。


他们的手上，滴淌着无数无辜者的鲜血。


真相，重要吗？


或许，政治本来就没有真相。
