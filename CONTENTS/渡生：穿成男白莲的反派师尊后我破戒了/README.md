# 渡生：穿成男白莲的反派师尊后我破戒了

### 作者：雪中送猫

### 分类：古代言情

### 状态：已完结，共 17 节

### 存档日期：2022-06-02

## 简介
含瑢万万没有想到，把自己抽筋扒皮的逆徒，不仅欺师灭祖，还想睡祖。


含瑢穿成了《神魔降临》的最大女反派。


不是一来就死的背景板，而是将男主温玹从小抚养，从生理到心理都极尽折磨，蹦跶了大半本书的魔头师父。


自从她来到临渊大陆，意识到自己坐上了火山口，第一件事情，就是从良。


她是真的想要好好对他，弥补原主带给他的伤害。


但一切似乎都太晚了。


含瑢穿到《神魔降临》的第一百九十八天，她终究没有扛过原著威力，甚至表现得不如原版女配，被男主提前搞死了。


果然反派不是常人能当的，特别是遇到一个杀伐果断又心狠手辣的男主。


再次醒来时，她身处秘境，除了自己的名字，没有半点记忆，也无半点修为。


除了元神仍在，整个人已与凡人无异。


她在兽穴里待了三天，才在溪边遇到一个少年。


一个眉宇柔和又眼神干净的少年。


含瑢看着少年，一双眼儿脉脉含羞，嘴角都快咧到耳根。


他为她解毒，她助他疗伤。


很快，破了伦常。


当含瑢记起了所有，冷汗从四肢百骸疯狂涌出。


她气自己，竟然对这个处心积虑将她弄死的男人，投怀送抱。


她恨温玹，明明知她失忆，却故意以情爱诱之，破戒欺师。


但眼下自己修为全无，不能与温玹撕破脸面。


她得逃出这个秘境！
