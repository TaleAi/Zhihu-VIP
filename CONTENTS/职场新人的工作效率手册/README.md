## 简介
智能设备高度普及的今天，也迎来了应用工具的大爆发，这些工具和方法，正在造就个人和职场效率的新形态，如果你不抓紧学习和掌握，将很快成为数字时代的「山顶洞人」。这里是一份全面实用的工具清单和实用指南，源于专业用户的实践经验，易于理解、干货满满，获得效率达人罗永浩和王俊煜大力推荐，献给想要变得更好的你。


  



![img](https://pic2.zhimg.com/v2-5901377d308cb197564fee06a5686862.png)

  



**你为什么需要这门课？**


* 你心里明白，很多加班的无奈，都源于糟糕的任务管理；
* 身处高速运转的信息时代，一不留神，就在信息洪流中迷失自己；
* 没法腾出时间来充实自己，担心自己陷入「穷忙」的境地；
* 花钱没有计划和节制，结果只能是「你不理财，财不理你」；
* 文件归档，邮件收发，密码管理，思维导图......太多小套路，你还需要补习；

很多优化效率的好习惯，很多免费好用易上手的小工具，这十堂课都可以教给你。


**适合人群**


* 渴望全方位提高工作效率的人
* 想要学会快速筛选信息，合理分配时间的人
* 需要提升邮件礼仪，让自己在商务沟通中更得体的人
* 经常莫名其妙月光，急需培养良好理财习惯的人
* 愿意花时间反思和总结工作习惯，相信磨刀不误砍柴工的人
* 希望掌握更多实用工具，相信科技让生活更美好的人
