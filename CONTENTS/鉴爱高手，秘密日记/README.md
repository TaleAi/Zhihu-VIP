**《年年不忘》**


男朋友的好兄弟吻了我。  

我还一脸错愕的时候，他却说：「这个姿势太费腰了，下次得换个姿势。」


**《我喜欢上了总监》**


职场小白喜欢上了部门总监，一次团建，关系不小心「究极进化」。  

同事说：总监高岭之花、生人勿近。  

小白腹诽：这说的是给我发「昨晚腰疼」那个人吗？


**《男朋友外面有狗了》**


环球影城开业，我刷个攻略，都能发现男朋友奸情。  

说是考公，其实狗男女亲亲我我。  

他考试在即，我拿着证据对他说，「只要上岸，我就举报。」
