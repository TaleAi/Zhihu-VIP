## 简介
太空争霸的时代又一次到来。


自美苏冷战的高潮阶段，太空就是两大阵营的决赛场。


美国政府开出 500 亿美元的天价支票将人类送上月球，开启的太空竞争也将苏联拖到崩溃的边缘。


自苏联解体，美国的「星球大战计划」也随即下线，人类探索太空的脚步开始放缓。


直到 40 多年后，中国的航天飞船、宇宙空间站、月球探测器、火星探测器一个又一个地进入太空，美国人深受刺激，各国企业也陆续加入太空开发的商业大军。


从 SpaceX、维珍银河、蓝色起源再到银河航天、翎客航天，越来越多的私人公司开始涉足太空领域，这让原本广袤的地外空间，突然显得拥挤了起来。


我们一起看看太空争霸的历史和未来：


* 美苏太空争霸烧了天量资金，难道仅仅为了争个大国脸面？
* 中国为什么能在短短五十年，突破地心引力，成为人类探索宇宙的领头羊？
* 马斯克的星链计划到底是服务于人民？还是服务于美军？
* 各国的太空布局是什么？未来的外太空会不会是大国争霸的下一个主战场？

本专栏将从历史、科技、商业、大国博弈等角度，为你解读即将到来的太空竞争，理解太空竞争的底层逻辑，和未来各国在太空经济中的布局。
