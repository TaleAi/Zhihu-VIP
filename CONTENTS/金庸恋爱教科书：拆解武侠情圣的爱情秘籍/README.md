## 简介
在金庸的江湖世界里，我们不仅记住了肝胆相照的侠气，同样难以忘怀那些缠绵悱恻的爱情。我们生活在当今的现实社会里，没办法真的飞檐走壁，而只能对书中的大侠心向往之，可金庸先生留给我们那几段旷世奇恋，却能够让我们有所收获，进而更好的经营我们自己的亲密关系。本次课程就将从两性视角，来跟大家详细拆解八位金庸情圣的爱情经营秘籍。


各位看官，不知道你是否抱怨过，为什么现在的男孩/女孩都那么难追，我明明已经这么努力了？


为什么他/她看我的眼神总是少了那么点爱意，还总愿意把我跟被人相比？


为什么我这么诚实正直的好少年/好少女,却总是输给隔壁的渣男和绿茶？


为什么我都说过我爱他/她,她/他却还是心猿意马，离我而去？


为什么我只是想要愿得一心人，白首不相离，可却每次都遭遇爱情的滑铁卢？


如果您有这一系列的苦恼的话，那不妨点开本期的课程，让我们看看金庸世界里的情圣们是如何升级打怪打通关的。


这一系类的课程满满干活，我们并不写散文鸡汤来讴歌爱情之美，而是逐步拆解，金庸世界里的八大情圣，到底养成了怎样的品质或者运用了什么技巧，让自己的爱情一帆风顺的。
