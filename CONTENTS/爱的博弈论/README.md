## 简介
任何一个置身于感情中的人，都有属于 TA 的专属人设。病娇 VS 冰山、绿茶 VS 渣男、真爱 VS 替身……在感情里角逐，谁都不无辜！不如看看他们有关爱的博弈论……


我精心制定了「狩猎计划」。分为 3 个部分：诱导、俘虏、圈养。计划太顺利让我发现了我爱的冰山，也是个病娇。发现了真相的我，有一种被欺骗的愤怒，和一种被挑战的兴奋。爱情这场游戏，我绝不会输。欲望才是爱情的主导。


*《束缚成魔》*


我千辛万苦找到了我爸小三微信后，竟然发现这个微信一直都是我的星标好友，原来害得我妈得重度抑郁症的罪魁祸首是她。得知真相之后，我开始蛰伏，等待报复的时机！


*《爸爸的秘密》*


通过 3 个男友成功步入婚姻殿堂，精确的目标分析和准确的情感目的，对我而言，爱情是一种消耗品，但是感情却被奉为呼吸。我的哭泣也有真心带来的情绪，我的爱情也有受伤的风险博弈。不过于我，爱情的赌博，从来只有操作，没有猜测。


*《上位》*
