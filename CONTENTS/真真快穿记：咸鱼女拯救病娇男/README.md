## 简介
狗系统，还我可可爱爱的男主！


陈真真，二十一世纪的「咸鱼」美少女，被系统带到了一个平行世界，目的只有一个：拯救病娇男主。


第一次，她穿到了男主小时候，天惹噜，长得好看还智商高的团子也太可爱了吧。把病娇小团子养成五好中学生，两个字形容这个过程：完美！


第二次，因为她的离开走歪了的男主开始祸害人间，再次承担重任的她决心把他「拨乱反正」，谁知道，这次，他不想再跟她当亲人，夭寿啦，她可是把他养大的姐姐……


第三次，男主被来去无踪的她气到彻底黑化，她一过去，就被翻过来覆过去这样那样了 10086 次。本着博爱无私的精神，她能否渡化病娇男神，走向人生巅峰？


而这个该死的系统，又会不会再次出来作妖？


且看陈真真的快穿之路！
