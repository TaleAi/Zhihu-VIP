## 简介
科技发展带来便利的同时也存在不少隐患，如何使得科技的发展更加符合善良的期许？本门课所讲述的五位哲学家的故事勾勒出现代技术哲学的整体概况；主讲人摒弃艰深词汇，用通俗的话语为你讲述当代生活的经典案例，与你讨论与反思科技中的伦理。


如今技术支撑着我们的生活，让我们享受诸多便利，成为我们与世界重要的交往方式。 


然而科技的进步也让我们常常陷入忧思：


转基因技术是否安全？ 


网络时代我们对自己隐私还有自治权吗？


当生命活动简化为手指在屏幕上滑动时 ，生活的丰富性该去哪寻找？ 


…… 


这些问题的答案关系到现代人最本质的生存状态，因此科技伦理讨论才意义非凡。


当科技无往不利的控制了我们的生活，它即是胜利之剑，也是破灭之枝。


如何使得科技的发展更加符合善良的期许？


本堂课将由人大哲学学院副教授为你讲述五位哲学家的故事与当代生活的经典案例，与你一同反思科学伦理中的哲学。
