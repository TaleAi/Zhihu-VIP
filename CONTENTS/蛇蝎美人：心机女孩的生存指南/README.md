## 简介
她们不是传统意义上的好女孩，耍心机，搞权谋，但不过是为了在险恶的人世间活下来。


《凤栖梧》


她是宫婢之女，被远嫁敌国，对方是缠绵病榻多年的老皇帝。几位皇子明争暗斗，她本想独善其身，却被推到了旋涡的中心。看来在这人吃人的后宫，想要活下去，只能做一朵面上的小白莲。她一面在几方势力之间虚与委蛇，一面同她选中的继位者小心筹划。在阴暗的地方待得太久了，这次她要攀上高位。


《戎色》


她是将军府的嫡长女，文韬武略，与太子定有婚约，梦想是做出自己的一番事业。前方战事吃紧，她为父分忧，上了战场。当战事大捷，回来时却发现自己的妹妹被一顶花轿抬进了东宫，果然是人心易变。男人她不稀罕了，她要的是权力。她入了朝，当了官，太子也当上了皇帝。她的复仇开始了。
