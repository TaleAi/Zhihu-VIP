## 简介
本专栏聚焦从 80 后到 00 后的当代「老实」青年，真实讲述的文风更贴近生活，现实里普通人遭遇的憋屈事情，在专栏文章中看主角反杀获得爽感


马善被人骑，人善被人欺。


老实人的世界总是充斥着许多自以为是的「恶人」们。


 


有的恶人，在婚姻。


我是一个草根出身的年轻富一代，半夜突然接到了一通陌生电话。


电话里，是一阵暧昧的嗯嗯啊啊……


而这个声音，属于我心爱的女朋友。


 


有的恶人，在家庭。


姐姐怀孕五月，我好心去她家陪伴，却不料遭遇了姐夫的「魔手」……


姐夫是有心还是无意？


我该怎么办？


 


有的恶人，在职场。


我历尽千辛万苦，终于考进理想的企业。


可不幸的是，遇到了一个极度自信又普通的男领导。


他不仅把我的每一个「好的」都当成示爱，还把肮脏的手伸到了我的腿上。


忍无可忍，这一次，我决定反击。


…………


 


本专栏带你开眼世间真恶人，围观那些被逼入绝境的老实人们，如何逆风翻盘！
