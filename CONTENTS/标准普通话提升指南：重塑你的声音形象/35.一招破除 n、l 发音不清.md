## 35.一招破除 n、l 发音不清
  



![img](https://pic2.zhimg.com/v2-83eafd69fd4aca8c0a2fb22d7d157cc6.webp)

这节课我要和你分享的是一招破除 n l 不分的现象。


我有一位学员，他说自己一直分不清 n 和 l 的发音，不知道怎么办，就算发出来了，他自己也不确定自己到底有没有发准。


比如有两个词语从来都是听众告诉他没有发清楚，第一个，中国的省份河南，他读成河蓝。还有一个词语就是老子和脑子。


我还有另外一位学员，他对语言研究很深，是一位语言学者，但是在发音上却难住了他。他跟我说，他发现自己不仅 n 和 l 发音发不清，更重要的是他不知道怎么能够听出两者的区别，到底要怎么练也就无从下手。


这两位同学的困惑在区分鼻音和边音的发音上很有代表性，它说明在深度纠音的实践活动当中听要优先于说，**如果你听不出来区别，当然也就无法分辨自己是不是说对了。**


**好，同学们现在让我们一起来听一下，大家听听看你自己能不能听出这其中的发音问题，大家可以跟着绕口令的文本边看边听这位同学的示范当中他是哪些音发错了。**


来，我们来听一下！

刘奶奶说，牛奶奶的牛奶不如柳奶奶的牛奶。


牛奶奶说，柳奶奶的牛奶会流奶。


柳奶奶听见了大骂牛奶奶，你的才会流奶。

在这一节深度纠音课程当中，我发现同学们反馈给我的普通话语音面貌中，模仿 10086 的声音大家都模仿得挺好，我们来听一下！        < p>

这位同学的 n 和 l 还是分得比较清楚的，但是在需要反应速度和发音的清晰度更高一些的绕口令练习当中，另外一位同学鼻音和边音分的不太清楚的发音面貌就出现了，我们来听一下他的普通话语音自测！


「刘奶奶听了大骂牛奶奶，你的榴莲牛奶才会流奶！


刘奶奶和牛奶奶泼牛奶，吓坏了刘奶奶。「


好，听完之后我们可以感觉得到，其实这位同学他非常奋力的想要把这两者的发音区分清楚，但是从听者的耳朵我们感觉到的却是两个音都不清晰。这是什么原因造成的呢？


造成的原因有两点。知识点来了，请同学们记住。


**第一点，根本原因是因为我们每个人所生活的环境不一样。**我们的发音器官在我们生长的早期发育阶段，适应了我们自己当地的一个语言系统。我们刚出生的时候对所有的语音都能够快速地学习，但是在出生一个月之后，这种区分所有语音快速学习所有语音的能力就会减弱，一直减弱到我们最后只能识别当地的语音面貌的因素了，其他相似的语音面貌就会被大脑自动合并。


所以，可能在你的语音体系当中 n 和 l 本来就是同一个发音，所以自然的你的大脑就会忽略它们的区别和差别，慢慢地就听不出也说不准了，我们可以听到刚才这位示范的同学，他应该是一位南方的朋友，他的困惑也真是这样的！


「我是南方人，很多朋友都说我说普通话不标准，特别是我的同事，我不知道怎么办，每天早上起来我在锻炼，可是真的没办法。」


这就是我刚刚说到的第一个原因。


**好，接着我们来说说第二个原因，第二点只要做到了我们就能够快速地深度纠音。我们要熟悉鼻音 n 和边音 l 它们不同的发音口形动作。**


**针对第二点，珊珊老师将教你一招来破除发音不清的情况，这个方法就是「自然成声法」，**特别有效，我帮助很多同学通过自然成声法来建立了一个正确的发音口形动作，帮他们找到感觉。


好，在自然成声法之前，我们要知道它们的发音要领，先要把造形做对，请同学们看到老师为大家准备的发音图示！


![img](https://pic3.zhimg.com/v2-3fbed0e029207270c6a82dcaf8c247e0.webp)

我们可以看到，**鼻音 n 和边音 l ，它们的共同点：一、都是舌尖中**音。二、都是浊音。这说明什么问题呢？


**舌尖中音说明我们的舌尖是着力点，浊音说明我们的声带要振动，**因为在语音学当中，发声的时候声带振动的音我们称为浊音，声带不振动的音我们称为清音。


**而两者的不同点是什么呢？不同点就在于气流到底从哪里出？鼻音的气流那肯定是要从鼻腔出啊，边音的气流要从口腔出。**


**那现在我们就知道如何去发这两个音了。**当我们发鼻音的时候，着力点在舌尖，舌尖顶住我们的上牙齿的齿龈，这时候我们口腔内部的软腭下垂，鼻腔的通道就打开了，随后气流冲击声带振动，送出去，n n n ，你听，气流就完全从鼻腔出来，发出的音就是鼻音。


**请注意！很多同学鼻音发不出来，不是在于舌尖没有顶住上齿龈，而是他们没有把软腭下垂，**所以发出来听到的感觉就是这样的, l l l ，鼻音不明显，因此大家要注意把软腭下垂下来。


**很多同学就问我，软腭怎么下垂呢？**很简单，你平时懒洋洋说话的时候，你的软腭就是下垂的，所以同学们平时很懒惰的时候，我们要记住那种感觉，说话懒懒散散的，没有调动兴奋状态的感觉，它也有用哦，就是用在你发鼻音的时候能帮你软腭下垂，发音更准。


接着我们来说一说边音的发音要领。着力点也是舌尖抵住上齿龈，这个时候注意了，区别点就在于发边音的时候，软腭我们要上提。


![img](https://pic1.zhimg.com/v2-256154e878279a810b83e6bd2e5cb0c1.webp)

**因为上提之后鼻腔的通路就阻塞了，气流只能从口腔出了，**而由于你在口腔中舌头做了一个阻挡，所以你的口腔气流是从舌头两边出去的，这时候发出来的音就是边音「l l l」，这时候就算你拿手拿出来，捏着你的鼻子发这个边音都不会被影响，我现在捏着鼻子的，请同学们听一下！「l l l」，因为气流是从口腔走的。


明白了这其中的原理之后，自然成声法就很简单了。我们先按照发边音的要求做好准备之后，我们用手捂住嘴巴，然后你试图来发一下刚才的音。


「…………」


我就用手捂住了我的嘴巴，大家有没有感觉到两腮鼓起，然后伴有一种很憋气的感觉，这就说明啊你符合要求了，然后我们再移开我们的手掌，带上我们汉字字音当中的完整拼音发出来！


「玲珑  嘹亮」边音是不是就发的很漂亮啊？


同样，自然成声法，我们现在发一发鼻音，请按照鼻音的发音要求做好准备，然后用拇指和食指捏住你的鼻孔，你试图在发一个鼻音。


「………」


这就是我用食指和拇指捏住鼻子之后发音的感觉，你听听你的！


是不是有很强的憋气的感觉？如果有的话就说明发音的部位和方法正确了，随后松开拇指和食指，再带上我们汉字拼音的全部发音，发出来的字音就是你们听！


「能耐   泥泞」这个鼻音就发的很清晰，很漂亮了。


**这就是自然成声法。**


![img](https://pic3.zhimg.com/v2-89130b08b03e6082debd9bf8246b285c.webp)

怎样一招破除鼻音和边音发音不清的情况呢，我们只需要掌握一招，这一招就是灵活控制软腭的升降。


鼻音和边音的发音要领当中，它们的共同点是在于都是舌尖前音，需要用舌尖抵住上齿龈。


它们不同点是在于发鼻音的时候，软腭下降，气流从鼻腔走；发边音的时候，软腭上提气流从口腔走。


掌握了这一点之后，我们可以通过用食指和拇指捏住我们的鼻腔通道来检测发鼻音的时候气流是否正确，同时我们还可以用手掌捂住嘴巴来检测发边音时气流是否正确。当气流正确后，再放开我们手的辅助，随后自然成声就好了。
