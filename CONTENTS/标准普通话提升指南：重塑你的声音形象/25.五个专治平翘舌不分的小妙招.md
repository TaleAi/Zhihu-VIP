## 25.五个专治平翘舌不分的小妙招
  



![img](https://pic4.zhimg.com/v2-83eafd69fd4aca8c0a2fb22d7d157cc6.webp)

这节课我要和你分享五个专治平翘舌不分的小妙招。


平翘舌不分可以说是普通话语音面貌当中的第一大现象了，几乎很多同学都会中招这个发音难点，当我问学习的同学们你们最希望立刻改掉的发音顽疾是哪一个的时候啊，大家几乎都会首先说：「我想改掉平翘舌不分的毛病」。


平翘舌发音清晰，和说普通话的时候，口腔的外部开度和内部的操控度有关系。‍‍怎么有关呢？请同学们看一下，我为大家准备的同步版文字资料当中有一张对比表：


外部开度


内部操控


读时都都需要展唇、用力


平舌音 z、c、s 属舌尖前音。


zh、ch、sh 发音时需要舌尖向上翘起，同时向后。


一目了然，‍‍外部开度它们都需要我们读的时候展唇、用力。


![img](https://pic1.zhimg.com/v2-9ecefdd9ba01c6e9006f5a340be33146.webp)

大家可以看到表格下方的图片，这就是展唇用力之后我们看到的视觉效果。‍‍你可以拿着镜子对照着，看看你在读的时候嘴唇外部开度有没有展唇、用力呀？能看见的都是比较明显的，‍‍难点在哪儿呢？‍‍在内部。因为内部操控度，你看不见！来，我们先看一下到底平舌音和翘舌音‍‍它们的内部操控度区别在哪里，然后我们再实践。‍


‍**他们之间的区别就在于，你的舌尖到底是往哪个方向用力的？‍‍平舌音 z c s 你的舌尖需要向前用力，‍‍翘舌音 zh ch sh 发音的时候，我们的舌尖要向上翘起，同时向后用力，‍‍**这时我们头脑当中就有了一个非常清晰的图景了。


![img](https://pic4.zhimg.com/v2-daf8b9ba5ee306d5758e4dfc626eb8d4.webp)

**为什么很多朋友在发平舌音和翘舌音的时候，听起来非常的模糊呢？**


就是因为大家在说普通话的时候，舌头太懒了，躺在那里一动也不动，‍‍**舌头既不向前用力，也不向后用力，最后呢，就导致发出的音，还常常介于平翘舌之间，久而久之，‍‍这个惯性就制约着你，让你平翘舌‍‍不分了，‍‍**比较常见的情况，听起来就是这样，让我们听听看！‍‍‍


‍为了加深同学们对于平翘舌发音内部操控的区别印象，请同学们看看我为大家准备的表格：


![img](https://pic2.zhimg.com/v2-f12dc8abfdef44c667eab81a9501c3ef.webp)

  



![img](https://pic4.zhimg.com/v2-eecab796779037ccdb12f5dee64bf83d.webp)

大家可以看到，**我单独为同学们标示出来了着重的注意点，‍‍z c s 的发音属于舌尖前音，zh ch sh r 的发音属于舌尖后音，同学们可以形象地记为舌尖向前用力和舌尖向后用力。**


**五招专治平翘舌不分**


**第一招就是：请同学们区分舌头的用力方向是向前还是向后。**


‍‍好，‍‍第一招区分清楚之后，你会发现有一些朋友在说翘舌的时候，翘舌发音发音度不够，‍‍总觉得缺了那么一小口没发到位。‍


‍写能写对，一读就错，这种现象是为什么呢？‍‍这很可能就是跟你口腔内部的空间构造习惯有关，‍‍也许你会说：口腔内部的构造还有不同吗？我们都是同样的张嘴说话。


当然有不同，‍‍声音发得很清楚的朋友，我们口腔内部的后声腔‍‍是打开了的，而大部分朋友‍‍平时都是关闭状态。


**来，你跟我做一个动作，‍‍打哈欠，**「**啊**」**，‍‍打哈欠的时候你注意体会一下，我们最里面靠近喉咙部分的软腭就挺起来了，挺起来之后，这个空间就更大了，‍‍这部分就是你的后声腔的口腔后部空间。**


但**平时你说话的时候这部分是坍塌的，‍‍所以我们在发翘舌音的时候，发音听起来不立体，非常的扁，‍‍造成平翘舌不清晰。**


同学们听听看，珊珊老师做示范，**从舌尖向前用力到向后用力，同时伴随着我口腔内部的后面部分空间打开，**你们听！


‍‍


同学们听，是不是越来越清晰，越来越洪亮？


**这就是平翘舌的区分的第二招，口腔内部的后声腔是否有打开。**


‍‍**在课后的深度纠音练习当中，我会带领同学们去做练习，来打开我们口腔内部的后声腔。**


**我们要注意到的第三点区分，‍‍平舌音之间的内部区分。‍‍‍**


同学们如果仔细听，你会发现，‍‍**平舌音 z c s 这三者之间它们都是有细微区别的。**


如果没有区分出来的话，我们的发音听起来就很有可能是这样的！


‍‍


同学们可以听出来，‍‍整个特别的平，‍‍在平舌音 z c s 上面我们几乎听不到区别。来，我们听一下！


‍‍那我们细化的区别到底发生在哪里呢？‍‍同学们，请看我为大家准备的这张表格：


![img](https://pic1.zhimg.com/v2-e55b1fb227feda0962b092db25c3d30e.webp)

Z 和 c 之间的区别，‍‍一个是送气，‍‍一个是不送气。‍


‍我们刚才听到的普通话面貌当中，为什么听起来非常的模糊，‍‍你不能够区分他们之间到底有什么听觉不一样，就是因为，‍‍对这一点没有掌握。


虽然**舌头的内部控制是做对了，但是气流控制没做对，‍‍所谓气流的控制就是送气和不送气的区别。**


**送气和不送气的区别是什么？**你们听！


‍‍**Z ‍‍是不是很轻？‍‍**


**C ‍‍是不是听起来声音更重？**


来，再听一下，z，‍‍c，‍‍z，‍‍c，‍‍z，‍‍c，‍‍z，‍‍c，‍‍同学们还可以看到珊珊老师把自己的发音可视化之后，为大家呈现出来的图片：


![img](https://pic4.zhimg.com/v2-a82ab37bd37c60ab16bac114a22362ca.webp)

我在发 z 的时候，明显看到这个波形更小，发 c 的时候，这个波形是不是幅度更大呀？这就是因为我在发这两个音的时候，我做出了其中的区分，‍‍z c 它们的发音部位和发音方法大体都相同，只是两者之间吐出的气流强弱不一样而已，‍‍**发 z 吐出的气流弱，发 c 吐出的气流强。**请同学们记住，这就是有效纠音的第三招。


**接着我们来说‍‍第四招：**


平舌音 z 和 c 之间的区别大家知道了，那还有一个平舌音 s，它的发音有什么不一样呢？我单独讲 s 这个发音是因为，‍‍它和前两者的区别，不是送气和不送气的区别，‍而是在于我们在发音的时候，口腔内部的气流是从哪儿出去的。


‍‍来，大家跟我做一下，舌尖轻轻地向前平伸，‍‍然后靠近我们的上牙齿齿背，‍‍对，‍‍这时候我们口腔内部就形成一条缝隙，‍‍这条窄窄的缝隙就是我们气流的出路，**我们发 s 这个音的时候，气流就是从我们刚才口腔内部造形中的这个缝隙里挤出来的。**


‍‍来，跟我一起发三遍，‍‍s，‍‍s，‍‍s‍‍


**接着我们来说最后的第五招：**


平舌内部的区别大家都知道了，**翘舌内部也有细微的区别，否则我们的翘舌音全部发成一样的了。没有注意区别，发音就会模糊的。**


‍‍**第五招，它们的区别是什么呢？**请同学们跟我做动作，舌尖翘起来，轻轻地顶住你口腔内硬腭的前部，请大家看到我们同学们标注出来的同步文字版资料：


![img](https://pic3.zhimg.com/v2-e55b1fb227feda0962b092db25c3d30e.webp)

**动作做好了，就是开始进行的气流强弱的控制了，很简单，也是送气还是不送气，**有了前面我们学平舌的基础，这里我们学翘舌就很容易了。


‍‍**Zh ‍‍不送气 ，ch 送气，‍‍它们气流的强弱不一样，‍‍**来，‍‍大家听！


zh，‍‍ch，‍‍zh，ch，‍‍zh，‍‍ch，‍‍大家能感觉到强弱不同吧？同样你可以看到我在做示范的时候啊，声音的可视化图片：


![img](https://pic3.zhimg.com/v2-d34596d109db408996926037142f2727.webp)

你也能够看到，波形完全不一样，这就是其中的细微区别。


‍‍**那么 sh 这个音又和这两个音有什么不一样呢？‍‍**


有了前面的平舌基础，翘舌也是一样的，zh ch sh 它们发音造形上几乎都是相同的，**只是在发 sh 这个音的时候，阻塞的力度没有那么大，口腔内部有一条窄缝，气流就从这条窄缝当中溜出去了。**


![img](https://pic2.zhimg.com/v2-89130b08b03e6082debd9bf8246b285c.webp)

现在让我们来总结一下这节课的知识点。


‍‍如果你不幸中招了普通话发音平翘舌不分的话，你可以从五个方面来纠正它，‍‍平舌音和翘舌音都需要我们在外部的口腔开度上展唇用力，它们的区别更重要的是在内部操控上。‍‍同学们内部的舌头可别偷懒：


一、平舌音舌尖向前用力，翘舌音舌尖向后用力。‍‍


二、平舌音发音比较省力，翘舌音发音更费力，翘舌音要发得够立体，够到位的话，我们要把口腔的后声腔也要打开。


三、注意平舌音内部的区分，在平舌音 z c s ‍‍它们的内部，这三者之间它们的发音造形都是相同的，舌尖向前平伸，抵住上齿背，‍‍它们的不同点是在于，z，‍‍不送气，气流较弱，c，送气，气流较强。


四、同学们要注意平舌音 s‍它的发音造形和 z c 是一样的，但是，‍‍它的气流由于我们口腔内部有一条窄缝，我们的气流就从窄缝当中溜走了，阻塞感没有那么强。


‍‍五、翘舌音当中也有区别，‍‍它们的共同点是在于都是需要‍‍舌尖上翘，抵住硬腭的前部， 它们之间的区别是在于 zh，‍‍不送气，气流较弱；ch，送气，气流较强。


翘舌音 sh 它的气流离开的方式，由于我们口腔内部有一条窄窄的缝隙，所以阻塞感没有那么强，就从缝隙当中流走了‍。‍这一点和平舌音 s 是相同的。
