## 36.课后深度纠音：声母 n、l 的对比练习
  



![img](https://pic1.zhimg.com/v2-83eafd69fd4aca8c0a2fb22d7d157cc6.webp)

亲爱的同学们，大家好，接下来你将跟随珊珊老师一起进入第 16 节课后的深度纠音练习。


‍‍这节课的深度纠音练习，我们要做的是鼻音 n 和边音 l‍‍这两个声母之间的对比变音练习，在课程中我们已经熟悉了它们的发音要领，普通话当中的鼻音 n 和边音 l 是完全不一样的发音部位，我们再来加深一下印象！


‍‍鼻音，‍‍顾名思义，气流一定是从鼻腔走的，‍‍要从鼻腔走，我们就要关闭掉口腔当中的气流通道，舌头向上卷，用舌尖抵住你的上齿龈，‍‍门就建起来了，这就阻挡了气流从口腔出去的通路。‍‍随后我们要把气留更多地进入到鼻腔，这时候你就需要通过把软腭降下来，打开鼻腔的通路，‍‍这时候发出的声音就带有鼻音了。


‍‍把手放在你的鼻翼上面，‍‍感受鼻音的震动，跟我一起做五遍「n n n n n」


‍‍与之相对应的边音气流就不能从鼻腔走了，这时候我们要堵塞鼻腔的通路，升起你的软腭，倒吸一口凉气，‍‍这时候有没有感觉到你的软腭就向上升啦，还可以打哈欠，‍‍啊，这时候你的软腭都是自然地向上升的，‍‍所以，‍‍软腭上升，堵住鼻腔通路之后，舌尖再抵住上齿龈，‍‍好，发出 l，你听，这完全不带有鼻音嘛！


‍‍同学们还可以做一个测试，用手捏着你的鼻子，然后我们再来发这个音，你听，完全不受影响‍‍「l l l l l‍‍」


复习完发音要领之后，现在请同学们跟我一起做对比变音练习。


请同学们看到我为大家准备的同步版文字练习材料，和我一起大声朗读吧！
