## 简介
李白斗酒诗百篇、赵匡胤杯酒释兵权、曹操对酒当歌人生几何，没有一杯酒解决不了的事儿，那些爱喝酒的帝王将相、文人墨客身上都发生过什么趣事？知名网络作家房昊曰天带你一探究竟。


一杯酒里到底藏着多少故事？


「醉里挑灯看剑，梦回吹角连营。」  

「酒醒只在花前坐，酒醉还须花下眠。」  

「葡萄美酒夜光杯，欲饮琵琶马上催。」


古人一壶酒，世间多少事。


知名记者出差去大唐长安，为何在采访李白时被「怼」的哑口无言？  

 东篱之下，经历过风风雨雨的菊花与陶渊明畅谈，无意中发现了哪些秘密？  

 囚禁、捆绑、上树，同样是喝酒，为什么石曼卿能把喝酒变成行为艺术？


杯酒之间，我将带你看遍古代名人身上的奇闻逸事。


这些故事并非天马行空，每个故事都有史料可考，脑洞与专业并存。


谈笑间看点秘闻，煮酒中学点诗句。


本专栏根据真实历史人物的历史文献记载改编，杯酒史话，醉看往昔。
