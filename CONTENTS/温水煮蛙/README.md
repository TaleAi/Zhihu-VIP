# 温水煮蛙

## 简介
如果有个好看又强大的病娇喜欢你，你会怎么办？天降一个「完美」追求者，是美好的灰姑娘逆袭故事，还是恐怖的悬疑推理小说？一切，都没那么简单。所有的偶遇，可能都是一场蓄谋已久的狩猎。


一团乱麻的原生家庭对一个人来说，是一道无法摆脱的桎梏。作为在这段家庭关系里碰得灰头土脸的普通女孩，我却突然遇见了一个不费吹灰之力，就可以解决我所有问题的男人。


这个人还是个有颜有钱、脾气百依百顺的豪门公子哥，豪门准公婆还温和得不像话……简直就是言情宠文里的顶配。 


他的出现，像极了为我量身定做；而他的人，像是会读心术一般。


我的顾虑，我的自卑，我的警惕，在他面前全都被步步瓦解。他用最光明正大而又悄无声息的方式，一点点渗透了我的生活。如同是一只温水里的青蛙，我被困在他给的舒适里一点点沦陷。


可事情似乎，没这么简单。他无处不在的身影，身边人若有似无的提醒，怪事的接踵而至，让我不得不怀疑，他的背后，是否藏着什么秘密？我们的恋爱，究竟是甜甜的童话故事，还是他另有目的的安排？


如果说每个灰姑娘都可能会有一个「霸道总裁爱上我」的言情梦，那么这个故事，就是揭开美梦背后的冰山一角。跟着故事中的女主，以第一视角，一步步去发现那些不为人知的秘密吧……


你确信你看到的善，真的是善吗？你看到的恶，又真的是恶吗？
