## 简介
犹记当日初相见，花落肩头惊艳。笑似春风拂面，眼底风月无边。少年白衣翩然，颦笑乱我心弦。


她这一生仿佛过了几世。


穿越而来成为平平无奇的丫鬟，恪守本分，机智护主，因缘巧合遇见了一位翩翩佳公子。


初次听闻他的名字，她便觉得在某段不为人知的岁月里，她曾将他的姓名在心里描摹了千万次。


于是，心起涟漪，相思暗许。


万没有想到，丫鬟的身份竟是一场精心的设计。


她忘却的前尘往事里，有与另一位白衣男子的海誓山盟。


她忘了他，也不爱他。


于他而言，前者虽伤筋动骨，后者才是致命一击。


命运面前，他深感无能为力，眼睁睁地看着她喜欢上别人，眼睁睁地看着过往的画面与现实交叠在一起。


他固守着一份没有回应、近乎绝望的喜欢，一面强装云淡风轻地对她讲，


若放手能换她平安喜乐，那他甘愿成全。


直到那段记忆解封，蓦然回首，她终于发现，


一眼万年的倾慕是你，铭心刻骨的爱恋是你，毫无保留的欢喜也是你，往后余生都是你。
