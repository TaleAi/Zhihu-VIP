## 简介
在医学史上，推动医学进步的除了大名鼎鼎的医生和生物学家，还有很多籍籍无名的小人物。

在这个病人故事集里，你可以看到原本一如你我的普通人，染上无法治愈的疾病后，怎样推动医学的进步，造福人类的发展。


 现代医学发展到今天，很多以前的「疑难杂症」，都已有了根治方法，就连让人闻风丧胆的癌症，也有了治愈的可能。 


但是你可能不知道，在人类与疾病作斗争的漫长历史中，许多被我们忽略的名字作出了不可忽视的贡献。他们是医学史上的「明星」： 


既是侏儒，又是巨人的亚当•雷纳尔，提高了我们对垂体瘤与激素生长水平的认识； 


记忆只有七秒的职业病人 HM 先生，以一己之力推进了神经科学对「记忆」的认识； 


长着三条腿的弗朗西斯科•伦蒂尼，成功地纠正了大众对寄生胎的歧视； 


还有被误解的艾滋「零」号病人盖尔坦•杜加，永生的海拉，发现色盲的色盲科学家道尔顿…… 


愿你有所收获。
