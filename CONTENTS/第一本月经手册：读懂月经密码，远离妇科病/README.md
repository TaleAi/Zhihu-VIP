## 简介
妇科医师将为你讲解那些你难以启齿的问题，教你如何轻松辨别月经情况、识别身体健康状态、正确处理异常问题。


女生在经期时面对各种情况，总会陷入恐慌，不知道自己是怎么了又觉得难以启齿不知道该询问谁。你是否也存在以下困惑：


* 月经的颜色鲜红、发黑，还有血块？为什么我的和别人的不一样，这是正常的吗？
* 经期或短或长，太任性？淋漓不尽又是怎么回事？
* 非经期的出血是怎么回事？褐色分泌物意味着什么？

本专栏将由妇科医师从专业的角度为你剖析究竟怎样的月经是正常的，如何判断自己是否需要去医院检查。
