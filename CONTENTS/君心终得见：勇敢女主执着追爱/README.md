# 君心终得见：勇敢女主执着追爱

## 简介
勇敢可贵，真爱无敌。


她们是独立勇敢的女主，或遭遇挫折，或并不完美，却仍有勇气迈出追逐的步伐，将各路男神收入麾下，终成眷属。


**尘封在我心**


送年货撞上疫情，我被迫在竹马家隔离，并在住进去的第一天就与他「坦诚」相见。


本以为能近水楼台先得月，却意外发现他的邻居，是他三年前暗恋的女神。


伤心过度下，我灌醉自己笨拙撩拨，却反被竹马表白。


「我不想和你做朋友了，还是换一种关系吧。」


我这才知道，什么叫真正的送货上门……


**觅旧欢**


我出身青楼，却因和公主样貌相似被送去和亲。


作为对战败国的羞辱，我被赐给了身有残疾、命不久矣的九皇子陆斐。


原以为很快就要守寡，却在日常相处中对陆斐渐生情愫，笨拙试探引诱。


一次刺杀中的相救，我意外发现了陆斐「残疾」的真相，和他布下的精妙棋局。


待一切尘埃落定，真正的公主却忽然出现，揭穿了我的真实身份。


也是这个时候，我才知道，原来从一开始，陆斐就知道我是谁……
