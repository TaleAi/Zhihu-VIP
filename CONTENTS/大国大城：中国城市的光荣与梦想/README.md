## 简介
每一座城市都有自己的「记忆」。


这个记忆可以是人物、地标或者往事。


只要稍微提起它们，就能立马想起这座城市穿越历史风云，从草创到辉煌的跌宕历程。


城市的历史，不应该是大而全的「图表数字」，不应该是冰冷枯燥的「标签」。


应该是生意盎然、栩栩如生的「人物」和其背后的「故事」。


本专栏聚焦于各大城市的「人物」、「地标」或「往事」：


* 广州篇：  

霍英东的广州往事，冒死为中共运输药品；  

白天鹅宾馆始建，给所有华商资本的定心丸；  

海珠桥两次轰炸、三次建修，广州人的爱恨和希望；
* 哈尔滨篇：哈尔滨本土「流氓」乔四的传奇故事；  

中央大街， 使得「摩登」最早在哈尔滨流行；  

田家烧锅与中东铁路和哈尔滨建城始末；

通过找寻这些熟悉标签背后隐秘的历史，带你领略一座城市的风情，以及城市背后的国家和民族命运。
