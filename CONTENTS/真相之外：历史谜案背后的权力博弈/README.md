## 简介
两宋时期反转再反转的真假柔福帝姬案，唐高宗时期太子李弘暴毙之谜·····翻开历史长卷，总有一些谜一样的事件，至今依然争议不断。谜案背后，往往伴随着权力的博弈和抗争，而博弈背后，或许，另有真相。



**真假柔福帝姬案**


南宋时期，一个从敌营逃回的女子自称是柔福帝姬（公主），而且得到了南宋官方的承认。更不可思议的是，多年以后，南宋皇室不顾脸面，公开打自己的脸，承认这个「公主」是假冒的。也由此，在历史上留下了一桩说不清道不明的悬案。


**太子李弘暴毙之谜**


太子李弘是唐高宗和武则天最看中的儿子，他才华横溢，为人仁孝，却年纪轻轻就突然暴毙。关于他的死因，一直争议不断，有说他是因病自然死亡的，也有说他是被母亲武则天毒死的。李弘究竟是怎么死的呢？


······ 


看似不可思议的谜案背后，充满了猜忌、阴谋与杀机，伴随着各个利益集团的博弈与抗争。


本专栏中，百万畅销书作家王伟，深挖有争议的历史谜案背后的史料，结合当时的社会背景，对这段历史进行了更深层次的剖析，为我们提供了现有真相之外的另一种可能，也让我们对当时的社会制度和政治经济，有更立体深入的认识。


来，拨开历史迷雾，一起踏上寻找另一种真相的旅行吧～
