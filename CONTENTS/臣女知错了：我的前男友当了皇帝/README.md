# 臣女知错了：我的前男友当了皇帝

### 作者：墨卿

### 分类：古代言情

### 状态：已完结，共 20 节

### 存档日期：2022-06-03

## 简介
退婚半年后，前未婚夫他成了皇帝……


他登基后的第三天，就召我入宫。


原以为他念在昔日旧情，会高抬贵手放我一马，谁知该男人心性大变，好像换了一个人似的。把我叫进宫，不是抄书就是给他守夜。


他捏着我的下巴，一脸气愤，「朕说过，你会后悔的！」


我扑通一声，熟练地给他跪了，「陛下，臣女知错了！」


……


那方宝座，真的会改变一个人的心性吗？


而我喜欢的少年，又真能如他少年时承诺的那样，爱护我一生一世吗？
