## 18.维生素（上）：维生素 C 怎么补？
与脂溶性维生素同类的还有一类维生素，叫水溶性维生素。这两类维生素就像兄弟俩一样，同属于维生素这个大家族。不过这两「兄弟」的性格爱好各不相同。


上节我们讲了，脂溶性维生素喜欢跟脂类在一起，而水溶性维生素喜欢水，但它们却又特别「娇气」，很容易在烹调过程中流失，还容易受到光、热和氧化作用的破坏，所以人体要吸收它们就比较困难。可是它们又不能在人体内储存，很容易被排出体外，这就要求我们必须每天进行补充，才能保证身体内有足量的水溶性维生素，满足机体的需要。


水溶性维生素包括维生素 C 和 B 族维生素。我们就先来了解一下维生素 C，看看维生素 C 对我们的身体有哪些功效？我们该怎样补充，才能满足身体所需呢？


首先我们看一下维生素 C 的营养功效。


**一、维生素 C 的营养功效**


维生素 C 也叫抗坏血酸。那为什么要叫这个名字呢？我先给你们讲个小故事。


在 13 世纪时，以罗马教皇为首的西欧人对东部地中海沿岸的国家发动侵略战争，但行军过程中，很多士兵患上了出血病，牙龈出血、鼻子出血，皮肤稍微有点儿破损也总不好，最后死了不少人。后来人们就把这种病称为坏血病。


随着医学的发展，人们发现吃含维生素 C 的食物能防治坏血病。1747 年时，苏格兰一位医生还专门用新鲜的蔬果来治疗患坏血病的人，结果发现柑橘和柠檬的效果最好。现在我们知道，这是因为这两种水果中维生素 C 含量特别丰富，所以后来人们就管维生素 C 叫抗坏血酸了。


除此之外，维生素 C 还能增强人体免疫力，预防感冒；促进人体肌肤再生，并参与胶原蛋白的合成，让女性的肌肤更加美白、润滑；还能帮助人体对铁元素的吸收，你在每餐食物中多摄入 50-100 毫克的维生素 C，就能让身体对植物性食物中的铁吸收率增加 2-3 倍。


现在你们了解到维生素 C 对身体的重要性了吧？


不过，维生素 C 却不能被我们的身体自行合成，只能从外界摄取，而且它也是所有维生素中人体每天需要量最多的维生素。但维生素 C 偏偏又是个任性而脆弱的小家伙，很容易从人体内流失，还会在空气中遇热、遇碱时被破坏。有时我们明明吃了一大堆富含维生素 C 的食物，结果可能并没有摄入多少，所以人们也认为水溶性维生素最难补。


那我们是不是对它一点办法也没有呢？当然不是，掌握了正确的方法，我们一样能摄入足量的维生素 C。接下来，我们就看看怎样防止维生素 C 被破坏或流失。


**二、防止维生素 C 被破坏和流失的方法**


水果和蔬菜中的维生素 C 含量最为丰富，尤其是色彩斑斓的水果和绿黄色蔬菜，比如鲜枣、柑橘、柠檬、草莓、猕猴桃、柚子、橙子，以及小白菜、花菜、菠菜、卷心菜、芥菜、苦瓜、青椒、西红柿、豌豆等。


但是，如果你在吃这些食物时，把水果榨汁、加热，或对蔬菜进行焯水、长时间高温炖、炒等，都会使其中的维生素 C 大量流失。那我们该怎么吃呢？


**在吃水果时，最好洗净后直接吃，不要榨汁，也不要削皮。**因为你在榨果汁过程中，高速旋转的刀片就会破坏掉水果中的所有细胞，并且又把所有物质混合在一起，这就会令水果中的维生素 C 与空气或其他细胞中的氧化酶相遇并发生作用，导致维生素 C 大量流失，所以你喝的果汁中维生素 C 含量其实已是微乎其微了。果皮中的维生素 C 的含量要比果肉多 2-5 倍，从这一点上讲，我主张大家在吃一些不是必须去皮的水果时，尽量不去皮，比如苹果、梨、葡萄等，洗净后连果皮一起吃掉最好。


**在处理蔬菜时呢，尽量不要长时间清洗和浸泡，并且一定要先洗干净后再切，**否则你切完后再洗，其中的维生素 C 就会通过切口溶于水中流失掉了。如果能生吃的蔬菜，也尽量不加热，如西红柿、黄瓜等。如果要烹饪的话，尽量通过急火快炒或勾芡等方式降低维生素 C 的损失。另外在炒豆芽、土豆丝时，也可以在炒加少许醋，除了能提鲜外，还能保护其中的维生素 C。


维生素 C 虽然很容易流失，但我们的身体各项功能又必须有维生素 C 的参与，那么我们该如何来补充它呢？接下来我们就看一下。


**三、维生素 C 怎样科学补充**


根据中国营养学会推荐，我国成年居民每日膳食维生素 C 的推荐摄入量为 100 毫克。怀孕女性要在此基础上要增加 15 毫克，哺乳期女性则要增加 50 毫克。但婴幼儿和青少年应适当减少，一般婴幼儿每天摄入 40-50 毫克就够了，青少年摄入量在 50-100 毫克左右。


前面我们了解到一些水果和蔬菜中富含维生素 C，所以补充维生素 C 最安全有效的方法就是多吃这类新鲜的水果和蔬菜。比如鲜枣，每 100 克鲜枣果肉中就含有 250 毫克的维生素 C，100 克鲜枣有多少呢？根据大小不同，大约为 5-8 颗。也就是说，你每天吃 5-8 颗鲜枣，就能为身体提供 250 毫克的维生素 C 了。


再比如，100 克新鲜菠菜中含有维生素 C 约为 60 毫克，也就是我们单手握住的一把的量；100 克新鲜彩椒中含有维生素 C 可达 70 多毫克，其大小相当于女性的一个拳头大小。当然，焯水或烹饪后会使其中的维生素 C 损失一部分，但总体来说，只要我们平时注意多摄取新鲜的水果和蔬菜，通过一日三餐是完全可以满足身体对维生素 C 的需求的。


不过这里还有个很有趣的地方，我要提醒大家注意一下哦！就是人体对维生素 C 的摄入和吸收量，并不是你摄入得越多，它被人体吸收的就越多。当你摄入 30-50 毫克维生素 C 时，人体对它的吸收率是 100%；但当你摄入 90 毫克以上时，它的吸收率就会下降到 80%；以此类推，你摄入得越多，人体对它的吸收率就越低。而当你摄入超量时，如超过 1000 毫克，那么多余的维生素 C 反而会损伤身体，引起腹泻、腹痛等，长期过量还会出现结石。


这就提醒大家，如果你想通过服用维生素 C 制剂来补充，必须在医生指导下服用。但如果你是从新鲜的蔬果中摄取的话，那么完全不必担心摄入会超量。因为不安全摄入量折算出来的食物量，就算你是个「大胃王」的话，一天也吃不下这么多食物。这里主要是提醒服用维生素 C 制剂的朋友注意的。


**（本期小结）**


这节课我们主要讲了水溶性维生素中的一种，也就是维生素 C，它对人体的作用，以及日常膳食中如何防止维生素流失，我们该怎样合理补充维生素 C 等内容。我再来简单总结一下：（一）维生素 C 为水溶性维生素，易溶于水，它们无法在人体内自行合成，也不能大量储存，常会随着尿液排出体外，所以必须每天补充；（二）维生素 C 性质很不稳定，容易在空气中遇水、遇热、遇碱时被破坏；（三）一些新鲜的蔬菜和水果中富含维生素 C，日常膳食应注意增加摄入量；（四）如果想通过制剂补充维生素 C，一定要遵医嘱。


**维生素 C 的营养功效**


1.被称为抗坏血酸，可防治坏血病。


2.增强人体免疫力，预防感冒等。


3.提高人体对铁元素的吸收律，每餐多摄入 50-100 毫克维生素 C，可使身体对植物性食物中的铁吸收率增加 2-3 倍。


**富含维生素 C 的食物**


1.水果：鲜枣、柑橘、柠檬、草莓、猕猴桃、柚子、橙子等。


2．蔬菜：小白菜、花菜、菠菜、卷心菜、芥菜、苦瓜、青椒、西红柿、豌豆等。


**怎样防止维生素 C 被破坏或流失**


1.避免长时间清洗和浸泡富含维生素 C 的食物。


2.先洗后切，尽量生吃，不要榨汁、加热。


3.烹饪时尽量采取急火快炒或勾芡等方式，还可在烹炒时加入少许醋，保护其中的维生素 C。


**日常怎样补充维生素 C**


1.食补：每天保证科学、均衡的饮食，食物中一定要包括富含维生素 C 的新鲜的蔬菜、水果等。


2.补充维生素 C 制剂，一定要在医生指导下服用。
