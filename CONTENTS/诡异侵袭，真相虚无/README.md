## 简介
真相究竟是什么？你有勇气揭开吗？


**《偷窥日记》**


我有偷窥癖，偷窥对象是我的美女房客。


每天，我都通过针孔摄像头，窥视她的生活。


我曾以为，我是最了解她的人。


直到今天，通过监控，在她卫生间的浴缸里——


我看到了，我的尸体。


**《操控》**


我的老公伙同小三设计我。


他们计划逼疯我，夺取公司与财产。


他们以为自己胜券在握了。


可是，成为疯子，正是我想要的啊……


**《土枪》**


打鸟是我们村过年的传统项目。


初一这天，第一次举枪，我打中了十米开外的稻草人。


暮色里，我隐约看到稻草人身后闪过一个人影……


糟糕，我是不是射中了他。


两天后表弟告诉我，同村的傻娃子，失踪了。


跟我一道去打鸟的强子，也失踪了。
