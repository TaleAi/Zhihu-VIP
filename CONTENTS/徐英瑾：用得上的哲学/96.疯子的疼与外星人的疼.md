## 96.疯子的疼与外星人的疼
**导语**


很多情侣在恋爱中都会烦恼这样一个问题：对方到底爱不爱自己，有多爱？为了证明这个问题，常常就会导致两个人的矛盾。在这种情况下，如果懂得对于人类感觉的同一性理论和功能主义理论解释的话，可能就不会为这个问题困扰了。那么这两个理论是如何来解释爱的呢？我们先从它们如何解释疼痛说起。


本集节目将包含以下要点：1. 用同一性解释疼痛；2. 用功能主义解释疼痛；3. 关于爱的功能主义解释。


  



各位听友大家好，我是徐英瑾，非常开心和大家再来讨论用得上的哲学这个话题。今天我们所讨论的话题也是接续上面一次节目的话题，还是说疼。


上一次我们讲的是私人语言之不可能性，如果具体到疼痛问题上，也就是说疼痛并不是某个人的私有财产，只能够为他本人所理解。今天我们要接续这个话题往下谈，我们要谈谈这么一个概念，即「疯子的疼」和「火星人的疼」之间的区别，哪种疼算真的疼。


**用同一性解释疼痛：疯子的疼不是疼**


大家听到这样一个题目以后可能会觉得有点莫名其妙：到底什么是疯子的疼和火星人的疼？火星人又是谁？这里面有一个哲学上的梗。


有一个美国哲学家大卫·刘易斯，他写了一篇论文，这篇论文的题目叫《疯子的疼和火星人的疼》。他为什么要提这个话题？刘易斯在维特根斯坦之后，继续对疼痛问题进行了深入的思考，并且他的思考也涉及到了语言哲学和心灵哲学的交汇之处。


在刘易斯看来，有两种关于疼痛的理论看似都是有道理的，一种关于疼痛的理论是由神经科学给出的，就叫所谓的同一性理论。同一性理论我们在讲心灵哲学的时候也已经说过了，这里的「同一性」指的是神经状态和特定的心理状态之间的同一。


比如什么叫疼痛？按照一种非常简化的神经科学故事，肯定有一些特定的神经结构它发生了些变化，你才能够感到疼痛。哲学家往往会简化神经科学的故事，仅仅说一种叫 C 纤维的东西它被激发了以后，这种神经状态就对应了疼。所以我看你疼不疼，我就看你的神经状态本身的活跃程度，是不是 C 纤维激发了，如果激发了，你也就疼了。


由此拓展开，如果我有一个非常精密的设备来监测你的大脑的所有的运作，甚至是化学物质的分泌，你说你现在是不是处在幸福感之中，可不是由你自己嘴上说了算，你可能会假装你很幸福，其实你心里一点都不幸福，但是机器如果能够监测到你的大脑分泌了很多多巴胺，毫无疑问我坚信这件事情是对的。为什么？因为心脑同一论告诉我们，所有的心理活动背后都是有一个类型的神经活动托底的。


讲明白这个理论以后，大卫·刘易斯就要挤对人了，他试图给出关于这个理论的一个反例，这个反例是什么？是疯子的疼。


什么叫疯子的疼？就是这个疯子他真的感受到疼的话，你如果能够有非常厉害的设备去监测他的神经活动，你的确发现他的 C 纤维得到了激发，这和我们正常人得到激发是一样的，按照同一性理论来说，我管他是不是疯子，显然他就是处在疼痛状态。


但是这个故事并没有那么简单，它还有另外一方面。大家先想想看自己是在什么情况下疼的？被打了的时候，疼；被开水烫了，「这开水好烫」；不小心一脚踩到图钉上，血流出来了，实在是太疼了；或者晚上吃了一些不干净的东西，胃经挛了，好疼……都是因为我们身体受到了损害，而且疼的时候就会伴之以大喊大叫，掉眼泪，哭天喊地。


刘易斯笔下的疯子就非常奇怪了，他什么时候疼？他数学题目做出来的时候，「好疼，我这数学题目做出来了」；考试在班上得了第一名，「疼死我了，我怎么考试第一名」；女朋友亲他一下，「女朋友亲我了，疼」……这在我们看来没有一件和疼相关。他脚上踩了个图钉，他应该疼了，他笑得好开心，「今天被图钉戳了一个洞」……就这么一个人，他所有的疼痛和我们疼痛是相反的。


这有点像我们以前所说的光谱颠倒的案例。光谱颠倒就是指你给他闪红灯，但是他看到的是绿灯，但是他嘴上说的还是「我看到了红灯」，为什么？他的神经接错了一次，他的语言和他的感觉器官之间的神经又接错了一次，结果负负得正。


所以，我们这里所说的疯子的例子，和前面所说的光谱颠倒的例子有一点是相似的，他的神经接错了，但是他不是接错两次，他就接错一次，如果接错两次负负得正，你在外面就看不出他的异常了，但是疯子因为他只接错了一次，所以他的语言表现还是和大家相反的，否则你怎么说他是疯子？所以在这种情况下，这些疯子的行为就体现出和我们正常人系统性的偏差了。


在这样的一个情况下，你愿意不愿意说这些疯子他们处在疼痛状态中呢？请注意，比如他考试考了一百分的时候，他疼得不得了，在这种情况下，他的神经活动的确体现为 C 纤维的激发，和我们在疼痛的时候的神经活动是一样的，只是非常奇怪的是，他并不是在应该疼的时候疼的。


就好比说如果不用疼痛来说，什么时候我们可能会发生一些肠胃的不舒适？是在吃了不干净的食物的时候，肠胃会产生不舒适感，如果我们吃的都是非常干净并且养胃的东西，我们的胃应该感到很舒服。如果也有一些人，他系统性的基本反应和我们不一样，他们会在吃到很不干净的东西的时候，他们的肠胃很舒服，他们吃到了很干净的食物，他们会发生各种各样的腹泻或者胃痉挛的问题，因此我们也说他们显然是群不正常的人，和我们是系统性相反。


好了，有人就会说：你现在讲到这里，我们大家都有点头大了，难道世界上真有这种疯子吗？我们是不是能够在医学报告的文献里面发现这种类型的疯子？


我认为这个问题不是很重要，可能就是我们哲学家编出来的。刘易斯说，可能世界上没有这个疯子，我编出这样一个思想实验的 point 是什么？我只是想问大家这个问题，即疯子在产生所谓疼的感觉的时候，你认不认为疯子所具有的那种疼是一种真正的疼？


很多人都倾向于说 no，也就是说他们会认为，疯子的疼尽管他的基本的神经科学的基础是和我们人类一样的，但是他疼得也太不是时候了。


我们感觉到疼这件事它是有两面性的，一面它可能体现为神经的一些反应，另外一方面可能会体现为疼所扮演的因果角色，也就是说肯定你的身体遭到了一些损害了，在这样的情况下你才会感到疼。但是这家伙完全是反过来的，身体没有受到损害反而感到疼了，身体受到损害反而感到爽了，这不是很怪吗？所以这些人的疼痛所扮演的因果角色，和咱们的疼痛所扮演的因果角色是系统性的不同的，所以咱们就得出了一个结论，他们的疼不是我们的疼。


**用功能主义解释疼痛：火星人的疼是不是疼？**


讲到这里就特别好玩了，讲到这里我们关于疼痛的一个标准就不是同一性理论了，同一性理论讲的是疼痛的本质就在于它的神经学基础，现在我们不是按照神经学基础作为评判标准，我们拿什么做评判标准？我们拿疼痛在你的整个行为序列里面扮演的因果角色。这种观点是什么？就是我们以前在心灵哲学里面所讲的另外一种理论，就是功能主义。


疼痛的功能是什么？就是避免各种各样的伤害。比如一个人如果老感觉不到疼，脚上踩了个图钉他都觉得好舒服，就再踩个图钉，你再踩几下你脚就烂了，对你的生活绝对没有好处，你的身体会因此迅速坏掉，所以为了使得我们的生物体能够在这个残酷的世界中生存下去，我们就会感到疼，知道疼了以后就知道怕，知道怕了以后就不会去做蠢事，这具有非常显著的进化论意义，这就是功能主义的解释。至于你最后的疼痛，它是实现为 C 纤维的激发还是什么，站在功能主义的立场上它不是特别的关心。


比如举个例子，如果你要做出一个机器人，让机器人也会有疼痛的感觉，机器人怎么可能有 C 纤维的激发？机器人它是什么？它是建立在以硅元素为基础的物质载体上，它并不是建立在以碳元素为基础的物质载体上，咱们人类是建立在碳元素上，所以你希望机器有神经，岂不是缘木求鱼吗？这是完全做不到的。


现在我们讲到了功能主义以后，大家就觉得，那就好啦，有了功能主义理论以后，咱们就不用管疯子了。但大卫·刘易斯又开了另外一个脑洞，就是这篇文章的另外一部分，它讨论的是什么？它讨论的是火星人的疼。


火星人是对外星人的一种总称，并不是真的火星上有人，火星上显然是没人的，我们已经知道了。所谓的火星人，在大卫·刘易斯的笔下，他的身体结构显然和我们有很大的不同，他根本就没有 C 纤维，他身上都是很多管子和管道，疼痛也并不体现为 C 纤维的激发，而是体现为这里的水管里面的水压大一点，身体另外一个地方的水管里面的水压小一点，可能他的身体的水管到处都有一些压力阀，能够检测他体内水管的流动情况，这就是火星人的身体状况。


我们也不管这件事情在生物学上是不是可能，是不是有一个物种可以进化成这样子，咱们就不管这事了，这是个哲学的思想实验。


这些火星人他们的行为和我们的行为之间的分别是什么？应该说没有太大的分别。如果从行为角度上来看，你揍那个火星人一通，火星人如果打不过你，他也会逃，一边逃的时候，一边往外喷水，类似于我们地球人在被别的地球人暴打的时候，口吐白沫，落荒而逃的样子，他也四处乱喷水。


这时候，如果有一台机器能够检测他们身体各处水管的压力，就会发现水管的压力也发生一些变化了，很有可能他们在疼痛的时候，他们某个疼痛地方的水管的压力就会比较高，而这种高的压力就会传导到他身体的其他地方，让他做出紧急避险的动作，大致就是这么一回事，「啪啪啪」，火星人就这样逃走了。


好，我现在讲完火星人的故事了，我就再追问大家一个问题：当火星人被暴打以后，哇哇乱叫、嘴里乱喷水的时候，火星人是不是处在疼痛状态？


我不知道大家的直观是什么，我倾向于认为这些火星人真的疼了。至于他背后的生物学故事和我们没有太大的关系，我认为这一点是可以理解的，因为火星人他的进化环境和我们地球不一样，也许他进化出像水管这样的身体结构是适应于那样的环境，这些底层的故事相对是可以被忽略的。


好，讲到这一步的时候，我们关于疼痛这件事情好像就有了比较清楚的一个观点了，这个观点就是好像同一性理论是有问题的，但是关于因果的功能主义的角色理论，或者说对疼痛作出功能主义角色刻画的这样的一种理论，听上去好像更加的有道理。也就是说功能主义比同一性理论好像更胜一筹，更符合我们的直觉。


我个人认为，功能主义能够胜出还有一个很重要的理由，也就是说站在语言哲学的立场上来看，我们的疼痛指派是咱们日常语言机能的一部分，这在心灵哲学里面有一个专门的词，就是常识心理学。常识心理学就是我们在日常语言当中怎么样来指派一种心理状态，把疼痛、忧虑、焦躁这些心理词，指派给张三、李四这些不同的心理主体。


这种能力在原始人那里恐怕也就有了，在唐朝人、清朝人那里显然也都有了，尽管唐朝人、清朝人也根本不可能懂神经是怎么回事，他们可能完全不知道现代医学。这就说明什么问题？这就说明疼痛指派主要是根据一些宏观的现象，而不是根据微观的底层的现象，所以仅仅站在神经科学的层面上来做出疼痛的理论，有可能它的视野就会变得比较狭隘。


**我们玩弄的是一个双重标准的游戏**


但是讲到这一步，我觉得站在刘易斯的立场上他还是要做点修正，因为刘易斯本人他还是比较主张不能够对同一性理论赶尽杀绝，尽管他是更同情功能主义理论。


为什么？因为同一性理论在很大程度上也是有道理的，也就是说至少对于咱们这个物种来说，疼痛真的是体现为 C 纤维的激发。如果真的出现疯子的话，按照刘易斯的观点，这些疯子的疼痛虽然他 C 纤维的激发的现象是存在的，但我们为什么要把他排除到疼痛的例子之外？这是因为他的疼痛所扮演的因果角色和我们不一样，所以在这种情况下，我们可以认为疯子是咱们这个物种当中的例外。


请注意，这里我们玩弄的是一个像双重标准一样的游戏。疯子是咱们地球人的成员，但是我们认为疯子的疼痛不算疼痛，我们用的是功能主义的一个标准把他排除出去了，但是我们在讨论疯子以外的其他正常人的时候，我们用的却不是功能主义的标准，我们也可以使用神经科学的标准，否则你生了病以后，医生凭什么给你做 CT？CT 难道不就是用神经科学的标准来衡量你得了什么病吗？这里貌似就出现了双重标准。


但这里为什么允许出现这种双重标准？实际上在正常人那里，神经科学的标准和行为因果的标准它们没有产生冲突，因为它们没有产生冲突，所以你用哪个标准都是 OK 的。但是在疯子那里，关于疼痛的行为因果或者是功能主义的标准，与关于疼痛的神经科学标准打架了，这就很麻烦了，你就得分一个大小了，你到底要听谁的意见。


这就好比在一个公司里面，经理的意见和董事长的意见如果都一致，这时候你就不要考虑到底是遵从董事长的意见还是经理的意见了，因为这是一回事。但是如果两个人产生冲突了，你就得思考了，谁的官大，当然董事长他的权力更大，你就得听董事长的了。


所以按照刘易斯的观点，虽然从骨子里面我们可以说功能主义的标准是通吃天下的，但是如果功能主义的标准和神经科学的标准在正常人那里没有产生冲突，从医学实践的方便上来看，你使用神经科学的标准是非常 OK 的，是没有问题的。这个观点听上去也为那些神经科学的拥戴者挽回了一点面子，所以这种观点听上去就比较圆融。


从总体上来说，今天我们这期节目的观点，也应和了维特根斯坦对于私人语言说的反驳态度，因为私人语言说它的实质就是把疼痛的标准说成是当下的感觉，不愿意把疼痛的标准诉诸到公共的尺度里面去加以考量，在这个问题上，我认为大卫·刘易斯的观点接续了维特根斯坦。


**如何用得上？**


讲到这里，我就要谈一谈怎么把今天这一集节目的内容用到日常生活中去了，在这里我想谈谈爱情。


大家也许会很震惊，怎么说了半天疼，最后来谈到爱情了，这两者相关吗？难道我像那个所谓的疯子一样，在谈到爱情的时候会感到疼吗？当然不是，我谈到爱情的时候还是觉得甜蜜的。但大家要想明白，今天所说的这个疼只是对于各种各样感觉的一个总称，因此，如果我们关于疼痛的功能主义的解释方案能够行得通的话，也可以把它用到其他的感受上，比如关于爱情的感受。


我们知道，爱情的感受和疼痛一样，当然它是甜蜜的，但是它和疼痛一样都是难以名状、难以言说的。有时候我们会用一些词，「我喜欢上他了」「喜欢是什么感觉？」「说不清楚，反正就是那种甜丝丝的感觉，小鹿乱撞」——你以为胸口真的有小鹿啊？你真被小鹿撞一下就撞死了，但是我们老说这话。所以，爱情肯定带来某些感受了，而且我相信这些感受也可能伴随一些神经的活动，可能也牵涉到了微观层面上的一些生物学的事件，比如说多巴胺的分泌等等，或者你很激动要又一次见面了，肾上腺激素就开始分泌了。


也有人专门从生物学的角度来研究所谓的七年之痒。老夫老妻之间的七年之痒最后体现为什么？就体现为那些和激情有关的化学物质，在老夫老妻那里估计都被消磨得差不多了，两个人见面的时候也不会小鹿乱撞了。「都老夫老妻了，孩子都好几岁了，我们还是讨论到哪去买学区房」，基本上就像合伙人的态度了。


这很可能就引出了一种观点，我们可以从生物学的角度或者是神经科学的角度、生物化学的角度去讨论爱情的物理基础，这种观点是什么？是对于爱情的同一性理论的解读。但是，这种解读是不是一定很正确？


我们就会看到有一些人，他和一般的人有点偏差，这个偏差也不至于偏差到像疯子的程度，也就是说爱你的时候他觉得心里像刀锥一样的疼，不是这样的，他可能比较冷淡，如果你去测他的化学物质，分泌的可能也很少。女孩子抓住男孩子的手说：「你爱我吗？」男孩子说：「我爱你。」「你爱我爱得到底有多深？」男孩子他就说：「我很烦，我晚上工作，我领导要我加班。」「你没有那么爱我！」女孩子嘴一撅就走了。


很多女性在问这个问题的时候，她们要追求的似乎是一个表面的、肤浅的口头承诺，但是对于爱情的承诺它本身是否有真实的情感基础，我认为是有很大的问号的。因为我要指出，有些人天生就比较木纳，不喜欢说话，这种男性应该说比例还是蛮高的，或者有些人天然的他的情绪就不是很激烈，他并不像意大利人那样热情似火。


在这种情况下，你怎么判断他爱不爱你？你就要从爱这件事，来看看它在整个行为因果角色里面所扮演的角色了，这还是要用疼痛的功能主义理论去套。疼痛的功能角色是什么？让你趋利避害，让你受到伤害可以逃。爱的标准是什么，就是如果他真爱你的话，在遇到重大关头的时候他会先来救你。所以你不要去问「如果我和你妈同时掉水里，应该先救谁」这样的问题，而是如果真的碰到这个问题，看他是怎么做的。


当然，有人说为了测试他专门做出这个局，这可能是不太道德的，但是如果两个人真实地面对了一些共同的挑战，而男性在这种情况下的确做出了一些具有责任感的行为，你就可以从他的这个行为的后果上来判断他是爱你的。


不过有些人非要杠精，就要说了：那个男人其实并不爱我，他之所以救我，只是出于责任心。对于这个问题分得太清楚，是增加人生的困扰，明明那个男孩子已经帮了你的大忙了，你还觉得他是出于责任心，而人生的痛苦就在于某些问题过于较真，这样会平添烦恼的。
