## 简介
埋藏在暗涌之下的脑洞一旦开启便来势汹汹。


如果人的影子也具有思维意识，并且让影子和本体交换了身份，会演绎出怎样一种新的人生？


如果人性有力学性质，那么受力方向是向上还是向下？在听到世界上最好听的音乐之后，情绪是飞升，还是坠落？


*《我和影子交换了余生》*


男主深夜醒来，发现自己跟影子交换了身份，男主变成了一具没有厚度不受控制的影子，而原来的影子则操控着男主的躯体去杀了男主的前女友，并且还去做了变性手术让男主成为了一个女人，甚至还爱上了曾经抢走男主女朋友的富家公子，一段崭新的人生就此开启！


*《终极乐章》*


何适带着自己的研究成果「终极乐章」——可以导致试听者人发疯和自杀的音乐，拼命逃亡。最终在好友的帮助下，重新开始了终极乐章的研究，也开始了一段关于人性方向和情绪重力的探索......


*《复制人生》*


你可以模仿我，但是你永远不可能比我更像「我自己」。


车窗外、房屋内、楼道中一双眼睛紧紧地跟随着，突然出现的陌生男人，楼道中消失不见的垃圾，长久没有见面的室友，这一切让陈辰陷入到了自我怀疑之中，到底是什么在盯着她？
