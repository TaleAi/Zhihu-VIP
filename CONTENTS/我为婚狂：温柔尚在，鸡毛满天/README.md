**— 一日三婚 —**


我一天参加了三场婚礼，新娘都是同一个。


婚礼上，新郎笑眯眯地问我：「怎么样，我老婆漂亮吧？」


我憋了半天，没告诉他，在另外两个婚礼现场，我也看到他老婆了，搀着别的男人走花环拱门呢。


更离谱的是，晚上洞房时，新娘跟我跑了。


  



**— 剩女难为 —**


30 岁的陆琪和自己不太喜欢、各方面不出众的林洋谈起了恋爱。


起初无感的她被林洋的关怀打动，很快就谈婚论嫁，结果却被林洋的妈妈处处刁难。陆琪不断妥协退让，最终在结婚前夕决定做回真实的自我。
