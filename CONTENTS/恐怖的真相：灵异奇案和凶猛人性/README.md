## 简介
专注暗黑罪案故事的孙大胖子又回来了，这次他带来的是从各行各业朋友口中听到的，很多普通人一辈子都遇不到的诡异邪性故事，还有自己经历过的神秘事件，希望继续和你一起观世人，听世间，审心声。


我是孙达雱（pang），熟悉我的人都叫我孙大胖子。


当了 15 年律师，我认识了不少各行各业的朋友，有给有钱老板做司机的，有深度调查记者，有在知名企业做高管的，还有豪宅销售部经理…...


他们给我讲的这些故事形形色色，骇人听闻又近在咫尺：


地下室里，做手工娃娃的老头正在亲吻像真人一般大的娃娃，他究竟是变态还是别有原因？


山中土地被村民奉为龙脉，无人森林中传出一声声恐怖的喘息声，是龙吟还是邪恶的阴谋？


健身房里勾搭的美女竟在他豪宅顶楼供奉着的小鬼泥像，究竟是财神还是邪物？


贫贱夫妻暗地做着肮脏的不法勾当，背后的原因却令人心酸。


看完这些故事后，你会发现：有时，人性的阴暗比鬼怪还要可怕。
