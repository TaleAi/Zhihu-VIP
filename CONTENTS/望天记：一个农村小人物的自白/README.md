## 简介
你向命运抗争过吗？我抗争过……



每个人都妄想着逃离自己出生的环境，这似乎是一个定律。同样地，「想逃又逃不出去」，这也是一个定律。


徐希出生在面朝黄土背朝天的农村，他从小就厌恶这里的环境和人，时时刻刻望着天空，渴望着更大的世界。


为了脱离大碗集这样一个闭塞、落后、野蛮、无知的农村，徐希努力读书，最终考上了大学。


即使身体已经远行，但是精神依然没有高飞。


大碗集有形又无形，时时刻刻扣在他的身上。他永远在逃，又永远逃不掉。


作品展现了一系列真实到失真的故事，让我们在大跌眼镜的同时感受到深深的悲哀和无助，命运薄我如斯，我必拼命一搏。


结果是人定胜天？还是命定之局？
