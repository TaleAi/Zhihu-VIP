## 简介
没有一场战争是突然爆发的。


战争是政治的延续，每场矛盾不是从谈判桌走向战争，就是从战争回到谈判桌。


一百多年来，人类在这颗蓝色星球上的战争就从未停歇过。


当战争打响时，往往令局外人猝不及防，但当我们回溯历史，又往往发现战争爆发的苗头早已出现蛛丝马迹，只是我们从未察觉——


* 如果当天那个士兵没走丢，七七事变还会发生吗？
* 因个人情仇踏入萨拉热窝，斐迪南大公之死如何导致「一战」的爆发？
* 俄乌战争的前世今生，你又知道多少？

在这个专栏里，我们将以每场战争的引爆点作为切入口，去还原与探讨，一场战争的爆发究竟是必然还是偶然。
