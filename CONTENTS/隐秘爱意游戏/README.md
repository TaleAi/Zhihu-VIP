## 简介
人们渴望爱人，渴望被爱，爱是他们共情的理由，也是他们能够接纳罪恶的唯一方式。


母与子的爱，妻与夫的爱，陌生人瞬间的怜悯之心与末路人对爱意最后的渴求与挣扎……这个专栏，讲述那些在爱中沉浮、犯下不可饶恕的罪过的人的故事。


在无尽的荒唐中撞击人心——我看似拥有一切实则别无选择，因为我不是为了自己而活着的，我的一切如针尖上的罗马帝国版脆弱，而支撑着那一切的，正是我对你的爱。
