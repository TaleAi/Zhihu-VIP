## 简介
尸体不会撒谎，而是会说话的证据。女法医为你破译死亡背后的秘密与真相，根据真实案例改编，挑战心理极限的罪案。


我是个法医。女的。


我的工作内容是和人打交道，活人或者死人。


经常有人问我，每天面对伤口和尸体，会不会恶心得吃不下饭，说这话时，他们看我的眼神活像在看一只怪物。


我就轻松地回答他们，我的胃口一向很好，如果正在经手一起碎尸案，晚餐就吃红烧肉。 


问话的人没有得到期待的答案，半信半疑，就会故作可爱地冲我做个鬼脸。不过在我看来，成年人无论男女，扮鬼脸时都不怎么可爱。 


其实，尸体远没有人们所想象得那么可怕。真正可怕的，是活人。人性的狠毒和黑暗，一次次地拉低我对人类道德底线的认知。如果没有法律，也许整个人类社会就是弱肉强食的丛林。甚至比丛林更可怕，因为人类有其他动物不具备的谋害同类的智商。


他们残忍嗜血，费尽心机。
