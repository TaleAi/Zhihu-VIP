《奠梯》


为了弥补年少的遗憾，老人在每一个平行时空回到过去，用五十年的孤独换取和初恋相伴一年的时光……


  



《玉脂狂妖》


当瘦成为整个社会「美」的唯一标准，在抗击脂肪的科学前端，他疯狂发愿：「我要让这世上的人，都变成胖子。」


  



《当洋葱精遇上吸血鬼》


吸血鬼家里开网店，洋葱精兼职快递员，从此邻居总能看见一个帅小伙儿在隔壁家门口一件一件地脱衣服……


  



《玄妃》


穿越到古代，却发现侍寝要抽卡？她换掉自己的一手烂牌，幸灾乐祸地来到皇帝面前：「卡组里包含了保卫龙椅、帝都沦陷、亡国之君、马嵬兵变、歪脖子树、藩王勤王、君主立宪等事件，皇上，鉴定你是欧皇还是非皇的时候到了……」 


  



《用灵魂和恶魔做交易后》


学渣虔诚许愿成为恶魔的仆人，睁开眼的瞬间，恶魔却对他说：「我的仆人，明天早上六点半起床开始读书吧。」
