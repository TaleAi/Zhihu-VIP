## 6.当洋葱精遇上吸血鬼
001 


洋葱在经过了艰苦卓绝的修炼后，终于成精了。 


洋葱精在人间找了份工作，那就是快递员。因为送货快，态度好，脸圆圆的，笑起来讨人喜欢，很快就变成了一片区域的小负责人。 


到达自己负责的新区域收快递的第一天晚上，洋葱精接到了一个电话。 


「我要寄快递。」那个人说，「我是开网店的。」 


002 


洋葱精拉着小车，到了小哥家门口。小哥脸色惨白，面无表情地拉开门，露出门后一堆箱子。 


「单子。」他伸手。洋葱连忙给他面单。 


小哥趴桌上填单子，看单子上名字，他姓李。 


洋葱：「李哥啊，那你慢慢填……」 


李哥：「不用，给你的很快就能填掉。」 


洋葱：「啊？还有其他那么多箱子呢？」 


李哥看看洋葱身上顺 X 快递的 LOGO：「那堆是给另外一家圆 X 快递的。」 


洋葱把顺 X 的外套一脱，露出里面圆 X 的 LOGO：「没关系！圆 X 快递我也接的！」 


李哥看着他递来一打圆 X 的单子。 


洋葱：「哥你还要寄啥快递？」 


李哥：「还有韵 X……」 


洋葱又脱了一层外套，露出韵 X 的服装。 


李哥：「……」 


李哥：「你是洋葱精吗？」 


洋葱：「不是。」 


——为什么做快递呢？这就是洋葱精得天独厚的优势了。一般的快递员，最多一个人偷偷接三家的活；可洋葱精就不一样了。如果你一层一层一层一层剥开他的皮，你会发现圆 X 申 X 韵 X 天 X 中 X 顺 X…… 


洋葱：「哥你放心，不管你要寄什么快递，都能直接找我！」 


003 


李哥几乎每天晚上都叫洋葱过去收快递。 


李哥：「你为什么能穿那么多件各种快递公司的外套啊？」 


洋葱：「因为我皮厚呀！」 


李哥：「我这个件要寄海外的，你身上穿了 EMS 的衣服没？」 


洋葱：「穿了穿了！李哥等等哦，我要脱一会儿！」 


洋葱开始一层一层脱皮，脱了厚厚的一堆，露出最里面的 EMS 服装。 


于是，最近邻居谈起李哥都压低了声音：经常晚上看他窗户影里，有个小年轻在脱衣服，哎呀，每天下午都脱，怎么这样啊，啧啧啧…… 


有天货多，李哥填了好久的单子。他在屋里写快递单：「哎小洋啊，你吃午饭了没？」 


洋葱：「还没吃呢。」 


李哥：「要不帮你点个外卖吧，我这还要写一会儿呢……」 


刚说完，就听见脱衣声——李哥一扭头，洋葱又脱了一堆衣服，戴上了一顶鸭舌帽。 


洋葱快递员工服底下居然还有 X 了吗外卖的外套。 


洋葱说：「你要吃啥？直接在我这下单就行了。」 


李哥：「我不饿，就是问你要不要吃饭……」 


李哥好像总不吃晚饭，人高高瘦瘦，脸色白得吓人。 


洋葱自己给自己叫了份外卖，自己给自己打了个五星好评，拿着盒饭回到李哥家，蹲着吃起来。 


等他回过神来，李哥站在他身后。 


洋葱：「李哥，你也要吃点吗？」 


话没说完，李哥直接扑在他身上，咬在他脖子上…… 


004 


李哥咬了一嘴洋葱汁：「呸呸呸呸呸！」 


洋葱捂着脖子：「啊啊啊，李哥你不能吃我啊我生吃没营养的！」 


李哥被辣清醒了：「卧槽你真的是洋葱精！」 


洋葱：「被，被看穿了……」 


李哥：「是咬穿的。」 


洋葱：「你……你该不会做炒饭就少了只洋葱吧？」 


李哥生无可恋地看着他。 


李哥：「你就没问过为啥我总不吃饭，总晚上寄快递吗？」 


洋葱：「啊？搞网店微商的难道不都是这样的吗？」 


李哥：「没事儿，你走吧。」 


洋葱反应过来了：「啊，难道李哥你也不是人？那你是啥？」 


李哥：「是吸血鬼。」 


洋葱：「牛逼啊！那可是贵族啊！挣得多吗？」 


李哥：「呃……还行吧，卖卖泰国佛牌，里面加一滴我的血，能驱蚊子，卖得特别好。」 


屋里还弥漫着一股洋葱汁的味道。 


洋葱：「那你刚刚吃了我的汁，要不要紧啊？」 


李哥：「你到底是大蒜还是洋葱？」 


洋葱很难过：「以前遇到过一只大蒜精，后来他去四川旅游，就再也没回来过……」 


知道对方不是人了，两人就更加有了一种革命情谊。吸血鬼李哥说：「能帮我叫个外卖吗？」 


洋葱：「吸血鬼也能吃外卖啊？」 


李哥：「叫份鸭血。」 


005 


李哥是一个从来没牛逼过的吸血鬼，大概是在八几年开放外国友人来华探亲那时候被吸血的，那是个没 WIFI 没手机没电脑的年代，八点后联防队就在街上巡逻有没有可疑分子了——可想而知，白天没法出门，晚上出门啥事都干不了是有多痛苦。 


后来借助了自己不用睡觉的优势，李哥成功成为了一个网店小老板，24 小时客服在线。 


洋葱听了，忍不住抹眼泪：「李哥，太不容易了啊李哥。」 


眼泪落在地上成了洋葱汁，李哥也跟着抹眼泪：「没事，你也不容易。一颗洋葱要成精，想必要经历很多凶险。」 


洋葱：「没啥凶险，就是尿尿时候味儿有点大。」 


李哥：「……」 


洋葱每天给李哥寄快递和送鸭血。有天发现李哥很苦恼地看着电脑。 


有个人乱给差评。一看就是职业差评师。 


李哥叹了口气，出门了。过了两个小时回来了。 


那个人把差评改成了五星好评。第二天，有八卦小媒体报道，有个大学生精神错乱，号称自己晚上被一个吸血鬼网店老板威胁。 


洋葱有时候送快递也受气，服务业都不容易。李哥说：「没事，谁给你气受了你和我说，我就当是给自己加餐了。」 


好在洋葱是一颗心胸宽广的洋葱。 


李哥经常沉迷工作，忘记吃饭，洋葱就给他带新鲜的鸡鸭血：「这是我楼下卖鸡大妈自己放血做的，绝对不含添加剂的！」 


吸血鬼在中国的生活还是很艰难的，毕竟新鲜鸡鸭血不多，只能吃不那么新鲜的，据说每年都有那么几只吸血鬼死于食物中毒。 


不知不觉，万圣节了。 


洋葱很兴奋：「李哥你今年陪我一起过节好不好！我们一起去外面吓人！」 


不过李哥是吸血鬼，万圣节好像是他对家的节日。 


但李哥很爽快地答应了：「没事，身在曹营心在汉，我们中国的吸血鬼，管什么西洋的神……」 


006 


洋葱把自己的头变成了洋葱，李哥翻出了不知道有几百年的吸血鬼三件套，和他一起上街去吓人。 


结果旁边走过一只丧尸，洋葱尖叫着扑在李哥身上。 


丧尸一摘头套，就是个高中生。 


「叫什么叫！吓死人了！」 


洋葱：「不好意思啊不好意思……」 


又遇到一只丧尸，洋葱不叫了。这只丧尸的服装和化妆制作非常精良，看不出哪里假。 


丧尸嗅了嗅：「你是吸血鬼吧？」 


李哥：「……」 


丧尸：「旁边那只味道好重，看不出是啥……」 


洋葱：「……」 


万圣节的路上，不仅有洋葱精和吸血鬼，还有真的丧尸。 


玩了一晚上，洋葱和李哥回了家。终于不用送快递好好过了个万圣节！ 


李哥摸摸他的洋葱头：「快点把头变回来吧。」 


结果洋葱玩得太累了，睡过去了，变回了一颗洋葱。李哥很纠结地思索了一会儿，把洋葱塞进了冰箱。 


第二天，洋葱被冻得瑟瑟发抖，从冰箱里掉出来：「李哥……你为什么要把我……放冰箱？」 


李哥：「怕坏。」 


有时候，如果发快递弄得晚了，洋葱就干脆睡李哥家。洋葱睡着的时候就会变回洋葱，第二天永远都是在冰箱里醒来的。 


尽管和李哥说了不用进冰箱，但李哥还是很担心，要是在外面放久了，洋葱会不会霉掉。就这个问题，大家探讨了很久，终于有一天李哥没把洋葱放冰箱。 


——洋葱醒过来的时候，发现自己在李哥被窝里。 


007 


小日子过了也没多久，有天，有两个男的找到了快递公司，说要找送几号件的快递员。 


洋葱迎了出去，一看，两人手里拿着李哥卖的佛牌和快递盒。 


对方说：「商品有点问题，就是想确认一下，退货地址是不是单子上的这个地址啊？」 


洋葱：「是啊。」 


对方点头，没再说啥。 


那天夜里，李哥没再让洋葱去寄快递，打家里电话也没人接。 


洋葱想，会不会是吸血鬼出去打牙祭了啊，毕竟，每天吃鸡鸭血也是够艰难的。 


洋葱去了李哥家，用李哥给的备用钥匙开了门。一开门就傻眼了，李哥家乱成一团，好像经历过一场大战。 


李哥出事了！ 


洋葱急得变回洋葱原地乱滚，怎么办怎么办……对了，那两个男的！ 


早上故意来问他地址的两个男的，肯定有问题！ 


还好，几年快递当下来，洋葱精练就了职业技能：记地址。 


单子上，不只有李哥的寄件人地址，还有一个收件人地址。 


洋葱连忙骑着小摩托来到那地方，夜深人静，那地方还挺偏僻的。 


洋葱抱着个快递盒上去敲门：「你好，顺 X 快递！」 


门里起初很安静。洋葱脱掉顺 X 的外套，露出圆 X 的外套，又拍门：「你好，圆 X 快递！」 


里面还是没反应。 


洋葱：「你好！中 X 快递！」 


继续没反应。 


洋葱：「你好！EMS！」 


…… 


终于，里面的人被他吵得受不了一把拉开了门：「谁大晚上的送快递啊？」 


洋葱一口洋葱汁喷在那人脸上，直接把人辣得满地打滚。他往里面冲，就看里屋站着白天见过的男人。还有张椅子，李哥就被绑在椅子上。 


洋葱：「李哥李哥！」 


李哥：「快跑！他们都是吸血鬼猎人！」 


一个男的冲着洋葱举起枪，砰的一声脆响，洋葱身上给打了个孔。洋葱汁从里面喷出来，淋了男人一身。 


男人：「好辣！好辣！这啥玩意儿？」 


洋葱给李哥解开绳子，拉了人就跑。 


008 


两人骑着小电驴，洋葱的伤口一路都在洒洋葱汁。 


李哥：「不行，他们追上来了……」 


洋葱：「没事！」脱下一件新的快递外套给李哥套上。 


李哥：「还在跟着！」 


洋葱又脱了一件新的外套给两个人变装。 


李哥：「还在……」 


洋葱脱到最后，剩下美 X 外卖和 X 了么外卖的外套。路上有人看着两家外卖巨头的外卖员同乘一车的奇景，一脸懵逼。 


他们终于把猎人给甩掉了。洋葱停下车，一句话都说不出来，倒落在地，变回了一颗带着弹孔的洋葱。 


洋葱没有再变回来过，好像妖力散尽，成为了一颗死洋葱。 


但李哥把洋葱一直带在身边，搬了家，改了名字，洋葱就揣在怀里。 


晚上，李哥把洋葱放被窝里，轻轻拍拍：「你什么时候能变回来啊？」 


这个世上少了一只洋葱精，好像也没怎么样。但李哥又变成了一个人了。邻居说，新搬来的那个男的，爱好挺独特的，每天就摸一颗洋葱，洋葱都快给摸出包浆来了。 


有天晚上，李哥带着洋葱在外面晒月亮，临时有事离开了一会儿，邻居家小孩就拿起洋葱回去研究了。等李哥发现的时候，洋葱的皮都快给剥光了。 


邻居：「就一颗洋葱嘛，干嘛发那么大火啊，神经病！」 


李哥拿着一颗洋葱芯回了家。 


——可是，这个晚上，好像从洋葱芯里传出了声音。 


洋葱：「呼啊……呼啊……」 


但是李哥伤心地睡着了，没听见。 


第二天，李哥睁开眼睛，发现身边睡着洋葱精。 


好手好脚。 


李哥：「你活过来了？！」 


洋葱精吓得醒过来：「啊啊啊李哥！」 


李哥：「我还以为你变成死洋葱了！」 


洋葱：「对方没把芯打碎，我就不会死啊……」 


洋葱没死，但伤太重，昏昏沉沉睡了下去。厚厚的洋葱皮把他和外界隔开了，要不是被熊孩子掰开，估计还要睡很久。 


李哥：「那，那你的皮没了要不要紧？」 


洋葱：「呃，应该没事……」看看身上一丝不挂，又说：「就是……没衣服穿了……」 


洋葱可能还要修炼很久，才能修炼出以前的层数，自己变衣服穿。 


洋葱抱着被子——这几年，自己大概只能给一家快递公司打工了…… 


所以，看到快递员送快递的时候，闻闻对方身上有没有洋葱味，说不定你遇到的就是只洋葱精呢。 


END
