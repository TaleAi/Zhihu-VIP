帮你找到适合自己需求的耳机与音频设备


介绍耳机与以耳机为核心的随身听音系统，普及耳机与音频设备相关工作基本知识，引导读者形成一套自己的耳机审美评价体系，帮助读者了解自己的听音习惯与购买需求，在各类耳机「评测」与体验内容中找到真正可用信息，进而找到适合自己需求的耳机与音频设备。
