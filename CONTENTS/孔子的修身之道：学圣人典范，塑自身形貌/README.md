## 简介
有人说，中华民族不信教、不信神，却信仰一个人——孔子。可以说，孔子奠定了中华民族的文化基因。但我们真的了解孔子吗？封建帝制代言人、封建礼教代言人、封建旧势力代言人……几千年来，孔子受到了太多歪曲。透过真实可感的孔子故事，重新定义你心中的孔子。


「天不生仲尼，万古长如夜。」


一个家道中落、父母双亡的「苦孩子」，


如何通过修身一步步成为深受中国人崇敬的偶像？


明明可以安稳富贵、名利双收，他为何「明知不可为」而坚持周游列国，几欲丧生？


一些教材、教辅书籍断章取义，使孔子的思想更加难以理解。


本堂课透过《史记》《左传》《四书集注》等历史典籍，卸下圣人的光环，借鲜为人知的背后故事为你展现一个真实的孔子，重新体会国学经典，看孔子如何逆风翻盘，并掌握这套「勤于学，志于道，游于艺，据于德，依于仁」五维度的修身之法。
