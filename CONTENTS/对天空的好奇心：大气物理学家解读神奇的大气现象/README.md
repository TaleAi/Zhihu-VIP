## 简介
在我们童年的时候，空气还很清新，夜空还能看到闪亮的繁星。你是不是也对天空充满了好奇，有着探测大气奥秘的梦想？别小看了这小小的好奇，正是这些最平常的好奇心，推动了人类科学史的不断发展。


我们日常生活中见到的很多神奇的大气现象，比如说彩虹、火烧云、海市蜃楼、雾凇等，都充满了一种神秘和美，能够激发儿童的好奇心。


龙卷风是从哪里来的？天空为什么是蓝色的？


雾霾会影响我们的健康嘛？怎么才能躲避闪电？


......


地球是人类唯一的家园，而大气则是我们生活的主要环境，大气的各种状态，如温度，湿度，污染指数等等，或多或少都在影响着我们每个人的生活质量。同时，大气也是地表、海洋等陆面构造同高层空间之间联结的纽带。因此，孩子们学习了大气科学的基本知识，也将为了解整个地球系统的平衡打下基础。


本专栏通过对这些大气现象的探索，可以养成小朋友细致观察和独立思考的习惯，形成系统分析问题的能力。当前，大气科学还有很多未知领域需要去挖掘，让小朋友从小就开始了解大气科学知识，将为社会培养更多的具有原创性思维的科学工作者。
