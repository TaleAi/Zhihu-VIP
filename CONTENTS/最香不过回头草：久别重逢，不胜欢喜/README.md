## 简介
久别重逢，开启大型追妻/追夫现场，回头草真香！


**导演的地下恋情**


容钦，导二代，年纪轻轻已经拿了好几个电影大奖。这次，他却挑中了一个名不见经传的编剧的本子，而且在片场他那双眼睛似是要长在姜编剧身上。十年前，两人是大学校园里人人艳羡的「神仙爱情」；十年后，一个还想追一个只想逃……


**男友不正经**


五年前，顾可舟不声不响就突然去留学，于水水在心里问候了他祖宗十八代。五年后，顾可舟突然回国，还马上宣示主权，「你是我的，从前是现在是以后也是，谁都别想抢走！」。那五年前玩失踪算什么？我于水水不接受！


**傅医生请接招**


得知手中艺人的主治医生是傅泽，铁血经纪人木子立刻拿出自己珍藏多年的爱马仕包包，穿上高定套装，脚踩细高跟，摇曳生姿地杀进了医院。三年前两人冲动分手，木子恨了他三年，也想了他三年。如今好不容易相逢，她当年追了三年才追到手的男神，怎么可能轻易放他离开！


……


久别重逢，认认真真谈恋爱，勤勤恳恳撒狗粮。
