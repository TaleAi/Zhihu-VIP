## 简介
本专栏涵盖清代的风水学、建筑学、美学、哲学、景观学、丧葬祭祀、宗教等文化，让你除帝王后妃纠葛外，掌握更多清史知识，成为最懂清史的现代人。

康熙的三个皇后为何各个早亡？乾隆的皇后为何也难得善终？首任富察皇后为何客死他乡？继后那拉皇后，为何被抹去历史痕迹？后宫中默默无闻的后妃，唯一的出路竟是殉葬？ 


以上不过是真实历史中清宫帝王后妃命运的一角。在清东陵和清西陵这两座皇家陵墓，埋葬着清代所有的帝王后妃，无数展现人性悲欢的情爱故事和扣人心弦的政治斗争，我们的专栏都会为你一一呈现。
