## 简介
悬疑编剧的日常工作，是研究真实案件的卷宗。本专栏根据真实案件的报道和资料，还原1988-2020年特大案件的作案过程，披露凶手的成长经历，讲述警察缉凶过程。


滥杀、灭口、变态杀人，每当一个惨烈案件发生后，所有人都会问：杀人者是什么样的人？为什么作案？心里在想什么？


他们是精神病？杀人成性？还是天生恶魔？


作为悬疑编剧，每次书写谋杀的过程，就像是探索人性的弱点。对于大部分人来说，每一次血腥现场的呈现，让人们自然认为凶手是怪异变态。但当你深入凶手的人生经历，真真实实看到他时，才发现，他和你我无异，很正常很平和，甚至很像许久未见的老朋友，直至挥起屠刀那一刻，才听到绝望呐喊。


在本专栏里，用凶手视角，真实过程与细节，带你走进曾经中国九大恶魔，那份不为人知的内心世界。 


妻子眼中体贴老实丈夫，是如何成为连环强奸杀人魔，笼罩了一座城几十年的梦魇？


老师骄傲好学生，为什么手握铁锤，砸死了四个亲密无间的同学？


 一个事业有成中年男人，手握屠刀，杀死了自己妻子父母和两个亲生孩子后，为何心情无比舒畅？


 不想当无名之辈的连环杀人犯，将所有作案经历写在日记上，他的执念从何而来？


**本专栏中，悬疑编剧通过研究新闻报道和陈年卷宗，还原 1988-2020 年九件特大案件犯罪的经过与结局。**
