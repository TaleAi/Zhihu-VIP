## 简介
疾病无小事，科学面对最重要。


* 每到重要场合，就会腹泻是怎么回事？
* 你知道我们每个人都是潜在的「狐臭」患者吗？
* 「十男九痔，十女十痔」，得了痔疮坐立难安，该怎么办？

相对于感冒发烧，狐臭、痔疮、脚气这些令人尴尬的「小毛病」，可能对我们的生活会造成更大的困扰。


也因为害怕尴尬，面对这些困扰时，我们往往处于有苦不好意思谈的状态，所以这类问题，也成为我们轻信偏方、被收割智商税的重灾区。


本专栏邀约多位相关领域的专业医生，通过科普疾病的来源、正确的治疗方法，帮助读者正确认识这些「难言之隐」，全面了解解决困扰的科学方法。
