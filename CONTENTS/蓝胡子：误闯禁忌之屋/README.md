## 简介
遇见童话里的爱情，别高兴太早，也许他是有着不能触碰秘密的「蓝胡子」。


张芳的女儿死后没多久，就来我家当了保姆。


这两年，她很爱我女儿，几乎把她当成亲生女儿来养，可没想到，一次意外，我不仅发现了她和我老公的私隐。还有她隐藏在保姆身份下的骇人一面。


——《娃娃》  

那个喜欢了六年的男人，


成了我的姐夫。


也成了我的金主。


我曾自私的想，只要我姐姐死了，这男人，是不是就属于我了？


三年前，我姐姐真的失踪了，我也终于如愿跟了翟逸星，这三年，翟逸星对我很好，出手大方，温柔宠溺，可就在我二十三岁生日这天，却发现了他那个肮脏又可怕的秘密。


——《血色》
