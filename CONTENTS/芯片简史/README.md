## 简介
讲述芯片从诞生、发展到广泛应用的过程


我们生活在芯片的世界里，就像鱼儿生活在水里。芯片是如何诞生的，又是如何把触角延伸到我们周围？全景式地讲述了晶体管和芯片不到百年的历史发展中的最重要的发明，让大众更全面深入地了解芯片，同时激发年轻读者对于这个领域的兴趣。


全文大致按时间顺序讲述了芯片从诞生、发展到广泛应用的过程。首先讲述了晶体管与芯片的诞生，然后讲述了晶体管与芯片的演进的过程，尺寸不断减少，形态结构逐渐演变。接着讲述了芯片如何变得更加多样化的过程，种类与功能朝着信号感知、大功率驱动、光电显示等领域发展。接下来讲述了芯片设计和制作的发展历程，如何从一个想法和一堆沙子变成了电路，并且制造出来。
