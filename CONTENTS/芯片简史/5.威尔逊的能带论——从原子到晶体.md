## 5.威尔逊的能带论——从原子到晶体
在晶体中，原子的分布规律对其性能有很大影响，但如何才能知道晶体的内部结构呢？普通的光学显微镜根本无法观察原子内部结构，因为可见光的波长有几百纳米，而原子间的距离连 1 纳米都不到，这个问题困扰着研究固体物理学家。


与此同时，关于 X 射线是波还是粒子的疑问困扰着德国物理学家劳厄（M.Von Laue）。如果 X 射线是光，那么应该像光那样互相干涉产生明暗相间的条纹图案，但是 X 射线的波长过短，只有可见光的千分之一。如果 X 射线通过与其波长差不多尺度的光栅细缝也许会产生衍射，但是那里去找这么小尺度的光栅来实现衍射呢？在一次散步时劳厄想到了晶体，也许其内部原子的间距足够小，可以作为一种天然的光栅。他让一束 X 射线照射蓝色的硫酸铜晶体，果然在晶体后面的感光照相纸上他们看到了有规则的亮点。劳厄证明了 X 射线是一种波，也无意中找到了窥测晶体内部结构的方法。


英国的布拉格父子（W.H.Bragg, W.L.Bragg）听到消息后，立刻用晶体做了类似实验。小布拉格考虑了 X 射线照射到上下两层晶体反射后相互干涉的结果，推导出了 X 射线的波长和晶格间距的简单公式，只需通过衍射点阵就能揭开晶体内部的结构图案。这下物理学家就有了 X 射线这个非常趁手的工具去窥探晶体内部的秘密。人们通过 X 射线在晶体里衍射形成的规则斑点发现，晶体内部都是按照非常有秩序的点阵周期性排列，例如硅和锗中的原子分布是金刚石的点阵结构。


知道了晶体内部结构，带入到薛定谔方程中就能求解出电子在其中的行为。可以想象，薛定谔方程会很复杂，但幸好对于内部有着规则结构的晶体，人们能够把方程的时间和空间变量分开，从而求解出波函数。海森堡的博士生 23 岁的布洛赫（Felix Bloch）第一个提出了在晶体中求解薛定谔方程的波函数的方法。他发现原来规则点原子结构可以对波函数进行的周期性调制。这使得人们更好地理解了晶体里电子的行为，但是对于导电性的认识布洛赫仍没有完全想清楚。


1931 年初，布洛赫所在的海森堡研究所来了一位 25 岁的英国小伙威尔逊（A. H. Wilson），他跟随海森堡研究固体物理。威尔逊研读了布洛赫等人的文章后认为，理解物质导电的关键是原子内部的电子如何吸收能量跃迁到最外层。他提出了一个新的理论来解释物质导电性，这就是后来被称为经典的能带理论[]。海森堡听了威尔逊的理论后非常兴奋，马上叫来布洛赫讨论。但布洛赫却连连摇头，说「大错特错」，他认为区分导体和绝缘体靠的是原子间的电子的行为。但是经过一个多星期的努力思考，布洛赫终于被说服了，接受了威尔逊的理论。


威尔逊认为，晶体中的两个原子距离比较远时，它们的电子拥有相同的能级。但是当两个原子逐渐接近时，它们最外层中能级相同的电子被两个原子所共享，它们的能态相互叠加，不同的组合方式就叠加出两个稍微不同的能量，于是就形成了不同的能级，泡利不相容原理再一次发挥作用。这就像两个学生的一门课的期末成绩都是 80 分，但他们俩接触交流后就发现其中一个平时成绩好，而另外一个期末考试成绩高，于是就区分出二者。推而广之，当众多原子组成晶体时，许多相邻的原子彼此靠近，共享的电子态将延伸而且分出更多的不同能态。就像一个学校上千名学生，经典物理认为学生可以考相同的分数，而量子力学中一定要区分出每个学生的成绩高低，这样每两个学生之间的分数差别就要小于 0.1 分。由于原子数量众多，电子的能态虽不相同却非常接近，就形成了一簇密集的能态，即像一条带状的能带[2]。


能带理论：


电子总是先填满低层的能带，然后再向上逐渐填充。这有点像进入电影院的观众先坐前排的座位，然后逐渐坐满后面的。经典物理中的麦克斯韦-玻尔兹曼分布认为一个座位可以坐两个或更多电子，而在量子物理中一个座位只能坐一个电子，必须从前到后对号入座，这样的分布叫做费米-狄拉克分布。电子能够填充的最上一层称为价带。而无法完全填满的能带叫做导带[]。价带中电子较为拥挤，不容易移动和形成电流。导带没有全部占满，所以其中的电子在电场下容易移动，形成电流。在导带和价带之间有一个空白，那里没有电子存在，称为禁带，就好像影院里前半部和后半部之间的过道，分割了两部分座位。


但是电子能级概念比电影院里的座位复杂。让我们再打个比喻。申请电子邮箱时，用户名不能重复。如果一个人用自己姓名对应的英文字母例如 wangwei 去申请邮箱，很可能会被拒绝，因为这个名字已经被占用了，这就像电子对特定轨道和能级的独占性。系统会立刻推荐一堆可用的名字，例如 wangwei1， wangwei2， wangwei3 等，这就像电子能级的随机性。这些用户名彼此相近又不完全相同，组成了一条以 wangwei 为特征的用户族群。类似地，接下来又会有 wangwen1， wangwen2， wangwen3 等的用户组成另一族比较密集的用户族群。这就像电子能级的聚集性而形成的能带。而在 wangwei 和 wangwen 两个用户族群之间，因为字母拼写规则的原因，几乎没有 wangwej， wangwek， 和 wangwem，这些地方就像禁带一样几乎是空白的。


为什么会存在禁带呢？这要回到微观世界。既然电子是一种波，它就对应一定的频率和波长。特定的频率的电子波在周期性的晶格中传播，在特定波长下反射回来的波跟前行的波刚好抵消，那么在此波长或频率的电子波就无法存在于晶格中。因为电子波的频率对应于能量，所以就不存在对应的能量等级，从而形成了能量的禁区即禁带。


只要温度超过绝对零度，总有一些价带电子从原子振动中吸收了能量跨越了价带跃迁到导带，就像爆米花吸收了热量突然从锅里蹦起来，所以温度升高半导体更容易导电，这解释了法拉第遇到的负温度电阻率的问题。尽管电子从环境温度中吸收的能量远小于跨越禁带所需的能量，但实际上仍有不少电子成功跃迁到导带。在宏观世界里这是不可能发生的，你不可能轻轻用力就把一块砖抛到十楼那么高，但是在微观世界里这就是量子力学的统计规律在起作用了。虽然单个电子跃迁的几率很小，但电子的总数很庞大，总有一些价带电子跃迁到导带，形成电流。


那么导带中的电子是形成电流的唯一原因吗？也不是。当电子跃迁后，在价带留下了一个空位（空穴）。本来满满当当的价带有了空位后，变得不那么拥挤，那里的电荷也可以在电场作用下流动起来。那么空位到底是不是一种实体电荷呢？因为空位是电子留下的，而电子带负电，所以为了计算方便，人们将空穴当做一种带正电的准粒子，就像电影院里坐了许多人时，与其费力地数人头，不如去数空座位；又或者当中间某个座位没有人，临近的观众会一个接一个地朝那里移动，看上去就好像是空位朝相反方向移动一样[]。这样半导体中既有负电荷又有正电荷，如果二者相等则抵消，否则其中一个占主导就会表现出正电荷或负电荷特性，于是用霍尔效应测量半导体就能测量到带正电的电荷，这就解释了此前人们对半导体带电粒子到底是正电荷还是负电荷的疑惑。


乍看之下，电子越多就应该越容易导电。但事实并非如此。威尔逊领悟到并不是有了电子就能导电，而是一定要在能带中留出一些空位，电子才能流动起来。如果价带里充塞满电子，电子反而很难移动。就像饭店老板都希望就餐者越多越好，每张餐桌就是一个原子核，就餐者是原子的外围电子。但如果顾客坐满了，一直赖着不走，就不会有更多的顾客进来，老板就要着急了。只有当一些人走开了，才能带来更多的人流。


能带理论为解释半导体提供一幅清晰的图景：如果把每个电子比作一辆汽车，车流相当于电流。价带就像地面上车辆拥堵的马路，车辆一点点向前挪动，前面的车向前挪一下，就产生一个空位（空穴），后面的车就能向前跟进，车流向前移动，空位反方向移动。而导带像高架桥，上面的汽车不多，所以可以开得很快。马路和高架桥之间隔着一定高度即能量差，称为禁带宽度。如果一些汽车吸收能量能突然从拥堵不堪的马路「跃」到上方的高架桥上，这样不仅这辆汽车可以在高架桥上继续移动，低处马路上的其余汽车因为有了空位也可以开始移动。这样一来，不仅电子能导电形成电流，空穴也能形成电流。尽管电子和空位的移动都能引起电流，但是二者引起的效果还是有较大差别的，这从汽车的例子也能看出。一辆车在空旷的高架桥上移动的速度远远比拥挤马路上的一个空位移动得快。常温下硅晶中电子的迁移率（电子在电场下的移动速度）大约是空穴迁移率的将近 3 倍，所以工程师设计高速电路时经常考虑用电子而不是空穴作为移动电荷。


能带理论很好地解释了半导体的独特魅力：它的高架桥不太高也不太低，总有一部分电子可以跃上去从而导电；如果从外部施加反向电压桥就变得更高，阻断电子飞跃从而关断。而绝缘体呢，高架桥太高，电子不可能跃上去，都堵塞在马路上，无法形成电流。而金属的能带则跟半导体和绝缘体大不相同，它的价带和导带重合，没有禁带，所以金属中的电子毫不费力地在导带中自由穿梭，形成电流，所以金属非常适合作为导体。但是，由于没有禁带，金属中的电子也很难停下来，这样就不容易关断电流。这样看来，只有半导体既能方便地开通又能较为容易地关断，最适合作为开关。


威尔逊发现能带理论时是二十世纪三十年代初，没多久德国纳粹上台，犹太人的厄运来临。一些著名科学家如爱因斯坦等不得不「跃」过大西洋来到美国。这种单向的人才流动给这个新兴国家带来了宝贵的智力资源，而他们在德国留下了众多空位，后来这些空位被没那么优秀的人所填补。


进一步地，威尔逊研究了半导体中掺加杂质对半导体导电性的改变和影响，并且对一种氧化铜半导体展开了研究。威尔逊的理论犹如一道光芒照射进半导体的研究领域，但是这道光花费了十多年才真正照亮了在黑暗中摸索半导体性质的贝尔实验室的肖克利、巴丁和布拉顿等人的对半导体晶体管的研究之路，首先照亮并引起了光电流的是贝尔实验室无意中发现的 PN 结。现在，我们就进入到半导体晶体的世界。
