本课程由 Momself 提供，基于 EFT 情绪取向疗法（Emotionally Focused Therapy）专治「 一个人瞎忙活，另一个不努力 」


他每个毛病我都知道，但还是忍不住要吵，吵来吵去总是在吵一件事。


老公安于现状，怎么让他更上进？


学了不少技巧，都是我在学习我在改，怎么让他跟我一起努力？


他永远都不知道我想要的是什么，需要安慰需要抱，他就只会说「你要我做什么？为什么 你总是不满意？」 


工作生活压力大，动不动就看对方不顺眼，我抱怨一下，他就说我烦。


性生活频率变低，我们的关系出问题了吗？


龚利琴老师将通过 ETF 情绪趋向疗法带你找到问题的答案，让爱你的人更爱你。
