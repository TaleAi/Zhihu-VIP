# 猛龙过江：拳手李景亮的战役

### 作者：李景亮 等

### 分类：真实故事

### 状态：已完结，共 11 节

### 存档日期：2022-06-12

## 简介
本专栏讲述了格斗王李景亮在拳场摸爬滚打、与铁哥白手创建拳天下、走出国门的真实成长经历。


格斗真残酷！


耳朵被打碎了，不哭，站起来去战斗；


眼眶被锤爆了，不怕，醒醒脑去反杀；


打赢当英雄，打输成狗熊，这是现实的残酷。


格斗会上瘾！


赢了一场比赛，还想再赢一场。


输了一场比赛，下一场一定要扳回来。


输输赢赢，突破自我，这是跟自我较劲的乐趣。


李景亮，江湖人送外号「吸血魔」，在 UFC 第一次比赛中就获得胜利，其后更是屡战屡胜，成为了继张铁泉、居马别克之后第三位进军 UFC 的中国选手。


在本专栏里，将以李景亮的视角，讲述他与格斗的不解之缘和更多不为人知的过往：


他与格斗的第一次结缘是在什么时候？


真实的格斗训练到底有多残酷？


是去 ONE FC 挣双倍钱，还是去 UFC 实现梦想？


同样学格斗的好哥们，最后因为运气不同都怎么样了？


……


本专栏由李景亮口述，花椒、杜尚先生、迈克尔、Eriiic 撰写整理。


仅以此专栏感谢铁哥、毛毛姐，以及支持过中国拳击事业的朋友。
