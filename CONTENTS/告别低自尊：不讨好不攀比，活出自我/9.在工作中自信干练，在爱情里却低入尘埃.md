## 9.在工作中自信干练，在爱情里却低入尘埃
**- 导言**


不知道你还记不记得，我在第一节课中提到的来访者小米，她的「讨好行为」让她和室友的关系日渐恶化。但是，在和室友相处之外，小米其实是一个开朗活泼的、顶着名牌大学光环的大学生，很多和她不是那么熟悉的大学同学，甚至包括我，对她的第一印象，都是「自信阳光」的。


还有我在第二节课中提到的来访者吉米，他也并不是一直「玻璃心」和「暴脾气」。他的女友告诉我，吉米只有在受到别人批评和建议的时候才会突然「爆炸」，平时，吉米是一个谦谦有礼、体贴入微的「暖男」男友，女友的朋友们甚至都很羡慕她，找了个这么好的男朋友。


这种情况在我的临床工作中是很常见的。虽然，许多来访者都认为自己是个自卑敏感的人，但是，他们的外表看起来并不是很自卑，甚至有些人看起来「气场强大」。而且，当我仔细询问过他们后，我就发现，生活中的他们也不总是「自卑敏感」的。在一些方面，比如，在工作上，有些来访者可以很自信、很干练地完成工作任务。但是，在恋爱关系中，这些来访者像是变了一个人似的，一下子唯唯诺诺、低入尘埃了起来。或者反之，我常常会听到这些来访者发自内心的呼喊：「老天爷啊，为什么我不能像处理工作（恋爱）一样处理我的恋爱（工作）关系呢？」


我相信，你也曾经在内心呼喊过类似的话。为什么一个人，会在生活的不同方面，表现出那么不同的心态呢？


**- 课程正文**


这个问题的答案，也许和低自尊的「激活点」分不开。


激活点，就像是生活中打开低自尊心态的「开关」。当你一不小心碰到了这个「开关」之后，上节课中我们说到的「低自尊的核心信念」就会冒了出来，不断地向你重复着：「我没有用，我有缺陷，我是个失败者，我不够好……」。自然而然地，你会产生非常不舒服的情绪和症状：例如自卑、敏感、多疑、嫉妒的情绪，或者自我批评、自我打击的想法。


在这样不舒服却又压倒性的情绪和想法中，你不得不赶紧行动起来，做出一些「应激行为」，例如讨好、攻击他人/自己、逃避、或者神游身外，来应对大脑里的这些变化。


为什么生活中会有这些「开关」呢？这要从我们大脑的记忆存储系统说起。


在上几节课中，我们知道了低自尊的根源，是童年缺爱的经历。但是，这些缺爱的经历不会像普通的经历一样，被存储在大脑里。


当我们说到记忆，大多数时候，我们都在说有意识的记忆，或者叫显性记忆 ( Explicit Memory )，比如今天课上我说的所有知识，都是我在其他书本里、课堂上学习到的知识——这些知识就是显性记忆。


其实呢，我们的大脑中还有一种无意识的记忆，也叫做隐性记忆 ( Implicit Memory )，比如说，我的办公室有个密码锁，我可以在五秒钟之内摁对密码进入办公室。但如果你要问我密码是多少，我却要仔细地想一段时间，才能告诉你。


大脑神奇的地方就在于，它能够把同一件事情，编码成不同的记忆——有一些是显性的记忆，而有一些呢，则是隐性记忆。


有这样一个相关实验：在实验室中，科学家们把含有威胁的「有害」刺激物（比如微弱的电击），和一些没有任何威胁的中性刺激物（比如中性的图像、中性的声音、或者中性的嗅觉）相配对。比如，当被试者听到滴滴声的同时，实验员就会在他脚上轻轻地电一下。结果，人们的大脑就会自动地把中性刺激物和有害刺激物强烈地联系到一起去，以至于，人们一看到这个中性刺激物，就会在并没有有害刺激物的情况下，自动发生无意识的、被威胁的反应。比如，被试者一听到滴滴声，哪怕没有被电击，也会感到肌肉紧张、呼吸急促、心跳加快。这样的具体实验有很多很多，感兴趣的话，你也可以搜一搜 Pavlovian fear conditioning。


再说个身边的例子。小的时候，我兴高采烈地去公园里玩，结果被一只蜜蜂狠狠地蛰了一下，我哭着跑去告诉我妈：「妈妈，我被蜜蜂蛰了，好痛好痛！」这就是我的显性记忆。但是从此，年幼的我看到任何黑黄相间的东西，比如说，黑黄相间的袜子，就会哭起来。甚至，直到现在，我猛然看到一个黑黄相间、小小的东西，都会忍不住打一个激灵，这就是隐性记忆在作祟。我的大脑把一件事情——被蜜蜂蛰，分别编码成了显性和隐性记忆。


那么，大脑的这一神奇功能，和低自尊的激活点有什么关系呢？不要着急，让我来慢慢跟你解释。


第一节课中我们说到，小米关于童年缺爱经历的回忆是：父亲一辈子仕途不顺，一旦工作不顺心，他就会打骂小米，把对领导的怒火发泄到她身上。所以，小米要不断地去讨好父亲，让父亲开心，这样她就会觉得自己很懂事、可以分担父亲的重任，因此逃过打骂。这是小米的显性记忆，有逻辑，很 make sense，而且很具体。


然而，聪明的你应该很快就发现问题了：小米现在的寝室室友们，并不是她的父亲啊！室友们也并没有工作不顺心，更不会真的打骂她啊！为什么小米还是忍不住去讨好室友们呢？


小米告诉我：「我最怕最怕的，就是寝室里的沉默，这让我想到了父亲心情不好的时候，也是在家里一声不吭，我只要胆敢惹怒他，一定会遭到暴打。所以，现在大家一旦都不说话，我就特别恐慌，觉得有什么不好的事情要发生了。我得赶紧调节气氛，牺牲一下自己的需求，让大家脸上露出满意的笑容。」——由此可见，小米对于童年缺爱经历的隐性记忆，是「沉默的环境」。


于是，在我们看来，「寝室里的沉默」是一个中性的、没有任何威胁的状况。但是，在小米的大脑中，「寝室里的沉默」却与含有威胁的童年缺爱经历，强烈地联系到了一起。


在小米的感受中，「寝室里的沉默」不是中性的，它是打开缺爱记忆的「开关」，给她带来了无限「脑补」的场景：一层又一层关于被父亲打骂的回忆，叠加着必须戴上「照顾者」面具的熟悉感、和其背后的毒性羞耻感，覆盖到了小米眼前的事实之上。


面对「脑补」中的威胁，小米的身体又开始大量分泌压力激素，而这些激素通过刺激脑回路，甚至还会强化原始的缺爱记忆——这叫做记忆重建现象（ Memory reconsolidation ）。小米大脑中的理性管家「新皮质」被迫下班，让位给了应激反应的专家「杏仁体」，向全身心发出警报：「危险！危险！前方有危险！」


这个时候的小米，很自然地会深陷于低自尊的泥沼之中。不论平时她看起来多么自信阳光，此时此刻，她看不到也感受不到自己的价值，因为她的大脑告诉她，她现在正处在一个不安全的情景中，一切以「脱离危险」为优先——赶紧戴上「照顾者」的面具，把自己真实的想法掩藏起来！


等到压力激素褪去，「杏仁体」休息了，「新皮质」重新上班了，小米才会重新感受到，刚才不断讨好的自己是多么可悲、多么不可思议。


我相信，小米的经历，你一定也很熟悉。在一些场景中，你「按道理」应该做这样的反应，但是实际上却做出了那样的反应。


比如说，当有人对你提出无理要求时，「按道理」你应该感到被冒犯，并且果断地拒绝对方，但实际上你却犹犹豫豫地不敢说不，生怕得罪了对方；再比如说，当你不喜欢亲密伴侣的某个举动时，「按道理」你应该去和 TA 沟通，但实际上你却假装什么都没发生，委屈自己去配合伴侣的喜好；还有，当你向领导做工作报告时，尽管工作进度一切顺利，「按道理」你应该充满自信，但实际上你却惶恐不安，觉得自己做的还不够好……


在这些时刻中，你的情绪和行为反应，明显和现实中发生的场景不吻合。明明是一个正常的生活场景，你却深深地感到自己不值得、不够好、不应该，让你产生了「生而为人，我很抱歉」的想法。那么，这些时刻，就是你低自尊的激活点。


在生活中，激活点可以分为两类：内部激活点和外部激活点。内部激活点是你感觉到的、或者体验到的情绪、思想、记忆、身体知觉等；外部激活点是你遇到的场景、碰到的人，或者处在的地理位置。


值得一提的是，在亲密关系中，低自尊的「开关」可能尤其多。这是因为，你和伴侣的相处，与你和父母、和主要抚养人的相处是有一定相似性的，都是依恋关系。因此，你可能会在亲密关系中，看到某一些特定的「开关」会反复出现。


**- 思考题**


在这一节的思考题中，我们就来一起找找，生活中低自尊的激活点。


你可以跟着我回想一下，在过去的一周之内，有没有哪些时刻，你的情绪和行为反应明显和现实中发生的场景不吻合？有没有哪些时刻，你感到了难以消化的负面情绪，例如自卑、嫉妒、多疑、敏感、玻璃心、想要讨好？或者，有没有哪些时刻，你认定自己一无是处、不值得被爱、永远不够好？


当想到了这些时刻，你也许心里会有点不舒服、不开心，也可能感觉有点丢脸，觉得自己怎么这么卑微、这么怂呢？没关系，这很正常，这说明现在你的理性管家——新皮质是在线的，它在帮你很理性地分析状况。


现在，请跟我一起来大声说一句有魔力的句子，来帮助你真正打开心扉，面对这些时刻。


这个有魔力的句子就是——咦？我为什么会这么做呢？——一定要大声地说出来哦！


这句话，会让你对自己当时的行为举止，产生一些好奇，让你变成了一个研究自己行为的心理学家。


然后，你可以在内心问自己：在这些时刻中，你所处的场景、地点，所经历的人际互动等外部环境是怎样的呢？同时，你的情绪、思想、身体直觉等感性体验又发生了哪些变化？


这时，你的回答都可能是你「激活点信息库」中非常重要的信息。当你掌握了足够多的信息后，你也许就会发现属于自己的激活点规律。在未来，当类似的场景、身体感觉、情绪、或者脑海中的声音再次出现的时候，你就能准确地意识到，你的低自尊心态很可能已经、或者即将要被激活了。


此时，你可以提醒自己，你目前的想法很可能已经不再客观，情绪和行为的反应也很可能已经不再理性，你即将做出的决定很有可能是一时冲动的应激行动，而非深思熟虑的抉择。


**- 总结**


好了，本节课就先讲到这里。在这节课中，我们学习到了，大脑可以把同一件事情编码成显性记忆和隐性记忆，因此，许多看似正常的生活情境，却能唤醒我们的童年缺爱回忆，让我们的身心再次进入紧急应激状态：理性管家「新皮质」下班，应激反应专家「杏仁体」登场，我们又不得不戴上「面具」，再次体验其背后的「毒性羞耻感」：生而为人，我很抱歉。
