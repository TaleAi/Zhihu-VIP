## 简介
义和拳是一种什么拳？太平天国的宣传工作搞得很魔性？美国军方有一本叫作《像鬼子一样战斗》的格斗教材？为什么宋朝时候会有日本女性专程来接种生混血儿？……这些知识点严肃又不奇怪，只是不会在考试里出现，所以历史书不会告诉你。


  



知名人文历科普作家刘鹤（ID：幻想狂刘先生）的另类历史课，收录刘先生在历史的学习和研究过程中的随笔文章，回顾历史事件中奇特、诡怪、冷僻的隐藏知识点。


  



「识的这个老妈否——浅谈太平天国的特色修辞发明学」


作为世界历史上影响力最大的农民起义之一的太平天国，其领导人洪秀全有一个特殊的爱好——搞文化建设。迫于水平有限，洪天父的文学创作实数在水平之下，可是当作「打油诗」来读竟然朗朗上口甚至有些幽默，在农民起义军中也魔性的传播起来……洪天王决定创立一套脱胎于汉语但异于汉语的全新话语体系，但是土匪要是有了文化，历史就要出点意外了。


知识点：文化土匪洪秀全；论洪秀全的文学水平；太平天国战时的内部宣传；


  



「1900 ——义和神拳志怪录」


1898 年的戊戌变法和 1899 年的义和团运动代表了知识分子和农民阶层在西方文明的威胁下做出的不同反应和选择，前者的鲜血为辛亥革命埋下伏笔，后者却被归为「愚昧」「残暴」「没文化」，历史书没告诉你义和团是怎么和这些词汇画上等号的？这要从义和拳的宗教性盖过了技击性说起。


知识点：义和拳是什么拳；真实的义和团运动是什么样子；义和拳是传武吗；


  



「汉江怪物——当代半岛名族主义杂谈」


为什么全世界都抵触「韩国起源说」，而韩国却一而再再而三的宣扬事件万物皆起源韩国？为什么明面上朝韩关系紧张，但民间往来密切又友好？为什么中日韩文化类型相似的三个国家，却没有形成共同记忆，从上至下的不理解，形成现在看似文化撕裂的东北亚局势?


知识点：万物皆起源韩国；「三重国恨」；申采浩；


  



「白刃相向彻骨寒——旧日本陆军白刃战史话」


热兵器时代，为什么「拼刺刀」的白刃战成为中日战场上出场率最高的作战方式？日军的白刃战技术水平到底怎样？让我们从日军白刃战教材《1915 剣術教範》和美军解析日军刺刀技术的教材《How to fight like a jap》（如何像鬼子一样战斗）中窥见一斑。


知识点：抗日战争；白刃战；大刀与刺刀；
