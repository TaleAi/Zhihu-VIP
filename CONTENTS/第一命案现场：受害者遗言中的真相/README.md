## 简介
有什么巨短却超级吓人的恐怖故事？


这世界每一天都有形形色色的命案发生。


凶杀案一般可以分为两种，一种是激情杀人，一种是预谋杀人。无论是哪一种，都会存在犯罪现场。


有人说这是受害者留给我们最后的遗言。


但第一个读到遗言的人，往往是犯罪者。他们将受害者的遗言藏在自己的规则之下，从而掩饰自己的杀人行为，也因此犯罪现场的调查就变成了警察和罪犯的第一次博弈较量。


从警多年，我见识了各种光怪陆离充满诡谲的现场，这其中最具冲击和震撼力的，我称之为第一命案现场，残忍第一、诡谲第一、狡诈第一……


有凶手将尸体塞进鱼缸，摆成了招财猫的形态，就为了隐藏真实的案发案场。


也有人在浴室里杀人后逃脱，利用密室巧妙的制造「不可能犯罪」。


杀人后纵火的凶手，为了将尸体和犯罪现场燃烧殆尽，以残忍的方式扯烂受害者的遗言。


……


我更习惯将案发现场当做犯罪者的「病状」。


杀人是最严重的犯罪，也是人类历史上最恐怖的顽疾。而犯罪现场就是顽疾发作时的病状，杀人动机是症结。


这病有的治。


借用一句老话：法网恢恢，疏而不漏。
