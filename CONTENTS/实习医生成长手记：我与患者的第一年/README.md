## 简介
本专栏是一则别具一格的医生见闻录，是一部幽默从容的实习医生成长笔记。作者伊声在毕业前夕来到本市最著名的三甲医院实习，在这里，他辗转急诊室、风湿科、骨科、妇产科、整容外科……见证了最罕见的病例，最令人扼腕的离别，最蛮不讲理的医闹事件，最善良的医生同行。
「在这场实习结束后，我决定当一辈子医生。」


念医学院的第五年，我申请到了建安医院的实习机会，救死扶伤是我自小的梦想，建安医院是我的 dream hospital。


与我同行的，是我最好的哥们大麦。大麦从没想过要当医生，临床医学是他转专业失败的结果。


从来到建安医院的第一天起，我记述下了这里的每一桩救死扶伤的故事，每一则揪心不已的医闹事件：


因吞咽饭团被噎死的肾内科住院老汉，家属将死亡责任赖在主治医师身上。搭上一整年工资作赔偿的大夫，默默地担下这个责任：他并没有离开医院，他仍然坚持做医生。


逃避警察追捕跳下四层楼的黑帮老大，被医院的手术天才救治成功。天才主刀说：「生命平等，医生首先看到的是疾病。」


收治了几乎无药可医的高空坠楼者，医院最终放弃对她的资源扶持。是什么让白衣天使眼睁睁地看着一条生命的逝去？


产科的准妈妈得知双胞胎必有一畸胎时，是什么让她毅然决然地坚持满月并将孩子产下？在一场意外失去孩子的事故背后，看起来彬彬有礼的初产妇到底对婆家和医生隐藏了怎样的秘密？


一名由父母带到整容外科的特殊病人，ta 的首要诉求竟然是搞清自己是男是女？


经历了做医生的实战考验，大麦和我，将何去何从？  




注：本专栏为非虚构真实故事集，所述事件均真实发生过。出于对当事人隐私的保密，文中医院和医患姓名皆为化名。
