## 简介
人生不完美，但我爱这有缺憾的自己。


短暂的人生中，我会遇到很多人，我能做的只有捧出一颗赤诚的心，与顺路的人结伴而行。


如若等不到同路人，那就将满身尖锐化作铠甲，做一个独行侠。


《**小叔如父**》


小叔是江家的大冤种。我是江家的小冤种。


我们都是不被父母喜爱的孩子。


后来，小叔和我越来越有出息，曾经对我们非打即骂的亲人却眼巴巴地凑了上来。


可惜，已经被消磨殆尽的亲情，却再也挽不回来了。


**《玫瑰和雏菊都会盛开》**


我闺蜜是白富美，家世好，人美心善，聪明优秀。


我就像她身边的陪衬，普通又平凡。


有人问：人生来就不平等，你嫉妒吗？


嫉妒？当然不嫉妒，他们只看到了闺蜜光鲜亮丽的一面，却没看到她背后吃的苦。


所谓「如人饮水，冷暖自知」，我和她，不同的人生会有不同的幸福。
