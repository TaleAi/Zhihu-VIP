## 简介
阅读不应该是一件功利的事，但也确实存在一些方法，能够帮助你更好地汲取书中的知识，让你的阅读避免成为「无用功」。在碎片化到读一本书已经成为「深度阅读」的时代，本专栏希望通过书评，来讲一讲更深的深度阅读。


你是否有这样的困惑：


终于下定决心翻开一本书？十分钟就想玩手机。


坚持读完一本书却很难记住讲了什么？


已经读过不少书但收获感觉不如预期？


本专栏不仅仅是手把手教你如何写一篇书评，更是借助书评这一典型体裁，以一种评论视角，看待阅读。一篇好的书评背后，是一个人阅读能力、分析能力、深度独立思考能力、观点表达能力综合体现。基于这些能力，你可以用一种新的眼光阅读和理解书籍，并且获得更多知识和乐趣。
