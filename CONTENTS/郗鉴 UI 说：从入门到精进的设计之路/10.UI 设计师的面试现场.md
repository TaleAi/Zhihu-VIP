你好，请坐，这里郗鉴 UI 说，我是郗鉴。


今天跟大家聊的话题是 UI 设计师的面试现场。是不是等了很久了这个话题，我们终于要聊面试了。面试其实是一个后期的工序，所以说你要想成为 UI 设计师，不要先想怎么面试。这个我觉得是非常的不正确的，你作品非常好了以后，然后你去准备一个面试的流程，然后才是去面试。不是说咱们一步就就找工作去面试了，那不是我们这个行业，可能是这种销售部门，对吧？我们这行业其实前期的准备是非常耗时间的，我们准备什么？


不管大家是处于一个什么情况，不管大家是校招、实习、转行、换工作，不管是哪个方面，我们都需要一个时间去积累作品。有人说不好意思，我问一下我之前不是 UI 设计师，我之前可能是搞体育的。对吧？我就是特别喜欢。UI 我特别想做 UI 这个行业，我怎么积累作品？既然您喜欢 UI 总得拿出点证据对吧？咱们说谈恋爱也是，你说你爱我，你为什么说你爱我，对吧？你有什么证据对吧？你给我怎么证明一下，所以说最好的证明就是作品了这个作品。


判断一个设计师的是不是能够胜任这工作非常重要。我看你作品，我就知道你大概能做成什么样，我们这儿这个要求你能不能满足。转行的设计师其实也应该花一段时间去学习，学习有方法有很多，你怎么学都行，总之最后是要一份很棒的作品集，至少你要让面试官要知道，这个人他能干活，对吧？他能胜任，我们就想要这样的人。对吧？然后你再一看你是转行的，对吧？你之前是搞体育的，打篮球的，也就 OK 对吧？实际上你这个做的是我满意的了，对吧？

在大学的目的就是学习，然后学习完以后，你就赶紧沉淀，把你的作品进大学这段时间画的画什么也好，然后你做的一些图标也好，做的一些练习也好，整理成作品集。即使你不是 UI 行业，我也要告诉你，设计类的在校生都应该这样，就说你之前在学校的所有的活动都是应该围绕着设计而来，围绕你的专业，对吧？然后你可以学学其他的东西，当然很有益处，比如说哲学，比如体育都很好，但是你的生活的重心应该是设计。这个会造成一种误解，因为在学校可能比如说老师可能更喜欢一个，比如说有能的领导能力的孩子，对吧？但是有领导能力的孩子成不了，就不能直接成为设计师，对不对？你想能成直接成为总监吗？不可能，对吧？你必须先做设计师吗？所以你设计能力还得好，所以一切的生活中心，你的学习重心应该还是设计。


你可以联系一些设计的周围技能，手绘、摄影，对吧？挺丰富的，你可以来学一学，所以这是在校生跟准备找实习的。我想换份工作怎么办？还是做作品。做什么？其实有很多的设计师在工作中的项目很单一，我是做一个社交 APP 的 UI 设计师。我一年的工作是做 1.0，第二年做 2.0，第三年做 3.0，第四年都 4.0 对吧？做五年一看都 APP 它都是蓝的跟白的对吧？索然无味是不是？你们说对吧？所以我们这样的设计师 UI 设计师工作 1 到 3 年，大家的问题就是能不能真正的把自己的水平拔高，在作品中作品集中能够体现出来，这不要放出来都是得一样。


包括有些 UI 设计师可能在工作中做的东西比较杂，今天做一个电商，明天做一个线下的一个海报，这种东西其实它很难帮助你再找一份比这好的工作，他顶多给你找一个跟这一样的工作，对吧？怎么办？我们想拔高，这跳槽不是被拔高吗？对吧？做好作品。


这作品集你要让人看到以后觉得你比你现在档位高一个档次，你的项目是不是得丰富，说我去哪找项目接私活吗？你就可以做练习对吧？但是我们做练习的时候要有的放矢，就不是说我就看今天看一个网上有一个哥们做了一个概念的一个东西，我要做概念。是可以做的事 OK 的，但是我说的还是更加管用的一些偏真实方向的一些设计，比如说我们做一个网站，你就不能光为了酷炫而做，你要考虑到后面的我们，假如说我们模拟一个电商的平台，它的战略层的想法是怎样的，对吧？然后它框架层是怎样的，表现层我是用了什么样的一个色彩的一个结构，我的 logo 长什么样？

来说作品集这作品集不是一个大长条。作品集目前比较公认的好形式，是那种这个长方形举行的，我会建议大家做成 1920×1080 是一张，然后你可以做四十几张，就第一页是封面，然后一番没落在一番，第一个 APP 在翻 APP 的一些原型图，再翻设计稿，然后第二个 APP 然后是网站、图标、手绘一些东西，最后谢谢，这是我的电话，如果您对我有兴趣，请拨打电话。这是一个完整的作品集，一般这格式是 pdf，pdf 是电子书的格式，对吧？

实习生的统一的按校招以后，第一批的是按照学历学校他有一个范畴，然后我们这种已经是设计师，然后在再次跳槽的时候会有这个问题，就是涨多少？一般来说我看一个数据，说是每次跳槽是 30% 左右是正常的，所以给大家来个参考最后还会有一个人来面你，但是这个人他可能陷到沙发里了，然后他说话可能也跟设计没什么关系，这时候你不要瞧不起人家，有可能是 CEO 对吧？有的 CEO 会老忘记自己介绍一下，但我是 CEO 他不他坐那就跟你聊，所以你要好好聊，但是如果能进入最后一面的话，表示这个希望很大了，如果在二面跟三面之前，他跟你说想做个面试题，你可以放心大胆的做，如果你对它这公司有意思的话，就做面试题，这好多警惕性很高的朋友觉得这是不是骗子，我可以告诉你，不是面试题，就是这个想测试一下你的真实水平，你万一作品集里作品是这个，这个不是你特别原创的，对吧？万一作品集体作品，它跟公司实际项目它不太像，对吧？所以做个面试题也是正常的，面试题特点就是要做得快做得好，然后写设计说明给他看就行了。


然后如果你完成这些步骤以后你就入职了，入职以后也要戒骄戒躁，然后希望大家也是注意一下，不要轻易地选择跳槽一定是比较慎重，在一个公司里其实呆一年以上两年两三年都很正常，也是都对自己会非常好，因为你只有在年限的情况下，你才能从一个设计师在这公司里熟悉环境，然后可以慢慢的向上发展。咱别每次都当一个初级设计师，今天在 A 公司好不容易换手了咱去 B 公司了对吧，这也不合适，所以这是我想跟大家来聊的面试现场，不知道你们对我这期满不满意，如果有更多的面试问题的话，欢迎在评论区留言，这期节目就到这里了，我是郗鉴，再见~
