## 简介
UI 是什么，如何入门？
学好设计，还需要了解哪些心理学知识？
设计师岗位的人，有哪些高风险操作需要避免？
科技不断发展，设计师的核心竞争力是什么？

在这门音频课程中，资深设计师郗鉴将亲授你从入门到精进的独门心得。



郗鉴是谁？


* 站酷推荐设计师，西见网络科技创始人
* 前腾讯视觉设计师、Adobe 认证界面设计师
* 设计过移动版腾讯网、腾讯世界杯项目组等产品
* 后完成《二代妖精》《幕后玩家》等电影 FUI

在这门课中，郗鉴将解答一个 UI 设计小白的种种困扰：


* 空有热情，不知如何入门
* 前路茫茫，不知如何进阶
* 性格内向，不会与人交流

从入门到进阶，你将得到全方位的「过来人」指点：


* UI 基础知识、设计历史、字体历史、色彩心理学等硬核内容；
* 如何看待科技发展对设计产生的影响；
* 如何提升自己、在闲暇时间内应怎样选择知识输入；
* 与产品经理，技术人员对接工作的技巧。
