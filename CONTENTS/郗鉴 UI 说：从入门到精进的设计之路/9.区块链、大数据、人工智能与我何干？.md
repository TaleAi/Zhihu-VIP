## 9.区块链、大数据、人工智能与我何干？
你好，请坐，这里是郗鉴 UI 说。


今天想跟大家聊一个非常有意思的话题，这个区块链大数据，人工智能与我何干？很多时候有很多朋友都来问我，郗鉴，你看现在 VR 跟 AR 技术这么火爆，我们要不要学些 VRUI 对吧？还有些朋友说旗舰区块链人工智能发展起来了，我怎么能够赶紧学习，我感觉到很要被时代淘汰了，对吧？你同龄设计师正在淘汰你，对吧？还有些朋友担心，人工智能会不会取代设计师，所以今天想跟大家聊一聊技术跟我们设计之间的一个关系。


其实想聊这个话题，我又想扯一扯历史了，最早之前从包豪斯的时代开始技术，我们设计研究的一个课题，因为我们的设计要随着技术而改变。这个是真的吗？咱们从头看起。UI 设计叫 user interface design 它是用户图形界面设计的一个门类。UI 设计在最早的时候，我们互联网最早的时候，他们是一起出现了再 web 1.0 的时代，UI 设计就出现了。那么 web 1.0 是什么？web 1.0 指的是这个网站或者是某个平台，然后它向用户去发布内容，用户像在看电视一样接收这样的内容。 


Web 1.0， web 是网络的意思了，也是网站的意思，所以其实当时指的都是 PC 端，web 1.0 的时代，UI 设计有什么样的风格？UI 设计当时是水晶按钮，然后加一些图片，对吧？图片比较少，然后文字会比较多，为什么会是这样的风格？因为当时的技术很有限，当时的技术 html 以及当时的带宽都无法支持很复杂的交互。所以当时我们 UI 设计适应了外部 1.0 的形式，包括它的产品形态，在后面出现了 web 2.0，我相信您一定听过。 Web 2.0 很出名，web 2.0 说的是什么呢？ 


Web 2.0 说的是我们每一个用户，都是一个内容生产的中心。今天我们都是内容生产的中心了，对吧？我们发朋友圈，然后我们看微博，我们既是内容的消费者，也是内容的生产者，这种形式叫也叫做 UGC。我们 web 2.0 的时代 UI 设计也发生了一定的变革，随着技术跟产品形态的变化，UI 设计也适应了这种变化。比如说更大的布局更大的图片，甚至我们 UI 设计，也产生出这种 timeline 的形式，这种时间线的形式，时间线的形式不太好解释，就比较像朋友圈了，对吧？你往下翻，然后是过去的时间线，最上面是现在的时间线，后来就出现了移动互联网，我们的互联网，这个服务从 PC 端变成了手机端，飞跃非常之大。


有人说这有什么飞跃，这不是都是提供互联网服务吗？您就错了，因为移动端的互联网它的使用场景不一样，它有几个特点，我随便说几个，比如说随时随地，对吧？我们随时随地可以使用互联网了，对吧？而过去的 PC 端你只能在电脑前坐着，然后就是碎片化，因为使用移动端的时候，我们可能在做着另一件事情，然后中间去使用它，作为一个时间的消遣，或者是一个时间的补充，然后我们就去返回去干其他事情了。


然后还有很多的使用场景，跟用户的画像不太一样的地方，都改变了我们的设计。比如说在乔布斯发布了第一代 iPhone iPhone 3GS 的时候，UI 设计也改变了。乔布斯称之为革命性的用户界面，它有什么革命性的？首先第一代 iPhone 的所有的图标都是圆角，圆角矩形，一直到今天都是圆角矩形。大家知道为什么不是矩形吗？因为矩形在屏幕上看着很扎手，因为对于刚刚接触这种一个大黑玻璃的用户来说，点击的感觉是很难找到的。


我们在现实中点击的感觉比较容易找到，比如说咱们家开关到比如说咱们一个按钮，而在整个屏幕上的一个虚拟按钮的话，它是很难找到感觉的，所以远角看起来更舒服，并且还会用到一些玻璃以及金属的质感，这样的话更增强了点击的这种感受了，对吧？所以它会让整个的我们的用户，从一个没有移动端操作经验的人，过渡到一个可以适应的这么一个经验。如果当时所有的界面都是扁平化的话，用户无疑是会非常迷惑的，但是在后来 ui 设计，从拟物化走向了扁平化，这是为什么呢？ 


因为你在环视周围的时候，你发现所有人都在玩手机了，对吧？而且我们每天玩手机的时间简直快大于工作时间了，对吧？我们可能跟手机陪伴的时间零零散散加起来都快超过八个小时了，所以我们这么长时间的操作手机，重复盯着一些玻璃按钮，水晶效果，对吧？金属拉丝你肯定会觉得不太舒服，你想关注内容，觉得这些东西是一些视觉上的噪音了，对吧？就觉得看这些东西怎么这个细节怎么这么多好烦，对吧？所以扁平化时代来了，也适应着我们的用户的一个发展跟进步。


UI 设计，同时又改变了自己我们的按钮，没有那么多效果了。对吧？我们的设计突出了内容了，淡化了我们的装饰。整体上我们以内容输出为准，我们现在可以看到 iOS 的大标题风格简直看上去就像是一张报纸一样，对吧？为什么他想突出的是内容？突出是这个内容，而并非装饰性的设计的元素。我们如果是这么做的话，设计的元素就会比较少了，所以 UI 设计又从别的领域拿来了一些非常适合用户缓解他们的阅读压力的东西了。比如说动效，我们 UI 设计从传统动画行业中把动效的这些知识拿了过来，所以我们现在今天 UI 设计师需要懂得一些动效的这个原理。 


由于按钮跟图标的装饰没有过去那么强了，所以我们又从原画插画这样的领域借鉴过来一些图形表达的方式。所以我们今天看到很多的 APP 虽然它的按钮并不花哨，但是它会有一些插图或者是这种图形的东西来缓解长期阅读，造成了这种视疲劳，所以这也是很好的。所以 UI 设计也随着移动互联网技术改变了自己。


移动互联网上还有很多的产品形态，我们做设计的时候也会根据产品形态的不同而调整我们自身的设计，比如说共享经济是什么？共享经济就是把社会的闲置资源调用起来的一种产品形式，比如说共享充电宝、共享单车、共享汽车，对吧？共享雨伞等等之类的。虽然共享经济有很多的问题，比如说安全的问题，是吧？比如说使用损坏的问题，环境的问题，对吧？但它终究是一个比较新的一个模式，这样的产品形态下的 UI 设计也需要调整自身的一些一些细节，对吧？来突出它整个产品的一个主线。 


还有比如说大家可能比较熟悉的，在新闻里老看到的互联网网贷的公司，也叫做 P2P 这种产品形态，P2P 其实就是怎么说，就是一个私营的公司，他开始做过去我们认为是银行才能做的事情，比如说这边是来存钱的人，然后这边是来借钱的人，然后他中间去收取一个利益。但是互联网网贷前段时间去爆雷了，很多，大家应该也能够看到爆雷的，就是因为这个产品形态其实没有太大的问题，但是人是很重要的问题，跟共享经济一样人。


是这个产品最后是否走上正道的一个主要的环节，他没有太多的人来借钱，所以他把之后存钱的人的钱给之前的人当利息，所以这样的话就变成了一个庞氏骗局，对吧？这个也是一个不太好的事情，但是它这个产品形态本身，我们是可以介绍一下，还有付费知识，付费知识是什么呢？付费知识或者叫知识付费，因为我们更渴求知识，说明我们社会的这个层次，或者说我们的人群的对学习的一个向往是越来越强的了，对吧？还有直播这种形式，直播也分为很多种，最简单的就是大家比较熟悉的这种这种刷游艇的形式，对吧？所以如果你是在这样的直播公司做视觉设计的时候，也需要考虑到用户的场景以及这个产品的形态，它的带来的一些问题，以及如何用视觉的方法，如何用交互的方法来解决这些问题。


总体来说其实我们 UI 设计伴随着技术一起发展的未来，我们应该关注一些什么样的技术？他们会对我们的设计带来怎样的一个改变？相信很多朋友应该都比较熟悉这个词，叫区块链，英文是 blockchain。区块链技术，其实它本身有一句话叫技术无罪，是吧？这个区块链本身是没有问题的，区块链技术是一种加密的形式、去中心化的互联网这样的一个技术，它并不仅仅是这个虚拟，货币，它其实更多的是一种这种就是去中心化互联网的一种技术。我们知道现在的互联网是中心化的，互联网就有一些问题，去中心的话，互联网无疑让互联网更加的自由，更加的开放，也更加的公平了，对吧？


但是我们现在往往我们的咱们在吃饭时候聊天的时候，往往把区块链技术混同于虚拟货币技术，什么挖矿，大家应该比较熟了，对吧？这个虚拟货币是什么？虚拟货币其实它应用了区块链的技术，然后比如说我发行多少货币，对吧？然后你可以用多少钱来买它。然后由于它的数量是固定的，然后随着关注度以及他这能够消费的地方，所以它会越来越火爆，虚拟货币的本质其实就是这个。其实我们货币的本质跟虚拟货币就差不多，我们本身这个钱是一个比较抽象的概念。


比如说我们中国，我们所有中文跟钱有关的字，都是有一个被这是为什么？因为古代中国我们是个内陆国家，对吧？所以贝壳比较难找，我们拿贝壳来当前。贝壳其实这个也不是牛不是马对吧？但是如果你有足够的贝壳，你就可以买到这些东西，它是一个指代物，它比较抽象，它就是用一个东西来指代。后来整个世界都用黄金来指代了，对吧？因为黄金它数量比较稳定，互联网货币它为什么不能用一个更加虚拟更加抽象的形式来表示前的概念，所以其实这个也是没有问题的，技术本身没有问题，还是存在于人身上，对吧？用我妈妈那代人的话叫投机倒把，对吧？有很多这样的创业者，然后发行一些毫无意义的虚拟货币，对吧？他也不关注技术，他就是来做这件事情。


包括之前我刚才讲了互联网经济有很多不太好的例子，对吧？P2P 、知识付费、直播都有不太好的例子，对吧？但是它的技术本身是好的，如果我们在这样的公司在这样的产品模式下做设计的话，我们也可以用设计的形式来帮助用户更好的理解这项技术。区块链就跟大家来简单的介绍到这里，还有些人很关注大数据，其实大数据也是一个非常就是有影响力的一个词，忘比如说现在的快递柜其实就非常的智能，它根据你的消费习惯，会提前知道你要买什么东西，他会为你提前预留出位置，并且推算出你应该该买啥了。


这个其实很可怕，我们其实在这种互联网的非常发达的基础之下，简直就是裸体站在这里了，对吧？我下个月买什么你都知道，你都给我放好格子了，对吧？所以这个就特别的挺吓人的，对吧？再比如说我们在网络上一些电商平台，我们经常看一个东西的时候，他会猜我们喜欢。这也是蛮恐怖的，其实他就把我们贴成标签，比如说你是一个男生，你多大年龄，你在什么城市，你之前买过什么，所以根据很多百分之多少多少的人跟你相似的标签的人都买了，所以我也给你推荐这个。


而且他推荐的是越来越准了，其实这令我毛骨悚然，我没看某电商平台的时候，我都不知道我想买它，看完了觉得这不是我想买的吗？所以这挺可怕的，但是大数据作为我们一个技术来说，我们能直接应用的设计其实并不多，因为它其实更是一个有点偏后台的技术对吧？所以这个事我们大概了解他是怎么回事，他怎么工作的就 OK 了人工智能人工智能也是最近非常火的词，对吧？人工智能叫 artificial intelligence，简称就是 AI 了，对吧？人工智能出了一一些非常棒的产品，比如说我们国内的公司开发出来了一些可以做运营图，做海报，甚至做 UI 界面的一些机器人，你们害怕吗？ 


害怕被机器人取代吗？其实如果让我来说的话，大家完全不必紧张。有一个有一个表述的一个实验叫中文房间，我不知道大家听没听过。中文房间的意思就是说，如果我把一个美国人关在一个屋子里，我每天拿纸条写上中文给他塞进去，比如说今天你要吃什么，对吧？美国人在屋子里是个封闭的空间，但是给他扔一本中英对照的词典。然后他看着这个我的词，查我的词，然后他根据那个词在写这个词再递给我，我们俩是可以交流了。但是他懂中文吗？


他不懂。对吧？所以这个很像目前的 AI 阶段。AI 现在其实并不高级，它并不理解我在做的事情是什么，他只是知道你给我一个命令，我应该怎么去反馈你，他是还是一个比较初级的 UI 阶段了，对吧？这样的初级的 AI 阶段的人工智能是没有能力代替设计师的，但是它可以代替很多的行业。比如说我们现在很多的政府部门的窗口，对吧？医院挂号的窗口，对吧？我觉得医院挂号的工作人员完全就可以变成一张纸，贴个二维码就行了。


我们一扫然后就可以挂号了，对吧？所以这个本身就是可以被人工智能或者被互联网的程序所取代的，但是我们设计其实是很难被取代的，对吧？我们设计本身是一种感性思维的一种产物，对吧？我们虽然要懂一些交互的理论以及逻辑的知识，但是我们主要是靠我们的审美以及我们的看很多设计产生出来的一些灵感来驱动的，所以这个是。A.I. 我认为很很多年之内它无法解决的。所以大家也不用担心，人工智能能不能代替我们，其实人工智能已经来了。 


因为现在很多公司都在投入人工智能的研究，比如说在 Photoshop 中就有。人工智能。在 Photoshop 中哪有，人工智能我们会注意到现在有很多，否则效果最新的功能中包括一些抠图，对吧？包括一些人脸的识别都用的，人工智能他分析出来人的几亿张照片的一些共同特点，比如说它两个眼睛一个鼻子是吧，然后头发是一些零碎的这种随风飘舞的这种线条，对吧？人工智能就学习到了它内部叫 Adobe sensei 是吧？Sensei 日语老师的意思，所以人工智能就学习到这是这个人的一个图像，所以在抠图的时候他更加智能了，他知道大约是个人的眼睛，这个大约是个头发，我的操作者大概是想把头发抠出来，所以人工智能已经来了，所以设计师现在也不用像过去一样，就是来过分的关注，比如说怎么做个火焰字。


但我记得很老的设计师，其实他有一套怎么做火焰字的一个表，比如说先转一下，然后再加个风的效果，然后再加一个颜色，再加个滤镜，对吧？现在我们不用了。我们设计师现在的设计师很幸福，我们更多关注的是创意，是图形是我们的一些思维层面的东西，而技术方面他越来越会被 AI 提高我们的效率，所以这对大家应该是件好事情，所以大家不用担心说 UI 设计被人工智能取代了不可能，但是人工智能会帮助我们提高我们的工作效率。人工智能也是一个偏后台的一个一个操作，所以其实它对于我们的设计是有帮助的，但是我们的设计像大数据一样，我们没有说直接是因为人工智能的运算的逻辑算法的逻辑而改变我们的设计风格，这还不太不太可能，最后想跟大家来聊一聊。


VR 跟 AR 技术了，VR 是虚拟现实 ，AR 是增强现实的意思。现在我们应不应该来关注一些 VR 跟 AR 的 UI？ 我认为咱们分开说，VR 略早，为什么现在的 AR 这个技术很好，技术也趋向成熟了，但是卡在应用上了。很多年前有一款也要改变世界的产品，这个公司也很好，很大，叫 Google。所以这个产品是 Google Glass。是一款眼镜，它是增强现实的一款眼镜。戴上这款眼镜的话，你可以就不用低头了，对吧？你就可以看着这个你所处的环境，看着你的人，看着你的朋友，他的头上可能会弹出一个联系人的介绍之类的，对吧？


技术很棒，对吧？他卡在应用上。卡在用户能不能接受这个产品上了。所以 google glass 其实并没有改变世界。iPhone 改变世界了。对吧？但还有很多的技术，没有像 iPhone 这么幸运。所以在这个阶段我们关注一些非常前沿的技术是好的，但是你要是认为这个技术是你的未来，还有点早，我们要看到市场用户是不是接受这份技术，它能不能被应用在现实生活中，这很重要。


VR 目前来说，应用的比较多的可能是每个商场的游戏机，比如上一趟 50 块钱，然后让你体验一下蹦极的感觉，对吧？然后给你戴个眼镜，然后拿一两个手柄。所以现在的 VR 的应用这个场景比较窄，我目前这是我个人观点，所以在大商场里老能见着，但是它并不是一个用户已经很认可的一个设备。所以 VRUI 我认为尚需时日目前是一个不错的趋势，但是他最后是怎样的，我们还不知道。AR 无疑要好得很多，为什么？因为 AR 应用很简单，我们每个人都有一份 AR 设备，就是我们的手机。 


VR 的话，你需要给你的手机至少买个十几块钱的一个盒子戴在眼睛上，对吧？把手机挂眼睛上那才行。而 AR 完全不需要。所以最新的苹果的 iOS 版本中就包含了 AR 的组件，就是增强现实。比如说我拿手机照这摄像头，启动了从家里的那个点到我的桌子，他可以通过近大远小的透视关系给你量出来是多少米，或者比如说宜家，我可以从宜家的 AR 的程序中，把一些个桌子放在咱们这个家的中间，看看效果如何。所以说 AR 更好应用，对吧？VR 它由于设备跟市场的原因，它我们还是先观察比较稳妥，ARUI 我们用学习吗？其实 AR 的 UI 跟我们现在做的界面的 UI 没什么区别，对吧？包括按钮，包括信息的显示什么的，就弹窗之类的，没有什么区别，所以大家也不用惊慌。
