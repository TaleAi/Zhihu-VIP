## 4.勤有功戏无益
你好，请坐。这里郗鉴 UI 说，今天我们的主题是勤有功戏无益。


勤有功戏无益，其实是我之前一本书的一个章节，题目这里我主要想告诉大家的是，只要大家勤奋努力，就会有很好的收获。勤有功戏无益的完全的句子应该是「勤有功，戏无益。戒之哉，以勉励」。这是南宋的一个官员叫王应麟说过的一句话，那么我们作为设计师来说，怎么样才算努力，怎么样才算不荒废时间，是吧？怎么样才算不贪玩？这是一个很有意思的事，首先我想跟大家来聊的是我当年的一个经历。


在我刚毕业的时候，那会儿是 2011 年毕的业，一年毕业以后我就到了北京，然后开始工作，找工作的过程非常的艰辛，很多人以为现在找工作设计师找工作好像是比较困难！其实我感觉哪年的工作也不是很好找，对吧？当年也是跟很多朋友一样，海投对吧？海面当时我有一张纸保留，到了，现在我去一公司就划掉一行纸，上列了这么 20 多家公司，好不容易找到一家不错的公司，一个团队，它人数也不是很多，然后我记得很清楚，让我当场做一个 404 页面，当场我这没有电脑，他就给我借了一台财务用的电脑，说你这个来搞，现在是 11 点，你什么时候做完什么时候叫我。


这个我又不太好意思当着她的面下载一些素材，就完全凭自己的感觉来画，拿鼠标画画的就不是非常的好，后来面试官就说好像跟你的作品集里还差很多，怎么回事，紧张吗？是吧？当时我这个脸我记得通红，因为其实本身我对鼠标画的图形当时不是很熟练，然后说这是这机器可能不太熟，后来我就入职了这家公司。


其实说找工作一直到刚找到工作这段时间，所有人都是比较勤奋的，包括我当年也是都非常的勤奋对吧？因为老觉得陌生的环境会让人紧张，我记得当时好像吃饭都是我要拿余光扫描，周围的人都撤了我再去吃，是吧？这个工作上也会每天的去努力想提高一下，但是给我的工作其实并不是十分的重要，当然也可以理解，我这当时是一个刚毕业的小后生，难堪重任，给我一些比较轻松，P 图的活我也能理解。但是后来这个轻松 P 图的活就变成了海量 P 图的活，我就活生生变成了人肉 PS 机，去个水印什么加个框，这都是我干的事，我也觉得稍微有点没什么成就感，其实也是挺难受的。他们也觉得我作图确实不行，当年，对吧？后来我就会发现我做的事情确实不太重要，但是他们也是因为我对我这个作品不太认可，所以我需要提升我的作品，在工作中提升。


好像没什么戏，因为就我当时的环境来说好像很难，说是让老设计师手把手的来教学，不太现实，他每天忙的也在加班是吧？我当年也没有这么发达的网络资源，说这找一个效果，就是一大堆视频都出来了，现在怎么办？于是每天下了班以后，我就会在我租的房子里去练一些设计，去练练图标什么的。当时我在北京就我一个人，所以确实非常的孤独，我觉得其实很多时候人是与孤独做斗争的。太寂寞了是吧？当时我拿着我偏色的笔记本电脑在床上练图标，因为没有桌子。


我租的那个房子是家徒四壁，是四面白墙一个窗户，没有桌子，这真是很惨。然后做了一半，我下地要上个厕所，差点走不到厕所就尿裤子了，因为这个腿麻了。对吧？所以这个还是很艰难的，就是说你说回家躺一会儿对不对？加班加到 9、10 点才能到家，躺一会儿的话才能缓一缓。当年之所以能够那么努力那么拼，也是因为确实我没有什么别的更好的选择，很多人说你为什么会当年那么努力，没什么选择，又不会干别的，对吧？然后就做了这些练习，很大量的练习。


很多练习现在你们可能没有看到，我其实临摹了大量设计师的图标，我也自己原创了好多的图标，但是都做得非常之差。这个也正常，对吧？确实没有一个很方便的通道去学习，就只能自己摸索了，而且我觉得做得太差的，我也不想发到网络上是吧？所以我就一直电脑里有一大部分图标都是当年练出来的。


后来我觉得我做得还不错了，但当然现在来看的话，确实是只有自己觉得自己做得不错。于是我就想参加一个比赛，正好彼时站酷上有一个比赛，所以说我确实是在站酷上获得了不少的机会，站酷这比赛在当年参赛的设计师已经有很多非常厉害的人了，主要是做一个图标，做一组图标，做这组图标的时候，我这个其实没什么信心，我也是照猫画虎，照着天气预报那种普通的天气图标来做的。


然后在公司里的时候，我也是希望，比如说中午吃饭的时候，我能够不睡午觉，能够稍微改一改。很巧的事，中午没有休息的时候，我在改改我自己的参赛图标，然后总监从后面飘过：「这是干啥？」我说正在参加比赛，他说你这做的不行，你看你自己都虚了，对吧？你看这没有什么意义，你这参赛浪费时间，只会让你自己遭受打击是吧？我不知道你们是不是身边也有这样的人，就说他们可能不太会给你鼓励是吧？你在努力的时候，他们老是觉得你做东西没有什么用，但是好在当年我也没太听别人的话，我这人主意还是比较正的，于是我就继续的来做我自己的东西。


后来就得了奖了，对吧？得了一个大抱枕，对吧？是那种极其普通简陋的抱枕，但是这个是我人生中第一次参加 UI 比赛得的奖品，虽然是这个没几块钱，但是我觉得是非常重要的一个里程碑，同事们狂笑不止，说干了一个月挣了个两三块的抱枕。但是我非常喜欢，我非常的觉得有纪念价值。


后来呢后来我参完赛了以后，也会去练一些图标，慢慢的工作中有一些需求，我觉得我能干了，与是我就主动出击，找总监来要一些，我觉得我能够搞定的项目。后来也比较顺利，我就从 P 图领域走出来，正式变成 UI 设计师了，所以说这个机会是自己创造。


当时其实很多同行跟我一边大的设计师，下了班以后也会去参加很多的活动，包括参加一些分享会，对吧？参加很多这样聚会，然后练练瑜伽什么的，但我认为当时当年的我重点应该是提升视觉表达能力，对吧？所以这些事儿我就不参与了，而且当年这些活动都比较费钱，我当年非常穷，然后也舍不得参加，参加一次能吃好几天饭，这也其实帮了我。其实我们会发现很多的活动，我不是说所有活动，就是一部分的活动，其实是没有什么价值的。


有一本书叫做 Rework，中文翻译叫重来，就说的是大部分的会议跟你没什么关系，我觉得这个也适用于一些分享会。大部分的这种会议，尤其是人数更多的会议，其实对我们实际的帮助可以说是不是很大，所以说如果你在年轻的时候，有一个机会让你去参加一个全世界的设计大会，还有一个机会是你可以在家做一天图，是吧？我比较鼓励我们表达能力视觉表达能力不太强的同学，还是要做图，与 Photoshop 成为朋友，是吧？不要这个觉得人脉可能更加重要，现阶段可能不是。


后来我就这个比较顺利了，后来又进入了一些大公司，在大公司的工作中，其实也会发现自己有一些能力的欠缺，比如说当年不会画图标的时候，他们觉得我图标设计能力这图形表达能力不行，后来我图标练得不错了，他们又觉得你只会画图标，觉得你这个产品思维能力不强，我就想我下了班以后，咱们再提高，咱们怎么能够提高产品思维。当年我的这个想法就是我想自己做一个 APP，然后也确实是节衣缩食做出来一款，APP 从整个产品的思维到原型图到设计稿到切图标注，这个就差编程不会，对吧？找了我的特别好的好朋友来帮我做开发。


上线以后很多的大量的用户也使用了我这个小产品，我觉得这个产品思维能力我是不是有说服力了，对吧？因为我自己做过产品了，我觉得比如说你说什么需求的时候，我也大概知道他是一个怎么样的，对吧？所以这个也是个好的锻炼的方法，后来又有很多崭新的这种平台出现了，比如说每年发布的 iPhone 以及这些智能手表，对吧？我们现在就是这种多屏，对吧？我们生活中有很多的屏幕智能多屏，对吧？这样的东西都会需要我们去学习去了解，所以我也在不断的努力，跟上这个时代，别被抛弃了，但是确实如果让你去做一张图，跟你去玩一局游戏相比的话，当然游戏的吸引力很大了，所以其实很早之前，我是不太敢，比如说在手机上安一个王者荣耀，对吧？我觉得这些游戏太令人着迷了，对吧？不管是从设计上还是从打击感上，对吧？这一玩确实是上瘾，不是说人家不好，是做的太好了。


所以说在我需要做东西的时候，我可能会短期把一些游戏都删掉，然后再我确实觉得人生很无聊的时候再去装起来。很多朋友，其实如果有很多选择的时候就容易迷茫，对吧？比如说你同时放着电视剧，一个你的男神演的电视剧，或者是打开你喜欢的游戏，对吧？在面对 Photoshop 的时候，你就觉得没什么意思了，所以说勤有功戏无益，对吧？所以我们还是要在大家能拼的时候多拼一下，多把自己能力提高一下，因为这个人生很长，你会发现在后期确实你没有年轻的时候脑子那么快乐，没有那么能学东西了，到那个时候你再去扩展一下其他的能力，对吧？然后你也可以稍微的增加一些你的游戏时间，娱乐的时间，对吧？在年轻的时候还是要多拼一下的，我觉得会比较好。


那么现在我的生活中其实还是主线还是在作图中度过的，但是我也会偶尔玩一些游戏，看一些电影之类的，其实像看电影玩游戏其实也会对我们的眼界做出一些补充，比如说电影里有很多非常好的设计，你可以看到银翼杀手里面是怎么样塑造一个赛博朋克的这么一个设定。


游戏中很多游戏，它这种设计它 UI 的界面也是值得我们去学习的，但是不能纯粹为了玩而玩，为了娱乐而娱乐，还是我们要时刻谨记设计师是一个 24 小时的职业，不管你在干什么，请大家尽量以设计师的眼光去看待周围的方方面面，然后我们就会把所有看到东西再反哺的设计里，我们的设计就会越做越好了。


这就是我今天想跟大家分享的勤有功戏无益， 当然不是说这个完全不能「戏」，只是在勤的基础上，我们适当的「戏」用这些休息的动力，然后再去补充勤。这就是我想跟大家分享的，如果大家有什么想法的话，也可以多与我互动，好，下一期再见。
