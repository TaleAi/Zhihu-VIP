## 3.设计也有心理学，你了解么？
你好，请坐。这里是郗鉴 UI 说，我是郗鉴。


今天我们的主题非常有意思，想跟大家聊一聊，什么叫做情感化设计。情感化设计这个词，我们在日常的工作中以及我们在面试的过程中经常会听到这个词，对吧？这个词看起来很神秘，我们也会在很多的前辈的设计必看书单中找到一个名字叫做《设计心理学》的书，《设计心理学》无疑就暗示出来了我们 UI 设计师也必须要懂得一定的心理学知识了，今天就想跟大家好好的来聊一聊我们可能会必须懂得的一些心理学的小理论，首先我们为什么要去学这个东西？


我们为什么要去学心理学？


原因就是我们的用户在使用我们的产品的时候，经常会做一些我们好像看起来不可思议的操作，比如说这个功能不就放在那，你怎么看不到对吧？为什么用户老会点这个按钮而不是那个功能对吧？其实背后都有一定的逻辑在里面，都有一定的规律在里面，那就是心理学了。


我们的界面，是通过视觉的语言来表述的我们的一个信息的，那是觉得这个信息其实在用户认知的时候，会有一定的偏差以及一定的偏好，我们研究好我们的设计心理学的方面的知识，会更好的帮助我们设计界面，用户也会按照我们的产品逻辑，找到应该去注意到的功能。


心理学其实很广泛，心理学我们其实需要了解的心理学有很多，比如说色彩心理学，对吧？不同的颜色给我们不同的心理暗示，是不是？比如说我们看到了黑色白色我们觉得高端，所以很多大牌的这种厂商都是用黑跟白这种极简的颜色来设计的，再比如说红色跟暖色系的颜色会给我们一种冲动，一种热血沸腾的感觉，当然肯德基麦当劳主色调都是红色了，为什么？因为你坐那就想走，对吧？因为我们坐不住在红色场景中，因为红色本身就是血液的颜色，对吧？我们这个动物性就展示出来了，我们看到这个颜色就觉得有一些紧张暖色系的，像红色橙色也多用于电商设计中，为什么？因为提高用户的冲动消费的可能性，对吧？你看大红色专题 618 来了是吧？你赶紧想下单了。 


所以色彩心理学我们应该去了解它每个颜色对应的一种人心理的一种暗示，除了色彩心理学之外，其实还有一个心理学的种类叫做认知心理学。什么叫认知心理学呢？本身设计心理学这册丛书的作者阿诺曼博士，他就是认知心理学的专家。认知心理学主要研究人怎么样去认识一个事物并记住它的。也就是说我们怎么样让能够让用户熟悉我们的界面，然后在脑子中会大概的记得这功能应该在哪，这就属于认知心理学研究的范畴了。


认知心理学里面又有一个分支分支，可能对我们来说会更重要一些，它就是格式塔心理学，格式塔心理学主要研究的是我们人怎么样去逐层递进认知事物的一个理论，它对于我们有什么这个作用？我们如果掌握了格式塔的理论的话，我们就可以让用户按照我们安排的剧本来操作界面了，你想让他在哪个地方下单，然后他可能会主要找到这个功能，对吧？它就可能会增加它的点击的几率，这个也是蛮神奇的。但其实我举一个特别简单的例子，你可能就会明白了，比如说有的时候我们都会发现安装一些软件的时候，我们会附带安装了很多的什么工具栏之类的，其实很有意思，我本来是不想安装的，你怎么能强迫我安装，对吧？


其实人家也是很聪明的流氓软件的研发人员是吧？也运用了格式塔心理学的方法，比如说第一步问你要不要安装软件下一步对吧？同不同意我们用户守则下一步对吧？然后您是不是需要安装如下插件非常小的字是吧？你也这习惯了，就点下一步就完成了。其实电脑中无形多了很多的软件，这就是运用了格式塔心理学的效果。格式塔的心理学主要有什么理论？主要在认知方面就可以总结出我们认知事物有五个大的规律，叫做格式塔的五大律。


第一个叫做接近率，什么叫接近律？就说其实我们在视觉方面来看两个物体更加接近的时候，我们会下意识的认识到他们俩可能这关系更近，对吧？所以说我们如果是排一个界面的版的话，相关的信息他们之间的距离应该是更近一些的。对吧？这个道理听起来很简单，就是谁跟谁离这近他们就会有关系，但是真正让你操作起来的时候，大家可能会有一些失误，很多的界面就是因为相关信息不紧密是吧？造成它可能会用户会误解，比如这个人的头像跟下面的图片更近了，是不是代表他是发的是下面一张图，对吧？所以接近率很有意思，并且有的时候我们会排好几个接近率。


比如说一张图片，一个作者一个标题，一个时间，他们之间的间距完全是由他们背后的逻辑关系决定的。对吧？那么这个作者跟作者头像的关系是不是应该比作者跟时间的关系更近？作者头像跟作者与标题之间的距离是不是应该比它跟时间的关系更近？也就是说我们在排界面的时候，必须要搞清楚背后信息的逻辑关系，谁跟谁更近，我们就把谁搞到一起。也就是我们为什么说 UI 设计师接到需求以后，一定要弄明白这个需求再去做设计，因为我们要理解背后的逻辑。 


第二个非常重要的定律叫做相似率，跟刚才这个率其实比较接近接近率是吧？然后我们下面是相似率，相似率率是法律的律，相似律讲的是什么？就是说我们在认知事物的时候，我们的刺激的要素，如果它逻辑上关系，他们是一致的话，这些刺激的要素也应该一样，什么叫刺激的要素？就是我们这些信息的大小颜色与形状等等的要素都要一致，你别突然的点赞是吧？都是灰色，突然出现一个绿色，突然这个放大了，就用户会觉得很奇怪，是吧？为什么？


这个地方怎么跟别的地方不一样，对吧？所以这叫相似率，刚才是接近率，然后我们说的是相似率。相似率的作用也很重要，它会暗示用户长的一样颜色一样字体大小一样的信息，它们都是一个类别的。有些设计师觉得这个设计就是好看与不好看，这样的区别，很容易的会破坏我们相似率，让用户找不着北，比如说小标题的样式，设计师会设计 N 种——其实都是错误的，如果是逻辑上它们是一个概念的话，尽量让他们统一这种设计感。


第三个绿叫做闭合率，什么叫闭合率？就说我们人脑会自动补完一些图形，比如说一个缺口的圆，你会认为它是一个圆，而不是一个弧线，对吧？就类似于这样的道理，就很多我们看到我们认为是不完整的形状，它其实是一个本身就不完整的图形，我们在大脑中会自动给它补全了，闭合率也很有意思，有的时候我们会做一些界面的时候，可能没有办法，比如说横向排版怎么样？暗示用户我们后面还有内容，我们给他露一半露一半图，看到这一半图，他就下意识认为这个图后边可能还有一个图，对吧？他就意识到这个操作是可以去进行滑动的，就是说我们用一半的图形暗示了他，后面还有东西，也应用了我们格式塔里面的闭合率。


第四个绿很重要，叫做连续率连续率，就是刚才我举的流氓软件的例子，就说我们会把很多的操作以是或者是视觉把它们线性化，我们操作，比如说点下一步我们就下意识地培养了一种习惯，对吧？下一步同意就是下一步这样也很容易被利用，对吧？比如说突然有一部给你一个流氓软件的话，你就很容易的上当了，对吧？


第五个律叫做成员特性率。什么叫成员特性率？就是说我们在一组信息之中，他们可能是一个逻辑跟相似率是一样的，对吧？但是如果同样的逻辑下，要突出一个功能的话，我们就要使用成员特性率，来让它突出了。类似于这样格式塔五大率这样的概念在我们工作中会经常的被用到，除了格式塔的五大率之外，还有三大记忆率，同学们如果有兴趣的话，可以在网上找一些相关的格式塔心理学的资料，我这里就不多说了！


除了格式塔心理学色彩心理学之外，其实还有一种我们应该掌握的用户的一个心理的一种理论，就是我们今天的主角——情感化设计了。


什么是情感化设计，这个很有意思。其实最早的概念就是由这位认知心理学家诺曼提出来的。情感化设计主要就是让我们用设计的一些手段，让用户在使用我们产品的时候感觉到舒适，感觉到一种情感上的满足。


我相信您一定听过马斯洛理论，对吧？就是马斯洛需求的金字塔。马斯洛理论就是把我们的需求按照等级划分了这么五种，对吧？最低等的就是生理需求，生理需求之上就是安全需求。这两层就是说最基本的是金字塔的基石，我们首先要满足我们的生理需求，然后是安全需求。在往上就更高级一些了：爱和归属感，再往上是尊重，最后是自我实现。它就是一个从低层次到高层次的一个序列。其实再往上还有自我超越的需求等等的这样的层级。


就说我们人的需要是分为很多等级的，情感化设计，其实我觉得跟这个理论还有一些相似的地方，他也是我们的设计如何能够更好的去满足用户的一个需求的一个递进，比如说情感化设计，他把我们的设计水平分为了三种，第一个层次叫做本能水平的设计，本能水平就是说我们是视觉动物，对吧？所以我们对于外形的理解跟观察是出于本能的，如果我们的视觉设计符合本能水平的思维的话，就让人会接受起来会非常的快，这个非常的快，比如说我们很多的运营图的设计，对吧？情人节七夕等等这样的节日，然后它我们给予它一些图形化直观的这种触及，就会让用户感觉到情感的传达了，对吧？


第二是行为水平的设计，第二个层次。这个水平上的设计主要就是说要关注产品是否能够帮助用户有效的完成他应该完成的一些任务，同时希望这些任务完成的时候有一种有趣的体验，最高一个层次的水平叫做反思水平的设计，反思水平的设计，就是说我满足了大家的一切的需基本需求之外，同时还会让用户感觉到有一种文化身份的一种认同感。然后它会让用户对我们的品牌产生忠诚度，反思水平就是这个非常难达到的，对吧？


我们可以看到，其实很多公司也会发给员工 T 恤衫，对吧？但是你愿意穿吗？那就是反思水平，如果我们想让你的用户也好，以你的品牌为荣的时候，我们需要付出的努力就很大了，对吧？举几个例子可能会帮大家理解的更快，本能水平的设计，比如说我们的一些运营产品，对吧？运营图与一些这种广告图都属于本能水平的设计，什么是行为水平的例子？比如说微信的摇一摇，对吧？摇摇它摇摇，它首先是一个挺有趣的操作，对吧？然后它也会有效地帮我们完成很多任务，因为微信摇摇不只是可以摇人，对吧？它还可以比如说电视节目的时候同步，它可以帮你找到一些相关的信息。


反思水平的这个例子，我想举的是 google doodling，google doodling 可能是大家都很熟悉的，就是说每一个节日，比如说马克吐温的生日，比如说气息之类的，他都会把 google 本身的 logo 变成一个图形与设计结合的这么一种 logo，我们就会感觉到这个是今天很有意思，是吧？今天是马克吐温的生日，再点击这动画的时候，它就会跳出一个专题了。还有比如说韩国 Naver 公司旗下的 Line 产品，Line 本身就是像微信一样，它是一个即时通讯的工具，但是它衍生出来了一个卡通品牌叫 line friends。 


Line friends 其实在功能上来说，其实他就是一个花花绿绿的小人，对吧？就是一些贴纸什么的，但是不要小看这种人对于美好事物的向往，我们现在周围年轻的年轻人都非常喜欢布朗熊，还有小兔子，是吧？还有他们俩的小黄鸭，就很像周冬雨的，对吧？还有一个暴脾气的老板，一个小王子等等的组合的形象，这一套形象其实对应了我们日常生活中的方方面面，对吧？比如说你谈恋爱的时候，那就是小熊跟小兔子的关系，老板非常的损，对吧？非常让你生气，老让你加班，他有这样的形象对应，也就是说我们在现实生活中的很多情绪其实都可以通过来粉丝来投射，那么这个就到达了一个我们说反思水平的设计，就是我们在这个设计中找到了认同感跟身份，对吧？是跟身份的感觉，我们同时还会看到很多人会穿一些比如说腾讯的 T 恤，对吧？它这个会感到很光荣，为什么腾讯要那么强调企鹅的形象，QQ family，对吧？也就是反思水平设计的力量了，对吧？所以我们在设计一个产品的时候，我们不光要考虑到基本能让用户找到一些他需要的功能，比如说我们会把一些需要重要的功能突出，同时我们也要考虑能不能带给他一些反思水平设计层面上的东西，比如说我们是不是应该为我们的产品提供一个卡通形象。


在用户比如说到 404 的页面、到空白消息的页面的时候，使用形象来强化用户对于品牌的一个感知，在一些特殊的节日我们是不是可以设计一些闪屏，这都是一些反思水平设计层面上的东西。


有的同学可能会说，我们公司没有这样的需求怎么办？我想告诉大家的是，如果大家是设计师，而不是一个人肉 ps 机的话，我们在做设计的过程中其实需要更加的主动。我们有的时候我们向我们提需求的产品经理也好，大 boss 也好，他们可能没有了解到比如说像情感化设计这方面的知识，那么我们正好可以借此机会给予普及，并且我们可以提出我们是不是也可以在我们产品中加入这样情感化设计的一些思维，但是情感化设计在是我们 UI 设计的视觉表达里容易落地成为比如说插图，比如说一些有意思的 icon 这样的东西，那么就需要我们设计师除了可以摆非常工整的界面之外，你也需要具有一定的图形化的能力，包括我们的插画也好，我们图标也好，这样的能力，如果你只会摆介面的话，你只能是一个本能水平的设计，对吧？行为水平的设计无法到达反思水平了。


这个是非常有意思的一个小知识，也希望对大家有一些启迪。
