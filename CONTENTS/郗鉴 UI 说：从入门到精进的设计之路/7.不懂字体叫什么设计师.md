## 7.不懂字体叫什么设计师
你好，请坐。这是郗鉴 UI 说，我是郗鉴。


作为设计师，不管我们从事的是平面设计也好，还是互联网设计也罢，我们都要对字体设计有一定的了解。为什么？因为字体它就是文字，而文字是视觉传达所有元素中最直白的。如果你看不懂一幅海报没关系，我底下会写着是什么的展览，对吧？


一个 APP 图标你不理解的话没关系，我底下写一行字告诉你这是什么，所以文字是传达信息最准确的一个元素，而字体它可以让整个文字呈现出一种这一种审美，一种我们希望它有的意境，所以了解字体对我们表达我们想表达的事情非常的重要。


人类其实有文字的历史并不长，只有几千年，文字其实是一个文明进化到一定程度的一个产物，我们当今这个世界文字如果按照时间的主线来分的话，应该还是能分为东西方世界两条线索，这个字体的历史就藏在里面了。


咱们先去看看西方世界，大家应该能猜到了，当然就是欧洲的历史了，欧洲从希腊雅典的时候就诞生了很多的现代文明的影子了，对吧？比如说法律、共和制讨论这些事情，还有会议这样的事情，而且字体也出现了。


最早的字体，我们从罗马体讲起，罗马体又这又叫衬线体，这字体是后来的人去考古去发现罗马时代的一些碑文的时候，发现上面刻着非常优美的字母，这个字母非常的漂亮，然后它会有一些他因为楔形文字在上面，它会有些工具的产生，是吧？


产生了这样的工具的脚，就产生了我们所谓现在的最早的衬线。衬线就是一个字母装饰的线，比如说 A 的上边跟 A 的下边它都会有一定的装饰的线条，如果有的话代表它就是衬线体了，所以最早的衬线体就是罗马体了。Roman 如果大家在大学中写论文的话，应该你听过这么一款字体叫 Times New Roman，但是这个字体跟罗马体还是有一定的不同的，总之以罗马体代表的这种很有历史的字体成为了当时能够在欧洲随处可见，包括报纸，包括很多的设计招贴上都能看到的字体。


后来有了书写的工具得了进化，欧洲人他也不用石头跟强了，对吧？我从中国学到了造纸术，对吧？那么就有了鹅毛笔了，对吧？鹅毛笔书写的时候，它由于这个工具它比较的细是吧，然后比较尖，所以它一定会侧过来鹅毛笔来写字，对吧？所以写的这个字就非常的纤细，非常的漂亮，那就是我们后来的这种叫做什么？叫做手写体。咱们在电视剧里老看到莎士比亚，对吧？或者是亚里士多德，他们很多欧洲的圣人们，他们写字的时候都是一根鹅毛笔在上面写，对吧？所以他们写这个字体都是 script 手写体。 


再后来有了更先进的工具了，字体也跟着一步一步的演变，总之在衬线体统治世界的时期，基本上都是由于工具产生的一些衬线所造成的衬线字体，衬线字体非常的好看，然后后来也有专门为了制造好看的衬线而产生的字体，比如说 Old English。大家可能好像没怎么听过这字体，如果打出来一个 J 你就认识了，或者你打出一个周杰伦的英文，你就认识了。为什么？因为周杰伦、林俊杰，之前他们的专辑的封面就用到了这种 Old English 的字体。怎么说，有些人也会认为有点杀马特，因为这个字体看着是吧，就感觉像皇家的字体一样，对吧？后来呢这个字体慢慢就发展开来了，有专门从事活字印刷字体设计的公司了。


因为在国外公司成立的很早，对吧？有专门成立专门制造字体的这种字体公司，也有字体设计师出现了，不是我们像今天在站酷上看到的那些很厉害的国内的设计师，其实在很早之前就有这样的厉害的设计师了，对吧？他们有粉丝，他们也有站酷的「派对」，当然是线下的了，对吧？在欧洲的那个时期 18 世纪，然后也有一些线下的 party，然后大家聚集过来，然后来一起讨论。


这里面有很多的小小的故事，这个没办法一一的分享，大家可以找找相关的资料，让我印象比较深的一款字体衬线体的话，我还是要提到一款字体，这款字体叫 Garamond。这款字体，你要看到它衬线的感觉，如果大家家里有信教的亲戚，你就会知道了，这是圣经上印的字体。


Garamond 是一个新教的一个传教士，然后他为了这个也是用活字印刷技术印刷圣经，然后他想创作一款字体。后来被天主教的教皇的人发现，然后就把它给烧死了，然后在它烧死刑场的时候，他的徒弟默默地也在那拿着他的书稿在那哭泣，后来他徒弟接着他的遗志完成了这款字体。后来很多的圣经的版本中也用到了这字体。很多字体其实它背后都有非常有意思的小故事。


有的时候我们的朋友打开 Photoshop 以后看到字体感觉就是一片懵，就是感觉他们好像就是一个盒子里的不同的工具一样。其实不是的，很多字体的年纪比我们大多了，它们只是以电子的形式来到了我们身边，其实他们在很早以前都是活字印刷的签字。因为我妈妈的缘故，其实我可能更了解印刷一点，为什么，因为我妈妈就在印刷厂工作，我从小就在活字印刷签字堆里玩，不过大家不要担心千不是不是有毒的铅，有很多金属可以代替。


当时会有一个一个大叔，然后他是排版工。排版工的职责非常有意思，当时我还能依稀记得当时是他会有一个稿子，比如说这个叫「我爱站酷」，当然一本书的内容可能是几万字几十万字了，但就比如说「我爱站酷」四个字，他怎么排？他得倒着排，然后他得从一个盒子里专门找标点符号，找到句号、逗号、不同的符号排上。


我记得还有的有一个很有意思的事，他会找到一个非常细的，但是上面什么都没有字的一个签字来排版，这就是空格。中文的活字印刷跟英文的活字印刷还不一样，为什么？因为英文有的时候我们得补好多的空格，我们中文活字印刷其实相对简单，为什么？因为我们都是方块字对我们字间距不用管，我们字间距就这么大，对吧？所以其实要排个英文的话比排中文要难。


由于我当时我是在内蒙古长大的，所以其实我也见过蒙文的排版，所以都非常有意思，然后小小的铅字也不要小看它，确实在很早之前是人工做出来的，是吧？老师傅做出来的，非常的有情怀，所以当时在欧洲的那些字体设计师们也是这样，就凭着情怀然后来做设计的。当时很多设计师都是必须会字体设计，然后你才能成为一名平面设计师。不可能说我就做图形，不可能，对吧？我们要从学徒做起，然后要做很多年的字体的学习，然后做平面的一些学习，才能成为一个真正的平面设计师。所以像当时的比如说一个叫 Gill 的英国人，它设计了 Gill sans，如果你们感兴趣的话可以搜索一下。


这个 Gill 就是一名多才多艺的设计师，他又会摄影，又会画画，又会写诗歌，对吧？长得也帅，然后也会做字体，所以当时的欧洲的设计师，是吧，都是这个调调。到 18 世纪末 19 世纪初，这个事儿就有点变化了。如果你听过我之前第二集的时候，我讲过设计史的话，大家应该有印象了，万国博览会工业革命来了，工业革命来了带来的是各个行业的颠覆，这可就跟信息革命一样，所以包括字体设计也经历了一些变革。其实这个问题的核心还是当包豪斯这样的学院，做出了一些适合工业生生产的现代主义设计风格的时候，大家就发现了衬线体不能满足我们的审美了，衬线体雕龙画凤这感觉好像不是很现代，对吧？


现在的产物都机器铲了，都是这种方块，这种圆很标准的这种形状几何体，对吧？不信大家看看现在的宜家无印良品，对吧？都是极简风格，这些字体太花哨了。大家也在期盼着有没有这样一款，这叫什么？这叫没有衬线的字体，所以我们起名叫无衬线字体。这种无衬线的字体能不能出现几款？在德国，因为德国当时比较先进，德国就有设计师完成了很多的无衬线字体，然后但是最流行的无衬线字体是来自瑞士的。在瑞士有一家设计公司，我们现在都知道瑞士的平面设计很发达，它为什么发达？因为在当年它就已经有了一个非常专业的设计公司了，这公司的设计能远销整个欧洲。


这公司有几代人一直在完善一套字体，这套字体的名字起名叫做瑞士的，但是这个是用瑞士语说的这个字体名字读出来叫 Helvetica，大家已经了解了，说 Helvetica 怎么说，它简直就是字体界的披头士，他是个摇滚巨星。为什么这么说？因为在 Helvetica 刚出来的时候，好多人觉得很丑很丑陋，甚至很多设计师说这不叫设计，你这没有设计，你看你一点装饰线都没有，对吧？你就是横平竖直的这种感觉，这叫什么设计，对吧？但是它非常符合当时生产的环境，而且它这个字的间距，它的正负空间都设计得非常好，Helvetica 一下子就火爆了起来。 


怎么火的呢？比如说很多的厂家，当时全世界很多的厂家都直接用 Helvetica 做 logo，因为我们知道 logo 现在的老板应该是这样子的，就是我们要做个 logo，肯定要设计师鼓捣几天对吧？你不能直接用个字体给我打字，对吧？但是 Helvetica 就能比如说谁，比如说日本的公司，咱们应该听过无印良品 MUJI 这四个字母就是 Helvetica 直接打出来的。3M、Jeep、 三星是经过 Helvetica 设计的，所以说这个世界简直迅速就被 Helvetica 占领了。当时很多人说如果你不知道用什么，就用 Helvetica。


有人说这 Helvetica 就像肯德基一样，对吧？吃完就走是吧？非常的快。这不像很多字体一样，就是吃的很慢，对吧？所以 Helvetica 就一下子就点燃了世界。Helvetica 完成以后得到了很多的定单，但是 Helvetica 其实还是觉得有一些不完美，因为字体发展到那么一个极致的情况。


一个字体会有很多的字组，它的字组就不太满足整个的需要了，于是这家公司又研发了新 Helvetica，如果大家在网络上找的话，你应该能找到叫 Helvetica Neue，Neue 是 new 的意思，Helvetica Neue 的字组非常之多，大概有一百多种，所以 Helvetica 就成了一个工业革命之后一个设计史上不容忽视的名字了，它跟我们说的包豪斯一样，成为了改变过世界的一个设计的元素，包括我们非常敬爱的乔帮主，乔布斯当年在大学的时候学了字体设计，他学这课的时候就认识了 Helvetica，于是他就买了版权，然后在他生产的第一代第二代苹果电脑中就预计了 Helvetica 字体，直到前段时间，苹果才正式的停用了 Helvetica ，研发了自己的无衬线字体。


所以这么多年 Helvetica 简直是横行世界，对吧？以至于比尔盖茨说乔布斯做了，我也要做，比尔盖茨在我心中多少有一些是吧，抄袭的感觉。但是 Helvetica  当然是有版权的了。你要是直接用的话要告诉你的，所以这手下人就说了，盖茨老板这个不能用，盖茨说是吗？这个要多少钱？人家一说盖茨一犹豫。太贵了，这字体还要这么多钱。找人做也做一个对吧？于是微软下了一个订单，照着 Helvetica ，又做了一款字体，这款字体大家一定都熟了，叫什么？叫 Arial，好读。


他就是觉得 Helvetica 太难，Arial 非英语国家的人都能读出来吧？Arial 字体的字间距以及正负的空间以及笔画跟 Helvetica 差别非常的小，简直就可以判断抄袭了。所以在当时的字体圈子，只要说我要用 Arial 做设计，你要被笑死的对吧？所以现在也是现在我们打论文的时候可能会用到 Arial 字体，但是同学们做设计的时候，比如说我们要做一个严肃的设计，然后需要一些气氛，你要是用 Arial 的话还是会被内行所取笑的。自从 Helvetica 火爆了以后，也诞生了出了很多的无衬线体，比如说 DIN，都得到了市场的好评，为什么？因为整个世界变成了现代主义设计风格了。


衬线体 serif 跟无衬线体 sans-serif，它们俩其实还是相爱相杀的。衬线体的设计师们难道就放弃了吗？也不是。衬线体的设计师也设计出了很多适合现代的叫现代衬线体，比如说像 Bodoni，你就知道了非常的大牌，这就是现在这种服装大品牌常用的字体，以及 Didot，一用你就感觉知道是谁了，就是长得非常像时尚杂志的标题，时尚杂志标题常用字体 Didot 一看就很大牌，对吧？所以说这就是怎么说，你方唱罢我登场，衬线体跟无衬线字体，在欧美国家就非常的火爆了。


我们转眼回头我们来看看亚洲，亚洲的核心当然应该是我们中国了，在很早的时候我们的祖先就有了文字了。最早的文字当然也不是写纸上了，他们是砌石头上，写龟壳上，我们占卜用。其实文字很早之前跟宗教是扯不开关系的，包括刚才说的像圣经这样的事情，对吧？所以我们叫什么？叫甲骨文，甲骨文对吧？从甲骨文往后我们中国的字体就非常的写意了，为什么？因为我们中国的书写工具其实改变得不多，从有了纸以后我们都是毛笔直，以前也是毛笔，对吧？直以前也是毛笔吗？毛笔沾墨水来写字，其实它工具的情况下，它的这个字体相对来说比较的有意思了，可以研究一下它这个字体，由于它比是软的，它可以写出各种的自体来，比如说我们常常见的常听说的这种书法体，对吧？书法题里有什么草书是吧？然后有各式各样的书法，还有什么？还有楷书、隶书等等的字体，大家应该都大概听过，除了这种大的分类的话，还有很多的书法家也自成一派，比如说颜真卿颜体等等这样的字体，对吧？


包括后来我们中国活字印刷也有很多的字体被保留下来，对吧？比如说我们最常见的宋体，对吧？宋体然后传播到了日本，因为宋体其实这个不是我们想当然跟宋代是有关系的，所以在日本其实它被叫做明朝体，这有一定的误读，所以日本明朝体就是我们中国的宋体，当然我们当时是文化出国，所以中国周边的国家基本使用的字体跟文字都是我们的中国字，这是非常骄傲的了。但是由于咱们这个国家这个在封建的时代，没有这种资本主义的这种这个公司没有字体设计公司，再加上我们的审美倾向于书画同体，就说一方面我们是要尊重之前的字体，对吧？另一方面书画同体，它跟画画相等了，所以说字体的发展，所以说它发展并不如西方字体发展的快。


在工业革命然后战争之后，我们的中文字体其实也进行了一些改革，其实主要就是在学习一些西方的字体，然后到了现在我们的中国字体正在慢慢的腾飞。我们都知道中国字体有很多的字体，公司非常的有名，到此其实我们就可以大概的来总结一下了，字体知识一共我们就是说是一共有两块，一块西文字体，一块是我们中文字体，西文字体分为衬线跟无衬线，serif 跟 sans-serif，中文字体大多是书法字体，对吧？


然后由我们后面来根据西方这种无衬线体来的这种黑体的体系，以及我们书法字体的体系，我们现在 UI 的时代就是互联网的时代，我们也知道了一些为互联网而生的字体，在苹果字体里就有 SFUI 叫 San Francisco UI。因为苹果总部在旧金山，所以 SFUI 生来就是为了 UI 而生的，它字体在苹果的最小设备也是 iWatch 上看着也是非常清晰的。
