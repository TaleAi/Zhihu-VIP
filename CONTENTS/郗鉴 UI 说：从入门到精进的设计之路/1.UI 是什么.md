## 1.UI 是什么
你好，请坐，这里是郗鉴 UI 说。这是我们第一期节目，我叫郗鉴，认识您很高兴。


我曾经就职在腾讯人网新浪网等互联网公司，我也曾经设计过一些用户量很大的互联网产品，以后在这个时段，我会跟大家每一期都交流一些 UI 设计相关的话题，希望你能够喜欢。我曾经碰见过很多想转 UI 设计师的朋友，他们对于 UI 设计这个行业还有很多的困惑，不知道从哪说起。我也碰到过一些已经转行成功的朋友，但也有一些职业上的困惑。很多我的朋友也是资深设计师，他们对于未来行业的发展还有一些担忧和迷惑。


这个节目主要就是想跟大家来聊一聊，来好好的探讨一下 UI 设计师的过去现在和将来。没期时间不会太长，刚好是 20 分钟左右，刚好您可以听着我的节目从公司走回家，或者吃个饭，听着我的节目下一下饭，或者在您睡不着的时候，失眠的时候可以听着我的节目，刚好可以助眠，当然我希望我不这么的催眠。


今天我们就先来聊一个非常小的话题，就是 UI 设计师的现状。那么什么是 UI 设计师在我这个工作刚开始的时候，每年回家的时候我存在一个很大的困惑，就是怎么跟我的周围的朋友们来讲我在干什么？


什么是 SNS，什么是 UGC， 什么是 UI？很多同学们都觉得，你这怎么老说英语，对吧？现在呢，每年在回去的时候同学就问我，你们公司搞不搞这个区块链，对吧？你这个对吧？你这个知识付费弄得怎么样了？对吧？所以说大家对于整个行业的关注度越来越高，互联网成为了一个我们生活中不可缺少的部分了。


但是作为 UI 设计师来说，有的时候也会比较困惑，有的时候面试官会问我，你觉得什么是 UI 设计？你觉得具备什么样能力的人才能叫做 UI 设计师？这时候我们就会稍微有点懵，对吧？我要怎么回答？


仿佛平时我们大概脑子里知道我们在干什么，不就是做一些界面，但是让我们总结成一个书面语的话，往往我们大家会这个不知道该怎么从哪说起，现在我们来聊聊什么是 UI 设计，首先 UI 这两个字母代表的是 user interface ，就是用户图形界面。User interface 已经发展了很久，从很早的时候就有图形用户界面系统了，在乔布斯第一代电脑就有了。


发展到今天其实这个模式没有变，图形用户界面系统主要就是使用一些我们日常生活中了解的一些这种概念也好、具象的图形也好来代替一个复杂的电脑原理，比如说我们在电脑里知道回收站，生活中回收站就是扔垃圾的地方，所以我们在电脑中也会把不要的东西扔到回收站清空，这个操作看似很简单，其实电脑理解的并不那么简单，电脑会把这些文件全部选中，然后隐藏掉，等新的磁盘满的时候再复写过去！


但是我们需要知道这些吗？不需要对吧？作为一个普通的电脑用户来说，Ta 可能是一个职场上的这种白领，Ta 也可能是一个家庭主妇，Ta 没必要知道背后电脑操作的一个逻辑，Ta 只需要知道我这东西我不要了，我想把它删掉，OK 这个概念就完美的符合了 Ta 这个要表达的含义了。这个过程叫做隐喻，隐藏的隐，比喻的喻隐喻的这个过程，也就是我们 UI 设计师需要思考的过程。


其实 UI 设计也是这个人跟计算机沟通的一个翻译官，我们是一个人机交互的一个桥梁。人机交互在历史中发展过很多阶段，比如说第一个阶段，打孔纸时代，好比说我们跟计算机我们作为两个种族来说，我们第一步是要写情书，所以说最早的时候人跟计算机交流不是通过键盘，也不是通过鼠标，是通过打孔纸。这种纸是有一个眼，代表个是百千万的位数，把打空指输入电脑以后，电脑会进行一个运算，这过程就是人机交互的这种初级阶段了。我们在卷福演的电影《模仿游戏》里面就看到过图灵计算机，它这个是怎么样跟人类沟通的。


后来有的键盘鼠标我们跟计算机沟通是通过代码，当然这个过程现在也没有改变，对吧？我们程序员还是要去写代码的。那代码就是这个计算机相对来说更容易理解了，对吧？但是这是有一个学习成本的，就是说你不可能要求我们的父母辈去学代码，那是很恐怖的。在很早之前尝试过让我们所有人去学到 Dos，大家肯定在上学的时候学过一点点的 Dos 或者是 CC sharp 之类的语言，但是可能一毕业都忘光了，代码就存在一个学习成本很高，你要想让普通用户去学习的话，是这个真的是有点开玩笑了。


后来有了两位跨时代的人物，乔布斯跟比尔盖茨，他们推动了个人电脑的发展，但是他们都选择了图形用户界面系统作为人机交互的第三个阶段，就是说我们用鼠标来模拟，用光标来模拟我们手中的鼠标，然后有不同的图标，代表不同的功能以及不同的隐喻，这就是图形用户系统的诞生。我们的职责就是人机交互的一个桥梁。


人机交互未来会怎么样？对吧？你不是说了三个阶段，它还会有第四个阶段吗，第四个阶段 UI 设计师会失业吗？我不知道，第四个阶段可能应该我猜是语言了，因为我们已经度过了这么多人机交互的一个渠道，当然更直接的应该是直接说话，对吧？咱们就是拿手指头猜对吧？拿纸条写都过去了，应该就该咱们直接聊天了，所以 Siri 所以 google assistant 等就会迸发出来，就是人机交互进入了语音时代，但是我认为现在技术并不成熟，很多时候我们会觉得 Siri 很蠢，所以语音时代可能还要再远一些。 


对吧？满大街的这种 VR 设备，还有 AR 设备，这些可以作为人机交互的平台吗？当然可以了，在很遥远的未来，我相信我们会实现电影里的那样的科幻场景，我们一回家一位虚拟的、漂亮的、你喜欢明星样子的这种虚拟的人物出现：「欢迎回来，您今天辛苦啦！」但是我认为那也是一个非常遥远的未来了，所以目前人机交互的主流仍然是图形用户界面系统，也就是 UI 设计师所专注的领域。


我们做 UI 设计师的时候还要注意一些事情，就是大家可能耳熟能详的词汇了，比如说什么叫做 UED， 就是 User Experience Design？叫用户体验设计，这是我们会帮助用户建立一个他体验上非常愉快的一个产品。同样还有一个词汇，我们老听，叫 UCD—— User-center Design，以应用户为中心的设计，对吧？


这两个词汇现在已经非常的常见了，就是说我们在做设计的时候一定强调是以用户为中心，但是怎么以用户为中心，我们后面的我们的节目还会为大家讲到互联网团队是怎么样通过一些用户研究的方式来调查用户怎么样能够用的更爽，产品用的不迷惑，用的更爽，用得更简单。咱们的后期的节目都会讲到 UI 设计，如果你并不想从事这个行业，我认为这个节目您也应该听一听，因为我会带来一些互联网行业才有的一些这种新的趋势。 


比如说我们在后面会讲什么是区块链，什么是现在说的这种赋能是吧？什么是这种，我们说互联网金融，都会给大家来讲这部分的知识，帮助大家来判断自己所在的公司从事的行业是否有前途。另外如果您确实对 UI 设计比较感兴趣，以及对设计都感兴趣，可能您现在并不是设计专业本身出身的人。那么在我们后面的节目里，我也会为大家来介绍一些设计的基础知识，比如说我会为大家来介绍整个设计使得一个简略的过程，设计有哪些风格，以及每些风格有哪些特点，都会在后面的节目为大家来慢慢的介绍。


同时如果大家对这种比如说字体，比如说我们这种设计的板式也有困惑的话，后面节目同样会为大家来慢慢来介绍。所以说值得您期待。讲到这里我又突然想到了曾经有一些朋友在担心 UI 这个行业还能做多久？那么这个我正好在我们第一期节目里跟大家来探讨一下。我认为 UI 行业能做多久？这个问题可以从两个方面说起，一个是这个行业还能存在多久，另一个是我们还能做多久。


第一个这个行业还能存在多久这个问题，如果你真的在担心，那么真的是有点杞人忧天了，因为互联网行业作为我们这种信息革命后的这种对吧大潮之中的，所以你应该能看到这个趋势，互联网行业是短时间内是不会消亡的，而且它会随着自己的新的技术而不断更新，也就是说以这种信息技术为基层的这种技术会不断的更新，这个技术与人之间的鸿沟，就需要我们设计师来做一个补充来协调作为人机交互的一个平台，所以说我们的行业的未来完全不用大家操心。


虽然我们会看到很多设计师会觉得现在找不到 UI 的这个工作，对吧？但是我们也要把眼光放远，信息产业已经是一个非常强壮的一个产业了，它并不是一个什么朝阳产业，它已经非常如日中天了，对吧？那么我们能做多久？这个问题其实我们还要仔细的去思量一下，首先有一些朋友她觉得我是女生，我可能是不是应该在我比如 30 岁以后把我的重心调整在家庭之中，我是不是在公司的日子会越来越不好过！同样我有很多的例子，但这里不方便讲，人家的名字都是这个已经成了家，并且他是 30 岁以上的女性 UI 设计师做的都非常的棒，在行业在公司里在团队里简直就是一个领头羊的作用，所以说同样女生不用担心。


那么男生也是，同样的有一份调查数据显示，UI 设计师现在的比例男女比例是 45% 跟 55%，也就是说这个行业里也几乎就是一半对一半，一半男生一半女生，所以这个对于我们来说男女性别其实并不存在一个很严重的差异，大家主要关心的可能是年龄了，对吧？比如说我已经 30 了，或者我 30 以上，好像我们在这种周围的朋友之中以及网络上没有看到过，好像是这个有年纪非常大的 UI 设计师。 


其实这里也存在一个误会，因为首先我们这个行业从互联网发展在中国发的发展到现在，其实时间并没有特别长，所以我们也算一个新兴的行业，对吧？没有一个非常有前辈的这种感觉的这种氛围是正常的，但是我相信随着时间的推移，我们会出现很多很厉害的 UI 设计师的，就是你们幻想的白发苍苍的这种前辈，对吧？他们也会执着于 UI 设计之中，让我们有一个更好的榜样。现在由于第一代的 UI 设计师其实也是年富力强的，并没有大家能够想象到像日本的原研哉这样的就是白发苍苍这样的设计师，对吧？ 


同时就是说大家也要放松心态，30 岁并不是一个很可怕的事情，对吧？40 岁也不是很可怕的事情，只要你还处在一线，你还在接触更新的东西，更新的技术来，然后让你不断的进化，你是没有一个担忧的，往往担忧的都是自己专业还不够强的人，自己专业不够强，老在担心会不会被淘汰，所以如果你有这样的忧虑，不如把时间交付给你的努力，你每天去锻炼自己的技术，去学习新的东西，新的表达方式，这样的话你就不会有这样的担忧了，以后的未来我们会看到很多这样的前辈，然后也会看到他们不断在努力，现在已经有这样的前辈了对吧？以后可能他们就白发苍苍了，我们就更有一个榜样的作用了对吧？所以现在不要迷茫，因为我们这行业还是很新的。


还有就是说有些朋友会觉得是不是我应该过了一个年龄以后把我们注意力转到家庭之中，当然家庭是我们生活中的一部分，我们不可以忽视，但是我要强调一个叫所谓职人精神，就是我们在职场之中我们就要做好我们自己应该做的事情，所以说设计跟生活也不完全冲突。如果你现在还有一个比如说我 30 岁以后我就要去回家开一个小咖啡馆，这样的梦想的话，我建议你可以打消了，把我们专业做好，它是可以让我们做到老的。好，第一期节目我们就到这里，在后面的节目中我再来预告一下，我们会为大家来讲整个设计史的一个演变过程，以及什么是情感化设计。
