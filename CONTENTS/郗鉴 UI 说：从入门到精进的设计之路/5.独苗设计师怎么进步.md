## 5.独苗设计师怎么进步
你好，请坐，这里郗鉴 UI 说。


我是郗鉴，很多朋友经常会问我，这个公司里只有自己一个设计师怎么进步？有的时候找工作的时候，我们也会找到一些就是说只有一个设计职位的工作，我们该不该接受？很多朋友，包括我之前也是抱着一个想法，就说我去一家公司，我的目的是学习。其实我们想一想，这逻辑上是有问题的，对吧？因为我们其实并不是选择的一所学校，我们是选择了一个工作的单位，人家每个月买下我们的时间，让我们来输出一些设计一些他们需要的产品，本来就没有包括让大家成长的义务，我们是不是其实也不太应该去要求说这个公司必须给我一个成长的环境。


这个输出就是指我们为公司提供，他们希望我们提供的这样的设计的作品，包括设计的理念也好，包括我们人员的参与也好，这是这个是等价交换。提高，其实指的是输入，我们说一个是输出，一个是输入，input， output。 输入是指我们自身提高自己的能力，提高完自己的设计能力以后，我们可以更好的为公司服务，我们输出更好的设计的作品，包括我们随着产品思维的增强，我们也可以帮助我们的产品团队来构思一下这个产品未来的规划之类的，所以说这个输入很重要，但是公司并不能帮助你完成这个环节，公司能够提供给我们的附加价值，除了工资之外就是经验了，现在很多的招聘都会强调经验的重要性，但是我们的能力并不是仅仅由经验构成的，这样的话进步会很慢。


其实我们应该去多寻找一些资源，在我刚毕业的时候资源很少学习的资源，哪种插画风格的资源几乎都是需要自己去摸索的，而现在这个时代最方便的就是资源唾手可得，只要你在各种网站上一搜是吧，大量的 UI 类、插画类，这些教程都有，有中国的有外国的。所以现在学习变得空前容易了，我们是独苗技师这个命题，其实也是可以破解的，只要我们保证自己定期的输入就好了。


输入其实不仅仅是学个教程对吧？我们可能还需要有产品思维的能力，我们有图形表达的能力，界面排版的能力等等诸多的能力，这些能力除了在网络上寻找一定的资源之外，我们还能怎么获得？当然了看书对不对？看书这个书是个好东西，对吧？好多人说设计师不看书，我我觉得是对我们的偏见，这个书一定要看起来，而且我们要看好书。作为 UI 设计师来说，其实相关的书其实并不是很多，相较于平面设计室内设计的画册，我们这个行业的书籍可能并不是很多。 


除了那些 UI 正相关的书籍，就说它本身就是 UI 类的书目，我们可以去看，同时交互类的书籍我们也可以去了解一下，比如说设计心理学，比如说栅格系统之类的书，同时我认为我们这个行业也要了解一下设计的历史在历史上有什么样的风格，我们其实可以去借鉴和参考。


咱们现在很流行的赛博朋克，还有蒸汽朋克之类的，其实都是在历史上出现过的一些风格，然后重新进入了我们的视线，所以说了解历史也很重要，像中外设计史、包豪斯了其实都是可以去了解的。除了书籍能够补充我们的知识之外，然后互联网还要提供了大量的资源，除此之外其实我们还可以去参加一些比如说站酷之类的活动，线下的活动认识同行其实对我们自己也是有帮助的，如果你的好友列表里都是你的初中同学小学同学，那么你真的是需要扩展一下业内的人脉了，但是人脉当然不是第一位的了，我们能力是第一位的。人脉当然也很重要，但是不要把它放在一个非常高的位置，因为你认识谁其实并不重要，你是谁才重要对吧，我们当然平时去认识结交一些设计圈的朋友也是很好的。


然后我们还能怎么提高，当然就是一个最重要的环节了，就是靠自己的练习，这个练习并不能拿上班的东西来练手，对吧？如果你是理发师的话，你不能拿真人来练，对吧？我们肯定是拿个西瓜来练是吧？所以作为设计师也是一样的，我们需要花很多的时间投入到练习之中，这练习可能产出的不怎么样，产出的可能是个抄袭品，对吧？因为我们要学习人家对吧？这个东西绝对不能直接商用，你需要有这个过程去学习风格也好技法也好，这个时候我建议的是我们不要使用工作的项目来练，我们怎么来练？


当然就是下班以后回家了，如果一位设计师在他的职业生涯中除了工作之外，不怎么打开 PS 这位设计师可能会被淘汰，说的一点都不过分，如果你不保持努力向上爬的话，你就再向后退，因为我们这个行业年轻人很多，努力的人也很多，想上向向上爬的人更多，所以你只要不进步你就会倒退，一定要自己做练习，不管是多大岁数的 UI 设计师。很多设计师随着年龄的增长，不是太愿意去投入具体的项目练习了，就不会说是下班以后做个网站做一个图标之类的，更愿意去学习一些理论知识。这本身是没错的！如果我们的手上的能力已经到达了一个级别，这级别当然应该是很高的级别，然后我们去学习一些思维方面的东西是没错的，但是很多人是年龄到了这个级别儿，手上的功夫并没有到这个级别，然后就跟着大六一起去学学什么产品思维之类的，我认为那也是错误的。


如果我们 UI 设计师需要有产品思维的话，首先要保证他的手上功夫一定要拿得出手。如果我们手上功夫还做的跟新人一样，那你就不要去学产品思维，因为你不学产品思维，你不会失业的，但是你的手上的功夫你画的图形化的东西很差的话，那就很抱歉了，即使你有产品思维，你也比不了产品经理对吧？你凭何来维持你的生计？


所以说这个是我们的看家的本事，就是我们一定要做练习，这个练习是知识补充代替不了的，这是最重要的，而且是在下班之后，这个是我很强调的，除此除了这些之外，我们还能怎么提升，其实我挺建议大家没事的时候尝试在输出中找输入，这个就是我们可以在一些工作环境与非工作环境中尝试去沉淀一下自己，沉淀自己，就包括我们去给大家去讲我们设计理念。


比如说我们的公司中做一个 APP 在设计的过程中，我们除了产品经理让我们做的事情以外，我们还想了很多情感化设计的东西，记得不记得上节课讲过，然后我们还加入了一些插图，插图的风格非常的流行，有可能是拿 iPad 的玩成的，对吧？所以这些东西我们给它沉淀一下，别让它随着版本的迭代就被大家遗忘了，我们沉淀一下，然后可以在公司进行一个小小的演讲，小小的分享，来跟大家分享一下在设计中我是怎么完成我们这个公司的产品的，参与的人如果没有设计师的话，也可以是跟你打过交道的产品经理、程序员、以及老板之类的，这个环节也会增强你在团队中的话语权。


他看到你的时候这个潜意识告诉我，这是一位厉害的设计师，咱们要从头到尾埋头干是吧？然后这个让我干嘛我就干嘛，你说放哪就放哪，那就是小美工对吧？所以我们一定要积极主动的做一些事情，如果你发现你的工作中老是被动的话，你也是同样挺危险的了，在除了工作的环境之中也有很多类似的小型分享，大家也可以参与，然后也可以给你周围的朋友也好给同道中人去讲解一下你在你工作中的收获，也可以听听他们在工作中的收获，这样对我们的发展也非常的好，并且很多设计师不擅长说话，是你吗？对不对？


我们觉得设计师是不是可以不用与人交流，你就大错特错了，我们跟前边的这个环节，也就是产品经理需要打交道，需要沟通原型图跟我们的下游环节就是开发 RD，也需要交流沟通。所以其实我们设计师的行业是个与人沟通的行业，虽然我们并不是像运营部门那样需要与人亲密的沟通，但是我们的工作也要求我们需要有良好的沟通能力，所以大家不能只盯着电脑，我们也要锻炼一下怎么样能够传递我们的思想，通过我们的语言如何能够更好的传递我们的思想，所以这个分享环节是非常重要的。


我还记得第一次我在我当时的公司做分享的时候，非常的害怕，当然当年我在的公司有三个设计师是吧？所以搞了一个小小的分享会，参加了分享会的人，有产品经理有技术，然后其实人不多，人其实也就是二三十人，但是站在这二三十人面前，对吧？大家好像就有点熟又不太熟，然后 PPT 一开就有点懵了。 


我从哪讲起？我当时挺害怕的。如果你也是觉得有点紧张的话，在面试的时候甚至跟一个陌生人沟通都很害怕的话，我们需要锻炼了。人是个社会动物，是吧？我们需要锻炼侃侃而谈，来谈谈我们的理念，来谈一谈格式塔，来谈一谈情感化，设计三个水平，对吧？这些对我们的成长都是非常重要的。当然这也会锻炼我们设计 PPT 的能力，PPT 呈现的时候其实也是有一些技巧的，比如说我们不应该在 PPT 中加入特别多的内容，这个不好，因为你想象一下做的很远，然后屏幕上都是字就很可怕了。


所以 PPT 一般来说会比较简洁的介绍一个标题，内容是由你口述说出来的，并不是说直接打在 PPT 上，然后多图少文对吧，然后尽量制造一些有意思的环节，因为人是比较喜欢轻松幽默的对吧？所以你可以尽量设计一些梗，设计一些包袱，在你整个的演讲之中，不要让大家干坐着看你对吧？而且演讲的时候也是面带微笑，去给大家讲述。这个能力在你之后的沟通，甚至在你面试的时候都用得上，所以这也是一个提高自己的很好的一个办法。


另外还有一个也是用输出代替输入，就是我们要把项目完成以后做一个 review，做一个项目的回顾，然后包括我们可以设计一个设计规范，很多朋友可能觉得设计规范好像是老设计师给年轻设计师搞了一个源文件的一个大表格，对吧？你需要什么按钮，你拖进去对吧？字体多大？我告诉你，其实我们如果只是一个人来做这项目的话，也需要完成设计规范。


设计规范不仅是对他人的要求，也是对设计师本身的要求，我们很轻易地就会在界面中加入六七种字体，六七种字号按钮，同样的功能，可能换个页面做的时候你就忘记了，对吧？你就用了一个新的样式，这些都是错误的，这些都应该在设计规范中被规避，所以说哪怕一个人我们也要完成一套设计规范，设计规范包含的内容就是比如说字体的大小，对吧？咱们每一个 APP 大概也就这么六七种对吧？你不能够太多的花样了。太多的花样的话，在格式塔认知上，咱们上节课同样讲过的就会存在问题，太文字的大小各式各样的太多了，用户就会看到不太舒服。我们要重复来使用相同的样式，然后图标要规范，有的时候我们画图标的时候，也会不注意，那也是错误的。我们在完成感性部分以后，要再把它放到设计规范里看看，如果把图标都单拎出来，他们是不是可以再统一一下尺寸再统一一下设计的样式！


除此之外，还有比如说像按钮规范、图片的尺寸规范。图片的尺寸其实不是指的具体的几像素，而是说这图片的一个比例的问题，因为只要是差不多的尺寸，从后台调取了一张图片的话是可以放大缩小的，对吧？只要你不是太夸张就可以，但是比例是不变的，他同时有这张图片的构图，在这个图片的比例我们也需要在设计规范中规定下来，一般的比例就是 1:1，4:3，16:9 这三种对吧？


我们在设计的时候也不能随心所欲地画一个矩形就完事了，我们要考虑到最后运营这个产品的时候，用哪个比例会更加的方便，当你从网上寻找资源来学习，然后我们参加很多的活动，来分享看书下班做练习，然后每个项目都保证它有一个定期的回顾与设计规范的完成。那么即使在这公司里只有您一位设计师，那么我们也可以把这个工作做的井井有条，把我们的职业发展的道路也做得津津有味。


好，这就是我的一个小小的分享，就是关于独苗设计师如何进步的，谢谢大家，咱们下期再见。
