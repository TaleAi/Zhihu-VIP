## 简介
入骨相思，深藏心底。


相思相望不相亲，他们彼此深爱，终究还是遗憾错过。


**殇江**


那日我夫君带回一位姑娘，眉眼像极了他那前未婚妻。


我是公主，他是驸马，按理说这是不合礼法的。


但放眼京城，无人为我作主。


父皇对我漠不关心，皇兄对我别有用意，皇嫂待我笑里藏刀，而我唯一的妹妹已横死深宫。


就连亲自求来的枕边人，也时时盘算着要夺我性命。


不过无妨，我早已习惯于深宫的尔虞我诈。


天下熙熙皆为利往，就让我陪你们演这一出出大戏，为我自己，也为我妹妹的骨肉，在这盘根错节的皇城里，挣出一条生路、一线生天。
