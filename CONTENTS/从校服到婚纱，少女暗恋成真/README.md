**《告白》**


我做梦都不敢这么做，隔壁住的竟然是何予？！


我的男神！


我可觊觎他太久了


**《你和世界我都要》**


亲哥接了个兼职，临时有事去不了。


为了那四位数的酬劳，我去了。


嚯，好家伙，居然是租对象。


就这样，我成了男神的女朋友，租的！
