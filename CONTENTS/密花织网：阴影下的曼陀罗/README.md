## 简介
这是一个女性情感悬疑专栏。生活的琐碎之下，也许还有另一个曲折离奇的故事。


婆婆得了阿兹海默症，老公不孕不育，丈夫赌博成瘾……


你不知道每一桩生活的琐事之下，有怎样的离奇秘密，怎样的扑朔迷离。


纠葛在家庭、亲人、生活之中的女人们，究竟将发现怎样的真相？


「噩梦」：婆婆得了阿兹海默症后，本来对我如同亲生女儿的她仿佛变了人，对我又打又骂，还喊我婊子，我不堪侮辱，幸好还有丈夫关爱。但我和丈夫之间也有难言之隐，丈夫有问题我们没法要孩子。为了给婆婆养病，我们搬到乡下，但婆婆似乎更受刺激甚至拿刀砍我，而我也发现镇上的人看我的眼光指指点点，更有人直接调侃我情人去了哪里，我非常迷惑。而且最重要的是，一到晚上，丈夫似乎就会变成一个人，对我非常粗暴。直到有一天，我怀孕了……


「养鬼」：我和老公结婚七年，本来已经分房睡了。不知道为什么，老公突然焕发了第二春，天天缠着我同房。而每次同房后，他就会立即回到自己的房间。我非常奇怪，有一天晚上过后，我透过门缝发现老公似乎在拜什么。后来我趁着老公不注意，安装了摄像头，我这才惊恐的发现，老公拜的台子上竟然放着我前男友的骨灰盒！而他喃喃念的是，「我把我老婆献给你，求你让我发财」。那么，每天晚上缠着我的就是……


更多秘密，仍在等待真相……
