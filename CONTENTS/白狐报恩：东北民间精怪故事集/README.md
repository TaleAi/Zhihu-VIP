## 简介
在那个特殊的年代，民间流传着许多精怪故事，白狐报恩，黄皮子讨封，刺猬复仇……一个个离奇的故事背后，是对人性的探索。


我曾从一位叫赵娥的老人口中，听过一个「白狐报恩」的故事，她小时候曾救过一只白狐，家里因此大发横财，只是富贵没持续多久，家里便遭受厄运，家破人亡。
