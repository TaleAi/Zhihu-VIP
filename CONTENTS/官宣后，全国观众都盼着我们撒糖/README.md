## 简介
人间最美的事莫过于年少时遇上最耀眼的你——影帝的追星日常。


《影帝失忆后喊我乖乖》


影帝陆深失忆了，非说我是他女朋友。


他指着我和顶流的合照，问我是不是移情别恋了。


晚上，热搜爆了。


当红影帝发帖问：「如何挽回女朋友的心？」


热评第一的网友说：「肯定是男人不行，满足不了女朋友。」


陆深扭头看我：「你觉得我不行吗？」


啊？


我哪知道您行不行？


陆深捧住我的脸。


「那试试。」


《宁班长来听我的演唱会》


最近挖出时间胶囊的视频走红。


其中还有顶流歌手周子昂十八岁时写下的纸条。


「二十八岁的周子昂，你有和西西结婚吗？」


众人都以为是创作者故意蹭热度。


却没想到周子昂会回应。


「还没。」
