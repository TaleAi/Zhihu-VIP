## 简介
微博百万粉丝优秀历史科普作家刘鹤（笔名：幻想狂刘先生）首部作品，高晓松作序推荐。
一部以武器和战争为主题，涉及火枪、火炮、长枪、刀、剑、弓弩、战场着装、武术等众多领域的战争文化史。真正从战场带回来的真知灼见，用剑和戟可在黑铁上的硬核力作。在钢筋水逆构筑的森林里，让遵循古道的现代骑士为你讲述人类战争的残酷史。


《兵者不祥》是一部以武器和战争为主题的战争文化史。书名出自《道德经》：「兵者不祥之器,非君子之器,不得已而用之」，意象复杂而深远。「兵者」既指武器，又指向战争本身。


本书对武器和战争的种种技巧、谋略进行了细致入微的描述和分析。作者一方面赞叹人类运 用的技巧和勇气，以及在战 争中表现出的高度智慧和精妙谋略，另一方面又对战争本身的荒谬性和残酷性进行了深刻的反思，其中涉及世界多种文明的历史、文化、交流和冲突。作者以西欧和东亚作为世界文明桥梁的两端切入，把看似无序的一系列历史故事，绘制成一幅人类文明史上的纷争画卷。


此外，作者还将常年参加中世纪格斗赛事「诸国之战」的战斗技巧和战场经验灌注书中，希望将一些现代社会中不可多得的宝贵实践知识传递给读者。
