真实的考古究竟是什么样的？众说纷纭的考古谜题，真相到底如何？解锁本专栏，知乎艺术话题优秀答主，浙江大学考古学硕士项木咄，用破案式的考古思维，带你围观最真实的考古现场，探寻「考古未解之谜」，一窥令人目瞪口呆的历史真相。


秦始皇陵为什么一直没被挖掘？地宫中的九层夯土，是不是暗合了九层妖塔之说？


三星堆出土的青铜人，到底是不是外星人？

……


考古工作中，总有些真相难以探寻，成了长久困扰人们的「未解之谜」。


其实，真相远没有那么邪乎。


本专栏里，作者会逐一给大家介绍考古遗址中那些广泛传播的、被人误读的「未解之谜」，再用破案式的考古学思维，详细讲述考古学家们是如何运用各种知识和常识，抽丝剥茧揭开这些「未解之谜」背后的真相的。
