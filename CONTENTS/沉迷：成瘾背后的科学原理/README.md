## 简介
吃不完的鸡，刷不完的屏，拔不完的草，闭不上的嘴...... 生活中处处埋着让人玩物丧志、欲罢不能的钩子，让我们沉迷其中。上瘾现象不再是「废柴」的专利，许多有着钢铁般意志的人，也难免沦陷。上瘾的背后，是商业社会对人的生理机制的巧妙利用。这门课将从心理学和脑科学角度，剖析成瘾现象背后的科学原理和社会因素，让我们理性地对待成瘾，更好地掌控自己的生活。
