## 简介
本专栏是法医红眸记录的基于真实案件改编的罪案故事。从业十年来，他手持银色解剖刀，为生者权，为死者言。在他眼中，尸体会说话，且只说真话。真相就在刀下！


作者红眸，曾为法医，毕业于北方某医科大学法医专业，从业十年来处理了 300 多宗刑事案件，临床鉴定 800 余「人」。


央视报道过他，美国《时代周刊》用了两个版面刊登他的照片。


红眸法医职业技术过硬，对现场勘验、尸体解剖、临床鉴定、毒物分析都有丰富的一线经验。他曾下水捞尸体，也曾深入化粪池里找凶器。


正因为如此，此专栏故事内容也过硬。


里面每个案件都参考了真实案件原型，**涉及暴力人身侵害案，拐卖儿童案，暴力杀妻抛尸案，投毒案，无差别随机杀人案……**


故事经过严格的法医学取证、辅以缜密的线索推理和强大的犯罪心理学侧写。


专业，精彩，烧脑。
