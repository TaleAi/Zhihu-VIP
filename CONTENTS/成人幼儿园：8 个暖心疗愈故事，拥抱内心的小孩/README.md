## 简介
回到童年，重新长大。


不是所有的痛，时间久了就会痊愈。


不是长大了，就会更坚强，不再受伤害。


如果能有机会，回到童年重新长大，你会怎么做？


  



许苑是一个三岁小姑娘的妈妈，她明知道丈夫有外遇也没有勇气面对。


作为家庭主妇，她默默承受着亲戚朋友的各种质疑和轻视，再把怒火都发泄到年幼的女儿身上，使女儿性格懦弱古怪。


一次在幼儿园她失控了，对女儿大发雷霆，引起了到幼儿园来交流经验的退休老院长的注意。院长问她愿不愿意再当一次小朋友，加入老院长退休后创办的「成人幼儿园」。


这里除了她之外，还有许许多多无法适应成人世界的「小朋友」。


有二十多岁一直家里蹲的宅男


有三十多岁突然被裁员的上班族


甚至还有六十五岁退休大爷……


在这所特殊的幼儿园里，他们能找到小时候的自己，再次勇敢长大吗？
