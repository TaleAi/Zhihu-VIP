## 简介
女孩的短篇脑洞悬疑故事。


城市的背面，隐含着一套动物世界的运行法则。


穿梭在钢铁森林的少女，是白领，也是猎豹。


红色高跟鞋，也可以是锋利的爪子。


她们捕捉最原始的欲望，也躲避危险的猎枪。


所以别太惊讶，穿白纱的新娘背负着腐朽的十字架。


当她突破禁锢，撕下头纱，她光脚踩在土地上，获得了源源不断的，生存的力量。
