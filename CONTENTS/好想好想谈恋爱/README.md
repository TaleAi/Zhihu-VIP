## 简介
母胎 solo 乖乖女，单亲家庭独女，中产家庭娇娇女，丧偶阿姨，开房车的酷姐姐……她们年龄不同、经历各异，但都有一个共同点——好想好想谈恋爱啊！本专栏收录不同女性的追爱故事，她们面临不同的困境和牵绊，也在过程中收获了自我成长。


**- 霸道总裁之恋 -**


孟小羊独自一人生活在大城市，租房、宅女、社畜，母胎单身，业余时间看了不少甜宠电视剧和霸道总裁文。孤单久了，她也想获得一段轰轰烈烈的爱情。


她认识了一个女同事的男朋友，这位男朋友碰巧就是一个很霸道总裁式的男人，孟小羊心里很羡慕女同事。随着交往越来越多，这位男朋友对孟小羊也霸总了起来，使她陷入了暗恋，不可自拔。


**- 顺风车之恋 -**


徐远峰有很多的兴趣爱好，包括摇滚乐、摄影、房车旅行等。她从前是一个广告行业从业者，事业有成，挣了不少钱后，就开始一个人疯疯癫癫过上了开着房车到处跑的生活。


别人问起她想找男友的标准，她说：「能跟着我一起疯，话还少」。


一次路上，她遇到了想搭顺风车的男人，还带着一条狗。
