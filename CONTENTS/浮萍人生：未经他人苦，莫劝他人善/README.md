# 浮萍人生：未经他人苦，莫劝他人善

## 简介
一些都市青年的原生家庭困局。这里有重庆伢子巷里互相欺骗的父与子，有苦苦隐藏被拐往事的煎包店女老板，有不愿把人生献给弟弟的姐姐，也有携手反抗婆婆的新媳妇与前妻……一个个看似离奇的故事，包裹着他们作为普通人未曾停止挣扎的一生。


《山风忽来》
------


伢子巷里住满了跟忽来一样的人，他们老弱、堕落、贫穷，背井离乡。与周围的高楼广厦格格不入，像是一条细细的伤疤，时代的断层。


忽来在这里住了 3 年，他爸老郑，住了 13 年。


他们怨恨对方，鄙视对方，欺骗对方，却也爱着对方。


这就是父与子，永远那样拧着的关系。


  



《孽障》
----


19 岁时，周玉霞被人拐卖给 42 岁的光棍，生了一个儿子，3 年后逃出来隐瞒过去重新生活。


10 年后，她当年生的儿子在网上寻亲，志愿者找到了她。


此时的周玉霞已经成了生煎店里的老板娘，老公踏实温柔，女儿年幼可爱。


当年的痛苦经历，成了她幸福人生的一颗炸弹。


  



《及时止损：爸妈，给女儿一条活路》
-----------------


29 岁那年，董晓妮终于如愿远嫁，自认用一笔彩礼买断了和父母弟弟的亲情。


她不想做扶弟魔，她想有自己的人生。


可是快活的日子还没过一年，她就再次接到了母亲的电话。


  



《浮生列传：悔婚记》
----------


何晓雯嫁给二婚带女的李博伟，是真的为了爱情。


丈夫温柔浪漫，继女乖巧可爱，婚前的何晓雯沉浸在幸福当中，对丈夫前妻的提醒充耳不闻。


直到过门之后，她见识到了「绿茶」婆婆的手段。
