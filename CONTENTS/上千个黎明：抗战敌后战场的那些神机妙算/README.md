## 简介
伟大的抗日战争，距今已爆发 90 年。本专栏记述了 1931 年「九一八事变」到 1945 年抗战胜利，这一段用鲜血与泪水写就的中国敌后抗战历史。八路军和新四军主导了一场又一场以少胜多，直戳敌人要害的决斗。战场背后，有着无数鲜为人知的内幕。

本专栏的 8 场战役，为你还原真实的抗战敌后战场。八路军和新四军极为有效、灵活多样的战法，以少胜多，积小胜为大胜，使日伪军犹如困兽，陷入人民战争的汪洋大海之中不可自拔。


山林之王杨靖宇、平型关战役始末、号称日本「名将之花」的阿部规秀被歼灭内幕、对日寇的最后一战……在敌后战场，八路军和新四军中涌现出无数英烈，他们善战、顽强，他们令日寇闻风丧胆又失魂落魄，他们是一个个勇敢的战士，更是民族存亡之际，拼到最后一刻的、大写的中国人。
