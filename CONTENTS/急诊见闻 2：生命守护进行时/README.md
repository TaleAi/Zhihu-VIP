## 简介
急诊抢救的使命，就是要守护生命。


擅长用生动病例科普医学知识的李鸿政医生，带着新专栏《急诊见闻 2：生命守护进行时》来了。


这个专栏里，有着多年重急症工作经验的李医生，将继续分析各种常见病或突发性急症会导致的危险可能，探寻隐藏在病症表象下的真实病因：


* 病人恐水，就一定是狂犬病吗？
* 腹痛有多少种可能性？
* 胸痛就是心肺有问题吗？

同时，普及多种常见病症相关的医学常识，以及日常生活适用的急救知识。


除了惊险刺激的抢救案例，新专栏里，李医生还会跟我们一起探讨值得深思的问题：


* 面对高昂的费用，ICU 里的病人，是否要坚持抢救？
* 医生总要求病人先做各种检查，有没有必要？
* 异地就医，医保是否能够报销？报销比例如何？

了解李医生和他的同事们做诊断、做决策的全过程，还可以帮普通读者换位思考，对医生有更客观科学的理解，缓解医患关系。


每一篇故事结尾，还有李医生专门准备的「科普小课堂」。


**关键时刻，这些知识能救人一命。**


愿你有所收获～
