## 简介
孩子发热、惊厥、抽搐……家长如何及时处理？


当宝贝小脸通红、浑身滚烫，是不是让你心疼坏了？甚至感觉有些手足无措？


相信所有父母都是这种心情，只要孩子一发烧，便会全然不顾酷暑或者严寒，直奔医院。


其实，发烧是人体对外来病原的有效免疫反应，也是对身体的一种自我保护。因此对孩子的成长来讲，发烧不完全是一件坏事，它几乎是每个孩子成长过程中必经的考验。


所以遇到孩子发烧，家长正确的做法不是惊慌失措或匆忙送医，而是冷静处理、缓解宝宝的病情。


不过，照顾发烧的宝宝，不仅需要你有足够的耐心，还要有正确的护理知识武装自己。否则，面对无助而可怜的小宝宝，你可能真的难以应付。


本课程特邀原北京儿童医院医生鲁靖，教你判断宝宝发烧的「黄金标准」，改变你以前对于宝宝「头疼脑热」的认知理念，给你一套宝宝发烧的家庭护理指南。


相信你能先人一步，掌握宝宝发烧更科学、更权威的护理知识，成为更优秀的父母。
