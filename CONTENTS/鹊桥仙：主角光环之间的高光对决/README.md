## 简介
这是一个甜虐交织，上天入地，包含了现代、民国、仙侠和后宫的言情合集。


当虐文女主光环博弈恶毒女配光环；当白月光光环颠覆替身光环；当朱砂痣与病娇天帝互相囚禁；当女扮男装遇见女装大佬……丝丝入扣，环环对决，该是怎样的精彩纷呈。


《但为良药》


意外穿书，在遍地男女主的小说里，成为了除了主角光环没有，什么光环都有的炮灰女配，又阴错阳差地被替身光环搞进了监狱，凭借倒霉光环遇见了病娇暴躁军官上校男主，要如何靠着那一堆不着调的光环苟到大结局。


《心有独钟》


沈迦萝穿进了虐文，开局即地狱，家徒还四壁，却有一个花花世界花花钱的妈；跟爸要钱被拒绝，又遇原男主大虐虐；身为男主，他没心没肺没脑子，可就是不会没自信，开口就是我选你。


谈谈情，虐渣渣，装柔弱，扮无辜，勇斗病娇绿茶壶，用用聪明脑袋瓜，反正狭路相逢，她必须胜！


《白切黑月光》


天地自蛮荒归来，身后跟着一个女子，那是天界众仙的白月光——月华仙子。


而任鹊羽，区区凡女，却是天界众仙眼里的罪人，只因她白月光替身的角色没有拿捏得死死的。


所以月华仙子心悦天帝，是四海八荒皆称的佳话，而她爱慕帝主，则是三界六道俱知的笑话。


可她自己心里清楚，她不是痴心妄想，她只是没有喝下那碗孟婆汤。


《倾君渡》


说好当霸总白月光的替身三年，就给三千万，却不想合约到了期，霸总竟想续，争执中到了异世界，霸总成了皇帝，她那影帝初恋又是出征归来的霍大将军！


而国师对她『好女嫁三郎』的预言，更是将好好的大婚礼堂，变成了她与三个男人拜堂的修罗场。


于是身兼皇后、摄政王王妃与国师夫人的她懵了，究竟谁是我的新郎，而我又是谁的新娘？


除此之外，还有另外六个阴谋阳谋，缠绵悱恻的反套路言情陆续上线！
