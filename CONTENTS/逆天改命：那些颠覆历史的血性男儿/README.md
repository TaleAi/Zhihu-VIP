## 简介
人生苦短，热血难凉。


古往今来，每到至暗时刻，往往天降奇才，拨云见日；遭遇危急关头，总有人横空出世，力挽狂澜。


他们原本也是普通人，有的胸怀凌云壮志，也有的只想游戏人间，有的背负血海深仇，也有的只是个闲云野鹤，有的绞尽脑汁出人头地，也有的仅仅是被时势推到了风口浪尖。


本专栏聚焦那些在重大历史时刻力挽狂澜的血性男儿郎：


**看「面涅将军」狄青，如何在文弱的北宋张扬武德。**


**看「但愿海波平」的戚继光，如何狂虐不可一世的小鬼子。**


**看「金刀计」主角王猛，化腐朽为神奇，打造强横一时的前秦帝国。**


**看「明犯强汉者虽远必诛」的陈汤，强悍折断北匈奴复苏脊梁，热血守住西域版图。**


**看「有志者事竟成」的耿弇，如何耍出教科书级的花式兵法，事了拂衣去，深藏功与名。**


要么领风弄潮，要么搏击沧海，人生可以翻转，月明云开、浓墨重彩。
