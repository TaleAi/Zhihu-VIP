## 简介
想做设计，怕不挣钱？做室内设计该如何为自己制定详细的职业规划？设计是一种职业，但是设计师绝不是单纯的职业。本专栏将告诉你怎么从设计快速进阶到室内设计师，希望你在设计师的职业路上少走很多弯路。


想做设计，怕不挣钱？


无法把握设计行业的前景，总担心自己被淘汰？


想了解设计师的日常状态是怎么样的？


为什么很多设计师总收不到设计费？


其实，关于「室内设计入行」这个命题，我常被我刚毕业的助理，还有知乎上许多在读大学生问到过。其中，大部分人都关心做设计能不能挣钱，究竟是要考研、去设计院、还是做独立设计。当然也有同学问我究竟应该怎么做设计等诸如此类问题。


设计师并非盲目的学习或刷经验就能获得进步和成功。康德认为，按照时间，没有任何知识是先行于经验的，都是以经验开始的。设计师也并不是纯粹以知识+创造性来解决居住需求后获得物质变现的。浅层次设计的核心有可能是「图」，但深层次的设计核心一定是人和行为。


所以，设计是一种职业，但是设计师绝不是单纯的职业。那么，究竟室内设计师应该掌握什么样的平衡和自我提升？我将从我的个人经验出发，给大家讲讲入行前的准备、入行后的生存与竞争。同时，在刷足经验后的个人品牌建立方面，也用我的经验来给大家一个参考。希望你在室内设计师的路上少走弯路。
