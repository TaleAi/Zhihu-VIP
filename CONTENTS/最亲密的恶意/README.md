## 简介
往往，最致命的恶意都来自最亲近的人。在每一个甜蜜的举动下，隐藏着一把阴暗的匕首。在这个专栏里，你会看到关于混乱关系的复仇，脑洞大开的反转，真实经历的还原。


**《商圈反击录》**


我没想到，一夕之间，我会失去一切。


事业，家庭，未婚妻，甚至她腹中的孩子。


都因为我在网上开了个店。


  



**《渣爹报复记》**


那天，我拿到全国排名第一大学的录取通知书。


我妈因为护士长，脱不开身陪我，但是却在我们同一屋檐的三人群里发语音说，今天破例要开瓶八二年的拉菲。


可等回到家，却发现等待的不是爸妈为我准备的一桌好菜，而是…...


  



**《养女的前世今生》**


不知道为什么，我二哥林承从小就讨厌我。


虽说我是半路出家的养女，可是爹妈一向对我视如己出。


撇开这层关系，哪怕是对陌生人都不应该这样啊，何况还是青梅竹马。


我曾奢望，他有大哥林应对我一半的好，我也就知足了……
