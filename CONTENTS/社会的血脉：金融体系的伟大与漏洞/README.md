## 简介
如何成为一个清醒的现代人？



我们知道，现代市场经济的基本形态是以债务和信贷为核心的。


在理解现代经济的过程中，信贷体系不可忽略，它带给这个系统超乎想象的合理性与伟大，也在其中深埋了许多的不合理与漏洞。


我们不仅要理解金融这个伟大系统的规则，也要看到这个系统的 bug 是如何伤害到我们的经济与生活：


* 华尔街精英如何利用规则的漏洞牟取暴利？
* 货币如何取代黄金，并依据自身规则左右国与国之间的博弈？
* 历史上值得关注的经济体系是根据哪些规则建立的？它们对我们现在所生活的世界有什么影响？

在试图找出一个系统的不合理前，我们必须把它的合理性给理解透彻了，再来谈不合理；在谈阴谋论前，我们必须先把明谋给吃透了，再来谈阴谋。


本专栏将会带你一起看透那些翻涌在金融市场上的阴谋与阳谋，在这个纷繁复杂的社会中找回一丝清醒。
