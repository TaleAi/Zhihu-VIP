## 简介
罗振宇、吴晓波、吴声、贺学友、杨飞、徐志斌联袂推荐的营销思维课。
「如何用低预算做出大营销」几乎是所有做品牌、市场、公关的从业者的痛点。唐文基于十余年的营销经验，提炼出最新的营销方法，将经典营销理论予以升级，以「故事」视角阐述商战，要言不烦地用生活化、最新案例讲透晦涩的 10 个营销理论，深入浅出教你用低预算做营销。


大多数市场、品牌、公关从业者都会遇到这些问题：


  • 身处缺钱的中小企业，如何「用小预算做大营销」？


  • 先前项目积累的营销经验，在新的产品面前不奏效了。


  • 经济寒冬下创业，新产品如何「低成本实现增长」？


  • 传统营销人员想转型互联网，一筹莫展。


  • 瑞幸咖啡、RIO……想从当今大火的品牌的营销案例中，偷师到什么方法？


这门课，抛弃过去企业营销打价格战、砸广告、拼渠道的「花钱式营销」，直面客户的应用性场景，从真实情况着手解决营销难题。


针对营销常见的用户研究、产品定位、增长等问题，唐文基于对营销本质的观察，创业试错与服务宝洁、蓝色光标等企业的十余年营销经验，提炼出最新的营销方法，并将经典营销理论予以升级。


课程亮点：
-----


**1.系统完整的课程逻辑**


从营销的起点，即如何理解用户、选品类、推广到增长、定价，直至最后达成销售，是一次完整的飞行体验。10 节课教会你如何「准备起飞」、「选跑道」、「造声势」到最后「跑起来」并最终「拿奖」。


**2.原创营销方法+经典营销理论**


「定位」、「故事经济学」、「增长黑客」……既强调营销专业的基本功课，又总结了互联网时代下的营销新打法。 「品牌关注窗」、「轻营销」……凝结了唐文十余年营销经验总结出的原创方法，帮你尽可能多的掌握各类产品的营销方法。


**3.「故事」视角阐述案例**


情侣们为什么爱吃辣味火锅？ 可口可乐和白雪公主之间有什么内在联系？ 瑞幸咖啡靠裂变营销就能战胜星巴克吗？ 揭示生活里处处隐藏着的营销策略，以「故事」化生活化视角解读商战中的营销战略，即使你是零基础也能帮你建立营销思维。


**营销思维可以帮专业的市场、品牌、公关从业者：**


• 看清用户真实心理，达成销售


• 穿透竞争本质，选准赛道


• 分析增长技巧，找到突破口


**但也可以帮你：**


• 刷新消费观，有效主动地掌控自己的消费心理


• 触类旁通，利用营销思维解决问题、打造个人品牌


• 熟悉营销套路，拓展商业视野


罗振宇、吴晓波、吴声、贺学友、杨飞、徐志斌联袂推荐。
