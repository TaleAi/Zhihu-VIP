## 简介
认知能力的提升，意味着世界观、格局、思考问题方式的改变，乃至改变人生。这个专栏，在思考中进化你的原始大脑，获得一套完整的思维框架，帮你高效地学习和掌握知识，让行动价值最大化，快速解决生活或工作上的难题。


* 很多人很聪明也很勤奋，但却很难出成绩，原因是什么？
* 上级和老板总是能想到你没有想到的地方，他们是怎么做到的？
* 来自同一个学校，年龄、学习成绩都差不多的同学，工作以后取得的成就用天差地别形容也不为过，这其中又隐藏着什么玄机呢？
* 在职场中遇到天花板，停留在中层管理者（经理、高级经理），为什么无论如何也达不到高管（总监、副总裁）的层次？
* 任何时代都有很多机会，但只有少数人能抓住，为什么你没有？

**——你需要提升自己的「认知思维」。**


计算机在硬件配置差不多的情况下，能够做什么和做得多好主要取决于软件（操作系统和应用程序）。


从先天而言，同一时期人类的大脑大多在同一起跑线上，这是硬件基础。人与人的不同，主要在于人类大脑运行的软件基础的差别，我们把这个人类大脑的操作系统称为认知能力。


**当你形成自己的思维模型和认知框架后，就可以在实践中打碎再重塑。思维能力达到螺旋式上升，就是认知升级的终极形态。**


这个专栏，帮助你逐渐形成思维模型，拉开你和他人的差距，拥有个人持续性的竞争优势。认知能力的提升，意味着世界观、格局、思考问题方式的改变，乃至改变人生。


  



●  4 大升级模块：


  重新定义思维——用思考进化原始大脑，获得一套完整的思维框架


  重建学习之路——辨析对你最有价值的知识，高效地学习和掌握知识


  有效实践应用——让行动价值最大化，快速解决生活或工作上的难题


  高速成长迭代——创造自己的概念，超越同龄人，从更高的维度看世界


●  34 个进阶之法：解密职场精英人士从未公开的升级之法，从走出思维误区到作出正确的决策和判断，从认知升级到格局破局。


●  核心方法论：思想精华+商业实践，列举大量实践证据，借助思维导图，深度剖析认知的本质，帮你掌握成功真正的逻辑。
