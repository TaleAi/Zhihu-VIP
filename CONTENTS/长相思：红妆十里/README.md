## 简介
这是一个言情故事合集，沙雕、甜文、宫斗……你爱看的言情故事这里都有。


《一枕偏安醉晚南》


家中惊逢巨变，父母相继离世之后，阿箬从没想过，自己还能拥有一次改变人生的机会。


这一回，她回到一切故事的开端，做出了与前世截然不同的选择。


可原来苍天竟这般弄人，从前她所知所晓都并非全貌，让她前生一世悲怆。


也兴许是苍天对她有所垂怜，才能让她重回这人间。


这一世，解未解之结，圆未圆之愿，去爱从前没能来得及去爱的人。


还好，她都有做到。


《精神小伙不请自来》


前一世的貂纱第一次见到天界最尊贵霸道的太子爷的时候，她就知道如果爱情是一种毒，那她注定毒入膏肓、无药可救。


她死过一回才知道恶毒舔狗女配竟是她自己。


笑死，原来根本没有爱情，只有毒，还是让她死无葬身之地的毒。


重生后，她只想从此不谈地久天长，单刀赴会做气质流氓。


毒解了的她面对这些剧毒的男人避之不及，只想和他们断绝一切关系，可是他们却变得不再一样……她逃，他们追，他们都插翅难飞。
