## 简介
其实现在有很多朋友都想要学习吉他，但作为一个彻彻底底的小白，光是在买吉他前就已经犯了愁。本专栏将从吉他的硬件知识、不同价位、入门教材等方面介绍购买吉他的注意事项，带你选购真正适合自己的吉他。


现在网上出售的吉他各种各样，品牌繁杂，价格有高低，外形又多样。身边如果没有专业的人，想要通过自己查找零碎的资料去了解吉他的话是很困难的。


在自己一无所知的情况下该如何挑选一把适合自己的吉他呢？


在购买吉他前，我们需要了解一些吉他硬件的相关知识。吉他虽然是乐器，但是它也同手机、电脑等数码产品一样，也看品牌、硬件配置的类型与质量。不同的用料、不同的配件以及产地构成了吉他的音色与价格。所谓知己知彼，在了解吉他相关配置知道其大致的音色方向后，再根据自己的喜好及预算来选购。


在购买前我们需要摆正自己的心态，不被所谓的「性价比」所蒙蔽双眼，明确自己的需求。另外在购买渠道以及后续的学习上，本专栏也会帮助各位排雷，减少不必要的「学费」，让想要入门学习吉他的朋友们少踩坑。无论是购买吉他前后需要注意的问题，还是吉他到手后的养护及学习，本专栏都能让大家有更安心的购买过程以及有效的学习提升。
