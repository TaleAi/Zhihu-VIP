## 简介
总有些词句，跨越千年，把我们「砸」得泪流满面。


那些平淡生活里的相思、遗憾、难以实现的梦想，随着时光流逝都渐渐淡化了。


但总有些古老的词句，能跨越千年，再次唤醒旧日的记忆。 


潘安 · 爱情是疲惫生活里的英雄梦想
------------------


美男子的代言人潘安，开创了悼念亡妻的悼亡诗题材。


他出道就是神童，妙笔织就云霞，皇帝当场叫好，无论是才华还是颜值都是天下一流。


是怎样的姑娘，让他在十二岁开始就矢志不渝，一生一世一双人，失意或者得势都不能令他变更分毫？当斯人远去，潘安的世界发生了什么样的改变？


李清照 · 也曾看轻天下英豪
--------------


少女时代的李清照，是意气风发的，扬着笑脸且自负的。南渡之前的一篇《词论》，俨然有自认天下第一的趋势。


她既有少女的娇羞，又有和喜欢的男生凑在一起打牌喝酒的豪爽。


后来那个凄凄惨惨戚戚的李清照，是怎么出现的呢？


**辛弃疾 · 嘴上认命，从未甘心**
-------------------


这么多豪放词里，苏东坡有仙气，贺铸有侠气，辛弃疾得一股悲慨英雄气。


求而不得四个字，多少志犹未已心。北方金戈铁马一身本领，来南方后却只能用来平定农民起义。


辛弃疾的后半生里，他跟朋友举杯豪饮时，听到的都是梦碎的声音……


  



在本专栏中，房昊将与你一同探索诗词背后的故事，在中国古人深切的情感中，找寻当代人的灵魂共鸣……


**我们不在一个时代，但拥有相似的爱怨与惆怅。**
