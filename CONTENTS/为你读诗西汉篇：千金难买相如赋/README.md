## 简介
本专栏为你深度解析「长门赋」，带你领略汉赋华美瑰丽的语言艺术。


每个朝代都有自己引以为傲的文学载体。


唐诗、宋词、元曲……


汉朝的载体是什么？**恐怕就是汉赋了**。


汉赋想象之宏大，文字之瑰丽，后世难以望其项背。


其中，又以司马相如的赋为当世最高。


在本专栏中，**卫东老师用声音带你深入赏析司马相如的「长门赋」和「上林赋」，**用心体会汉赋中蕴藏着的精彩与魅力。
