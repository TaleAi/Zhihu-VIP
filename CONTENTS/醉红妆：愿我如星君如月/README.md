## 简介
祈望我是星星你是明月，每一个夜里你我光明洁白的光彩辉映着。


《与君同归》


宫里人人都觉得，我是个是傻子。


不过是丞相送进宫的棋子，竟敢妄图与皇后娘娘争宠。


但其实大家都不知道，贵妃这份职业，我不想干，但刀架在我脖子上，我不得不干。


「如何能在不被人察觉的情况下，让皇上讨厌我？」是我每天都在思考的生存问题。


太难了！
