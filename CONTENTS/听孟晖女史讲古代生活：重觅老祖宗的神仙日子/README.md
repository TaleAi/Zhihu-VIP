## 简介
听孟晖女史讲古代生活，与千百年前的古人相遇，重觅老祖宗的神仙日子。


唐人描绘花鸟，总是选择春光明媚的时刻。宋人却异常迷恋寂寥的秋与冬，琉璃世界，白雪红梅，人与天地之间似乎更容易融为一体。


但是漫长的冬季也容易让人情绪消沉，对此古人有个妙策，那就是亲朋好友们一起宴聚「消寒」。美味佳肴连同聚会当中的欢乐气氛与温情，无疑是提振人心的灵丹妙药。


「消寒」的传统就一直延续下来，在士大夫文人中渐渐演变成了文化活动，与宴者在饮酒吃肉之余，一般都要吟诗联句，甚至弹奏古琴，现场作画。于是漫漫冬日中，一颗颗富有才思的心灵不再孤独，反而因碰撞而激发出喜悦。


孟晖女史以独特的美学目光、严谨而优美的文笔，带着花间的胭脂气息，缓缓走来。那些精心复原的细节、细腻的语言，以及她清雅温婉的声音，将把我们带回那个熠熠生辉、令人迷醉的时代，与过去的伟大重逢。

倘若你对唐宋雅士、古典闺阁心有向往


倘若你需要一条高效的途径
