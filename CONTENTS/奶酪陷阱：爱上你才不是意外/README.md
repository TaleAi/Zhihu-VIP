## 简介
这是一个套路与反套路对垒的感情故事集。



有甜到齁嗓的恋爱，也有刀中撒糖的感情；有针尖对麦芒的 cp 博弈，也有欢喜冤家的轻松过招；分分合合聚聚散散，只为成全一辈子的，命中注定。

恋爱像是一块奶酪，甜的头脑发昏的时候才发现已经步入别人的陷阱……
