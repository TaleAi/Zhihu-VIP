**《泥儿子》**  

听我太奶奶说，旧时候的人将一种「外表裹泥，里面包肉」的泥塑娃娃叫「泥儿子」。


他们把「泥儿子」供奉起来，期盼能给家里招个男丁，来继承香火。


但这玩意养在家里，是会招灾的，邪门得很。
