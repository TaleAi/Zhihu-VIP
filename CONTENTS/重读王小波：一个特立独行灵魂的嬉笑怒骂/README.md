## 简介
没有一场非正常死亡，他的作品寂寂无人问。在这场非正常死亡以后，他被捧成了新的权威，自名为「王小波门下走狗」者，成群结队。原因何在？重读王小波，一起来了解这个特立独行的灵魂的嬉笑怒骂。


「我对自己的要求很低：我活在世上，无非想要明白些道理，遇见些有趣的事。倘能如我愿，我的一生就算成功。」——王小波，一个特例独行的有趣灵魂。


历史上从来不缺生前默默无闻，在死后才被人发现价值的名人，比如曹雪芹、梵高、卡夫卡。但王小波在世时肯定想不到，他自己也是这样的人。没有一场非正常死亡，他的作品寂寂无人问。在这场非正常死亡以后，他的全集、选集一版再版，尽管他终生致力于解构和反对权威，但他还是被捧成了新的权威，自名为「王小波门下走狗」者，成群结队。


那么，王小波究竟有着怎样的文学地位和现实地位，才能受到如此的认可，甚至是追捧，这节课带你了解他。


本课程内容精选王小波的经典著作，精心策划解读角度，围绕现代人的常见困惑，诸如爱情、成长、舒适区、自我、妥协、幸福、焦虑和自由等，全方面解读王小波作品以及生活中所蕴含的独特理解，带你领略这个有趣灵魂的嬉笑怒骂，或许也能给迷茫中的自己一些启发与慰藉吧。
