## 简介
不是每个人都能上北大，但每个人都能掌握北大学霸的学习方法，Stephanie老师将带你走进自学力课堂，提供给你一套北大学霸系统的自主学习方法，让你不再为学习能力弱而苦恼。


你有没有遇到这些难题？


每次上手新工作，都苦于没人带、无从下手，想努力都没方向；


报了各种班，花了上万块学费依然通不过考试；


拼尽全力，苦学 1 万小时，却发现依然原地踏步。


为什么同样没人带，有些人碰上什么工作都无师自通？


为什么上同样的课，有些人学起来就毫不费力，轻松考高分？


造成差距的原因只有一个，就是自学能力。
