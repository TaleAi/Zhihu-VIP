## 简介
本专栏由一直致力于中亚考古美术研究的邵学成博士主讲，他不断奔走在阿富汗一线文化遗产调查现场，将通过自己的亲历观察和研究，为我们介绍阿富汗文物背后所承载的文明、历史。


  



你知道，汉代张骞出使西域，到达的是哪里吗？


你了解，唐代玄奘去西天取经，他途径最重要的国家是哪个吗？


没错，就是**阿富汗**。


  



虽然挨着我们的边境线，却因为高山的阻隔，让阿富汗蒙上了一层神秘的面纱。


地理上：阿富汗作为欧亚大陆东西方文明交融的十字路口，是多种文化的集散地。


历史上：亚历山大大帝曾率领大军东征到达这里，带来了西方希腊、波斯的文化和审美。


在南面：也受到了来自印度、佛教文明的影响，催生出了阿富汗自身独特的文化和审美。


![img](https://pic1.zhimg.com/v2-16f5498153c4535f1e71bf6d466be670.png)

从埃及经由波斯传来的玻璃容器、印度的精美象牙制品，还有从希腊化城市阿依哈努姆出土的雕像，以及大量的黄金制品，这些被视为阿富汗人们的骄傲的文化遗产，成为了一把解开东西方文化交流之谜的钥匙。


  



对于阿富汗，大家可能会好奇：


1. 带有东西方文明密码的珍贵文物如何保存至今？
2. 阿富汗的二维野性审美和近现代艺术史中的野兽派和立体派的兴起又有何关联？
3. 迷雾般的亚历山大城、神秘的大月氏和当地的游牧民族，到底有哪些千丝万缕的联系？

让我们带着这些疑问，一起跟随邵学成的讲述，走进神秘的阿富汗。


  



你将收获
----


1. 为什么阿富汗是世界文明的十字路口？
2. 亚历山大大帝东征阿富汗留下一座什么城？
3. 「东方庞贝」贝格拉姆城是怎么被发现的？
4. 玄奘取经看到的巴米扬大佛怎么样？
5. 在巴克特里亚，如何寻找希腊文明的足迹？
