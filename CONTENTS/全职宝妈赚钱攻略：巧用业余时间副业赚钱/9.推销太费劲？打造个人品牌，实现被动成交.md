## 9.推销太费劲？打造个人品牌，实现被动成交
通过前面两个模块的学习，你已经找到了自己的优势定位，同时也通过各种引流方式拥有了第一批流量。而当你有了流量后，只能说你的副业之路向前迈进了一小步，如果不通过一系列有效的操作，你辛苦吸引来的用户很容易就会流失掉，更不用说实现变现了。


那这个模块，我们就来学习用户运营，我会通过 4 节课来讲解，分别是个人品牌打造、朋友圈运营、社群运营和视频号运营。今天是我们课程的第 8 课，这一节课我会给你讲解：如何打造你的个人品牌，实现被动成交。


个人品牌的网络俗语也叫个人 IP，为什么要打造个人 IP 呢？课程开始前我想给你讲个案例，我有一个学员是在微信里卖护肤品的，也就是我们所说的微商。平时她的朋友圈里大多数都是自己产品的广告，每次她跟客户推销自己的产品时，都觉得很难，不管她把产品介绍的多好，客户总是难成交。


后来我给她建议，让她把自己打造成一个护肤小达人的形象，然后持续在小红书、抖音以及朋友圈输出关于护肤的方法和技巧。她老老实实按照这个路径坚持了三个月之后，就开始有陌生的客户主动找她来咨询皮肤问题，比如长痘怎么办？t 区很爱冒油有黑头怎么办？皮肤很敏感怎么办？随着咨询的人越来越多，她成交的客户也越来越多，很快升级成了总代。


听完这个案例，相信你也会有相同的感受：同样都是售卖产品，一个人有自己鲜明的 IP 特色，而另一个人只是单纯推销卖货，你肯定更倾向于向第 1 种人去购买产品。所以我们说一个人的个人品牌价值千万就是这样的道理，当今时代，每一个想要做副业或者创业的人，都必须要学会去打造自己的个人 IP。


那么我们该如何塑造个人品牌，并且让别人被我们的品牌所吸引，实现被动成交呢？接下来会给大家分享三个重点。


第一个重点，让别人知道你是做什么的。


要想让别人知道你是做什么的，你需要通过以下两点来操作：


首先你必须要有一个明确的 IP 人设，前面我们讲了优势定位，接下来你就可以根据你的优势定位来打造你的 IP 人设。那么如何打造 IP 人设呢？可以分为 3 个步骤：


第一步：根据你的优势定位，取一个好名字


对于起名字我的建议是：品类+真名（或昵称），比如：美妆测评师阿雅、绘本桐妈、设计师阿爽、创业导师万小傲等等。这样取名字不仅好传播，还可以帮你快速锁定你的目标客户。


然后是第二步：写一个好的自我简介


由于名字字数比较少，能体现的信息有限，此时你就可以用一个简短的自我介绍来补充名字所不能体现的信息，一个好的自我介绍需要包含以下 2 点内容：


1、你的标签：标签是基于你的领域所提取的关键词，比如你所在的领域是美妆测评，但是美妆测评这个范围比较广，那么你的标签里就可以加入一些具体的关键词，比如科学护肤、平价美妆种草、网红美妆避坑、口红试色等等；


2、你的优势：要体现你和别人不一样的地方，比如李佳琦，他的优势就是：涂口红世界纪录保持着，战胜马云的口红一哥。作为普通人，你可能没有如此亮眼的标签，但只要有温度，有真情实感，或者有个性，依然可以成为你的优势，你只需要把它提炼出来就可以了。


通过前两步，你就会得出一个明确的 IP 人设，比如以我为例，我的 IP 人设就是：


【名字】：辣妈直升机创始人万小傲


【标签】：副业咨询、副业实操、创业导师、IP 孵化


【优势】：成功孵化 300 个宝妈 IP，帮助她们轻松创业，实现左手带娃右手养家


最后是第三步：在你的 IP 矩阵里不停地去重复


名字和介绍确定后，接下来你要做的事情就是不停地去重复它，尤其是在你还没有太多粉丝的时候，只有不断的重复才能让你的用户形成长久且深刻的印象。


比如你新加入一个社群，就可以积极的把你的自我介绍发到群里，吸引你的目标用户；


比如你对外展示的所有的自媒体，包括图文类和视频类，名字和介绍都要完善，最好都是使用你定好的这个名字和介绍；


比如你做短视频内容，你就可以在你的内容开头和结尾处，重复你的名字和介绍，抖音红人设计师阿爽，在早期自己的视频结尾处就会提到：「我是设计师阿爽，爱设计超过爱男人」，听起来既个性又让人印象深刻。


要想让别人知道你是做什么的，除了 IP 人设的打造，其次就是对外展示你的专业。比如根据你定位的领域，你可以通过短视频账号、朋友圈、直播分享、社群等，持续性输出这个领域的干货知识。


可能有宝妈会有这样的疑惑：老师，我的专业度还不强怎么办？这个领域有那么多的大佬，我的内容真的有人看吗？


首先千万不要说到「专业」这个词就退避三舍，觉得自己不行。其实输出知识也是一个学习的过程，当你觉得自己专业度还不够时，就更加需要去多看相关领域的专业书籍，学习一些专业的课程，多和你相同领域的人去交流，这就是一个学习输入知识的过程。


而当你学习输入了这些知识后，你可以通过自己的理解，把这些内容在你自己的平台输出表达出来。比如你今天看了一本专业的书籍，从里面学会了一个对你非常有用的方法，那你也可以把这个方法通过朋友圈或者短视频的形式发布出去。这些并不需要多么高大上的理论，你要做的就是把觉得对你有用的知识或者方法结合自己的实践和感受表达出来就好，越真实越有用。


这样日积月累之后，你的个人品牌就会慢慢的沉淀出来，当然这个事情需要你坚持去做，只有你坚持下去，才能让别人对你是做什么的产生记忆。


然后我们来说第二个重点：让别人知道你做的怎么样。


让别人了解你的实力最好的展示方式就是：客户证言。也就是我们所说的「消费者现身说法」。好的客户证言，还能激发更强的信任感和购买欲。


可能你会问了，老师我现在才刚开始去做个人品牌，还没有付费的客户，怎么样去证明我的服务结果呢？


你可以从身边找一些同学朋友，然后把你的产品给他们去试用，在这个过程中用你的专业知识去服务他们，我相信只要你的服务到位，给到的专业的帮助价值到位，让他们拿到了满意的结果，那他们一定也会愿意去写一些好评反馈。


那么你要做的就是下意识的去搜集这些好评反馈，并且对外展示出来，这样子才能够让初次了解你的陌生人，知道你做的怎么样。


最后我们来说第三个重点，让别人知道为什么他要选择你。


如何让别人选择你呢？这里面有两个小技巧：


第 1 个技巧：权威展示。既然要让别人选择你，你总得有两把刷子吧。比如你在相关领域得过的奖、拿到过的证书、你的作品、你研究该领域的时长、曾经服务过的知名的人物，受邀参加过的一些活动，这些都是很好的权威展示。你只需要从中找出最值得宣发的点，摆好事实就行了，大多数人还是很容易被权威所吸引的。


第 2 个技巧：个性展示。酒香不怕巷子深的时代已经过去了，身处快速变化的互联网时代，你更加需要适当频繁大胆地去展示表达自己、展示自己的个性。这里给你四点建议：


一、展示你独特的见解：学会独立思考，学会表达你在这个行业和领域里的一些独特见解和观点；


二、展示你的价值观：两个人一起相处，最重要的就是志趣相投，三观相合，多输出与你目标群体相同的价值观，势必更容易吸引到相同领域的人，比如我们你是一个妈妈群体，多输出一些帮助宝妈开辟第二职业，实现自身价值的内容一定会很受妈妈全体的欢迎。


三、学会讲故事：把你的创业经历、产品经历、或者记忆深刻的经历以故事的框架呈现出来，加入一点点关键的血肉，那么这些经历就变得更能打动人了，这样大家才会知道创业的不易和产品的优秀之处。


四、展示你的个人调性：你的审美、口头禅、个人性格这些属于你独特的个性都可以进行适当展示。


有了这些展示，在潜在客户的印象里，你就是一个真实立体有血有肉的真人，这样才会对你更感兴趣，愿意去多接触。之所以叫做个人品牌，就是它具备了很强的人物属性，需要有情感，有温度，有性格，做到真正的有料有趣。


这就好比你去喝咖啡，两家咖啡馆的品质完全相同，一家是中规中矩的咖啡吧；另一家咖啡馆呢，店老板把自己做咖啡独到的心得、为什么想开咖啡馆的故事展示了出来，而且专门做了一个店主自己旅游全球搜罗的关于咖啡的小知识，小故事和照片展示墙，而且她自己的穿衣风格和整个店的装修风格，以及店内音乐融为一体，有她独特的调调，显而易见你肯定更愿意选择第二家咖啡馆。


好，课程到这里就要接近尾声了，我来为你总结一下，这节课我们学到了个人品牌的重要性，以及如何打造个人品牌的三个重要因素：


第一，让别人知道你是做什么的，打造你的个人 IP 人设，然后不断去输出你的专业知识。


第二，让别人知道你做的怎么样，通过你服务的成功的案例和好的结果反馈去展示。


第三，要让别人知道，他为什么要选择你。要学会展示自己在这个专业领域的权威，以及你自己的独特的个性。


相信只要你做到了以上 3 点，那么一定会有人主动来链接你，让你实现被动成交，再也不用那么费劲地去推销产品。


今天的课程就到这里，今天的作业是：发一条你的专业知识朋友圈，告诉大家你是做什么的吧。
