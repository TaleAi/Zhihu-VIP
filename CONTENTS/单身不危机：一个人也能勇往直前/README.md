## 简介
单身，也是人生中极为重要的一个阶段。面对未来可能的亲密关系、家庭、婚姻、孩子，抑或永恒的孤独，你，准备好了吗？本门课程从心理学出发，洞察单身人群的心理特点；从关系着手，讨论亲密关系里的边界。


单身，已经越来越成为新时代的主流，一个人的我们，也可以过得很好。虽然单身，却没有丧失爱的能力，哪怕只有自己，也要直面生活、努力前进。


一个人，只有将自己的生活安排好了，才可能将快乐和充实带给别人。所以单身其实是生活的试炼，把自己所有的生存经验整合重组，找到自我，完善自我。


一个人，我们需要面对的问题很多，比如保有自己爱的能力，比如清楚自己与他人关系的边界，比如给自己一颗定心丸，去面对生活的不确定和永恒的孤独。


叔本华说，一个人只有独处的时候，才能完全成为他自己。


谁要是不热爱独处，那他就是不热爱自由。因为只有当一个人独处的时候，他才是自由的。自由，意味着开放的自我和未来的无限可能。这是一堂特殊的单身课，探索单身的边界，涉及关系、情感、婚姻、孤独等人生重要议题，指向未来更好的自我。


来，一起开启有品质的单身生活，拥抱未来更好的自己吧！


![](https://pic2.zhimg.com/v2-fcf32ac4130700b4ccba9f00daebb7b3.jpg)


**-你将收获：**


无论单身与否，教你面对与人相处时的关系边界问题，与他人建立良性互动；


教你在走入亲密关系之前，学会爱自己，爱他人，培养自己爱的能力；


看清现实，排解焦虑，学会把握当下，去做，然后去承担；


从心理学层面深度剖析孤独，直面我们独自存在的孤独；


一个人生活的时候，你需要注意的那些：一人食、锻炼、穿衣打扮……


![](https://pic4.zhimg.com/v2-50c624f9469dc2f68ca0f24bfb1e3ea2.jpg)


无论是期待着未来进入稳定的亲密关系；


还是已经抱有独身主义的偏向；


或者更多的是对未来的人生不确定性的担忧；


或者还想趁着单身完成一个人的心愿……


你都可以从这节课出发，去尝试探索生活的边界。
