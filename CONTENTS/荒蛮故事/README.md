## 简介
一组荒诞不经的犯罪故事集，逼仄之地的人性大爆炸。


黑色故事，看起来荒诞，但是又与现实有着巨大的联系。


夸张的手法，写尽人的本性和底层的残酷。


**《巴山的故事》**


大家都叫我傻瓜，因为我是一名天生的唐氏综合征患者。我也因此常被嘲笑。后来我爱上了一个姑娘，我以为她美丽善良，却想不到她骗光了我所有的钱。在去追她的路上我又遭遇了意外，我在一道闪电后失去了意识，醒来后却和那个霸凌了我七年的男人互换了身体…


**《杀人计划》**


为了母亲我要杀掉我父亲的情妇，可在执行杀人计划时我屡屡挫败，最后只得将她绑到了偏远的别墅中。在那我与她相处了三日，我发现我对她的感情悄然起了变化，她也远不及我想象的那般简单…


**《心仪》**


我是一名 33 岁的大龄剩女，孑然一身又得了绝症。为了能最后挥霍一把，我卷了公司的钱跑路。却没想到会在生命的倒计时爱上一名警察。我对他的感情是真的，但是在他面前我只有谎言。
