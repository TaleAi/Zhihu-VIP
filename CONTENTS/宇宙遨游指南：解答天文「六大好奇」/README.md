## 简介
本专栏由国家天文台宇宙暗物质与暗能量研究团组首席科学家陈学雷老师主讲，通过对星体、星系、黑洞、引力波等「冷知识」的解读，带你走入奇妙的天文世界。


  



我们为什么要了解宇宙？


1970 年，赞比亚修女 Mary Jucunda 曾致信美国航空航天局 Ernst Stuhlinger 博士，问了同样的问题。博士的回信很长，但其中有一句话让人印象深刻：**太空探索将给人类提供一面审视自己的镜子。**


  



但是，我们有多了解宇宙呢？


为什么会有白天和黑夜？


宇宙是怎么诞生的？


火星上真的能种土豆吗？


有没有地外文明存在？


我们对于天文其实还一无所知，只能从一部部科幻电影中找寻答案。


这次我们邀请了国家天文台宇宙暗物质与暗能量研究团组首席科学家，博士生导师陈学雷带我们一同走进天文的世界。作为天文学家的他，将从星体、星系、宇宙等方面全方位为我们梳理你最应该了解的天文知识，解答你最好奇的天文问题。从入门到进阶，「宇宙漫游指南」带你探索宇宙的第一站。


  



你将收获

* 1 次奇妙的太空之旅
