## 简介
双向奔赴、暗恋成真的甜文合集。


你是我明目张胆的偏爱，众所周知的私心。


即便知道自己会输，但我依然会坚定地选择你，不顾一切地奔向你。


因为遇见你，我变得自卑、偏执，也因为遇见你，我变得勇敢、温柔。


最难的不过是浪子回头，最幸运不过是你也爱我。


在这风华正茂的年纪，我们都是为爱冲锋的勇士。


世间繁花无数，唯有你是我的玫瑰。
