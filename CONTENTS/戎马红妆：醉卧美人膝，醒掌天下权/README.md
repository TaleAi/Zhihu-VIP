## 简介
这是一个古言权谋专栏。且看废后、女将军、不受宠的皇女，如何运筹帷幄，问鼎天下。



**- 帝隐歌 -**


大夏九年，新帝李乘风登基，册封苏行云为皇后，同时册封自己曾经的恋人霍隐歌为郡主。


二女相争，一个赢得了皇后宝座，另一个得了郡主封号也并不亏本。李乘风很是得意，觉得自己是个厚道人。


可是众人不知，当霍隐歌沉默着接过那道圣旨，他们的命运就此天翻地覆。


  



**- 主沉浮 -**


一朝穿越，陆沉变成了不受宠的二皇女。


父君早死，女帝不爱，后院内宅里，正君和侧君撕扯得不可开交。


最要命的是，大绥王朝已经走到末路，外有北疆蛮族虎视眈眈，内有旱灾流民暴乱。


谁要和这个封建王朝一起陪葬啊？！


陆沉愤怒了！


挽狂澜于既倒，扶大厦于将倾，或许也只有新任女帝，才能为大绥带来一线生机。
