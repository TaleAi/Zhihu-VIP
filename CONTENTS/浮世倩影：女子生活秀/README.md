## 简介
女子之力，遇强更强。


**《胖女孩也有春天》**


外貌焦虑的女主为了减肥，选择了极端的方法——催吐。


当「催吐」的习惯行为变成身体的记忆，她再也停不下来，在往鬼门关去的路上越走越远。


其实减肥，难，也不难。


难的，是我们的执行力在欲望面前会变得脆弱，我们不够自信，内心深处就不相信自己能通过运动和膳食均衡瘦下来。


哪怕一开始确实有运动+合理饮食，可一旦陷入平台期就开始焦虑、惶恐，开始怀疑自己。


陷入死循环之后的我们，很容易走进能快速看见成果的极端误区。


那个时候我们不会去过度关注外在的数值，更多的，是找到和自己和平相处的方式，用心去感受身边一切的美好。


  



**《单亲妈妈职场大乱斗》**


因原生家庭的不幸，童年的阴影，导致追究欲扭曲。


以致踏上社会后近乎变态的要强、执拗，成为一名奋斗 X。


丈夫去世后，独立生下孩子。


为避免孩子重蹈她的覆辙，更加拼命，身心俱疲又遇重疾……


直到有一天，默默照顾的他那个男人打动了她，生病期间反复思考前半生，获得平安喜乐……
