## 简介
心思缜密的作精女主主动出击，步步为营，花式攻略各路高冷霸气的禁欲系男神。


**白桃乌龙**


老公出轨白月光，我果断离婚走人。走出民政局，却发现自己怀了娃。


够狗血吗？还远远不够。


去医院产检，医生竟是前男友那朵高岭之花。


他一口一个要对我的孩子负责，还用实际行动把我宠上了天。


他到底想干吗？在线等，挺急的。


 


**攻略那个医生**


我俩第一次见面不太光彩。


我是酒后吞灯泡被送医院的憨憨，他是帅到叫人垂涎的急诊医生。


我关注了他的知乎，发现他刚回答一个问题——「你身边有哪些人硬生生活成了段子？」


我把他当未来老公，他竟然把我当段子？
