## 简介
科学饮食，拥有好身材。


你还在节食减肥吗？还在为了减肥而不吃早餐吗？


主食应该怎么吃？热量高的食物就一定会吃胖吗？


减肥也能吃零食吗？你爱吃的糖都有哪些危害？


……


不少人对饮食和瘦身之间的关系有误解，以为盲目节食就能达到减肥的目的，却不知这样做其实有损身体健康。


本课程特邀国家一级营养师张淋琳，为想要科学瘦身的你带来科学的饮食管理指导，针对不同饮食与肥胖的关系，全面讲解正确的饮食方法，结合中国传统医学理论，教你成为真正的易瘦体质。


同时，本课程明确在不同时期的饮食管理侧重点，关心你的营养健康，科学指导瘦身过程中的正确饮食。


另外，本课程也适合备孕期及产后妈妈学习，为你消除女性的「肥胖焦虑」，拥有全面科学的营养方案。
