## 5.被神化的蔬菜，吃素就能减肥吗？
**一、开篇**


今天这节课我们来说说蔬菜。


你可能会觉得说，蔬菜有什么好说的，多吃蔬菜健康、多吃菜少吃肉就行了。


实际上蔬菜不是你想象的那么简单的，相反，蔬菜的问题还非常多，而且现在我们身体出现这么多问题，很大程度上与我们没有吃好蔬菜有关。这节课我们就来说一说，蔬菜究竟要怎么吃。


**二、认识青菜**


首先我们要理顺下，到底什么是蔬菜？


通俗地讲，蔬菜是指我们吃的植物性食物中，除了粮食、水果和植物油脂以外的食物，都是蔬菜。粮食、油脂我们前面讲过，水果我们下节讲，这节我们主要讲蔬菜。


当然，粮食、油脂、蔬菜、水果这些植物类食物之间是有交叉的，就像土豆，原来就是属于蔬菜，属于我们日常吃的菜，但是现在国家已经把土豆归入主粮了，也算粮食。还有一些既是蔬菜也是水果，比如西红柿、圣女果、黄瓜，我更倾向于把它们当水果，尤其对肥胖人群来讲，为什么？我们在下节讲水果时再讲，这节我们主要讲蔬菜。


蔬菜的问题比较严重，严重程度远远超过你的想象。问题主要集中在两个方面：


1、蔬菜对我们健康的价值被严重忽视；2、对于如何吃蔬菜存在非常多的误区。


我们先来看看蔬菜的营养价值。我前面讲过，维持我们身体健康的营养物质有 7 大类：碳水化合物、蛋白质、脂肪、维生素、矿物质、膳食纤维和水。


**蔬菜能给我们补充的主要是维生素、矿物质和膳食纤维，蔬菜这一类食材就能满足人体三种营养素的需求，所以，对我们的身体健康的意义非常大。**


有人可能会有疑问，说水果也能满足维生素、矿物质和膳食纤维的需要，这没错，水果也能满足人体对这 3 中营养素的需要，但是，水果含糖量太大，而糖对我们身体的危害非常大，这我们在下节讲水果的时候会详细讲，这里大家只要记住蔬菜才是补充这三种营养素的最优选择就行了。


通常来说，人们对蔬菜没那么喜爱，蔬菜没有肉食香，没有水果甜，吃蔬菜比较难获得愉悦感，因此，健康意识薄弱的人很少有喜欢蔬菜的，最明显的例子就是孩子，因为小，不懂健康，几乎完全按本能选择食物，所以大部分孩子不爱吃蔬菜。


不爱吃蔬菜、忽视蔬菜价值的后果，给大家介绍一个名词「隐形饥饿」。就是看似你酒足饭饱，一日三餐都是吃的饱饱的，但实际上你的身体是缺乏营养的、是饥饿的，对营养物质的饥饿是被掩盖了，所以叫「隐性饥饿」，而这些缺乏的营养素主要就是指维生素、矿物质和膳食纤维，而补充这 3 种营养素，正是蔬菜的强项。


维生素、矿物质和膳食纤维这三种营养到底有多重要呢？


我们就举一个维生素 C 的例子，维生素 C 的发现就是从缺乏维生素 C 造成的严重后果开始的。


在大航海时代，西方人在舰船上最忌惮的疾病就是坏血病。不管是哥伦布还是麦哲伦，躲过风暴礁石或许不难，然而再高超的航海技术也阻挡不了海员因为坏血病大批死亡。直到后来，英国一位海军医生发现柑橘或柠檬可以治疗坏血病，人们这才逐渐意识到新鲜水果蔬菜的重要性。


此后又经过多年的探索，维生素 C 这种物质才逐渐浮出水面，而它的另外一个名字就叫做「抗坏血酸」。


而现在各种疾病高发，高血压、高血脂、高血糖、心脏病、脑中风、各种癌症、白血病、痛风、风湿关节炎等等，有很大一部分原因就是因为我们的营养不均衡，碳水化合物、脂肪、蛋白质摄入过多，而维生素、矿物质、膳食纤维摄入严重不足。


实际上，如果营养问题解决了，不用吃药，就通过饮食调理，80% 的疾病是可以避免的。


**三、如何正确吃菜？**


蔬菜该怎么吃才健康呢？


要说清这个问题，我们有必要先把蔬菜分一下类，不同类别蔬菜的最佳食用方法有所差别。当然，我们的分类方法主要是从吃的角度来分的，搞农业、搞蔬菜研究的，他们有更专业更精细的划分方法。


从消费者的角度，蔬菜一般分这 7 类：


第 1 类，根茎类蔬菜，比如萝卜、马铃薯、藕、甘薯、山药、芋头、茭白、洋葱、生姜、大蒜、蒜薹、大葱、韭黄等，我们吃的主要是这些植物的根部或者杆茎部。


第 2 类，瓜果类蔬菜，比如冬瓜、南瓜、西葫芦、丝瓜、黄瓜、茄子、西红柿、苦瓜、辣椒等，我们吃的主要是植物孕育种子的果子，这些被当做蔬菜的果子，大多数都叫瓜，像西葫芦，有的地方就把它叫茄瓜。


第 3 类，叶子类蔬菜，只要是吃的是叶子的，都算这一类。以绿叶蔬菜为主，包括白菜、菠菜、油菜、卷心菜、苋菜、韭菜、香菜、芥菜、芥兰、茴香等等，当然也有其他颜色的，比如苋菜，是红色的。


第 4 类，花菜类，这类蔬菜以花、肥大的花茎或花球为产品器官，如花椰菜、金针菜、青花菜、紫菜蔓、芥蓝等。


第 5 类，嫩种子类蔬菜，成熟的种子算粮食，就不算蔬菜了。嫩种子类蔬菜包括各种嫩的种子和豆荚，比如菜豆（含矮生菜豆、蔓生菜豆变种蔬菜）、豇豆、豌豆、蚕豆、毛豆（即大豆）、扁豆、刀豆等，还有嫩玉米。


第 6 类，芽苗类，最常见的就是各种豆芽。近些年流行起来各种苗菜，比如豌豆苗、花生苗、穿心莲苗等等，都属于芽苗类。


第 7 类，菌菇类，就是由各种菌生长成的食物，包括各种蘑菇，银耳、木耳，以及顶级食材黑松露等等。


**1、烹饪方法**


至于蔬菜怎么吃才健康，不同种类的蔬菜最适合的烹饪方式不同。


**像叶菜类、花菜类以及芽苗类**，因为富含维生素，而维生素受热容易分解，所以这类蔬菜营养成分损失最小的食用方式就是生吃，能凉拌吃最好，不能生吃的，就快炒，这几类蔬菜最不适合长时间炖煮。


**而菌菇类和部分根茎类**，就不怕长时间高温，适合长时间炖煮。简单概括来讲，只要是绿颜色的蔬菜，一般都不适合长时间炖煮，最好的食用方法是生吃，然后是快炒。


**其他类蔬菜**烹饪方式几乎不受限制，可按自己的饮食习惯来安排。


这里面还有一点需要说明，富含淀粉的蔬菜，主要是部分根茎类蔬菜和瓜果类蔬菜，比如土豆、山药、芋头、玉米、南瓜、红薯等，虽然是蔬菜，但我的建议是把它们当成主食来对待，比如你吃土豆烧牛肉，我建议你把里面的土豆等同于米饭或者馒头，我们前面讲主食的时候减过，要控制主食的量，尤其是精白米精白面，如果你吃了一碗土豆，那就别吃主食了，如果你是肥胖人群，就更要限制这些富含淀粉的蔬菜摄入。


**2、蔬菜的保质期的问题**


这里是指做熟的菜，也就是剩菜能不能是的问题。这个误解比较多，搞不清楚的话，一刀切的吃或者不吃都不对。


能不能吃，也要看是什么菜：


**主食类的食物**，比如馒头、米饭、饼这些，一次吃不完，放冰箱，冷藏的话，放 1-2 天都没问题，下次吃的时候只要充分加热就完全可以吃。


**肉类的菜**，如果是凉拌的肉，即使放冰箱冷藏，也不要超过 5 小时，也就是最多这顿吃不完，下一顿还能吃，再吃不完就只能倒掉了。而如果是炒煮过的肉菜，如果冷藏，存放一天也没问题，而如果冷冻，放一周以内也没问题。之所以炒煮过的肉菜能存放的时间这么长，就是因为你再次吃的时候还有个加热环节，会杀死可能会产生的细菌，比较安全。


**至于叶菜类、花菜类和芽苗类的菜，是不能吃剩菜的**，而且建议做熟后半个小时内要吃完，因为叶子类的蔬菜，特别容易腐败变质，只要超过半个小时，就可能变质和产生细菌。所以，一旦发现菜多了，就优先吃绿叶类菜，因为这些菜不能保存。


**至于其他根茎类和菌菇类的蔬菜**，也是冰箱冷藏不要超过 5 小时，一旦超过就要倒掉。


有些人可能非常重视健康，外面吃饭从不打包，在家也是每次吃饭剩下的统统倒掉，我觉得这是浪费，虽然中国现在富裕了，但我们也尽量不要浪费食物，确实不能再吃的，我们果断的倒掉，还能吃的，只要保存得当是可以再次吃的。


**3、蔬菜的摄入量的问题，一天吃多少合适？**


按照「中国居民平衡膳食宝塔」建议，我国每人每天应吃 200-400 克水果和 300-500 克蔬菜。也就是每人每天 1 斤左右的蔬菜，我觉得国家的这个标准是最低标准，远远不够，同时按质量来衡量多少，大家很难操作，因为很少有人吃饭时会拿称来称重量，所以我给大家一个简单的衡量蔬菜摄入多少的方法，就是每顿饭中蔬菜要占一半，另一半是肉蛋奶和主食，照这个量，如果蔬菜的种类和品质还不错，基本上维生素、矿物质和膳食纤维就不缺了。


关于蔬菜量的问题，还有一点需要强调，就是蔬菜的种类问题，这个问题很严重。蔬菜的种类比人们想象的多得多。在中国，人们常吃的蔬菜就有 100 多种，但很多人家里总吃那几样会做的蔬菜，见到新奇的蔬菜因不知如何做而不敢买，久而久之，商贩们常卖的蔬菜也就那么十几种了。


由于每类蔬菜中所含的营养物质成分不一，比如菠菜、卷心菜、空心菜等绿叶菜富含 B 族维生素、维生素 C 和多种无机盐，胡萝卜、辣椒等富含维生素 C 和胡萝卜素，只有吃得种类多，才能营养均衡。


但很多家庭常吃的蔬菜不超过 10 种，这是不对的，吃蔬菜不仅要保证数量，更要保证绿叶菜、茄果类、薯芋类、白菜类、瓜类、根茎类等各类蔬菜都要摄取到，我给的建议是，不要求每天吃多少种，放在一个月之内，我们每个人吃过的蔬菜要超过 20 种。


关于蔬菜要多样性，还有一个问题，就是某一天因为工作或者其他原因，一整天几乎没有吃蔬菜，有问题没？是不是每顿饭都得要蔬菜多样性？


这个没有必要，前几次课我讲过，营养问题一定是有个时间持续性的问题，一次营养不够、一次不均衡，都不会有问题的。就比如一日三餐，很多地方的饮食习惯是一日三餐中有一餐是正餐，很丰盛，其他两餐是简餐，非常简单，这都是没问题的。而且不仅是一天之内没问题，持续较长的时间，甚至几天，都影响不是太大，比如几天时间，只有馒头和白菜，有问题没？


没有，只要几天过后吃的丰富一些，营养再均衡起来就行了。所以，这里教大家一个生活技巧，在你因为一些特殊原因，吃饭不太方便的时候，比如到一个陌生的地方，水土不服或者不习惯当地饮食，这时候方便面就是一个最好选择，大家提到方便面时，立马就认为它是不健康的，确实，跟正常饮食比起来，方便面里就是碳水化合物和脂肪，其他营养素都非常非常少，确实不太健康，但是，在你出差或者出国，饮食不便时，方便面就是最安全的食物，缺乏维生素、矿物质、膳食纤维这些营养素，回来再补充就行了。


**4、蔬菜的农药残留的问题**


现在市场上有不少去农药残留的商品，有药水还有机器，都说能去除农残，所以也导致大家担心我们吃的蔬菜到底有没有农残的问题。


如果是 10 年前，可能这些担心是有必要的，因为那时新闻报道过蔬菜农残中毒的事件，但现在大家根本不用担心这个问题，只要是从正规的超市或者农贸市场买的蔬菜，基本上没啥问题，因为我们国家的管理越来越到位了，高毒的农药已经不让生产了，现在用的农药基本都是低毒和易降解的，而且国家管理也严格，基本不用再担心农残了，至于市场上的药水和机器，大部分是在做营销，不用太在意。


当然，农残不用太担心了，但蔬菜也是有品质差别的，大家可能听说过几个概念：无公害蔬菜、绿色蔬菜、有机蔬菜，很多人搞不清楚，这里我给大家梳理下：


无公害蔬菜，是以规范农业生产、保障基本安全、满足大众消费为目标，达到中国普通蔬菜质量水平，是最基本的市场准入条件；


绿色蔬菜，达到了国家的更高标准，满足人们对食品质量更高层次的需求，达到发达国家普通食品质量水平。绿色蔬菜分为 A 级和 AA 级两类，AA 级等同于有机蔬菜，通常绿色蔬菜指 A 级。


有机蔬菜，是蔬菜生产过程中完全不使用农药、化肥、除草剂、生长调节剂等化学物质，不使用基因工程技术，是满足更高层次的安全消费。


所以，可以把蔬菜分为 3 个档次，即无公害蔬菜是基本档次，A 级绿色蔬菜是第 2 档次，AA 级绿色蔬菜和有机蔬菜为最高档次。大家在挑选蔬菜时，经济条件好的，尽量选择有机的和 AA 绿色的，没有条件的，选无公害的就行，实际上现在市场上的蔬菜基本都达到了无公害的标准，否则是不允许上市销售的。


**四、关于蔬菜常见的错误认知**


**1、维生素片可以代替蔬菜**


因为工作或者其他原因，没条件吃蔬菜，吃维生素药片行不行？


这个问题非常普遍，但是不能简单回答行还是不行。


如果你有条件通过蔬菜补充维生素等，但是你懒或者为了满足口感食欲不吃蔬菜，想靠吃药片来补回来，这样就不行，因为通过蔬菜水果补充和通过药片补充，效果差别很大。但如果确实不方便，平时只能吃馒头和咸菜，这时吃点药片补充维生素，也是可以接受的，效果虽然差些总比没有强啊。


为什么通过食物和通过药片补充维生素差别很大呢？


我们举个非常具体的例子，现在广告做的很猛的一个产品，是一个维生素 C 药片，说它这个维生素 C 是从针叶樱桃里提取的，是天然的，所以价格将近 100 块钱一瓶，说医院开的是人工合成的，所以才几块钱一瓶。因为消费者不懂，所以就觉得天然的比人工合成的好，所以反而很多人去买 100 元的。


实际上，只要做成药片，所有的维生素 C 几乎没有任何差别。100 元的维生素 C 药片和几块钱的维生素 C 药片，对我们身体的作用几乎一模一样。这是商家利用了「天然维生素」这个概念，是一种营销手段。


不过，直接通过吃针叶樱桃补充维生素 C 和通过吃药片补充维生素 C，我们身体的吸收情况是有很大差别的，确实是通过食物补比通过药片补，要好很多。


科学家做过实验，分别通过橙汁和通过药片，补充相同数量的维生素 C，结果就发现，天然来源的维生素 C 在我们体内停留的时间更长，而药片来源的维生素 C 很快就代谢掉了，天然来源的维生素具有「缓释」功能。


「缓释」能带来 2 点好处：


1、在体内停留时间长，利用率就高；


2、在身体中停留的时间短，转了一圈就随着尿液排出去，会有一些副作用。


维生素 C 没有明显毒性，稍稍过量也并无什么风险，然而事实上，确有不少人因为长期补充维生素导致肾结石或其他肾功能疾病，这正是代谢过快造成的后果。


而且，喝橙汁的时候，不仅摄入了维生素 C，还会摄入黄酮类物质，而黄酮类物质是被证明能促进人体对维生素的吸收的，通常情况下，通过橙汁补充维生素 C 的吸收率比单纯补充维 C 片要高出 35%。这就是我们鼓励通过食物补充营养素，而不鼓励通过吃药片来补充营养素的根本原因。


**2、吃素减肥**


你肥不肥胖，与吃不吃素一毛钱关系没有，从自然界的例子来看，并不一定吃肉的就胖，吃素的就瘦，河马胖得很，是吃草的。拿我们人类举例，你去寺庙里看看，那些吃素的和尚，绝对是胖和尚比瘦和尚多。


我经常给人讲，常吃肉夹馍会发胖，但发胖的因素是那个馍，而不是馍里面的肉。我在第一节课里已经讲过肥胖的原理，我们之所以会发胖，主要就是因为血糖，血糖高胰岛素就分泌，胰岛素水平高，身体就开始储存脂肪，所以，我们要减肥，主要的任务是如何控制住血糖，而血糖的高低，与吃素和吃荤几乎没有关系，而且，总体来讲，导致血糖波动的，素食里更多，反而肉食里能引起血糖波动的比较少。


所以要想靠吃素减肥，基本上都是一厢情愿，根本不可能成功，而且，胡乱吃素减肥，很容易导致营养不良，能用吃素来减肥的，基本上不懂营养学，不可能做到营养均衡，长期吃素，一定会造成蛋白质和脂肪的长期摄入不足，最终会导致身体抵抗能力严重受损，有很大可能会造成严重后果。


所以不要盲目吃素了，你吃素不仅不能减肥，还有可能吃出各种健康问题，得不偿失。

**凉拌萝卜缨（补维 C 蔬菜佳品）**


配料：白萝卜缨、大蒜、小红椒、醋、生抽、香油、蜂蜜。


方法：


1、将白萝卜缨洗净，大蒜切碎，小红尖椒洗净切小段；


2、烧沸水，将洗净的白萝卜缨放入焯烫 15－20 秒后捞出，控去水分；


3、待萝卜缨稍晾凉些，切成小段，然后将所有材料和调料混合拌匀即可；


4、萝卜缨味道有点辛辣，带点淡淡的苦味，所以吃的时候可以适量加点蜂蜜，如果喜欢这种味道的可以不加。
