## 简介
《鬼谷子的局》堪称当代现象级的历史畅销小说，本音频课程，从解读小说出发，导引您穿越至先秦战国时代：一场场腥风血雨的战场，一幕幕惊心动魄的外交论辩，尽显王侯将相、英雄圣贤之未了余情。通过先人们的权力角逐、智慧博弈，来反思当今各个领域之纵横捭阖！


《鬼谷子的局》的故事内容与立意紧扣当前的国家文化发展战略。故事以二千多年前的天下乱局为时代背景，以百家诸子为人物主角，讲述中国战国中期的有志之士如何思考、争鸣、行动以寻求乱世治理方案的故事。


诸子百家不但为二千多年来中国风行不衰的文化传统奠定了的根基，设定了高度，且使中国成为世界三大思想策动力的源泉之一。


故事紧扣「天下（列国）」这个主题，全方位重构中国战国时代列国的风土民情，再现国家治理、军事外交、社会生活的方方面面，场面宏大，堪称史诗巨作。


《鬼谷子的局》以普世人性为内核，以人物托事件，以事件推动人物成长， 从细微处构建人物的精神内在，从而使人物在成长中栩栩如生，使情节在交错中环环相扣。
