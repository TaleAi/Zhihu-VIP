## 简介
生动展现唐朝三百年历史的风度与气质。


三百年的大唐天下，单就个体朝代而言，是中国历时最久的皇朝。这三百年的历史，之所以吸引人，更多的在于它的特异性。


如果走入唐朝，你也许会觉得像是到了外国，因为你的所见所闻，并不是我们熟悉的所谓的「典型的」中国人和中国文化。唐人的心态是开放的，眼光是世界性的，唐文化的特点是外向型的，是积极开拓的。


这一切，都让三百年的大唐成为后来中国人心目中最辉煌的一梦。


在这个梦想的天空中，闪烁着无数耀眼的星辰。


有一些一直闪耀到今天，如天纵英武的唐太宗，古今无二的女人武则天，英姿勃发的白袍将军薛仁贵，德才兼具的宰相兼神探狄仁杰；也有一些，似乎已经黯淡了，但当年却曾如流星划过天空，夺人眼目，像那经济天才、一代财神刘晏，会昌名相、贵族气度的李德裕……


由这些名字所串起来的唐代历史，无疑会是一部波澜壮阔的唐代政治军事历史。你将跟着这些名字，来一次惊心动魄的唐朝之旅。


本专栏不仅有宏大的历史叙事，又兼顾一个个真实生命的喜怒哀乐，甚至是所谓历史家不屑的琐碎细节。


现代人更关心的历史其实依然是人，是人性，是人心。历史就是由人创造的，就是由一个个有思想、有喜怒哀乐的人创造的。所以，读史就是读人，读人心。


此外，历史之所以有趣，并不在于它总是必然的，而在于它也总是充满了偶然性，带给人意外，带给人惊诧，更带给人深思。
