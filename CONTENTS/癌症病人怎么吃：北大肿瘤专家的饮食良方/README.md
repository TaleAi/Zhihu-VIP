## 简介
来自北京大学肿瘤医院主治医师献给癌症病人的专属饮食处方。正确的饮食习惯能让癌症患者保证手术获得最佳的效果，并在术后快速恢复并减少并发症率——手术前如何补充营养？术后不同时期，该怎么差别对待病人的饮食？如何用饮食帮助巩固药效，降低并发症？这个专栏，用通俗的语言、真实的病例，手把手教你如何从饮食上帮助癌症病人合理补充营养，提高身体免疫力，有效提高手术和化疗的治愈率。


每次去医院看病，不管头疼发热的小病，还是事关人身安危的重疾，医生都会叮嘱一句：「注意饮食」。


* 一般的忌辛辣、油腻，清淡饮食
* 禁食「发物」，否则会加重病情，短期内让伤口不易愈合
* 吃头孢药物绝对不可以饮酒

**更何况是癌症。**


癌症能够治愈。


患的是低死亡率癌症、发现时为早期、又有医术高超的医生、合理的术前准备与术后恢复……这些都能大大提高癌症的治愈率。但还有一点容易被忽略：癌症病人的饮食


——这可能是癌症患者，以及患者家属们唯一能下工夫的事情了。


**营养良好、体力和免疫力充足，才能对抗手术后感染，顺利走完化疗、放疗过程，度过漫长的追踪期，最终摆脱癌症。**


这份来自北京大学肿瘤医院主治医师献给癌症病人的专属饮食处方：


* 手术前吃什么能提高免疫力？
* 术后不同时期，该怎么差别对待病人的饮食？
* 如何用饮食帮助巩固药效，降低并发症？

**——针对术前、术后、放疗、化疗等疗法疗程以及不同癌症类型，提出以科学为本、简单明了的饮食指南。**


* 民间谣传的食谱，该不该信？
* 术后一定不能吃「发物」吗？
* 流食、软食、可溶性纤维食物？怎么补充营养最方便？

**——来自一线资深医生的经验分享，语言风趣幽默，案例生动鲜活，帮助你破除养生的迷雾，用科学知识帮助病人合理饮食、走向康复。**
