## 简介
剧情向古言专栏，又是为美好女主流泪的一天。


逆境之中的不负深情。


吾之良人，尔之卿卿。


因为有你，我即使身处困顿也不放弃心中温良；


因为有你，我即使千帆历尽也依旧怀有希望；


因为有你，我即使满目沧桑也能和歌而唱……


甜中带虐，虐中带甜，古言女主大赏，尽在此处。


《浮云散》


她是宫中三公主，六岁那年母妃薨逝，皇上将鸩酒喂到她嘴边想让她为母妃殉葬，十六岁那年太皇太后新丧，摄政王身穿孝服反了她弟弟成颢的江山，而她封后的那年只有十七岁，冬日暖阳天气甚好，她好似从过往的一场场梦魇中窥到了一丝天光……


《失东隅》


阿蛮从小就知道于桑和镇中其他人不一样，他冷僻孤傲脾气臭腿瘸嗓子还哑，可阿蛮怎么都想不到这样的于桑长大竟然会攀墙跳窗夜会佳人，更想不到自己有一日会哭着上了于家的花轿，走出小镇踏进京城，入了皇宫得见皇后，会怀疑于桑是不是个太监？！


《名伶》


京都最有名的戏院汇聚了天下最动人的嗓音，只是因为戏院里的小伶人若不能十三岁功成登台，最后只能是被灌下哑药死路一条。她三岁入戏院，白日唱着才子佳人梦，夜里却在恐惧下入睡，如此十年，她登台的第一曲便是唱给戏院的主人……笼中鸟为何歌唱，笼中鸟缘何被放，当往事成烟回首再看，唱戏的人终究成了戏中人。
