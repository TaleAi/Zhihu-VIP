## 简介
为你塑造有趣的灵魂，学会高效率的聊天沟通技巧，变身高情商的社交达人。


我们常说「好看的皮囊千篇一律，有趣的灵魂万里挑一。」


皮囊已然固定，灵魂尚可塑造。一个有趣的灵魂，最直接的表现就是高情商，会聊天。


千万不要小看「聊天」技能在生活中的作用，俗话说：「智商决定你的下限，情商决定你的上限。」高情商的人能够通过聊天交到朋友，达成交易，收获爱情。「不会说话，社交恐惧，尬聊高手」成为很多人的困扰。


集体活动总是插不上话，不能融入集体；


酒桌饭局闷头吃饭，接不上别人抛出来的话题；


喜欢的人一聊天就消失，尴尬又窘迫。


你想学会高效率的聊天沟通技巧，变身高情商的社交达人？


找对方法，勇敢尝试才是重点。跟着李骆老师学习沟通术，让你走到哪里都有朋友、搞定客户轻松聊天、撩妹撩汉不在话下。


本专栏将围绕职场、社交、生活、两性多领域话题，实景案例型课程，拒绝枯燥，轻松易学；24 个细分场景话术，深入浅出，即学即用，解决你的实战型困难；快速改善你的沟通思维逻辑，模板式语言教程，让你以不变应万变。
