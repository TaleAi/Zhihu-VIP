## 简介
从古希腊的古典时代到希腊化时代，涌现出一大批不仅在当时，也在千年之后大放光彩的人物，本课程用 8 节小课，在专业史实的基础上，力求用精心设计的场景和丰富的音效，为你讲透这一个个鲜活的人物的故事。

古希腊智者普罗泰戈拉有句名言：人是万物的尺度。现代西方人将自己的源头追溯到古希腊，正是因为古希腊追求人的价值。本节目精选古希腊文明史上那些如流星一样照耀夜空的人物，以一种轻松、幽默的方式呈现他们的故事，相信你会在他们的故事中获得启发和情感共鸣。
