##### **大咖评论**


丸山健太郎（日本咖啡教父）：


当各大便利店都以 100  

  

日元的价格销售咖啡时，消费者开始有了比较。这种良性的比较，是差别化的开始。我觉得中国是个竞争很激烈的国家，咖啡厅的数量很多，不出彩便很难生存下去，所以大家都去买很贵、很稀少的咖啡豆。


而大部分客人真正需要的是一杯普通好喝的咖啡，在中国，这一块的供给是缺失的。其实想要制作出一杯评分 85 分、普通好喝的咖啡也并非一件容易事。
