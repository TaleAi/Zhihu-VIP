## 简介
椒房锁明月，寒蝉泣夜凉。
鸾凤岂堪困，振翼上扶摇。


自立自爱古风大女主合集。


**《拂花枝》**


我是一个不受宠的皇后，满后宫为了争宠打破头的时候，我在想我儿子的那个亲爹，什么时候才能起兵造反，来京城和他亲哥掰腕子。


他们两兄弟鹬蚌相争，我这个渔翁在后面排队等得很急呀。


  



**《迟来雪》**


我虽然长得像已故皇后，但选秀被嫌弃，进宫被冷落，就在我安于现状的时候，皇帝找上我，远在边疆的二皇子也不放过我。  

他们都把我当替身，那我就只能搞事业了，恋爱脑？狗都不当！
