## 简介
贼老天，前一世她就是个孤儿，穿越后又被父母抛弃，被牙婆卖进王府做末等丫鬟，睡得比狗晚，起得比鸡早。好不容易熬到能出府的年纪，却又阴差阳错卷入皇子夺嫡大战，无法脱身……


穿越而来成了秦王府平平无奇的丫鬟，陌生的时空里，她牢记身份，每一走都走得格外谨慎。


本以为可以安稳待到「退休」出府，见义勇为救下在王府落水的皇长孙，她的人生天翻地覆。


皇长孙的意外落水和不久后的骤然离世、皇城里突如其来的疫病、太子妃的难产死亡······一桩桩看似意外、实则疑云重重的事件，似乎都跟她这小丫鬟扯上了关系。


为保命，她被迫卷入阴谋，深陷波谲云诡的夺嫡风暴中心。越演越烈的夺嫡大战中，究竟是太子能胜出，还是呼声渐高的大皇子高宏朗能胜出？此前她所经历的种种事，究竟是哪一方在背后搞鬼？


心惊胆战的日子里，她的感情也铁树开了花。


阴郁寡言、却又处处待她特别的秦王高偃，放浪形骸、却总在危机时刻护她周全的异性王胡元离，一心追求她的太子首席侍卫顾阳，一个比一个难以琢磨。他们究竟是喜欢她，还是利用提防她？她该如何选择？


她一个现代魂、古代壳的末等丫鬟，能在步步惊心的权谋争斗里保全自己，收获属于自己的幸福吗？
