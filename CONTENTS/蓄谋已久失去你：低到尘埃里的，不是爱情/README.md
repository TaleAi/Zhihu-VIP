## 简介
姜朵一门心思想得到迟倦的爱，哪怕卑微得好像他的私生饭也在所不惜。


迟倦心目中，女人分为三种：年抛、月抛、日抛。


好死不死，姜朵对上了他的胃口，成了他为数不多的「年抛」对象。


一年结束，迟倦轻轻松松地扔了这件叫「姜朵」的衣服，笑着、闹着，左拥右抱着。


姜朵也很给他面子，死缠烂打依依不舍，甚至还把他的衣服私藏起来，宛若疯狂的私生饭。


迟倦笑得很散漫，「游戏而已，你当真了？」


低到尘埃里的姜朵，矢志不渝的姜朵，也会放弃吗？
