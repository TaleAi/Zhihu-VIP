## 30.爪哇岛：爪哇和 Java 语言有什么关系呢？
今天要带你去一个海岛，这个岛叫做爪哇岛。让我们打开地图，找到一个叫印度尼西亚的国家了。印度尼西亚由大大小小很多岛屿组成，人们把它称作「千岛之国」，其实印度尼西亚的海岛总数超过一万七千个。在印度尼西亚的这些岛屿里，爪哇岛算不上大岛，论面积只能排在第五位，但它的人口最多，超过了 1.4 亿，相当于我们中国总人口的十分之一。爪哇岛是世界上人口最多的岛屿。


爪哇岛的英文名字是 Java。提到这个名字，世界上很多人首先想到的不是这个海岛，而是一种叫做 Java 的计算机语言。语言是我们人类互相沟通交流的一种方式，但是计算机听不懂我们人类的语言，我们为了给计算机下达命令，就设计出了计算机语言，让计算机能按照我们的命令来工作。


Java 语言的标志十分有意思，是一杯咖啡，你知道这是为什么吗？这是由于 Java 语言的一位创始人，名字叫詹姆斯·高斯林的加拿大科学家，他非常喜欢喝爪哇产的咖啡，所以在给这种语言正式命名时，他选择了「爪哇」这个单词，还把这种语言的标志设计成一杯热气腾腾的咖啡。后来，由于 Java 语言简单易学、功能强大，非常受计算机专业人员的欢迎。Java 标志上的爪哇咖啡也成为了世界上知名度最高的咖啡之一。


爪哇岛出产的咖啡品质非常高，味道香醇、口味独特，深受人们的喜爱。呢？那么为什么爪哇咖啡的品质这么好呢？我们告诉你啊。这与当地的气候关系很大。爪哇岛处于赤道多雨气候类型区，一年到头气温都比较高，且雨水丰沛。咖啡是典型的热带植物，种植咖啡，并将其作为饮品的文化从非洲传到了阿拉伯，后传到了欧洲。17 世纪，荷兰人来到爪哇岛，发现这里夏季气温很高，而且雨水很多，很适合咖啡的生长。于是他们就把咖啡引种到当地，结果这里种植出咖啡的产量和品质都非常好。到 20 世纪初，爪哇咖啡在全世界知名度已经很高了。


除了咖啡之外，荷兰殖民者统治爪哇岛的时期，他们强迫当地人大量种植甘蔗、橡胶树、茶叶等经济作物。并将岛上许多丰富的物产运回荷兰。荷兰人为了把这些东西都运出去，就在岛上修建了港口、铁路和公路。这些设施进一步带动了当地经济的发展。慢慢地，爪哇岛成了印度尼西亚经济最发达的地区，也就吸引来了很多人。刚才我们讲过，爪哇岛上的人口有 1.4 亿，这个数字占印度尼西亚总人口数的一半还多。


位于印度尼西亚的爪哇岛上不仅有印度尼西亚当地人，还有很多华人。这是为什么呢？原来，我国古代和近代一直都有「下南洋」的说法。这里的「南洋」是个比较模糊的地理范围，大体上是指今天的南亚和东南亚地区，包括越南、马来西亚、新加坡、菲律宾、印度尼西亚等国家。在我国古代，广东、福建一带不是经济发达的地区，那里的人们经常要为生活发愁。而且遇到改朝换代的时候会发生战争，人民的生活就更加困难。所以，有一些南方沿海的人们就坐上船，到海外寻找更安全、更适合于生活的地方。


到了清朝的中晚期，欧洲人来到了南亚和东南亚地区，在这里发展农业、修造建筑，特别需要劳动力，就开出了一些优惠的条件，吸引其他地方的人们到这里参与劳动。再加上当时的清王朝社会动荡、人民生存和发展的条件很差，所以从那时候起，大量华人来到以爪哇岛为代表的南亚地区工作，并从此在这些地方定居下来。这些华人把自己的文化传统也带到了当地。我们本人就好几次去过南亚和东南亚国家，在这些地方经常看到当地人用汉字、说汉语。


中国和这些地区的文化交流不只是民间的。，在明代，永乐皇帝派郑和率领船队开展了一系列航海活动，这就是著名的「郑和下西洋」，实际上，郑和的船队也到过爪哇岛，而且在第一次上岛的时候，当地人由于不明真相，还袭击了郑和船队，造成船员伤亡。后来郑和与明朝皇帝运用了外交智慧，把这场外交风波圆满化解，开辟了中外友好的局面。其实中国和爪哇的文化交流也不仅限于明朝以后。在郑和到达爪哇之前，中国古人就已经来到过这片土地。你可能听说过《水浒传》吧？这是我国古代四大名著之一。在这本书里，就有一句话说「那怒气直钻过爪哇国去了」。这是个比喻的说法，因为以当时的交通条件，爪哇离中国很远，所以怒气跑到爪哇国去，意思是一下子就不生气了。


但是到了现代，爪哇相对我们来说就不远了。印度尼西亚的首都雅加达就坐落在爪哇岛上，从北京坐飞机到雅加达，需要 7 个多小时，而从广州飞到那里只需要 5 个小时。如果你有兴趣，今后可以去爪哇岛，体验当地的美景和文化。当然啦，现在的物流这么发达，你在家里就可以买到世界各地的产品，当然也包括爪哇出产的咖啡。你想不想亲自品尝一下呢？
