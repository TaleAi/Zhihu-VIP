## 简介

地理与人文，从古至今便有着密不可分的关系。


不少孩子，从小就有一颗对外在世界的强烈好奇心，对于遥远的异国他乡，怀有一些浪漫的幻想。


而这些其他的地域和国度，真实的人文地理环境究竟怎么样呢？


荷兰为什么叫「风车之国」？阿根廷的牛为什么比人还多？冰岛又为何被称作「冰火之国」？澳大利亚为什么是「骑在羊背上的国家」？
