## 简介
汇报是职场的必备技能，也是展现自己工作能力的重要机会。在当代职场中，汇报不再是简单的做一份 PPT 或一次会议上的发言，如何写好工作邮件进行汇报、如何利用微信来工作汇报……这些都是职场人需要学会的工作技巧。本门课程总结了工作中会遇到的各种汇报场景，结合具体案例，生动幽默地帮你克服汇报中的常见问题。


无论你从事哪个行业，不管你是设计大牛还是代码大神，工作汇报可能都是让你最头疼的职场问题之一。


汇报离你的日常生活并不遥远，小到一次微信上的沟通，大到一次对客户的提案，我们无时无刻不在与人沟通、向人汇报。


**公司新人做汇报：**标新立异可能被认为不成熟，随大流又会让人觉得没能力；


**回复邮件有诀窍：**收件抄送要分清，正文书写要得体，附件签名要仔细；


**微信工作要注意：**一个不合适的表情可能会让你失去一位重要客户，一句不得当的回复可能会让你被领导和同事误解；


**两式三招 PPT ：**理清框架与联系，构图清晰有逻辑；


**客户提案有深意：**把握时长，明确要点，摸清客户花钱背后的真正诉求，用实力与方案回应；


**年终总结难搞定：**拒绝复制粘贴，按模块进行自我复盘，掌握全年的成长轨迹……


除了这些场景，我们在工作汇报中还会遇到内向胆小、情绪紧张、有啰里啰嗦的口头禅等等各种问题；除了这些对象，我们在工作汇报中难免还会遇到要向大领导汇报、向企业老总汇报、向投资人汇报等等各种情况。

不知道如何高效汇报工作，说了一大堆，老板却觉得没重点、不清晰；


想让客户接受提案，拿下投资，却不知如何制作一份完美方案；


性格内向，羡慕外向者在台上热情洋溢、侃侃而谈；


在汇报工作时紧张焦虑，结结巴巴，影响效果，状况频出……


12 节工作汇报技巧课，帮你快速提高工作能力。

多场景，解决职场汇报中的各种问题；


多角度，不管面对客户、同事、领导、大领导，都能从容应对；


不枯燥，语言风趣幽默，分享贴近你日常生活的职场案例；
