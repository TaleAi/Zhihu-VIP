## 简介
十篇文章帮你进行「考研大扫盲」


考研周期太长，不知道能不能坚持？


考研太难，机会成本太高？


工作挣钱比考研更实际？


作为站在人生的十字路口上的应届本科毕业生，如果不能准确抓住时机为未来发展储备实力，就会被这个人人奔跑的时代远远甩在后面。


对自己最好的投资就是努力学习。只有通过学习不断提升自我，才有资本和资格去实现自我，唯有能力、素质和格局，会伴你一生，回馈于你，给你真正的安全感。


临近大学毕业，不少同学对未来感到迷茫，到底是选择工作还是读研深造？中公考研政治研究院总结 7 年的考研辅导实践经验，筛选广大学子选择考研之前最关心的问题。从考研可行性、考研基本常识和考研必备技能三个方面对考研进行了全面剖析，帮助同学们在全面了解考研的基础上做出最适合自己的选择。


不用再辛苦检索困惑的问题，十篇文章帮你进行「考研大扫盲」。
