# 速读名人传记：24个人生进阶故事

### 作者：李源

### 分类：文学

### 存档日期：2022-06-03

## 简介
把历史和现实连接起来，把这些名家名人请下神坛，化身成你自己、你的父母、你身边的小人物。看看这些智者的人生观、成长历程，学习他们如何提升自己、解决问题，以及他们为人处世的方式和思考问题的方法。不管是孩子，还是成人，希望你在这个过程中，既可以听到 24 个好故事，又能收获 24 个心法，陪你走过未来的路。


**看故事容易**，**提炼方法难。**


但是**浮躁、偏娱乐化、没有太多营养**的体验，跟阅读经典传记所带来的**长久激励、深度启迪**是有很大差距的。


因此提炼这些大咖的思维方式和处事方法，帮助人们**节约时间**去了解名人故事，给予自己激励；并掌握他们的人生经验，为我们的生活和学习**提供方法**，最终让我们的生活和工作变得更加让人满意。


想让自己更好地读书学习，但是不知道具体该看什么书？


想培养自己的兴趣和特长，但选择太多眼花缭乱？


想让自己不断进步蜕变，但又总是三分钟热度？


  ······


其实学习名人的成长故事，可以解决上面的很多问题，为什么这么说呢？


传记留下的是历史名人的言传身教。读一个名人的传记就是在学习一个人的人生经验和智慧。古人说：「少年养志」，那又该如何养志？通过读名人传记是一个很好的方法，历史名人往往能够给我们极大的正面激励。


名人的经历，足以成为我们的榜样。他们的传记，可以激发人的志向，增强人的信念，最终还能让我们学到名人的独门心法。用来解决现实中遇到的各种问题。


著名作家茨威格曾说:「读伟人的传记吧，与勇敢的心灵作伴!」我在这里要说的是：「读名人的传记吧，里面有你想要的所有答案。」
