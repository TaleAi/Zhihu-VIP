## 简介
本课程由曾任世界 500 强高管的徐琳老师主讲，通过讲述她亲身经历的种种故事，打破了一些狭隘的固有观念，帮你掌握既会做人、又会做事的本领。


 本专栏适合所有现代女性。无论你是温婉的小女子，还是豪情的女汉子，都会面临一些来自工作和生活上的烦恼。


一位曾经的世界 500 强高管、如今的女性创业者，通过讲述她亲身经历的种种故事，打破了一些狭隘的固有观念，帮你掌握既会做人、又会做事的本领。


如何规划好自己的职业路径，实现升职加薪？


如何摆脱熬夜加班，成为效率达人?


又该如何看待职场潜规则？


踏入职场，不应只顾盲目努力，要成为一名会做事的人。


而作为现代女性，除了会做事，也要会做人。譬如，要懂得把控与男性的社交尺度、要知道如何优雅地与人说「不」、要学会消化生活中的种种负面情绪……


作者将自己二十多年在职场和生活上的感悟浓缩成本专栏，试图帮助女性朋友们解决现实生活中常会遇到的难题，让她们走出舒适区，挑战更多可能。
