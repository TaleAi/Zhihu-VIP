## 简介
投资是什么？「投资理财是一生的功课」，「是概率的问题」，「是生活的一种手段，而非目的」；投资需要什么？首先需要「树立一个正确的投资理念」，需要知道「如何从市场中』取钱』」，还需要懂点儿「行为金融学道理」，需要确认「上市公司的价值在哪里」、「企业的核心价值在哪里」。


投资理念如何指导实践？


「有了理念，财富管理也很重要」，要根据自身所处的阶段「管理配置资产」、还要理解「投资理财的本质就是指数成长」，而其实投资与现实最密切的联系就体现在当「别人百无聊赖去逛街、网上买买买，你打开行情看看股票、债券和商品」。


如果你是一个仍旧徘徊在投资大门前的小白，老邢专门写给你的一百封信，也许会帮你推开投资理财这扇大门，带你走上这条通往财富的道路。


老邢的 100 封投资信即将陆续发出，收信人就是你。


每封信中，老邢都会向你介绍投资理财中不可不知的「秘密」，也许关于时下的财经热点，也许感发于多年的实战经验，封封都是干货。
