## 简介
明月千里，相思难寄。


每个人都有自己的月亮。


但这个专栏里，他们虽然一心向着自己的明月，却不像那些贪心不足的追月人，想要把月亮拉下凡尘。


他们选择自己斩尽繁杂苦难，站到月亮的面前。


**二重奏**


我逛商场的时候被狼尾冷面帅哥搭讪了。


帅哥俯视着我，问得毫不犹豫，「请问你的火罐儿在哪儿拔的？」


原本澎湃的内心瞬间平静，我扬起微笑，「爬。」


本以为只有这一面之缘，结果我被他堵在了校庆晚会的后台。


「你把我的微信拉黑了？」


请停止引诱我，我马上把你放出来。 


**月华如昼**


我是大昭最有权势的贵女，国公府嫡女，父兄皆是大昭名将。


宫宴的时候，喝醉了偷溜到御花园，结果遇见了被敌国送来的质子。


我这才知道，质子原来是个极有意思的人。


突逢刺杀，质子舍身为我挡刀。


我以为这是他的善良。


但很久以后，我才恍然明白，原来所有人，都早已在他的圈套之中。
