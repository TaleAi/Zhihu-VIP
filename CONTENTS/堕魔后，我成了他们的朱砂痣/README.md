# 堕魔后，我成了他们的朱砂痣

## 简介
心中无男人，结局必成神。


被众仙当成白月光的替身后，她纵身一跃，从仙界跳了下去。


所有人都以为她死了，就连白月光本人也松了口气。


但谁也没想到，百年后，她又回到了仙界，参加太子的婚典。


只不过，这次她的身份已经变成了魔尊。


她本来只是想凑个热闹的。但天知道为什么，百年前折辱她、逼迫她的仙君们，这次却一个接一个地出现在她身边，试图确认她就是百年前那个可怜的小透明。


难道他们没看到身边的白月光，已经面色苍白、摇摇欲坠了吗？


她对这帮男人烦不胜烦。然而因为一个阴谋，她竟和他们一同进入了幻境。在这里，她发现了仙人魔三界的大秘密。


生死关头，她会原谅这些曾伤害过她的仙君吗？


三界岌岌可危，作为关键人物，她会伸出援手，拯救这个并不美好的世界吗？
