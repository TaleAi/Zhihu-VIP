## 简介
在本门课程中，一颉老师将为你盘点那些历史上最荒淫残暴的帝王们。他们把杀人当乐趣，视人命如草芥。剥皮、砍头、锯耳朵，可谓无所不用其极。这些皇帝里面，没有最残暴，只有更残暴，让我们一起来看看吧。


在长达五千年的历史长河中，中国共出现了八十多个王朝和四百多位帝王。


他们之中有创世英雄，也有乱世枭雄；有盛世明君，也有荒唐昏君；有治世英才，也有唯恐天下不乱的蠢材……


在这些帝王之中，被评价为暴君的帝王，不占少数。只是这历史由后人所写，评价由后人来定，难免有失公允，毕竟有些「暴君」在历史上也有着不小的功绩。但今天我们要讲的这些却是功绩不多，但十分残暴的皇帝！


比如沉迷酒池肉林，研究酷刑的商纣；让百姓走在街上都不敢随意交谈的周厉王；为博美人一笑，烽火戏诸侯的周幽王；杀掉一百多个侄子的昭成帝......


这些皇帝里面，没有最残暴，只有更残暴！他们把杀人当乐趣，视人命如草芥。剥皮、砍头、锯耳朵，可谓无所不用其极。今天这门课我们就来盘点一下那些历史上最荒淫残暴的帝王们。
