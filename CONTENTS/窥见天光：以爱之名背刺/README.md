## 简介
这是一个揭露人性黑暗的复仇悬疑故事合集——


被最亲近的人背刺、欺骗，最后到底该何去何从。


各种复仇和悬疑脑洞，揭露人性的黑暗地带。


被偷走人生的拐卖女孩，把人贩子送进监狱后拿回了属于自己的幸福；


被「吃绝户」的富家独生女，幡然醒悟后给了渣男致命一击；


收养了白眼狼女儿的好心夫妇，最后终于狠下心来斩断羁绊；


被男朋友诓骗去献祭蛟龙的无知少女，实际上是传承千年的寻鳞师……


踽踽独行在人性的黑暗泥沼，在挣脱后得以窥见天光！


天理循回，报应不爽，黑暗里终能再见光明。
