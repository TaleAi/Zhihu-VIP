## 简介
她们是不被定义的古代女子。


女子应该是什么样的？


这个问题的答案是「没有应该」。


即便是在古代，她们可以盐，可以甜，可以霸道嚣张，可以乖顺谦和，可以追求爱情，更可以拼搏事业。


这个专栏，静看古言女主的万千模样。


**《意未央》**


某一日，夫君性情大变，他逼死了我爹娘，又囚禁了我。


他不再爱我，只以折磨我为乐。


人怎么会发生这么大的变化，我一定要找到原因。


直到我见到曾经讨厌的晋王，才知道，一切早已注定。
