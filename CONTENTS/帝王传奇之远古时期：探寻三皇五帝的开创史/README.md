## 简介
本课程为帝王传奇系列第一弹，由超人气主播一颉老师主讲。传奇帝王，或英明贤德，开创伟业；或荒淫无度，残暴昏庸；或惨遭不测，英年早逝……至高无上的权力之下，他们的命运迥然不同。让我们一起透过他们的人生，感悟世界的精彩。


「千古兴亡多少事？悠悠。不尽长江滚滚流。」


中华上下五千年，沧海桑田，朝代更替。中华民族在这五千年的漫长岁月中书写了悠久的历史，创造了璀璨的文化。


而皇帝是中国历史长流中最「顶尖」的一批人，这里的「顶尖」指的是权力。自公元 221 年起嬴政登基称帝到 1912 年末代皇帝退位，这其中两千多年的历史中，中国一共经历了四百多位皇帝。


他们或英明贤德，开创伟业；或荒淫无度，残暴昏庸；又或是惨遭不测，英年早逝……至高无上的权力之下，他们的命运迥然不同。


道德三皇五帝，功名夏后商周，五霸七雄闹春秋，顷刻兴亡过手。历史是一面镜子，让一颉老师带你钩沉他们的史迹，传述他们的轶闻。透过他们的人生，感悟世界的精彩。


本课程为帝王传奇系列第一弹，主要讲述的是远古时期三皇五帝的精彩故事。在这里，你将会看到尝遍百草，播种五谷的神农氏；将禅让制变成世袭制的君主启；沉迷美色，昏庸无度的周幽王……他们的故事等你来倾听。
