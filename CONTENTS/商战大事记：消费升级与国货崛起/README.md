# 商战大事记：消费升级与国货崛起

## 简介
任何企业或品牌在商业上的行为，都不是孤立存在的。


从微观来看，它受企业自己的品牌、产品乃至合作方、竞争方所影响；


从宏观来看，经济政策、产业发展乃至当下消费者的消费行为，都会影响企业和品牌下一步的商业决策。


分析商业斗争背后的影响因素，不仅能让我们更理智地看待商业斗争，还能读懂当下经济、产业、市场大环境对企业，乃至我们生活的影响。


* 百事 VS 可口，争的真的是口味吗？
* 品牌 VS 主播，真的只是一时兴起的毁约吗？
* ……

本专栏选取 21 世纪以来，具有代表性的企业或品牌之间的商业斗争，从微观到宏观，抽丝剥茧地分析出斗争背后的影响因素，帮助读者理清在互联网、经济高速发展的这十多年里，产业转型、消费升级、乃至国货崛起给商业模式带来的巨大改变。
