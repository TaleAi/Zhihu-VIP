## 简介
每一代人，都有每一代人的存在危机。在复杂的社会中，如何维持独立的思想？面对文化的冲击，如何保有精神的高地？本专栏带你拨开历史尘雾，回到时代现场。缕述影响整个 20 世纪中国学术文化进程的 20 位晚清著名文人与五四知识分子，带你了解他们的人生选择、思想起伏、心路历程。


面临复杂人生局面，想要保持思考和探索出路？


在平凡生活中，想要自我超越，不愿甘于平庸？


对晚清名士、五四学人的精神怀有敬佩和认同？


跟随知名学者陈平原，解锁独家视角，打破标签化认知，从文学作品和学术著作入手解读，直抵思想的来龙去脉，真正读懂 20 位中国近现代名人内心深处的光明面与阴暗面。


透视时代风云，探究其思想的来源和去向。那个时代的他们所具有的特立独行的精神气质、趣味和人格魅力，为处在变化中的年轻人提供 N 种突破平庸的思考路径与借鉴样本。
