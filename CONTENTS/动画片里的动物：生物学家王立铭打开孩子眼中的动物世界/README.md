## 简介
章鱼为什么能变色？树懒是不是真的懒？蜘蛛丝能不能真的吊起蜘蛛侠……这些具体的问题和知识背后，是孩子们无拘无束的自由想法，是他们好奇地张望这个世界的眼光。让我们跟随浙江大学生命科学院教授王立铭一起，去探索动画片中包含的生物学知识，用科学解开孩子们眼中「千奇百怪」的幻想世界。


很多孩子都有一些共同的特征，比如喜欢听爸爸妈妈讲故事，也热爱看动画片。


同样，还喜欢没休止地问「为什么」，直到大人们的解释枯竭——这一点，大概所有孩子们都一样。


孩子的世界里没有太多的「理当如此」或者「见怪不怪」。孩子的世界，也没有什么思考问题的圈圈和范围。这些「为什么」，带着孩子们的思维在跨度巨大的思想世界里跳跃飞翔。这让孩子们的世界要远比大人的世界更立体、更丰富多彩、更有想象力。


章鱼为什么能变色？树懒是不是真的懒？蜘蛛丝能不能真的吊起蜘蛛侠……这些具体的知识并没有那么重要，但是这些问题的背后，是孩子们无拘无束的自由想法，是他们好奇地张望这个世界的眼光。


本专栏也期待着能用科学知识，给孩子们的好奇心以及天马行空的想法提供一层新的畅游空间，这是一个很好玩、很激动人心、也很有力量的新世界。
