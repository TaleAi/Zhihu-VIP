## 简介
漆黑中凝视着你的眼睛，带来无法言说的恐惧——大学生与都市青年的惊悚日常，令人欲罢不能的超恐怖小说集。


我目睹了楼上的邻居坠楼的瞬间，


他身上的痘痘逐个在空气中爆开，


半空中的血雾里，他向我露出笑脸。


后来我才意识到，那是解脱的快乐。


——《无数》


  



朋友离奇死亡，


出租屋里狼藉一片，


短短几天之内，他的尸体已经腐化成一片白骨。


没有人能不畏惧衰老，除非，他已经体会过生命力流逝的绝望


——《衰老》


  



工厂里每周都在死人，


所有死去的人，都和我是同一天生日。


清查真相的过程，屡战屡败，


我知道，如果问题不能解决，那么，下一具尸体


是我。


——《死神有没有来》
