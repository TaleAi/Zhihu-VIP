## 简介
逃离至暗时刻


美国「新精神分析学派」心理学家埃里克森通过临床观察以及对大量案例的分析提出，人在每一个阶段都有其特殊的目标、任务和冲突。


一个人从出生到死亡，心理一共经历了 8 个连续发展的阶段，如果某个阶段任务没有完成，我们很可能会停滞在那里。


而我们在生活中遇到的种种问题：


* *没有安全感，总是患得患失；*
* *脆弱、敏感，总像是被事情、被世界推着走，自己没有话语权和选择权；*
* *自卑、不自信，总觉得自己是没有价值的；*
* *对亲密关系充满了怀疑，渴望但不敢靠近... ...*

极有可能是当下或之前的发展阶段中，没有「打好怪」，因此没有「升好级」。


本专栏将会从埃里克森八阶段发展理论出发，提炼出每个阶段最需要发展的能力，结合心理学、管理学、社会学等多视角，告诉你「是什么-为什么-怎么办」，一步步带你打怪升级。
