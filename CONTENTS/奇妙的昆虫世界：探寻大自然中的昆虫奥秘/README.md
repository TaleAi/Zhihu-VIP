## 简介
在大自然的旅途中，随时随处都会见到昆虫。在公园的草坪里，在春游的路途中，在池塘边的石头缝里，都有各种各样我们熟悉的或未曾了解过的昆虫。昆虫只是简单的小虫子吗？其实它在我们的生活中扮演着重要的角色，了解昆虫的故事，亲近大自然。


昆虫的世界有很多奥秘，你知道吗？


屎壳郎可以通过星星发光位置的记忆，来辨别自己和巢穴的位置；


能悬飞停留在空中的不只有蜂鸟，还有长喙天蛾；


大部分的蚊子都不咬人，在会咬人的那些种类里，也只有要生宝宝的蚊子妈妈才会吸血；


蟑螂和螳螂还有白蚁是亲戚，有着共同的祖先，在动物分类学上的关系也非常近；


......


本专栏庞虹教授主要从事昆虫的物种多样性起源与演化、鞘翅目昆虫的分类、天敌瓢虫资源的利用等方面的研究。为人母的庞虹教授，以切身育儿经验，引导小朋友们对大自然的兴趣与好奇心，在贴近大自然的过程中，探索神奇的「虫虫世界」。


从认识身边的这些小小生灵开始，认识自然与科学，认识这个世界的规律，去探索更多神奇的未知。
