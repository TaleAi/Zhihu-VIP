## 简介
以系统性的结构深论前沿科技，以亦庄亦谐的文风讲解知识。


关于科学知识，有很多是我们一知半解的，比如：


量子力学有哪些原理作为支撑？为何我们轻易就会被它忽悠了？


为什么说芯片是我们国家的「卡脖子」技术？战斗机的速度为什么用「马赫」表示？


宇宙是如何一步步演化到如今这模样的？杨振宁为何反对我国现在建造大型对撞机？


……


传播严肃科学知识的重要意义在于，要使这个社会、这个民族有科学的氛围。


氛围是一种无处不在的暗示，潜移默化中影响着很多人对已知的思考，对当下行为的反省。更重要的是，可以帮助人们拾起对未来的信心。


科学的传承，才使人类可以站在巨人的肩膀上去更高更远处，而非不断重复昨天。


本专栏特邀两位年轻科学爱好者，以系统性的结构深论前沿的科技，覆盖量子物理、生物、通信、军事科技等多个模块，亦庄亦谐的文风将高难复杂的理性知识通俗易懂地表达出来，让听众轻松理解深刻的硬核前沿科学知识。
