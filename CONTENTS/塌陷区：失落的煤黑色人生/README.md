# 塌陷区：失落的煤黑色人生

### 分类：现代言情 · 悬疑推理 · 真实故事

### 状态：已更新至第 15 节, 共 20 节

## 简介
矿上三代居民的烟火故事，讲述一座矿城呼吸的脉络，以及生灭的蛛丝马迹。发廊女、下岗矿工、畸零寡妇、漂泊的青年、求死不得的老人……他们像极了我们生命中的过客，摇摆不定，找不到人生的意义和合适的落脚点。


◆一座名为「水冘」的煤矿养育了三代人。


煤矿养活矿工，矿工建立家庭，家庭带动周边一系列生产力。


◆他们像「矿难」一般度过一生。


发廊女、下岗矿工、畸零寡妇、漂泊的青年、求死不得的老人……他们在煤黑色的阴霾之下，咀嚼着人生的况味。


这是一个既不属于城市也不属于农村的地方。


故事，就开始于这样的魔幻之地。


**《入水》** 一次矿井事故，让玻璃厂女工失去丈夫，往后的生活，让女人像活在一滩水中。


**《黄桃罐头》** 成年人的崩溃往往是因为一件小事。


**《爸爸妈妈的游戏》** 在没有性教育的年代，孩子在什么情况下长大？


**《澡堂》** 矿上给满身煤灰，黑乎乎、脏兮兮的矿工洗澡的的澡堂是什么样子？


**《塌陷区》** 据说塌陷区每年都会死几个人，死在那里的人必须拉别人下水才能投胎。


**《悬索桥恋人》** 两个人在遇到危机时会产生强烈的依恋，彼此扶持度过摇晃的悬索桥，只是一旦危机解除，两人也会分道扬镳……
