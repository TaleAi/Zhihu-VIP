## 简介
说书十八年，张大春将为你带来一个全新的三国。如果你不知道三国该从何读起，如果你想听精彩的三国故事，了解背后的权谋与人心、精密细节与智谋布局，如果你感到自己只读到冰山一角，而三国永远读不完，那么，你也会爱张大春讲的三国。


有人说，三国里藏着每个中国人必须要有的情商。


细读三国，不仅是看故事，更是看人。


计谋权术、战略战法、内政外交、人情世故，


张大春精研细读，结合史料，


为你展现另一种三国。

**课程大纲**


[**《张大春细说三国（上）》**](https://www.zhihu.com/remix/albums/1127613746109186048)


**第一模块  东汉末年，三国鼎立形成之前时局动荡，各地诸侯割据。（**1-83 集）



> 东汉末年，时局动荡。黄巾起义后，董卓挟持汉献帝独揽大权，王允设计杀死董卓。董卓的部属又杀死了王允全家。以袁绍为首的十七镇诸侯又以讨伐董卓的名义相互厮杀，从黄巾起义中起家的曹操、孙坚、刘备在多次战争中慢慢扩大势力。


**第二模块  三国纷争：曹操崛起，刘备投靠刘表。**（84 -131 集）



> 曹操的势力逐渐扩大，终于和北方大军阀袁绍进行了生死较量——官渡之战，巩固了自己地位的同时还统一了北方。刘备投靠刘表后，又被刘表的老婆和小舅子离间，不得不退守新野。徐庶投靠刘备后，向刘备推荐了诸葛亮。


**第三模块 三国纷争：孙刘联盟，赤壁之战三分天下。**（132-225 集）



> 刘备三顾茅庐说动诸葛亮出山，并和孙权建立同盟，一起对抗曹操。赤壁之战中，刘备和孙权大胜。刘备趁机占领荆州，和孙权反目。随后刘备攻打益州，又和孙权联盟夺下汉中，之后自立为汉中王，三国鼎立正式形成。


  



**《张大春细说三国（下）》**


**第四模块  三国纷争后期：曹刘称帝，刘备病死托孤。**（1-135 集）



> 孙权杀关羽夺荆州，吴蜀绝交。曹操病死，曹丕逼汉献帝退位称帝建魏。一年后，刘备在成都称帝。为了替关羽报仇，率兵攻打东吴，兵败后退到白帝城，在那里病死。死前托孤，诸葛亮立刘禅为帝。


**第五模块  三国归晋：蜀吴灭亡，司马炎统一天下。**（135-203 集）



> 诸葛亮死后，蜀汉就不行了。魏国在曹睿死后，大权掌控在司马家。魏景元四年，蜀汉灭亡。魏咸宁二年，司马昭的儿子司马炎代魏称帝，西晋建立。咸宁六年，东吴灭亡，三国时代结束。
