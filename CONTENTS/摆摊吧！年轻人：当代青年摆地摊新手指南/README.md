## 简介
70 后「前浪」给想摆地摊的「后浪」们传授的创业致富实战技巧与经验。


摆地摊门槛低、资金流动性强、回款快、弹性大，加上国家政策的扶持风口，让我们赶上了做副业、踏入零售行业的好时机。


而且，练摊儿是城市里消失了有些年头儿的行当，对于年轻人来说，肯定会是一个全新的人生体验。


摆摊真能赚钱，但前提是你要知道做低成本小生意中各个环节中的要点，钱是怎么从别人兜里赚到你兜里的，每个环节都要赋予「卖点」，有用户观。


如果你现在想的是我卖什么最赚钱？我要投入多少钱摆地摊？那么你可能走入了误区，这些问题并不是你首先要考虑的。在不提前考察市场、了解实际情况就下海行动，是无准备无计划的盲从。


俗话说：外行看热闹，内行看门道。在本专栏中，我将给你讲讲摆地摊的门道：如何进货、如何吼货、如何低成本投入获得高额回报、如何在竞争中脱颖而出、如何获得优质客户、有哪些需要规避的潜在风险等等。


这套课程浓缩了我多年来做副业：自营地摊、管理地摊的经验和心得。我将从市场分析、成本控制、经营模式、实操经验、风险管理等六个方面手把手教会你摆地摊，不赔钱，利用碎片时间合理有效的赚钱。
