## 简介
天上的神，地下的鬼，中间的人，都被万事万物困住过。在问题面前，人神鬼都一样在渴求答案快快显现，而答案往往就藏在被忽视的人身上。


我还记得年少时的梦，长大了娶你，是我写给命运的答案，你却停留在洋溢青春的秋天，凝固成一朵永远不凋零的花；


我还记得第一次与你见面，像只疯猴子，满脑子的难缠问题却深藏心事，痛彻地眼神里投射出锋利的光，那是死了也要爱的决然；


我还记得你自愿为家庭画地为牢，可那真的是一座牢，感情不是绑架，不是个体自由的湮灭之地；


……


在这个专栏里，邢二狗会写给你烧脑爆炸的悬疑故事、极致的复仇爽文以及脑洞爆炸的情感故事。


你会在文字中或悲或喜，或离奇或真挚，或唏嘘或温暖，全是因为爱会解答故事里的所有问题。


爱才是天上地下，唯一的答案。
