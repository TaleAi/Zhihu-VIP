# 他的情书：牙科诊室重逢的爱情

### 存档日期：2022-06-03

## 简介
高岭之花弟弟 X 辣手摧花姐姐。作为高中校友的他们，时隔 7 年，在牙科诊室重逢。从高中起，他们就是一对欢喜冤家。恩怨的源头，是 7 年前一封离奇的情书。那封情书，让她「自作多情」地成了全校的笑话。7 年后，他们逐渐揭开了当年关于「他的情书」的误会，并挖掘出一段横跨 7 年却始终沉静的纯爱。


姜柔在追那个转校生，搞得人尽皆知。


结果别人根本不甩她。


因为此事她被全校嘲笑了一年。


好友问起此事，「你说他是不是要你考进年级前三才答应你呀？」


姜柔，「？」


她一手捏着全是叉叉的成绩单，一手捏着那个转校生写给自己的情书。


简直离谱！


到底谁追谁？
