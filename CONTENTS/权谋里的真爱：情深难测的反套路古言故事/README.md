## 简介
当爱情遇上权谋，秘密便多了起来。而那些秘密背后的小心算计，用心经营，是最爱你的人，留给你的温柔和深情。本专栏里，「脑洞砖家」都关月，为你带来 3 个有秘密的反套路古言故事。在那些主角或遗忘、或被刻意隐瞒的秘密里，藏着另一个人最缱绻最深沉最无奈的温柔。 


**囊**


可曾听闻狸猫换太子？她想过，但是真的发生的时候，她退缩了。同一张皮囊，一边是青梅竹马的护卫，一边是情根深种的帝王。当二人身份颠倒，泼天的权谋暗算里，她又该何去何从？


**竹马三生**


傲娇竹马 PK 天降姻缘，究竟谁能虏获刁蛮公主？天降会英雄救美，竹马有门当户对；天降令公主暗许芳心，竹马有皇帝圣旨赐婚。当理想与现实激烈碰撞，刁蛮小公主到底该如何抉择？


……


一场场精心策划的阴谋，一段段至死方休的爱恨，在这里上演。
