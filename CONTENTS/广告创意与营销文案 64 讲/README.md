## 简介
新媒体时代，广告营销的创意和策略发生了颠覆性改变。广告不再靠砸钱买「注意力」，而是要看你是否能创造出「讨论力」和「分享力」。如何提取品牌身上的病毒基因，并且传播给消费者成了广告取胜的关键所在。本次课程将从广告洞察、创意、文案、视觉、投放五个方面选择经典案例并做深度剖析，通过系统化讲解，让你快速掌握互联网时代的广告核心创意方法。


**明星企业御用营销大咖主讲**

**段子不断，轻松高效学干货**


把广告案例包装成一个个段子，赵宁在风趣幽默的言谈中拆解创意思路和底层逻辑，听他的课，你绝对不会犯困，而是让你在潜移默化中学到有用有料的知识、经验和见解。
