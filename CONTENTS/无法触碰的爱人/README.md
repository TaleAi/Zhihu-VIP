## 简介
我很好，他不配。


他明明就在身边，我们却好像隔着遥远的距离。


无法得到回应的情意，爱得再深也是枉然。


无法碰触的爱人，或许放手，也是一种不错的选择。


**无法触碰**


我死的第一天，男朋友就把白月光带回了家。


他们在我买的沙发上旁若无人地接吻，吃着我亲手包的芹菜馅饺子，用着我送给他的游戏机。


有一天，白月光好奇地问：「安安呢？」


男朋友语气平静，「前几天跟我吵了一架，跟公司申请出差了。」


哦，他还不知道我已经死了。


**丑爆的校花**


门没关好，听见外卖小哥偷偷给人打电话，「看清楚了，跟照片不一样，本人丑爆了。」


下一秒，我收到网恋男友的微信：「我们分手吧。」


昨天刚被票选为校花的我：？
