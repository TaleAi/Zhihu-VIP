宇宙是人类最终极的浪漫。


也许从第一次仰望星空开始，人类就产生了无尽的好奇和探索欲——


早期的神话，天的那边永远有神，然后我们学会了观星象指导生活；


后来的人，羡慕鸟的自由自在，然后我们探索出了飞上天的方式。


从幻想到实现，唯一不变的，是我们对天空、对宇宙无尽的追求。


这个专栏，将为你搭起一条通往星星之路——


* 从牛顿力学，到地球大炮，为了逃离重力，人类做了什么？

让我们跟随科学发展的脚步，一起探索知识的边界，而这次的宇宙之旅，将从怎样离开地球表面开始。
