## 简介
以经典为师，解读世界公认的经典或得到权威组织认可的著作，结合案例和实践，让高深的理论轻松易懂，让你成为「教育行家」。


如果把「父母」视为职业，我们真的称职吗？


我们应该怎样教育眼前这位初来作到的小天使？是给予 TA 系统化的教育，还是家庭理念中默认正确的价值观？


是跟他用儿语交流还是用成年人的交际逻辑和语言方式？


在他学会新的技能时，是该众星捧月式的表扬，还是低调地给予肯定？


到底是该放养，还是规定条条框框？


回答这些问题，光凭感觉或者人云亦云是不行的。为人父母的我们，应该进行系统化的学习。


在信息甚器尘上的时代，我们要怎么学习才最有效？有一个答案毋庸置疑便是读经典。


本专栏将为你解读经典，以经典为师，让你成为「教育行家」。为你解读世界公认的经典或得到权威组织认可的著作，剔除众所周知的常识性内容，剔除视角平庸的见解，结合案例和实践，让高深的理论轻松易懂。
