## 10.空巢综合症：整天找事儿的婆婆，媳妇如何与她相处？
搞不定婆婆，也就很难搞定婚姻。这里是，教你搞定婆婆的，婆媳关系圣经课。


在这节中，我要分享的主题是 一个整天找事的婆婆，你该如何跟她相处？对于一个没事找事，一个不折腾就不舒服，一个不让别人难受，她就很难受的婆婆，我们要不要把婆婆的所作所为，一五一十地告诉老公呢？我会在 这节课里告诉你。


学员小 c 自从生完孩子以后，就每天和婆婆一起，在家里照顾孩子。但是婆婆每天在家里各种唠叨。今天说小 c 不会照顾孩子，总是让这个孩子哭，说再这样照顾下去，孩子迟早会出毛病。


婆婆甚至说，等孩子一岁以后，要带着孩子回家生活。偶尔呢，婆婆又说城市里的交通太差，每次买菜都要坐很久的汽车，然后提议让小 c 和老公去商量，应该搬家。


一开始小 c 觉得婆婆就是随便说说，但是时间一长，小 c 一听到婆婆说自己不会带孩子，就很烦。一听说婆婆要把孩子带回老家带，就非常焦虑。那段时间，只要老公一回来，小 c 就赶紧跟老公说：「今天你妈又说我不会照顾孩子了，你妈又过来说要搬家了。今天你妈又想跟我吵架了。」


虽然小 c 觉得，自己是无辜的，老公一定会向着自己，为自己主持公道，结果呢，小 c 老公的态度越来越差。到后来甚至跟小 c 说：「行了，你别讲了，我不想听了。」


小 c 太委屈了，她不知道做错了什么。明明是婆婆不停地没事找事，老公却怪起自己来了。


无独有偶，另一个婆婆是每天给儿子打电话，今天说她胃不舒服，要儿子请假过来带自己上医院，后天说自己的头疼，要让儿子请假过来看医生。小两口带婆婆去了医院，却发现完一没有了那种感觉。


这个婆婆的儿媳妇就非常不理解，不知道这样的婆婆到底想干什么。


身为妻子，她开始跟老公抱怨，说婆波没事找事，占用老公时间不说，还弄得大家忙乱不堪。原本是为了老公着想，但也换来了老公的一顿数落。


先来说一下，一个婆婆没事找事，到底是什么心理？


反复折腾的老人，往往有空巢综合症。他们已经进入老年，社会关系渐渐变少，身体的功能在衰退，而此时儿女们也都越来越忙，陪伴在他们身边的时间越来越少。


老人内心世界里的孤独感越来越重。她害怕一个人呆着，希望获得更多的爱与陪伴。担心儿子有了儿媳，以后就再也不关系自己了。 又担心儿媳妇会要求儿子，不要管自己。


所以，这些老人会通过提各种要求，发泄各种不满，来表达自己对于爱和陪伴的需要。


这种老人需要的很简单：他们需要的是爱和陪伴，需要的是关心和照顾。了解了这一点，当你发现，你的婆婆经常没事找事各种折腾你和你老公的时候，你就知道怎样应对了。


第一、给婆婆更多的平和的爱。


当你觉得婆婆找事儿的时候，一定要看一看是不是她得到的爱太少了？可能平时儿女们对她关心并不多。


有没有人陪她说一说话？


除了送礼物，我们还有其他对她表达爱的方式吗？


最后再和婆婆进行一次无伤害的沟通。告诉婆波：「我们都很关心您，只是我们有时候工作太忙，又要照顾您的孙子，所以有时候不能及时回来陪您。但是，我们心里都有您，这一点您放心。」


你会发现，婆婆不是不能沟通，是沟通要讲究方式方法。婆婆也不是心眼有多坏，只是她们用了错误的方式来表达爱的需求而已。


第二、无论是当着婆婆的面，还是背着婆婆，高情商的妻子们，会有意无意地跟老公说：「妈妈不容易，咱们多陪陪他们吧。」


因为你的顾全大局，你的老公会感激你。


第三、要不要把婆婆的烦人行为告诉老公？


如果你很烦你的婆婆，由你来告知老公，让老公知道他的妈妈到底有多烦，这绝对不是一个高情商的作法。


因为，工作一天的老公，不会觉得他妈妈没事找事，反而觉得是你在没事找事。本来他可以安静地休息一会儿，但是你却跟他叨念他妈妈有多么不好。在他看来，是你在增加他的心理负担。所以，他会对你非常愤怒和反感。


在他看来，只要你不告诉他，一切麻烦都不存在。


所以呢，当婆婆没事找事的时候，由妻子来告知老公，只会激化夫妻矛盾。


从另一个角度来说，自己的妈妈是什么样子，儿子们也一清二楚，就算你不算，他的儿子也知道。


再退一步说，假如你老公永远都觉得他妈妈是天下最好的人，那么你就更没有必要去在老公耳边吹风了。你敢说他妈妈任何不好，都会引起巨大的反感。


无论是在婆婆面前，还是在老公面前，妻子们需要保持适当的沉默，同时适当示弱。


当婆婆不断对你提各种要求的时候，小事情你能办到的，可以适当地去满足婆婆。只要你感觉到吃力，你就可以用示弱的办法，来化解掉。


当婆婆不断要求你这，要求你那儿的时候，你可以说：「我以后好好向您学习啊，但是现在我还做不来，您别急，我慢慢学。」


当婆婆不断指责你，这不对，那不好的时候，你要需要态度良好地跟婆婆说：「好的好的，我下次一定注意。」


总结一下，当你的婆婆经常折腾，没事找事，有一个原则就是，爱！


爱是一工的根源，爱是一切的答案。当我们家庭关系遇到问题时，不管别人有什么样的问题，我们一定有自己的问题，静下心来思考一下：我们有哪些需要提升的地方。


大多时候，可能老公找茬、婆婆找茬、自己的父母找茬、孩子不听话，往往都是因为缺少了爱。


爱是解决一切焦虑和痛苦的灵丹妙药。当你真正把你的爱给出去，你才会收获更多的爱。当然，如果你说，你做不到，你根本给不出爱，那你就需要寻求心理咨询师的帮助，从提升你的心理能量做起，让你心中的爱，像蓄水池一样，被慢慢蓄起来。


这时候，你的婚姻会更幸福，你的家庭会更和睦。
