## 简介
当你穿进书中世界，一个腹黑又邪恶的男人对你开展霸道虐爱剧情，稍不如意就会有性命之忧，你到底是要自救、还是去救赎？爱情的美好可否打败这致命的寒冰？也许褪去变态冷血的保护色，他也能对你捧起一颗真心……本专栏是腹黑偏执类自我的爱情展现，隐藏在冷漠病态背后的情感诠释。当内心愈陷愈深，爱情也将成为救赎之路的起点。


本专栏目前收录三个中篇故事，分别为《鬼畜总裁应对手册》、《替身娇妻自救手册》及《倾尽时光去爱你》。


《鬼畜总裁应对手册》


如果穿越成炮灰女配，马上要被变态男主绑架囚禁做成人体标本怎么办？


一个怀揣梦想熬夜加班的编剧，打个盹的功夫，一睁眼发现穿越进自己多年前写的小说里，还是书中开局就死的炮灰角色。只有打破「炮灰早死」定律，她才能重新回到现实……


她在刀尖上跳舞，与死神周旋。当她以为自己即将顺利通关时，猛然发现自己穿越的世界其实是由小说魔改的影视剧剧本！


面对三个版本的魔改剧情，她这个注定早死的炮灰，能否反转剧情，活到最后，收获属于她的爱情？





《替身娇妻自救手册》


替身，变态男，虐恋，遇上哪个都是死无全尸，何况还是三种梗一起遇到？


穿越到这本小说里命运的大手时刻想要扼住我的喉咙。


我反其道而行之，玩儿转豪门攻略，脚踢心机婊，智斗变态男。


正当如鱼得水之际，却发现一切都不一样了，是渣男洗白了，还是故事出错了？


褪去变态冷血的总裁外壳，他居然也能深情款款，白月光突然出现，死而复生，真爱是谁？


谁说替身女主不受虐就不能逆袭，哪怕一手烂牌照样打得精彩！






《倾尽时光去爱你》


自知命不久矣，她决定手刃害死父母的仇人，却阴差阳错对庆城首富动了刀子。


他命令她做他的协议妻，彼此利用，互相伤害，卷入无边无尽的阴谋漩涡。


她为了弟弟，自愿牺牲自己，任由他把她当成棋子。


可当意外得知真相，她倾尽力气还给他一切后，彻底逃离他的身边……
