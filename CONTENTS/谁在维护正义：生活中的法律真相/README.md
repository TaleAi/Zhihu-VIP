## 简介
一次不公正的审判，其恶果甚至超过十次犯错。


公平与不公平仅一线之隔……


这条线，就是「人」的意志。


法律、道德、正义……这些概念从来不是一个虚词，从法律的诞生，到法律的逐步完善，都由人来参与，并一步一步构筑成我们生活的全部事实。


大多数人都只知道遵守法律，做好公民，但法律的涵义，远远大于此。


参与法律行为的「人」，如何利用法律达到自己的目的？


以人为主体的「法律活动」，又是如何影响个体，影响世界的？


本专栏希望能够帮助大家从法律的角度去观察我们的生活和人们的行为，更深刻地了解法律的意义。
