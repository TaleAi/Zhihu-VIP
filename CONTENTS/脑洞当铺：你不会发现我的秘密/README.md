## 简介
有哪些脑洞设定曾令你眼前一亮，念念不忘？来，走进脑洞当铺，用你的好奇心当一个好故事。在这里，为你讲故事的每个人都把脑洞开得清奇又独特，他们绝不会为你讲述那些可以预见的故事，热血的江湖恩仇、诡谲的骗保陷阱、暖心的爱情幻梦·····他们笔下的每一个故事，都有着一个接一个令人始料未及的神转折。这些不按常理出牌的故事，你或许猜中了开头，但万万想不到结局。


偶然登陆上司账号的小员工，意外窥探到一场亦真亦假的谋杀案，他到底该不该报警；


为调查一起宠物离奇失踪案，杂志社记者卷入一场荒诞诡异的案中案，几次死里逃生；


说谎身体就会爆炸的世界里，人们只说真话，世界已经没有谎言，可这时，大街小巷却意外传来了此起彼伏的爆炸声；


······


走进这个荒诞离奇、脑洞大开的故事集，你会看到一个个光怪陆离的反套路故事。这些包含了悬疑、荒诞、科幻、惊悚、温情等各种趣味元素的故事，用一波接一波的逆天反转，带给你十分过瘾的阅读体验，而故事背后，也折射出当下的年轻人对工作、生活、感情以及自我的思考，细细嚼来发人深省。


温馨提示：故事里的每一个细节都需要仔细斟酌，请认真阅读，大胆想象。只要你的脑洞够大，真相永远不止一个。


祝你阅读愉快～
