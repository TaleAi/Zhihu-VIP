## 简介
有人说，艺术史太难了，我记不住。但，如果我能听到大师当年有血有肉的人生经历和作画过程，我就会深深地记住那些鲜活的作品。本专栏精选 21 位艺术大师，涵盖西方艺术的各个阶段，从上千年艺术发展中抽丝剥茧，为你浓缩出关键要点：一段历史、两个阶段、三个人物，用最短的时间丰富你的知识，对西方艺术发展形成系统认知。


西方艺术史上，有哪些必看的名家名作？


当你感觉一幅作品「好」的时候，你该怎么和别人安利你喜欢的作品和艺术家？


如果你对某一个艺术家感兴趣，你可以去哪里找到他的更多资料，形成你自己的专属艺术知识库？


我们想要在心里跟每一位大师对话和共鸣，就像我们也在巴洛克时期生活过，听曼陀铃和古典吉他的演奏，给自己的裙边缝上层层的褶皱。


让我们一起跟随著名漫画家慕容引刀的讲述，听他还原历史场景，将同一时期不同艺术家和艺术流派之间的脉络关联和趣味掌故讲清讲透，从上千年艺术发展中抽丝剥茧，为你浓缩出关键要点：一段历史、两个阶段、三个人物，用最短的时间丰富你的知识，对西方艺术发展形成系统认知。
