## 简介
科幻为什么令人着迷？科幻是一种特殊的文学类型，让我们突破日常生活的思维框架。但是，你真的看懂背后的科学知识了吗？经典科幻作品到底经典在哪儿？本专栏特邀知名科普作家汪诘老师，带你深入解析科幻文学史上的十多部经典作品，全景式地扫描 200 多年的科幻发展史，让你把作品理解到骨子里，触摸到科幻的内核。


**近几年，国人对科幻的热情大有走进黄金时代的势头：**


刘慈欣先生的《三体》一举斩获雨果奖最佳长篇，并成为了现象级作品；


由大刘作品《流浪地球》改编的同名电影，在 2019 年创下了中国票房史上第二的佳绩；


郝景芳的《北京折叠》和陈楸帆的《荒潮》都获得众多赞誉……


**科幻为什么令人着迷？**


科幻是一种特殊的文学类型，让我们突破日常生活的思维框架。


我们的想象力可以跨越宇宙数千亿年的历史，也可以前往数百亿光年外的世界，在宇宙法则的背景中进行各种大胆的思想实验，让心灵踏上星辰大海的征途，痛快淋漓。


**但是，你真的看懂背后的科学知识了吗？经典科幻作品到底经典在哪儿？**


本专栏将带你深入解析科幻文学史上的十多部经典作品，从科幻的古典时代到黄金时代，从太空歌剧、未来史诗、赛博朋克、时间旅行、超级英雄、反乌托邦到末日灾难，科幻绝不只是让你感到兴奋的肾上腺素，科幻能带来的远超出你的想象。


请跟随**知名科普作家汪诘**的讲解，全景式地扫描 200 多年的科幻发展史，用科学的视角看科幻，用科幻的思维谈科学。
