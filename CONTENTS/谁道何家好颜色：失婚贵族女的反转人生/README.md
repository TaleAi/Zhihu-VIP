# 谁道何家好颜色：失婚贵族女的反转人生

### 状态：已完结，共 16 节

## 简介
我何颜，当朝丞相的宝贝闺女，蜜罐里长大的豪门贵女，即将嫁给苦追七年才追上的男神的幸运儿；我的完美人生，在新婚那晚画上句号，彻底反转……


新婚那晚，最好的闺蜜沛儿跪在我面前：


她说，我给你戴了绿帽子！


她说，我怀了你丈夫的孩子！


我跟沛儿在洞房里大打出手，这一架打得太惨，结果是，我爱了七年的男人，带着我最好的闺蜜匆忙离开，留下我独守空房。


我的世界轰然倒塌。


后来，我一面假意与看似悔过的渣丈夫斡旋，一面暗中筹划和离，却渐渐发现往日完美生活的背后，有越来越多的疑点：


失足落水而早逝的二姐，似是被我的婆婆陈贵妃谋杀；而看似忠良的爹爹，似乎是个大奸臣；还有我的渣丈夫、太子大伯哥，每个人似乎都戴上了面具，好坏难辨，令人琢磨不透。


他们都来跟我说，让我信他！爹说他是忠臣，为国为民而谋，让我信他；渣丈夫说他爱惨了我，过去一切都是情非得已，让我信他；太子说他韬光养晦多年，只为给我二姐报仇，让我信他。


真相究竟如何？而我，究竟该信谁？
