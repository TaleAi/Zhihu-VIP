## 简介
自从戴口罩以来，你的脸上是否出现以下问题？爆痘，刺痛，搔痒，红肿，脱皮，过敏，以上这些问题统称「口罩脸」，本课程将提供你全套皮肤损伤解决方案，让您有机会零距离接触顶级皮肤健康专家，让你戴口罩也可以更美。


什么是「口罩脸」？


简单来说，就是因长期佩戴口罩的而出现的各种皮肤问题。随着节后开始复工复产复课，大家需要戴口罩的时间逐步延长，马上又到了闷热的夏天，皮肤会更加敏感和脆弱，我们到底如何才能减少因「口罩脸」而带来的皮肤伤害呢？


刘东阳老师将带你走进科学护肤课堂，拯救你的「口罩脸」。
