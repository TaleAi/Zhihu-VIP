## 简介
贼老天，我才十四岁，我真的不想当皇帝姑父的皇后啊！


因为皇后姑母忽然病逝，原本只是太子平妻候选人的我，一跃成了后宫的第一大 boss——皇后。


但是苍天可鉴，臣妾真的不想啊！


再怎么说，我还只是个十四岁的少女，过惯了跟大哥二哥太子表哥胡作非为的日子，在后宫处处受辖制，可不是干什么都不痛快吗？


更何况看似风平浪静的后宫，其实处处有坑：


年长我三十岁的皇帝姑父，虽然爱我姑母甚重，但为了后宫与前朝、太子与八皇子势力间的平衡，还是想让出身显赫的我生下他的嫡子；


后宫第二大 boss 王贵妃，动不动就来试探我，被我惩治过后，还想找他儿子，太子表哥的最大竞争对手八皇子来求我罢手；


傻乎乎的太子表哥，整日被心狠手辣的太子妃哄骗得团团转，对政事一窍不通的他能斗得过美名远扬的八皇子吗？


还有这个该死的八皇子，长得帅也就算了，人品还那么好，还那么关心我，我能抵挡他的个人魅力吗？


这个专栏，且看娇宠小皇后如何在后宫活出自己的精彩~
