# 毫不费力的向上管理：别紧张，管理上级没那么难

## 简介
作为知名猎头公司负责人，我想告诉你，管理者也是普通人。


职场人想要的很简单，努力工作，升职加薪。


可往往当你努力后，发现事情并没有那么简单。


拥有生杀大权的上级，对你所知甚少，缺乏信任，更无从肯定你的价值。


向上管理的本质，是协助上级，建立良好的信任基础；继而为自己开展工作争取支持，完成最终目标，实现双赢。


但是，很多职场人会认为：


* **工作做好就好，向上管理没必要**

除非想好了躺平，能接受工作多年仍杯水车薪，


否则默默耕耘只会让你保持原地不动，


更可能，在公司突然洗盘时你会被动「毕业」。


* **忌惮等级关系，不敢问也不敢聊**

这也不是旧社会，低头就能保平安。


依靠有话语权的人，才能打开格局，争取资源倾斜，用实力说话。


何必怕漏短？


漏短是必然的，更是必要的。


管理者需要公开透明。


对员工的认知、能力了解越多，越利于个性化管理；


对团队的工作情况了解越多，越利于掌控全局。


* **执行起来有麻烦，向上管理就很难**

或许你意识到了向上管理的重要性，


但操作起来没那么顺利，比如：


有些事不知道该不该汇报，


有些话不知道当讲不当讲……


向上管理不是和盘托出，也需要掌握技巧。


本专栏的作者 Carrie 是资深人力资源行业从业者，Lydia 为畅销书作家、知乎 24w+粉丝答主，他们将帮你梳理上下级关系，打造自身职业形象，增强沟通技能与汇报技巧，与上级建立强联系，让你成为综合职业素养合格的职场人。
