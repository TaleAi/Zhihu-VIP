## 简介
都市之外，小城才是人间的底色。


小城的爱情，从来不是两个人的事。


相亲奇遇：螺狮粉味道弥漫的柳州，从大城市回到家乡的女孩遭遇了父母的逼婚大战，万般无奈下只得去相亲，没想到相亲第一面，来的不光是男孩，还有一大帮亲戚。历经盘问、条件比对层层关卡，她和他终于进入交往阶段，可没想到，看起来体面安稳的男方家庭，还有一场暴风雨在等着她……


蓝颜知己：女孩好不容易从临汾的一个小镇走出，成了第一个女大学生，本以为自己将要迎来平稳安乐的人生，却在上学期间遇到一个富二代。她以为自己遇见真爱，却遭遇玩弄，女孩从小到大的男闺蜜陪她度过了最黑暗的时期，青梅竹马最终走入婚姻，没想到孩子出生之后，她才了解到当年的真相……


…… 


长辈、亲友。


家长里短、人情往来。


两个人的故事，其中多少过客匆匆。


小小一座城，承载无数悲欢喜乐。 


小城故事多，不只你与我。
