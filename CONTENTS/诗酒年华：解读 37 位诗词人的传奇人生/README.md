## 简介
37 位诗词人的传奇人生，读懂唐诗宋词的悲欢离合


总有一句诗词，能拨动心弦，让人泪流满面。


再细腻的情感，再莫名的思绪，再飘忽的心境，都能被一言说穿，这就是诗词的力量。


读懂了诗词，也就读懂了人生。


逐字逐句地翻译，枯燥乏味地解释，并不是打开诗词最正确的方式。


何来、往何去，诉与谁、谁与共……这些要素，人物的经历，才是读懂唐诗宋词的关键所在。


苏轼「小舟从此逝，江海寄余生」写于何地，秦观「郴江幸自绕郴山，为谁留下潇湘去」作于何时，张孝祥为何会「扣舷独啸」，辛弃疾又为何要「栏杆拍遍」？


「感月吟风多少事，如今老去无成」，生活刚刚稳定的李清照，为何会伤秋悲春、心灰意冷？「老夫不管春催老，只图烂醉花前倒」，以天下为己任的杨万里，又为何会有如此「佛系」之语？


一切答案，都系于诗词人一身。


他们自幼成长的家庭，毕生所处的时代，失之交臂的金榜，擦肩而过的爱情，遭遇的所有坎坷，经历的一切幸运，都会变成文字和音韵，全部写进诗词里。


诗人词人，才是解开唐宋的终极密钥。


本次课程，就是以诗词人为主，采用「词人+作品」的方式，结合南唐覆灭、庆历新政、王安石变法、元佑更化、建炎南渡、崖山海战等重大历史事件，演绎 37 位最具代表性诗词人的传奇人生，写尽悲欢离合、荣辱浮沉以及国仇家恨，带你重回唐宋，探析古人的性格与命运，领略文人的气节与风骨，读懂诗词人的才华与作品。
