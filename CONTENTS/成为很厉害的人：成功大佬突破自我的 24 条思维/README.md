## 简介
跟大佬学高效人生经营术，23 种成功路径，帮你突破瓶颈，找到发展最优解，打开更广阔的人生视角。


太多成功的人本身就是一本行走的教科书，我们要做的，只是学习。


比如


很多人不知道如何驱动团队，如何说服别人。


马云在创业初期，天天喊着我们上市了就如何如何，为什么他的员工没有认为他就是一神经病，而就连蔡崇信这样见过大世面高学历的人才都能被马云忽悠进去？


很多人有强迫症，工作中非常受影响却无可奈何。


乔布斯的强迫症远近闻名，那么他是如何将性格劣势——极端的强迫症用于工作中，产生令人惊艳的产品的呢？


很多人以性格内向为自卑不知如何改变。


技术出身的马化腾，性格内向，不善言辞。可是他是怎么熟练处理复杂人际，并一手建立起了自己的商业帝国的呢？


要学的太多，但问题是，如何学呢？


我们真正要学习和效仿的是高手的思维模式和行为方式。


本课程的主讲人鱼樵作为国内顶尖商业记者，在与他们接触和观察的过程中，会把每个大佬身上的人物特性提炼出来。


比如马云的「会忽悠」，丁磊的「文艺范」，巴菲特的「判断力」……


再与心理学、管理学等知识和现代人的生存现状结合起来，转化为我们普通人都能用的思维方式和做事方法。


不讲鸡汤，只讲策略和方法。不谈成功，只注重人生的转折和细节。


用他们的实战解决我们的问题，让你持续性的效仿、学习、积累、执行。


带你在大佬身上挖金矿， 在最短的时间内，最高效地吸收高手最有价值的精华。少走弯路，撬动自己的个人成长。
