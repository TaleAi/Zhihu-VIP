## 简介
集合穿越、沙雕、虐恋的古言短篇。


设想一下，如果穿越成为虐文的女配，你要怎么逆袭？


如果你是大奸臣的女儿，你会如何选择？


如果一个好看强大的病娇喜欢你，你该怎么办？


也许这些不按套路出牌的故事，能带给你另一种精彩。


《我来姑苏，不做妾》


原本开评弹坊的她醉酒落水，一朝穿越回旧时苏州。年少时受尽嫂子虐待，温润如玉的男主成为她熬过苦难日子的支撑。她在这场爱里倾尽所有，却不愿委身做妾。


《我不做将军夫人好多年》


她魂穿成了炮灰女配，因家族联姻嫁给了不爱自己的大将军。但她可是个拥有现代思想的独立女性，不仅没有受情伤，反而跟男主斗智斗勇，把人虐得死去活来。


《见鹿》


她是个小丫鬟，重生后成为前世自己伺候的小姐，却面临即将被害死的既定结局。她周旋在深情男主与恶毒女配之间，她的目标只有一个，活下来。


……


那些前尘往事纠葛，都化作后来的欲说还休；那些看似易得的简单幸福，往往藏着有心人的良苦经营。在套路与反套路的情网中，逃不脱的，始终是故事中的人。
