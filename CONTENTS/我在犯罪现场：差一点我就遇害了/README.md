## 简介
那天我在现场，目睹了人间种种惨状。


犯罪故事天天都在发生，而受害人大多只「活」在新闻里，受害人内心如何挣扎、抉择，外人很难得知。


本专栏集合了各种受害人亲历故事，以第一视角来还原犯罪现场：被追杀、被绑架、被抢劫......


* 某电视台编导加完班，回家时发现被杀的男友尸体吊在衣柜，而凶手正虎视眈眈地猫在她家，预谋着下一步的行动....
* 一女孩跟随父母回老家探亲，却在婶婶家里，撞见一个被铁链锁住的伤痕累累的少女，女孩「啊啊」地开始求救……
* 一银行柜员正坐在柜台办理业务，突然来了个劫匪，直接把汽油喷进柜台，并丢进一个打着的火机。瞬间，柜台烧起来了！
* ……

受害者往往也容易成为犯罪者，于是有人选择了报复，有人选择了转行，有人选择了自首......


生活不易，唯有承受。


本专栏皆为真实故事，为保护当事人已化名。
