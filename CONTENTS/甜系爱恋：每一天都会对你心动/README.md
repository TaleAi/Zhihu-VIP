## 简介
一看到你，心脏就怦怦直跳，你听到了吗？


从懵懂到心动，从暗恋到明恋。  

你是救赎，是理想，是心底不敢触碰的梦。  

我想告诉你，我的心脏为你而跳动，请跟我谈个恋爱吧。


《那一刻心动》


我的绯闻 cp 和我成了同班同学，所有人都以为我们是一对儿，可我们只是朋友。


当我要澄清的时候，他按住我的手，微微一笑：「不如，将错就错？」


《炽热温度》


校霸说我和他有一段情，我觉得他在胡诹。


可他一直缠着我，说什么要帮我找回记忆。


好像……的确……我和他有过一段情。
