## 3.什么是早泄? 你需要一个科学的认识
早泄属于最常见的射精障碍，总患病率估计为 20%-30%，说明「快男」还是比较普遍的。今天我们重点讲一下早泄，希望给大家一个全新而科学的认识。


**早泄的诊断标准，我是不是真「快男」？**


早泄又称为「较快或过早射精」，具有以下三个特征：


**射精特别短；**


**短到没法控制；**


**双方都不满意。**


因此，早泄比较严格的诊断标准也是 3 项：(1)男性射精潜伏期(ejaculatory latency time, ELT)≤1 分钟左右；(2)射精无法自控；(3)患者和或性伴侣出现心理苦恼，例如：痛苦、烦恼、挫折和/或回避性亲密行为。必须满足以上 3 个标准时，方能诊断为早泄。


*小知识：对大多数男人来说，射精前的平均时间大约是五分钟。*


知道了诊断标准，我们再来看看生活中几个真实的案例。


**案例 1**：「小 A 是一个成功的商人，生意经营的不错，应酬不少，经常陪客人喝酒。最近他发现偶尔几次的同房当中，虽然硬度正常，基本上五分钟左右就射了。这种情况在过去的半年当中发生了两次，当这个次数达到第 3 次的时候，他感觉非常苦恼，四处求医，最后去了一家私立的男科医院，经历了手术和各种调理，前前后后花费了不止 10 万，感觉还是没效果。看到这个案例，你可能会问，小 A 这是早泄吗？」


**点评：**我们知道要诊断早泄，必须符合 3 点，小 A 这不是早泄，虽然感觉射精无法自控，还伴有一定的心理焦虑，但时间没有＜1 分钟。小 A 这种状态，不认为是一种病态，算是一种正常的性功能波动。如果减少应酬，少饮酒，改变不良生活习惯，以及积极调整心态，或许性生活时间会有很大提高。


**案例 2**：小 B 是个健身教练，平时身体非常好，最近他发现性生活时间变短了，其实没有非常严重，从半个小时下降了十分钟左右的时间。但他觉得自己之前非常厉害，最近控制力没有以前好了。又因为他一直身体非常健康，因此当他发现这个问题之后，变得非常紧张和焦虑，赶紧到门诊来看。这是早泄吗？


**点评：**小 B 这种情况根本就不是早泄，仅仅是他自己感觉射精控制力没以前好了，出现紧张和焦虑，有点「杞人忧天」的感觉，对早泄出现了错误的认知。需要依靠心理治疗，纠正患者的错误认知，必要时进行心理辅导。


**案例 3**：小 C 今年 28 岁，因为早泄问题，前前后后谈过几个女朋友，可是一旦进入亲密期，她们就会发现小 C 有严重早泄，基本上性生活都是秒射（不到一分钟），不能自控，后期双方之间的感情慢慢的就淡了，直到最后分手，无一例外都是这种情况。小 C 心理上逐渐有了变化，害怕相亲，害怕同居，害怕做爱。小 C 也试图找到解决方法，无论是物理锻炼还是吃补药，结果还是没有改善。


**点评：**小 C 这种情况，属于早泄，符合(1)射精≤1 分钟左右；(2)射精无法自控；(3)患者和性伴侣出现心理苦恼。需要正规就医，必要时进行药物和心理治疗。


**案例 4**：小 D 是个夜班人员，经常是晚上上班，白天睡觉，生物钟也随着他的工作而改变。熬夜的环境也让小 C 养成了抽烟喝酒的习惯，平时一般一天一包烟，二两白酒。最近他就发现和妻子的性生活出了问题，妻子很不满意，一个是秒射，另一个是勃起硬度下降。表现为有的时候可能插不进，即使勉强插入了也会在中途疲软，甚至会在疲软的时候射精或者精液流出来，而不是射出来。


**点评：**这个案例中小 D 属于早泄合并 ED（勃起功能障碍）。ED 和早泄并非一码事，大约 30% 的早泄男性合并 ED，常常导致尚未完全勃起就过早射精了。通常先治疗勃起功能障碍，ED 改善后，一般早泄的问题也会迎刃而解（具体治疗见第四节）。


**早泄的全球形势不容乐观**


根据上面这种比较严格的定义，大约 4% 的男性存在早泄，但在实际调研中发现，抱怨射精过快的男性竟然高达 30%，明显「理想与现实」不符，说明我们可能低估了早些的发生率。


此外，基于文化差异和对待性的态度不同，早泄的患病率在不同国家和不同族群中也有所不同，你应该猜到了，亚洲国家的早泄患病率往往较高，而欧洲国家的患病率通常较低。


**早泄是怎么导致的？**


实际上，现代医学研究发现：在大多数早泄病例中，病因并是不明确。负面的条件反射和阴茎高敏感性可能是早泄的致病因素，但这两个机制到目前为止尚未得到足够的实验支持。此外，像抑郁、焦虑等心理因素也可能对早泄有影响。


说到这里，估计有很多人非常关心一个问题：手淫（自慰）是否会造成早泄？


从上面讲到早泄的发病机制上来看，自慰似乎能够关联其中（负面的条件反射+焦虑），可能会影响早泄的发生，但是至今没有很充分的证据。鉴于，亚洲国家的早泄患病率往往较高，我到觉得，我们对于自慰和性的认识是不是出了问题？


很多医生和学者，都认为手淫是很正常、很美好的一件事情，很多人迫于外界压力以及自慰的羞耻感，给自己带来了巨大的心理负担和焦虑，从而可能通过心理系统影响到了我们的射精过程。如果我们改变了对手淫的认识和态度，摘下看待自慰的有色眼镜，或许很多早泄会改善，因为手淫非但不会导致早泄，手法的快慢缓急训练反而对延长射精时间有效。


**安慰一下快男，医生眼中的早泄**


其实，在我的一本正经的理解中，多数年轻人发生的早泄根本就不能算是一种疾病。知名泌尿外科医生下下老师曾经说过：「其实早泄是写入人类基因的。原始社会，做这事一定要越快越好，如果野外环境，还要充分缠绵享受，说不定旁边有个狮子、灰狼之类的盯着你呢。好吧，速战速决是上策，完事赶紧跑。进入文明社会后，这事才有了私密性，也才有条件慢慢来，充分体验其中乐趣。」


当然，如果严格按照上面国际性医学学会对于早泄的定义，早泄是「病」，因为影响到了人的生理、心理，以及家庭的和谐。有早泄的问题，还是要治疗的，也是能够治疗的，下面我们来重点聊聊。
