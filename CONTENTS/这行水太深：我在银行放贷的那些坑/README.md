## 简介
银行水最深的部门就在信贷部：油水多、风险多、涉案多。都说钱是万恶的源泉，在巨大金钱利益面前，更是在考验着人性的善与恶，骗贷事件也因此层出不穷。


我是边江，银行从业人员，7 年来一直战斗在银行信贷业的第一线，见证过房地产公司的潮起潮落，也饱尝骗贷客户失联的辛酸；给朴实的种瓜农民雪中送过炭，也把岌岌可危的小企业推入过鬼门关。


名利不是怪物，银行本就是金钱的母体，但终日面对诱惑，银行人注定要面对比寻常人更艰难的抉择。


在这个严密缝合的庞大机器中，时时会有罪恶和贪婪滋生：


法官为了逃避担保罪责，不惜将合同吞入口中；


办事人员为了回扣和业绩，将贷款放给并不存在的客户；


客户经理为了收回一笔逾期贷款，往往搜肠刮肚极尽所能，甚至大半夜受到死亡威胁；


而也有的人明明已经去世两年，却还每月定期从银行取钱；


......


凝视深渊太久，深渊将回以凝视。希望你看完这些人性贪婪的骗贷故事，能更加恪守原则，驱逐内心的恶念。当你战胜了那些欲望和挣扎，就会变得更加强大。
