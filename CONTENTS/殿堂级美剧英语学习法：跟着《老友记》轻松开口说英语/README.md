## 简介
本课程由资深同声传译周邦琴 Albert 老师主讲，通过讲解英语口语入门经典美剧《老友记》，在剧情发展中学习正宗的英语口语表达方式。


  



**学习英语多年，还是不敢张口说英语？**


常用单词都认识，使用时却说不出？


考试时分数很高，一练口语错误不断？


看得懂英文资料，听不懂老外聊天？


 


是你没有找到正确的英语口语学习工具


**选对素材，精听精练，有效输出**


**才能真正提升英语听说能力**


现在，和 Albert 一起


用美剧口语学习排行榜稳居第一的《老友记》


轻松张口说流利英文


 


为什么老友记可以是升英语听说能力？
-----------------


**经典性**


* 《老友记》是 90% 学习口语的入门美剧
* 20 多年来经久不衰，堪称殿堂级美剧
* 史上多次高居美剧排行榜榜首

 


**实用性**


* 剧情覆盖生活中各个场景，整部剧以人物间的谈话为主
* 语言接地气、生活化、发音标准，对提升听力和口语有很大帮助

 


**趣味性&文化性**


* 经典美式情景喜剧
* 轻松有趣，助于增强对语言的理解记忆
* 轻松学习英语的同时了解美国文化

 


课程亮点
----


**第一步｜创造语言使用场景：模拟交际**


筛选《老友记》40+生活应用场景，


全面模拟、实操国外日常生活。


  


**第二步｜激活现存单词储备：植入高频实用句型**


梳理英美母语者每天挂在嘴上，


我们却 70% 都不知道的简单句型，


把原有的单司储备串联起来。


 


**第三步｜打破中英转换习惯：培养英语思维**


解析英美构词思维和表达文化，不用学语法，就能摆脱 Chinglish。


 


**第四步｜训练发音习惯和本能：拆解、模仿、跟读**


分为常速朗读+慢速朗读，一快一慢，雕刻你的发音习惯；


强化多种音变形式，感受语调和节奏在表达思想中的作用和震撼。


  



你将获得
----


1. 无障碍英语对话，搞定日常交际
2. 无字幕听说习惯，轻松看剧听广播
3. 标准地道英语发音，展现交际自信和魅力
4. 不学语法，也能表达西式逻辑
