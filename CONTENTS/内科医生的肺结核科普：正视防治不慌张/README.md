## 简介
本专栏由阿源老师为大家科普结核病的相关知识，帮助大家了解、认识、治疗和预防结核，消除大家对结核病的恐惧。


突然开始频繁咳嗽，是不是患上肺结核的「警告」？


接触到结核病人后需要吃药吗？怎样才能避免被传染？


结核病到底是如何确诊的，现代医学有怎样的筛查手段？


得了结核病还能不能治愈呢？


结核病已经成为世界上传染病中的第一杀手，由不得我们不重视。虽然经过多年的努力，国内结核病的发病率和新发、复发病例都呈下降趋势，但是患病总数依然位居世界前列。


对结核病以及抗结核药的恐惧，常常让人害怕就医、害怕用药，有的人因此拖到了重症，甚至导致家人被感染，给自身和家庭都带来了伤害，甚至因此一蹶不振，觉得自己这辈子就毁了。


因此，本专栏请到了阿源老师为大家科普结核病的相关知识，帮助大家了解、认识、治疗和预防结核，面对不同的筛查结果应该如何处理，结核患者需要如何「养生」，希望大家可以正确认识结核病，消除内心对结核病的恐惧。
