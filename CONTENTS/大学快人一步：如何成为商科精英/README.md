# 大学快人一步：如何成为商科精英

## 简介
专业选择，志愿填报前必看不可的史上首本专属商科学生的职业规划手册！
本专栏聚焦对商科领域感兴趣的高中生及大学生学生，既可以为选择志愿的学生进行方向指导，也可为已经入学的大学生提供职业生涯规划。专栏详细介绍了涵盖金融、财会、市场营销、咨询、商业分析、供应链管理、人力资源管理、行政管理、酒店管理等方向的热门商科专业，细致分析了各个领域的优缺点、与自己的匹配度和发展潜力，作者结合近十年来数千小时授课和数万学员职业规划咨询心得、各行业调研成果，献上深入行业的一手资料，以启发式讲解为导向，不单有犀利的观点，更具问题思考方式上的指导作用。


以数据和案例为核心，拒绝传统职业规划的「干讲大道理 心灵鸡汤 成功学」。


读完本专栏，宏观上，你将拥有更平衡的生活与职业发展问的人生定位； 微观上，你将更明晰该如何做好职业定位和规划，如何锁定自己的「真爱」，如何在战略之下将每一步规划执行到位。
