## 简介
美人多深情，君子多好逑，帝王将相，英雄豪杰，一遇粉黛尽低头。
被家族抛弃郎君屠戮、被逼做棋子卷入阴谋、被丈夫当成筹码送入敌营……女人本倾国，奈何又情重！
在男权当道的古代，她们是否能拨云见日，破茧成蝶寻回幸福？


**《美人思倾国》**  

穿成失宠贵妃，开局就险些被皇帝勒死。面对时刻准备灭她全家的皇帝丈夫，随时有可能挂掉的老爹，和一后宫想弄死她的莺莺燕燕……看她如何翻云覆雨，搅动朝堂！


**《江山作聘》**


谭娇娇最后悔的事，就是掳走了摄政王世子，还强行入了洞房！


结果一着不慎竟然落入了他的圈套，她为了活命只能忍辱负重！


朝堂刀光，江湖剑影，当海河晏清而他以江山为聘时……谭娇娇跑路成功。


**《卿君侧》**


身为燕国皇后，却被皇帝亲手送给敌军首领当礼物。刀架在脖颈，她以骨肉血亲和他做了交易，却是满盘皆输家破人亡……


哀莫大于心死。她凤凰泣血，复仇还朝……从此，再没有人能够拿捏她！
