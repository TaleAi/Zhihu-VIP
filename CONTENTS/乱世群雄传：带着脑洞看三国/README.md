## 简介
带着脑洞看三国，古今多少事，皆在笑谈中。


在人们的印象中，三国是一个群雄逐鹿的时代。


这个时代，英雄辈出，纷争并起。


给我们留下深刻印象的历史故事数不胜数。


但在这些故事之外，其实还有更多事。


白帝城托孤，阿斗继位后，他是怎么治理这个国家的？果真是乐不思蜀？


诸葛亮鞠躬尽瘁一生，工作事业之外，面对家庭又有怎样的生活?


张飞长坂坡吓退曹军，勇武过人，但在张苞的眼中，这是一位怎样的父亲？


跟着知名脑洞作者寸君，看看那些三国演义之外的传奇故事。
