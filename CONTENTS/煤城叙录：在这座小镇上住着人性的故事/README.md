# 煤城叙录：在这座小镇上住着人性的故事

### 作者：田烨然

### 分类：悬疑推理 · 真实故事

### 状态：已完结，共 10 节

### 存档日期：2022-06-12

### 原始地址：https://www.zhihu.com/xen/market/remix/paid_column/1251835042290393088


## 简介
在这座以产煤为主的小镇里，各色各样的普通人用自己的方式生活着，那里有爱情，有亲情，有魔幻，有伤感。


《那座煤矿的收购，只花了一块钱》


在国家的煤矿资源整合政策下来之前，县城里的收矿人无所不用其极，他们没有王法，只有办法。


《那个国道上一百五十块就可以带走的小姐》


李老二去洗浴中心泡了个澡，来了个小姐。一场意外以后，他带着她跑了。警局里，他说要娶她。


《送您最后一块烧白起》


烧白起就是烧豆腐，是当地的特色小吃。老李是做烧白起的一把好手。忽然来了文化局要评非遗传承人，又来了年轻人小刘说要跟老李学手艺。老李忙忙活活而后病了俩月才发现是个局。


《英雄、劫匪、便利店》


我对一个便利店有深刻的感情，在妻子带着孩子跟我离婚以后，我打劫了便利店。恰好，便利店里的几个人质跟我都有着几分关系。不要考验人性。


《窈窕姥爷》


姥姥过世了，姥爷陷入了巨大的悲伤。一日，姥爷突然穿着姥姥的衣服走了出来，学着姥姥的语气跟我们讲话、做饭，行为举止全变了。自那以后，姥爷就变成了姥姥。


《煤运昌盛》


从矿上的挖煤工一路做到矿长，他经历的与黑社会斗争、和红灯区小姐结婚、应对一场没有发生的地震、矿井坍塌事故等等。


《南街往事（上）》


高中的一段时间，我的父母出差，我被寄宿在舅舅家。舅舅家是幢三层小楼，一层自住，二三层隔成了 20 平米一个的小间，一起二十多间。没错，是做红灯区的营生的。里面有一个我管她叫乐乐姐，跟别人都不一样，身上有仙气。据说是因为家庭中落，从大城市沦落到这里的。


《南街往事（下）》


非典来势汹汹，在这个时候，一个嫖客突然身亡。嫖客的儿子找到皮条商油焖叔，非说是父亲可能是得非典死的，肯定是那个姑娘给传染的。非不依不饶拉着一干小姐去医院检查身体。结果小姐们查出来什么病也没有， 连感冒也没几个。消息传开，油焖叔的生意更好了……


《三万块了无影踪》


为了 900 块，被骗三万块的全过程。


《90 后货车司机的日与夜》


这是一个普通的 90 后货车司机的挣扎。
