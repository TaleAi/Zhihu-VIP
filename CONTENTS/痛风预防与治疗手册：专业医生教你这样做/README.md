## 简介
什么是「痛风」？如何止痛、降尿酸、防止复发？


当下社会，生活水平提高了，很多人也被如影随形的痛风疯狂纠缠多年，无法抽身。


痛风，像一个刻骨铭心的情感骗子，它来的时候，你不知道怎么去拒绝，它走的时候，你不知道怎么去自爱。


其实，痛风是一个挺简单的疾病，让我们整得特别复杂。


这事不单单医生有「责任」，更多的是作为患者太迷茫，不知道什么是痛风、不知道发作时如何止痛、不知道不痛了如何降尿酸，更不知道怎么去科学预防？


本专栏由知名医学科普作家、上海长海医院肾内科陈医生主讲，重点从科学认识尿酸与痛风、痛风发作时和缓解后的治疗，饮食、运动等生活中应该必知的预防等几个方面，做「看得懂、用得上」的专业解析，教会大家轻松攻克痛风这个顽疾。


如果你是一位被反复痛风折磨得死去活来的老患者，亦或者是刚经历过刻骨铭心之痛的痛风新人，希望读完本专栏，大家可以对痛风有一个科学的认识，怎么预防和发作时的治疗都可以弄得清清楚楚、明明白白。
