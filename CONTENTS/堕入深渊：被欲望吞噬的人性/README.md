## 简介
天网恢恢，疏而不漏


他们并不是天生的恶人。


如果命运的车轮稍稍转个方向，或许，他们就会走上另一条人生路。


但是人性中羸弱的一面，最终占了上风。


面对利益的引诱，欲望的沟壑，他们最终选择了营私利己。


* *包子铺老板，被利益熏了心，罔顾人命。*
* *烧腊店员工，觊觎上了本不属于自己的东西，狠下杀手。*
* *学业优异的研究生，只因情商低下，与上司发生龃龉，最终举起了屠刀。*

人性总是复杂的，恶意总会在某个角落蠢蠢欲动。


本专栏将通过纪实的手法，还原案情的真实细节，揭露人性中潜藏的黑暗面，警醒世人，任何时候，都要保有防范之心，以免陷入危险的境地。
