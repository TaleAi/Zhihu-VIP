## 简介

你是不是有以下这些情况？


眼角、唇周、脸颊有小小的细纹；照镜子时常常觉得自己「憔悴」，但不知道问题出在哪；


很用心护肤了，尝试了各种并不便宜的护肤品，但皮肤并没有变好；


想尝试医美项目，期待医美解决自己的各类皮肤问题，却又不敢迈出第一步；


……


对于「抗衰老」，多数人都有下面这些误解：


**误区一：「抗衰老」= 花钱，要用贵的护肤品+做医美**


实际上，对「抗衰老」来说，真正重要的事情，都是免费的。想要做好抗老工作，并不需要花很多钱。


反而护肤品和医疗美容，能起到的作用有限。如果不懂得甄别，反而可能因为使用了错误浓度的产品，或者混用了不合适的成分，导致皮肤变薄敏感，反而加速皮肤老化。 


**误区二：「抗衰老」是光鲜亮丽的明星才要考虑的，普通人就算了吧**


普通人更需要抗衰老，预防衰老。越早具备「抗衰老」的意识，越能留住时光、保持健康活力状态。年龄只是数字，预防衰老是每个人保持健康的必修课。


因为，「衰老」不仅是体现在外表的年轻，更是整体身体机能的减退。做好抗衰工作，不仅能让人外表更加年轻紧致，更可以让人少生病、有活力。


**实际上，抗衰老是一个由内而外的综合体系。除了美容护肤方面，更重要的是改变内在的「生活方式」。**  


本专栏中，周倩医生提出了独创的「抗衰金字塔」理论。结合生活方式、饮食和睡眠、情绪管理、护肤、医疗美容等多个方面，打造了一套普通人也能从中获益的抗衰老方案。
