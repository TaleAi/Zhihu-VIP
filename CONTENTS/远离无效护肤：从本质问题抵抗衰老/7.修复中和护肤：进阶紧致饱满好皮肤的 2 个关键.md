## 7.修复中和护肤：进阶紧致饱满好皮肤的 2 个关键
你好，我是周倩。


今天我们继续往金字塔往上进阶，详细聊聊在它的中层我们需要做些什么呢？


![img](https://pic2.zhimg.com/v2-80145098872fd6a6f90aca64a3e0c0ae.webp)

**修护：保证皮肤屏障健康的基础**


在护肤的第二层，首先要考虑的是「修护」环节。


最近几年，在我的公众号后台咨询皮肤敏感问题的人越来越多。仔细了解之后我发现，有很多「敏感」的情况，是不正确的护肤方法「作」出来的：


有的是因为不考虑皮肤的耐受度就往脸上抹各种高浓度的果酸、维 A 酸等「猛药」，还有的是想要快点改善皮肤问题，往脸上叠加一层层的美白精华、抗老精华、祛痘精华等等。


结果造成了皮肤屏障受损，脸部敏感泛红、刺痛、出现红血丝，越护肤，皮肤反而越差。


**皮肤屏障如果受损，就相当于房间没有了外墙和窗户，皮肤面对伤害的防御力大幅下降，稍微一点紫外线、干燥、炎热等情况，就会伤害到皮肤。**


这时候，皮肤需要耗费大量的精力来自我修复，如果这时候还不能给皮肤自我修复的机会，继续上「猛药」和各种功效性的产品，那么皮肤在炎症和敏感中恶性循环，反而更容易老化。


有很多成分都能帮助皮肤修复，这里我想重点说一个成分：神经酰胺。


神经酰胺其实是我们皮肤中天然存在的一种成分,也是皮肤屏障(角质层)中非常重要的组成部分。


神经酰胺有着非常好的渗透性和亲肤性,能够快速的渗入皮肤表皮细胞并与角质层中的水分结合,形成一道锁水屏障，保护我们肌肤到同时，使皮肤柔软富有光泽。


此外，神经酰胺还可以辅助后续的美白，抗炎抗氧化，是有效对抗肌肤衰老的好帮手。

**抗炎抗氧化：选取功效性产品搭配使用**


我们再来说说「抗炎」和「抗氧化」。


在「衰老三元凶"中，比较有效的抗糖手法目前只推荐少吃糖、多运动。但通过护肤来进行抗炎、抗氧化，还是可以取得一定效果的。


如果你近几年来关注过护肤品的广告，一定对这两个词不陌生。


**现在市面上抗炎抗氧化的产品其实非常多，比如维生素 C、E、虾青素、辅酶 Q10、维生素 A/C/E、SOD、谷胱甘肽、艾地苯、富勒烯、还有成分界的万金油-烟酰胺等等，这些都是市面上被广泛使用，研究数据非常多的一些抗氧化成分。**


**此外，姜黄、葡萄籽、茶叶等等含原花青素、多酚一类的植物提取物，也都具有不错的抗氧化、抗炎修护的作用。**


![img](https://pic2.zhimg.com/v2-93b2cd895ce01245ea2fc9141b63efae.webp)

这节课，我想重点跟你分享抗氧化届的常青树——维生素 C 和维生素 C 衍生物。


维生素 C 具有很强的还原抗氧化能力，具有清除自由基和抗氧化的作用。而且，维生素 C 还能通过抑制酪氨酸酶的活性，达到美白效果。


**这里我想多提一句，如果你营养均衡不挑食，那么没必要口服维生素 C 补充剂。**


因为过量维生素 C 并不会被人体吸收，反而会分解代谢为草酸随尿液排出。长期过度服用可出现草酸尿以致形成泌尿道结石。如果每天都补充大量维 C，可引起腹泻、皮疹、胃酸增多、胃液反流.......


当然，近几年大火的酸类成分，比如果酸、水杨酸，也具有一定的抗炎特性。这类成分的原理通过剥落老旧角质，加速表皮细胞新陈代谢，在抵抗初老方面也是很「能打」的。而且对油性皮肤来说，刷酸还能起到改善皮肤出油、黑头粉刺等问题的作用。


**但作为医生，我不推荐大家自己在家盲目跟着网上各种「教程」进行刷酸的尝试。**


毕竟酸的浓度、配方和类型丰富多样，稍有不慎就可能损伤皮肤屏障。如果你想要刷酸，我更建议你去皮肤科，在专业医生的指导下进行。


![img](https://pic1.zhimg.com/v2-3d5d6c74640afdea753ab7e83d468cab.webp)

**小结**


我们来总结一下：


· 在做好了基础护肤后，可以根据自己的肌肤情况选择修复或者抗炎、抗氧的进阶需求。


· 修复肌肤的要点是不要「作」，选择精简成分的护肤品，可以尝试含有神经酰胺成分的产品。


· 抗炎抗氧的成分有很多，其中维生素 C 是最经典和最常见的一种。尽管功效强大，外用也足够了，如果你平时营养均衡，没必要口服维生素 C 补充剂。


· 部分酸类也有一定的抗炎作用，但不推荐你在家自己刷酸，一定要在专业人士的指导下进行。
