## 简介
本课程精选鲁迅五部经典短篇，以原文朗读和文章解析的方式，深度解析小说情节与人物，带你走进中国现代文学奠基人鲁迅大师的精神世界。


 他是白话文学的开山之人，《狂人日记》打开白话文小说大门； 


他是难以逾越的文学高峰，作品在教材课本中一直屡见不鲜； 


但他始终怀着「甘为孺子牛」的谦卑，写下令一众「正人君子」所深恶痛绝的文章； 


一声呐喊，万众彷徨，他是时代里闪耀的明灯。 


鲁迅先生的文字在很多时候都能恰如其分的揭示事物的本质，给人以深刻的指导意义。网上流传过一句话，「如果你不知道某句名言出自谁，就说是鲁迅说的。」虽为戏言，也更加证明鲁迅先生所写文字是何等的一针见血，入木三分。每个年代的人都要读一读鲁迅，今天也不例外。 


本课程精选鲁迅五部经典短篇，以原文朗读和文章解析的方式，深度解析小说情节与人物，带你领略一代文学巨匠的深刻文字，走进中国现代文学奠基人鲁迅大师的精神世界。
