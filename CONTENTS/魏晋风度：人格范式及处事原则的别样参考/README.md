## 简介
本专栏精选魏晋时期的曹操、曹丕、曹植、阮籍、嵇康、陶渊明等 28 篇精美诗歌，搭配优美赏析，带你领略魏晋风度。


每个人到最后千篇一律无非都是死，为什么还有那么多人奋不顾身地使自己的生命绽放升华？


魏晋时期是中国历史上少有的思想解放、文艺繁荣的时代。那个时代的士人们尊崇思想独立与行为自由的人格范式，绽放自己生命的魅力，


他们


或慷慨悲壮如曹操，写下了「慨以当慷，忧思难忘」的诗句；


或虚浮放诞如刘伶，戏谑地「以天地为屋，以房屋为衣裤」，纵酒佯狂；


或豪爽清奇如嵇康，激愤地向山涛写信「绝交」，临终时却托孤于他；


或雅致通达如陶潜，「采菊东篱下，悠然见南山」的隐居精神，被后世奉为「古今隐逸诗人之宗」，影响深远。


![img](https://pic3.zhimg.com/v2-fb2d5d13edc47b29d175ff300947031b.jpg)

魏晋风度不仅是一种混杂着士人的现实痛苦和理想追求的矛盾体，也是一种文化人在乱世中追求真理和追求自我的处世原则。


本专栏带你走进这些文人的内心世界，领略真正的魏晋风度。
