## 简介
黎明前，誓与敌人斗争到底。


在本专栏中，你将读到建党百年来那些令人拍案叫绝的红色奇案。


他们是散落于历史尘埃中的理想主义者，他们走上革命之路的那一刻，便做好了随时随地为信仰而牺牲的准备——


1931 年，七个中共地下交通员执行绝密任务，从苏区瑞金运送黄金到上海，不料最后一环失败。此案让中央记挂 18 年。真相揭开，一段隐姓埋名的地下党传奇往事被世人所知；


1935 年，300 名红军士兵越过六盘山，却在一夜间离奇失踪。苦苦追查了 57 年真相的中央，在得知战士们死因的一刻，无不潸然泪下……


1940 年，毛主席的秘书沈辉潜逃，三十年后，一个叫沈之岳的「特务之王」在台湾公之于众，将曾经发生在延安的惊心动魄间谍案揭秘。


1934 年，一个衰弱的叫花子抵达中共中央驻地瑞金，倒下前仅凭最后的意志把四本字典交给联络人。收到字典后，中央连夜部署人员将字典里潜藏的密码字誊抄出来。当四本字典全部誊抄完毕，一个偌大的围剿计划浮现。
