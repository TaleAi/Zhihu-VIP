## 3.测一测你的社恐指数
上一节我们提到，由于我们的主观感受并非准确，所以对社恐症的测量也需要借助测评工具，从认知、躯体和行为三个角度来评估一个人是否是真正的社恐患者。


考虑到对躯体和行为的评估方法受到时空的限制，无法做情境反应观测，所以在这里我们推荐几个认知测评工具：目前国际上比较受认可的认知测评量表有简明标准恐惧症自评量表（Marks, 1979），社交恐惧和焦虑量表（SPAI），LSAS 量表，和社交恐惧量表（SPS）、社会交往焦虑量表（SIAS）。


其中社交恐惧量表（SPS）在临床应用中效果较好、也是广泛使用的量表之一，根据下列题目，选出最符合你感受和行为的选项，可以获得测评结果。


1、经常想到亲人会有不幸？


是 否


2、有时担心会给自己或所爱的人带来伤害？


是 否


3、经常检查灯和水龙头关好没有？


是 否


4、在人群中受到推搡觉得反感？


是 否


5、有洁癖？多次反复地刷洗衣服和家具？是否老洗手？


是 否


6、是否老是对自己和自己所干的事不满意，尽管已经努力想干好？


是 否


7、你是否总是尽量想提前离开有可能使你遭遇尴尬的境地？


是 否


8、是否能轻易做出困难的决定？


是 否


9、你是否觉得有一种做某种多余事的必要？


是 否


10、经常觉得身上衣服有些不对劲？


是 否


11、有过回家检查门窗是否锁好的情况吗？


是 否


12、老舍不得扔掉已没用的旧东西？


是 否


13、是否老在想一般都是不由自主做的事？


是 否


14、是否有过老重复说同一句话或数一些没有必要数的东西的时候？


是 否


15、睡觉前会把衣服整齐码好吗？


是 否


16、你是否干一些不重要的事也很认真？


是 否


17、你周围的东西是否随时都要放在同一个地方？


是 否


18、会老做一些无足轻重的老动作吗？


是 否


测试结果结算：请从做出「是」的回答数的多寡看出其患上恐惧症的程度。


0~4 个说明是正常范围。


5~10 个说明患有轻度恐惧症。


10~15 个说明患有中度恐惧症。


15 个以上就得注意了，有患上潜伏性神经官能症恐惧的危险。
