## 简介
你有社交恐惧症吗？活跃于网络社交平台，一到现实却变成社恐青年？实际上，社恐者的恐惧来源，不止是真实社交环境中的种种难题，更多是来自社恐者自身的心理问题。本专栏带你一起了解社交恐惧症，通过心理学层面的深层探索，掌握与人沟通的技巧。


你有社交恐惧症，或是社交焦虑症吗？


为什么活跃于网络社交平台，一到现实就变成了社恐青年呢？


究竟是天生抵触社交，还是现实社交能力的缺失所造成的？


当我们因为惧怕真实社交而转身投入虚拟社交后，我们的社交能力只会越来越差——现实社交能力的缺失，会让我们在学习、在工作中失去许多展示自己优势的机会。


实际上，社恐者的恐惧正是来源于此，不止是真实社交环境中的种种难题，更多的是社恐者自身的心理问题。


本专栏中，我们带你一起了解社交恐惧症，通过心理学层面的深层探索，掌握与人沟通的技巧。
