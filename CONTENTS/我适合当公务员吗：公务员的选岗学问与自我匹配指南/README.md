那么在这件可以影响人生的「大事」上，如何才能做出正确的决定？


本专栏邀请到黑化肥发灰老师，他曾经参加过公考 10 次，4 次进面，2 次上岸，有县、市、省三级政府部门工作经历，对体制内各单位情况都有广泛了解，同时对公务员考试报名、考试等相关问题有深入研究。在知乎，有 3600 多人次曾向他咨询过考公相关的问题，黑化肥发灰在咨询中也积累出丰富的考公真实案例。针对「公务员」大家提问过最多、最热的问题，他将在本专栏中进行系统性地解答。希望能帮助第一次参与公考的同学们，解决关于公务员考试报名、选岗等疑问。
