## 简介
不小心进入光怪陆离的书中世界，不但没有成为主角，还成了最后要被牺牲掉的炮灰。
女主表示，这趴我不可！
面对黑化的备胎男配、残疾的偏执哥哥，该如何救赎他们注定悲惨的一生？
若你水深火热，我必伸出援手。


《救赎备胎男配》 


姜如很不幸，穿入了书中，成为了一个死在变态杀人犯刀下的无辜炮灰甲。


「救命，我真的不想死。」 


作为已经了解剧情的吃瓜群众，姜如很惜命，竭尽所能地避开任何威胁生命的危险因子，可是却偏偏玩了火。


「张子鸣，如果我喜欢你，你是不是可以不杀我？」


「考虑考虑。」


于是，姜如本着牺牲小我完成大我的精神，完美拯救了一位失足黑化少年，开展了不一样的炮灰人生。


《救赎偏执哥哥》


陆灵觉得自己的人生只能用罪无可恕来形容，这种恶毒女配的套路，落在了她的头上，让她恨不得再一次掐死自己看能不能找一个好一点的身体穿。


不过既来之则安之，开局是恶毒女配，不代表轨道要按照原剧情来。


陆灵安慰自己说道。


原主的锅，她来背。


原主的伤，她来抗。


可是拯救病态少年的任务……


「齐程，我愿意用一生，来弥补对你的伤害。」


「陆灵，如果你是因为我这双腿留在我身边，我情愿我这辈子都站不起来！」


偏执是病，得治。
