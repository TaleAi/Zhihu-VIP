## 简介
投资就是在做各种选择，选择一个，就意味着放弃另一个。只有拥有投资人的思维，站在投资人的立场去思考问题，才能像投资人一样做出科学实用的决策。这门课程，告诉你什么是投资的本质，并帮你剖析股价涨跌规律，快速搭建科学投资体系，拥有一名成熟投资人才能拥有的科学而实用的思维方式。


投资就是在做各种选择，大到人生路该如何走，项目该怎么投，都是投资。


选择一个，就意味着放弃另一个。


乐观、理性、自律、还要有好奇心和想象力——这都是投资人最基本的修养。


但，大道理「知易行难」。现实当中，我们总会忍不住犯了这样或那样的错误。有人总结过散户的几大常见痛点，包括没耐心、没纪律、追涨杀跌等等。


  




![img](https://pic1.zhimg.com/v2-09224a63192d8f291c09348639d984f5.jpg)

**我们之所以一遍遍地在同一个地方跌倒，除了因为人性中非理性的「动物精神」，更重要的是思维方式上的偏差。**


**——成为一名合格的投资人，首先要拥有投资人的思维。**


只有拥有投资人的思维，站在投资人的立场去思考问题，才能像投资人一样做出科学实用的决策。


这门课程，作者将十年来积累的那些卷帙浩繁的知识和方法，抽丝剥茧并压缩而成的一门通俗易懂的课程。无论是投资老手，还是没有任何基础的「小白」，也能通过这门课的学习，拥有一名成熟投资人才能拥有的科学而实用的思维方式。


课程亮点：


* **五大思维模块**

![img](https://pic4.zhimg.com/v2-214f8df95e3ebf2634c1d7a319216af2.jpg)

宏观思维、金融思维、产业思维、财务思维、企业思维。


全面提升投资知识框架，建立优秀投资者思维。


* **基本面分析原则**

![img](https://pic2.zhimg.com/v2-44cde25ac4b6e4231e74215bb5b9391c.png)

在基本面方面，帮你提升关于宏观、行业、企业经营等方面的认知；


在资金面方面，帮你提升关于货币政策和资金流动的认知；


在情绪面方面，帮你了解重大政策、重大事件会如何影响市场情绪。
