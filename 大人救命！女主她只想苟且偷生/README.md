**- 我的保护神大人 -**


绝对不会红的美艳女明星 x 伪装成小鬼的傲娇保护神


我，傅常，世界上最倒霉的倒霉女明星。


胸不算小，腿不算短。一张脸从娘胎里还算妖媚惑人。


小小地上过某扑女神备选名单之一。


但就是火不了。


直到有一天，我养的小鬼突然出现，问我，


「你想火吗？」


  



**- 女明星倒霉事件 -**


想啥啥不灵的沙雕女明星 x 冷酷铁面吸血鬼杀手


丁丁是一个女明星，她的特异功能是「贼尼玛倒霉」+ 「心想事不成」。


幸好，她遇到了一个总裁，这个总裁帅气多金，且对她深情不悔，


但万万没想到，总裁原来早已经有了妻儿，


一夕之间，全国都知道了丁丁做小三的事情。


丁丁心灰意冷决定跳楼自杀，却在楼顶撞见了杀人现场。


而那个杀手，貌似……是个吸血鬼？！


  



**- 一脚踏碎凌霄殿 -**


克夫女 x 俊美脱俗罗汉转世


我是一个克夫女。


我嫁了三任丈夫，任任都被克死，最后一任还是我的青梅竹马。


这天我上山准备吊死自己，却遇到一个拥有天人之姿的男子，


他对我双手合十说：「阿弥陀佛，都是命运的安排，姑娘，请你嫁给我吧。」
