## 简介
「三言二拍」是中国古典短篇小说的代表作，里面讲述了古人们的爱情、复仇、鬼神故事，包罗万象。本专栏是对「三言二拍」世界的现代演绎，用通俗、幽默的方式，让读者发现，原来古人和我们很像。


县太爷竟离奇变鱼，惨遭同事分食！


少女父母为贼人所害，靠托梦里的俩谜语找到凶手！


大舅哥成为女装大佬，假冒我老婆，又成了我妹夫怎么办？


《唐伯虎点秋香》官方原著，看唐伯虎如何发糖！


史上最帅小姐姐：骑得最烈的马，射得最快的箭，撩遍男女！


本专栏采用周刊形式书写，说书口气讲述，给你古人事迹「现场报道」的还原，仿佛你就走在古代的大街上，看到、听到种种奇怪、爆笑、温馨的故事。
