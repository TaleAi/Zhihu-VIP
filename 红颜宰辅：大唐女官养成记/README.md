## 简介
「红颜宰辅」上官婉儿不是美女也不是才女，而是一个在朝堂上拼杀的政治家。


一身风雅奉君王的大唐女官上官婉儿，她是执掌中枢 30 年从不出错的最强「女秘书」。


她的一生，与李治、李显，李旦，武则天，李隆基、李重茂等「六位帝皇丸」息息相关；在属于狄仁杰、武三思、来俊臣、等千古名臣的朝堂，她长袖善舞，屹立不倒。


本专栏里的上官婉儿，褪去了美女与才女的光环——我们只讲述她身为一个属于朝堂的政治家的故事。


不屑后宫明争暗斗，偏上朝堂搅动风云，男人权势，怎敌我红颜手腕。不再耽溺于闺阁间的小情小爱，而是登上金銮殿与须眉男儿一争高下，这是唐朝女子最夺目的鲜妍明媚。
