## 简介
《道德经》在科技发达的今天依然广受推崇，它究竟讲了些什么？让我们踏着伟人的足迹，跟随曾仕强教授，一起走进《道德经》的奥秘。


「道可道非常道」，几千年来，老子的《道德经》流传甚广，不仅文人百姓口耳相传，历代帝王也无比尊崇。


![img](https://pic1.zhimg.com/v2-78a576aa1d2b9f89dfc298c779fefa95.png)

作为道家的开山之作，中华文明的智慧源泉，《道德经》文意深奥，包涵广博。


它究竟讲了什么？作为普通人,我们如何能够在轻松自在的状态下修习中华至高经典？


如何轻松领悟《道德经》的精髓？如何在日常生活和工作的修行中，运用《道德经》的智慧，参透为人处世的道理，追寻人生的意义，找到幸福之道呢？


名师讲解、正本清源；通俗易懂、妙趣横生，让我们踏着伟人的足迹，跟随曾仕强教授，一起走进《道德经》的奥秘，领悟古圣先贤的智慧在当代生活的应用之法。
