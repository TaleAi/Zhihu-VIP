# 单身女人，让男神倒追你

### 存档日期：2022-06-02

## 简介
成为婚恋关系中高质量的长期伴侣


写给这样的单身女人：


* 你是职业女性，甚至有自己的事业
* 你工作能力不错、颜值不错，不乏人追，但还是单着
* 你希望寻找彼此平等、沟通顺畅、互相激励的爱情，而不是做男人的附庸
* 你看重精神上的互动，喜欢聪明、热情、有趣的伴侣

在外人看来你条件优越，但自己还是觉得真爱难寻。这话在不知情的人看来有些矫情，但事实上，你的确面临着各种困扰：


* 社交圈狭窄，选择面小，难以遇到让自己中意的人
* 不乏追求者，但常常是烂桃花，反而让自己分心和烦恼
* 遇到心动的人，又不知道该如何把握机会
* 常常有选择错误的经历，潜意识里还是对自己缺乏信心和安全感
