《攻略直男计划》：


我在 A 大火了，就是火得不太光彩，一切还得从对面宿舍楼穿红裤衩的男生说起。


事情发生在前几天晚上，我在阳台上晾衣服，眼睛一抬，突然有个光裸着的东西闯进了视线。


是男人的肉体……那肉体移动着，似乎也是在晾衣服。


原本我该乖乖移开视线，可……那男生正背对着我，身上只穿着条四角裤。


背上线条流畅，宽肩细腰，那身材，简直斯哈斯哈！还有翘着的屁股蛋子！我眼睛不听使唤地移了回去。


我流着哈喇子，直到手中的衣服已经晾完，才恋恋不舍地进了宿舍门。


然后，我把他挂上了表白墙……


……
