## 简介
恐龙是最能吸引小朋友的话题之一。即使谁也没有真正见过恐龙，但这并不能减少我们对恐龙的探索，反而加强了对恐龙的好奇心。尤其是孩子们，对未知的恐龙世界充满好奇与敬畏，了解恐龙也是拓展思维的一种方式。


世界上最早被发现的恐龙化石，是禽龙的化石；


恐龙是最早长出羽毛的动物；


和平中华盗龙的性格，一点也不平和；


......


恐龙是一类远古生物，它非常神秘，也非常形象。


古生物学家徐星表示，在做科普的过程当中，会收到很多小朋友的问题：「叔叔，你怎么知道恐龙会跑这么快？」，或者说：「叔叔你怎么知道这个恐龙是生活在水里的？」这时，就会给小朋友举例说明，告诉他们恐龙的特征。比如棘背龙的嘴巴形状有些像鳄鱼——是用来吃鱼的。


科学家不仅能通过嘴巴骨骼的形状像鳄鱼来分析它的生活环境，而且可以通过研究骨骼里保存的一些化学元素，分析出这种动物很可能生活在水里。本专栏让孩子在课程中，参与到古生物学家的恐龙探索之旅，珍惜孩子对自然的好奇心，培养孩子科学的思维方式，严密的推理能力。
