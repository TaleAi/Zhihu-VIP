## 18.如何吃出肤如凝脂好气色？
今天已经是咱们「变美计划」课程中的最后一节了，师姐也收集了很多同学对于自己的肤色问题的反馈。


其中师姐发现很多同学都有一个集中的问题，那就是：自己的肤色太白了，看起来没有活力。很多同学和师姐当时的反应大概都会是：现在不都是以白为美么，为什么还有人担心自己太白了？


当师姐进一步问到，有没有什么其他的表现。一部分同学都会提到自己的例假周期不太稳定，而且皮肤的白并不是那种白皙的感觉，而是有点苍白。


其实，这样的皮肤表现，反映着的则是例假的问题。对于女同学来说，皮肤的状况可以说是例假问题的一面镜子。比如：皮肤苍白有可能是例假推迟造成的、皮肤长痘可能是例假提前等等。


当然也有很多同学通过查阅一些网络上的文章，就会把自己的例假推迟和肤色苍白归因于贫血，但真的是这样么？


今天师姐就来给大家详细分析一下，到底原因是什么？肤色不好的妹子吃什么食物才能调理例假周期，提亮肤色？


贫血可没这么简单


课程的开始，师姐先给大家吃一颗定心丸：虽说贫血可能导致肤色苍白和例假推迟，但反之不一定成立，肤色苍白或是例假推迟可不一定是贫血。我们来学习下关贫血的知识，就明白了。


贫血的定义


贫血在医学上的定义是：人体外周血红细胞容量减少，低于正常范围下限的一种常见的临床症状。可以用血红蛋白浓度的测量来判断，成年男性血红蛋白浓度<120g/L，成年女性血红蛋白浓度<110g/L，孕妇血红蛋白浓度<100g/L 就有贫血。


简单来说，贫血指的就是我们血液中运输氧气的「小货车」不够了，出现血液缺氧的情况。


什么原因可能会贫血


贫血的原因多种多样，有缺铁也有慢性疾病导致的贫血等等。但就原理来说，大致可以归为三类：


（1）运氧的小货车由于种种原因，如：缺乏提元素或者造血功能障碍，导致生产小货车速度变慢了，出现了火车供应不足的情况。这也就是，红细胞生成减少导致的贫血。


（2）运氧的小货车，由于大量失血，出现了遗失造成的。这就是，血红细胞丢失过多导致的贫血。


（3）运氧的小货车，由于发生了机械故障，出现了数量减少。这也就是红细胞破坏过多导致的贫血。


简单来说就是，营养缺乏、大量失血和疾病这三种影响因素，可能会导致我们出现贫血。我们可以对比下，自己是否有这样的情况呢？


我猜会有同学想到，我们每个月例假的时候，不是都会「大量失血么」，这个原因会不会导致我们贫血呢？


答案是不会！虽说在医学上，确实存在着长期例假失血过多导致的贫血的情况，但是这指的是一个例假周期失血超过 80 毫升才会出现。而正常情况下，大多数健康成年女性的例假期失血量仅有 20~60 毫升，是不足以造成贫血的。


另外，只要我们按照正常饮食的标准，吃肉并且每天都可以吃 2~3 种以上蔬菜的话，也几乎是不会出现贫血的。


那到底是什么让我们姨妈出走，肤色苍白无神呢？


是什么让我的姨妈出走了呢？


有关大姨妈的二三事


如果师姐现在问大家这样一个问题：「一个完整的例假周期有多久，分为哪几个周期？」


我想大多数同学都很难回答上来，而且多数人即便心中有答案，也存在着偏差。我们通常都会以为，一个例假周期仅仅只有 7 天。


但实际上，根据研究显示平均一个例假周期总计时长为 28 天，切分为四个阶段，分别为：


![img](https://pic3.zhimg.com/v2-719bb9436555de8d66ecf9c603d5b2b6.webp)

雌激素和孕激素水平很高。如果没有受精，荷尔蒙水平就会下降，月经周期会再次开始。


所以我们常说的那个「7 天例假」，仅仅是整个例假周期中的月经期。此外，我们所说的姨妈出走，很可能也只是正常情况。


根据研究和调查的结果来看，经期比预计时间推迟 1~7 天都是很正常的，不需要担心。而且我们可以看到一个完整的例假周期是 28 天，并不是根据自然月来的，因此短时间的推迟或提前都没有太大的问题。


那么如果真的是姨妈出走，身体出现异常，可能会受哪些因素影响呢？


例假都受什么影响


根据我们刚才对于例假周期的划分来看，其实例假的周期性变化，其实就是激素变化的一个反馈。


我们体内的激素本身就很容易因种种原因波动，例如：情绪波动（过度紧张、愤怒、抑郁）、过度劳累、压力过大、过度节食、过度运动、熬夜等，都有可能导致经期推迟，紧急避孕药也可能导致月经推迟。


其中最为常见的几个可能原因就是：


①在断糖期间爆碳水


这是因为，当我们在进行断糖饮食的情况下，血糖和胰岛素的水平都非常稳定。突然这个时候，吃很多的淀粉类食物或是甜食，会一下打破体内的宁静。血糖和胰岛素水平瞬间飙升，随后在几十分钟后，又会急速下降，仿佛坐上了过山车。


胰岛素作为体内激素平衡中的一环，当它出现大幅波动时，体内的荷尔蒙也会受到波及，最终导致例假推迟，较为严重的甚至有可能会出现长时间闭经的情况。


②吃太少


有很多同学，虽然名义上是在断糖，但又畏惧吃脂肪而长胖，所以干脆把碳水和脂肪都降得很低，几乎每天就靠饿着。减重效果是有了，但是身体开始受不了了。


由于缺少饮食中的必需营养元素，比如油脂中的维生素 A、D、E 以及 Omega-3 脂肪酸，身体机能开始下滑，激素水平也开始出现了大幅波动。这样的情况，最终也会让姨妈离家出走。


③压力大


有的同学可能听说过，压力胖和过劳肥，其实压力同样会影响我们的例假周期。这种看不见摸不着的情绪，影响着我们身体的各项机能。


这是由于我们体内存在一种名为皮质醇的压力调节激素，诸如：学习工作压力大、睡眠少、喝太多咖啡、运动过量都会增加这种激素的分泌。当它反过来作用到我们的激素调节中枢——下丘脑时，影响多种激素的分泌，最终造成姨妈出走。


④体脂太低


相信这个体脂低是每个同学都梦寐已久的，毕竟谁不想拥有一个 A4 腰附加马甲线呢。但是，其实当体脂过低的时候，也会把我们姨妈扫地出门。


实际上，脂肪细胞不仅仅是让我们变胖才存在的，它同样是一个内分泌器官，他能释放多种激素分子，影响着全身多种组织。


当我们体内的脂肪含量过低时，影响姨妈的如：黄体生成素、促卵泡素、催乳素以及雌激素等异常，最终也会出现姨妈离家出走的现象。


总而言之，以上的这些情况，都可以归结为一类，那就是「挑战生命」。当我们的生存状态受到挑战时，身体会调整到保命优先模式，关掉对生存没太大用处的生殖系统。换个角度来想一想，其实这可以称得上是大自然赐给女性的保护机制。


有什么方法可以找回出走的姨妈么？


那么如果我们的姨妈出走了，有没有什么切实可行的方法把它找回来呢？


下面师姐就来教大家，通过吃这些食物，找回出走的姨妈，恢复正常的生理功能。


找回姨妈吃什么？


1.姜汤


自古以来，姜汤在人们的印象中就是暖身的。其实，姜中的姜黄素对于调理女性生理周期也是有帮助的。


在过往的多项研究中，专家们发现，姜黄素可以刺激子宫和骨盆区域血液的流动，有助于找回我们出走的姨妈。此外，姜黄素还有缓解痉挛的作用，可以帮助子宫扩张，改善月经推迟的问题。


2.芹菜


我们通常对于芹菜的印象都是含有丰富的膳食纤维，但其实芹菜中的维生素 C 以及芹菜酚的含量也很丰富。


有研究显示芹菜酚这样的成分，有助于刺激子宫收缩，帮助月经正常排出，对调节例假周期很有益处。此外，丰富的维生素 C，还可以帮助我们有效降低体内炎症水平，间接帮我们找回姨妈。


3.富含维生素 C 的浆果和蔬菜


蓝莓、草莓、西蓝花、菠菜、红辣椒、青椒以及西红柿


一部分研究认为，维生素 C 可以在经期时段，帮助我们提升雌激素水平并降低孕激素水平。这样的激素变化有助于宫缩和子宫内膜的脱落，也就是加速月经的排出。


同时维生素 C 超强的抗氧化特性，还有助于帮助我们改善皮肤光泽，调节体内器官的炎症水平，降低慢性病的发生概率。


4.牛奶&鸡蛋


全脂牛奶和蛋黄中都含有丰富的脂溶性维生素，例如：维生素 A、D、E、K 等。其中维生素 D 在研究中被发现，与女性的例假周期有关。通过补充维生素 D 不仅可以调节经期，甚至还可以有效治疗多囊卵巢综合征导致的月经不调。


此外，维生素 D 还具有其他健康益处，其中包括：降低部分疾病的风险，帮助减轻体重和降低抑郁风险。


其实除了饮食上面的改变，生活习惯上的一些改变，也会对调整例假周期有帮助，下面就来看看师姐给大家总结的找回姨妈小妙招吧。


找回姨妈怎么做？


1. 定期运动


运动有很多健康益处，而且还可以帮助调理我们的例假周期。一方面，它有助于我们控制体重，另一方面还有助于降低压力激素水平，保证其它激素的动态平衡。


此外，最新的研究显示，通过运动的方式可以有效改善痛经的情况。那些每周做 3 次时长为 30 分钟有氧运动的女性，在经期时的疼痛有了明显减少。


保证充足睡眠


就像我们前面说到的，睡眠不足会导致压力激素——皮质醇水平的上升。而这些又会引起雌激素和孕激素受到抑制，这些是控制我们经期的重要激素。


缓解压力、保证充足睡眠，可以让这些激素处于正常释放的状态。很多时候，我们只需要调节我们的生活习惯就能解决大部分的问题。，


保持健康的体重


体重是影响我们例假周期一个非常重要的因素，与体重健康的女性相比，体重超重的女性更容易出现月经不调以及痛经情况。这是由于脂肪细胞，不仅会造成肥胖，还会对激素和胰岛素的影响。


反之，通过饮食上的管理，减轻体重，从而帮助调节例假周期，以促使我们的经期恢复正常。


其实，肤色和例假周期这两件事是相辅相成的。如果我们发现自己的皮肤出现了一些问题，那么很可能意味着我们的激素出现了波动，这也就意味着我们的例假可能会出现一些异常。反过来，例假的异常也会呈现出皮肤上的变化。


  



课程的结尾，建议大家：即便出现了例假周期异常或是皮肤苍白的情况，也不要过于焦虑或者紧张，先对自己的饮食和生活习惯的改变进行排查，找到引发的原因。


过度的焦虑和紧张反而会导致压力激素水平的上升，最终导致我们的姨妈迟迟不来。还有，千万不要信那些红糖水、红枣的说法，这些都是伪科学。它们不仅改善不了我们的例假问题，还可能会造成胰岛素水平升高，进一步推迟经期。


如果例假长期不规律的话，如：超过 2 个月姨妈都没有来，那么我们除了遵循师姐给大家的建议外，正确的做法应该是挂号看妇科，根据医生的要求检查 B 超和各项激素，找到问题对症下药。
