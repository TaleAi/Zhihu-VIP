## 简介
杰出人才能从纷繁的信息中提炼价值，复杂世界的运行需要我们升级思维模式。


「为什么我管理不好时间？」


「为什么读了那么多书却仍然过不好这一生？」


「为什么效率总是这么低？」


「为什么别人能想到好的方案，我却不行？」


我们的迷茫与焦虑，我们的困惑和不解，我们经历的种种失败，并不是我们本身的能力与别人有多么大的差距。而是我们的思维方式出了问题。


知乎上有很多问题是围绕：那些很厉害的人和我们的思维方式有什么不同？


格局高大上的人是怎么思考问题的？


很多时候我们不能突破自身的瓶颈或者碰到了职场天花板，是因为有限的思维能力和决断力。


杰出的人才总是能从纷繁的信息中提炼价值，而未经训练的思维模式往往很难做出正确的判断，而现在这个复杂世界的运行需要我们升级思维模式。


如果你想在职场上发现更多可能；


如果你想换一种方式打开人生；


如果你想对自我能力进行深一步的挖掘，不再自我设限；


如果你想提升脑力，升级学习工作方法，合理分配精力；


那你必须培养正确的高版本的思维模式。


由畅销书《反本能》、《暗理性》的作者、心理学大咖卫蓝老师，对 20 + 个经典的心理实验案例进行分析，手把手带你拆解 30 + 个提高效能思维模式，带你打破固有认知牢笼，重塑思维模式。让你更深刻精准地看待问题，加深对信息的理解，轻松应对生活问题。


学习这门课程将给你带来：


生活上，在不确定的环境中提前布局的能力；


工作中，能把控风险，掌握平衡，助你权衡利弊；


情绪上，心态不再容易崩盘，找准自我定位；


这是系统化的认知世界的思维方式，相信一定能缓解你的人生焦虑，打通人生跃迁之路。
