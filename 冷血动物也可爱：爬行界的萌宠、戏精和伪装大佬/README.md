## 简介
自然界中有哪些长相可怕的动物、有哪些长得比较逆天的动物？有哪些气质超群的动物、有哪些可爱却不常见的动物？如果你对奇特的动物感兴趣，


大自然在创造了无数美好事物的同时，也创造了许多千奇百怪的事物，其中以动物界和植物界居多，它们有些就生活在我们身边，有些我们可能一辈子也遇不上。


你知道有一种动物，它的乳头长在屁股上吗？


猫咪是深受人类喜爱的动物，当它没有毛时，它还依旧可爱吗？


有一种叫作巨蜥的蜥蜴，它其实只有 24 厘米！


还有一种生物，靠吃自己母亲的身体为生......


。这个世界上存在的所有生物，都是大自然带来的惊喜，大千世界，无奇不有，未知，更令人沉迷。就让我们一起来探索那些你看一眼就忘不掉的奇特动物吧。
