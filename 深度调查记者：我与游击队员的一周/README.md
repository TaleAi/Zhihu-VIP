# 深度调查记者：我与游击队员的一周

## 简介
一段颠覆你想象的战地采访，将带你深入了解「哥伦比亚革命武装力量－人民军」。


「电视机不知被谁关掉了，游击队员们起身离开，仿佛多一秒都不想停留。我也准备回棚屋休息，刚出会议室，突然听见有人大喊：快把灯灭了，天上有『啄木鸟』！一时间身边的游击队员都紧张极了。那是政府的战斗机，游击队员敏锐的耳朵让我们知道，此时危险正在逼近。那一瞬间，没有人说话，似乎也没有人思考……」


这不是一个悬疑的战地小说，也不是一个勇闯丛林的冒险故事。这是一份关于「哥武」的真实记录。


「哥武」，全称「哥伦比亚革命武装力量－人民军」，在大部分人的印象中，它是一个令人畏惧的恐怖组织：从 20 世纪 80 年代起参与毒品交易；绑架超 1 万人，其中包括政客、军人、商人和记者；组织多次暗杀、爆炸等恐怖事件。在「哥武」与哥伦比亚政府长达半个多世纪的内战中，造成了至少 22 万人死亡、数百万人流离失所。


美国、欧盟在内的多个国家和组织都将它列入恐怖组织名单。但是与此同时，「哥武」没有放弃与政府间的和平谈判，有时能换来短暂的和平，有时则换来新一轮的屠杀。


作为第一位深入丛林营地对「哥武」进行采访的亚洲记者，刘骁骞在这个专栏里，带着自身的思考，与我们分享了这段不同寻常的采访之旅。


「哥武」的游击队员都是冷血杀手吗？


平常的日子里他们都在做些什么？


对于战争、和平、未来，他们有着怎样的态度？


在这专栏里，让我们以一名记者之眼，重新了解一个真实的「哥武」。
