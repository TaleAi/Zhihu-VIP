## 简介
应该如何开发右脑？什么时候学习语言最合适？大脑生下来以后是一成不变的吗？如何提高记忆力？……本专栏特邀北师大脑科学教授薛贵，引导孩子与家长，正确认识脑科学，让家长从科学的角度去认识和理解孩子的行为，寻求隐藏在孩子行为背后的科学道理。


21 世纪必然是脑科学的世纪，这可能是人类面临最后的挑战。


心理学和脑科学的使命是为了理解大脑的工作原理，从而有助于我们发展更好的人工智能。


但同时，了解人脑的工作原理还需要更好地帮助我们发展更强的人的智能，提高整个社会的智力资本，和人工智能抗衡并维护人类的尊严。人工智能和人脑智能的巅峰对决，这样的精彩，现在的孩子，未来的希望，他们不应该只是一个观众。


但人脑的工作方式确实非常复杂，对大脑的认识才刚刚开始，而公众对脑科学又充满了渴求，这就带来了脑科学被误传的风险。社会上流传了很多所谓的「神经神话」，加上利益的驱使，不断加深了公众对大脑的误解。


在脑科学领域，家长们会经常问到关于儿童教育的问题：应该如何开发右脑？什么时候学习语言最合适？大脑生下来以后是一成不变的吗？如何提高记忆力？等等。


这样的问题背后，是公众对大脑存在相当大的误解。同时，也是因为现有的科学实在没有办法解答他们的很多疑问。


本专栏特邀北师大脑科学教授薛贵，引导孩子与家长，正确认识脑科学，让家长从科学的角度去认识和理解孩子的行为，寻求隐藏在孩子行为背后的科学道理。
