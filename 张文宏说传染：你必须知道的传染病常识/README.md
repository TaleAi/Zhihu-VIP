## 简介
新冠疫情的爆发再一次向人类敲响了防控传染病的警钟，而对我们每个人来说，了解传染病常识成为一种必须。上海华山医院感染科主任张文宏，将系统地介绍传染病知识，教会我们如何在生活中保护自我和家人的健康。


2020 年蔓延全球的新冠疫情，让全人类都措手不及。它让我们再一次意识到要对自然界与微生物界保持加倍的敬畏之心。科学防控传染病，也成为了一种生活新常态。 有哪些可能引发传染的病毒与细菌？我们身边潜伏着哪些常见传染病？新冠、流感、乙肝、埃博拉、鼠疫、艾滋病……虽然都叫传染病，又有哪些异同？新常态下，每个人都应该懂一点儿传染病常识，以更好地保护自己和家人。 


《张文宏说传染》不是一本短时效的防疫手册，而是一部提供系统性知识的科普著作。它是上海华山医院感染科主任张文宏首次出版的面向大众的科普书，采用简明易懂的语言，让没有医学背景的读者也能看得懂、信得过、用得上。 


《张文宏说传染》将 28 种常见传染病划分为 7 大类别，涉及不同的人体部位和生活场景，在「传染病的前世今生」「传染病知识小科普」「张爸敲黑板」等不同的板块中，娓娓道来每一种疾病的历史、特性和防护手段，解答我们的常见疑问。


传染病的历史，就是一部人类繁衍和生存的历史。历史上的每次大流行，都让我们倍感谦卑与警觉。在抗击传染病的过程中，我们始终要怀抱希望，相信随着科学不断发展，人类控制住传染病，只是时间问题。
