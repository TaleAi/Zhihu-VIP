你看不见黑暗，是因为有他们把黑暗挡在了你看不见的地方。


在风雨如磐的岁月里，是他，抱救中华之志，唤起民族的觉醒。


在白色恐怖的暗战下，是他，潜伏敌营 14 年，传递重要情报。


在碧海蓝天之上，是他，不惜以血肉之躯，将觊觎我国的飞机抵挡在外。


在 150 年不遇的洪涛中，是他们，靠顽强的意志，挽救百姓于灾难之中。


在炮火纷飞的战乱国家里，是他们，不畏牺牲，将同胞接回祖国。


有多少人，能让你发自内心的佩服和敬仰？


2021，看看他们，看懂影响中国的力量，了解中国近现代史、当代史中最有尊严的事件。


你便会知道，为什么我们中华民族的复兴大业一定能够实现。


1921-2021，「人间观察团」为你揭开百年历史中不为人知的细节。


为你细细讲述他们的传奇与无畏。


呈现中华气节，彰显中国精神。
